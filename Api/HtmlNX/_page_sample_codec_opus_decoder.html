Opus decoder sample program.

Overview

This sample plays audio while decoding Opus data using the Opus decoder. An audio renderer is used to play the audio.

File Structure

This sample program is located in Samples/Sources/Applications/CodecOpusDecoder/ .

Required Environment

You must be able to use audio output.

How to Use

When the sample program is executed, the BGM starts playing automatically. During playback, input from the keyboard or DebugPad can be used to control the playback position of the BGM being played.

Input Operation
A Button Start seeking to a different playback position.
Left/Right Buttons Control seek position (in units of samples).
Up/Down Buttons Control seek position (in units of OpusFrameSize).
L/R Buttons Control seek position (go to the beginning or end of the waveform).
âˆ’ Button/SELECT Display instructions.
START or SPACEBAR Exits the program.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

307 extern "C" void nnMain()
308 {
309 #ifdef USE_HARDWARE_DECODER
310  NNS_LOG("CodecOpusDecoder sample (using nn::codec::HardwareOpusDecoder)\n");
311 #else // USE_HARDWARE_DECODER
312  NNS_LOG("CodecOpusDecoder sample (using nn::codec::OpusDecoder)\n");
313 #endif // USE_HARDWARE_DECODER
314 
315  nn::mem::StandardAllocator allocator(g_HeapBuffer, sizeof(g_HeapBuffer));
316  nn::mem::StandardAllocator pcmBufferAllocator(g_PcmBuffer, sizeof(g_PcmBuffer));
317 
318  // Initialize the file system.
319  InitializeFileSystem();
320 
321  // Initialize HID.
322  InitializeHidDevices();
323 
324  // Display the instructions.
325  PrintUsage();
326 
327  // Load the Opus data.
328  const auto& opusFile = OpusFiles[OpusFileIndex];
329  const auto Filename = opusFile.filename;
330  nn::fs::FileHandle handle;
331  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::fs::OpenFile(&handle, Filename, nn::fs::OpenMode_Read));
332  int64_t size;
333  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::fs::GetFileSize(&size, handle));
334  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::fs::ReadFile(handle, 0, g_OpusData, static_cast<std::size_t>(size)));
335  nn::fs::CloseFile(handle);
336 
337  // Get the sample rate and the number of channels from the header.
338  nn::codec::OpusBasicInfo opusBasicInfo;
339  std::memcpy(&opusBasicInfo, g_OpusData, sizeof(opusBasicInfo));
340  NN_ABORT_UNLESS(opusBasicInfo.header.type == nn::codec::OpusInfoType_BasicInfo);
341  int sampleRate = opusBasicInfo.sampleRate;
342  int channelCount = opusBasicInfo.channelCount;
343  NN_ABORT_UNLESS(channelCount <= OpusChannelCountMax);
344 
345  // Initialize the Opus decoder.
346 #ifndef USE_HARDWARE_DECODER
347  nn::codec::OpusDecoder decoder;
348 #else // #ifndef USE_HARDWARE_DECODER
349  const int32_t noOption = 0; // Currently, there are no options that may be specified.
350  nn::codec::HardwareOpusDecoder decoder(noOption);
351 #endif // #ifndef USE_HARDWARE_DECODER
352  void* workBuffer = std::malloc(decoder.GetWorkBufferSize(sampleRate, channelCount));
353  NN_ABORT_UNLESS_NOT_NULL(workBuffer);
354  nn::codec::OpusResult opusResult = decoder.Initialize(sampleRate, channelCount, workBuffer, decoder.GetWorkBufferSize(sampleRate, channelCount));
355  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
356 
357  //Build the audio renderer.
358  nn::audio::AudioRendererParameter audioRendererParameter;
359  nn::audio::InitializeAudioRendererParameter(&audioRendererParameter);
360  audioRendererParameter.sampleRate = 48000;
361  audioRendererParameter.sampleCount = 240;
362  audioRendererParameter.mixBufferCount = 2;
363  audioRendererParameter.voiceCount = channelCount;
364  audioRendererParameter.sinkCount = 1;
365  NN_ABORT_UNLESS(nn::audio::IsValidAudioRendererParameter(audioRendererParameter));
366 
367  std::size_t audioRendererWorkBufferSize = nn::audio::GetAudioRendererWorkBufferSize(audioRendererParameter);
368  void* audioRendererWorkBuffer = allocator.Allocate(audioRendererWorkBufferSize, nn::os::MemoryPageSize);
369  NN_ABORT_UNLESS_NOT_NULL(audioRendererWorkBuffer);
370 
371  nn::audio::AudioRendererHandle audioRendererHandle;
372  nn::os::SystemEvent systemEvent;
373  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::OpenAudioRenderer(&audioRendererHandle, &systemEvent, audioRendererParameter, audioRendererWorkBuffer, audioRendererWorkBufferSize));
374 
375  std::size_t configBufferSize = nn::audio::GetAudioRendererConfigWorkBufferSize(audioRendererParameter);
376  void* configBuffer = allocator.Allocate(configBufferSize, nn::os::MemoryPageSize);
377  NN_ABORT_UNLESS_NOT_NULL(configBuffer);
378  nn::audio::AudioRendererConfig config;
379  nn::audio::InitializeAudioRendererConfig(&config, audioRendererParameter, configBuffer, configBufferSize);
380 
381  nn::audio::FinalMixType finalMix;
382  nn::audio::AcquireFinalMix(&config, &finalMix, 2);
383 
384  nn::audio::DeviceSinkType sink;
385  int8_t bus[2] = { 0, 1 };
386  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::AddDeviceSink(&config, &sink, &finalMix, bus, sizeof(bus) / sizeof(bus[0]), "MainAudioOut"));
387 
388  // Prepare a memory pool to maintain the sample data added to WaveBuffer.
389  nn::audio::MemoryPoolType pcmBufferMemoryPool;
390  NN_ABORT_UNLESS(nn::audio::AcquireMemoryPool(&config, &pcmBufferMemoryPool, g_PcmBuffer, sizeof(g_PcmBuffer)));
391  NN_ABORT_UNLESS(nn::audio::RequestAttachMemoryPool(&pcmBufferMemoryPool));
392 
393  nn::audio::VoiceType voice;
394  nn::audio::VoiceType::BehaviorOptions option;
395  option.isPlayedSampleCountResetAtLoopPoint = false;
396  option.isPitchAndSrcSkipped = true;
397  nn::audio::AcquireVoiceSlot(&config, &voice, sampleRate, channelCount, nn::audio::SampleFormat_PcmInt16, nn::audio::VoiceType::PriorityHighest, nullptr, 0, &option);
398  nn::audio::SetVoiceDestination(&config, &voice, &finalMix);
399  nn::audio::SetVoicePlayState(&voice, nn::audio::VoiceType::PlayState_Play);
400  if(channelCount == 1)
401  {
402  nn::audio::SetVoiceMixVolume(&voice, &finalMix, 1.0f, 0, 0);
403  nn::audio::SetVoiceMixVolume(&voice, &finalMix, 1.0f, 0, 1);
404  }
405  else
406  {
407  nn::audio::SetVoiceMixVolume(&voice, &finalMix, 1.0f, 0, 0);
408  nn::audio::SetVoiceMixVolume(&voice, &finalMix, 1.0f, 1, 1);
409  }
410 
411  // Prepare buffers for streaming playback.
412  // Pass in buffers initialized to 0, which get filled with the results of decoding the Opus data as part of the main loop.
413  nn::audio::WaveBuffer waveBuffer[BufferCount];
414  const std::size_t pcmBufferSize = OpusFrameSampleCount * channelCount * sizeof(int16_t);
415  void* pcmBuffer[BufferCount];
416  for (int i = 0; i < BufferCount; ++i)
417  {
418  pcmBuffer[i] = pcmBufferAllocator.Allocate(pcmBufferSize, nn::audio::BufferAlignSize);
419 
420  std::memset(pcmBuffer[i], 0, pcmBufferSize);
421 
422  waveBuffer[i].buffer = pcmBuffer[i];
423  waveBuffer[i].size = pcmBufferSize;
424  waveBuffer[i].startSampleOffset = 0;
425  waveBuffer[i].endSampleOffset = static_cast<int32_t>(pcmBufferSize / sizeof(int16_t) / channelCount);
426  waveBuffer[i].loop = false;
427  waveBuffer[i].isEndOfStream = false;
428  waveBuffer[i].pContext = nullptr;
429  waveBuffer[i].contextSize = 0;
430 
431  nn::audio::AppendWaveBuffer(&voice, &waveBuffer[i]);
432  }
433 
434  auto pcmBufferForPreRoll = static_cast<int16_t*>(pcmBufferAllocator.Allocate(pcmBufferSize, nn::audio::BufferAlignSize));
435 
436  // Start the audio renderer.
437  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::StartAudioRenderer(audioRendererHandle));
438 
439  // Get the Opus data itself in addition to the size of the Opus data.
440  unsigned char* p = reinterpret_cast<unsigned char*>(g_OpusData + opusBasicInfo.dataInfoOffset);
441  nn::codec::OpusInfoHeader dataInfoHeader;
442  std::memcpy(&dataInfoHeader, p, sizeof(dataInfoHeader));
443  NN_ABORT_UNLESS(dataInfoHeader.type == nn::codec::OpusInfoType_DataInfo);
444  std::size_t remainedCount = dataInfoHeader.size;
445  p += sizeof(dataInfoHeader);
446  NN_ABORT_UNLESS(opusBasicInfo.frameDataSize != 0, "This sample only supports CBR format opus file.");
447 
448  const auto opusDataSampleCount = static_cast<int>(remainedCount / opusBasicInfo.frameDataSize * OpusFrameSampleCount);
449 
450  int seekDestSamplePosition = opusDataSampleCount - 1;
451  int playbackSamplePosition = 0;
452  bool isSeekRequested = false;
453  for (;;)
454  {
455  systemEvent.Wait();
456  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::RequestUpdateAudioRenderer(audioRendererHandle, &config));
457 
458  nn::hid::NpadButtonSet npadButtonCurrent = {};
459  nn::hid::NpadButtonSet npadButtonDown = {};
460  nn::hid::AnalogStickState analogStickStateL = {};
461  nn::hid::AnalogStickState analogStickStateR = {};
462 
463  GetHidState(npadButtonCurrent, npadButtonDown, analogStickStateL, analogStickStateR);
464 
465  if(npadButtonCurrent.Test< ::nn::hid::NpadButton::Plus >())
466  {
467  break;
468  }
469 
470  if(npadButtonDown.Test< ::nn::hid::NpadButton::Minus >())
471  {
472  PrintUsage();
473  }
474 
475  bool isSeekPositionDisplayRequired = false;
476  if(npadButtonCurrent.Test< ::nn::hid::NpadButton::Left >())
477  {
478  isSeekPositionDisplayRequired = true;
479  --seekDestSamplePosition;
480  }
481 
482  if(npadButtonCurrent.Test< ::nn::hid::NpadButton::Right >())
483  {
484  isSeekPositionDisplayRequired = true;
485  ++seekDestSamplePosition;
486  }
487 
488  if(npadButtonCurrent.Test< ::nn::hid::NpadButton::Up >())
489  {
490  isSeekPositionDisplayRequired = true;
491  seekDestSamplePosition += OpusFrameSampleCount;
492  }
493 
494  if(npadButtonCurrent.Test< ::nn::hid::NpadButton::Down >())
495  {
496  isSeekPositionDisplayRequired = true;
497  seekDestSamplePosition -= OpusFrameSampleCount;
498  }
499 
500  if(npadButtonDown.Test< ::nn::hid::NpadButton::L >())
501  {
502  isSeekPositionDisplayRequired = true;
503  seekDestSamplePosition = 0;
504  }
505 
506  if(npadButtonDown.Test< ::nn::hid::NpadButton::R >())
507  {
508  isSeekPositionDisplayRequired = true;
509  seekDestSamplePosition = opusDataSampleCount - 1;
510  }
511 
512  seekDestSamplePosition = std::max(std::min(seekDestSamplePosition, static_cast<int>(opusDataSampleCount - 1)), 0);
513 
514  if(isSeekPositionDisplayRequired)
515  {
516  NNS_LOG("Seek Position: %d%s\n",
517  seekDestSamplePosition,
518  seekDestSamplePosition > opusFile.loopEnd ? " (position is after loopEnd, so playback will jump to loopStart immediately)" : ""
519  );
520  }
521 
522  if(npadButtonDown.Test< ::nn::hid::NpadButton::A >())
523  {
524  NNS_LOG("Seek to: %d\n", seekDestSamplePosition);
525  isSeekRequested = true;
526  }
527 
528  // If playback is complete for any buffers, start the Opus decoding process.
529  if (auto pReleasedWaveBuffer = nn::audio::GetReleasedWaveBuffer(&voice))
530  {
531  nn::audio::WaveBuffer* pWaveBuffer = nullptr;
532  int16_t* pPcmBuffer = nullptr;
533  for (int i = 0; i < BufferCount; ++i)
534  {
535  if (pReleasedWaveBuffer == &waveBuffer[i])
536  {
537  pWaveBuffer = &waveBuffer[i];
538  pPcmBuffer = static_cast<int16_t*>(pcmBuffer[i]);
539  break;
540  }
541  }
542 
543  // At the end of the Opus file, seek to loopStart.
544  if(remainedCount == 0)
545  {
546  seekDestSamplePosition = opusFile.loopStart + opusBasicInfo.preSkipSampleCount;
547  isSeekRequested = true;
548 
549  NNS_LOG("Seek to loopStart (Seek to: %d)\n", seekDestSamplePosition);
550  }
551 
552  if(isSeekRequested)
553  {
554  isSeekRequested = false;
555 
556  const auto seekDestFrameIndex = seekDestSamplePosition / OpusFrameSampleCount;
557 
558  int idleDecodeFrameCount;
559  if(seekDestFrameIndex < OpusPreRollFrameCount)
560  {
561  decoder.Reset();
562  idleDecodeFrameCount = seekDestFrameIndex;
563  }
564  else
565  {
566  idleDecodeFrameCount = opusFile.idleDecodeFrameCount;
567  if (idleDecodeFrameCount == 0)
568  {
569  // When the decoder is not emptied, resetting the decoder can make noise less likely to occur.
570  decoder.Reset();
571  }
572  if (idleDecodeFrameCount == 0 &&
573  (opusFile.loopStart % OpusFrameSampleCount < opusBasicInfo.preSkipSampleCount ||
574  opusFile.loopStart % OpusFrameSampleCount >= OpusFrameSampleCount - opusBasicInfo.preSkipSampleCount))
575  {
576  NNS_LOG("warning: loopStart position is in [0, %d) or [%d, %d) in opus frame,"
577  "so noise might be heard without idle decode\n",
578  opusBasicInfo.preSkipSampleCount, OpusFrameSampleCount - opusBasicInfo.preSkipSampleCount, OpusFrameSampleCount);
579  }
580  }
581 
582  const auto idleDecodeStartDataOffset = (seekDestFrameIndex - idleDecodeFrameCount) * opusBasicInfo.frameDataSize;
583 
584  // Update information such as the playback position to the jump destination.
585  p = reinterpret_cast<unsigned char*>(g_OpusData + opusBasicInfo.dataInfoOffset + sizeof(nn::codec::OpusInfoHeader) + idleDecodeStartDataOffset);
586  remainedCount = dataInfoHeader.size - idleDecodeStartDataOffset;
587  playbackSamplePosition = idleDecodeStartDataOffset / opusBasicInfo.frameDataSize * OpusFrameSampleCount;
588 
589  // Allow decoding to idle to adjust the decoder context.
590  for(int i = 0; i < idleDecodeFrameCount; ++i)
591  {
592  std::size_t consumed;
593  int sampleCount;
594  opusResult = decoder.DecodeInterleaved(&consumed, &sampleCount, pcmBufferForPreRoll, OpusFrameSampleCount * channelCount * sizeof(int16_t), p, remainedCount);
595  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
596  NN_ABORT_UNLESS(sampleCount == OpusFrameSampleCount);
597 
598  p += consumed;
599  remainedCount -= consumed;
600  playbackSamplePosition += sampleCount;
601  }
602 
603  // Decode the frame corresponding to the new seek position.
604  {
605  std::size_t consumed;
606  int sampleCount;
607  opusResult = decoder.DecodeInterleaved(&consumed, &sampleCount, pPcmBuffer, OpusFrameSampleCount * channelCount * sizeof(int16_t), p, remainedCount);
608  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
609  NN_ABORT_UNLESS(sampleCount == OpusFrameSampleCount);
610 
611  const auto seekDestSamplePositionInFrame = seekDestSamplePosition % OpusFrameSampleCount;
612 
613  p += consumed;
614  remainedCount -= consumed;
615 
616  pWaveBuffer->size = sampleCount * sizeof(int16_t) * channelCount;
617 
618  pWaveBuffer->startSampleOffset = seekDestSamplePositionInFrame;
619  pWaveBuffer->endSampleOffset = sampleCount;
620  pWaveBuffer->loop = false;
621  pWaveBuffer->isEndOfStream = false;
622 
623  nn::audio::AppendWaveBuffer(&voice, pWaveBuffer);
624 
625  playbackSamplePosition += sampleCount;
626  }
627  }
628  else
629  {
630  // Decode normally.
631  std::size_t consumed;
632  int sampleCount;
633  opusResult = decoder.DecodeInterleaved(&consumed, &sampleCount, pPcmBuffer, OpusFrameSampleCount * channelCount * sizeof(int16_t), p, remainedCount);
634  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
635  NN_ABORT_UNLESS(sampleCount == OpusFrameSampleCount);
636  p += consumed;
637  remainedCount -= consumed;
638 
639  pWaveBuffer->size = sampleCount * sizeof(int16_t) * channelCount;
640 
641  // If necessary, skip forward by an amount equal to the internal encoder delay.
642  if(playbackSamplePosition < opusBasicInfo.preSkipSampleCount)
643  {
644  pWaveBuffer->startSampleOffset = opusBasicInfo.preSkipSampleCount;
645  }
646  else
647  {
648  pWaveBuffer->startSampleOffset = 0;
649  }
650 
651  playbackSamplePosition += sampleCount;
652 
653  // At loopEnd, trim as necessary and then jump back to loopStart.
654  if(playbackSamplePosition >= opusFile.loopEnd + opusBasicInfo.preSkipSampleCount)
655  {
656  const auto trimSampleCount = playbackSamplePosition - (opusFile.loopEnd + opusBasicInfo.preSkipSampleCount);
657  pWaveBuffer->endSampleOffset = std::max(0, sampleCount - trimSampleCount);
658  remainedCount = 0;
659  }
660  else
661  {
662  pWaveBuffer->endSampleOffset = sampleCount;
663  }
664 
665  pWaveBuffer->loop = false;
666  pWaveBuffer->isEndOfStream = false;
667 
668  nn::audio::AppendWaveBuffer(&voice, pWaveBuffer);
669  }
670  }
671  }
672 
673  // Issue a request to detach the memory pools that were previously being used.
674  nn::audio::RequestDetachMemoryPool(&pcmBufferMemoryPool);
675  // Wait for the memory pools to be detached.
676  while (nn::audio::IsMemoryPoolAttached(&pcmBufferMemoryPool))
677  {
678  nn::audio::GetReleasedWaveBuffer(&voice);
679 
680  systemEvent.Wait();
681  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::RequestUpdateAudioRenderer(audioRendererHandle, &config));
682  }
683  // Release MemoryPool.
684  nn::audio::ReleaseMemoryPool(&config, &pcmBufferMemoryPool);
685  // Release the buffers.
686  for (int i = 0; i < BufferCount; ++i)
687  {
688  pcmBufferAllocator.Free(pcmBuffer[i]);
689  }
690  pcmBufferAllocator.Free(pcmBufferForPreRoll);
691 
692  // Finalize the audio renderer, and free the memory.
693  nn::audio::StopAudioRenderer(audioRendererHandle);
694  nn::audio::CloseAudioRenderer(audioRendererHandle);
695  nn::os::DestroySystemEvent(systemEvent.GetBase());
696  allocator.Free(audioRendererWorkBuffer);
697  allocator.Free(configBuffer);
698 
699  // Finalize the Opus decoder, and free the memory.
700  decoder.Finalize();
701  std::free(workBuffer);
702 
703  // Finalize the file system.
704  FinalizeFileSystem();
705 
706 } // NOLINT(readability/fn_size)

Sample Program Description

This sample program plays audio while decoding pre-encoded Opus data.

This sample program has the following flow.

  • Initialize the file system.
  • Load Opus data.
  • Initialize the Opus decoder.
  • Initialize and start the audio renderer.
  • Run the Opus decoder, and play the decoded results.
  • Finalize the audio renderer.
  • Finalize the Opus decoder.

This sample program uses nn::codec::OpusDecoder for the Opus decoder by default. If you define the USE_HARDWARE_DECODER macro and build it, the nn::codec::HardwareOpusDecoder will be used. The following description uses nn::codec::OpusDecoder as a representative example. If you are using , read the document with HardwareOpusDecoder replaced for OpusDecoder.

First, initialize the file system. Use a resource file that is converted during build. Appropriately change the location of the resource directory.

Load the necessary Opus data from a file. The file to be loaded can be selected by OpusFileIndex in the source code from the data available in the OpusFiles array.

  • OpusFileIndex == 0
    • Play the input Opus data from beginning to end in a looped manner.
    • Assume that prediction is enabled, and reset the decoder upon looping or start decoding from 3,840 samples earlier.
      • This sample assumes that one Opus frame equals 960 samples, so it starts decoding from four Opus frames earlier.
  • OpusFileIndex == 1
    • Play the input Opus data starting from the beginning. At loopEnd (which is not necessarily of end of the data), jump to loopStart (which is not necessarily the beginning of the data, and then play the segment from loopStart to loopEnd in a looped manner.
    • Assume that prediction is disabled, and start decoding from one Opus frame earlier upon looping.
  • OpusFileIndex == 2
    • Play the input Opus data starting from the beginning. At loopEnd (which is not necessarily of end of the data), jump to loopStart (which is not necessarily the beginning of the data, and then play the segment from loopStart to loopEnd in a looped manner.
    • Assume that prediction is disabled and that the position of loopStart will not be contained within the endpoints of the Opus frame during encoding, and continue playback without emptying the decoder upon reaching a loop.

For more information about how enabling and disabling prediction affects program behavior, see the reference documentation and Help for the OpusEncoder.exe converter tool.

The Opus decoder is initialized based on the header information.

You must pass the sample rate, number of channels, and a working buffer sized appropriately according to the previous two parameters to the nn::codec::OpusDecoder::Initialize() function.

Use the nn::codec::OpusDecoder::GetWorkBufferSize() function to get the working buffer size.

Get the audio renderer ready. For more information about audio renderers, see the API Reference and the Samples/Sources/Applications/AudioRenderer/ sample.

The main loop implements a streaming playback process. Several PCM buffers are allocated, and audio is played via nn::audio::VoiceType while sequentially decoding the Opus data in buffers for which playback is complete.

After all of the Opus data has been decoded and played, the sample automatically seeks to the beginning of the waveform and continues playback in a looped manner.

After a request to close the problem is received, the repeating process is ended, the audio renderer is stopped, the audio renderer and the Opus decoder are finalized, and the associated memory is freed.
