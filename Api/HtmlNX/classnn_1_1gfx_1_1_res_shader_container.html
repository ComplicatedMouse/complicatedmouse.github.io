Detailed Description

Class representing shader container resources.

Details
Maintains multiple shaders as shader container variations.

Examples
GfxPrimitiveRenderer/GfxPrimitiveRenderer.cpp, GfxSimple/GfxSimple.cpp, GfxSimpleCompute/GfxSimpleCompute.cpp, MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Template Parameters

Parameters

[in]pMemoryPoolA memory pool that holds a shader container.
[in]memoryPoolOffsetThe byte offset to the shader container in pMemoryPool.
[in]memoryPoolSizeThe size, in bytes, of the pool in pMemoryPool.
Precondition

  • pDevice is initialized.
Postcondition
  • The shader container is not initialized.

Details
When pMemoryPool == NULL, initialize using a memory pool with a shader container.
