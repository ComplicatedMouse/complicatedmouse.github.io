nn::os::MessageQueueType Struct Reference

This structure defines a message queue object. More...

#include <nn/os/os_MessageQueueTypes.h>

Detailed Description

This structure defines a message queue object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeMessageQueue()
  • FinalizeMessageQueue()
  • SendMessageQueue()
  • TrySendMessageQueue()
  • TimedSendMessageQueue()
  • JamMessageQueue()
  • TryJamMessageQueue()
  • TimedJamMessageQueue()
  • ReceiveMessageQueue()
  • TryReceiveMessageQueue()
  • TimedReceiveMessageQueue()
  • PeekMessageQueue()
  • TryPeekMessageQueue()
  • TimedPeekMessageQueue()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, MessageQueueType* messageQueue, MessageQueueWaitType waitType);

To use these functions, include nn/os.h.

Also see the description of the nn::os::MemoryPermission_ReadWrite utility class.

Examples
OsMessageQueue/OsMessageQueue.cpp, OsMultiWait/OsMultiWait.cpp, and OsMultiWaitUtility/OsMultiWaitUtility.cpp.
