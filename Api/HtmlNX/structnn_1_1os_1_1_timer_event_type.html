nn::os::TimerEventType Struct Reference

#include <nn/os/os_TimerEventTypes.h>

Detailed Description

This structure defines a timer event object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeTimerEvent()
  • FinalizeTimerEvent()
  • StartOneShotTimerEvent()
  • StartPeriodicTimerEvent()
  • StopTimerEvent()
  • WaitTimerEvent()
  • TryWaitTimerEvent()
  • SignalTimerEvent()
  • ClearTimerEvent()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, TimerEventType* timerEvent)

To use these functions, include nn/os.h.

Also see the description of the nn::os::TimerEvent utility class.
