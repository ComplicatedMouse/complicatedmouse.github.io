◆ Open()

NN_DETAIL_G3D_VIEWER_API ViewerResult nn::g3d::viewer::ViewerServer::Open ( )

Requests that communication be enabled between the server and 3DEditor.

Returns
Returns the execution result.
Return Values:

ViewerResult_Success - The process ended normally.

ViewerResult_ViewerDisabled - nn::g3d::viewer has been invalidated.

ViewerResult_ServerNotInitialized - The server is not yet initialized.

ViewerResult_ServerClosing - The sever is closing.

ViewerResult_UnknownError - Unknown error.

Details
To put the server in the state where communication is possible, you must call nn::g3d::viewer::ViewerServer::Poll() after you call this function. Use the nn::g3d::viewer::ViewerServer::IsOpened() function to determine whether communication is possible.

◆ Close()

NN_DETAIL_G3D_VIEWER_API void nn::g3d::viewer::ViewerServer::Close ( void  )

Requests a disconnection of communication between the server and 3DEditor.

Details
To put the server in the disconnected state, you must call nn::g3d::viewer::ViewerServer::Poll() after you call this function. Use the nn::g3d::viewer::ViewerServer::IsOpened() function to determine whether the server is disconnected.
