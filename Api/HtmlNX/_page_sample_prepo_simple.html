Describes the sample program for creating and saving play reports.

Overview

This sample program demonstrates how to use the play report library. For more information about play reports, see NintendoSDK Documents and the Play Report Guide.

File Structure

This sample program is located in Samples/Sources/Applications/PrepoSimple/ .

Required Environment

A user account associated with a Nintendo account is required for saving report data for that user. However, using the debugging features makes it possible to save report data without an associated Nintendo account. For more information, see NintendoSDK Documents.

How to Use

Nothing specific.

Important Information

A network connection is required for sending report data to the server.

Execution Procedure

Build the sample program and then run it.

Description

This sample program has the following flow.

  • Specify an event ID and a working buffer for an nn::prepo::PlayReport object.
  • Write keys and values to the report object.
  • Save the data.
