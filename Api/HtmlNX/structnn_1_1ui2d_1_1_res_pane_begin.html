Detailed Description

The data block that begins the definition for child panes.

Details
DataBlockKindPaneBegin is specified for blockHeader[kind].

Child panes are defined between ResPaneBegin and ResPaneEnd.

The ResPane object before ResPaneBegin is the parent pane.

  • ResPane : pane0
  • ResPaneBegin
  • ResPane : pane1
  • ResPane : pane2
  • ResPaneEnd

In this case, pane1 and pane2 are children of pane0.

Instead of a ResPane object, you can also specify a ResPicture, ResWindow, ResTextBox, ResBounding, or ResCapture object.
