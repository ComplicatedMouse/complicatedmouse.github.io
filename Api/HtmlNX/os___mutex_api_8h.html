Macro Definition Documentation

◆ NN_OS_MUTEX_INITIALIZER

#define NN_OS_MUTEX_INITIALIZER (   recursive)

Statically initializes an nn::os::MutexType object.

Parameters
[in]recursiveWhether to allow recursive locking in the mutex.

Details
Macros that are expanded in the list used to statically initialize nn::os::MutexType objects. For recursive, specify a bool type indicating whether a recursive lock is set. The lock level value is 0.

For more information about mutexes, see nn::os::InitializeMutex().

As shown in the following example, this macro can only be used for static initialization of nn::os::MutexType objects. Objects initialized in this way do not need to be explicitly destroyed by calling nn::os::FinalizeMutex.

Example:

nn::os::MutexType g_StaticMutex = NN_OS_MUTEX_INITIALIZER(false);
