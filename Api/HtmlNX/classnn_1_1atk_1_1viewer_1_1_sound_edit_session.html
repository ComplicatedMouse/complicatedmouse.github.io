Detailed Description

(Beta version.) The class for managing sound editing sessions.

Details
Periodically makes connection requests to SoundMaker when a session is opened.

When SoundMaker allows the connection, the project content on SoundMaker is replaced by the content of the SoundArchive object associated with the specified SoundArchiveEditor.

Initializes SoundEditSession.

Details
This function initializes memory used by SoundEditSession, and prepares to start the connection with SoundMaker.

Always call this function before using SoundEditSession.

buffer is the work buffer used for communications.

You must specify a size that is greater than the value returned by SoundEditSession::GetRequiredMemorySize().

size_t nn::atk::viewer::SoundEditSession::GetRequiredMemorySize ( const Configs &  configs) const

Gets the size of memory needed to use SoundEditSession.

Details
To use SoundEditSession, you must allocate a buffer of the size obtained by this function, and specify it for the buffer and bufferLength parameters of the Initialize() function.

At that time, in the configs parameter of the Initialize() function, specify the same Configs structure that was passed to the GetRequiredMemorySize() function.

◆ RegisterSoundArchiveEditor()

Result nn::atk::viewer::SoundEditSession::RegisterSoundArchiveEditor ( SoundArchiveEditor *  soundArchiveEditor)

Registers the specified SoundArchiveEditor.

Details
The specified SoundArchiveEditor becomes the target for connection with SoundMaker.

Note:

Currently, only one SoundArchiveEditor can be registered, and any attempt to register multiple objects fails.

There are plans to enable matching of the sound archive binary with the sound project in SoundMaker. When that happens, there will be support for multiple sound archives and it will be possible to register multiple objects.

Parameters
[in]soundArchiveEditorThe SoundArchiveEditor that is the edit target.

void nn::atk::viewer::SoundEditSession::Update ( )

Updates the sound editing session.

Details
While the session is open, connection requests are sent to SoundMaker in addition to queries about items edited in SoundMaker.

If a session is not open, this function does nothing even if it is called.
