Detailed Description

The structure for words registered in the user dictionary.

Details

  • This is used when you want to present your own words as conversion candidates.
  • When the string specified in reading is entered on the software keyboard, word is set as the predictive conversion result.
  • For example, if you register the Japanese word 任天堂, then specify にんてんどう for reading and 任天堂 for word.
  • If the user enters characters in katakana, they are converted to hiragana when the dictionary is queried. For this reason, use hiragana to register the readings for words.
  • Only UTF-16 is supported in the case of both notations and readings.
