Detailed Description

This file defines some classes that handle the model and material data, uniform updating, and drawing for all of the objects in the scene.

PBRObject::Mesh

This class holds the vertex and index buffers for a single mesh and handles updating the material uniforms for it.

PBRObject::Model

This class is a collection of meshes that make up one model. This class also handles setiing up/updating the uniforms related to postion/orientation.

PBRObject::Object

This class holds a reference to one model and handles drawing it. Models are unique, but there may be many Objects referencing a Model in the scene.
