Samples for NintendoSDK.

Introduction

This page describes NintendoSDK sample programs.

Samples

The sample code is located in Samples/Sources/Applications/.

Some samples are using the libnns* sample libraries, which bring together the processes that are shared among samples. The source code for those libnns* libraries can be found in the Samples/Sources/Libraries/ directory.

Source code from the samples and libnns* libraries can be copied and pasted into applications, but that material is designed to be easy to understand. There is some redundant code, and some of the parameters that are used might not be optimized. Check the content carefully.

When working with bugs, if you have any questions, the samples and the libnns* libraries will have the same level of support as ordinary libraries.

Attention
The samples and demo programs use image data and sound data that are (with some exceptions) the property of Nintendo. This data cannot be used outside of the samples and demo programs.
