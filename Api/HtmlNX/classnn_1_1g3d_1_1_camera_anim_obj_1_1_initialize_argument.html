Detailed Description

The use of internal features is prohibited. Parameters passed to the CameraAnimObj::Initialize() function for initialization.

Constructor & Destructor Documentation

◆ InitializeArgument()

nn::g3d::CameraAnimObj::InitializeArgument::InitializeArgument ( )

Constructor.

You must either call Reserve() on all camera animation resources used in the instance, or use SetMaxCurveCount() to specify the parameters required to calculate the buffer size.
