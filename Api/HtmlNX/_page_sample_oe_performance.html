A description of the process of selecting performance configurations using the application operating environment library.

Overview

This sample program describes how to select performance configurations using the application operating environment library.

For more information about using the application operating environment, see the Operating Environment Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OePerformance/ .

Required Environment

This sample program can only be built and run on the NX platform.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program updates the performance configurations to use in normal mode and in boost mode.

This sample program has the following flow.

  • Set the default performance combinations.
    • Set the default performance combination in normal mode.
      • To specify the performance mode, specify a nn::oe::PerformanceMode enumerated type value for the parameter.
      • To specify the performance configuration, specify a nn::oe::PerformanceConfiguration enumerated type value for the parameter.
      • Additions might be made to the nn::oe::PerformanceConfiguration type definitions in the future.
      • The nn::oe::SetPerformanceConfiguration function always succeeds.
    • Set the default performance combination in boost mode.
  • Set the GPU clock rate in normal mode to a value that is higher than the default performance value.
  • Set the GPU clock rate and the memory clock rate in boost mode to values that are lower than the default performance value.
  • Destroy all performance settings and restore the defaults when the program ends.
