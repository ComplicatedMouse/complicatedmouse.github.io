Detailed Description

The class for handling the resampler.

Details
The resampler performs frequency conversion for entered sample data.

The nn::audio::Resampler class is a utility class that wraps calls to functions that use nn::audio::ResamplerType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessResamplerBuffer() that take the nn::audio::ResamplerType type for their arguments.

Examples
AudioOutWithResampler/AudioOutWithResampler.cpp.
