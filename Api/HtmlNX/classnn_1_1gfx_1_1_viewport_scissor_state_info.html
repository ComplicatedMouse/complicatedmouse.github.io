◆ SetViewportStateInfoArray()

void nn::gfx::ViewportScissorStateInfo::SetViewportStateInfoArray ( const ViewportStateInfo *  pViewportStateInfoArray,
int  viewportStateCount 
)

Sets the array of viewport state information.

Parameters
[in]pViewportStateInfoArrayPointer to the array of viewport state information.
[in]viewportStateCountNumber of viewport states.
Precondition
  • viewportStateCount >= 0 && viewportStateCount <= 65535.
Postcondition
  • The viewport state information array and count are set.

Details
The content being pointed to can be destroyed after the object is initialized. When scissoring is enabled, the number of viewport states must match the number of scissor states.
