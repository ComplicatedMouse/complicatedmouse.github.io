nn::os::MutexType Struct Reference

#include <nn/os/os_MutexTypes.h>

Detailed Description

This structure defines a mutex object.

Details
Directly altering member variables is prohibited.
Use the main structure through the following functions.

  • InitializeMutex()
  • FinalizeMutex()
  • LockMutex()
  • TryLockMutex()
  • UnlockMutex()
  • IsMutexLockedByCurrentThread()
  • NN_OS_MUTEX_INITIALIZER()

To use these functions, include nn/os.h.

Also see the description of the nn::os::Mutex utility class.
