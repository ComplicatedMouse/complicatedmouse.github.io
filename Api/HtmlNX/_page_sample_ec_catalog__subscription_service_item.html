Getting Information About Subscription Service Items

Table of Contents

  • Overview
  • File Structure
  • Description

Describes the sample program for getting information about subscription service items (passes).

Overview

This sample displays information about subscription service items. This is the screen when Subscription service item list has been selected from the start screen. The names of the subscription service item courses are retrieved and displayed, together with course descriptions, disclaimers, and other information. Selecting a course redirects to a page where extensions, automatic renewal extensions, and pricing information for subscription service items are retrieved and displayed (see SubscriptionServiceItemScene.cpp). To get the price information, a separate function for that purpose is used. Selecting a subscription service item jumps to Nintendo eShop. The term can be displayed in Japan and other countries. In addition, legal language about automatically-recurring billing is displayed if automatic renewal information exists. If the item is an automatically renewed item that is free the first time (see note), it is spotlighted. The order of the items can be changed using item_tags. Note: The following conditions apply to automatically renewed items that are free the first time. The item can be purchased a maximum of one time (max_purchase_count == 1) and it is an item that is automatically renewed (automatic_renewal != null) and it is free (price == 0).

File Structure

This sample program is located in Samples/Sources/Applications/EcCatalog/SubscriptionServiceItem/ .

For a general overview of the process, see the description in Main.cpp. This description only details the shop jump functionality. The shop jump within this sample occurs using Main.cpp.
