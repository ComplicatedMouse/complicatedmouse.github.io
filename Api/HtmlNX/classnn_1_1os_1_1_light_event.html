Detailed Description

Class for handling lightweight events.

Details
Lightweight events are a synchronization mechanism for communicating flags between threads. Lightweight events do not support multi waits like nn::os::Event, but they operate faster.

The nn::os::LightEvent class uses nn::os::LightEventType objects. It is a utility class that wraps calls to synchronization control functions.

The member functions provided by this class are implemented by issuing APIs such as nn::os::WaitLightEvent(), that take the nn::os::LightEventType type as arguments.

◆ Wait()

void nn::os::LightEvent::Wait ( )

Waits until a lightweight event is in a signaled state.

Details
Waits until a lightweight event is in a signaled state.

If the clear mode for lightweight events is nn::os::EventClearMode_AutoClear, the target event is cleared automatically in conjunction with the waking of the thread by this function. If the clear mode for lightweight events is nn::os::EventClearMode_ManualClear, the target event is not cleared when this function wakes the thread.

For more information, see nn::os::WaitLightEvent().

◆ TryWait()

bool nn::os::LightEvent::TryWait ( )

Polls whether lightweight events are in the signaled state.

Returns
Returns whether polling of the lightweight event was successful.

Details
Polls whether a lightweight event is in a signaled state. Returns true if the lightweight event is in a signaled state, or false otherwise.

If the clear mode for lightweight events is nn::os::EventClearMode_AutoClear, the target event is cleared automatically in conjunction with the detection of the signaled state. If the clear mode for lightweight events is nn::os::EventClearMode_ManualClear, the target event is not cleared despite detection of the signaled state.

For more information, see nn::os::TryWaitLightEvent().

Waits for a lightweight event object to enter a signaled state, with a timeout. Returns true if the lightweight event enters the signaled state within the specified timeout, or false otherwise.

If the clear mode for lightweight events is nn::os::EventClearMode_AutoClear, the target event is cleared automatically in conjunction with the detection of the signaled state. If the clear mode for lightweight events is nn::os::EventClearMode_ManualClear, the target event is not cleared despite detection of the signaled state.
