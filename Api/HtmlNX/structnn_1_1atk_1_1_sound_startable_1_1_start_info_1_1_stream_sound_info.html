Detailed Description

A structure relating to a stream sound.

Details
This structure is passed when stream sounds are played.

Specify the path to a stream sound binary in externalPath to use that data for playback instead of the stream sound binary referenced by the stream sound set in the sound archive. If NULL is specified, the original stream sound binary code as set in the sound archive is used.
