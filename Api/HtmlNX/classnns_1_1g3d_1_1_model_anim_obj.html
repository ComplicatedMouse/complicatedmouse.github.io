Detailed Description

This class manages the model objects and animation objects.

Examples
G3dDemo/Edit.cpp, G3dDemo/NnsG3dSimple.cpp, G3dDemo/Outline.cpp, G3dDemo/Parallel.cpp, G3dDemo/Tessellation.cpp, and G3dDemo/Town.cpp.

◆ Calculate()

void nns::g3d::ModelAnimObj::Calculate ( )

Performs update processing for models and animations that do not affect GPU processing.

Details
This function is equivalent to calling the following functions. CalculateAnimations() CalculateWorld() CalculateBounding()

Note that calling this function after the animation is updated by the viewer library, overwrites the animation managed internally in the class.

Examples
G3dDemo/Edit.cpp, G3dDemo/NnsG3dSimple.cpp, G3dDemo/Outline.cpp, G3dDemo/Parallel.cpp, and G3dDemo/Tessellation.cpp.
