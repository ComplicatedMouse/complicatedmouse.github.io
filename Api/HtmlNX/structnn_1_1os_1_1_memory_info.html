Detailed Description

The structure for storing the obtained memory information.

Details
This structure is used by the nn::os::QueryMemoryInfo() function.

The following table describes the values stored in the member variables.

Member Variable Description
totalAvailableMemorySize The total amount of memory available to this program.
totalUsedMemorySize The total amount of memory being used by this program. Includes the total size of the physical memory assigned to the address area by totalMemoryHeapSize, programSize, and the virtual address memory management API.
totalMemoryHeapSize The total size of the memory heap. This size is the same size as allocated by nn::os::SetMemoryHeapSize().
allocatedMemoryHeapSize The amount allocated from the memory heap. This amount is the same as the sum of successful calls to the nn::os::AllocateMemoryBlock() function. It does not include memory freed by the nn::os::FreeMemoryBlock() function.
programSize The total amount being used by the program's code and data.
totalThreadStackSize The total stack size for the threads created by nn::os::CreateThread(). Does not include the stack size for threads destroyed by nn::os::DestroyThread(). Also does not include the stack size for the main thread.
threadCount The current number of threads in use. Includes the main thread.
Windows Platform Only:
  • totalAvailableMemorySize takes the amount of available physical memory.
  • A value of 0 is always assigned to programSize, making this member variable meaningless.
