Detailed Description

This file defines a version of the PBRObject classes used for frustum culling.

PBRObject::MeshCulled

This class adds a bounding box used to test against the camera frustum.

PBRObject::ModelCulled

This class adds the ability to draw indirectly with data from the culling pass.

PBRObject::ObjectCulled

This class adds the ability to draw indirectly with data from the culling pass.
