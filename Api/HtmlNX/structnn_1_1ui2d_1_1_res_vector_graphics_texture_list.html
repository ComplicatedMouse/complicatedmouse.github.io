Detailed Description

The data block that stores the list of vector graphics data reference information.

DataBlockKindVectorGraphicsDataList is specified for blockHeader[kind].

blockHeader[size] includes the data up to nameStrPool.

The following data structure uses this structure.

  • ResVectorGraphicsData vectorGraphicsData[dataCount]
  • uint8_t nameStrPool[]
