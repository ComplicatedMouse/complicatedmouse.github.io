Result Value Search

Searches the error information of API functions that return an instance of the nn::Result class and displays the details.

Enter a 32-bit hexadecimal value, which is the internal representation of an nn::Result object.

Usage

  1. Check the 32-bit internal representation of the nn::Result object.
    • You can get the 32-bit internal representation using the nn::Result::GetInnerValueForDebug() function.
    • You can also check the same value in the log when the nn::Result value is passed to the NN_ABORT_UNLESS_RESULT_SUCCESS() macro. If execution is stopped by the NN_ABORT_UNLESS_RESULT_SUCCESS() macro, a log entry like the following is output.
      Failed: result
      Module: 3
      Description: 8
      InnerValue: 0x00001003
      The InnerValue value above is the 32-bit internal representation.
  2. Input the 32-bit value into the text box at the top of this page.
  3. The search results are displayed.

Search Results

The search results include information on the Result value, and a table of Result values that can catch the located Result value. The Search Results section shows the information for the Result with the matching detailed value. Each field in the search results and table is described below.

  • Name
    • The name of the Result class.
  • Text
    • A description of the Result class.
  • Module
    • The module that the Result belongs to.
  • Description
    • The detailed value of the Result in the module.
  • Range
    • The range of detailed values that the Result catches.

Cautions

Will not work properly in Microsoft Help Viewer. Use a web browser.
