◆ NN_OS_ALIGNAS_GUARDED_STACK

#define NN_OS_ALIGNAS_GUARDED_STACK   NN_ALIGNAS(4096)

Macro to specify the alignment of a guarded stack space.

Details
Macro made to meet the alignment rules of a guarded stack space. The following usage aligns a buffer definition for the stack equivalent to the constant nn::os::GuardedStackAlignment.

Example:

NN_OS_ALIGNAS_GUARDED_STACK uint8_t threadStack[8192];
