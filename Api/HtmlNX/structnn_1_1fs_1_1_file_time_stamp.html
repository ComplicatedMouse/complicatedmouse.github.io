Detailed Description

The structure that defines file timestamp information.

Details
The file's creation date and the times it was last modified and last accessed are all defined in Unix time.

Member Data Documentation

◆ isLocalTime

bool nn::fs::FileTimeStamp::isLocalTime

If true, gets the local time, which includes any time zone difference.

If false, gets the Coordinated Universal Time (UTC), which does not include any time zone difference.
