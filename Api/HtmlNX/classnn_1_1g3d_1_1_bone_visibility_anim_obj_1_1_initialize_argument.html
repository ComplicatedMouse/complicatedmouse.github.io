Detailed Description

The use of internal features is prohibited. Parameters passed to the BoneVisibilityAnimObj::Initialize() function for initialization.

Constructor & Destructor Documentation

◆ InitializeArgument()

nn::g3d::BoneVisibilityAnimObj::InitializeArgument::InitializeArgument ( )

Constructor.

Details
For all bone visibility animation resources and all model resources used in the instance, you must either call Reserve() or specify the necessary parameters for calculating the buffer size using SetMaxBoneCount(), SetMaxMatCount(), SetMaxBoneAnimCount(), SetMaxMatAnimCount(), and SetMaxCurveCount().
