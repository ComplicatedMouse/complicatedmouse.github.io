nn::os::ReaderWriterLockType Struct Reference

#include <nn/os/os_ReaderWriterLockTypes.h>

Detailed Description

The structure for defining a readers-writer lock object.

Details
Directly altering member variables is prohibited.
Use the main structure through the following functions.

  • InitializeReaderWriterLock()
  • FinalizeReaderWriterLock()
  • AcquireReadLock()
  • TryAcquireReadLock()
  • ReleaseReadLock()
  • AcquireWriteLock()
  • TryAcquireWriteLock()
  • ReleaseWriteLock()
  • IsReadLockHeld()
  • IsWriteLockHeldByCurrentThread()

To use these functions, include nn/os.h.

Also see the description of the nn::os::ReaderWriterLock utility class.
