Detailed Description

Sink to output the rendering results to memory.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • AddCircularBufferSink()
  • RemoveCircularBufferSink()
  • GetRequiredBufferSizeForCircularBufferSink()
  • ReadCircularBufferSink()

To use these functions, include nn/audio.h.

Examples
AudioCpuRenderer/AudioCpuRenderer.cpp, and AudioSink/AudioSink.cpp.
