Detailed Description

Class for calculating a SHA-512 hash.

Details
This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state
  • Ready.
  • Complete.

◆ GetContext()

size_t nn::crypto::Sha512Generator::GetContext ( Sha512Context *  pContext) const

Gets the hash context, an intermediate state for hash calculations.

Parameters
[out]pContextPointer to the buffer storing the hash context.
Returns
Returns the number of bytes for input data for which there has not yet been any hash calculations.
Precondition
  • The instance is in a ready state.
Postcondition
  • The hash context is written to pContext.

Details
You can resume the hash computations by initializing using InitializeWithContext() with the obtained context.

Data that was entered using Update() but has not yet reached the size of a block is buffered in an unprocessed state until the block size is reached. Because the computation result does not consider this unprocessed data in the context, you must use either of the following methods to correctly restart the hash computation when this function returns a value other than 0.

  1. Enter data from a position rolled back by the size of the return value when the hash computation is restarted
  2. Save the data that can be obtained with GetBufferedData() together with the context, and enter that data first when the hash computation is restarted
