Detailed Description

(Beta version.) The canceller object that cancels a process on a timeout.

virtual bool nn::atk::viewer::TimeoutCanceller::IsCanceled ( void *  param)
virtual

Gets whether there is a cancellation.

Details
The process continues until either the target process ends or this function returns true.

Parameters
[in]paramSpecify the user parameter.
Returns
If the function returns true, the target process is canceled.

If the function returns false, the process continues.
