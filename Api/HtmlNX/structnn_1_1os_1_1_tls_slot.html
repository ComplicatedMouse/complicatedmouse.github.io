nn::os::TlsSlot Struct Reference

#include <nn/os/os_ThreadLocalStorageCommon.h>

Detailed Description

This type represents a TLS slot number.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • AllocateTlsSlot()
  • FreeTlsSlot()
  • GetTlsValue()
  • SetTlsValue()
  • GetUsedTlsSlotCount()

To use these functions, include nn/os.h.

Also see the description of the ThreadLocalStorage utility class.

Examples
OsThreadLocalStorage/OsThreadLocalStorage.cpp, and PerfSimple/PerfSimple.cpp.
