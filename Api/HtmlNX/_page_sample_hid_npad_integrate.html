Description of a sample program that uses multiple Npad features in tandem.

Overview

Describes how to run with multiple Npad features in tandem.

File Structure

This sample program is located in Samples/Sources/Applications/HidNpadIntegrate .

Required Environment

Either pair controllers to the development hardware or connect Joy-Con controllers in advance.

How to Use

Run the sample program and press any of the controller buttons. You switch the scene using the Plus + R Button or Plus + ZR Button combinations on the controllers. You can also switch using the Minus + L Button or Minus + ZL Button combinations or a left/right swiping operation.

Press both the ZR Button and the R Stick or the ZL Button and the L Stick to modify the Npad connection state. Switches to the play format modification screen, where the various settings can be configured. Press the ZR Button and R Stick, or the ZL Button and L Stick again to return to the original scene.

You can switch to six-axis sensor calculations, whenever you want, while displaying another scene. Press the following buttons to switch. Six-axis sensor: Right + L Stick or A Button + R Stick. Calculation processing is automatically enabled when switching to relevant scenes, however.

Press the Plus + R Stick or the Minus + L Stick to end the sample program. You can also end the sample program via Nintendo Target Manager.

Important Information

Make sure that the controller is sufficiently charged before using it.

Execution Procedure

Build the sample program and then run it.

Description

The sample program includes the following scenes.

  • Display pad input states.
  • Control the vibration motor.
  • Display the six-axis sensor states.
  • Screen for modifying Npad connection state.

For more information about scene switching, see How to Use earlier in this document.
