Detailed Description

template<class T, class NodeTraits>
class nn::util::IntrusiveDic< T, NodeTraits >

An intrusive dictionary.

Template Parameters
TThe element type. An element must either have an IntrusiveDicNode object as a member, or be a type that inherits IntrusiveDicBaseNode.
NodeTraitsThe type that indicates the characteristics of an element. If the element has an IntrusiveDicNode object as a member, you must specify IntrusiveDicMemberNodeTraits. If the element inherits IntrusiveDicBaseNode, you must specify IntrusiveDicBaseNodeTraits.
