Detailed Description

Structure representing vibration values.

Details
A vibration value is a set of amplitude and frequency values at a particular instant. It is represented as a set of four values: amplitude and frequency values for both low and high bandwidths.

Examples
HidVibrationRealTimeConverter/HidVibrationRealTimeConverter.cpp, and HidVibrationRollingBall/HidVibrationRollingBall.cpp.
