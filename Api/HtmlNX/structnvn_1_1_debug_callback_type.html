enum nvn::DebugCallbackType::Enum

Enumerated values for the type DebugCallbackType.

Enumerator
API_ERROR 

A debug callback is being issued because of an error in API usage.

API_WARNING 

A debug callback is being issued because of any warning in API usage.

TOOLS_ERROR 

A debug callback is being issued because of an error in tools usage.

TOOLS_WARNING 

A debug callback is being issued because of any warning in tools usage.

INITIALIZATION_ERROR 

A debug callback is being issued because of an error during driver initialization.

Note
To get notification of errors during the initialization phase a special debug callback must have been installed using nvnInstallGlobalDebugCallback.
