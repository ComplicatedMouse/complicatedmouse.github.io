Detailed Description

Class for handling semaphores.

Details
The semaphore feature uses a number counter to establish synchronization among threads.

The nn::os::Semaphore class uses nn::os::SemaphoreType objects. It is a utility class that wraps calls to synchronization control functions. The member functions provided by this class are implemented as functions that call functions like nn::os::AcquireSemaphore() that take the nn::os::SemaphoreType type for their arguments.
