Sets up the shader program.

Parameters
[in]pDevicePointer to the device.
Precondition
  • pDevice is initialized.

Details
This function is called indirectly from the ResShaderArchive::Setup() function. Do not call this function directly because shader programs might currently be shared within the library.

Delays shader initialization until the Update() function is called.
