◆ SetDefaults()

BufferBuilder & nvn::BufferBuilder::SetDefaults ( )

Set default state for the BufferBuilder object.

This method does not modify the device associated with the builder, which is programmed by SetDevice.

Note
This is semantically equivalent to calling
BufferBuilder::SetStorage(NULL, 0, 0);
