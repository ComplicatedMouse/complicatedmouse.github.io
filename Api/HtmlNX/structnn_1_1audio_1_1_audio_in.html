Detailed Description

Structure for managing an audio input.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • OpenDefaultAudioIn(AudioIn *pOutAudioIn, const AudioInParameter &parameter)
  • OpenAudioIn(AudioIn *pOutAudioIn, const char *name, const AudioInParameter &parameter)
  • CloseAudioIn()
  • StartAudioIn()
  • StopAudioIn()
  • GetAudioInState()
  • GetAudioInName()
  • GetAudioInSampleRate()
  • GetAudioInChannelCount()
  • GetAudioInSampleFormat()
  • AppendAudioInBuffer()
  • GetReleasedAudioInBuffer()

  • ContainsAudioInBuffer()

    To use these functions, include nn/audio.h.
