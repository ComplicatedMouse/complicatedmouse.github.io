Overview

This page describes a sample program that can be used to check the behavior of dead stripping.

When dead stripping is enabled, any functions that are not directly referenced inside the program itself or from an NRS file specified in the linker are removed. As a result, functions are removed unless they are called via the addresses obtained by the nn::ro::LookupSymbol() function, just like happens in this sample program. As a workaround, you can add symbols you want to keep to the base list provided by the SDK in ExportDynamicSymbol.lst. The symbols added to this list are not removed.

File Structure

This sample program is located in Samples/Sources/Applications/RoSimple/RoDeadStripStaticApplication .

Required Environment

This program does not run in a PC environment.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

First build Samples/Sources/Libraries/ro/RoDeadStripDynamicModule , and then build and run this sample program.
