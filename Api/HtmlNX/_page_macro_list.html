NintendoSDK API Reference  14.1.0
List of Macros

Lists the macros provided by the SDK.

Platform Information Macros

Macro Name Description
NN_BUILD_APISET_GENERIC Defined when using the general API set.
NN_BUILD_TARGET_PLATFORM_WIN Defined when the application operating platform is Windows.
NN_BUILD_TARGET_PLATFORM_OS_WIN Defined when the operating system of the application operating platform is Windows.
NN_BUILD_TARGET_PLATFORM_ADDRESS_32 Defined when the address of the application operation platform is 32-bit width.
NN_BUILD_TARGET_PLATFORM_ADDRESS_64 Defined when the address of the application operation platform is 64-bit width.
NN_BUILD_TARGET_PLATFORM_ENDIAN_BIG Defined when the endian of the application operating platform is big-endian.
NN_BUILD_TARGET_PLATFORM_ENDIAN_LITTLE Defined when the endian of the application operating platform is little-endian.

Platform Information Macros (NX)

Macro Name Description
NN_BUILD_APISET_NX Defined when using the NX API set.
NN_BUILD_TARGET_PLATFORM_NX Defined when the application operating platform is the NX development environment.
NN_BUILD_TARGET_PLATFORM_OS_NN Defined when the operating system of the application operating platform is an operating system used on the NX development hardware.

Assert

Macro Name Description
NN_ENABLE_ASSERT Enables assertions. It cannot be defined at the same time as NN_DISABLE_ASSERT.
NN_DISABLE_ASSERT Disables assertions. It cannot be defined at the same time as NN_ENABLE_ASSERT.
NN_ASSERT (condition, ...) Tests whether a condition is satisfied.
NN_ASSERT_EQUAL (lhs, rhs) Tests whether the values are equal.
NN_ASSERT_NOT_EQUAL (lhs, rhs) Tests whether the values are not equal.
NN_ASSERT_NULL (pointer) Tests whether the pointer is a null pointer.
NN_ASSERT_NOT_NULL (pointer) Tests whether the pointer is not a null pointer.
NN_ASSERT_POINTER (pointer) Test that it is a valid address for a pointer.
NN_ASSERT_LESS (lhs, rhs) Tests whether the left side is less than the right side.
NN_ASSERT_LESS_EQUAL (lhs, rhs) Tests whether the left side is less than or equal to the right side.
NN_ASSERT_GREATER (lhs, rhs) Tests whether the left side is greater than the right side.
NN_ASSERT_GREATER_EQUAL (lhs, rhs) Tests whether the left side is greater than or equal to the right side.
NN_ASSERT_STRING_EQUAL (lhs, rhs) Tests whether the content of the C strings is the same.
NN_ASSERT_STRING_NOT_EQUAL (lhs, rhs) Tests whether the content of the C strings is not the same.
NN_ASSERT_ALIGNED (pointer, align) Tests the alignment of the specified address and pointer.
NN_ASSERT_RANGE (value, begin, end) Tests whether the value is included within the range.
NN_ASSERT_MINMAX (value, min, max) Tests whether the value is greater than or equal to the minimum and less than or equal to the maximum.

Compile-Time Assert

Macro Name Description
NN_STATIC_ASSERT (condition) Tests whether a conditional expression is true at compile time.

Abort

Macro Name Description
NN_ABORT (...) Stops execution.
NN_ABORT_UNLESS (condition, ...) Tests whether a condition has been satisfied and stops execution if it has not.
NN_ABORT_UNLESS_EQUAL (lhs, rhs) Tests whether the values are equal and stops execution if they are not.
NN_ABORT_UNLESS_NOT_EQUAL (lhs, rhs) Tests whether values are equal and stops execution if they are not.
NN_ABORT_UNLESS_NULL (pointer) Tests whether the pointer is a null pointer and stops execution if it is not.
NN_ABORT_UNLESS_NOT_NULL (pointer) Tests whether the pointer is a null pointer and stops execution if it is.
NN_ABORT_UNLESS_POINTER (pointer) Test that it is a valid address for a pointer, and stops if it is not.
NN_ABORT_UNLESS_LESS (lhs, rhs) Tests whether the left side is less than the right side.
NN_ABORT_UNLESS_LESS_EQUAL (lhs, rhs) Tests whether the left side is less than or equal to the right side.
NN_ABORT_UNLESS_GREATER (lhs, rhs) Tests whether the left side is greater than the right side.
NN_ABORT_UNLESS_GREATER_EQUAL (lhs, rhs) Tests whether the left side is greater than or equal to the right side.
NN_ABORT_UNLESS_STRING_EQUAL (lhs, rhs) Tests whether the content of the C strings is the same.
NN_ABORT_UNLESS_STRING_NOT_EQUAL (lhs, rhs) Tests whether the content of the C strings is not the same.
NN_ABORT_UNLESS_ALIGNED (pointer, align) Tests the alignment of the specified address and pointer.
NN_ABORT_UNLESS_RANGE (value, begin, end) Tests whether the value is included within the range and stops execution if it is not.
NN_ABORT_UNLESS_MINMAX (value, min, max) Tests whether the value is greater than or equal to the minimum and less than or equal to maximum, and stops execution if it is not.
NN_ABORT_UNLESS_RESULT_SUCCESS (result) Tests whether the specified nn::Result is successful or not, and stops if it is not.

Log Output

Macro Name Description
NN_ENABLE_LOG Enables log output. It cannot be defined at the same time as NN_DISABLE_LOG.
NN_DISABLE_LOG Disables log output. It cannot be defined at the same time as NN_ENABLE_LOG.
NN_LOG (...) Adds the string specified in the parameter as a log entry.
NN_VLOG (format, args) Adds the string specified in the parameter as a log entry.
NN_PUT (text, length) Adds the string specified in the parameter as a log entry.
NN_STRUCTURED_LOG (module, severity, verbosity, ...) Adds the string specified in the parameter as a log entry.
NN_STRUCTURED_VLOG (module, severity, verbosity, format, args) Adds the string specified in the parameter as a log entry.
NN_STRUCTURED_PUT (module, severity, verbosity, text, length) Adds the string specified in the parameter as a log entry.

Break

Macro Name Description
NN_DIAG_BREAK () Inserts a breakpoint command at the locations where it is called.

Operating System Library Features

Macro Name Description
NN_OS_ALIGNAS_THREAD_STACK Macro to specify the stack space alignment.
NN_OS_ALIGNAS_FIBER_STACK Macro to specify the alignment of a fiber stack space without guards.
NN_OS_ALIGNAS_GUARDED_STACK Macro to specify the alignment of a guarded stack space.
NN_OS_MUTEX_INITIALIZER (recursive) Statically initializes an nn::os::MutexType object.
NN_OS_CONDITION_VARIABLE_INITIALIZER () Statically initializes a nn::os::ConditionVariableType object.

HTC and HTCS Library Features

Macro Name Description
NN_ENABLE_HTC Enables the HTC and HTCS libraries. It cannot be defined at the same time as NN_DISABLE_HTC.
NN_DISABLE_HTC Disables the HTC and HTCS libraries. It cannot be defined at the same time as NN_ENABLE_HTC.

Features for the spy Library

Macro Name Description
NN_ENABLE_SPY Enables the spy library when defined within spy_Config.h. Defined for debug and develop builds by default. The htc library must be enabled when enabling the spy library with this macro's definition.
NN_ENABLE_SPY Defined when the spy library is enabled.
