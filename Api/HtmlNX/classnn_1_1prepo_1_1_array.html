Detailed Description

Class for handling arrays.

Details
All values added to the array must be of the same type.

When adding nn::prepo::Struct, match the keys, value type, number and order of key values to nn::prepo::Struct.

There are no restrictions on the total number of values that can be added to an array. However, the total size of the report must be less than or equal to nn::prepo::ReportBufferSizeMax.

void nn::prepo::Array::SetBuffer ( void *  buffer,
size_t  size 
)

Sets a buffer.

Parameters
[in]bufferThe buffer.
[in]sizeThe buffer size.
Precondition
  • buffer != nullptr.
  • size > 0.

Details
An array is stored in the buffer that is set by this function. The buffer must not be freed until after this object has been added to the PlayReport class.

Use the following formula to calculate the optimal size of the buffer to set with this function.

  • (the size of the value) x (the number of elements)

Calculate the size of the value using the following methods.

  • String
    • StringBufferSizeMin + (the size of the string)
  • Structure
    • StructBufferSizeMin + (the size of the structure)
  • Other
    • PrimitiveValueSizeMax

For more information about calculating the sizes of structures, see Struct::SetBuffer().
