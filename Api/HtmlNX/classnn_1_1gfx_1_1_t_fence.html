◆ Sync()

template<typename TTarget >
SyncResult nn::gfx::TFence< TTarget >::Sync ( TimeSpan  timeout)

Synchronizes fences to a signaled state.

Parameters
[in]timeoutMaximum length of time to wait for synchronization.
Returns
Returns the synchronous processing result.
Return Values:

SyncResult_Success Succeeded.

SyncResult_TimeoutExpired The time specified by timeout passed before the fence went into a signaled state.

Precondition
  • The fence is set in the queue.

Details
Control is not returned until either the fence goes into a signaled state or the specified time has passed.
