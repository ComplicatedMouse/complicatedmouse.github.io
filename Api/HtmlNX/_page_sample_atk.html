Audio Toolkit Library Samples

Describes the audio toolkit library samples.

Introduction

Describes the audio toolkit (atk) library samples. The audio toolkit library features reside in the nn::atk namespace.

Samples

Sample code is located in the directories starting with the prefix Atk under Samples/Sources/Applications/. The following table lists the samples.

Page Content
Sample for Playing Sounds Using an Additional Sound Archive Describes a sample program showing how to play sounds using an additional sound archive in the nn::atk library.
Applying Effects to Playback Sample A sample program for the effect features of the nn::atk library.
Using Filters for Playback Sample This sample program shows how to use the filter feature of the nn::atk library.
Group Loading Playback Description of the sample program for features relating to loading sound data using nn::atk library groups.
Using the Mix Volume Settings for Playback Sample This sample program shows how to use the mix volume settings of the nn::atk library.
Using Sound Profile Describes the sample program demonstrating the Sound Profile feature of the nn::atk library.
Simple Audio Playback Sample Sample program for the basic features of the nn::atk library.
Sound3D Playback Sample This sample program shows how to use the Sound3D feature of the nn::atk library.
Playing Sounds Using Sound Actors Sample This sample program shows how to use the sound actor feature of the nn::atk library.
Playing Stream Sounds This is a description of the sample program that uses stream sound in the nn::atk library to play sound.
Playback with a defined submix structure. Describes a sample program for playing sounds with a user-defined submix structure using the nn::atk library.
In-Game Editing Sample This sample program shows how to use the in-game editing feature of the nn::atk library.
