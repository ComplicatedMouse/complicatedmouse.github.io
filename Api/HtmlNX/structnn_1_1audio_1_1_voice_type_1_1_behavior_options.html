bool nn::audio::VoiceType::BehaviorOptions::isPitchAndSrcSkipped

Skips pitch conversion and sampling rate conversion (SRC).

When this option is enabled, the sampling rate specified in the argument of nn::audio::AcquireVoiceSlot() must be the same as the sampling rate set in the VoiceType connection target.
