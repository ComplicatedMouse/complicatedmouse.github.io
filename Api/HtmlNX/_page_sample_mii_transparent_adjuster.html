This sample program shows a feature to prevent transparent pixel color bleeding due to linear interpolation.

Overview

This sample program lets you use the Face Library's feature to prevent color bleeding to compare the extent to which bleeding occurs when an image is enlarged.

File Structure

This sample program is located in Samples/Sources/Applications/MiiTransparentAdjuster/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This program creates a Mii icon and lets you compare how the image differs when nn::mii::TransparentAdjuster is and is not used.

Conversion Process

  • /Samples/Sources/Applications/MiiTransparentAdjuster/Resources/SampleShader.glsl

A simple shader for rendering textures..

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Load the Mii resource file.
  • Initialize the Mii resource (nn::mii::Resource).
  • Get nn::mii::CharInfo.
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Initialize the texture to which the icon will be written.
  • Initialize the texture for storing the result.
  • Initialize the texture for the depth stencil.
  • Initialize the shader for the icon texture (nn::mii::IconShader).
  • Initialize nn::mii::IconGpuBuffer.
  • Create a sampler for the Mii.
  • Create the texture view and register it to the descriptor pool.
  • Use nn::mii::TextureShader to render the Faceline and Mask textures.
  • Use nn::mii::IconShader to render the Mii icon.
  • Initialize nn::mii::TransparentAdjuster and set the parameters.
  • Initialize nn::mii::TransparentAdjusterGpuBuffer and set the parameters.
  • Use nn::mii::TransparentAdjuster to output the resulting texture from the Mii icon image.
  • Show the Mii icon texture to the left on the screen, and the resulting texture to the right.
