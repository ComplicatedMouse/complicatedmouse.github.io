)

Initializes.

Details
The user passes the work memory that AudioSpyModule will use. Set the parameters needed for initialization to the InitializeArg structure, and get the required size of memory using GetRequiredMemorySize().

Parameters
[in]argsThe initialization parameters.
[in]bufferThe pointer to the work memory that AudioSpyModule will use.
[in]bufferSizeThe size of the work memory.
Returns
Returns true if successful, or false otherwise.
Precondition
  • buffer is aligned to nn::DefaultAlignment.
  • bufferSize >= GetRequiredMemorySize()
  • IsInitialized() == false.
  • IsRegistered() == false.
  • args.pAudioRendererParameter != nullptr.

◆ PushWaveform()

bool nn::spy::audio::AudioSpyModule::PushWaveform ( const void *  buffer,
size_t  bufferSize,
nn::os::Tick  tick 
)

Sends the waveform data.

Details
Sends the waveform data obtained by nn::audio::ReadCircularBufferSink().

The tick parameter takes the tick when the nn::audio::RequestUpdateAudioRenderer() function called before nn::audio::ReadCircularBufferSink() completed its processing.

Parameters
[in]bufferPointer to the buffer storing the sample waveform data.
[in]bufferSizeThe size of the sample data stored in the buffer.
[in]tickThe tick when nn::audio::RequestUpdateAudioRenderer() completed its process.
Returns
Returns true if successful, or false otherwise.
Precondition
  • tick <= nn::os::GetSystemTick().
  • tick must be greater than the value it was when called the last time.
  • IsInitialized() == true.
  • The parameters for sending waveform data are specified by Initialize().
See also
Initialize

When a data request comes from Spy.exe, the presence or absence of a data request is notified to each DataInfo object.
