TMemoryPool< Target > *  pMemoryPool,

size_t  memoryPoolSize 
)

Initializes a buffer.

Parameters
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
[in]pMemoryPoolPointer to the memory pool in which to place the buffer.
[in]memoryPoolOffsetOffset, in bytes, to the memory pool where the buffer is placed.
[in]memoryPoolSizeSize, in bytes, of the memory pool for the buffer to use.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The buffer is not initialized.
  • If IsMemoryPoolRequired is true, pMemoryPool != NULL.
  • If IsMemoryPoolRequired is true, memoryPoolSize is larger than the size set by info.
Postcondition
  • The buffer is initialized.

Details
If IsMemoryPoolRequired is false and pMemoryPool is specified, the content of the specified memory pool is used as the default.
