◆ NN_PERF_INITIALIZE_METER

#define NN_PERF_INITIALIZE_METER (   pDevice,
  loadMeterCenterInfo,
  pMemory,
  memorySize,
  pMemoryPool,
  memoryPoolOffset,
  memoryPoolSize 
)    nn::perf::LoadMeterCenter::Initialize( pDevice, loadMeterCenterInfo, pMemory, memorySize, pMemoryPool, memoryPoolOffset, memoryPoolSize )

Initializes LoadMeterCenter.

Parameters
[in]pDevicePointer to the device.
[in]loadMeterCenterInfoInformation for initialization purposes.
[in]pMemoryPointer to the initialization buffer.
[in]memorySizeBuffer size, in bytes.
[in]pMemoryPoolPointer to the memory pool to which the GPU timestamp is written.
[in]memoryPoolOffsetOffset to the memory pool to which the GPU timestamp is written.
[in]memoryPoolSizeSize of the memory pool to which the GPU timestamp is written.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The number of sections set in info, for GpuMeter to measure, is a value of 1 or greater.

Details
Creates the following meters.

  • CpuMeter objects for frame metrics: 1.
  • CpuMeter objects for core metrics: The number specified in LoadMeterCenterInfo.
  • CpuMeter objects for user metrics: The number specified in LoadMeterCenterInfo.
  • GpuMeter objects for GPU metrics: 1. All of the other CpuMeter objects and GpuMeter are attached to the CpuMeter object for frame metrics.
Examples
FrameworkDemo/Main.cpp, GfxPrimitiveRenderer/GfxPrimitiveRenderer.cpp, and PerfSimple/PerfSimple.cpp.

Use frameMeter to record the frame end time.

Call this function when the frame ends.

Details
Performs the following operations.

  • Finishes measuring the frame time.

◆ NN_PERF_END_FRAME_NO_CLEAR_PROFILE

#define NN_PERF_END_FRAME_NO_CLEAR_PROFILE ( )    nn::perf::LoadMeterCenter::EndFrameMeasure(false)

Use frameMeter to record the frame end time.

Call this function when the frame ends.

Details
Prior profiling results are not cleared. The results are accumulated. If you want to clear the profiling results, use the NN_PERF_END_FRAME() function, which acts internally to clear the results, or call the NN_PERF_CLEAR() function anywhere. Performs the following operations.

  • Finishes measuring the frame time.
  • Switches the tallies and the buffers for the measurement results for each CpuMeter object and GpuMeter.
  • Tallies the measurement results using LoadMeterCenter.
