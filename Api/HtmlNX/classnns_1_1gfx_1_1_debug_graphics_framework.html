Detailed Description

The graphics framework class for debugging.

Details
Inherits GraphicsFramework, but has the following additional features. A feature to get nns::gfx::PrimitiveRenderer::Renderer A utility function to initialize nns::gfx::PrimitiveRenderer::Renderer A feature to get nn::gfx::util::DebugFontTextWriter Automatic measurement of performance and rendering of performance meters via the nn::perf::LoadMeterCenter class

Examples
G3dDemo/Parallel.cpp, G3dDemo/ShapeAnimation.cpp, G3dDemo/SkeletalAnimation.cpp, G3dDemo/Tessellation.cpp, G3dDemo/Town.cpp, HidVibrationGenerator/HidVibrationGenerator.cpp, HidVibrationPlayer/HidVibrationPlayer.cpp, and ImguiSimple/ImguiSimple.cpp.

◆ FrameworkMode

enum nns::gfx::GraphicsFramework::FrameworkMode
inherited

The corresponding framework mode, using this class.

Enumerator
FrameworkMode_DeferredExecution 

Mode that submits a command prior to Vsync, and then starts GPU processing after the submission and after waiting for Vsync.


The behavior is different for generic builds and for NX builds.
The command buffer multiple buffering count must be at least 2.

FrameworkMode_DeferredSubmission 

Mode that submits a command after Vsync, and then starts GPU processing after the submission.


Mode that enables easier compatibility with GL.
The command buffer multiple buffering count must be at least 2.

FrameworkMode_Immediate 

Mode where command generation, submission, and execution all occur within a single frame.


The GPU idles between command generation and submission, even though display delays are minimized.
The command buffer multiple buffering count does not need to be at least 2.

◆ InitializePerf() [2/2]

void nns::gfx::DebugGraphicsFramework::InitializePerf ( )

Initializes the measurement features.

Details
The CPU and GPU loads are measured automatically by ProcessFrame(), and the performance meters are rendered automatically by EndFrame(). You must use nn::os::SetThreadCoreMask() to assign the thread performing the metrics to a specific core for the measurement interval. A singleton of nn::perf::LoadMeterCenter is being initialized, so an error is generated when the application acts to initialize nn::perf::LoadMeterCenter. Initializes using LoadMeterCenterInfo with SetCoreCount(1), SetCpuSectionCountMax(256), SetGpuSectionCountMax(256), and the most appropriate BufferCount set for FrameworkMode. If FrameworkMode has changed, call FinalizePerf() and InitializePerf() to rebuild the measurement features.

Details
This function does the following.

  • If enabledCopyImage is true, it adds a command to copy the content of the color buffer to the scan buffer.
