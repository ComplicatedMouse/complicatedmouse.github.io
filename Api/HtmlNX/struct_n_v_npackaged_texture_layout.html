Detailed Description

Data structure describing the organization of packaged texture data in memory.

This structure specifies the memory layout of packaged texture data produced by the NVN texture packager. By default, the driver and texture packager select an optimal layout for a texture based on its size and format. This structure allows applications to provide the driver with a layout descriptor produced by the NVN texture packager so that old packaged texture assets can continue to be used even if the default layout changes in a new driver.
