Detailed Description

Structure representing an audio input buffer.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • SetAudioInBufferInfo()
  • GetAudioInBufferDataPointer()
  • GetAudioInBufferDataSize()
  • AppendAudioInBuffer()
  • GetReleasedAudioInBuffer()
  • ContainsAudioInBuffer()

To use these functions, include nn/audio.h.
