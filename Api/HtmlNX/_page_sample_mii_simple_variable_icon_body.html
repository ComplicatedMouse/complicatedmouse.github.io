This sample program renders Mii icons with certain body physiques.

Overview

This sample renders a Mii icon to reflect the height and physique set for the Mii. The program demonstrates how to use nn::mii::VariableIconBody and nn::mii::VariableIconBodyShader to create a Mii icon with a body that reflects the physical settings configured for the Mii.

File Structure

This sample program is located in Samples/Sources/Applications/MiiSimpleVariableIconBody/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

By changing the USR_SRGB_GAMMA_RESOURCE and USE_SRGB_SWAP_CHAIN_FORMAT macros you can switch the gamma setting and the SwapChain format for displaying. Switch the settings in a way that matches the gamma used in your application and confirm the results.

Execution Procedure

Build the sample program and then run it.

Description

This sample program creates and displays a Mii icon with a body.

Conversion Process

  • /Samples/Sources/Applications/MiiSimpleVariableIconBody/Resources/SampleShader.glsl

A simple shader for rendering icons.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Load the Mii resource file.
  • Initialize the Mii resource (nn::mii::Resource).
  • Get nn::mii::CharInfo.
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Initialize the texture to which the icon will be written.
  • Initialize the texture for the depth stencil.
  • Initialize the shader for the icon with a certain body physique texture (nn::mii::VariableIconBodyShader).
  • Create nn::mii::VariableIconBodyGpuBuffer.
  • Create nn::mii::VariableIconBody.
  • Use nn::mii::TextureShader to render the Faceline and Mask textures.
  • Use nn::mii::VariableIconBodyShader to render the Mii icon with certain body physique.
  • Render the texture for the Mii icon with certain body physique.
