◆ Draw() [1/4]

void nns::g3d::RenderModelObj::Draw ( nn::gfx::CommandBuffer *  pCommandBuffer,
int  passIndex,
int  viewIndex,
int  bufferIndex,
DrawCullingCallback  pDrawCullingCallback 
) const

Draws a model.

Parameters
[in]pCommandBufferThe pointer to nn::gfx::CommandBuffer.
[in]passIndexRendering pass index.
[in]viewIndexThe view index.
[in]bufferIndexThe buffer index.
[in]pDrawCullingCallbackPointer to the function that determines whether to render.

Details
Draws a model. passIndex must be within the range specified when generating RenderModel. viewIndex must be within the range of the view count of the model. bufferIndex must be an index that is valid for skeletons, shapes, materials, and all uniform blocks set by nns::g3d::RenderUnitObj::BindUniformBlock(). Specifically, specifying 1 for bufferIndex when only one buffer is specified by nns::g3d::RenderUnitObj::BindUniformBlock(). View frustum culling is performed when SetViewVolume() is used to set nn::g3d::ViewVolume. When SetCalculateLodLevelFunctor() is used to set nn::g3d::ICalculateLodLevelFunctor, the set function object is called to determine the rendering LOD level. A model with LOD level 0 will be rendered if this is not set. If SetDrawLodEnabled() is used to specify a rendering LOD level, rendering will be performed at that level. If SetDrawShapeEnabled() is used to specify a shape to render, only that shape will be rendered.

Examples
G3dDemo/Edit.cpp, G3dDemo/NnsG3dSimple.cpp, G3dDemo/Outline.cpp, G3dDemo/Parallel.cpp, G3dDemo/ShapeAnimation.cpp, G3dDemo/Tessellation.cpp, and G3dDemo/Town.cpp.
