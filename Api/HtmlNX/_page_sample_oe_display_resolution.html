Describes the process of getting the default display resolution using the application operating environment library.

Overview

This sample program describes how to get the default display resolution using the application operating environment library.

For more information about using the application operating environment, see the Operating Environment Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OeDisplayResolution/ .

Required Environment

This sample program can only be built and run on the NX platform.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program gets the default display resolution and outputs it to a log. The default display resolution changes under the following circumstances.

  • The device is attached or detached from the SDEV cradle or the Switch dock.
  • The system's operating mode is TV mode and the resolution is changed in the settings screen of DevMenu or HOME Menu.

This sample program has the following flow.

  • Initialize the application operating environment library.
  • Get the current default display resolution.
  • Get the system event when the default display resolution is changed.
  • Wait for the default display resolution to change.
  • Get the current default display resolution.
