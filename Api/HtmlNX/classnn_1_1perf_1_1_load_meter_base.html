◆ Next()

virtual void nn::perf::LoadMeterBase::Next ( )
virtual

Switches the buffer where the measurement results from the load meter and its attached LoadMeterBase are saved.

Call this function at the end of every frame.

Details
The Next() function of the attached LoadMeterBase is called recursively, so only call the Next() of the parent load meter. You cannot get normal measurement results if you do not call this function.

Reimplemented in nn::perf::GpuMeter, and nn::perf::CpuMeter.
