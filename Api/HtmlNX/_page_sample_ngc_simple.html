Describes the sample program that uses the NGC library for simple processing of profanity strings.

Overview

This sample uses the NGC library to judge and mask profanity strings.

File Structure

This sample program is located in Samples/Sources/Applications/NgcSimple/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

If this sample program is executed in a development hardware target environment, an assertion failure occurs when executing CheckProfanityWords and MaskProfanityWordsInText if there is no application management data in the directory. For more information about creating application management data, see Creating Metadata for Placing in NSPD in Features > Creating Application Images > Tools Reference > AuthoringTool. This method will no longer be necessary in a future release.

Execution Procedure

Build the sample program and then run it.

Description

This sample program has the following flow.

  • Initializes nn::fs.
  • Sample count processing for characters that represent numbers.
    • Specify a string that includes characters representing numbers and execute the count processing.
    • Display the count result.
  • Profanity text check sample.
    • Create and initialize an instance of the ProfanityFilter class.
    • Get the version information for the profanity word patterns installed on the system.
    • Add Japanese and Chinese to the pattern list and execute the check processing.
    • Display the check results.
  • Profanity character mask sample.
    • Create and initialize an instance of the ProfanityFilter class.
    • Set the masking method to Overwrite.
    • Add Japanese to the pattern list that is being used and execute mask processing.
    • Display the number of strings that were masked.
    • Display the strings after mask processing.
    • Set the mask method to "one asterisk character."
    • Add Japanese to the pattern list that is being used and execute mask processing.
    • Display the number of strings that were masked.
    • Display the strings after mask processing.
  • At sign (@) sample.
    • Create and initialize an instance of the ProfanityFilter class.
    • Configure the setting for skipping the at sign (@) check.
    • Perform the at sign (@) check.
    • Display the check results.
    • Configure the setting for not skipping the at sign (@) check.
    • Perform the at sign (@) check.
    • Display the check results.
