◆ isAlternateCharSpaceWithOriginalWidth

bool nn::font::ScalableFont::InitializeArg::isAlternateCharSpaceWithOriginalWidth

If set to true, the value of alternateChar is ignored, the alternate character for the font becomes a blank space, and the width is the same as that of the character that would have been displayed.

If set to false, functionality is dependent on the value of isAlternateCharSpaceWithOriginalWidthForNotReadyChar. The default is false.

◆ isAlternateCharSpaceWithOriginalWidthForNotReadyChar

bool nn::font::ScalableFont::InitializeArg::isAlternateCharSpaceWithOriginalWidthForNotReadyChar

This member is used only when isAlternateCharSpaceWithOriginalWidth is set to false.

If set to true, the regular alternate character is used for characters not included in the font (the alternate character specified by alternateChar, which is the question mark (?) by default). For characters that are included in the font but have not yet been plotted, a blank space is used for the alternate character and the width is the same as that of the character that would have been displayed. If you set this member to false, the font uses the normal alternate character. The default is false.

◆ isDrawingHyphenMinusInsteadOfNonBreakingHyphen

bool nn::font::ScalableFont::InitializeArg::isDrawingHyphenMinusInsteadOfNonBreakingHyphen

This indicates whether to use the hyphen-minus(U+002D) glyph when rendering a non-breaking hyphen(U+2011) character.

If true, the U+002D glyph will be used instead of U+2011. Set this member if the font does not contain the U+2011 glyph. The default is false.
