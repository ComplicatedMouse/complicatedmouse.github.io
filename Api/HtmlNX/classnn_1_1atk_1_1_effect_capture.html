Detailed Description

Class for handling the sample data capture process.

Details
The EffectCapture class wraps the API for nn::audio::CaptureType. After EffectCapture has been appended by SoundSystem::AppendEffect , the class must be retained until the effect has been deleted by SoundSystem::ClearEffect or SoundSystem::RemoveEffect.

You can create any kind of audio capture process by inheriting this class and performing signal processing on the waveform passed as the argument by ProcessSamples.

When allocating this type of memory, it must be aligned to a nn::atk::EffectCapture::EffectCaptureAlignment byte boundary.

◆ SetChannelCount()

bool nn::atk::EffectCapture::SetChannelCount ( int  channelCount)

Sets the number of channels.

Details
If this function is used to specify the number of channels, apply a number of effects corresponding to the number of channels specified in order starting from the lowest value of nn::atk::ChannelIndex. To set both the effects and the applicable channels at the same time, use SetChannelIndex instead of this function.

The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. If this function is used together with SetChannelIndex, the get settings that are applied are those of the function called last.

Parameters
channelCountNumber of channels to configure.
Returns
Returns true when configuration is successful.
Precondition
  • 0 < channelCount <= ChannelCountMax.

◆ SetChannelIndex()

bool nn::atk::EffectCapture::SetChannelIndex ( const ChannelIndex *  pChannelIndices,
int  channelCount 
)

Configures channels.

Details
You can specify the channels for applying effects and the number of channels for the effect. You do not need to set the number of channels with SetChannelCount when using this function.

The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. If SetChannelCount is used together with this function, the settings of the function called later are applied.

Parameters
[in]pChannelIndicesThe channels in which to apply the effect. A ChannelIndex-type array.
[in]channelCountThe number of channels in which to apply the effect. Specify the number of elements in the array passed to pChannelIndices.
Returns
Returns true when channel configuration is successful.
Precondition
  • pChannelIndices != nullptr.
  • (Each value in the pChannelIndices array) < nn::atk::ChannelIndex_Count.
  • Two or more of the same channel number do not exist in each of the pChannelIndices array values.
  • 0 < channelCount <= ChannelCountMax.
See also
SetChannelCount
