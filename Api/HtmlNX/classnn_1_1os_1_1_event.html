Detailed Description

Class for handling events.

Details
Events make it possible to establish synchronization among threads.

The nn::os::Event class uses nn::os::EventType objects. It is a utility class that wraps calls to synchronization control functions. The member functions provided by this class are implemented as functions that call functions like nn::os::WaitEvent() that take the nn::os::EventType type for their arguments.

Examples
AudioEchoback/AudioEchoback.cpp, and WebSocketChat/WebSocketChat.cpp.

bool nn::os::Event::TryWait ( )

Polls whether events are in the signaled state.

Returns
Returns whether polling of the event was successful.

Details
Polls whether an event is in a signaled state. Returns true if the event is in a signaled state, or false otherwise. If the event is set to nn::os::EventClearMode_AutoClear, it is cleared to a non-signaled state as soon as a signaled state is detected. For more information, see nn::os::TryWaitEvent().

Waits for an Event object to enter a signaled state, with a timeout. Returns true if the event enters the signaled state within the specified timeout, or false otherwise.
