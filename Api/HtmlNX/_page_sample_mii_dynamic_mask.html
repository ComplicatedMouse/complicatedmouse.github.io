This sample program dynamically switches the facial expression of the Mii character.

Overview

This sample program dynamically switches the facial expression of the Mii character. It demonstrates how to dynamically change one Mask texture held by nn::mii::CharModel.

File Structure

This sample program is located in Samples/Sources/Applications/MiiDynamicMask/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

Periodically overwrites a single Mask texture to display the Mii character with a dynamically changing expression.

Conversion Process

  • /Samples/Sources/Applications/MiiDynamicMask/Resources/SampleShader.glsl

A simple shader for rendering Mii characters.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Create a sampler for the Mii.
  • Gets nn::mii::CharInfo.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize the Mii resource (nn::mii::Resource).
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Render the Faceline texture.
  • Render the Mask texture.
  • Initialize the shader for Mii rendering.
  • Initialize the drawing process for Mii rendering.
  • Set the constant buffers for Mii rendering.
  • Overwrites the Mask texture and draws the Mii character in order to change the expression.
