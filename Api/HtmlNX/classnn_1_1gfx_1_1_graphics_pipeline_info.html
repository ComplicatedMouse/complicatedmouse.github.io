Detailed Description

The class representing the information for initializing a graphics pipeline.

Examples
MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
