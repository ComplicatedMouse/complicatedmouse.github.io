◆ SetupBlockBuffer()

bool nn::g3d::ModelObj::SetupBlockBuffer ( nn::gfx::Device *  pDevice,
nn::gfx::MemoryPool *  pMemoryPool,
ptrdiff_t  offset,
size_t  memoryPoolSize 
)

Passes a memory pool and builds a uniform block.

Parameters
[in]pDevicePointer to the device.
[in]pMemoryPoolPointer to the memory pool.
[in]offsetOffset to the region to use for the memory pool.
[in]memoryPoolSizeThe size of the available memory pool beyond offset.
Returns
Returns true if building succeeded, or false otherwise.
Precondition
  • pDevice is initialized.
  • pMemoryPool is initialized.
  • The position of the memory pool specified by offset is aligned to the value of nn::gfx::Buffer::GetBufferAlignment().
  • memoryPoolSize is at least the size of CalculateBlockBufferSize().
  • The uniform block is not built.
Postcondition
  • The uniform block is built.

Details

  • We recommend using CpuCached as the memory pool characteristic when accessing the CPU from the post-build dynamic vertex buffer, or when the SetShapeAnimCalculationEnabled() function is called to enable application of shape animations.
  • Performance will be lower if you set CpuUncached.
Examples
G3dDemo/ShapeAnimation.cpp, and G3dDemo/Simple.cpp.

◆ CalculateSkeleton()

void nn::g3d::ModelObj::CalculateSkeleton ( int  bufferIndex)

Calculates the matrix palette from the world matrix.

Parameters
[in]bufferIndexIndex to the uniform block that applies the calculation.
Precondition
  • The index is within the range of the number of uniform block buffers held by the skeleton.

Details
The calculation result is applied to the skeleton's uniform block.

Because buffers accessed by the graphics processing unit (GPU) are overwritten, this process must be called after GPU access for the previous rendering process completes. It is assumed that double buffering has been set as a build option, or that it is called after waiting for rendering to complete.

Examples
G3dDemo/ShapeAnimation.cpp, G3dDemo/Simple.cpp, and G3dDemo/SkeletalAnimation.cpp.

◆ CalculateShape()

void nn::g3d::ModelObj::CalculateShape ( int  bufferIndex)

Calculates render resources for shapes.

Parameters
[in]bufferIndexIndex to the uniform block that applies the calculation.
Precondition

Details
Performs calculations for shapes that are not dependent on rigid bodies or views. The calculation result is applied to the shape's uniform block.

Because buffers accessed by the graphics processing unit (GPU) are overwritten, this process must be called after GPU access for the previous rendering process completes. It is assumed that double buffering has been set as a build option, or that it is called after waiting for rendering to complete.

Examples
