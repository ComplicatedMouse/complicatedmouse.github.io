nn::profiler::ScopedCodeBlock::ScopedCodeBlock ( uint64_t  id)
explicit

Creates a scoped code block based on an integral ID.

Parameters
[in]idAn identifier to denote this code block.
Returns
The created code block.

Details
On creation, nn::profiler::PushCodeBlock() is called with the id provided. The identifier provided is the integer that will appear in the profiler GUI to denote this specific code block. When the created variable falls out of scope, nn::profiler::PopCodeBlock() is called automatically.

Here is an example on creating a ScopedCodeBlock:

nn::profiler::ScopedCodeBlock block(10);
