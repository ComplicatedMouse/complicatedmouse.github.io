Detailed Description

The utility of the I3dl2Reverb effect applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform I3dl2Reverb effect processing. Specify the sample data obtained by the nn::audio::ReadAuxSendbuffer() function as the input for this effect to output environmental reverb effects that conform in part to the Interactive 3D Audio Level 2 (I3DL2) specification. The I3dl2Reverb effect applied with this effect is identical to the processing performed by nn::audio::I3dl2ReverbType. However, this effect processing is performed on the CPU thread that calls nn::audio::ProcessAuxI3dl2Reverb().

Directly altering member variables is prohibited. Use the main structure through the following functions.
