enum nvn::CommandBufferMemoryEvent::Enum

Enumerated values for the type CommandBufferMemoryEvent.

Enumerator
OUT_OF_COMMAND_MEMORY 

The command buffer has run out of command memory.

Applications should respond to this event by providing new memory by calling CommandBuffer::AddCommandMemory. Failing to do so will result in program termination.

OUT_OF_CONTROL_MEMORY 

The command buffer has run out of control memory.

Applications should respond to this event by providing new memory by calling CommandBuffer::AddControlMemory. Failing to do so will result in program termination.
