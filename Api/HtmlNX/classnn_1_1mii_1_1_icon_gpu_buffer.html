Detailed Description

Buffers for drawing textures of icons.

Details
This class is for the buffer and other resources needed to draw textures of icons using IconShader. You need an instance of this class for each icon texture being drawn.

Attention
It is unnecessary if you are not using IconShader.
Examples
MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Initializes IconGpuBuffer.

Attention
Both pMemory and pMemoryPool must be retained until the call to IconGpuBuffer::Finalize().
You cannot reinitialize an instance of IconGpuBuffer that has been initialized already. Call this function after calling Finalize().
Parameters
[in]pMemoryThe memory that IconGpuBuffer will use.
[in]memorySizeThe size of pMemory.
[in]pDeviceThe device.
[in]pMemoryPoolThe memory pool that IconGpuBuffer will use.
[in]memoryPoolOffsetThe offset to the memory pool that IconGpuBuffer will use.
[in]memoryPoolSizeThe size of the memory pool that IconGpuBuffer will use.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && ! IconGpuBuffer::IsInitialized() && The offset and size are normal.
Postcondition
If successful, IconGpuBuffer::IsInitialized() = true.

Sets whether to clear the color target view and the depth stencil view.

Parameters
[in]clearColorTargetViewSpecifies whether to clear the color target view.
[in]clearDepthStencilViewSpecifies whether to clear the depth stencil view.

Details
By default, both clearColorTargetView and clearDepthStencilView are set to true. When clearColorTargetView = true, that view is cleared to the color set by the SetBackgroundColor() function. When clearDepthStencilView = true, that view is cleared to depth=1.0, stencil=0.
