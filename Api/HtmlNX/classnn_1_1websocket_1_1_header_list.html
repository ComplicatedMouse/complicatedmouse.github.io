Detailed Description

The header list.

Details
Manages the list of HTTP headers sent and received during HTTP communication.

Note
You can use range-based for statements because the begin() and end() functions are implemented.
Examples
WebSocketChat/WebSocketChat.cpp, and WebSocketEcho/WebSocketEcho.cpp.
