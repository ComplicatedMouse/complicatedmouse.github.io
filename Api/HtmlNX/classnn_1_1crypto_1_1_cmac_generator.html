Detailed Description

template<typename BlockCipher>
class nn::crypto::CmacGenerator< BlockCipher >

Class for calculating CMAC.

Template Parameters
BlockCipherThe implementation class for the block cipher to use.

Details
CMAC is a cipher mode to create a message authentication code, MAC, using a block cipher stipulated using NIST SP 800-38B.

This class is a template class that takes the block cipher implementation class as a template parameter. The AES block sizes currently only support a 128-bit block cipher. Use Aes128CmacGenerator or a similar class in your actual code.

The implementation of the block cipher class specified in the template must satisfy the following conditions. For more information such as the parameters of each function, see the reference for the AesEncryptor class.

  1. Has a BlockSize constant that defines the size of the block to process.
  2. Has a EncryptBlock function that encrypts a single block.

This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state.
  • Ready.
  • Complete.

◆ Initialize()

template<typename BlockCipher >
void nn::crypto::CmacGenerator< BlockCipher >::Initialize ( const BlockCipher *  pBlockCipher)

Initializes the MAC calculation.

Parameters
[in]pBlockCipherInstance of the block cipher implementation class.

Details
Specify an instance of the class that implements the block cipher to use for encryption in pBlockCipher. The instance must be initialized with the required key, and other settings, so that EncryptBlock is ready to be called.

The instance specified in pBlockCipher is not copied. Do not perform any operations on pBlockCipher until the processing sequence is complete.
