Detailed Description

template<size_t KeySize, typename HashFunction>
class nn::crypto::RsaOaepEncryptor< KeySize, HashFunction >

Class for encryption using RSA-OAEP.

Template Parameters
KeySizeSize of the key to use, in bytes.
HashFunctionImplementation class for the hash function to use.

Details
RSA-OAEP is an encryption method that uses an RSA encryption algorithm per PKCS #1.

This class is a template class that takes the key size and hash function implementation class as template parameters.

The key size can be specified in byte (8-bit) units up to a maximum of 512 bytes (4096 bits).

The implementation of the hash function class specified in the template must satisfy the following conditions. For more information about parameters of each function, see Sha1Generator or similar classes.

  1. Has a BlockSize constant that defines the size of the block to process.
  2. Has a HashSize constant that defines the output size of the hash value.
  3. Has an Initialize function that begins processing.
  4. Has an Update function that updates the hash value on input.
  5. Has a GetHash function that gets the final hash value.

This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state
  • Ready.
  • Complete.

Details
Adds the parameters and initializes the encryption process. An internal copy of the parameters is created.

The modulus and exponent are always treated as unsigned numbers in big-endian format.

A value up to three bytes in size, such as 65537, can be specified for the RSA public exponent.

Parameters

[in]labelSizeByte size of the label.
Precondition
  • The instance is in a ready state.

Details
The label is data that can attached that is separate from the encryption target message.

This function can be split among multiple calls. However, note that the entered data is treated as a single contiguous set of data, so the order of input is significant.

All label input must be completed before Encrypt() runs.

If you already know the hash value for the label, you can add it using SetLabelDigest() instead of this function.

Encrypts.

Uses a stack as the work area.

Parameters
[out]pOutCipherPointer to the buffer that receives encrypted data.
[in]cipherSizeByte size of the buffer indicated by pOutCipher.
[in]pRawMessagePointer to the buffer storing the data to encrypt.
[in]rawMessageSizeSize in bytes of the data to encrypt.
[in]pSeedPointer to the Seed data.
[in]seedSizeSize of the Seed data.
Returns
Returns whether encryption succeeded.
Invalid parameters are returned when initialization fails.
Precondition
  • The instance is in a ready state.
  • cipherSize == KeySize.
  • rawMessageSize <= KeySize - 2 * HashSize - 2.
Postcondition
  • The instance is in a completed state.
  • The encrypted result is written to pOutCipher.

Details
Encrypts the data series passed to the function.

For the Seed data, specify a random number of HashSize bytes that is different for each process of encryption.

RequiredWorkBufferSize bytes are used from the stack as a work buffer.
