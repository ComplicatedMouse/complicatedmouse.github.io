Detailed Description

The data block that stores the list of texture image reference information.

Details
The list of textures referenced by the pane is defined ahead of the pane.

DataBlockKindTextureList is specified for blockHeader[kind].

blockHeader[size] includes the data up to nameStrPool.

The following data structure uses this structure.

  • ResTexture textures[texCount]
  • uint8_t nameStrPool[]
