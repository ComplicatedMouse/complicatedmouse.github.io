Detailed Description

template<typename StorageT, typename Tag = void>
struct nn::util::BitPack< StorageT, Tag >

Bitfield substitution type.

Details
Use BitFlagSet for types that are only used in a collection of flags. nn::util::BitPack objects, such as nn::util::BitPack8, nn::util::BitPack16, and nn::util::BitPack32, are usually trivial type. Explicitly initialize them for use.

Example:

nn::util::BitPack32 variable{};
