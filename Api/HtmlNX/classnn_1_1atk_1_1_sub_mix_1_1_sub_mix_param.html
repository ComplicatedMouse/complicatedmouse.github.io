◆ SetSoundSendClampEnabled()

void nn::atk::SubMix::SubMixParam::SetSoundSendClampEnabled ( int  bus,
bool  isEnabled 
)

The use of internal features is prohibited.

Sets whether to clamp the send volume from the sound to the submix to a maximum value of 1.0f. If true, the send volume from sound to submix is clamped to a range from 0.0f to 1.0f. If false, the send volume from sound to submix is 0.0f or greater. The default is true for all buses.

Parameters
[in]busThe bus for which to configure this setting.
[in]isEnabledWhether to enable clamping.
Precondition
0 <= bus <= nn::atk::OutputReceiver::BusCountMax.
