◆ NN_OS_CONDITION_VARIABLE_INITIALIZER

#define NN_OS_CONDITION_VARIABLE_INITIALIZER ( )

Statically initializes a nn::os::ConditionVariableType object.

Details
Macros that are expanded in the list are used to statically initialize nn::os::ConditionVariableType objects.

For more information about condition variables, see nn::os::InitializeConditionVariable().

As shown in the following example, this macro can only be used for static initialization of nn::os::ConditionVariableType objects. Objects initialized in this way do not need to be explicitly destroyed by calling nn::os::FinalizeConditionVariable().

Example:

nn::os::ConditionVariableType g_StaticCondVar = NN_OS_CONDITION_VARIABLE_INITIALIZER();
