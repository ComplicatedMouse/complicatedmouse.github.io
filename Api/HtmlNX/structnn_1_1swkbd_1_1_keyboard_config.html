◆ isPredictionEnabled

bool nn::swkbd::KeyboardConfig::isPredictionEnabled

Specifies whether to use predictive text input.


Specify true to use the feature.

Details
The value specified with isPredictionEnabled is valid when all of the following criteria are met.

  • KeyboardMode is KeyboardMode_LanguageSet1, KeyboardMode_LanguageSet1Latin, or KeyboardMode_Alphabet.
  • invalidCharFlag is not set.
  • passwordMode is PasswordMode_Show.

◆ invalidCharFlag

uint32_t nn::swkbd::KeyboardConfig::invalidCharFlag

The flags for characters that are prohibited from being entered.


Specify the bit flags (the set of nn::swkbd::InvalidChar values) for the group of characters (keys) that you want to prohibit from being entered.

◆ headerText

char16_t nn::swkbd::KeyboardConfig::headerText[HeaderTextMaxLength+1]

The string to show in the header text.


Specify a string that succinctly shows what to be entered.
This is only shown on top of the preview field when the preview field consists of one line.
You can specify up to nn::swkbd::HeaderTextMaxLength characters.

◆ subText

char16_t nn::swkbd::KeyboardConfig::subText[SubTextMaxLength+1]

The string to show in the sub text.


Specify a string that supplements the content of the header text.
This is only shown below the header text when the preview field consists of one line.
You can specify up to nn::swkbd::SubTextMaxLength characters.

◆ guideText

char16_t nn::swkbd::KeyboardConfig::guideText[GuideTextMaxLength+1]

The guide text.

This is shown when no characters have been entered in the preview field.
Specify a string that can be shown when nothing has been entered, presenting an example or otherwise indicating the purpose of making an entry.
Also specify this text if there is a description that you want to display, even when there are multiple lines of text.
You can specify up to nn::swkbd::GuideTextMaxLength characters.

◆ passwordMode

PasswordMode nn::swkbd::KeyboardConfig::passwordMode

The mode for hiding the entered text in password keyboard mode.


If this is not specified, the entered text will be shown on screen.

Details
If KeyboardMode is one of the following, it is forcibly handled as PasswordMode_Show.

  • KeyboardMode_SimplifiedChinese
  • KeyboardMode_TraditionalChinese
  • KeyboardMode_Korean
  • KeyboardMode_LanguageSet2

◆ isUseNewLine

bool nn::swkbd::KeyboardConfig::isUseNewLine

Specifies whether to enable line feeds.


Set to true to enable the action equivalent to pressing the ENTER key for a newline.
If InputFormMode is not InputFormMode_MultiLine, the newline feature does not become enabled.

◆ isUseUtf8

Specify true to use UTF-8 for the string that is returned when the software keyboard ends.
Specify false to use UTF-16 for the string that is returned when the software keyboard ends.

◆ isUseBlurBackground

bool nn::swkbd::KeyboardConfig::isUseBlurBackground

Specifies whether to blur the screen that is currently displaying when the software keyboard starts.


When the software keyboard starts, the screen of the running application is captured and that captured image is rendered as the background for the software keyboard.
Specify true to render this captured image in a blurry manner.

◆ separateTextPos

The insertion position of the delimiter can be specified.


A delimiter is inserted after the specified position.

◆ isCancelButtonDisabled

bool nn::swkbd::KeyboardConfig::isCancelButtonDisabled

Specify true to prevent the software keyboard from being terminated with the B Button or X Button.


While it is more difficult for the user to cancel via input, cancellation may occur if the following procedure is performed.

  1. Press the HOME Button while the keyboard is displayed and start system settings or other software from the HOME Menu.
  2. If you then resume the application, the nn::swkbd::ShowKeyboard() function returns nn::swkbd::ResultCanceled.

    If you want to make it more difficult to accept empty input, set a value of 1 or more for textMinLength.
