Detailed Description

The structure that represents vibration file information.

Details
Use this structure to store the information obtained when you parse the vibration file using nn::hid::ParseVibrationFile.

When a vibration file that does not contain loop information is parsed, loopStartPosition and loopInterval both store 0, and loopEndPosition stores the same value as sampleLength.
