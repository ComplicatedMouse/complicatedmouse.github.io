bool nn::atk::detail::SoundArchiveLoader::LoadData ( SoundArchive::ItemId  id,
SoundMemoryAllocatable *  pAllocator,
uint32_t  loadFlag = LoadFlag_All,
size_t  loadBlockSize = 0 
)

Loads the data in a sound archive.

Details
The memory required to load data is allocated by pAllocator. Returns false if sufficient memory cannot be allocated.

This function synchronously loads sound data. When loading sound data asynchronously, call this function from a thread used for loading. This function is thread-safe provided that no operations are performed during the loading process on the associated sound archive or on a sound heap passed in an argument.

Parameters
[in]idItem ID of the data to load.
[in]pAllocatorAllocates memory for loading data.
[in]loadFlagFlag that loads related data.
[in]loadBlockSizeThe block size (in bytes) when data is loaded in blocks.
Returns
Returns true if loading succeeded, or false otherwise.

◆ CancelLoading()

bool nn::atk::detail::SoundArchiveLoader::CancelLoading ( )

Cancels the executing instance of LoadData.

Details
If a nonzero value is specified for loadBlockSize, LoadData loads data in blocks of that size. This function cancels the block loading and returns control from LoadData.

Even if loading is canceled, the memory required for the original load remains secured. You must call a deallocation process such as SoundHeap::LoadState to enable the heap.

Returns
Returns true if data is currently loaded, or false otherwise.
