Detailed Description

Class for handling friend presence.

Details
Can get the presence information configured by the friend. The information that can be obtained using this class can be set on a friend's console via UserPresence.

No cache remains in console storage because presence functionality conveys information in real time.

You cannot get valid presence data in the following situations.

  • Not connected to the network.
  • Something is wrong with the friends server and friend presence is not available.
  • The friend has restricted the range of their presence disclosure.

bool nn::friends::FriendPresence::IsSamePresenceGroupApplication ( ) const

Checks whether the presence group of the application being played is the same as that of the running application.

Returns
Returns a value denoting whether the presence group of the application being played is the same as that of the running application.

Details
To determine whether the presence group is identical, the presence group IDs are compared to see if they match.

This function returns false if there is no valid presence data.
