This sample offline HTML directly plays back videos.

Overview

This offline sample HTML directly plays back videos. This sample demonstrates the following features.

  • Directly playing back a video

File Structure

The video that this sample plays back is Samples/Sources/Applications/WebDemo/html_document/sample/shooting_movie.htdocs/img/shooting_movie.mp4.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

Input Operation
A Button Switches between playback and pause.
B Button Exits the offline web applet and returns to the menu, via the application.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and directly play back the video. Configured to automatically close the video player and to return to the application when the video file has played through to the end.

Directly Playing Back a Video

Set the video's file path as the nn::web::ShowOfflineHtmlPageArg argument, and set the argument for nn::web::ShowOfflineHtmlPageArg::SetBootAsMediaPlayer() to true.
