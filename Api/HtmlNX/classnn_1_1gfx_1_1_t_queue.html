◆ Present()

template<typename TTarget >
void nn::gfx::TQueue< TTarget >::Present ( TSwapChain< Target > *  pSwapChain,
int  presentInterval 
)

Copies the content of the swap chain to the display.

Parameters
[in]pSwapChainPointer to the swap chain.
[in]presentIntervalScreen V-sync count.
Precondition
  • The queue is initialized.
  • pSwapChain != NULL.
  • pSwapChain is initialized.
Postcondition
  • The command to draw to the display is submitted.

Details
Display V-sync is not used when presentInterval is set to 0. Calls might not return until vertical synchronization has occurred because of things like the dependency on low-level graphics APIs. To make sure that synchronization is occurring in the CPU, call Sync after this.
