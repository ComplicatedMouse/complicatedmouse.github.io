◆ SetCurrentApplicationFrame()

void nn::spy::SpyController::SetCurrentApplicationFrame ( int  value)

Sets the current application frame.

Parameters
[in]valueThe current application frame value.

Details
The application frame set in this function is added to the data sent by nn::spy. By setting the application frame, a grid display by application frame will be possible in the Spy.exe timeline. Call this function at the beginning of application frame processing as much as possible.

See also
GetCurrentApplicationFrame
Examples
SpyAtk/SpyAtk.cpp, and SpyAudio/SpyAudio.cpp.

◆ RegisterModule()

bool nn::spy::SpyController::RegisterModule ( SpyModule &  module)

Registers a Spy module.

Details
Module registration is performed between Initialize() and Open().

Parameters
[in]moduleThe Spy module.
Returns
Returns true if registration succeeded, or false otherwise.
Precondition
  • Initialize() was called.
  • Finalize() was not called.
  • Open() was not called, or Close() was called after Open() was called.
Examples
SpyAtk/SpyAtk.cpp, and SpyAudio/SpyAudio.cpp.
