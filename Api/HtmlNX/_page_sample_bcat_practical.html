This sample program describes practical uses of the data distribution feature.

Overview

This section describes practical uses of the data distribution feature.

File Structure

This sample program is located in Samples/Sources/Applications/BcatPractical/ .

Required Environment

You must create a user account that is associated with a network service account. Also requires an environment where network connections are possible.

How to Use

Use either Npad or DebugPad.

Important Information

Because some features are under development, the data distribution library may be restricted in some cases. For more information about use restrictions, see NintendoSDK Documents.

Execution Procedure

Build the sample program and then run it. To go back and confirm immediate synchronization, delete the data distribution cache storage after exiting the sample program.

Description

Sample program to perform immediate synchronization and access distribution data.

Cautions are noted for error handling and immediate synchronization. For more information, see the comments in the various scene functions.
