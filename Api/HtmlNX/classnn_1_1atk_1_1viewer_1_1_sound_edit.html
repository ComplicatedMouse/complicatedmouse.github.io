Detailed Description

A class that provides features for in-game editing of sounds.

Details
This class provides an easy way to embed features for in-game editing of sounds in your application. If you want to customize, use SoundEditSession, and SoundArchiveEditor instead of this class. (Note that those classes are beta versions.)

Examples
AtkViewer/AtkViewer.cpp.

◆ GetRequiredMemorySize()

size_t nn::atk::viewer::SoundEdit::GetRequiredMemorySize ( const Option &  param) const

Gets the size of memory required to use the in-game sound editing feature.

Details
To use the in-game sound editing feature, you must allocate a buffer of the size obtained by this function, and specify it for the InitializeArg parameter of the Initialize() function. At that time, in the param parameter of the Initialize() function, specify the same Option structure passed to the GetRequiredMemorySize() function.

Parameters
[in]paramThe Option object.
