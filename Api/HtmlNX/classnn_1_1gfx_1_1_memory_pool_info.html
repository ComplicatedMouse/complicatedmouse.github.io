◆ SetMemoryPoolProperty()

void nn::gfx::MemoryPoolInfo::SetMemoryPoolProperty ( int  value)

Sets the properties of the memory pool.

Parameters
[in]valueThe properties of the memory pool, described as a combination of nn::gfx::MemoryPoolProperty objects.
Postcondition
  • The memory pool properties are set.

Details
To specify a value, you must specify one of the following: nn::gfx::MemoryPoolProperty_CpuInvisible, nn::gfx::MemoryPoolProperty_CpuUncached, or nn::gfx::MemoryPoolProperty_CpuCached; and one of the following: nn::gfx::MemoryPoolProperty_GpuInvisible, nn::gfx::MemoryPoolProperty_GpuUncached, or nn::gfx::MemoryPoolProperty_GpuCached.
