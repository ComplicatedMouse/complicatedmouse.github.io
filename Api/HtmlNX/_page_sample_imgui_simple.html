Required Environment

Nothing specific.

How to Use

In the development hardware environment, the following input devices are supported. DebugPad Keyboard (a USB keyboard, or the PC keyboard using Input Director) Mouse (a USB mouse, or the PC mouse using Input Director)

In the Windows environment, you can operate the sample using the mouse and keyboard.

Only ASCII characters are accepted for text input.

The sample is operated in the following ways using the debug pad. D-Pad Up/Down/Left/Right: Navigate and change values. A Button: Activate item, press and hold to change a value, enter a child window, etc. B Button: Close pop-up menus, remove from child, clear selection, etc. Y Button (press): Access menu, collapse window, close window, configure window, etc. Y Button (hold) + D-Pad: Change window size. Y Button (hold) + Stick: Move window. Y Button (hold) + ZL/LR Button: Change window focus (similar to ALT+TAB). X Button: Enter text. L/R button (hold): Manually change the speed. Control stick: Scroll.
