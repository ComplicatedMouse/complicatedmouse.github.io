Detailed Description

template<int TViewportCount>
class nn::gfx::util::ViewportScissorStateHolder< TViewportCount >

A wrapper class that allocates the necessary additional memory as a member to nn::gfx::ViewportScissorState.

Template Parameters
TViewportCountThe number of viewports. Specify a value that is equal to or greater than the viewport state information count specified in nn::gfx::ViewportScissorState::SetViewportStateInfoArray().
