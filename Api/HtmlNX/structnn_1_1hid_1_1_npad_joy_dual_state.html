Detailed Description

The structure representing the input states for the NpadStyleJoyDual play style modeled on holding two Npad Joy-Con controllers.

Details
The digital buttons available when using two Joy-Con controllers. Provides the input state for the (directional buttons, A, B, X, Y, L, R, ZL, ZR, Joy-Con (L) SL, Joy-Con (L) SR, Joy-Con (R) SL, Joy-Con (R) SR, Plus, Minus) and the left and right control sticks. The attributes::IsConnected bits are set when either of the right and left Joy-Con controllers are connected. The individual connection states for the right and left controllers are available using attributes::IsLeftConnected and attributes::IsRightConnected.

There are no rules governing the input from the directional buttons.
