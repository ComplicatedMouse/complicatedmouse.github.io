Detailed Description

The class representing LoadMeterCenter class information.

Passed as a parameter when initializing the LoadMeterCenter class.

Examples
FrameworkDemo/Main.cpp, G3dDemo/Parallel.cpp, GfxPrimitiveRenderer/GfxPrimitiveRenderer.cpp, and PerfSimple/PerfSimple.cpp.

Member Function Documentation

◆ SetCoreCount()

void nn::perf::LoadMeterCenterInfo::SetCoreCount ( int  coreCount)

Sets the number of CPU cores for the application to use in the current environment.

Parameters
[in]coreCountThe number of CPU cores.

Details
CpuMeter objects for measuring each core are created in the number set. To get accurate GPU measurements, you must also specify the number of CPU cores.

Examples
FrameworkDemo/Main.cpp, G3dDemo/Parallel.cpp, GfxPrimitiveRenderer/GfxPrimitiveRenderer.cpp, and PerfSimple/PerfSimple.cpp.
