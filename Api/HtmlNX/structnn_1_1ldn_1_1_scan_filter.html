Detailed Description

The structure that stores scan filters.

Details
This structure sets the criteria for filtering the Scan results.

First, set values for all of the members you want to use for filtering among networkType, networkId, and ssid. Only the networks that have values matching the values of these parameters will be included in the search results.

Afterwards, store the logical OR of the constant defined in ScanFilterFlag to the flag to notify the LDN of the members used for filtering. If, for example, you want to use networkType and networkId.intentId for filtering, specify ScanFilterFlag_NetworkType | ScanFilterFlag_IntentId.
