Overview

This sample implements the drawing of a spinning textured cube to introduces how to use textures and samplers. A spinning textured cube is rendered to show the using of tiled textures and linear textures, and different samplers.

File Structure

The sample's source file and Visual Studio solutions can be found at Samples/Sources/Applications/NvnSimple03TexturedCube

TexturedCube.h, TexturedCube.cpp A class that implements the drawing of a spinning textured cube.
NvnSimple03.vs Dependent vetex shader which is used for rendering the textured cube
NvnSimple03.fs Dependent fragment shader which is used for rendering the textured cube

Description

A large part of the work involved with texturing with NVN is implemented in the GraphicsTextureManager class, which this sample utilizes. Building on the previous demo, WorldSpaceTriangles, this class shows how to use textures when rendering primitives, which involves the following:

  • Setting up NVN texture pool and sampler pool objects
  • Initializing NVN texture and sampler objects and registering them to the respective pools
  • Looking up sampler location from shader reflection info and using it to bind a texture when rendering This demo also shows both tiled textures and linear textures in action.

The NvnSimpleTextureConverter tool, whose source code is included in this solution, is used to convert the source images into texture asset files with tiling that is optimized for the NX GPU. See Properties > Build Event >Pre-Build Event for texture conversion command line details.
