Detailed Description

template<int THistoryLengthMax>
class nn::gfx::util::TransientMemoryPoolAllocatorHolder< THistoryLengthMax >

Class that wraps TransientMemoryPoolAllocator and the memory block needed for its initialization, and creates an object.

Template Parameters
THistoryLengthMaxThe maximum number of TransientMemoryPoolAllocator history entries that will be recorded.

Initializes TransientMemoryPoolAllocator, which will be wrapped with this object.

Parameters
[in]pMemoryPoolMemory pool.
[in]baseOffsetThe start of the offset for the memory to manage.
[in]sizeThe length, in bytes, of the memory to manage.
[in]allocatableAlignmentMaxThe maximum alignment required for the memory allocated from this allocator.
Precondition
This object must be uninitialized.
The preconditions for each parameter are the same as those for TransientMemoryPoolAllocator::Initialize().
Postcondition
This object is initialized.
See also
TransientMemoryPoolAllocator::Initialize()

Details
Initializes an instance of TransientMemoryPoolAllocator, to be wrapped by this object using the specified parameters. The memory region wrapped in this object is used for the memory block to use for initialization. HistoryLengthMax can specify the maximum number of history entries. Calling this instance for the same instance is not thread-safe.
