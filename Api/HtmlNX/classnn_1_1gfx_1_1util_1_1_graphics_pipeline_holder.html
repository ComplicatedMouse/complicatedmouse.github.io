Detailed Description

template<int TBlendTargetCount, int TVertexAttributeCount, int TVertexBufferCount>
class nn::gfx::util::GraphicsPipelineHolder< TBlendTargetCount, TVertexAttributeCount, TVertexBufferCount >

A wrapper class that allocates the necessary additional memory as a member when using nn::gfx::Pipeline as a graphics pipeline.

The necessary value is taken as a template parameter because the amount of memory to allocate is set statically.

Template Parameters
TBlendTargetCountThe number of blend targets. for the nn::gfx:: :: BlendStateInfo that is passed to nn::gfx:: :: GraphicsPipelineInfo::SetBlendStateInfo(), Specify a value that is equal to or greater than the blend target state information count specified in nn::gfx::BlendStateInfo::SetBlendTargetStateInfoArray().
TVertexAttributeCountThe number of vertex attributes. For the nn::gfx::VertexStateInfo that is passed to nn::gfx::GraphicsPipelineInfo::SetVertexStateInfo(), Specify a value that is equal to or greater than the vertex attribute state information count specified in nn::gfx::VertexStateInfo::SetVertexAttributeStateInfoArray().
TVertexBufferCountThe number of vertex buffers. For the nn::gfx::VertexStateInfo that is passed to nn::gfx::GraphicsPipelineInfo::SetVertexStateInfo(), Specify a value that is equal to or greater than the vertex buffer state information count specified in nn::gfx::VertexStateInfo::SetVertexBufferStateInfoArray().
