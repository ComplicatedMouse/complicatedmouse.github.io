Detailed Description

Class that manages animation bind states.

Member Enumeration Documentation

◆ BindFlag

enum nn::g3d::AnimBindTable::BindFlag

The use of internal features is prohibited. Flag for binding.

Enumerator
BindFlag_Enabled 

Enabled.

BindFlag_SkipCalculate 

Skips the update process, but applies the final result.

BindFlag_SkipApply 

Skips the apply process, resulting in a futile update process running on its own.

BindFlag_Disabled 

Skips both the calculate and the apply processes.
