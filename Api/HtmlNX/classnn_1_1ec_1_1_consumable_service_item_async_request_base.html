Detailed Description

Asynchronous request base class for consumable service items.

Details
This class is a base class that defines the common processing for asynchronous requests for consumable service items. Processing that changes based on the request content is defined in each derived class. No instances are generated by the base class by itself.

The base class provides the following functionality.

  • Cancel() cancels an asynchronous request.
  • Wait() waits for the asynchronous request to complete.
  • TryWait() performs wait verification for asynchronous request completion.
  • GetEvent() gets the wait event for receiving the asynchronous request completion notification.
  • GetErrorCode() gets the error code for asynchronous request results.

The derived classes provide the following functionality.

  • Begin() starts an asynchronous request.
  • End() ends the asynchronous request.
