This sample program shows how to draw Mii characters using compressed textures.

Overview

This is a sample program for drawing Mii characters using Faceline and Mask textures that have been converted to compressed textures. Uncompressed Faceline and Mask textures are created using the TextureShader class provided in the Face Library and then nn::gfx::util::TextureCompressor is used to create compressed versions of the textures. This sample program presents the procedure for using these compressed textures to display a Mii face model in 3D space.

File Structure

The sample program is located in Samples/Sources/Applications/MiiCompressedTexture/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

In this sample program, Faceline and Mask textures that have been converted into textures in compressed format using the runtime compression library are used to display a Mii character.

Conversion Process

  • /Samples/Sources/Applications/MiiCompressedTexture/Resources/SampleShader.glsl A simple shader for rendering Mii characters.
  • /Sources/Resources/GfxUtilTextureCompression/TextureCompression_ShaderVariation.glsl
  • /Sources/Resources/GfxUtilTextureCompression/TextureCompression_VertexShader.glsl Shaders for runtime texture compression.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Create a sampler for the Mii.
  • Get nn::mii::CharInfo.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize the Mii resource (nn::mii::Resource).
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Render the Faceline texture.
  • Render the Mask texture.
  • Prepare the compressed texture.
  • Prepare the compressed texture view.
  • Prepare the shader for compression.
  • Create compressed textures using the runtime compression library.
  • Register the compressed textures to CharModel.
  • Initialize the shader for Mii rendering.
  • Initialize the drawing process for Mii rendering.
  • Set the constant buffers for Mii rendering.
  • Render the Mii.
