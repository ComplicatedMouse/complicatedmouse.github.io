Detailed Description

The class for handling the Reverb effect utility applied to nn::audio::AuxType.

Details
This utility processes reverb effects using the sample data obtained by nn::audio::AuxType as the input. It specifies the sample data obtained by the nn::audio::ReadAuxSendBuffer() function as the input for this effect, and outputs the result of applying the Reverb effect, which emulates the reverberation of sounds in a closed space.

The nn::audio::AuxReverb class is a utility class that wraps calls to functions that use nn::audio::AuxReverbType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessAuxReverb() that take the nn::audio::AuxReverbType type for their arguments.

Examples
AudioEffectAuxUtility/AudioEffectAuxUtility.cpp.
