Detailed Description

Specifies a wrap mode when mapping a normalized texture coordinate to texels in a texture map.

When sampling from a texture, shaders typically provide normalized coordinate values in the range [0,1] that are mapped to texels in a texture map. Values outside the range [0,1] are legal, and are mapped to texels using the wrap mode.

enum nvn::WrapMode::Enum

Enumerated values for the type WrapMode.

Enumerator
CLAMP 

Texture coordinates are clamped to the range [0,1] before filtering.

REPEAT 

When filtering, texels outside the [0,1] range are mapped back to values in [0,1] by using only the fractional portion of the coordinate.

MIRROR_CLAMP 

Texture coordinates are mirrored around zero and then treated like CLAMP.

MIRROR_CLAMP_TO_EDGE 

Texture coordinates are mirrored around zero and then treated like CLAMP_TO_EDGE.

MIRROR_CLAMP_TO_BORDER 

Texture coordinates are mirrored around zero and then treated like CLAMP_TO_BORDER.

CLAMP_TO_BORDER 

When filtering, texels outside the [0,1] range are treated as sampling the border color.

MIRRORED_REPEAT 

Texture coordinates are mirrored around zero and then treated like REPEAT.

CLAMP_TO_EDGE 

When filtering, texels outside the [0,1] range are treated as sampling the texel on the nearest edge or corner of the texture.
