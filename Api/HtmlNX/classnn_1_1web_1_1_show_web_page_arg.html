◆ SetCallbackUrl()

void nn::web::ShowWebPageArg::SetCallbackUrl ( const char *  pCallbackUrl)

Sets the callback URL.

Parameters
[in]pCallbackUrlThe callback URL.

Details
If page transitions in the webpage reach the page URL beginning with the address specified with this method, the webpage stops being displayed.

The specified value must end with a null terminator. The number of characters must be CallbackUrlMaxLength – 1 or fewer.

The value is overwritten if this function is called a second time.

◆ SetCallbackableUrl()

void nn::web::ShowWebPageArg::SetCallbackableUrl ( const char *  pCallbackableUrl)

Sets a URL that can be used in a callback function.

Parameters
[in]pCallbackableUrlThe URL that can be used in a callback function.

Details
URLs set for use in callback functions are used to restrict the webpages that may perform termination processing for displaying the webpage. The callback URL may stop displaying the webpage only when transitioning from a webpage beginning with a URL that has been set for use in a callback function. If no URLs set for use in a callback function are specified, all pages may use termination processing via the callback URL.

The specified value must end with a null terminator. The number of characters must be CallbackUrlMaxLength – 1 or fewer.

The value is overwritten if this function is called a second time.
