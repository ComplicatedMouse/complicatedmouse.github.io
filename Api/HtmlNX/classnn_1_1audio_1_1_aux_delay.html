Detailed Description

The class for handling the Delay effect utility applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform the Delay effect process. It specifies the sample data obtained by the nn::audio::ReadAuxSendBuffer() function as the input for this effect, and produces the result of applying the Delay effect, which emulates the reverberation of sounds in a closed space.

The nn::audio::AuxDelay class is a utility class that wraps calls to functions that use nn::audio::AuxDelayType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessAuxDelay() that take the nn::audio::AuxDelayType type for their arguments.

Examples
AudioEffectAuxUtility/AudioEffectAuxUtility.cpp.
