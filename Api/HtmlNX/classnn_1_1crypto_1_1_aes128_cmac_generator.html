Detailed Description

Class for performing CMAC calculations using a 128-bit key AES.

Details
This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state.
  • Ready.
  • Complete.

Initializes for CMAC calculations using a 128-bit key AES.

Parameters
[in]pKeyPointer to the key data.
[in]keySizeSize in bytes of the key data.
Precondition
  • keySize == 16.
Postcondition
  • The instance is in a ready state.

Details
This function takes key data as input and initializes the MAC calculation. An internal copy of the key data is created.

size_t  macSize 
)

Completes the MAC calculation and gets the final MAC.

Parameters
[out]pMacPointer to the buffer storing the MAC.
[in]macSizeSize of the buffer pointed to by pMac in bytes.
Precondition
  • macSize >= MacSize
  • The instance is in a ready or complete state.
Postcondition
  • The MAC is written to pMac.
  • The instance is in a completed state.

Details
This function completes the MAC calculation. To calculate a different MAC, call Initialize() to begin a new calculation.
