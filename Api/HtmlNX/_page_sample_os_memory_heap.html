A sample program for the memory heap feature.

Overview

This sample shows the memory heap feature.

For more information about using the memory heap feature, see the Memory Heap Feature Manual and the OS API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OsMemoryHeap/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • The memory heap is allocated with nn::os::SetMemoryHeapSize() in nninitStartup().
  • In nnMain(), the size information for the allocated memory heap is sent to a log.
  • Execute the main processing for the sample in MemorySample().
    • Use nn::os::AllocateMemoryBlock() to get the memory block.
    • Get the address and size of the memory block.
    • Use nn::os::FreeMemoryBlock() to return the memory block.
    • Change the obtained memory block size and loop.
  • Return to nnMain() and end.

First, use the nninitStartup() function to set the memory heap size and prepare an environment where the memory heap can be used. Normally, this processing should only be executed when the application starts, it is performed in the nninitStartup() function.

In MemorySample(), nn::os::AllocateMemoryBlock() and nn::os::FreeMemoryBlock() are repeated, while increasing the size, until no more memory can be obtained.

The memory heap used in nn::os::AllocateMemoryBlock() will not exceed the specified size. If no fragmentation occurs, the entire memory heap can be used with AllocateMemoryBlock(), without wasting a single byte.
