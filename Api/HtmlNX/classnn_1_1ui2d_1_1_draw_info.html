Detailed Description

The class for drawing information.

Details
Use this class to control the overall method of drawing and to pass information during drawing.

Before drawing, you must set the GraphicsResource object using the SetGraphicsResource() function.

The content is updated during processing, so the instance cannot be shared by multiple threads. Create duplicates to use on each separate thread.

See also
Layout
Examples
Ui2dDemo/Main.cpp, and Ui2dViewerSimple/main.cpp.

◆ SetUi2dConstantBuffer()

void nn::ui2d::DrawInfo::SetUi2dConstantBuffer ( nn::font::GpuBuffer *  pConstantBuffer)

Sets the constant buffer that ui2d uses for drawing.

Parameters
[in]pConstantBufferPointer to the constant buffer that ui2d uses for drawing.

Details
Specify a GpuBuffer with an allocated size that is at least as large as the value of BuildResultInformation::requiredUi2dConstantBufferSize that can be obtained when Layout::Build is called. To display data set to a shape other than a square in a specially shaped picture pane, a vertex buffer and an index buffer for the display of meshes is allocated inside the region of the GpuBuffer that is passed as the constant buffer. In that case, set nn::gfx::GpuAccess_VertexBuffer and nn::gfx::GpuAccess_IndexBuffer in addition to nn::gfx::GpuAccess_ConstantBuffer for GpuAccessFlag.

Examples
Ui2dDemo/Main.cpp.

◆ ResetDrawState()

void nn::ui2d::DrawInfo::ResetDrawState ( )

Resets the rendering state information.

Details
Retains the rendering state to reduce the number of times configuration is performed when scanning pane trees and rendering. When the application performs other rendering in between rendering the panes, discrepancies occur in the rendering states and produce inappropriate rendering results. For this reason, call this function to reset the rendering information before returning to rendering panes.

void nn::ui2d::DrawInfo::SetDefaultViewportScissorInfo ( const nn::gfx::ViewportStateInfo &  viewportInfo,
const nn::gfx::ScissorStateInfo &  scissorInfo 
)

Sets the viewport and scissor setting information that will be reset after changing the viewport and scissor settings inside the layout runtime.

Details
After scissor settings are changed by a scissor pane, they are reset to the scissor settings set by this method. The viewport information is also used for the following purposes other than resetting settings, so make sure to specify valid values using this function when using these features.

  • Calculation of scissor pane drawing position.
  • Calculation of projection positions for perspective projection texture mappings.
Parameters
[in]viewportInfoViewport information.
[in]scissorInfoScissor information.
