Detailed Description

A structure relating to a sequence sound.

Details
This structure is passed when a sequence sound is played.

To play sequence data stored in memory instead of the sequence data set in the sound archive, specify the address for that sequence data in seqDataAddress. If you specify NULL, the original sequence data as set in the sound archive will be played.

startLocationLabel specifies where to start playback in the sequence data. The specified label string is used instead of the label string set in the sound archive. If NULL is specified, the sequence is played using the original starting position as set in the sound archive.

Use bankIds to specify the bank ID to associate with the sequence data. The ID that you specify is used instead of the bank ID set within the sound archive. Specifying NULL plays back by using the bank ID that was originally set in the sound archive.
