void nn::pcm::Finalize ( )

Finalize the pcm library.

Postcondition
  • The library is in a state prior to initialization.

Details
Finalize the pcm library. The library only enters an uninitialized state when this function is called the same number of times as the library initialization function.

◆ ReadCurrentPower()

int nn::pcm::ReadCurrentPower ( MeasuringPoint  point)

Reads the power consumption, in milliwatts (mW), immediately next to the specified point.

Parameters
[in]pointSpecifies the power measurement identifier.
Returns
Power consumption, in milliwatts (mW).
Precondition
  • The library is initialized.

Details
Gets the power consumption near the specified point in milliwatts (mW). This function will always return 0 if the specified power measurement point is not available. Use nn::pcm::IsSupported() to determine whether the power measurement point is available. To access the measurement device in the function, this function might block for as long as several milliseconds.

Examples
PcmBasic/PcmBasic.cpp.
