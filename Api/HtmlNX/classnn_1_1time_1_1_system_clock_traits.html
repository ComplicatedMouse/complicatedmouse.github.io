Detailed Description

The class that defines the type handling the NintendoSDK system clock.

Details
The type handling the clocks that provide functionality in line with std::chrono::system_clock, meeting the requirements of the C++ standard TrivialClock, is defined with a different name for this class. As a result, all of the time_point instances returned by those clocks can be treated as the same type.
