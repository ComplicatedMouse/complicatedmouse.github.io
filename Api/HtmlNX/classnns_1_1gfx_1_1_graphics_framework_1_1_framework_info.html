◆ SetVirtualWindowCoordinate()

void nns::gfx::GraphicsFramework::FrameworkInfo::SetVirtualWindowCoordinate ( const nn::gfx::util::ScalableViewport::WindowCoordinate &  virtualWindowCoordinate)

Sets the virtual window coordinates.

Parameters
[in]virtualWindowCoordinateThe virtual window coordinates.

Details
The virtual window coordinates configure the coordinate system used for purposes such as setting the viewport and the screen layout for text. Configure the settings as follows if you want to change the rendering resolution while the application is running.
SetDisplayWidth() and SetDisplayHeight() configure the maximum area to render in the application. SetPhysicalWindowCoordinate() sets the area that is actually rendered. SetVirtualWindowCoordinate() sets the virtual window coordinates you want to handle in the application.

Examples
EcCatalog/Main.cpp, HidVibrationGenerator/HidVibrationGenerator.cpp, HidVibrationPlayer/HidVibrationPlayer.cpp, and ImguiDemo/ImguiDemo.cpp.

◆ SetPhysicalWindowCoordinate()

void nns::gfx::GraphicsFramework::FrameworkInfo::SetPhysicalWindowCoordinate ( const nn::gfx::util::ScalableViewport::WindowCoordinate &  physicalWindowCoordinate)

Configures the physical window coordinate settings.

Parameters
[in]physicalWindowCoordinateThe window coordinates to render to.

Details
The physical window coordinates set the coordinate system of the window that actually performs rendering. Configure the settings as follows if you want to change the rendering resolution while the application is running. SetDisplayWidth() and SetDisplayHeight() configure the maximum area to render in the application. SetPhysicalWindowCoordinate() sets the area that is actually rendered. SetVirtualWindowCoordinate() sets the virtual window coordinates you want to handle in the application.

Examples
EcCatalog/Main.cpp, HidVibrationGenerator/HidVibrationGenerator.cpp, HidVibrationPlayer/HidVibrationPlayer.cpp, and ImguiDemo/ImguiDemo.cpp.
