Detailed Description

Buffers for rendering Faceline textures.

Details
Combines things like the buffers required when rendering Faceline textures using TextureShader. You need an instance of this class for each Faceline texture you render.

Attention
It is unnecessary if you are not using TextureShader.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Initializes FacelineGpuBuffer.

Attention
Both pMemory and pMemoryPool must be retained until the call to FacelineGpuBuffer::Finalize().
You cannot re-initialize an instance of FacelineGpuBuffer that has been initialized already. Call this function after calling Finalize().
Parameters
[in]pMemoryThe memory FacelineGpuBuffer will use.
[in]memorySizeThe size of pMemory.
[in]pDeviceThe device.
[in]pMemoryPoolThe memory pool FacelineGpuBuffer will use.
[in]memoryPoolOffsetThe offset to the memory pool FacelineGpuBuffer will use.
[in]memoryPoolSizeThe size of the memory pool FacelineGpuBuffer will use.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && ! FacelineGpuBuffer::IsInitialized() && The offset and size are normal.
Postcondition
If successful, FacelineGpuBuffer::IsInitialized() = true.
