Time Library Samples

Describes the time library samples.

Introduction

Describes the time library samples. The time library features are in the nn::time namespace.

Samples

Sample code is located in the directories starting with the prefix Time under Samples/Sources/Applications/. The following table lists the samples.

Page Content
StandardUserSystemClock Sample A sample program dealing with nn::time::StandardUserSystemClock.
CalculateSpanBetween Describes the sample program for nn::time::CalculateSpanBetween.
Calculate User-Initiated System Clock Changes Sample Describes the sample program showing how to calculate the amount by which the user intentionally changed the user clock (nn::time::StandardUserSystemClock).
ClaimPeriodicBenefitWithUserSystemClock This sample program demonstrates how to use the user clock time (nn::time::StandardUserSystemClock) to perform a clock-operation aware determination of whether a periodic benefit can be claimed.
