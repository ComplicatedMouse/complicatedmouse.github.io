Describes the nn::fs::SetAllocator() sample program.

Overview

This sample uses nn::fs::SetAllocator().

For more information about file and directory operations, see the File System Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/FsSetAllocator/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

This sample creates a directory named C:/Windows/Temp/FsSmplFile/, and then deletes it after the sample is complete.

Note that these entries exist while the sample is running, and that the sample may not run correctly if it cannot access these paths.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • Set an allocator using the SetAllocator function.
  • Call functions for allocating and freeing memory.

Call the SetAllocator function to set an allocator that can be used to allocate memory with the fs library. Set an allocator that returns 8-byte aligned addresses.
