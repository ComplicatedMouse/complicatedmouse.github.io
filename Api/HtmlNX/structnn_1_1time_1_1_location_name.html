Detailed Description

The structure that handles location names.

Details
Specified in the LoadTimeZoneRule() function.

If necessary, you can cache LocationName when you want to keep it fixed while opening an application. You can also save LocationName to the save data, at this point, if you want to use this value again the next time you open an application. Note, however, that discrepancies can arise in values and determinations, calculated from the cached LocationName, when determining items such as time passed until the current time or date, if different from values on the system side.

For more information, see the API For Handling Time Zones documentation.
