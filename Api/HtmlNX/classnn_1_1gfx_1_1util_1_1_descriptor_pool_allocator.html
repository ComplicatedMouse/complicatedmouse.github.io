Detailed Description

This allocator is for breaking slots in a descriptor pool into smaller units for use.

Details
Management-region memory is allocated and freed with a callback function specified by the user.

[in]pFreeFunctionThe callback function for freeing a management region.
[in]pFreeFunctionUserDataThe user-defined parameter called when freeing the management region.
[in]pDescriptorPoolThe descriptor pool.
[in]baseSlotIndexThe index of the start of the descriptor slot to use.
[in]slotCountThe number of descriptor slots to use.
[in]isThreadSafeFlag indicating whether to thread-safe call the Allocate() and Free() functions.
Precondition
IsInitialized() == false.
pAllocateFunction must be a pointer to a valid callback function.
pFreeFunction must be a pointer to a valid callback function.
pDescriptorPool must be a pointer to a valid descriptor pool.
baseSlotIndex >= 0.
slotCount >= 1 && slotCount <= SlotCountMax.
Postcondition
IsInitialized() == true.

Details
This function is not thread-safe.
