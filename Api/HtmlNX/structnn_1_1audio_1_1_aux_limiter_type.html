#include <nn/audio/audio_AuxUtilityEffectLimiterTypes.h>

Detailed Description

The utility of the Limiter effect applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform Limiter effect processing. By specifying the sample data obtained by nn::audio::ReadAuxSendBuffer() as the input for this effect, you can produce a Limiter effect that restricts the sample value within a specific area. This effect processing occurs in the CPU thread that calls the nn::audio::ProcessAuxLimiter() function.

Directly altering member variables is prohibited. Use the main structure through the following functions.
