Detailed Description

Defines the animation targets.

Details
This structure stores animation targets and curve information.

Depending on the animation target, specify one of the following enumerated types for target.

  • nn::ui2d::AnimTargetPane
  • nn::ui2d::AnimTargetVisibility
  • nn::ui2d::AnimTargetPaneColor
  • nn::ui2d::AnimTargetMatColor
  • nn::ui2d::AnimTargetTexSrt
  • nn::ui2d::AnimTargetIndirectSrt
  • nn::ui2d::AnimTargetTexPattern
  • nn::ui2d::AnimTargetFontShadow
  • nn::ui2d::AnimTargetPerCharacterTransform
  • nn::ui2d::AnimTargetPerCharacterTransformCurve
  • nn::ui2d::AnimTargetWindow
  • nn::ui2d::AnimTargetMaskTexMtxSrt
  • nn::ui2d::AnimTargetDropShadow
  • nn::ui2d::AnimTargetProceduralShape

The following data structures are in this structure.

  • ResHermiteKey keys[keyCount]

or

  • ResStepKey keys[keyCount]
