Detailed Description

The startup parameters for calling the controller support UI.

Member Function Documentation

◆ SetDefault()

void nn::hid::ControllerSupportArg::SetDefault ( )

Sets the startup parameters for calling the controller support UI to their default values.

Details
The following values are set as the defaults.

  • playerCountMin : 0
  • playerCountMax : 4
  • enableTakeOverConnection : true
  • enableLeftJustify : true
  • enablePermitJoyDual : true
  • enableSingleMode : false
  • enableIdentificationColor : false
  • enableExplainText : false

bool nn::hid::ControllerSupportArg::enableTakeOverConnection

Specify false to disconnect all controllers.

◆ enableLeftJustify

bool nn::hid::ControllerSupportArg::enableLeftJustify

Specifies whether to left-justify the controller numbers when controller support is ended.

When false is specified, there may be gaps in controller numbers when controller support is ended. Disabled in single-player special mode.

◆ enablePermitJoyDual

bool nn::hid::ControllerSupportArg::enablePermitJoyDual

Specifies whether to permit actions when both controllers are being held in a dual-controller grip.

When false is specified, actions cannot be made when both controllers are being held in a dual-controller grip. This is designed for times like during local communication when you want to prohibit the dual-controller grip.

◆ enableSingleMode

bool nn::hid::ControllerSupportArg::enableSingleMode

Specifies whether to start controller support in dedicated single-player mode.

After single-player special mode is enabled, connections from controllers that are attached to the console will be accepted.

◆ identificationColor

nn::util::Color4u8Type nn::hid::ControllerSupportArg::identificationColor[ControllerSupportPlayerCountMax]

Specifies the colors to use to identify the individual controller numbers shown in the controller support UI.

If enableIdentificationColor is false, the values specified here will not be applied.

◆ explainText

char nn::hid::ControllerSupportArg::explainText[ControllerSupportPlayerCountMax][ExplainTextMaxBufferSize]

The text to use for the individual controller numbers shown in the controller support UI.

You can specify up to nn::hid::ExplainTextMaxLength characters. If enableExplainText is false, the values specified here will not be applied. Check how the text actually displays to make sure it is not too long and otherwise displays appropriately.
