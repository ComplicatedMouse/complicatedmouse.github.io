Detailed Description

This class represents a length of time.

Details
Within this class, a length of time is managed in nanoseconds as a 64-bit integer. This class is used to prevent incorrect units in functions that take time as an argument.

Instances of this class can be created from integer values of various units by using the From*() functions such as FromNanoSeconds(). This class does not provide implicit conversions from integers to this type to avoid ambiguous units, but implicit conversion of 0 to this type is possible.
