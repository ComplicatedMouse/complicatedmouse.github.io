pure virtual

Assemble a series of images into a single mipmap chain.

The images must be given in order (level0, level1, level 2...), and must be sized correctly.

Parameters
[in]srcThe array of input images to assemble into mip chain.
[in]numSourceImagesNumber of images in the src[] array; also indicates the number of mip levels in the resulting image.
[out]dstDestination output mipmapped image.
Returns
TP_OK if the operation was successful or an error code otherwise.
