Detailed Description

The class representing requests to get authorization from a Nintendo Account.

Details
This class performs the following operations related to a request to get authorization from a Nintendo Account.

  • Gets the event that receives notification of the completion of the request process.
  • Gets the state indicating whether the request process is complete.
  • If the request process is complete, gets the result together with the authorization code and the ID token for the request.
  • If the request process is not complete, requests cancellation of that request.

The NintendoAccountAuthorizationRequestContext object takes one of the following states.

  • Invalid
    • The object is not associated with any request.
    • The only call possible in this state is a destructor.
  • Valid
    • This object is associated with a specific request.
    • In this state, these operations can be performed on the request to make a link.

◆ GetResultWithInteractionIfRequired()

Result nn::account::NintendoAccountAuthorizationRequestContext::GetResultWithInteractionIfRequired ( )

Gets the result of the procedure to get authorization.

Returns
Returns a nn::Result that represents the process result.
  • nn::ResultSuccess Indicates that the procedure to get authorization succeeded.
  • nn::account::ResultCancelledByUser Indicates that procedure to get authorization was explicitly canceled intentionally by the user. Do not handle this result as an error.
  • nn::account::ResultCancelled Indicates that the procedure to get authorization was canceled by a call to the Cancel() function. Do not handle this result as an error.
  • nn::account::ResultNetworkCommunicationError Indicates that a problem with network communication occurred during the process. Make every effort not to ignore this error when it occurs. Use the nn::err::ShowError() function to notify the user.
  • nn::ssl::ResultSslService Indicates that a problem with network communication occurred during the process. Make every effort not to ignore this error when it occurs. Use the nn::err::ShowError() function to notify the user.
Precondition
  • *this is valid.
  • The procedure to get authorization is complete.

Details
Gets the result of the procedure to get authorization. If interaction with the user is required for getting authorization, the system UI is displayed and an attempt is made to get authorization through interaction with the user.

This function sometimes blocks the calling thread to display the system UI.

Result nn::account::NintendoAccountAuthorizationRequestContext::GetResult ( )

Gets the result of the procedure to get authorization.

Returns a nn::Result that represents the process result.
  • nn::ResultSuccess Indicates that the procedure to get authorization succeeded.
  • nn::account::ResultCancelled Indicates that the procedure to get authorization was canceled by a call to the Cancel() function. Do not handle this result as an error.
  • nn::account::ResultNintendoAccountAuthorizationInteractionRequired, Indicates that interaction with the user is required for getting authorization. If this error occurs, use the GetResultWithInteractionIfRequired() function and display the system UI to interact with the user. must be handled.

Precondition
  • *this is valid.
  • The procedure to get authorization is complete.

Details
Gets the result of the procedure to get authorization. If interaction with the user is required for getting authorization, the function returns nn::account::ResultNintendoAccountAuthorizationInteractionRequired.

Only use this function for the purpose of momentarily avoiding displaying the system UI. In normal situations, use the GetResultWithInteractionIfRequired() function.

Creates a request to get authorization from a Nintendo Account linked to a network service account.

Parameters
[out]pOutRequestThe address where the created request will be placed.
[in]handleThe UserHandle indicating the target user.
[in]paramThe parameter to use for authentication and authorization procedures with the Nintendo Account server.
[out]workBufferThe work memory required for storing the request.
[in]workBufferSizeThe size (in bytes) of workBuffer.
Returns
Returns a nn::Result that represents the process result.
  • nn::ResultSuccess Succeeded.
  • nn::account::ResultNetworkServiceAccountUnavailable, The network-service account is not available. Make the network service account available by running EnsureNetworkServiceAccountAvailable(), which allows you to resolve this error. must be handled.
Precondition
  • pOutRequest != nullptr.
  • workBuffer != nullptr.
  • reinterpret_cast<uintptr_t>(workBuffer) % nn::os::MemoryPageSize == 0.
  • workBufferSize >= nn::account::RequiredBufferSizeForNintendoAccountAuthorizationRequestContext.
  • workBufferSize % nn::os::MemoryPageSize == 0.
Postcondition
  • *pOutRequest is a valid request.
  • While the created request is valid, a range of workBufferSize bytes in workBuffer cannot be accessed.

Details
Creates a request for the application to get authorization from a Nintendo Account linked to the network-service account for the specified user. After the request is created, the process of getting authorization can be conducted asynchronously and the result can be obtained from the NintendoAccountAuthorizationRequestContext object specified in the parameter.

This function communicates with the Internet.

The memory space that is specified for work memory cannot be accessed while it is storing a request with a valid NintendoAccountAuthorizationRequestContext object.
