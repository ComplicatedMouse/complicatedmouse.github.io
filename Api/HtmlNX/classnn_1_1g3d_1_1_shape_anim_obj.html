◆ BindFast()

virtual void nn::g3d::ShapeAnimObj::BindFast ( const ResModel *  pModel)
virtual

Binds to the animation target.

Parameters
[in]pModelPointer to the ResModel instance of the animation target.
Precondition
  • The ResShapeAnim resource specified by pModel is bound in advance using ResShapeAnim::PreBind.

Details
By using ResShapeAnim::PreBind to bind resources together in advance, a comparatively faster binding is performed without looking up names with BindFast.
