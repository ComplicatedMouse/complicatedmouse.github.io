#include <nn/account/account_Types.h>

Detailed Description

The handle for dealing with a user in the Open state.

Details
When you use the OpenUser() function to change the state of a user to the Open state, a UserHandle object is created to indicate that user. When you destroy this UserHandle, the user indicated by that handle reverts to the Closed state. You can destroy UserHandle with CloseUser().
