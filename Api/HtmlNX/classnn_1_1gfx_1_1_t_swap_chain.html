Initializes the swap chain.

Parameters
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
[in]pMemoryPoolPointer to the memory pool in which to place the scan buffer.
[in]memoryPoolOffsetOffset to the memory pool where the scan buffer is set, in bytes.
[in]memoryPoolSizeSize of the memory pool the scan buffer uses, in bytes.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The swap chain is not initialized.
  • If IsMemoryPoolRequired is true, pMemoryPool != NULL.
  • If IsMemoryPoolRequired is true, memoryPoolSize is larger than the size obtained by CalculateScanBufferSize.
Postcondition
  • The swap chain is initialized.

Details
The memory pool being passed must have been created to be compressible.

◆ AcquireNextScanBufferIndex() [1/2]

template<typename TTarget >
AcquireScanBufferResult nn::gfx::TSwapChain< TTarget >::AcquireNextScanBufferIndex ( int *  pOutScanBufferIndex,
TSemaphore< Target > *  pSemaphore,
TFence< Target > *  pFence 
)

Gets the scan buffer index for the next Present target.

Parameters
[out]pOutScanBufferIndexThe pointer for receiving the scan buffer index that is the next Present target.
[in]pSemaphoreThe pointer to the semaphore that is signaled when the target scan buffer becomes available.
[in]pFenceThe pointer to the fence that is signaled when the target scan buffer becomes available.
Returns
Returns the result of acting to acquire the scan buffer.
Return Values:

AcquireScanBufferResult_Success Acquiring the scan buffer succeeded.

AcquireScanBufferResult_Failed Acquiring the scan buffer failed.

Precondition
  • The swap chain is initialized.
  • pOutScanBufferIndex != NULL.
  • pSemaphore != NULL.
  • pSemaphore is initialized.
  • pSemaphore is not in the signaled state.
  • If pFence != NULL, pFence is initialized.
  • If pFence != NULL, pFence is not in the signaled state.

Details
In the GL version, Sync does not have any effect on pSemaphore and pFence.

Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
