Detailed Description

Class for handling barriers.

Details
Barriers are synchronization objects used to perform waits on multiple threads.

The nn::os::Barrier class is a utility class that wraps calls to synchronization control functions related to nn::os::BarrierType objects. The member functions provided by this class are implemented by issuing functions like nn::os::AwaitBarrier() that take the nn::os::BarrierType type as arguments.
