Detailed Description

Buffers for rendering Mask textures.

Details
This class is for the buffer and other resources needed to draw Mask textures using TextureShader. You need an instance of this class for each Mask texture you render.

Attention
It is unnecessary if you are not using TextureShader.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Details

Attention
Both pMemory and pMemoryPool must be retained until the call to MaskGpuBuffer::Finalize().
You cannot reinitialize an instance of MaskGpuBuffer that has been initialized already. Call this function after calling Finalize().
Parameters
[in]pMemoryThe memory that MaskGpuBuffer will use.
[in]memorySizeThe size of pMemory.
[in]pDeviceThe device.
[in]pMemoryPoolThe memory pool that MaskGpuBuffer will use.
[in]memoryPoolOffsetThe offset to the memory pool that MaskGpuBuffer will use.
[in]memoryPoolSizeThe size of the memory pool that MaskGpuBuffer will use.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && ! MaskGpuBuffer::IsInitialized() && The offset and size are normal.
Postcondition
If successful, MaskGpuBuffer::IsInitialized() = true.

void nn::mii::MaskGpuBuffer::SetColorTarget ( nn::gfx::Device *  pDevice,
nn::gfx::Texture *  pTexture,

int  resolution,
int  mipCount 
)

Sets the texture to write to.

Parameters
[in]pDeviceThe device.
[in]pTextureThe pointer to the texture to write to.
[in]infoTextureShaderInfo for TextureShader.
[in]resolutionThe resolution of the texture to write to. Specify the resolution of the Mask texture set using nn::mii::CharModelInfo::SetDynamicTextureResolution(int facelineHeightValue,int mask).
[in]mipCountThe mipmap count of the texture to write to.
Precondition
pDevice != NULL. && pTexture != NULL &&: pTexture is initialized. && A valid texture format is set in info. && 0 < resolution && 0 < mipCount.

Details
Sets the texture to write to. Mask textures are textures with same size for the width and height. nn::gfx::ColorTargetView is constructed internally. Processes after Finalize is run on the previous instance of nn::gfx::ColorTargetView. This passed instance of MaskGpuBuffer was configured during TextureShader initialization. Pass TextureShaderInfo.
