Detailed Description

Structure for storing information used to determine whether a periodic benefit can be claimed.

Details
This class stores information used to implement the nn::time::ClaimPeriodicBenefitWithUserSystemClock() feature.

This structure satisfies the standard C++ requirements for being trivially copyable and can be copied with std::memcpy().

See also
nn::time::ClaimPeriodicBenefitWithUserSystemClock()
Examples
TimeClaimPeriodicBenefit/TimeClaimPeriodicBenefit.cpp.

Parameters

Precondition

Returns
Returns whether the time was successfully obtained.
Return Values:

true Success.

false Failure (no penalty period is currently in effect).

Details
We do not recommend using this function.
Use the nn::time::PeriodicBenefitClaimContext::GetSpanToEndOfPenaltyAtLastClaimForDebug function instead.

Gets an nn::time::SteadyClockTimePoint object representing the instant when a penalty period ends on the basis of the last determination results from nn::time::ClaimPeriodicBenefitWithUserSystemClock().

In some cases, the obtained time may have already occurred, or the returned time may not be sequential to the last value obtained from nn::time::StandardSteadyClock.

This function is only intended for use during development and cannot be used in the retail environment. For this reason, the following types of implementations are prohibited.

  • Displaying the time returned by this function or other information calculated based on this time in a UI in the retail environment
  • Using the time returned by this function to determine whether a benefit can be claimed
See also
nn::time::StandardSteadyClock::GetCurrentTimePoint(), nn::time::GetSpanBetween()

bool nn::time::PeriodicBenefitClaimContext::GetSpanToEndOfPenaltyAtLastClaimForDebug ( nn::TimeSpan *  pOut) const

Gets the time until the penalty period ends.

Using this function for retail products is prohibited.

Parameters
[out]pOutThe time when the penalty period ends.
Precondition
  • pOut != nullptr.
Returns
Whether getting the time succeeded.
Return Values:

true Success.

false Failure.

Details
Returns the time until the penalty period ends on the basis of the last determination results from nn::time::ClaimPeriodicBenefitWithUserSystemClock().

Gets a negative nn:TimeSpan value if the penalty time has already ended.

Returns false if one of the following apply.

  • The nn::time::ClaimPeriodicBenefitWithUserSystemClock() function has not been called using the target nn::time::PeriodicBenefitClaimContext object.
  • No penalty period occurred with the preceding nn::time::ClaimPeriodicBenefitWithUserSystemClock() function.
  • A monotonic increase clock occurred after the preceding nn::time::ClaimPeriodicBenefitWithUserSystemClock() function.

This function is only intended for use during development and cannot be used in the retail environment. For this reason, the following types of implementations are prohibited.

  • Displaying the time returned by this function or other information calculated based on this time in a UI in the retail environment
  • Using the time returned by this function to determine whether a benefit can be claimed
