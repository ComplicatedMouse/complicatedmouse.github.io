Description of the Opus decoder sample program.

Overview

This sample plays back audio while decoding with the Opus decoder any PCM data that was encoded with the Opus encoder. An audio renderer is used to play the audio.

File Structure

This sample program is located in Samples/Sources/Applications/CodecOpusEncoder/ .

Required Environment

You must be able to use audio output.

How to Use

Roughly a minute's worth of background music is played and the program ends automatically when you run the sample program.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

151 {
152 #ifdef USE_SILK_CODING
153  NNS_LOG("CodecOpusEncoder sample (using SILK coding)\n");
154 #else // USE_SILK_CODING
155  NNS_LOG("CodecOpusEncoder sample (using CELT coding)\n");
156 #endif // USE_SILK_CODING
157 
158  nn::mem::StandardAllocator allocator(g_HeapBuffer, sizeof(g_HeapBuffer));
159  nn::mem::StandardAllocator pcmBufferAllocator(g_PcmBuffer, sizeof(g_PcmBuffer));
160 
161  // Initialize the file system.
162  InitializeFileSystem();
163 
164  // Load sound source data.
165  const char Filename[] = "asset:/SampleBgm0-1ch.wav";
166  nn::fs::FileHandle handle;
167  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::fs::OpenFile(&handle, Filename, nn::fs::OpenMode_Read));
168  int64_t size;
169  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::fs::GetFileSize(&size, handle));
170  auto const pWaveDataHead = new uint8_t[static_cast<size_t>(size)];
171  NN_ABORT_UNLESS_NOT_NULL(pWaveDataHead);
172  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::fs::ReadFile(handle, 0, pWaveDataHead, static_cast<std::size_t>(size)));
173  nn::fs::CloseFile(handle);
174 
175  const int8_t* pWaveData = reinterpret_cast<const int8_t*>(pWaveDataHead);
176 
177  // Get the sample rate and the number of channels from the header.
178  nns::audio::WavFormat wavFormat;
179  auto parseResult = ParseWavFormat(&wavFormat, pWaveDataHead, static_cast<size_t>(size));
180  NN_ABORT_UNLESS(parseResult == nns::audio::WavResult_Success);
181  const int channelCount = wavFormat.channelCount;
182  const int encoderSampleRate = wavFormat.sampleRate;
183  const int64_t sampleCountPerChannel = wavFormat.dataSize / (wavFormat.bitsPerSample >> 3) / channelCount;
184 
185  // Initialize the Opus encoder.
186  nn::codec::OpusEncoder encoder;
187  void* encoderWorkBuffer = std::malloc(encoder.GetWorkBufferSize(encoderSampleRate, channelCount));
188  NN_ABORT_UNLESS_NOT_NULL(encoderWorkBuffer);
189  nn::codec::OpusResult opusResult = encoder.Initialize(encoderSampleRate, channelCount, encoderWorkBuffer, encoder.GetWorkBufferSize(encoderSampleRate, channelCount));
190  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
191 
192  // Set the bit rate of the encoder.
193  const int HigherBitRate = 60000;
194  const int LowerBitRate = 12000;
195  encoder.SetBitRate(HigherBitRate);
196 
197  // Initialize the Opus decoder.
198  const int decoderSampleRate = 48000;
199  nn::codec::OpusDecoder decoder;
200  void* decoderWorkBuffer = std::malloc(decoder.GetWorkBufferSize(decoderSampleRate, channelCount));
201  NN_ABORT_UNLESS_NOT_NULL(decoderWorkBuffer);
202  opusResult = decoder.Initialize(decoderSampleRate, channelCount, decoderWorkBuffer, decoder.GetWorkBufferSize(decoderSampleRate, channelCount));
203  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
204 
205  //Build the audio renderer.
206  nn::audio::AudioRendererParameter audioRendererParameter;
207  nn::audio::InitializeAudioRendererParameter(&audioRendererParameter);
208  audioRendererParameter.sampleRate = decoderSampleRate;
209  audioRendererParameter.sampleCount = 240;
210  audioRendererParameter.mixBufferCount = 2;
211  audioRendererParameter.voiceCount = 1;
212  audioRendererParameter.sinkCount = 1;
213  NN_ABORT_UNLESS(nn::audio::IsValidAudioRendererParameter(audioRendererParameter));
214 
215  std::size_t audioRendererWorkBufferSize = nn::audio::GetAudioRendererWorkBufferSize(audioRendererParameter);
216  void* audioRendererWorkBuffer = allocator.Allocate(audioRendererWorkBufferSize, nn::os::MemoryPageSize);
217  NN_ABORT_UNLESS_NOT_NULL(audioRendererWorkBuffer);
218 
219  nn::audio::AudioRendererHandle audioRendererHandle;
220  nn::os::SystemEvent systemEvent;
221  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::OpenAudioRenderer(&audioRendererHandle, &systemEvent, audioRendererParameter, audioRendererWorkBuffer, audioRendererWorkBufferSize));
222 
223  std::size_t configBufferSize = nn::audio::GetAudioRendererConfigWorkBufferSize(audioRendererParameter);
224  void* configBuffer = allocator.Allocate(configBufferSize, nn::os::MemoryPageSize);
225  NN_ABORT_UNLESS_NOT_NULL(configBuffer);
226  nn::audio::AudioRendererConfig config;
227  nn::audio::InitializeAudioRendererConfig(&config, audioRendererParameter, configBuffer, configBufferSize);
228 
229  nn::audio::FinalMixType finalMix;
230  nn::audio::AcquireFinalMix(&config, &finalMix, 2);
231 
232  nn::audio::DeviceSinkType sink;
233  int8_t bus[2] = { 0, 1 };
234  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::AddDeviceSink(&config, &sink, &finalMix, bus, sizeof(bus) / sizeof(bus[0]), "MainAudioOut"));
235 
236  // Prepare a memory pool to maintain the sample data added to WaveBuffer.
237  nn::audio::MemoryPoolType pcmBufferMemoryPool;
238  NN_ABORT_UNLESS(nn::audio::AcquireMemoryPool(&config, &pcmBufferMemoryPool, g_PcmBuffer, sizeof(g_PcmBuffer)));
239  NN_ABORT_UNLESS(nn::audio::RequestAttachMemoryPool(&pcmBufferMemoryPool));
240 
241  nn::audio::VoiceType voice;
242  nn::audio::AcquireVoiceSlot(&config, &voice, decoderSampleRate, channelCount, nn::audio::SampleFormat_PcmInt16, nn::audio::VoiceType::PriorityHighest, nullptr, 0);
243  nn::audio::SetVoiceDestination(&config, &voice, &finalMix);
244  nn::audio::SetVoicePlayState(&voice, nn::audio::VoiceType::PlayState_Play);
245  nn::audio::SetVoiceMixVolume(&voice, &finalMix, 1.0f, 0, 0);
246  nn::audio::SetVoiceMixVolume(&voice, &finalMix, 1.0f, 0, 1);
247 
248  // Prepare buffers for streaming playback.
249  // Pass in buffers initialized to 0, which get filled with the results of decoding the Opus data as part of the main loop.
250  nn::audio::WaveBuffer waveBuffer[BufferCount];
251  void* pcmBuffer[BufferCount];
252  for (int i = 0; i < BufferCount; ++i)
253  {
254  const std::size_t pcmBufferSize = OpusSampleCount * OpusChannelCount * sizeof(int16_t);
255  pcmBuffer[i] = pcmBufferAllocator.Allocate(pcmBufferSize, nn::audio::BufferAlignSize);
256 
257  std::memset(pcmBuffer[i], 0, pcmBufferSize);
258 
259  waveBuffer[i].buffer = pcmBuffer[i];
260  waveBuffer[i].size = sizeof(g_PcmBuffer[i]);
261  waveBuffer[i].startSampleOffset = 0;
262  waveBuffer[i].endSampleOffset = static_cast<int32_t>(sizeof(g_PcmBuffer[i]) / sizeof(int16_t));
263  waveBuffer[i].loop = false;
264  waveBuffer[i].isEndOfStream = false;
265  waveBuffer[i].pContext = nullptr;
266  waveBuffer[i].contextSize = 0;
267 
268  nn::audio::AppendWaveBuffer(&voice, &waveBuffer[i]);
269  }
270 
271  // Start the audio renderer.
272  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::StartAudioRenderer(audioRendererHandle));
273 
274  // Get the PCM data itself.
275  auto p = reinterpret_cast<const int16_t*>(pWaveData);
276  int bufferCount = BufferCount;
277  const size_t encodedDataSizeMaximum = nn::codec::OpusPacketSizeMaximum;
278 
279  // Frames can be changed at the encoding level.
280  // Use encoding that switches, in a cycle, from 5 to 10 to 2.5 to 20 milliseconds.
281  const int encodeFrameDurationCount = 4;
282 #if !defined(USE_SILK_CODING)
283  const auto codingMode = nn::codec::OpusCodingMode_Celt;
284  const int encodeFrameDurations[encodeFrameDurationCount] =
285  {
286 // // If the encodeFrameDuration value is set to less than 5000, note that the processing may
287 // // not be performed in real time depending on the environment.
288 // 2500,
289 // 5000,
290  10000,
291  20000,
292  10000,
293  20000
294  };
295 #else // ! defined(USE_SILK_CODING)
296  const auto codingMode = nn::codec::OpusCodingMode_Silk;
297  const int encodeFrameDurations[encodeFrameDurationCount] =
298  {
299  10000,
300  20000,
301  10000,
302  20000
303  };
304 #endif // ! defined(USE_SILK_CODING)
305  // You can specify an encryption mode before calling EncodeInterleaved().
306  encoder.BindCodingMode(codingMode);
307 
308  const int encodeSampleCountMaximum = encoder.CalculateFrameSampleCount(20000);
309  auto encodedDataBuffer = new uint8_t[encodedDataSizeMaximum];
310  auto decodedDataBuffer = new int16_t[encodeSampleCountMaximum * channelCount];
311  auto inputDataBuffer = new int16_t[encodeSampleCountMaximum * channelCount];
312  int sampleCountTotal = 0;
313  int encodeFrameDurationIndex = 0;
314 
315  for (auto remaining = sampleCountPerChannel; ; )
316  {
317  systemEvent.Wait();
318  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::RequestUpdateAudioRenderer(audioRendererHandle, &config));
319 
320  // If playback is complete for any buffers, start the Opus decoding process.
321  if (auto pReleasedWaveBuffer = nn::audio::GetReleasedWaveBuffer(&voice))
322  {
323  nn::audio::WaveBuffer* pWaveBuffer = nullptr;
324  int16_t* pPcmBuffer = nullptr;
325  for (int i = 0; i < BufferCount; ++i)
326  {
327  if (pReleasedWaveBuffer == &waveBuffer[i])
328  {
329  pWaveBuffer = &waveBuffer[i];
330  pPcmBuffer = static_cast<int16_t*>(pcmBuffer[i]);
331  break;
332  }
333  }
334  if (remaining > 0) // < sampleCountPerChannel / encodeSampleCount)
335  {
336  // You can switch the bit rate at the encoding level.
337  // Switches between low and high bit rates for every 10,000
338  // samples processed, for the following processing.
339  if (sampleCountTotal >= 10000)
340  {
341  int bitRate = encoder.GetBitRate();
342  if (HigherBitRate == bitRate)
343  {
344  encoder.SetBitRate(LowerBitRate);
345  }
346  else
347  {
348  encoder.SetBitRate(HigherBitRate);
349  }
350  sampleCountTotal -= 10000;
351  }
352 
353  // Encoding.
354  const int encodeSampleCount = encoder.CalculateFrameSampleCount(encodeFrameDurations[encodeFrameDurationIndex++]);
355  encodeFrameDurationIndex = encodeFrameDurationIndex % encodeFrameDurationCount;
356  size_t copySampleCount = encodeSampleCount;
357  // Fractional processing.
358  if (remaining < encodeSampleCount)
359  {
360  copySampleCount = static_cast<int>(remaining);
361  std::memset(inputDataBuffer, 0x0, sizeof(int16_t) * channelCount * encodeSampleCount);
362  }
363  std::memcpy(inputDataBuffer, p, sizeof(int16_t) * channelCount * copySampleCount);
364  size_t encodedDataSize = 0;
365  nn::codec::OpusResult encodeStatus = encoder.EncodeInterleaved(
366  &encodedDataSize,
367  encodedDataBuffer,
368  encodedDataSizeMaximum,
369  inputDataBuffer,
370  encodeSampleCount);
371  NN_ABORT_UNLESS(encodeStatus == nn::codec::OpusResult_Success);
372  p += encodeSampleCount * channelCount;
373 
374  // Decoding
375  std::size_t consumed = 0;
376  int sampleCount = 0;
377  const int decodeBufferSize = encodeSampleCount * channelCount * sizeof(int16_t) * decoderSampleRate / encoderSampleRate;
378  opusResult = decoder.DecodeInterleaved(
379  &consumed,
380  &sampleCount,
381  pPcmBuffer,
382  decodeBufferSize,
383  encodedDataBuffer,
384  encodedDataSize);
385  NN_ABORT_UNLESS(opusResult == nn::codec::OpusResult_Success);
386  NN_ABORT_UNLESS(consumed == encodedDataSize);
387 
388  // Add decoded results to the Voice playback buffer.
389  pWaveBuffer->size = sampleCount * sizeof(int16_t);
390  pWaveBuffer->endSampleOffset = sampleCount;
391  pWaveBuffer->loop = false;
392  pWaveBuffer->isEndOfStream = false;
393  nn::audio::AppendWaveBuffer(&voice, pWaveBuffer);
394 
395  remaining -= encodeSampleCount;
396  sampleCountTotal += encodeSampleCount;
397 
398  }
399  // If no data remains, wait until playback of all the buffers is complete.
400  else if (--bufferCount == 0)
401  {
402  break;
403  }
404  }
405  }
406 
407  // Release the buffers.
408  for (int i = 0; i < BufferCount; ++i)
409  {
410  pcmBufferAllocator.Free(pcmBuffer[i]);
411  }
412  // Issue a request to detach the memory pools that were previously being used.
413  nn::audio::RequestDetachMemoryPool(&pcmBufferMemoryPool);
414  // Wait for the memory pools to be detached.
415  while (nn::audio::IsMemoryPoolAttached(&pcmBufferMemoryPool))
416  {
417  nn::audio::GetReleasedWaveBuffer(&voice);
418 
419  systemEvent.Wait();
420  NN_ABORT_UNLESS_RESULT_SUCCESS(nn::audio::RequestUpdateAudioRenderer(audioRendererHandle, &config));
421  }
422  // Release MemoryPool.
423  nn::audio::ReleaseMemoryPool(&config, &pcmBufferMemoryPool);
424 
425  // Finalize the audio renderer, and free the memory.
426  nn::audio::StopAudioRenderer(audioRendererHandle);
427  nn::audio::CloseAudioRenderer(audioRendererHandle);
428  nn::os::DestroySystemEvent(systemEvent.GetBase());
429  allocator.Free(audioRendererWorkBuffer);
430  allocator.Free(configBuffer);
431 
432  delete[] inputDataBuffer;
433  delete[] decodedDataBuffer;
434  delete[] encodedDataBuffer;
435  delete[] pWaveDataHead;
436 
437  // Terminate the Opus decoder and encoder, and destroy the memory.
438  encoder.Finalize();
439  decoder.Finalize();
440  std::free(encoderWorkBuffer);
441  std::free(decoderWorkBuffer);
442 
443  // Finalize the file system.
444  FinalizeFileSystem();
445 
446 } // NOLINT(readability/fn_size)

Sample Program Description

This sample program plays back the PCM data that results when decoding the Opus data created when encoding PCM data.

This sample program has the following flow.

  • Initialize the file system.
  • Load PCM data.
  • Initialize the Opus encoder.
  • Initialize the Opus decoder.
  • Initialize and start the audio renderer.
  • Run the Opus encoder and decoder and play back the decoding results.
  • Finalize the audio renderer.
  • Finalize the Opus encoder.
  • Finalize the Opus decoder.
  • Finalize the file system.

First, initialize the file system. It uses sound source data in the WAV format for resource files. Change the resource directory as needed.

Loads the sound source data and the PCM data from a file.

The Opus encoder is initialized based on the header information.

You must pass the sample rate, number of channels, and a working buffer sized appropriately according to the previous two parameters to the nn::codec::OpusEncoder::Initialize() function.

Use the nn::codec::OpusEncoder::GetWorkBufferSize() function to get the working buffer size.

Initialize the Opus decoder. For more information about nn::codec::OpusDecoder, see the API Reference and the Samples/Sources/Applications/OpusDecoder/ sample.

Get the audio renderer ready. For more information about audio renderers, see the API Reference and the Samples/Sources/Applications/AudioRenderer/ sample.

The main loop implements a real-time streaming playback process. The Opus decoder is used to decode PCM data that was encoded with the Opus encoder. This data may have been encoded by the distributor of the audio data. Decoded PCM data can be decoded by the downloader of the distributed audio data. PCM data is added to a buffer where playback has ended, and playback occurs consecutively using nn::audio::VoiceType.

Encodes all PCM data. Halts the audio renderer when playback ends and finalizes the Opus encoder and decoder, and the audio renderer, and then destroys the memory.
