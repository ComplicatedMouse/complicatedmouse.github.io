Detailed Description

The class that builds the FogAnimObj instance.

Constructor & Destructor Documentation

◆ Builder()

nn::g3d::FogAnimObj::Builder::Builder ( )

Constructor.

You must either call Reserve() on all fog animation resources used in the instance, or use SetMaxCurveCount() to specify the parameters required to calculate the buffer size.
