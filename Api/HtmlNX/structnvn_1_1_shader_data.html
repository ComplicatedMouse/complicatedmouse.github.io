Detailed Description

Data structure representing the shader data.

The data structure is for one shader stage. The contents of this struct contain everything NVN needs to set up the program for use with Program::SetShaders

Examples
NvnSimple/NvnSimpleSharedLib/GraphicsShader.cpp.
