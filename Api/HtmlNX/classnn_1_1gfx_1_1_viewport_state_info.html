A helper function for setting the default value for each parameter.

Postcondition

Details
Equivalent to calling the following.

  • SetOriginX( 0.0f );
  • SetOriginY( 0.0f );
  • SetWidth( 0.0f );
  • SetHeight( 0.0f );
  • SetMinDepth( 0.0f );
  • SetMaxDepth( 1.0f );
