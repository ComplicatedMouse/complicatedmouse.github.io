◆ allocateTrackFlags

uint16_t nn::atk::SoundArchive::StreamSoundInfo::allocateTrackFlags

A bit flag representing tracks that are included in the stream sound.

This represents track 0, track 1, track 2, and so on in order starting at the least-significant bit. Tracks for valid bits can be manipulated.
