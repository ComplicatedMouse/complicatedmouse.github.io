#include <nn/audio/audio_EffectCaptureTypes.h>

Detailed Description

An effect for incorporating the process of getting sample data.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • AddCapture(AudioRendererConfig* pOutConfig, CaptureType* pEffect, FinalMixType* pFinalMix, void* buffer, size_t bufferSize)
  • AddCapture(AudioRendererConfig* pOutConfig, CaptureType* pEffect, SubMixType* pSubMix, void* buffer, size_t bufferSize)
  • RemoveCapture(AudioRendererConfig* pOutConfig, CaptureType* pEffect, FinalMixType* pFinalMix)
  • RemoveCapture(AudioRendererConfig* pOutConfig, CaptureType* pEffect, SubMixType* pSubMix)
  • GetCaptureChannelCountMax(const CaptureType* pEffect)
  • GetCaptureInput(const CaptureType* pEffect, int8_t* pOutInput, int* pOutCount, int count)
  • SetCaptureInput(CaptureType* pEffect, const int8_t* input, const int count)
  • ReadCaptureBuffer(CaptureType* pEffect, int32_t* pOutData, int count)
  • GetCaptureSampleRate(const CaptureType* pEffect)
  • GetCaptureSampleCount(const CaptureType* pEffect)
  • IsCaptureEnabled(const CaptureType* pEffect)
  • SetCaptureEnabled(CaptureType* pEffect, bool enable)
  • IsCaptureRemovable(const CaptureType* pEffect)
  • GetCaptureLostSampleCount(const CaptureType* pEffect)
  • ResetCaptureLostSampleCount(CaptureType* pEffect)

To use these functions, include nn/audio.h.
