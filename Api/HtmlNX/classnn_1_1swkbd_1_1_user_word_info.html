void* nn::swkbd::UserWordInfo::pWorkBuf

Working buffer used when creating a user dictionary.

Specify a buffer at least as big as the size returned by nn::swkbd::UserWordInfo::GetRequiredWorkBufferSize() (in bytes). This space must be aligned to nn::os::MemoryPageSize.
