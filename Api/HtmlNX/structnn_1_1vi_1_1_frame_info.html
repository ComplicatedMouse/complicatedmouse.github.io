Detailed Description

Contains frame status information.

Details
Applications should check the status member to determine what fields are available:

  • FrameStatus_Unknown: No fields are valid.
  • FrameStatus_Enqueued: frameNumber and enqueueTime are valid.
  • FrameStatus_Presented: All fields are valid.

Values of unavailable fields are undefined.

All timestamps will relate to values from nn::os::GetSystemTick().

Examples
ViTimestamp/ViTimestamp.cpp.
