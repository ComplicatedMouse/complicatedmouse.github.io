virtual

Searches for text based on the text ID.

UTF-16 version.

Details
The function is called for each TextBox object when a TextSearcher is set as a parameter of the Layout class Build member function.

Note that the function is called even if no text ID is set in the TextBox object. In that case, the messageId parameter takes NULL.

Search for the text corresponding to the specified text ID and assign it to the pTextInfo parameter. If the function returns without setting the text parameter of TextInfo, the string specified by Layout Editor is entered in the text box.

Parameters
[in]pTextInfoInformation about the text settings. forceAssignedTextLength is an input parameter, and all of the other settings are output parameters. For more information about configuring these settings, see TextInfo.
[in]messageIdThe text ID set to the text box. Becomes NULL if the text ID has not been set.
[in]pLayoutThe layout storing the text box. Note that the layout is still being built. You can use GetName.
[in]pTextBoxThe text box where text is set.
[in]pResourceLayoutThe layout that includes the resource to use when building the text box. When this layout is overwritten using the parts properties, the function returns the pointer to that overwritten layout. In all other cases, the same pointer as that for the pLayout parameter is passed.
