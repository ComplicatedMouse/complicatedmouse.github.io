Detailed Description

The structure representing the input states for the NpadStyleFullKey play style modeled on holding an Npad Switch Pro Controller.

Details
Provides input states for the digital buttons (A, B, X, Y, L, R, ZL, ZR, +, -, and the directional buttons) and the right and left control sticks, modeled on the Switch Pro Controller.

There are rules governing the input from the +Control Pad: If up/down or left/right are pressed at the same time, "up" and "left" take precedence. You can get the controller connection state using attributes::IsConnected.
