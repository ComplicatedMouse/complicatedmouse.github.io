)

Sets an emission time matrix and manually emits particles.

Details
A pointer to user data can be specified as a second argument. You must be careful to avoid destroying the content that user data pointer indicates until particle generating processes are completed. To manually emit two or more particles within one frame and have each particle take on different user data, you must prepare an amount of user data equivalent to the maximum number of particles to be emitted (which is specified as an argument of the nn::vfx::System::CreateManualEmitterSetId() function) and pass their addresses to the EmitParticle() function. Note that the scale component for the specified emission matrix is used for emitter size, not particle size.
