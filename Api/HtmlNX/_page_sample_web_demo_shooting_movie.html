Overview

This offline HTML sample is for displaying the game technique used in a 3D shooter. This sample demonstrates the following features.

  • Playing video in full-screen mode
  • Displaying animation images inline

File Structure

This sample offline HTML is located in Samples/Sources/Applications/WebDemo/html_document/sample/shooting_movie.htdocs.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

Input Operation
A Button Plays video in full-screen mode.
B Button Exits the offline web applet and returns to the menu, via the application.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and display the offline HTML from this sample.

Playing Video in Full-Screen Mode

Select HOW TO PLAY from the offline HTML to play the video in full-screen mode. Placing either a video element or an a element that links to a video file can play the video in full-screen mode when those are selected. Configured to automatically close the video player and to return to the web page when the video file has played through to the end.

Displaying Animation Images Inline

Replacing the video file with an animated GIF (img/shooting_loop.gif) will display that animation inline.
