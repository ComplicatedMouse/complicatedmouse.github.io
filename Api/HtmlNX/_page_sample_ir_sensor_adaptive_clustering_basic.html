Describes the sample program of simple controls to use the IR Motion Camera adaptive clustering mode.

Overview

Describes how to use the IR Motion Camera adaptive clustering mode.

File Structure

This sample program is located in Samples/Sources/Applications/IrSensorAdaptiveClusteringBasic .

Required Environment

When running in a Windows environment, connect the Bluetooth dongle to the computer ahead of time, and pair the controller to the computer. When running in an SDEV or EDEV environment, pair the controller to the SDEV or EDEV ahead of time.

How to Use

Run the sample program and press any of the controller buttons. After the Bluetooth connection is made, the LED on the controller stops flashing and remains lit.

After connection, press a button on the controller to get the current adaptive clustering mode state for that controller. The retrieved state is output to the console log.

To end the sample program, press either the + Button or the - Button.

Important Information

Make sure that the controller is sufficiently charged before using it. This sample program does not draw anything on the screen. We recommend making function calls from a thread other than the main loop because the IR Motion Camera API has functions that block the thread for extended periods. For more information, see the API Reference.

Execution Procedure

Build the sample program and then run it.

Description

This sample program has the following flow.

  • Set NpadID.
  • Get the IR Motion Camera handle from the NpadID.
  • Get the IR Motion Camera state.
  • Get the adaptive clustering mode processing result.
