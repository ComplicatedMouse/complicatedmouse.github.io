Member Enumeration Documentation

◆ Operation

template<typename CharType >
enum nn::font::TagProcessorBase::Operation

An enumerated type for processing that must be performed by the caller (TextWriter).

Enumerator
Operation_Default 

Do not leave space before the next character at the start of a line.

Leave space before the next character anywhere except at the start of a line.

Operation_NoCharSpace 

When processing a tag in a position other than the beginning of a line, by default, a space is left between it and the next character, but do not leave this space.

Operation_CharSpace 

When processing a tag at the beginning of a line, by default, no space is left between it and the next character, but leave a space in this case.

Operation_NextLine 

Adjust x-coordinates during a line break, but skip all other required processing.

Do not adjust y-coordinates. Also, do not leave space before the next character.

Operation_EndDraw 

Stops drawing a string midway.

Operation_MaxOperation 

This is an internal feature so its use is prohibited.

virtual

Gets the height of the row.

Details
The default implementation returns the value calculated by line feed width of the font × y-scale + line spacing. When using tags to change character scale, use this method as an override to adjust the row height when there is more empty space in the row height than expected.

Parameters
[in]pTextWriterPointer to TextWriter.
