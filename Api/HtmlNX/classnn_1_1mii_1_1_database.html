◆ Initialize()

nn::Result nn::mii::Database::Initialize ( )

Start process.

Returns
Returns nn::ResultSuccess when successful.
Postcondition
Succeeding databases become available.

Details
Starts use of the database. To use the database, you must first call this function and allow processing to succeed. pResult is enabled when nn::ResultSuccess() is called.

◆ IsUpdated()

bool nn::mii::Database::IsUpdated ( int  sourceFlags)

Determines whether the state has been updated.

Parameters
[in]sourceFlagsSourceFlag.
Returns
Returns true if there has been an update.

Details
Determines whether there has been an update, addition, or deletion for a Mii.
Returns true when a Mii maintained by the system has been updated. Updates occur during transitions between applications and applets.
This function can return true when returning from applications or applets.
This is a one-shot member function.
Note that calling this function will clear the state.

◆ IsFullDatabase()

bool nn::mii::Database::IsFullDatabase ( ) const

Determines whether the database is full.

Returns
Returns true when full.

Details
Use this member function instead of determining the equivalence of GetCount(SourceFlag_Database) and DatabaseMiiCount. The database can be full even when GetCount(SourceFlag_Database) is not equal to DatabaseMiiCount.

Converts 3DS/Wii U StoreData to CharInfo.

Parameters
[out]pInfoSpecifies the storage location of converted CharInfo.
[in]ver3StoreDataSpecifies Ver3StoreData to convert.
Returns
Returns a nn::Result that represents the process result.
  • nn::ResultSuccess Successful conversion.
  • nn::mii::ResultBroken must be handled.

Details
Converts 3DS/Wii U StoreData to CharInfo.
Nicknames that contain profanity have the number of characters of the nickname replaced with "?".
Nicknames that contain an invalid character code are replaced by "???".
For character codes that can be used as Mii nicknames, see the character filter in NintendoSDK/Resources/Mii/fontset.
