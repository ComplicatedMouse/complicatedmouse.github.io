Detailed Description

Structure representing the sound profile information.

Indicates the profile information in the specified audio frame.

Details
Call nn::audio::SetPerformanceFrameBuffer and store the value calculated from the obtained nn::audio::PerformanceInfo.

nwFrameProcess stores the time to process the sound thread that is measured within the atk library.

nwFrameProcess

   - The time spent to process parameter updates for players, sounds, and tracks.
   - Processing takes longer in proportion to the number of sounds, the number of tracks for these sounds,

the number of channels, and the number of note-ons.

voiceProcess

   - The time spent to process voice rendering in the <tt>nn::audio</tt> audio renderer used internally.
   - Processing takes longer in proportion to the number of voices in playback.

mainMixProcess

   - The time spent for processing the mix that performs effect processing in the <tt>nn::audio</tt> audio renderer used internally.
   - Processing takes longer in proportion to the number and

type of effects being appended by SoundSystem::AppendEffect.

rendererFrameProcess

   - The time spent for all processing performed by the <tt>nn::audio</tt> audio renderer used internally.
   - This time is the sum total of the time used for <tt><var>voiceProcess</var></tt>, <tt><var>mainMixProcess</var></tt>, <tt><var>finalMixProcess</var></tt>,

sinkProcess, circularBufferSinkProcess and the time spent for audio renderer system processing.

◆ rendererFrameProcess

TimeRange nn::atk::SoundProfile::rendererFrameProcess

The total time spent for processing by the internally used nn::audio audio renderer.

Corresponds to the value that can be obtained using nn::audio::PerformanceInfo::GetTotalProcessingTime. Note that this total includes the time spent for the audio renderer's system processes, so it does not necessarily match the sum of the times of voiceProcess, mainMixProcess, finalMixProcess, sinkProcess, and circularBufferSinkProcess. In particular, be careful on a PC environment when this may be blocked by processing from other processes on the same PC, resulting in greater discrepancies.

◆ isRendereringTimeLimitExceeded

Corresponds to the value that can be obtained using nn::audio::PerformanceInfo::IsRenderingTimeLimitExceeded, which indicates whether the audio renderer processing time has exceeded the time limit for rendering.

◆ _additionalSubMixProcess

TimeRange nn::atk::SoundProfile::_additionalSubMixProcess

The use of internal features is prohibited. The time spent for submix processing in the nn::audio audio renderer used internally when additional effect buses in SoundSystemParam can be used.

This time includes the processing time for effects for the additional effect buses. Do not reference this value if the additional effect buses cannot be used. In addition, note that the time spent for _additionalSubMixProcess will be added to rendererFrameProcess when additional effect buses can be used.
