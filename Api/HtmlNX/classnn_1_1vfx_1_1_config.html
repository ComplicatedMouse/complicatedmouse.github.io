Detailed Description

The config to set when creating an effect system.

Details
You must specify a heap with nn::vfx::Config::SetEffectHeap() or nn::vfx::Config::SetEffectDynamicHeap() to start the effect system. If necessary, set the buffer size appropriately. For more information, see the effect tutorial documentation.

See also
nn::vfx::Config::SetEffectHeap(), nn::vfx::Config::SetEffectDynamicHeap()
Examples
VfxBinarySegmentation/main.cpp, VfxCustomShader/main.cpp, VfxManualEmitterSet/main.cpp, and VfxSimple/main.cpp.
