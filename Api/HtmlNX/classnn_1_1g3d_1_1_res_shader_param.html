typedef size_t(* nn::g3d::ResShaderParam::ConvertShaderParamCallback) (void *pDst, const void *pSrc, const ResShaderParam *pShaderParam, const void *pUserPtr)

The conversion callback invoked when there is a transfer to buffers for uniform blocks.

Parameters
[out]pDstPointer to the location to use for writing the conversion result.
[in]pSrcPointer to the source data.
[in]pShaderParamPointer to a shader parameter.
[in]pUserPtrUser pointer.
Returns
Size of the written data.

Details
This callback function is executed if the shader parameters specified for the callback function were changed when calling nn::g3d::ModelObj::CalculateMaterial() or nn::g3d::MaterialObj::CalculateMaterial(). The user pointer specified in nn::g3d::MaterialObj is passed to pUserPtr.

If the shader parameter specifying the callback function has a dependent shader parameter, use nn::g3d::ResShaderParam::GetDependPointer() to get the pointer to the dependent shader parameter.

◆ SetDependPointer()

bool nn::g3d::ResShaderParam::SetDependPointer ( void *  pSrcShaderParam,
const void *  pDependParam 
) const

Sets a pointer to the dependency target parameter if the shader specifies a shader parameter within itself as the dependency target.

Parameters
[in]pSrcShaderParamPointer to the source data of the shader itself.
[in]pDependParamPointer to the dependency target parameter.
Returns
Returns true if configuration is successful.

Details
If other shader parameters are set as the dependency target parameter in the shader, configuration will fail and return false.

If other shader parameters are set as the dependency target in the shader, the dependency target pointer is specified automatically in nn::g3d::MaterialObj::Builder::Build() and this function does not need to be used.
