Detailed Description

Class for handling asynchronous processing.

Details
It is possible to wait on and cancel asynchronous processing. You must not destroy objects passed to asynchronous functions until asynchronous processing completes.

Examples
FriendsProfileImage/FriendsProfileImage.cpp, and FriendsSimple/FriendsSimple.cpp.

nn::Result nn::friends::AsyncContext::Cancel ( )

Requests the interruption of processing.

Returns
Returns the result of processing.

Details
This function just requests processing interruptions. Asynchronous processing does not complete immediately after calling this function. When an interrupted asynchronous process completes, the value for AsyncContext::GetResult is returned as ResultCanceled.

◆ GetResult()

nn::Result nn::friends::AsyncContext::GetResult ( ) const

Gets the result of asynchronous processing.

Returns
Returns asynchronous processing results.
Return Values:

ResultCanceled Indicates that there was a cancel request for an asynchronous API function.

ResultInternetRequestNotAccepted Indicates that an asynchronous API function was run prior to an Internet connection request.

ResultHttpError An error occurred during communication.

ResultServerError The server returned an error.

nn::account::ResultNetworkCommunicationError An error occurred during communication.

nn::account::ResultNetworkServiceAccountUnavailable The network service account is not in an available state.

Details
Returns the asynchronous processing result. Note that this is not the result of an asynchronous function call.

You must not call this function before the flag obtained with AsyncContext::HasDone is true.

Display an error message using the error viewer when any of the following are the asynchronous processing results.

  • ResultInternetRequestNotAccepted
  • ResultHttpError
  • ResultServerError

There is no need to display errors when communication has no effect on application progress, such as when getting profile data in the background.

Perform error handling in conjunction with the account library manual when the asynchronous processing result belongs in the nn::account namespace.

Examples
FriendsProfileImage/FriendsProfileImage.cpp, and FriendsSimple/FriendsSimple.cpp.
