This sample program demonstrates how to generate icons using pre-multiplied alpha values when blending with the background.

Overview

This sample demonstrates how to generate icons using the pre-multiplied alpha feature of the icon shader provided in the Face Library. This sample compares the results obtained when directly rendering icons and face models with the pre-multiplied alpha feature enabled and disabled. Left: Icon generated using the pre-multiplied alpha feature. Middle: Rendered Mii face model. Right: Normal icon. Note that although the color in the region where the translucent glasses overlap with the background is essentially the same in the left and middle images, it is completely different in the right image.

File Structure

This sample program is located in Samples/Sources/Applications/MiiPremultipliedAlphaIcon/ .

Description

This sample program generates Mii icons to provide a comparison of the resulting images when SetPremultipliedAlphaBlendForBackground() is enabled and disabled.

Conversion Process

  • Samples/Sources/Applications/MiiPremultipliedAlphaIcon/Resources/SampleShader.glsl

This shader renders icon textures and the same Mii characters as in the Mii applet.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Load the Mii resource file.
  • Initialize the Mii resource (nn::mii::Resource).
  • Get nn::mii::CharInfo.
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Initialize the texture to which the icon will be written.
  • Initialize the texture for storing the result.
  • Initialize the texture for the depth stencil.
  • Initialize the shader for the icon texture (nn::mii::IconShader).
  • Initialize nn::mii::IconGpuBuffer.
  • Create a sampler for the Mii.
  • Create the texture view and register it to the descriptor pool.
  • Use nn::mii::TextureShader to render the Faceline and Mask textures.
  • Use nn::mii::IconShader to render the Mii icon with the pre-multiplied alpha feature disabled.
  • Use nn::mii::IconShader to render the Mii icon with the pre-multiplied alpha feature enabled.
  • Use the same shader used to render the icons to directly render the Mii character's face model as the render target (center image).
  • Draw the texture with the pre-multiplied alpha feature disabled (left image) and the texture with the pre-multiplied alpha feature enabled (right image) on the screen using a dedicated blend state for each.
