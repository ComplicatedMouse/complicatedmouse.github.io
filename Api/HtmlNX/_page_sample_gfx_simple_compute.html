A description of the simple GPU calculation sample program.

Overview

This sample program shows how to use compute shaders for basic calculations using the gfx library.

File Structure

This sample program is located in Samples/Sources/Applications/GfxSimpleCompute/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program loads a compute shader resource and uses the GPU in parallel to perform calculations on a simple array. This sample is roughly as small as possible while performing GPU processes using gfx. The resource used is converted at build time starting from a pre-build event. A shader converter is used to convert the resource.

This sample program has the following flow.

  • Initialize the device.
  • Initialize a memory pool.
  • Initialize the queue.
  • Initialize the compute shader resource.
  • Initialize the command buffer.
  • Initialize the timestamp query buffer.
  • Initialize an unordered access buffer.
  • Write the initial values for calculation to the buffer.
  • Create the command list.
  • Execute the command list.
  • Wait for the GPU to complete the process.
  • Get the total execution time.
  • Verify the calculation results.
  • Free all objects used.
