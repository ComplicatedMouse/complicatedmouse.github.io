Detailed Description

A sample program that renders Mii characters the same as the MiiEdit applet.

Macro Definition Documentation

◆ USE_HAREWARE_DEGAMMA

#define USE_HAREWARE_DEGAMMA

A sample of a shader implementation comparable to the MiiEdit applet.

Details
This sample performs shader rendering of Mii characters the same way they are displayed in icons and the MiiEdit applet. The Mii characters are rendered using the nn::mii::Material class provided in the Face Library. Processing with linear gamma is performed inside the shader. Defines whether to render using the hardware's degamma setting.
