Detailed Description

The effect operation handle class.

Details
This class can be retrieved by emitter sets that have been created.

Examples
VfxBinarySegmentation/main.cpp, VfxCustomShader/main.cpp, VfxManualEmitterSet/main.cpp, and VfxSimple/main.cpp.
