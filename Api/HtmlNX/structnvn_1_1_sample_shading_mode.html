Detailed Description

Sample shading mode used when executing a fragment shader.

◆ Enum

enum nvn::SampleShadingMode::Enum

Enumerated values for the type SampleShadingMode.

Enumerator
DEFAULT_FRAGMENT 

Default mode.

The fragment shader is evaluated either by fragment or by sample, depending on whether a keyword such as gl_SamplePosition is specified within the fragment shader.

FORCE_SAMPLE_SHADING_ON 

Force the fragment shader to be evaluated per sample.

FORCE_SAMPLE_SHADING_OFF 

Force the fragment shader to be evaluated per fragment, overriding any behavior from any keywords in the fragment shader.
