nn::os::SemaphoreType Struct Reference

This structure defines a semaphore object. More...

#include <nn/os/os_SemaphoreTypes.h>

Detailed Description

This structure defines a semaphore object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeSemaphore()
  • FinalizeSemaphore()
  • AcquireSemaphore()
  • TryAcquireSemaphore()
  • TimedAcquireSemaphore()
  • ReleaseSemaphore(SemaphoreType* semaphore)
  • ReleaseSemaphore(SemaphoreType* semaphore, int count)
  • GetCurrentSemaphoreCount()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, SemaphoreType* semaphore)

To use these functions, include nn/os.h.

Also see the description of the nn::os::Semaphore utility class.

Examples
OsMultiWait/OsMultiWait.cpp, OsMultiWaitUtility/OsMultiWaitUtility.cpp, and OsSemaphore/OsSemaphore.cpp.
