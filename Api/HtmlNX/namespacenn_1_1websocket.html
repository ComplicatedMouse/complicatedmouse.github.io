◆ SetLogLevel()

void nn::websocket::SetLogLevel ( LogLevel  level)

Sets the log level.

Parameters
[in]levelThe log level.

Details
Setting the log level controls the types of log entries generated within the library.

The following table shows the available log levels and the corresponding types of information.

Log Level Content
LogLevel::Trace Information for each communication process.
LogLevel::Information The communication content, time, and other information associated with the connection process.
LogLevel::Warning Error information for errors that allow the process to continue.
LogLevel::Error Error information for errors that cancel the process.
LogLevel::None No output.

When a lower-level log level is specified, entries for all higher-level log levels are also included.

Attention
In release builds, no logs are generated, regardless of the specified log level.
Note
The default value is LogLevel::Error.
