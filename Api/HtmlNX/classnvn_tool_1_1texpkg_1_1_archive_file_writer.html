Detailed Description

This module is responsible for writing and appending to an NVNFD archive.

◆ Valid()

virtual bool nvnTool::texpkg::ArchiveFileWriter::Valid ( void  ) const
pure virtual

Whether opened file handle is valid.

If append is disabled, this returns whether the file has been successfully created. Otherwise returns whether the archive was successfully read, or created if no previous archive exists.

Returns
true if the file is ready to be written to, false otherwise.
