◆ ptr

void* nn::swkbd::String::ptr

Pointer to the start of the memory space storing the entry result string.


When nn::swkbd::KeyboardConfig::isUseUtf8 is true, the entry result string is stored in UTF-8 format.
When nn::swkbd::KeyboardConfig::isUseUtf8 is false, the entry result string is stored in UTF-16 format.
Set a buffer of the allocated size, as obtained with the nn::swkbd::GetRequiredStringBufferSize() function.
This space must be aligned to nn::os::MemoryPageSize.

Examples
SwkbdSimple/SwkbdSimple.cpp, and WebSocketChat/WebSocketChat.cpp.
