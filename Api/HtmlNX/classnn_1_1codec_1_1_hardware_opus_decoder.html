Detailed Description

The Opus decoder class for hardware use.

Details
Decodes mono and stereo sound Opus data.

Examples
CodecOpusCodecProcessingTime/CodecOpusCodecProcessingTime.cpp, and CodecOpusDecoder/CodecOpusDecoder.cpp.
