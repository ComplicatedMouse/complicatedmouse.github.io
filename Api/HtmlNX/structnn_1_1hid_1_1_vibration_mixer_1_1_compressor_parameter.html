Detailed Description

This structure represents the compressor coefficient.

Details
You can use VibrationMixer to set the compressor coefficient to compress the amplitude of the vibration. This makes it easier to sense weaker vibrations and prevents unnecessarily strong vibrations.

For the amplitude of the entered vibration value, values over threshold are compressed to 1/ratio.
If you set (threshold, ratio) = (0.0f, 1.0f), the input value is used as the output value as is.
If you set (threshold, ratio) = (0.0f, 2.0f), the input value is half.
If you set (threshold, ratio) = (0.5f, 2.0f), the amplitude exceeding 0.5f becomes 1/2.0f.
If the input is 1.0f, the output is 0.75. If the input is 0.3f, the output remains 0.3f.
If ratio exceeds MaxRatio, threshold becomes the maximum of the amplitude.
