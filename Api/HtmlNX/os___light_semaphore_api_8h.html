Detailed Description

Declarations for API resources related to light semaphores.

Details
The light semaphore feature that uses a number counter to establish synchronization among threads.

Light semaphores differ in the following manner from the semaphore feature that supports nn::os::SemaphoreType.

  • When the semaphore counter is 1 or higher during the operation for getting a semaphore (P operation), the local thread does not enter a waiting state due to other light semaphore processing functions.
  • During the semaphore return operation (V operation), the local thread does not enter a waiting state due to other light semaphore processing functions.
  • You cannot specify this as the target for multiple wait.
