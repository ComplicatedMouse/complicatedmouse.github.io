Detailed Description

This class receives cancellation requests from CancellationTokenSource.

Details
CancellationToken is a class that receives cancellation requests from specific CancellationTokenSource objects. The thread that executes asynchronous processing can poll the CancellationToken object to determine whether there is a cancellation request, and then realize cancellation of cooperative asynchronous processing with the thread that requested cancellation.

For more information, see the reference for CancellationTokenSource.

◆ IsCancellationRequested()

bool nn::async::CancellationToken::IsCancellationRequested ( ) const

Determines whether a cancellation request was received.

Returns
Returns true if a cancellation request was received.

Details
Calling CancellationTokenSource::RequestCancel() "RequestCancel()" for the CancellationTokenSource that generated *this causes this function to return true.

If *this was generated by the default constructor, it always returns false.
