◆ Initialize()

bool nn::spy::atk::AtkSpyModule::Initialize ( const InitializeArg &  args,
void *  buffer,
size_t  bufferSize 
)

Initializes.

Details
The user passes the work memory that AtkSpyModule will use. Set the parameters needed for initialization to the InitializeArg structure, and get the required size of memory using GetRequiredMemorySize().

Parameters
[in]argsThe initialization parameters.
[in]bufferThe pointer to the work memory that AtkSpyModule will use.
[in]bufferSizeThe size of the work memory.
Returns
Returns true if initialization succeeded.
Precondition
  • buffer is aligned to nn::DefaultAlignment.
  • bufferSize >= GetRequiredMemorySize()
  • nn::atk::SoundSystem::IsInitialized() == true.
  • IsInitialized() == false.
  • IsRegistered() == false.
  • args.isPerformanceMetricsInitializationEnabled == true when args.isPerformanceMetricsEnabled == true
  • args.audioPerformanceMetricsInfoCount >= InitializeArg::AudioPerformanceMetricsCountMin
  • args.isWaveformInitializationEnabled == true when args.isWaveformEnabled == true
  • When args.isWaveformInitializationEnabled == true, nn::atk::SoundSystem will be initialized if nn::atk::SoundSystem::SoundSystemParam::enableCircularBufferSink == true.

◆ PushWaveform()

bool nn::spy::atk::AtkSpyModule::PushWaveform ( const void *  buffer,
size_t  bufferSize,
nn::os::Tick  tick 
)

Sends the waveform data.

Details
Sends the waveform data obtained using nn::atk::SoundSystem::ReadCircularBufferSink().

nn::atk does not provide the time of completion of nn::audio::RequestUpdateAudioRenderer(), so for the tick parameter, specify the tick just prior to the call to nn::atk::SoundSystem::ReadCircularBufferSink().

Parameters
[in]bufferPointer to the buffer storing the sample waveform data.
[in]bufferSizeThe size of the sample data stored in the buffer.
[in]tickThe tick just prior to the call to nn::atk::SoundSystem::ReadCircularBufferSink().
Returns
Returns true if successful, or false otherwise.
Precondition
  • tick <= nn::os::GetSystemTick().
  • tick must be greater than the value it was when called the last time.
  • IsInitialized() == true.
  • The parameters for sending waveform data are specified by Initialize().
See also
Initialize
nn::audio::ReadCircularBufferSink
