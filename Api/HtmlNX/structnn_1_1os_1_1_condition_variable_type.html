nn::os::ConditionVariableType Struct Reference

#include <nn/os/os_ConditionVariableTypes.h>

Detailed Description

This structure defines a condition variable object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeConditionVariable()
  • FinalizeConditionVariable()
  • SignalConditionVariable()
  • BroadcastConditionVariable()
  • WaitConditionVariable()
  • TimedWaitConditionVariable()
  • NN_OS_CONDITION_VARIABLE_INITIALIZER()

To use these functions, include nn/os.h.

Also see the description of the nn::os::ConditionVariable utility class.
