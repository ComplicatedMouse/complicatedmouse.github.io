Detailed Description

Represents a four-dimensional vector using a single precision floating point data type.

It is also used to represent quaternions.

Details
Type that provides vector and matrix calculations.

In addition to vectors, this structure is used to represent quaternions in functions that handle quaternions, such as nn::util::QuaternionAdd. When the vector elements in order are x, y, z, w, and the quaternion imaginary elements are i, j, k, the quaternion, q, is represented as q = xi + yj + zk + w.

We assume that calculations that use this type will switch to an implementation that uses the SIMD unit in environments that can use the SIMD unit. A generic environment is provided for environments in which the SIMD unit cannot be used.

Details
Type that provides vector and matrix calculations.

Attention
This type is optimized for calculations. As a result, it can differ based on memory layout, size, and environment. For this reason, direct access to internal elements is prohibited. To access elements, use dedicated macros and functions.
Because calculations within the SIMD are to be optimized, this type calculates memory alignment. When allocating this type in memory, make sure that it meets the required alignment. The required alignment may differ for each internal implementation. Each implementation enumerates the required alignment.
  • General purpose implementation version: 16 byte
  • NEON implementation version: 16 byte

The general purpose implementation version specifies an alignment that can satisfy the required alignment of all other SIMD implementations.

Examples
G3dDemo/Tessellation.cpp, G3dDemo/Town.cpp, NvnTutorial/BoundingVolume.cpp, NvnTutorial/ViewFrustum.cpp, nvntutorial/BoundingVolume.h, and nvntutorial/ViewFrustum.h.

void QuaternionLerp ( Vector4fType *  pOutValue,
const Vector4fType &  from,
const Vector4fType &  to,
float  t 
)
related

Calculates the linear interpolation between two quaternions.

Parameters
[out]pOutValueLocation where the result is stored.
[in]fromFirst quaternion.
[in]toSecond quaternion.
[in]tInterpolation coefficient.

Details
Performs linear interpolation between two vectors (from and to) using a ratio of t:1 – t.

Pseudo-code for the algorithm implemented by this function:

pOutValue->v.x = from.x + t * (to.x - from.x);
pOutValue->v.y = from.y + t * (to.y - from.y);
pOutValue->v.z = from.z + t * (to.z - from.z);
pOutValue->v.w = from.w + t * (to.w - from.w);
