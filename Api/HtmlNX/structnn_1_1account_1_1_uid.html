Detailed Description

User identifiers added to the account system.

Details
Identifiers to uniquely identify users added to the account system.

InvalidUid ensures that the value is not valid as a user identifier. It can also evaluate whether a particular Uid value is invalid, using a Boolean operator.

NN_EXPLICIT_OPERATOR nn::account::Uid::bool ( ) const

Evaluates the validity of a Uid value.

Returns
Returns *this != InvalidUid.

Details
Evaluates the validity of a Uid value. In this content, a valid Uid value is one that is not InvalidUid. Even when a Uid value is taken to be valid, that does not ensure that it indicates a user added to the account system.
