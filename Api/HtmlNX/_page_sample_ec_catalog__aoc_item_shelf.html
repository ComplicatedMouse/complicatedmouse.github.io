Describes the AOC shelf list sample program.

Overview

This sample shows how to display the AOC shelf list. This is the screen when AOC item shelf list is selected from the start screen. The major classifications for the item_tags data in AOC Items are equipment and available time. The minor classifications include sword, shield, and spear, in addition to early, middle, and late. If you create a search list from the item_tags data that includes minor classification values like sword and then specify that list when you jump to Nintendo eShop, only certain items are displayed.

File Structure

This sample program is located in Samples/Sources/Applications/EcCatalog/Aoc/ .

Description

For a general overview of the process, see the description in Main.cpp. This description only details the shop jump functionality. The shop jump within this sample occurs using Main.cpp.
