◆ NN_UTIL_CALCULATE_COMPRESS_ZLIB_WORKBUFFER_SIZE

#define NN_UTIL_CALCULATE_COMPRESS_ZLIB_WORKBUFFER_SIZE (   memLevel)    (137024 + (1 << ((memLevel) + 9)))

Macro to calculate the size (in bytes) of the working buffer required by CompressZlib().

Details
To specify the memory usage of the CompressZlib() function, prepare a working buffer of the size calculated by this macro and pass it to CompressZlib().
