Detailed Description

A lightweight wrapper for an nvn::Program object.

Manages the details of nvn::Program creation based on shader program data compiled by GLSLC. Also provides reflection information for vertex attributes, uniform blocks, and inidividual uniforms (see Lookup* methods).

See GraphicsShader::Initialize for implementation details on how nvn::Program is set up, and how reflection information is obtained and recorded.

See NvnSimpleShaderCompiler project for details on how GLSLCoutput, which is used as the input for creating an instance of this class, is generated.
