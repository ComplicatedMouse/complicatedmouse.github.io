bool nn::audio::PerformanceInfo::SetBuffer ( const void *  frameBuffer,
size_t  bufferSize 
)
noexcept

Sets and initializes a buffer for performance information.

Parameters
[in]frameBufferA buffer for storing the performance information.
[in]bufferSizeSize of the buffer for storing the performance information.
Returns
Return false if there is no performance information in the specified buffer.
Precondition
  • frameBuffer != nullptr.

Details
The nn::audio::SetPerformanceFrameBuffer() function sets a buffer to which all subsequent performance information will be output, and returns the buffer that stores all performance information that has been output so far. This function initializes a new buffer that includes a reference to the performance information contained in the buffer returned by the nn::audio::SetPerformanceFrameBuffer() function. After calling the nn::audio::SetPerformanceFrameBuffer() function to change the buffer to which audio renderers output performance information, pass the address returned by that function as a parameter to this function.

noexcept

Determines whether the time limit for rendering has been exceeded.

Returns
Whether the time limit for rendering has been exceeded.

Details
If the time limit for rendering has been exceeded, processing slowdowns might create noise in the rendering results.

NX Platform Only:
For more information about the rendering time limit, see nn::audio::SetAudioRendererRenderingTimeLimit() and DSP Resource Limitations.
Windows Platform Only:
For more information about the rendering time limit, see nn::audio::SetAudioRendererRenderingTimeLimit(). This function always returns false on the Windows platform.

◆ GetFrameIndex()

uint32_t nn::audio::PerformanceInfo::GetFrameIndex ( ) const
noexcept

Gets the current PerformanceFrame index.

Returns
Returns the index.

Details
PerformanceFrame has an index for each frame, and the value is incremented sequentially in every audio frame. You can check for missing PerformanceFrame by checking the sequentiality of the index values before and after the MoveToNextFrame() function. The function returns 0 if the index exceeds the maximum value that can be represented by uint32_t.

AudioPerformanceMetrics/AudioPerformanceMetrics.cpp.
