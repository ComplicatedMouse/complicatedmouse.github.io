Overview

This offline HTML sample is for displaying a game manual. This sample demonstrates the following features.

  • Independently changing spatial navigation focusing
  • Limiting L Stick controls to spatial navigation
  • Hiding the footer
  • Calling the web applet from the offline HTML
  • Using the Javascript extension to play sound effects

File Structure

This sample offline HTML is located in Samples/Sources/Applications/WebDemo/html_document/sample/instruction_manual.htdocs.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

Input Operation
L Stick or the directional buttons Moves the spatial navigation.
A Button, Taps Sets options.
B Button Either goes back to the previous page or exits the offline web applet, returning to the menu, via the application.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and display the offline HTML from this sample.

Independently Changing Spatial Navigation Focusing

Uses the focus-ring-visibility attribute from the meta element to hide the spatial navigation focus frame. Uses the CSS ::before and ::after pseudo-elements to display a proprietary focus frame while the focus is on the button element.

Limiting L Stick Controls to Spatial Navigation

Uses nn::web::ShowOfflineHtmlPageArg::SetPointerEnabled() to limit L Stick controls to spatial navigation.

Hiding the Footer

Uses nn::web::ShowOfflineHtmlPageArg::SetFooterEnabled() to hide the footer.

Calling the Web Applet From the Offline HTML

Select the Official Website option from the offline HTML to transition to a URL that starts with http://localhost/. The offline web applet considers this URL to be a callback URL and exits the offline web applet, returning the URL to the application. The application uses nn::web::OfflineHtmlPageReturnValue::GetLastUrl() to get the URL to be opened using the web applet. The web applet then starts and displays the web page. Displays the offline HTML once more after returning from the web page.

Using the Javascript Extension to Play Sound Effects

Uses the ns-se-disable attribute from the button element to disable standard sound effects. Uses the window.nx.playSystemSe() Javascript extension to play system sound effects.
