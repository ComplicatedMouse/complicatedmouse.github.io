Detailed Description

A set of parameters for stream data.

Details
You can get this information by using SoundArchivePlayer::ReadStreamSoundDataInfo or StreamSoundHandle::ReadStreamSoundDataInfo.

◆ loopEnd

int64_t nn::atk::StreamSoundDataInfo::loopEnd

The loop end position when the stream data is looped, as a number of samples from the start of the stream.

Represents the end of the data, as a number of samples if stream data is not looped. (Points to the sample that follows the last sample played.)

◆ compatibleLoopStart

int64_t nn::atk::StreamSoundDataInfo::compatibleLoopStart

The loop start position when the stream data is looped, as a number of samples from the start of the stream.

Details
Due to hardware restrictions, a change was made in version 0.3.0 such that the loopStart member variable now returns the pre-corrected value. If you want to get the value like before, use this parameter instead. Note, however, that the use of this member variable is not recommended, and it will be deprecated in the future.
