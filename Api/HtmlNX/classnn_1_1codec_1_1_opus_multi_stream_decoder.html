Detailed Description

The Opus multitrack stream decoder class.

Details
In addition to mono and stereo sound, decodes Opus data for sound sources with three or more channels.

Opus data handled by nn::codec::OpusDecoder is single stream and can also be handled by this class. However, this class requires a work buffer size larger than nn::codec::OpusDecoder. Unless you have a specific reason not to, we recommend using nn::codec::OpusDecoder for decoding mono and stereo sound Opus data.

Examples
