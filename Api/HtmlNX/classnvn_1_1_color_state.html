◆ SetLogicOp()

ColorState & nvn::ColorState::SetLogicOp ( LogicOp  logicOp)

Specifies the logical operation to perform for all color targets.

If programmed to anything but COPY, arithmetic blending has no effect. Unlike blending, there is no support for separate logical operations for each color target.

Parameters
logicOpLogical operation to perform on all color targets.

◆ SetLogicOp_fastpath()

ColorState & nvn::ColorState::SetLogicOp_fastpath ( LogicOp  logicOp)

Specifies the logical operation to perform for all color targets.

Fastpath variant.

If programmed to anything but COPY, arithmetic blending has no effect. Unlike blending, there is no support for separate logical operations for each color target.

Note
: This is functionally equivalent to ColorState::SetLogicOp, but should be faster in common cases. Not compatible with the NVN debug layer.
Parameters
logicOpLogical operation to perform on all color targets.

◆ SetAlphaTest()

ColorState & nvn::ColorState::SetAlphaTest ( AlphaFunc  alphaTest)

Specifies a comparison function to use for alpha testing.

Using AlphaFunc::ALWAYS disables the alpha test entirely.

Note
Applications may get better performance from performing an alpha test in the fragment shader and discarding fragments that fail. Doing so allows the fragment shader to skip other color calculations for failing fragments.
Parameters
alphaTestComparison function used by the alpha test.
