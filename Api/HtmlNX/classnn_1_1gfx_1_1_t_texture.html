Calculates the alignment required of the pointer to the base level data of the mipmap data and the overall size of the mipmap data.

Parameters
[out]pOutSizePointer for storing the overall size of the mipmap data.
[out]pOutAlignmentPointer for storing the alignment required of the first of the mipmap data.
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
Precondition

  • pOutAlignment != NULL.
  • pDevice != NULL.
  • pDevice is initialized.

Details
This function gets the values that were obtained by CalculateMipDataSize() and CalculateMipDataAlignment(). Depending on the low-level API, this can reduce the computational overhead compared to making individual calls to these functions.

Initializes a texture.

Parameters
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
[in]pMemoryPoolPointer to the memory pool where the texture mipmap data is placed.
[in]memoryPoolOffsetOffset to the memory pool where the texture's mipmap data is placed, in bytes.
[in]memoryPoolSizeThe size, in bytes, of the memory pool the texture uses.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The sampler is not initialized.
  • If IsMemoryPoolRequired is true, pMemoryPool != NULL.
  • If IsMemoryPoolRequired is true, memoryPoolSize is larger than the size obtained by CalculateMipDataSize.
Postcondition
  • The sampler is initialized.
Examples
