Overview

Describes how to call the controller support applet from the sample application title screen.

File Structure

This sample program is located in Samples/Sources/Applications/HidControllerSequence/ .

Required Environment

Pair the controller to the development hardware ahead of time.

How to Use

There are only rights for choosing master player controls.

You can select the number of game players using either the up and down directional buttons or the control stick. Press the A Button to begin the game with the selected number of players.

Important Information

Make sure that the controller is sufficiently charged before using it.

Execution Procedure

Build the application and run it.

Description

The following processing occurs at the title screen.

  • Confirm that controllers are connected.
  • Select the number of players.
  • Confirm the number of players.
  • Check the number of connected controllers.
  • Transition to the game screen.

The controller support applet is called at the title screen under the following circumstances.

  • When the master player controller is lost
  • When there are not enough wireless controllers for the specified number of players
