Detailed Description

The class that builds the ShapeAnimObj instance.

Examples
G3dDemo/ShapeAnimation.cpp.

Constructor & Destructor Documentation

◆ Builder()

nn::g3d::ShapeAnimObj::Builder::Builder ( )

Constructor.

Details
You must either call Reserve() on all shape animation resources and model resources used in the instance, or use SetMaxShapeCount(), SetMaxVertexShapeAnimCount(), SetMaxKeyShapeAnimCount(), and SetMaxCurveCount() to specify the parameters required to calculate the buffer size.
