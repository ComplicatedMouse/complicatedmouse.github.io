Detailed Description

The class that defines the window coordinates.

Details
nn::gfx::util::ScalableViewport manages virtual windows and physical windows, and performs conversion between viewports.

Examples
EcCatalog/Main.cpp, HidVibrationGenerator/HidVibrationGenerator.cpp, HidVibrationPlayer/HidVibrationPlayer.cpp, and ImguiDemo/ImguiDemo.cpp.
