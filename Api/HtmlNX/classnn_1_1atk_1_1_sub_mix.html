Gets the buffer size required for initialization.

Details
If output destinations will change in the course of processing, use the function that directly specifies the number of output destination buses and channels.

Parameters
[in]srcBusCountThe number of output source buses.
[in]srcChannelCountThe number of output source bus channels.
[in]pDstReceiverThe output line. You must pass an output destination that has already been initialized.
Returns
Returns the buffer size required for initialization.

  • SoundSystem is already initialized.
  • 1 <= srcBusCount <= (the busCountMax set when SoundSystem was initialized).
  • 1 <= srcChannelCount <= nn::audio::MixBufferCountMax.
  • srcChannelCount × srcBusCount <= nn::audio::MixBufferCountMax.
  • pDstReceiver is initialized.
  • 1 <= pDstReceiver->GetBusCount() <= (the busCountMax set when SoundSystem was initialized).
  • 1 <= pDstReceiver->GetChannelCount() <= nn::audio::MixBufferCountMax.
  • pDstReceiver->GetChannelCount() * pDstReceiver->GetBusCount() <= nn::audio::MixBufferCountMax.

Initializes the object as a submix.

Details
Bus count: srcBusCount, channel count: srcChannelCount, destination bus count: dstBusCount, destination channel count: dstChannelCount. Initializes the object as a submix. If the destinations of the submix output will change in the course of processing, specify the maximum number of output destination buses and output destination channels.

Parameters
[in]srcBusCountThe number of output source buses.
[in]srcChannelCountThe number of output source bus channels.
[in]dstBusCountThe number of output destination buses. If the output destinations will change in the course of processing, specify the maximum number of output destination buses.
[in]dstChannelCountThe number of output destination channels. If the output destinations will change in the course of processing, specify the maximum number of output destination buses.
[in]bufferThe buffer to use for initialization.
[in]bufferSizeThe size of the buffer to use for initialization.
Returns
Returns true if it initialized successfully, or false otherwise.
Precondition
  • SoundSystem is already initialized.
  • SubMix is not initialized.
  • 1 <= srcBusCount <= (the busCountMax set when SoundSystem was initialized).
  • 1 <= srcChannelCount <= nn::audio::MixBufferCountMax.
  • srcChannelCount × srcBusCount <= nn::audio::MixBufferCountMax.
  • 1 <= dstBusCount <= (the busCountMax set when SoundSystem was initialized).
  • 1 <= dstChannelCount <= nn::audio::MixBufferCountMax.
  • dstChannelCount × dstBusCount <= nn::audio::MixBufferCountMax.
  • buffer != nullptr.
  • buffer is aligned to nn::atk::SubMix::SubMixAlignment.
  • bufferSize is larger than the value obtained by the GetRequiredMemorySize() function.
Postcondition
  • SubMix is initialized.

Initializes the object as a submix.

Details
Initializes the object as a submix with srcBusCount number of buses, srcChannelCount number of channels, and the number of output destination buses and channels referenced from pDstReceiver. If the output destinations will change in the course of processing, use the API that directly specifies the number of output destination buses and channels.

Parameters
[in]srcBusCountThe number of output source buses.
[in]srcChannelCountThe number of output source bus channels.
[in]pDstReceiverThe output line. This parameter is used for referencing the numbers of buses and channels.
[in]bufferThe buffer to use for initialization.
[in]bufferSizeThe size of the buffer to use for initialization.
Returns
Returns true if it initialized successfully, or false otherwise.
Precondition
  • SoundSystem is already initialized.
  • SubMix is not initialized.
  • 1 <= srcBusCount <= (the busCountMax set when SoundSystem was initialized).
  • 1 <= srcChannelCount <= nn::audio::MixBufferCountMax.
  • srcChannelCount × srcBusCount <= nn::audio::MixBufferCountMax.
  • pDstReceiver is initialized.
  • 1 <= pDstReceiver->GetBusCount() <= (the busCountMax set when SoundSystem was initialized).
  • 1 <= pDstReceiver->GetChannelCount() <= nn::audio::MixBufferCountMax.
  • pDstReceiver->GetChannelCount() * pDstReceiver->GetBusCount() <= nn::audio::MixBufferCountMax.
  • buffer != nullptr.
  • buffer is aligned to nn::atk::SubMix::SubMixAlignment.
  • bufferSize is larger than the value obtained by the GetRequiredMemorySize() function.
Postcondition
SubMix is initialized.

◆ SetBusVolume()

void nn::atk::SubMix::SetBusVolume ( int  bus,
float  volume,
int  fadeFrame 
)

Sets the bus volume.

The default volume is 1.0f.

Details
When the value is set to 1.0, output is not scaled, but the ultimate value that gets used is the product of this value multiplied by Send and ChannelVolume. There are no restrictions on the value that can be set, but the ultimate value that gets used is clamped between the values of nn::audio::SubMixType::GetVolumeMin() and nn::audio::SubMixType::GetVolumeMax().

Parameters
[in]busThe bus for which the volume will be set.
[in]volumeThe volume.
[in]fadeFrameThe number of frames that elapse when the volume changes.
Precondition
  • SubMix is initialized.
  • 0 <= bus < srcBusCount when SubMix was initialized.
  • fadeFrame >= 0.

◆ SetChannelVolume()

void nn::atk::SubMix::SetChannelVolume ( int  channel,
float  volume,
int  fadeFrame 
)

Sets the channel volume.

The default volume is 1.0f.

Details
When the value is set to 1.0, output is not scaled, but the ultimate value that gets used is the product of this value multiplied by Send and BusVolume. Note that the value set with this function affects all channels with the same index in all of the buses in the SubMix instance.

There are no restrictions on the value that can be set, but the ultimate value that gets used is clamped between the values of nn::audio::SubMixType::GetVolumeMin() and nn::audio::SubMixType::GetVolumeMax().

Parameters
[in]channelThe channel for which to set the volume.
[in]volumeThe volume.
[in]fadeFrameThe number of frames that elapse when the volume changes.
Precondition
  • SubMix is initialized.
  • 0 <= channel < srcChannelCount when SubMix was initialized.
  • fadeFrame >= 0.
