Enumerated values for the type AutomaticCheckpointsMask.

Enumerator
DRAW 

Enables automatic checkpoint event markers for all draw commands.

If set, this flag indicates that automatic checkpoint event marker should be added for all draw commands.

COMPUTE_DISPATCH 

Enables automatic checkpoint event markers for all dispatch compute commands.

If set, this flag indicates that automatic checkpoint event marker should be added for all dispatch compute commands.

COPY 

Enables automatic checkpoint event markers for all copy commands.

If set, this flag indicates that automatic checkpoint event marker should be added for all copy commands.

DEBUG_GROUP 

Enables automatic checkpoint event markers for all debug group push commands.

If set, this flag indicates that automatic checkpoint event marker should be added for all debug group push commands.

ALL 

Enables automatic checkpoint event markers for all supported commands.

If set, this flag indicates that automatic checkpoint event marker should be added for all supported commands. This is the default setting.
