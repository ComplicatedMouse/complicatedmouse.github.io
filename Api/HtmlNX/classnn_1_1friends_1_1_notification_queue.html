Detailed Description

Class for handling the notification queue.

Details
This class manages the notification queues created for each user account.

Notification queues maintain a maximum of NotificationCountMax notifications. All NotificationType_FriendPresenceUpdated notifications are deleted and replaced with a single NotificationType_FriendListUpdated notification when the notification queue is full.

Notifications with the same content are bundled into a single notification. Only one notification will be stored in the notification queue, for example, even when a friend updates their presence multiple times. A notification with identical content will be added once more when the application uses NotificationQueue::Pop to get notifications.

Examples
FriendsPresence/FriendsPresence.cpp.
