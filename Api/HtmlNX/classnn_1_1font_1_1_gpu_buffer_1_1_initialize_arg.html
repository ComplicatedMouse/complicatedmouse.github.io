◆ SetAnalyseModeFlag()

InitializeArg& nn::font::GpuBuffer::InitializeArg::SetAnalyseModeFlag ( bool  analyseModeFlag)

Initializes GpuBuffer in the mode for measuring the amount of memory in use.

Details
In this mode for measuring the amount of memory in use, the process of actually allocating memory is not conducted. Instead, the total amount of required memory is registered when Allocate is called. In measurement mode, nothing needs to be set for InitializeArg except this flag and GpuAccessFlag. The measurement result can be obtained by calling nn::font::GpuBuffer::GetAllocatedSize any time between the calls to nn::font::GpuBuffer::Map and nn::font::GpuBuffer::Unmap. Note that other methods for getting information do not return meaningful values. Synchronous processing when memory is being allocated is not supported.

Parameters
[in]analyseModeFlag
Returns
Returns a reference to the object.
