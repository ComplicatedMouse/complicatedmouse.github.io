Detailed Description

Defines an Npad ID for identifying an Npad.

Details
An NpadId is defined for each controller number and handheld controller. A controller number is assigned to each controller that is connected wirelessly or using a USB cable. In most cases, controller numbers are assigned sequentially in the same order in which the controllers are connected.

NpadID::Handheld is used when you want to get input from a portable controller.

The correspondence between physical controllers and the controller numbers assigned to them may change. For example, the IDs assigned to controller A that was originally connected as NpadId::No1 and controller B that was originally connected as NpadId::No2 may get switched by the user in the system menu. In addition, controllers get disconnected when they go to sleep. For this reason, the IDs assigned to each controller may change when they wake up and get reconnected.
