nn::os::LightSemaphoreType Struct Reference

This structure defines a light semaphore object. More...

#include <nn/os/os_LightSemaphoreTypes.h>

Detailed Description

This structure defines a light semaphore object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeLightSemaphore()
  • FinalizeLightSemaphore()
  • AcquireLightSemaphore()
  • TryAcquireLightSemaphore()
  • TimedAcquireLightSemaphore()
  • ReleaseLightSemaphore(LightSemaphoreType* semaphore)
  • ReleaseLightSemaphore(LightSemaphoreType* semaphore, int count)
  • GetCurrentLightSemaphoreCount()

To use these functions, include nn/os.h.

Also see the description of the nn::os::LightSemaphore utility class.
