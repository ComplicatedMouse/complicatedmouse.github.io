#include <nn/os/os_FiberTypes.h>

Detailed Description

This structure defines a fiber object.

Details
A fiber object has the following internal states.

  • NotInitialized state: Clearly uninitialized (for example, immediately after nn::os::FinalizeFiber() is called).
  • Ready state: The fiber function is ready for execution (after initialization or while stopped).
  • Running state: The fiber function is actually running.
  • Completed state: The fiber function has finished running and returned control.

Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeFiber()
  • FinalizeFiber()
  • SwitchToFiber()
  • GetCurrentFiber()

To use these functions, include nn/os.h.

Examples
OsFiber/OsFiber.cpp, and OsUserExceptionHandlerSample.cpp.
