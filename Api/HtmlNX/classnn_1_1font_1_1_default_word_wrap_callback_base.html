Detailed Description

template<typename CharType>
class nn::font::DefaultWordWrapCallbackBase< CharType >

The default implementation of the callback function that is called during word wrapping.

Details
This class performs general word wrapping. Inherit nn::font::WordWrapCallbackBase and create a callback class to adjust the line feed method in greater detail.

Gets the byte size of the tag.

Details
Tags are normally exempt from word wrapping, so using this function to return the number of bytes comprising the tag skips the word wrap processing for the tag for a specified size.

This function is called each time when character codes shorter than 0x20 appear.

Parameters
[out]pNonBreakingLengthProhibits line feeds, for the specified number of characters, for the string following the tag. The units are in characters rather than bytes.
[in]pStrPointer to the start of the tag.
[in]pEndThe end of the text that the user passed into the word-wrapping function.
Returns
The number of bytes to skip. Returns 0 if not a tag.
