Describes the sample program for capstone reverse assembly.

Overview

Serves as a sample for the capstone library that reverse-assembles instructions written to the specified memory.

File Structure

This sample program is located in Samples/Sources/Applications/CapstoneSimple/ .

Required Environment

There must be a link to libcapstone.a to use this functionality. There is no need to get libcapstone.a separately from NDI. It is included in the Capstone package.

How to Use

Nothing specific.

Important Information

An additional copyright notice is needed when including this library in retail devices. For more information, see SLIM.

Execution Procedure

Build the sample program and then run it.

Description

This sample program has the following flow.

  • Configures the memory needed to use the capstone library.
  • Configures instructions to be reverse-assembled as AArch64(NX64) for ARM.
  • Uses cs_disasm to reverse-assemble and display all instructions at the same time.
  • Uses cs_disasm_iter to reverse-assemble and display instructions one by one.
  • Uses nn::diag::GetBacktrace to reverse-assemble and display obtained backtraces.
