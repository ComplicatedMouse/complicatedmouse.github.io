Environment Functions for the Shop Service Access Utility Feature

The module's API for setting the shared resources and other resources necessary to use the ShopServiceAccessor class.

Operation of the ShopServiceAccessor class is assured only between the calls to this API's InitializeForShopServiceAccessors() and FinalizeForShopServiceAccessors() functions.

Alias Replacement
For some of the request strings that are specified when requests are made via ShopServiceAccessor, you can use a feature to replace the alias symbol with an appropriate value when the communication request is made. This feature can be used with the following request elements.
  • URL path field strings.
Alias Symbols
Alias symbols are strings represented by keywords in the following prescribed format.
  • Strings enclosed in curly brackets { }.
Available Alias Symbols
  • applicationId : Replaced by the ID of the running application.
  • country : Replaced by the country of residence registered to the Nintendo Account and linked to the specified user account.
Usage Example
const char* path = "/v1/applications/{applicationId}/consumables? country={country}&lang=ja";
