Initializes the vertex state.

Parameters
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
[in]pVertexShaderPointer to the target vertex shader of the vertex state.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • If pVertexShader is not NULL, it is initialized.
  • The vertex state is not initialized.
  • If the value returned by GetRequiredMemorySize with the same info argument is larger than 0, the required memory is set to that size or greater.
Postcondition
  • The vertex state is initialized.

Details
pVertexShader is bound by name if it is not NULL.
