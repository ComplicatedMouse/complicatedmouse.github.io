Detailed Description

The class for handling structures.

Details
Restrictions on strings that can be used as keys are the same as for the nn::prepo::PlayReport class.

There are no restrictions on the total number of key values that can be added to the structure. However, the total size of the report must be less than or equal to nn::prepo::ReportBufferSizeMax.

void nn::prepo::Struct::SetBuffer ( void *  buffer,
size_t  size 
)

Sets a buffer.

Parameters
[in]bufferThe buffer.
[in]sizeThe buffer size.
Precondition
  • buffer != nullptr.
  • size > 0.

Details
A structure is stored in the buffer that is set by this function. The buffer must not be freed until after this object has been added to the PlayReport class or the Array class.

Use the following formula to calculate the optimal size of the buffer to set with this function.

  • (the size of the value) x (the number of elements)

Calculate the size of the value using the following methods.

  • String
    • KeySizeMax + StringBufferSizeMin + (the size of the string)
  • Other
    • PrimitiveKeyValueSizeMax
