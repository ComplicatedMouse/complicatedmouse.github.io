Detailed Description

template<class T, class Tag = T>
class nn::util::IntrusiveDicBaseNode< T, Tag >

An intrusive dictionary node.

Uses a methodology that defines the node as a base class.

Template Parameters
TThe type of an intrusive dictionary element.
TagA type to differentiate nodes when a node inherits features from multiple parents using inheritance.
