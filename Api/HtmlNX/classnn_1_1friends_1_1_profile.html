Detailed Description

Class for handling user profiles.

Details
This class is used to get the URL for the user's nickname and profile image. The application must download the profile image.

The URL for the profile image uses the HTTPS schema. Use a downloader, such as libcurl, that supports the HTTPS schema.

Examples
FriendsProfileImage/FriendsProfileImage.cpp, and FriendsSimple/FriendsSimple.cpp.
