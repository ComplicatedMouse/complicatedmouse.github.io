Detailed Description

The structure that handles calendar information.

Details
The structure that handles the date and time composed of the Western calendar year, month, day, hour, minute, and second.

Addition and subtraction with nn::TimeSpan is possible, but the result is rounded down to the nearest second.

Examples
TimeClaimPeriodicBenefit/TimeClaimPeriodicBenefit.cpp, and TimeStandardUserSystemClock/TimeStandardUserSystemClock.cpp.

◆ IsValid()

bool nn::time::CalendarTime::IsValid ( ) const

Determines whether valid calendar information is being kept.

Returns
Determines whether the object is valid.
Return Values:

true Valid

false Disabled

Details
Always returns false when the year is 0 or less.

Returns true when all of the following conditions are met.

  • The year, month, and day are a valid date (determined to be equivalent to nn::time::IsValidDate()).
  • 0 <= hour <= 23.
  • 0 <= minute <= 59.
  • 0 <= second <= 59.

Details
Advances the date and time, indicated by this structure, by the amount specified in timeSpan. Calculations are performed with no consideration for time difference changes from daylight savings time and other factors. Leap years are taken into consideration.

For timeSpan, values less than a second are discarded.
