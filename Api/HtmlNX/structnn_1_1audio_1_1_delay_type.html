nn::audio::DelayType Struct Reference

Delay effects. More...

#include <nn/audio/audio_EffectTypes.h>

Detailed Description

Delay effects.

Details
The effects provided by Delay are regular delay effects.

Directly altering member variables is prohibited. Use the main structure through the following functions.

  • AddDelay(AudioRendererConfig* pOutConfig, DelayType* pDelay, void* buffer, size_t bufferSize, SubMixType* pSubMix, nn::TimeSpan delayTimeMax, int channelCountMax)
  • AddDelay(AudioRendererConfig* pOutConfig, DelayType* pDelay, void* buffer, size_t bufferSize, FinalMixType* pFinalMix, nn::TimeSpan delayTimeMax, int channelCountMax)
  • RemoveDelay(AudioRendererConfig* pOutConfig, DelayType* pDelay, SubMixType* pSubMix)
  • RemoveDelay(AudioRendererConfig* pOutConfig, DelayType* pDelay, FinalMixType* pFinalMix)
  • GetRequiredBufferSizeForDelay(nn::TimeSpan delayTimeMax, int sampleRate, int channelCountMax)
  • GetDelayInputOutput(const DelayType* pDelay, int8_t* outInput, int8_t* outOutput, int* pOutCount, int count)
  • SetDelayInputOutput(DelayType* pDelay, const int8_t* input, const int8_t* output, int count)
  • GetDelayTimeMax(const DelayType* pDelay)
  • GetDelayChannelCountMax(const DelayType* pDelay)
  • GetDelayTime(const DelayType* pDelay)
  • SetDelayTime(DelayType* pDelay, nn::TimeSpan time)
  • GetDelayFeedbackGain(const DelayType* pDelay)
  • SetDelayFeedbackGain(DelayType* pDelay, float feedbackGain)
  • GetDelayDryGain(const DelayType* pDelay)
  • SetDelayDryGain(DelayType* pDelay, float dryGain)
  • GetDelayInGain(const DelayType* pDelay)
  • SetDelayInGain(DelayType* pDelay, float inGain)
  • GetDelayChannelSpread(const DelayType* pDelay)
  • SetDelayChannelSpread(DelayType* pDelay, float channelSpread)
  • GetDelayLowPassAmount(const DelayType* pDelay)
  • SetDelayLowPassAmount(DelayType* pDelay, float lowPassAmount)
  • IsDelayEnabled(const DelayType* pDelay)
  • SetDelayEnabled(DelayType* pDelay, bool enable)
  • IsDelayRemovable(const DelayType* pDelay)
  • GetDelayParameters(const DelayType* pDelay)
  • SetDelayParameters(DelayType* OutDelay, const DelayParameterSet* pParameterSet)

To use these functions, include nn/audio.h.
