Detailed Description

The class for handling the I3dl2Reverb effect utility applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform I3dl2Reverb effect processing. It specifies the sample data obtained by the nn::audio::ReadAuxSendBuffer() function as the input for this effect, and outputs the result of applying an environmental reverb effect that partially conforms to the Interactive 3D Audio Level 2 (I3DL2) specification.

The nn::audio::AuxI3dl2Reverb class is a utility class that wraps calls to functions that use nn::audio::AuxI3dl2ReverbType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessAuxReverb() that take the nn::audio::AuxI3dl2ReverbType type for their arguments.

Examples
AudioEffectAuxUtility/AudioEffectAuxUtility.cpp.
