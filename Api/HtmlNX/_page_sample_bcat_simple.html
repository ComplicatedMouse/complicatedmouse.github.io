This sample program introduces the data distribution feature.

Overview

This section covers all of the API in the data distribution library.

File Structure

This sample program is located in Samples/Sources/Applications/BcatSimple/ .

Required Environment

You must create a user account that is associated with a network service account. Also requires an environment where network connections are possible.

How to Use

Nothing specific.

Important Information

Because some features are under development, the data distribution library may be restricted in some cases. For more information about use restrictions, see NintendoSDK Documents.

Execution Procedure

Build the sample program and then run it.

Description

This sample program has the following flow.

  • Synchronizes immediately.
  • Mounts the data distribution cache storage.
  • Enumerates the files that have already been received.
