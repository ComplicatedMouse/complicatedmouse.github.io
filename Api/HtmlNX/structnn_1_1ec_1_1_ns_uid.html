Detailed Description

The ID for uniquely identifying the individual page for a product in the shop.

Details
Different IDs are assigned in the development environment, the production environment, and the like, so be sure to use the value obtained from the server. You can use the nn::ec::MakeNsUid function to create an NSUID from a string representation.
