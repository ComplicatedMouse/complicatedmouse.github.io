Parameters

[in]pCharModelThe Mii character model to add.
[in]pMaterialCallbackCallback function for changing the values for the shader parameter and render info that were defined independently by the user.
[in]pMaterialCallbackParamThe callback function parameter used as the pMaterialCallback parameter.
[in]pDrawParamCallbackThe callback function that returns the appropriate nn::mii::DrawParam* functions from the material and bone names.
[in]pDrawParamCallbackParamThe callback function parameter used as the pDrawParamCallback parameter.
Precondition
pResModel is a valid pointer.
pCharModel is a valid pointer.
pCharModel is initialized.
pMaterialCallback is a valid pointer.
pDrawParamCallback is a valid pointer.
Postcondition
pCharModel to be assigned to pResModel.
