Detailed Description

The Opus multi-stream decoder class for hardware use.

Details
In addition to mono and stereo sound, decodes Opus data for sound sources with three or more channels.

Opus data handled by nn::codec::HardwareOpusDecoder is single-stream and can also be handled by this class. However, this class requires a work buffer size larger than nn::codec::HardwareOpusDecoder. Unless you have a specific reason not to, we recommend using nn::codec::HardwareOpusDecoder for decoding mono and stereo sound Opus data.

Examples
CodecOpusMultiStreamDecoder/CodecOpusMultiStreamDecoder.cpp.
