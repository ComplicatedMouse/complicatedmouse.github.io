◆ NN_OS_ALIGNAS_FIBER_STACK

#define NN_OS_ALIGNAS_FIBER_STACK   NN_ALIGNAS(16)

Macro to specify the alignment of a fiber stack space without guards.

Details
This macro is made to satisfy the alignment rules of a fiber stack space where stack guards will not be set. The following use will align a buffer definition for the stack equivalent to the constant nn::os::FiberStackAlignment.

Use the NN_OS_ALIGNAS_GUARDED_STACK() macro to adjust the alignment of a fiber stack space that will include stack guards.

Example:

NN_OS_ALIGNAS_FIBER_STACK uint8_t fiberStack[8192];
