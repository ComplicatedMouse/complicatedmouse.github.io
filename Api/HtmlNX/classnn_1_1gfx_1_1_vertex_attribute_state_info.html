◆ SetShaderSlot()

void nn::gfx::VertexAttributeStateInfo::SetShaderSlot ( int  value)

Sets the slot number in the shader.

Parameters
[in]valueSlot number.
Precondition
  • value >= -1 && value <= 32767.
Postcondition
  • The slot number is set in the shader.

Details
Set the slot number to the value retrieved from the shader. If SetNamePtr is set to anything other than NULL, that value is prioritized. Specify -1 to set an invalid slot number.

Examples
G3dDemo/ShapeAnimation.cpp, G3dDemo/Simple.cpp, and MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp.
