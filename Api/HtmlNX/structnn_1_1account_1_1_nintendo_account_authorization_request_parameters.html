Detailed Description

Parameters to use for the OAuth 2.0/Open ID Connect procedures with the Nintendo Account server.

Details
These parameters are sent to the server when getting authorization from the Nintendo Account server.

These parameters can be set in the application and optionally in the independent server. Each parameter must have a length of at least 1 and must be terminated with '\0'.

For more information about the OAuth 2.0/Open ID Connect procedures with the Nintendo Account server, see the Independent Server Configuration Manual.
