◆ operator bool()

constexpr nn::net::Ssid::operator bool ( ) const
explicitconstexprnoexcept

Checks whether this object is a valid SSID object.

Returns
Returns true if it is a valid SSID object, or false if it is not.

Details
A valid SSID object is an object that simultaneously satisfies both of the following conditions.

  • size <= SsidSizeMax.
  • value[size] == 0.

These conditions are always satisfied by SSID objects that have been created by the Ssid::Empty(), Ssid::FromAscii(), and Ssid::FromBinary() functions.
