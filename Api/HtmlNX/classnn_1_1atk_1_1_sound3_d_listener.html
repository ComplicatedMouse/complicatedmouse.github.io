Detailed Description

The 3D sound listener class representing listening points in 3D space.

Details
Listeners are used by registering them to the Sound3DManager class. The sound parameters are calculated and set by using information in the listener matrix that is configured for the listener.

Because this type has members that are of the nn::util::Vector3fType type, it must be aligned when allocating in memory. Align to a nn::atk::Sound3DListener::BufferAlignSize byte boundary.

◆ SetMatrix()

void nn::atk::Sound3DListener::SetMatrix ( const nn::util::Matrix4x3fType &  mtx)

Sets a listener matrix.

Details
Sets the listener matrix for a 3D listener. The set matrix is used to calculate 3D sound parameters.

When a new listener matrix is set, the difference between the new matrix and the previous matrix is calculated, and the velocity of the listener is set automatically. The velocity of the listener is reflected inn the change in pitch due to the Doppler effect. The velocity can also be set explicitly by using SetVelocity.

When the coordinates jump to a different position, the velocity is calculated based on the transverse distance. This can produce abrupt changes in pitch if the movement is extremely fast. Note that in such cases, you must explicitly set the velocity by calling SetVelocity after calling this function.

Parameters
[in]mtxThe listener matrix.

◆ SetInteriorSize()

void nn::atk::Sound3DListener::SetInteriorSize ( float  interiorSize)

Sets the interior size.

Details
The interior size is the size of the area in which pan varies, and is set as a distance (radius) from the listener. Pan varies more smoothly with a larger interior size. Pan varies more drastically with a smaller interior size.

Although volume attenuates as the distance from the listener increases, the amount of attenuation depends on the setting for the maximum volume distance (as set by SetMaxVolumeDistance).

void nn::atk::Sound3DListener::SetMaxVolumeDistance ( float  maxVolumeDistance)

Sets the maximum volume range.

Details
If the distance between the listener and actor is smaller than the maximum volume distance, the sound played by that actor is played without any volume attenuation. If the distance exceeds the maximum volume distance, the sound is played at lower volume, attenuating according to the distance.

◆ SetUnitDistance()

void nn::atk::Sound3DListener::SetUnitDistance ( float  unitDistance)

Sets the unit distance for volume attenuation.

Details
Generally, sound volume is attenuated as the distance between the 3D sound actor and 3D sound listener increases. The attenuation velocity is determined by using this unit distance combined with an attenuation coefficient that is set for each sound.

The sound volume is decreased according to the attenuation coefficient for every unit-distance separation of the listener from the actor. Attenuation is performed according to the attenuation curve that is set for each sound.

Parameters
[in]unitDistanceThe unit distance for attenuating the volume.

◆ SetUnitBiquadFilterValue()

void nn::atk::Sound3DListener::SetUnitBiquadFilterValue ( float  value)

Sets the effect strength of the biquad filter.

Details
The value set by this function is used for the "effect strength" in the following equation, which is used in the default implementation of the 3D sound calculation (Sound3DCalculator::CalculateBiquadFilterValue).

Biquad filter value =
{ ( distance between actor and listener - maximum volume distance ) / volume attenuation unit distance }
x effect strength.

(In the default implementation (Sound3DCalculator::CalculateBiquadFilterValue), if the biquad filter value calculated from this equation is larger than the value set by SetMaxBiquadFilterValue, it is rounded to the value set by SetMaxBiquadFilterValue.)

The default is 0.5f.

[in]valueThe biquad filter value.
