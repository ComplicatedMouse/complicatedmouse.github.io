Initializes.

This function is deprecated. Please check the details for usage conditions and substitute functions.

Deprecated:
The Initialize member function, which brings together the arguments used at initialization by InitializeArg, is now available. Use that function instead.

Details
This function remains for compatibility reasons. It allocates memory from the passed memory pool and creates the buffer. Allocate memory from the memory pool and create the buffer.

Parameters
[in]pDeviceDevice.
[in]gpuAccessAccess method from the GPU.
[in]bufferSizeSize of the buffer to create.
[in]bufferCountCount for multi-buffering for the buffer to create.
[in]pMemoryPoolMemory pool from which memory for the buffer to create is allocated.
[in]memoryPoolOffsetOffset from the start of the memory pool of the region from which memory is allocated.
Returns
Returns whether buffer creation succeeded.
