Detailed Description

Allocator that manages the region in the buffer that is temporarily used.

Details
Dynamically allocates the buffer region that is temporarily used during rendering. Also provides a feature to deallocate all at the same time after rendering completes.

Gets the buffer and the range for the offset to be managed, and then initializes.

Parameters
[in]pBufferThe buffer.
[in]baseOffsetThe start of the offset for the region to manage.
[in]sizeThe size, in bytes, of the region to manage.
[in]allocatableAlignmentMaxThe maximum alignment required for the region allocated from this allocator.
[in]historyLengthMaxThe maximum number of history entries to record.
Precondition
IsInitialized() == false.
pBuffer must be a pointer to a valid buffer.
baseOffset >= 0.
baseOffset % allocatableAlignmentMax == 0.
size >= 1 && size <= SizeMax.
allocatableAlignmentMax <= AlignmentMax.
allocatableAlignmentMax must be a power of 2.
historyLengthMax >= 1.
There must be a memory block configured to have the required alignment size, set with SetMemory().
Postcondition
IsInitialized() == true.
IsRecording() == false.
GetHistoryLength() == 0.

Details
This object targets the region with the range specified in the buffer for management. A contiguous valid range must be specified for the region.

baseOffset must be an integer multiple of allocatableAlignmentMax. You can specify an alignment with a value that is at most equal to that specified in allocatableAlignmentMax, using Allocate().

Calling this instance for the same instance is not thread-safe.

void nn::gfx::util::TransientBufferAllocator::Free ( )

Releases the oldest recorded region from among those recorded by this object.

Precondition
IsInitialized() == true.
GetHistoryLength() > 0.
Postcondition
Releases the oldest region recorded in the history.
The value returned by GetHistoryLength() is decremented by 1.

Details
The region allocated between the calls to Begin() and End() is treated as a single unit and released. The regions allocated individually using Allocate() cannot be released on an individual basis. The released region becomes immediately available for allocation. Calling this function multiple times on the same instance is not thread-safe. Calling this function and the function that allocates regions at the same time on the same instance is thread-safe. End()

◆ FillHistory()

void nn::gfx::util::TransientBufferAllocator::FillHistory ( )

Enters the history for the 0th allocation.

Precondition
IsInitialized() == true.
IsRecording() == false.
Postcondition
GetHistoryLength() == GetHistoryLengthMax().

Details
Adds to the history for the 0th allocation until the history is full. Equivalent to the following code.

TransientBufferAllocator transient;
while(transient.GetHistoryLength() < transient.GetHistoryLengthMax())
{
transient.Begin();
transient.End();
}
