Detailed Description

Structure for storing protrusion information.

Details
a protrusion is the part of a shape protruding from what has been identified as a palm. For a shape corresponding to a hand, a protrusion would probably be a group of extended fingers or the arm.

◆ pointCount

int nn::irsensor::Protrusion::pointCount

Number of points in the silhouette of the protrusion.

Points are HandAnalysisSilhouetteState::points[Shape::firstPointIndex + (firstPointIndex + i) % Shape::pointCount] with 0 <= i < pointCount. If pointCount is 0, protrusion is not valid.
