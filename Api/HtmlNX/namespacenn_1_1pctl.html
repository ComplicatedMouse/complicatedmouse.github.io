◆ TryBeginFreeCommunication()

bool nn::pctl::TryBeginFreeCommunication ( bool  isShowUi)

Attempts to start features for freely communicating with others.

Parameters
[in]isShowUiBoolean value representing whether to display within the function that permission to use features for freely communicating with others was not granted.
Returns
Returns true if the features for freely communicating with others are available. Returns false if features are restricted and cannot be used.

Details
For applications that offer features for freely communicating with others, using these features must be permitted in parental controls. This function confirms whether features for freely communicating with others are permitted and also notifies the system that the application has started using features for freely communicating with others. The duration of time when the application is using features for freely communicating with others is called the in-use state.

While features for freely communicating with others are in the in-use state, restrictions for complying with parental controls are placed by the system. For example, if permission to use features for freely communicating with others is revoked while features are in use, the system prompts the user when resuming the application to request permission for using features for freely communicating with others.

If this function returns true, it indicates that features for freely communicating with others are now in the in-use state. To end features for communicating freely with others, call nn::pctl::EndFreeCommunication.

If this function returns false, it indicates that features for freely communicating with others are not permitted in parental controls, so it could not enter the in-use state. Do not provide features for communicating freely with others.

If true is specified for isShowUi, this function displays inside the function that features for communicating freely with others could not enter the in-use state.

Note
If true is specified for isShowUi but permission to use features for communicating freely with others is changed outside the application while the function is internally displaying on the screen, the return value may vary depending on the settings of focus handling mode.
Examples
PctlSimple/PctlSimple.cpp.

◆ EndFreeCommunication()

void nn::pctl::EndFreeCommunication ( )

Ends using features for freely communicating with other users.

Details
This function notifies the system that features for freely communicating with others is no longer in use. Between when this function is called and the next time nn::pctl::TryBeginFreeCommunication is called, features for freely communicating with others are considered to not be in use because those features are not offered during this time. If this function is called when features for freely communicating with others are not in the in-use state, this function does nothing.

Examples
PctlSimple/PctlSimple.cpp.

◆ IsFreeCommunicationAvailable()

bool nn::pctl::IsFreeCommunicationAvailable ( )

Determines whether features for communicating freely with others is available.

Returns
Returns true if the features for communicating freely with others are available. Returns false if they are unavailable.

Details
This function checks whether features for communicating freely with others are permitted, and returns true if they are permitted. Note that true is returned when this function is called while features for communicating freely with others are in the in-use state. Calling this function does not start or end using features for freely communicating with other users.

To start using features for freely communicating with other users, you must call TryBeginFreeCommunication.
