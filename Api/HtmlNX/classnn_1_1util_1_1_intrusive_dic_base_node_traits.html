Detailed Description

template<class T, class Tag = T>
class nn::util::IntrusiveDicBaseNodeTraits< T, Tag >

Class that defines the access method to an intrusive dictionary node.

Used when a node is defined as a base class.

Template Parameters
TThe type of an intrusive dictionary element.
TagA type for the IntrusiveDicBaseNode that is accessed.
