Gets the shader information.

Details
Gets the shader information associated with the shader resource name specified by pName.

If the shader is not owned, the function attempts to load it using the LoadShader() function.

Specify a name in the format xxx_xxx for pName to get the merged archive shader that includes the relevant variation.

Parameters
[in]pDeviceDevice.
[in]pNameThe name of the shader resource.
Returns
Returns the shader information. The function attempts to load the shader if it is not owned. If it cannot load the shader, an invalid value is set to the shader object.

Generates a dynamically created texture name from the BuildArgSet parts hierarchy modifier information and name, and gets the texture information.

Details
NULL can be specified for pNewNameCopy if it is not needed. Internally, AcquireTexture is called by the generated name. The behavior if it is not found conforms to AcquireTexture.

Parameters
[out]pNewNameCopyA buffer to write in if a final name string is needed. Does nothing if NULL is specified.
[in]newNameCopyBufferSizeThe size of the pNewNameCopy buffer.
[in]buildArgSetThe reference to BuildArgSet.
[in]overridedWhether it is an overwritten dynamically created texture.
[in]pDeviceThe device.
[in]pNameThe name of the dynamically created texture.
Returns
Returns the TextureInfo found.
