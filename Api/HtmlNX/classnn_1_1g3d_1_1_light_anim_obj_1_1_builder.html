Detailed Description

The class that builds the LightAnimObj instance.

Constructor & Destructor Documentation

◆ Builder()

nn::g3d::LightAnimObj::Builder::Builder ( )

Constructor.

Details
You must either call Reserve() on all light animation resources used in the instance, or use SetMaxCurveCount() to specify the parameters required to calculate the buffer size.
