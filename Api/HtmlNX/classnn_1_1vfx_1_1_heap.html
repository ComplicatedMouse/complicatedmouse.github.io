Detailed Description

The heap class used exclusively for effects.

Details
The memory allocation and memory release member functions must be implemented in the derivative classes. An instance of this class must be set in the nw::vfx::Config object that is passed when creating an effect system.

Examples
VfxBinarySegmentation/main.cpp, VfxCustomShader/main.cpp, VfxManualEmitterSet/main.cpp, and VfxSimple/main.cpp.
