class nn::gfx::TTimestampBuffer< TTarget >

Graphics abstraction layer representing the buffer used for timestamps.

Template Parameters
TTargetThe target low-level graphics API.

Details
This class represents the layout of content written by the GPU. It does not express low-level graphics API objects.
