enum nvn::SyncWaitResult::Enum

Enumerated values for the type SyncWaitResult.

Enumerator
ALREADY_SIGNALED 

The sync object was already signaled when the wait was requested.

CONDITION_SATISFIED 

The sync object not signaled when the wait was requested, but was signaled prior to the end of the wait period.

TIMEOUT_EXPIRED 

The condition was not satisfied before the wait period expired.

FAILED 

An error occured while waiting on the sync object.

Waiting on a sync object that has not been signaled with a fence command is an error and will produce a result of FAILED.
