◆ Attach() [1/2]

bool nn::ui2d::ArcResourceAccessor::Attach ( void *  pArchiveStart,
const char *  pResourceRootDirectory 
)

Register the archive data.

Details
Align the archive data passed to pArchiveStart to nn::ui2d::ArchiveResourceAlignment bytes.

This enables the GPU to handle the texture data included in the archive data without copying it to another location. The data in the files in the archive is aligned automatically by Layout Archiver.

Define nn::ui2d::ArchiveResourceAlignment to accommodate the largest alignment requirement.

Parameters
[in]pArchiveStartThe pointer to the archive data.
[in]pResourceRootDirectoryThe name of the directory that will be the root of the resource search.
Returns
Returns true if registration succeeds or false if it fails.
