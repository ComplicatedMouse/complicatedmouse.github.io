Detailed Description

The class for handling the polyphase resampler.

Details
The polyphase resampler performs frequency conversion for entered sample data.

The polyphase resampler works most efficiently when the ratio of input and output sample rates is a rational expression where the denominator and the numerator are both positive integers of 255 or less. If the values for inputSampleRate and outputSampleRate do not satisfy this condition, the polyphase resampler converts the output sample rate to the nearest value that satisfies this condition and then performs its process. You can use nn::audio::PolyphaseResampler::CalculateActualOutputSampleRate() to calculate the actual output sample rate and check whether that actual value differs from the specified value (either before or after initialization of PolyphaseResampler) and whether that difference is tolerable.

The nn::audio::PolyphaseResampler class is a utility class that wraps calls to functions that use nn::audio::PolyphaseResamplerType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessPolyphaseResamplerBuffer() that take the nn::audio::PolyphaseResamplerType type for their arguments.
