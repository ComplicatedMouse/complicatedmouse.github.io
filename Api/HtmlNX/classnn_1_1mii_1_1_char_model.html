Gets the size of the dynamic memory pool.

Parameters
[in]pDeviceSpecifies the device.
[in]infoA CharModelInfo that is already set.
Precondition
pDevice != NULL.
Returns
If successful, returns the required size. If it fails, it returns 0.

Details
Gets the size of the dynamic memory pool (memory pool for textures generated by rendering) needed to generate CharModel. This memory pool will be handled as a render target after initializing, so the memory pool properties must be set as Compressible. If you partitioned the memory pool and then initialized it, use this function to get the size of the dynamic memory pool.

Initialization of CharModel.

Parameters
[in]pMemoryThe memory for CharModel to use.
[in]memorySizeSpecifies the size of pMemory.
[in]pDeviceSpecifies the device.
[in]pMemoryPoolThe memory pool for CharModel to use.
[in]memoryPoolOffsetThe offset to the memory pool used by CharModel.
[in]memoryPoolSizeThe size of the memory pool used by CharModel.
[in]resourceThe CharModel build resource.
[in]pTemporaryBuffThe pointer to the temporary buffer.
[in]modelInfoGeneration conditions.
[in]charInfoThe CharInfo of the Mii to be generated.
Returns
Returns a nn::Result that represents the process result.
  • nn::mii::ResultBroken must be handled.
  • nn::mii::ResultInvalid must be handled.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && modelInfo.IsValid() == true && ! CharModel::IsInitialized() && pTemporaryBuffer != NULL && The offset and size are normal. && Resource::IsInitialized() && charInfo is not corrupted.
Postcondition
If successful, CharModel::IsInitialized() = true.

Details

Attention
Both pMemory and pMemoryPool must be retained until the call to CharModel::Finalize().
Pass the memory pool that was set in MemoryPoolProperty_CpuCached and MemoryPoolProperty_Compressible to pMemoryPool. Textures that will be rendering targets are included, so Compressible settings are required.
resource may need to be retained until the call to CharModel::Finalize() based on the Resource settings conditions.
You cannot reinitialize an instance of CharModel that has been initialized already. Call this function after calling Finalize().
Only required when initializing pTemporaryBuffer. You can destroy it after initialization is complete.
Examples

Attention
The DrawParam that is obtained is set permanently at the conditions in which it was obtained. Note that the texture descriptor slot is changed with SetTextureDescriptorSlot(). mask is ignored other than DrawType_Mask. There is dependency between DrawType and the model generation type (CreateModelType, CreateNoseType). The following DrawType depend on CreateModelType.
  • DrawType_Forehead
  • DrawType_Hair
  • DrawType_Hat
The parameters returned for these DrawType differ depending on the value of CreateModelType. In addition, it returns nullptr if specifying a CreateModelType that does not contain the generation flag that was set during initialization. The following DrawType depend on CreateNoseType.
  • DrawType_Nose
  • DrawType_Noseline
The parameters returned for these DrawType differ depending on the value of CreateNoseType. In addition, it returns nullptr if specifying a CreateNoseType that does not contain the generation flag that was set during initialization. Other DrawType do not depend on CreateModelType or CreateNoseType.
