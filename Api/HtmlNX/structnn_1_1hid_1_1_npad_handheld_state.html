Detailed Description

The structure representing the input states for the NpadStyleHandheld play style modeled on holding an Npad handheld controller.

Details
Provides input states for the digital buttons (A, B, X, Y, L, R, ZL, ZR, +, -, and the directional buttons) and the right and left control sticks, modeled on the handheld controller.

There are rules governing the input from the +Control Pad: If up/down or left/right are pressed at the same time, "up" and "left" take precedence.
