Detailed Description

Class for information required to initialize the LoadMeterBase class.

Parameters passed when the CpuMeter and GpuMeter classes are initialized.

◆ SetBufferCount()

void nn::perf::LoadMeterInfo::SetBufferCount ( int  bufferCount)

Specifies the number of buffers held by LoadMeter for storing the measurement results.

It determines how many previous frames of measurement results to keep.

Parameters
[in]bufferCountThe number of buffers in the array for storing measurement results. You must specify at least two double buffers.

Details
The results tabulated by Next() when the frame ends are the measurement results for the previous (bufferCount - 2) frames.

◆ SetCoreCount()

void nn::perf::LoadMeterInfo::SetCoreCount ( int  coreCount)

Sets the number of CPU cores for the application to use.

Parameters
[in]coreCountThe number of CPU cores.

Details
This setting is used to correctly measure the GPU load when GPU commands are issued in parallel. This value is ignored when initializing CpuMeter.
