◆ GetKeyCodeMap()

void nn::kpr::GetKeyCodeMap ( KeyCodeMap *  pOutValue)
noexcept

Gets the key input character code conversion table.

Parameters
[out]pOutValueCharacter code conversion.
Precondition
  • pOutValue != nullptr.
Postcondition
  • *pOutValue is a valid character code conversion table.

Details
Gets the character code conversion table corresponding to the keyboard input settings when called. It is always enabled unless the character code conversion table that was obtained cannot be rewritten. To apply changes in the keyboard input settings, you must get the character code conversion table again.

Gets the character code corresponding to the key input HID Usage ID.

Parameters
[out]pOutValueCharacter code.
[in]usageIdKey input HID usage ID.
[in]modifiersModifier key information for key input.
[in]mapCharacter code conversion.
[in]modeConversion mode for the character code conversion table.
Precondition
  • pOutValue != nullptr.
  • 0x0000 <= usageId && usageId <= 0xffff.
  • map is the valid character code conversion table.

Details
When there is no corresponding character, sets 0 for the pOutValue->code value. If an unsupported conversion mode is specified, the same results are obtained as when KeyCodeMapMode_Default.
