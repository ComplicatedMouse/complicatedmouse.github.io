Detailed Description

◆ SetDefault()

void nn::gfx::TextureViewInfo::SetDefault ( )

A helper function for setting the default value for each parameter.

Postcondition
  • The various parameters are configured.

Details
Equivalent to calling the following.

  • SetImageDimension( nn::gfx::ImageDimension_2d );
  • SetDepthStencilTextureMode( nn::gfx::DepthStencilFetchMode_DepthComponent );
  • SetImageFormat( nn::gfx::ImageFormat_Undefined );
  • EditSubresourceRange().SetDefault();
  • SetChannelMapping( nn::gfx::ChannelMapping_Red, nn::gfx::ChannelMapping_Green, nn::gfx::ChannelMapping_Blue, nn::gfx::ChannelMapping_Alpha );
  • SetTexturePtr( NULL );
Examples
GfxPrimitiveRenderer/GfxPrimitiveRenderer.cpp, GfxShaderCompile/GfxShaderCompile.cpp, ImguiDemo/ImguiDemo.cpp, MiiCompressedTexture/MiiCompressedTexture.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
