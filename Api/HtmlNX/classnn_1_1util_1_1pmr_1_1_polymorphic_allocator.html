template<typename T >
T* nn::util::pmr::PolymorphicAllocator< T >::allocate ( std::size_t  n)

Allocates memory.

Parameters
[in]nThe number of objects to allocate.
Returns
Returns a pointer to the allocated memory space.

Details
Returns a contiguous region of memory that is large enough to be used by n number of T-type objects. This function only allocates memory and does not call the T constructor. It is equivalent to the following implementation.

return static_cast<T*>(this->resource()->allocate(n * sizeof(T), alignof(T)));

Instantiates an object.

Template Parameters
UThe type of the object to construct.
ArgsThe type of the U constructor arguments.
Parameters
[in]pA pointer to the object's memory space.
[in]argsThe U constructor arguments.

Details
Calls the constructor and instantiates a U object in the memory space specified by the pointer p. Uses the convention std::forward<Args>(args)... for the U constructor arguments.

This function supports uses-allocator construction using the *this allocator. If std::uses_allocator<U, PolymorphicAllocator<T>>::value == true, the constructor that can take *this for the argument is called automatically.
