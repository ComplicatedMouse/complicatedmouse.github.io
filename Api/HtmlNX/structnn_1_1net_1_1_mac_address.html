◆ FromAscii()

static constexpr MacAddress nn::net::MacAddress::FromAscii ( const char *  ascii)
staticconstexprnoexcept

Generates a MAC address from a string.

Parameters
[in]asciiThe ASCII string.
Returns
Returns the generated MAC address.

Details
For the ascii parameter, specify a 12-digit hexadecimal string, separated by a delimiter every two digits. As the delimiter, you can use either a colon (:) or a hyphen (-). The hexadecimal string can have either or both uppercase and lowercase letters. Examples of valid strings include the following.

  • 00:22:aa:bb:cc:dd
  • 00-22-AA-BB-CC-DD
  • 00:22-Aa:bB-Cc:dD

The string does not need to be ended with a terminating character. If the string does not comply with this format or if nullptr is passed to the argument, the function returns ZeroMacAddress.
