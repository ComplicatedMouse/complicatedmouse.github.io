Program RGBA channel masks for a single color target.

Parameters
indexNumber of the color target to update. Must be less than 8 (the value of nvn::DeviceInfo::COLOR_BUFFER_BINDINGS).
rEnable (TRUE) or disable (FALSE) writes to the first (R) component.
gEnable (TRUE) or disable (FALSE) writes to the second (G) component.
bEnable (TRUE) or disable (FALSE) writes to the third (B) component.
aEnable (TRUE) or disable (FALSE) writes to the fourth (A) component.
