void nn::g3d::ResBoneAnim::Initialize ( BoneAnimResult *  pResult,
const ResBone *  pBone 
) const

Initializes the animation for the specified bone.

Parameters
[out]pResultPointer to the buffer for storing animation results.
[in]pBonePointer to the bone.

Details
If the animation does not have initial values for scaling, rotation, and translation, initialize it using the scaling, rotation, and translation values of the bone indicated by pBone. If the animation does not have initial values for scaling, rotation, and translation, and the pBone value is NULL, initialize to scaling 1, rotation 0, and translation 0.
