Detailed Description

This class extracts the vibration value from the sound data.

Details
It is a feature for converting sound data to vibration values in real time. This tool is the result of porting the vibration converter PC tool to the development hardware.

The tool automatically performs the following tasks.
Applies a band-path filter to the entered sound data.
Extracts the amplitude and frequency for each nn::hid::VibrationConverter::ChunkDuration of the filtered waveform.
Perform this processing for both the low-frequency and high-frequency bandwidths to determine the vibration value.
The supported input sampling rates are 32,000 Hz and 48,000 Hz.

Examples
HidVibrationRealTimeConverter/HidVibrationRealTimeConverter.cpp.

Parameters

[in]higherCutoffLowMaximum frequency to extract (for low-frequency bandwidth).
[in]lowerCutoffHighMinimum frequency to extract (for high-frequency bandwidth).
[in]higherCutoffHighMaximum frequency to extract (for high-frequency bandwidth).
Precondition
  • sampleRate is 32,000 Hz or 48,000 Hz.
  • lowerCutoffLow is between 10.0 Hz and 2500.0 Hz.
  • higherCutoffLow is between 10.0 Hz and 2500.0 Hz.
  • lowerCutoffHigh is between 10.0 Hz and 2500.0 Hz.
  • lowerCutoffHigh is between 10.0 Hz and 2500.0 Hz.
  • lowerCutoffLow < higherCutoffLow.
  • lowerCutoffHigh < higherCutoffHigh.

Details
You can set maximum and minimum values for the frequency to extract for both low-frequency and high-frequency bandwidth. You can set values in the range from a minimum of 10 Hz to a maximum of 2500 Hz.
The default frequency is [128.0 Hz, 200.0 Hz] for low-frequency bandwidth, and [256.0 Hz, 400.0 Hz] for high-frequency bandwidth.

Converts sound data to vibration values.

Parameters
[out]pOutValueStorage location for the converted vibration values.
[in]pValuesStart address for the memory where the sound data is extracted.
[in]countSound data count.
Precondition
  • pOutValue != nullptr.
  • pValues != nullptr.
  • Valid sound data is written in continuous areas starting with pValues.
  • The format of pValues is mono channel 16-bit PCM.
  • count is the number of samples for nn::hid::VibrationConverter::ChunkDuration hours.

Details
Converts the sound data (mono channel 16-bit PCM) for nn::hid::VibrationConverter::ChunkDuration hours to vibration values.
The number of sound data to input changes depending on the configured sampling rate.
If the sampling rate is not configured with the constructor, it is nn::hid::VibrationConverter::DefaultSampleRate.
