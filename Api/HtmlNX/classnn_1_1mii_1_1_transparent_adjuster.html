Detailed Description

The shader for preventing transparent pixel color bleeding due to linear interpolation.

Details
Calculates the color value for transparent (a=0) pixels based on the surrounding opaque pixels (a!=0) pixels.

Examples
MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Initializes an instance of TransparentAdjuster.

Attention
The memory space specified by pMemory must be retained until the call to TransparentAdjuster::Finalize().
You cannot re-initialize an instance of TransparentAdjuster that has been initialized already. Call this function after calling Finalize().
The initialized resShaderFile must be retained in the initialized state until TransparentAdjuster::Finalize() is called.
Parameters
[in]pMemorySpecifies the memory for TransparentAdjuster to use.
[in]memorySizeThe size of pMemory.

Postcondition
