Detailed Description

template<typename R, typename... Args>
class nn::util::SharedFunction< R(Args...)>

Specialization of SharedFunction<T>.

Class template for managing IFunction<R(Args...)> using a shared pointer.

Template Parameters
RThe type of the values returned by the function.
ArgsThe type of the arguments of the function.

Details
SharedFunction is a class template that handles std::shared_ptr<IFunction<R(Args...)>> in a similar to the std::function interface. This class template offers the following advantages over directly managing IFunction objects with shared pointers.

  • Objects of this class template can be created directly from lambda expressions and the like using implicit type conversion.
  • This class template implements operator(), so you can call IFunction objects as easily as with std::function.

In addition, this class template supports the specification of memory allocators, so it also can be used as a utility to compensate for the std::function memory allocator support that was removed in C++ 17.

This class template has features and an interface resembling std::function. Although it can replace std::function in many cases, note that the specifications differ greatly in the following respects.

  • This class template shares the same function objects among multiple, different objects. If the call to a function object has side effects, the call to one SharedFunction object in some cases affects the result of the call to another SharedFunction object.
  • The copy constructor and copy assignment operator of this class template acts only to copy shared pointers. The actual function objects are not copied.

template<typename R , typename... Args>
template<typename F >
nn::util::SharedFunction< R(Args...)>::SharedFunction ( F  f)

Creates a SharedFunction object with a call target.

Template Parameters
FThe type of the function object.
Parameters
[in]fThe function object.
Postcondition
  • If either of the two following conditions are satisfied, static_cast<bool>(*this) == false.
    • F is a function pointer type and f == nullptr.
    • F is any SharedFunction type and static_cast<bool>(f) == false.
  • Otherwise, static_cast<bool>(*this) == true.

Details
Creates an object from f whose base type is IFunction<R(Args...)> and stores it as the call target of *this. f must be a value that can be used as an argument for IFunction<R(Args...)>::Make().

This function performs dynamic memory allocation. The memory resource obtained by pmr::GetDefaultResource() is used for the memory allocator.
