Detailed Description

A set of parameters for wave sound data.

Details
You can get this information by using SoundArchivePlayer::ReadWaveSoundDataInfo or WaveSoundHandle::ReadWaveSoundDataInfo.

bool nn::atk::WaveSoundDataInfo::loopFlag

◆ loopEnd

int64_t nn::atk::WaveSoundDataInfo::loopEnd

The loop end position when the wave sound data is looped, as a number of samples from the start of the wave sound.

Represents the end of the data, as a number of samples if stream data is not looped. (Points to the sample that follows the last sample played.)

◆ compatibleLoopStart

int64_t nn::atk::WaveSoundDataInfo::compatibleLoopStart

The loop start position when the wave sound data is looped, as a number of samples from the start of the wave sound.

Details
Due to hardware restrictions, a change was made in version 0.3.0 such that the loopStart member variable now returns the pre-corrected value. If you want to get the value like before, use this parameter instead. Note, however, that the use of this member variable is not recommended, and it will be deprecated in the future.
