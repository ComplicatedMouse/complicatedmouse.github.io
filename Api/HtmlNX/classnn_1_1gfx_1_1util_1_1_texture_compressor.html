Adds a shader to use for texture compression.

Parameters
[in]targetFormatCompression format for adding the shader.
[in]pShaderPointer to the shader that will compress to suit the format.
Precondition
  • IsInitialized()
  • IsSupportedFormat(targetFormat)
  • pShader != NULL.
  • nn::gfx::IsInitialized(*pShader)

Details

  • You must add a shader for each compression format.
  • A pointer will be overwritten when you pass a pointer for a separate shader as an argument for the same format.

void nn::gfx::util::TextureCompressor::MakeCommand ( nn::gfx::CommandBuffer *  pCommandBuffer,
const nn::gfx::DescriptorSlot &  srcTextureDescriptorSlot,
const nn::gfx::DescriptorSlot &  samplerDescriptorSlot,
const TextureCompressorTargetInfo &  targetTextureInfo 
) const

Creates a command to compress a texture.

Parameters
[in]pCommandBufferPointer for the command buffer that writes the compression.
[in]srcTextureDescriptorSlotDescriptor slot for the compression source texture.
[in]samplerDescriptorSlotDescriptor slot added by the sampler to use when loading compression source texture values.
[in]targetTextureInfoCompression destination texture information.
Precondition
  • IsInitialized()
  • pCommandBuffer != NULL.
  • nn::gfx::IsInitialized(*pCommandBuffer)
  • IsSupportedFormat(destFormat)
  • destTextureInfo must store the compression destination information using SetTextureInfo() and SetTextureStorage().
  • The number of mipmaps configured to targetTexturInfo must be 15 or less.
  • The length of the array configured to targetTexturInfo must be 65535 or less.
  • A shader that corresponds to the compression format must be added.
Postcondition
  • Commands that perform texture compression are recorded to the command buffer passed using pCommandBuffer.

Details

  • You must not destroy any shaders added when creating compression commands until those commands run.
  • You must not destroy any compression destination textures until those commands run.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp.
