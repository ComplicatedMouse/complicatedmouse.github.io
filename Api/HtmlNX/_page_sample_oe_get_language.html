Describes the sample program that gets language settings using the application operating environment library.

Overview

This section shows how to get language settings using the application operating environment library.

For more information about language settings, see Writing the NMETA File in NintendoSDK Documents.

For more information about using the application operating environment, see the Operating Environment Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OeGetLanguage/ .

Required Environment

This sample program can only be built and run on the NX platform.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program gets language settings, and outputs them to the log. For more information about language settings, see nn::oe::GetDesiredLanguage().

This sample program has the following flow.

  • Store the configured language and the string to display in a map.
  • Get the language setting.
  • Output the obtained language setting.
