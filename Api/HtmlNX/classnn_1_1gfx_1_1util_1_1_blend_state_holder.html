Detailed Description

template<int TBlendTargetCount>
class nn::gfx::util::BlendStateHolder< TBlendTargetCount >

A wrapper class that allocates the necessary additional memory as a member to nn::gfx::BlendState.

The necessary value is taken as a template parameter because the amount of memory to allocate is set statically.

Template Parameters
TBlendTargetCountThe number of blend targets. Specify a value that is equal to or greater than the blend target state information count specified in nn::gfx::BlendStateInfo::SetBlendTargetStateInfoArray().
