Detailed Description

The structure that stores the exception information.

Details
This structure is defined using typedef to match the target environment from separately defined, separately named structures, as in the following table. For more information, see the descriptions of each structure in the API Reference.

|Name of Structure for Individual Target Environment|Description|| |:--—:|:-—| |nn::os::UserExceptionInfoDetailAarch32Status32|Running in the ARMv7A operating environment.| |nn::os::UserExceptionInfoDetailAarch32Status64|Running on ARMv8A in the AArch32 operating environment.| |nn::os::UserExceptionInfoDetailAarch64|Running on ARMv8A in the AArch64 operating environment.| |nn::os::UserExceptionInfoDetailWin32|Running in the Windows environment.|
