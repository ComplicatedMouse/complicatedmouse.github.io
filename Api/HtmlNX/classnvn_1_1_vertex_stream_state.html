◆ SetDivisor()

VertexStreamState & nvn::VertexStreamState::SetDivisor ( int  divisor)

Specifies the frequency divisor for vertices in vertex binding N.

If zero, specifies that the vertex binding holds a separate set of values for each vertex in a rendered primitive. If set to a non-zero value X, a separate set of values will be used for each group of X instances, with the same set of values used for each vertex in a single instance.

Parameters
divisorFrequency divisor for vertex attributes using the specified binding point.
