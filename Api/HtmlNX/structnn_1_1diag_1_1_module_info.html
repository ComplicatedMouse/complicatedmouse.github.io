Detailed Description

The module information.

NX Platform Only:
  • id is the Build ID in elf format.
  • age is always 0.
Windows Platform Only:
  • id is the GUID in pe format.
Examples
BreakpadSimple/BreakpadSimple.cpp, and OsUserExceptionHandlerSample.cpp.
