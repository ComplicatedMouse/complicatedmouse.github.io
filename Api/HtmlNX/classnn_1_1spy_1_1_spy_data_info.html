Parameters
[in]dataNameThe data name. Enter the name using an ASCII string with no more than 128 characters including the final character.
[in]dataVersionThe data version. The version is in 2-byte increments and represents the 4 numerical values, namely major, minor, micro and (not in use). The version number 0x00000000 is considered an invalid value.
