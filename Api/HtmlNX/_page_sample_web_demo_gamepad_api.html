Sample offline HTML to read controller operations using the Gamepad API.

Overview

Offline HTML sample to read controller operations using the Gamepad API. This sample demonstrates the following features.

  • Reading controller operations using the Gamepad API
  • Disabling standard controller operations

File Structure

This sample offline HTML is located in Samples/Sources/Applications/WebDemo/html_document/sample/gamepad_api.htdocs.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

When you operate the controller, the data read using the Gamepad API appears on the screen. Holding down any button for at least five seconds will exit the offline web applet, returning to the menu via the application.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and display the offline HTML from this sample.

Reading Controller Operations Using the Gamepad API

Controller operations are read as Gamepad object properties, and appear on the screen.

Disabling Standard Controller Operations

The following method disables standard controller operations.

  • Use nn::web::ShowOfflineHtmlPageArg::SetPointerEnabled() to limit L Stick controls to spatial navigation.
  • Use the window.nx.footer.unsetAssign() Javascript extension to disable standard controller operations.
  • Use the viewport attribute from the meta element to disable the zoom feature.
