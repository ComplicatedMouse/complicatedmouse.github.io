Detailed Description

Class for storing various types of time information.

This class satisfies the standard C++ requirements for being trivial to copy, and can be copied with std::memcpy().

To continue working with this class while the application is not running, store an instance of the object in save data, and then pass it back to the application the next time the application starts.

This class's member functions are thread-safe.

See also
nn::time::CalculateStandardUserSystemClockDifferenceByUser(), nn::time::CalculateSpanBetween()
Examples
TimeCalculateSpanBetween/TimeCalculateSpanBetween.cpp, TimeCalculateUserSystemClockDifference/TimeCalculateUserSystemClockDifference.cpp, and TimeClaimPeriodicBenefit/TimeClaimPeriodicBenefit.cpp.

Constructor & Destructor Documentation

◆ ClockSnapshot()

nn::time::ClockSnapshot::ClockSnapshot ( )

Constructor.

Details
The created object stores the following information.

  • nn::time::PosixTime, nn::time::CalendarTime, and nn::time::CalendarAdditionalInfo are time values representing 1970-01-01 00:00:00 UTC.
  • nn::time::LocationName is a value representing Coordinated Universal Time (UTC).
  • nn::time::SystemClockContext and nn::time::SteadyClockTimePoint are invalid values.

static void nn::time::ClockSnapshot::CreateWithStandardSystemClock ( ClockSnapshot *  pOut)
static

Creates the object with the latest system time.

Parameters
[out]pOutA pointer to a buffer storing the object.
Precondition
  • The time library is initialized.
  • pOut != nullptr.

Details
Creates the object using the latest system times obtained from nn::time::StandardUserSystemClock and nn::time::StandardNetworkSystemClock.

If the network clock has not been corrected, the network clock time stores a value of 1970-01-01 00:00:00 UTC.

Examples
TimeCalculateSpanBetween/TimeCalculateSpanBetween.cpp, TimeCalculateUserSystemClockDifference/TimeCalculateUserSystemClockDifference.cpp, and TimeClaimPeriodicBenefit/TimeClaimPeriodicBenefit.cpp.

Details
We do not recommend using this function. If you want to know how much time has elapsed in the running application, use the system tick value that you can get from the nn::os::GetSystemTick function.
