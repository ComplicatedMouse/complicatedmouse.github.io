Detailed Description

The handle class that receives the asynchronous result.

Details
Operations on this class interface are not thread-safe. Implement mutual exclusion in the appropriate manner if you intend to use multithreading to call destructors and perform other synchronous processes after you have obtained this asynchronous handle as the result of the success of the ShopServiceAccessor::Request() function.
