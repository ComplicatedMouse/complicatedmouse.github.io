Detailed Description

The display name within the application.

Details
Display names within the application must be UTF-8 encoded.

The number of characters that can be recorded to the system is represented by InAppScreenNameLengthMax. When the display name within the application is too long, the last character on My Page will be replaced with an ellipsis (...).

When adding a display name within the application, the number of characters can exceed InAppScreenNameLengthMax without an issue. Any characters that exceed that limit will be automatically trimmed by the system.

Specify a language code that is a language selected by the application. If the language is dependent on the system settings, specify the language code obtained with nn::oe::GetDesiredLanguage.

When specifying the nickname of the user account for the display name within the application, there is no issue with specifying the language code obtained with nn::oe::GetDesiredLanguage, regardless of the application's language settings.

The language code serves as a hint for font selection when displaying the display name within the application. An unknown language code might be set to handle display names within applications for services other than the target platform.

If an unknown language code is set, reference the language code obtained with nn::oe::GetDesiredLanguage.
