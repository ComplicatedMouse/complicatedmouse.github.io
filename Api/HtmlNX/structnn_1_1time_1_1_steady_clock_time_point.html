Detailed Description

This function is deprecated.

Please check the details for usage conditions and substitute functions.

Deprecated:

Structure representing the values that the monotonic increase clock indicates.

Details
We do not recommend using this architecture. If you want to know how much time has elapsed since the started application was last ended, use the nn::time::ClockSnapshot class and the nn::time::CalculateSpanBetween function.

The nn::time::GetSpanBetween function can be used to calculate the time that elapsed between two values. To calculate the elapsed time in an interval while the application is not running, save the content of this structure to save data, and use it for the next start.
