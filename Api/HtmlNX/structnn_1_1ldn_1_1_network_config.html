Details
Settings for a local network created by an access point.

intentId identifies the purpose of the local network. For more information, see IntentId.

nodeCountMax sets the maximum number of nodes that can connect to the network. The maximum number of nodes that can connect can be set as a value between 1 and NodeCountMax, and this number includes the access point itself.

channel specifies the wireless channel. The available wireless channels for the ldn library are either 1, 6, or 11 in the same 2.4-GHz band as the production device, or 36, 40, 44, or 48 in the 5-GHz band that is only available to development hardware. The frequency band is determined automatically by the specified channels, so there is no need to separately switch between the 2.4-GHz band and the 5-GHz band. If no particular wireless channels need to be specified, pass AutoChannel to make the system search for the most appropriate and uncongested channel among channels 1, 6, and 11 in the 2.4-GHz band. The 5-GHz band channels will not be selected when specifying AutoChannel. Specifying a wireless channel is only effective for development hardware, and production devices will ignore these specifications and the system will always determine the wireless channel.

localCommunicationVersion is the local communication version. Indicate a lack of communication compatibility by incrementing the local communication version in advance when communication specifications will change for things like applying a patch to an application. In the local communication version, set the value in the range from LocalCommunicationVersionMin to LocalCommunicationVersionMax.
