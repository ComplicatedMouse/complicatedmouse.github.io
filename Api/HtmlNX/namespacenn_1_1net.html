◆ GenerateRandomMacAddress()

MacAddress nn::net::GenerateRandomMacAddress ( )
noexcept

Generates a random MAC address.

Details
The MAC address generated by this function has 1 as the U/L bit and 0 as the I/G bit. In other words, the macAddr return value satisfies the following conditions.

  • macAddr.IsBroadcast() == false
  • macAddr.IsGlobal() == false
  • macAddr.IsLocal() == true
  • macAddr.IsMulticast() == false
  • macAddr.IsUnicast() == true

This function is not used for secure random numbers. Use GenerateCryptographicallyRandomMacAddress() when you need a MAC address that is difficult to guess.

◆ GenerateCryptographicallyRandomMacAddress()

MacAddress nn::net::GenerateCryptographicallyRandomMacAddress ( )
noexcept

Generates a random MAC address.

Details
The MAC address generated by this function has 1 as the U/L bit and 0 as the I/G bit. In other words, the macAddr return value satisfies the following conditions.

  • macAddr.IsBroadcast() == false
  • macAddr.IsGlobal() == false
  • macAddr.IsLocal() == true
  • macAddr.IsMulticast() == false
  • macAddr.IsUnicast() == true

Although this function can be used for secure random numbers, it is slow. Use GenerateRandomMacAddress() when you need a MAC address that is difficult to guess.
