Detailed Description

A class that represents the results of sound playback.

Details
ResultCode_ErrorNotEnoughInstance indicates that the necessary number of instances have not been created for the sound you are attempting to play. For playback using a sound archive, you must increase the number of instances set within the sound archive.

SoundStartable::StartSound.

enum nn::atk::SoundStartable::StartResult::ResultCode

Codes representing the results of sound playback.

Enumerator
ResultCode_Success 

Playback was successful.

ResultCode_ErrorLowPriority 

Could not play because (1) the number of sounds that can be played simultaneously in the player has been exceeded, and (2) this sound had a lower priority than other sounds.

ResultCode_ErrorInvalidLabelString 

Playback failed because the sound corresponding to the specified label string could not be found.

ResultCode_ErrorInvalidSoundId 

Playback failed because the sound corresponding to the specified sound ID could not be found.

ResultCode_CanceledBySinglePlay 

Playback of the sound corresponding to the specified sound ID was stopped by the single playback feature configured to prioritize the oldest.

ResultCode_ErrorCanceledForPrioritizingOldest 

This function is deprecated.

Please check the details for usage conditions and substitute functions.

Deprecated:
This function is scheduled for removal. Instead, use the ResultCode_CanceledBySinglePlay function. Playback of the sound corresponding to the specified sound ID was stopped by the single playback feature configured to prioritize the oldest.
ResultCode_ErrorNotDataLoaded 

(Currently not used.)

ResultCode_ErrorNotSequenceLoaded 

Playback failed because sequence data has not been loaded, and there is no player heap.

ResultCode_ErrorNotBankLoaded 

Playback failed because bank data has not been loaded, and there is no player heap.

ResultCode_ErrorNotWsdLoaded 

Playback failed because wave sound data has not been loaded, and there is no player heap.

ResultCode_ErrorNotWarcLoaded 

Playback failed because wave archive data has not been loaded, and there is no player heap.

ResultCode_ErrorNotEnoughPlayerHeap 

Playback failed because the player heap has not been properly prepared.

ResultCode_ErrorCannotOpenFile 

(Currently not used.)

ResultCode_ErrorNotAvailable 

Playback failed either because SoundArchivePlayer::Initialize was not called, or because the functions to initialize the SoundArchive set in SoundArchivePlayer (FsSoundArchive::Open,, MemorySoundArchive::Initialize,, and AddonSoundArchive::Initialize) were not called.

Playback failed.

ResultCode_ErrorCannotAllocateTrack 

(Currently not used.)

ResultCode_ErrorNotEnoughInstance 

Allocation of the sound instance failed and playback failed because the priority was too low.

You can set the number of sound instances in SoundMaker in Project Settings, on the Sound Archive tab. To increase the priority of the sound, increase the player priority.

ResultCode_ErrorInvalidParameter 

Playback failed because an invalid parameter was specified.

Currently, this error is generated if the actorPlayerId specified in StartInfo is invalid (has a value other than 0 through 3).

ResultCode_ErrorInvalidSequenceStartLocationLabel 

Playback failed because the startLocationLabel variable specified by the sequenceSoundInfo parameter of StartInfo is not valid.

ResultCode_ErrorActorNotInitialized 

Playback failed because SoundActor or Sound3DActor was called prior to initialization.

ResultCode_ErrorInvalidWarcId 

(Currently not used.)

ResultCode_ErrorInvalidBankData 

No wave archive data was found and playback failed because the loaded bank data is invalid.

ResultCode_ErrorCannotLoop 

Playback failed because looping was enabled for audio that cannot be looped, such as when looping is enabled for Opus stream sound with no looping information.

ResultCode_ErrorUser 

Not used inside the sound library.

The user is free to use ResultCode_ErrorUser through ResultCode_ErrorUnknown – 1.

ResultCode_ErrorUnknown 

Playback failed for an unknown reason.
