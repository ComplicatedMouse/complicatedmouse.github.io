Gets the UUID string representation.

Parameters
[out]pOutStringSpecifies the pointer to the buffer assigned to the string.
[in]sizeSpecifies the size of the buffer passed to pOutString.
Precondition
  • pOutString != nullptr.
Returns
Returns pOutString.

Details
Writes the string representation of the UUID to the buffer passed by pOutString. When a value of 1 or greater is passed to size, the string in pOutString is null-terminated. To completely get the string representation, a value of StringSize or greater must be specified for inCount.
