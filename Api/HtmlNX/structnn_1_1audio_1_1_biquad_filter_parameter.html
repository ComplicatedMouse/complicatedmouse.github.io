Detailed Description

Biquad filter parameters.

Details
Can be enabled or disabled by using enable.

Biquad filter transfer functions are represented as follows. numerator and denominator are treated as fixed-point values in Q14 format.

        numerator[0] +  numerator[1] * z^{-1}  +  numerator[2] * z^{-2}
H(z) = ------------------------------------------------------------------
          (1 << 14)  - denominator[0] * z^{-1} - denominator[1] * z^{-2}
