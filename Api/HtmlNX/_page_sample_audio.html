Audio Library Samples

Describes the audio library samples.

Introduction

Describes the audio library samples. The audio library features reside in the nn::audio namespace.

Samples

Sample code is located in the directories starting with the prefix Audio under Samples/Sources/Applications/. The following table lists the samples.

Page Content
Audio CPU Renderer Sample Describes the sample program for the audio CPU renderer.
Simultaneously Using Audio Input and Output Features Sample This sample program shows how to simultaneously use the audio input and output features.
Audio Effects Sample This sample program shows how to use the audio effects feature.
Audio Effects Using Aux Sample Describes the sample program for audio effects that use Aux.
Audio Memory Pool Sample Describes the audio memory pool sample program.
Multi-Audio Renderer Sample This sample program shows how to use the multi-audio renderer.
Audio Playback Sample This sample program shows how to use audio playback.
Resampler Sample Describes the resampler sample program.
Audio Renderer Performance Metrics Sample This sample program shows techniques for gathering performance information about audio renderers.
Audio Renderer Sample Sample program for the audio renderer.
Audio Renderer Sinks This sample program describes how to use audio renderer sinks.
SplitterType Sample Program This sample program demonstrates how to use nn::audio::SplitterType.
Handling of Various Settings for Audio Output Devices Sample This sample program describes the settings unique to audio output devices and the settings for application-specific downmixing.
