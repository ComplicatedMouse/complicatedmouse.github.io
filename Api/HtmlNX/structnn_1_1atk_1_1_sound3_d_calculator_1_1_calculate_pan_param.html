Detailed Description

The parameter structure for pan calculations.

Details
This structure is the parameter structure that is referenced when the pan is calculated by CalculatePan.

The stereoSpeakerAngle parameter sets the speaker position to use when the pan is calculated. The speakers are placed symmetrically on the left and right, and the angle from the front of both speakers is specified in radians. The speaker is positioned at 45Â° in stereo mode.

surroundPanOffset sets the offset value for the surround pan. The value to set has the same scale as the normal surround pan value. If it is set to -1.0f, a surround pan that was placed in the center emits sound from the front. If it is set to 1.0f, a surround pan that was placed in the center emits sound from the back.
