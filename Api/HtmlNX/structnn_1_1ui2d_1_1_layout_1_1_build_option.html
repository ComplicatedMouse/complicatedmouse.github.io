◆ isRootPaneParts

bool nn::ui2d::Layout::BuildOption::isRootPaneParts

Indicates whether to make the root pane the parts pane.

Details
The default is false.

Normally, the layout root pane is nn::ui2d::Pane (null pane). However, if this option is set to true, the root pane is nn::ui2d::Parts (parts pane). As a result of this setting, when the built layout is attached to another layout, it can take the same state as it takes as a part placed on LayoutEditor. It can be searched for inside the attached layout when FindPaneByName is run.

Note that if the built layout is attached to another layout, to take the same state as a part placed on LayoutEditor, you must enter it not only in the ChildList for the pane but also in the PartsPaneList for the layout. For more information, see GetPartsPaneList. For more information, see GetPartsPaneList.

See also
FindPaneByName
GetPartsPaneList
