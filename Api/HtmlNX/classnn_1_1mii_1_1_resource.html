Initializes Resource.

Attention
pMemory must be retained until Resource::Finalize() is called.
You cannot reinitialize an instance of Resource that has been initialized already. Call this function after calling Finalize().
Parameters
[in]pMemoryThe memory that Resource will use.
[in]memorySizeThe size of pMemory.
[in]pTextureResourceThe pointer to a texture resource file.
[in]textureResourceSizeThe size of texture resources.
[in]pShapeResourceThe pointer to a shape resource file.
[in]shapeResourceSizeThe size of shape resources.
[in]infoResourceInfo.
[in]pDeviceThe device.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && modelInfo.IsValid() != NULL. && ! Resource::IsInitialized(). && The offset and size are normal. && charInfo is not corrupted. && pTextureResource and textureResourceSize are normal. && pShapeResource and shapeResourceSize are normal. && Both pTextureResource and pShapeResource && satisfy ResourceMemoryAlignment to request resource alignment.
Postcondition
If successful, Resource::IsInitialized() = true.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
