Detailed Description

The friend list filter.

Details
Various filters can be applied when getting the friend list. When the filter is zero-initialized, you get all friends.

For example, to filter on favorite friends who seem available to play, specify the following parameters.

  • presenceStatus: PresenceStatusFilter_OnlineOrOnlinePlay
  • isFavoriteOnly: true
  • isSameAppPresenceOnly: true
  • isSameAppPlayedOnly: false
  • isArbitraryAppPlayedOnly: false
  • presenceGroupId: 0

Determinations are made based on the last application played within the play log and friend presence when you set isSameAppPlayedOnly and isArbitraryAppPlayedOnly to true. The play log is a log of applications the user has recently played. Older entries can be removed. You can also make the log private or delete it whenever you want. As a result, you cannot determine with certainty whether a particular application has been played.

When you set isArbitraryAppPlayedOnly to true, you must set the presence group ID of an application to presenceGroupId.
