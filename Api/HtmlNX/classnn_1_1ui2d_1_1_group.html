◆ Group() [2/4]

nn::ui2d::Group::Group ( const char *  pName)
explicit

Constructor.

Details
Initializes in a state with no panes, and a state where the group name is specified.

The pointer to the string specified by the parameter is maintained. Do not specify a pointer to a string temporarily allocated on the stack or elsewhere.

Use a string no longer than nn::ui2d::GroupNameStrMax.

Parameters
[in]pNameThe group name. Specify a string that is not NULL.

◆ Group() [4/4]

nn::ui2d::Group::Group ( const Group &  group,
Pane *  pRootPane 
)
explicit

Copy constructor.

Details
If you assign nullptr to pRootPane, the same pane as the copy source is registered as the root pane. If you assign the pointer to a pane to pRootPane, the functions searches again, looking below that pane for a pane with the same name as the source. This feature is used to reference the newly copied pane in the destination when copying a layout.

Parameters
[in]groupA reference to a group resource.
[in]pRootPanePointer to the root pane.
