The initialization process.

Details
To internally create the render target, specify a memory pool set to nn::gfx::MemoryPoolProperty_CpuInvisible | nn::gfx::MemoryPoolProperty_GpuCached | nn::gfx::MemoryPoolProperty_Compressible. If you are using nns::gfx::GraphicsFramework, use a memory pool set to nns::gfx::GraphicsFramework::MemoryPoolType_RenderTarget.

Parameters
[in]pDeviceThe gfx device.
[in]pInfoTypeParameter information at initialization time.
[in]pMemoryPoolThe memory pool to use when creating the internal render target.
[in]memoryPoolOffsetOffset in the memory pool.
[in]pAllocateFunctionMemory allocation function.
[in]pFreeFunctionMemory deallocation function.
[in]pAllocateFunctionUserDataPointer to the user data passed to the functions for allocating and deallocating memory.
