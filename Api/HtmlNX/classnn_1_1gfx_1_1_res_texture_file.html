Initializes a texture resource file.

Template Parameters
TTargetThe target low-level graphics API.
Parameters
[in]pDevicePointer to the device.
[in]pMemoryPoolThe memory pool where the texture resource file is located.
[in]memoryPoolOffsetThe offset, in bytes, to the location of the texture resource file in pMemoryPool.
[in]memoryPoolSizeThe size, in bytes, of the pool in pMemoryPool.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
Postcondition
  • The texture resource file is initialized.

Details
If pMemoryPool == NULL, the function uses a memory pool that holds texture resources to initialize a texture resource file.
