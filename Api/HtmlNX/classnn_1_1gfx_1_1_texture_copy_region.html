Detailed Description

Class that indicates a range of texture copy operations.

Examples
EcCatalog/Main.cpp, G3dDemo/Town.cpp, ImguiDemo/ImguiDemo.cpp, and VfxSimple/main.cpp.

◆ SetDefault()

void nn::gfx::TextureCopyRegion::SetDefault ( )

A helper function for setting the default value for each parameter.

Postcondition
  • The various parameters are configured.

Details
Equivalent to calling the following.

  • SetOffsetU( 0 );
  • SetOffsetV( 0 );
  • SetOffsetW( 0 );
  • SetWidth( 1 );
  • SetHeight( 1 );
  • SetDepth( 1 );
  • EditSubresource().SetDefault();
  • SetArrayLength( 1 );
Examples
EcCatalog/Main.cpp, G3dDemo/Town.cpp, ImguiDemo/ImguiDemo.cpp, and VfxSimple/main.cpp.
