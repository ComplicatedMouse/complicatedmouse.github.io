nn::fs::FileHandle Struct Reference

Handle for working with a file. More...

#include <nn/fs/fs_File.h>

Detailed Description

Handle for working with a file.

Details
If an operation on a file handle fails due to an abnormal internal state, the error state flag is set on that file handle for all subsequent operations. If any operation other than nn::fs::CloseFile() is performed on a file handle in an abnormal state, a notification is sent to the developer to prompt the proper handling of the situation.
