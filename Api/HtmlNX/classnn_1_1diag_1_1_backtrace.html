Detailed Description

The class that gets the backtrace.

Details
nn::diag::Backtrace is a class that gets information while transversing the stack frames, in order.

NX Platform Only:
This class does not work correctly for programs built with the -fomit-frame-pointer option.

Details
Starts to get a backtrace using register information.

The assumed use is for getting a backtrace for threads where exceptions occurred, from the user exception handler.

Call this function, passing the register information for the thread where an exception occurred, based on the exception data in the user exception handler.

Windows Platform Only:
This constructor cannot be used in a Windows environment.

◆ Step()

bool nn::diag::Backtrace::Step ( )

Performs one trace of the stack frames in the direction of the caller.

Returns
Returns true if the caller was traced, or false otherwise. If the function returns false, it has traced to the end, and it will always return false after that. The initial stack frame information is obtained using the constructor. Call Step() after getting the stack frame information.
