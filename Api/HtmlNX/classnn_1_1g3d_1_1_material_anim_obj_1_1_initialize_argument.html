Detailed Description

The use of internal features is prohibited. Parameters passed to the MaterialAnimObj::Initialize() function for initialization.

Constructor & Destructor Documentation

◆ InitializeArgument()

nn::g3d::MaterialAnimObj::InitializeArgument::InitializeArgument ( )

Constructor.

For all material animation resources and all model resources used in the instance, you must call Reserve() or use SetMaxMaterialCount(), SetMaxMaterialAnimCount(), SetMaxParamAnimCount(), SetMaxCurveCount(), and SetMaxTextureCount() to set the parameters needed to calculate the buffer size.
