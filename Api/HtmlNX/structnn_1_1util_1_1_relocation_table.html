◆ Relocate()

void nn::util::RelocationTable::Relocate ( )

Fixes the location of a binary file in memory and enables data access.

Precondition
IsRelocated() is false for the corresponding binary file header.

Details
Converts the offset value held by BinTPtr within the binary file to a pointer, and sets a flag indicating that conversion is complete in the binary file header.

◆ Unrelocate()

void nn::util::RelocationTable::Unrelocate ( )

Restores the binary file to a movable state in memory and disables data access.

Precondition
IsRelocated() is true for the corresponding binary file header.

Details
Reverse converts the pointer held by BinTPtr within the binary file to an offset value, and sets a flag indicating that conversion has not yet occurred in the binary file header.
