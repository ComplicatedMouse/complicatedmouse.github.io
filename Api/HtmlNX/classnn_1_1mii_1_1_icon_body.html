Initializes IconBody.

Parameters
[in]pMemoryThe memory that IconBody will use.
[in]memorySizeSpecifies the size of pMemory.
[in]pDeviceSpecifies the device.
[in]pMemoryPoolThe memory pool that IconBody will use.
[in]memoryPoolOffsetThe offset to the memory pool that IconBody will use.
[in]memoryPoolSizeThe size of the memory pool that IconBody will use.
[in]pBodyA pointer to the body resource file.
[in]bodySizeThe size of pBody.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && The offset and size are normal. && charInfo is not corrupted.
Postcondition
If successful, IconBody::IsInitialized() = true.

Details
You can free pBody after initialization.

Attention
pMemory and pMemoryPool must be retained until IconBody::Finalize().
You cannot reinitialize an instance of IconBody that has been initialized already. Call this function after calling Finalize().
You need to flush the memory pool after calling this function when the memory pool that was passed at initialization was CpuCached.
Examples
MiiSimpleIconBody/MiiSimpleIconBody.cpp.
