Detailed Description

The heap class for sound data.

Details
In sound heaps, you can set a callback function to call when a memory block is freed by the sound heap. This feature is used in the sound library to perform processing to safely free the data when a memory block that contains sound data is freed.

Examples
AtkAddonSoundArchive/AtkAddonSoundArchive.cpp, AtkEffect/AtkEffect.cpp, AtkFilter/AtkFilter.cpp, AtkGroup/AtkGroup.cpp, AtkMixMode/AtkMixMode.cpp, AtkProfile/AtkProfile.cpp, AtkSimple/AtkSimple.cpp, AtkSound3D/AtkSound3D.cpp, AtkSoundActor/AtkSoundActor.cpp, AtkStreamSound/AtkStreamSound.cpp, AtkSubMix/AtkSubMix.cpp, and AtkViewer/AtkViewer.cpp.

◆ SaveState()

int nn::atk::SoundHeap::SaveState ( )

Saves the current state of the sound heap.

Details
The hierarchical level immediately after heap creation is 0. The hierarchical level is incremented by one each time this function is called. By calling LoadState you can restore the state that existed immediately after the specified level was saved.

Saving the heap state consumes a small amount of the heap. This function call fails if there is not enough free space in the heap.

Returns
Returns the hierarchical level after the state has been saved. If the state fails to be saved, the function returns -1.

◆ LoadState()

void nn::atk::SoundHeap::LoadState ( int  level)

Restores the state of the sound heap to the saved state.

Details
The sound heap is restored to the state it was in immediately after it was saved by SaveState. All memory blocks allocated from the time the state was saved to the current time are deallocated. The callback function set by Allocate is invoked for each memory block that is freed.

Set level (the hierarchical level) to a value equal to or less than the current level. Specifying a value of 0 has the same effect as calling Clear. The heap returns to the state it was in immediately after it was saved, and the current hierarchical level takes the assigned value.

Parameters
[in]levelThe hierarchical level to restore.
