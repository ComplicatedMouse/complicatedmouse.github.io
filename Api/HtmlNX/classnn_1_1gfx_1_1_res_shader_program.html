◆ Initialize()

template<typename TTarget >
ShaderInitializeResult nn::gfx::ResShaderProgram::Initialize ( TDevice< TTarget > *  pDevice)

Initializes the shader.

Template Parameters
TTargetThe target low-level graphics API.
Parameters
[in]pDevicePointer to the device.
Returns
Returns the result of shader initialization.
Return Values:

ShaderInitializeResult_Success Succeeded.

ShaderInitializeResult_InvalidFormat The shader format specified in info is not valid in this environment.

ShaderInitializeResult_SetupFailed Shader setup failed due to a compiling failure or other problem.

Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The shader is not initialized.
  • The shader container that the shader is associated with is initialized.
Postcondition
  • The shader is initialized.

Details
Performs processing equivalent to the processing of the Initialize function in nn::gfx::Shader.

Parameters
