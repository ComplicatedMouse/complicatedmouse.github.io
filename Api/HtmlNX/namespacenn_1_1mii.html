nn::gfx::Device *  pDevice 
)

Gets the color target format for rendering dynamically created textures.

Parameters
[in]textureFormatTexture format
[in]pDevicePointer to the device.
Returns
Returns the color target format based on the received texture format.
Precondition
Formats that textureFormat can use as a texture. && pDevice != nullptr && The texture target format must be either UnormSrgb or Unorm.

Details
Converts to a Unorm format when an image in the UnormSrgb format was passed. Remains as is when the Unorm format is passed. Because dynamically created textures such as the Mask and Faceline textures must be handled in the SRGB space, the color target format must be Unorm, even when rendering textures in UnormSrgb format.
