◆ FullKeyIndex

enum nn::settings::FullKeyIndex

Identifier that uniquely identifies the controller to emulate.

Details
A maximum of nn::settings::FullKeyDeviceCountMax controllers, that support NpadStyleFullKey, can be emulated. This enumerator is used to uniquely identify the controller to emulate. The values and order of the enumerators do not have any particular meaning. They are assigned to Npads in order of the youngest open Npads when an assigned button on a generic gamepad, or a keyboard button, is pressed.

◆ GetDebugPadGenericPadMap()

void nn::settings::GetDebugPadGenericPadMap ( DebugPadGenericPadMap *  outValue)
noexcept

Gets the generic gamepad assignments mapped to DebugPad.

Details
The generic gamepad assigned to DebugPad can be identified using the DebugPadGenericPadMap::genericPadId identifier. The identifier is a value that is specific to a platform and is defined per platform. The assignments to the various digital buttons (such as DebugPadGenericPadMap::buttonA) are specified using the values defined by GenericPadButton. For example, if GenericPadButton_1 is specified as DebugPadGenericPadMap::buttonA, the button press state of button 1 on the generic GamePad corresponds to the A Button on the DebugPad. If 0 is specified as the setting value, it will be handled as if no buttons on the generic gamepad correspond to that digital button. The assignments to the various coordinate axes (such as DebugPadGenericPadMap::axisAnalogStickRX) are specified using the values defined by GenericPadAxis. For example, if GenericPadAxis_X is specified for DebugPadGenericPadMap::axisAnalogStickRX, the x-axis coordinate of the generic GamePad with the DebugPadGenericPadMap::genericPadId identifier corresponds to the x-axis of the right control stick on the DebugPad. If 0 is specified, it is handled as if no coordinate axes on the generic gamepad correspond to that coordinate axis. Use DebugPadGenericPadMap::attributes to determine whether the directions of increase and decrease are reversed for the coordinate values. For example, if DebugPadGenericPadMap::attributes.Get<DebugPadGenericPadMapAttribute::IsXAxisInverted>() is true, the directions of increase and decrease of coordinate values along the x-axis of the generic gamepad are being used in reverse. If this function is called when the generic gamePad is not assigned to DebugPad, it returns an internally resolved assignment.

Parameters
[out]outValueThe buffer to load the generic gamePad assignment to Xpad
Precondition
  • outValue != nullptr.

void nn::settings::GetFullKeyGenericPadMap ( FullKeyGenericPadMap *  pOutValue,
FullKeyIndex  index 
)
noexcept

Gets the generic gamepad assignment settings for emulating controllers that support NpadStyleFullKey.

Details
The generic gamepad assigned to the controller uniquely identified by FullKeyIndex can be identified by the FullKeyGenericPadMap::genericPadId identifier. The identifier is a value that is specific to a platform and is defined per platform. Assignments for various digital buttons, such as FullKeyGenericPadMap::buttonA, use the values defined by GenericPadButton. For example, if GenericPadButton_1 is specified for FullKeyGenericPadMap::buttonA, the state where button 1 is pressed on the generic gamepad with the FullKeyGenericPadMap::genericPadId identifier will be associated with button A on the controller that supports NpadStyleFullKey. If 0 is specified as the setting value, it will be handled as if no buttons on the generic gamepad correspond to that digital button. Assignments for various coordinate axes, such as FullKeyGenericPadMap::axisAnalogStickRX, use the values defined by GenericPadAxis. For example, if GenericPadAxis_X is specified for FullKeyGenericPadMap::axisAnalogStickRX, the x-axis coordinate of the generic GamePad with the FullKeyGenericPadMap::genericPadId identifier corresponds to the right control stick x-axis on the simulated controller. If 0 is specified, it is handled as if no coordinate axes on the generic gamepad correspond to that coordinate axis. Use FullKeyGenericPadMap to determine whether the increase and decrease direction of the coordinate values are reversed. As an example, if the value of FullKeyGenericPadMap::attributes.Test<GenericPadAxisAttribute::IsXAxisInverted>() is true, the x axis of the generic gamepad is used with the increase and decrease directions reversed. If relevant functions are called when a generic gamepad has not been assigned to the controller specified by FullKeyIndex, the assignments that are resolved internally are returned.

Parameters
[out]pOutValueThe buffer for loading the generic gamepad assignments for controllers that support NpadStyleFullKey.
[in]indexFullKeyIndex for uniquely identifying controllers that support NpadStyleFullKey.
Precondition
  • pOutValue != nullptr.
