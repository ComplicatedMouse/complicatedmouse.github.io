◆ SetAllocator()

void nn::gfx::SetAllocator ( const Allocator *  pAllocator)

Sets the memory allocator.

Precondition
  • If pAllocator != NULL, pAllocator->GetAllocateFunction() != NULL and pAllocator->GetFreeFunction() != NULL are satisfied.
Postcondition
  • If pAllocator != NULL, the allocator is set.
  • If pAllocator == NULL, the allocator is not set.

Details
If an allocator is set, memory that is allocated dynamically in the gfx library is allocated through the allocator. If an allocator is not set, memory that is allocated dynamically in the gfx library is allocated though methods that depend on the gfx implementation. Call this function only once before calling nn::gfx::Initialize. The behavior is undefined if this function is called after calling nn::gfx::Initialize.
