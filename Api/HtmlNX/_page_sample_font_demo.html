This sample program shows how to use the font library.

Overview

This documentation describes the FontDemo sample program.

File Structure

This sample program is located in Samples/Sources/Applications/FontDemo .

Required Environment

Nothing specific.

How to Use

The Joy-Con, DebugPad, mouse, and touch panel can be used to perform the following operations.

Input Operation
Select items using up or down on the +Control Pad or the mouse. Select the sample.
Press the A Button or left mouse button. Execute the sample.
Press the B Button or right mouse button. Returns to the menu.

Cautions

Nothing specific.

Execution Procedure

You can build the sample program and select which sample to run from the menu.

Description

Each sample is the most basic demo for loading fonts from disc and displaying them.

Check the following sample by selecting from the menu or changing the g_SampleMode value, or by entering a number starting from 0 as the parameter value.

  • SampleMode_Simple: Displays text using a bitmap font.
  • SampleMode_PairFont: Displays text using a pair font (a combination of multiple fonts).
  • Displays SampleMode_WordWrapping strings with word wrapping enabled.
  • SampleMode_ScalableFont: Displays text using a scalable font format such as TTF (OTF) or BFTTF. When the sample is executed, the text images are created in memory and then displayed.
  • This sample shows the SampleMode_TagProcessor tag processor.
  • This sample displays the contents of the SampleMode_ViewingTextureCache on the screen.

Other: Scaling font sizes, specifying line spacing, specifying character spacing, changing colors, color gradations, italics, drop shadows, showing/hiding the border of outline fonts ...and many other capabilities.

For more information, see the API Reference. You can also refer to the layout library source code and font library code samples. (To find related code, search for "font::".)

Use the dedicated converter to create bitmap fonts. (Located in Tools/Graphics/FontConverter.)
