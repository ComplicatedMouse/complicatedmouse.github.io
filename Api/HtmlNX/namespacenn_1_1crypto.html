Detailed Description

Namespace for the encryption feature library.

Details
The crypto library provides the following classes. See the links for a description of each class.

Block Cipher

  • AES: AesEncryptor / AesDecryptor

Cipher Modes (templates that take a block cipher as an argument)

  • CBC mode: CbcEncryptor / CbcDecryptor
  • CTR mode: CtrEncryptor / CtrDecryptor
  • CCM mode: CcmEncryptor / CcmDecryptor
  • GCM mode: GcmEncryptor / GcmDecryptor
  • CMAC mode: CmacGenerator

Cipher Mode (AES 128-Bit Key)

  • AES-128-CBC: Aes128CbcEncryptor / Aes128CbcDecryptor
  • AES-128-CTR: Aes128CtrEncryptor / Aes128CtrDecryptor
  • AES-128-CCM: Aes128CcmEncryptor / Aes128CcmDecryptor
  • AES-128-GCM: Aes128GcmEncryptor / Aes128GcmDecryptor+
  • AES-128-CMAC: Aes128CmacGenerator

Cipher Mode (AES 192-Bit Key)

  • AES-192-CTR: Aes192CtrEncryptor / Aes192CtrDecryptor

Cipher Mode (AES 256-bit Key)

  • AES-256-CTR: Aes256CtrEncryptor / Aes256CtrDecryptor
  • AES-256-GCM: Aes256GcmEncryptor / Aes256GcmDecryptor

Hash Functions

  • SHA-1: Sha1Generator
  • SHA-256: Sha256Generator
  • SHA-512: Sha512Generator
  • SHA-3: Sha3Generator
  • MD5: Md5Generator

MAC

  • HMAC: HmacGenerator
  • CMAC: CmacGenerator

RSA Cryptographic Primitive

  • RsaCalculator

RSA encryption

  • (RSA-OAEP) RsaOaepEncryptor / RsaOaepDecryptor

RSA Signature

  • RSA-PKCS1: RsaPkcs1Signer / RsaPkcs1Verifier
  • RSA-PSS: RsaPssSigner / RsaPssVerifier
