◆ PushMaskTexture()

void nn::ui2d::detail::VectorGraphicsDrawInfo::PushMaskTexture ( nn::gfx::DescriptorSlot *  pSlot)

Sets the texture for the mask texture stack.

Details
This data structure supports the state where masks are nested, such as nested composition when rendering shapes. When updating the hierarchical structure of the layer, if a mask is set, this structure creates a mask texture and adds it to this stack. When rendering with masks applied, the masks on the upper layer are all set as textures and blended in the shader for application. TexMapMax is the maximum number of stacks.

Parameters
[in]pSlotThe DescriptorSlot of the texture to add to the stack.
