Detailed Description

The class for handling busy mutex objects.

Details
The busy mutex is a synchronization object for implementing mutually exclusive control among threads.

The nn::os::BusyMutex class is a utility class that wraps calls to synchronization control functions related to nn::os::BusyMutexType objects. The member functions provided by this class are implemented as functions that call functions such as nn::os::LockBusyMutex() that take the nn::os::BusyMutexType type for their arguments.

This class fulfills the BasicLockable and Lockable requirements in the C++ standard, so it can be used as a type argument for class templates such as std::lock_guard and std::unique_lock. As a result, multiple member functions exist that perform the same operations, such as Lock() and lock().
