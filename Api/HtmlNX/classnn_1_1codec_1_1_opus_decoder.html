Detailed Description

The OPUS decoder class.

Details
Decodes mono and stereo sound Opus data.

Examples
CodecOpusCodecProcessingTime/CodecOpusCodecProcessingTime.cpp, CodecOpusDecoder/CodecOpusDecoder.cpp, and CodecOpusEncoder/CodecOpusEncoder.cpp.
