Detailed Description

template<size_t KeySize, typename HashFunction>
class nn::crypto::RsaPkcs1Signer< KeySize, HashFunction >

Class that generates RSA-PKCS1 signatures.

Template Parameters
KeySizeThe size of the key to use in bytes.
HashFunctionThe implementation class for the hash function to use.

Details
RSA-PKCS1 signatures are a signature method that uses the RSA signature algorithm stipulated with PKCS #1.

This class is a template class that takes the key size and hash function implementation class as template parameters. Use ,Rsa2048Pkcs1Sha256Signer or a similar class in your actual code.

The key size can be specified in byte (8-bit) units up to a maximum of 512 bytes (4096 bits).

The implementation of the hash function class specified in the template must satisfy the following conditions. For more information such as the parameters of each function, see the reference for Sha1Generator or a similar class.

  1. Has a BlockSize constant that defines the size of the block to process.
  2. Has a HashSize constant that defines the output size of the hash value.
  3. -# The Asn1ObjectIdentifier object ID has a DER-encrypted binary array.
  4. -# The Asn1ObjectIdentifierSize object ID has a constant defined by the size of the DER-encrypted binary array.
  5. Has an Initialize function that begins processing.
  6. Has an Update function that updates the hash value on input.
  7. Has a GetHash function that gets the final hash value.

This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state.
  • Ready.
  • Complete.

The modulus and exponent are always treated as unsigned numbers in big-endian format.
