Detailed Description

The structure representing the input states for the NpadStyleJoyRight play style modeled on holding an Npad Joy-Con (R) controller.

Details
Provides input states for the digital buttons (A, B, X, Y, R, ZR, Joy-Con (R) SL, Joy-Con (R) SR, and + Button) and the Right Stick, which comprise the input dealt with for Joy-Con (R) operations. Other input states are treated as noninputs, where the digital button is not depressed and the control stick is in a neutral position. You can get the controller connection state using either attributes::IsConnected or attributes::IsRightConnected.
