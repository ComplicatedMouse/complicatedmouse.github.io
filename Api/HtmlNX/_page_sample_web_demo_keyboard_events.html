Sample offline HTML to read controller operations using keyboard events.

Overview

Offline HTML sample to read controller operations using keyboard events. This sample demonstrates the following features.

  • Reading controller operations using keyboard events
  • Using the Javascript extension to play sound effects

File Structure

This sample offline HTML is located in Samples/Sources/Applications/WebDemo/html_document/sample/keyboard_events.htdocs.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

When you operate the controller, the data read using keyboard events appears on the screen.

Input Operation
B Button, X Button Exits the offline web applet and returns to the menu, via the application.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and display the offline HTML from this sample.

Reading Controller Operations Using Keyboard Events

Controller operations are read as KeyboardEvent object keyCode properties, and appear on the screen.

Using the Javascript Extension to Play Sound Effects

Uses the window.nx.playSystemSe() Javascript extension to play system sound effects when buttons are pressed.
