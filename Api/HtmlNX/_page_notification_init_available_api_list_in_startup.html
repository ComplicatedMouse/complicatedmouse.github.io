A list of APIs that can be used from nninitStartup().

Description

List of APIs that can be used within nninitStartup(). The functions, classes, and macros provided by the following libraries can be used.

  • nn::Result class
  • nn::TimeSpan class
  • nn::util library
    • NN_LOG() macro
    • NN_ASSERT() family of macros
    • NN_ABORT() family of macros
  • nn::init library
    • All APIs and classes can be used.
  • nn::os library
    • All APIs and classes can be used.
  • nn::oe library
    • You can use nn::oe::SetCpuBoostMode().
