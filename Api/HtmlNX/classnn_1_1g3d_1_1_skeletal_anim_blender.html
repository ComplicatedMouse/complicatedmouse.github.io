◆ Blend() [2/2]

void nn::g3d::SkeletalAnimBlender::Blend ( SkeletalAnimObj *  pSrcAnimObj,
SkeletalAnimObj *  pReferenceAnimObj,
float  weight 
)

The blend calculation uses the difference between two animations.

Parameters
[in]pSrcAnimObjPointer to the skeletal animation to which to apply subtraction.
[in]pReferenceAnimObjPointer to the skeletal animation that performs subtraction.
[in]weightBlend weight.

Details
This function requires that BeginBlend() or SetBlendMode() set BlendMode_Add.

Call the SkeletalAnimObj::Calculate() and blend its results. If the calculations have already finished, SkeletalAnimObj::Calculate() is skipped. Bones for which the SKIP_APPLY flag has been specified are not blended in pSrcAnimObj.

pReferenceAnimObj is subtracted from pSrcAnimObj and the calculated difference is blended with the value multiplied by the cumulative value. The blend calculation of this function depends on the cumulative blend calculations before the function call. Note that the ultimate blend result varies depending on the timing of the function call.

◆ ApplyTo()

void nn::g3d::SkeletalAnimBlender::ApplyTo ( SkeletonObj *  pSkeletonObj) const

Applies the blend result to the target.

Parameters
[in]pSkeletonObjPointer to the skeleton to apply the blend results to.

Details
This function is not thread-safe because it performs cache processing internally. Do not call the same instance from multiple threads. It does not apply to bones with a cumulative weight of 0. Bones that do not have a cumulative weight of 1 are normalized before the blending result is applied.

static void nn::g3d::SkeletalAnimBlender::CalculateBoneAnimDiff ( BoneAnimDiffResult *  pOutBoneAnimDiffResultArray,

)
static

Calculates the difference between two animations.

Parameters
[in,out]pOutBoneAnimDiffResultArrayPointer to the location for writing the animation differences.
[in]resultCountThe number of elements in pOutBoneAnimDiffResultArray.
[in]pSrcAnimObjPointer to the skeletal animation to which to apply subtraction.
[in]pReferenceAnimObjPointer to the skeletal animation that performs subtraction.
Precondition
  • pOutBoneAnimDiffResultArray != NULL.
  • resultCount >= pSrcAnimObj->GetAnimCount().
  • pSrcAnimObj != NULL.
  • pReferenceAnimObj != NULL.

Details

  • pReferenceAnimObj is subtracted from pSrcAnimObj and the calculated difference is blended with the value multiplied by the cumulative value.
  • Call the SkeletalAnimObj::Calculate() and blend its results.
  • If the calculations have already finished, SkeletalAnimObj::Calculate() is skipped.
  • Sets an invalid state for bones for which the SKIP_APPLY flag is specified for pSrcAnimObj.
  • Writes values to the elements of pOutBoneAnimDiffResultArray for which the index matches the bone index.
