Detailed Description

Class for handling friends.

Details
You can reference the friend information specific to a particular user account. The content that can be referenced for each user account differs even when the same user is a friend in multiple user accounts.

Examples
FriendsPresence/FriendsPresence.cpp, and FriendsSimple/FriendsSimple.cpp.

nn::Result nn::friends::Friend::GetProfileImage ( size_t *  outSize,

Gets user profile images cached in console storage.

Parameters
[out]outSizeSize of the obtained profile image.
[out]bufferStorage destination buffer for the profile image.
[in]sizeSize of the storage destination buffer for the profile image.
Returns
Returns the result of processing.


Return Values:
ResultProfileImageCacheNotFound There is no cache for the friend's profile image. Display the default user icon or separately get a URL to download the image.
Precondition
  • outSize != nullptr.
  • buffer != nullptr.
  • size > 0.
  • The specified user account is in an open state during initialization.

Details
The profile image is saved as a 256 x 256 JPEG image. The maximum file size is 100 KB.

To get a profile image that is a different size, get the URL using GetProfileList and download it.

◆ IsNewly()

bool nn::friends::Friend::IsNewly ( ) const

Determines whether the friend relationship was newly established.

Returns
Returns whether the friend is a new one.

Details
Newly established friends will be marked as such on My Page. This flag becomes false when the user views a marked friend in My Page.

This mark is only intended to be displayed in My Page. It must not appear within an application.
