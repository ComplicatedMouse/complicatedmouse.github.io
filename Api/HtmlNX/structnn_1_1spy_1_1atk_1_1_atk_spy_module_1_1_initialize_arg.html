◆ audioPerformanceMetricsCount

int nn::spy::atk::AtkSpyModule::InitializeArg::audioPerformanceMetricsCount

The number of entries in the buffer for collecting the nn::audio performance metrics.

(The default is 5.)

The nn::atk sound thread gets performance metrics once per audio frame (= 5ms) and stores the information in a buffer.

The minimum value for audioPerformanceMetricsCount is calculated as follows, where T is the interval in milliseconds for calls to the AtkSpyModule::PushPerformanceMetrics() function.

(Minimum value of <tt><var>audioPerformanceMetricsCount</var></tt>) = max( <tt>AudioPerformanceMetricsCountMin</tt>, (T / 5) + 1 );
