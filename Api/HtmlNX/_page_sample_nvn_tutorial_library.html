The NVN Tutorial Library contains common code shared by the NVN Tutorials.

Overview

The NVN Tutorial Library contains common code shared by the NVN Tutorials. The source files that make up the library are listed and linked below along with a short description of what it implements. More implementation details can be found in the inline comments in each file. The project file for the library is included each of the tutorials' visual studio solution files and is built before the tutorial runs.

File Structure

AlignedAllocator.h, AlignedAllocator.cpp This file defines a simple aligned allocator that takes care of keeping track of the original allocation while returning the aligned pointer.
AssetFileDataHolder.h, AssetFileDataHolder.cpp This files defines some structures that hold the data provided by an asset file created by Tutorial02 as well as a class that holds lists of each section type that made up an single asset file.
AssetFileLoadingHelper.h, AssetFileLoadingHelper.cpp This file defines a helper class used to load in asset files generated by Tutorial02.
Camera.h, Camera.cpp This class keeps track of the main rendering camera's position and orientation.
DebugTextData.h This file contains the raw texture data for the debug text renderer font.
DebugTextRenderer.h, DebugTextRenderer.cpp This file defines a singleton class that renders text to the screen.
DeferredDirectionalLightPass.h, DeferredDirectionalLightPass.cpp This class handles the directional light pass for the deferred renderer.
DeferredPointLightObject.h, DeferredPointLightObject.cpp This class holds the information for the deferred point light and it's shadow map.
FrameBufferManager.h, FrameBufferManager.cpp This file defines 2 classes:
GraphicsObject.h, GraphicsObject.cpp This files defines a simple base class that handles render state setting for a given loaded model.
GraphicsObjectCubes.h, GraphicsObjectCube.cpp This file defines a class that derives from the base GraphicsObject class for drawing the graphicsObjectCube.out asset file.
GraphicsObjectSpecializedCube.h, GraphicsObjectSpecializedCube.cpp This file defines a class that derives from the base GraphicsObject class for drawing the graphicsObjectSpecializedCube.out asset file.
InputManager.h, InputManager.cpp This is the base class for the platform specific input managers.
InputManager.Nx.h, InputManager.Nx.cpp This class handles initializing the input system for NX and keeps track of the current and previous frames' user input state.
InputManager.Windows.h, InputManager.Windows.cpp This class handles initializing the input system for Windows and keeps track of the current and previous frames' user input state.
LoadScene.h, LoadScene.cpp This file defines some helper functions for loading the various objects, textures, shaders, lights, and global data that makes up the scene.
ManagedCommandBuffer.h, ManagedCommandBuffer.cpp This file defines a class that manages memory for command buffers by using multiple chunks of of GPU memory.
ManagedUniformBuffer.h, ManagedUniformBuffer.cpp The ManagedUniformBuffer class is used to create uniform buffers from the double buffered memory used by the UniformBufferManager.
MemoryPool.h, MemoryPool.cpp This file defines a wrapper around the NVNmemoryPool object that keeps track of the memory allocated for the pool, the size of the pool, and the offset into the pool at which the next memory write could take place.
Model.h This file defines a model structure used for reading/writing model data.
MultiThreadedAssetFileLoadingHelper.h, MultiThreadedAssetFileLoadingHelper.cpp This file defines a helper class to load asset files in parallel on multiple threads.
OutputFileHeaders.h This file defines the data structures that make up the the overall structure of the output asset file.
PBRModelBinaryFileHeaders.h This file defines the structure of the converted binary model files.
PBRModelData.h This file defines some helper structs that organize data used for rendering.
PBRObject.h, PBRObject.cpp This file defines some classes that handle the model and material data, uniform updating, and drawing for all of the objects in the scene.
RenderThreadPool.h, RenderThreadPool.cpp This file defines a class that handles running multiple threads in parallel that create command buffers to render objects from a work queue.
ScreenSpaceQuadDepthPass.h, ScreenSpaceQuadDepthPass.cpp This class renders a depth texture to a full screen quad and linearizes those values based on the near/far planes so that it is easier to visualize.
ScreenSpaceQuadPass.h, ScreenSpaceQuadPass.cpp This class renders a texture to a full screen quad.
ShaderHelper.h, ShaderHelper.cpp This file defines a helper function for getting the attribute location function pointer for a given shader program.
ShaderTypeEnum.h, ShaderTypeEnum.cpp This file defines some helper functions that give a more generic interface to the helper shader header files create by the offline asset file generating tool.
SkyBox.h, SkyBox.cpp This class handles render the sky box for the scene.
StatTracker.h, StatTracker.cpp This file defines a helper class that sets up command buffers to report GPU statistic counters and reset those counters.
TextureIDManager.h, TextureIDManager.cpp This file defines a class that wraps the texture and sampler registering process.
TextureUtil.h, TextureUtil.cpp This file defines some helper functions for creating textures.
TutorialBaseClass.h, TutorialBaseClass.Windows.cpp, TutorialBaseClass.Nx.cpp This class is shared among all tutorials in order to abstract away per-operating system details related to NVN initialization and operating system integration.
TutorialUtil.h, TutorialUtil.cpp This file defines the interface for utility functions used in the tutorials.
UniformBuffer.h, UniformBuffer.cpp This file defines a helper class for setting up uniform blocks.
UniformBufferManager.h, UniformBufferManager.cpp This file defines a class that manages memory for uniform buffers by using multiple chunks of of GPU memory.
