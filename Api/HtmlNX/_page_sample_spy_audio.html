A sample program using the nn::spy and nn::spy::audio libraries.

Overview

This sample program uses the nn::spy and nn::spy::audio libraries to monitor the state of nn::audio.

File Structure

This sample program is located in Samples/Sources/Applications/SpyAudio/ .

Required Environment

You must be able to use audio output.

How to Use

Play sound effects using input from the keyboard or DebugPad, and use the Spy tool to check the nn::audio processing load and the output waveforms. Operation instructions are displayed in the console after you start the sample. Communication begins after configuring the connection target from the spy tool ribbon and activating the connection.

Important Information

You cannot communicate with Spy.exe using the Release build. Spy is a feature for application-development purposes and is disabled in the Release build to prevent accidental inclusion in retail products. If you want to use Spy in the Release build, you must edit spy_Config.h to define the NN_ENABLE_SPY macro.

Execution Procedure

  1. Build the sample program in either Debug or Develop mode and then run it.
  2. The program will start as long as NintendoTargetManager is not running.
  3. From the PC, start the spy tool, and then connect to the sample program.

399  // Update the debug pad.
400  nn::hid::DebugPadButtonSet UpdateDebugPad()
401  {
402  g_PadLastButtonSet = g_PadButtonSet;
403 
404  {
405  nn::hid::DebugPadState state;

416  nn::hid::NpadFullKeyState state;
417  nn::hid::GetNpadState(&state, id);
418  g_PadButtonSet |= MapNpadButtonSet(state.buttons);

420 
421  if (styles.Test<nn::hid::NpadStyleHandheld>())

424  nn::hid::GetNpadState(&state, id);

473  void Mainloop()
474  {
475  PrintUsage();
476 
477  while (NN_STATIC_CONDITION(true))
478  {
479  WaitAudioRendererEvent();
480 
481  // Record the application frame start time.
482  g_SpyController.SetCurrentApplicationFrame(g_AppFrameCount++);
483 
484  nn::hid::DebugPadButtonSet triggered = UpdateDebugPad();
485 
486  // Play sound effects.
487  if (triggered.Test<nn::hid::DebugPadButton::A>())
488  {
489  PrintMessage("PlaySe(0) ...");
490  bool result = PlaySe(0);
491  PrintResult(result);
492  }
493  if (triggered.Test<nn::hid::DebugPadButton::B>())
494  {
495  PrintMessage("PlaySe(1) ...");
496  bool result = PlaySe(1);
497  PrintResult(result);
498  }
499  if (triggered.Test<nn::hid::DebugPadButton::X>())
500  {
501  PrintMessage("PlaySe(2) ...");
502  bool result = PlaySe(2);
503  PrintResult(result);
504  }
505  if (triggered.Test<nn::hid::DebugPadButton::Y>())
506  {
507  PrintMessage("PlaySe(3) ...");
508  bool result = PlaySe(3);
509  PrintResult(result);
510  }
511 
512  // Enable or disable reverb.
513  if (triggered.Test<nn::hid::DebugPadButton::L>())
514  {
515  PrintMessageLine(IsEnableReverb() ? "EnableReverb(false)" : "EnableReverb(true)");
516  EnableReverb(!IsEnableReverb());
517  }
518 
519  if (triggered.Test<nn::hid::DebugPadButton::Start>())
520  {
521  PrintMessageLine("Exit");
522  return;
523  }
524 
525  RequestUpdateAudioRenderer();
526  nn::os::Tick tick = nn::os::GetSystemTick();
527 
528  PushAudioPerformanceMetrics();
529  PushWaveform(tick);
530  PushSpyDebugInfo();
531  }
532  }

Sample Program Description

This sample program has the following flow.

  • Initialize htcs and audio.
  • InitializeSpy()
    • Initialize nn::spy::SpyController.
  • InitializeSpyAudio()
    • Initialize nn::spy::audio::AudioSpyModule.
    • Register nn::spy::audio::AudioSpyModule in nn::spy::SpyController.
  • StartSpy()
    • Start communication with the Spy tool.
  • Main loop.
    • Send the application frame using nn::spy::SpyController::SetCurrentApplicationFrame(), as necessary.
    • Send audio performance information using nn::spy::audio::AudioSpyModule::PushPerformanceMetrics().
    • Send the output waveform using nn::spy::audio::AudioSpyModule::PushWaveform().
    • Send the debug data, in terms of the amount of data buffer used for the Spy tool, using nn::spy::DebugModule::PushDataBufferUsage().
  • StopSpy()
    • End communication with the Spy tool.
  • FinalizeSpyAudio()
    • Unregister nn::spy::audio::AudioSpyModule from nn::spy::SpyController.
    • Finalize nn::spy::audio::AudioSpyModule.
  • FinalizeSpy()
    • Finalize nn::spy::atk::AtkSpyModule.
  • Finalize htcs and audio.
