The text.

If NULL, the string specified in Layout Editor is set. To empty the text box, specify the null character.

◆ bufferLength

int32_t nn::ui2d::TextSearcher::TextInfo::bufferLength

The length of the buffer to allocate for storing text.

Specify as a number of characters, not as a number of bytes. If the specified value is 0, the length is set to value specified for textLength. If you want to use the length that was specified with the Restrict String Length option in Layout Editor, copy forceAssignedTextLength.

◆ forceAssignedTextLength

int32_t nn::ui2d::TextSearcher::TextInfo::forceAssignedTextLength

When the Restrict String Length option is selected in LayoutEditor,.

this member variable is passed to SearchText with the specified number of characters. If the option is not selected, this member is set to -1. Not used for output.
