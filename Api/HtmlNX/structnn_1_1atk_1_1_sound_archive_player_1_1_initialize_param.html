Detailed Description

Structure used to initialize the sound archive player.

Details
Use this structure with the version of the Initialize function that takes this structure as an argument.

Calling the Initialize function using this structure enables file read cache during stream sound playback and provides a way to control the frequency of access to the file system.

For more information about userParamSizePerSound, see Initialize.

◆ pSetupBuffer

void* nn::atk::SoundArchivePlayer::InitializeParam::pSetupBuffer

The buffer used by the instance and player heap generated in the sound archive player.

It must pass a memory region of the size obtained by GetRequiredMemSize. It must be aligned to a BufferAlignSize byte boundary.

You must pass a memory region of the size obtained by GetRequiredMemSize, which takes InitializeParam as a parameter, when using the ability to specify a buffer for stream sound instances. At this point, the maximum amount of memory that can be saved is nn::audio::MemoryPoolType::SizeGranularity. For more information, see GetRequiredStreamInstanceSize.

Examples
AtkAddonSoundArchive/AtkAddonSoundArchive.cpp, and AtkStreamSound/AtkStreamSound.cpp.

◆ pStreamBuffer

void* nn::atk::SoundArchivePlayer::InitializeParam::pStreamBuffer

The buffer used for streaming sound playback.

It must pass a size obtained by GetRequiredStreamBufferSize or pass a memory region up to StreamBufferTimesMax times the size. Specify NULL if the size obtained by GetRequiredStreamBufferSize is 0. It must be aligned to an nn::audio::BufferAlignSize byte boundary. This buffer must be managed by the memory pool.

Examples
AtkAddonSoundArchive/AtkAddonSoundArchive.cpp, and AtkStreamSound/AtkStreamSound.cpp.

◆ enablePreparingStreamInstanceBufferFromSetupBuffer

bool nn::atk::SoundArchivePlayer::InitializeParam::enablePreparingStreamInstanceBufferFromSetupBuffer

The flag that indicates whether the library will automatically configure the buffer for stream sound instances using a part of pSetupBuffer.

The default value is true. Set the flag to false to save memory used and the time for initialization and finalization when you configure the buffer for instances. For more information, see GetRequiredStreamInstanceSize.

◆ pStreamInstanceBuffer

The buffer for stream sound instances.

You must pass the size obtained by GetRequiredStreamInstanceSize. It must be aligned to an nn::audio::BufferAlignSize byte boundary. This buffer must be managed by the memory pool. For more information about using the buffer, see GetRequiredStreamInstanceSize.
