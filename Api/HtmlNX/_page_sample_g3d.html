3D Graphics Library Samples

Describes the 3D graphics (G3D) library samples.

Introduction

Describes the 3D graphics (g3d) library samples. The g3d features reside in the nn::g3d namespace.

Samples

The nn::g3d sample code is found in Samples/Sources/Applications/G3dDemo/. When this code is executed, the launcher menu will be displayed. If you are using the development hardware, you can select and execute the demo by using the Joy-Con, DebugPad, or touch screen. If you are using a PC, select and execute the demo through keyboard operations.

Input Operation
Up and down on the +Control Pad or the up and down arrow keys on the keyboard Selects the demo.
A Button or A key Executes the demo.

To select and execute the demo, you can use the touch screen when using the development hardware or the mouse when using a PC.

Sample code that uses nn::g3d::mii is located in the directories starting with the prefix MiiG3d under Samples/Sources/Applications/. The following table lists the samples.

Page Content
Simple G3dDemo Sample This sample program shows displaying a simple 3D model. It is an example of model rendering code that uses g3d.
G3dDemo NnsG3dSimple This documentation describes the NnsG3dSimple sample program. Use nns::g3d to display the model.
G3dDemo Editing Sample This documentation describes the Edit sample program. This sample program shows how to use the viewer library for real-time editing with 3DEditor.
G3dDemo SkeletalAnimation Describes the SkeletalAnimation sample program.
G3dDemo ShapeAnimation Sample This documentation describes the ShapeAnimation sample program. Uses shape animation to display models.
G3dDemo Outline Sample This documentation describes the Outline sample program. Use the geometry shader to render the outline of a model.
G3dDemo Parallel Sample Description of the Parallel sample program. This program uses parallel processing to update models.
G3dDemo Tessellation Sample Describes the Tessellation sample program. The sample program uses tessellation to change the number of mesh divisions corresponding to the distance from the camera.
G3dDemo Town Sample This documentation describes the Town sample program. This sample program is close in content to a real game.
MiiG3dSimple Sample This sample program shows the feature to assign Mii models to nn::g3d::ResModel.
MiiG3dHeadwear Sample This sample program shows putting a hat on a Mii character in g3d.

Note

It can take from between five to ten minutes to convert the resources. Note that during the conversion process, no log is output within Visual Studio. A log will be output, however, after conversion completes.
