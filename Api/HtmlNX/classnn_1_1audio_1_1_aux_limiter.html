Detailed Description

The class for handling the Limiter effect utility applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform Limiter effect processing. It specifies the sample data obtained by the nn::audio::ReadAuxSendBuffer() function as the input for this effect, and outputs the result of applying the Limiter effect, which limits the sample values to a certain range.

The nn::audio::AuxLimiter class is a utility class that wraps calls to functions that use nn::audio::AuxLimiterType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessAuxLimiter() that take the nn::audio::AuxLimiterType type for their arguments.

Examples
AudioEffectAuxUtility/AudioEffectAuxUtility.cpp.
