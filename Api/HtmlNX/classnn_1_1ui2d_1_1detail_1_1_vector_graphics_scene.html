void nn::ui2d::detail::VectorGraphicsScene::Finalize ( )

Finalizes the instance.

Details
Because adding mask support causes dynamic rendering textures to be created internally, a device pointer is now required during termination processing. If a Build() function that specifies layout instance is called, use the Finalize() function to pass the device pointer.
