Detailed Description

A BiquadFilter effect.

Details
BiquadFilter provides a generic biquad filter effect.

Directly altering member variables is prohibited. Use the main structure through the following functions.

  • AddBiquadFilter(AudioRendererConfig* pOutConfig, BiquadFilterType* pBiquadFilter, SubMixType* pSubMix)
  • AddBiquadFilter(AudioRendererConfig* pOutConfig, BiquadFilterType* pBiquadFilter, FinalMixType* pFinalMix)
  • RemoveBiquadFilter(AudioRendererConfig* pOutConfig, BiquadFilterType* pBiquadFilter, SubMixType* pSubMix)
  • RemoveBiquadFilter(AudioRendererConfig* pOutConfig, BiquadFilterType* pBiquadFilter, FinalMixType* pFinalMix)
  • GetBiquadFilterInputOutput(const BiquadFilterType* pBiquadFilter, int8_t* outInput, int8_t* outOutput, int* pOutCount, int count)
  • SetBiquadFilterInputOutput(BiquadFilterType* pBiquadFilter, const int8_t* input, const int8_t* output, int count)
  • IsBiquadFilterEnabled(const BiquadFilterType* pBiquadFilter)
  • SetBiquadFilterParameter(BiquadFilterType* pBiquadFilter, const BiquadFilterParameter& parameter)
  • BiquadFilterParameter GetBiquadFilterParameter(const BiquadFilterType* pBiquadFilter)
  • SetBiquadFilterEnabled(BiquadFilterType* pBiquadFilter, bool enable)
  • IsBiquadFilterRemovable(const BiquadFilterType* pBiquadFilter)

To use these functions, include nn/audio.h.
