Detailed Description

Class that expresses four-dimensional vectors using a single precision floating number type.

Details
Type that provides vector and matrix calculations.

Utility class that inherits nn::util::Vector4fType and adds a member function.

Quaternions are expressed with another type. See nn::util::Quaternion.

We assume that calculations that use this type will switch to an implementation that uses the SIMD unit in environments that can use the SIMD unit. A generic environment is provided for environments in which the SIMD unit cannot be used.

Attention
This type is optimized for calculations. As a result, it can differ based on memory layout, size, and environment. For this reason, direct access to internal elements is prohibited. To access elements, use dedicated macros and functions.
Because calculations within the SIMD are to be optimized, this type must be aligned to 16 bytes. When allocating this type in memory, make sure that it meets the required alignment.
Examples
NvnSimple07DynamicResolution/DynamicResolution.cpp, NvnSimple07DynamicResolution/DynamicResolution.h, NvnTutorial/DeferredDirectionalLightPass.cpp, NvnTutorial/DeferredPointLightObject.cpp, NvnTutorial/FrustumCullingPass.cpp, NvnTutorial/LoadScene.cpp, NvnTutorial12DeferredRenderer/NvnTutorial12.cpp, NvnTutorial13ScreenSpaceAO/NvnTutorial13.cpp, NvnTutorial14FrustumCulling/NvnTutorial14.cpp, nvntutorial/BoundingVolume.h, nvntutorial/DeferredDirectionalLightPass.h, nvntutorial/DeferredPointLightObject.h, nvntutorial/FrustumCullingPass.h, and nvntutorial/ViewFrustum.h.
