explicit

Constructor.

Parameters
[in]pHeapThe VFX heap instance to use in the viewer library.
[in]pConnectionHeapThe VFX heap instance to use for communications. Passes from the communication thread because an in-use thread-safe heap is required.
[in]pSystemThe VFX system instance.
[in]noUseInternalThreadSpecify true to use an external thread.
[in]priorityThe priority of the thread. The default argument is specified as nn::os::LowestThreadPriority, so it is necessary to increase the priority, etc., if there is difficulty in communicating with EffectMaker. For more information, see Embedding the Viewer Runtime.

Details
Define the NN_ENABLE_HTC or NN_DISABLE_HTC compilation switch in the application. These options switch between connecting and disconnecting communication.
