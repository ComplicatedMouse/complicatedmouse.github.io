enum nvn::CompareFunc::Enum

Enumerated values for the type CompareFunc.

Enumerator
NEVER 

Comparison always fails.

LESS 

Comparison passes if the reference value is less than the texture value.

EQUAL 

Comparison passes if the reference value is equal to the texture value.

LEQUAL 

Comparison passes if the reference value is less than or equal to the texture value.

GREATER 

Comparison passes if the reference value is greater than the texture value.

NOTEQUAL 

Comparison passes if the reference value is not equal to texture value.

GEQUAL 

Comparison passes if the reference value is greater than or equal to the texture value.

ALWAYS 

Comparison always passes.
