Detailed Description

Abstract connection base class.

Details
Abstract base class for connections for connecting to and communicating with a server.

Note
Classes that inherit this class must be implemented to run communication processes in nonblocking mode.

)
pure virtual

Sets a function to use to cancel the connection process.

Parameters
[in]pFunctionThe function.
[in]pParamUser parameters for the function.

Details
The cancellation function set using this function is called periodically starting when the Connect() function is called until the Connect() function is complete. If the cancellation function returns true, the connection process is canceled.

Return Values:

>=1 The size of the data that was referenced.

0 The server connection was closed.

-1 An error occurred. Get the error details using the GetLastError() function.

Precondition
  • pBuffer != nullptr.
  • size > 0.

Details
Unlike Receive(), this function does not remove receive data from the receive queue. If you call this function again immediately afterward, it returns the same data.
