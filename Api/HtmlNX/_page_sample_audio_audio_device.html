This sample program describes the settings unique to audio output devices and the settings for application-specific downmixing.

Overview

The purpose of this sample program is to describe how to use the following features. #1 Getting the device to which audio is output, the number of output channels, and the time when output switches to another device. #2 Setting the volume for each audio output device. #3 Setting the application-specific downmixing feature.

File Structure

This sample program is located in Samples/Sources/Applications/AudioDevice/ .

Required Environment

You must be able to use audio output.

How to Use

After this sample program runs, BGM starts playing. You can also use the buttons to play sound effects.

Input Operation
A Button Play a sound effect.
[B] Play a sound effect.
[X] Play a sound effect.
[Y] Play a sound effect.
[L] Play a sound effect.
[Left/Right] Select the device to set the volume for.
[Up/Down] Adjust the volume of the selected device.
[Select/Minus] View the sample description.
START or SPACEBAR Exits the program.

Important Information

This sample assumes that you already know how to use basic audio renderers. For a detailed example of using the audio renderer, see Samples/Sources/Applications/AudioRenderer/ .

Execution Procedure

Build the sample program and then run it.

Description

Descriptions of the following features are presented in the sample code.

#1 Getting the device to which audio is output, the number of output channels, and the time when output switches to another device. Use the nn::audio::GetActiveAudioOutputDeviceName() function to get the name of the device to which audio is currently being output. Use nn::audio::GetActiveAudioDeviceChannelCountForOutput() to get the current number of output channels. The output device and number of output channels changes dynamically depending on the operating state of the platform, but you can get notification when the device is switched via the system event obtained by the nn::audio::AcquireAudioDeviceNotificationForOutput() function. Comments relating to the relevant code are provided inside the code as comments #1-1 and #1-2.

#2 Setting the volume for each audio output device. You can set an output volume separately for each audio output device. Use the nn::audio::SetAudioDeviceOutputVolume() function to set the volume. Comments relating to the relevant code are provided inside the code as comments #2-1 and #2-2.

#3 Setting the application-specific downmixing feature. There are two downmix processes: system default downmixing, and application-specific downmixing. The latter process can be used only in the audio renderer, and a description is provided in this sample program. For a comprehensive description of both downmix processes, see the Downmix Processing section of Audio Library. To use application-specific downmixing, first set the parameter using nn::audio::SetDownMixParameter() and then enable it using nn::audio::SetDownMixParameterEnabled(). Comments relating to the relevant code are provided inside the code as comments #3-1 and #3-2.
