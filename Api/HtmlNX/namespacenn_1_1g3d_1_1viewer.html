Enumeration Type Documentation

◆ CallbackType

enum nn::g3d::viewer::CallbackType

Type of server callback.

Enumerator
CallbackType_ModelFileLoaded 

Invoked when 3DEditor has requested the loading of model resources.

The game application must copy and accept, manage, and render resource files. You can register an edit target by calling nw::g3d::viewer::ViewerServer::QueueAttachModelCommand().

CallbackType_ModelFileUnloaded 

Invoked when 3DEditor has requested the unloading of model resources.

A resource file can be discarded by a game application at the time it is no longer necessary.

CallbackType_ModelAttached 

Called after the model is attached.

CallbackType_ModelDetached 

Called after the model is detached.

CallbackType_ShaderAssignUpdated 

Called when an edit is performed along with a change in the model resource structure, such as changing the material shader binding with 3DEditor or adding or deleting the sampler.

Because the viewer library dynamically changes model resources, if it is necessary to bind to model resources and have data, this function can perform construction and destruction.

CallbackType_MaterialUpdated 

Called after a material update occurs for a model instance.

Denotes the material update process for the target model. The ResMaterial available from the nn::g3d::MaterialObj that maintains the target model is replaced with post-update resources when this callback is invoked.

CallbackType_SamplerParamUpdated 

Called when sampler parameters were modified in 3DEditor.

Modifying sampler parameters enables you to recreate the display list and re-register the sampler to the descriptor pool.

CallbackType_RenderInfoUpdated 

Called when rendering information is edited with 3DEditor.

You can use this function to reflect from the targeted model, material, or rendering information to the game side.

CallbackType_SendRenderInfoRequested 

Called when there is a request to send render information definitions from 3DEditor to the game application.

This callback is used to send the render information definitions for the specified material needed by 3DEditor.

CallbackType_BoneBindUpdated 

Called when editing binding information using the 3DEditor model preview feature.

Binds the target model to the bone specified from the model that is the parent.

CallbackType_ShapeUpdated 

Called when a shape was updated within the viewer library.

Shapes are updated when models or shapes flash after selecting them. This happens when you need to restructure models within the viewer library or when switching display of LOD models in 3DEditor.

CallbackType_ModelLayoutUpdated 

Called when editing layout information using the 3DEditor model preview feature.

This function can reflect from the targeted model and layout information to the game side.

CallbackType_SendModelLayoutRequested 

Called when the layout information required by 3DEditor is sent on the game application side.

Reflects the specified model position information in 3DEditor.

CallbackType_TargetSelected 

Called when models, materials, bones, or shapes are selected in 3DEditor.

Performs viewer processing for the selection.

CallbackType_ModelUserScriptExecuted 

Called when a user script is run on a model in 3DEditor.

CallbackType_ModelTextureBindingsUpdated 

Called when the model's texture binding has been updated.

CallbackType_TextureFileLoaded 

This function is called when a texture file is loaded.

CallbackType_TextureFileUnloaded 

This function is called when a texture file is unloaded.

CallbackType_SceneAnimBound 

Called when setting scene animation in the bind target with 3DEditor.

You can use this function to associate the game application with the targeted camera animation, light animation, and fog animation.

CallbackType_SceneAnimUnbound 

Called when removing a scene animation from the bind target with 3DEditor.

You can use this function to disassociate the game application from the targeted camera animation, light animation, and fog animation. We recommend implementing this process as a pair of functions with SceneAnimBound.

CallbackType_ApplySceneAnimRequested 

Applies the results of the calculation of the scene animation that is the bind target with SceneAnimBound.

This process is executed only for scene animations that are the bind target with SceneAnimBound.

CallbackType_ShaderAttached 

Called immediately after a shader is attached with 3DEditor.

Enables, for example, switching to an object that a render shader has been attached to.

CallbackType_ShaderDetached 

Called immediately after a shader is detached with 3DEditor.

Enables, for example, switching to an ordinary object from one that a render shader has been attached to.

CallbackType_ShaderProgramUpdated 

This process runs following a change to the shader program after delayed compilation when a shader is being attached.

CallbackType_ShadingModelUpdated 

The use of internal features is prohibited. The process executed when the shading model is edited by 3DEditor when attaching a shader.

CallbackType_CameraUpdated 

The use of internal features is prohibited. This function is executed when the client requests updating the camera transform or other elements.

CallbackType_BoneTransformUpdated 

The use of internal features is prohibited. This function is executed when the client requests updating the bone transform.
