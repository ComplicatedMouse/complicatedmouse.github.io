Detailed Description

The class for storing drawing resources that are shared by multiple Layout objects.

Details
Set to DrawInfo using the DrawInfo::SetGraphicsResource() function.

Examples
Ui2dDemo/Main.cpp, and Ui2dViewerSimple/main.cpp.

Sets and configures the instance of RectDrawer created in the application.

Details
For the memory pool, specify a pool that has the properties nn::gfx::MemoryPoolProperty_CpuUncached and nn::gfx::MemoryPoolProperty_GpuCached. Use CalculateMemoryPoolSize() to get the size and CalculateMemoryPoolAlignment() to get the offset alignment. Initialize and finalize RectDrawer in the application.

The value passed to lodBias is set by nn::gfx::SamplerInfo::SetLodBias. Use this function to use the mipmap to adjust LodBias.

Parameters
[in]pDeviceThe gfx device.
[in]charMaxThe maximum number of characters that can be drawn in the layout.
[in]pExternalMemoryPoolThe memory pool.
[in]externalMemoryPoolOffsetThe offset to the memory pool region to use.
[in]externalMemoryPoolSizeThe memory pool size.
[in]pFontDrawerInstance of RectDrawer created in the application.
[in]lodBiasThe offset value applied to the calculated mipmap level.
