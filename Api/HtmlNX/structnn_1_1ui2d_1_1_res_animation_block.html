Detailed Description

The data block that defines an animation.

Details
DataBlockKindPaneAnimInfo is specified for blockHeader[kind].

The following data structures are in this structure.

  • uint32_t fileNameOffsets[fileCount]
  • uint8_t namePool[]
  • uint32_t animContOffsets[paneCount]
  • ResAnimationContent animCont0
  • :

The fileNameOffsets array stores the offsets to the texture resource names that are referenced from fileNameOffsets.

The structure stores offsets to the animation content (from the start of the data block) in the animContOffsets array.
