Detailed Description

The structure representing the POSIX standard time.

Details
Maintains the time as the elapsed seconds since 1970-01-01 00:00:00 in Coordinated Universal Time (UTC).

Addition and subtraction with nn::TimeSpan is possible, but the result is rounded down to the nearest second.

Examples
TimeStandardUserSystemClock/TimeStandardUserSystemClock.cpp.
