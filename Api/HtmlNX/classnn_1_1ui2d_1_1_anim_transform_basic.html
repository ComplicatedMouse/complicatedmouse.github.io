Sets the resource used by the animation.

Parameters
[in]pDeviceDevice.
[in]pResAccessorThe resource accessor.
[in]pResThe pointer to the animation block for the resource.

Details
Sets the resource needed for the animation data and the animation block data bound to the animation by the Bind() function.

It also allocates the BindPair array needed by the Bind() function, allocating as many objects as there are animation targets included in the animation block.

To implement this function you need to register animation block data using the SetAnimResource() function.

◆ ForceBindPane()

virtual void nn::ui2d::AnimTransformBasic::ForceBindPane ( Pane *  pDstPane,
const Pane *  pSrcPane 
)
virtual

Forces the binding of an animation of another pane.

Details
Binds the animation of the pane specified by pSrcPane to the pane specified by pDstPane.

Binds animations in panes that were originally separate, so it could cause buggy results if, for example, the panes are of different types. Use this feature with care.

Use the UnbindPane function to unbind a bound animation.

Parameters
[in]pDstPanePointer to the pane to forcibly bind to.
[in]pSrcPaneName of the pane with the animation you want to forcibly bind.

Implements nn::ui2d::AnimTransform.

Determines whether the data is Wait animation data.

Details
This member function determines whether the data is Wait animation data. A Wait animation has data with a frame length of 0 and a Playback Mode that has been set to Wait in the LayoutEditor settings for animation segment tags.

Returns
