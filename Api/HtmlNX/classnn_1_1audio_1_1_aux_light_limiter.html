Detailed Description

The class for handling the LightLimiter effect utility applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform LightLimiter effect processing. By specifying the sample data obtained by nn::audio::ReadAuxSendBuffer() as the input for this LightLimiter effect, you can produce an effect that is restricted to a specific range of sample values.

The nn::audio::AuxLightLimiter class is a utility class that wraps calls to functions that use nn::audio::AuxLightLimiterType objects. The member functions provided by this class are implemented as functions that call functions like nn::audio::ProcessAuxLightLimiter() that take the nn::audio::AuxLightLimiterType type for their arguments.
