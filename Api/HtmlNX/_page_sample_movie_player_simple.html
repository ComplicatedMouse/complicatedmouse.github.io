This sample program demonstrates how to play videos using the MoviePlayer API.

Overview

This sample program plays specified movie files.

File Structure

This sample program is located in Samples/Sources/Applications/MoviePlayerSimple/ .

Required Environment

No additions are necessary.

Using the Demo

If you execute the sample program, the specified video will be played back automatically. The program exits automatically after playback is complete.

  • A Button: Pause/resume playback.
  • B Button: Stop playback.
  • X Button: Exit the application.
  • L Button: Rewind (250-ms intervals by default).
  • R Button: Fast-forward (250-ms intervals by default).
  • ZR Button: Increase the rewind and fast-forward intervals (in 250-ms increments, up to a maximum of 10 seconds).

Cautions

Nothing specific.

Execution Procedure

<MoviePlayerSimple executable> <movie path> <optional parameters>

<movie path>: A video file in a supported format.

  • Containers: mp4, m4a, webm, mkv.
  • Video codecs: h264, vp8, vp9.
  • Audio codecs: aac, vorbis.

If the target platform is Windows, only some formats are supported.

  • Container: webm
  • Video codec: vp9
  • Audio codec: vorbis

<optional parameters>

  • -dm 1: Enables movie::DecoderMode_Cpu mode. (If the target platform is Windows, Cpu mode is the only mode supported.)
  • -dm 2: Enables movie::DecoderMode_NativeTexture mode. The application passes NVN_FORMAT_RGBA8 format NVN textures to the video decoder. (Default)

Example

  • Get movie data from the host PC: MoviePlayerSimple.nsp c:/H264_AAC_1280x720_30sec_01.mp4
  • Get movie data from an SD card (for development purposes): MoviePlayerSimple.nsp sdcard:/H264_AAC_1280x720_30sec_01.mp4
  • If the target platform is Windows: MoviePlayerSimple.exe c:/VP9_Vorbis_1280x720_30sec_01.webm

Description

Plays a specified movie file.
