◆ ShowApplicationError()

void nn::err::ShowApplicationError ( const ApplicationErrorArg &  arg)

Displays the application-specific errors.

Parameters
[in]argThe class used to store application-specific error information.
Precondition
  • The message for dialog-box display, or the message for full-screen display, or both messages are configured in arg.
  • The language is configured in arg.

Details

  • Blocks the caller while the error is being displayed.
  • This function is one of the error display functions.
    • This function is not for general message display. Do not use this function to display warnings, notifications, or other messages that are not error messages.
  • You must configure Application/ApplicationErrorCodeCategory in the NMETA file to use this function. For more information, see Writing the NMETA File in NintendoSDK Documents.
NX Platform Only:
This function uses the error viewer applet to display errors. For more information, see System Features Guide > Library Applets > Error Viewer Applet.

Displays the error that prompts the user to update if the currently installed add-on content version is unacceptable.

Details

Shows an error prompting an update when the running application has an unacceptable version.

Details
