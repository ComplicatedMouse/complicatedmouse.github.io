Initializes the sound library.

Details
The work memory used by the sound system is passed by the user. Set the parameters needed for initialization in the SoundSystemParam structure, and get the required memory size using nn::atk::SoundSystem::GetRequiredMemSize.

The thread that processes sounds and the thread that loads sound data from storage are initialized and run using the priority and stack size values that were specified in the SoundSystemParam structure.

Parameters
[in]paramThe initialization parameters.
[in]workMemThe pointer to the work memory for the sound system.
[in]workMemSizeThe size of the work memory.
Returns
Returns true if the sound library was initialized successfully.
Precondition
  • workMem is aligned to WorkMemoryAlignSize.
  • param.soundThreadStackSize is aligned to nn::os::ThreadStackAlignment.
  • param.taskThreadStackSize is aligned to nn::os::ThreadStackAlignment.
  • param.rendererSampleRate == 32000 || param.rendererSampleRate == 48000.
  • param.voiceCountMax >= 1 && param.voiceCountMax <= SoundSystem::VoiceCountMax.
  • param.busCountMax >= 1 && param.busCountMax <= .OutputReceiver::BusCountMax.

◆ SetOutputMode()

static void nn::atk::SoundSystem::SetOutputMode ( OutputMode  mode,
OutputDevice  device = OutputDevice_Main 
)
static

Sets the output mode for the sound.

Details
The default output mode is OutputMode_Stereo. If you want to know the current output mode, use GetOutputMode.

Calculating the Mix Parameters

The output modes set with this function influence the method used to calculate mix parameters (front L/R and rear L/R volume. Referred to respectively in this document as fL, fR, rL, rR.

For example, if the mode is OutputMode_Monaural, fL and fR are both set to the same value and rL and rR are both set to 0, regardless of the pan and surround pan values set in SoundHandle. 

   - fL = fR = M (value commensurate to the volume).
   - rL = rR = 0.0f.

If the mode is OutputMode_Stereo, fL and fR are set independently to apply the pan (if the pan is centered, they take the same value), but surround pan is ignored and rL and rR are set to 0. 

   - fL = L (value commensurate to volume and pan).
   - fR = R (value commensurate to volume and pan).
   - rL = rR = 0.0f.

If the mode is OutputMode_Surround, fL, fR, rL, and rR are set independently to apply the pan and surround pan values (if the respective pans are centered, they take the same value). 

   - fL = surFL (value commensurate to volume, pan, and surround pan).
   - fR = surFR (value commensurate to volume, pan, and surround pan).
   - rL = surRL (value commensurate to volume, pan, and surround pan).
   - rR = surRR (value commensurate to volume, pan, and surround pan).

Downmixing the Mix Parameters

In addition to calculating the values for front L/R and rear L/R, the mix parameters are then downmixed, in conjunction with the final number of speaker outputs. For more information about downmixing, see Audio Renderer > Sink.

This function sets the master volume.

Details
The master volume affects all sounds played by the sound library. It does not affect sounds played by means other than the sound library.

The value specified by this function operates independently of other volume parameters, and they are all layered.

The volume parameter takes a ratio of 0.0 or greater. Volume does not change if 1.0 is specified. If 0.0 is specified, there is no sound. The default value is 1.0.

Note that after this value is combined with the other volume parameters, the final volume is clamped to the range from nn::audio::VoiceType::GetVolumeMin() to nn::audio::VoiceType::GetVolumeMax(). For more information about combining volume parameters, see Sound Parameter Control Operations.

The volume changes over the time specified by fadeTimes. If this function is called again while the volume is changing, the volume starts to change over the newly specified time, using the volume at that point as the base point.
