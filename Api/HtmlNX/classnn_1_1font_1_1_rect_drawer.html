static size_t nn::font::RectDrawer::GetWorkBufferSize ( nn::gfx::Device *  pDevice,
uint32_t  charCount 
)
static

Gets the size of the work buffer required for RectDrawer.

Details
Also adds the size of the memory pool. If you are using Initialize() to separately specify the memory pool, subtract the value of CalculateMemoryPoolSize() from the return value of this function.

Parameters
[in]pDeviceThe device.
[in]charCountThe number of characters that can be drawn in one process.
Returns
Returns the memory size of the vertex buffer and command buffer. (The memory space must be aligned to GetWorkBufferAlignment.)

nn::gfx::MemoryPool *  pExternalMemoryPool,
ptrdiff_t  externalMemoryPoolOffset,
size_t  externalMemoryPoolSize 
)
virtual

Initializes.

Details
For the memory pool, specify a pool that has the properties nn::gfx::MemoryPoolProperty_CpuUncached and nn::gfx::MemoryPoolProperty_GpuCached. Use CalculateMemoryPoolSize() to get the size and CalculateMemoryPoolAlignment() to get the offset alignment.

Parameters
[in]pDeviceThe device.
[in]pWorkBufferThe vertex buffer. It must be aligned to GetWorkBufferAlignment().
[in]charCountThe number of characters.
[in]pExternalMemoryPoolThe memory pool.
[in]externalMemoryPoolOffsetThe offset to the memory pool region to use.
[in]externalMemoryPoolSizeThe memory pool size.
