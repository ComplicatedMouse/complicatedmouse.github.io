Detailed Description

ImGui supplemental library.

Details
Use nn::gfx for rendering and then transfer the user input received from nn::hid to ImGui.

A copyright notice is needed when including ImGui in products. Use the software legal information system (SLIM) for the copyright notice. https://slim.mng.nintendo.net/slim/

◆ Initialize() [1/2]

void nns::dbgui::Initialize ( nn::gfx::Device *  pDevice,
nn::gfx::Queue *  pQueue,

)

Initializes the library.

Parameters
[in]pDeviceAn nn::gfx::Device instance.
[in]pQueueAn nn::gfx::Queue instance.
[in]infoLibrary settings.
Precondition
  • IsInitialized() == false.
Postcondition
  • IsInitialized()

Details
Initializes the internal library data. Sets the rendering parameters according to the device and the structure of the queue. The command buffer instance passed to Render must be created from the device passed to this function. In addition, the queue for rendering can only use the queue passed to this function. The reason is that the queue structure can result in different rendering settings.

◆ SetMouseState()

void nns::dbgui::SetMouseState ( const nn::hid::MouseState &  mouseState)

Transfers the mouse input state to ImGui.

Parameters

Precondition

Details
Controls the ImGui mouse pointer and button. There are two ways to control the mouse with this library:

  • SetMouseState()
  • SetTouchState()

Entries from TouchScreen are prioritized. When SetTouchState is called, any entries from SetMouseState are ignored until either EndFrame or Render are called a few times.

◆ Render()

void nns::dbgui::Render ( nn::gfx::CommandBuffer *  pCommandBuffer)

Renders the frame.

Parameters
[in]pCommandBufferAdds the rendering command to the command buffer.

Details
Call this function instead of ImGui::Render for ImGui to render the frame. Assume that all configured GPU states will be invalidated because GPU states can be overwritten. If the NewFrame() function was not called, the same configuration as the previous frame is rendered again. Implicitly call the EndFrame() function. Note: Only nn::gfx::Queue instances passed during initialization can be used.

Precondition
