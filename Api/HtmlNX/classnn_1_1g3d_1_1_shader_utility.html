Parameters
[out]pKeyPointer to the buffer that stores the shader key.
[in]keyLengthShader key length.
[in]pResShadingModelPointer to the shading model resource.
[in]pResShaderAssignPointer to the shader assignment information.
[in]materialPassWhether to enable warnings.

Details
Set materialPass to true to display a warning if the settings of the shader and material do not match.

Binds the shading model to the material.

Parameters
[in]pResMaterialPointer to the material resource.
[in]pResShadingModelPointer to the shading model resource.

Details
Records the size of the UniformBlock writing to the shader parameter, and the individual shader parameter offsets to the material.

Examples
