Describes the process of getting pseudo device IDs using the application operating environment library.

Overview

This sample program describes how to get pseudo device IDs using the application operating environment library.

For more information about using the application operating environment, see the Operating Environment Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OePseudoDeviceId/ .

Required Environment

This sample program can only be built and run on the NX platform.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program gets pseudo device IDs and outputs them to a log.

This sample program has the following flow.

  • Initialize the application operating environment library.
  • Get the pseudo device ID.
  • Display the pseudo device ID.

The following is an example of the results from running this sample program.

Your Pseudo Device ID is ab40c421-6e12-5180-87e4-1b13c4ce4dfa.
