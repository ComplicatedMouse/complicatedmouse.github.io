This function takes the range arrays for two submeshes and calculates a new range array that comprises both of those range arrays.

Parameters
[out]pDstPointer to a SubMeshRange array.
[in]pLHSPointer to a SubMeshRange array.
[in]pRHSPointer to a SubMeshRange array.
Returns
Returns the number of submeshes.

Details
pLHS and pRHS must end with a SubMeshRange for which index and count are both 0. pDst ends with a SubMeshRange for which index and count are both 0. This function returns the number of submesh elements in the new array not including the terminating element. If the LOD level is different when combining, the level that will be reduced more (the larger one) is used.
