Detailed Description

This structure provides a feature to distribute the sample data of the VoiceType / SubMixType input source to multiple output destinations.

Details
Specify this structure as the VoiceType / SubMixType output destination to send the sample data stored by VoiceType / SubMixType to multiple SubMixType and FinalMixType objects. SplitterType does not perform sample rate conversion. For this reason, all SplitterType output destinations must have the same sample rate as that specified for SplitterType.

Directly altering member variables is prohibited. Use the main structure through the following functions.
