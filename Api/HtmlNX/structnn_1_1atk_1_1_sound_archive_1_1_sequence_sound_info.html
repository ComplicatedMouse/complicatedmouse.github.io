◆ allocateTrackFlags

uint32_t nn::atk::SoundArchive::SequenceSoundInfo::allocateTrackFlags

A bit flag representing tracks that are included in the sequence sound.

This represents track 0, track 1, track 2, and so on in order starting at the least-significant bit. Tracks for valid bits can be manipulated.

◆ isReleasePriorityFix

bool nn::atk::SoundArchive::SequenceSoundInfo::isReleasePriorityFix

Flag indicating whether the channel priority is fixed for the sound produced by the sequence sound at time of release.

If not fixed, the player priority is lowered at time of release.
