This sample program shows how to use the ui2d library.

Overview

This documentation describes the Ui2dDemo sample program.

File Structure

This sample program is located in Samples/Sources/Applications/Ui2dDemo/ .

Required Environment

Nothing specific.

How to Use

The Joy-Con, DebugPad, mouse, and touch panel can be used to perform the following operations.

Input Operation
Select items using up or down on the +Control Pad or the mouse. Select the sample.
Press the A Button or left mouse button. Execute the sample.
Press the B Button or right mouse button. Returns to the menu.

Important Information

Nothing specific.

Execution Procedure

You can build the sample program and select which sample to run from the menu.

Description

Each sample uses a simple workflow to initialize, run the main loop, and then finalize.

Check the following samples by selecting from the menu or changing the g_SampleMode value, or by entering a number starting from 0 as the parameter value.

  • SampleMode_Simple: Simple sample.
  • SampleMode_MultiArcResource: Sample that uses multiple archive resources and resource accessors.
  • SampleMode_TextureChange: Sample that changes the texture of a pane to a texture loaded by the system.
  • SampleMode_UserShader: Sample that loads a customized shader to the ui2d library.
  • SampleMode_FontChange: Sample that changes the font in the program.
  • SampleMode_ComplexFont: Sample that loads the complex font.
  • SampleMode_CannonButton: Sample program for handling buttons.
  • SampleMode_CapturePane: Sample program that captures textures.
  • SampleMode_TextureCacheAdvanced: Sample program of advanced uses of texture cache.
