Describes the resampler sample program.

Overview

This sample uses the resampler to change the sampling rate.

File Structure

This sample program is located in Samples/Sources/Applications/AudioOutWithResampler/ .

Required Environment

You must be able to use audio output.

How to Use

Run the sample program to play square waves. You can operate the program using input from a keyboard or from DebugPad.

Input Operation
START or SPACEBAR Exits the program.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program combines Samples/Sources/Applications/AudioOut/ with the resampler.

Convert the sampling rate to convert a 44.1kHz waveform for 48kHz AudioOut.

This sample program has the following flow.

  • List the usable audio outputs.
  • Open the default audio output.
  • Get and verify audio output information.
  • Prepare the resampler.
  • Prepare and register the playback buffer.
  • Start playback.
  • Play the audio.
  • End playback.
  • Close audio output.
  • Free the playback buffer.

Use the nn::audio::InitializeResampler() function to initialize the resampler. The sampling rate and number of channels for the input/output audio data must be specified at this point, and cannot be changed after initialization. In addition, a work buffer of size nn::audio::GetRequiredBufferSizeForResampler() must be passed during initialization. This buffer cannot be deallocated until you finish using the resampler.

Use the nn::audio::ProcessResamplerBuffer() function to perform resampling. The number of input/output samples for resampling is determined by the sampling rate ratio. You must use the nn::audio::GetResamplerOutputSampleCount() function to calculate the output buffer size in advance and then prepare a buffer of at least this size.
