◆ SetBufferCount()

InitializeArg& nns::gfx::GpuBuffer::InitializeArg::SetBufferCount ( int  bufferCount)

Specifies the number of buffers for multiple buffering.

Details
A number of nn::gfx::Buffer instances equivalent to the specified number are created, and it is possible to switch between and use these buffers. If you want to use double-buffering, specify 2. For triple-buffering, specify 3.

Parameters
[in]bufferCountThe number for multi-buffering.
Returns
Returns a reference to the object.
