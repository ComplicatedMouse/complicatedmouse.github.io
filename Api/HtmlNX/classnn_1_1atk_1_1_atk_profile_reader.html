◆ Initialize()

template<typename TProfile >
void nn::atk::AtkProfileReader< TProfile >::Initialize ( void *  buffer,
size_t  bufferSize,
int  profileMaxCount 
)

The use of internal features is prohibited. Initialize the profile reader.

Details
Initialization requires a buffer that is at least the size of the value obtained with the GetRequiredMemorySize function. In addition, specify the same value as GetRequiredMemorySize for the profileMaxCount of the argument.

You can get the profile information by calling the Read function.

Parameters
[in]bufferA buffer of at least the size obtained from GetRequiredMemorySize.
[in]bufferSizeThe size of buffer.
[in]profileMaxCountThe maximum number of profiles that can be stored. Specify the same value as GetRequiredMemorySize.
Precondition
  • buffer != NULL.
  • bufferSize >= value obtained from GetRequiredMemorySize.
  • profileMaxCount > 0.

◆ Read()

template<typename TProfile >
int nn::atk::AtkProfileReader< TProfile >::Read ( TProfile *  pProfile,
int  maxCount 
)

The use of internal features is prohibited. Gets the profile information.

Details
The pProfile specified by the argument can store a maximum of maxCount task profiles, but if there are not that many profiles for the profile reader to read, only the profiles that can be read are stored there.

Parameters
[in]pProfileThe location for storing the profile information that was read.
[in]maxCountThe maximum number of profiles that are read.
Returns
Returns the number of stored task profiles.
Precondition
  • pProfile != NULL.
