Getting Information for Other Related Titles

This section describes the sample program that gets information about other related titles.

Overview

This sample gets the information for other related titles. After selecting Related Title/Bundle List on the start screen, this is the screen after selecting sale status (on sale items only / all items). When a title is registered, the application ID for a different, related title can be specified as an item tag. You can read information about related titles and use it to purchase those titles in the shop. The title information shows information about the title, including the title name, a description, a disclaimer, information about the application type, and price information. To get the price information, a separate function for that purpose is used. Selecting the title jumps to eShop. If you select on sale items only for sale status (on sale items only / all items), a list of items currently on sale is displayed. If you select all items, all items are displayed except those for which sales_status is not_onsale in the price information.

File Structure

This sample program is located in Samples/Sources/Applications/EcCatalog/RelatedTitleBundle/ .
