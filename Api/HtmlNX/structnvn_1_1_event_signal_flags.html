enum nvn::EventSignalFlags::Enum

Enumerator
NO_WAIT_MEMORY 

Do not wait for completion of memory transactions from commands prior to the SignalEvent command.

By default, an event is not signaled until all GPU memory transactions from prior commands have completed. When this bit is set, the event may be signaled without waiting for all memory transactions.
