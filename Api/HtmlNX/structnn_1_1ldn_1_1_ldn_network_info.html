Detailed Description

A structure that stores ldn-specific network information.

Details
nodeCount indicates the number of nodes where isConnected == true. The nodes where isConnected == true are not stored packed into the front, so to get information about the connected nodes, search NodeInfo for nodes that satisfy isConnected == true.

The data and size of the data that the access point specified using the SetAdvertiseData function is stored in advertiseData and advertiseDataSize.

This type of variable requires a large memory area, so we do not recommend allocating it in the stack. If it is allocated in the stack, be careful to prevent the stack from overflowing during execution.
