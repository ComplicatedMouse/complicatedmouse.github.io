#include <nn/audio/audio_AuxUtilityEffectLightLimiterTypes.h>

Detailed Description

Utility that applies a LightLimiter effect to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform LightLimiter effect processing. The utility specifies the sample data obtained by nn::audio::ReadAuxSendBuffer() as the input for the effect, and outputs the result of applying the LightLimiter effect, which limits the sample values to a certain range. This effect processing occurs in the CPU thread that called the nn::audio::ProcessAuxLimiter() function.

Directly altering member variables is prohibited. Use the main structure through the following functions.
