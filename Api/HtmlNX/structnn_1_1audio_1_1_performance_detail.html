Detailed Description

Detailed performance information linked to each VoiceType, SubMixType, and FinalMixType object.

Details
You can get information about the specified VoiceType, SubMixType, and FinalMixType objects by using nn::audio::SetPerformanceDetailTarget(). parentId contains the ID of each VoiceType, SubMixType, and FinalMixType obtained by nn::audio::GetVoiceNodeId(), nn::audio::GetSubMixNodeId(), and nn::audio::GetFinalMixNodeId(). The NodeId obtained here is identical to the NodeId of the VoiceType, SubMixType, and FinalMixType objects specified with nn::audio::SetPerformanceDetailTarget(). startTime and processingTime specify the time that processing started and the amount of time spent processing in the audio renderer, respectively. detailType specifies the types of processes executed with the specified VoiceType, SubMixType, and FinalMixType objects. parentEntryType specifies the VoiceType, SubMixType, and FinalMixType objects specified with nn::audio::SetPerformanceDetailTarget(). The user can use parentEntryType and parentId to uniquely identify the VoiceType, SubMixType, and FinalMixType objects registered to the audio renderer by this PerformanceDetail.
