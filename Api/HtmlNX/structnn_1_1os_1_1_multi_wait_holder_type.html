nn::os::MultiWaitHolderType Struct Reference

This structure defines a multiwait object holder. More...

#include <nn/os/os_MultipleWaitTypes.h>

Detailed Description

This structure defines a multiwait object holder.

Details
Directly altering member variables is prohibited.
Use the main structure through the following functions.

  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, EventType *event)
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, TimerEventType *timerEvent)
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, SystemEventType *event)
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, MessageQueueType *messageQueue, MessageQueueWaitType waitType)
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, SemaphoreType *semaphore)
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, ThreadType *thread)
  • FinalizeMultiWaitHolder()
  • LinkMultiWaitHolder()
  • UnlinkMultiWaitHolder()
  • UnlinkAllMultiWaitHolder()
  • MoveAllMultiWaitHolder()
  • SetMultiWaitHolderUserData()
  • GetMultiWaitHolderUserData()

The various types of the WaitAny() function will return this object.

  • WaitAny()
  • TryWaitAny()
  • TimedWaitAny()

To use these functions, include nn/os.h.

Examples
AudioEchoback/AudioEchoback.cpp, LdnBasic/LdnAccessPoint/LdnAccessPoint.cpp, LdnBasic/LdnStation/LdnStation.cpp, MovieDecoderPlayer/MovieDecoderPlayer.cpp, MoviePlayerSimple/MoviePlayerSimple.cpp, and OsMultiWait/OsMultiWait.cpp.
