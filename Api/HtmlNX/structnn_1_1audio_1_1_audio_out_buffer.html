Detailed Description

Structure representing an audio output buffer.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • SetAudioOutBufferInfo()
  • GetAudioOutBufferDataPointer()
  • GetAudioOutBufferDataSize()
  • AppendAudioOutBuffer()
  • GetReleasedAudioOutBuffer()
  • ContainsAudioOutBuffer()

To use these functions, include nn/audio.h.

Examples
AudioCpuRenderer/AudioCpuRenderer.cpp, AudioEchoback/AudioEchoback.cpp, AudioOut/AudioOut.cpp, AudioOutWithResampler/AudioOutWithResampler.cpp, and CodecAacDecoder/CodecAacDecoder.cpp.
