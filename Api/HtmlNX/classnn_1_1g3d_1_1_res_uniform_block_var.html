enum nn::g3d::ResUniformBlockVar::Type

Enumerates uniform block types.

Enumerator
Type_None 

The type is unspecified.

Type_Material 

Represents a uniform block for a material.

Type_Shape 

Represents a uniform block for a shape.

Type_Skeleton 

Represents a uniform block for a skeleton.

Type_Option 

Represents a uniform block for options.

Type_Num 

The use of internal features is prohibited.
