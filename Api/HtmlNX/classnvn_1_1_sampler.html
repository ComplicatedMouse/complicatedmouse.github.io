Detailed Description

GPU-accessible object controlling how texture data are fetched and filtered.

Examples
NvnSimple/NvnSimpleSharedLib/GraphicsTextureManager.cpp, and nvnsimple/NvnSimpleSharedLib/GraphicsTextureManager.h.
