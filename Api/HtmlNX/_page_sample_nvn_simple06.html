Overview

This sample implements the drawing of some cones by different command sets to introduces how to use multiple command sets from a single NVN command buffer. 9 cones are drawn by different command sets which have different materials and translations.

Description

In this sample, a single command buffer is utilized to record multiple command sets that are different from one another. The first command set records common initial rendering commands for the scene. The additional command sets record material and translate information for each shape instance that is drawn. The recorded command handles are then submitted to the nvn::Queue at once.
