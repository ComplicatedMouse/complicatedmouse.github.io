◆ IsValid()

bool nns::dbgui::InterfaceInfo::IsValid ( ) const

Determines whether valid values are set.

Returns
Returns true when valid values are set.

Details
Checks the following. Valid values have been set for every descriptor pool and descriptor pool index. The rendering dimensions have been set 1 or greater. The buffer count is at least 1. The memory callback function is configured. The TTF buffer property is set to a valid value. The keyboard input method is set to a valid value. All navigation key assignments are set. The buffer size has been configured to be at least 1.

◆ SetIniFilePath()

void nns::dbgui::InterfaceInfo::SetIniFilePath ( const char *  filePath)

Sets the path for the ImGui configuration file.

Parameters
[in]filePathPath for the ImGui configuration file.

Details
If you call this function, the configuration file is written automatically. The process depends on the file system, so it makes use of the standard C library functions fopen, fclose, fread, and fwrite. The file system is accessed when NewFrame () is called, which can prolong the processing time. If you cannot use the file system, use SetIniConfiguration. This function and SetIniConfiguration are mutually exclusive.

See also
SetIniConfiguration
