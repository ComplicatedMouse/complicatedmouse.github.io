Copy constructor.

Details
Creates a copied pane from the pane specified by the argument.

Note that the pane is copied based on its state at that time, and not on its original state in the layout data. In other words, if the copying occurs after an animation has played, the pane is copied in that played state.

Note, however, that this function does not copy the parent-child relationships of panes. In other words, the pane is created without a parent and with an empty child list. Explicitly call the AppendChild() function on the pane to add it to the pane tree.

Even for animations, nothing is bound.

To generate a new pane, a font constant buffer larger than the requiredFontConstantBufferSize value automatically calculated when the layout is constructed must be allocated and created.

This constructor cannot copy the information about capture textures, capture effects, or vector graphics textures. To copy panes that use these features, use Window::Window(const Pane& pane, nn::gfx::Device* pDevice, Layout* pLayout).
