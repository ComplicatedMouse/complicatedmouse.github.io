Detailed Description

(Beta version.) Manages a sound remote-control session.

Details
When a session is open, this class accepts remote control requests from SoundMaker.

viewer::Result nn::atk::viewer::SoundControlSession::Initialize ( void *  buffer,
size_t  bufferLength,
const Configs &  configs 
)

Initializes SoundControlSession.

Details
This function initializes the memory used by the SoundControlSession object and prepares to start the connection with SoundMaker.

Make sure that you call this function before using SoundControlSession.

buffer is the work buffer used for communications.

You must specify a size that is greater than the value returned by SoundControlSession::GetRequiredMemorySize().

Parameters
[in]bufferThe work buffer for communication processes.
[in]bufferLengthThe length of buffer.
[in]configsThe SoundControlSession settings. Specifies Configs passed to GetRequiredMemorySize().
Returns
Returns the execution result.

◆ GetRequiredMemorySize()

size_t nn::atk::viewer::SoundControlSession::GetRequiredMemorySize ( const Configs &  configs) const

Gets the size of memory needed to use SoundControlSession.

Details
To use SoundControlSession, you must allocate a buffer of the size obtained by this function, and specify it for the buffer and bufferLength parameters of the Initialize() function.

At that time, in the configs parameter of the Initialize() function, specify the same Configs structure that was passed to the GetRequiredMemorySize() function.

Parameters
[in]configsThe SoundControlSession settings.
Returns
Gets the necessary memory size.
