Detailed Description

A multiphase memory allocator interface with the same interface as N4529.

Details
A multiphase memory allocator interface with an N4529 (http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2015/n4529.html#memory.resource) equivalent interface.

is_equal() may be used to compare MemoryResource. When there are two MemoryResource objects a and b, and the memory space allocated with a can be correctly freed in b, and the reverse is also true, a.is_equal(b) returns true. If a.is_equal(b) is true, a and b are considered to be equivalent values.

The concrete class of this class must override do_allocate(), do_deallocate(), and do_is_equal().
