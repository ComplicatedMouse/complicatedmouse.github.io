Enumerator

Enabled.

Skips the update process, but applies the final result.

Skips both the calculate and the apply processes.

◆ RevertTo()

void nn::g3d::MaterialAnimObj::RevertTo ( ModelObj *  pModelObj) const

Reverts the parameters overwritten by own material animation to the material source values.

Parameters
[in]pModelObjThe model instance to be reverted.
Precondition
  • m_pRes != NULL.
  • IsTargetBound()
  • pModelObj != NULL.

Details

  • Note that if parameters of model materials have been changed by multiple material animations, When this function is called, the parameters will be overwritten with values from the material resource.
  • Processing is skipped for materials where Apply was disabled using SetBindFlag().
