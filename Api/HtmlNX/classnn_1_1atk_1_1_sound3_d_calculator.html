Detailed Description

Consolidates the calculations of standard parameters for 3D sounds.

Details
This function is called from the Sound3DEngine class.

The functions of this class can be used when customizing and creating a unique 3D sound engine.

Calculates the standard volume and priority parameters for 3D sounds.

Details
Volume and priority are calculated together by this function because they are linked for 3D sounds.

This function has been retained for compatibility with earlier versions. To reduce calculations, the current version of the nn::atk library calls an overload of this function with the actorDistance parameter.

Calculates the standard volume and priority parameters for 3D sounds.

Details
Volume and priority are calculated together by this function because they are linked for 3D sounds.

In the current version of the nn::atk library, to lighten the computational load, the actorDistance parameter is calculated in advance and then passed to the following functions:

  • CalculateVolumeAndPriority (this function)
  • CalculatePitch
  • CalculateBiquadFilterValue

For more information, see Sound3DEngine::UpdateAmbientParam.
