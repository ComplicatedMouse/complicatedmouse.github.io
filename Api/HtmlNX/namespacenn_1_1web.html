◆ ShowWebPage()

nn::Result nn::web::ShowWebPage ( WebPageReturnValue *  pOutputWebPageReturnValue,
const ShowWebPageArg &  arg 
)

Opening a webpage.

Parameters
[out]pOutputWebPageReturnValuePointer to the WebPageReturnValue instance for receiving the results of display a webpage.
[in]argShowWebPageArg instance for maintaining the information for opening the webpage.
Returns
Result value indicating the processing results.
Return Values:

nn::ResultSuccess Webpage display succeeded and the display was terminated by user operation.

nn::web::ResultTerminated Webpage display was forcibly canceled and the display was terminated.

Precondition
  • pOutputWebPageReturnValue != nullptr.

Details
This function maintains the instance specified by arg. Displays the webpage based on the URL.

This function blocks the caller until display of the webpage is terminated. If the return value is not nn::ResultSuccess, do not reference the value of pOutputWebPageReturnValue because that value is not valid.

If RequestExitWebPage is called when the webpage is not opened, the function returns nn::web::ResultTerminated.

NX Platform Only:
This function uses the web applet to display the webpage. For more information about the web applet and webpages that can be displayed, see the Web Applet section of the System Features Guide.
Examples
WebDemo/WebDemo_Main.cpp, and WebSimple/WebSimple_Main.cpp.

◆ RequestExitWebPage()

void nn::web::RequestExitWebPage ( )

Sends an exit request to a page that is open.

Precondition
  • ShowWebPage is complete.

Details
If this function was called in the background-focus state, the opened page is momentarily shown when transitioning to the in-focus state. To avoid this problem, you must not call this function in the background-focus state.

◆ ShowOfflineHtmlPage()

nn::Result nn::web::ShowOfflineHtmlPage ( OfflineHtmlPageReturnValue *  pOutputOfflineHtmlPageReturnValue,
const ShowOfflineHtmlPageArg &  arg 
)

Open the offline HTML (documentation) added to the application.

Parameters
[out]pOutputOfflineHtmlPageReturnValuePointer to the OfflineHtmlPageReturnValue instance for receiving the results of opening offline HTML.
[in]argShowOfflineHtmlPageArg instance for maintaining the information for opening offline HTML.
Returns
Result value indicating the processing results.
Return Values:

nn::ResultSuccess Offline HTML display succeeded and the display was terminated by user operation.

nn::web::ResultTerminated Offline HTML display was forcibly canceled and the display was terminated.

Precondition
  • pOutputOfflineHtmlPageReturnValue != nullptr.

Details
This function maintains the instance specified by arg. Displays the offline HTML in the path.

This function blocks the caller until display of offline HTML is terminated. If the return value is not nn::ResultSuccess, do not reference the value of pOutputOfflineHtmlPageReturnValue because that value is not valid.

If RequestExitOfflineHtmlPage is called when the offline HTML (document) is not opened, the function returns nn::web::ResultTerminated.

NX Platform Only:
This function uses the offline web applet to display offline HTML. For more information about the Offline Web Applet and offline HTML, see Offline Web Applet in the System Features Guide.
Examples
WebDemo/WebDemo_Main.cpp, and WebOfflineSimple/WebOfflineSimple_Main.cpp.
