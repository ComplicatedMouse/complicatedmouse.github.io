◆ ShapeUpdated()

virtual void nn::g3d::viewer::ViewerCallbacks::ShapeUpdated ( const ShapeUpdatedArg &  arg)
virtual

Called when a shape was updated within the viewer library.

Shapes are updated when models or shapes flash after selecting them. This happens when you need to restructure models within the viewer library or when switching display of LOD models in 3DEditor.

◆ ApplySceneAnimRequested()

virtual void nn::g3d::viewer::ViewerCallbacks::ApplySceneAnimRequested ( const ApplySceneAnimRequestedArg &  arg)
virtual

Process executed when reflecting the calculation result of the scene animation that is set in the bind target.

With this timing, you can set it to apply to any animation calculation result target you like.

◆ TargetSelected()

virtual void nn::g3d::viewer::ViewerCallbacks::TargetSelected ( const TargetSelectedArg &  arg)
virtual

The process executed when a material or other object has been selected in 3DEditor.

If a model has been selected, the index of TargetSelectedArg is always nullptr. The content specified by the index of TargetSelectedArg is invalidated when this callback ends. For deferred processing, copy the contents of index.
