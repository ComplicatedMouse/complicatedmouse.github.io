nn::gfx::DepthStencilView* nns::gfx::DepthStencilBuffer::GetDepthStencilView ( int  index)

Gets the depth stencil view for the specified index.

Parameters
[in]indexThe index of the depth stencil view to get.
Returns
Returns the depth stencil view for the specified index.

Details
If the image dimensionality is nn::gfx::ImageDimension_2dArray, the 0th index is the view of 2DArray. The other indexes correspond to individual color target views inside 2DArray.
