Detailed Description

template<size_t ModulusSize, size_t ExponentSize>
class nn::crypto::RsaCalculator< ModulusSize, ExponentSize >

Class that implements RSA cipher primitive functionality.

Template Parameters
ModulusSizeThe byte size for the RSA modulus to use.
ExponentSizeThe byte size for the RSA exponent to use.

Details
This class provides the potential residue calculation functionality for large moduli and exponents that comprise RSA cipher primitive functionality. When actually using encryption and signatures, use in combination with an algorithm that considers message padding.

This class is a template class that takes the RSA modulus and exponent sizes as template parameters. RSA moduli and exponents can be specified in byte (8-bit) units up to a maximum of 512 bytes (4096 bits).

This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state
  • Ready.
