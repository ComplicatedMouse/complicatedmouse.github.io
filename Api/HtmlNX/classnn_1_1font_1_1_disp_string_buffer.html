◆ SetDoubleDrawnBorderEnabled()

void nn::font::DispStringBuffer::SetDoubleDrawnBorderEnabled ( bool  isDoubleDrawnBorderEnabled)

Sets whether to render the border of the special border format in two cycles.

Details
When SetDoubleDrawnBorderEnabled is set to true for DispStringBUffer, which was not set to nn::font::DispStringBuffer::InitializeArg::SetDoubleDrawnBorder(true), the size in use of the constant buffer increases and may exceed the size obtained in advance with nn::font::DispStringBuffer::GetRequiredConstantBufferSize. To avoid this problem, set DispStringBuffer (that switches SetDoubleDrawnBorderEnabled to nn::font::DispStringBuffer::InitializeArg::SetDoubleDrawnBorder(true)) in advance, and then initialize.

Parameters
[in]isDoubleDrawnBorderEnabledWhether to render the border of the special border format in two cycles.
