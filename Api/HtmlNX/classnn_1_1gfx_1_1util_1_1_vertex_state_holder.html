Detailed Description

template<int TVertexAttributeCount, int TVertexBufferCount>
class nn::gfx::util::VertexStateHolder< TVertexAttributeCount, TVertexBufferCount >

A wrapper class that allocates the necessary additional memory as a member to nn::gfx::VertexState.

The necessary value is taken as a template parameter because the amount of memory to allocate is set statically.

Template Parameters
TVertexAttributeCountThe number of vertex attributes. Specify a value that is equal to or greater than the vertex attribute state information count specified in nn::gfx::VertexStateInfo::SetVertexAttributeStateInfoArray().
TVertexBufferCountThe number of vertex buffers. Specify a value that is equal to or greater than the vertex buffer state information count specified in nn::gfx::VertexStateInfo::SetVertexBufferStateInfoArray().
