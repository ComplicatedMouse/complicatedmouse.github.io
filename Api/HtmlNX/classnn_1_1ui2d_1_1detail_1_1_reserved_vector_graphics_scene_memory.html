Detailed Description

The memory space allocated in advance to avoid dynamic memory allocation inside vector graphics scenes.

Details
Assigns memory from the space allocated ahead of time by the Initialize() function, assigning memory from the start of this space while shifting the pointer. You cannot free and reuse the assigned memory space.
