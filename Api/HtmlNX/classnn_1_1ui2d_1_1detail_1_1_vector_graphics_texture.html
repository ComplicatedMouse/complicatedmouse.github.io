◆ SetReferencedFlag()

void nn::ui2d::detail::VectorGraphicsTexture::SetReferencedFlag ( )

Sets the referenced flag.

The referenced flag must be set in every frame so that it will be reset by the call to Draw().

Details
This function is used to avoid unnecessary rendering of vector graphics textures when the referencing pane is hidden, has a transparency of 0, or is otherwise not being displayed. VectorGraphicsTexture is managed separately from the referencing pane, so the referenced flag is used to determine whether the texture is being used in the referencing pane. If this flag is not set, the content of the texture is not rendered because it is assumed that the texture itself is not being used.

)
static

Creates an internal reference name for the vector graphics texture.

Details
With vector graphics textures, when the same vector graphics texture is being used in multiple locations, you must also separate out the individual instances of the texture because the circumstances in the various locations may differ. Names that are not intended to be duplicated are specified when the layout data is converted to binary, so those names are restored from the resource.

Parameters
[out]pResultThe output buffer. Treated as a buffer with a size of resultSize.
[in]resultSizeThe maximum size of pResult.
[in]pBnvgFileNameThe name of the BNVG file being referenced.
[in]pResThe vector graphics texture resource.
