Parameters

)

Initializes the archive and the resources.

Details
Fails if the archive format is invalid.

Align the archive data passed to pArchiveStart to nn::ui2d::ArchiveResourceAlignment bytes.

For more information, see nn::ui2d::ArcResourceAccessor::Attach.

Parameters
[in]pArchiveStartThe starting address of the archive.
[in]pResourceRootDirectoryThe name of the root directory of the resource.
[in]pMemoryPoolThe memory pool. If the binary is placed in a memory pool, specifies that memory pool. Specify nullptr when nothing has been placed yet.
[in]memoryPoolOffsetThe offset to the resource position from the start of the target memory pool.
[in]memoryPoolSizeThe target memory pool size.
Returns
Returns true if the function was successful in configuring the archive, or false otherwise.
