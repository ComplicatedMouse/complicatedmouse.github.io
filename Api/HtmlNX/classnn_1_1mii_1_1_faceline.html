Detailed Description

The class for creating Faceline textures.

Details
The class that maintains the DrawParam instance needed when creating Faceline textures.

Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

)

Initializes Faceline.

Parameters
[in]pMemorySpecifies the memory Faceline will use.
[in]memorySizeThe size of pMemory.
[in]pDeviceThe device.
[in]pMemoryPoolSpecifies the memory pool Faceline will use.
[in]memoryPoolOffsetSpecifies the offset to the memory pool Faceline will use.
[in]memoryPoolSizeSpecifies the size of the memory pool Faceline will use.
[in]pTemporaryBufferThe pointer to the structure for a temporary buffer.
[in]resourceThe Faceline build resource.
[in]charInfoThe CharInfo of the Mii to be generated.
[in]isVerticalFlipFlag indicating whether to vertically invert.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && Resource::IsInitialized() && ! Faceline::IsInitialized() && pTemporaryBuffer != NULL && The offset and size are normal. && charInfo is not corrupted.
Postcondition
If successful, Faceline::IsInitialized() = true.
Attention
Both pMemory and pMemoryPool must be retained until the call to Faceline::Finalize().
Pass the memory pool that was set in MemoryPoolProperty_CpuCached to pMemoryPool.
resource may need to be retained until the call to Faceline::Finalize() based on the Resource settings conditions.
You cannot re-initialize an instance of Faceline that has been initialized already. Call this function after calling Finalize().
Only required when initializing pTemporaryBuffer. You can destroy it after initialization is complete.
Set the vertical inverse flag, isVerticalFlip, in conjunction with the application coordinate system settings.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
