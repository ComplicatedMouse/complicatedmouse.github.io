Detailed Description

Class for handling thread-local storage.

Details
Thread-local storage (called "TLS") is a way of maintaining thread-specific data (different for each thread) using a single TLS slot.

The nn::os::ThreadLocalStorage class is a utility class that wraps calls to TLS-related features that use nn::os::TlsSlot objects.

The member functions provided by this class are implemented as functions that call functions like nn::os::GetTlsValue() that take the nn::os::TlsSlot type for their arguments.
