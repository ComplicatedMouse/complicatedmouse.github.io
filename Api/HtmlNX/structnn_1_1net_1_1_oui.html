◆ FromAscii()

static constexpr Oui nn::net::Oui::FromAscii ( const char *  ascii)
staticconstexprnoexcept

Generates an OUI from a string.

Parameters
[in]asciiThe ASCII string.
Returns
Returns the generated OUI.

Details
For the ascii parameter, specify a six-digit hexadecimal string, separated by a delimiter every two digits. As the delimiter, you can use either a colon (:) or a hyphen (-). The hexadecimal string can have either or both uppercase and lowercase letters. Examples of valid strings include the following.

  • 00:22:aa
  • 00-22-AA
  • 00:22-Aa

The string does not need to be ended with a terminating character. If the string does not adhere to this format or if nullptr is passed to the argument, the function returns ZeroOui.
