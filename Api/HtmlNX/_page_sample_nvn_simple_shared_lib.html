Nvn Simple Shared Library

Table of Contents

  • Overview
  • File Structure

The NVN Simple shared Library contains common code shared by NvnSimple 01-06.

Overview

The NVN Simple shared Library contains helper classes and functions to interface with NVN while exposing underlying NVN objects to facilitate direct NVN usage. Also includes utilities for File I/O, font and text rendering, a simple primitive renderer with a local illumination lighting model, and other shared framework code. The project file for the library is included each of the samples' visual studio solution files and is built before the sample runs.

File Structure

FileCache.h, FileCache.cpp A simple class that allows you to load file content into a cache and subsequently lookup actual file content data based on file path.
FileSystem.h, FileSystem.cpp Functions to read files like shader or texture files which are already conveted into NX format.
Font.h, Font.cpp A font class that supports all the font asset types that nn::fontll::ScalableFontEngine supports, namely TTF, OTF, and BFTTF (which includes the NX system fonts).
GraphicsCommandBuffer.h, GraphicsCommandBuffer.cpp A lightweight wrapper for an nvn::CommandBuffer.
GraphicsContext.h, GraphicsContext.cpp Functions which initializes fundamental display device-level graphics context and bootstraps NVN Device.
GraphicsMemoryPool.h, GraphicsMemoryPool.cpp A lightweight wrapper for an nvn::MemoryPool.
GraphicsRenderState.h, GraphicsRenderState.cpp A simple convenience data structure that combines commonly used nvn render state objects.
GraphicsShader.h, GraphicsShader.cpp A lightweight wrapper for an nvn::Program object.
GraphicsStructures.h Contains commonly used data structures to represent graphical data such as vectors and colors.
GraphicsSystem.h, GraphicsSystem.cpp A simple NVN-based rendering framework suitable for demo applications.
GraphicsTextureManager.h, GraphicsTextureManager.cpp A simple class to help setup prerequisite NVN resources for rendering textured primitives, manage and register application-defined texture and samplers.
GraphicsUtilities.h, GraphicsUtilities.cpp Contains a number of utility functions to do a variety of graphics rendering-related tasks such as setting up vertex stream/attribute data, vectors and transforms, and time measurement.
SceneBaseClass.h The base class of demo scenes.
SimplePrimitiveRenderer.h, SimplePrimitiveRenderer.cpp A primitive renderer with support for a simple local illumination shading model.
SimpleTextRenderer.h, SimpleTextRenderer.cpp A simple class that can be used to display text using Fonts to the screen.
SimpleFramework.h, SimpleFramework.cpp Contains framework code that is used throughout all the NvnSimple graphics rendering applications.
TimeUtilities.h, TimeUtilities.cpp Contains Time measurement-related helper functions.
UserInput.h, UserInput.cpp Contains routines for utilizing user input.
