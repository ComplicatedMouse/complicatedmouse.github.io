nn::os::EventType Struct Reference

This structure defines an event object. More...

#include <nn/os/os_EventTypes.h>

Detailed Description

This structure defines an event object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeEvent()
  • FinalizeEvent()
  • SignalEvent()
  • WaitEvent()
  • TryWaitEvent()
  • TimedWaitEvent()
  • ClearEvent()
  • InitializeMultiWaitHolder(MultiWaitHolderType *multiWaitHolder, EventType *event)

To use these functions, include nn/os.h.

Also see the description of the nn:os::Event utility class.
