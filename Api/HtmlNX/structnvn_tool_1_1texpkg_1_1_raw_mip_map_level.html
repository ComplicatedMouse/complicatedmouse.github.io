Detailed Description

Reference to the data in one texture mipmap level in an image.

A reference to the data in one texture mipmap level in an image, which is a simple W x H x D image. The offset into the data for each 3D depth slice may be queried, taking into account alignment requirements for different formats. This structure does NOT own any memory. It is merely a weak pointer struct with some helper functions.
