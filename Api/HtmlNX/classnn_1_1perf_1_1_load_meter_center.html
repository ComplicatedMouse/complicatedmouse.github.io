Detailed Description

The class that manages multiple CpuMeter and GpuMeter objects, and profiles the measurement results.

Details
Profiles using the measurement results stored for all meters.

Initializes LoadMeterCenter.

Parameters
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
[in]pMemoryPointer to the initialization buffer.
[in]memorySizeBuffer size, in bytes.
[in]pMemoryPoolPointer to the memory pool to which the GPU timestamp is written.
[in]memoryPoolOffsetOffset to the memory pool to which the GPU timestamp is written.
[in]memoryPoolSizeSize of the memory pool to which the GPU timestamp is written.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The number of measurement sections for GpuMeter is set as a value of 1 or greater in info.

Details
Creates the following meters.

  • CpuMeter objects for frame metrics: 1.
  • CpuMeter objects for each core metric: The number specified by LoadMeterCenterInfo.
  • CpuMeter objects for user metrics: The number specified in LoadMeterCenterInfo.
  • GpuMeter objects for GPU metrics: 1. All of the other CpuMeter objects and GpuMeter are attached to the CpuMeter object for frame metrics.

◆ EndFrameMeasure()

static void nn::perf::LoadMeterCenter::EndFrameMeasure ( bool  isClearProfile)
static

Records the frame end time using the CpuMeter object for frame metrics.

Parameters
[in]isClearProfileWhether to initialize the profiling results.

Details
Call this function when the frame ends. Performs the following operations internally.

  • Finishes measuring overall frames.
  • Initializes profiling results (when isClearProfile == true).
  • Switches the tallies and the buffers for the measurement results for each CpuMeter object and GpuMeter.
  • Tallies the measurement results using LoadMeterCenter. Note: Alternatively, you can use the Clear() function to initialize the profiling results at any time.
