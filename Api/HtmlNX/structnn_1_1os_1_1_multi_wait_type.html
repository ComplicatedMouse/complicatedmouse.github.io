nn::os::MultiWaitType Struct Reference

#include <nn/os/os_MultipleWaitTypes.h>

Detailed Description

This structure defines a multiwait object.

Details
Directly altering member variables is prohibited.
Use the main structure through the following functions.

  • InitializeMultiWait()
  • FinalizeMultiWait()
  • WaitAny()
  • TryWaitAny()
  • TimedWaitAny()
  • LinkMultiWaitHolder()

To use these functions, include nn/os.h.

Also see the description of the nn::os::MultipleWait utility class.

Examples
AudioEchoback/AudioEchoback.cpp, LdnBasic/LdnAccessPoint/LdnAccessPoint.cpp, LdnBasic/LdnStation/LdnStation.cpp, and OsMultiWait/OsMultiWait.cpp.
