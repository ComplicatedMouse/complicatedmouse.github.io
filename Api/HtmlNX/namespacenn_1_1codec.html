Decodes using the AacDecoder instance.

Parameters
[in]pDecoderThe AacDecoderType structure.
[out]pOutConsumedThe size of the data used at decoding.
[out]pOutSampleCountThe numbers of output samples (per channel).
[out]outputBufferThe output PCM buffer.
[in]outputSizeThe size of outputBuffer.
[in]inputBufferThe input data buffer.
[in]inputSizeThe size of inputBuffer.
Returns
Returns the execution result.
Precondition
  • pDecoder != nullptr.
  • *pDecoder is initialized.
  • pOutConsumed != nullptr.
  • pOutSampleCount != nullptr.
  • outputBuffer != nullptr.
  • inputBuffer != nullptr.

Details
When there are two input data channels, output data is written to outputBuffer in left-right order. A processing failure could result in the content of outputBuffer being modified.

The maximum size of the input data is equal to nn::codec::AacPayloadSizePerChannel multiplied by the number of channels.
