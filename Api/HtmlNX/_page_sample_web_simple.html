A simple sample program that uses the nn::web library to start the web applet and open a webpage.

Overview

A simple sample program that uses the nn::web library to start the web applet and open a webpage.

File Structure

This sample program is located in Samples/Sources/Applications/WebSimple/ .

Required Environment

This sample program is only supported in the NX development hardware environment.

How to Use

Operation instructions are displayed on the screen.

Input Operation
A Button Opens the specified webpage.
B Button Opens a webpage with the settings to exit the web applet automatically after it reaches the specified page.
X Button Specifies the startup screen and the webpage background and opens a webpage.
Y Button Starts the web applet using dedicated video startup and plays video content.
+ Button Ends the sample application.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

131 void UseCallbackUrl() NN_NOEXCEPT
132 {
133  // URL of the webpage that is opened when starting the web applet.
134  static const char* requestUrl =
135  "https://nintendo-sdk-sample.cdn.nintendo.net/page/callback_url_sample/index.html";
136  // Callback URL.
137  static const char* callbackUrl =
138  "https://nintendo-sdk-sample.cdn.nintendo.net/page/callback_url_sample/callback_url/index.html";
139  // Callback available URL.
140  static const char* callbackableUrl =
141  "https://nintendo-sdk-sample.cdn.nintendo.net/page/callback_url_sample/callbackable_url/index.html";
142 
143  nn::web::ShowWebPageArg showWebPageArg(requestUrl);
144  // If a callback URL is specified, the webpage stops displaying after the web applet reaches this URL.
145  showWebPageArg.SetCallbackUrl(callbackUrl);
146  // If a callback available URL is specified, the webpage stops displaying only end if the web applet reaches the callback URL from this URL.
147  showWebPageArg.SetCallbackableUrl(callbackableUrl);
148 
149  nn::web::WebPageReturnValue webPageReturnValue;
150  const nn::Result result = nn::web::ShowWebPage(&webPageReturnValue, showWebPageArg);
151  if (result.IsSuccess())
152  {
153 
154  nn::web::WebExitReason exitReason = webPageReturnValue.GetWebExitReason();
155  NN_LOG("Exit Reason %d\n", exitReason);
156  if (exitReason == nn::web::WebExitReason_CallbackUrlReached)
157  {
158  if (webPageReturnValue.GetLastUrlSize() > 0)
159  {
160  NN_LOG("LastUrl %s\n", webPageReturnValue.GetLastUrl());
161  }
162  }
163  }
164  else
165  {
166  NN_LOG("Cancel\n");
167  }
168 }

211 bool Update() NN_NOEXCEPT

Sample Program Description

The preceding sample program has the following main steps.

  • Initialize and configure the background color settings for rendering.
  • Main loop.
    • Update key input.
    • Process how characters are displayed.
    • Start the offline web applet and display the webpage if there is specific key input.

Specifying Accessible URLs

You must specify the URLs that can be opened in the web applet in an accessible-urls.txt text file and include them as additional information with the ROM. In this sample, only the Nintendo website and the website containing the sample webpages are accessible.
