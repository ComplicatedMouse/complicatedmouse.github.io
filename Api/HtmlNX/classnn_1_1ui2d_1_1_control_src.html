Detailed Description

The class that stores required information when creating a control.

Details
This class is passed as an argument when ControlCreator::CreateControl() is called and provides the information required to create a control.

The ControlSrc class provides three kinds of information: control name information, functional animation information, and functional pane information.

The control name is the name of the control selected in the Control Type subwindow of the Parts Settings when a layout is being created with the layout library. This name is not directly input using Layout Editor, but rather selected from a list of names defined in the control definitions file.

Functional animation specifies the animation tag names of the animations assigned for control features (such as movement when a button has been pressed).

Functional animations are determined ahead of time (for example, "the first one for when a button is selected, and the second one for when a button is pressed" ) and written in that order to the control definitions file. The index of the functional animation determines which feature the animation data has been specified for.

Functional panes are used basically the same way as functional animations, but you specify a pane rather than animation data. It is used, for example, for panes that specify the button's touch-detection area.

Note that both functional animation and functional pane can be set as Not Specified by Layout Editor. In this case, NULL is returned by GetFunctionalAnimName and GetFunctionalPaneName.
