Detailed Description

The class for loading files created by the layout archiver.

Performs operations on files created by the layout archiver.

◆ ReadEntry()

int nn::ui2d::ArcExtractor::ReadEntry ( int *  pHandle,
ArcEntry  pEntries[],
int  count 
) const

Reads an entry.

  • The filename is assigned to entry only if that filename is included in the archive.
  • If the filename is not included in the archive, the hash is converted to a string and that string is assigned instead of the filename (when the -n option is specified at conversion time).
    Parameters
    [in,out]pHandleEntry ID where you begin reading. Initialize to 0 if reading from the start. Pass this argument to the next ReadEntry object for continuous reading.
    [out]pEntriesLocation of the information read by the function.
    [in]countThe number to read.
    Returns
    If reading is successful, returns the number of sets of entry information actually read.
