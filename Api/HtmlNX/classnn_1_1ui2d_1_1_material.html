Copy constructor.

Details
Creates a copied material from the material specified by the argument. If this constructor is used to copy a material belonging to a different layout, the capture texture reference information is not updated and is in a state referencing the capture texture maintained by the layout at initialization. To copy the material as a material for a different layout, use the copy constructor that specifies MaterialCopyContext.

Note that the material is copied based on its state at that time, and not on its original state in the layout data. In other words, if the copying occurs after an animation has played, the pane is copied in that played state.

In addition, the state is with no animations bound.

Parameters
[in]materialThe material to copy.
[in]pDeviceDevice.

Details
Creates a copied material from the material specified by the argument. If appropriate parameters are set for copyContext, changes the capture texture reference based on these parameters.

Note that the material is copied based on its state at that time, and not on its original state in the layout data. In other words, if the copying occurs after an animation has played, the pane is copied in that played state.

In addition, the state is with no animations bound.

Creates a copied material from the material specified by the argument. If appropriate parameters are set for copyContext, changes the capture texture reference based on these parameters.

Note that the material is copied based on its state at that time, and not on its original state in the layout data. In other words, if the copying occurs after an animation has played, the pane is copied in that played state.

In addition, the state is with no animations bound.

Parameters
[in]materialThe material to copy.
[in]copyContextAdditional information for copying.
