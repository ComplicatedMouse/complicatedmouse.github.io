Describes a simple sample program including a demonstration of how to initialize OpenGL.

Overview

This simple sample demonstrates how to use gll to load OpenGL commands, how to call the loaded features, and how to determine whether certain features are supported. In this sample, several features are loaded and then used to display a simple screen.

File Structure

This sample program is located in Samples/Sources/Applications/GllSimple/ .

Required Environment

You must be able to use the screen display.

How to Use

Tap the screen or click with the mouse to end the sample.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program starts by using gll to load OpenGL commands. Then it uses gll to determine whether certain features are supported, and takes different actions depending on the results of this determination. Finally, it uses the loaded OpenGL features to perform a simple rendering process.

This sample program has the following flow.

  • Get the display and initialize the layers.
  • Create the OpenGL context.
  • Initialize OpenGL.
  • Execute different processes depending on whether certain features are supported.
  • Render the results to the screen.
