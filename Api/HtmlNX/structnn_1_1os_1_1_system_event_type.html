nn::os::SystemEventType Struct Reference

#include <nn/os/os_SystemEventTypes.h>

Detailed Description

This structure defines a system event object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • DestroySystemEvent()
  • WaitSystemEvent()
  • TryWaitSystemEvent()
  • TimedWaitSystemEvent()
  • ClearSystemEvent()
  • InitializeMultiWaitHolder(MultiWaitHolderType* multiWaitHolder, SystemEventType* event) To use these functions, include nn/os/os_SystemEvent.h.

Also see the description of the SystemEvent utility class.

Examples
FriendsFriendInvitation/FriendsFriendInvitation.cpp, LdnBasic/LdnAccessPoint/LdnAccessPoint.cpp, and LdnBasic/LdnStation/LdnStation.cpp.
