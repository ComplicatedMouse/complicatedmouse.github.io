Detailed Description

Information for Resource creation.

Details
This is the accessor for the information needed to create Resource.

◆ SetDefault()

ResourceInfo& nn::mii::ResourceInfo::SetDefault ( )

Sets the default values.

Details
The default values are as follows. ResourceInfoData::textureQuality = TextureQuality_Middle. ResourceInfoData::shapeQuality = ShapeQuality_Middle. ResourceInfoData::read = ResourceReadType_OnMemory. ResourceInfoData::gamma = GammaType_Linear.

Returns
Returns a reference to self.
