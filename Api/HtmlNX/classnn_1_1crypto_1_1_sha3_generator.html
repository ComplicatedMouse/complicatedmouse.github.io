Detailed Description

template<size_t OutHashSize>
class nn::crypto::Sha3Generator< OutHashSize >

Class for calculating a SHA-3 hash.

Template Parameters
OutHashSizeSize of the output hash, in bytes.

Details
This template class takes the output hash size as its template argument. Use the Sha3224Generator, Sha3256Generator, Sha3384Generator, or Sha3512Generator type in your actual code.

This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state
  • Ready.
  • Complete.

Parameters

Precondition
  • The instance is in a ready state.
Postcondition

Details

For SHA-3, use Update() to apply all of the entered information to the hash context.
