Overview

This sample implements the drawing of some animated triangles in world space coordinates to introduces the uniform data setup. Its shader code takes in model view and projection matrices as uniform data as a means of applying transformations to the triangles.

File Structure

The sample's source file and Visual Studio solutions can be found at Samples/Sources/Applications/NvnSimple02WorldSpaceTriangles

Description

This application draws two animated triangles in world space coordinates. Compared with ScreenSpaceTriangles, the main addition that this class makes is the usage of uniform blocks. Illustrated is how to lookup uniform reflection information via the GraphicsShader class, how to prepare blocks of memory to contain uniform data, and how to bind those blocks of uniform data when generating GPU rendering commands.
