Detailed Description

Specifies whether the depth values of rasterized polygon primitives should be offset.

When rasterizing polygon primitives, the GPU can offset depth values for the primitive to primitives with similar depth or to ensure that a wireframe outline of a filled primitive is always on top of the filled primitive. Offset can be enabled separately for polygons drawn in point, line, or filled modes.
