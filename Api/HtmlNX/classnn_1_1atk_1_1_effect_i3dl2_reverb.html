Detailed Description

Interactive 3D Audio Level 2 (I3DL2) reverberation for the specified number of channels.

Details
The EffectI3dl2Reverb class wraps the API resources for nn::audio::I3dl2ReverbType. This class must be retained after EffectI3dl2Reverb has been appended by SoundSystem::AppendEffect and until the effect has been removed by either SoundSystem::ClearEffect or SoundSystem::RemoveEffect.

When allocating this type of memory, it must be aligned to a nn::atk::EffectBase::EffectAlignment byte boundary.

◆ SetChannelMode()

bool nn::atk::EffectI3dl2Reverb::SetChannelMode ( ChannelMode  mode)

Sets the maximum number of I3DL2 reverb effect channels.

Details
If a number of channels has been specified by this function, the I3DL2 reverb effect is applied to that set number of channels, in order from the lowest value of nn::atk::ChannelIndex. Use SetChannelIndex instead of this function if you also want to set the channels to which I3DL2 reverb is applied. Either 1, 2, 4, or 6 channels can be used as input channels. The default is ChannelMode_6ch.

Parameters
[in]modeThe enumerated type for specifying the number of channels.
Returns
Returns true if a maximum number of channels has been successfully set.
See also

Details
The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. In addition, the function will fail if the number of input channels is not 1, 2, 4, or 6. If this function is used together with SetChannelIndex, the get settings that are applied are those of the function called last.

◆ SetChannelIndex()

bool nn::atk::EffectI3dl2Reverb::SetChannelIndex ( const ChannelIndex *  pChannelIndices,
ChannelMode  channelMode 
)

Configures the I3DL2 reverb effect channel settings.

Details
You can specify the channels for applying effects and the number of channels for the effect. If you are using this function you do not need to set the number of channels with SetChannelMode. Either 1, 2, 4, or 6 channels can be used as input channels.

The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. In addition, the function will fail if the number of input channels is not 1, 2, 4, or 6. If SetChannelMode is used together with this function, the settings of the function called later are applied.

The I3DL2 reverb effect requires channel settings to be configured in a fixed order. For more information about channel order, see nn::audio::SetI3dl2ReverbInputOutput.

Parameters
[in]pChannelIndicesThe channel that applies the effects. A ChannelIndex-type array. The number of pChannelIndices elements must be greater than the number of channels set in channelMode.
[in]channelModeThe enumerated type that indicates the number of channels to which effects are applied.
Returns
Returns true when channel configuration is successful.
Precondition
  • pChannelIndices != nullptr.
  • (Each value in the pChannelIndices array) < nn::atk::ChannelIndex_Count.
  • Two or more of the same channel number do not exist in each of the pChannelIndices array values.

Enables or disables I3DL2 reverberation.

The default is false.

Before calling either SoundSystem::ClearEffect or SoundSystem::RemoveEffect, first use this function to disable I3DL2 reverberation and then call IsRemovable to confirm that the effect can be removed. Note that effects do not immediately become deletable after they are disabled using this function.
