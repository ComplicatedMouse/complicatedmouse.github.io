Detailed Description

This allocator is for breaking memory in a large memory pool into smaller units for use.

Details
Management-region memory is allocated and freed with a callback function specified by the user.

Member Enumeration Documentation

◆ AllocationMode

enum nn::gfx::util::MemoryPoolAllocator::AllocationMode

This mode is for allocation and release.

Details
Specifies the behavior when allocating and releasing segments. By default, a value of AllocationMode_NoOption is specified.

Enumerator
AllocationMode_NoOption 

If a segment longer than 256 is released, the region is combined with adjacent empty regions.

AllocationMode_CoalesceAdjacentArea 

Constantly combine with adjacent empty regions upon release.

While the running efficiency is lower than AllocationMode_NoOption, it can suppress fragmentation within the allocator.
