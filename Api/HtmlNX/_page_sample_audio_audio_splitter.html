This sample program demonstrates how to use nn::audio::SplitterType.

Overview

This sample program uses nn::audio::SplitterType to construct an audio graph.

File Structure

This sample program is located in Samples/Sources/Applications/AudioSplitter/ .

Required Environment

You must be able to use audio output.

How to Use

When the sample program is executed, the BGM and sound effects will be played back automatically. You can use input from the keyboard and DebugPad during playback to manipulate the playing BGM, sound effects, and sine waves.

Input Operation
Left/Right Buttons Adjusts the mix volume for the splitter output destinations.
A Button Changes the splitter output destinations.
[B] Releases and reconnects the splitter.
[X] Prints a description.
START or SPACEBAR Exits the program.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program describes the basic operations for complicated routing using nn::audio::SplitterType to route a single output source to multiple output destinations.

The sample program contains the following descriptive sections. 0. Sample Program Overview Presents the audio graph that is constructed by this sample program. This construct is the basis on which AudioRendererParameter and the other parameters are configured.

  1. Preparing to Use SplitterType Objects Describes the AudioRendererParameter settings. To use SplitterType objects, you must set values for splitterCount and splitterSendChannelCount.
  2. Connecting the SplitterType Object Constructs the audio graph depicted in section 0. Describes how to connect the SplitterType object and other basic procedures.

The following three sections describe how to change various SplitterType settings.

  1. Switching the SplitterType Connection Targets
  2. Adjusting the SplitterType Mix Volume
  3. Releasing the SplitterType Object
