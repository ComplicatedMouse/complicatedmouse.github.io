Parameters
[out]pOutFontDataSizeThe data size.
[in]pFontNameThe font name.
[in]resTypeThe resource type.
[in]pUserDataThe user data to pass to the callback function.
Returns
Returns a pointer to the font data. This function returns NULL if there is no data.

You can load fonts set in the BFTTF font in the FCPX file, even if the return value data in the nn::pl::GetSharedFontAddress() is returned as is.

Details
It is necessary to initialize the texture cache before calling this function.

[in]pRegisterTextureViewSlotThe texture view slot used to initialize bitmap fonts in the composite font.
[in]pUserDataForDescriptorSlotAllocatorUser data passed to DescriptorSlotAllocator.
[in]pTextureCacheThe texture cache used for initializing the scalable fonts in the composite font.
[in]fontFaceHeadThe start of the font face. Pass the return value of nn::ui2d::ComplexFontHelper::SetupTextureCacheArg().
