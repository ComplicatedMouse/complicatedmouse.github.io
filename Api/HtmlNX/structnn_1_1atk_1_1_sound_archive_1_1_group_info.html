Detailed Description

The structure for group settings information.

Details
Defined by the sound archive. The structure for getting group information.

groupFileSize contains the size of the group file stored in the sound archive.

If the group Output Method in SoundMaker is set to Embed, this size is equivalent to the required memory size when the relevant group is loaded using SoundDataManager::LoadData.

However, if Output Method is set to Link, the size differs from the memory size required when using LoadData because the group file contains only the item information for the relevant group and not any actual data.
