◆ USR_SRGB_GAMMA_RESOURCE

#define USR_SRGB_GAMMA_RESOURCE

The gamma of the resource to use.

Commented out. Render the icon to a texture using linear gamma. Use SrgbWrite to write to the texture. Valid. Render the icon to a texture using SRGB gamma.
