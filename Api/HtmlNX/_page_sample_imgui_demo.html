Sample demonstrating use of the nns::gfx::dbgui library.

Overview

This sample demonstrates how to create a simple ImGui GUI while using the dbgui library.

File Structure

This sample program is located in Samples/Sources/Applications/ImguiDemo/ .

Required Environment

Nothing specific.

How to Use

The following input devices are supported. DebugPad (development hardware only) Handheld (development hardware only) JoyDual (development hardware only) Keyboard (on the development hardware, a USB keyboard or the PC keyboard with Input Director) Mouse (on the development hardware, a USB mouse or the PC mouse with Input Director) TouchScreen (Handheld mode only for development hardware)

Text input is supported with the USB keyboard. On the development hardware, the software keyboard is also supported. This sample uses a basic implementation for Japanese text on the Windows operating system.

Debug Pad/DualJoy Controls: D-Pad Up/Down/Left/Right: Navigate and change values. A Button: Activate item, press and hold to change a value, enter a child window, etc. B Button: Close pop-up menus, remove from child, clear selection, etc. Y Button (press): Access menu, collapse window, close window, configure window, etc. Y Button (hold) + D-Pad: Change window size. Y Button (hold) + Stick: Move window. Y Button (hold) + ZL/LR Button: Change window focus (similar to ALT+TAB). X Button: Enter text. L/R button (hold): Manually change the speed. Control stick: Scroll.

On a debug pad, you can switch the mouse operation by simultaneously pressing + Button and the L Button. Stick: Move mouse pointer. A Button + D-Pad Right: Left-click mouse. B Button + D-Pad Down: Right-click mouse. X Button + D-Pad Down: Middle-click mouse.

Press the + Button on the DualJoy to activate a mouse pointer controlled via the six-axis sensor on the right Joy-Con. A Button + D-Pad Right: Left-click mouse. B Button + D-Pad Down: Right-click mouse. X Button + D-Pad Down: Middle-click mouse.

Press the R Button and the + Button or the Start Button at the same time to enable/disable the debugging GUI.
