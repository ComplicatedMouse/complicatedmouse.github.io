Detailed Description

Rendering parameter class.

Details

◆ ModulateType

enum nn::mii::DrawParam::ModulateType

Modulation type.

Details
Enumerated type indicating the modulation method when rendering. The various emulation shader codes are as follows. ModulateType_Constant

color = const[0].rgba;

ModulateType_Texture

color = texture(s_tex,uv);

ModulateType_RgbLayered

tex = texture(s_tex,uv);
color = vec4(tex.r * const[0].rgb
+ tex.g * const[1].rgb
+ tex.b * const[2].rgb ,tex.a);

ModulateType_Alpha

tex = texture(s_tex,uv);
color = vec4(const[0].rgb,tex.r);

ModulateType_AlphaOpa

tex = texture(s_tex,uv);
color = vec4(tex.r * const[0].rgb,1.0f);

ModulateType_Glass The translucent appearance for eyeglasses will vary based on the gamma type. Processing to adjust the alpha has been added for the linear gamma appearance to approach that of the sRGB gamma.

tex = texture(s_tex,uv);
color = vec4(tex.r * const[0].rgb, pow(tex.g, const[1].g));

ModulateType_IconBody The constant color for this modulation mode does not include any color. Process after getting a favorite color.

color = favoriteColor.rgba;
Enumerator
ModulateType_Constant 

0: constant color rendering (enabled attributes: Position,Normal)

ModulateType_Texture 

1: texture rendering (enabled attributes: Position,Normal,Uv)

ModulateType_RgbLayered 

2: modulation rendering for the various RGB channels (enabled attributes: Position,Normal,Uv)

ModulateType_Alpha 

3: modulation rendering for the A channel with texture alpha output (enabled attributes: Position,Normal,Uv)

ModulateType_AlphaOpa 

4: modulation rendering for the A channel with a fixed alpha (enabled attributes: Position,Normal,Uv)

ModulateType_Glass 

5: modulation rendering for the LA channel with output after adjusting the texture alpha (enabled attributes: Position,Normal,Uv)

ModulateType_IconBody 

6: constant (favorite) color rendering (enabled attributes: Position,Normal)

ModulateType_End 

Terminator (7)

ModulateType_Min 

The minimum value (0)

ModulateType_Max 

The maximum value (6)

◆ ParamFormat

const ::nn::gfx::AttributeFormat nn::mii::DrawParam::ParamFormat = ::nn::gfx::AttributeFormat_8_8_8_8_Unorm
static

Parameter vertex attribute format.

Details
Parameter vertex attribute elements. r = Ratio of Blinn specular reflection to anisotropic specular reflection (1.0= Blinn, 0.0=anisotropic) g = Proportion of specular reflection (0.0=No specular reflection, 1.0=Full specular reflection) b = unused a = Rim light width (0.0 to 1.0)

Examples
MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, and MiiShaderExample/MiiShaderExample.cpp.
