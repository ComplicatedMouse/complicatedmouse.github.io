nn::ldn::SessionId Struct Reference

Identifiers instantiated per session. More...

#include <nn/ldn/ldn_NetworkId.h>

Detailed Description

Identifiers instantiated per session.

Details
This is a 128-bit identifier that is randomly generated for each session. It is the equivalent of the latter half of NetworkId.

Specifically, when the access point uses the CreateNetwork function to build a network, a SessionId is automatically assigned by the system. Because a random SessionId is assigned each time a network is created, it is not normally possible to reuse the same NetworkId or SessionId.
