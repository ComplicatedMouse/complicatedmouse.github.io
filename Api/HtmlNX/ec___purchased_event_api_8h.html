Functions for Knowing About Shop Purchases

This group of functions is used for detecting purchases made with the shop app on NX devices and purchases made on the Nintendo website.

These functions can detect purchases even when they are made on a Switch system that has not been registered as the primary console.

These functions do not support the detection of downloadable content purchases. To see when you can begin using a purchased downloadable content item, call the nn::aoc::GetAddOnContentListChangedEvent() function.
