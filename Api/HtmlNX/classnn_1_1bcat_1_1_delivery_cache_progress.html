int64_t nn::bcat::DeliveryCacheProgress::GetWholeTotal ( ) const

Gets the total size of the download processing.

Returns
Total size of the download processing.

Details
The size obtained by this function is not the total size of all files being distributed but the total size of all new and updated files.

Returns a value if the progress status is one of the following.

  • DeliveryCacheProgressStatus_Download
  • DeliveryCacheProgressStatus_Commit
  • DeliveryCacheProgressStatus_Done

Call the DeliveryCacheProgress::Update function to update the status and value.
