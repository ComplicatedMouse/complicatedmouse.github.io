Overview

This section describes the sample program that uses dynamic modules.

For more information, see the Ro Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/RoSimple/RoStaticApplication/ .

Required Environment

This program does not run in a PC environment.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

First build Samples/Sources/Libraries/ro/RoDynamicModule/ , and then build and execute this sample program.
