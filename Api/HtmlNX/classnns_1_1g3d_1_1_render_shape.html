bool nns::g3d::RenderShape::Initialize ( nn::gfx::Device *  pDevice,
nn::g3d::ResShape *  pResShape,
RenderMaterial *  pRenderMaterial 
)

Initializes RenderShape.

Parameters
[in]pDevicePointer to the device.
[in]pResShapePointer to the target shape.
[in]pRenderMaterialPointer to the RenderMaterial that manages the rendering information for the material referenced by the shape.
Precondition
  • RenderShape is not initialized.
Postcondition
  • RenderShape is initialized.
Returns
Returns true if initialization was successful, or false otherwise.

Details
After initialization, the following vertex state is applied during shape rendering.

The nn::gfx::VertexState object that is initialized after investigating the pResShadingModel vertex attribute information and the vertex attribute information of the material referenced by pResShapep, and vertex attribute information corresponding to the material is adopted as the respective vertex attributes.

If there is no vertex attribute information corresponding to the material, compare the vertex attributes and assign the first vertex attribute that matches.

Configures rendering settings in the command buffer.

Parameters
[out]pCommandBufferThe pointer to nn::gfx::CommandBuffer.
[in]pRenderUnitObjPointer to the RenderUnitObj render target.
[in]viewIndexThe view index.
[in]bufferIndexThe buffer index.

Details
Sets the command buffer for configuring settings for shape rendering. Specifically, the following states and objects are set.

  • VertexBuffer
  • VertexState
  • BlendState
  • RasterizerState
  • DepthStencilState
  • ResShaderProgram
  • Sampler
  • Uniform Block
  • Shader Storage Block
