Detailed Description

Namespace for the NGC library.

Details

Feature Overview

NGC is short for "not good word check," the check for profanity.

The NGC library is used to check for profanity. The library provides features for taking appropriate measures when strings entered by users are determined to be offensive or sexual in nature. The NGC library can check UTF-8 encoded strings.

Enumeration Type Documentation

◆ ProfanityFilterPatternList

enum nn::ngc::ProfanityFilterPatternList

This enumerator lists profanity string patterns for each language.

Details
Each list contains bad words that have been registered for testing purposes to confirm that the library has been incorporated correctly in your application. For more information about which test-use bad words are contained in each list, see the description of each enumerator. "badworda" is registered as an exact-match test word that is common to all lists, while "warumoshi", "ワルモシ", and "qxzq" are registered as partial-match test words that are common to all lists. Only strings that exactly match these test-use bad words are detected.

◆ CheckProfanityWordsInTextForPlatformRegionChina()

nn::Result nn::ngc::CheckProfanityWordsInTextForPlatformRegionChina ( bool *  pCheckResult,
const char *  pText 
)

Checks whether there are any phrases in the specified text that would be problematic to display.


The check is conducted on a different standard than the standard used by the function provided by the nn::ngc::ProfanityFilter class.

Parameters
[out]pCheckResultThe pointer for getting whether the text has any illegal strings.
[in]pTextThe pointer to the text to check.
Up to nn::ngc::TextLengthMaxForPlatformRegionChina characters can be checked, and the text must be followed by a null terminator.
Store them using UTF-8 character encoding.
Returns
Returns a nn::Result that represents the process result.
  • nn::ResultSuccess The checking succeeded.
  • nn::ngc::ResultInvalidPointer An invalid pointer was specified.
  • nn::ngc::ResultInvalidSize The specified string has an invalid length.

Details
This function checks whether there are any illegal strings among the specified strings and stores the result in pCheckResult.

The processing by this function can be time-consuming (it blocks). The processing time can vary depending on the system version and the length of the strings passed to the function.
The frame rate can decline if this function is called from the main thread, so we recommend running it from another thread.

The pattern list dynamically references objects saved outside of the ngc library.
As a result, the checked content can vary depending on the version of the SDK or on development hardware that was initialized with that version.
Do not write code that assumes that results for specific strings will remain constant, from the perspective of compatibility.

This function might yield a different result than the function provided by the nn::ngc::ProfanityFilter class.

This function is valid if the platform region is China.

This function is thread-safe.

◆ MaskProfanityWordsInTextForPlatformRegionChina()

nn::Result nn::ngc::MaskProfanityWordsInTextForPlatformRegionChina ( char *  pText)

Checks whether there are any phrases in the specified text that would be problematic to display, and returns a string with those parts masked.


The check is conducted on a different standard than the standard used by the function provided by the nn::ngc::ProfanityFilter class.

Parameters
[in,out]pTextThe pointer to the text to check. Any illegal strings in the text are replaced with asterisks.
For this reason, the application must specify a writable buffer.
Up to nn::ngc::TextLengthMaxForPlatformRegionChina characters can be checked, and the text must be followed by a null terminator.
Store them using UTF-8 character encoding.
The number of bytes in the output string will never exceed the number of bytes in the input string.
Returns
Returns a nn::Result that represents the process result.
  • nn::ResultSuccess The checking succeeded.
  • nn::ngc::ResultInvalidPointer An invalid pointer was specified.
  • nn::ngc::ResultInvalidSize The specified string has an invalid length.

Details
This function checks whether there are any illegal strings among the specified strings and masks those illegal strings with asterisks.

If masking is not required, use the CheckProfanityWordsInText() function.

The processing by this function can be time-consuming (it blocks). The processing time can vary depending on the system version and the length of the strings passed to the function.
The frame rate can decline if this function is called from the main thread, so we recommend running it from another thread.

The pattern list dynamically references objects saved outside of the ngc library.
As a result, the checked content can vary depending on the version of the SDK or on development hardware that was initialized with that version.
Do not write code that assumes that results for specific strings will remain constant, from the perspective of compatibility.

This function might yield a different result than the function provided by the nn::ngc::ProfanityFilter class.

This function is valid if the platform region is China.

This function is thread-safe.

Variable Documentation

◆ WordLengthMax

const int nn::ngc::WordLengthMax = 64

The maximum number of characters per word, including the null terminating character, when single words are being checked.

This value represents the number of characters. To change this value into a number of bytes, use a function like nn::util::Strnlen to get the number of elements in the string.
