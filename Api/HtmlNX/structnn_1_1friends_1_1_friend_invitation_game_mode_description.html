Detailed Description

Game mode description for friend invitations.

Details
This structure is used to describe the game mode in the details screen of the friend invitation on the user page.

The user page selects the string to display based on language for the system settings of the user displaying the friend invitation. For this reason, we strongly recommend describing the applicable items in all languages described in the supported languages in the NMETA file.

The game mode description is encoded in UTF-8, and must be null-terminated. Specify an empty string for languages for which game mode descriptions will not be registered.

For more information about what happens with the user display when a language set to an empty string is selected, see the Nintendo Switch Online Play Guide.

Attention
Set the fixed string determined by the application for the description text. Strings freely entered by the user may not be included in the description.
