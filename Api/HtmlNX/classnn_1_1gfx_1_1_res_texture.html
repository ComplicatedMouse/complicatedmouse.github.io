◆ Initialize()

template<typename TTarget >
void nn::gfx::ResTexture::Initialize ( TDevice< TTarget > *  pDevice)

Initializes a texture.

Template Parameters
TTargetThe target low-level graphics API.
Parameters
[in]pDevicePointer to the device.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The texture is not initialized.
  • The texture resource file to which the texture belongs is initialized.
Postcondition
  • The texture is initialized.

Details
Performs processing equivalent to that of the Initialize function of the nn::gfx::Texture and nn::gfx::TextureView classes.

Examples
G3dDemo/Simple.cpp, GfxPrimitiveRenderer/GfxPrimitiveRenderer.cpp, GfxSimple/GfxSimple.cpp, HidControllerSequence/HidControllerSequence_Main.cpp, VfxBinarySegmentation/main.cpp, and VfxCustomShader/main.cpp.
