Detailed Description

The structure representing the input states for the NpadStyleJoyLeft play style modeled on holding the Npad Joy-Con (L) controller.

Details
Provides input states for the digital buttons (the directional buttons, L, ZL, ZR, SL, SR, and - Button) and the Left Stick, which comprise the input dealt with for Joy-Con (L) operations. Other input states are treated as noninputs, where the digital button is not depressed and the control stick is in a neutral position. There are no rules governing the input from the directional buttons. You can get the controller connection state using either attributes::IsConnected or attributes::IsLeftConnected.
