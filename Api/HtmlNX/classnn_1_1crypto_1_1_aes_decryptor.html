Detailed Description

template<size_t KeySize>
class nn::crypto::AesDecryptor< KeySize >

Class that implements decryption with an AES cipher.

Template Parameters
KeySizeSize in bytes.

Details
This class implements decryption using AES (Rijndael) cipher.

This class is a template class that has the key size as a template parameter. Use the AesDecryptor128, AesDecryptor192, or AesDecryptor256 type in your actual code.

Unless you are sufficiently well-versed in encryption and sufficiently understand what is going on, there is no need to directly use DecryptBlock() or GetRoundKey() from this class.

This class has the following states. The state at the time of instantiation is uninitialized.

  • Uninitialized state
  • Ready.

◆ GetRoundKey()

template<size_t KeySize>
const Bit8* nn::crypto::AesDecryptor< KeySize >::GetRoundKey ( ) const

Gets the round key.

Returns
Pointer to the round key data array.
Precondition
  • The instance is in a ready state.

Details
The block size for the round key, as a result of key expansion during Initialize(), is always calculated as one plus the number of prescribed rounds for each key size. This function can get a pointer to the start of that round key data array. The total round key size is defined with RoundKeySize.
