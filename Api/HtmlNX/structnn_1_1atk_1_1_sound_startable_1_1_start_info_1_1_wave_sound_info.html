Detailed Description

A structure relating to a wave sound.

Details
This structure is passed when wave sounds are played.

Specify the address to waveform data in waveAddress to play that data stored in memory instead of the waveform data set in the sound archive. If NULL is specified, the original waveform data as set in the sound archive is used.

◆ isContextCalculationSkipMode

bool nn::atk::SoundStartable::StartInfo::WaveSoundInfo::isContextCalculationSkipMode

The use of internal features is prohibited. If an offset was specified for ADPCM playback, playback will occur without first calculating the context used for playback.

This results in reducing the CPU processing when specifying offset, but playback will occur in the state where the waveform data in binary does not match the output results.
