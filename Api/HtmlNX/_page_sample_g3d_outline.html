This documentation describes the Outline sample program. Use the geometry shader to render the outline of a model.

Overview

This sample displays a model's outline using geometry shaders.

File Structure

This sample program is located in Samples/Sources/Applications/G3dDemo/ .

Required Environment

Nothing specific.

How to Use

Use input from the Joy-Con or DebugPad, or the keyboard and mouse when using a PC, to perform the following operations.

Input Operation
START Button, SPACEBAR, Joy-Con + Button: Returns to the menu.
Press the Right Stick or click the left mouse button: Camera rotation. When using the mouse, the camera rotates based on the direction and distance of the click from the center of the screen.

Important Information

Nothing specific.

Execution Procedure

Build the sample program. From the menu, select Outline, and then run it.

Description

This sample program begins by loading shader archives and model resources. It renders the model on the first pass, and then renders the outline on the second pass before displaying that outline.

This sample program has the following flow.

  • Load and initialize the shader archive.
  • Load and initialize the model resource, and initialize the texture.
  • Register a sampler in the descriptor pool.
  • Use nns::g3d to set up models.
  • Use nns::g3d to set up models for outline.
  • Begin the loop.
  • Execute the command list created in the previous frame.
  • Update each matrix.
  • Update models.
  • Generate the model rendering command for the first pass.
  • Generate the outline rendering command for the second pass.
  • Return to the start of the loop.
