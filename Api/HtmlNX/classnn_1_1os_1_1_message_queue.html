Detailed Description

Class for handling message queues.

Details
A message queue is a feature for establishing synchronization among threads by sending and receiving data.

The nn::os::MessageQueue class is a utility class that wraps calls to synchronization control functions that use nn::os::MessageQueueType objects.
The member functions provided by this class are implemented by issuing functions like nn::os::SendMessageQueue() that take the nn::os::MessageQueueType type as arguments.
