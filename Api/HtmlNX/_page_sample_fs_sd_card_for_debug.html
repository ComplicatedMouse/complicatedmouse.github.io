Description of a sample program showing how to use SD card file system operations for debugging.

Overview

The following describes a sample program that uses the SD card file system for debugging.

For more information about file and directory operations, see the File System Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/FsSdCardForDebug/ .

Required Environment

On the NX32 and NX64 platforms, the development hardware must have a FAT32-formatted SD card inserted in the SD card slot.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • Mounts the SD card using the nn::fs::MountSdCardForDebug function.
  • Demonstrates operations on a FAT32 file system on the SD card.

You can use the nn::fs::MountSdCardForDebug function to mount the SD card. To use this function, make sure that you include nn/fs/fs_Debug.h. Note that this function is only for debugging purposes and cannot be used in the retail product.

On the NX32 and NX64 platforms, this sample program creates a file named Sample_FsSdCardForDebug.txt in the root directory of an SD card. A FAT32-formatted SD card must be inserted in the development hardware.

On the Win32 and x64 platforms, the sample program creates the path sdcard/Sample_FsSdCardForDebug.txt in the execution directory.
