WordWrapConfig& nn::font::WordWrapConfig::SetLineCountMax ( int  value)

Sets the maximum number of lines.

Parameters
[in]valueSpecify the maximum line count, where zero (0) is the default.
Returns
Returns a reference to WordWrapConfig.

Details
When a maximum number of lines has been specified, stops when an upper limit on the number of lines is exceeded when processing the WordWrapping::CalculateWordWrapping() function.

Specify 0 for value to make the number of lines unlimited.

The % symbol following numbers specifies whether to prohibit cases where the start of the line wraps.

Details
