◆ TextCheckCallback

typedef TextCheckResult( * nn::swkbd::TextCheckCallback) (void *pOutDialogTextBuf, size_t *pOutDialogTextLengthSize, String *pStr)

The type for the callback function for checking the entry result string.


Check the entry result string stored in pStr and return nn::swkbd::TextCheckResult as necessary.
If you will be displaying a dialog box, specify a string for pOutDialogTextBuf to show in that dialog box.

Parameters
[out]pOutDialogTextBufThe buffer for the string to display in the dialog box.
This uses the buffer specified by nn::swkbd::ShowKeyboardArg::textCheckWorkBuf.
Enter using the character encoding specified by nn::swkbd::KeyboardConfig::isUseUtf8 and add a terminating null character to the end of the string.
Make sure that the number of entered characters matches the number set for pOutDialogTextLengthSize.
[out]pOutDialogTextLengthSizeThe number of characters to show in the dialog box.
Make sure this is no more than nn::swkbd::DialogTextMaxLength.
[in]pStrThe entry result string.
It is null-terminated.
String::bufSize stores the size of the buffer, including the null terminator, in bytes.

enum nn::swkbd::KeyboardMode

The software keyboard layout types.

Details
Specify the mode that corresponds to the content being entered so that you can use the keyboard with the appropriate layout.

Enumerator
KeyboardMode_LanguageSet1 

Language set 1 keyboard mode (European, American, and Japanese languages).

KeyboardMode_Numeric 

Numeric keyboard mode.

Details

  • A keyboard specially designed for the entry of numbers.
  • Predictive text input is always disabled.
KeyboardMode_ASCII 

ASCII keyboard mode.

Details

  • A keyboard on which only ASCII characters can be entered.
  • Predictive text input is always disabled.
KeyboardMode_LanguageSet1Latin 

Mode that prioritizes input of Latin characters from language set 1.

Details

  • The available keyboard types are the same as for LanguageSet1.
  • Regardless of which keyboard started the previous time, the keyboard that displays first, is always a keyboard for single-byte alphanumeric characters.
  • When a keyboard is closed, it is not saved as "the keyboard that started up last time."
KeyboardMode_Alphabet 

European and American languages keyboard mode.

Details

  • The available keyboard types are the keyboards from LanguageSet1, with Japanese omitted.
KeyboardMode_SimplifiedChinese 

Simplified Chinese keyboard mode.

Details

  • The Simplified Chinese keyboard and symbol keyboard are available.
  • The predictive text input feature is always enabled.
KeyboardMode_TraditionalChinese 

Traditional Chinese keyboard mode.

Details

  • The Traditional Chinese keyboard and symbol keyboard are available.
  • The predictive text input feature is always enabled.
KeyboardMode_Korean 

Korean keyboard mode.

Details

  • The keyboard and symbol keyboard are available.
  • The predictive text input feature is always enabled.
KeyboardMode_LanguageSet2 

Language set 2 keyboard mode (European, American, Japanese, Chinese, and Korean languages).

Details

  • In addition to the keyboards in LanguageSet1, Simplified Chinese, Traditional Chinese, and Korean keyboards are available.
  • The predictive text input feature is always enabled.
KeyboardMode_LanguageSet2Latin 

Mode that prioritizes input of Latin characters from language set 2.

Details

  • The selectable keyboards are the same as the keyboards for LanguageSet2.
  • Regardless of which keyboard started the previous time, the keyboard that displays first, is always a keyboard for single-byte alphanumeric characters.
  • When a keyboard is closed, it is not saved as "the keyboard that started up last time."
  • The predictive text input feature is always enabled.

◆ ShowKeyboard()

nn::Result nn::swkbd::ShowKeyboard ( String *  pOutResultString,
const ShowKeyboardArg &  showKeyboardArg 
)

Shows the software keyboard.

Parameters
[out]pOutResultStringThe entry result string. Byte size returns 0 when canceled.
[in]showKeyboardArgThis settings parameter is required when showing the software keyboard.
Returns
Returns the result of showing the software keyboard. Returns ResultSuccess if there were no particular issues.

Details
The callee of this function blocks until the software keyboard ends. In addition, even if this function is called while a system UI such as the error viewer is displayed, this function waits internally and moves out of wait status after the UI that was started first is no longer being used.

Some settings must be configured in advance, such as ShowKeyboardArg::workBuf and ShowKeyboardArg::workBufSize in showKeyboardArg.

Examples
SwkbdSimple/SwkbdSimple.cpp, and WebSocketChat/WebSocketChat.cpp.
