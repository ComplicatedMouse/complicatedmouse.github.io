Detailed Description

Class for handling system events.

Details
System events are synchronization objects used by SDK for event notification. For more information, see the related documentation.

The nn::os::SystemEvent class is a utility class that wraps calls to synchronization control functions that use nn::os::SystemEventType objects.
The member functions provided by this class are implemented as functions that call functions like nn::os::WaitSystemEvent() that take the nn::os::SystemEventType type for their arguments.
