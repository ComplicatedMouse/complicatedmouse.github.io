◆ invalidUidList

Uid nn::account::UserSelectionSettings::invalidUidList[UserCountMax]

The list of user Uid items to block from selection.

Details
Renders a user ID non-selectable, scanning the list from the top, until either an InvalidUid is encountered or all Uid values have been checked.

◆ isSkipEnabled

bool nn::account::UserSelectionSettings::isSkipEnabled

Whether to skip UI display when the user has enabled the option to omit user selection on that device.

Details
We recommend specifying that selection will be skipped from a location where users would expect it, and where there would be no sequence bugs. Calling the ShowUserSelector() function will fail when isSkipEnabled == true and any of the following conditions are met.

  • A valid Uid, equal to a value of at least 1, is set for invalidUidList.
  • additionalSelect == true

The default for DefaultUserSelectionSettings is false.

◆ isNetworkServiceAccountRequired

bool nn::account::UserSelectionSettings::isNetworkServiceAccountRequired

Determines whether a network service account is required.

Details
This function is used for selecting a user in scenes where a network service account is required. If true is specified, the IsNetworkServiceAccountAvailable() function returns true when called immediately after a user has been selected.

The default for DefaultUserSelectionSettings is false.

Examples
EcCatalog/Main.cpp.

◆ showSkipButton

bool nn::account::UserSelectionSettings::showSkipButton

Determines whether to display a button in the UI for skipping the user selection process.

Details
This function is used for scenes where user selection is optional and play can continue without making a selection. When the skip button is selected, nn::ResultSuccess returns with InvalidUid stored as the selected user ID.

The default for DefaultUserSelectionSettings is false.

◆ additionalSelect

bool nn::account::UserSelectionSettings::additionalSelect

When using a mode in the UI for selecting users other than the first user.

Details

  • The UI display will then change to one where selection of additional users is intended.
  • Opened users cannot be selected.

The default for DefaultUserSelectionSettings is false.

◆ isUnqualifiedUserSelectable

bool nn::account::UserSelectionSettings::isUnqualifiedUserSelectable

Determines whether restricted (unqualified) users are selectable.

Details
Set this flag to true to make restricted (unqualified) users selectable.

The default value of this flag in DefaultUserSelectionSettings is false.
