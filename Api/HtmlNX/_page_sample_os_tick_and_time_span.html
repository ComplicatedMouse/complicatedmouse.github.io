A sample program for the time management feature.

Overview

This sample shows the time management feature.

For information about using the time management feature, see the Time Management Feature Manual and the OS API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OsTickAndTimeSpan/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • SampleCodeFunction() is issued from nnMain().
  • SampleCodeFunction() first gets the tick cycle and displays it.
  • Next, it gets the current tick value.
  • The current tick value is converted to nn::TimeSpan format (in nanoseconds).
  • The tick value and the nn::TimeSpan value (converted to milliseconds) are displayed.
  • Wait one second.
  • This process is repeated 10 times.
  • Return to nnMain() and end.

SampleCodeFunction() gets the tick value in one-second intervals, converts that value to nn::TimeSpan format, and then displays the result after converting it to milliseconds.

On the other hand, in the nn::TimeSpan format, 1 count corresponds to one nanosecond.
