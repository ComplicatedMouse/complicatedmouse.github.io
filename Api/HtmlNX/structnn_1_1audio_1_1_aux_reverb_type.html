Detailed Description

Utility for reverb effects applying to nn::audio::AuxType.

Details
This utility processes reverb effects using the sample data obtained by nn::audio::AuxType as the input. By specifying the sample data obtained using the nn::audio::ReadAuxSendBuffer() function as the effect input, you can output a reverb effect that emulates the reverberation of sounds in a closed space. The reverb effect implemented this way is identical to the processing performed by nn::audio::ReverbType. However, this effect processing is performed on the CPU thread that called nn::audio::ProcessAuxReverb().

Directly altering member variables is prohibited. Use the main structure through the following functions.
