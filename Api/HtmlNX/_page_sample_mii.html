Mii Library Samples

The face library sample backgrounder.

Introduction

This page describes the face library sample programs. The face library features are in the nn::mii namespace.

Samples

Sample code is located in the directories starting with the prefix Mii under Samples/Sources/Applications/. The following table lists the samples.

Samples Content
Drawing Mii Characters Using Compressed Textures Sample This sample program shows how to draw Mii characters using compressed textures.
Displaying Mii Characters Retrieved From a Database Sample This sample program displays a Mii character retrieved from a database.
Dynamic Facial Expressions of Mii Characters Sample This sample program dynamically switches the facial expression of the Mii character.
Displaying Mii Characters Wearing Headwear Sample This sample program displays Mii characters wearing accessories, hats, and other headwear.
Mii Texture Mipmap Sample This sample program creates Faceline and Mask texture mipmaps.
MiiPremultipliedAlphaIcon This sample program demonstrates how to generate icons using pre-multiplied alpha values when blending with the background.
Mii Shader Sample This sample program renders Mii characters the same as the MiiEdit applet.
Simple Mii Character Rendering Sample This sample program renders Mii characters.
Simple Mii Icon Rendering Sample This sample program renders Mii icons.
Rendering Mii Icons With Bodies Sample This sample program renders Mii icons with bodies.
Rendering Variable Body Physiques of Mii Characters Sample This sample program renders Mii icons with certain body physiques.
Rendering Mii Characters Obtained From the System Sample This sample program renders Mii characters using Mii resources obtained from the system.
Preventing Transparent Pixel Color Bleeding With Mii Characters Sample This sample program shows a feature to prevent transparent pixel color bleeding due to linear interpolation.
