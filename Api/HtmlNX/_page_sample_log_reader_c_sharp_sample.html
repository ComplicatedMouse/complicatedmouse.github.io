This sample program demonstrates how to use the LogReader library in C#.

Overview

This section describes how to create host tools that use the LogReader library.

For more information about using the LogReader library, see the Features and API Reference sections of the AutoTestAssistToolKit documentation.

File Structure

This sample program is located in Samples/Sources/Tools/AutoTestAssistTools/CSharpSamples/LogReaderSample/ .

Required Environment

Target Manager must be installed ahead of time.

How to Use

To get the log from a development hardware target, enter the numerical value that corresponds to that device from the generated list of targets.

Important Information

You must connect to the target development hardware with TargetManager before you run this program. It is easier to check operations if the application generating the log has been started on the target development hardware when you run this program.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • Try to get and display 10 lines of the log.
  • Wait for the appearance of a log that matches the regular expression and display it.
  • Move to the line of the log that was added 10 seconds ago and display it.
  • Move to the newest line of the log and display it.
  • Try regular expression matching with a line prior to the current line of the log and display it.
