Sets the required memory.

Parameters
[in]pMemoryPointer to the memory to set.
[in]sizeSize of the memory to set, in bytes.
Precondition
  • If pMemory is not NULL, it is aligned to RequiredMemoryInfo_Alignment.
  • If pMemory is NULL, size == 0.
Postcondition
  • If pMemory is not NULL, the required memory is set.
  • If pMemory is NULL, the required memory is not set.

Initializes the graphics pipeline.

Parameters
[in]pDevicePointer to the device.
[in]infoInformation for initialization purposes.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The pipeline is not initialized.
  • If the value returned by GetRequiredMemorySize with the same info argument is larger than 0, the required memory is set to that size or greater.
Postcondition
  • The pipeline has been initialized as a graphics pipeline.
