Detailed Description

The class for creating a Mask texture.

Details
The class that stores the DrawParam necessary for creating a Mask texture.

Examples

nn::Result nn::mii::Mask::Initialize ( void *  pMemory,
size_t  memorySize,
nn::gfx::Device *  pDevice,

size_t  memoryPoolSize,
TemporaryBuffer *  pTemporaryBuffer,
const Resource &  resource,
const CharInfo &  charInfo,
int  expression,
bool  isVerticalFlip 
)

Initializes Mask.

Details
The default values after initialization are as follows.

Attention
Both pMemory and pMemoryPool must be retained until the call to Mask::Finalize().
resource may need to be retained until the call to Mask::Finalize() based on the Resource settings conditions.
You cannot reinitialize an instance of Mask that has been initialized already. Call this function after calling Finalize().
pTemporaryBuffer is only required when initializing. There is no problem with destroying the instance after initialization.
Pass the memory pool that was set in MemoryPoolProperty_CpuCached to pMemoryPool.
Set the vertical inverse flag, isVerticalFlip, in conjunction with the application coordinate system settings.
Parameters
[in]pMemoryThe memory that Mask will use.
[in]memorySizeThe size of pMemory.
[in]pDeviceThe device.
[in]pMemoryPoolThe memory pool Mask will use.
[in]memoryPoolOffsetThe offset to the memory pool Mask will use.
[in]memoryPoolSizeThe size of the memory pool Mask will use.
[in]pTemporaryBufferThe pointer to the temporary buffer.
[in]resourceThe Mask build resource.
[in]charInfoThe CharInfo of the Mii to be generated.
[in]expressionThe flag for the expression to use.
[in]isVerticalFlipThe up/down inversion flag.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && Resource::IsInitialized() && ! Mask::IsInitialized() && pTemporaryBuffer != NULL && The offset and size are normal. && charInfo is not corrupted. && Specifies the same ExpressionFlag as when calculating memory.
Postcondition
If successful, Mask::IsInitialized() = true.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
