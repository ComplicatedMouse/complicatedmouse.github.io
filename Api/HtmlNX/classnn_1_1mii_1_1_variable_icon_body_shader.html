Detailed Description

Shader for creating icons with certain body physiques.

Details
The shader class for creating icon textures with certain body physiques. Prepared as a utility to simplify the creation of icon textures with certain body physiques. It is okay to prepare a shader in the application and not use this class.

Examples
MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp.
