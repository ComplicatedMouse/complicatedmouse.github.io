Detailed Description

Shader for creating icons.

Details
The shader class for creating icon textures. Prepared as a utility to simplify the creation of icon textures. It is okay to prepare a shader in the application and not use this class.

Examples
MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Initializes.

Attention
pMemory must be retained until IconShader::Finalize() is called.
You cannot reinitialize an instance of IconShader that has been initialized already. Call this function after calling Finalize().
The initialized resShaderFile must be retained in the initialized state until IconShader::Finalize() is called.
Parameters
[in]pMemoryThe memory that IconShader will use.
[in]memorySizeSpecifies the size of pMemory.
[in]resShaderFileAn initialized shader file.
[in]shaderCodeTypeThe shader code type.
[in]pDeviceSpecifies the device.
[in]infoThe information required for IconShader initialization.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pShader != NULL.
Postcondition
If successful, IconShader::IsInitialized() = true.
