int nn::atk::ProfileReader::Read ( SoundProfile *  profile,
int  maxCount 
)

Gets the profile data.

Details
Profile data consumes one instance of SoundProfile for each audio frame (equivalent to five milliseconds).

If the function is called at intervals of 1/60 second, a minimum of four instances are consumed. Considering factors like dropped frames, it is a good idea to pass three times this number or 12 instances of the SoundProfile structure to profile. (1/60 second = 16.67 milliseconds, which converts to about three to four audio frames.)

In other words, if the function is called at an interval of T, the minimum necessary number of SoundProfile instances is calculated as follows.

(Minimum required number) = (T / 5) + 1;

The sound library stores 32 instances of the SoundProfile structure.

Parameters
[in]profileThe location where loaded profile data will be stored.
[in]maxCountThe maximum number of audio frames to go back to get profile data. Can specify up to 32.
Returns
Returns the number of sets of profile data retrieved.
