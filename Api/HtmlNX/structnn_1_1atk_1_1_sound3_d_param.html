Detailed Description

The parameters for 3D sounds.

Details
3D sound is calculated by using these parameters.

Sounds are configured on SoundMaker by entering values for controlFlag, decayCurve, decayRatio, dopplerFactor, and soundUserParam.

The actorUserParam parameter takes the value set using Sound3DActor::SetUserParam.

dopplerFactor represents the applied strength of the Doppler effect. Sound does not change when this argument is set to 0. The higher the value, the greater the change.
