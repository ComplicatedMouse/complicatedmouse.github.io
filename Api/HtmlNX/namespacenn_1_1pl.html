◆ RequestSharedFontLoad()

void nn::pl::RequestSharedFontLoad ( SharedFontType  sharedFontType)

Requests priority loading of the specified shared font.

Parameters
[in]sharedFontTypeThe shared font type.

Details
The system begins to automatically load shared fonts, and will load the shared fonts in order. The system ensures that all of the shared fonts will be in the loaded state at the end. This function requests that the system prioritizes loading of the specified shared font. Calling this function will not affect shared fonts that have already been loaded.

bool nn::pl::GetSharedFont ( int *  pOutFontCount,
SharedFontData  outData[],
int  outCountMax,
settings::LanguageCode  languageCode 
)

Gets all shared fonts in the preferential order set for each language.

Parameters
[out]pOutFontCountThe location for storing the number of accessible fonts.
[out]outDataThe location for storing the font data.
[in]outCountMaxThe number of elements in outData. The number must be at least pl::SharedFontType_Max to get the data for all of the shared fonts.
[in]languageCodeLanguage.
Returns
Returns true if the fonts were successfully obtained with all of the fonts loaded.
Precondition
  • pOutFontCount != nullptr.
  • outData != nullptr.
Postcondition
  • If the return value is true:
    • outData[0 ~ min(*pOutFontCount, outCountMax) - 1] stores valid font data.

Details
Gets all shared fonts in the order of priority set for the language specified by languageCode. To draw all characters as accurately as possible using shared fonts, you must combine several fonts in the correct order of priority (the order that determines which font data to use for each character). This order of priority differs for different languages. For example, for Chinese, the Chinese-language fonts have higher priority than the fonts for Japan, North America, and Europe. If the priority for using the outData indexes obtained by this function is set so the data of index 0 has the highest priority, then you can draw characters with the appropriate fonts for the specified language. For more information about using combinations of fonts, see the font library manual.
