◆ NVPA_MetricsContext_GetMetricNames_Begin()

NVPA_Status NVPA_MetricsContext_GetMetricNames_Begin ( NVPA_MetricsContext *  pMetricsContext,
NVPA_MetricsEnumerationOptions *  pOptions 
)

Outputs (size, pointer) to an array of "const char* pMetricName".

The lifetime of the array is tied to MetricsContext. The names are sorted. Enumerates all metrics at all levels. Includes:

  • counter.{sum,avg,min,max}
  • throughput.{avg,min,max}
  • <metric>.peak_{burst, sustained}
  • <metric>.per_{active,elapsed,region,frame}_cycle
  • <metric>.pct_of_peak_{burst,sustained}_{active,elapsed,region,frame}
  • <metric>.per.{other, other_pct}
