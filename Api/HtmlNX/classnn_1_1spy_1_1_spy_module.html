◆ OnRequested()

virtual void nn::spy::SpyModule::OnRequested ( bool  isRequested)
protectedvirtual

The use of internal features is prohibited.

Sends a notification of a data request from Spy.exe.

When a data request comes from Spy.exe, the presence or absence of a data request is notified to each DataInfo object.

When a session ends, isRequested is set to false.

For inherited classes, call the implementation of the base class first.

Parameters
[in]isRequestedSpecify true to request data.

Reimplemented in nn::spy::atk::AtkSpyModule.
