Detailed Description

A virtual interface type that can be customized for use with the Visit API

Details
The Visitor class defines a virtual interface that is used by the Visit API to report the entries stored in the MediaData instance. When reporting, the MediaData instance will, for each entry it contains, call into the Visitor with the name of the entry (const char*) and the entry's value (movie::Buffer entries are passed to the Visitor by const&).
