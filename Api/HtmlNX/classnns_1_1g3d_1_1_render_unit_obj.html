◆ Draw() [1/2]

void nns::g3d::RenderUnitObj::Draw ( nn::gfx::CommandBuffer *  pCommandBuffer,
int  passIndex,
int  viewIndex,
int  bufferIndex,
DrawCullingCallback  pDrawCullingFunction 
) const

Draws a shape.

Parameters
[in]pCommandBufferThe pointer to nn::gfx::CommandBuffer.
[in]passIndexRendering pass index.
[in]viewIndexThe view index.
[in]bufferIndexThe buffer index.
[in]pDrawCullingFunctionPointer to the function that performs culling.

Details
Draws a shape. passIndex must be within the range specified when generating RenderModel. viewIndex must be within the range of the view of the model. bufferIndex must be an index that is valid for skeletons, shapes, materials, and all uniform blocks set by nns::g3d::RenderUnitObj::BindUniformBlock(). Specifically, specifying 1 for bufferIndex when only one buffer is specified by nns::g3d::RenderUnitObj::BindUniformBlock().

View frustum culling is performed when SetViewVolume() is used to set nn::g3d::ViewVolume. When SetCalculateLodLevelFunctor() is used to set nn::g3d::ICalculateLodLevelFunctor, the set function object is called to determine the rendering LOD level. A model with LOD level 0 will be rendered if this is not set. If SetDrawLodEnabled() is used to specify a rendering LOD level, rendering will be performed at that level. If SetDrawSingleSubmeshEnabled() is used to specify a rendering submesh, only that submesh will be rendered.
