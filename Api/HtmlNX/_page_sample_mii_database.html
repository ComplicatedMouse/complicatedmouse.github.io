Overview

This sample program displays a Mii character retrieved from a database. This sample program shows how to get Mii data using nn::mii::Database.

File Structure

This sample program is located in Samples/Sources/Applications/MiiDatabase/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Mii test mode must be enabled for execution with the NX development hardware. For information, see the DevMenu and DevMenuCommand development support features.

Execution Procedure

Build the sample program and then run it.

Description

This sample program gets Mii data from a database and uses that data to draw the Mii character.

Conversion Process

  • /Samples/Sources/Applications/MiiDatabase/Resources/SampleShader.glsl

A simple shader for rendering Mii characters.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Create a sampler for the Mii.
  • Initialize the database.
  • Import the sample database.
  • Get the database from nn::mii::CharInfo.
  • Free the database.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize the Mii resource (nn::mii::Resource).
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Render the Faceline texture.
  • Render the Mask texture.
  • Initialize the shader for Mii rendering.
  • Initialize the drawing process for Mii rendering.
  • Set the constant buffers for Mii rendering.
  • Render the Mii.
