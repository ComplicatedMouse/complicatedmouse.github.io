This page describes the sample program of the feature for accessing MIFARE tags using the nfc library.

Overview

The nfc library feature for accessing MIFARE tags provides a way to read and write MIFARE format NFC tags via near-field communication (NFC). This feature for accessing MIFARE tags using the nfc library provides a way to link applications with MIFARE tags.

This sample program supports the following MIFARE operations.

  • Reading and writing.

In this sample program, counters are written to the first 1 byte of block address 1, and the first 1 byte of block address 6. (0x00 is written to the other 15 bytes.) Each time data is written, the counter at block address 1 is increment by one, and the counter at block address 6 is incremented by two.

File Structure

This sample program is located in Samples/Sources/Applications/MifareSimple/ .

Required Environment

Prepare a MIFARE tag.

How to Use

The state of the nfc library is displayed when you run the sample program. Use a gamepad to change the state of the nfc library.

Important Information

This sample program assumes you have a MIFARE tag with 0xFF written to all six bytes and to which you can use Key A to read and write to block addresses 1 and 6. If your MIFARE tag is different than this scenario, appropriately change the key and block addresses specified in the program.

Execution Procedure

Build the sample program and then run it.

Description

The MifareSimple sample program increments the counters regardless of the state of the MIFARE tag.
