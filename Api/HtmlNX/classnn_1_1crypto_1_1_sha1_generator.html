Detailed Description

Class for calculating a SHA-1 hash.

Details
This class has the following states. The state immediately after instantiation is uninitialized.

  • Uninitialized state
  • Ready.
  • Complete.
Note
Because of the reported vulnerabilities of SHA-1, we do not recommend using MD5 except when necessary to maintain compatibility with past systems.

Completes the hash calculation and gets the final hash value.

Parameters
[out]pHashPointer to the buffer storing the hash value.
[in]hashSizeSize of the buffer pointed to by pHash.
Precondition
  • hashSize >= HashSize.
  • The instance is in a ready or complete state.
Postcondition
  • The calculated hash value is written to pHash.
  • The instance is in a completed state.

Details
This function completes the hash calculation. To run a different hash calculation, call Initialize() to begin a new calculation.
