Detailed Description

The reverb effect for the specified number of channels.

Details
The EffectReverb class wraps the API for nn::audio::ReverbType. After EffectReverb has been appended by SoundSystem::AppendEffect, the class must be retained until the effect has been deleted by SoundSystem::ClearEffect or SoundSystem::RemoveEffect.

When allocating this type of memory, it must be aligned to a nn::atk::EffectBase::EffectAlignment byte boundary.

See also
EffectBase Class
Examples
AtkEffect/AtkEffect.cpp, and AtkSubMix/AtkSubMix.cpp.

Sets the maximum number of reverb effect channels.

Details
If this function is used to specify the number of channels, apply a number of reverb effects corresponding to the number of channels specified in order starting from the lowest value of nn::atk::ChannelIndex. Use SetChannelIndex instead of this function to also set the channels to which reverb is applied. Either 1, 2, 4, or 6 channels can be used as input channels. By default, the value is ChannelMode_4ch.

Parameters

Returns

Details

◆ SetChannelIndex()

bool nn::atk::EffectReverb::SetChannelIndex ( const ChannelIndex *  pChannelIndices,
ChannelMode  channelMode 
)

Configures the reverb effect channel settings.

Details
You can specify the channels for applying effects and the number of channels for the effect. If you are using this function you do not need to set the number of channels with SetChannelMode. Either 1, 2, 4, or 6 channels can be used as input channels.

The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. In addition, the function will fail if the number of input channels is not 1, 2, 4, or 6. If SetChannelMode is used together with this function, the settings of the function called later are applied.

The reverb effect requires channel settings to be configured in a fixed order. For more information about channel order, see nn::audio::SetReverbInputOutput.

Parameters
[in]pChannelIndicesThe channels in which to apply the effect. A ChannelIndex-type array. The number of pChannelIndices elements must be greater than the number of channels set in channelMode.
[in]channelModeAn enumerated type representing the number of channels in which to apply the effect.
Returns
Returns true when channel configuration is successful.
Precondition
  • pChannelIndices != nullptr.
  • (Each value in the pChannelIndices array) < nn::atk::ChannelIndex_Count.
  • Two or more of the same channel number do not exist in each of the pChannelIndices array values.

◆ SetHighFrequencyDecayRatio()

void nn::atk::EffectReverb::SetHighFrequencyDecayRatio ( float  highFrequencyDecayRatio)

Gets the reverb's high frequency attenuation rate.

When this ratio is small, low-frequency sounds predominate and high-frequency sounds attenuate faster. When set to 1.0, both the high and low frequency components will attenuate at the same speed. The value is in the range from HighFrequencyDecayRatioMin to HighFrequencyDecayRatioMax.

Parameters
[in]highFrequencyDecayRatioThe reverb's high frequency attenuation rate.
