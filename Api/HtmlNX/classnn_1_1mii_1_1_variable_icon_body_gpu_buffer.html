Detailed Description

Buffers for drawing textures of icons with bodies.

Details
This is the class for the buffer and other resources needed to draw textures of icons with bodies using VariableIconBodyShader. You need an instance of this class for each icon texture being drawn.

Attention
It is unnecessary if you are not using VariableIconBodyShader.
Examples
MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp.

Parameters

[in]clearDepthStencilViewSpecifies whether to clear the depth stencil view.

Details
By default, both clearColorTargetView and clearDepthStencilView are set to true. When clearColorTargetView = true, that view is cleared to the color set by the SetBackgroundColor() function. When clearDepthStencilView = true, that view is cleared to depth=1.0, stencil=0.
