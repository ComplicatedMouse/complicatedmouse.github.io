◆ SetShapeAnimateFlag()

void nn::ui2d::detail::VectorGraphicsShapePathProcessor::SetShapeAnimateFlag ( bool  flag)
protectedinherited

Sets whether the shape is animated.

Details
This flag must be set to true during initialization when the shape changes. The total length of the pass may change if the shape is animated. The total length of the pass is cached the first time to reduce the load caused by trim processing. However, if the length might change, it must be recalculated each frame.

Parameters
[in]flagSets whether the shape changes depending on the animation.
