Detailed Description

The use of internal features is prohibited. Parameter passed to SkeletalAnimBlender::Initialize() to perform initialization.

Constructor & Destructor Documentation

◆ InitializeArgument()

nn::g3d::SkeletalAnimBlender::InitializeArgument::InitializeArgument ( )

Constructor.

You must either call Reserve() on all skeleton resources used in the instance, or use SetMaxBoneCount() to specify the parameters required to calculate the buffer size.
