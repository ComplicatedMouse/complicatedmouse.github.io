Detailed Description

template<int THistoryLengthMax>
class nn::gfx::util::TransientDescriptorAllocatorHolder< THistoryLengthMax >

Class that wraps TransientDescriptorAllocator and the memory block needed for its initialization and creates an object.

Template Parameters
THistoryLengthMaxThe maximum number of TransientDescriptorAllocator history entries to record.

Initializes the TransientDescriptorAllocator instance that will be wrapped with this object.

Parameters
[in]pDescriptorPoolThe descriptor pool.
[in]baseSlotIndexThe index of the start of the descriptor slot to use.
[in]slotCountThe number of descriptor slots to use.
Precondition
This object must be uninitialized.
The preconditions for each parameter are the same as those for TransientDescriptorAllocator::Initialize().
Postcondition
This object is initialized.
See also
TransientDescriptorAllocator::Initialize()

Details
Initializes an instance of TransientDescriptorAllocator, to be wrapped by this object using the specified parameters. The memory region wrapped in this object is used for the memory block to use for initialization. HistoryLengthMax can specify the maximum number of history entries. Calling this instance for the same instance is not thread-safe.

Details
