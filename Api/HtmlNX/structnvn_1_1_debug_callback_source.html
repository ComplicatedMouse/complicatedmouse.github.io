enum nvn::DebugCallbackSource::Enum

Enumerated values for the type DebugCallbackSource.

Enumerator
API 

A debug callback is being issued while processing an API command.

TOOLS 

A debug callback is being issued while processing tools command.

INITIALIZATION 

A debug callback is being issued while initializing the NVN driver.

Note
To get notification of errors during the initialization phase a special debug callback must have been installed using nvnInstallGlobalDebugCallback.
