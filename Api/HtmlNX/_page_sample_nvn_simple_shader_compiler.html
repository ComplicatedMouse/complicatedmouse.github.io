Overview

The NvnSimpleShaderCompiler is a simple GLSL compiler for NVN applications, built using GLSLC. This tool loads vertex shader and fragment shader source files and use CLSLC to compile them into binary GPU code.

File Structure

ShaderCompiler.cpp A simple GLSL compiler for NVN applications, built using GLSLC.

System Requirements

Only runs on Windows.

Operation Procedures

Use Command Line like following:

NvnSimpleShaderCompiler.exe <vertex shader %path> <fragment shader %path> -o <output shader binary %path>

Precautions

None.

Execution Procedure

Build the Visual Studio Solution, supply the proper command line, and run it.
