nn::os::LightMessageQueueType Struct Reference

#include <nn/os/os_LightMessageQueueTypes.h>

Detailed Description

This structure defines a light message queue object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeLightMessageQueue()
  • FinalizeLightMessageQueue()
  • SendLightMessageQueue()
  • TrySendLightMessageQueue()
  • TimedSendLightMessageQueue()
  • JamLightMessageQueue()
  • TryJamLightMessageQueue()
  • TimedJamLightMessageQueue()
  • ReceiveLightMessageQueue()
  • TryReceiveLightMessageQueue()
  • TimedReceiveLightMessageQueue()
  • PeekLightMessageQueue()
  • TryPeekLightMessageQueue()
  • TimedPeekLightMessageQueue()

To use these functions, include nn/os.h.

Also see the description of the nn::os::LightMessageQueue utility class.
