This page describes a sample program that adds the components required by NintendoSDK to instances of Visual Studio.

Overview

This sample program adds the components required by NintendoSDK to all Visual Studio instances that support NintendoSDK.
The sample program can be run using PowerShell.

File Structure

The sample program is located in Samples/Sources/Tools/VisualStudioInstallerUsage.

Required Environment

PowerShell 5.1 or later is required for running this sample program.

How to Use

Nothing specific.

Important Information

When you run this sample program, the components are added to the Visual Studio instances installed in the execution environment.

Execution Procedure

Use PowerShell to run Samples/Sources/Tools/VisualStudioInstallerUsage/AddNintendoSDKRequiredComponents.ps1 as an administrator. The following command is an example.

powershell -File Samples/Sources/Tools/VisualStudioInstallerUsage/AddNintendoSDKRequiredComponents.ps1

Description

This sample program adds the components required by NintendoSDK to all Visual Studio instances that NintendoSDK supports.
The sample program executes the following processes.

  • Run vswhere to enumerate the Visual Studio instances that are installed in the execution environment.
  • Branch the components to add and the workloads depending on the version of Visual Studio.
  • Run vs_installer on each instance of Visual Studio to add the required components.
