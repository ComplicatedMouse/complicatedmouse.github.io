Detailed Description

Security information for a local network created by an access point.

Details
securityMode is used to set the communication protection settings. For more information, see SecurityMode.

passphrase and passphraseSize are used to generate the key for encrypting wireless communication, so specify a difficult-to-guess string or bit array in the range from PassphraseSizeMin to PassphraseSizeMax. Communication cannot be established between stations that do not have a matching passphrase.
