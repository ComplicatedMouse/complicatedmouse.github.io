Overview

This offline HTML sample is for displaying results within a race game. This sample demonstrates the following features.

  • Displaying custom fonts using Web Fonts
  • Applying text styles using CSS
  • Passing the application state to the offline HTML using a URL parameter
  • Restoring via the system when the offline web applet ends
  • Customizing controller operations using Javascript extensions

File Structure

This sample offline HTML is located in Samples/Sources/Applications/WebDemo/html_document/sample/race_result.htdocs.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

Input Operation
A Button Exits the offline web applet and returns to the menu, via the application.
X Button Switches the font.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and display the offline HTML from this sample.

Displaying Custom Fonts Using Web Fonts

All text appears using a custom typeface (document_sample_font.ttf) that suits the design of the racing game. To confirm the display, press the X Button to switch to the default font.

Applying Text Styles Using CSS

Cascading style sheets (CSS) are used to style text and apply various effects without using image data.

Passing the Application State to the Offline HTML Using a URL Parameter

When starting the offline web applet, pass race result data using a URL parameter. You can pass data like the stage, course, vehicle type, ranking, driver, and time to the offline HTML. The application determines rankings randomly. The offline HTML uses the passed parameters to change the displayed content.

Restoring via the System When the Offline Web Applet Ends

Restore the offline HTML to display it again, via the system, when the offline web applet ends and you restart the offline web applet. Pass whether you will restore to the offline HTML using a URL parameter. When restoring the display, the animation within the offline HTML for displaying the rankings is skipped.

Customizing Controller Operations Using Javascript Extensions

Change the functionality of specified keys using the window.nx.footer.setAssign() Javascript extension.
