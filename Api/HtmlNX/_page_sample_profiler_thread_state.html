This sample shows how to use the Profiler Thread State tracing feature.

Overview

File Structure

This sample program is located in Samples/Sources/Applications/ProfilerThreadState .

System Requirements

The Nintendo NX CPU Profiler PC executable, found in Tools/NintendoNxCpuProfiler, should be running.

Operation Procedures

It is not possible to interact directly with the sample.

Cautions

This demo does not display anything on the screen.

Execution Procedure

  • Execution procedure
    • Build the sample program and then run it.
    • Once it is running, click Sync in the Nintendo NX CPU Profiler PC tool.
    • On the Sampled Profile tab, click Thread State to enable it, and click on Start to start taking a profile.
    • Profiling can be stopped by either clicking on Stop or waiting for the internal buffers to fill.
    • To view the results from the profile, select the Threads tab on the left pane, then click on the specific threads.
  • Explanation of results
    • Click on threads "Sample Thread 1", "Sample Thread 2" and "Sample Thread 3".
    • Data for each thread will be visible on the right pane under Thread State Data.
    • A red vertical bar shows a thread is blocked.
    • Double click on a red vertical bar to show more details in a pop up window.

105 // Function run by each thread which will all attempt to access a shared resource
106 void ThreadSample(void *arg)
107 {
108  int64_t resultFromCalculations = 0;
109  int fibonacciSeed = 0;
110 
111  while (!g_ShouldExit)
112  {
113  fibonacciSeed = (rand() % 5) + 15;
114  resultFromCalculations = CalculateFibonacci(fibonacciSeed);
115  if (resultFromCalculations != 0)
116  {
117  nn::os::LockMutex(&g_Mutex);

119  // Calculation that uses the shared resource

Sample Program Description

This sample shows how to use the Profiler Thread State tracing feature. The intention is to provide an example on expected usage of this feature.

This sample is to be used in conjunction with the Nintendo NX CPU Profiler PC executable. The executable can be found in Tools/NintendoNxCpuProfiler.

The sample shows the basic method used to both Initialize and Finalize the profiler. It also shows how to use the Thread State button.
