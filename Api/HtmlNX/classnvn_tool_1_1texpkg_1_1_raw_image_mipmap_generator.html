Detailed Description

This module is responsible for mipmap generation.

Currently, only box filtering is supported.

pure virtual

Generate mipmap for N levels, starting from baseLevel = 0.

Parameters
[in]srcSource image with baseLevel 0 defined to generate mipmaps from.
[out]dstDestination image to store finished image. baseLevel 0 will be bitwise-copied. Alignment / padding will be exactly the same as the source image.
[in]numLevelsNumber of mipmap levels to generate.
Returns
TP_OK if successful, TP_UNSUPPORTED_FORMAT if source format is not supported, TP_ERROR_UNKNOWN if src or dst is NULL, TP_INVALID_ARGUMENT_FORMAT if filter is unsupported, other < 0 error code otherwise.

RawImage *  dst,
uint32_t  minSize 
)
pure virtual

Generate mipmap for until all dimensions are below minSize x minSize x minSize, stating from baseLevel = 0.

A minSize = 1 would generate until 1 x 1 x 1.

Parameters
[in]srcSource image (baseLevel 0 must be defined).
[out]dstDestination image to store finished image. baseLevel 0 will be bitwise-copied. Alignment / padding will be exactly the same as the source image.
[in]minSizeGenerate texture mipmaps until size is < minSize.
Returns
TP_OK if successful, TP_UNSUPPORTED_FORMAT if source format is not supported, TP_ERROR_UNKNOWN if src or dst is NULL, TP_INVALID_ARGUMENT_FORMAT if filter is unsupported, other < 0 error code otherwise.
