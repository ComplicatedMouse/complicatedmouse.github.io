Detailed Description

template<typename T>
class nn::Span< T >

A class template representing views for continuous areas.

Template Parameters
TThe element types.

Details
nn::Span is a subset of std::span, to be added to C++ 20. (See http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2018/p0122r6.pdf.) In the future when NintendoSDK supports C++ 20, it will become an alias template for std::span.

Internally, it maintains a pointer to the start of the contiguous region and the length.

Span is just a view to the contiguous region. It does not own the region. The lifetime maintenance of the region must be taken care of separately.

You can create an instance of Span, omitting the use of T, the element type, by using MakeSpan().
