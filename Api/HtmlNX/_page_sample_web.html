Web

The description of the sample program that uses web applet and offline web applet features.

Introduction

The following describes the web applet and offline web applet sample.

Samples

Sample code is located in the directories starting with the prefix Web under Samples/Sources/Applications/. The following table lists the samples.

Page Content
WebSimple A simple sample program that uses the nn::web library to start the web applet and open a webpage.
WebOfflineSimple A simple sample program that uses the nn::web library to start the offline web applet and then opens offline HTML.
WebDemo The sample program that shows how to use web applet and offline web applet features.
WebDemoInstructionManual This sample offline HTML is for displaying a game manual.
Web Applet Pause Menu Sample This sample offline HTML is for displaying a game pause dialog box.
WebDemoQuickHelp This sample offline HTML is for displaying quick Help after pausing a game.
WebDemoRaceResult This sample offline HTML is for displaying results within a race game.
3D Shooter Movie in the Web Applet Sample This sample offline HTML is for displaying the game technique used in a 3D shooter.
Direct Playback Via Web Applet Sample This sample offline HTML directly plays back videos.
WebDemoKeyboardEvents Sample offline HTML to read controller operations using keyboard events.
WebDemoGamepadApi Sample offline HTML to read controller operations using the Gamepad API.
Spatial Navigation in the Web Applet Sample Sample offline HTML to change spatial navigation with various proprietary focus effects, using CSS.
