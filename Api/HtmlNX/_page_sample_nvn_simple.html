NVN Simple

Simple sample programs for NVN library.

Introduction

These sample applications are designed to help application developers get started with native graphics programming using the NVN graphics API and NX Addon. The graphics applications each begin with the prefix NvnSimple, and are ordered with increasing complexity, starting with NvnSimple01ScreenSpaceTriangles. In an effort to aid learning, we have designed the source code and project structure with the goals of simplicity and clarity. Much of the low-level interactions with the NVN API can be found in the libnns_NvnSimpleSharedLib-spec.NX.autogen project.

We hope that progressing through the applications one by one can act as a walkthrough on how to get started with utilizing the exciting native graphics APIs on NX platform.

Operating environment

No extra system requirements for running these samples with default build configurations. But NvnSimple samples support several tools to help debug and optimize your applications shown below.
-NVN Graphics Debugger (NVNGD)
-Low-Level Graphics Debugger (LLGD)
-Nintendo NX CPU Profiler (NXCP)
these tools are not default installed, so make sure these tools are installed if you want to use them.

Build configurations and tool support

To enable development tools mentioned above, editing the following properties for integration/linking:

Add Preprocessor macro shown below in project libnns_NvnSimpleSharedLib-spec.NX.autogen.
"Visual Studio > Configuration Properties > C/C++ > Preprocessor > Preprocessor Definitions"

Tools Preprocessor
NVN Graphics Debugger (NVNGD)
Low-Level Graphics Debugger (LLGD) 		USE_NVNGD
Nintendo NX CPU Profiler (NXCP) USE_NXCP


For NX32/NX64 platforms, all these tools can be used. For using these tools, add following input files to link the libraries.
"Visual Studio > Configuration Properties > Linker > Input" in application projects.

Additional Dependencies Additional NSO Files
NVN Graphics Debugger (NVNGD) Nvidia_gfx_debugger.nss Nvidia_gfx_debugger.nso
Low-Level Graphics Debugger (LLGD) llgd-target.nss llgd-target.nso
Nintendo NX CPU Profiler (NXCP) libnn_profiler.a


For Win32/x64 platforms, only NVN Graphics Debugger (NVNGD) is supported.
For using NVNGD, DLL files from the NintendoSDK/NvnGraphicsDebugger/target/windows-desktop-nvn-[x86, x64]/ directory are necessary.
Add build event shown below to copy DLL files that are used.
"Visual Studio > Configuration Properties > Build Events > Post-Build Event" in application projects.

Platform Command Line
Win32 robocopy
$(NintendoSdkRoot)/../NvnGraphicsDebugger/target/windows-desktop-nvn-x86 *.dll > nul
exit 0
x64 robocopy
$(NintendoSdkRoot)/../NvnGraphicsDebugger/target/windows-desktop-nvn-x64 *.dll > nul
exit 0


Caution

Note that application which uses these tools' library, cannot be uploaded as a Master ROM into Nintendo Develop Portal(NDP). Disable all the codes used for using these tools and remove the link to these tools' library before master up.

Samples

The NvnSimple sample programs can be found in Samples/Sources/Applications/ with prefix "NvnSimple".
Please refer to the links below for more information on the samples.

Page Contents
Nvn Simple 01 : Screen Space Triangles This sample implements the drawing of some triangles in screen space coordinates to introduces the basics of shader and vertex data setup.
Nvn Simple 02 : World Space Triangles This sample implements the drawing of some animated triangles in world space coordinates to introduces the uniform data setup.
Nvn Simple 03 : Textured Cube This sample implements the drawing of a spinning textured cube to introduces how to use textures and samplers.
Nvn Simple 04 : Render To Texture This sample renders a scene into an application-specific texture, which is then used to draw a secondary scene that is then displayed to screen.
Nvn Simple 05 : Font Rendering This sample implements the displaying of English, Japanese, and Korean strings to introduces how to render font with nn::fontll scalable font engine.
Nvn Simple 06 : Multi Command Set This sample implements the drawing of some cones by different command sets to introduces how to use multiple command sets from a single NVN command buffer.
Nvn Simple 07 : Dynamic Resolution This sample introduces how to use nvn::Window::SetCrop to adjust render resolution.
Nvn Simple Texture Compiler The NvnSimpleTextureConverter is a simple offline converter texture into a hardware optimized format for NX platform.
Nvn Simple Shader Compiler The NvnSimpleShaderCompiler is a simple GLSL compiler for NVN applications, built using GLSLC.
Nvn Simple Shared Library The NVN Simple shared Library contains common code shared by NvnSimple 01-06.
