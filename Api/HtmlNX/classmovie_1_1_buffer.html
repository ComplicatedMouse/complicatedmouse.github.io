Detailed Description

Buffer utility for passing media data between decoders and extractors.

Details
Buffer utility class can be used for passing decoder buffers to extractors. Buffer can be constructed with buffer object allocated data memory, client passed data memory or to hold only buffer information.

Examples
MovieDecoderPlayer/MovieDecoderPlayer.cpp.
