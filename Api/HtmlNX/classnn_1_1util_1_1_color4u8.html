Detailed Description

Class that supports colors expressed as unsigned 8-bit integers for each of the four components.

Examples
FrameworkDemo/Main.cpp, G3dDemo/ShapeAnimation.cpp, G3dDemo/SkeletalAnimation.cpp, G3dDemo/Town.cpp, HidControllerSequence/HidControllerSequence_Game.cpp, HidControllerSequence/HidControllerSequence_Main.cpp, HidControllerSequence/HidControllerSequence_Start.cpp, HidControllerSequence/HidControllerSequence_Title.cpp, PerfSimple/PerfSimple.cpp, and WebDemo/WebDemo_Main.cpp.
