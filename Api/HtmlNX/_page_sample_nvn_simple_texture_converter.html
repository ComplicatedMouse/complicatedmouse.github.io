The NvnSimpleTextureConverter is a simple offline converter texture into a hardware optimized format for NX platform.

Overview

The NvnSimpleTextureConverter demonstrates how to use NVN texture packager to convert tiles images into a hardware optimized format for NX platform. This tool can convert TGA and DDS file formats.

File Structure

TextureAsset.h Defines the format of a texture asset file converted with NvnSimpleTextureConverter.exe.
TextureConverter.cpp A simple texture converter for NVN applications.

System Requirements

Only runs on Windows.

Operation Procedures

Use Command Line like following:

NvnSimpleTextureConverter.exe <texture image %path> -o <output image %path>

Precautions

None.

Execution Procedure

Build the Visual Studio Solution, supply the proper command line, and run it.

Description

This tool uses command line to loads a TGA or DDS file which is supported by NVN texture packager library and convert raw image into hardware optimized format. Then the hardware optimized texture is written to file as a TextureAsset. Note that we are writing out the byte stream as-is, so the endianness of the machine where this tool is being used must match the endianness of the target platform from which the output file will be loaded.
