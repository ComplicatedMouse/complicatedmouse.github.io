Detailed Description

Effect used to incorporate an effect process that the user can define.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • GetRequiredBufferSizeForAuxSendReturnBuffer()
  • AddAux(AudioRendererConfig* pOutConfig, AuxType* pAux, SubMixType* pSubMix, void* pSendBuffer, void* pReturnBuffer, size_t bufferSize);
  • AddAux(AudioRendererConfig* pOutConfig, AuxType* pAux, FinalMixType* pFinalMix, void* pSendBuffer, void* pReturnBuffer, size_t bufferSize);
  • RemoveAux(AudioRendererConfig* pOutConfig, AuxType* pAux, SubMixType* pSubMix)
  • RemoveAux(AudioRendererConfig* pOutConfig, AuxType* pAux, FinalMixType* pFinalMix)
  • SetAuxEnabled()
  • IsAuxEnabled()
  • GetAuxInputOutput()
  • SetAuxInputOutput()
  • ReadAuxSendBuffer()
  • WriteAuxReturnBuffer()
  • GetAuxSampleRate()
  • GetAuxSampleCount()
  • GetAuxBufferStatistics()
  • ResetAuxBufferStatistics()

To use these functions, include nn/audio.h.
