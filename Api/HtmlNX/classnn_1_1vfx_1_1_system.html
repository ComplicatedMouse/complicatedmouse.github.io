Detailed Description

Manages all effects.

Details
Through this class, the user can create and destroy emitter sets, register and destroy resources, and calculate and render effects.

Examples
VfxBinarySegmentation/main.cpp, VfxCustomShader/main.cpp, VfxManualEmitterSet/main.cpp, and VfxSimple/main.cpp.

Registers a resource.

Keep the resource in memory even after registering it.

Details
Becomes a parameter for the binary placed in the memory pool prepared in advance on the user side, and for the target memory pool. Memory pool creation does not occur at the level of the textures or shader binaries included within the binary because initialization is based on the passed memory pool.

Parameters
[in]pHeapThe heap to allocate memory from.
[in]pResourceThe effect binary in memory. It must be aligned to 128 bytes.
[in]pMemoryPoolMemory pool. When an effect binary (pBinary) was placed in a memory pool, specifies that memory pool.
[in]memoryPoolSizeThe destination memory pool size.
[in]memoryPoolOffsetOffset within the memory pool.
[in]resourceIdThe resource ID to register. The specifiable upper limit for the ID is specified in nn::vfx::Config.
[in]shaderDelaySetuptrue to use shader setup.
[in]residentResourceThe resource containing the texture that is shared.
Returns
Returns true if the resource is successfully registered.
See also
nn::vfx::System::ClearResource, nn::vfx::Heap, nn::vfx::Resource
Examples
