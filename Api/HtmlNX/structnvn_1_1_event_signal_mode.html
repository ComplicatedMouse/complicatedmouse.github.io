Detailed Description

Method used by CommandBuffer::SignalEvent to update event memory.

When CommandBuffer::SignalEvent is called, this mode specifies how memory associated with the Event object is updated.

enum nvn::EventSignalMode::Enum

Enumerated values for the type EventSignalMode.

Enumerator
WRITE 

Store the value provided by the SignalEvent command to event memory.

This mode is supported on all platforms.

ADD 

Atomically add the value provided by the SignalEvent command to event memory.

This mode is not supported on all platforms. Support is determined by querying the EVENTS_SUPPORT_REDUCTION_OPERATIONS device property.
