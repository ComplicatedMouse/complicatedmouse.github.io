Detailed Description

The use of internal features is prohibited. Parameter passed to SkeletalAnimObj::Initialize() to perform initialization.

Constructor & Destructor Documentation

◆ InitializeArgument()

nn::g3d::SkeletalAnimObj::InitializeArgument::InitializeArgument ( )

Constructor.

Details
You must either call Reserve() for all skeletal animations and all skeleton resources used in the instance, or use SetMaxBoneCount(), SetMaxBoneAnimCount(), and SetMaxCurveCount() to specify the parameters required for calculating the buffer size.
