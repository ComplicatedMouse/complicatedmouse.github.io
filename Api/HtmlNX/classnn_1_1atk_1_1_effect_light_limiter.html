Detailed Description

Lightweight limiter effect for the specified number of channels.

Details
The EffectLightLimiter class wraps the API for nn::audio::LightLimiterType. After EffectLightLimiter has been appended by SoundSystem::AppendEffect, the class must be retained until the effect has been deleted by SoundSystem::ClearEffect or SoundSystem::RemoveEffect.

Also, when allocating this type on memory, it must be aligned to a nn::atk::EffectBase::EffectAlignment byte boundary.

The effect parameters are initialized as follows. 

   - <tt>GetAttackTime</tt> = <tt>GetDefaultAttackTime</tt>.
   - <tt>GetReleaseTime</tt> = <tt>GetDefaultReleaseTime</tt>.
   - <tt>GetLookAheadTime</tt> = <tt>GetDefaultLookAheadTime</tt>.

   - <tt>GetOutputGain</tt> = <tt>DefaultOutputGain</tt>.
   - <tt>IsEnabled</tt> = <tt>true</tt>.

The default channel mode is ChannelMode_6ch and effects are assigned to channels in order of nn::atk::ChannelIndex, starting from the smallest value.

◆ SetChannelMode()

bool nn::atk::EffectLightLimiter::SetChannelMode ( ChannelMode  mode)

Sets the maximum number of lightweight limiter effect channels.

Details
If this function is used to specify the number of channels, apply a number of lightweight limiters corresponding to the number of channels specified in order starting from the lowest value of nn::atk::ChannelIndex. Use SetChannelIndex instead of this function to also set the channels to which the lightweight limiter is applied. Either 1, 2, 4, or 6 channels can be used as input channels. The default is ChannelMode_6ch.

Parameters
[in]modeThe enumerated type for specifying the number of channels.
Returns
Returns true when configuration for the maximum number of channels is successful.
See also
GetRequiredMemSize
GetChannelMode
SoundSystem::AppendEffect
SetChannelIndex

Details
The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. In addition, the function will fail if the number of input channels is not 1, 2, 4, or 6. If this function is used together with SetChannelIndex, the get settings that are applied are those of the function called last.

Configures the lightweight limiter effect channel settings.

Details
You can specify the channels for applying effects and the number of channels for the effect. If you are using this function you do not need to set the number of channels with SetChannelMode. Either 1, 2, 4, or 6 channels can be used as input channels.

Parameters
[in]pChannelIndicesThe channel that applies the effects. A ChannelIndex-type array. The number of pChannelIndices elements must be greater than the number of channels set in channelMode.
[in]channelModeThe enumerated type that indicates the number of channels to which effects are applied.
Returns
Returns true when channel configuration is successful.
Precondition
  • pChannelIndices != nullptr.
  • (Each value in the pChannelIndices array) < nn::atk::ChannelIndex_Count.
  • Two or more of the same channel number do not exist in each of the pChannelIndices array values.
See also
GetChannelIndex
SetChannelMode

Details
The function successfully configures the channel settings before nn::atk::SoundSystem::AppendEffect, but fails after nn::atk::SoundSystem::AppendEffect and continues to fail until the effect has been removed by nn::atk::SoundSystem::RemoveEffect. In addition, the function will fail if the number of input channels is not 1, 2, 4, or 6. If SetChannelMode is used together with this function, the settings of the function called later are applied.
