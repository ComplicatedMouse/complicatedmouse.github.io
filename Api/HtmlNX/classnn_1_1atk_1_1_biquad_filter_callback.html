Detailed Description

An abstract class representing a biquad filter.

Details
Inherits the BiquadFilterCallback class, enabling the user to implement a biquad filter.

For more information about using BiquadFilterCallback, see SoundSystem::SetBiquadFilterCallback.

pure virtual

Returns the biquad filter coefficient.

Details
The biquad filter coefficient in the nn::atk library is called when necessary. Classes that inherit BiquadFilterCallback must implement filter coefficients in this function.

type is the type number when the callback class is registered with SoundSystem::SetBiquadFilterCallback. A value in the range from BIQUAD_FILTER_TYPE_USER_MIN to BIQUAD_FILTER_TYPE_USER_MAX for BiquadFilterType is entered.

value is a value between 0.0f and 1.0f that indicates how to apply the filter. The filter coefficient that corresponds to this value is stored in coef.

Parameters
[out]pOutValueStructure that stores the calculated filter coefficient.
[in]typeThe filter type. (Values are in the range from BIQUAD_FILTER_TYPE_USER_MIN to BIQUAD_FILTER_TYPE_USER_MAX.)
[in]valueHow to apply the filter (0.0f - 1.0f).
