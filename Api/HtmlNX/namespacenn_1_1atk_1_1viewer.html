Detailed Description

Namespace of the sound in-game edit library.

This library allows you to edit the sound archive currently being used in a game from SoundMaker. For more information about using the library, see the description of each class and the edit demo.

Notes

  • This library contains classes that are still in beta.
  • The features provided by these classes may be changed without notice.
