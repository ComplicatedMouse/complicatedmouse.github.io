◆ NN_UTIL_SCOPE_EXIT

#define NN_UTIL_SCOPE_EXIT

Macro that writes the process when an object goes out of scope.

Details
If you code a block after this macro, the process in the block is executed when objects go out of scope. A semicolon is required after the closing parenthesis inside the block.

Example:

{
FILE* f = std::fopen(path, "r");
NN_UTIL_SCOPE_EXIT
{
std::fclose(f);
};
... Call to std::fread or similar
// When scope ends, std::fclose(f) is called automatically.
}
