Constructor that takes four 4-byte numbers as seeds.

Calls init(seed_0, seed_1, seed_2, seed_3) internally, and initializes the random number generator. Instances of the Random class with the same seed value generate the same random number sequence. Note that you cannot pass a value of 0 to all parameters. Execution stops on an assertion failure. When assertions are disabled in the retail version, the object is instantiated as init(0) instead.

◆ Initialize() [2/3]

void nn::vfx::detail::RandomGenerator::Initialize ( uint32_t  seed)

Initializes the random number generator using the seed passed as an argument.

Details
This function expands the 4-byte seed provided as an argument into a 16-byte number because XorShift requires a 16-byte number for initialization. The seed value can be any number in the range from 0 to 0xFFFFFFFF, but avoid seeds that follow a single pattern (such as 0, 1, 2, ...).

Parameters
[in]seedA random number seed.
