Describes the sample program for the AOC item list.

Overview

This sample shows how to display AOC item information. After selecting AOC item list on the start screen, this is the screen after selecting sale status (on sale items only / all items). A list of AOC item names is displayed, and information is obtained and shown about the selected AOC item, including the name, price, description, disclaimers, and bundle information. To get the price information, a separate function for that purpose is used. A shop jump occurs when an item is selected. The price information also supports displaying discount prices and discount periods. The display of the discount period supports display formats for Japan, the EU, and elsewhere (including the US). For the EU, inc tax message (VAT) information appears under the price. If you select on sale items only for sale status (on sale items only / all items), a list of items currently on sale is displayed. If you select all items, all items are displayed except those for which sales_status is not_onsale in the price information.

File Structure

This sample program is located in Samples/Sources/Applications/EcCatalog/Aoc/ .

Description

For a general overview of the process, see the description in Main.cpp. This description only details the shop jump functionality. The shop jump within this sample occurs using Main.cpp.
