Detailed Description

The use of internal features is prohibited. Parameters passed to the ModelObj::Initialize() function for initialization.

◆ Skeleton()

void nn::g3d::ModelObj::InitializeArgument::Skeleton ( SkeletonObj *  pSkeleton)

Sets the shared skeleton.

Parameters
[in]pSkeletonPointer to SkeletonObj.
Precondition
  • The SkeletonObj being set is the instance that handles the same skeleton resource as that of the resource being set.

Details
By passing a constructed SkeletonObj, the skeleton can be shared with another ModelObj. Sharing enables you to skip calculation processing.

◆ CalculateMemorySize()

void nn::g3d::ModelObj::InitializeArgument::CalculateMemorySize ( )

Calculates the amount of memory required to create the ModelObj instance.

Precondition
  • Buffering of at least one uniform block for the specified skeleton.
  • Buffering of at least one uniform block for the specified shape.
  • Buffering of at least one uniform block for the specified material.
  • At least one view has been set.
Examples
G3dDemo/ShapeAnimation.cpp, and G3dDemo/Simple.cpp.
