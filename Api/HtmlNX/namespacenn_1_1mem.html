Detailed Description

Namespace of the dynamic memory allocation library for fast allocation and deallocation.

Details
This namespace has been prepared for a dynamic memory allocation library for fast allocation and deallocation. Heaps with the same features as those in the Cafe SDK and CTR-SDK are located in the nn::lmem namespace.

NX Platform Only:
Memory allocation features such as malloc() are used through internal nn::mem namespace features. For more information about defining malloc() independently, see the nn::init namespace and About malloc() and Other Functions That Get Memory in NintendoSDK Documents.
