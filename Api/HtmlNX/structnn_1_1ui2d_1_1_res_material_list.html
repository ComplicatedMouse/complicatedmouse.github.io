Detailed Description

A list of material information.

Details
The list of materials referenced by the pane is defined ahead of the pane.

DataBlockKindMaterialList is specified for blockHeader[kind].

The following data structures are in this structure.

  • uint32_t materialOffsetTable[materialCount]
  • ResMaterial material0
  • :

materialOffsetTable has the offsets to each instance of material information (from the start of the data block).
