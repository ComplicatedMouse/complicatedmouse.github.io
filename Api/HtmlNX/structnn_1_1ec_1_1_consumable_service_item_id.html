Detailed Description

ID that identifies the purchased consumable service item.

A maximum of 16 characters can be stored as a string.

Details
nn::ec::ConsumableServiceItemId is a server parameter required for handling various requests for consumable service items. It is equivalent to the item ID. The item ID must be registered in Nintendo eShop. For more information about registration and the ID, see Preparations for Starting Development in the Nintendo Switch Service Item Programming Manual.
