Detailed Description

The structure that represents the context of the clock that returns the date and time.

Details
The context is updated whenever the clock is adjusted, manipulated, or reset. You can determine whether there have been any such adjustments or manipulations by comparing the context.

To detect such adjustments or manipulations while the application is not running, save the content of this structure to save data, and pass it to the next launch.
