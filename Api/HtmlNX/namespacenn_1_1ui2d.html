Detailed Description

Namespace for the ui2d library and the default control.

Details
Controls add user interface and other functions to a layout, and default controls are implemented by the layout library. Library users may extend the default controls, or implement their own controls and not use the default controls at all.

The buttons of the default control are implemented as a derived class of the AnimButton class.

To use the default Control in layouts, specify DefaultControlCreator for the parameter of the ui2d::Layout::Build() function.

Attention
To use the functions of this namespace , the NintendoSDK NintendoWare package must be installed.

◆ ArchiveResourceAlignment

const int nn::ui2d::ArchiveResourceAlignment = 4 * 1024
static

It is the alignment required of the starting address of the archive resource.

This alignment is necessary so the GPU can handle the texture data included in the archive data without copying it to another location. The data in the files inside the archive is aligned automatically by Layout Archiver.

This number is defined to accommodate the largest alignment requirement.
