This sample program demonstrates file system operations on the host PC.

Overview

This sample uses file system operations on a host PC.

For more information about file and directory operations, see the File System Library API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/FsHost/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

This sample creates two entries, C:/Windows/Temp/FsSmplDir/ and C:/Windows/Temp/FsSmplFile, /FsSmplFile, and then deletes them after operations are complete.

Note that the demo may not work correctly when these entries exist or when the demo cannot access these paths.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • An example of file system operations on the host PC using the MountHost function.
  • An example of file system operations on the host PC using the MountHostRoot function.

The file system on a host PC can be used by using the MountHost and MountHostRoot functions.

File system operations occur when mounted with the MountHost function using path names that begin with mount name:/. When mounted with MountHostRoot, the absolute path within the host PC is used.
