Ec Library Samples

Describes the samples for using Nintendo eShop features.

Introduction

This section describes the samples for using Nintendo eShop features. The module with the Nintendo eShop features is in the nn::ec namespace.

Samples

Sample code is located in the directories starting with Ec under Samples/Sources/Applications/. The following table lists the samples.

Page Content
EcCatalog Sample Program (Sample of Getting E-Commerce Shop Information and Jumping to the Shop) This section describes the sample programs that demonstrate how to get e-commerce shop information (lists of consumable items, AOC items, AOC shelves, related titles, and subscription service items) and how to jump to shops.
EcConsumableServiceItemManager This document describes the sample program that demonstrates the user save data manager for consumable service items.
Sample That Checks For Purchase Reports Describes the sample program that checks for reports of purchases.
