Sample offline HTML to change spatial navigation with various proprietary focus effects, using CSS.

Overview

Offline HTML sample to change spatial navigation with various proprietary focus effects, using CSS. This sample demonstrates the following features.

  • Independently changing spatial navigation focusing
  • Blocking touch screen controls

File Structure

This sample offline HTML is located in Samples/Sources/Applications/WebDemo/html_document/sample/spatial_navigation.htdocs.

Required Environment

This sample is only supported in the NX development hardware environment.

How to Use

Input Operation
L Stick or the directional buttons Moves the spatial navigation.
B Button, X Button Exits the offline web applet and returns to the menu, via the application.

Cautions

Nothing specific.

Execution Procedure

Build the WebDemo sample program and then run it. From the menu, select this sample.

Description

Select this sample from the menu to start the offline web applet via the application and display the offline HTML from this sample.

Independently Changing Spatial Navigation Focusing

Uses the focus-ring-visibility attribute from the meta element to hide the spatial navigation focus frame. Uses the CSS :focus pseudo-class to specify a style while the focus is on for the button element.

Blocking Touch Screen Controls

Used nn::web::ShowOfflineHtmlPageArg::SetTouchEnabledOnContents() to disable touch controls.
