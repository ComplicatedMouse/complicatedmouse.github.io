A sample program for the timer event feature.

Overview

This sample shows the timer event feature.

For more information about using the timer event feature, see the Timer Event Feature Manual and the OS API Reference.

File Structure

This sample program is located in Samples/Sources/Applications/OsTimerEvent/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • Two timer events and two threads are created in nnMain().
  • The two threads are ThreadFunction1() and ThreadFunction2().
  • The first timer event is started as a periodic timer event.
  • The second timer event is started as a one-shot timer event.
  • ThreadFunction1() waits for the periodic timer event with the WaitTimerEvent() function.
  • ThreadFunction2() waits for the one-shot timer event with the WaitTimerEvent() function.
  • The period timer event is stopped at the end of ThreadFunction1().
  • Execution waits for the two threads to complete in nnMain().
  • The two threads and two timer events are destroyed in nnMain()

ThreadFunction1() and ThreadFunction2() both use the nn::os::WaitTimerEvent() feature to wait for the separately started timer events.

The first timer event starts periodically. After the thread wakes in WaitTimerEvent(), the countdown for the next event starts automatically and g_TimerEvent1 is signaled automatically every period specified in StartPeriodicTimerEvent().
