◆ InitializeGraphics()

void nv::InitializeGraphics ( void *  address,
size_t  size 
)

Specify the address and size of memory that is donated to graphics firmware.

Details
This function must be called before using any graphics APIs. The relative order of nv::InitializeGraphics, nv::SetGraphicsAllocator and nv::SetGraphicsDevtoolsAllocator calls does not matter, as long as the application makes all its calls before using any graphics API's. Failure to do so will result in the application being aborted. The donated memory block will be unmapped from process address space and cannot be used afterwards.

Parameters
[in]addressAddress of memory being donated to graphics firmware. Must be aligned to nv::GraphicsMemoryAlignment.
[in]sizeSize of memory being donated to graphics firmware. Must be a multiple of nv::GraphicsMemoryGranularity.

◆ FinalizeGraphics()

void nv::FinalizeGraphics ( )

Release resources from graphics subsystem.

Details
This function must be called once the client is no longer making use of the graphics subsystem. It will release the resources in use from the memory donated in nv::InitializeGraphics, and also free the memory received from the allocator previously set up with the nv::SetGraphicsAllocator function. Initialized resources for all graphics APIs must be released before calling nv::FinalizeGraphics to release all allocated memory resources. nv::FinalizeGraphics releases memory resources indirectly allocated in graphics subsystem.

Examples
GfxShaderCompile/GfxShaderCompile.cpp, GfxSimple/GfxSimple.cpp, GfxSimpleCompute/GfxSimpleCompute.cpp, MovieDecoderPlayer/MovieDecoderPlayer.cpp, MoviePlayerSimple/MoviePlayerSimple.cpp, ViTimestamp/ViTimestamp.cpp, and VkSimple/VkSimple.cpp.

◆ IsCapableNoVsyncForDebug()

bool nv::IsCapableNoVsyncForDebug ( )

Query status of No-Vsync capability.

Details
This function can be used only for debugging. The function returns the capability to render without vsync waits. In other words, whether the driver allows to render with SwapInterval = 0.

Returns
Returns true if rendering without vsync wait is allowed, and false otherwise.
