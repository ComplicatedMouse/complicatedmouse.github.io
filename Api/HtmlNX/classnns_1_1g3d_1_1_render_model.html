Detailed Description

This class manages the vertex state, depth stencil state, blend state, rasterizer state, and shading model of the model.

Examples
G3dDemo/Edit.cpp, G3dDemo/NnsG3dSimple.cpp, G3dDemo/Outline.cpp, G3dDemo/Parallel.cpp, G3dDemo/ShapeAnimation.cpp, G3dDemo/SkeletalAnimation.cpp, G3dDemo/Tessellation.cpp, and G3dDemo/Town.cpp.

bool nns::g3d::RenderModel::CreateDrawPass ( nn::gfx::Device *  pDevice,
int  passIndex,
nn::g3d::ResShaderArchive *  pResShaderArchive 
)

Creates the rendering pass.

Parameters
[in]pDevicePointer to the device.
[in]passIndexRendering pass index.
[in]pResShaderArchivePointer to nn::g3d::ResShaderArchive that contains the shading models configured by the shader assignments of the model.
Precondition
  • IsInitialized()
  • pDevice != NULL.
  • nn::gfx::IsInitialized(*pDevice)
  • 0 <= passIndex < m_PassCount.
  • pResShaderArchive != NULL.
Returns
Returns true if the settings succeeded, or false if it failed.

Details
Create nns::g3d::RenderShape and nns::g3d::RenderMaterial and perform initialization for nn::g3d::ResShape and nn::g3d::ResMaterial under pResModel. Initialization of nns::g3d::RenderMaterial takes nn::g3d::ResShaderArchive of nns::g3d::RenderMaterial::Initialize() as the argument.

Examples
G3dDemo/Edit.cpp, G3dDemo/NnsG3dSimple.cpp, G3dDemo/Outline.cpp, G3dDemo/Parallel.cpp, G3dDemo/ShapeAnimation.cpp, G3dDemo/SkeletalAnimation.cpp, G3dDemo/Tessellation.cpp, and G3dDemo/Town.cpp.

bool nns::g3d::RenderModel::CreateDrawPass ( nn::gfx::Device *  pDevice,
int  passIndex,
nn::g3d::ResShaderArchive **  pResShaderArchiveArray,
int  resShaderArchiveCount 
)

Creates the rendering pass.

Parameters
[in]pDevicePointer to the device.
[in]passIndexRendering pass index.
[in]pResShaderArchiveArrayPointer to nn::g3d::ResShaderArchive that contains the shading model used in combination with the model.
[in]resShaderArchiveCountThe number of elements in the pointer array to nn::g3d::ResShaderArchive.
Precondition
  • IsInitialized()
  • pDevice != NULL.
  • nn::gfx::IsInitialized(*pDevice)
  • 0 <= passIndex < m_PassCount.
  • pResShaderArchiveArray != NULL.
  • resShaderArchiveCount == m_pResModel->GetMaterialCount().
Returns
Returns true if the settings succeeded, or false if it failed.

Details
Create nns::g3d::RenderShape and nns::g3d::RenderMaterial and perform initialization for nn::g3d::ResShape and nn::g3d::ResMaterial under pResModel. Initialization of nns::g3d::RenderMaterial takes nn::g3d::ResShaderArchive of nns::g3d::RenderMaterial::Initialize() as the argument. nn::g3d::ResShaderArchive element in array index 0 is used for index 0 of nns::g3d::RenderMaterial.
