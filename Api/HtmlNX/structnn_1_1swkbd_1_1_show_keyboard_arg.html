◆ workBuf

void* nn::swkbd::ShowKeyboardArg::workBuf

The work buffer to pass to the software keyboard.


Stores the string specified by the nn::swkbd::SetInitialText() function as the initial entry text and the user dictionary specified by the application.
Set a buffer of the allocated size, as obtained with the nn::swkbd::GetRequiredWorkBufferSize() function.
This space must be aligned to nn::os::MemoryPageSize.

Examples
SwkbdSimple/SwkbdSimple.cpp, and WebSocketChat/WebSocketChat.cpp.

◆ textCheckWorkBuf

void* nn::swkbd::ShowKeyboardArg::textCheckWorkBuf

The work buffer to use when the application checks the entered text (the entry result string) after input has been accepted on the software keyboard.


Set a buffer of the allocated size, as obtained with the nn::swkbd::GetRequiredTextCheckWorkBufferSize() function. This setting is unnecessary if the application does not need to check the entered text.
