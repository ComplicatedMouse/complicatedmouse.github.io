◆ isValid

bool nn::irsensor::Finger::isValid

Is this finger valid? If false, the remaining properties have no meaning.

It can be false when the finger has not been identified or is folded.

◆ tipDepthFactor

float nn::irsensor::Finger::tipDepthFactor

Estimation of tip depth, relative to hand.

The value is in the range from 0 to 1, where 0 indicates that the finger is away and 1 indicates the point where it is closest to the camera.
