Detailed Description

The list of shape information.

Details
Defines the list of shape information accessed by the pane ahead of the pane.

DataBlockKindShapeList is specified for the kind parameter of blockHeader.

The following data structures are in this structure.

  • :

shapeInfoOffsetTable has the offsets to the various shape information (the offsets from the start of the data block).
