Detailed Description

A structure that stores general network information.

Details
This structure also stores information for networks constructed by general access points, not just the networks created by the ldn library.

channel is the wireless channel where the access point has created a network. One wireless channel is stored among channels 1, 6, and 11 in the 2.4-GHz band and 36, 40, 44, and 48 in the 5-GHz band.

linkLevel stores communication quality between LinkLevelMin and LinkLevelMax. The station calculates the signal strength between itself and the access point based on the communication quality. The signal strength between stations is not reflected in the communication quality, so in some cases many frames may be dropped between stations even with good communication quality. The communication quality is LinkLevelMin if the station was already disconnected from the access point. The access point calculates the communication quality from the minimum signal strength between itself and all stations. However, the communication quality will always be LinkLevelMax if the station does not exist. The following situations may cause poor communication quality.

  • There is a problem with the surrounding signal strength.
  • The access point is too far away from the station.
  • There is an obstacle between the access point and the station.

The communication quality is based on the signal strength of the recently-received frames. Even if the application does not receive all frames, the communication quality is still updated because the system sends and receives control frames. However, note that the communication quality may be slow to update or lack accuracy.

One of the NetworkType values is stored in networkType.
