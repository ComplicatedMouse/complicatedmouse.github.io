◆ RevertTo()

void nn::g3d::BoneVisibilityAnimObj::RevertTo ( ModelObj *  pModelObj) const

Restore the visibility that overwrites own bone visibility animation to the bone resource value.

Parameters
[in]pModelObjModel instance that restores visibility.
Precondition
  • m_pRes != NULL.
  • IsTargetBound()
  • pModelObj != NULL.

Details

  • If you plan to add changes to the bone visibility for a model using multiple bone visibility animations, note that calling this function will overwrite bone visibility with bone resource values.
  • Processing is skipped for bones where Apply was disabled using SetBindFlag().
Examples
G3dDemo/SkeletalAnimation.cpp.
