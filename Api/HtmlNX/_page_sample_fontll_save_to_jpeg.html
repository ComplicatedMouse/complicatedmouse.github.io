This sample program shows how to use the fontll library to create glyph (character) images.

Overview

This sample (FontllSaveToJpeg) creates glyph images.

File Structure

This sample program is located in Samples/Sources/Applications/FontllSaveToJpeg/ .

Required Environment

This sample program saves JPEG files using the host PC file system (nn::fs::MountHost). As a result, it cannot be executed in an environment that cannot use the host PC file system, such as when it is installed on development hardware and started on its own.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it. After execution is complete, the glyph image is saved to the folder that contains the executable file for the sample (under Samples/Sources/Applications/FontllSaveToJpeg/Binaries).

Description

This sample loads font data to generate glyph images, converts the images to JPEG, and saves the images.
