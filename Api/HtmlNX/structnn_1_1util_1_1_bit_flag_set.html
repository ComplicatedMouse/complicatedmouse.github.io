Detailed Description

template<int N, typename Tag = void>
struct nn::util::BitFlagSet< N, Tag >

A type for handling a POD bit flag set.

Details
It is possible to maintain POD state while safely using bit flag operation features equivalent to std::bitset(). The data members held by BitFlagSet depend on the number of bits set in the N template parameter. If N <= 32, it stores an uint32_t array of length 1. For any larger values, it stores an uint64_t array of length (N + 64 â€“ 1) / 64. Note the higher cost of passing arguments and larger stack size when a large N value is specified. Ensures that the value is a trivial type and can be handled as a standard layout type. Because it is a trivial type, it must be explicitly initialized. Use BitPack for types that might store values other than bit flags.

Template Parameters
NThe number of bits stored by BitFlagSet.
TagTag to instantiate as a different type.
