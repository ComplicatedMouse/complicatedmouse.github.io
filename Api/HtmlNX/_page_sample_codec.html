Codec Library Samples

Describes samples of the codec library for audio and video compression.

Introduction

Describes samples of the codec library for audio and video compression. The codec library's features for audio and video compression are in the nn::codec namespace.

Samples

Sample code is located in the directories starting with the prefix Codec under Samples/Sources/Applications/. The following table lists the samples.

Page Content
AAC Decoder This section describes the AAC decoder sample program.
Measuring Opus Encoder and Decoder Processing Times Sample Description for the sample program to measure the Opus encoder and decoder processing times.
Opus Complexity This is a sample program for testing the impact of Opus complexity on codec quality and performance in various settings.
Opus Decoder Sample Opus decoder sample program.
Opus Encoder Description of the Opus decoder sample program.
Opus Multistream Decoder Sample This is a description of the sample program of the Opus multistream decoder.
