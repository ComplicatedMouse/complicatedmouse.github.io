size_t nn::ui2d::BuildResultInformation::requiredUi2dConstantBufferSize

The size of the constant buffer for ui2d needed for rendering the initialized layout data.

If the shape is set to something other than a square, this includes the sizes of the vertex buffer and index buffer for the mesh.
