Initializes a descriptor pool.

Parameters

[in]pMemoryPoolPointer to the memory pool where the descriptor pool is placed.

[in]memoryPoolSizeThe size of the memory pool, in bytes.
Precondition
  • pDevice != NULL.
  • pDevice is initialized.
  • The descriptor pool is not initialized.
  • pMemoryPool != NULL.
  • memoryPoolSize is at least the size calculated by CalculateDescriptorPoolSize.
Postcondition
  • The descriptor pool is initialized.
