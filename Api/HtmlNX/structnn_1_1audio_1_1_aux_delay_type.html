#include <nn/audio/audio_AuxUtilityEffectDelayTypes.h>

Detailed Description

Utility for the Delay effect applied to nn::audio::AuxType.

Details
This utility uses the sample data obtained by nn::audio::AuxType as input to perform the Delay effect process. It specifies the sample data obtained by the nn::audio::ReadAuxSendBuffer() function as the input for this effect, and produces the result of applying the Delay effect, which emulates the reverberation of sounds in a closed space. The delay effect implemented this way is identical to the process performed by n::audio::DelayType. However, this effect process is performed on the CPU thread that called nn::audio::ProcessAuxDelay().

Directly altering member variables is prohibited. Use the main structure through the following functions.
