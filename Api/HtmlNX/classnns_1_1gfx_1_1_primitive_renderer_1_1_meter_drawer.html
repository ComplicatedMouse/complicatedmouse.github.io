◆ SetScale()

void nns::gfx::PrimitiveRenderer::MeterDrawer::SetScale ( int  scale)

Sets the scale that indicates how many segments to render for the standard frame rate.

Parameters
[in]scaleThe scale.

Details
If 1 is specified, only processing that fits within the standard frame rate is rendered. Specify a value of 2 or more for the scale if the processing does not fit within the standard frame rate. If 0 is specified, the scale is adjusted automatically to match the load.

Examples
FrameworkDemo/Main.cpp.

Renders the load meter.

Parameters
[in]pCommandBufferPointer to the command buffer for storing the rendering command.
[in]pRendererPointer to the PrimitiveRenderer.
[in]pRootMeterPointer to the parent load meter to render. (The attached child load meters are also rendered.)

Details
Use MatrixIdentity to set the default values for the model, view, and projection matrices set internally for PrimitiveRenderer.
