Detailed Description

This class manages the depth stencil state, blend state, randomizer state, and shading model of the material.

nn::g3d::ResMaterial *  pResMaterial,

)

Initializes RenderMaterial.

Parameters
[in]pDevicePointer to the device.
[in]pResMaterialPointer to the target material.
[in]pResShaderArchivePointer to nn::g3d::ResShaderArchive.
Precondition
  • RenderMaterial is not initialized.
Postcondition
  • RenderMaterial is initialized.
Returns
Returns true if initialization was successful, or false otherwise.

Details
Finds the shading model that was linked to pResMaterial from pResShaderArchive and performs initialization. Initialization is not performed if an appropriate shading model was not found in the shader archive.

After initialization, the following state is applied during material rendering.

The blend state is the nn::gfx::BlendState object that is initialized after calling nn::gfx::BlendStateInfo::SetDefault() and specifying one nn::gfx::BlendTargetStateInfo that was subject to the nn::gfx::BlendTargetStateInfo::SetDefault() function.

The depth stencil state is the nn::gfx::DepthStencilState object that is initialized after calling the nn::gfx::DepthStencilStateInfo::SetDefault() function and configuring the following.

  • SetDepthTestEnabled(true)
  • SetDepthWriteEnabled(true)

The rasterizer state is the nn::gfx::RasterizerState object that is initialized after calling the nn::gfx::RasterizerStateInfo::SetDefault() function and configuring the following.

  • SetScissorEnabled(true)

SetBlendState(), SetDepthStencilState(), and SetRasterizerState() can be used to change to a suitable state during rendering.
