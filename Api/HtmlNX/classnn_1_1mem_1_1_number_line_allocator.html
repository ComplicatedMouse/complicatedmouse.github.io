Detailed Description

The class for integer allocators that can retain separate regions for management and allocation or deallocation.

Details
The nn::mem::NumberLineAllocator class can assign any integer region represented by the indexes 0 up to 0xFFFFFF.
The NumberLineAllocator management region can be managed using a dedicated callback and separated from the region handled by NumberLineAllocator.

Member Enumeration Documentation

◆ AllocationMode

enum nn::mem::NumberLineAllocator::AllocationMode

This mode is for allocation and release.

Details
Specifies the behavior when allocating and releasing segments.
By default, a value of AllocationMode_NoOption is specified.

Enumerator
AllocationMode_NoOption 

If a segment longer than 256 is released, the region is combined with adjacent empty regions.

AllocationMode_CoalesceAdjacentArea 

Constantly combine with adjacent empty regions upon release.

While the running efficiency is lower than AllocationMode_NoOption, it can suppress fragmentation within the allocator.

◆ GetAllocatableSize()

int nn::mem::NumberLineAllocator::GetAllocatableSize ( ) const

Gets the largest size that can be allocated from the allocator.

Returns
Returns the size of allocatable memory.
Precondition
The allocator is initialized.

Details
Gets the largest size that can be allocated from the allocator.
When there are multiple available memory regions, gets the size of the largest among them.

The size returned by this API will always be allocatable. However, sometimes the algorithm might allocate a region greater than the size returned by this API.

This function is thread-safe if true was passed to the isThreadSafe parameter when the constructor was run or Initialize() was called.

◆ Dump()

void nn::mem::NumberLineAllocator::Dump ( uint64_t  offset,
int  factor 
) const

Displays the information in the allocator.

Parameters
[in]offsetThe offset from the starting index number when displaying the information.
[in]factorA constant used to multiply the length of the segment.
Precondition
The allocator is initialized.

Details
Returns the internal state of the allocator. When offset and factor are specified, the information displayed for the allocator is offset from the starting index by the amount specified by offset, and the segment length handled by the allocator is multiplied by factor.
If offset is set to 0 and factor is set to 1, the information for the allocator is displayed without any modification.

For more information about the export format, see NumberLineAllocator in NintendoSDK Documents.
