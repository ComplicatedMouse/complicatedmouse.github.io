Detailed Description

Shader for creating textures.

Details
The shader class for creating Mask and Faceline textures. Prepared as a utility to simplify the creation of Mask and Faceline textures. It is okay to prepare a shader in the application and not use this class.

Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.

Initializes TextureShader.

Attention
pMemory must be retained until the call to TextureShader::Finalize().
You cannot reinitialize an instance of TextureShader that has been initialized already. Call this function after calling Finalize().
The initialized resShaderFile must be retained in the initialized state until TextureShader::Finalize() is called.
Parameters
[in]pMemoryThe memory that TextureShader will use.
[in]memorySizeSpecifies the size of pMemory.
[in]resShaderFileAn initialized shader file.
[in]shaderCodeTypeThe shader code type.
[in]pDeviceSpecifies the device.
[in]infoInformation for initializing TextureShader.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL
Postcondition
If successful, TextureShader::IsInitialized() = true.
Examples
MiiCompressedTexture/MiiCompressedTexture.cpp, MiiDatabase/MiiDatabase.cpp, MiiDynamicMask/MiiDynamicMask.cpp, MiiG3dHeadwear/MiiG3dHeadwear.cpp, MiiG3dSimple/MiiG3dSimple.cpp, MiiHeadwearExample/MiiHeadwearExample.cpp, MiiMipMap/MiiMipMap.cpp, MiiPremultipliedAlphaIcon/MiiPremultipliedAlphaIcon.cpp, MiiShaderExample/MiiShaderExample.cpp, MiiSimple/MiiSimple.cpp, MiiSimpleIcon/MiiSimpleIcon.cpp, MiiSimpleIconBody/MiiSimpleIconBody.cpp, MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp, MiiSystemResource/MiiSystemResource.cpp, and MiiTransparentAdjuster/MiiTransparentAdjuster.cpp.
