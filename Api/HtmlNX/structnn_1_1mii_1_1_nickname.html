Detailed Description

The structure representing the Mii nickname.

Details
The structure representing the Mii nickname, including the terminating character. The characters stored in this structure follow the endianness of the applicable platform.

Parameters
[in]fontRegionThe FontRegion of this nickname.
[in]fontRegionMaskThe mask for the supported FontRegion.
Precondition
IsValid() == true.

Details
Used in the CharInfoAccessor::GetNickname() function. Does nothing if fontRegion is included in fontRegionMask. If not included, replaces characters other than ASCII characters with "?" and returns the name.
