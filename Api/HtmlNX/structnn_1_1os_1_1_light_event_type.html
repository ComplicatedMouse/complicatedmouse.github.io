nn::os::LightEventType Struct Reference

This structure defines a lightweight event object. More...

#include <nn/os/os_LightEventTypes.h>

Detailed Description

This structure defines a lightweight event object.

Details
Directly altering member variables is prohibited. Use the main structure through the following functions.

  • InitializeLightEvent()
  • FinalizeLightEvent()
  • SignalLightEvent()
  • WaitLightEvent()
  • TryWaitLightEvent()
  • TimedWaitLightEvent()
  • ClearLightEvent()

To use these functions, include nn/os.h.

Also see the nn::os::LightEvent utility class.
