The class for icons with certain body physiques.

Details

Initializes VariableIconBody.

Parameters
[in]pMemorySpecifies the memory for VariableIconBody to use.
[in]memorySizeSpecifies the size of pMemory.
[in]pDeviceSpecifies the device.
[in]pMemoryPoolSpecifies the memory pool VariableIconBody will use.
[in]memoryPoolOffsetSpecifies the offset to the memory pool VariableIconBody will use.
[in]memoryPoolSizeSpecifies the size of the memory pool VariableIconBody will use.
[in]pBodyA pointer to the body resource file.
[in]bodySizeThe size of pBody.
Returns
Returns nn::ResultSuccess if successful.
Precondition
pMemory != NULL. && pDevice != NULL && pMemoryPool != NULL && The offset and size are normal. && charInfo is not corrupted.
Postcondition
If successful, VariableIconBody::IsInitialized() = true.

Details
You can free pBody after initialization.

Attention
pMemory and pMemoryPool must be retained until VariableIconBody::Finalize().
You cannot re-initialize an instance of VariableIconBody that has been initialized already. Call this function after calling Finalize().
You need to flush the memory pool after calling this function when the memory pool that was passed at initialization was CpuCached.
Examples
MiiSimpleVariableIconBody/MiiSimpleVariableIconBody.cpp.
