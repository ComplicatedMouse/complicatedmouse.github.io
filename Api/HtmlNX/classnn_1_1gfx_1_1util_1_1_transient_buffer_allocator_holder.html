Detailed Description

template<int THistoryLengthMax>
class nn::gfx::util::TransientBufferAllocatorHolder< THistoryLengthMax >

Class that wraps TransientBufferAllocator and the memory block needed for its initialization, and creates an object.

Template Parameters
THistoryLengthMaxThe maximum number of TransientBufferAllocator history entries to record.

Initializes TransientBufferAllocator, which will be wrapped with this object.

Parameters
[in]pBufferThe buffer.
[in]baseOffsetThe start of the offset for the region to manage.
[in]sizeThe size, in bytes, of the region to manage.
[in]allocatableAlignmentMaxThe maximum alignment required for the region allocated from this allocator.
Precondition
This object must be uninitialized.
The preconditions for each parameter are the same as those for TransientBufferAllocator::Initialize().
Postcondition
This object is initialized.
See also
TransientBufferAllocator::Initialize()

Details
Initializes an instance of TransientBufferAllocator, to be wrapped by this object using the specified parameters. The memory region wrapped in this object is used for the memory block to use for initialization. HistoryLengthMax can specify the maximum number of history entries. Calling this instance for the same instance is not thread-safe.

Details
The initialization and destruction of the obtained TransientBufferAllocator occurs with the Initialize() and Finalize() functions for this object. The lifetime of the obtained TransientBufferAllocator is the same as the lifetime for this object. This function is thread-safe.
