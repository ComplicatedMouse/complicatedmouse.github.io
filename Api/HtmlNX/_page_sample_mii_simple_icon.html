This sample program renders Mii icons.

Overview

This sample program creates a Mii icon using the shader provided in the Face Library and then renders that Mii icon. It presents the procedure for creating Mii icons using nn::mii::IconShader.

File Structure

This sample program is located in Samples/Sources/Applications/MiiSimpleIcon/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program creates and displays a Mii icon.

Conversion Process

  • /Samples/Sources/Applications/MiiSimpleIcon/Resources/SampleShader.glsl

A simple shader for rendering icons.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Load the Mii resource file.
  • Initialize the Mii resource (nn::mii::Resource).
  • Get nn::mii::CharInfo.
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Initialize the texture to which the icon will be written.
  • Initialize the texture for the depth stencil.
  • Initialize the shader for the icon texture (nn::mii::IconShader).
  • Initialize nn::mii::IconGpuBuffer.
  • Create a sampler for the Mii.
  • Create the texture view to which the icon will be written and register it to the descriptor pool.
  • Use nn::mii::TextureShader to render the Faceline and Mask textures.
  • Use nn::mii::IconShader to render the Mii icon.
  • Render the Mii icon texture.
