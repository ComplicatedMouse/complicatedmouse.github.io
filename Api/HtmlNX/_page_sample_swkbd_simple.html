A sample program that starts up the software keyboard applet using the nn::swkbd library.

Overview

Start the software keyboard applet.

File Structure

This sample program is located in Samples/Sources/Applications/SwkbdSimple/ .

Required Environment

This sample program can only be built and run on the NX platform.

How to Use

After executing the sample program, press the A Button to start the software keyboard applet.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

The processing flow is described in the nn::swkbd library programming manual.
