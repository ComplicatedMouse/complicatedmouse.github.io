Overview

This sample program renders Mii characters the same as the MiiEdit applet. It demonstrates how to use nn::mii::Material to render Mii characters the same as the MiiEdit applet.

File Structure

This sample program is located in Samples/Sources/Applications/MiiShaderExample/ .

Required Environment

Nothing specific.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program renders Mii characters in the same way as the MiiEdit applet.

Conversion Process

  • /Samples/Sources/Applications/MiiShaderExample/Resources/SampleShader.glsl

A shader that renders Mii characters in the same way as the MiiEdit applet.

Processes of the Sample Program

This sample program has the following flow.

  • Initialize the various resources for drawing.
  • Create a sampler for the Mii.
  • Get nn::mii::CharInfo.
  • Initialize the shader for the Faceline and Mask textures (nn::mii::TextureShader).
  • Initialize the Mii resource (nn::mii::Resource).
  • Initialize material for rendering the Mii (nn::mii::Material).
  • Initialize nn::mii::CharModel.
  • Initialize nn::mii::Faceline.
  • Initialize nn::mii::Mask.
  • Assign descriptor slots.
  • Initialize nn::mii::FacelineGpuBuffer.
  • Initialize nn::mii::MaskGpuBuffer.
  • Render the Faceline texture.
  • Render the Mask texture.
  • Initialize the shader for Mii rendering.
  • Initialize the drawing process for Mii rendering.
  • Set the constant buffers for Mii rendering.
  • Render the Mii.
