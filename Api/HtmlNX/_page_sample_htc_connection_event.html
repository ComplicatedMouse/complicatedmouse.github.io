Description of the sample program for getting the communication state with the host.

Overview

This section describes a sample program using an API for getting the communication state with the host.

For more information, see the HTC API Reference and documentation.

File Structure

This sample program is located in Samples/Sources/Applications/HtcConnectionEvent/ .

Required Environment

Requires the installation of Target Manager.

How to Use

Nothing specific.

Important Information

This sample program does not display anything on the screen. The execution results are output to a log.

Execution Procedure

Build the sample program and then run it.

Sample Program Description

This sample program has the following flow.

  • Use the nn::htc::BindHostConnectionEvent() and nn::htc::BindHostDisconnectionEvent() functions to get the connection notification event and disconnection notification event with Target Manager.
  • Wait for the connection notification event to be signaled.
  • Use the nn::htc::GetEnvironmentVariableLength() function to get the environment variable size from the host.
  • Wait for the disconnection notification event to be signaled.

This sample operates according to the connection state to Target Manager, so verify as required by connecting to or disconnecting from Target Manager.
