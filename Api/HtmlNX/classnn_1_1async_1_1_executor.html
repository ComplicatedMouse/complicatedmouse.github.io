Detailed Description

The abstract base class representing the executor of asynchronous processes.

Details
Executor is the abstract base class with the functions for executing asynchronous processes. Normally, a derived class of Executor manages the memory resources and thread resources and executes the asynchronous processes by allocating thread resources to the objects (tasks) representing those asynchronous processes. You can get the result using a Future object.

Currently, there is no public interface of the Executor class. For more information, see the reference for the derived class provided by the async library.
