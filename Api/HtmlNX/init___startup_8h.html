void nninitStartup ( void  )

Startup initialization routine.

Details
This function is used by the application for memory management initialization.

This function is not called by applications, but is implemented in the application. Define this function using C linkage. If this is function has not been uniquely defined in the application, the default implementation as defined by the libnn_init library is used.

Depending on the use of C++, memory allocation may be needed before nnMain(), such as by a static object constructor. The application's proprietary memory management system must be initialized beforehand. This function is called before the static object constructor, allowing for the memory management system to be initialized in it.

This function is called when the initialization required in order to run the application has not been completely executed, limiting the use of most of the SDK APIs.

For the features that can be used in nninitStartup(), see List of available APIs from nninitStartup().
