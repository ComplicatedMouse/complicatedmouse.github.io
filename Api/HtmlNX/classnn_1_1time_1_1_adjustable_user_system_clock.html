Detailed Description

Class representing a user clock that can be adjusted at any time according to the system time.

Details
This clock indicates the current time held by the device. You can get a value that can be converted into a time expression that a human can understand. The clock time is not affected by modifications to the time made by the user or the system unless the Adjust function is called, which allows rollbacks and large increases in clock time to be managed appropriately and makes this clock easy to use in applications.

Synchronize the system time (nn::time::StandardUserSystemClock) by calling the Adjust function, at which point the indicated time may be rolled back or forward by more than the actual elapsed time. In addition, the clock is incremented when the device is powered down, and the value is reset when the power supply is lost as a result of a dead battery. When this happens, you can detect clock manipulations and resets, but you cannot rely on the calculation results for the time elapsed before and after those events.

This class's member functions are thread-safe. This class satisfies the standard C++ TrivialClock requirements and provides features equivalent to std::chrono::system_clock. You cannot, however, detect clock manipulations or a value reset when you are only using the C++ standard interface.

◆ Adjust()

static void nn::time::AdjustableUserSystemClock::Adjust ( )
static

Adjusts the clock time to the latest system time.

Precondition
  • The time library is initialized.

Details
Adjusts the time that can be obtained with the GetCurrentTime function to match the latest system time.

Calling this function may cause the time returned by the GetCurrentTime function to be rolled back or rolled forward by a large amount.

Details

Calling the Adjust function may change the value returned by this function.
