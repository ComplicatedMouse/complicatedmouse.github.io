Detailed Description

Class for automatically starting and ending measurements using constructors and destructors.

Precondition
LoadMeterCenter is initialized.

Details
Measures using the CpuMeter stored in LoadMeterCenter that corresponds to the running CPU core. Use this function only if the running CPU core is not switched while the section is being measured.
