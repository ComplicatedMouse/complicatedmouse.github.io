Detailed Description

This context class increases the animation speed.

The speed of animation is increased by recording the number of frames pointed to by the keys before and after the frame that was evaluated last time, determining whether a key search is necessary for the next evaluation time, and performing a key search as necessary. It cannot be used when reverse playback or curve is baked.
