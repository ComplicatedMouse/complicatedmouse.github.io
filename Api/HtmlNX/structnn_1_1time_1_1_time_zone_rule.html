#include <nn/time/time_TimeZoneRule.h>

Detailed Description

The type representing time zone rules.

Details
Loaded by the LoadTimeZoneRule function and used for converting back and forth between local time and absolute time.

The internal specification might be changed in the future. Do not store this in save data or elsewhere for use at the next startup.
