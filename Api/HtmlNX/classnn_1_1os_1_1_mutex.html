Detailed Description

Class for handling mutexes.

Details
Mutexes are synchronization objects for implementing exclusive control among multiple threads.

The nn::os::Mutex class is a utility class that wraps calls to synchronization control functions related to nn::os::MutexType objects. The member functions provided by this class are implemented by issuing functions like nn::os::LockMutex() that take the nn::os::MutexType type as arguments.

This class fulfills the BasicLockable and Lockable requirements in the C++ standard, so it can be used as a type argument for class templates such as std::lock_guard and std::unique_lock. As a result, multiple member functions exist that perform the same operations, such as Lock() and lock().

Examples
AtkEffect/AtkEffect.cpp, AudioEchoback/AudioEchoback.cpp, and ImguiDemo/ImguiDemo_UserInputs.cpp.
