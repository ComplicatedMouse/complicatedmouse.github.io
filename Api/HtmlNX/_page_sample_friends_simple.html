Description of a sample program to use the basic features of the friends library.

Overview

This program is a sample for introducing the basic features of the friends library. Also see the API reference for the nn::friends library.

File Structure

This sample program is located in Samples/Sources/Applications/FriendsSimple/ .

Required Environment

You must create a user account that is associated with a network service account. Also requires an environment where network connections are possible.

How to Use

Nothing specific.

Important Information

Nothing specific.

Execution Procedure

Build the sample program and then run it.

Description

This sample program has the following flow.

  • Open the user account.
  • Ensure the ability to get the friend list.
  • Get the friend list.
  • Get the list of friend profiles.
  • Ensure the ability to get the blocked-user list.
  • Get the list of blocked users.
  • Close the user account.
