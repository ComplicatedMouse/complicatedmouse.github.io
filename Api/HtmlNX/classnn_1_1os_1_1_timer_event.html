Detailed Description

Class for handling timer events.

Details
Timer events are a means of synchronization that you can use to postpone signaling for the specified amount of time, in addition to the normal event features.

The nn::os::TimerEvent class is a utility class that wraps calls to synchronization control functions that use nn::os::TimerEventType objects. The member functions provided by this class are implemented by issuing functions like nn::os::WaitTimerEvent() that take the nn::os::TimerEventType type as arguments.

Examples
OeDisableAutoSleep/OeDisableAutoSleep.cpp, OeIdlePolicy/OeIdlePolicy.cpp, and WebSocketEcho/WebSocketEcho.cpp.

◆ Stop()

void nn::os::TimerEvent::Stop ( )

Stops the timer event's timer.

Details
Stops the timer event's timer. This API function does not change the timer event's signaled state. Also, even if there are waiting threads, this API function does not wake them. For more information, see nn::os::StopTimerEvent().

◆ Wait()

void nn::os::TimerEvent::Wait ( )

Waits for a timer event to enter a signaled state.

Details
Waits for a timer event to enter a signaled state. If the timer event is set to nn::os::EventClearMode_AutoClear, the timer event is cleared to a non-signaled state as soon as a signaled state is detected. For more information, see nn::os::WaitTimerEvent().

Examples
OeDisableAutoSleep/OeDisableAutoSleep.cpp.

bool nn::os::TimerEvent::TryWait ( )

Polls whether timer events are in the signaled state.

Returns
Returns whether polling of the timer event was successful.

Details
Polls whether a timer event is in a signaled state. Returns true if the timer event is in a signaled state; otherwise, returns false. If the timer event is set to nn::os::EventClearMode_AutoClear, the timer event is cleared to a non-signaled state as soon as a signaled state is detected. For more information, see nn::os::TryWaitTimerEvent().
