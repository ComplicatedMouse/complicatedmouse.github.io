Detailed Description

The data block that stores the list of capture texture image reference information.

Details
The list of capture textures referenced by the pane is defined ahead of the pane.

DataBlockKindCaptureTextureList is specified for blockHeader[kind].

blockHeader[size] includes the data up to nameStrPool.

The following data structure uses this structure.

  • ResCaptureTexture textures[texCount]
  • uint8_t nameStrPool[]
