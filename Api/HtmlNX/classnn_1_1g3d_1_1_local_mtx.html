Detailed Description

Class that indicates a bone local matrix.

When flag is 0, scale and mtxRT are used as is and are calculated. When the next bit is enabled, the library performs the special calculation or optimization calculation that corresponds to that bit.

ResBone::Flag_SegmentScaleCompensate
ResBone::Flag_ScaleUniform
ResBone::Flag_ScaleVolumeOne
ResBone::Flag_RotateZero
ResBone::Flag_TranslateZero
