int nn::pdm::QueryApplicationPlayStatistics ( ApplicationPlayStatistics  outValues[],
const ApplicationId  applicationIds[],
int  count 
)

Gets play data for the specified application.

Parameters
[out]outValuesThe location for storing the play data.
[in]applicationIdsThe application ID of the application for which to get play data.
[in]countThe number of elements in outValues and applicationIds.
Returns
The number of play data items stored in outValues.
Precondition
  • outValues != NULL.
  • applicationIds != NULL.
  • count > 0.
  • PlayLogQueryCapability in the NMETA file is set correctly.
Postcondition
  • Return value >= 0 && Return value <= count.
  • If count > 0,
    • Valid ApplicationPlayStatistics values are stored in outSizes[0] to outSizes[Return value - 1].

Details
Gets play data for the specified application on the console where this function is executed. To get play information for other applications, you must set Application/PlayLogQueryCapability in the NMETA file. For more information, see Writing the NMETA File in NintendoSDK Documents.

  • Return value >= 0 && Return value <= count.
  • If count > 0,

Details
Gets play data for the specified application for the specified user on the console where this function is executed. To get play information for other applications, you must set Application/PlayLogQueryCapability in the NMETA file. For more information, see Writing the NMETA File in NintendoSDK Documents.
