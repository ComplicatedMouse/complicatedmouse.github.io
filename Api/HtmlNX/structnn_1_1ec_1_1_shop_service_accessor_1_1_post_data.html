Detailed Description

Parameter container for the post data field.

Details
The container class that specifies the memory region where the post data field resides. For the region length specified for size, you can use the result string of the std::strlen() function if you are using a terminated URL-encoded string.

NN_EXPLICIT_OPERATOR nn::ec::ShopServiceAccessor::PostData::bool ( ) const

Evaluates the status of the container settings for availability.

Returns
Returns the evaluation result.
Return Values:

true The container can be used as a post data container.

false The container cannot be used as a post data container.

Details
This process does evaluate the content of the post data string. Rather, it evaluates whether the values of the member elements in the container can be used as input.
