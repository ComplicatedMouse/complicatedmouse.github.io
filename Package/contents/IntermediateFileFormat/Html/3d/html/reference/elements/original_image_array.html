<original_image_array>

Description

An element of the <original_image> array.

Parent Elements

  • <texture>: Element specified in texture files (FTXA and FTXB files).

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in the order that face and slice_index attributes are defined in <original_image>.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_image> [1 – 65,535]: Element that represents 2D image data based on a <texture> element.
