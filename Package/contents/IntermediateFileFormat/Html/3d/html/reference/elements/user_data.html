<user_data>

Description

Element used to store original user data that can be obtained at runtime.

Parent Elements

  • <user_data_array>: A collection of <user_data> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
name string
[0-9A-Za-z_]+		 The name of the user data.
Configurable

Description

The following elements appear. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <user_int> [1]: Element that stores integer type user data.
  • <user_float> [1]: Element that stores float type user data.
  • <user_string> [1]: Element that stores an ASCII string of user data.
  • <user_wstring> [1]: Element that stores a Unicode string of user data.
  • <user_stream> [1]: Element that stores stream type user data.
