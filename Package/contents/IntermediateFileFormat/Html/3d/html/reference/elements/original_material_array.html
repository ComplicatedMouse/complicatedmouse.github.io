<original_material_array>

Description

A collection of <original_material> elements.

Parent Elements

  • <model>: Element specified in model files (FMDA and FMDB files).

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in the character code order of <original_material> mat_name.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_material> [1 – 65,535]: Element that defines shader parameter-related material settings in the DCC tool.
