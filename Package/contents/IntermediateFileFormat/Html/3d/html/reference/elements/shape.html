<shape>

Description

Element used to define the shape of a model.

Parent Elements

  • <shape_array>: A collection of <shape> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
name string
[0-9A-Za-z\-\._]+		 Represents the name of the element.
It is paired with the reference source <vertex_shape_anim> shape_name. Specifies the "bone name_material name" used for export from a DCC tool. This bone name is the same as the node name in the DCC tool. It is converted to the parent bone of the shape. When names conflict, a number such as "_1 " or "_2" is added to the end of the name.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shape_info> [1]: Element that represents information related to <shape>.
  • <mesh_array> [1]: A collection of <mesh> elements.
  • <key_shape_array> [0 or 1]: A collection of <key_shape> elements.
