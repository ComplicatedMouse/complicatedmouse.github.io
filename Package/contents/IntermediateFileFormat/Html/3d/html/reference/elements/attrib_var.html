<attrib_var>

Description

Element that defines shader option variables.

DCC tools do not output this element.

Parent Elements

  • <attrib_var_array>: A collection of <attrib_var> elements.

Attributes

Name Types and Values Description
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Array of vertex attribute variables.
It is paired with the reference source <attrib_assign> id.
hint string
[0-9A-Za-z_]*		 Represents hint information used when binding vertex attribute variables.
When applying a hint according to output from the DCC tool, specify this information using the same command naming conventions as used with the <vtx_attrib> hint.
type enum
int
int2
int3
int4
uint
uint2
uint3
uint4
float
float2
float3
float4		 Represents the logical type of the vertex attribute variable.
Definition is the same as <vtx_attrib> type.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <vertex_symbol> [1]: Element that defines variable symbols used with the vertex shader.
  • <ui_label> [0 or 1]: Element that defines the UI labels for editing variables.
  • <ui_comment> [0 or 1]: Element that defines the UI comment for editing variables.
  • <ui_group> [0 or 1]: Element that defines the UI groups for editing variables.
  • <ui_order> [0 or 1]: Element that defines the order of precedence for an element while editing variables in the UI.
  • <ui_editable> [0 or 1]: Element that defines whether variables can be edited using the UI.
