<shader_param_anim_array>

Description

A collection of <param_anim> elements.

DCC tools do not output this element.

Parent Elements

  • <per_material_anim>: Element that specifies the target material of a material animation.

Attributes

Name Types and Values Description
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.

Description

In the case of child elements, the <param_anim> id elements are sorted in the order that shader id elements are defined.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <param_anim> [1 – 65,535]: Element that defines parameters for a shader parameter animation.
