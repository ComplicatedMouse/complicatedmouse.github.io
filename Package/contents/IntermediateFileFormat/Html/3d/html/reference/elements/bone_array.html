<bone_array>

Description

A collection of <bone> elements.

Parent Elements

  • <skeleton>: Element that brings together <bone> elements used to define the coordinate system.

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted according to depth priority beginning from the root bone. Siblings are sorted in the order of the character code used for <bone> name.
The sort order influences runtime operations such as calculating the world matrix.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <bone> [1 – 65,535]: Element that defines the coordinate system used for features such as skinning.
