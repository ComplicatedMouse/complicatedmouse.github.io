<user_int>

Description

Element that stores variable type user data.

Parent Elements

  • <user_data>: Element that stores original user data that is obtained at runtime.

Attributes

Name Types and Values Description DCC
count int
0 – MAX		 Specifies the number of instances of data.
Configurable

Description

Specifies the number of integer values indicated by count.
