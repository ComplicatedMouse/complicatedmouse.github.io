<shader_option>

Description

This element defines shader options.

DCC tools do not output this element.

Parent Elements

  • <shader_option_array>: A collection of <shader_option> elements.

Attributes

value_policy enum
unset

default		 Specifies the reference resolution settings for the value.
unset Indicates that it is not explicitly set. Handled as refer when resolving a reference.
refer Uses the value of the parent material. Handled as override not referencing a parent material.
override Uses the value of the local material.
default Uses the default value of the shader definitions.
child_restriction enum
unset
none
force_refer
disallow_refer
value_policy_to_refer		 Specifies the restrictions for resolving references of parameters that have the same identifier in the child material.
unset Indicates that it is not explicitly set. Handled as none when the reference is resolved.
none No limit.
force_refer Fixes the value of the child material as the value of the material that configured this setting.
disallow_refer Prohibits setting refer as the value_policy of the child material. Handled as override when refer is set.
value_policy_to_refer Handles the value_policy of the child material as refer.
child_value_restriction string
-		 The restrictions for the value range of parameters that have the same identifier in the child material. Specifies using the same format as choice in <option_var>.
