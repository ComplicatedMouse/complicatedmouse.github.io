tex_name

Description

Element that specifies the name of the <texture> used in the sampler.

Parent Elements

  • <sampler>: Element that defines texture samplers.

Attributes

Name Types and Values Description DCC
value string
[0-9A-Za-z\-\._]*		 Represents the name of the <texture> to be used by the sampler.
Specify a name based on the texture filename, excluding the extension.
