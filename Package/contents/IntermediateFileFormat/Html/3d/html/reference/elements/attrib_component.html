<attrib_component>

Description

Element that defines constituent vertex attribute variables.

DCC tools do not output this element.

Parent Elements

  • <attrib_component_array>: An array of <attrib_component> elements.

Attributes

Name Types and Values Description
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 The identifier of the constituent vertex attribute variables.
It is paired with the reference source <attrib_assign> id.
value int
MIN – MAX		 The defined component value within the shader.
Example: 1 when layout(location = 3, component = 1) in vec2 uv;.

Description

There is no content.
