<bone_anim_array>

Description

A collection of <bone_anim> elements.

Parent Elements

  • <skeletal_anim>: Element specified in skeletal animation (FSKA and FSKB) files.

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted according to depth priority beginning from the root bone. Siblings are sorted in the order of the character code used for <bone_anim> bone_name.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <bone_anim> [1 – 65,535]: Element that animates each of the scale, rotation, and translation attributes included under <bone>.
