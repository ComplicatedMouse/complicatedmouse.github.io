<input_array>

Description

A collection of <input> elements.

Parent Elements

  • <vtx_buffer>: Element that defines the vertex buffer.

Attributes

Name Types and Values Description DCC
length int
1 – 255		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are not sorted. They are saved in the order specified by the user.
When exporting from a DCC tool, output follows the order defined in <input> attrib_index.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <input> [1 – 255]: Element that specifies the order and quantization type when vertex attributes are included in the vertex buffer.
