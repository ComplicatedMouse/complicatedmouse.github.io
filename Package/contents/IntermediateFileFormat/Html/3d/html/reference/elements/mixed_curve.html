<mixed_curve>

Description

An element that represents a mixture of hermite curves, linear curves, and step curves. Do not use this element because it is currently being validated.

Parent Elements

  • <bone_anim_target>: Element that animates each parameter included under <bone>.
  • <param_anim_target>: Element that animates shader parameters.
