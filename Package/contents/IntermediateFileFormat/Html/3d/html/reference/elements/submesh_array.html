<submesh_array>

Description

A collection of <submesh> elements.

Parent Elements

  • <mesh>: Element that represents vertex indices used to define a polygon.

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are not sorted. They are saved in the order specified by the user.
A DCC tool only ever outputs one of these elements.
Range specifications using the offset and count attributes of <submesh> must specify the entire range of vertex indices in order starting from zero.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <submesh> [1 – 65,535]: Element that defines rendering calls.
