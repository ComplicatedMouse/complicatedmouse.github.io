<param_anim_target>

Description

Element used to animate shader parameters.

DCC tools do not output this element.

Parent Elements

  • <param_anim>: Element that defines parameters for a shader parameter animation.

Attributes

Name Types and Values Description
component_index int
0 – 15		 Specifies whether the nth value in the shader parameter should be animated.
base_value float
MIN – MAX		 Specifies initial animation values.
Specify the 0 frame value when an animation is attached.

Description

There are no contents when there is no animation and the target is set using a fixed value for base_value.

The following elements appear. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <hermite_curve> [0 or 1]: Element that represents hermite curves that will be smoothed between keys.
  • <linear_curve> [0 or 1]: Element that represents linear curves for which linear interpolation is performed between keys.
  • <step_curve> [0 or 1]: Element that represents step curves for which interpolation is not performed between keys.
  • <mixed_curve> [0 or 1]: Element that contains a mixture of hermite curves, linear curves, and step curves. Do not use this element because it is currently being validated.
