<per_material_anim>

Description

Element specifying materials that will undergo material animation.

Parent Elements

  • <per_material_anim_array>: A collection of <per_material_anim> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
mat_name string
[0-9A-Za-z\-\._]+		 The name of the <material> to animate.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shader_param_anim_array> [0 or 1]: A collection of <param_anim> elements.
  • <tex_pattern_anim_array> [0 or 1]: A collection of <pattern_anim> elements.
  • <material_visibility_anim> [0 or 1]: Element that animates the visibility attributes included under <material_info>.
