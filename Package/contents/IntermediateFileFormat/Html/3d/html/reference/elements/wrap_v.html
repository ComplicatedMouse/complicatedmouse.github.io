<wrap_v>

Description

Element that specifies the v-direction wrap for the texture sample.

Parent Elements

  • <sampler>: Element that defines texture samplers.

Attributes

Name Types and Values Description DCC
value enum
repeat
mirror
clamp
mirror_once		 Specifies wrapping in the v-direction.
repeat Repeats the texture.
mirror Flips and repeats the texture.
clamp Stretches pixels at the edges of the texture.
mirror_once Mirrors the texture only once in the negative coordinate direction and then stretches pixels at the edges of the texture.
