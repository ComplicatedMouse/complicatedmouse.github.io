<original_linear>

Description

This element represents a linear curve exported from a DCC tool that will be linearly corrected between keys.

Parent Elements

  • <original_color_anim_target>: Element that animates color in the DCC tool.
  • <original_texsrt_anim_target>: Element that animates a texture SRT in the DCC tool.

Attributes

Name Types and Values Description DCC
count int
1 – 65,535		 The number of resources held by the curve. Configurable
pre_wrap enum
clamp
repeat
mirror
relative_repeat		 Specifies the operation performed when evaluating from the first key to the previous frame.
clamp Returns the value of the first key.
repeat Returns a value that is used repeatedly on the curve.
mirror Returns the value used to repeatedly rotate the curve.

 Configurable

clamp
repeat

relative_repeat		 Specifies the operation performed when evaluating from the last key to the next frame.
clamp Returns the value of the last key.
repeat Returns a value that is used repeatedly on the curve.
mirror Returns the value used to repeatedly rotate the curve.

 Configurable
