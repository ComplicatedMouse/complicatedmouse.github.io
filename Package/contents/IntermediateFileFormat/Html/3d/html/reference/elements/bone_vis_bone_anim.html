<bone_vis_bone_anim>

Description

Element that animates each of the visibility attributes included under <bone>.

The following conditions determine whether a bone is used in rendering.
rigid_body || (matrix_index[0] != -1) || (matrix_index[1] != -1)

Parent Elements

  • <bone_vis_bone_anim_array>: A collection of <bone_vis_bone_anim> elements.

Description

There are no contents when there is no animation and the target is set using a fixed value for base_value.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <step_curve> [0 or 1]: Element that represents step curves for which interpolation is not performed between keys.
