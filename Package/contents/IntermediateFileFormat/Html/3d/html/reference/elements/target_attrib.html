<target_attrib>

Description

Element used to define vertex attributes of the key shape.
This element defines the vertex attributes that will undergo shape animation and their order.

Parent Elements

  • <target_attrib_array>: A collection of <target_attrib> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
attrib_name string
[0-9A-Za-z_]+		 The name of <shape> used by the key shape.
Configurable

Description

There is no content.
