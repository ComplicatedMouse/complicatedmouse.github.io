<bone_visibility_anim>

Description

Element specified in bone visibility animation (FVBA and FVBB) files.

The name of a bone visibility animation is taken from the filename, excluding the extension.

Parent Elements

  • <nw4f_3dif>: Root element for the 3D intermediate file format.

Attributes

Name Types and Values Description DCC
version enum
4.0.0		 -

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <process_log_array> [0 or 1]: A collection of <process_log> elements.
  • <bone_visibility_anim_info> [1]: Element that represents information related to <bone_visibility_anim>.
  • <bone_vis_bone_anim_array> [0 or 1]: A collection of <bone_vis_bone_anim> elements.
  • <user_data_array> [0 or 1]: A collection of <user_data> elements.
  • <stream_array> [0 or 1]: A collection of <stream> elements.
  • <comment> [0 or 1]: Element that stores metadata such as editing comments.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
  • <user_tool_data> [0 or 1]: Element that stores data for original user tools.
