<ui_label>

Description

Element that defines the UI label to edit for the variable.

DCC tools do not output this element.

Parent Elements

  • <option_var>: Element that defines the option variables of a shader.
  • <attrib_var>: Element that defines shader option variables.
  • <sampler_var>: Element that defines shader sampler variables.
  • <uniform_var>: Element that defines uniform shader variables.
  • <render_info_slot>: Element that defines rendering information associated with shading models.
  • <group>: Element that defines groups of user interfaces used for editing.
  • <page>: Element that groups together items that can be edited on their own page.

Attributes

Name Types and Values Description
value string
.+		 Label for the UI with a variable to edit.

Description

There is no content.
