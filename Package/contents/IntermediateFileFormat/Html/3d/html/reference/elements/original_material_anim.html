<original_material_anim>

Description

Element that specifies the target material for material animation in the DCC tool.

Parent Elements

  • <original_material_anim_array>: A collection of <original_material_anim> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
mat_name string
[0-9A-Za-z\-\._]+		 The name of the <material> to animate.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_color_anim_array> [0 or 1]: A collection of <original_color_anim> elements.
    This element is not found in files with the extension .ftsa or .ftsb.
  • <original_texsrt_anim_array> [0 or 1]: A collection of <original_texsrt_anim> elements.
    This element is not found in files with the extension .fcla or .fclb.
