<scene_anim_info>

Description

Represents information related to <scene_anim>.

Parent Elements

  • <scene_anim>: Element specified in scene animation (FSNA and FSNB) files.

Attributes

Name Types and Values Description DCC
frame_resolution int
1 – MAX		 The frame resolution used when baking, quantizing, or creating frames for the application.
Specifies how many times to split the frame.
 Configurable
dcc_preset string
-		 Preset name for the DCC plug-in.
 Configurable
dcc_magnify float
0 – MAX		 The overall scale specified by the DCC tool.
 Configurable
dcc_start_frame int
MIN – MAX		 Represents the start frame of the animation specified by the DCC tool.
 Configurable
dcc_end_frame int
MIN – MAX		 Represents the end frame of the animation specified by the DCC tool.
 Configurable
dcc_fps float
0 – MAX		 Represents the frame rate of the animation specified by the DCC tool.
 Configurable
bake_all bool
true or false		 Specify true if all animations have been baked by the DCC tool.
 Configurable
bake_tolerance_rotate float
0 – MAX		 Specifies the allowable rotation error when baking animations.
Specify a number of degrees as an absolute value.
 Configurable
bake_tolerance_translate float
0 – MAX		 Specifies the allowable translation error when baking animations as an absolute value.
 Configurable
bake_tolerance_color float
0 – MAX		 Specifies, as an absolute value, the allowable color error when baking animations.
 Configurable
quantize_tolerance_rotate float
0 – MAX		 Specifies the allowable rotation error when quantizing animations.
Specify a number of degrees as an absolute value.
 Configurable
quantize_tolerance_translate float
0 – MAX		 Specifies the allowable translation error when quantizing animations.
This value is specified as a ratio of the range of possible animation result values.
 Configurable
