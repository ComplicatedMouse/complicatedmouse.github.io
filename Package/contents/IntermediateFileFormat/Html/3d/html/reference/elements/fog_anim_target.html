<fog_anim_target>

Description

Element used to animate each parameter of a user-defined fog.

Parent Elements

  • <fog_anim>: Element that animates user-defined fogs.

Attributes

Name Types and Values Description DCC
target enum
dist_attn_start
dist_attn_end
color_r
color_g
color_b		 Specifies the animation target.
dist_attn_start Animates the distance to start distance attenuation.
dist_attn_end Animates the distance to end distance attenuation.
color_r Animates the R component of the color.
color_g Animates the G component of the color.
color_b Animates the B component of the color.

 Configurable
base_value float
MIN – MAX		 Specifies initial animation values.
Specify the 0 frame value when an animation is attached.
Configurable

Description

There are no contents when there is no animation and the target is set using a fixed value for base_value.

The following elements appear. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <hermite_curve> [0 or 1]: Element that represents a hermite curve that will be smoothed between keys.
  • <linear_curve> [0 or 1]: Element that represents linear curves for which linear interpolation is performed between keys.
  • <step_curve> [0 or 1]: Element that represents step curves for which interpolation is not performed between keys.
