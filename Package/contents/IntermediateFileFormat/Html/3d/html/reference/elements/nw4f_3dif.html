<nw4f_3dif>

Description

The root element for the 3D intermediate file format.

For more information, see Text Intermediate File.

Parent Elements

  • The root element.

Attributes

Name Types and Values Description DCC
version enum
4.0.0		 Represents the version number of the intermediate file.
Configurable

Description

One element matching the file extension appears.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <file_info> [0 or 1]: Element used to save a record of the creation of the project file and changes made to it.

  • The following elements appear. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

    • <model> [1]: Element specified in model files (FMDA and FMDB files).
    • <material> [1]: Element that defines materials.
    • <texture> [1]: Element specified in texture files (FTXA and FTXB files).
    • <skeletal_anim> [1]: Element specified in skeletal animation (FSKA and FSKB) files.
    • <shader_param_anim> [1]: Element specified in shader parameter animation (FSPA and FSPB) files, color animation (FCLA and FCLB) files, and texture SRT animation (FTPA and FTPB) files.
    • <tex_pattern_anim> [1]: Element specified in texture pattern animation (FTPA and FTPB) files.
    • <bone_visibility_anim> [1]: Element specified in bone visibility animation (FVBA and FVBB) files.
    • <mat_visibility_anim> [1]: Element specified in material visibility animation (FVMA and FVMB) files.
    • <shape_anim> [1]: Element specified in shape animation (FSHA and FSHB) files.
    • <scene_anim> [1]: Element specified in scene animation (FSNA and FSNB) files.
    • <shader_config> [1]: Element specified in shader configuration files (FSCA and FSCB files).
    • <shader_definition> [1]: Element specified in shader definition files (FSDA and FSDB files).
    • <shader_variation> [1]: Element specified in shader variation files (FSVA or FSVB files).
    • <material_anim> [1]: Element specified in material animation (FMAA and FMAB) files.
    • <shader_graph> [1]
    • <transform_tree> [1]
    • <transform_tree_anim> [1]
    • <transform_tree_visibility_anim> [1]
