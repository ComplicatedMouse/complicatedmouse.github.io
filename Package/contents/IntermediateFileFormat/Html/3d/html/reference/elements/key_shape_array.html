<key_shape_array>

Description

A collection of <key_shape> elements.

This element is omitted if <shape> does not correspond to a shape animation.

Parent Elements

  • <shape>: Element that defines the shape of a model.

Attributes

Name Types and Values Description DCC
length int
1 – 255		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

When exporting from a DCC tool, <key_shape> index="0" corresponds to the base shape, and all other indices, starting from index="1", correspond to the target shapes in their order in the DCC tool.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <key_shape> [1 – 255]: Element that defines the key shape for shape animations.
