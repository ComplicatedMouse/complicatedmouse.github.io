<inv_model_matrix>

Description

Inverse matrix element for the bind pose.

This element is defined if <bone> matrix_index is not "-1 -1".

Parent Elements

  • <bone>: Element that defines the coordinate system used for features such as skinning.

Attributes

There are no attributes.

Description

Specifies a 3x4 matrix consisting of twelve float values for the local bone coordinate system based on the model coordinate system for the bind pose.
