<stream>

Description

This element stores a data array of vertex attributes or vertex indices.

Parent Elements

  • <stream_array>: A collection of <stream> elements.

Attributes

Name Types and Values Description DCC
stream_index int
0 – MAX		 Represents an index of the array.
It is paired with the stream_index attribute of the reference sources <vtx_attrib>, <mesh>, <texture_info>, <original_image>, <original_hermite>, <original_linear>, <original_step>, <shader_src>, <user_int>, <user_float>, <user_string>, <user_wstring>, <hermite_curve>, <linear_curve>, and <step_curve>.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
type enum
float
int
byte
string
wstring		 Specifies the type of value in the data array.
float Indicates that the data array value is a real number.
int Indicates that the data array value is a signed 32-bit integer.
byte Indicates that the data array value is an unsigned 8-bit integer (hexadecimal notation).
string Indicates that the value is an ASCII string.
The character code is UTF-8, the character set is ASCII, and CRLF is used for returns.
Use &amp, &lt, and &gt to escape &, <, and >, respectively.
wstring Indicates that the value is a Unicode string.
The character code is UTF-8, the character set is UCS2, and CRLF is used for returns.
Use &amp, &lt, and &gt to escape &, <, and >, respectively.

 Configurable
count int
1 – MAX		 Specifies the type of value in the data array.
Always 1, if the type is string or wstring.
Configurable
column int
0 – 16		 Indicates the number of columns to use when laying out values in an intermediate file.
Specify a number of columns that is easy to read according to the content of each column.
Always 0, if the type is string or wstring.
Configurable

Description

Indicates the number of columns per line by fixed size.
