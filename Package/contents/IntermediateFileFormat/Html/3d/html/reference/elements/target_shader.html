<target_shader>

Description

This element defines the shader for which shader variations are to be generated.

DCC tools do not output this element.

Parent Elements

  • <target_shader_array>: A collection of <target_shader> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
shading_model_name string
[0-9A-Za-z_]+		 The name of the shading model for which shader variations are generated.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shader_program_array> [1]: A collection of <shader_program> elements.
