<original_texsrt_anim>

Description

Element that specifies the target sampler for texture SRT animation in the DCC tool.

Parent Elements

  • <original_texsrt_anim_array>: A collection of <original_texsrt_anim> elements.
    This element is not found in files with the extension .fcla or .fclb.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
hint string
[0-9A-Za-z_]+		 Represents hint information used when binding with texture samplers using a tool.
Specifications are the same as those for <sampler> hint.
It is paired with the reference source <param_anim> original_hint.
Configurable
mode enum
maya
3dsmax
softimage		 Specifies the calculation method for texture SRT.
mayaSpecifies the Maya calculation method.
3dsmaxSpecifies the 3ds Max calculation method.
softimageSpecifies the Softimage calculation method.

 Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_texsrt_anim_target> [1 – 5]: Element that animates a texture SRT in the DCC tool.
