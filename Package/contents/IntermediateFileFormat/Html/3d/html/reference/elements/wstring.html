<wstring>

Description

Element that stores Unicode strings.

Parent Elements

  • <user_wstring>: Element that stores a Unicode string of user data.

Attributes

There are no attributes.

Description

The character code is UTF-8, the character set is UCS2, and the line feed character is CRLF.
Use &amp, &lt, and &gt to escape &, <, and >, respectively.
