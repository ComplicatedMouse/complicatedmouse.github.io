<block_var>

Description

Element that defines the uniform block variables of a shader.

DCC tools do not output this element.

Parent Elements

  • <block_var_array>: A collection of <block_var> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Represents the uniform block variable ID.
type enum
none
material
shape
skeleton
option		 Represents the uniform block variable type.
none Represents a uniform block without a type specification.
material Represents a uniform block for a material.
shape Represents a uniform block for a shape.
skeleton Represents a uniform block for a skeleton.
option Represents a uniform block for options.

Description

At least one of the following elements must be included in the file: <vertex_symbol>, <geometry_symbol>, <fragment_symbol>, <compute_symbol>, <tessellation_control_symbol>, or <tessellation_evaluation_symbol>.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <vertex_symbol> [0 or 1]: Element that defines variable symbols used with the vertex shader.
  • <geometry_symbol> [0 or 1]: Element that defines variable symbols used with the geometry shader.
  • <fragment_symbol> [0 or 1]: Element that defines variable symbols used with the fragment shader.
  • <compute_symbol> [0 or 1]: Element that defines variable symbols used with the compute shader.
  • <tessellation_control_symbol> [0 or 1]: Element that defines variable symbols used with the tessellation control shader.
  • <tessellation_evaluation_symbol> [0 or 1]: Element that defines variable symbols used with the tessellation evaluation shader.
  • <uniform_var_array> [0 or 1]: A collection of <uniform_var> elements.
