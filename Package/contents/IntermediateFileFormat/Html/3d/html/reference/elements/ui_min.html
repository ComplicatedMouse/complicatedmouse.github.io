<ui_min>

Description

Element that defines the minimum value for the UI for editing a variable.

DCC tools do not output this element.

Parent Elements

  • <uniform_var>: Element that defines uniform shader variables.

Attributes

Name Types and Values Description
value float
MIN – MAX		 The minimum value for the UI for editing a variable.

Description

There is no content.
