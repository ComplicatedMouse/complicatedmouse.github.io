<shader_storage_block_var_array>

Description

A collection of <shader_storage_block_var> elements.

DCC tools do not output this element.

Parent Elements

  • <shading_model>: Element that defines shading models.

Attributes

Name Types and Values Description
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.

Description

Child elements are not sorted. They are saved in the order specified by the user in the shader source.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shader_storage_block_var> [1 – 65,535]: Element that defines the shader storage block variables of a shader.
