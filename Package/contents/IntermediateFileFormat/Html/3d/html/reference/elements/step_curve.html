<step_curve>

Description

The element representing step curves for which interpolation is not to be performed between keys.

Parent Elements

  • <pattern_anim>: Element that animates texture sampler textures.
  • <material_visibility_anim>: Element that animates the visibility attributes included under <material_info>.
  • <shape_anim_target>: Element that animates <key_shape> weight.
  • <bone_anim_target>: Element that animates each parameter included under <bone>.
  • <camera_anim_target>: Element that animates each parameter of a user-defined camera.
  • <light_anim_target>: Element that animates each parameter of a user-defined light.
  • <fog_anim_target>: Element that animates each parameter of a user-defined fog.
  • <param_anim_target>: Element that animates shader parameters.
  • <transform_node_anim_target>
  • <pattern_anim_target>: Element that animates texture sampler textures.
  • <bone_vis_bone_anim>: Element that animates each of the visibility attributes included under <bone>.
  • <mat_vis_mat_anim>: Element that animates the visibility attributes included under <material_info>.
  • <transform_node_visibility_anim>
