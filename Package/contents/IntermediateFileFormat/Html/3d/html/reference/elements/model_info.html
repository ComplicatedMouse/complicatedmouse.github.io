<model_info>

Description

Element representing information related to <model>.

Parent Elements

  • <model>: Element specified in model files (FMDA and FMDB files).

Attributes

Name Types and Values Description DCC
material_count int
0 – 65,535		 It is the material (<material>) count.
 Configurable
bone_count int
0 – 65,535		 It is the bone (<bone>) count.
 Configurable
shape_count int
0 – 65,535		 It is the shape (<shape>) count.
 Configurable
smooth_skinning_shape int
0 – 65,535		 It is the number of smooth skinning shapes. Configurable
rigid_skinning_shape int
0 – 65,535		 It is the number of rigid skinning shapes.
 Configurable
smooth_skinning_matrix int
0 – 65,535		 Smooth skinning matrix index.
 Configurable
rigid_skinning_matrix int
0 – 65,535		 Number of rigid skinning matrices.
 Configurable
unite_pos_quantize bool
true or false		 When the quantized format of the vertex coordinate is standardized by the format with the highest precision inside the model, true is specified.
When exporting from a DCC tool, always specify false.
 Fixed
dcc_preset string
-		 Preset name for the DCC plug-in.
 Configurable
dcc_magnify float
0 – MAX		 The overall scale specified by the DCC tool.
 Configurable
dcc_start_frame int
MIN – MAX		 Represents the start frame of the animation specified by the DCC tool.
When exporting a rigid body that includes an animation, models are exported set to the start frame.
 Configurable

There is no content.
