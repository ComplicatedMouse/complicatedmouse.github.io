<shape_info>

Description

Element that represents information related to <shape>.

Parent Elements

  • <shape>: Element that defines the shape of a model.

Attributes

Name Types and Values Description DCC
mat_name string
[0-9A-Za-z\-\._]*		 The name of the <material> used in rendering.
Configurable
bone_name string
[0-9A-Za-z\-\._]+		 The name of the <bone> used in rigid body rendering or visibility animation.
Configurable
original_material_name string
-		 Specify a material name for raw shape data.
Specify multiple names delimited by commas when multiple shapes are combined, for example, during shape compression.
Configurable
original_bone_name string
-		 Specify a bone name for raw shape data.
Specify multiple names delimited by commas when multiple shapes are combined, for example, during shape compression.
Configurable
vertex_index int
0 – 65,534		 Represents the index of the <vertex> used in shape definitions.
 Configurable
vertex_skinning_count int
0 – 128		 Represents the number of bones that influence a single vertex.
This value indicates the number of bone indices and bone weights per vertex.
This value is 0 in the case of rigid bodies.
Configurable
local_coordinate bool
true or false		 Specify true when vertices are stored in local bone coordinates (when the shape is either rigid body or rigid skinning).
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
original_shape_hash string
[^"']*		 Specify a hash value for raw shape data.
When exporting from a DCC tool, always specify an empty string.
Fixed
original_shape_count int
1 – 65,535		 Specifies the number of shapes before applying shape compression.
When exporting from a DCC tool, always specify 1.
Fixed
delete_near_vertex_mode string
[^"']*		 Specifies an applied approximated vertex delete mode.
When exporting from a DCC tool, always specify an empty string.
Fixed
divide_submesh_mode string
[^"']*		 Specifies the submesh division mode that is already applied.
When exporting from a DCC tool, always specify an empty string.
Fixed
optimize_primitive_mode string
[^"']*		 Specifies the primitive optimization mode that is already applied.
When exporting from a DCC tool, always specify an empty string.
Empty string Does not optimize primitives.
default Indicates that primitive optimization has already been applied using standard settings.
force Indicates that primitive optimization over time has already been applied.
brute_force Indicates that primitive optimization over a longer time has already been applied.
forsyth This has been optimized using the algorithm described on http://tomforsyth1000.github.io/papers/fast_vert_cache_opt.html.

 Fixed
polygon_reduction_mode string
[^"']*		 Specifies the polygon-reduction optimization mode that is already applied.
When exporting from a DCC tool, always specify an empty string.
Fixed
