<modify>

Description

Element that saves a record of changes to the intermediate file.

Specify <modify> only when <file_info> and <create> elements are present.
DCC tools do not output this element.

Parent Elements

  • <file_info>: Element used to save a record of the creation of the project file and changes made to it.

Attributes

Name Types and Values Description
tool_name string
-		 The name of the tool used to modify the intermediate file.
tool_version string
-		 The version of the tool used to modify the intermediate file.

Description

There is no content.
