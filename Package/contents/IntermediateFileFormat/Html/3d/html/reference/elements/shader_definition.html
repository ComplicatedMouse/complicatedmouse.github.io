<shader_definition>

Description

Element specified in shader definition files (FSDA and FSDB files).

The name of the shader archive to be generated from the shader definition file is taken from the name of the shader definition file, excluding the extension.
DCC tools do not output this element.

Parent Elements

  • <nw4f_3dif>: Root element for the 3D intermediate file format.

Attributes

Name Types and Values Description
version enum
4.1.0

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shader_definition_info> [1]: Element that represents information related to <shader_definition>.
  • <force_include_array> [0 or 1]: A collection of <force_include> elements.
  • <shading_model_array> [1]: A collection of <shading_model> elements.
  • <shader_src_array> [1]: A collection of <shader_src> elements.
  • <stream_array> [0 or 1]: A collection of <stream> elements.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
