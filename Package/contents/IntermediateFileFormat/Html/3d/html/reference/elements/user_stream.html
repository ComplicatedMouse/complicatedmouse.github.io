<user_stream>

Description

Element used to store stream type user data.

DCC tools do not output this element.

Parent Elements

  • <user_data>: Element that stores original user data that is obtained at runtime.

Attributes

Name Types and Values Description
size int
0 – MAX		 Specifies the number of bytes representing data.
stream_index int
-1 – 65,534		 Represents the index of the <stream> element storing the data.
Specify byte for the type attribute of <stream>, and 16 for column.
Specify -1 if size is zero.

Description

There is no content.
