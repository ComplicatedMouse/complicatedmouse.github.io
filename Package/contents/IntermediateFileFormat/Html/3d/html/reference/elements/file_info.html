<file_info>

Description

Element used to save a record of the creation of the project file and changes made to it.

Parent Elements

  • <nw4f_3dif>: Root element for the 3D intermediate file format.

Attributes

There are no attributes.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <create> [1]: Element that saves a record of the creation of the intermediate file.
  • <modify> [0 or 1]: Element that saves a record of changes to the intermediate file.
