<shader_config_info>

Description

Element that represents information related to <shader_config>.

DCC tools do not output this element.

Parent Elements

  • <shader_config>: Element specified in shader configuration files (FSCA and FSCB files).

Attributes

Name Types and Values Description
code_page int
0 – 65,535		 Specifies the code page representing the character code of the shader source code.

Description

There is no content.
