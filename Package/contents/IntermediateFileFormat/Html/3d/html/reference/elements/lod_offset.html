<lod_offset>

Description

Vertex index offset value in each LOD level.

DCC tools do not output this element.

Parent Elements

  • <vertex>: Element that defines vertex data.

Attributes

Name Types and Values Description
count int
1 – 255		 Specifies level count excluding LOD base level.

Description

Specifies the number of integer values indicated by count.
