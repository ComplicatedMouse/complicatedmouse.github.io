<tex_pattern_mat_anim_array>

Description

A collection of <tex_pattern_mat_anim> elements.

Parent Elements

  • <tex_pattern_anim>: Element specified in texture pattern animation (FTPA and FTPB) files.

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in the character code order of the <tex_pattern_mat_anim> mat_name.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <tex_pattern_mat_anim> [1 – 65,535]: Element that specifies the target material of a texture pattern animation.
