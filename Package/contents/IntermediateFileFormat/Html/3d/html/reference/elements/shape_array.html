<shape_array>

Description

A collection of <shape> elements.

Parent Elements

  • <model>: Element specified in model files (FMDA and FMDB files).

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in order of the <bone> index referenced by <shape_info> bone_name.
To reference the same <bone>, sort in the character code order of <shape> name.
The sort order influences runtime operations such as calculating hierarchical billboards.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shape> [1 – 65,535]: Element that defines the shape of a model.
