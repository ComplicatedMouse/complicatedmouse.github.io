<user_string>

Description

Element that stores user data consisting of an ASCII string.

Parent Elements

  • <user_data>: Element that stores original user data that is obtained at runtime.

Attributes

Name Types and Values Description DCC
count int
0 – MAX		 Specifies the number of instances of data.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <string> [0 – 65,535]: Element that stores ASCII strings.
