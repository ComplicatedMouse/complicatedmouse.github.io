<vtx_buffer_array>

Description

A collection of <vtx_buffer> elements.

Parent Elements

  • <vertex>: Element that defines vertex data.

Attributes

Name Types and Values Description DCC
length int
1 – 255		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

When exporting from a DCC tool, only one <vtx_buffer> element is output because all vertex data is output in interleaved form.
If <shading_model> is assigned in <shader_assign>, <vtx_buffer> is re-organized according to the content of <interleave>.
<input> is registered in <vtx_buffer> if <vtx_buffer> is defined in each <interleave> and <interleave> id_set matches <attrib_assign> id.
Cases where <attrib_assign> id does not match are included in the <interleave> id_set="" that corresponds to <vtx_buffer>.
In the end, <vtx_buffer> components that do not include <input> are deleted.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <vtx_buffer> [1 – 255]: Element that defines the vertex buffer.
