<shader_param_anim>

Description

Element specified in shader parameter animation (FSPA and FSPB) files, color animation (FCLA and FCLB) files, and texture SRT animation (FTPA and FTPB) files.

The name of a shader parameter animation is taken from the filename, excluding the extension. Note, however, that in the case of color and texture SRT animation files, the filename includes the extension when converting to binary for use on actual hardware.

Parent Elements

  • <nw4f_3dif>: Root element for the 3D intermediate file format.

Attributes

Name Types and Values Description DCC
version enum
4.0.0		 -

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <process_log_array> [0 or 1]: A collection of <process_log> elements.
  • <shader_param_anim_info> [1]: Element that represents information related to <shader_param_anim>.
  • <shader_param_mat_anim_array> [0 or 1]: A collection of <shader_param_mat_anim> elements.
  • <original_material_anim_array> [0 or 1]: A collection of <original_material_anim> elements.
  • <user_data_array> [0 or 1]: A collection of <user_data> elements.
  • <stream_array> [0 or 1]: A collection of <stream> elements.
  • <comment> [0 or 1]: Element that stores metadata such as editing comments.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
  • <user_tool_data> [0 or 1]: Element that stores data for original user tools.
