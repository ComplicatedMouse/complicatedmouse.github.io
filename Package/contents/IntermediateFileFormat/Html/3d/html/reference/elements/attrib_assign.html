<attrib_assign>

Description

This element specifies vertex attributes assigned to the shader.

Specifies the logical OR of vertex attribute assignments for all shapes bound to the material.
DCC tools do not output this element.

Parent Elements

  • <attrib_assign_array>: A collection of <attrib_assign> elements.

Attributes

Name Types and Values Description
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Represents the <attrib_var> to be assigned.
attrib_name string
[0-9A-Za-z_]*		 Represents the name of <vtx_attrib> to assign to the shader.
value_policy enum
unset
refer
override
default		 See the description for value_policy in <shader_option>.
child_restriction enum
unset
none
force_refer
disallow_refer
value_policy_to_refer		 See the description for child_restriction in <shader_option>.
