<vtx_attrib>

Description

Element that defines vertex attributes.

Parent Elements

  • <vtx_attrib_array>: A collection of <vtx_attrib> elements.

Attributes

Name Types and Values Description DCC
attrib_index int
0 – 254		 Represents an index of the array.
It is paired with the attrib_index attribute of the reference source <input> or <attrib_assign>.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
name string
[0-9A-Za-z_]+		 Represents the name used for binding vertex attributes on actual hardware.
It is paired with the attrib_name attribute of the referring <attrib_assign> or <target_attrib>.
When exporting from a DCC tool, specify a name using the command naming conventions given below.
_p0 – _p254: position
_n0 – _n254: normal
_t0 – _t254: tangent
_b0 – _b254: binormal
_c0 – _c254: color
_u0 – _u254: uv
_i0 – _i31: blend index
_w0 – _w31: blend weight
Configurable

count int
1 – MAX		 Specifies the number of vertices.
 Configurable
stream_index int
0 – MAX		 The index of the <stream> element used to store actual vertex attribute data.
The binormal vector points in the opposite direction as V of UV.
Specify 1 for the W value of tangent and binormal vectors if the original UV faces the front, or -1 if it faces the back.
The tangent and normal vectors are orthogonal to the binormal vector. Each vector has a length of 1.
The upper left corner of the texture image is treated as 0 in texture coordinates.
When blendweight is 0, blendindex specifies the blendindex having the largest blendweight among vertices.
Configurable
