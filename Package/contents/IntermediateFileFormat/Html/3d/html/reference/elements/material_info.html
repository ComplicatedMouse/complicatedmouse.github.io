<material_info>

Description

Element that represents information related to <material>.

Parent Elements

  • <material>: Element that defines materials.

Attributes

Name Types and Values Description DCC
visibility bool
true or false		 Specifies the visibility <shape> bound to the material.
When exporting from a DCC tool, specify true.
Fixed
compress_enable bool
true or false		 Specify true in the case of materials that can be compressed during material compression by the intermediate file optimizer.
Configurable
child_sampler_structure_restriction enum
none
disallow_add_own_sampler		 Specifies the restrictions for the sampler composition of the child material. When exporting from a DCC tool, specify none.
none No limit.
disallow_add_own_sampler Prohibits adding independent samplers to the child material.

 Fixed

Description

There is no content.
