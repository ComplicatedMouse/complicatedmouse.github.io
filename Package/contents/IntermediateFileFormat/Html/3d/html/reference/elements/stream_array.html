<stream_array>

Description

An element of the <stream> array.

There is no text portion in binary intermediate files because they have been binarized.

Parent Elements

  • <material>: Element that defines materials.
  • <model>: Element specified in model files (FMDA and FMDB files).
  • <material_anim>: Element specified in material animation (FMAA and FMAB) files.
  • <shader_config>: Element specified in shader configuration files (FSCA and FSCB files).
  • <shader_definition>: Element specified in shader definition files (FSDA and FSDB files).
  • <shape_anim>: Element specified in shape animation (FSHA and FSHB) files.
  • <skeletal_anim>: Element specified in skeletal animation (FSKA and FSKB) files.
  • <scene_anim>: Element specified in scene animation (FSNA and FSNB) files.
  • <shader_param_anim>: Element specified in shader parameter animation (FSPA and FSPB) files, color animation (FCLA and FCLB) files, and texture SRT animation (FTPA and FTPB) files.
  • <shader_variation>: Element specified in shader variation files (FSVA or FSVB files).
  • <transform_tree_anim>
  • <tex_pattern_anim>: Element specified in texture pattern animation (FTPA and FTPB) files.
  • <texture>: Element specified in texture files (FTXA and FTXB files).
  • <bone_visibility_anim>: Element specified in bone visibility animation (FVBA and FVBB) files.
  • <mat_visibility_anim>: Element specified in material visibility animation (FVMA and FVMB) files.
  • <transform_tree_visibility_anim>

Attributes

Name Types and Values Description DCC
length int
1 – MAX		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in the order that instances of stream_index appear in the reference file.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <stream> [1 – 2,147,483,647]: Element that stores a data array of vertex attributes or vertex indices.
