<vertex_stage>

Description

Element that defines the shader vertex shader stage.

DCC tools do not output this element.

Parent Elements

  • <shading_model>: Element that defines shading models.

Attributes

Name Types and Values Description
src_index int
0 – 65,534		 Represents the index specified in <shader_src> that defines the shader source for the vertex shader.

Description

There is no content.
