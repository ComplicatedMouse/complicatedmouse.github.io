<original_color_anim_array>

Description

A collection of <original_color_anim> elements.
This element is not found in files with the extension .ftsa or .ftsb.

Parent Elements

  • <original_per_material_anim>: Element that specifies the target material of a material animation in the DCC tool.
  • <original_material_anim>: Element that specifies the target material of a material animation in the DCC tool.

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in the order defined in <original_color_anim> hint.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_color_anim> [1 – 65,535]: Element that specifies the color for color animation in the DCC tool.
