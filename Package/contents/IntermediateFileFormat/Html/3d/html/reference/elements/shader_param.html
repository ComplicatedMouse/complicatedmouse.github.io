<shader_param>

Description

This element is used to set shader parameters.

The order of shader parameters is determined at the time of shader assignment and matches the order data is arranged in UBO.
DCC tools do not output this element.

Parent Elements

  • <shader_param_array>: A collection of <shader_param> elements.

Attributes
