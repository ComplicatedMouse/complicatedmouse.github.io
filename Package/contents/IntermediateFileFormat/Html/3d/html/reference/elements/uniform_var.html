<uniform_var>

Description

Element that defines uniform shader variables.

DCC tools do not output this element.

Parent Elements

  • <uniform_var_array>: A collection of <uniform_var> elements.

Attributes

Description

At least one of the following elements must be included in the file: <vertex_symbol>, <geometry_symbol>, <fragment_symbol>, <compute_symbol>, <tessellation_control_symbol>, or <tessellation_evaluation_symbol>.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <vertex_symbol> [0 or 1]: Element that defines variable symbols used with the vertex shader.
  • <geometry_symbol> [0 or 1]: Element that defines variable symbols used with the geometry shader.
  • <fragment_symbol> [0 or 1]: Element that defines variable symbols used with the fragment shader.
  • <compute_symbol> [0 or 1]: Element that defines variable symbols used with the compute shader.
  • <tessellation_control_symbol> [0 or 1]: Element that defines variable symbols used with the tessellation control shader.
  • <tessellation_evaluation_symbol> [0 or 1]: Element that defines variable symbols used with the tessellation evaluation shader.
  • <ui_label> [0 or 1]: Element that defines the UI labels for editing variables.
  • <ui_comment> [0 or 1]: Element that defines the UI comment for editing variables.
  • <ui_group> [0 or 1]: Element that defines the UI groups for editing variables.
  • <ui_order> [0 or 1]: Element that defines the order of precedence for an element while editing variables in the UI.
  • <ui_item> [0 or 1]: Element that switches the UI type for editing variables.
  • <ui_min> [0 or 1]: Element that defines the minimum value for editing a variable in the UI.
  • <ui_max> [0 or 1]: Element that defines the maximum value for editing a variable in the UI.
  • <ui_visible> [0 or 1]: Element that enables or disables the display of an element while editing variables in the UI.
  • <ui_default_min> [0 or 1]: Element the defines the default minimum value for editing a variable in the UI.
  • <ui_default_max> [0 or 1]: Element the defines the default maximum value for editing a variable in the UI.
