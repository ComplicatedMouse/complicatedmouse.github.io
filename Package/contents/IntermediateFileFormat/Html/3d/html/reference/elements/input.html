<input>

Description

This element specifies the order and quantization type when vertex attributes are included in the vertex buffer.

Parent Elements

  • <input_array>: A collection of <input> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
attrib_index int
0 – 254		 Represents the index of <vtx_attrib> input to the shader.
Configurable

Description

There is no content.
