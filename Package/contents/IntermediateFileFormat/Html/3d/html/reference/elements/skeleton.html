<skeleton>

Description

Element that brings together <bone> elements used to define the coordinate system.

Parent Elements

  • <model>: Element specified in model files (FMDA and FMDB files).

Attributes

There are no attributes.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <skeleton_info> [1]: Element that represents information related to <skeleton>.
  • <bone_array> [1]: A collection of <bone> elements.
