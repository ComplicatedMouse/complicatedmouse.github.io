<tex_pattern>

Description

This texture pattern element is used in texture pattern animations.

Parent Elements

  • <tex_pattern_array>: A collection of <tex_pattern> elements.

Attributes

Name Types and Values Description DCC
pattern_index int
0 – 65,534		 Represents an index of the array.
This index is referenced by the evaluated value of curves that include <pattern_anim_target>.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
tex_name string
[0-9A-Za-z\-\._]*		 The name of the <texture> used in the texture pattern animation.
Specify a name based on the FTX filename, excluding the extension.
Configurable

Description

There is no content.
