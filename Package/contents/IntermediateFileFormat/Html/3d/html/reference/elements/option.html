<option>

Description

This element defines shader options set for the shader program.

DCC tools do not output this element.

Parent Elements

  • <option_array>: A collection of <option> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Represents an identifier for the <option_var> element to set.
choice string
[^"']+		 Specifies option values in the same manner as used for <variation> choice.

Description

There is no content.
