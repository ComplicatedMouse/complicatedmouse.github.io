<shader_definition_info>

Description

Element that represents information related to <shader_definition>.

DCC tools do not output this element.

Parent Elements

  • <shader_definition>: Element specified in shader definition files (FSDA and FSDB files).

Attributes

Name Types and Values Description
code_page int
0 – 65,535		 Specifies the shader source code page.
config_path string
[^*?\\|"'<> ]+		 Represents the path to the shader configuration file serving as the data source for generating the shader definition file.
This path can be a filename or a relative path from the root path set for the shader converter.

Description

There is no content.
