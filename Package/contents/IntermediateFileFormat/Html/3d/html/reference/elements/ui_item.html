<ui_item>

Description

Element that switches the type of the UI for editing a variable.

DCC tools do not output this element.

Parent Elements

  • <option_var>: Element that defines the option variables of a shader.
  • <uniform_var>: Element that defines uniform shader variables.
  • <render_info_slot>: Element that defines rendering information associated with shading models.

Attributes

Name Types and Values Description
value enum
auto
color
check
combiner		 The type of the UI for editing a variable.
auto Automatically detects the UI type.
color Specifies a color-type UI.
check Specifies a check box UI.
The value of <ui_min> is used if the check is invalid, and the value of <ui_max> is used if valid.
A value of 0 is assumed if <ui_min> is not defined, and a value of 1 is assumed if <ui_max> is not defined.
combiner This setting is for development use. Do not use it.

Description

There is no content.
