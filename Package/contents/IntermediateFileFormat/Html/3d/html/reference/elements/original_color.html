<original_color>

Description

Element that defines color setting for materials in the DCC tool.

Parent Elements

  • <original_color_array>: A collection of <original_color> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
hint string
[0-9A-Za-z_]+		 Represents hint information used when binding colors with a tool.
It is paired with the reference source <shader_param> original_hint.
diffuse: Diffuse color.
opacity: Indicates an opaque color.
ambient: Indicates an ambient color.
emission: Indicates an emission color.
specular: Indicates a specular color.
Configurable
color float[3]
MIN – MAX		 Specify the value of color components in order of RGB.
Configurable

Description

There is no content.
