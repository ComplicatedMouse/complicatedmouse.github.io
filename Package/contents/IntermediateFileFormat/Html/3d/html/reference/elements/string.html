<string>

Description

Element that stores ASCII strings.

Parent Elements

  • <user_string>: Element that stores an ASCII string of user data.

Attributes

There are no attributes.

Description

The character code is UTF-8, the character set is ASCII, and the line feed character is CRLF.
Use &amp, &lt, and &gt to escape &, <, and >, respectively.
