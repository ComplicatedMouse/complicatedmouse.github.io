<mat_visibility_anim_info>

Description

Element that represents information related to <mat_visibility_anim>.

DCC tools do not output this element.

Parent Elements

  • <mat_visibility_anim>: Element specified in material visibility animation (FVMA and FVMB) files.

Attributes

Name Types and Values Description
frame_count int
0 – 16,777,215		 Frame count of the animation.
loop bool
true or false		 Specify true if loop playback is used on the animation.
frame_resolution int
1 – MAX		 The frame resolution used when baking, quantizing, or creating frames for the application.
Specifies how many times to split the frame.
dcc_preset string
-		 Preset name for the DCC plug-in.
dcc_start_frame int
MIN – MAX		 Represents the start frame of the animation specified by the DCC tool.
dcc_end_frame int
MIN – MAX		 Represents the end frame of the animation specified by the DCC tool.
dcc_fps float
0 – MAX		 Represents the frame rate of the animation specified by the DCC tool.
bake_all bool
true or false		 Specify true if all animations have been baked by the DCC tool.

Description

There is no content.
