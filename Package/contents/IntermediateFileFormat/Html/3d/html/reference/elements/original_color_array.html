<original_color_array>

Description

A collection of <original_color> elements.

Parent Elements

  • <original_material>: Element that defines shader parameter-related material settings in the DCC tool.

Attributes

Name Types and Values Description DCC
length int
1 – 255		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted in the order defined in <original_color> hint.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_color> [1 – 255]: Element that defines color setting for materials in the DCC tool.
