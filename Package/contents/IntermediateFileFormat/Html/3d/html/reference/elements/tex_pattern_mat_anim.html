<tex_pattern_mat_anim>

Description

Element specifying materials that will undergo texture pattern animation.

Parent Elements

  • <tex_pattern_mat_anim_array>: A collection of <tex_pattern_mat_anim> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
mat_name string
[0-9A-Za-z\-\._]+		 The name of the <material> to animate.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <pattern_anim_target> [1 – 255]: Element that animates texture sampler textures.
