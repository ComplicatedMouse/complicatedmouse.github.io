<skeletal_anim_info>

Description

Element that represents information related to <skeletal_anim>.

Parent Elements

  • <skeletal_anim>: Element specified in skeletal animation (FSKA and FSKB) files.

Attributes

Name Types and Values Description DCC
frame_count int
0 – 16,777,215		 Frame count of the animation.
Configurable
loop bool
true or false		 Specify true if loop playback is used on the animation.
 Configurable
frame_resolution int
1 – MAX		 The frame resolution used when baking, quantizing, or creating frames for the application.
Specifies how many times to split the frame.
 Configurable
dcc_preset string
-		 Preset name for the DCC plug-in.
 Configurable
dcc_magnify float
0 – MAX		 The overall scale specified by the DCC tool.
When baking this animation outside the DCC tool, multiply the value by bake_tolerance_translate.
 Configurable
dcc_start_frame int
MIN – MAX		 Represents the start frame of the animation specified by the DCC tool.
 Configurable
dcc_end_frame int
MIN – MAX		 Represents the end frame of the animation specified by the DCC tool.
 Configurable
dcc_fps float
0 – MAX		 Represents the frame rate of the animation specified by the DCC tool.
 Configurable
bake_all bool
true or false		 Specify true if all animations have been baked by the DCC tool.
 Configurable
scale_mode enum
standard
maya
softimage		 The scaling calculation method of the bone matrix.
Specifies a value according to the DCC tool.
standard Calculates scale values using the standard method.
maya Calculates scale values, taking into account the segment scale correction applied by Maya.
softimage Calculates scale values, taking into account the scale calculation method used by Softimage.

 Configurable
rotate_mode enum
euler_xyz
quaternion		 The rotation calculation method of the bone matrix.
When exporting from a DCC tool, specify euler_xyz.
euler_xyz Calculates the Euler angle rotation in order of XYZ.
The Euler angle rotation is specified in degrees.
quaternionCalculates using quaternions.

 Fixed

 Configurable
bake_tolerance_translate float

 Configurable
quantize_tolerance_scale float

 Configurable

 Configurable
