<attrib_component_array>

Description

The <attrib_component> array element.

DCC tools do not output this element.

Parent Elements

  • <packed_attrib_var>: Element that defines shader option variables after composition.
    This element is added if one component is specified for multiple vertices, using the layout declaration for shader vertex attributes.

Attributes

Name Types and Values Description
length int
1 – 255		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.

Description

Child elements are sorted in order of component value in the shader source.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <attrib_component> [1 through 255]: Element that defines constituent vertex attribute variables.
