<fog_anim>

Description

Element used to animate user-defined fogs.

Parent Elements

  • <fog_anim_array>: A collection of <fog_anim> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
fog_name string
[0-9A-Za-z\-\._]+		 The name of the user-defined fog to animate.
Configurable
frame_count int
0 – 16,777,215		 Frame count of the animation.
Configurable
loop bool
true or false		 Specify true if loop playback is used on the animation.
 Configurable
dist_attn_func string
[0-9A-Za-z_]*		 Specifies the distance attenuation function to use with the user-defined fog.
When exporting from a DCC tool, specify an empty string.
Fixed

Description

Each <fog_anim_target> target element contains a pair of dist_attn_* and color_* attributes.
<fog_anim_target> is sorted in the order that targets are defined.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <fog_anim_target> [0 – 5]: Element that animates each parameter of a user-defined fog.
  • <user_data_array> [0 or 1]: A collection of <user_data> elements.
  • <comment> [0 or 1]: Element that stores metadata such as editing comments.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
  • <user_tool_data> [0 or 1]: Element that stores data for original user tools.
