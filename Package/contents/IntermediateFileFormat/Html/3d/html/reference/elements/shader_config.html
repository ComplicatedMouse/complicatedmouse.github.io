<shader_config>

Description

Element specified in shader configuration files (FSCA and FSCB files).

DCC tools do not output this element.

Parent Elements

  • <nw4f_3dif>: Root element for the 3D intermediate file format.

Attributes

Name Types and Values Description
version enum
4.1.0

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shader_config_info> [1]: Element that represents information related to <shader_config>.
  • <include_path_array> [0 or 1]: A collection of <include_path> elements.
  • <force_include_file_array> [0 or 1]: A collection of <force_include_file> elements.
  • <shader_array> [1]: A collection of <shader> elements.
  • <stream_array> [0 or 1]: A collection of <stream> elements.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
