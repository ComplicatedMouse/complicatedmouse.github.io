<pattern_anim>

Description

Element used to animate texture sampler textures.

Parent Elements

  • <tex_pattern_anim_array>: A collection of <pattern_anim> elements.

Attributes

Name Types and Values Description DCC
sampler_name string
[0-9A-Za-z_]+		 Represents the name used for binding with a texture sampler on actual hardware.
Specifications are the same as those for <sampler> name.
Configurable
hint string
[0-9A-Za-z_]*		 Represents hint information used when binding with texture samplers using a tool.
Specifications are the same as those for <sampler> hint.
Configurable
base_value float
MIN – MAX		 Specifies initial animation values.
Specify the 0 frame value when an animation is attached.
Configurable

Description

There are no contents when there is no animation and the target is set using a fixed value for base_value.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <step_curve> [0 or 1]: Element that represents step curves for which interpolation is not performed between keys.
