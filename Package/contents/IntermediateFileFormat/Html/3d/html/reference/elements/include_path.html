<include_path>

Description

This element defines the include path for the shader source.

DCC tools do not output this element.

Parent Elements

  • <include_path_array>: A collection of <include_path> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
path string
[^*?\\|"'<> ]+		 Represents the include path for the shader source.
Specify as a relative path from the shader configuration file.

Description

There is no content.
