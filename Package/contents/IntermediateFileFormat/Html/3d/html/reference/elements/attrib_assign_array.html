<attrib_assign_array>

Description

A collection of <attrib_assign> elements.

DCC tools do not output this element.

Parent Elements

  • <shader_assign>: Specifies the shader to assign to the material.

Attributes

Name Types and Values Description
length int
1 – 255		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.

Description

In the case of child elements, the <attrib_assign> id elements are sorted in the order shader id elements are defined.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <attrib_assign> [1 – 255]: Element that specifies vertex attributes assigned to the shader.
