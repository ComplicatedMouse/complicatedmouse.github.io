<render_info>

Description

Element used to define rendering information.

DCC tools do not output this element.

Parent Elements

  • <render_info_array>: A collection of <render_info> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
name string
[0-9A-Za-z_]+		 The name of the rendering information.
type enum
int
float
string		 Specifies the type of value used to represent rendering information.
count int
0 – MAX		 Specifies the number of values used to represent rendering information.
value_policy enum
unset
refer
override
default		 See the description for value_policy in <shader_option>.
child_restriction enum
unset
none
force_refer
disallow_refer
value_policy_to_refer		 See the description for child_restriction in <shader_option>.
child_value_restriction string
-		 See the description for child_value_restriction in <shader_option>.

Description

Specifies rendering information values as an alphanumeric string.
Multiple values can be specified by separating them with a space, tab, or return character.
