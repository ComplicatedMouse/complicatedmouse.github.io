<original_texsrt>

Description

Element that defines texture SRT settings for materials in the DCC tool.

Parent Elements

  • <original_texsrt_array>: A collection of <original_texsrt> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
hint string
[0-9A-Za-z_]+		 Represents hint information used when binding with texture samplers using a tool.
Specifications are the same as those for <sampler> hint.
It is paired with the reference source <shader_param> original_hint.
Configurable
uv_hint string
[0-9A-Za-z_]*		 Specifies hint information that can be used when binding vertex attributes.
Value is specified in the same manner as <vtx_attrib> hint.
Configurable
mode enum
maya
3dsmax
softimage		 Specifies the calculation method for texture SRT.
mayaSpecifies the Maya calculation method.
3dsmaxSpecifies the 3ds Max calculation method.
softimageSpecifies the Softimage calculation method.

 Configurable
scale float[2]
MIN – MAX		 Specifies texture SRT scale values in the order XY.
Configurable
rotate float
MIN – MAX		 Specifies the texture SRT rotation value.
Configurable
translate float[2]
MIN – MAX		 Specifies texture SRT translation values in the order XY.
Configurable
