<original_color_anim_target>

Description

Element that animates color in the DCC tool.

Parent Elements

  • <original_color_anim>: Element that specifies the color for color animation in the DCC tool.

Attributes

Name Types and Values Description DCC
target enum
color_r
color_g
color_b		 Specifies parameters that will undergo color animation.
color_rAnimates the R component.
color_gAnimates the G component.
color_bAnimates the B component.

 Configurable
base_value float
MIN – MAX		 Specifies initial animation values.
Specify the 0 frame value when an animation is attached.
Configurable

Description

There are no contents when there is no animation and the target is set using a fixed value for base_value.

The following elements appear. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_hermite> [0 or 1]: Element that represents a hermite curve exported from the DCC tool and smoothed between keys.
  • <original_linear> [0 or 1]: Element that represents a linear curve exported from the DCC tool that is linearly corrected between keys.
  • <original_step> [0 or 1]: Element that specifies a step curve exported from the DCC tool that is not corrected between keys.
