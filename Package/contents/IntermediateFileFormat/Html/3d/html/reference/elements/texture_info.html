<texture_info>

Description

This element represents information related to <texture>.

Parent Elements

  • <texture>: Element specified in texture files (FTXA and FTXB files).

Attributes

hint string
[0-9A-Za-z_]*		 Specifies a hint about how the texture is used.
Specifies values like albedo, opacity, emission, normal, tangent, specular, and reflection similar to those used with <sampler> hint.
Operations of each tool may change depending on hint (such as normalizing normal vectors if normal is specified).
Configurable
linear bool[4]
true or false		 Specifies whether the gamma values for the converted texture corresponding to <original_image> have been linearized.
If you specify true for one of the R, G, B, or A channels of <original_image>, the gamma values are converted from sRGB to linear in the resulting texture.
Channels for which you specify false, have the same gamma values as <original_image> in the resulting texture.
However, when quantize_type is a format that begins with srgb_, the resulting texture has the same gamma values as <original_image>, no matter what you specify, because the gamma values of the texture are linearized when the texture is fetched by the GPU.
Specify true true true false or true true true true for formats that begin with srgb_.
When using hardware with a two-channel format, the R and G channels undergo gamma correction when <original_image> is an RGB image, and the R and A channels undergo gamma correction when <original_image> is an RGBA image.
When exporting from a DCC tool, specify flags according to the configuration settings.
