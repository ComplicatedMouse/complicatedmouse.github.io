<shader_assign>

Description

Specifies the shader to assign to the material.

DCC tools do not output this element.

Parent Elements

  • <material>: Element that defines materials.

Attributes

Name Types and Values Description
shader_archive string
[0-9A-Za-z_]+		 The name of the shader archive.
Specify a name based on the filename of the shader definition file, but without a file extension.
shading_model string
[0-9A-Za-z_]+		 Represents the name of the <shading_model> to be assigned.
revision int
0 – MAX		 Revision number of the <shading_model> to be assigned.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <render_info_array> [0 or 1]: A collection of <render_info> elements.
  • <shader_option_array> [0 or 1]: A collection of <shader_option> elements.
  • <sampler_assign_array> [0 or 1]: A collection of <sampler_assign> elements.
  • <shader_param_array> [0 or 1]: A collection of <shader_param> elements.
  • <attrib_assign_array> [0 or 1]: A collection of <attrib_assign> elements.
