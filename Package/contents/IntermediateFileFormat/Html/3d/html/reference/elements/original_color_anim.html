<original_color_anim>

Description

This element specifies the color for color animation in the DCC tool.

Parent Elements

  • <original_color_anim_array>: A collection of <original_color_anim> elements.
    This element is not found in files with the extension .ftsa or .ftsb.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
hint string
[0-9A-Za-z_]+		 Represents hint information used when binding colors.
It is paired with the reference source <param_anim> original_hint.
diffuse: Diffuse color.
opacity: Indicates an opaque color.
ambient: Indicates an ambient color.
emission: Indicates an emission color.
specular: Indicates a specular color.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <original_color_anim_target> [1 – 3]: Element that animates color in the DCC tool.
