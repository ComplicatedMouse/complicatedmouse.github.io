<bone_vis_bone_anim_array>

Description

A collection of <bone_vis_bone_anim> elements.

Parent Elements

  • <bone_visibility_anim>: Element specified in bone visibility animation (FVBA and FVBB) files.

Attributes

Name Types and Values Description DCC
length int
1 – 65,535		 Represents the number of elements in the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

Child elements are sorted according to depth priority beginning from the root bone. Siblings are sorted in the order of the character code used for <bone_vis_bone_anim> bone_name.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <bone_vis_bone_anim> [1 – 65,535]: Element that animates each of the visibility attributes included under <bone>.
