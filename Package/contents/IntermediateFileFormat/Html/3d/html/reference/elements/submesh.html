<submesh>

Description

This element defines rendering calls.

This element is used for things like culling using a program.

Parent Elements

  • <submesh_array>: A collection of <submesh> elements.

Attributes

Name Types and Values Description DCC
submesh_index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
offset int
0 – MAX		 Specifies the index offset to the vertex index at which the render call should start.
When exporting from a DCC tool, always specify 0.
Fixed
count int
1 – MAX		 Specifies the number of vertex indices to be rendering starting from the offset position.
Configurable

Description

There is no content.
