<vertex_shape_anim>

Description

Element that animates shapes defined in <shape>.

Parent Elements

  • <vertex_shape_anim_array>: A collection of <vertex_shape_anim> elements.

Attributes

Name Types and Values Description DCC
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
shape_name string
[0-9A-Za-z\-\._]+		 The name of the <shape> element to animate.
Configurable
base_name string
[0-9A-Za-z\-\._]+		 The name of the <key_shape> used as the base shape.
The weight used for the base shape is 1 – (the total weight of <shape_anim_target>).
Configurable

Description

When exporting from a DCC tool, child elements are exported in the same order as the target shapes in the DCC tool.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <shape_anim_target> [1 – 255]: Element that animates <key_shape> weight.
