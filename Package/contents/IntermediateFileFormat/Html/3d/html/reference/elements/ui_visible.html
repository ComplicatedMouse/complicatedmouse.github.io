<ui_visible>

Description

Element that enables or disables the UI display to be edited for the variable.

DCC tools do not output this element.

Parent Elements

  • <option_var>: Element that defines the option variables of a shader.
  • <sampler_var>: Element that defines shader sampler variables.
  • <uniform_var>: Element that defines uniform shader variables.
  • <render_info_slot>: Element that defines rendering information associated with shading models.

Attributes

Name Types and Values Description
value enum
true
false
param_only
anim_only		 Specifies the display mode of the UI used to edit variables.
true Displays the UI for editing variables.
false Does not display the editing UI.
param_only Displays the UI for editing parameters, but does not display the UI for editing animations.
If it is a child element of <sampler_var>, the same operations result as when true is specified.
anim_only Displays the UI for editing animations without displaying the UI for editing parameters.
If it is a child element of <sampler_var>, the same operations result as when false is specified.

Description

There is no content.
