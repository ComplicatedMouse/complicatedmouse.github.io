<tessellation_evaluation_shader>

Description

Element that defines the tessellation evaluation shader settings.

DCC tools do not output this element.

Parent Elements

  • <shader>: Element that defines the shader configuration.

Attributes

Name Types and Values Description
path string
[^*?\\|"'<> ]+		 The tessellation evaluation shader source path.
Specify as a relative path from the path attribute of <include_path>.

Description

There is no content.
