<search_path>

Description

Element that specifies the search path when resolving a file reference.

DCC tools do not output this element.

Parent Elements

  • <texture_search_path_array>: Array element of <search_path> used for texture search.
  • <material_search_path_array>: Array element of <search_path> used for material search.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
path string
[^"']*		 Specifies the path to search. This path is either a relative path to the folder or an absolute path. You can include environment variables surrounded by %. The relative path indicates a relative path from the reference source file. You can specify wildcards (*, ?). When searching for subfolders or multiple wildcards are specified, and there are multiple files with the same name, they are sorted in alphabetical order regardless of case, and files are applied in descending order.
recursive bool
true or false		 Specify true to search recursively, and false otherwise.

Description

There is no content.
