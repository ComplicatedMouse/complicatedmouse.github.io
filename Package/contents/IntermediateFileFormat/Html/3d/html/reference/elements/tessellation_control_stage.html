<tessellation_control_stage>

Description

Element that defines the shader tessellation control shader stage.

DCC tools do not output this element.

Parent Elements

  • <shading_model>: Element that defines shading models.

Attributes

Name Types and Values Description
src_index int
0 – 65,534		 Index specified in <shader_src> that defines the shader source for the tessellation control shader.

Description

There is no content.
