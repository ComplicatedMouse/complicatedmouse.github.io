<vertex_symbol>

Description

Element that defines variable symbols used with the vertex shader.

DCC tools do not output this element.

Parent Elements

  • <option_var>: Element that defines the option variables of a shader.
  • <attrib_var>: Element that defines shader option variables.
  • <sampler_var>: Element that defines shader sampler variables.
  • <image_var>: Element that defines shader image variables.
  • <block_var>: Element that defines the uniform block variables of a shader.
  • <shader_storage_block_var>: Element that defines the shader storage block variables of a shader.
  • <uniform_var>: Element that defines uniform shader variables.
  • <buffer_var>: Element that defines the buffer variables of a shader.

Attributes

Name Types and Values Description
name string
[0-9A-Za-z\.\[\]_]+		 Represents the name of the variable used by the fragment shader.

Description

There is no content.
