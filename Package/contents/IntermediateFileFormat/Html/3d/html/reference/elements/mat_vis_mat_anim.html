<mat_vis_mat_anim>

Description

Element that animates the visibility attributes included under <material_info>.

DCC tools do not output this element.

Parent Elements

  • <mat_vis_mat_anim_array>: A collection of <mat_vis_mat_anim> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
mat_name string
[0-9A-Za-z\-\._]+		 The name of the <material> element to animate.
base_value float
MIN – MAX		 Specifies initial animation values.
Specify the 0 frame value when an animation is attached.

Description

There are no contents when there is no animation and the target is set using a fixed value for base_value.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <step_curve> [0 or 1]: Element that represents step curves for which interpolation is not performed between keys.
