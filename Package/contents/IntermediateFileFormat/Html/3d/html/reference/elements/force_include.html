<force_include>

Description

This component defines shader sources that are forcibly included.

DCC tools do not output this element.

Parent Elements

  • <force_include_array>: A collection of <force_include> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
src_index int
0 – 65,534		 Represents the index given in <shader_src> that defines the shader source forcibly included.

Description

There is no content.
