<group>

Description

Element for defining groups of user interfaces used for editing.

DCC tools do not output this element.

Parent Elements

  • <group_array>: A collection of <group> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
name string
[0-9A-Za-z_]+		 The group name.
page_name string
[0-9A-Za-z\-\._]*		 The <page> name the group belongs to.
condition string
[^"']*		 Controls whether the UI inside the group can be edited.
The UI can be edited if the condition is true, but cannot be edited if it is false.
UI elements that appear in the formula can always be edited.
For the formula, in addition to alphanumeric characters and each operator (basic arithmetic operations, bitwise operations, and logical operations), three types of variables (option, uniform, and renderinfo) are available. For variables, connect a type name and an identifier with "." and add "$" as a prefix as in $renderinfo.mode. If a type name is omitted, it is handled as the identifier of option. In addition, true and false can be used as special characters. true is evaluated as 1 and false as 0. The following restrictions are applied when using variables. uniform with the second and higher dimensional value of parameters is not supported. renderinfo with two or more render information values (count of 2 or greater) or variable-length renderinfo (optional is set to true) is not supported.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <ui_label> [0 or 1]: Element that defines the UI labels for editing variables.
  • <ui_comment> [0 or 1]: Element that defines the UI comment for editing variables.
  • <ui_order> [0 or 1]: Element that defines the order of precedence for an element while editing variables in the UI.
  • <ui_group> [0 or 1]: Element that defines the UI groups for editing variables.
