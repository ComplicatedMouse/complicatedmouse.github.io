<param_anim>

Description

This element is used to define parameters that will be used as the target for shader parameter animation.

DCC tools do not output this element.

Parent Elements

  • <shader_param_anim_array>: A collection of <param_anim> elements.
  • <param_anim_array>: A collection of <param_anim> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Specifies the <shader_param> id to animate.
Specifications are the same as those for <shader_param> id.
original_hint string
[0-9A-Za-z_]*		 Specifies the hint attribute of <original_color_anim> or <original_texsrt_anim> when animations are assigned, based on the original <original_color_anim> or <original_texsrt_anim>.
Specify an empty string if settings are not based on <original_color_anim> or <original_texsrt_anim>.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <param_anim_target> [1 – 65,535]: Element that specifies a target for a shader parameter animation.
