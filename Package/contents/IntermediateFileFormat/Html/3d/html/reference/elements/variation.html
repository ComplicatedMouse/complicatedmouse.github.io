<variation>

Description

This element limits the range of values specified usign shader option variables.
If this element is omitted, a variation that uses only default values is defined.

DCC tools do not output this element.

Parent Elements

  • <variation_array>: A collection of <variation> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Represents the ID of the option variable specifying the variation.
choice string
[^"']+		 Represents the limits placed on the range of values of the option variable.
The choice attribute specified by the shader option value is overwritten.
The method of specification differs depending on the variable type.
To specify a single value, specify that value as is.
Specifying * automatically determines the value based on the shader source.
Type (Mode)Notation Used in This Document<shader_option>
value
bool"bool"0 or 1
Integer
range		"[min, max]"
min and max are integers.		Integer
Enumerated List
list		"add, sub, mul, div"
Enumerated values consisting of alphanumeric strings and underscores (_).		Any one of these enumerated values
default string
[0-9A-Za-z+\-_]*		 Represents the option variable default value.
Specify a single value using the <option_var> choice notation.
If default is blank, the default is automatically determined based on the value specified in the source.
If the default in the source does not fall in the range specified by choice, the first candidate in the specified range is taken as the default.
branch bool
true or false		 Specifies whether to treat an option variable as a static branch.
An option variable is treated as a static branch if true or as a variation if false.
