<packed_attrib_var>

Description

Element that defines shader option variables after composition.
This element is added if one component is specified for multiple vertices, using the layout declaration for shader vertex attributes.

DCC tools do not output this element.

Parent Elements

  • <packed_attrib_var_array>: An element of the <packed_attrib_var> array.

Attributes

Name Types and Values Description
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 The identifier of the vertex attribute variable after composition.
It is paired with the reference source <attrib_assign> id.
type enum
int
int2
int3
int4
uint
uint2
uint3
uint4
float
float2
float3
float4		 Represents the logical type of the vertex attribute variable after composition.
Definition is the same as <vtx_attrib> type.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <attrib_component_array> [1]: An element of the <attrib_component> array.
