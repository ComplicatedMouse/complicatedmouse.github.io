<force_include_file>

Description

This component defines shader source files that are forcibly included.

DCC tools do not output this element.

Parent Elements

  • <force_include_file_array>: A collection of <force_include_file> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
path string
[^*?\\|"'<> ]+		 The path to the shader source file being forcibly included at the start of the shader source.
Specify as a relative path from the include path.

Description

There is no content.
