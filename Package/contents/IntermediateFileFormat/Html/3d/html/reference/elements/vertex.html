<vertex>

Description

Element that defines vertex data.

When exporting from a DCC tool, this is output to the <shape> element.
<vertex> elements are merged by the intermediate file optimizer.

Parent Elements

  • <vertex_array>: A collection of <vertex> elements.

Attributes

Name Types and Values Description DCC
vertex_index int
0 – 65,534		 Represents an index of the array.
It is paired with the reference source <shape_info> vertex_index.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <lod_offset> [0 or 1]: Element that specifies the vertex index offset value in each LOD level.
  • <vtx_attrib_array> [1]: A collection of <vtx_attrib> elements.
  • <vtx_buffer_array> [0 or 1]: A collection of <vtx_buffer> elements.
