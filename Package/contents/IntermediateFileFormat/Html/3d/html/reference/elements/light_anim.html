<light_anim>

Description

Element used to animate user-defined lights.

Parent Elements

  • <light_anim_array>: A collection of <light_anim> elements.

Attributes

Description

This element includes a set of <light_anim_target> target attributes: position_*, direction_*, aim_*, dist_attn_*, angle_attn_*, color0_*, and color1_*.
<light_anim_target> is sorted in the order that targets are defined.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <light_anim_target> [0 – 20]: Element that animates each parameter of a user-defined light.
  • <user_data_array> [0 or 1]: A collection of <user_data> elements.
  • <comment> [0 or 1]: Element that stores metadata such as editing comments.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
  • <user_tool_data> [0 or 1]: Element that stores data for original user tools.
