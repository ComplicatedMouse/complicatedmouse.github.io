<bone_anim>

Description

Element that animates each of the scale, rotation, and translation attributes included under <bone>.

The following conditions determine whether a bone is used in rendering.
rigid_body || (matrix_index[0] != -1) || (matrix_index[1] != -1)

Parent Elements

  • <bone_anim_array>: A collection of <bone_anim> elements.

Description

Regardless of whether there are any animations, scale, rotate, and translate animations of XYZ values are specified under <bone_anim_target>.
Specify W value rotation animations only when the <skeletal_anim_info> rotate_mode is specified as quaternion.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <bone_anim_target> [9 or 10]: Element that animates each parameter included under <bone>.
  • <user_data_array> [0 or 1]: A collection of <user_data> elements.
