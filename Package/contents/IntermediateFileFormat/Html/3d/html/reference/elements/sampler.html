<sampler>

Description

Element that defines texture samplers.

Parent Elements

  • <sampler_array>: A collection of <sampler> elements.

Attributes

Name Types and Values Description DCC
sampler_index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
Configurable
name string
[0-9A-Za-z_]+		 Represents the name used for binding texture samplers on actual hardware.
It is paired with the name attribute of the reference source <pattern_anim_target>.
When exporting from a DCC tool, specify a name using the command naming conventions given below.
_a0 – _a254: albedo
_o0 – _o254: opacity
_e0 – _e254: emission
_n0 – _n254: normal
_t0 – _t254: tangent
_s0 – _s254: specular
_r0 – _r254: reflection
Configurable
hint string
[0-9A-Za-z_]*		 Represents hint information used when binding texture samplers with a tool.
It is paired with the hint attributes of the reference sources <input> and <bone_anim>.
When exporting from a DCC tool, specify the hint using the command naming conventions given below.
albedo0 – albedo254
opacity0 – opacity254
emission0 – emission254
normal0 – normal254
tangent0 – tangent254
specular0 – specular254
reflection0 – reflection254
Configurable
required bool
true or false		 Specify true to make the sampler with this name required for the child material, and false otherwise. When exporting from a DCC tool, specify false.
Fixed

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <tex_name> [1]: Element that specifies the name of the <texture> used in the sampler.
  • <wrap_u> [1]: Element that specifies the u-direction wrap for the texture sample.
  • <wrap_v> [1]: Element that specifies the v-direction wrap for the texture sample.
  • <wrap_w> [1]: Element that specifies the w-direction wrap for the texture sample.
  • <filter_mag> [1]: Element that specifies the filter for enlarging the texture.
  • <filter_min> [1]: Element that specifies the filter for reducing the texture.
  • <filter_mip> [1]: Element that specifies a filter between texture mipmaps.
  • <filter_max_aniso> [1]: Element that specifies the maximum value of the anisotropic filter.
  • <lod_min> [1]: Element that specifies the minimum LOD value for the texture sample.
  • <lod_max> [1]: Element that specifies the maximum LOD value for the texture sample.
  • <lod_bias> [1]: Element that specifies the LOD bias value for the texture sample.
