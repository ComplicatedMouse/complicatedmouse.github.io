<sampler_assign>

Description

This element assigns texture samplers to a shader.

DCC tools do not output this element.

Parent Elements

  • <sampler_assign_array>: A collection of <sampler_assign> elements.

Attributes

Name Types and Values Description
index int
0 – 254		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 Represents the ID of the <sampler_var> to be assigned.
sampler_name string
[0-9A-Za-z_]*		 Represents the <sampler> name to be assigned to the shader.
value_policy enum
unset
refer
override
default		 See the description for value_policy in <shader_option>.
child_restriction enum
unset
none
force_refer
disallow_refer
value_policy_to_refer		 See the description for child_restriction in <shader_option>.

Description

There is no content.
