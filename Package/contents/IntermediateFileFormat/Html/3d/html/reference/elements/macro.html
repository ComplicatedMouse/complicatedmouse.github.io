<macro>

Description

This element defines shader macros.

DCC tools do not output this element.

Parent Elements

  • <macro_array>: A collection of <macro> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
name string
[0-9A-Za-z_]+		 The node name.
value string
[^"']*		 The node name.

Description

There is no content.
