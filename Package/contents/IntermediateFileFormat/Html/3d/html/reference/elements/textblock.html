<textblock>

Description

Element used to define text blocks associated with shading models.

DCC tools do not output this element.

Parent Elements

  • <textblock_array>: A collection of <textblock> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
id string
[0-9A-Za-z_]+		 The text block identifier.
text string
-		 The text in the text block.

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <ui_group> [0 or 1]: Element that defines the UI groups for editing variables.
  • <ui_order> [0 or 1]: Element that defines the order of precedence for an element while editing variables in the UI.
