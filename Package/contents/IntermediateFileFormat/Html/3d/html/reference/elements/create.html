<create>

Description

This element saves a record of the creation of the intermediate file.

Parent Elements

  • <file_info>: Element used to save a record of the creation of the project file and changes made to it.

Attributes

Name Types and Values Description DCC
tool_name string
-		 The name of the tool used to create the intermediate file.
 Configurable
tool_version string
-		 The version of the tool used to create the intermediate file.
 Configurable
src_path string
[^*?\\|"'<> ]*		 The full path when there is a source file when an intermediate file is created.
 Configurable

Description

There is no content.
