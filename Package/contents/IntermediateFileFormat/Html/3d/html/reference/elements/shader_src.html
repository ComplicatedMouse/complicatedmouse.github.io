<shader_src>

Description

Element that defines shader definitions.

DCC tools do not output this element.

Parent Elements

  • <shader_src_array>: A collection of <shader_src> elements.

Attributes

Name Types and Values Description
src_index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
include_path string
[^*?\\|"'<> ]*		 Represents the include path for the shader source.
path string
[^*?\\|"'<> ]+		 Represents a relative path from the include path for the shader source code.
stream_index int
0 – 65,534		 Represents the index of the <stream> element storing the shader source.
Specify wstring for the type attribute of <stream>.
Specify 1 for the count attribute of <stream>.
Specify 0 for the column attribute of <stream>.
The character code is UTF-8, the character set is UCS2, and CRLF is used for returns.
Use &amp, &lt, and &gt to escape &, <, and >, respectively.

Description

There is no content.
