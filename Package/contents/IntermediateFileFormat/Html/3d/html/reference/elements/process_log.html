<process_log>

Description

Element that saves a record related to batch processing.

DCC tools do not output this element.

Parent Elements

  • <process_log_array>: A collection of <process_log> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
process string
[0-9A-Za-z_]+		 Represents the type of batch processing.
For details on processing, see the Help documentation for the intermediate file optimizer.
argument string
-		 Optimization process argument.
Specify an empty string if there is no description of argument in the description of process.

Description

Gives a summary of the optimization process result.
For more information, see the help documentation for the intermediate file optimizer.
