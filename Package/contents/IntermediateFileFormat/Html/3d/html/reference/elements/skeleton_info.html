<skeleton_info>

Description

Element representing information related to <skeleton>.

Parent Elements

  • <skeleton>: Element that brings together <bone> elements used to define the coordinate system.

Attributes

Name Types and Values Description DCC
scale_enable bool
true or false		 If true is specified, scaling is calculated when the bone matrix is calculated.
When exporting from a DCC tool, always specify true.
Fixed
scale_mode enum
standard
maya
softimage		 The scaling calculation method of the bone matrix.
Specifies a value according to the DCC tool.
standard Calculates scale values using the standard method.
maya Calculates scale values, taking into account the segment scale correction applied by Maya.
softimage Calculates scale values, taking into account the scale calculation method used by Softimage.

 Configurable
rotate_mode enum
euler_xyz
quaternion		 The rotation calculation method of the bone matrix.
When exporting from a DCC tool, specify euler_xyz.
euler_xyz Calculates the Euler angle rotation in order of XYZ.
The Euler angle rotation is specified in degrees.
quaternionCalculates using quaternions.

 Fixed
motion_mirroring_enable bool
true or false		 Specify true when performing motion mirroring.
When exporting from a DCC tool, always specify false.
Fixed

Description

There is no content.
