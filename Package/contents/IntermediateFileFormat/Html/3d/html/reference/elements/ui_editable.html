<ui_editable>

Description

Element for selecting whether variables can be edited using the UI.

DCC tools do not output this element.

Parent Elements

  • <attrib_var>: Element that defines shader option variables.

Attributes

Name Types and Values Description
value bool
true or false		 Specifies true if the variable can be edited by the UI.

Description

There is no content.
