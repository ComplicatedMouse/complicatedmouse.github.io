<shader_variation_info>

Description

Element that represents information related to <shader_variation>.

DCC tools do not output this element.

Parent Elements

  • <shader_variation>: Element specified in shader variation files (FSVA or FSVB files).

Attributes

Name Types and Values Description
shader_archive string
[0-9A-Za-z_]+		 The name of the shader archive.
Specify a name based on the filename of the shader definition file, but without a file extension.

Description

There is no content.
