<material_anim>

Description

Element specified in material animation files (FMAA or FMAB files).

The name of the material animation is taken from the filename, excluding the extension.

Parent Elements

  • <nw4f_3dif>: Root element for the 3D intermediate file format.

Attributes

Name Types and Values Description DCC
version enum
4.1.0		 -

Description

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <process_log_array> [0 or 1]: A collection of <process_log> elements.
  • <material_anim_info> [1]: Element that represents information related to <material_anim>.
  • <texture_search_path_array> [0 or 1]: Array element of <search_path> used for texture search.
  • <tex_pattern_array> [0 or 1]: A collection of <tex_pattern> elements.
  • <per_material_anim_array> [0 or 1]: A collection of <per_material_anim> elements.
  • <original_per_material_anim_array> [0 or 1]: A collection of <original_per_material_anim> elements.
  • <user_data_array> [0 or 1]: A collection of <user_data> elements.
  • <stream_array> [0 or 1]: A collection of <stream> elements.
  • <comment> [0 or 1]: Element that stores metadata such as editing comments.
  • <tool_data> [0 or 1]: Element that stores data for NintendoWare tools.
  • <user_tool_data> [0 or 1]: Element that stores data for original user tools.
