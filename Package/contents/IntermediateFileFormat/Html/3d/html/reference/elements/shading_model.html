<shading_model>

Description

Element that defines shading models.

DCC tools do not output this element.

Parent Elements

  • <shading_model_array>: A collection of <shading_model> elements.

Attributes

Name Types and Values Description
index int
0 – 65,534		 Represents an index of the array.
Serving as a hint to improve the readability of the file, this value is ignored when loaded by a tool.
name string
[0-9A-Za-z_]+		 Shading model name.
It is paired with the reference source <shader> name.
Also paired with the reference source <shader_assign> shading_model.
material_shader bool
true or false		 Specify true if the shading model is for materials.
revision int
0 – MAX		 Revision number for the shading model.

Description

At least one of the following elements must be included in the file: <vertex_stage>, <geometry_stage>, <fragment_stage>, <compute_stage>, <tessellation_control_stage>, or <tessellation_evaluation_stage>.

The elements appear in the following order. Numeric values inside square brackets ([ ]) indicate the number of times the element may appear.

  • <vertex_stage> [0 or 1]: Element that defines the shader vertex shader stage.
  • <geometry_stage> [0 or 1]: Element that defines the shader geometry shader stage.
  • <fragment_stage> [0 or 1]: Element that defines the shader fragment shader stage.
  • <compute_stage> [0 or 1]: Element that defines the computer shader stage of the shader.
  • <tessellation_control_stage> [0 or 1]: Element that defines the shader tessellation control shader stage.
  • <tessellation_evaluation_stage> [0 or 1]: Element that defines the shader tessellation evaluation shader stage.
  • <macro_array> [0 or 1]: A collection of <macro> elements.
  • <option_var_array> [0 or 1]: A collection of <option_var> elements.
  • <attrib_var_array> [0 or 1]: A collection of <attrib_var> elements.
  • <packed_attrib_var_array> [0 or 1]: An element of the <packed_attrib_var> array.
  • <sampler_var_array> [0 or 1]: A collection <sampler_var> elements.
  • <image_var_array> [0 or 1]: An element of the <image_var> array.
  • <block_var_array> [0 or 1]: A collection of <block_var> elements.
  • <shader_storage_block_var_array> [0 or 1]: A collection of <shader_storage_block_var> elements.
  • <render_info_slot_array> [0 or 1]: A collection of <render_info_slot> elements.
  • <textblock_array> [0 or 1]: A collection of <textblock> elements.
  • <interleave_array> [0 or 1]: A collection of <interleave> elements.
  • <group_array> [0 or 1]: A collection of <group> elements.
  • <page_array> [0 or 1]: A collection of <page> elements.
