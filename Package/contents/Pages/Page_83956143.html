Tips

Tips and hints for using SoundMaker are explained.

Shortening Conversion Times

Large projects that use large numbers of waveform files can impede normal work, especially due to the length of conversion times.

The following method is considered an effective way to address this issue.

Creating a Sound Project for Sound Sets Edited Most Frequently

Although you ultimately create a sound archive for use by the application, you can speed up normal work by creating several sound projects, and registering those that are changed most frequently instead of working with just one sound project that includes all sounds.

Getting SoundMaker Version Information

You can copy version information about SoundMaker to the clipboard by clicking the Copy Info button that displays in the splash window when you select Help > About This Program from the menu.
