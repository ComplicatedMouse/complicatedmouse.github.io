Position (±)

This section describes the feature for setting the position of the magnetic force.

Overview

The Position (±) feature can be used to set the position of the magnetic force.
The following overview describes the settings ().

Position (±)

Sets the magnetic force position, based on the xyz of the emitter's coordinate system.
The following formula calculates the value added to the velocity of particles as a result of the magnetic force.

  • Value to add to speed: Magnetic force position - Position of the particle at that time - Velocity of the particle at that time) × Magnetic force
