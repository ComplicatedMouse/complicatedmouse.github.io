Emitter

This tab configures the emitter shape and transform settings.

Emitter Settings

The Emitter Settings group () is the group of features for configuring the shapes of emitters and their sizes.
This group has the following settings.

Emitter Shape

Feature for determining the particle emission shape and area.
Scale Feature to apply a scale to the selected emitter shape.
Random Emission Position Feature for emitting particles from random locations.

Emitter Transform

The Emitter Transform feature group used to configure settings related to the position, rotation, and scaling of emitters and for their animations.
This group has the following settings.

Emitter Transform Position(±) The feature for setting the position when placing emitters.
Emitter Transform Rotation(±) The feature for setting the rotation when placing emitters.
Emitter Transform: Scale The feature for setting the scale when placing emitters.
Update Random Width The feature for determining whether to update the random number values set for Position(±) and Rotation(±) in the emitter transform settings every time particles are emitted.
Info

The position, location, and scale of the emitter can be animated in the Curve Editor Window.
