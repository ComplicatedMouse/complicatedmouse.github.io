Items set with animation in the parts layout cannot be overwritten in the body layout.

Question

Animations have been set in a parts layout using Show/Hide.
When these items are overwritten in the body layout where this pane is located with the setting fixed to Hide, the animations are nevertheless played in the game.

Answer

The specification is designed such that items set with animation in a parts layout cannot be overwritten in the body layout.
As a result, the animations in the parts layout are played in the game application even if you overwrite the settings in the body layout.

As a workaround, add a null pane in which animations are not set in the parts layout, and overwrite that pane in the body layout, as shown below.
