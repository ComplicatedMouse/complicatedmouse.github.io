Trimming

This section describes the feature for trimming regions where particles are not displayed.

Overview

The trimming feature creates particles in a way that optimizes the rendered surface area by considering useless alpha regions of the texture.
This can help reduce the processing load because it reduces the superfluous regions that are not used for rendering.
The following overview describes the settings ().

Trimming

Select the check box () to trim to the appropriate size based on the alpha shape of the texture registered to Texture 0.

Info

This feature is ignored when it is used at the following times.

  • When the particle shape is a primitive.
  • When texture pattern animation is being used.
