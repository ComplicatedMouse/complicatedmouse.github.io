Show Ground

This section describes the Show Ground feature for displaying the ground in EffectViewer.

Overview

The Show Ground feature displays a ground or floor in EffectViewer, making it easier to mentally grasp the positions of effects when manipulating the camera.
There are many different kinds of grounds, and you can select the one suitable for the feature that you want to verify.
This section provides an overview of the settings ().

Show Ground

Use the check box () to switch between showing and hiding the ground.
The ground is hidden when the check box is cleared. Use this setting when you want to verify the rendering of individual particles.
Select the check box to display the Show Ground () group of feature settings. The group has the following settings.

Ground Display Type

The type of ground to display in the viewer is selected from among the following presets.

 Grid

Shows the grids. This type is convenient when you are determining the size of effects.
Blocks

Displays blocks rendered opaque, with the grid surface filled with a gray color.
Because it is rendered opaque, it is easier to verify the rendering state of particles that use color buffer refraction and soft particles.

Size

Select the size of one grid or one block from levels 1 through 10.
This allows you to standardize the particle size when creating effects.

Number of Divisions

When displaying a floor as a grid, select the number of subdivisions per grid as 1 or 10.
