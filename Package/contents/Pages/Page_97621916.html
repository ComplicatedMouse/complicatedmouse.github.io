Emitter Billboard

This section describes the feature for orienting the emitter toward the direction of the camera.

Overview

The Emitter billboard feature applies an offset to the rotation angle of the emitter.
If set to No, you can set the angle of rotation of the emitter separately around each axis.

Info

To use the emitter billboard feature on a child emitter, configure the child emitter as follows.
█ are parent particles, and █ are child particles.

Child emitters do not billboard correctly. Configure the following settings.
Particle follow type: Perfectly follow emitter SRT
Emitter Assignment for Particles: ON

Emitter Billboard

Select the emitter billboard settings from among the following presets.

None Do not billboard the emitter. For emitters to be able to rotate, this must be selected.
Emitter billboard Billboard based on the axes of the origin, rotating the emitter so the z-axis remains parallel to the camera lens axis and the y-axis is tied to the up direction of the camera.
Emitter y-axis billboard Billboard based on separate axes, rotating the emitter around only the y-axis while keeping the z-axis parallel to the camera lens axis.

If set to No, the Emitter Rotation Settings group () appears. The group has the following settings.

Rotation (±)

Sets the offset for emitter rotation around the x-axis.
