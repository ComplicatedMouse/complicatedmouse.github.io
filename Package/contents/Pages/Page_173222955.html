3D News

This document is a graphical introduction to the features that were added in NintendoSDK 0.16.0.

Added platform settings to the Team Settings window You can now check the platform settings in the 3DEditor Team Settings window.
Wrap and mirror settings specified in Maya are applied during model merge and export The wrap and mirror settings specified with the Maya plug-in can now be applied.
Shapes With Different Skinning Weights Can Be Compressed Shapes with different skinning weights when exported with shape compression enabled in the Maya plug-in can now be compressed.

3DEditor

Added Platform Settings to the Team Settings Window

You can now check the platform settings in the Team Settings window.

Related Links For more information, see Team Settings.

Maya Plug-In

Wrap and Mirror Settings Specified in Maya Are Applied During Model Merge and Export

When configuring the following setting in the config file, the wrap and mirror settings for the place2dTexture node specified in Maya are now applied during merge and export.

sampler_merge_priority="true"

If this setting is not configured, the behavior is the same as before and does not change.

Related Links For more information, see Using the Export Plug-In.

Shapes With Different Skinning Weights Can Be Compressed

When exporting shapes with shape compression (Compress Shape) enabled, until now shapes with different skinning weights were not merged. They can now be merged by configuring the following setting in the config file.

compress_ignoring_vertex_skinning_count="true"

Related Links For more information, see Using the Export Plug-In.
