File

Describes the File menu in LayoutEditor.
Create New The procedure for creating a new layout window.
Open Opens a previously saved layout window file.
Open Additional Adds additional layout window data to the currently selected layout window.
Save Saves the current layout window.
Save As Saves the layout window with a new name.
Closing Closes the layout windows.
Export Binary Opens the Export Binary File window.
Recently Used Files Displays a list of recently used files. Selecting one of the files opens it in the program.
Re-export the layout files under the parts root folder

The path of the folder that contains parts, is known as the parts root folder.

This option loads all layout files under the parts root folder in the tool, and then re-saves them.
Batch synchronize all layout files contained within the specified folder

The layout for all bodies that use parts in the specified folder is reflected by changing the extended user information in the parts layout.

Any parameters that would be overwritten in the body layouts are left unchanged.
Convert files in the specified folder to separated mode. Specify a folder in the dialog box that opens after running to batch-convert the layout files in that folder to separated mode.
Finalizing Closes LayoutEditor.

Added Thumbnail Save Feature

You can add a feature to save images for use in thumbnails in LayoutEditor.

When the setting is enabled in the project settings file, thumbnail images are saved as LayoutFilename.flyt.png in the .NwThumbs folder next to layout files when layout files are saved, during batch synchronization, and at other times.
The Update Thumbnail Image in the Specified Folder menu is also added to the File menu in LayoutEditor.

Saved images can be displayed using the Thumbnail Explorer Extension.

File Menu in LayoutEditor with Added Menu Option

A Update Thumbnail Image in the Specified Folder option added to the File menu.
