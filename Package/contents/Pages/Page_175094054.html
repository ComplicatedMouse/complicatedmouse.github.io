Fields

This page describes fields, which can be used to apply changes to particle movement in directions other than the initial velocity direction.
You can set fields to apply force to particles in directions other than their initial velocity.
The field features in EffectMaker are in the following locations.

  • The Air resistance and Gravity fields, often used in making effects, are in the Emission tab of the emitter node.
  • There are Field Nodes that can be added when you want to use them.

Related Links For more information, see the following pages.

Air Resistance This field includes emitter nodes.
Describes the Air resistance feature that decelerates or accelerates the initial velocity of all emitted particles.
Gravity This field includes emitter nodes.
Describes the Gravity feature that applies gravity to emitted particles.
Field Node A field node. Multiple types of fields can be added as nodes.

Field Types

Fields are broadly separated into to following two types.

Position Fields These fields update the position of the particle for each frame.
Velocity Fields These fields update the velocity of the particle for each frame.

The following fields are Position Fields. All other fields are Velocity Fields.
Position fields are not affected by Air resistance.

GPU Noise This section describes the GPU Noise field, which moves particles in accordance with the Random Trajectory.
Spin This section describes the Spin field, which sets all particles moving like a twister.
Convergence This section describes the Convergence field, which moves particles so they converge at a specified location.
Add to Position This section describes Add to Position field, which moves particles in the specified direction.
Info

While Simple Collision is a velocity field, the behavior after landing is adjusted by moving the position.

Fields Unaffected by Momentum Random

The following fields are unaffected by Momentum Random.

Random This section describes the Random feature for eliminating any orderliness to the movements of particles.
GPU Noise This section describes the GPU Noise feature for moving particles in an orderly way that nevertheless appears random.
Add to Position This section describes Add to Position field, which moves particles in the specified direction.
Curl Noise This section describes the Curl Noise feature, which places turbulence in space to control the movement of particles.
