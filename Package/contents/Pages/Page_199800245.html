LayoutEditor

This section introduces the new features added to LayoutEditor.

Added "Controls for Switching the Connection Destination"

Added to the toolbar controls for switching the connection destination.


Added a drop-down menu for switching the connection destination (PC/development hardware) to the toolbar of the LayoutEditor window.
The operation for switching the connection target is now easier to understand.

Switching Controls on the Toolbar
　PC Connect to the PC.
　Development Hardware Connect to the development hardware.

The same feature that was previously in the Tools Menu has been discontinued.
