Feature Status Determination

Overview

Feature status determination is a feature for determining, after initialization, whether a specific function was loaded and whether a specific extension is supported.

Before executing feature status determination, you must first successfully initialize the target function.

Determining Whether a Specific Function Was Loaded

You can determine whether a specific function was loaded by checking whether the pointer for that function is null.

Usage Examples

For example, you can write the following when you want to call glDepthBoundsEXT if it was loaded.

// nngllInitializeGl() was successful.
if( glDepthBoundsEXT )
{
    glDepthBoundsEXT( 0.0f, 1.0f );
}

Example of Use in a Sample

See the GllSimple sample.

Determining Whether a Specific Extension Is Supported

You can determine whether a specific extension is supported in the context during initialization by passing that extension name to the NN_GLL_IS_SUPPORTED() macro parameter.

Usage Examples

For example, you can write the following when you want to enable conservative rasterization if GL_NV_conservative_raster is supported.

// nngllInitializeGl() was successful.
if( NN_GLL_IS_SUPPORTED( GL_NV_conservative_raster ) )
{
    glEnable( GL_CONSERVATIVE_RASTERIZATION_NV );
}
