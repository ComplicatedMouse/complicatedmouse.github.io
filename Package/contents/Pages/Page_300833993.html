Handling in Programs When Using Scissor Panes

About the Scissor Pane

The Scissor Pane is used for clipping the rendering results of panes in the hierarchy of a specified region.

You can use the feature by placing the Scissor Pane in LayoutEditor.

The Scissor Pane acts internally to set the scissor using hardware features and to restore the original scissor setting after rendering, but the program must specify ahead of time the regions to which the viewport and scissor should revert.

Specify the regions to which the viewport and scissor should revert, in the following way, when you initialize the nn::ui2d::DrawInfo class.

// Configure the default settings for the viewport and scissor.
const int width = 1280; // Specifies the default width.
const int height = 720; // Specifies the default height.

nn::gfx::ViewportStateInfo  viewportInfo;
viewportInfo.SetDefault();
viewportInfo.SetWidth(static_cast<float>(width));
viewportInfo.SetHeight(static_cast<float>(height));

nn::gfx::ScissorStateInfo  scissorInfo;
scissorInfo.SetDefault();
scissorInfo.SetWidth(width);
scissorInfo.SetHeight(height);

g_pDrawInfo->SetDefaultViewportScissorInfo(viewportInfo, scissorInfo);
