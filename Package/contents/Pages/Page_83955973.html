Example of Working Collaboratively Using Sound Projects

Sound projects and sound sets can be used freely in any manner.
When one person creates a project, there is no problem working on a single sound project and a single sound set.

Collaborative work can be performed by splitting up sound sets based on the purpose of the sound even when a single person is working on the project alone or by sharing sound set files amount SoundMaker users for large-scale projects with more than one staff member.
