Introduction

Purpose of This Documentation

This document describes the features provided by the play data management (pdm) library in NintendoSDK.

Info

The play data management library is available in NX Add-On 5.0.0 and later.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Related Documentation

Also see the Play Data Management Library API Reference.

How the Play Data Management Library Is Provided

See the following pages.

Features Provided by the Play Data Management Library

The play data management library provides the following features.

  • Get play data about applications that have previously been started on the system.

Getting Play Data for Applications

Applications can use the nn::pdm::QueryApplicationPlayStatistics function, which takes application IDs as inputs, to get play data about those applications.

Play data includes the following information.

  • Startup count
  • Play time

The obtained information can be divided up by user accounts used in the target applications to get play information on a device-by-device basis.

To use this feature, the NMETA file created when creating an application must have PlayLogQueryCapability set to WhiteList and have the application IDs of the applications for which to get information listed in PlayLogQueryableApplicationId.

For more information about writing NMETA files, see Writing the NMETA File.

Info

The system region for storing play information is limited in size, and old records may be deleted as necessary. For this reason, the startup counts and play times included in the play information are not guaranteed to increase monotonically. In addition, this information may be deleted, such as when the system is initialized.

In most normal use cases, the upper limits on the amount of play information stored will never be reached. However, never implement any process that depends on this information increasing monotonically.

Restrictions

The play data management library is subject to the following restrictions.

Getting Play Information in the Windows Environment

Play information cannot be retrieved in the Windows environment.
