Texture Files

Texture files are image files used in emitter set files.
This page describes how to use texture files in EffectMaker.

Where to Save Texture Files

Emitter set files (ESET files) store only the names of texture files and do not store the paths for the texture files.
As a result, when opening an emitter set file, EffectMaker searches for a texture file with the same name as the one stored in the emitter set file.

Save texture files in a searchable folder on the same drive as the associated emitter set file.
The following image shows the search order and the searchable region of folders for EffectMaker, where EmitterSet is the emitter set file (ESET file) to be opened and Texture is the texture file corresponding to the name stored in the emitter set file.

  1. When the EmitterSet.eset emitter set file is opened, EffectMaker first searches for texture files that are in the same folder.
  2. EffectMaker then searches in the Textures folder directly below the folder that the emitter set file is in.
  3. EffectMaker then searches in the Textures folder above the folder that the emitter set file is in until it finds a texture file.
  4. Finally, EffectMaker searches in the highest level Textures folder on the drive. This folder is the last folder that can be searched.
    If no texture file has been found by this point, EffectMaker displays a warning.
Info

EffectMaker makes no distinction between uppercase and lowercase characters in the Textures folder name.
