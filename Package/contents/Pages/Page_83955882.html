Plug-In for Setting Material Attributes - 3ds Max

Plug-In for Setting Material Attributes

Sets the NintendoWare material attributes for materials. Note that the settings made with this plug-in are not applied to what is shown on the 3ds Max screen.

Configure Set Material Attribute using either of these methods.

Settings Materials Individually

The NintendoWare data creation plug-ins have been designed to enable searching and configuration of multiple materials simultaneously.

However, you can also configure the NintendoWare material information separately for individual materials in the Material Editor. To do so, use the following procedure.

To add NintendoWare material information, select a material and then select Add Material Attribute from the Nintendo menu of the Material Editor.

The information in the following image is added automatically when you use the data creation plug-in. Here you can directly change the settings for individual materials.
To delete the NintendoWare information that has been added to a material, select Delete Material Attribute from the Nintendo menu.

You can also set user data for materials.
To open the settings window, select a material and select Set Material User Data from the Nintendo menu of the Material Editor.
For information about how to configure the settings, see Plug-In for Setting User Data (Set User Data).

If material settings have been configured for Multi/Sub-Material, the same material information is exported for all sub-materials.
Also, you can set different material information for each sub-material.
If you do not set material information for sub-materials, multi/sub-material information is exported.

Info

Setting or deleting additional information specific to NintendoWare has no effect on the work view or data structure on 3ds Max.

Settings for Multiple Materials at The Same Time

Select Nintendo > Set Material Attribute to display the window.

You can use this window to change the settings for multiple materials simultaneously, or to search for materials.
You can select the materials assigned to objects by selecting multiple objects.

Materials listed in Selected Material are the settings target.
When the content of a Set is changed, the change becomes applied to that item in those materials.
The settings can be changed for more than one material simultaneously. The settings, however, will not be changed for items that are not updated.

  • Compress
    Sets a flag that prevents compression even when the Compress Material option of the Export plug-in is selected.
    Select Compress if possible to use compression if other materials have the same attributes. This setting is the default.
    Select Don’t compress to export the material as an independent material without compression, even if the attributes are the same as other materials.

Searching for Attributes

Enter criteria in the Search field and then click Search to search through the materials registered in the Material Editor. The names of relevant materials are displayed in the Selected Material field.

The name of the material selected in the Target field becomes the settings target, and its name is displayed in the Set field.
