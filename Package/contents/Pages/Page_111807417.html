Hierarchy Window

The Hierarchy window can be used to create hierarchies between panes.

A




  • Cut
  • Copy
  • Paste
  • Delete
 Copies and pastes panes.
  • Lock Pane
  • Unlock Pane
  • Hide Pane
  • Display Pane
 Performs operations on the selected pane.
  • Select Hierarchy
Selects the hierarchy under the selected pane.
  • Lock Hierarchy
  • Unlock Hierarchy
  • Hide Hierarchy
  • Display Hierarchy

Locks, shows, or hides the hierarchy under the selected pane.

Works the same as the operations performed in E and F.
  • Create Group
  • Edit Group
 Edits Groups.
  • Display Parts Layout of Selected Pane
  • Open Parts Layout of Selected Pane in Another Window
  • Properties
 Opens the parts layout or the property window.
  • Display Selected Pane Automatically ()

Switches the display of the selected pane between Automatic and Manual modes.

You can also use B to switch between Automatic and Manual modes.


When using Automatic mode, the hierarchy will be opened automatically when clicking the pane where the hierarchy is collapsed in the Hierarchy window.

When using Manual mode, the hierarchy will be not opened automatically when clicking n the pane where the hierarchy is collapsed in the Hierarchy window.
B
Switches the display of the selected pane between Automatic and Manual modes. Selects and clears the Display Selected Pane Automatically () option in A.
C
Expands all of the hierarchies in the selected layout window.
D
Closes all of the hierarchies in the selected layout window.
E

Hide Hierarchy / Display Hierarchy

Click the Show/Hide icon () or the blank space on the left to display or hide the panes in the layout window.

Drag in the location where the icon appears to switch between showing and hiding for multiple panes.

Hold the ALT key and click the icon to hide all items other than the ones that have been clicked. (Support is planned.)


This option works the same as the Hide Hierarchy and Display Hierarchy features in A.
F

Lock Hierarchy / Unlock Hierarchy

Click the blank space on the right or the Lock/Unlock icon () to lock or unlock the panes in the layout window.

Drag in the location where the icon appears to switch between locking and unlocking for multiple panes.


This works the same as the Lock Hierarchy and Unlock Hierarchy features in A
G Hierarchy display area

Displays the hierarchy.

Drag items to change the hierarchy.
Click the icon to expand the hierarchy.
Click the icon to collapse the hierarchy.
