Profiler-NX-FileSystemTrace

Overview

Controls whether File System Trace data should be recorded in the profile data or not. This will only enable the feature if the File System Trace button is enabled for selection at the time this cmdlet is called. If the File System Trace button is disabled, file system tracing will be disabled.

Profiler-NX-FileSystemTrace [-EnableFileSystemTrace] <Boolean>

[-EnableFileSystemTrace] <Boolean>

If true, record file system tracing in the profile data. If false, turn off recording file system trace data.

Examples

Basic Usage

Profiler-NX-FileSystemTrace $true
