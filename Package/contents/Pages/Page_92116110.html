Features

Feature Descriptions

These sections describe the various features of EffectMaker, along with examples of configured settings.

Emitter Set Node This page describes the emitter set node that allows you to gather multiple emitters and previews into a single set within a workspace.

Emitter Node

This page describes the emitter node that allows you to configure settings related to particle creation and emission.
The icon displays differently, depending on whether particle behavior calculations are performed in the CPU, GPU, or GPU+SO.
Field Node This page describes the field node that applies changes to the speed of particles emitted from the emitter.
Emitter Plug-In Nodes This page describes the emitter plug-in node that provides a group of additional features for the existing particle system.
Custom Action Nodes This page describes the custom action node that adds user-proprietary data on a per-emitter granularity.
Emitter Extended Parameter Node This page describes the emitter extended parameter, which can be used to configure data available to the game application.
Preview Node This page describes the preview node that allows you to check aspects of emitter sets that can be controlled in the game application.
Viewer Nodes This page describes the viewer node that controls the viewer features and display.
Model Settings Nodes This page describes the model node that registers models and animations in EffectMaker.
Options This page describes the Option feature used to configure the EffectMaker and Viewer environment.
