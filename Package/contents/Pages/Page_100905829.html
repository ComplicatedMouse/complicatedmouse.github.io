NVN Graphics Debugger

Overview

The NVN Graphics Debugger is a dedicated graphics debugger for NVN. It provides features for debugging draw calls, states, and resources, in addition to features for performance analysis and capturing rendered frames.

NVN Graphics Debugger is intended to be used for debugging applications. ROMs that use this library cannot be uploaded to the Nintendo Developer Portal (NDP) as master ROMs.

Packages

The NVN Graphics Debugger is provided as a separate package from the NintendoSDK NX Add-On. It may also be released later than other NintendoSDK packages.

The following table lists the packages.

Package Name Description
NVN Graphics Debugger A dedicated graphics debugger is for NVN.

Placement

The NVN Graphics Debugger is located below the directory where NintendoSDK is installed, as shown below.
