Root CA Certificates

Overview

Root CA certificates that can be used in the web applet can be largely categorized into the following two groups.

  • Certificates read from the SDK SSL library.
    • For more information, see Features > Network > SSL Library > Built-in Data.
  • Additional certificates loaded in the web applet.
