Control Settings Window

Use the Control Settings window to configure controls for parts.

By adding control definitions you can define new types of functionality.
For example, if developers prepare an extended program, the designers can create buttons and other parts with unique capabilities.

The following operations can be performed in the Control Settings Window.

Control Settings Window
A Control Type

Select the type of control to configure from the drop-down list.

  • Normal layout [ None ]
  • Locator part [ Locator ]
  • Standard button [ NormalButton ]
  • Check button [ CheckButton ]
  • Decision button [ DecisionButton ]
  • Selection button [ SelectCutton ]
  • Touch Screen button [ TouchOffButton ]
  • Check button for the Touch Screen [ TouchOffCheckButton ]
  • Drag button [ DragButton ]
  • Drag button for the Touch Screen [ TouchDragButton ]
B Control Description Displays a description of the selected control.
C Name Sets the name of the control.
