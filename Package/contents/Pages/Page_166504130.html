LayoutEditor

This section introduces the new features in LayoutEditor.

The width can now be adjusted on a per-line basis in the Text Box Pane

The width of the Text Box Pane in LayoutEditor can now be adjusted on a per-line basis for greater expressiveness.

Similar to as in a newspaper or magazine, this useful feature allows you to adjust the layout of text along with images.
You can freely change how multiple lines of text display in a single pane.

Property Window > Text Box Pane > Specify the width of each line

To adjust the width of each line, separately in each line, enter numeric values for the starting place and the ending place of the text.

This feature lets you freely set the width of the lines in the text box pane relative to the horizontal (X direction) size that has been set for the base pane.
Set the starting place (on the left) and the ending place (on the right) for the line, entering negative values that define how much inward to narrow the line by from either direction.
You cannot enter positive values, since this would elongate the width beyond the size of the text box pane.

You can set the width for as many as 16 separate lines.

Specify Width Per Line (for All 16 Lines) in the Property Window Text Box Pane With Per-Line Width Settings Applied
Enable Feature on/off switch Selecting the check box enables the feature.
Size X: The size of the text box pane in the X direction.

Shows the size of the text box pane in the X direction.
This is the size in the X direction configured for the base pane in the Property Window.

When you adjust the width on a per-line basis, the size of each line changes, so this number serves as a reference value.
A Setting to narrow the line from the left (1st line). Sets the width from the left for the 1st line, in units of pixels. The starting place of the 1st line of text.
B Setting to narrow the line from the right (1st line).

Sets the width from the right for the 1st line, in units of pixels. The ending place of the 1st line of text.
C Setting to narrow the line from the left (2nd line). Sets the width from the left for the 2nd line, in units of pixels. The starting place of the 2nd line of text.
D Setting to narrow the line from the right (2nd line). Sets the width from the right for the 2nd line, in units of pixels. The ending place of the 2nd line of text.
E Setting to narrow the line from the left (3rd line). Sets the width from the left for the 3rd line, in units of pixels. The starting place of the 3rd line of text.
F Setting to narrow the line from the right (3rd line). Sets the width from the right for the 3rd line, in units of pixels. The ending place of the 3rd line of text.
G Setting to narrow the line from the left (4th line). Sets the width from the left for the 4th line, in units of pixels. The starting place of the 4th line of text.
H Setting to narrow the line from the right (4th line). Sets the width from the right for the 4th line, in units of pixels. The ending place of the 4th line of text.
I Setting to narrow the line from the left (5th line). Sets the width from the left for the 5th line, in units of pixels. The starting place of the 5th line of text.
J Setting to narrow the line from the right (5th line). Sets the width from the right for the 5th line, in units of pixels. The ending place of the 5th line of text.
K Field for entering values to narrow the line from the left. Sets the line width from the left side rightward, in units of pixels. Values can be set for up to 16 lines.
L Shows the line width that results from specifying the width values. When the Specify the width of each line feature is used, displays the result of changing each line width.
The displayed numbers are linked to the numbers entered for narrowing each line from the left and right.
M Field for entering values to narrow the line from the right. Sets the line width from the right side leftward, in units of pixels. Values can be set for up to 16 lines.

In the properties window, in Color interpolation, you can now drag colors in the color picker separately for RGB and A.

In LayoutEditor, the color picker in the properties window has been split into RGB and A, and you can drag colors separately to each part.

Drag to the left side of the color button in the properties window to copy the RGB color, and drag to the right side to copy the A color.

Color Interpolation Settings (Property Window)
  • You can separately drag the RGB and A values.
  • RGB is to the left, and A is to the right.
