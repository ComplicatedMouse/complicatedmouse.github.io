Attach Path

Overview

The attach path is used to search for reference files during runtime collaboration model attach and shader attach.
If multiple attach paths have been set, the search is performed in the order they were set.

Specify a folder path for the attach path. You can set the absolute path.
Path strings may include environment variables surrounded by percent signs (%). 3DEditor interprets such paths by replacing any environment variables surrounded by percent signs (%) with the value the environment variable represents.
