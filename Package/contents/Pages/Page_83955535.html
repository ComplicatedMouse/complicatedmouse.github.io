Shape Property Window: General Page

This page is used to check and edit the general properties of the selected shape.

A

The name of the bone to which the shape is bound.
Used for rendering bone visibility animation and shapes whose skinning type is rigid body.
B

The name of the material to use when rendering the shape.
C

The shape’s material assignment can be changed.

Click this button to open the Change Material window, and then select the material file.
D

The skinning type of the shape.
Select smooth skinning, rigid skinning, or rigid body.
If the skinning type is smooth skinning, the number of bones influenced by a single vertex is also displayed.
E

Applied approximated vertex delete mode information.
It is blank when exported from a DCC tool.
F

Applied submesh division mode information.
It is blank when exported from a DCC tool.
G The primitive optimization mode to apply.
H

Hash value for raw shape data.
I Number of shapes before applying shape compression. Always 1 when exported from a DCC tool.
J The bone name of the original shape before optimization.
K

The material name of the original shape before optimization.

L

The number of vertices to process when rendering the shape.

Switches the display of the processed vertex count appropriately according to the settings of the connection target platform.
M

The number of vertices of the shape.
N

The polygon type of the shape.
O

The type of primitive used to build the polygon of the shape.
Select triangles, triangle fans, or triangle strips.
P

The primitive topology type.
Q

The number of shape indices.
R The index quantization type of the shape.



Changing Materials

This feature changes the material assigned to the shape.

A

Displays a list in C of the material files for the model intermediate files described in the path in Team Settings.
B

Displays a list in C of the material files for the model intermediate files in the model folder.
C Displays a list in C of the material files for the model intermediate files loaded in 3DEditor.
E Changes the material if you press the OK button.
F Cancels the material change.


Optimization Modes

The following primitive optimization modes are available.

Optimization Mode Description
None applied Does not optimize primitives.
Minimum optimization already applied Primitive optimization is applied using standard settings.
Strong optimization has been applied Primitive optimization is applied over time.
Extra-strong (brute force) optimization has been applied Primitive optimization is applied over even more time.
