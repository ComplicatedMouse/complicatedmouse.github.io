Debugging

Overview

This page describes the debugging features provided by the ec library.

List of Debugging Features

Debugging Overview
Notifications of Access to Shop Services

This feature displays an overlay notification on the screen when the application requests access to the shop service server.

Used for checking compliance with the guidelines for accessing the shop item catalog service server.
Response Simulation Feature

This function simulates access to the eShop service server from the application.

Register the planned shop item catalog service server response and use it to validate the behavior of the application.
JSON Compatibility Check Feature This function adds dummy elements not included in the original response to the JSON response obtained from the eShop service server and then from the application.
Used for checking compliance with the guidelines for accessing the shop item catalog service server.
