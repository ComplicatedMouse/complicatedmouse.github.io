Origin (V Direction)

This page describes the Origin (V Direction) feature, which sets the origin position when scaling or rotating characters for per-character transformation.

Overview

The Origin (V Direction) feature sets the origin position when scaling or rotating with the per-character transformation.
The table later in this section provides an overview of the Origin (V Direction) settings.

Origin (V Direction)

Use the radio buttons to set the origin position for the character.

Center The center of the character is the origin.
Bottom The bottom of the character is the origin.
