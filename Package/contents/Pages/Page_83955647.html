Search Feature

Search Feature

Use the Search window or Search tab to find items in a sound project based on characters and special conditions. You can also search for references from a list.

Using the Search Window

To open a Search window, select Edit > Search.
You can also use the CTRL+F shortcut key for this operation.

Using the Search Window

Categories Tabs

Items that can be searched are divided into sound set items and instruments.
Select the tab of the category you want to search.

Search Range

Select the range to search from the drop-down list.

Entire Project

Search the entire project.

Specified Sound Set

You can select Specified sound set when the Item tab in the Search window is selected.
The Select sound set button becomes active when Specified sound set is set, which makes it possible to open the Select sound set dialog box.
The Select sound set dialog box shows a list of the sound set names registered in the project. Select the check boxes for the sound sets you want to search.

Current Bank

In the Search window, when the Instrument tab is selected and the Bank tab is active, you can select Current Bank.

Current List

Search the list that is currently displayed in the active tab.
This action is not shown if there is no active tab.

Search Target

Use the check boxes to specify what items to search when searching for the string specified in Find What.

You can specify any of the following as search targets.

  • Name
  • File Path
  • Comments
  • Item references
  • Starting playback position (for sequence sounds)
Info

• Stream sound tracks do not have Name and are not found in name searches.
• To search for items that are using the player (bank), select the Item reference check box.

Option

You can use the asterisk (*) as a wildcard by selecting the Use wildcards check box in Search String.

  • When cleared:

The asterisk symbol (*) is treated as a character for partial match queries.

  • When selected:

The asterisk symbol (*) is treated as a wildcard for matching.

Select Continue search from beginning of range to loop back and search from the start after you have reached the end of the search range in a Find Previous or Find Next search.

Search Button

Find Next (Find Previous)

From the selected position in the active tab, a search is made for the next (or previous) item that matches the conditions. The position of the found item is displayed selected in the Sound Set tab or the Bank tab.
Clicking this button does not close the Search window.

Shortcut Keys

Shortcut keys have been configured for searches.

Search Next: F3 (ENTER also works if the focus is on the Find What text box.)

Search Previous: SHIFT+F3 (SHIFT+ENTER also works if the focus is on the Find What text box.)

Reference Search

To search for items that are using a particular player or bank, right-click either the Player list or Bank list, and select Find All References from the shortcut menu.

The entire project is searched for items that reference the target item, and the results are listed in the Search tab.

During conversion, warning messages about unreferenced items are displayed in the Conversion Log window.
