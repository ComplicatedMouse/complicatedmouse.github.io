Colocate Textures, Fonts, and Parts

This section describes the Colocate Textures, Fonts, and Parts feature.

The Colocate Textures, Fonts, and Parts feature automatically creates a set of associated files when the layout data is transferred.
Select the layout data you want to transfer and run Colocate Textures, Fonts, and Parts to extract and group (colocate) the data you want to transfer into a single folder.
There is no need to select layouts and textures one-by-one and remake the paths for the data to transfer.

Colocate Textures, Fonts, and Parts Window
A Colocation path

Enter the path to write the colocated data to.

The colocated and written data reference the specified path.
B
Create a subfolder with the layout name

When selected, a folder with the same name as the layout is created and specified as the folder to write the colocated data to.

By sending and receiving this folder, you can easily deliver layout data.
C
Sort textures for each layout in subfolders (with textures for parts going into Common)

When selected, the colocated textures are sorted into subfolders for each layout.

Use this setting when you want to sort the textures of multiple layouts.

Parts textures are sorted into Common.
D Colocation target path

Specifies the folder to write the colocated data to.

When B Create a subfolder with the layout name is selected, a folder to write to is created using the layout name and specified as the colocation target.
E Run Colocates data to the specified path.
