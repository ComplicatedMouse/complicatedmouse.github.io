Creating Distance Field Textures (Basic)

This section describes how to create distance field textures using the Photoshop Distance Field plug-in.
By using distance field textures, you can enlarge textures while maintaining their distinct edges.

Preparing to Create the Texture

To prepare to create the texture, start an instance of Photoshop with the plug-in installed.

Related Links For more information about setting up the Photoshop plug-in, see the Introduction to the Photoshop plug-in.

Setting the Image Mode

For the Distance Field Filter plug-in to run, the image mode for the file must be either Grayscale mode or RGB Color mode, and the file must be 8 bits/channel.
Use the following procedure to change the image mode. In this example we set the file to RGB color and 8-bits/channel.

1 From the Photoshop menu, select Edit > Image > Mode > RGB Color.
2

From the Photoshop menu, select Edit > Image > Mode > 8 Bits/Channel.

Changing the Interpolation Method

To improve image quality, we recommend applying the Distance Field Filter plug-in to a high-resolution image and then shrinking the image.
To prevent image deterioration due to excessive interpolation when the image is shrunk, we recommend setting the interpolation method as follows.
Related Links For more information, see Improving Texture Quality.

1 From the Photoshop menu, select Edit > Preferences > General.
2

In the General dialog box that opens, select Nearest Neighbor (preserve hard edges) from the Image Interpolation presets.
3. Click OK to confirm the settings.

Opening the Distance Field Filter Plug-In

Open the Distance Field Filter plug-in.

1 Make the window with the loaded DistanceField the active window.
2

From the Photoshop main menu, select Filter > Nintendo > Nintendo Distance Field.
3. Open the Nintendo Distance Field dialog box.

Setting the Maximum Distance for Gradations

Set several pixels for Max Distance, which is the maximum distance from the black/white boundary.
The following image shows the Max Distance settings, and the table provides details about the settings.


Setting Value Description
Max Distance
 		96 As shown in the preceding image, Distance information (gradation) is created for every 96 pixels away from the black/white boundary.

Shrinking the Image

The resolution of the image we will actually use is 128×128, so here we shrink the image.

1 From the Photoshop menu, select Edit > Image > Image Size.
2

In the Image Size dialog box that opens, set 128 pixels for both Width and Height.
3. Click OK to confirm the settings.
