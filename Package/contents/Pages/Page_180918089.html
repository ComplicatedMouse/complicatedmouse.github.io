Pattern Tables

Use the pattern table to specify which pattens to use for a texture pattern animation and to change the order in which the patterns are animated.

A Easy Input

Make entries using a hyphen (-) to specify a sequence, and a comma (,) to specify individual frames.

In the example shown, the sequence of 1, 2, and 3 is specified by 1-3, while 0,1,2 specifies individual frames separated by commas.

The upshot is that the pattern is animated in the order 1 , 2 , 3 , 0 , 1 , 2.
B Enter in Units of Frames Specify the number of pattern tables to use and the frames you want to link, one frame at a time.
