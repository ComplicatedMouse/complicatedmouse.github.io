DrawParam

DrawParam

Overview

nn::mii::DrawParam maintains the parameters required for rendering individual parts that make up the face model, face outline texture, and mask texture. Get and render the DrawParam for each part from each class.

Information Contained in DrawParam

Types of Modulation

DrawParam contains the type determining which modulation to use for rendering. For more information on types of modulation, see the Modulation description page.

Vertex Attribute Buffers

The types of vertex attribute buffers that are able to be obtained from DrawParam, are related to the nn::mii::DrawParam::AttributeType enumerated type. They are as follows.

Type Description
nn::mii::DrawParam::AttributeType_Position Position
nn::mii::DrawParam::AttributeType_Normal Normals
nn::mii::DrawParam::AttributeType_Uv Texture Coordinates
nn::mii::DrawParam::AttributeType_Tangent Tangents
nn::mii::DrawParam::AttributeType_Param Special Parameters

You can get nn::gfx::Buffer and the size, stride, and format information of each vertex attribute. For more information, see the API reference.

Texture Descriptor Slot

If you need textures to render parts, you can get the pointer of the descriptor slot that the texture is registered to.

You can use the nn::mii::DrawParam::GetTextureDescriptorSlot() function to get the descriptor slot. Depending on the modulation type, the part may not require a texture. If there is no texture, the GetTextureDescriptorSlot() function returns nullptr.

Constant Color

You can get the constant color that is used in modulation. Use the nn::mii::DrawParam::GetConstantColor() function to get the constant color. You cannot get colors for the body portion of a Mii character. Get and use your favorite color.

Culling Mode

You can get the culling mode specified during rendering. Use the nn::mii::DrawParam::GetCullMode() function to get the culling mode. Change the rendering settings and render according to the obtained culling mode.
