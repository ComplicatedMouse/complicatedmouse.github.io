Sample

The Win32 version Demo1 does not start.

Symptoms

For Samples\Sources\Applications\Demo1, Demo1 does not start if you select Win32 as the platform and build it.

The location where the error occurs differs depending on the build combination.

Workaround

Currently none.

Fix Schedule

This bug will be fixed in a future release.
