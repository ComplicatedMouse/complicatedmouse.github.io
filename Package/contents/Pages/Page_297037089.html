Capture Effects (Mask)

The Capture Effects (Mask) page is used for setting the masking method that uses the mask image registered to a pane.

Enable Mask

Use the check box to set whether to enable the mask feature.
When the check box is selected (), the Mask settings appear.

Mask

Configures various settings for the mask image used for masking.

Changing the Mask Image

To mask the pane, select one of the textures registered in the texture window.

Related Links For more information about the texture window, see here.

Swapping the Pane and Mask Image

Sets how to mask the pane.

Swap the pane and mask image ()

When cleared (default)

Masks the pane's rendering result using the image set for Mask.

Swap the pane and mask image ( )

When selected

Masks the image set for Mask using the pane's rendering result.

Settings

Configures the mask image settings.

Color Image

Displays the color image for the selected mask image.

Alpha Image

Displays the alpha image for the selected mask image.

Text Box

Displays the file name of the selected mask image.

Format

Displays the format of the selected mask image.
Size Displays the size of the selected mask image.
Repeat Method Sets the repeat method for the mask image.
Filtering Specifies information about the Filtering section.
SRT Transformation Matrix Configures the SRT transformation matrix settings.

Repeat Method

Configure the repeat method for picture data.

U Direction

Sets the repeat method to the u direction.

V Direction

Sets the repeat method to the v direction.

Select the repeat method for both the u direction and the v direction from among the following presets.

Clamp

 This setting stretches the top, bottom, left, and right edges of the mask image.

Repeat

 This setting repeats the mask image.

Mirror

This setting repeats the mask image, flipping it each time.

Filtering

Select among the following presets for filtering the mask image.

When Expanding:

Configures the filtering when expanding the texture.

When Shrinking:

Configures filtering when shrinking the texture.

Select the filtering for both expanding and shrinking the texture from among the following presets.

Near ()

 Displays each pixel distinctly.

Linear ()

 Interpolates to smooth out pixel edges.

SRT Transformation Matrix

Enter information about the SRT transform matrix for the mask image.

U Scale:

Sets the scale value for the U direction.

V Scale:

Sets the scale value for the V direction.

Rotation:

Rotation values.
U Translation: Sets the translation value for the U direction.
V Translation: Sets the translation value for the V direction.
Reset Resets SRT to the initial values.
