4 Editing the Child Material

This section describes how to edit the child material (IceCreamMt) to give it different toppings.

Toppings

The toppings are expressed by adding textures.

There are three tasks to perform for each of the toppings, IceCream_Chips, IceCream_Chocolate, and IceCream_Marble.

Sprinkles Topping

Double-click the IceCream_Chips material in the file view to open that material.

Select Shader > Multi-texture: > Multi-texture blend mode: and change the mode from None to Alpha blend.


Drag the IceCreamMt_alb_ChildA.ftxb texture to albedo texture 1 in the following location.

Location %NINTENDO_SDK_ROOT% \ Samples\Sources\Applications\G3dDemo\Resources\SampleIceCream\textures



A topping is displayed on the ice cream cone on the left, as shown in the following figure.

Chocolate Topping

Using the same procedure described earlier, double-click the IceCream_Chocolate material to open it.

Select Shader > Multi-texture: > Multi-texture blend mode: and change the mode from None to Alpha blend.

Drag the IceCreamMt_alb_ChildB.ftxb texture to albedo texture 1.

Marble Topping

Using the same procedure described earlier, double-click the IceCream_Marble material to open it.

Select Shader > Multi-texture: > Multi-texture blend mode: and change the mode from None to Alpha blend.

Drag the IceCreamMt_alb_ChildC.ftxb texture to albedo texture 1.
