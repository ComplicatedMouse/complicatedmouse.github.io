Creating Bubbles

The Fluctuation setting and GPU Noise field are available as features that use sine waves for animations.
This page describes the procedure for creating various bubble effects using these features.

Specifying Emitter Settings

The Bubble () particles are transformed using scale and rotation.

Selecting an Emitter Node

Click to select the Bubble node () in the Bubble_Template emitter set () in the workspace.

Selecting the Particle Tab

Select the Particle tab () in the property window. The particle-related setting items appear.

Configuring the Particle Tab Settings

Configure the following item in the particle settings () to animate bubble particles.

Item names Value Description
Particle center(±) X,Y = 0,0.5 Shift the particle's rotation axis and change the movement path for bubbles that will rise vertically.

Set the following items in the Fluctuation setting ().

In Apply fluctuation (), enable Scale ().
After Apply fluctuation () appears, set the following items.

Setting Value Description
Amplitude 0.3 Configures the width of the contraction and expansion that results from the effects of water pressure.
Period 50 Configures a cycle of contraction and expansion animation that results from the effects of water pressure to 50 frames.

Select the Set the Y Scale Separately () check box ().
This selection enables the separate animation of the bubble in the Y scale, or vertical direction.
Set the Y Scale Separately () will appear. Configure the following items.

Setting Value Description
Amplitude 0.3 Use the same amplitude as for the X scale (horizontal direction).
Period 50 Use the same period as for the X scale (horizontal direction).
Initial Phase 20 Shift the phase by 20 frames from the Y scale (vertical direction) to animate the vertical and horizontal axes separately.
You will end with an animation that gently expands and contracts, maintaining the bubble's volume.

Add a GPU Noise Field to the Hierarchy

Add a GPU Noise field beneath the Bubble_Template () emitter in the hierarchy.

Adding Emitters to the Hierarchy From Presets

Using the following procedure, add a GPU Noise field beneath the Bubble object () directly beneath Bubble_Template () in the hierarchy.

1
In the workspace, right-click the Bubble object () beneath the Bubble emitter set () in the hierarchy.
2
From the menu that appears, select Add field () and then Add GPU noise ().
3.

GPU Noise () is added under the Bubble node (), and is in a selected state.

GPU Noise Settings

Set the GPU Noise field to use a sine wave to create a movement path for the bubbles.

Configuring the GPU Noise Tab

Make sure that GPU Noise is selected in the workspace.
Configure the following items () under Basic Settings () for GPU Noise.

Setting Value Description
Noise Strength X,Y,Z = 0.4,0.1,0.4 Adjust the amplitude value such that the bubble will rise while loosely wobbling in the horizontal direction.
Use a smaller amplitude to create a smaller rising bubble with a tighter wobble.
Base Period 300

Adjust a single period such that a bigger bubble will move somewhat slowly through water.

Use a smaller period to create a smaller bubble that rises faster.
