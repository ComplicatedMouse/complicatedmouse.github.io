Profiler-Copy

Overview

Tells the profiler to copy the contents of a specified tab. By default, the data is copied onto the clipboard. The tab contents are also returned as a result of this cmdlet.

Profiler-Copy [-Tab] <ScriptingTabs> [[-ExtendedParam] <ScriptingCopyExtensions>] [-NoClipboard]


[-Tab] <ScriptingTabs>

Specifies the tab to copy from. The following values are supported for this argument:

    • Assembly: Assembly Tab
    • CallTree: Call Tree Tab
    • CodeCoverage: Code Coverage Tab
    • Counters: Counters Tab
    • Files: Files Tab
    • Functions: Functions Tab
    • GraphicsGroups: Gfx Groups Tab
    • Info: Info Window
    • Instrumented: Instrumented Tab
    • Last: Last Tab
    • Logs: Logs Tab
    • Shaders: Shaders Tab
    • Threads: Threads Tab
    • Warnings: The Warnings window accessible from the Overview tab.


[[-ExtendedParam] <ScriptingCopyExtensions>]

Optional. The Call Tree and Code Coverage tabs have more than one way to copy data from them.

After each option below, compatible tabs will be listed.

    • All: Compatible with CallTree and CodeCoverage. Default value. Used to copy all data on the tab.
    • NotSeen: Compatible with CodeCoverage. Used to copy only the not seen functions list.
    • Seen: Compatible with CodeCoverage. Used to copy only the seen functions list.
    • Selected: Compatible with CallTree. Used to copy only the selected functions from the Call Tree.


[-NoClipboard]

Optional. Do not copy the contents of the specified Tab to the Windows clipboard. The contents will only be available as a return value.

Returns

A string containing the contents of the Tab, as if the copy button was clicked.

Side-effect. The clipboard will also contain the return value unless the -NoClipboard option was specified.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Copy Functions
Profiler-Copy CallTree Selected
Profiler-Copy CodeCoverage -ExtendedParam Seen


$contents = Profiler-Copy Functions -NoClipboard
