Variations

Modulation provides a means of changing the colors of the various parts of the Mii face model.

Modulation-Related Drawing Parameters

The nn::mii::DrawParam class stores drawing parameters for the various parts of the Mii face model, the face outline texture, and the mask texture.

You can get the following modulation-related parameters from nn::mii::DrawParam. Set these parameters to uniforms so the modulations are performed appropriately when the Mii face model is rendered.

Parameter Type Description
mode The type of the modulation. The types are enumerated by nn::mii::DrawParam::ModulateType.
const0 Constant color 1.
const1 Constant color 2.
const2 Constant color 3.
texture Texture

Types of Modulation

The Face Library uses the following types of modulation.

They are associated with the nn::mii::DrawParam::ModulateType enumerated type. For more information, see the API Reference for nn::mii::DrawParam::ModulateType.

Modulation Type Description
nn::mii::DrawParam::ModulateType_Constant Renders using a constant color (const0).
nn::mii::DrawParam::ModulateType_Texture Renders the unmodified texture.
nn::mii::DrawParam::ModulateType_RgbLayered Renders after multiplying the RGB components of the texture by the constant colors (const0 - const2).
nn::mii::DrawParam::ModulateType_Glass Renders after multiplying the R component of the texture by constant color const0. For the alpha, uses an adjusted alpha value obtained by adjusting the texture's G component using const1. For more information, see the API reference.
nn::mii::DrawParam::ModulateType_Alpha Renders using constant color const0 as the color and the alpha of the texture as the alpha.
nn::mii::DrawParam::ModulateType_AlphaOpa Renders after multiplying constant color const0 by the alpha of the texture. The alpha value is fixed to 1.0f.
nn::mii::DrawParam::ModulateType_IconBody No color is included in the constant color. Get and use your favorite color.
