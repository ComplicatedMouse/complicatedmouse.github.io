prepo Library

Introduction

If you are using play reports, read the Play Report Guide (日本語/English).

Purpose of This Documentation

This document describes the features provided by the Play Report (prepo) library included in NintendoSDK.

Info

The prepo library is available in NintendoSDK 10.1.0 and later.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Related Documentation

Along with this documentation, refer to the prepo library API reference.

prepo Library Features

Play Report provides a means by which information about events inside applications can be collected by a server and analyzed.

The prepo library provides features for storing and sending information about events in the application.

The collected data can be analyzed on a server. For more information, see the Play Report Guide (日本語/English).

You can check the data stored by the prepo library by using DevMenuCommand.

Restrictions

The following restrictions apply to the prepo library.

Using in the Windows Environment

The library is only implemented as a stub in the Windows environment. Data is not stored or sent.
