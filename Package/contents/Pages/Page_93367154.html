Emitter Scale

Scale

This section describes the Scale feature for applying scaling to the selected emitter shape.

Overview

This feature applies scaling to the emitter shape. It does not affect the particle speed.
The following table shows the properties to which scale is applied.

Emitter Size Particle Size Particle Speed
Yes No No

The following is an overview of the setting.

If any emitter shape other than Point has been selected, the Scale () settings are displayed. The group has the following settings.

Scale

Sets the scale factor by which the emitter shape will be multiplied.
If Emitter Time Animation is specified, the scale is fixed to the value used when the particle is emitted.

Example Configurations

The following image shows what happens when Scale is applied to a Circle (Evenly Divided) emitter shape.

A XYZ = 1 (Default)

Displays the Circle (Evenly Divided) emitter shape in its normal size with a diameter of 10.
B XYZ = 5 Displays the emitter shape at five times its normal size.
