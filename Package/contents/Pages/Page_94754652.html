Apply Fluctuation

This section describes the Flux Settings feature for fluctuating the alpha and scale of the particle.

Overview

The Flux Settings feature is a way to animate fluctuations of the alpha and scale of the particle with waveforms.
You can use waveforms to create an effect of periodic brightening and dimming to the animation of alpha and scale.
This section provides an overview of the settings ().

Apply Fluctuation

Use the following options to specify where to apply flux.

Alpha Fluctuates alpha values.
Scale Fluctuates scale values.
