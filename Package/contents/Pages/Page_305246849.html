Fragment Operations

This section describes the Fragment operations feature for setting the methods for alpha testing and integrating rendered pixels.

Overview

The following table shows the picture pane settings.

Current Settings Select the current settings from the presets.
Alpha Test

Sets the alpha test.

Using the default settings disables the alpha test and reduces the binary size.
Blend

Configures the settings for blending.

Use default settings and configure standard translucency to reduce the binary size.

In Fragment Operations (), you can specify one of the preset blending methods for blending with the background framebuffer.

Usage Examples

The following are examples using the detailed combiner mode for getting the expected results using the Blend preset Multiplicative.

Blended Image Blending Result Description

Pane x Background

The pane texture images are combined in the standard combiner mode.

The colors in the alpha part are multiplied by the background, so this does not produce the expected result.

Pane x Background

The alpha part of the pane is adjusted in detailed combiner mode to be white.
By using Multiplicative Blending in this state, it can be neatly combined with the background as seen in the Combined Results.
