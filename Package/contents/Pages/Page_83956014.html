How to Use

This plug-in converts monochrome images to distance field images (a format in which the shortest distance from a black/white boundary in the image is expressed in grayscale).
By using distance field images, you can enlarge textures while maintaining distinct edges.
To improve the quality of textures, apply this plug-in to high-resolution images and then use scaled-down versions of the images.

Select the RGB channel, gray channel, or alpha channel. Then select Filter > Nintendo > Nintendo Distance Field.

  • Max Distance
    Specify the maximum distance (in pixels) from a black/white boundary to which to apply the effect.
    The black regions in the monochrome image are converted to a value from 0 to 128, depending on their distance from a black/white border (0 is used at and beyond the maximum specified distance).
    The white regions in the monochrome image are converted to a value from 128 to 255, depending on their distance from a black/white border (255 is used at and beyond the maximum specified distance).
    The default slider control has a maximum value of 64, but you can specify a maximum distance of up to 250 pixels by entering the value directly.
  • Edge Wrap
    Select this option to process edge pixels to repeat the image.
    Disable this option if edge pixels of the image are being extended.
  • Normalize
    Select this option to normalize distance field values to a minimum of 0 and a maximum of 255.

Click OK to run the filter.

To zoom in or out on the preview image, click the + or - button or use the mouse wheel.

If the image is not already monochrome, all values of 128 or higher are changed to white and all values less than 128 are changed to black.
If you select the RGB channel, the R component is used as the input value, and the same values are output to the RGB components.

When the Edge Wrap option is disabled, a preview image is generated using a similar algorithm. The actual result of applying the filter may differ slightly from that shown in the preview.

Notes

These notes pertain to the use of the Distance Field Filter plug-in.

Image Mode

This option cannot be executed on files with an image mode other than RGB color, grayscale, or that do not use 8 bits per channel.
