Particle Time Curve Editor

The Particle Time Curve Editor provides a graphical way to set particle time animations, with time on the horizontal axis and values on the vertical axis.
This section will desctibe the particle time curve editor, using the scale animation UI as an example.

A Presets One-click generation of animation curves in shapes commonly used for effects.
B Toolbar A group of tools for operating the curve editor.
C Curve View

Displays the nodes that constitute an effect in tree format.

Right-clicking opens the workspace Edit menu.
D Help Mouseover the icon to show a list of shortcuts useful in animation editing.

Presets

The presets are buttons that you can click to generate animation curves that are often used for effects.
The following table describes the animation behavior for each button.

2 key Expands the size of the particle from a scale of 0 to the standard scale value, over the particle lifetime.
2 key Shrinks the size of the particle from the standard scale value to a scale of 0, over the particle lifetime.
3 key

Expands the size of the particle from a scale of 0 to the standard scale value, by 50% of the particle lifetime.

Then shrinks the size of the particle from the standard scale value back to a scale of 0, from 50% to 100% of the particle lifetime.
4 key

Expands the size of the particle from a scale of 0 to the standard scale value, by 25% of the particle lifetime.

It then shrinks the size of the particle from the standard scale value back to a scale of 0, from 75% to 100% of the particle lifetime.
3 key

Starts the animation from a state in which the particle is already equal to the standard scale value in size.

It then shrinks the size of the particle from the standard scale value back to a scale of 0, from 75% to 100% of the particle lifetime.
4 key

Shrinks the size of the particle from the standard scale value to half of the standard scale value, by the time 25% of the particle lifetime remains.

Then shrinks the size of the particle from half of the standard scale value to a scale of 0, from 75% to 100% of the particle lifetime.
