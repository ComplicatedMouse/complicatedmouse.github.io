Move History Using Waveform

This section describes the Move history using waveform feature that moves a stripe by introducing fluctuations to the historical samples.

Overview

The Move history using waveform feature adds a periodical movement to a stripe by using waveforms (sine waves) to move the historical samples.
By using waveforms to move the history, you can make the end of the stripe sway back and forth.
The following overview describes the settings ().

Enabling this feature displays the related groups. The group has the following settings.

Move History Using Waveform

These settings configure the behavior when particles are emitted.

Initial Rate of History Sets the initial velocity of the history in a random direction, changing the amplitude of the waveform.
History Change Period Sets the period of the waveform that changes in the direction of the specified axis.
Direction Change Ratio Sets the rate of change of the waveform.
History Acceleration(±) Sets the acceleration applied to the entire history.
History Air Resistance Specifies the overall air resistance exerted on the history.

The figure below describes which settings adjust which parts of the waveform.

A The amplitude of the waveform can be adjusted using Initial rate of history.

B The phase is determined randomly on the XYZ axes.

C The period can be adjusted using History change period and Direction change ratio.
