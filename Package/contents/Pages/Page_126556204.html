Characters overlap in outline fonts.

Question

Characters overlap each other in outline fonts.

Answer

This problem is somewhat unavoidable, especially for fonts with heavy borders.
Increase the font spacing to avoid the issue.
In addition, for the LA4Packed format, render the characters separate of the borders.
Rendering only the borders the first time and only the characters themselves the second time will create a different appearance and impression, so consider using this method as a countermeasure.
This feature is available in the layout module.
