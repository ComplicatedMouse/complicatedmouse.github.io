Scale

This tab is for configuring settings for particle scale.

Basic Scale

Basic Scale () is a group of features for configuring the scale of the emitted particles.
This group has the following settings.

Standard value

Feature that sets the scale for particle emission.

Randomness (%)

Feature that changes the scale randomly when particles are emitted.

Scale (Particle Time Animation)

Scale (Particle Time Animation) () is a group of features for configuring particle scale animations using Curve Editor.
This group has the following settings.

Scale Animation

Feature that sets the animation for particle scale using Curve Editor.

Other Scale Settings

Other Scale Settings () is a group of features for setting the colors to use for the Color Combiner calculation formulas.
This group has the following settings.

Link the Angle of View With the Scaling Restrictions This is a feature that suppresses the effects of changing the camera angle-of-view on the particle display size when using the Camera Distance Scale Restriction feature.
Camera Distance Scale Restriction (Near) This feature limits scaling so particles in front of the camera are no larger than necessary.
Camera Distance Scale Restriction (Far) This feature keeps particles from becoming smaller than necessary as they move away from the camera.
Deform Vertex Based on Speed Feature that applies a scale in the particle's y-axis direction in proportion to the particle's movement speed.
