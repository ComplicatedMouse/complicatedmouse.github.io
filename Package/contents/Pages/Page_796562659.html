Layout and Font News

LayoutEditor

This section describes LayoutEditor features.

Added the Paste Without Animation menu.

Added Paste Without Animation and Paste With Animation for choosing whether animations are included when pasting a copied pane into a layout.
Similarly, when duplicating the selected pane, Duplicate Without Animation and Duplicate With Animation have been added to determine whether animations are included.

The displayed menu varies as follows, depending on whether the Copy/Paste Animations menu is On or Off.

Copy/Paste Animation - On Copy/Paste Animation - Off

The menu changes to allow pasting and duplication with animations excluded.
When pasting or duplicating, animation is included.

The menu changes to allow pasting and duplication with animations included.
When pasting or duplicating, animation is not included.

When copying or duplicating frames without animation, the values for the current frame are copied or duplicated for settings with animation.
By adjusting the current frame in the Curve Editor window, you can copy or duplicate it in the desired state.

Added Perform gamma correction of the effect if working in linear color space to Capture Effects.
When enabled in the linear workflow, this option applies degamma correction so that the midtones of the gradient created using Capture Effect are displayed correctly.
The default settings for this function can be changed in the project configuration file.
