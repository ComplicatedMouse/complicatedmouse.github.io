Selecting the color format.

Question

How do I select the color format?

Answer

  • Select A8 for standard fonts.
  • For outline fonts (fonts that have alpha information separate from brightness), specify LA8 and 2-channel format.
    • Outline fonts where the outline can be turned on and off use the LA4Packed format.
      • If compression noise is a concern, use the LA4PackedNoCompress format. However, this format doubles the size.
  • The options that begin with RGB are for image fonts.
    • Use the SRGB format if you have adopted a linear work flow (check with the lead developer).
  • With A4 and A8 or LA4 and LA8, the result is the same whether you choose the format ending in 4 or the format ending in 8. (They are interpreted as BC4 format internally.)
    • The formats remain so that data used on past platforms can be converted using the same settings.
