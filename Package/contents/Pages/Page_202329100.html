Panes

This page describes the additional Panes menu for LayoutEditor.
Create Pane Creates a pane.
Picture panes are created from the Texture window.
Hierarchy Sets the parent/child relationship between panes.
Remove Hierarchy Removes the set parent/child relationship.
Reflect Parent Parallel Translations to Child When selected, parallel translation of the child pane is performed in conjunction with parallel translation of the parent pane.
Adjust the size by dragging the adjustment point When selected, you can drag the adjustment point to adjust the size.
When not selected, dragging the adjustment point adjusts the scale.
Create Group Creates a group with the selected panes.
Edit Group Edits the group that includes the selected panes.
Lock Selected Panes Locks the selected panes.
Unlock All Unlocks all locked panes.
Hide Locked Pane Hides locked panes so they are no longer displayed in the Layout window or the viewer.
Lock Boundary Panes Locks the boundary panes.
Hide Selected Panes Hides the selected panes so they are no longer displayed in the Layout window or the viewer.
Show All Panes Shows all hidden panes.
Lock Hidden Panes Locks hidden panes.
Sort

Sorts panes after you select multiple panes and the sorting method from the Sort List.
Resize to the Texture Size Resizes the picture pane to be the same size as the texture.
Create a capture pane with the select pane as a child Creates a capture pane with the selected pane as a child.
Properties Opens the property window of the selected pane.
Reload Parts Pane Reloads the parts pane in the Layout window.
Convert Objects Under Selected Null Pane to Parts Converts objects under the selected null pane to parts.
Open Parts Layout of Selected Pane Opens the parts layout of the selected pane.
Create Hierarchy Creates a hierarchy from multiple selected panes.
Removing Hierarchy Removes a hierarchy from panes.
Create a null pane as a child of the selection pane. Creates a null pane as a child of the selected pane.

Sort List

To reorder the panes, select multiple panes and select Sort.

Sort List Commands
Align Left

Aligns the selected panes to the left edge of the pane whose left edge serves as the standard.

Of the selected panes, the uppermost pane in the Hierarchy window is the standard pane.
Center Horizontally

Aligns the selected panes to the horizontal center of the pane whose horizontal center serves as the standard.

Of the selected panes, the uppermost pane in the Hierarchy window is the standard pane.
Align Right

Aligns the selected panes to the right edge of the pane whose right edge serves as the standard.

Of the selected panes, the uppermost pane in the Hierarchy window is the standard pane.
Align Top

Aligns the selected panes to the top edge of the pane whose top edge serves as the standard.

Of the selected panes, the uppermost pane in the Hierarchy window is the standard pane.
Center Vertically

Aligns the selected panes to the vertical center of the pane whose vertical center serves as the standard.

Of the selected panes, the uppermost pane in the Hierarchy window is the standard pane.
Align Bottom

Aligns the selected panes to the bottom edge of the pane whose bottom edge serves as the standard.

Of the selected panes, the uppermost pane in the Hierarchy window is the standard pane.
Distribute Horizontally

Within the range of the selected multiple panes, panes are placed so that the distance between the centers of the panes is evenly distributed horizontally.

Info

This makes the distance between the centers of panes equal, so when there are panes of different sizes, the distances from the pane edges may not be equal.
In the future, we plan to add a mode to make the distances from the edges of panes equal.
Distribute Vertically

Within the range of the selected multiple panes, panes are placed so that the distance between the centers of panes is evenly distributed vertically.

Info

This makes the distance between the centers of panes equal, so when there are panes of different sizes, the distances from the pane edges may not be equal.

In the future, we plan to add a mode to make the distances from the edges of panes equal.
