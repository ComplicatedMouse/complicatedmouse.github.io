Search for Panes Window

This section describes the names and uses of the various parts of the Search for Panes window.

You can use this window to search for various panes and rename panes.

Search for Panes Window
A

Search Target

Selects the search target from a list.

  • Pane Name
  • Comments
  • User Information
B Operations

Displays the following menu when clicked.

  • Find Next
  • Find Previous
  • Replace
  • Add to Selection
  • Select All
  • Extended user information name
  • Extended user information value
C Find What Searches for the string that was entered in the text box.
D Replacement text Replaces the string entered in Find What with the string entered in the text box.
E Options

Select each feature to enable it.

  • Regular Expressions: Allows regular expressions to be used as the Find What string.
  • Automatic Select: Automatically selects panes that were searched.
  • Replace: Enables the Replacement text and Replace buttons.
F Search results Displays the pane name that was searched or the pane name after replacement.
G Find Next Selects the next pane that matches the search criteria.
H Replace Click this button to replace.
I Replace All Replaces all panes matching the search conditions in a batch.
