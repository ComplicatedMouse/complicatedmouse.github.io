Create and Set Segment Tags For Feature Animation Window

The following operations can be performed in the Create and Set Segment Tags For Feature Animation window.

A Name of an unconfigured feature animation
 Select the check box to add this feature animation to the Edit Animation Segment Tags window.
B Animation segment This displays the start frame and the end frame of the animation segment.
C Animation segment settings and length Sets the start frame and the end frame of the animation segment. The result is shown as Length:
D Description If a description has been set for the animation segment, it is displayed here.
E Description entry field Use this field to enter a description for the animation segment.
