Amplitude

This section describes the Amplitude feature for setting the magnitude of the flux.

Overview

These waveform settings can be configured when sine wave is selected for the waveform type.
This section provides an overview of the settings ().
When you select either setting in Apply Flux, the Apply Flux group () of settings appears. The group has the following settings.

Amplitude

Sets the size of the flux. Fluctuates between the minimum and maximum values, obtained with the following expressions.
Minimum value: The value of the parameter to which flux was applied minus the amplitude value.
Maximum value: The value of the parameter to which flux was applied.
