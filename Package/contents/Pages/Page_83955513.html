Bone Visibility Animation

This page describes the settings for bone visibility animations.

Types of Bone Visibility Animation Settings

Bone visibility animations can be set by animating the Visibility attribute of the transform node or the mesh node.

When you animate the visibility attribute of a transform node

The on/off state of the visibility affects all lower nodes in the hierarchy.
If the visibility of the lower nodes in the hierarchy is fixed in the ON state, the same animations used for nodes higher in the hierarchy are played.
If the visibility of the lower nodes in the hierarchy is also animated, that animation is combined with the animation used for the higher nodes in the hierarchy, and the resulting animation is played.

Animating the Visibility Attribute of a Mesh Node

The on/off state of the visibility affects only the polygons in that same hierarchy layer.

To enable key setting in the visibility attribute of a mesh node, select the mesh node, open Channel Control, and set Visibility to Keyable.

Important Information About Bone Visibility Animations

For bone visibility animations, the animation curve data is always automatically created and exported from baked data.

Info

Because bone visibility animation is set on a node basis, when the Compress Bone export option has been set to Cull, Merge, or Unite, make sure that nodes for which bone visibility animation is set are not compressed. (You can set the compress disable flag using plug-in for disabling node compression.)

Do not set Compress Bone to Unite All when using bone visibility animation.

Make sure that nodes for which bone visibility animation is set are not gathered together when Unite Child is selected.

By default, bone visibility animation for all nodes is exported to a binary file. However, in the config file export plug-in, you can configure the settings so that only nodes with animation set to visibility are exported.

In addition, you can use the plug-in for setting animation binary output to set the binary output flag for visibility animations (functionality available in NintendoSDK 10.0.0 and later).
