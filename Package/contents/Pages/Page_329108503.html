Enabling Drop Shadows

This section describes the Enable Drop Shadows feature that drops a shadow behind a pane.

Overview

The Enable Drop Shadows feature adds a shadow behind a pane.
Enable this feature when you want it to look like a pane is floating and three-dimensional.

Rendering Mode

The following images compare what happens when the Rendering Mode is changed.
This feature configures the method of blending the color in the area of shadow dropped from the pane and background color.

Normal Multiplicative (Default) Additive Subtractive

Blend translucently. Blend multiplicatively. Blend additively. Blend subtractively.

Angle

The following image compares what happens when the Angle value is changed for the drop shadow effect.

Angle = 0 is to the left of the pane, with the shadow rotating counterclockwise around the pane as the value increases.

Angle = 0 Angle = 120 (Default)

Distance

The following image compares what happens when the Distance value is changed for the drop shadow effect.

The higher the value, the greater the distance the shadow is dropped.

Distance = 5 (Default) Distance = 50

Spread

The following image compares what occurs when the Spread value is changed for the drop shadow effect.

The higher the value, the darker the shadow is displayed.

Spread = 0 (Default) Spread = 100

Size

The following image compares what happens when the Size value is changed for the drop shadow effect.

You can expand the area covered by the drop shadow by increasing this value.

Size = 5 (Default) Size = 100
