Maya Plug-In

The Import plug-in can now suppress the creation of material nodes.

A Create Material Select this option to create material nodes.
(blinn / shadingEngine / materialInfo)
If the option is cleared, the default material (lambert1) is assigned to all shapes.
B Create Texture Node Select this option to create texture nodes.
(file / place2dTexture / uvChooser / layeredTexture / bump2d)
If the option is cleared, only the material color, disable-compression flag, and user data are applied.
If Create Material is cleared, this option and the Create Texture File option are disabled.
Import Plug-In Describes the Import Plug-In features and how to use them.

The Import plug-in now has tabs for selecting the processing mode.

There are now tabs for switching between normal Import and Batch Convert.
