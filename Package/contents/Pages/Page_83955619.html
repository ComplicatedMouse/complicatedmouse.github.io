Random

This section describes the feature for eliminating any orderliness to the movements of particles.

Overview

Random randomizes the direction of movement of particles.
You can create behavior without any order by using a random seed.

Random

Random () is the set of features for randomizing the direction of movement of particles.

Random Width

Feature to set the value to add to the particle's speed.

Interval to apply randomness

Feature to set the interval over which to apply randomness.
