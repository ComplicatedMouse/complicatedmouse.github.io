Inheritance Settings: Alpha

This section describes the Alpha inheritance setting for child emitters.

Overview

The Alpha inheritance setting is a feature that makes child emitters inherit the alpha value of the parent emitter in the highest node.
The following overview describes the settings ().

Alpha

The feature to inherit alpha from the parent emitter is enabled or disabled using the check box ().
When this feature is enabled, particles multiply their own alpha value with the parent emitter’s alpha value when emitted.
You can separately specify Alpha 0 and Alpha 1 as the alpha values to inherit.

Enabling this setting displays the Per-frame inheritance setting. The following is a description of each item.

Per-Frame Inheritance

Select or clear the check box () to enable or disable this feature. When enabled, particles always multiply their own alpha value with the parent emitter’s alpha value.
