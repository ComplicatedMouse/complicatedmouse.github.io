NintendoSDK Overview

NintendoSDK

NintendoSDK is a development environment for Nintendo platforms. NintendoSDK provides both cross-platform features and the ability to add platform-specific features. Using a development environment for multiple platforms allows you to reuse much of your code and knowledge, making it easier to develop for other Nintendo platforms.

NintendoSDK provides the following cross-platform features.

  • Cross-platform APIs
  • Cross-platform tools
  • Cross-platform compiler tools and debugging software
  • A compatible environment for these features in Windows

The following features are platform-specific, and must be incorporated via add-on packages.

  • Features important to performance, such as those for low-level graphics layers
  • Features for using platform-specific hardware
  • Features for past platforms that would be difficult to provide compatibility for
  • A compatible environment for these features in Windows


Info
  • For more information about the major differences between NintendoSDK and previous development environments, see Differences From Previous Development Environments.
  • For a list of the specific features provided by NintendoSDK, see Features.


Feedback

We are seeking feedback from developers to make NintendoSDK an even better development environment. If we get feedback at the early stages of development, the feedback we receive can be incorporated at less cost.

Please send feedback to your contact for the NintendoSDK.

NintendoSDK NX Add-On

The NintendoSDK development environment for the NX platform. The NintendoSDK NX Add-On is necessary to develop for the NX platform.

Info

With the current version, all of the packages are installed if the NX Add-On is installed from a MegaZarf file.

Info
  • For information about the NX Add-On operating environment, see Operating Environment.
  • For information about the set of packages provided as the NX Add-On, see Package Structure.
  • For information about the features that have been added in the NX Add-On, see Features Added by NX Add-Ons.
  • For information about setting up the environment to use the NX Add-On, see Guide to Setting Up the Environment.
  • For information about developing applications using the NX Add-On, see Tutorial.
