Playback Method - Random

This section describes the Random playback method for texture pattern animations, where the pattern to show for each particle is randomly selected when particles are emitted.

Overview

The Random playback method can be used to randomly display individual patterns for each particle when particles are emitted.
The following overview describes the settings ().

Select to display the Random () and Pattern Tables () settings. The group has the following settings.

Number of Patterns

Use the dial () or the text box () to specify the number of patterns to display.
