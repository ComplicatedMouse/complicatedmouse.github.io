Range Slider

This section describes the features displayed in the Range Slider in the Curve Editor window.

The Range Slider sets the animation playback range.

Range Slider
A Start frame The animation starting frame.
B End frame The animation end frame.
C Playback Start Frame The frame at which the animation playback range begins playback.
D Last Playback Frame The frame at which the animation playback range stops playback.
E Animation Playback Range

This range will be played back repeated during animation playback.

(The frame range specified by C and D.)

This range is displayed in the Time Slider.
