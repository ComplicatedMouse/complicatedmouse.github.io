Label Color

This page describes the feature for setting colors with which to mark preview files when they are displayed in the Effect Browser.

Overview

The Label Color feature allows you to set a color for the preview file selected in the workspace.
The configured color marks the file when it is listed in the Effect Browser.
The following overview describes the settings ().

Label Color

For the preview label color, choose from among eight preset colors: Colors 0 to 7.
