Lifespan Randomness (%)

This section describes the feature to randomize the lifespan of individual particles.

Overview

The Lifespan Randomness (%) feature randomizes the lifespans of emitted particles within the range set for Lifespan.
You can use this feature to create more natural representations of effects like snowfall, where it would look unnatural if all particles died in unison.
The following overview describes the settings ().

Lifespan Randomness (%)

Randomly changes the particle lifespan.
The value set for Lifespan is used as the upper limit of the random value. The lower limit changes randomly. The following formula calculates a lower limit for the random number.

Example Configurations

The following image compares what happens when the Lifespan Randomness (%) value is changed (when the particle lifespan is set to 100).

A 0 The lifespans of all emitted particles are fixed to 100.
B 50 The lower limit of the random value for lifespan is half of the initial 100 frame lifespan, which in this case is 50 frames.
As a result, the particle lifespan varies randomly between 50 frames and 100 frames.
C 100

The lower limit of the random value for lifespan is 1 frame of the initial 100 frame lifespan.

As a result, the particle lifespan varies randomly between 1 frame and 100 frames.
