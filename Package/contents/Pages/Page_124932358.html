Image

The section describes the UI for the Image tab.
The Image tab can use images as the font file during input.

A Image Files Specifies an image file conversion source.
B Character Order Selects the character order used in the image file.
C Color Format

Specifies how to analyze the color for each pixel in the image.

Format names with ～_SRGB attached are texture formats that support linear workflow.
D Fix linear interpolation errors

When selected, corrects bleeding into surrounding colors caused by linear interpolation of glyphs.

This process is effective only for color formats that have alpha channels.
