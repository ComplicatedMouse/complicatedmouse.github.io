What are the rules for locking the region outside of key frame segments?

Question

How does locking work for the region outside of key frame segments?

Answer

You can lock outside of key frame segments by using LayoutEditor or by using the binary converter.
If you use the binary converter, specify the --bake-infinity option as a converter argument. This option is enabled by default.

If the end frame of the animation segment tag is the same as the end frame of the loop (with automatically generated keys), the value of the end frame will differ depending on which process is used for locking.
For this reason, we recommend that you consistently use the binary converter.

Tool processing Value of the start frame of the loop.
Converter processing Value of the end frame of the loop.
