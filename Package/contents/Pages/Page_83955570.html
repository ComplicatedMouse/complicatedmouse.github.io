Nintendo SDK 0.2.0

The following known bugs are confirmed to occur as of Nintendo SDK 0.2.0.

  • Cannot link to FS library functions.

Cannot link to FS library functions.

You cannot link to the following FS library functions because implementations are not included in the library.

  • nn::fs::ReadFile(FileHandle handle, int64_t offset, void* buffer, size_t size)
  • nn::fs::ReadFile(size_t* outValue, FileHandle handle, int64_t offset, void* buffer, size_t size)
  • nn::fs::ReadFile(size_t* outValue, FileHandle handle, int64_t offset, void* buffer, size_t size, const ReadOption& option)
