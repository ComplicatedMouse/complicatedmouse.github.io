Maya Plug-In

Added support for texture SRT animation that has been in a compound attribute of the place2dTexture node.

Added support to the Maya plug-in for connecting another compound attribute to the (repeatUV, translateFrame) compound attribute of the place2dTexture node and setting texture SRT animation for that.

The following figure shows an example of these settings.
(In this example, Translate X and Y of the locator are connected to Translate U and V of the place2dTexture node, and from there connected to the frame translations of a different place2dTexture node.)

In this example, the texture SRT is animated by moving the node in Maya.

The SRT animation is output as a result.
You can check it using SampleViewer.
