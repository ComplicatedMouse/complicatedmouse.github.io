Noise Table Interpolation

This page describes Noise Table Interpolation, a feature for reducing fluctuations when expanding the noise table.

Overview

Noise Table Interpolation enables you to reduce fluctuations in particle movement.
Particle movement fluctuation occurs when expanding the noise table because of lower densities at the table level.
Enable Noise Table Interpolation to reduce particle movement fluctuations and make movement smoother.
Note that the CPU load will increase when Noise Table Interpolation is enabled.
The following overview describes the settings ().

Info
Noise Table Interpolation is only available as an option when the emitter type is CPU Emitter.
You can select whether to interpolate every time there is a noise table lookup.
Interpolation occurs by default for GPU+SO emitters.
