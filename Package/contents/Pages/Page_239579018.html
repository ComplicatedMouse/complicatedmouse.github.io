Properties Window For a Circle Shape Picture Pane

This section describes the properties window when the picture pane shape is a circle.

Circle Shape Picture Pane

Shape Settings

The settings are used to configure the number of divisions in the picture pane circle.

When Circle is set for the picture pane, the pane takes the shape of a circle.
This setting specifies the number of divisions in that circle.

Number of Divisions:

Enter the number of divisions of the circle shape for the picture pane.

More divisions makes for a smoother circle, but also increases the processing load.
