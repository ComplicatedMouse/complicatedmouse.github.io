SLConverter

Introduction

Objective

SLConverter is a tool that converts the DirectX HLSL shader language into the GLSL shader language used by the NX.

The purpose of this tool is to enable the use of shaders, developed in HLSL, for the NX.

Operating Environment

Conforms to the NintendoSDK operating environment.

Intended Users

SLConverter is intended primarily for the following types of developers.

  • Developers who want to migrate existing shader files, written in HLSL, to the NX.

Advantages of the Tool

Using SLConverter has the following advantages.

  • Supports the DirectX effect file format. Can perform shader conversion for each internal technique.
  • Can accelerate shader conversion by using multiple threads.
  • Can optionally specify the position of VertexAttribute with the configuration file.
  • Can handle and convert matrix processing, and the like, that are different in HLSL and GLSL.
  • Can use some of the features of Shader Model 5.0. For more information, see Support for Shader Model 5.0.
