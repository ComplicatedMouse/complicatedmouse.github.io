Graphics/Tools/DirectXShaderCompiler

Overview

DirectXShaderCompiler (Tools/Graphics/GraphicsTools/dxc.exe) is the version forked from

microsoft/DirectXShaderCompiler with some minor build changes for the Nintendo environment. We may eventually also add new features or bug fixes to this branch.

DirectXShaderCompiler (dxc) is available from NintendoSDK 12.1.0.

Nintendo Specific Features

The Nintendo version of DXC and dxcompiler.dll have some bug fixes and improvements over the standard release, including:

Starting in SDK 13.2.0, additional SPIR-V debug information is provided when the -fspv-debug flag is provided on the command line. Specifically, line number information is emitted for the hidden wrapper function emitted by the compiler. Also, in the event that no source file is available (the original source was directly provided as a string to dxcompiler.dll) then the original source string is used for debugging.

Usage

dxc.exe is intended to be used to convert HLSL source code to SPIR-V, which may then be consumed by other tools. Typical usage would look like:

    %NINTENDO_SDK_ROOT%/Tools/Graphics/GraphicsTools/dxc.exe -spirv -T ps_6_0 pixelShader.hlsl -Fo pixelShader.spv
    %NINTENDO_SDK_ROOT%/Tools/Graphics/GraphicsTools/dxc.exe -spirv -T vs_6_0 vertexShader.hlsl -Fo vertexShader.spv

The "-spirv" option turns on conversion to SPIR V (the default for dxc is bytecode, so this option must be provided). The shader model to use is specified with the "-T" option, and the final output file is given by the "-Fo" option.

Running dxc.exe with the "-help"option will give a summary of options available.

More detailed documentation is available online from Microsoft's wiki at https://github.com/microsoft/DirectXShaderCompiler/wiki.


Files

Included with the DirectX shader compiler are the following files:

  • Tools/Graphics/GraphicsTools/dxc.exe: command line tool for converting files
  • Tools/Graphics/GraphicsTools/dxcompiler.dll: shared library containing the conversion functions
  • Tools/Graphics/GraphicsTools/dxcompiler.lib: link library for dxcompiler.dll
  • Include/GraphicsTools/DirectXShaderCompiler/dxcapi.h: definitions for using dxcompiler.dll
