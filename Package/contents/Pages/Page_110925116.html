Even though the file is a color image, only intensity (black and white) is selected for the texture format.

Question

A TGA file registered as a color image in the texture window is being handled as intensity (black and white) format.

Answer

LayoutEditor randomly selects pixels from the image file to determine whether the file is a grayscale image and which texture formats to offer as selections.
If the values for R = G = B for all pixels surveyed, LayoutEditor assumes it is a grayscale image.

Set Display Method to Detail View in the texture window, and change the format selection in the Format column of the table to color format.
