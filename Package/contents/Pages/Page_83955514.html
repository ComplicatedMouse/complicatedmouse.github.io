Skeletal Animation

Skeletal animation is set by animating Translate, Rotate, and Scale of the transform and joint nodes.

Animation curve data is always automatically created and exported from baked data in the following cases.

  • For animations that use IK, constrain, and dynamics.
  • Path animation
  • The Rotate Pivot attribute and Scale Pivot attribute differ.
  • The Rotate Order attribute is not xyz.
  • The Rotate Axis attribute is not (0, 0, 0).
  • The Joint Orient attribute is not (0, 0, 0).
  • Rotation interpolation is performed by using quaternions.

For the procedure for creating a model that supports the skeletal animation motion mirroring feature, see Motion Mirroring.

If you use the plug-in for setting animation binary output, you can disable playback of animations for specific nodes. You can also set individual binary output flags for translation, rotation, and scaling.
