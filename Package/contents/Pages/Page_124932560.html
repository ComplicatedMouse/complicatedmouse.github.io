Color Format

This section describes the image color formats able to be used with bitmap fonts.
The following list shows the color formats that can be specified during font conversion.

A4 , A8 1-channel
LA4 , LA8 2-channel, brightness, and alpha shapes are specified separately
LA4Packed Stores brightness and alpha in 8 bits in a special format Stored in the BC4 compression format
LA4PackedNoCompress Stores brightness and alpha in 8 bits in a special format Stored in noncompressed A8 format. Twice the size of LA4Packed
RGB565, RGB8 Compressed color, 1-bit alpha
RGB5A1, RGB5A3 Compressed color with alpha
RGBA8 Uncompressed color with alpha
RGBA8_SRGB RGBA8 with linear gamma correction
RGB565_SRGB RGB565 with linear gamma correction
RGB5A3_SRGB RGB5A3 with linear gamma correction

The following are the display results for the various color formats loaded as text panes in the LayoutEditor.

A Grayscale Format The image will be displayed with the black and white sections inverted.
If the font is created using black characters on a white background, the White color specified in the Color Interpolation Settings in LayoutEditor will be used for the color of the test.
B Color Format

The image will be displayed with the black and white sections as is.
The text color will normally be specified by the input image.
