Custom Fields

This section describes the Custom Fields feature for configuring fields that have been independently prepared for a game application.
The data can be used for game application features and workflow purposes.

Bits

The Bits group () is used to set custom fields using flags.
This group has the following settings.

Bits 0-15 A group of 16 flags that can be individually selected using the check boxes ().

Numbers

The Numbers group () is used to set custom fields using sliders.
This group has the following settings.

Number 0 & Number 15 Sets values using a 256-step dial () with values in the range from 0 to 255.
Info

Custom fields are processed by the GPU (by the computer shader). The emitter type switches to the GPU + SO emitter () when you create these nodes.
Related Links For more information about the GPU + SO emitter () , see here.

Example Configurations

Described next is an example of configuring a custom field for implementing wind and shock wave fields in a game application.
The wind custom setting applies a force in a fixed direction in the game scene and affects the moving speed of particles.
The blast wave field is a custom field that applies a radial force emanating from the center of an explosion, affecting the moving speed of particles.

Example UI for the Tree Leaves Effect Image Used in Game
A

Apply Wind Effect selected.

As shown in the image, the particles depicting tree leaves are influenced by wind and move in the direction of the arrow.
B Apply Blast Wave Effect cleared. As shown in the image, the particles depicting tree leaves move in the direction of the arrow and are not influenced by blast wind.
C
Wind Force Is 0.8 The speed of the moving tree-leaf particles is augmented by 0.8 of the wind speed.
