Specifying Output Image Colors

Specify colors for the various areas in the output image file.
The following items can be configured.

A null The color of the cell that is specified as NULL in the character order file.
B Grid The color of the grid to draw when Draw Grid is selected.
C Background The background color.
D Width Line The width line color.
E Margins The color of the one-pixel region between the cell and the grid.
