Emission

This tab is for configuring settings for particle emission.

Emission Settings

Emission Settings group () is the set of features for configuring settings related to how particles are emitted.
This group has the following settings.

One Time

Feature for determining the particle emission time.
Emission Start Frame Feature for setting the particle emission timing.
Emission Interval Type Feature for setting the particle emission interval.

Settings When Using Child Emitters

The following settings are for child emitters.
Related Links For more information about child emitters, see Child Emitters.

Emission Timing (%) Sets the timing of child particle emission.

Initial Velocity Settings

Initial Velocity Settings () is a group of features for configuring settings related to the initial velocity of emitted particles.
This group has the following settings.

All-Direction Initial Velocity Feature for setting the initial velocity in the emitter normal direction.
Specified-Direction Initial Velocity Feature for setting the initial velocity in the specified direction.
Y-Axis Initial Diffusion Velocity

Feature for setting the diffusion velocity from the y-axis in the emitter coordinate system.
Initial Diffusion Velocity Feature for setting the initial diffusion velocity in the specified axis.
Random Initial Velocity Feature to add a random value to the initial velocity, providing some texture.
Info

The results from adding the calculation results for each item, in order from the top, are the initial velocities at emission.

Other Motion Settings

Other Movement Settings () is a group of features for configuring settings related to the speed of particles after they are emitted.
This group has the following settings.

Air Resistance Increases or decreases the velocity of emitted particles.
Gravity Applies acceleration in a specified direction on emitted particles.
Inherited emitter velocity ratio A feature where articles inherit the emitter's velocity as their initial velocity when emitted.
Momentum Random Randomizes the momentum of individual particles.
