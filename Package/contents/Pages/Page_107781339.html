Curve Editor Window

The Curve Editor window provides a graphical way to create and edit animations, with time on the horizontal axis and values on the vertical axis.

A Toolbar Displays features as icons.
B Attribute Tree Displays the elements being animated in the pane.
C

Graph Display Area

Displays key frames and curves.

The horizontal axis represents time, and the vertical axis represents the values of attributes.

When drag operations are performed in Curve Editor, the layout window or property window display is updated accordingly.
D Time Slider

Displays the current frame, the playback range, and the key frames.

Click or drag to move the current frame.
E Playback Controls Controls animation playback.
F

Range Slider

Sets the range of the animation to play.
