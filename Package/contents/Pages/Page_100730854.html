Random Color Settings UI

Multiple colors can be added using the Random Color Settings UI.
The color for particles is selected at random from among the added colors when emitting.

A Behavior Type

Select Random to display this UI to add multiple colors.
B Color on/off
To enable or disable different random color elements, select or clear the check boxes ().
When a check box is selected, that color can be applied.
C Register colors To set colors, select these check boxes.

Related Links For more information, see the following pages.

Color Settings Describes the functionality for setting the Color 0 and Color 1 colors, which are used in the Color Combiner calculation formula.
Changing the color behavior type to random enables this UI for use.
