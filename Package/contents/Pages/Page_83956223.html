Cautions When Using Curve Editor

  1. If you use Curve Editor to create a horizontal linear curve, it is converted to a constant curve when data is saved. (When the saved animation is reloaded, there is just one key.)
  2. The curves of skeletal animations are also displayed in the Curve Editor, but 3DEditor does not support editing these skeletal animations.
    Note: Skeletal animations cannot be edited but are displayed in color in Curve Editor for readability.
  3. Although the curve editor displays the base shape for shape animations, 3DEditor does not currently support the editing of shape animations.

Uneditable Animations

Under some conditions, an animation may not be editable in CurveEditor.
In such cases, the reason is shown in the Curve View.

Displaying Animation Curves in Unique Colors

When using Curve Editor, a color is assigned to display each curve.
Note: Color animation curves are displayed using red, green and blue.

The color of the square displayed to the left of node names shown in the Curve Tree View is the same as the color used for that curve.

Valid Ranges When Editing a Curve

The area outside the valid range of values is colored purple when editing the curves for animations like color animations, texture pattern animations, and visibility animations.
The area outside the valid range for frames is shaded.

A The area outside the valid range of values for a curve being edited is colored purple.
B The area outside the valid range of frames for a curve being edited is shaded.
Info

You can set the valid ranges of a curve being edited. The ranges are based on the UI minimum value (min) and UI maximum value (max) defined by the shader.

Selecting Curves and Keys

The curve view can be used to select multiple curves, keys, and slopes.

Status View Display Selection Method
Unselected
Selects the area that contains nothing in the curve display region.
Curve selected
To select a range, click the curve.
Key selected
To select a key in the curve, click it, or select a range by using the mouse button.
If one or more keys are selected, the line of the curve being affected is highlighted and slope handles are displayed.
Slope selected
Either click the leading edge of the slope handle while a key is selected, or drag to select a range.
You can link or unlink slopes by using the associated icon on the toolbar.

Add Key

This section describes the procedure for adding a key to any frame of an existing curve.

1. Select the curve to which the key is to be added.
2. Rest the mouse pointer on the red line displayed in the Curve View for the frame you want to add a key, and press the S key.
3. The key is added to the selected location.
4. Select the added key, and enter numeric values for frame and value on the toolbar.

Related Links Keys can also be added using mouse operations in the Curve View.

Copying and Pasting

Curves and keys that are selected in the Curve View can be copied and pasted by using CTRL+C and CTRL+V.

There are two types of copying.

  • Copying curves
  • Copying keys

You can paste to the selected curve.

Related Links There are three modes you can use to paste curves and keys: Insert, Replace, and Merge. The paste mode can be selected in the Curve View.

Clamping Frames and Values

You can clamp frames and values when editing a curve.

Related Links You can enable and disable clamping by using the icons for clamping frames and clamping values on the Toolbar (Curve Editor).

If frame clamping is enabled, a green line is displayed to the left and right of the range being clamped.

If value clamping is enabled, a green line is displayed at the top and bottom of the range being clamped.

Specifying Pre-wrap and Post-wrap for a Curve

You can specify Pre-wrap or Post-wrap from a curve element in Curve Editor.

Pre-wrap clamp Returns the value of the first key. Pre-wrap specifies the operation for the frame prior to the first key.
repeat Returns a value that is used repeatedly on the curve.
mirror Returns the value used to repeatedly rotate the curve.
relative_repeat Returns the value used to repeat the curve relative to the value at the end of the curve.
Post-wrap clamp Returns the value of the last key. Post-wrap specifies the operation for frames after the last key.
repeat Returns a value that is used repeatedly on the curve.
mirror Returns the value used to repeatedly rotate the curve.
relative_repeat Returns the value used to repeat the curve relative to the value at the end of the curve.
