SoundMaker

None.

Runtime Libraries

Fixed a bug where the pan mode and pan curve settings were not applied to stream sounds.

Fixed a bug where pan mode and pan curve settings for stream sounds were not applied.
Previously, during playback of stream sounds, the pan mode and pan curve settings remained set to Dual and Sqrt(-3dB), regardless of what parameters had been configured in SoundMaker. Now, the pan mode and pan curve settings configured in SoundMaker are applied.
Note that because the default pan mode and pan curve settings for stream sounds created in SoundMaker are Balanced and Sqrt(-3dB), stream sounds for stereo sound sources exhibit the following changes by default.

  • Stream sounds sound slightly different when panning is applied using nn::atk::SoundHandle::SetPan or similar.
  • The volume is -3dB lower during playback.

Fixed a bug where pan curves differed between surround mode and stereo mode.
After the fix, the pan curve is the same for surround mode and stereo mode.

This same bug is also reproduced in NW4F, so you can switch the behavior in NW4F compatibility mode.
Consider using the following function when performing a port using NW4F data.

  • nn::atk::SoundSystem::SoundSystemParam::enableCompatiblePanCurve
