Window Pane

This section describes window panes, which enable the use of windows that register textures as layout assets in fixed positions within a rectangular frame.

Overview

Window panes allow you to combine multiple textures to create a window frame.
You can create a window that is like a window frame by configuring the frame (the four corners and the top, bottom, left, and right sides of the rectangle), and the window's content (the images for the interior portion of the frame).

The following are the main settings that can be configured in window panes.

  • The rectangle's position, shape, and transparency
  • The texture images for the rectangular frame and the content
  • Color and alpha settings
  • Blending with the texture colors
  • Blending with the background and other panes

The following table shows the window pane settings.

Base Pane

Sets information including the pane name, position, shape, and center position.
Window Pane Sets information including the inversion, rotation, and vertex color for the image.

Texture

Sets the texture image, texture image settings, and texture pattern.

Color Interpolation Settings

Sets the input values that affect the window pane's color and alpha.

Texture Combiner

Sets the blending method for two or more images.

Fragment Operations

Sets the blending method with the background and other panes.

Extended User Information

Sets user-defined extended information for the pane.
