Added Custom Shader Samples

Added custom shader samples that incorporate normal and light maps into an effect.

Usage Examples

The normal and light map incorporation state for an effect using primitive particles can be checked in EffectViewer.
You can use the custom shader provided as a sample by setting the sample project file (ProjConfig.xml) in Project Settings in the Option settings ().
The sample is in the following location.

\Samples\Sources\Applications\VfxCustomShader\Resources\

The following section describes how to use the custom shader sample.

  1. To open the Option Settings dialog box, from the EffectMaker main menu, select Tools > Options.
  2. Select Project Settings.
  3. Register the sample project file (ProjConfig.xml) to the Project Setting File Path ().
  4. To finalize the settings, click OK.
  5. Load the effect file CustomShader.eset () in ESET () in EffectMaker.
  6. In the workspace, select Emitter1 ().
  7. Select the Cstm Shader tab ().
  8. To display the sample settings, from the Custom Shader Setting presets (), select Custom Shader Sample.
  9. To start the viewer, from the menu bar, click Launch Viewer ().
  10. Check the EffectViewer () that started to confirm the sample content as follows.

The following section describes the shader provided in the sample.

A Color Changes Multiplies the RGB value set in the Parameter () Color () by the emitter.
B Rendering Position Changes Adjusts the effect rendering position by adding the XYZ values of the Parameter () Added Position () to the emitter position.
C Lighting Applies simple lighting.
D Normal Color You can check the particle normal as a color in the viewer.
E Normal Map

Applies normal maps. Registers textures for normal maps to the Texture tab () Texture 1 ().
F Light Map Applies a simple light map with the Parameter () IBL Color 0 () empty color, IBL Color 1 () as the ground color.

Per-Frame Inheritance

Added per-frame inheritance functionality for child emitters to inherit the parent emitter's alpha values.
When this feature is enabled, the alpha value of particles are always multiplied by the alpha value of the parent emitter.

Usage Examples

You can save your work in EffectMaker as a workspace file (WORK).
When you open a saved workspace file, you effectively restore the workspace to the state it was in when the workspace file was saved.

The VfxSample () in Favorites is in the Effect Browser () Favorites tab as a sample.
Inside it, Afterburner.work () describes how to use the workspace file.
  1. In the Effect Browser, select the Favorites tab ().
  2. In the navigation pane, in Favorites, select VfxSample ().
  3. In the file list, select Afterburner (), and then from the toolbar, click Export ().
  4. Changes the display in the property window ().
  5. Afterburner.eset () and the associated files Afterburner () and Afterburner () are loaded together in the workspace.
  6. To start the viewer, from the menu bar, click Launch Viewer ().
