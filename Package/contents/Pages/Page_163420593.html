NVIDIA CUDA

Overview

The CUDA package enables you to use CUDA in the NX environment.

Packages

CUDA is provided as a separate package from the NintendoSDK NX Add-On. It may also be released later than other NintendoSDK packages.

The following table lists the packages.

Package Name Description
NVIDIA CUDA A package that enables you to use CUDA.

Placement

CUDA is located in the following location in the directory where NintendoSDK is installed.

NintendoSDK/
CUDA/

Installing

Run setup.exe after getting the CUDA package from NDI. This starts the installer and installs CUDA in Windows.

Documentation

For basic information about CUDA, see the following documentation.

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\doc\html\index.html

Library

The libraries for NX are installed in the following directories.

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\lib\NX-NXFP2-a32
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\lib\NX-NXFP2-a64

Samples

The NX samples are installed in the following directory.

C:\ProgramData\NVIDIA Corporation\CUDA Samples\v8.0\9_NX

Important Information

The CUDA package provided for NintendoSDK NX Add-On cannot be updated to any version newer than 8.x, meaning it cannot be updated to 9.x or later.

CUDA 8.x supports Visual Studio versions up to Visual Studio 2015, but NintendoSDK 11.x.x and later do not support Visual Studio 2015.

As a result, the CUDA package cannot be used in the NintendoSDK 11.x.x and later Windows environment.

In the NX environment, the CUDA package can continue to be used with Visual Studio 2017 and later.
