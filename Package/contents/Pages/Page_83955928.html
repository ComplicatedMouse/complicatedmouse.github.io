Guide for Migrating From NW4F to NintendoSDK

Overview

This page describes the procedure for migrating from NintendoWare for Cafe (NW4F) to NintendoSDK-NintendoWare.
This cannot run on Cafe, so you must migrate to the Windows platform.

Migration Procedure

The following procedure can be used to migrate from the NW4F to the NintendoSDK-NintendoWare environment.

  1. Install the following packages.
    • NintendoSDK
    • NintendoSDK-NintendoWare
    • NintendoSDK-NintendoWare-Samples
    • NintendoSDK-Documents

  2. Create a Visual Studio project for a user application according to the instructions in Creating an NX Application Project.


  3. Change the converter path and options

    Although the procedure was already described, for more information about the change descriptions, see the following Changes section.

Converter Tools

The following changes were made.

  • The NW4F prefix has been deleted from the names of the EXE files for the tools.
    • Example: NW4F_EffectMaker.exe → EffectMaker.exe
  • The Softimage plug-in is not provided.
  • NW4F_AnimSoundMaker.exe is not provided.

Namespace

nw::g3d

nn::g3d

Transitioned from the nw namespace to the nn namespace.

In addition, some of the library names have been changed in some modules.
nw::font nn::font
nw::lyt nn::ui2d
nw::eft nn::vfx
