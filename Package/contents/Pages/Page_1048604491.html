Capturing Framebuffer

The capture pane can convert the contents of the framebuffer, within the specified capture range, into capture images.
This section describes the process for loading the captured framebuffer contents as textures into the picture pane, then adding a distortion effect.

Info

The results of the data created here cannot be checked in LayoutViewer or the layout window.
This functionality allows the state of the game background (framebuffer rendering) to be checked via import into the game application.
You can check an image of actual incorporation into the game in the Ui2dDemo demo for developers, below.

\Samples\Sources\Applications\Ui2dDemo\Ui2dDemo-spec.NX.autogen.vs2017.sln

This demo uses three rotating cube models as the game background and applies the distortion effects created in this tutorial.

Related Links For more information for developers about how to implement this in games, see Capture Features.

Specifying the Default Value for the FrameBuffer Capture Flag

Using a text editor, open the project configuration file in the specified Parts root and set the default configuration for the FrameBuffer capture flag.

1

Open the NW_LytProject.flpj project configuration file located in the following directory.
\Samples\Sources\Resources\Ui2d\
2

Make sure that the following is set to true in the project configuration file.
<DefaultCapturePaneFrameBufferCaptureFlag>true</DefaultCapturePaneFrameBufferCaptureFlag>
Specifies the default value set for the framebuffer capture flag when a new capture pane is created.
3

Save the edited project file and close the text editor.

Checking that the Famebuffer Capture flag is enabled

Verify that the Framebuffer Capture flag is enabled.

On the main menu, select Pane > Create Pane > Capture Pane.
In the hierarchy window, select the created capture pane (), and change the pane name to C_FrameBufferCaptureTex.
In the property window for C_FrameBufferCaptureTex (), check the following settings.

Capture Pane Enable Framebuffer Capture ON

Confirm that the check box is selected.
If this setting is enabled, the game application generates an image capture of the game background.

Creating a Picture Pane

On the main menu, select Pane > Create Pane > Picture Pane ().

In the Texture window, double-click C_FrameBufferCaptureTex.tga ().
If you double-click the texture, a picture pane () is created with a texture converted from the framebuffer capture contents registered.

Thumbnail Filename Format

C_FrameBufferCaptureTex RGBA8: (uncompressed_RGBA+8A)

In the Hierarchy window, select the created picture pane () and change the pane name to P_FrameBufferCaptureTex.
In the property window for P_FrameBufferCaptureTex (), specify the following settings.

Base Pane ()

 Pane Position and Shape Size X,Y = 256,256
