Using Reference Colors

This section describes how to configure properties to use reference colors.

Creating a Reference Color

Use the following steps to create a reference color.

1

From the main menu of LayoutEditor, select Window > Color Sample to open the Color Sample window.
2

In the Color Sample window, select Reference Color, and clear the Lock Reference Color check box.
3 In the Color Sample window, select Reference Color > New Reference Color File to create a new reference color.
You can skip this step if you have already loaded a reference color file.
4

Double-click the New icon () below the reference color name to open the Color Information window.
5

In the Color Information window, enter the following information and click OK.

  • The color (required)
  • The name (required)
  • Comments

Assigning a Reference Color to a Property

You can assign a reference color to a property by dragging it from the Color Sample window to a color icon in the property window.
The property displays the name of the assigned reference color, and a mark in the shape of a white triangle appears in the lower-right corner of the color icon.

Removing a Reference Color from a Property

You can open a property's color picker to change its color and remove any reference color assigned to it.

Updating a Property With Changes to a Reference Color

If you modify a reference color in the Color Sample window, the color information is changed wherever that color has been assigned.

Setting the Alpha Value for a Reference Color

You can include an alpha value for the reference color by selecting the check box for A in the Color Information window.

If the reference color includes an alpha value, a mark in the shape of a white triangle appears in the lower-right corner of the alpha icon for the property to which the reference color was assigned, and the alpha is fixed to that value.

If the reference color does not include an alpha value, you can set an alpha value for the property to which the reference color was assigned.

Saving the Reference Colors Settings

To save the edited reference color to a file, in the Color Sample window, select Reference Color > Save All Reference Colors.
