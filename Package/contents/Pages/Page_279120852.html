Added a feature to set animation keys from the property window.

Added a feature to set animation keys from the property window.

An animation marker () is displayed next to items corresponding to animation in the property window.
In the Edit Animation Segment Tags window, an animation marker () is displayed only when segment tags are selected in separated mode ().
You can set animation keys as follows from the shortcut menu that appears when you right-click the animation marker () or the item name.

You can also check the status of an item from the color of the animation marker.
The following table lists the display statuses and their descriptions.

 State where no animation key has been configured for this item.

State where an animation key has been configured for this item.
The value of an item has been confirmed in the current frame of the timeline.

The value of an item is unconfirmed.
The specified values become invalid if you operate the timeline or save the file.

Adding a key to the current frame of the timeline changes an item to the confirmed state ().

Added shortcut keys for capture pane editing.

Added shortcut keys for capture pane editing.
The shortcut keys are convenient to use in editing a capture pane () in the Layout window.

CTRL+ALT+C in the Layout Window

This shortcut switches between showing and hiding the capture pane (), including panes lower in the hierarchy.
We recommend using this shortcut because working by placing the capture pane in the layout area impedes editing work.

Layout Window

On the left are the capture pane () and the panes that are the capture target lower in the hierarchy.

On the right is the picture pane () for using the image generated from the capture pane () as a texture.

Use CTRL + drag to change the size of the capture pane.

When the size of the capture pane () and the capture area are changed, this shortcut does not change the positions of the panes lower in the hierarchy.

Not Using the Shortcut Using the Shortcut (CTRL)


Affects the positions of the panes lower in the hierarchy. Does not affect the positions of the panes lower in the hierarchy.
