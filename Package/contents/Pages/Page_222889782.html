3D News

This document is a graphical introduction to the features that were added in NintendoSDK 3.1.0.

3DEditor

This section introduces the new features in 3DEditor.

Simultaneous Selection and Manipulation of Multiple Items in File View

Multiple items now can be selected and manipulated at the same time in file view.

Related Links For more information, see the following pages.

File View

Describes the File View shortcut menu and other features.

Processed Vertex Count Displayed According to the Platform Connection Target

The processed vertex count displayed in the shape property window now switches according to which platform 3DEditor is set to connect to. Previously, it showed the count for the Wii U.

Related Links For more information, see the following pages.

Shape Property Window: General Page

Describes the general properties of the selected shape.


Improved Readability for the Samplers Page of the Material Property Window

When the property window is opened, the width of the column for textures in the Samplers page is now adjusted automatically to match the length of the texture names. In addition, the sample display region has been enlarged.

Related Links For more information, see the following pages.

Material Property Window: Samplers Page

Describes the page used to check and edit the sampler of the selected material.


Improved Readability for the Pattern Page of the Animation Property Window

The sizes of the pattern and texture items now can be adjusted by dragging the mouse.

Related Links For more information, see the following pages.

Animation Property Window: Pattern Page Describes the texture pattern animation.


Expanded Environment Variable Paths Are Now Shown in the Team Settings

You can now display a path specified with an environment variable such as %MY_PROJECT%, as expanded to D:¥MyProject, or similar.

Related Links For more information, see the following pages.

Team Settings Describes the dialog box displayed for the team settings file.

Parent Material Can Be Selected From the List in the Specified Folder in the Team Settings

By specifying <ParentMaterialPath> in the team settings, you can now specify the parent material by selecting a material from the list of materials in the model intermediate file in the specified folder.

By adding a path below 3dEditor.teamsettings, as shown in the example, you can now select parent materials by starting 3DEditor.

<TeamSettings>
    <FileIo>
        <ParentMaterialPath Recursive="true">%MY_PROJECT%\ParentMaterials</ParentMaterialPath>
        ...
    </FileIo>
    ...
</TeamSettings>


Note

This only displays the materials for the model intermediate files that are assigned the same shader definition and shading model.

Related Links For more information, see the following pages.

Parent Material Page Describes the dialog box to set and check the parent material of the material referencing feature.

This document is a graphical introduction to the features that were added to the Maya plug-in.

Added Support for Maya 2017

Added support for Maya 2017. The Time Editor and the Shape Editor settings are now applied to the exported animation intermediate files.

Items Can Be Batch-Replaced in the Multi-Export Plug-In

You can now select multiple list items and batch replace export item option settings.
