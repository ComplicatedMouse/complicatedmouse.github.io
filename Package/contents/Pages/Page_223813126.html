Info

When you run Start3dEditor.bat, which is the batch file used by this tutorial, the shaders (demo.fsdb and town.fsdb) are loaded automatically.

The shader definition for the model file (FMDB) exported from Maya and used by this tutorial is assigned automatically by the Post-Export Script of the TutorialDefault preset that is specified when the file is exported.

Use the property window for each material to check its shader definition.
