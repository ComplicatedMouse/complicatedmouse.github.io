Calculation Method

This section describes the Calculation Method feature, which specifies how particle face orientations are calculated.

Overview

The Calculation Method feature specifies how particle face orientations are calculated.
The following overview describes the settings ().

Calculation Method

Select the stripe calculation method from among the following presets.

Billboard Stripe This type of stripe always faces the camera.
Emitter Matrix Stripe This type of stripe faces the emitter.
Extend Vertically From Emitter This type of stripe is perpendicular to the emitter.
Billboard Stripe2 This stripe faces the camera while reducing twisting of the stripe face.

Blue For more information, see the following pages.

Stripe Calculated by Emitter Matrix Method is Missing Describes how to handle cases where the stripe is missing when using Emitter Matrix Stripe.
