Conventions Used in This Manual

The following conventions are used in this manual.

Menus

Menus and menu items are written as follows: Menu Name > Menu Item. For example, File > Open is used to refer to the Open item in the File menu.

Key Combinations

Key combinations (also known as key bindings and shortcut keys) use multiple keys at the same time and are written by using a PLUS SIGN (+). For example, CTRL+S indicates pressing the S key while holding down the CTRL key.
