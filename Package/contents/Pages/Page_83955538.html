Camera

This section describes the camera in scene animations.

Camera Types

3ds Max free cameras and target cameras are supported.
The camera rotation mode in the intermediate file is euler_zxy for free cameras, and aim for target cameras.
The camera projection mode in the intermediate file is ortho if the Orthographic Projection parameter is selected in the Camera Parameters panel, and persp otherwise.

Camera Scale

Set the camera scale to 1.0.

Aspect Ratio

The aspect ratio is exported by calculating it based on rendering settings.

Angle of View

The angle of view is configured using Field of View in the Camera Parameters panel. This is converted to the vertical angle of view and exported in the intermediate file.

Orthographic Camera View Volume

When you use an orthographic camera, the height of the orthogonal projection is calculated based on the distance to the target, using the following expression.

Info

tdist = Distance to the target
fovY = Field of view in vertical direction
ortho_height = tan( fovY / 2) * tdist * 2

Attributes for Which Animation Can Be Set

The following list of attributes can be animated.

  • Common
    • Camera's position
    • Clipping planes (near and far clip planes)
    • Camera's field of view. (For the orthographic camera, the field of view angle is converted to a height of orthographic projection and exported.)
  • Free Camera
    • Camera's rotation
  • Target Camera
    • Target's position
    • Camera's roll angle
Info

Values in global coordinates are output for the camera's position and rotation. If the camera target (equivalent to Aim) node has a parent node, this also applies to the position, rotate and scale values of the parent node.

Lights

This section describes the lights in scene animations.

Light Types

The following standard lights are supported.

  • Ambient lights (Ambient Light in the Environment dialog box)
  • Directional lights (Directional Light or Target Directional Light)
  • Point Light
  • Spotlight (target spotlight or free spotlight)

Attributes for Which Animation Can Be Set

The following list of attributes can be animated.

  • The Position, Rotation, and Multiplier (intensity) of a light
    (Translation is available only for point lights and spot lights.)
    (Rotation is available only for directional lights and spot lights.)
    (Multiplier is available only for point lights, directional lights and spot lights.)
    (Distance Attenuation is available only for point lights and spot lights.)
    Data is exported with the Multiplier attribute enabled (disabled if set to 0.0, otherwise enabled).
  • The light's color.
    (Color values are exported unmodified.)
  • The hot spot and fall off parameters of a spotlight.
  • The Apply, Start Frame, and End Frame parameters related to the distance attenuation of a light.
    (The start frame and end frame of animations are exported only when Apply is selected.)
Info
Global coordinate values are exported for the light position and direction. If a light has a parent node, the Position, Rotation and Scale attributes of the parent node are also applied.
