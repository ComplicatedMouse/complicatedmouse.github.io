Z Roll Speed

This section describes the Z roll speed feature which controls the rotational speed of the selected node.

Overview

The Z roll speed is a feature that rotates the selected node at a specified speed.
Rotates the selected node in the direction of movement in a circular movement route.

Info

The stripe calculation method is not in effect for billboard stripes.

The following overview describes the settings ().

Z Roll Speed

The value to be added to the angle every frame is set using the dial.

Example Configurations

The following images compare different Z roll speed values.
