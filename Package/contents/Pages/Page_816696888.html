Added Synchronize the playback states of all layouts to the menu

Added Synchronize the playback states of all layouts to LayoutEditor's Edit menu.
When multiple sets of layout data are open in the LayoutEditor, you can select whether to synchronize the playback position of the time slider for all layouts.

The following video compares adjusting the Curve Editor Window time slider with two layout files displayed in LayoutEditor .

Disable Synchronize the playback states of all layouts Enable Synchronize the playback states of all layouts

In the two open layouts, the states of the time sliders are not synchronized.
This is useful for checking animations with different scales for each layout.

In the two open layouts, the states of the time sliders are synchronized.

This is useful for checking animations with uniform lengths, such as fades.
Info

As described later, when using the function to save the animation editing state while in separated mode with this function also enabled, it is possible to save the file without noticing that the playback position of the time slider has changed.
For this reason, if the animation is in separated mode, Synchronize the playback states of all layouts is not enabled when the layout file is opened.

Preserving the Animation Edit State in Separated Mode

When animations are in separated mode, the animation editing state can be saved.
By enabling the project setting <SaveStateAsInitialState>, the edit state when the layout was saved can be set as the initial state to use when the file is next opened.

The following video compares the <SaveStateAsInitialState> settings disable and enable.
To see the effect, save the file when the fade-in animation that completes the layout display is finished, and then reload it.

Disable <SaveStateAsInitialState> Enable <SaveStateAsInitialState>

Opens with nothing displayed at the start of the fade-in.
In this way, some animations open in an undesirable state.		 Opens with the layout that was displayed when the file was saved.
This allows the layout content to be easily grasped when the file is opened.
