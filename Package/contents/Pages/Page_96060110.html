Soft Particles, Process as Vertex

This section describes the Process as Vertex feature for soft particles that processes particles on a per-vertex basis.

Overview

The Process as Vertex feature applied soft particle processing at the level of primitive polygon vertices.
This effect primarily applies to primitive particle shapes. The more primitive particles there are for the number of vertices, the nicer the alpha gradation will appear.

The following overview describes the settings ().

To display Process as Vertex (), in Process as Vertex, select Soft particles. The group has the following settings.

Alpha offset at beginning of fade Describes Alpha offset at beginning of fade, the feature that configures transparency where particles and models overlap.
Sampling Radius Describes the Sampling Radius feature to configure transparency by assigning a nonzero thickness from the location where the particles and model overlap.
