Pattern Division

This section describes the feature for dividing textures into specified numbers of parts.

Overview

Use the Pattern Division feature to divide the texture into a specified number of parts in the horizontal direction (U) and the vertical direction (V).
The divided pattern can be used as frames for texture pattern animation.
The following section provides an overview of this feature ().

Pattern Division

Independently sets the number of divisions of the texture in the horizontal direction (U) and the vertical direction (V).
You can directly enter the value in the text box () or use the Up () and Down () buttons to change the value by 1 each time.
