Position Follows Emitter

The section describes the Position follows emitter feature for determining whether to make the position of the magnetic force follow the emitter when it moves.

Overview

Use this feature to set whether to make the position of the magnetic force follow the emitter when it moves.
This setting is useful when the Particle follow type of an emitter is set to Do not follow emitter.

Position Follows Emitter

Use the check box () to select whether to make the position of the magnetic force follow the emitter when it moves.
The following formula calculates the value added to the velocity of a particle as a result of the magnetic force if set to follow the emitter.

  • Value to add to speed: (Emitter position at that time + Magnetic force position - Position of the particle at that time - Velocity of the particle at that time) × Magnetic force
