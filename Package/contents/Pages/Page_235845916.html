Introduction

Purpose of This Documentation

NintendoSDK includes the HlslCrossCompiler and SLConverter tools for converting from HLSL to GLSL. This page provides an overview of each tool in addition to information about selecting which tool to use.

Intended Audience

This documentation is intended for developers who may need to use an HLSL converter while developing with NintendoSDK.

HlslCrossCompiler

HlslCrossCompiler uses a version of hlslcc.dll (Tools/Graphics/GraphicsTools/hlslcc.dll) forked from HLSLcc (https://github.com/Unity-Technologies/HLSLcc). It includes various proprietary tests and improvements and supports compute shaders, vertex shaders, and pixel shaders in addition to most shader model 5.0 features. HlslCrossCompiler will continue to receive updates such as support for all shader stages and support for new features added in NVN.

SLConverter

This tool only supports vertex and pixel shaders and only supports use of a limited set of shader model 5.0 shader features. Also, no new feature additions are planned. This tool is intended for users who already using SLConverter.
