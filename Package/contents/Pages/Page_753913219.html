Profiler-Sort

Overview

Sorts the contents of a tab by the specified sorting order.

Profiler-Sort [-Tab] <ScriptingTabs> [-SortBy] <ScriptingSortMethods>


[-Tab] <ScriptingTabs>

Specifies the tab to apply the sort on. The following values are supported for this argument:

    • Counters: Counters Tab
    • Files: Files Tab
    • Functions: Functions Tab
    • GraphicsGroups: Gfx Groups Tab
    • Instrumented: Instrumented Tab
    • Last: Last Tab
    • Logs: Logs Tab
    • Shaders: Shaders Tab
    • Threads: Threads Tab


[-SortBy] <ScriptingSortMethods>

The ways in which the contents of a tab can be sorted. The sort done is the same as clicking on a column header in one of the tabs. As such, not all tabs support all of the below sorting methods.

    • Self: Sort by the column Self. Supported by the Functions, Last, and Threads tabs.
    • Total: Sort by the column Total. Supported by the Functions, Last, and Shaders tabs.
    • Core: Sort by the column or option Core. Supported by the Functions, Last, Threads, Instrumented, and Counters tabs.
    • Name: Sort by the column Name or option Code Block Name. Supported by the Functions, Last, Threads, Instrumented, Counters, Gfx Groups, and Shaders tabs.
    • Thread: Sort by the column Thread or option Thread ID. Supported by the Functions, Last, and Instrumented tabs.
    • ThreadName: Sort by the column Thread Name. Supported by the Logs and Threads tabs.
    • Module: Sort by the column Module. Supported by the Functions tab.
    • Opcodes: Sort by the column Opcodes. Supported by the Functions tab.
    • InCalls: Sort by the column Seen In Calls or option Blocks. Supported by the Functions and Instrumented tabs.
    • OutCalls: Sort by the column Seen Out Calls. Supported by the Functions tab.
    • MaxRecursiveDepth: Sort by the option Max Recursive Depth. Supported by the Instrumented tab.
    • AccumulatedTime: Sort by the option Time per Block (average). Supported by the Instrumented tab.
    • Group: Sort by the column Group. Supported by the Counters tab.
    • Domain: Sort by the column Domain. Supported by the Gfx Groups tab.
    • Average: Sort by the column Average. Supported by the Shaders tab.
    • Invocations: Sort by the column Invocations. Supported by the Shaders tab.
    • TimeOpen: Sort by the column Time Open. Supported by the Files tab.
    • AverageTransferRate: Sort by the column Transfer Rate. Supported by the Files tab.
    • AverageBufferSize: Sort by the column Buffer Size. Supported by the Files tab.
    • OpenCount: Sort by the column Opens. Supported by the Files tab.
    • AccessCount: Sort by the column Accesses. Supported by the Files tab.
    • Path: Sort by the column Path. Supported by the Files tab.
