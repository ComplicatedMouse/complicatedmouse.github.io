ConvertNxbinlog

Overview

ConvertNxbinlog is a command-line tool for converting NXBINLOG files to text files.

It supports conversions into plain text and JSON format.

Info

NXBINLOG is a file format for saving logs from the development hardware to an SD card. For more information about using SD card logs, see the SD Card Logging Mode section in Development Assistance Settings.

Package

ConvertNxbinlog is provided in the NintendoSDK NX Add-On.

Location

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/ConvertNxbinlog.exe

How to Use

Use the -i option to specify the NXBINLOG file to convert, and the -o option to specify the output file. Specify the -json option to convert the file into a JSON format file. If this option is omitted, the file is converted into a plain text file.

$ ConvertNxbinlog.exe --help
ConvertNxbinlog.exe: Convert nxbinlog to ohter format file.

Usage example: ConvertNxbinlog.exe -i<arg> -o<arg> [--json]

Available Arguments:
  -i, --input <arg>          Input file name.
  -o, --output <arg>         Output file name.
  --json                     Convert to json file.
