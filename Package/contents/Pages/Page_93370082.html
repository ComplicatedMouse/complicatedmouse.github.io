Emitter Shape - Point

This section describes the settings items for a Point emitter shape.
These are the particles when this emitter shape is being used.

Overview

With a Point emitter shape, particles emit radially from a point defined in world coordinates.
This emitter shape is good when you want to display just one stationary particle.

This feature becomes available when the emitter shape is set to Point.
There are no settings items specific to the Point shape.
