Scalable Fonts

This section describes scalable fonts, which provide a way to draw characters in a specified size using a single font file.

Overview

The scalable fonts feature renders fonts using the system internal scalable font data, or user-specified TTF (OTF).
You can use the composite font definitions file to define any character size when displaying a single font file in the game application.
You do not need to prepare separate character data for each display size.
