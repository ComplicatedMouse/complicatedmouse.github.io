Coordinates (±)

This section describes the feature for applying an offset to the position of the emitter.

Overview

Coordinates (±) provides an emitter position offset using world coordinates.
The following overview describes the settings ().

Coordinates (±)

Use the dial () or the text box () to specify the offset for the emitter position.

Example Configurations

The following figure compares what happens when the value for Coordinates (±) is changed.

A (X,Y,Z) = (0,0,0) Emitter position when not using an offset. The emitter is located at the center of the world coordinate system.
Configures the animation for the emitter position.
B (X,Y,Z) = (-20,0,-20) An offset is applied to the emitter position.
Plays the animation at the emitter position after applying the offset.
