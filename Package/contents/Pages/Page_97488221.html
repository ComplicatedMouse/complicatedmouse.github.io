Texturing

This section describes the texturing feature to configure texture mapping to stripes.

Overview

Texturing is a feature to configure texture mapping to stripes.
The following overview describes the settings ().

Texturing

Select how to map textures to stripes from among the following presets.

Paste All The texture image is applied to match the size of the stripe.
Fit Paste to Rendering Range The texture image is applied to a rendering area of a fixed size.
