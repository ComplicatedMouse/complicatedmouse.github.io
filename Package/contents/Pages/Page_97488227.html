Number of Divisions

This section describes the Number of Divisions feature for dividing up string-like particles.

Overview

The Number of Divisions feature can be used to divide up string-like particles.
Increasing the number of divisions makes string-like particles appear smoother, even when they make radical changes in direction.
The following overview describes the settings ().

Number of Divisions

Set the number of divisions for stripe particles using either the dial () or by entering a number in the text box field ().
You can make stripe particles look smoother by increasing the number of divisions.
