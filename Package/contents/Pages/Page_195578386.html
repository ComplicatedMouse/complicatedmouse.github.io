Edit

Describes the Edit menu in LayoutEditor.
Undo Undoes the last edit.
Redo Reapplies the last edit that was undone.
Cut Cuts the currently selected pane to the clipboard.
Copy Copies the currently selected pane.
Paste Pastes the cut or copied pane into the layout.
Duplicate Duplicates the selected pane on the layout.
Delete Deletes the selected pane from the layout.
Paste Without Animation Appears when the Copy/Paste Animation check box is selected.
It omits animation information when pasting panes, items, or animation keys.
Duplicate Without Animation Appears when the Copy/Paste Animation check box is selected.
It omits animation information when duplicating panes, items, or animation keys.
Paste With Animation Appears when the Copy/Paste Animation check box is cleared.
It includes animation information when pasting panes, items, or animation keys.
Duplicate With Animation Appears when the Copy/Paste Animation check box is cleared.
It includes animation information when duplicating panes, items, or animation keys.
Copy/Paste Animation When this check box is selected, animation information is also copied and pasted together with panes, items, and animation that are copied and pasted.
Key Auto-Configuration When this check box is selected, animation is automatically set or changed whenever a drag operation occurs in the layout window or a value is changed in the property window.
Synchronize the playback states of all layouts

If animations are present in the layout data open in the LayoutEditor, you can select whether to maintain the time slider playback position for each layout individually or to synchronize them.
When selected, animations within the layout open in the Layout Editor are synchronized.
Tool Environment Settings Opens the Tool Environment Settings window.
Info

When copying or duplicating frames without animation, the values for the current frame are copied or duplicated for settings with animation.
By adjusting the current frame in the Curve Editor window, you can copy or duplicate it in the desired state.
