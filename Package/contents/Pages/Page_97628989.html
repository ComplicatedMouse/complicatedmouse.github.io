Size After Emission

This section describes the feature for multiplying the size of particles emitted from the emitter by the specified value.

Overview

The Size after emission feature multiplies the scale of all particles emitted from the emitter by the specified value.
This section provides an overview of the settings ().

Size When Emitted

Use the dial () or text box () to set the value to multiply the scale of particles emitted from the emitter by.

Example Configurations

The following figure compares what happens when the Size after emission value changes.
The Emitter tab's scale value is set to X,Y,Z = 2.

A X,Y,Z=1 The default setting. The size of the particles when emitted is the same as the Emitter tab's scale value.
B X,Y,Z=2 Compared with A , the size of the particles when emitted increases 200% for all (X,Y,Z).
C X=2, Y,Z=1
 Compared with A , the size of the particles when emitted increases 200% only for X.
