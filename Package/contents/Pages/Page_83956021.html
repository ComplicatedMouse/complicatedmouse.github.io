How to Use

The plug-in for combining images, creates an image in Photoshop formed by merging all image files in the specified folder.
When rendering many polygons (such as particles), rather than loading a separate texture for each polygon, it can be useful to load a single large texture and render the polygons by adjusting texture coordinates to render only a part of the texture.

Select File > Automate > Nintendo Unite Images.

  • Input Folder
    Specifies the folder containing the image files to combine.
    You can use images in any file format supported by Photoshop. (You can also use images in the file formats supported by the plug-in.)
  • Number of Pixels in an Image
    Specifies the Height and Width of a single image, in pixels.
    Selecting custom from the drop-down list allows you to directly enter pixel values.
    Select Same Value to automatically use the same value for height as for width.
    If the height and width of the image differs from these values, files will be merged after changing the size. Use bi-linear to re-sample the image.
  • Number of Images in a Row (in a Column)
    Specifies the number of images being laid out in the Horizontal and Vertical directions.
    Selecting custom from the drop-down list allows you to directly enter the number of images.
    Select Same Value to automatically enter the same number of images for the vertical direction as set for the horizontal direction.
    Images are arranged in alphabetical filename order, starting from the upper left.
  • Unite Alpha
    Select this option to combine alpha channels.
    You can reduce the processing time by disabling this option if you know none of the images includes an alpha channel.
  • Fill Color Edge with Black
    Select this option to fill the one-pixel edge of the color channel with black for each image.
  • Fill Alpha Edge with Black
    Select this option to fill the one-pixel edge of the alpha channel with black for each image.

Click OK to create a combined image.

Notes

These notes are for using the plug-in for combining images.

Image Mode

The number of bits per channel in the combined image being created is the same as for the first of the source images in alphabetical order.

Alpha Channel

If there is more than one alpha channel in an image file, only the alpha channel at the top of the list in the channel window is combined.

If the Unite Alpha setting is enabled when EXR files are merged, the A component is read as the alpha channel. If the Unite Alpha setting is disabled, the A component is read as transparency.

Layers

If there is more than one layer in the image file, an image with combined display layers is created.
