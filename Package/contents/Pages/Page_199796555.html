Layout Settings Window

Layout Settings Window ()

In the Layout Settings window, you can configure layout settings for the selected layout window.

Select Comments, Environment Settings, or Extended User Information on the left of the Layout Settings window to display the different settings screens.
You can configure various settings in each layout window.

Comments

To display the comments settings screen, in the Layout Settings window, on the left, select Comments.

Set the comments for the selected Layout window.
The comments you set are only used for the configured Layout window.

Comments Settings Screen
Sets comments for individual Layout windows.
The set comments are only used for the comments settings screen.

Environment Settings

You can use Layout Environment Settings to change the colors and grid settings in the selected Layout window.

To display the environment settings screen, in the Layout Settings window, on the left, select Environment Settings.

Environment Settings Screen
A Color Specifies the background color of the layout window.
B Images

Specifies a background image.

Displays the file path of the loaded background image.
C Draw Grid Select this check box to display the grid in the layout window.
D Grid Color Specifies the grid color.
E Grid Size Specifies the intervals between the thick grid lines.
F Number of Divisions Specifies the number of divisions into which to divide the gaps between the thick grid lines.
G Screen Size Specifies the size of the layout screen.

Extended User Information

Select Extended User Information on the left of the Layout Settings window to display the screen for setting extended user information.

The specified extended user information is applied to the entire selected layout window.

Extended User Information Settings Screen

Configures extended user information for the entire selected layout window.

The extended user information for individual panes is configured in the property window for each pane.
