Model Files For Linking

This page describes the model-related files used in the Model Settings node.

Overview

EffectMaker allows you to preview an SRT transform by registering a model file and the animation files bound to that model.
In EffectMaker, you can register a maximum of five model files, and a maximum of four animation files for each model.

Where to Save Texture Files

Whenever EffectMaker registers a model file, it searches for a texture file with the same name based on the texture filename that is stored in the file.
Save texture files in a searchable folder on the same drive as the associated model file.

The following image shows the search order of texture files and searchable folders when the model file registered in EffectMaker is Model.(extension), and the texture file where the name for that file is stored is Texture.(extension).

  1. When the model file Model.(extension) is registered, EffectMaker first searches for texture files that are in the same folder.
  2. Next, EffectMaker searches in the Textures folder directly under the folder that contains the model file Model.(extension).
  3. Next, it searches in the Textures folder above the folder that contains the model file Model.(extension) until it finds a texture file.
  4. Finally, EffectMaker searches in the highest level Textures folder on the drive. This folder is the last folder that can be searched.
    If no texture file has been found by this point, EffectMaker displays a warning.

EffectMaker makes no distinction between uppercase and lowercase characters in the Textures folder name.
