File Events

You can set a user command file to be executed during file operations.
A temporary file path (for example, C:\Users\UserName\AppData\Local\Temp\tmp123A.tmp) that points to the list of all the files to be processed is passed as an argument to the command.

Pre-File Save Command

Pre-file save command () is a group of settings for specifying the user command file you want to execute immediately before files are saved.

Post-File Save Command

Post-file save command () is a group of settings for specifying the user command file you want to execute immediately after files are saved.

Pre-File Delete Command

Pre-file delete command () is a group of settings for specifying the user command file you want to execute immediately before files not being used in the Effect Browser are moved to the Unused folder.

Post-File Delete Command

Post-file delete command () is a group of settings for specifying the user command file you want to execute immediately after files not being used in the Effect Browser are moved to the Unused folder.

Pre-File Open Command

Pre-file open command () is a group of settings for specifying the user command file you want to execute immediately before files are opened.
