Shader Definitions Property Window

This window is used to check and edit the selected shader definition file.

A Displays the Shader Definition page.
This page displays information specified in the shader definition file.
B Displays the Shading Model page.
C Displays the Source Code page.
This page can be used to check and edit GLSL shader source code.
D Displays the File Information page.
