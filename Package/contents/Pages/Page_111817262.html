Panes

Panes are the smallest component of the layout data representing each individual image on the screen.
Use the LayoutEditor to position panes and create layouts.

Types of Panes

This section describes pane types.
LayoutEditor uses the following eight pane types.

Null Pane

This type of pane can control the upper and lower panes, in the Hierarchy window hierarchy, as a group. Nothing appears on the screen.

Picture Pane

This type of pane enables the use of added texture images as layout assets.

Text Box Pane

This type of pane uses added fonts as layout assets and enables string handling.

Window Pane

This type of pane enables the use of textures for fixed rectangular frame, and the use of windows as layout assets.

Boundary Pane

This pane enables determinations for things like button hits.

Capture Pane

This type of pane captures the framebuffer and layout rendering results, and enables use as a texture image.

Align Pane Describes the Align Pane, which is used for aligning and adjusting the positions and sizes of panes in the Hierarchy window.

Scissor Pane

This section describes the Scissor Pane, which clips the rendering targets of panes lower in the Hierarchy window in rectangular shapes.
