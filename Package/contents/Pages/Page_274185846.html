Error Displayed Indicating D3D Device Initialization Failed

An error message indicates that D3D device initialization failed.

Question

An error message indicates that D3D device initialization failed.

Answer

Check the following.

  • Restart the tool and open it again to see if the error is reproduced. (Is it reproducible?)
  • Is the error reproduced in other people's environments? (Is the problem dependent on the operator's PC environment?)
  • Can other data open? (Is the problem data-dependent?)
  • Is the error reproduced in a new layout (CTRL+N)? Is the curve editor shown? (Is the error reproduced in the simplest state?)


When you suspect that the problem is dependent on the operator's PC environment or data, try resetting the editing environment.

  • Procedure
    • Export the color sample (or it will be lost).
    • Close LayoutEditor.
    • Delete setting.xml in %LOCALAPPDATA%\Nintendo\LayoutEditor\X.X.X.0 (X.X.X.0 is the version you are using).
      • You can easily access this tool by pasting the preceding path in the address bar of Windows Explorer. (Press the Windows logo key+E to open it.)
    • Open LayoutEditor.
