Using Search Tabs

Overview

Use the Search tabs to find items in a sound project based on characters and special conditions.
Searches can also be conducted using the Search window, but this section describes the use of the Search tabs.

Search tab

Item lists that match the search criteria can be displayed in multiple tabs.
Whereas the Search window displays search results for individual sound set tabs or bank tabs, a Search tab can show all those results together.

Opening Search Tabs

To open tabs for sound set item searches, select Edit > Open item search tab.
You can also use the CTRL+SHIFT+F shortcut key for this operation.
To open tabs for bank instrument searches, select Edit > Open instrument search tab.

Search tabs appear on the right side of the main window, just like sound set tabs and bank tabs. Split views are also supported.

These search tabs hold the following information.

  • Search conditions
  • Search results
  • Filters

Searching for Attributes

Use the buttons and check boxes beneath the search field to select the search conditions you want to use.

Aa When this button is selected, SoundMaker searches for items exactly as entered. (For example, capitalization counts.)

* Selecting this button allows the use of the "*" wildcard in search strings.

  • When selected, the asterisk symbol (*) is treated as a string for partial match queries. (It matches zero or more characters.)
  • When not selected, the asterisk symbol (*) is treated as a wildcard for matching. (It matches zero or one character.)

The check boxes allow you to specify whether to search for the search string you entered in the names, file paths, or comments of items.

Note: Stream sound tracks do not have a name. Stream sound tracks are not found even if a name search is specified.

Select the Item reference check box to search for items that are using the player (bank).
Also, you can select the Starting Playback Position check box to include a starting playback time for the sequence sound your are searching for.

Item Search Tabs

Click the Select search target button in the Item search tab to open a dialog box for selecting sound sets.
The Select sound set dialog box shows a list of the sound set names registered in the project. Select the check boxes for the sound sets you want to search.
The names of the sound sets you have selected in this list are shown to the right of the button.
If you have selected all sound sets in the list, the string Entire Project is shown in this field.

Search Tab Features

  • To view an item in the Sound Set tab or Bank tab, double-click an index cell in the displayed list of search results.
  • You can apply filters to the search results by using the buttons at the top of the Search Results list.
    You can filter for Only Active List Items, Only Active Items, Item Type, and Item Color.
    Click Show All to select all item types and item colors, or select Hide All to clear all of these items.
    Selecting Only enabled items for List Item hides unnecessary list items in the search results.
    Note: List Item is not displayed in instrument search tabs.
  • Various parameters (list items) appear for the items in the search results. You can edit these parameters directly by clicking the cell or right-clicking to bring up the shortcut menu.
    Basic features are performed the same as for each list in the Sound Sets tab, but some features are restricted.
    Note: In instrument search results, you cannot make edits using the shortcut menu.
  • To sort items within the list, click the column names.
  • You can save and apply list item settings the same way as in the Sound Sets and Bank tabs.
    Register/Apply List Item Settings
  • When you search on instruments, the Bank item shows the bank that each instrument belongs to.
  • You can add Search tabs to your bookmarks in the same way as Sound Set tabs.
    To add a search tab to your bookmarks, right-click the tab and select Add to Bookmarks from the shortcut menu.
    Bookmarks record the search criteria and filters. Applying one of these bookmarks opens a new search tab and displays the search results.
Undo

When you make edits in a search tab, a message about Undo operations is displayed in the lower portion of the window.
After this message appears, if you make any edits in the Sound Sets tab, you cannot undo operations in the Search tab.
However, you can always undo individual edits made in the Search tab by using the Undo button in the Sound Sets tab.

Reference Search

To search for items that are using a particular player or bank, right-click the Player list or Bank list, and from the shortcut menu, select Find All References.

The entire project is searched for items that reference the target item, and the results are listed in the Search tab.

During conversion, warning messages about unreferenced items are displayed in the Conversion Log window.
