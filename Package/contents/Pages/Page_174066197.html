Atk Performance Panel

The atk performance panel displays atk performance information.

UI Reference

Item Description
atk SoundThread Displays the time from when atk sound threads are started to the time that they are finished.
atk TaskThread Displays the time from when stream sound data is loaded and the Opus decoding process is started to the time those processes are finished for atk task threads, along with other internal processing time.
DSP

Displays the time from when Voice, MainMix (or SubMix), FinalMix, and Sink processes are started to the time they are finished.

  • For Voice processes, this time is from when the process for the first item starts to the time when the process for the last item is finished.

Although the DSP process time itself is accurate, exact start and end times cannot be retrieved for DSP processes, and so the values displayed are only rough approximations.
Vertical orange line Represents the current time.

For more information about the displayed values, see Audio Performance Information.

In the PC versions of applications, start times, end times, and processing time may be affected significantly by other processes running on the host machine.

How to Operate

Operation Procedure
Click Moves the current time to the specified position.
Right-click + drag Scrolls horizontally.
CTRL+mouse wheel Magnifies and shrinks in the horizontal direction.
ALT+mouse wheel Magnifies and shrinks in the vertical direction.
