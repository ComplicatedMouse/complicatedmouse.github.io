Materials - 3ds Max

Materials

Material Names

Although you can assign the same name to multiple materials with 3ds Max, make sure that you assign a unique name to each material because there is no duplicating of material names in the intermediate file.

If multiple materials with the same name exist, the Export plug-in will operate internally to output the first material that it fetches with the existing name and then to alter the name of each subsequent material by adding the underscore symbol and a number (for example, name_1, name_2, and so on).

The user cannot check or change the export order for exported numbers and materials. This order of export may change depending on the version of the DCC tool, or the environment.

Material Types

Use Blinn, Phong, and Anisotropic standard materials.

If you use other materials, you can export them if the parameters are the same as for these standard materials, but there is no support for any other parameters.

When using anisotropic reflections, connections and binormal vectors (calculated from map channel 1) are exported even if normal mapping has not been set.

Valid Attributes

The following attributes are included in the intermediate files.

Also, you can set attributes for NintendoWare by using the plug-in for setting material attributes.

  • Diffuse → Diffuse color
  • Opacity
  • Ambient → Ambient color
  • Self-illumination → Emission color
  • Specular → Specular color

Multi/Sub-Object

Multi/sub-object materials are also supported.
When multi- or sub-object materials are used, the vertex data stored in intermediate files is divided by each sub-material for output.
