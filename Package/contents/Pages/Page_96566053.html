Child Emitter

An emitter located below another emitter in the hierarchy in the workspace is called a child emitter. The particles emitted from this child emitter are called child particles.
An emitter located above a child emitter in the hierarchy in the workspace is called a parent emitter, and the particles emitted from this emitter are called parent particles.

The following emitters are in a hierarchy.
● represents the parent particle, and ● represents the child particles emitted from it.

Child Emitter Settings and Differences in Behavior

The following table shows combinations of settings for which the child emitter behavior is the same when the Moving Object meets the following conditions.

  • Emitter: The parent emitter position is animated, and the speed of emitted particles is 0.
  • Particle: The parent emitter position is fixed, and emitted particles are accelerated.

Matching colors in the Matching Behavior column represent settings for which the child behavior is the same.
If the behavior is the same, we recommend clearing the Emitter Assignment for Particle option because it involves a lower processing load.
