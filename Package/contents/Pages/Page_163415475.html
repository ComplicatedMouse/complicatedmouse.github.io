Preparing Controllers

Introduction

This section provides information about using controllers for the NX, including the Joy-Con and Switch Pro Controller.

If you want to use controllers for the NX in your development process, you must prepare the environment as directed in this guide.

Overview Description Notes
Updating Controller Firmware

Update the controller internal firmware.

Connect the SDEV and the controller via Bluetooth and update the firmware.

To use the controller, you must update the firmware.

Connecting Controllers in the SDEV/EDEV Environment Controllers must be paired before they can be connected in the SDEV/EDEV environment. (Paring is the process of registering a controller in the system.)
Changing the Bluetooth Driver Software (Windows 7 Only) If you plan to connect the controller to a PC running Windows 7, you must change the Bluetooth driver software. This step is not needed for Windows 8.1 or 10.
Pairing Controllers in a PC Environment (Adding Devices) To connect the controller in a PC environment, you must use Windows functionality to pair it with your PC.
