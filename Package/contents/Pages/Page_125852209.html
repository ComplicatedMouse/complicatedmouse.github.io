5. New Material Animation Tutorial

5. Creating Material Animations

This section describes the process of creating material animations.
Using diffuse color RGB animation to make the Sample Tripod's eye blink.

This tutorial involves the following UIs.

Create New Animation Creates a new material animation.

Creating a Curve

 Create and edit animation curves.
Adjust the Key in the Color Picker Select a color to adjust the curve.

Info

This completed data is in the following location.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\Eye_Finish.fmab

Create New Animation

Right-click Default Animation Set, select Create New Animation, and then click Material.

The Create New Animation window opens. Set the following settings.

Name Eye
Frame count 100
Loop playback On

After clicking OK, Eye.fmab is created in File view.
A view of the Eye.fmab file is also added (as in the following figure).

Related Links For more information, see File View Basic Operations.

Selecting Diffuse Color

In File view, double-click Eye.fmab to open the Properties window.

The remainder of the operations on this page are controlled from the Properties window.


Related Links For more information, see Animation Property Window.

From the properties list on the left, select EyeMt > Shader Parameters > Diffuse.

Only the Diffuse Color option is selected, as indicated in the following figure.
Check that R, G, and B are all selected.

Note

There are two materials (Diffuse) for this model. Do not confuse it with BodyMt.

Related Links For more information, see Curve Tree View.

Creating a Curve

From the menu, click Add a key (). Enlarge the window size if you cannot find the option.

Create three key frames by clicking the middle mouse button.
The RGB components can be added in an overlapping state.

Related Links For more information, see Toolbar (Curve Editor).

Editing Keys

Adjusts each key frame.

Drag a key frame to change its value as described in the following figure.
The RGB curve is overlapping. The values of all of the RGB components must be changed.

Enter values in the text box (see the following figure) to adjust the value.

Enter key frame values like those in the following table.

Frame Value
0

0.0
50 1.0
100 0.0

The curve will become like the one in the following image. (The three RGB curves are overlapping at the same position.)

Info

To shift the focus to the overall curve, press A.
To shift the focus to the selected key, press F.
Click or drag at a position away from any keys or curves to clear the selection for the currently selected objects.

Adjust the Key in the Color Picker

Drag to select the curve. Drag a part of the curve other than a key to select it.
With the curve selected, click the white block indicated by the red box in the following figure.

Click the pink color in the color picker. Clicking the color will transmit the RGB values to the key.

The result is similar to the following image.

Close the Properties window to finish editing the curve.

Related Links For more information, see Time Slider (Curve Editor) and Curve View.

Info

You can also adjust the curve using the mouse. You can use the following methods.

On the menu, click Move multiple ().
You can click or drag with the left mouse button to select multiple keys or curves, and then drag the center mouse button to move them.

On the menu, click Move closest key ().
You can drag and move the key without having to select a key or curve. (Only one key at a time.)

Checking in the Viewer

Play back the animation and verify that the SampleTripod's eye is flashing pink.
