Specifications of Suffixes Indicating the Format of the Texture Name

Introduction

A suffix is attached to the name of textures converted in LayoutConverter to indicate the format of the texture. This document describes the specifications for the suffix.

Specifications of Suffixes Indicating Format

Suffix

The suffix indicating the format is represented by the following 2 to 3 characters.

First Character

The first character is a symbol that indicates whether the file is for indirect use.

For indirect use +
Not for indirect use. ^

Second and third characters

The second or third character is one of the following symbols indicating the texture format.

The format is specified in order from the beginning of enum (flyt.TexelFormat), which enumerates the format, according to the rules of "a, b, c, d, …, z, _A, _B, _C, _D, …_Z.

Because Windows does not differentiate between uppercase and lowercase characters in the filename, add a "_" before uppercase names.
