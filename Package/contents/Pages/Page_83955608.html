Shape Property Window: Mesh Page

This page is used to check mesh information for the selected shape.

A Mesh number.
B

Displays the primitive shape used for the vertex index.
C Displays the primitive topology type.
D Displays the vertex index type.
E Displays the number of vertex indices.
F Sub-mesh number.
G Position of the vertex index that starts the render call.
Always 0 when exported from a DCC tool.
H Render vertex index number, from the vertex index position that starts the render call.
