Noise Movement Speed (±)

This page describes the Noise Movement Speed (±) feature for configuring the movement speed for the noise table.

Overview

The Noise Movement Speed (±) feature makes it possible to configure the movement speed of the noise table along the XYZ axes.
Noise movement makes it possible to dynamically change the movement direction of particles.

The following images illustrate what happens when the noise movement speed value for the x-direction is changed.

The following overview describes the settings ().

Noise Movement Speed (±)

Use the dial () or the text box () to specify the noise table movement speed in the XYZ coordinate system. Larger values result in faster noise table movement and more significant changes in particle movement direction.
