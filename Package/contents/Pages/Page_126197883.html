Introduction

This document describes the features of the Face Library.

It starts by introducing topics that are common throughout the Face Library.

The namespace.

All of the features provided by the Face Library, such as types, constants, and variables, belong to the nn::mii:: namespace.

Unless explicitly specified otherwise, do not use any of the features in namespaces deeper than the nn::mii:: namespace. The hierarchy and specifications of features in these namespaces are subject to change without notice.

Header

Include the nn/mii.h header file to enable access to the features of the Face Library.

nn/mii.h is located in the following directory.

$NINTENDO_SDK_ROOT/Include/nn/mii.h

The other header files are located in the following directory. nn/mii.h includes these files.

$NINTENDO_SDK_ROOT/Include/nn/mii/

Do not include the header files in directories deeper than the nn/mii/ directory. The hierarchy and specifications of these directories are subject to change without notice.
