Physique

Physique can be exported.

However, Physique-specific features like bulge angle, tendons, and skin sliding are not supported. These are exported as a regular skin.
The exported data is nearly identical to that exported when Skin Update is set to Rigid for the Physique modifier.

The supported objects are Bone, Biped, Dummy, and Point Helper.

Info

Make sure that you do not perform the following operations after applying the Physique modifier.

  • Operations that change the number of vertices.
  • Delete nodes that are affected by skinning.

Shape Exported to the Model File

Physique is exported in the Figure mode, but the Biped pose is changed after it is exported.
Before exporting, make sure that you set Biped keys and the like and save the pose.

Info

If the total weight value of all the vertices does not equal 1.0, the weight ratios are retained but the value is adjusted to 1.0 for output.
