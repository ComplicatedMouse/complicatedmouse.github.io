Direction Change Ratio

This section describes the Direction change ratio feature for setting the change ratio when the history is moved using a waveform.

Overview

The Direction change ratio feature sets a ratio for the period of the waveform when the history is moved using a waveform.
The larger the value set for Direction change ratio, the finer the swaying of the history.

The following overview describes the settings ().

Direction Change Ratio

Set the change period for the history using the dial () or by entering a value in the text box ().
The larger the value, the shorter the period of the waveform and the finer the movement of the history.
The speed of the rate of change of the history is calculated with the following expression.

  • History change rate: sin (Time × History change period × Direction change ratio) × Initial rate of history

Blue For more information, see the following pages.

Initial Rate of History Sets the initial velocity of the history in a random direction, changing the amplitude of the waveform.
History Change Period Sets the period of the waveform that changes in the direction of the specified axis.
