Main Window

Main Window

This window is the main window of 3DEditor.

A Title Bar

Shows the name of the software and the file being edited.
B Main Menu Used to call various features.
For a description of short-cut keys, see here.
C Toolbar

Displays frequently used menu items as icons.
D File View

Displays all of the files loaded in 3DEditor in a tree view and shows the information for the selected file.
E Animation Controls Controls the playback of the animation in the view.

F

Log Bar

Retains log messages generated when assigning material shaders, applying original animations, or performing other operations.
You can enable or disable the visibility of the Log Bar using the triangle icon at the top of the pane, or from the View menu.

G

Status Bar

Displays descriptions of menu items and the file I/O status.

H

Object View

Displays the different types of objects that are included in the files loaded in 3DEditor, in a variety of ways.
This view can be turned on (visible) and off (not visible).
