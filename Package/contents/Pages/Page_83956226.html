Procedures Required With All Sounds

This section describes required procedures common to the creation of all sounds.
The procedure described in this section is absolutely required when you use any type of sound.

Creating a Player

What Is a Player?

The Player manages sounds created by SoundMaker when they are ultimately played back in the application using the atk library.

Players are created by SoundMaker. All sounds must have at least some player settings. For this reason, you must prepare a player ahead of time when you create new sounds for a newly created sound project or sound set.
If you create a stream sound or another type of sound without first creating a player, a player named DEFAULT_PLAYER will be created automatically.

Player Display

To create and edit players, use the Player list on the Sound Set tab.
Double-click the Player node in the project tree, or single-click if the Sound Set tab is open. The player list then appears on the Sound Set tab.

Creating Players

On the Sound Set tab, display the Player List. Click the Player List to change the focus to the list, and then, from the menu, select Project > Add Item > Player.
The Create New Player dialog box appears. Enter the required information.

Preparing Resources

SoundMaker is used to create all sounds that use waveform data, and standard MIDI files that are created by using typical formats as resources. Create resources using your favorite sequencer or wave editing tool.
This section describes the various file formats that can be used for SoundMaker resources.

The characters that can be used in a filename or path name are restricted. The following characters can be used.
　- The filename must begin with a single-byte letter or the underscore. [a-z, A-Z_]
- Other characters in the filename must be single-byte alphanumerics or the underscore. [a-z, A-Z, 0-9_]

Waveform File

Waveform files are a required resource when you create all sounds.

Waveform File Formats

Prepare waveform files that meet the following conditions.

Format
  • Linear PCM (.wav / .aiff / .aif)
  • nn::audio opus (.opus)
Number of Channels
  • 1ch / 2ch
    ※Data flow using the atk tool is not supported in 3ch and higher waveform files.
    ※You can achieve surround mode by applying the pan settings to the 2ch source.
Sample Rate
  • Optional (PCM16, ADPCM)
    ※ .opus is dependent on OpusEncoder.
Number of quantization bits
  • 8, 16, 24, or 32 bits
    ※However, this setting is changed to 16 bits during conversion.
Loop

One loop can be specified for the waveform file.
When you specify a loop, the range in this loop plays again and again.

Making Loop Specifications

To add loop information, you must use a separate application for editing waveform files.
In the waveform editing application, set the loop information for the starting point and ending point for the part of the waveform you want to loop.
If you want to loop the entire waveform, set the start of the waveform and the end of the waveform to the loop information.

You can only specify a single range to loop in each waveform file.

Loading in SoundMaker

Register the sound file with the loop information to your SoundMaker project. An "L" is displayed at the beginning of the Duration cell for looped sounds in the sound list on the Sound Set tab.
If "L" is displayed, the time shown for Duration is not for the length of the waveform itself, but the time until the end of the loop.
Note that in some cases this means that the time displayed does not represent the duration of just the looped portion.

MIDI Files

MIDI files are a required resource when you create sequence sounds.

MIDI File Format

Prepare type 0 or type 1 standard MIDI files (with the .mid filename extension).
For more information about the valid controls in files, see the Sequence Data Manual.

Text Sequence Files

A text sequence file (FSEQ file) is a sequence file especially for the NintendoWare sound development environment that is coded in text called sequence commands.
This resource is required when you create sequence sounds that require a higher level of control, which is difficult to achieve with MIDI files.

For more information about creating text sequence files or sequence commands, see the Sequence Data Manual.

Creating Stream Sounds

Displaying the Stream Sound List

Stream sounds are created and edited from the Stream Sound List and Stream Sound Track List on the Sound Set tab.
Double-click the Stream Sound node in the project tree (or single-click if the Sound Set tab is open). The Sound Set tab is split into upper and lower sections that display two kinds of lists. The upper list is called the Stream Sound List, and the lower list is called the Stream Sound Track List.

The waveform files that are set to stream sound in the stream sound list selection status are displayed in the Stream Sound List.

Creating New Stream Sounds

To create or add a stream sound, display the Stream Sound List, change the focus to the list on the Sound Set tab, and then select Project > Add Item > Stream Sound.
The Add New Stream Sound dialog box appears. Enter the required information.
If you specify a file in the path, the Create from filename button becomes active.

Creating Wave Sounds

Wave Sound Sets and Wave Sounds

Wave sounds are created in two stages: the creation of the wave sound set and the creation of the wave sounds that are used by the wave sound set.
First, create a wave sound set. Then create the wave sounds to use in that wave sound set.

During conversion, waveform data for each wave sound is collected together at the wave sound set level.
Although wave sounds are what actually get played as sounds, the waveform data for these wave sounds is loaded into memory in units of wave sound sets.

Displaying Wave Sounds

Wave sounds are created and edited by using the Wave Sound Set List and Wave Sound List on the Sound Set tab.
Click the Wave Sound Set node in the project tree. The Sound Set tab is split into upper and lower sections that display two kinds of lists.
The upper list is called the Wave Sound Set List, and the lower list is called the Wave Sound List.

Creating Wave Sound Sets

To create a wave sound set, click the list on the Sound Set tab to move the focus, and then, from the menu, select Project > Add Item > Wave Sound Set.
The Create New Wave Sound Set dialog box appears. Enter the required information.

Creating Wave Sounds

Wave sounds are created by using the wave sound list.

Click the Wave Sound Set node in the project tree, and then in the upper part of the Sound Set tab, select a wave sound set from the wave sound set list. The wave sounds inside this selected set appear in the lower part of the tab.

To create a wave sound, select the wave sound set you want to include it in and then select Project > Add Item > Wave Sound.

The Create New Wave Sound dialog box appears. Enter the required information. If you specify a file in the path, the Create from filename button becomes active.

Creating Sequence Sounds

Creating Banks

What Is a Bank?

To create a sequence sound, you need a set of tone programs that the sequence will reference. This set of tones is called a bank.

Banks are created as bank files (FBNK files) using a special SoundMaker format and referenced from a sound set.

More than one instrument can be created for the bank. You can set these instruments to program numbers 0 through 127, corresponding to MIDI's Program Change numbers.
Sequence sounds are always created in parallel with bank editing.

Bank List Display

New banks are created by using the Bank list on the Sound Set tab.
Select the Bank node in the project tree. The list displayed on the Sound Set tab is the Bank list.

Creating a New Bank

To create a new bank, while the Bank list is showing, select Project > Add Item > Bank.
Make the necessary entries in the Create New Bank dialog box that appears.
If you specify a file in the path, the Create from filename button becomes active.

Editing Bank Files

Bank files are edited on the Bank tab. Open the Bank tab by double-clicking the Bank list on the index cell that contains the icon for the bank that you want to edit, or by double-clicking the bank file in the project tree.

The Bank tab is divided into upper and lower sections. The upper section is called the Instrument List. This list enables you to create instruments, specify program numbers, and other activities relating to instruments.
The lower part of the Bank tab has two more tabs for switching between the display of the sample map and the percussion list.
You can use both the sample map and the percussion list to edit instrument regions.

For more information about Bank tab operations, see the Practical Guide.

Creating Sequence Sounds

Displaying the Sequence Sound List

Sequence sounds are created and edited from the Sequence Sound List on the Sound Set tab.
Double-click the Sequence Sound node in the project tree, (or single-click if the Sound Set tab is open).
The Sequence Sound List then appears on the Sound Set tab.

Creating Sequence Sounds

To create a sequence sound, select the Sequence Sound List and select Project > Add Item > Sequence Sound.
The Create New Sequence Sound dialog box appears. Enter the required information.
If you specify a file in the path, the Create from filename button becomes active.

Because many waves are often played overlapping each other when using sequence sounds, the default volume for them is set lower than that of stream sounds or wave sounds.

Creating Sequence Sound Sets

Sequence Sound Sets and Sequence Sounds

Sequence sound sets are created in a two-step process. First, you create the sequence sound set, and then you create the sequence sounds for that set.

For more information about creating sequence sounds, see Creating Sequence Sound and Creating Banks.

Displaying Sequence Sound Sets

To create and edit sequence sound sets, use the Sequence Sound Set list and the Sequence Sounds List on the Sound Set tab.
Double-click the Sequence Sound Set node in the project tree (or single-click if the Sound Set tab is open). The Sound Set tab is split into upper and lower sections that display two kinds of lists.
The upper list is called the Sequence Sound Set list, and the lower list is called the Sequence Sound List.

Creating Sequence Sound Sets

To create a sequence sound set, view the Sequence Sound Set List and then select Project >Add Item > Sequence Sound Set.
Type a name in the Create New Sequence Sound Set dialog box that appears.
