Parts Pane

This section describes the Parts pane, which provides a way to use general-purpose layouts as panes.

Overview

The Parts pane is a pane that can be used to take a general-purpose layout of the kind used for buttons and other game UI elements and place it as a part inside some other layout like a body layout.
The parts layout is saved as a file that is then placed as a Parts pane () in a body layout.
This helps reduce the cost of making revisions because, by changing just the parts layout, all of the information is rewritten for the Parts pane that the layout is placed in.

The following table shows the Parts pane settings.

Base Pane

Sets information including the pane name, position, shape, and center position.
Parts Pane

Sets how to use the Part Scaling Effect for the parts layout.

Extended User Information

Sets user-defined extended information for the pane.
