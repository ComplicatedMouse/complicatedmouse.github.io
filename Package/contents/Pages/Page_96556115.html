Rotation Axis

This section describes the Axial rotation features which is used to specify the center of rotation.

Overview

Axial rotation is the feature for specifying the rotational axis of the spin using an XYZ coordinate system based on the emitter.
Specifying Axial rotation makes it possible to select the axis that particles rotate around.
The following overview describes the settings ().

Rotation Axis

Specify one option for the spin rotation axis from the drop-down list.
The particles will rotate around the selected axis using emitter coordinates.
