Features

This section describes the features provided by the gfxutil library and how to use the API resources.

Features

Module Name Overview
DebugFontTextWriter Library that performs simple text rendering to use, for example, during debugging and development.
DescriptorPoolAllocator The allocator for portioning out the slots in a descriptor pool for use.
MemoryPoolAllocator The allocator for portioning out the memory in a large memory pool for use.
TextureCompressor Library for compressing textures during execution.
ScalableViewport The library for rendering layouts with accuracy when the resolution changes dynamically.
