Scalable Font Settings Properties

Scalable Font Settings Property Window

This section describes the UI of the Scalable Font Settings property window.



 A Comments You can add comments.
These comments are added to the ScalableFontDescriptionv node in the FCPX file.
B Character Range Group

Specifies the character range used in the composite font.
Up to 16 character ranges can be set.
C Hexadecimal check box When selected, displays the start and end positions in hexadecimal.
When cleared, displays the range start and end positions in decimal.
D Enabled check box

Feature for editing the start and end.

Range setting is enabled when this check box is selected.
E Start Position entry text box Specifies the start position of a character range.
F End Position entry text box Specifies the end position of a character range.
G Character Weight Specifies the character weight.
H Border Width Specifies the border width.
I TTC Font Index Specifies the TTC Font Index.
J Character Scale (Width) Sets character scale (width).
K Character Scale (Height) Sets character scale (height).
L Baseline Offset Specifies the offset of the baseline.
M Right and left bearings (margins) of a glyph Feature for editing the margins on the right and left of a glyph.
A margin can be deleted or the margins can be uniformly increased or decreased.
N Feature to forcibly render as monospace This function forcibly renders glyphs at the same width.
O Rendering Options Specifies the rendering options.
P Forcibly Overwrite Metrics

Forcibly overwrites original font data height information.

Resolves the following problems with adding glyphs to font data.

  • The maximum height (changed to make the line space wider).
  • The high operating cost of readjusting all the layout data

Use the WinAscent and WinDescent values in the OS/2 table in the font data to specify the ascent and descent values.

These values may be displayed differently depending on the tool. For example, they are called the Safe zone top and Safe zone bottom values in Fontographer.

Info

Use functions N and P in the table when fonts with added glyphs are supplied in the font header, and font values cannot be rewritten for licensing reasons.
