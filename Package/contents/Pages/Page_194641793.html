Playback Controls

This section describes the features displayed in the Playback Controls of the Curve Editor window.

Playback Controls control the playback of animations in the Curve Editor window and layout window.

Playback Controls
A








Move to the first frame.
Go back one frame.
Move to the previous key frame.
Reverse playback.
Stop.
Play.
Move to the next key frame.
Advance one frame.
Move to the last frame.
B

The current frame rate is displayed.

To change the frame rate, in the main menu, select Edit > Tool Environment Settings > Frame Rate.
