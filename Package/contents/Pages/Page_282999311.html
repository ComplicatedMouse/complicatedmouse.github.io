Border Pane

This section describes border panes, which can be used for detecting button hits and the like.

Overview

Border panes are a feature for specifying hit detection in panes.
The program can use a border pane placed in the layout for determining the hit detection area.

Border panes can be used in the following ways.

  • Hit detection when the cursor is over the pane

The following table shows the settings for border panes.

Base Pane

Sets information including the pane name, position, shape, and center position.

Extended User Information

Sets user-defined extended information for the pane.
