Animation Property Window

Animation Property Window

This window is used to check and edit the properties of the selected animation.

Related Links Material animations, texture pattern animations, and skeletal animations have properties that are not included in other animations. For more information, see here.

A Displays the animation page.
Select this when copying and pasting all animation properties.
B Displays the General page.
C Displays the Pattern page.
D Edits the animation in Curve Editor.
E Displays the Original Information page.
F Displays the File Information page.
G Displays the Additional Reference Paths page.
H Displays the Optimization Log page.
I Displays the User Data page.

Special Settings by File Type

Material animation, texture pattern animation, and skeletal animation have properties that the other animations do not have.

Additional Reference Paths for Each Model and Material Animation

You can specify an additional reference path for each model and material animation file.

Related Links Select Additional reference path to display the Additional Reference Paths page.

Material Animation, Texture Pattern Animation

In material animations and texture pattern animations,
the Pattern page was added to the property items.

Related Links Select Pattern to display the Pattern page.

Skeletal Animation

For skeletal animation, a Preview page is added to the property items.

Retarget source model and Enable mirroring playback may be set for the skeletal animation in the preview page.

  • To preview an animation's retargeting, select a retarget source model.

Related Links The retarget origin model may also be set from the file view animation set shortcut menu.

Info

Retargeting skeletal animation is valid for models with the same skeleton hierarchy.

  • You can flip and play animation by enabling mirroring settings.
    Note: Motion mirroring must be set for the model.

  • Related Links For more information, see Plug-In for Setting Motion Mirroring.

  • Related Links For more information about creating motion mirroring, see Motion Mirroring.

Bone Visibility Animation

For bone visibility animation, a Preview page is added to the property items.

You can flip and play animation by enabling mirroring settings.

  • Note: Motion mirroring must be set for the model.

  • Related Links For more information, see Plug-In for Setting Motion Mirroring.

  • Related Links For more information about creating motion mirroring, see Motion Mirroring.
