LayoutEditor

This section introduces the new features in LayoutEditor.

Function for Selecting Panes Using a Common Color

Functionality has been added for selecting panes using a common color.
For example, if you right-click the button in the property window for setting the color for white color blending, and then click Select panes with common white color blending, panes in the layout are selected that have the same white color.
You can open this menu from the color setting button in the property window, Color in the Color Sample window, or the Color key icon in the Curve Editor, and then select Main Menu > Common.
