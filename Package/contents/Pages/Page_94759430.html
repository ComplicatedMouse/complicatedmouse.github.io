This section describes the waveform types.

Overview

The Waveform types feature displays when you select either setting in Apply Flux.
Select from the presets the waveform type to use for animation.
This section provides an overview of the settings ().

Waveform Types

Select the behavior of flux animation from among the following presets. A description is provided for each preset, and the shape of the waveform is shown.

Sine wave

A gradual curved pattern using sine waves.

Sawtooth wave A rectilinear pattern with steep rises and gradual falls.

Rectangular wave An "all or nothing" pattern.
