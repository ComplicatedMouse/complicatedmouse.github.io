Project Settings

Project Settings

Settings can be configured individually for each game project.
We do not recommend configuring these settings for each user.

Info

Any changes to the settings in the light blue box () are overwritten by the settings in the project settings file when EffectMaker starts.

Project Settings File Path

The Project Settings File Path feature () is used to register the settings file for each game project.
You can use files in which rendering orders unique to game applications are written.

The Project Settings file describes the following settings for files.

The settings in the Project Settings file Other data in the file

Linear Workflow

Standard gravity

Info

Click the Reset icon () to reset settings to their defaults.

Preview Draw Setting

The Preview Draw Setting group () is used to show the draw settings set for each game project that is contained in the project settings file.

Linear Workflow

The Linear Workflow group () is used to toggle the linear edit mode on or off in the game project.

Related Links For more information, see the following pages.

Linear Edit Mode Describes color space as it relates to the color editing environment and displaying textures.

Standard gravity

The Standard Gravity feature () enables and disables the Apply Standard Gravity button on the Emission tab.
When it is enabled, the following settings are available.

Unit length

Dimensions in the world of the game (in centimeters).
FPS The basic frame rate of the game project.
Info

The following equation is for the use of standard gravity, where T is the FPS and M is the unit length.
Equation for calculating the gravity value = Gravitational acceleration 9.8 / ( T × T ) × M / 10000

Documentation file path

The Documentation File Path group () is used to set the location of the documents referenced by EffectMaker.
Sets Documentation File.
Click the OK button to confirm the settings, and they are applied when EffectMaker restarts.

This setting is found in both the User Settings and the Project Settings.
