TCP and UDP Ports Used by the SDEV


The SDEV uses the following TCP and UDP ports when connecting to a network.

49152 and higher TCP

Info

In the table, htc-gen1 refers to communication with the target using Target Manager, and htc-gen2 refers to communication with the target using Target Manager 2 and Nintendo SDK Daemon.

Note

In NintendoSDK 11.1.0 and NintendoSDK 11.2.0, configuring and controlling the SDEV from Target Manager always uses these three TCP ports: 60101, 60102, and 60103.
