Emitter Extended Parameter Node

This section describes the emitter extended parameter, which can be used to configure data available to the game application.
The data can be used for game application features and workflow purposes.

Flags

The Flags group () is used to set user data with flags.
This group has the following settings.

Bits 0-15

A group of 16 flags that can be individually selected using the check boxes ().

Numbers

Use the Integer Number Parameters group () to set user data with sliders.
This group has the following settings.

Numbers 1 to 2 You can set these to any integer value in the range of 0-1000000, using either the dial () or the text box field ().

Numbers

Use the Real Number Parameters group () to set user data with sliders.
This group has the following settings.

Numbers 1 to 2 You can set these to any real number in the range of 0-1000000, using either the dial () or the text box field ().

Below is an example of the use of emitter extended parameters in a game application.
