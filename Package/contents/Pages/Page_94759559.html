Sine Wave

This section describes sine waves.

EffectMaker uses sine waves for particle movements, alpha, and scale animations.
Waveforms composed of the elements shown in the following figure are called sine waves.

This figure defines the terms we use for the places that control the shape of the sine wave.
These terms are used in the labels for the fields in EffectMaker.

A Amplitude Amplitude The amplitude of fluctuation.
B Phase Phase The offset of the waveform.
C Period Period One cycle of the waveform (time).
