Development Hardware Setup

This page describes the procedure for setting up the development hardware (SDEV, EDEV, HDEV, and ADEV).

To run your NX application on the development hardware, refer to the described procedures for preparing and initializing each different kind of development hardware.

Preparing and Initializing

The process of initializing the development hardware deletes all installed applications and save data and deletes or resets the defaults for all configured settings, including network settings, account settings, and the various development hardware settings.

Initialize the development hardware when you will use it for the first time, when the device begins to behave unstably, and when you want to reset all of the settings.

Initialization requires the use of the NintendoSDK SystemUpdater for NX package.

Updating

You can update the firmware by initializing the development hardware, but this also resets all of the settings, as previously described.

To change the version of the firmware but leave the settings intact on the development hardware, use either of the methods described in the following links.

Depending on the procedure, you will need both the NintendoSDK SystemUpdater for NX package and the NintendoSDK DevKitVersionUpdater for NX package.
