Creating a Menu Screen

This section describes the procedure for creating the following kinds of menu screens.
Ui2dDemo is used for the data.

1 Creating a Parts Layout Describes the procedure for creating a parts layout generally used in the menu screen.
2 Creating a Body Layout Describes the procedure for creating a body layout that places the parts layout and completes it as a menu screen.
3 Using a Parts Layout Describes the procedure for loading and assembling parts in a body layout.
4 Creating Animations for Background Decorations Describes a sample procedure for animation creation in LayoutEditor to animate texture coordinates for a background decoration.
5 Creating Part Animations Describes a sample part animation creation method (a SELECT Button in a parts layout).
6 Checking Feature Animations Describes a sample revision procedure, while confirming a feature animation, for a SELECT Button in a parts layout.
7

Using Derivative Parts

Describes the procedure for using the derivative parts feature to create parts with different variations.
