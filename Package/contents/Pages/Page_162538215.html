No Effects Appear

Verify the following items if effects are not displayed in EffectViewer.

  • Tentatively close all effects within the workspace, create a new emitter, and make sure that the communication status for EffectMaker and the Viewer is normal.
  • Make sure that the number of emitters and particles is being counted in the Viewer.
  • When counted, data is being exported. Check whether the following types of display restrictions are in use.
      • Check whether the visibility button for the workspace node () is set to Hide ().
      • Check the status of the Render particles feature in the emitter node's Basic Settings tab.
      • Tentatively reset emitter position and particle speed settings, and check whether there are any effects outside of the Viewer screen.
      • Check whether One Time has been selected in the emission settings.
        If particles repeatedly display when looping () is enabled in the Preview Controls, you may have selected the One Time setting.
  • Check the following when no emitters or particles are being counted.
    • Check the status of the Viewer connection in the toolbar. If that status is Disconnected (), see EffectViewer Not Starting.
    • Make sure that no errors appear in either Message View or the Console.
