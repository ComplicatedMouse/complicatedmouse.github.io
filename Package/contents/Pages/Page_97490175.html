File Path

This section describes the File Path feature for checking the path where the preview file is stored.

Overview

The File Path feature allows you to check the path to where the preview file selected in the workspace is stored.
The following overview describes the settings ().

File Path

Displays the path to where a file is saved. The following table describes the buttons.

Overwrite Current Settings and Load Preview Overwrites the current settings and loads the preview file (PREV file) selected in the dialog box.
Load Additional Preview Loads the preview file (PREV file) selected in the dialog box, adding it under the emitter set.
Save Preview Saves the preview file (PREV file) selected in the dialog box.
Save Preview As Saves the preview file (PREV file) selected in the dialog box with a new name.
