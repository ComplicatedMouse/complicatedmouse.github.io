Introduction

The Settings Manager tool can be used from a PC to perform the following operations that are related to the system settings.

  • Import the settings
  • Export the settings
  • Reset the settings

How to Use

Import

Use this command to import the values for the system settings from a JSON file.

NintendoSDK 4.1.0 and Later

# Import the settings to the default development hardware.
Tools\CommandLineTools\Settings\SettingsManager.exe Import -r my_settings.json
 
# Import the settings to the explicitly specified development hardware.
Tools\CommandLineTools\Settings\SettingsManager.exe Import -t XAL0000000000 my_settings.json
 
# Import the settings to the host environment.
Tools\CommandLineTools\Settings\SettingsManager.exe Import my_settings.json
 
# Use the help option to discover information about the command arguments.
Tools\CommandLineTools\Settings\SettingsManager.exe Import --help

NintendoSDK 4.0.0 and Prior Versions

Note

In NintendoSDK 4.0.0 and prior versions of the Settings Manager tool, you need to specify absolute paths to files. Moreover, you cannot specify files located directly beneath root directories, such as c:\ and d:\.

# Import the settings to the default development hardware.
Tools\CommandLineTools\RunOnTarget.exe TargetTools\NX-NXFP2-a64\SettingsManager\Release\SettingsManager.nsp -- Import D:\files\my_settings.json
 
# Import the settings to the explicitly specified development hardware.
Tools\CommandLineTools\RunOnTarget.exe TargetTools\NX-NXFP2-a64\SettingsManager\Release\SettingsManager.nsp -t XAL0000000000 -- Import D:\files\my_settings.json
 
# Import the settings to the host environment.
Tools\CommandLineTools\Settings\SettingsManager.exe Import D:\files\my_settings.json

Export

Use this command to export the values for the system settings to a JSON file.

NintendoSDK 4.1.0 and Later

# Export the settings from existing development hardware.
Tools\CommandLineTools\Settings\SettingsManager.exe Export -r my_settings.json
 
# Export the settings from the explicitly specified development hardware.
Tools\CommandLineTools\Settings\SettingsManager.exe Export -t XAL0000000000 my_settings.json
 
# Export the settings from the host environment.
Tools\CommandLineTools\Settings\SettingsManager.exe Export my_settings.json
 
# Use the help option to discover information about the command arguments.
Tools\CommandLineTools\Settings\SettingsManager.exe Export --help

NintendoSDK 4.0.0 and Prior Versions

Note

In NintendoSDK 4.0.0 and prior versions of the Settings Manager tool, you need to specify absolute paths to files. Moreover, you cannot specify files located directly beneath root directories, such as c:\ and d:\.

# Export the settings from existing development hardware.
Tools\CommandLineTools\RunOnTarget.exe TargetTools\NX-NXFP2-a64\SettingsManager\Release\SettingsManager.nsp -- Export D:\files\my_settings.json
 
# Export the settings from the explicitly specified development hardware.
Tools\CommandLineTools\RunOnTarget.exe TargetTools\NX-NXFP2-a64\SettingsManager\Release\SettingsManager.nsp -t XAL0000000000 -- Export D:\files\my_settings.json
 
# Export the settings from the host environment.
Tools\CommandLineTools\Settings\SettingsManager.exe Export D:\files\my_settings.json

Reset

Use this command to reset the system settings to their default values.

NintendoSDK 4.1.0 and Later

# Reset the default development hardware settings.
Tools\CommandLineTools\Settings\SettingsManager.exe Reset -r
 
# Reset the settings on the explicitly specified development hardware.
Tools\CommandLineTools\Settings\SettingsManager.exe Reset -t XAL0000000000
 
# Reset the host environment settings.
Tools\CommandLineTools\Settings\SettingsManager.exe Reset
 
# Use the help option to discover information about the command arguments.
Tools\CommandLineTools\Settings\SettingsManager.exe Reset --help

Configuration File

Format

The settings file uses the following format. 

{
    "head": {
        "version": "1.0"            // The format version.
    },
    "body": {
        "my_settings": {            // The settings group.
            "my_key": {             //  The settings item.
                "current": true,    // Current value of the setting.
                "default": false    // Default value of the setting.
            }
        }
    }
}
