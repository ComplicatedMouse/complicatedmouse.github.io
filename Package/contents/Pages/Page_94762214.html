Gradient Editor

Use the gradient editor to configure color and alpha animations along the axis of Particle time.
You configure these using the Color/Alpha Settings group () in the Color tab ().

A Behavior Type

Select 8-Key Animation from the choices to display the gradation editor to animate colors.

Animation is enabled.
B Key

An animation key (). Click the key to register the color at that position (or timing).
Click a position to set the key there with the horizontal width of the gradation bar () as the time axis.

C Selecting a Key Click an animation key () to select it.
D Deleting a Key Click Delete Key () to delete an animation key ().
E Playback Method Toggle You can switch the playback method between linear playback () and loop playback ().
F Frame Length When Looping Can be set when Playback Method is set to loop. The entire animation is repeatedly played back in the specified number of frames.
G Random Starting Position

Select or clear this check box to set whether to change the particle color every time a particle is emitted ().

When enabled, the particle color is selected randomly from the overall gradation scale every time a particle is emitted.
H Presets Click this button to create frequently used alpha animations.

Presets

The presets are buttons that you can click to generate alpha animations that are often used for effects.
The following table describes the animation behavior for each button.

2 key

Changes the alpha value from 1.0 to 0.0 over the particle lifetime.

When using this preset, setting the lifetime to be short or using this effect at the same time as other effects, results in an effect in which particle generation is relatively subtle.
4 key

Change the alpha value from 0.0 to 1.0 by the time 25% of the particle lifetime remains.

Then, change the alpha value from 0.0 back to 1.0, from 75% to 100% of the particle lifetime.
This can be used when working with particles with longer lifetimes to make particle generation occur more gradually and subtly.
