Information Windows

There are three pop-up informational windows in the profiler:

  1. Function Details Window
  2. Thread State: System Call Details Window
  3. Thread State: Synchronization Object Details Window

Function Details Window

  • This window shows details for a specific function.
  • To open, hover over a function name, right-click and select Show Details.
  • The Function Details Window section shows the specifics of this window.

Thread State: System Call Details Window

  • This window shows details for a specific thread state.
  • Requires Thread State to be recorded on the Sampled Profile or Instrumented Profile tab.
  • To open, select a thread on the Threads tab, then on the Timeline tab in the Thread State Data section, double-click a thread state rectangle.
  • The Thread State: System Call Details Window section shows the specifics of this window.

Thread State: Synchronization Object References Window

  • This window shows details on a specific thread synchronization object.
  • Requires Thread State to be recorded on the Sampled Profile or Instrumented Profile tab.
  • To open, within a System Call Details Window, click on the View button next to the synchronization object.
  • The Thread State: Synchronization Object Details Window section shows the specifics of this window.
