Guide to Setting Up the Environment

Overview

This documentation presents essential information for developers who are building applications and libraries that use the NintendoSDK and want either to use an existing build system environment or want to construct their own build system.

For developers who want to construct a new build system environment, we recommend that you create a project from Visual Studio using the NintendoSDK wizard for creating a new project.
