Cell Size

This page describes the cell size option, which can be used to specify the cell size and glyph output position.
As described below, there are several different ways to configure this option.

A Specify Size/Offset Disables the top margin and left margin settings.
The glyph is automatically positioned in the center, according to the cell width and cell height.
B Specify Margin Disables the right margin and bottom margin settings.
Cells are calculated with the same value for the left margin as is used for the right margin. Cells are calculated with the same value for the top margin as is used for the bottom margin.
