Sample Map

The Bank tab is divided into upper and lower sections. The Instrument List appears in the upper section. Tabs for the Sample Map and Percussion List appear in the lower section.
The Sample Map is displayed in the lower section when the Sample Map tab is selected.

Role of the Sample Map

The Sample Map is a special editor for editing instruments.
You can edit instruments that are selected on the Instrument List.

The Sample Map is an instrument editor that features a typical layout that consists of a keyboard and an area above it that visually denotes the regions by using blocks. The Sample Map is suited for the creation of multiple-region instruments.

Sample Map Operations

This section describes the operations on the Sample Map.

Region Display

A light blue bar is displayed above the area where the regions are displayed. This bar is called the region header.
Click the region header to manipulate all velocity regions as a group.

Creating Key and Velocity Regions

You can split key and velocity regions by using the shortcut menu. To open the menu, right-click a region or a region header.

You can create new key regions by double-clicking blank regions that result from the deletion and movement of existing key regions.

A velocity region is treated as part of a key region. Accordingly, you cannot create a velocity region in a location where there is no key region.

Restrictions on Splitting Regions

You cannot split a velocity region by clicking a region header when multiple velocity regions are selected. You also cannot split a key region that has been split into several velocity regions and not all of the velocity regions are selected.

Specifying a Reference Wave File for a Region

The reference wave file for a region can be specified by double-clicking the region's area or by selecting Change Referenced Waveform from the shortcut menu.

The original key is automatically set to the appropriate value based on the specified waveform file's filename and additional information.
(For example, the A3 portion of the Test.A3.aif filename.)

Opening a Folder or Reference File

To open a folder or reference file, right-click the file to open the shortcut menu, and select either Open Folder or Open Referenced File.

Key Region Operations

You can swap the contents of a key region with another key region by dragging the region header. At this time, if velocity regions exist in the key regions that are being swapped, that information is also swapped.

You can copy the contents of a key region to another key region by dragging the region header while holding down the CTRL key.
As with swapping, if the key region that is being copied contains velocity regions, that information is also copied.

Velocity Region Operations

If you use the mouse to drag from any point that is not on a region header (in other words, from a velocity region), it is treated as an operation on the velocity region.

Operations are the same as those used for the region header: Move or replace using normal drag-and-drop operations, or copy by holding CTRL while dragging.

When the boundary of a velocity region is dragged, a tooltip appears that indicates the value.

Specifying Compression Formats for Regions

To specify a compression method for waveform files that are referenced by a region, select the Compression list box below the keyboard.

  • ADPCM
    Converts the waveform file to the proprietary ADPCM format.
    Compression automatically results in the transfer of a loop header to a block header. Note this when using playback controls that use loop position timing.
  • PCM16
    Converts the file to 16-bit PCM data.

Specifying Original Keys for Regions

The original key for a region is modified by using the slider that is located between the region display area and the keyboard.
At the keyboard position where the slider is set, playback of the selected region is at the original pitch.

Velocity Specifications

The velocity for playback using a keyboard can be specified with the slider on the left side of the area used to display the region.

Measuring the Number of Times Instrument Wave Files Are Accessed

This feature allows you to measure the number of times the instrument wave files that are used for all sequence sounds within a sound project are used.

Select Preview > Measure reference count of instrument waveform.

The Measure reference waveform of instrument dialog box appears. Set the time period over which to measure access, and then click Measure.
The measurement is made starting from the Starting Playback Position of each sound.

The number of accesses is for each sample map region and is highlighted in a shade of red that is proportional to the frequency of access.

If the sequence sound file path or the file itself is updated externally, the number of accesses is displayed with an asterisk (*), indicating that the result may not be accurate.

For information about measuring the reference waveforms of instruments, see Waveform Reference Count.
