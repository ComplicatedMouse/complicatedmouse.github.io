Using the Grouping Features

This section describes how to use the grouping features.
Groups are used to group together panes from different parent/child hierarchies, and to register one pane in multiple groups.

Creating Groups

Use the following procedure to create a group.

1 In the Layout window or the Hierarchy window, select multiple panes that you want to bring together in a group.
2 Right-click the selected panes and select Create Group.
Alternately, use the keyboard shortcut CTRL+G.
3. From the main menu, select Window > Groups to open the Group window.
4

Confirm that the panes selected in step 2 have been registered as children of the group.

Deleting Groups

Use the following procedure to delete a group.

1 From the main menu, select Window > Groups to open the Group window.
2 Select the group to delete.
3. Select Operation > Delete.

Editing Groups

Use the following procedure to edit the group name and comments.

1 From the main menu, select Window > Groups to open the Group window.
2 Select the group to edit.
3. Select Operation > Edit Selected Group.
4 In the Edit Selected Group window that appears, edit the name and comments for the group.
