Stop Emission

This section describes the feature for stopping the emitter from emitting particles.

Overview

The Stop Emission feature can be used to stop the emitter from emitting particles while a Fade command is running.
Game developers can use this feature to force particle emission to stop.
The following overview describes the settings ().

Stop Emission

Stop Emission is toggled on and off using the check box.

Info

We recommend that you stop emission and then enable alpha fadeout or scale fadeout when particles have infinite lifespans.
This prevents particles that have already been emitted from remaining.
Related Links For more information about infinite lifespans, see Infinite Lifespan.
