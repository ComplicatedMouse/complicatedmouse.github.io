Overview

You can change settings related to displaying information for past errors, exporting, and crash reports.

How to Use

Checking Error Information

Select the error information that you want to check from the list and press the A Button to confirm the selection. The error viewer starts, and the error description can be viewed.

Writing Error Information to a File

To save all error information in a file to a microSD card or a connected PC, click Export.

The save location for the file on a microSD card is /NintendoSDK/NxErrorReports. The save location on a PC is %USERPROFILE%/Documents/Nintendo/NxErrorReports.

The file saved by this feature corresponds to the file exported by the Error Features error export command.

Consider submitting this exported file if you are contacting Nintendo to resolve a fatal error for which you cannot determine the cause.
The contents are encrypted and cannot be read.

Changing Settings for Crash Reports

Select Settings from the drop-down menu at the top of the screen to transition to the settings screen.

For more information about each of the settings, see Enabling Crash Reports in Development Environments.
