Lifespan

This section describes the Lifespan feature, which can be used to configure particle lifespans.

Overview

A particle’s lifespan determines the time from when it is emitted from an emitter to the time the particle dies.

Info

Animations that are applied in terms of the time axis of a particle’s lifespan are known as particle time animations.

Related Links For more information about particle time, see Emitter Time and Particle Time.


The following overview describes the settings ().

Lifespan

Use the dial () and text box () to set how long particles remain alive after page emitted from the emitter.
