Introduction

Feature for handling the language setting in the system settings.

It is provided as an add-on because each platform has a different method for getting the current language setting.


The available language setting values are as follows. The values correspond to the language code in the Internet Engineering Task Force (IETF) tag format.

The language code values are available by calling the GetAvailableLanguageCodes() function.

Info

Chinese (Simplified) and Chinese (Traditional) are supported starting with NX Add-On 4.0.0.

Brazilian Portuguese is supported starting with NX Add-On 11.0.0.

Structure in the Windows Environment

There is no particular relationship between the language setting and the Windows system locale.

Using Language Settings

#include <nn/nn_Log.h>
#include <nn/settings/settings_Language.h> 
 
extern "C" void nnMain()
{ 
// Displays the language setting candidates.
    int count = nn::settings::GetAvailableLanguageCodeCount();
 
    nn::settings::LanguageCode* codes = new nn::settings::LanguageCode[count]();
 
    count = nn::settings::GetAvailableLanguageCodes(codes, count);
 
    for (int i = 0; i < count; ++i)
    {
        NN_LOG("%d %s\n", i, codes[i].string);
    }
 
    delete[] codes;
}
