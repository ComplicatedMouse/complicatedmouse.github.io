Build System

Introduction

The build system documentation contains information essential to the building of applications and libraries using NintendoSDK.

The intended audience is all developers engaged in development using NintendoSDK.

The Structure of This Documentation

Page Description
Overview Presents an overview of the NintendoSDK build system. Read this first if you are a developer using the NintendoSDK build system for the first time.
Guide to Setting Up the Environment Presents essential information for those building applications and libraries using NintendoSDK. The intended audience is developers who want to use the existing build system environment and developers who want to construct their own build systems.
Tutorial Uses an example to show how to make a build system using the NintendoSDK.
Manual Describes how to use the NintendoSDK build system.
Reference Presents the specifications of the NintendoSDK build system.
