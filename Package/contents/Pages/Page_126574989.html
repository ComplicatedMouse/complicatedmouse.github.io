User Data Resources

Overview

User data resources express any user data added when converting resources.

User Data Resources

User data resources are defined using nn::gfx::ResUserData.

Getting Names

Use GetName to get the names of user data resources.

Getting the Data Count

Use GetCount to get the data count held by the user data resources.

Getting the Type

Use GetType to get the data type held by the user data resources.

Getting Data

In the following table, data can be obtained by calling the function to get data corresponding to the type obtained using GetType.

Type Function to Get Data
nn::gfx::ResUserData::Type_Int GetInt
nn::gfx::ResUserData::Type_Float GetFloat
nn::gfx::ResUserData::Type_String GetString
nn::gfx::ResUserData::Type_Stream GetStream
