Animation plays back incorrectly in the game.

Question

Animation plays back incorrectly in a game application.

Answer

This issue occurs when multiple animations are played simultaneously in the game, and different animations are trying to update the same target (setting).
When this conflict occurs, each animation overwrites the results of the previous animations and the resulting animation is wrong.

Use the Animation Conflict Warning feature to display a warning symbol when this data is created and a comment indicating the location to fix.
