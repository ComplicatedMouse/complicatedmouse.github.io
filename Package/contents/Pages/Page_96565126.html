Friction Coefficient

This page describes Friction coefficient, a feature that effects speed when a particle collides with a collision.

Overview

Friction coefficient enables you to set a friction coefficient for multiplying speed when a particle collides with a collision.
You can decelerate or accelerate particles when they collide with collisions.
It is applied when Behavior When Colliding is set to Bounce.
The following overview describes the settings ().

Friction Coefficient

You set the bounce coefficient for multiplying speed during a particle collision using the dial () or text box ().

Values smaller than 1 decelerates particle speed after bouncing.
Values larger than 1 accelerates particle speed after bouncing.
A value of 1 maintains the same speed after bouncing.

Example Configurations

The following figure compares what happens when the value for Friction Coefficient is changed.

● Light blue 2 Particle bounces with twice the speed when colliding with a collision.
● Green 1 Particle bounces with the same speed when colliding with a collision.
● Red 0.5 Particle bounces with half the speed when colliding with a collision.
● White 0 Particles stop at the collision plane.
