Random Flip

This section describes the feature for setting whether to flip the texture coordinates when particles are emitted.

Overview

The Random Flip feature is for setting whether to flip the texture coordinates of individual particles when they are emitted.
You can enable or disable this feature for the u-coordinate and v-coordinate separately.
By registering a texture with no directionality for an effect where numerous particles are all emitted at the same time, you can make the texture look like different textures.

Info

This feature can be used together with texture pattern animations.
If you use anything other than Random as the Playback Method for the texture pattern animation, the animation will flip in every frame.
Related Links For information about texture pattern animation playback methods, see Playback Method.
