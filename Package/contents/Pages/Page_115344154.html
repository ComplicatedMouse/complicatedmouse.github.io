LayoutEditor

This section introduces UI elements newly added to LayoutEditor.

Added a button for generating segment tags and configuring settings for unspecified feature animations.

This feature simplifies feature animation and segment tag settings.

Click the Create and Set Segment Tags for Unconfigured Feature Animation button to open the window for creating and setting segment tags for use with a feature animation.
