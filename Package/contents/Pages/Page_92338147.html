Introduction

Purpose of This Documentation

This section describes the compiler and the tool chain Toolchain for NX used with NintendoSDK NX Add-On.

Toolchain for NX customizes LLVM/Clang for the NX and uses musl customized for the NX as the standard C library.
This documentation handles information for external documents for these toolchains and descriptions of the portions customized for the NX.

Intended Audience

This documentation is intended for developers who develop applications using NintendoSDK NX Add-On.
