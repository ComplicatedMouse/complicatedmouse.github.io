Workflow

This section describes the workflow from starting FcpxMaker and creating a composite font (FCPX), to completing the task.
The following subsections describe the procedures involved in performing a task.


1 Starting the Task Start FcpxMaker and begin the task.
2 Adding Nodes

Select the parent node from the Tree View, and then add child nodes using the Add button.
3 Property Editing

Select a node and edit the various items from the properties window, as needed.
4 Save

Save the file.
5 Completing the Task This completes the work in FcpxMaker.


1. Starting the Task

Start FcpxMaker to begin the task.

Starting FcpxMaker

To start FcpxMaker, double-click the following startup file.

\Tools\Graphics\LayoutTools\FcpxMaker.exe

You can also start FcpxMaker from the command line.
Enter the command-line arguments according to the following format.

FcpxMaker.exe <FCPX file path>

2. Adding Nodes

Add nodes in the Tree View to create the hierarchical structure of composite font definition (FCPX) files.
The Add button is enabled when child nodes can be added to a selected node.
Click the Add button to select the type of node to be added from the Add dialog box.


3. Editing Properties

When a node is selected, the properties corresponding to that node are displayed in the properties window.
Use the UI of the properties window to edit the parameters.
If an invalid parameter is set for a property, the node name is displayed in red in the Tree View.


4. Save

To save the composite font definition (FCPX) file, select Overwrite save or Save as from the menu.


5. Completing the Task

This completes the composite font (FCPX) creation process.

Exiting FcpxMaker

To exit FcpxMaker, perform one of the following.

  • From the main menu, select File > Exit.
  • Use the ALT+F4 shortcut key.
  • Click the Close () button in the upper-right corner of the main window.

A confirmation dialog box is displayed when an unsaved file exists. Click the Save as button in the dialog box when you want to save the file.
