Embellishing the Flames

This section describes the procedure for adding emitters to an existing emitter set.
Related Links Making Flames shows an example of adding billowing smoke to created fire effects.
The effect expression can be improved by adding multiple emitters and adding the elements necessary for flame expression.

Adding a New Emitter

Use the following procedure to add a new emitter.

1
Select the Fire_04 emitter set () in the workspace.
2
From the main menu, select File > New > Emitter ().
3.
Enter Smoke as the emitter name in the dialog box that appears, and then click OK.

Specifying Emission and Initial Velocity Settings

Adjust the number and motion of the particles to create a smoke movement.

Selecting the Emission Tab

In the Property window, from the tab (), select Emission. The timing at which particles are emitted from an emitter and initial velocity settings are displayed.

Configuring the Emission Tab

Set the following items ().
The smoke from the heat of the flame can be shown to rise.

Set the following items () in the emission setting ().

Item Value Description
Emission interval 2 Particles are emitted from the emitter at 2-frame intervals.

Set the following items () in the initial velocity setting ().

Item Value Description
All-Direction Initial Velocity 0 The initial speed passed to the particle with the default settings is OFF.
Y-Axis Initial Diffusion Velocity 0.07 The particle is emitted so as to disperse with respect to the y-axis. The result is the representation of smoke that slows down and then disperses.

Set the following items () in the other movement settings ().

Item Value Description
Air Resistance 0.96 The particles emitted in the direction of the y-axis from the emitter slow down and eventually stop because the air is colder higher up.
Gravity 1 Gravity is applied to the particle speed to match the movement of the rising flames.
Direction Y = 1.0 This changes the direction of gravity to upward. This can be used to express smoke rising in an updraft.

Particle Settings

Specifies when the emitted particle disappears.

Selecting the Particle Tab

In the Property window, from the tab (), select Particle. The properties for particles emitted from a particle emitter are displayed.

Configuring the Particle Tab Settings

Set the following items () in the Particle Lifetime setting ().

Item Value Description
Lifespan 80 Particles disappear on the 80th frame after they are emitted from the emitter.
Lifespan randomness 50 (%) The Lifespan value is used as the maximum, and the following formula is used to calculate the minimum.
80 - (80 x 0.5) = 40
Causes the particle lifespan to vary randomly between 40 and 80 frames.

Specifying Combiner Settings

From the presets, select the texture, color, and alpha setting combiner method.

Selecting the Combiner Tab

Setting the Combiner Tab

Set the following items () in Color / Alpha Combiner ().

Item Description of Settings Description
Texture 1. Channel = 1.0 Fixes to 1.0 (white) without using the RGB channel side of texture 1.
Color Combiner C0 × Comb_RGB Multiplies the texture color by the value of Color 0.
Alpha Combiner Comb_A × A0 × A1 This formula multiplies the animation for transparency set with A0 and the value set with A1 by the texture alpha value.

Scale and Rotation Settings

This demo creates a particle-scale animation to portray smoke that slows and dissipates as it rises.
To also add some fluctuation to the motions of the smoke, the demo adds a rotation animation that rotates slower than the one used for the Flame effect.

Changing the Rendering Order

Change the render order of emitters in the workspace.
Use the following procedure to rearrange emitter nodes, while reviewing changes in the viewer.

1

In the viewer, the flames ( Main) are hidden behind the smoke ( Smoke).

Select the Main emitter () within the Fire_04 emitter set () in the workspace.
2
Hold down the ALT key and drag the Main emitter () to a new position under the Smoke emitter ().
In the viewer, the flames ( Main) appear in front.

Related Links For more information about rearranging emitters, controlling render order with different operations, and descriptions with actions, see Render Order Controls for Emitters.
