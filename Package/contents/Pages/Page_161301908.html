Window Pane

The Window Pane window provides utilities for editing window panes in LayoutEditor.

Window panes are used to create window objects by combining multiple textures.
Window panes include an outer portion called the “frame” and an inner portion called the “contents”. Images can be set to both of these portions.

Frames

Enter information about frames.

Share Top Left Material

This setting specifies to use the upper-left settings to configure materials for all frames and for horizontal content.

Clear the check box to disable this feature.

This setting can be enabled when Number of Textures is set to a value greater than 1.

Copy the material information for the upper-left corner

The material settings for the upper-left corner can be copied to other locations.

  • All frames
  • All frames and content

If you select one of these copy targets, a window for selecting what to copy will be displayed.

Select the check boxes next to the items to copy and then click OK.

Vertex Color

Configure the vertex color settings for the content area.

 Simplified figure of a pane used to set the position to enter vertex color.
Click the position you want to specify color for in the figure. The Color Picker opens.
Use the Color Picker to specify the vertex color.

Set all to white

Specifies white (for RGB display, R:255, G: 255, B: 255, A: 255) for all vertex colors of panes being edited.
The default value of vertex colors is white, so this option can be used to reset editing.
This option will not be set If the Change RGB option () is not selected.

Change RGB ()

Select this option to edit RGB.
If this option is not selected, edits of RGB will not be applied.
Change A ()

Select this option to edit A (alpha).

If this option is not selected, edits of A will not be applied.

Content Area

Settings for the content area.
The following settings are the same ones displayed for Total Number of UV Coordinates, Mapping Settings, and UV Coordinates on the texture page.

Do not render content region

Select this check box if you do not want the content area to be drawn.

This results in no memory consumption for the GPU file process, unlike the previous method of setting the color alpha value to zero.
Degree of Magnification Enlarges only the pixels specifying the content part vertically and horizontally.
Number of Coordinates to Use Sets the number of texture coordinates to use in the pane.
Coordinates to Edit Sets the texture coordinates to edit.
