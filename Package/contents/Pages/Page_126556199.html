Creating Kerning Information (BFFKN) Files

Question

How do I create kerning information files (BFFKN)?

Answer

Currently, there is no way for individual users to create kerning information (BFFKN) files. The format is not publicly available.
If you are outputting a bitmap file, the converter manages the generating of the kerning information.

Do the following to embed kerning data into a font.

  • Create a font from a Windows font.
  • Export an image from the Windows font. (This exports a BFFKN file of the kerning within the Windows font.) Then edit the image and create the font again.

Info

Note that kerning information is not exported for Windows fonts that do not have kerning information in the first place.
