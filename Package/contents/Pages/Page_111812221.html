Making Fireworks

This tutorial describes the procedure to create the following fireworks effect using emitter hierarchies.

This tutorial describes the following UI operations.

Adds preset one level down

By registering often-used emitters as presets, you can use this UI to add them below a certain level in the hierarchy.

Batch edit nodes

 Edits multiple nodes to use the same settings among multiple emitters.

Preparing to Create the Effect

First, prepare to create the effect by starting EffectMaker and EffectViewer.

Related Link For more information about starting EffectMaker and EffectViewer, see Workflow.

Loading the Emitter Set

Load the effect into EffectMaker using the following procedure.

1
From the main menu, select File > Open ().
2

When a window opens, select and open the emitter set file (ESET file) from the following location.
\Samples\Sources\Resources\Vfx\Eset\Fireworks_Template.eset

3.

Loads the Fireworks_Template file () in the workspace.

Related Links For more information, see the following pages.

Workspace

Selecting the Camera Tab

In the Workspace window, select the Viewer node ().
In the property window, select the Camera tab.
The camera-related properties for the viewer are displayed.

Configuring the Viewer's Camera

Set the following items () for View and Projection () to make the effect easy to verify.

Setting Value Description
Position of the camera X,Y,Z = 0,30,100 Adjust the position of the camera and the look-at-point so that the effect is fully contained within the viewer screen.
Look-at point X,Y,Z = 0,20,0

Adding Fireworks Parts Lower in the Hierarchy

From the preset folder, load the emitters that become the fireworks effect parts below the Fireworks () emitter hierarchy.

Changing the Preset Folder Path

Use the following procedure to change the preset folder path from the default path, to the path where the presets for this tutorial are located.

1

From the main menu, select Tool > Options ().
2 In the opened Options dialog box, select User Settings in the navigation pane.
3.

Click the Load button () in the preset folder path (), and specify the following location in the window that opens.
\Samples\Sources\Resources\Vfx\Eset\Presets
4

Click OK () to confirm the settings and close the window.


The location for opening the presets, changes from the default location, to the location for this tutorial's presets.
Click the RESET button () to return this to the default path.

Related Links For more information, see Preset Folder Path Settings.

Adding Emitters to the Hierarchy From Presets

Use the following procedure to load parts for the fireworks effect to the hierarchy below FireBall (), directly under Fireworks_Template ().

1
In the workspace, in the hierarchy below the Fireworks () emitter set, right-click Fireball ().
2
From the displayed menu, select Adds a preset one level down () > Fireworks_Presets () > FlareLine (), in that order.
3.

Click the OK button () in the dialog box that displays to load FlareLine () in the hierarchy below FireBall ().
4

Use the same procedure to select and add the other presets TailSmoke () and Firework () to the hierarchy below FireBall ().

Completing the procedures to this point creates three child emitters below the FireBall () hierarchy, as follows.

 FlareLine Effect for a fireball flare.
TailSmoke Effect for smoke billowing from a burning fireball.
Firework Effect for an exploding firework.
Info
FlareLine (), TailSmoke () and Firework () in the hierarchy below FireBall () are called child emitters.
FireBall () is called the parent emitter for the child emitters.
Related Links For more information, see Child Emitters.

Changing Multiple Inheritance Settings

You can configure settings to be inherited by multiple child emitters.

Selecting Multiple Nodes

Use the following procedure to select all child emitters in the hierarchy below FireBall ().

1
In the workspace, select FlareLine ().
2
Press SHIFT and click Firework () to select all child emitters.

Selecting the Basic Settings Tab

With all child emitters in the selected state, select Basic Settings ().

Configuring the Basic Settings Tab

You can configure the Inheritance Settings () for multiple child emitters.
The check boxes are cleared () for all of the following settings items (), disabling them all.

Setting Description
Alpha0 Do not inherit the parent's alpha value and use its own alpha value.
Alpha1
Scale Do not inherit the parent's scale value and use its own scale value.
Rotate Do not inherit the parent's rotation value and use its own rotation value.

Related Links For more information, see Selecting and Editing Multiple Emitters Simultaneously.

Changing Child Emitter Inheritance Settings

Change the inheritance settings for the FlareLine () child emitter.

Selecting FlareLine

Use the following procedure to select only FlareLine ().

1
In the workspace, select the parent emitter FireBall () and clear the selections for child emitters.
2
Select FlareLine ().

Selecting the Basic Settings Tab

Select the FlareLine () child emitter, and then select Basic Settings ().

Configuring the Basic Settings Tab

Configure the inheritance settings () for the FlareLine () child emitter.
Change the check boxes to be selected () for the following settings items ().

Setting Description
Emitter Assignment for Particles
The FlareLine () child emitter follows the particles emitted from the FireBall () parent emitter.
Draw first before higher level
The FlareLine () child emitter is drawn before the FireBall () parent emitter.
Color Scaling
The color scaling of the FlareLine () child emitter is multiplied by the color scaling of the FireBall () parent emitter.
Color 0 Color
Color 0 of the FlareLine () child emitter is multiplied by Color 0 of the FireBall () parent emitter.
