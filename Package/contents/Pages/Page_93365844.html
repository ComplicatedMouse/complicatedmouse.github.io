Box Position, Scale, and Rotation

This section describes the features for configuring the box position, scale and rotation.

Overview

The Box Position, Scale, and Rotation feature enables you to configure the transform information for the box in world coordinates.
Configuring these settings enables you to set the range in which particles display using the Loop-In-Range feature.
This section provides an overview of the settings ().

Box Position

Sets the position of the box that determines the range in which particles display. The set value represents the distance from the center position in the world coordinate system to the center of the box.

Box Size

Set the size of the box.

Box Rotation

Sets the angle defining how much to rotate the box.
