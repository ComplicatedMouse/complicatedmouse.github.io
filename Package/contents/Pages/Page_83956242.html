Shortcut Menu in the Curve Tree View

The shortcut menu used in the curve tree view differs depending on the type of curve and group.
Generally, the shortcut menu provides selections for copying, pasting, and deleting curves or groups.

Info
Related Links You can also apply the original data for color animations and texture SRT animations exported from the DCC tool to create animation curves.

This section describes the shortcut menu for animations that include unique features.

Shader Parameter, Texture SRT, and Color Animations

The shortcut menu associated with shader parameter, texture SRT, and color animations includes a feature for displaying the material shader page.
Use this feature to jump to the Shader page of the Material Property window for the model selected in the shortcut menu of curve tree view.

Changing the Texture SRT/Texture SRT EX Texture Matrix Mode

You can select the texture coordinate calculation method from the shortcut menu of the texture matrix of a texture SRT or texture SRT EX group.
You can select from the following modes: Maya, 3ds Max, or Softimage.

Texture Pattern Animation

The shortcut menu for texture pattern animations includes a feature for displaying the Material Sampler page.

Only Display Selected Node or Curve

This feature allows you to display and edit the curves that are selected in the curve tree view only.

Show Only the Following Curve

Displays only the curves under the node selected in the curve tree view. The displayed curves can also be edited.


Show Only This Curve

Displays and edits only the curve selected in the curve tree view.

Apply Original

Creates an animation curve by referencing the shader definition file (FSDB file) from the original color animation/texture SRT animation data exported from the DCC tool.

Apply Specific Original Curve

Creates an animation curve by referencing the original data for a specific color animation/texture SRT animation exported from the DCC tool.

Selecting Apply Specific Original Curve displays a dialog box where you can select the original animation to apply.

To apply the animation curve from the original data for the selected animation, select the animation you want to apply and click OK.
