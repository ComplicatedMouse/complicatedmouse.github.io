UV Rotation Animation

This section describes the feature for rotating the UV of the registered texture.

Overview

The UV Rotation Animation feature enables you to animate the rotation of a registered texture by rotating its UV coordinates.
The following overview describes the settings ().

UV Rotation Animation

A feature for setting the angle of the texture UV coordinates on each axis separately.

Select this to display the UV rotation animation () settings. The group has the following settings.

Add-on value

Sets the value to add to the texture UV rotation angle in every frame.
This add-on value is applied after the emitter starts.

Initial value

Sets the default initial value for the texture UV coordinate rotation angle.

Random initial values

Specifies the use of random numbers in a set range for the initial values.
The upper and lower limits of the random seed for the initial value randomly change. The following formula calculates a lower limit for the random number.

A Add-on value

An add-on value of 1 for every frame is set in the negative ( - ) and positive ( + ) directions.
When the value is set in the negative direction, the texture rotates clockwise by one degree in each frame. When set in the positive direction, it rotates counterclockwise.
B Initial value A value of 45 degrees is set in the negative ( - ) and positive ( + ) directions.
When the value is set in the negative direction, the texture tilts clockwise by 45 degrees. When set in the positive direction, it tilts counterclockwise.
