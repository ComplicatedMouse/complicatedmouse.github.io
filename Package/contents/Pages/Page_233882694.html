nn::ui2d News ()

Fixed a bug where gamma conversion did not work correctly for the font's shadow color and the like when degamma-parameter was specified.

Fixed a bug where gamma conversion did not work correctly for the font's shadow color and for the per-character transformation color when the degamma-parameter option was specified in LayoutConverter.

Note that as a result of this bug fix, any adjustments that were made to the layout data to achieve correct display may now have the opposite effect and cause incorrect display. (The degamma-parameter option is specified so that you can use ported products and other past layout data that was created on the basis of an SRGB gamma workflow.)
