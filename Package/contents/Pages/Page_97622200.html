Scale

This section describes the feature for applying an offset to the scale of the emitter.

Overview

Scale configures an offset to the scale of the emitter using the world coordinate system.
Because this setting affects the size and speed of both the emitter and the particles, you can use it to modify the overall size of the effect.
The following table shows the properties to which scale is applied.

Emitter Size Particle Size Particle Speed
Yes Yes Yes

The following overview describes the settings ().

Scale
