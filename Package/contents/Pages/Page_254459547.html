Sequential Frames per Pattern

This section describes the Sequential frames per pattern feature that specifies the number of frames displayed per pattern.

Overview

Sequential frames per pattern is a feature for specifying the number of frames displayed per pattern.
The following overview describes the settings ().

Frames Displayed per Pattern

Use the dial () or text box () to set the number of frames displayed per pattern.

Example Configurations

The following image compares what occurs in clamped playback when the sequential frames per pattern value is changed.

A

Sequential frames per pattern = 5

Playback time per pattern (5 frames) / Particle lifespan (100 frames) = Up to 20 patterns can be played.
B Sequential frames per pattern = 10

Playback time per pattern (10 frames) / Particle lifespan (100 frames) = Up to 10 patterns can be played.
