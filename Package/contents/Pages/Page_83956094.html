An overview of sound sets and a description of the Sound Set tab are presented.

Overview

Role of the Sound Set Tab

The Sound Set tab is the pane in which sound set files (FSST files) are edited.
Use this tab (pane) to create sounds and perform most other SoundMaker tasks.

Sound Set Structure

Sound sets include all elements that are required by a sound archive. Switch the elements that are displayed by selecting different nodes in the Project Tree.
Detailed descriptions of the list elements are provided in later sections.
