Texture Combiner

Configures the merging method for the texture image and color added to the pane.

Switch to Advanced Combiner Mode

Changes the display mode from standard combiner mode to advanced combiner mode.

Stage Number Being Edited

Displays the stage number currently being edited.

Color Blending Settings

Configures the color blending. Performs indirect adjustments (warp/rotation).

Type

Specifies the type of color blending setting.

  • Overwrite
  • Multiplicative
  • Addition
  • Subtractive
  • Exclusion [Brights - darks; invert if white]
  • Color Dodge [Brightens previous stage; whiter ⇒ stronger effect.]
  • Color Burn [Darkens previous stage; darker ⇒ stronger effect.]
  • Overlay [Prev brightness ≥ 128 ⇒ multiply by inverse (brighten). Prev brightness < 127 ⇒ multiply (darken)]
  • Blend (Lighten)
  • Blend (Darken)
  • Indirect [Affects previous stage. Current stage is distortion source.]
  • Indirect [Affects prev stage; use blend of current & next stages as distortion source.]
  • Indirect [Distortion source = Self (stage 3), Target = previous stage 1 and previous stage 2]

Indirect Adjustment

(Can be animated)

Performs indirect adjustments.

  • Amount of warp
  • Rotation
Note
  • More than one simultaneous indirect texture setting is not supported.
  • When using Indirect [Affects prev stage; use blend of current & next stages as distortion source.], configure the second of three textures as an indirect texture.
  • When using Indirect [Distortion source = Self (stage 3), Target = previous stage 1 and previous stage 2], configure the third of three textures as an indirect texture.
