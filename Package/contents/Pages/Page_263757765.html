Introduction

This section describes state transitions within the nfc library.

The nfc library uses the following two types of states. To use the nfc library, you must properly understand these states.

  • nfc Library States
  • NFC Device States

State Definitions

The following states are defined for the NFC device.

Table 10 NFC Device States
State Description
DeviceState_Init Not currently searching for tags. This state occurs immediately after the nfc library is initialized.
DeviceState_Search Currently searching for tags.
DeviceState_Active A tag has been detected.
DeviceState_Deactive A tag was detected, and then the detected tag was removed.
DeviceState_Keep The tag session is maintained in this state.
DeviceState_Unexpected

The device has entered an unexpected state. This can occur if an unconnected device has been specified.
