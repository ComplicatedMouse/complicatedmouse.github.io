Capture Effect (Mask)

This section describes the Capture Effect (Mask) feature for clipping using a texture image registered to a pane.

Overview

Capture Effect (Mask) () uses the registered texture file as the image for the capture effect mask.
You can use this feature to clip the mask image to the shape of the pane, or conversely to use the alpha value of the mask image to clip the pane.

Restrictions

The following are restrictions for Capture Effect (Mask).

  • Display Restrictions
    • The display of the Layout window is not updated.
      To check the display, verify in LayoutViewer.
  • Runtime Restrictions
    • If the drawing result of the Capture Effect (Mask) feature is captured by the capture pane, or if the Capture Effect (Mask) feature is applied to a pane that uses a texture created in the capture pane, you can only get the correct effect if the actions are done in the same layout.
  • Using With a Pane-Size Animation
    • The capture effect uses the capture function, so on the included LayoutViewer, create a capture texture with the pane size of the starting frame. (The roughness is noticeable if enlarged.)
