6. Saving Data

This section describes how to save data after creating it.
There are various methods for saving.

This tutorial involves the following UI.

Save All in Specified Location Saves all of the files opened in 3DEditor, including the edited content, to the specified location.
Save Model Saves any model.
Save Material Animation Saves any material animation file.
Save Project File Saves the work in the 3DEditor.

Saving All in Specified Location

Saves all of the editing states.

From 3DEditor, select File > Save All in Specified Location. To save the content, select the folder to save to and click OK.

Info

Use Save All instead of saving individually.

During standard work, to update all of the edited content without any omissions, use File > Save All.
(To avoid overwriting the sample model, select Save All in Specified Location.)

Saving the Model

Saves the model. This includes the material settings.

In Object view, from the Schematic display, right-click SampleTripod.fmdb at the top of the model tree.
From the shortcut menu, select Save As.
Name the file and select a location to save it to.

Info

To update the file during standard work, on the shortcut menu, select Save.
(To avoid overwriting the sample model, select Save As.)

Saving the Material Animation

Saves the created material animation.

In File view, right-click Eye.fmab and select Save As.
Name the file and select a location to save it to.

Saving the Project File

Saves the work in 3DEditor.
You can restore the work state by loading the project file without having to load the individual files.

In File view, right-click Project.f3pj and select Save As.
Name the file and select a location to save it to.

Info

Saves the File view status.
The editing content of the materials and animations is not saved.
