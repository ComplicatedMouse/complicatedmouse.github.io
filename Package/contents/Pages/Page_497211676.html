LayoutEditor

This section introduces the new features in LayoutEditor.

Improved the feature for adjusting the time scale in Curve Editor.

You can now adjust the time scale in Curve Editor. In addition, the key tilt during the scale can also be adjusted.

Info

Two or more keys must be selected for each curve to adjust the time scale.

Runtime Libraries

This section introduces the new features added to the layout runtime.

Panes that use the Capture Effect feature can now be captured using the capture pane.

You can now capture panes that use the Capture Effect feature in the capture pane.
If the user uses a texture created in the capture pane in a picture pane and if the Capture Effect feature is applied to a pane, you can only get the correct effect in the same layout.

In the following images, the capture pane is used to capture a text box pane to which a Capture Effect was applied, and then a Capture Effect is used to apply a mask ().
Applying capture effects and capture panes to text box panes is an effective way to support multiple languages because you can change the content of the strings while maintaining the same character rendering effects.

1

Add a text box pane () with an border to the capture pane ().
2

Add the capture texture created in step 1 to a picture pane ().
Now also apply a mask using the Capture Effect feature ().

Info

The overwriting parts with capture textures does not render correctly if the process straddles layouts.
