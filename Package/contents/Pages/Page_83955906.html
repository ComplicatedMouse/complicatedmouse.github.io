This tab configures aspects of emitter sets that can be controlled in the game application, and settings linking 3D models and effects.

Matrix Settings

The Matrix Settings group () of features configure how scale is applied to particles.
This group has the following settings.

Matrix Settings Type　

Sets how scale is applied to particles.

Model Links

The Model Links group () is a set of features for linking effects with polygon models registered in EffectMaker.
This group has the following settings.

Matrix Application Type Determines whether to constrain the emitter set to the Rotate and Scale of the bound bone.
Model to Constrain Selects which model to link from among the model files registered in the Model Settings node.
Bone to Constrain Selects which bone you want to emit the effect from the linked model selected in Model to Constrain.

Blue For more information, see the following pages.

Combining Flames With a 3D Model Describes how to combine an animated 3D model with an effect.
Model Settings Nodes Contains model file and animation file settings to verify on the viewer.

Emitter Set Matrix

The Emitter Set Matrix group () is a set of features for offsetting all emitters in the emitter set using the value set in Emitter Transform.
Use it when you want to change the overall scale of the effect.
You can also use Emitter Set Matrix to apply an offset to the position from the bone to bind to when linking to a model.
This group has the following settings.

Coordinates (±) Applies an offset to the position of the emitter.
Emitter Billboard Orients the emitter toward the direction of the camera.
If set to No, you can set the angle of rotation of the emitter around each axis.
Scale Multiplies the value set for the emitter scale.
Info

The coordinate system in which to process the Emitter Set Matrix settings differs in the following way for emitters and particles:

Emitters Processed in world coordinates.
Particle Processed in emitter coordinates.
