Creating a New Sound Project

1. Overview

This section describes the ways to create a new sound project.

For more information, see Creating Sound Projects in the Basic Guide for SoundMaker.

There are three main ways to create a sound project. They differ by how you create the sound set to register to the project.

  • 2.1. Creating a New Sound Set
  • 2.2. Specifying an Existing Sound Set
  • 2.3 Empty Sound Project

2. Examples

2.1. Creating a New Sound Set

This method creates a sound project to which an empty sound set is registered.

1. Select Files > Create New Project.

2. Set the name and the path for the project, select Create a new sound set and click Next.

3. Select Complete to create a sound project with an empty sound set.

2.2. Specifying an Existing Sound Set

This method creates a sound project to which an already created sound set is registered.

1. Select Files > Create New Project.

2. Set the name and the path for the project, select Specify an existing sound set and click Next.

3. Specify the path to reference for the sound set and click Next.

4. Select Complete to create a sound project that references an existing sound set.

2.3 Empty Sound Project

Creates a sound project that does not reference any sound set.

1. Select Files > Create New Project.

2. Set the name and the path for the project, select Empty project and click Next.

3. Select Complete to create a sound project that does not reference a sound set.
