Contains settings for Viewer debug features.

Viewer Control

The Viewer Control group () is a set of features for controlling the connected viewer.
This group has the following settings.

Playback Speed Use this feature to change the playback speed on the viewer by multiplying the speed by the specified value.
Show Ground Features related to the ground shown in the viewer.
Glare Intensity This feature adjusts the glare post-effect embedded in the viewer.
Overlap Check for Particles

Feature that enables the checking of particle overlap.
Show Debug Information A feature to hide the effect information containing Show GPU Metrics.
Show GPU Metrics A feature that shows the details of the GPU load.
Show Parent Emitter Shape A feature that shows the emitter shape and emitter axis in the Viewer.
Show Child Emitter Shapes A feature that shows the child emitter shape and emitter axis in the Viewer.

Display Field Information

A feature that shows field and emitter plug-in node information in the viewer.
Use PC GamePad A feature for selecting whether to use the PCB GamePad and controller to control the EffectViewer.

Binary Transfer

The Binary Transfer group () is a set of features for loading the exported binary file to check how effects look in the viewer.

Related Links For more information, see the following pages.
