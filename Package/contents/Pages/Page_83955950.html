Specifying Emission and Initial Velocity Settings

Adjust the number and motion of the particles to create a flame-like movement.

Selecting the Emission Tab

In the Property window, from the tab (), select Emission.
The timing at which particles are emitted from an emitter and initial velocity settings are displayed.

Configuring the Emission Tab

Set the following items in the emission setting ().

Item Value Description
Emission interval 1 Particles are emitted from the emitter at every frame.

Set the following items in the other movement settings ().

Item Value Description
Air Resistance 0.95 The particles emitted in all directions from the emitter lose speed and eventually stop.
Gravity 1 More rising energetic fire is made by applying gravity to the particle speed.
Direction Y = 1.0 This changes the direction of gravity to upward.
This can be used to express flames rising due to an updraft.

Related Links For more information about editing values, see Editing Effect Data.

Particle Settings

Specifies when the emitted particle disappears.

Selecting the Particle Tab

In the Property window, from the tab (), select Particle. The properties for particles emitted from a particle emitter are displayed.

Configuring the Particle Tab Settings

Set the following items () in the Particle Lifetime setting ().

Item Value Description
Lifespan 50 Particles disappear on the 50th frame after they are emitted from the emitter.
Lifespan randomness 30 (%) The Lifespan value is used as the maximum, and the following formula is used to calculate the minimum.
50 - (50 x 0.3) = 35
Causes the particle lifespan to vary randomly between 35 and 50 frames.

Scale and Rotation Settings

This example creates a particle-scale animation to portray a fire that flares up brightly, and then burns out.
It also adds a small rotation animation to each particle to add a fluctuation to the movement of the flame.

Selecting the Scale Tab

In the Property window, from the tab (), select Scale.
The settings for particle tilt and rotation animation are shown.

Setting the Standard Value for the Particle Size

Set the particle size during emission using the following items () in Basic Scale ().
The configured values are used as standard values when applying a scaling animation to particles.

Item Value Description
Standard value X,Y,Z = 5 Sets the base size of particles during emission.
Randomness X,Y,Z = 30%

Establishing Randomness for the Standard value avoids particles from being emitted with a constant size.
This causes the particle size to vary randomly in the range between 70 percent of the standard value and the standard value.

Animating the Particle Scale

Use the following items () in Scale (Particle Time Animation) () to animate the particle size.

Related Links For more information about the operations for setting animations, see Curve Editor Operations.

Item Time (f) Time (%) Value Description
Scale Animation 0 0 1.0

Sets the value during emission to 1.0. The standard value is applied directly, unchanged.
25 50 1.0

Sets the value halfway through the lifespan to 1.0. This maintains the same size at the time of emission.
50 100 0.5

Sets the scale value to 0.5 in the final frame.
Matching the transparency animation creates an impression of a reduction in combustion energy.

Selecting the Rotation Tab

In the Property window, from the tab (), select Rotation.
The settings for particle tilt and rotation animation are shown.

Animating Particle Rotation

Set a rotation animation around the z-axis of particles using the following items () in Rotate Settings ().

Item Value Description
Rotation Axis
 		Z-axis ON This makes it possible to set particle rotation around the z-axis only, so configure as follows.
Random Spin Direction ON The direction of rotation during emission is varied randomly between clockwise and counterclockwise rotation.
Rotation speed(±)
4

Specify the random range for the Rotation speed(±) value of the rotation animation of particles.

Particles are animated by adding random rotation values in the range of 0 to 4 degrees in each frame.
The flame can be seen as fluttering by having multiple particles overlap.
