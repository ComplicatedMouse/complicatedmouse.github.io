Translation Speed

This section describes the Translation Speed feature for setting the speed of travel with the Translate Straight in Specified Direction auto-translation type.

Overview

Translation Speed sets the translation speed when Move forward in the specified direction is selected as the auto-translation type.
The speed of translation can be increased by increasing this value.

The following overview describes the settings ().

Translation Speed

Use the dial () or the text box () to specify the speed of automatic translation.
