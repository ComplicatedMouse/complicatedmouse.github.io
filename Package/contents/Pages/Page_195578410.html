Window

This page describes the Window menu in LayoutEditor.
Workspace Fix Docking Locks the state of docking windows.
Do Not Tab Layout Windows When multiple layout windows are being displayed, they will be shown in separate windows, and not as tabs.
Hide Toolbars Hides the toolbars in the LayoutEditor window.
Save Workspace Saves the state of the current workspace (the arrangement of sub-windows).
Restore Workplace Restores the saved workspace.
Reverts the workspace to the default state. Reverts the saved workspace to the default state.
Curve Editor Opens the Curve Editor window.
Textures Opens the Textures window.
Group Opens the Group window.
Hierarchy Opens the Hierarchy window.
Pane Information Paste Controls Opens the Pane Information Paste Controls window.
Editing Animation Segment Tags Opens the Edit Animation Segment Tags window.
Search for Panes Opens the Search for Panes window.
All Property Windows

Opens property windows.

If a pane is currently selected, this option only opens the property window for that pane.
Layout Settings Opens the Layout Settings Window.
Color Sample Opens the Color Sample window.
Animation Sharing Information Opens the Animation Sharing Information window.
Parts and Control Settings Opens the Parts and Control Settings window.
Parts Palette Opens the Parts Palette window.
