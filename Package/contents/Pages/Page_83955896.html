Emitter Set Node

This page describes Emitter Set Node features.

Basic Settings This page describes the Basic Settings tab used to view file information and configure basic settings for emitter sets.
User Data This page describes the User Data tab that sets additional user-specific information for emitter sets.
Info

This node has the following characteristics.

  • A number of emitter nodes and preview nodes can be gathered together in the workspace.
  • Makes an emitter set file that can be loaded in EffectMaker and saved as a single set of effect data.
  • There is always at least one emitter node and one preview node.

For more information about emitter set files, see Blue.
