Animation Sharing Information Window

Animation Sharing

Considering the case of creating layouts such as a game's menu screen, it is not uncommon to perform the same animation operation with many items.
For example, consider the following cases.

  • Animation that plays when the cursor collides with the Menu button
  • Animation that plays when a determination is performed using the Menu button

The animation sharing feature functions to efficiently create these kinds of data.
You can use the animation sharing feature to make many duplicates of the same animation and share animations without having to decide among developers and perform special processing.

Info

This feature cannot be confirmed by LayoutEditor.

Animation Sharing Information Window

This window is used to specify animation sharing.
To display the animation sharing information window, use the following command.

  • From the menu, select Window > Animation Sharing Information.

The following table describes the animation sharing information window UI.

Animation Sharing Information Window
A New Opens the dialog box for registering animation sharing information.
B Edit Opens the dialog box for registering animation sharing information.
C Create Sharing Group Opens the dialog box for creating a group for animation sharing.
D Highlight sharing group
E Delete Deletes the selected animation sharing information from the list.
F Animation Sharing Information

Dialog Box to Register Animation Sharing Information

The following image shows the UI for the animation sharing information registration dialog box to set the shared source pane and the target group for sharing.

Dialog Box to Register Animation Sharing Information
A Shared Source Pane Select the shared source pane from the drop-down list.
B Sharing Group Select the group that will do the sharing from the drop-down list.
