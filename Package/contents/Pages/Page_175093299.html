Emission Intervals for Emission Interval Type - Distance

Emission interval

This section describes the emission interval feature, which sets the interval at which an emitter emits particles in terms of the emitter’s movement distance.

Overview

The emission interval feature allows the particle emission to be set in terms of the emitter’s movement distance.
Setting the emission interval be in terms of distance rather than time enables you to set the particle emission interval independently of the emitter's movement speed.

The following overview describes the settings ().

Emission interval

Uses distance to set the emission interval for particles.
For example, if the emission interval is set to 1, the particle emission interval (distance) is 1.
