Emitter Node

This section describes the features of the Emitter node.

Basic Settings Configures the basic settings for the selected emitter.
Emitters Configures the emitter shape and transforms.
Emission Configures particle emission.
Particle Configures particles.
Combiner Configures particle appearance.

Textures

Configure how textures are applied to particles.

Color

Settings for the color and alpha of the effect.
Scale Configures the scaling of particles.
Rotate Configures settings related to particle rotation.
Cstm Shader Configures settings for the user-created shader when the option has been used to specify a custom shader definition file.
Info

There are three emitter behavior types, CPU (), GPU (), and GPU+SO ( ), for emitter node display in the workspace.
