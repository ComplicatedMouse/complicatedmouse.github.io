File path

This section describes the feature for checking the path to where an emitter set file is saved.

Overview

The File Path feature allows you to check the save destination path for the emitter set file selected in the workspace.
The following overview describes the settings ().

File path

The destination file path is displayed in the text box.
To copy the file path as a text string, right-click it and select Copy from the shortcut menu.
