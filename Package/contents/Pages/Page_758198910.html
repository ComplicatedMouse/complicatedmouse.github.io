Threads Tab

The Threads tab displays all of the threads that were sampled during a profile.

Note

These notes point out differences in the Threads tab when the Thread-Centric Data Views experimental feature is enabled. See Experimental Features Tab for additional information.

Note

The Thread rows aggregate data for all of the cores that the thread was observed on. Thread rows may be expanded to reveal the per-core information. If a thread has only been sampled on a single core, the thread row will not have any cores rows underneath it and will not be expandable.

Sorting Threads

Threads can be sorted by Self, Core, ID, or Name by clicking on each respective heading. By default, the threads are sorted by Core, and then sorted by Self within each core.

Threads Tab Toolbar

The following controls are in the Threads tab toolbar.

Hierarchy Buttons

Note

These buttons are only available when Thread-Centric Data Views are enabled.

The buttons listed below control the expanded state of the items in the list.

Expand All Button

The  Expand All  button will expand all rows.

Collapse All Button

The  Collapse All  button will collapse all rows.

Expand Selected Only Button

The  Expand Selected Only  button will expand all rows necessary to display all of the selected rows and collapse all other rows.

Cores Drop Down and Core Column

Click on the Cores drop-down in the toolbar to select between All Cores and individual cores. Only cores that were profiled will be available in the Cores drop-down. The default selection is All Cores and each new profile taken or loaded will reset the Cores drop-down selection to All Cores.

When a particular core is selected, such as Core 1, the Core column will be hidden since all functions shown belong to the same core. This can be helpful when focusing on a particular core and can help optimize horizontal space in the tab.

Deselect All Button

To quickly deselect all selected items, click on the Deselect All button.

Top Button

Note

This button is not available when Thread-Centric Data Views are enabled.

The Top button selects the first 50 items in the list.

Copy Button

The Copy button will copy to the clipboard whatever is currently shown using the current sorting and filtering. You can then paste the content into an email, document, or Excel spreadsheet. Because the copied content uses tab delimiters, it will retain the column organization when pasted into an Excel spreadsheet.

Selecting Threads

Click on a thread to select it and click again to deselect it. Selected threads will be graphed in the Timeline tab.

To quickly select the top 50 threads, click on the button marked Top in the Threads toolbar.

To quickly deselect all threads, click on the Deselect All button in the Threads toolbar.

Note

All of the core data for a thread is selected or deselected as a group.

Alternative Selection Methods

Multiple items can be selected or deselected with Shift+Click (select/deselect an item without holding the Shift key, then hold the Shift key and click on a second item). The range between the two clicks will be either selected or deselected.

To quickly select only a single item and deselect everything else, Alt+Click can be used. This deselects all and then selects the clicked upon item.

Filtering Threads

Threads can be filtered using the Filter text box at the bottom of the Threads tab. This behaves similar to the Filter text box on the Functions tab.
