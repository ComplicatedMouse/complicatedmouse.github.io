LayoutEditor Window

A list of the shortcut keys for the LayoutEditor window.

Operation Corresponding Menu Item Description
File

CTRL+O File > Open Opens saved layout files.
CTRL+S File > Save Saves the layout windows.
CTRL+SHIFT+S File > Save As Saves the layout windows with new names.

CTRL+Q

File > Quit

 Closes the layout windows.
Edit
CTRL+Z Edit > Undo Undoes the last operation.
CTRL+SHIFT+Z Edit > Redo Restores an undone operation.
CTRL+X Edit > Cut Cuts a selection.
CTRL+C Edit > Copy Copies a selection.
CTRL+V Edit > Paste Pastes the selection that was copied. Or pastes that which was cut.
CTRL+SHIFT+V Edit > Paste Without Animations
Edit > Paste With Animations		 The menu text varies depending on the state of animation that is being copied or pasted. Pastes by reversing the selection of Copy/Paste Animation.
CTRL+D

Edit > Duplicate

 Duplicates the selected pane and the panes under its hierarchy.
CTRL+SHIFT+D Edit > Copy Without Animations
Edit > Copy With Animations		 The menu text varies depending on the state of animation that is being copied or pasted. Copies by reversing the selection of Copy/Paste Animation.

DELETE

BACKSPACE

 Edit > Delete Deletes the data.
Panes
SHIFT+F2 Pane > Apply Parent Translations to Children Applies translations of the parent pane to the panes under its hierarchy.
SHIFT+F3 Pane > Adjust the size by dragging the mouse adjustment point

Adjusts the behavior of the mouse adjustment point every time there is input.

When enabled, you can drag the mouse adjustment point to adjust the size.
When disabled, you can drag the mouse adjustment point to adjust the scale.
CTRL+G Pane > Create Group Creates a group.
CTRL+ALT+SHIFT+G Pane > Edit Group Edits the selected group.
CTRL+2 Pane > Lock Selected Panes Locks the selected panes.
ALT+CTRL+2 Pane > Unlock All Unlocks all selected panes.
CTRL+3 Pane > Hide Selected Panes Hides the selected panes.
ALT+CTRL+3 Pane > Show All Hidden Panes Shows all hidden panes.
CTRL+P Pane > Properties Shows the properties.
CTRL+SHIFT+R Pane > Reload Parts Pane Reloads the parts pane.
P Pane > Create Hierarchy Sets the currently selected pane under the hierarchy of the last selected pane.
SHIFT+P Pane > Remove Hierarchy Removes the hierarchy relationship for the selected pane.
G Pane > Create a Null Pane as a Child of the Selected Pane Creates a null pane and sets it under the hierarchy of the selected pane.
Select
CTRL+A Select > Select All Selects all.

CTRL+ALT+LEFT BRACKET ([) Select > Select the pane behind the current one Selects the pane behind the current one.

Layout Window

A list of the shortcut keys for the layout window.

Operation Description
S

Generates a key for the Curve Editor.

Adds a key only for the curve selected in Curve Editor.
Arrow Moves a pane.
SHIFT+ARROW KEY Moves, in grid increments, when Snap to Grid is disabled.
CTRL Temporarily reverses how parent translation is applied to the panes under its hierarchy.
Dragging Between Panes After Hierarchy or Remove Hierarchy Sets or clears the hierarchy.
SPACE+ALT+Clicking Any Mouse Button Except Right Reduces.
SPACE+CTRL+Clicking Any Mouse Button Except Right Magnifies.
SPACE+ALT+Mouse Wheel

Changes the size of the window.

Note: A feature to use when performing operations using the Do Not Tab Layout Windows setting.

SPACE+CTRL+Mouse Wheel

SPACE+CTRL+Click and Horizontal Drag With Left Mouse Button

SPACE+ALT+Click and Horizontal Drag With Left Mouse Button

 Magnifies or reduces.
SPACE+Dragging With Any Mouse Button Except Right Scrolls up, down, left, and right.

Wheel button

(+SHIFT)

Scrolls vertically.

Press SHIFT to increase the amount of change.

CTRL+Mouse Wheel

(+SHIFT)

Scrolls horizontally.

Press SHIFT to increase the amount of change.

ALT+Mouse Wheel

(+SHIFT)

Magnifies or reduces.

Press SHIFT to increase the amount of change.
Clicking a pane Selects the pane.
SHIFT+Mouse click in the pane Reverses the selection for any clicked panes.
SHIFT+Range Selection Reverses the selection for any panes within the range.
Range Selection When Dragging Outside the Pane Selects panes within the range.
Pane Dragging Translates.
ALT+Drag pane Duplicates the selected pane, and the panes under its hierarchy, to the transfer target.
Hold Down SHIFT While Dragging Pane Translates horizontally, vertically, or in both directions.
Dragging Bounding Box Handles Changes size.
Hold Down SHIFT While Dragging a Bounding Box Handle Changes size while maintaining aspect ratio. The size can only be reduced if you grab a top, bottom, left, or right handle.
T+Dragging Bounding Box Handles Translates.
ALT+Dragging Bounding Box Handles Rotates.

Capture Pane
CTRL+ALT+C Shows or hides the capture panes, including panes lower in the hierarchy.
Use CTRL + drag the left mouse button to change the size of the capture pane. Changing the capture area of the capture pane does not affect the positions of the panes lower in the hierarchy.

Curve Editor Window

A list of the shortcut keys for the Curve Editor window.

Operation Description
CTRL+PLUS SIGN (+) Magnifies.
CTRL+MINUS SIGN (-) Reduces.
CTRL+UP ARROW Select the segment tag before the currently selected segment tag.
CTRL+DOWN ARROW Select the segment tag after the currently selected segment tag.
CTRL+ALT+1 to 5 Select the segment tag assigned to 1 through 5.
S Creates a key for the selected curve.
A Displays the entire selected curve.
F Displays all selected keys.

BACKSPACE

DELETE

 Removes a key.
CTRL+R Displays the selected pane in Curve Editor. (Use when automatic display is disabled.)
CTRL+LEFT ARROW Selects the previous key.
CTRL+RIGHT ARROW Selects the next key.
CTRL+1 Sets the slope of the selected key to the spline.
CTRL+2 Sets the slope of the selected key to the linear interpolation.
CTRL+3 Sets the slope of the selected key to zero.
CTRL+4 Sets the slope of the selected key to the step.
Left-click + drag Selects the pane.
Clicking or Dragging With a Mouse Button Other Than the Left Performs curve operations, including movement and adding.
Pressing T With Multiple Selections and Dragging the Operation Frame With the Right or Center Buttons Multiplies the key position scale.
Operating Curve With Ordinary or Nearest Key + Directional Buttons Moves the selected keys.
Operating Curve With Ordinary or Nearest Key + SHIFT + Directional Buttons Makes the movement of the selected key 10 times faster.
Curve Operations for Key Insertion or Addition + Directional Buttons Begins key insertion or addition.
SPACE+CTRL+Left Mouse Button Magnifies. (It does not matter which is pressed first: CTRL or SPACE.)
SPACE+ALT+Left Mouse Button Reduces. (It does not matter which is pressed first: ALT or SPACE.)
SPACE+SHIFT+Horizontal or Vertical Drag With Right Mouse Button Changes the horizontal or vertical view scale. (It does not matter which is pressed first: SHIFT or SPACE.)
SPACE+Left or Center Mouse Button Translates.
SPACE+Horizontal Dragging With Right Mouse Button Changes the view scale.
Mouse Wheel Magnifies and reduces the view.
Dragging Range Sliders With Left Mouse Button Advances time while updating the animation.
Dragging Range Sliders With Center Mouse Button Advances time without updating the animation.

Operation Description

 Displays only the panes under the hierarchy of the selected pane and hides all other panes.
Perform the same operation again to revert to the previous state.
