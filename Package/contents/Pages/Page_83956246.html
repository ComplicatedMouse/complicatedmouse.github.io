Creating Parent/Child Hierarchies

This section describes how to create parent/child hierarchies.
When creating a parent/child hierarchy, you can perform operations such as collecting and transferring the parent/child panes or consolidating and changing attributes like Show/Hide.

Use the following procedure to create a parent/child hierarchy.

1 From the main menu, select Pane > Hierarchy, or right-click and select Hierarchy from the shortcut menu on the pane.

1 From the main menu, select Pane > Remove Hierarchy, or right-click and select Remove Hierarchy from the shortcut menu on the pane.

Panes cannot be set as child panes in the following cases.

  • The pane being set as a child is the parent pane itself.
  • The pane being set as a child is the parent of the parent pane (or any other pane in the parent hierarchy).

Using the Hierarchy Window

In the Hierarchy window, you can display parent/child relationships between panes and edit parent/child relationships.
From the main menu, select Window > Hierarchy, or you can press F10 to open the Hierarchy window.

Creating or Removing Parent/Child Relationships in the Hierarchy Window

Use the following procedure to create a parent/child hierarchy.

1 Drag the intended child pane to the tree in the Hierarchy window.
2 Drag it to the intended parent pane.
3 The pane is displayed one step below the parent pane, and a parent/child relationship is created.

Use the following procedure to remove a parent/child hierarchy.

1 In the Hierarchy window, drag the child pane.
2 Drag it to RootPane.
3 The parent/child relationship is removed for the pane.

Operations Using Parent/Child Relationships

The following operations are possible using parent/child relationships.

Applying Parent Pane Translations to the Child Pane

When the position of the parent pane is changed, the child pane automatically moves following the parent.

When changing the position of the parent pane and leaving the child pane where it is, perform the following operations.

  • Select Menu > Pane, and clear the Apply Parent Translations to Children check box.
  • While pressing the CTRL key, drag and move the parent pane.

Batching and Editing Parameters for Panes With Parent/Child Relationships

When you click the eye icon on the left edge of the Hierarchy window, you can display or hide all panes with parent/child relationships at the same time.
Similarly, when you click the key icon, you can lock or unlock all panes with parent/child relationships at the same time.

Applying Parent Pane Transparency to the Child Pane

When you change the transparency of the parent pane, you can apply it to the child pane.
You can apply the transparency of the parent pane to the child using the following procedure.

1 Select the parent pane and open the property window.
2 In the property window, open the Base Pane.
3 In the transparency parameters, select the Apply transparency to children check box.
4 When you change the transparency of the parent pane, the transparency of the child pane follows the change.
