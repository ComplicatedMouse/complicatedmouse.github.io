Shortcut List

The following is a list of shortcut keys in EffectMaker.

Main Menu (These shortcut keys are for items that are displayed in the Main Menu.)
CTRL+N New > Emitter Set Creates a new emitter set.
CTRL+SHIFT+N New > Emitter Creates a new emitter directly under the emitter set.
CTRL+O File > Open Opens a file.
CTRL+W File > Close Closes the file selected in the workspace.
CTRL+ALT+W File > Close All Closes all of the files in the workspace.
CTRL+S File > Save Saves the node selected in the workspace.
CTRL+SHIFT+S File > Save As Saves the node selected in the workspace under a different name.
CTRL+ALT+S File > Overwrite and Save All Overwrites and saves all nodes in the workspace.
F2 File > Rename Allows the name of the node selected in the workspace to be changed.
CTRL+Q File > Quit Exits EffectMaker
CTRL+Z Edit > Undo Restores to a pre-edited state.
CTRL+Y Edit > Redo Reapplies a single edit that was undone.
CTRL+C Edit > Copy Copies the content of the node selected in the workspace.
CTRL+V Edit > Paste Pastes the copied content to the selected node.
CTRL+D Edit > Duplicate Duplicates the node selected in the workspace.
DELETE Edit > Delete Deletes the node selected in the workspace.
CTRL+SHIFT+H Enabling and disabling features Enables and disables the functionality of the selected node.
F9 Display > Property Window Displays the property window.
F10 View > Effect Browser Displays the effect browser.
F11 View > Curve Editor Window Displays the Curve Editor window.
F12 View > History Window Displays the History window in the docking area of the main window.
CTRL+SHIFT+M View > Toolbar Displays or hides the toolbar.
F5 Preview > Transfer again Transfers data to the Viewer.
CTRL+K Tools > Options Opens the Options dialog box.
F1 Help > Help Displays Help.
Property Window (These shortcut keys are only used in the workspace.)
ALT+drag operations on a node - Exchanges the selected emitter set node or the emitter node with the node of the same type above and below it.
CTRL+E Open in Explorer Opens the folder containing the selected emitter set node in Windows Explorer.
Property Window (These shortcut keys are only for tab operations.)
ESC - Transitions to tab operations even if a property is being edited.
CTRL+TAB or → - Switches to the rightmost tab.
CTRL+SHIFT+TAB or ← - Switches to the leftmost tab.
Texture Details Window
CTRL+the PLUS key on the numeric keypad (+) - Enlarges the texture image.
CTRL+the MINUS key on the numeric keypad (-) - Reduces the texture image size.
CTRL+1 - Displays the RGB channels.
CTRL+2 - Displays the R channel.
CTRL+3 - Displays the G channel.
CTRL+4 - Displays the B channel.
CTRL+5 - Displays the A channel.
CTRL+M Shows or hides the mipmap.
CTRL+D Switches the display direction for the mipmap between vertical and horizontal.
