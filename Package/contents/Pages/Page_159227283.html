This document is about the NintendoWare Spy feature.

What Is Spy?

Spy is a set of libraries and tools for collecting application information and making it viewable on a PC tool (Spy.exe) timeline.

Spy allows the internal information for applications, middleware, and the system itself to be easily made visible and recorded, and can help with debugging and performance tuning.

Note

Currently, it only supports some of the audio data.

Name
Description

nn::spy

The library that provides the communication infrastructure between the application and Spy.exe.

nn::spy::audio

The library for collecting nn::audio information and sending data using nn::spy.

nn::spy::atk

The library for collecting nn::atk information and sending data using nn::spy.

Spy.exe (PC tool)

The PC that receives Spy data and makes it viewable on a timeline.
