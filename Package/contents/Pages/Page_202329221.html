Display

This page describes the additional View menu for LayoutEditor.
Display Outlines When selected, only the outlines of panes are displayed.
Hide Pane Borders

Hide Pane Borders: Hides the pane borders.

Hide Pane Borders and Null Panes: Hides the pane borders and the null panes.

Display Pane Borders and Null Panes: Displays the pane borders and the null panes.

The three options cycle through in order each time this option is selected.
Hide Pane Borders and Null Panes
Show Pane Borders and Null Panes
Show Boundary Pane When selected, the boundary pane is displayed.
Highlight Parts Pane When selected, the parts pane is displayed in light blue to make is easy to distinguish from other panes.
Zoom In Zooms in to the center of the layout window.
Zoom Out Reduces.
Maximize Window Maximizes the layout window.
Minimize Window Minimizes the layout window.
View All Adjusts the layout window size so that the entire layout can be displayed.
Display All Selected Area Enlarges and displays the selected pane in the center of the layout window.
Display Grid When selected, the grid is displayed.
Snap to Grid When selected, the pane snaps to the grid.
Display Ruler When selected, the ruler is displayed.
New Guide Creates a new guide.
Snap to Guide When selected, the pane snaps to the guide when moved.
Display Guide When selected, the guide is displayed.
Lock Guide When selected, the guide is locked.
Clear Guide Deletes all guides in the layout.
