Blend State

Blend State

Overview

The blend state collectively expresses several states primarily for the fixed features for blending in the pipeline.

The blend state for each color target is expressed as a blend target state.

Information

Use nn::gfx::BlendStateInfo to define the blend state information.

Logic Operations

To enable or disable logic operations, use the SetLogicOperationEnabled function. To specify which logic operation to apply, use the SetLogicOperation function.
When a logic operation is enabled, blending is not performed.

Converting an Alpha Value to a Coverage Value

Use SetAlphaToCoverageEnabled function to set whether to convert alpha values to coverage values when multisampling.

Dual Source Blend

Use the SetDualSourceBlendEnabled function to enable or disable dual source blending, where two colors are output to the same output buffer.
When dual source blending is enabled, a restriction is imposed on the number of render targets that can be used.

Independent Blending

To use independent blend target states for each render target, set SetIndependentBlendEnabled to true.
If the SetIndependentBlendEnabled function is set to false, the setting configured for the 0th blend target state is applied to all blend target states specified by SetBlendTargetStateInfoArray, and the first and subsequent blend target states will be ignored.

Blend Constants

If you are using blend constants for the blend factors, those values are set using the SetBlendConstant function.
Separate blend constants are set for the red, green, blue, and alpha components.

Blend Target State

The blend target state array is set using the SetBlendTargetStateInfoArray function.

The elements in the blend target state array correspond to the color targets.

Blending

To enable or disable blending, use the SetBlendEnabled function.

Blending Method

The following functions control the method used for blending.

Element Function Parameter to Set
Color Components SetSourceColorBlendFactor Source blend factor
SetDestinationColorBlendFactor Destination blend factor
SetColorBlendFunction Blend function
Alpha element SetSourceAlphaBlendFactor Source blend factor
SetDestinationAlphaBlendFactor Destination blend factor
SetAlphaBlendFunction Blend function

Write Mask

Uses the SetChannelMask function to control which combination of elements to write to the color target.
To write to all elements, specify nn::gfx::ChannelMask_All.

Object

Use nn::gfx::BlendState to define the blend state object.

Initialization and Destruction

The blend state is initialized and finalized according to the rules for objects.

Setting a Blend State

When using an execution model that uses a pipeline, information about the blend state is included in the pipeline information.

When you are using an execution model that does not use a pipeline, you set the blend state by calling the SetBlendState function of the nn::gfx::CommandBuffer class.
