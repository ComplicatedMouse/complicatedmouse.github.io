File

The File menu provides operations related to file I/O and other general 3DEditor operations.

A

Displays the New submenu.
B Opens graphics intermediate files supported by 3DEditor.
When the loaded material parameters do not match the shading model, a dialog box appears where you can select which material to modify.
C

Opens project files supported by 3DEditor.
When the loaded material parameters do not match the shading model, a dialog box appears where you can select which material to modify.
D

Closes selected files.
When a model file (FMDB or FMDA file) is closed, all animation files and texture files that are referenced by that model are closed at the same time.
E

Closes all files.
If files being closed include ones that were edited, the Save Changed File dialog box is displayed.
Files opened on the startup path are not closed when Close all is selected.
F

Closes all files loaded into 3DEditor.
You can select the files to be closed from the Select files to close dialog box.
G

Saves the selected file and overwrites any existing file in the same location that has the same name.
If saving for the first time, you are prompted to specify a save location and filename.
H

The Save As operation. It saves the selected file by using the specified name.
A dialog box appears for specifying the save location and filename.
I

A dialog box for saving in a specified location opens, and the file is saved in the specified location.
J

Saves all files.
Note: Applies to files edited using 3DEditor. Files not edited using 3DEditor are not saved.
K

Renames the file selected in the file view.
L

Displays the most recently used files.
Loads the selected file into 3DEDitor.
M

Displays the most recently used projects.
The selected project is loaded into 3DEDitor.
N

Closes the 3DEditor.

New Item Menu

Use this submenu to perform operations that relate to creating new files.

A

Creates a new project.
B Creates a new material file.
C

Creates a new material animation file.
D

Creates a new bone visibility animation file.
E

Creates a scene animation file.


Save Changed File Dialog Box

When you close files by selecting Close all on the Files menu, this dialog box appears if the files being closed include edited files.

A Lists files that have been edited.
Saves selected files.
B Selects or clears all listed files.
C Saves and closes all selected files.
D Closes all files without saving any files even if selected.
E Cancels closing files.

Select Files to Close Dialog Box

This dialog box is displayed when you select multiple files for closing in 3DEditor.

A Lists files to close.
Closes selected files.
B Select files to close by extension.
Selects the check boxes of all files matching the selected extension.
C Exclude files to close by extension.
Clears the check boxes of all files matching the selected extension.
D Displays the number of files to be closed.
E Closes the selected files.
F Cancels closing files.

Specify Save Location Dialog Box

This dialog box is displayed when saving all files to a specified location.

A

The folder where files are saved.

Specifying a nonexistent folder displays a confirmation dialog box. Clicking OK creates a new folder and saves it.

B Specify any folder for saving from the Browse dialog box.
C The list of files to save.
By default, the check box for saving all files is selected. Clear the check box if there are files you do not want to save.
D Selects all check boxes for the selected extension.
E Clears all check boxes for the selected extension.
F The number of target files to save.
G Selecting this check box saves textures to the textures folder.
H Selecting this check box displays a confirmation dialog box when a file is to be overwritten.
I Saves files.
J Cancels all saving to the specified location.


Modify Materials Dialog Box

This dialog box appears when the parameters of loaded materials do not match the shading model.

A Displays a list of materials that must be modified.
Select the check boxes of the materials you want to make match the shading model.
All check boxes are selected by default.
B Select this check box if you want to automatically modify any materials that do not match the shading model without displaying the modify dialog box in the future.
The automatic modification setting is saved in user settings.
C Modifies the material.
D Cancels the modification of materials and closes the dialog box.
Info

You can enable or disable the Automatically modify and do not ask me again setting on the I/O Settings page located under Tools > User Settings.

Selecting Check Boxes for Multiple Files

If you hold down the CTRL key and drag to select multiple files listed in the dialog box, and then select (or clear) a check box, that same entry is made for all selected files.

Merge Dialog Box

Right-click the model file and select Merge Model Files to open the dialog box.
You can apply the settings from other model files to the selected model.

A Displays the settings to merge.
B Displays the file path and filename of the merge target.
C Displays the file path and filename of the merge source.
D Displays the settings and number of data items that can be merged.
E Used to specify the settings being merged for each item (tab).
Info

Overwriting some of the settings is called merging.
This does not mean all settings will be overwritten.


General Tab

Performs general merge settings for all models.

A Specifies whether to merge shapes.
B Specifies whether to merge the model's vertex data.
C Specifies whether to merge the "use bone scaling" settings.
D Executes or cancels merging.
To implement settings in all tabs and perform merging, click OK.

Bone Tab

Sets the merge settings for the model bones.
Merging bones overwrites the content of Billboard and Show shapes related to bones for each bone.

A

A list of bones in the model.

If multiple bones have the same name, the corresponding bone will be listed automatically.
B Configures individual settings for the bone selected from the list.
C

Configures merge settings for all bones shown in the list.
D Executes or cancels merging.
To implement settings in all tabs and perform merging, click OK.

Material Tab

Configures the merge settings for the materials of the model.
Merging materials overwrites all settings other than the Allow deletion during material-compression optimization setting for each bone.

















A A list of materials in the model.
If multiple materials have the same name, the corresponding material will be listed automatically.
B Configures individual settings for the material selected from the list.
C

Configures merge settings for all bones shown in the list.
D Executes or cancels merging.
To implement settings in all tabs and perform merging, click OK.
