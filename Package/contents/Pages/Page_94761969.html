Color Combiner Formula

This section describes the equations for the Color Combiner.

Overview

The Color Combiner Formula feature is for configuring the equation that is used when combining color elements.
By modifying the equation you can enhance the quality of the effect you are trying to create.
The following table provides an image and a brief description for each of the color elements that are used as inputs in the color combiner formula.

Comb_RGB

The RGB values in the previous stage combiner settings.
The alpha channel of the registered texture is output using RGB values.
█
C0 The RGB values for the color 0 value specified in the Color tab.
Here, the RGB values are set to 1.0, 0.5, and 0.0.
█ C1 The RGB values for the color 1 value specified in the Color tab.
Here, the RGB values are set to 0.0, 0.5, and 1.0.

Color Calculation Formula

Select which formula among the presets to use to combine the RGB values in the previous stage (Comb_RGB) with the RGB values set in the Color tab.
The following table provides examples of the combination results, output images, and formulas that can be used.

C0 Outputs color 0 as is.
C0 × Comb_RGB Outputs the result of multiplying color 0 and Comb_RGB.
C0 × Comb_RGB ＋ C1 × (1 － Comb_RGB) Outputs the result of combining color 0 (a white color) and color 1 (a black color) with the output result for Comb_RGB.
C0 × Comb_RGB ＋ C1 Outputs the result of multiplying color 0 and Comb_RGB and then adding color 1 to the product.
When using the glare post-effect, for example, you can use color 1 to control the brightness of the resulting glare effect.

Example Configurations

The following images compare two formulas in an environment where the Glare post-effect can be used.

The environment is the Gradient Editor settings.
The initial and final animation values are black.
Setting a blue color to a value of 1 or more during the animation enhances the glare effect.
A C0 × Comb_RGB ＋ C1 × (1 － Comb_RGB)

Applies color 0 to the light areas in the texture and color 1 to the dark areas in the texture.
In this example, only the dark areas in the texture are affected by the glare effect.
B C0 × Comb_RGB ＋ C1 Multiplies the texture by color 0 and then adds color 1.
In this example, the overall texture is affected by the glare effect.
