Main Menu

This section describes the 3DEditor main menu.

File

Menu commands specific to 3DEditor Projects and files, such as saving and specifying.
Edit

Menu commands specific to editing operations in 3DEditor, such as specifying and manipulating.
Display

Menu commands specific to the display of the Object View and column items in lists.
Preview

Menu for previewing models and animations in applications in which the viewer library is embedded (on a PC or the development hardware).
Tools

Menu commands specific to verifying and specifying configuration files related to 3DEditor operations.
Help Menu commands for opening the manual and displaying version information.
