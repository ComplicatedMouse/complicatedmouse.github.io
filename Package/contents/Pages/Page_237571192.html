2. Prepare the Maya Scene File

This section describes the procedure for importing the model.
An animation of 50 frames is created for the imported model.

In this tutorial, an existing FMDB file is imported to Maya for preparing the Maya scene. Normally you would open a Maya scene file.
Load the sample model from the package.

Open the Nintendo Import Window

From the Maya menu, select Nintendo > Import.

Configure the Settings in the Nintendo Import Window and Load the File

Configure each setting and load the file.

Load the file by performing the operations in the order described in the following images and table.

Order Category Operation
1 General Options

Set Magnify to 1.0000. Leave Namespace empty.
Select Create Material, Create Texture Node, and Create Texture File.
Keep LDR and HDR set to Auto.
2 Import Click the Import tab.
Import Options FMD File

Click the folder icon () and specify the following FMDB file.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal\TrafficSignal.fmdb
3 Import Options - Import fmd Click Import fmd. The configured file will be imported to Maya.

The model in the following figure is displayed when import succeeds. Close the Nintendo Import window.

The animation working area in Maya, is set to the range from frame 0 to frame 50.
