Particle Shape - Primitive

This section describes the Primitivesetting for particle shape.

Overview

When Primitive is specified as the particle shape, you can use freely shaped polygons created using the DCC tool for particles.
You can control the available attributes in your DCC tool.

The following overview describes the settings ().
When the particle shape is set to Primitive, the Primitive group () of settings is displayed. The group has the following settings.

Specify File

Registers, updates, and deletes primitive files. The following are button types.

 Load Open the file selection window and select a model file (FMDB file).
Open Sample Primitives Open the Primitive folder. You can put frequently used model files (FMDB) in this folder.
The Primitive folder is set to the following location.
Folder_where_EffectMaker_starts\Primitive
Reload Updates the specified model file (FMDB file).
Clear Releases the specified model file (FMDB file).

When you load a primitive, the following information about that primitive is shown in comments.
Use this information to check the load of the primitive and information about its attributes.

  • Number of vertices
  • Included attributes (pos (vertex positions) / normal / tangent / color (vertex color) / uv (texture coordinates) / LOD count)
