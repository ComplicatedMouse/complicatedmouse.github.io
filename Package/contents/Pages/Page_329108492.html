Enabling Brilliance

This section describes the Enable Brilliance feature that adds an effect where it appears that the pane is emitting light.

Overview

The Enable Brilliance feature adds an effect where it appears that the pane is emitting light.
Enable this effect when you want the pane to look like it is glowing.
The following is an overview of the settings.

Rendering Mode

The following images compare what happens when the Rendering Mode is changed.
This feature configures the method of blending the color in the area of the pane glow and background color.

Normal Multiplicative Additive (Default) Subtractive

Blend translucently. Blend multiplicatively. Blend additively. Blend subtractively.

Spread

The following images compare what happens when the Spread value is changed for the glow.

The higher the value, the more defined the edge of the translucent area becomes, which diminishes the bleed effect.

Spread = 0 (Default) Spread = 100

Size

The following images compare what happens when the Size value is changed for the glow.
The Spread value is set to 100.

The larger the value, the wider the glowing area.

Size = 5 (Default) Size = 25
