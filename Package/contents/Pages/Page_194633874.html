Library

Overview

The PC version of Vulkan only provides header files. The library does not need a static link, and the Vulkan library entry point gets function pointers from the graphics driver at runtime. The specific usage varies depending on the PC GPU vendor. For more information, see the Vulkan website.

The NX version of Vulkan provides a specialized library in addition to the header file.

Header Files

Vulkan provides the vulkan/vulkan.h header file for defining the C interface, and the vulkan/vulkan.hpp header file for defining the C++ interface.

These header files are placed in the following directory, along with other required header files.

$NINTENDO_SDK_ROOT/Include/vulkan
