Mii Development Support Features

Use the mii command to use the development support features of the Mii Face library.

  • Mii Testing Mode
  • Exporting a Mii Database
  • Importing a Mii Database
  • Exporting a Mii Database in CharInfo Format
  • Formatting a Mii Database 

$ RunOnTarget.exe DevMenuCommand.nsp -- mii --help
usage: DevMenuCommand mii get-testing-mode-enabled
       DevMenuCommand mii enable-testing-mode
       DevMenuCommand mii disable-testing-mode
       DevMenuCommand mii export-database <absolute_host_file_path>
       DevMenuCommand mii import-database <absolute_host_file_path>
       DevMenuCommand mii export-char-info <absolute_host_directory_path>
       DevMenuCommand mii format-database

Mii Testing Mode

To use developer support features you must enable testing mode.
You can use the following commands to get, enable, and disable testing mode.

# Get the testing mode state.
$ RunOnTarget.exe DevMenuCommand.nsp -- mii get-testing-mode-enabled
[SUCCESS]

# Enable testing mode.
$ RunOnTarget.exe DevMenuCommand.nsp -- mii enable-testing-mode
[SUCCESS]

# Disable testing mode.
$ RunOnTarget.exe DevMenuCommand.nsp -- mii disable-testing-mode
[SUCCESS]

Exporting a Mii Database

Use this command to export a Mii database to a specified file.

$ RunOnTarget.exe DevMenuCommand.nsp -- mii export-database <absolute_host_file_path>
[SUCCESS]

Importing a Mii Database

Use this command to import a specified file to a Mii database.

$ RunOnTarget.exe DevMenuCommand.nsp -- mii import-database <absolute_host_file_path>
[SUCCESS]
Note

Files exported from a platform other than NX cannot be used.

Exporting a Mii Database in CharInfo Format

Use this command to export the Mii characters from a database to a specified directory in CharInfo format.

$ RunOnTarget.exe DevMenuCommand.nsp -- mii export-char-info <absolute_host_directory_path>
[SUCCESS]

Formatting a Mii Database

Use this setting to format a Mii database.

$ RunOnTarget.exe DevMenuCommand.nsp -- mii format-database
[SUCCESS]
