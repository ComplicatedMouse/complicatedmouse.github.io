Property Window

The Property window is composed of a UI for operations on nodes and for previewing, and a UI for configuring node features.

A Preview Controls

Controls operations such as playing and stopping when connected to the viewer ().
B Workspace Toolbar Used to arrange the buttons related to node operations.
C Workspace

The nodes that constitute an effect are displayed in a tree structure.

Click a node to view a shortcut menu for that node.
D Tab

The tabs () display features for the selected node in the workspace, grouped by type.
E Group Inside the tabs (), groups () display groups of settings () that are grouped by type.
F Settings Settings () represent the names of features.
The UI on the right is for editing settings values and displaying information.
Info

To open the documentation in your default browser and display the Help for a setting, click the setting ().
