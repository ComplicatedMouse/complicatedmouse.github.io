Soft Particles, Masking Check

This section describes the Masking Check feature that applies translucency to particles masked by polygon models, using soft particle processing.

Overview

The Masking Check soft particle feature enables you to make particles masked by a polygon model transparent.
You can, for example, attenuate the alpha value for an effect to link it to the gradual hiding of a light source when using an effect like a flare.


The following overview describes the settings ().

To display the Masking Check, in Masking Check (), select Soft particles. The group has the following settings.

Sampling Radius

To check whether the particle targeted for masking is masked by the masking object, specify a circular range, using the screen coordinate system.
Use the dial () or the text box () to specify the circle's radius.
