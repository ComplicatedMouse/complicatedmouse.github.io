Profiler-NX-GpuTrace

Overview

Controls whether GPU Trace data should be recorded in the profile data or not. This will only enable the feature if the GPU Trace button is enabled for selection at the time this cmdlet is called. If the GPU Trace button is disabled, GPU tracing will be disabled.

Profiler-NX-GpuTrace [-EnableGpuTrace] <Boolean>


[-EnableGpuTrace] <Boolean>

If true, record GPU tracing in the profile data. If false, turn off recording of GPU trace data.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-NX-GpuTrace $true
