LayoutEditor

This section introduces the new features in LayoutEditor.

Key Auto-Configuration

Added a Key Auto-Configuration feature to LayoutEditor.
When “Enable Key Auto-Configuration” has been set, animation is automatically set or changed when an item is edited for which a curve has been in the layout window or property window.

Copy and Paste Values and Curves in the Property Window

Added functionality to right-click labels and other properties in the property window, and then copy and paste values from the shortcut menu. Curves can also be pasted when Main Menu > Edit > Copy/Paste Animation is enabled.

Feature for Saving Thumbnails From the Command Line

Added the --save-thumbnail option for saving thumbnail files to the LayoutEditor command-line options. It saves a file named Layout_filename.flyt.png in a .NwThumbs folder in the folder that contains the layout files. 

LayoutEditor.exe "D:\input.flyt" --save-thumbnail

Added Brick Repeat Feature

Added a brick repeat feature that makes it possible to specify an offset to be allocated between each successive mapping of a texture when repeating texture UV data in a pane.
This option can be designated in Mapping Method in the texture properties.

Added a Shortcut for Swapping Segment Tags From the Curve Editor Window

Added a shortcut for swapping Segment Tags in the Edit Animation Segment Tags window from the Curve Editor window.
As shown in the following image, you can now swap segments using the shortcuts added to the Segment Tags menu in the Curve Editor window.

The following table shows the keyboard shortcuts.

CTRL+UP ARROW Select the segment tag before the currently selected segment tag.
CTRL+DOWN ARROW Select the segment tag after the currently selected segment tag.
CTRL+1 through CTRL+5 Select among the segment tags assigned to numbers 1 through 5 in the Edit Animation Segment Tags window.
