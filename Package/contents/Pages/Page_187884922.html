Changes for Developers

This section describes the changes made to NintendoWare audio (nn::atk) in NintendoSDK 0.17.0.

Contents

  • Fixed a bug where the Fix Priority at Release feature was not disabled.

Fixed a bug where the Fix Priority at Release feature was not disabled.

Fixed a bug in SoundMaker where even if the Fix Priority at Release option was disabled, the priority would not be set to the minimum value when released.
Note that this fix will change the results of the sound emission priority determination process while an application is running, which may cause audio data to sound different from before.
