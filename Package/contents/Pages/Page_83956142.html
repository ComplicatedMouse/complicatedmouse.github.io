Team Settings File

Overview

Use the team settings file to adjust the operations of 3DEditor.

Use the 3DEditor --team-settings-file command-line option to specify the team settings file that 3DEditor will load.

If a team settings file has not been specified with the command-line option, 3DEditor uses the 3dEditor.teamsettings file in the same folder as 3dEditor.exe. If that file does not exist either, 3DEditor operates with its default settings.

The team settings file is loaded when 3DEditor starts.

Info

Any changes made by editing the settings file while 3DEditor is running are not applied. To apply changes, you must restart 3DEditor.


The following section introduces the main items that are set. For more information, see Team Settings File Format.

File I/O Settings <FileIo>

Describes the settings for file input and output.

You can configure the following primary settings.

  • Startup Path: Specifies files to open when 3DEditor starts.
  • Additional Reference Path: Used to search for reference files and to resolve the path of files included in the project starting from the startup path.
  • Commands for file operations: Specifies programs to call before or after files are saved or before files are opened.
  • Sets the command for generating shader variations for optimizing shaders.

Preset Settings <Preset>

You can configure texture formats as presets.

Default Settings <DefaultValue>

Describes the default values used by 3DEditor.

You can set the default values, such as the mipmap filter used by samplers.

You can also configure restrictions on the ranges for shader parameters, and which items can be edited using 3DEditor.

Error Settings <Error>

You can make settings such as the email address and subject title to use when sending error messages.

Preview Settings <Preview>

With these settings, you can specify the anticipated FPS and message transmission interval on the development hardware.

Color Settings <Color>

You can set whether to perform gamma correction.

Platform Settings <PlatformPresets>

Use this element to check the platform settings.

If the <PlatformPresets> element is not included in the team settings file, 3DEditor will use the default platform settings.

Platforms added to platform settings appear as selections in the platform area in the toolbar.
