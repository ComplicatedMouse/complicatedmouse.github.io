Output Waveform File

Each type of sound in the project can be exported as a waveform file.

The following types of sounds can be exported.

  • Stream sounds
  • Wave Sounds
  • Sequence sounds

Output Waveform File

Select a sound from a list in the sound set tab, and select Preview > Output Waveform File.
The Output Waveform File dialog box appears. Specify the output filename and other options and then click OK to export the waveform file.

Maximum Output Time: How much of the waveform to output (in seconds).
Number of Output Channels: Select either stereo or mono.

The same operation can be performed by selecting Output Waveform File from the shortcut menu on the list.

You can select multiple files to export.

The waveform file will be output as a 16-bit, 48,000 Hz WAV file.

Info

Sounds are saved as WAV files using 16-bit PCM compression regardless of the compression format that has been set for each sound.
