Maximum Number of Particles Emitted

This feature limits the number of particles that can be emitted.

Overview

The Maximum number of particles emitted feature limits the number of particles that are displayed to less than or equal to the specified value.
For example, you can use this feature to prevent unnecessarily high processing loads resulting from the number of particles that are simultaneously displayed becoming too large because the particles have a long lifetime.

The following overview describes the settings ().

Maximum Number of Particles Emitted

Use the dial () or the text box () to specify an integer threshold value for the maximum number of particles emitted.
