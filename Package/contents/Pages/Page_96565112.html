Bounce Rate

This page describes Bounce Rate, a feature for configuring the bounce coefficient after a particle collides with a collision.

Overview

Bounce Rate enables you to set a bounce coefficient for multiplying the y-axis speed when a particle collides with a collision.
You can restrict the height reached after a particle bounces.
The following overview describes the settings ().

Bounce Rate

This feature is applied when Behavior When Colliding is set to Bounce.
You set the bounce coefficient for multiplying the y-axis speed during a particle collision using the dial () or text box ().

Values smaller than 1 decelerates particle speed after bouncing.
Values greater than 1 accelerate particle speed after bouncing.
A value of 1 maintains the same speed after bouncing.

Example Configurations

The following figure compares what happens when the value for Bounce Rate is changed.

● Light blue 2 Particle bounces with twice the speed when colliding with a collision.
● Green 1 Particle bounces with the same speed when colliding with a collision.
● Red 0.5 The default setting. Particle bounces with half the speed when colliding with a collision.
● White 0 Particles move along the top of the collision plane without bouncing off it.
