Emitter Shape - Sphere

Describes the settings for Emitter Shape - Sphere, where particle emission locations are set within a sphere.
These are the particles when this emitter shape is being used.

Overview

With the Sphere emitter shape, particles emit in the direction of the normal in a widening spherical shape from random positions around the outer surface of a sphere located in world coordinates.
The following overview describes the settings ().

When the emitter shape is set to Sphere, the Sphere () settings are displayed. The group has the following settings.

Emitter Radius

Set the sphere radius using x y z.

Method of Arc Opening

Select how the arc opens from among the following presets.

Longitude Sets an angle in the range from 0 to 360 degrees for widening in the longitudinal (east-west) direction, based on the z-axis of the sphere.
Latitude Sets the angle between the y-axis zenith and the nadir, based on the y-axis zenith of the sphere, in the range from 0 to 180 degrees.
