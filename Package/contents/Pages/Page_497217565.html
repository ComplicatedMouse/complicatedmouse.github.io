Material Referencing Tutorial

This section uses the sample data included in the package to describe how to use material referencing.

Note

Sample models will be included in NintendoSDK 9.0.0 and later packages.


For this description, we use ice cream cones like those shown in the following figures by way of example.

Create child variations that reference a base. Change the base and the flavor also changes for the children.
  • Create a parent (base) material.
    We make the parent a vanilla ice cream cone and assign it to a model.
  • Each child material references this parent and becomes a vanilla ice cream cone.
  • Various toppings are added to the child materials.
  • Made some change to the parent (base).
    The vanilla ice cream cone is changed into a chocolate cone, an orange cone, a grape cone and the like.
  • The child materials reference the parent, so without doing any work they match the parent (with the toppings remaining unchanged).

Perform the tasks in the order described in the table to get results as shown in the preceding figures.

1 Preparation This section describes the startup procedures for various tools that are required to begin work.
2 Loading Samples

Load the sample (material and model) files included in the package .
3 Checking the Parent Material Settings Set the parent material to the models and check it in the viewer.
4 Editing Child Materials

Set the alpha mask textures that express the ice cream toppings.
5

Changing Parent Material and Syncing Child Materials

Change the color of the parent material and confirm that child materials also change as a result.
