Rotation Order

This page describes the rotational order, the functionality to configure the processing order when specifying two or more of the x, y, and z rotational axes.

Overview

Rotational Order is a feature for configuring the order of axes to which to apply rotation when the use of at least two axes is enabled in the check boxes () for Axial rotation ().
It allows the combination of a 3D model and rotational direction.

The following overview describes the settings ().
Rotational Order () is displayed when at least two axes are selected in Axial rotation. The group has the following settings.

Rotation Order

Selects the axis rotation order from the following presets.

Y → Z → X Particles rotate around axes in the order of y → z → x.
X → Y → Z Particles rotate around axes in the order of x → y → z.
Z → X → Y Particles rotate around axes in the order of z → x → y.
