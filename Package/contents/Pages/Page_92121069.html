Options

This page describes the Option window, which is used to configure the EffectMaker and Viewer environment.

User Settings These items are configured individually for each user.
Interfaces You can configure elements of the EffectMaker user interface.
Viewer Settings You can configure some aspects of the viewer.
File Events You can set a user command file to be executed during file operations.
Advanced You can set how to manage the memory and data used by EffectMaker.
Project Settings These settings are configured individually for each game project. Configuring these settings for each user is not recommended.
Custom Shaders Specifies files for custom actions used by EffectMaker.
Custom actions Specifies folders for custom shaders used by EffectMaker.
Emitter Extended Parameter Specifies folders for extended emitter parameters used by EffectMaker.
File Events You can set a user command file to be executed during file operations.
Platform Settings Information about the game platform.
