LayoutViewer

This section introduces features newly added to LayoutViewer.

Meter to Display the Processing Load as Show Debugging

Added a processing meter to LayoutViewer to enable users to better understand the processing load of layouts.

A Calc Shows the cost (in milliseconds) of computational processes by the CPU in the runtime.
B Draw Shows the cost (in milliseconds) of drawing by the CPU in the runtime.
C Gpu Shows the GPU cost (in milliseconds) in the runtime.
Info

The gray parts in the meter depict the viewer's display-related load outside of the layout.
