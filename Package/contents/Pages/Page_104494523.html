2. Loading a File

Load the layout file (FLYT file) with the layout data into LayoutEditor.
The following example loads the layout file located in the following location.

\Samples\Sources\Resources\Ui2d\expression\flyt\test\Font_Sample.flyt

Perform any of the following actions to load Font_Sample.flyt, the layout data file.

  • Open From the Menu
    From the main menu, select File > Open > All.
    In the dialog box, select Font_Sample.flyt and click Open.
  • Open Using Shortcut
    You can press CTRL+O to open the dialog box.
    In the dialog box, select Font_Sample.flyt and click Open.
  • Drag to open
    Select Font_Sample.flyt in Windows Explorer, and then drag it to the workspace window.

3. Previewing in the Viewer

While data is being edited, LayoutEditor can transfer the data to the Viewer so you can check how the results will ultimately be displayed.
The Viewer displays a screen corresponding to the layout screen region.
The following table shows the layout window, which appears within the LayoutEditor window, and LayoutViewer, which opens in a separate window.

The following methods can be used for previewing.

Send to Viewer

Sends layout data to the Viewer in the state in which it is being created.
Animations are not transferred.
If there is animation, the layout data is sent in the state of the currently displaying animation frame.

  • Send from menu
    From the main menu, select Tools > Transfer to Viewer.
  • Send using shortcut
    Press SHIFT+F5.

Transfer to Viewer (Animation Mode)

This method sends both layout data and animation data to the Viewer.
The length of the animation is defined by the segment of animation frames set in the curve editor.
(For more information about the curve editor and playback frame segments, see the description of animation features.)

  • Send from menu
    From the main menu, select Tools > Transfer to Viewer (animation mode).
  • Send using shortcut
    Press F5.

Exiting the Viewer

Use any of the following operations to exit the Viewer.

  • Close from the main window
    In the main window of the Viewer, click the Close icon ().
  • Close from the title bar
    In the Viewer window, from the title bar, select Close.
  • Close using shortcut
    Press ALT+F4.

4. Completing the Task

Use any of the following operations to close all open files and exit LayoutEditor.

  • Exit from main menu
    From the main menu, select File > Exit.
  • Exit from main window
    In the main window of LayoutEditor, click the Close icon ().
  • Exit using shortcut
    Exits in order, starting with the selected layout data.
    Press CTRL+Q.

If there are any files that have not been saved, a dialog box appears. Specify a save destination in the dialog box that opens, and then click OK.
