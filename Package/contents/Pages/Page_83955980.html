User Commands

User commands enable users to configure and run commands for set processes from the menus or toolbars.

From SoundMaker, you can run set processes for projects, such as the process that implements batch files, or the executable process for applications.

Configuring User Commands

Use the Project Settings dialog box to configure user commands.

User commands include:

  • A command name.
  • Command
  • An icon file path.

You can register a maximum of eight user commands.

The Command Name is the string that is displayed as the tool tip for the menu item or toolbar selection, and the Command is the actual string that is executed.

Set the icon for the menu item or toolbar button by specifying an image file in Icon File Path.

Configured commands appear in the Tools menu, under User Commands. They also appear on the toolbar.
Nothing is added to the toolbar when no commands are configured.
