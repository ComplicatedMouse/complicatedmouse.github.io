Operating Environment

The atk library depends on nn::audio, and depending on the platform, processing occurs either in the CPU or DSP.

CPU Rendering or DSP Rendering
Run the NX application in SDEV. Rendering is performed by nn::audio (DSP).
Run the NX application in Windows. Rendering is performed by nn::audio (CPU).

Resource Converter

Atk binary resource files (BFSAR, BFSTM, and so on) use shared files with PC and NX (SDEV and so on).

Note: SoundProjectConverter and WaveConverter do not have the option of specifying the platform.

SoundMaker

Connecting SoundMaker and NX Applications

You can switch the connected application with the combination of Connect to and Platform from the SoundMaker toolbar.

Platforms Connect To Description
PC PC Application Connects to a NintendoSDK application that runs on Windows.
NX PC Application

Connects to an NX application that runs on Windows.

Only available when the NX Add-on is installed.
NX Device Application

Connects to an NX application that runs on a device such as SDEV.

Only available when the NX Add-on is installed.

Connecting SoundMaker to the Hardware Version of AtkPlayer

The following features are available when NX is selected as the platform.

  • Start Hardware AtkPlayer (This item appears in the menu and as a tool button.)
  • Stop Hardware AtkPlayer (This item appears only in the menu.)

You can also set the connection target to the hardware version of AtkPlayer by selecting the hardware version of the application for Connect To.
After the hardware version of AtkPlayer is running use Connect to connect to SoundMaker.
When the connection is established, "connected" is displayed on the screen.

Info

You cannot connect to the PC version of AtkPlayer if the platform is set to NX and Connect To is set to PC Application.
To connect to the PC version of AtkPlayer, set the platform to PC and, for Connect To, select PC Application.

Information for the sounds being previewed is shown in the Command Prompt window and in the TargetManager window.

To read how to use the hardware version of AtkPlayer, see the How to Operate section of AtkPlayer.
