Rectangle

Property Window For a Rectangle Shape Picture Pane

This section describes the property window when the picture pane shape is a rectangle.


Rectangle Shape Picture Pane

Vertex Color

Sets the vertex color of a pane.

Simplified figure of a pane used to set the position to enter vertex color.

Click the position you want to specify color for in the figure. The Color Picker opens.

Use the Color Picker to specify the vertex color.
Set all to white

Specifies white (for RGB display, R:255, G: 255, B: 255, A: 255) for all vertex colors of panes being edited.

The default value of vertex colors is white, so this option can be used to reset editing.

This option is not set if the following "Change XXX " items are not not selected.
Change RGB ()

Select this option to edit RGB.

If this option is not selected, edits of RGB will not be applied.
Change A ()

Select this option to edit A (alpha).

If this option is not selected, edits of A will not be applied.

Texture coordinates

Sets the texture coordinates.

Number of Coordinates to Use Describes the texture setting method to use in the pane.
Coordinates to Edit Describes the texture mapping method.
Textures Using Coordinates Displays thumbnails of the textures using coordinates to edit.
Repetition Count Describes the UV repetition count presets.
UV Invert/Rotate Describes the UV coordinate editing presets.
UV Coordinate Advanced Settings Describes the advanced settings for UV coordinates.
