Coordinates (±)

This section describes the feature for applying an offset to the position of the emitter.

Overview

The Coordinates (±) feature applies an offset to the position of the model.
This section provides an overview of the settings ().

Coordinates (±)

Use the dial () or the text box () to specify the offset for each axis in the model's position.

Example Configurations

The following figure compares what happens when the value for Coordinates (±) is changed.

A XYZ = 0 , 0 , 0 The model's position is set as the same position as the origin.
B XYZ = 20 , 15 , 0 From the position of example A, the position when x, y, and z are changed to 20, 15, and 0.
