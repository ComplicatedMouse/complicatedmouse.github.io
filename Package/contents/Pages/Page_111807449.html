Color Sample Window

You can use the Color Sample window to add or edit colors used by LayoutEditor.

You can drag each of the colors registered in the Color Sample window to set the color for items in the Properties window.
Registering colors in the Color Sample window means it is no longer necessary to specify colors using the color picker each time when setting properties.
The registered colors do not depend on each layout window, and are convenient when handling data for many layout windows.

UI for Selecting Panes Using a Common Color

Panes that are using a registered color can be selected.
Right-click any registered color to display the Select panes with common colors menu. Shorcut menus contain the following options.

All Selects all panes in the layout that are using a common color.
Color Interpolation Selects panes that are using a common color for color interpolation.
Vertex Colors Selects panes that are using a common color for a vertex color.
Text Box Panes Selects panes that are using a common color in text box panes.
Shape Selects panes that are using a common color for shape.
Capture Effect Selects panes that are using a common color in a capture effect.
Textures Selects panes that are using a common color for a constant color.
Capture Pane Selects panes that are using a common color for a clear color.
Animation Keys Selects panes that are using a common color in an animation.
