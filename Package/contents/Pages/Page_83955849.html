Sampler

Sampler

Overview

Samplers provide the functionality for setting states when the GPU performs texture sampling.

gfx requires that a sampler be set in the command buffer when performing texture sampling. The sampler is set together with the corresponding texture as a pair, simultaneously.

Information

Use nn::gfx::SamplerInfo to define the sampler information .

Address Mode

The address mode indicates the mode when sampling texture with coordinates outside the range of the texture coordinates.
The u, v, and w coordinates are set using the functions SetAddressU, SetAddressV, and SetAddressW.

Filter Mode

The filter mode represents the filters to apply when sampling a texture.
Use the SetFilterMode function to combine enumeration values for comparison mode, the minification filter, the magnification filter, the mipmap filter, and reduction mode.

Comparison Function

When the filter mode is set to comparison mode, a comparison function is applied.
Specify this function using SetComparisonFunction.

Border Color Type

The border color type indicates the border color to apply when the address mode is set to clamping to a border.
In gfx, only the several predetermined border colors can be set.
Use the SetBorderColorType function to configure this setting.

Maximum Sampling Number for Anisotropic Filtering

The maximum sampling number for anisotropic filtering represents the maximum number of samples to apply when the filter mode is anisotropic filtering.
An integer value is configured using the SetMaxAnisotropy function, but there is no assurance that this exact value will be set.

Mipmap Range Minimum and Maximum Values

The lower-limit and upper-limit values for the mipmap range indicate the range of values to apply to the mipmap level that is calculated when sampling.
These values are set using the SetMinLod and SetMaxLod functions.

Mipmap Level Bias

The mipmap level bias represents the offset value to apply to the mipmap level that is calculated when sampling.
It is set using the SetLodBias function.

Object

Use nn::gfx::Sampler to define the sampler object.

Initialization and Destruction

Initializing and freeing the sampler generally follows the method for objects

Setting a Sampler

Setting to the Descriptor Pool

Set a sampler to the descriptor pool using the SetSampler function of the nn::gfx::DescriptorPool class.
Get the descriptor slot for the set sampler using the GetDescriptorSlot function of the nn::gfx::DescriptorPool class.

Setting to the Command Buffer

Set a sampler state command by pairing it with a texture in the SetTextureAndSampler function of the nn::gfx::CommandBuffer class. When using the descriptor pool, pass the descriptor slot as an argument. When not using it, pass the sampler.
The GL version has the limitation that bindings explicitly written to the shader are overwritten (See Uniform Blocks and Sampler Initialization Overwrites Bindings in the GLSL Shader). For this reason, at this time, set the shader slot number using the value dynamically obtained from the shader object.
