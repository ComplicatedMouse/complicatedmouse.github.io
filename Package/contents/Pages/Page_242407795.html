Question

When float color is enabled, there is glare in the preview if I specify Threshold-Based Alpha Interpolation in the Color Interpolation Settings even though there is no data that is meant to generate glare.

Answer

You cannot use Threshold-Based Alpha Interpolation and float color at the same time.
