Number of Pattern Tables To Use

This section describes the Number of pattern tables to use feature that specifies the number of pattern tables to use in animation.

Overview

The Number of pattern tables to use feature allows you to specify the number of pattern tables to use in an animation.
The following overview describes the settings ().

Number of Pattern Tables To Use

Use the dial () or text box () to set the number of patterns to use in an animation within the scope of the table.
The settings information is displayed in the Pattern Tables UI.
