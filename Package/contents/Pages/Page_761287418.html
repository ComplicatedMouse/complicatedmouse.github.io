Digitizer (NX Add-On)

Installing NX Add-On enables the digitizer API to be used in the following environments.

Structure in the SDEV Environment

Input Sent by Input Director

The input state is determined by the state of the digitizer as sent from the host PC by Input Director.

To send digitizer input, Input Director must be set to capture mode, which is when sending input from mouse and keyboard is enabled.

We recommend enabling the Allow my pen to act as mouse in legacy applications setting in advance to prevent unexpected digitizer input from the host PC.

The following equipment can be used as a digitizer.

  • Wacom Cintiq Pro 16

Structure in the EDEV Environment

The same as the SDEV environment.
