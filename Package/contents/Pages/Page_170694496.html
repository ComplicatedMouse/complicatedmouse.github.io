Build Procedure: Creating a Static Library

Creating a Static Library

Creating the a File

Using the ar tool, you can combine multiple object files and static library files together into a single static library file.

The following example shows a simple way to use the NintendoSDK ar tool. The regular ar tool is used the same way.

ar
    <output file>
    <input file>...

Archiver Option Restrictions

Currently there are no restrictions on archiver options.
