3D News

This document is a graphical introduction to the features that were added in NintendoSDK 4.0.0.

3DEditor

Parent Materials Can be Separated by Category

In the widows for adding and changing parent materials, you can now select parent materials from among categories specified in the team settings.

Related Links For more information, see the following pages.

Material Property Window: Parent Material Page

You can use the Edit Parent Material window to add and edit parent material models and parent materials.


By configuring the team settings in the following way, you can define all the files in a certain folder as belonging to a single category.

<FileIo>
  <ParentMaterialPath Name="BOY" >%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\boy</ParentMaterialPath>
  <ParentMaterialPath Name="TOWN" Recursive="true">%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\town\</ParentMaterialPath>
</FileIo>

Wildcard characters and other special notation are supported.

Related Links For more information, see the following pages.

Team Settings File Format This section describes the components of a team settings file. See the <ParentMaterialPath> element.


No More Restriction on Use of the Team Settings <SamplerNamePreset> Element

Until now, the <SamplerNamePreset> element of the team settings could only be used if the <FollowDccSamplerNameRule> element was set to true. That restriction has ended and it can now be used, even when the other element is set to false.

Note that the behavior has changed in NintendoSDK 4.0.0 and later versions, such that if <SamplerNamePreset> is defined, the sampler name written to that element will be shown as a selection candidate, even if <FollowDccSamplerNameRule> is set to false in the existing team settings.

Related Links For more information, see the following pages.

Deprecated the --preview_target (-pt) Command Line Option

The --preview_target (-pt) command line option has been deprecated. If this option is specified, the error dialog will be displayed when 3DEditor starts. There is no alternative. Revise the startup options so that this option is not specified.

The toolbar UI for selecting which platform to connect to has been rearranged for clarity. In addition, the icons for selecting the PC or the development hardware were replaced by written characters.
