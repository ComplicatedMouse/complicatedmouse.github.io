Plug-In for Setting Motion Mirroring

Sets the motion mirroring attribute to transform nodes and joint nodes.

This plug-in enables searching for settings and verifying whether the settings are correct.

How to Use

Select Nintendo > Set Motion Mirroring to display the window.

Selected transform nodes and joint nodes are displayed under Selected Transform Node.
If four or more nodes are selected, names are not displayed and only the number of nodes selected is displayed on the right edge.

Setting Method

When one or more transform nodes or joint nodes are selected, the current setting for the selected node is displayed under Set. If you select multiple nodes or groups, the setting for the first selected node is displayed.

Use Type to change the type of motion mirroring.

Motion Mirroring Type Description
Auto The type is set automatically to Center, Side, or None at binary conversion time based on the bone's position and the parent-child relationship.
Center Plays the inverse of the model's own bone animations.
This setting is for bones in the center of the model.
Side Plays the inverse of the bone animation for bones that have left/right symmetry.
This setting is for bones that have axial symmetry.
None Mirroring is not played.

You can select multiple nodes to change the settings for all selected nodes at the same time.

You can select a different node by selecting another node without closing the window.

Searching for Attributes

To select particular transform nodes and joint nodes, in Search, select the check boxes for the nodes, and then click Search.

Batch Setting From the Joint Node Side Attribute

From the menu, select Edit > Set Types from Joint Side, and use the side attribute to set the motion mirroring type for all joint nodes below the selected node. (Nothing is set for other nodes.)

Checking Whether Settings Meet the Motion Mirroring Specifications

From the menu, select Check > Check Motion Mirroring to check whether the attribute settings for all nodes below the selected node meet the motion mirroring specifications.

If there are not any problems, "OK" appears in the command response.
If there are problems, "NG" appears in the command response and details are sent to the script editor. In addition, the node with the problem is selected.

For the specifications of motion mirroring, see Motion Mirroring.
