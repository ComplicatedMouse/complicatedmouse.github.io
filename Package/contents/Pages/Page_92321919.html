Editing Effect Data

This page describes how to edit and save edits to effect data.

Preparations

Related Links Before beginning, complete the tasks described in Quick Start > Creating Effects.

The procedures described up to this point create the JustOne emitter () directly under the JustOne emitter set () in the workspace, as shown below.

Selecting Emitters

This section describes how to select emitters.
Click to select this JustOne emitter () in the JustOne emitter set () and display its settings on the right side of the property window.

Switching Tabs

At the top of the property window is a row of tabs () that group the features for the node that was selected in the workspace.
Click the tab () that you want to configure to switch to the settings () in the tab.
The following tabs appear when you select the JustOne emitter () of the JustOne emitter set ().

Info

If any settings () in this tab () have been changed from their default values, the tab color changes to orange ().

Editing Values

This section describes how to edit the selected emitter.
In the JustOne emitter set (), in the JustOne emitter () click the Emitter tab. Then in the Emitter Transform group of features, use the following procedure to edit the X value of Position (±) .

  • You can change the values of the text box () by dragging the dial () located to the right of the settings (). The following items describe the controls.
  • To increase or decrease the value, click and move left or right.
  • To increase or decrease the degree of increase or decrease of the value when moving left and right, click and move up or down.
  • You can also enter a value directly by clicking Value = 0 (). This tutorial sets the value to Value = 10.123 ().
  • Each click of the spin up button () increases the value by one step. Each click of the spin down button () decreases the value by one step.
  • After clicking the Adjust Default button () and intuitively setting the value using the dial (), click Adjust Default () again to round off the decimal fraction.
    For example, if you had Value = 10.123 () and clicked the Adjust Default button () once, the least-significant decimal place would be rounded down to Value = 10.12 ().
  • Random Settings () sets a random number for the preceding settings.

Editing these values adds an edited mark () to the right of JustOne () and JustOne () to indicate that the value has been edited.

Info

If any settings () have been changed from their default values, the color changes to orange ().

Saving Changes

Save saves the edited emitter set.
In the workspace, select the JustOne emitter set (), and do one of the following.

  • Right-click and select Save () from the shortcut menu.
  • From the main menu, select File > Save ().
  • The keyboard shortcut is CTRL+S.

When a dialog box appears, specify a filename and the save destination, and then click OK.
Confirm that the edited mark () to the right of JustOne () is no longer displayed.

Selecting and Closing an Emitter Set

This section describes how to close an emitter set.
In the workspace, select the JustOne emitter set (), and do one of the following.

  • Right-click and select Close () from the shortcut menu.
  • From the main menu, select File > Close ().
  • The keyboard shortcut is CTRL+W.

The JustOne emitter set () has been saved already, so the file is closed and removed from the workspace.

Info

If the emitter set has not been saved since being edited, a dialog box appears to prompt you to save it before closing.
