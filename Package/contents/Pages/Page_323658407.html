Overview

The demos in the vibration demo collection show how to use the HD rumble feature effectively; they can be run on an EDEV or SDEV unit.


The vibration demo collection enables you to experience how adding vibrations to a variety of in-game scenarios changes the feel of the controls and the feedback that you are given.


The vibration example collection currently includes the following vibration demos.

Name Description
UI Demo A demo that adds vibration to buttons, sliders, knobs, switches, and other UI components.

Location

The vibration demo collection is included with the NintendoSDK NX Add-On. You can find it at the following path.

TargetTools/NX-NXFP2-a64/VibrationDemo/Release/VibrationDemo.nsp

Description of the Demo

UI Demo

The UI demo enables you to experience vibration has been added to buttons, sliders, knobs, switches, and other UI components.

How to Operate

The UI demo allows you to interact with UI components on the screen using the Joy-Con and touch screen.

(Some UI components can be controlled by the gyro sensors.)


The following four types of components are shown at the top of the screen, with one each on the left and on the right.

  • Buttons
  • Sliders
  • Knobs
  • Switches

Controls on the left of the screen work differently from those on the right.


You can use the directional buttons and control sticks to change which UI component currently has focus.

When a UI component has focus, it is emphasized by a light blue border.

You can interact with each UI component by pressing the A Button while it has focus.


When you press the A Button to interact with a slider or knob, its border changes color from light blue to yellow.

In this state, you can use the directional buttons and control sticks to manipulate the UI.

Press the B Button if you would like another UI component to be in focus.


You can find instructions for controlling each UI component in the following table.
UI Component How to Operate
Buttons Use the A Button to begin pressing the UI button.
Sliders Change the current value using the left and right directional buttons or by moving the control sticks left and right.
Knobs

Change the current value using the left and right directional buttons or by moving the control sticks left and right.

Change the current value by rotating the right Joy-Con (as shown in the following figure) while holding down the A Button.
Switches Use the A Button to switch this setting on and off.

Enabling and Disabling Vibration
  • L Button: Enable vibration.
  • R Button: Disable vibration.


You can look at the upper-left corner of the screen to determine whether vibration is currently enabled.
