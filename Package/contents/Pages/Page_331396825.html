NvPerf Library

Overview

The NvPerf library is a library for measuring GPU performance in the NX environment.

Getting the metrics for various units in the GPU and identifying the locations of bottlenecks can be very useful in improving GPU performance.

The NvPerf library is intended to be used for application debugging purposes. ROMs that have used this library cannot be uploaded to the Nintendo Developer Portal (NDP) as master ROMs.
