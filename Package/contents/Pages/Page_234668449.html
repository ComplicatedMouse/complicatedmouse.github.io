Starting Maya

Start Maya by double-clicking () the batch file for the Maya version you will use in the following folder.

Maya%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\ResourcesStartMaya-2017.bat
StartMaya-2018.bat
StartMaya-2019.bat

Related Links For more information about the startup batch file, see the Introduction for the Maya plug-in.

Info

These Maya startup batch files use a config file (Export.ini) that is designed for the tutorial.
They are configured to output the color animation, texture SRT animation, and texture pattern animation, together, in the format of an FMAB file.

Related Links For more information, see Converting FSP (FCL, FTS), FTP, and FVM Files to FMAB Files in the NintendoSDK Migration Guide for NintendoWare 3D.
