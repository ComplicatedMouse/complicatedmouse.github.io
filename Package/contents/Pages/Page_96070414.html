Rotation Speed

Rotation speed(±)

This section describes the Rotation Speed feature for setting the speed at which particles rotate.

Overview

The Rotation Speed feature adds the specified value for each frame to the particle tilt to achieve rotation animation at constant velocity.
The following overview describes the settings ().

Rotation speed(±)

Sets a standard rotation speed. The specified value is added to every frame for animation.

Use Random Settings () to specify the ranges within which to randomize the values.
The specified random value is added to the particle rotation speed.
For example, if 1 degree is specified, a random value between -2 and 2 will be added to the Rotation Speed value, and this will be the final rotation speed.
