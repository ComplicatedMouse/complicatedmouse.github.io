History Air Resistance

This sections describes the History air resistance feature for increasing or decreasing the overall speed of the history using air resistance.

Overview

The History air resistance feature increases and decreases the speed of the history.
If an initial rate or an acceleration has been set for the history, the value set for History air resistance will effect the stripe.
If the initial rate or the acceleration for the history has been set to 0, the value set for History air resistance will not effect the stripe.
The following overview describes the settings ().

History Air Resistance

Set the air resistance to apply to the history using the dial () or by entering a value in the text box ().
A value greater than 1.0 speeds up the history, while a value of less than 1.0 slows it down.
