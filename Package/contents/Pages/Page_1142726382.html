Question

Display of the alpha part differs between the texture thumbnail and in LayoutViewer.

Texture Thumbnail LayoutViewer

Answer

If the registered texture does not have an alpha component, use the Texture window to change the format to a single-channel texture like BC4 or A8.

Info

For texture thumbnails, if the TGA file does not have an alpha component, the alpha part is simply shown as white, regardless of the format.
With LayoutViewer, if a TGA file of this type is set as a two-channel texture such as BC5 or LA8, the R channel is deliberately assigned for the alpha by LayoutConverter.
For this reason, the display of the alpha part differs in texture thumbnails and in LayoutViewer.
