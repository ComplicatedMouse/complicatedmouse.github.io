3DEditor

Enabled automatically adding an additional reference path for each file when a texture that is not in the search path is assigned.

Added the <AutoSaveTextureSearchPaths> option to the file input/output settings in Team Settings.

If you add a reference path here, an additional reference path is automatically set in the FMD file if there is a texture that is not in the search path.

Photoshop Plug-In

Accelerated the process of exporting textures.

The processes of exporting large textures and of exporting numerous slices of large images now go faster.

For example, when an image that is 8192 × 8192 pixels is sliced and exported as four PNG files of 4096 × 4096 pixels each, it now takes roughly 3 seconds instead of roughly 5 seconds (measured on a PC with a Core i7-8700K 3.70GHz processor).



Maya Plug-In

Information about export items that fail to be exported with the Multi-Export plug-in is now displayed using error messages.

To make errors and warnings easier to notice, information about export items that failed to be exported is now displayed using error messages.
When warnings have occurred during export, the total number of warnings is displayed with a yellow background.

Errors are displayed with a red background.

The names of items that fail to export are displayed in the script editor as follows.

If you run the multi-export process using the following script, the error information and the total number of warnings are displayed in dialog boxes.

if (NintendoMultiExportDirect() > 0)
{
    confirmDialog -message (getLastError()) -button "OK";
}

if (NintendoMultiExportGetTotalWarningCount() > 0)
{
    confirmDialog -message ("Total number of warnings: " + NintendoMultiExportGetTotalWarningCount()) -button "OK";
}

Added support to the Import plug-in for situations where the FMT files are under FMD_file_folder/Materials.

If the FMD file references external materials, the FMT files that are in the following folders are now imported.

  1. FMD_file_folder
  2. FMD_file_folder/Materials

The FTX files (the image files) in the following folders are imported for the textures referenced by that FMT file.

  1. FMT_file_folder
  2. FMT_file_folder/textures
