Getting Started

The Nintendo NX CPU Profiler analyzes code and system performance on SDEV and EDEV development kits.

You can expect to:

  • Inspect and visualize function and thread performance.
  • Understand and diagnose thread synchronization issues.
  • Optionally log detailed performance characteristics of user defined areas of code.
  • Review static and dynamic checks that automatically detect common performance issues.

For a full list of features, see the Overview of Features section.

Start the Profiler

You can find the profiler application, , in your SDK directory (NintendoSDK\Tools\NintendoNxCpuProfiler). If you do not have this directory, please use the NDI client to download the Nintendo NX CPU Profiler package.

Requirements

The profiler requires .NET 4.6.1 or later and DirectX 11. For the best support of high DPI monitors (such as 4K monitors), we recommend having .NET 4.6.2 and Windows 10 version 1703 or later.

Note: Since the profiler uses DirectX 11 to render graphs, a dedicated graphics card provides the best interactive experience.

Take your first Profile

We'll first walk you through the fastest way to take a profile for the first time. If you have an EDEV or want to enable deeper data collection of thread synchronization, GPU, audio, or user logged data, then please see the Full Integration Setup section.

  1. Launch your game on an SDEV, but do not launch it as a debug session through Visual Studio. If using Visual Studio, launch it with Start without Debugging.
  2. Launch the Nintendo NX CPU Profiler (NintendoSDK\Tools\NintendoNxCpuProfiler\Nintendo CPU Profiler.exe). If you don't have it, then download the Nintendo NX CPU Profiler package through the NDI client.
  3. Click the  Sync button on the Home tab to connect to your SDEV.

  4. In the Select a Profiling Scenario pop-up dialog, go to the Select NSS section of the Attach tab, click the  button to browse and find your game's .nss file, and then click OK.
  5. To take a profile, click the  Start button on the Sampled Profile tab. Anytime after starting, click the  Stop button to end collecting data.

  6. Wait for the data to load and then you can explore your game's profile data.
