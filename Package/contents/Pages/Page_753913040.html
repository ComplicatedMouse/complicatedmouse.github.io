Profiler-Filter

Overview

This option is the default form of Profiler-Filter. It will result in the Filter/Find box on the tab being modified.

Profiler-Filter [-Tab] <ScriptingTabs> [-Negate] [-RegEx] [-MatchCase] [-FilterName] <String>


[-Tab] <ScriptingTabs>

Specifies the tab to filter items on. The following values are supported for this argument:

    • CallTree: Call Tree Tab
    • CodeCoverage: Code Coverage Tab
    • Counters: Counters Tab
    • Files: Files Tab
    • Functions: Functions Tab
    • GraphicsGroups: Gfx Groups Tab
    • Instrumented: Instrumented Tab
    • Last: Last Tab
    • Logs: Logs Tab
    • Shaders: Shaders Tab
    • Threads: Threads Tab


[-Negate]

When specified, the results are negated.

[-RegEx]

When specified, the filter is a regular expression.

[-MatchCase]

When specified, perform a case-sensitive filter. When it is not specified, the filter is case-insensitive.


[-FilterName] <String>

The text to use for the filter. It is recommended that this text be placed in quotation marks, especially if this is a regular expression filter.

Limit (DEPRECATED)

This option will either limit the visible functions to 1000 lines or show all functions in the tab.

DEPRECATED: This option no longer has any effect since the Functions and Last tab always show all functions.

Profiler-Filter [-Tab] <ScriptingTabs> -Limit <Boolean>

[-Tab] <ScriptingTabs>

Specifies the tab to filter items on. The following values are supported for this argument:

    • Functions
    • Last

-Limit <Boolean>

Whether or not to limit the visible results.

Show Selected

This option will set the Function Tab to only show the selected functions.

Profiler-Filter [-Tab] <ScriptingTabs> -ShowSelected <Boolean>

[-Tab] <ScriptingTabs>

Specifies the tab to filter items on. The following values are supported for this argument:

    • Functions

-ShowSelected <Boolean>

Whether or not to show only the selected items.

By Core

This option will filter the currently visible results by only showing the results from the specified core.

Profiler-Filter [-Tab] <ScriptingTabs> -Core <Object>


[-Tab] <ScriptingTabs>

Note

This command is not enabled for the Functions Tab when Thread-Centric Data Views are not enabled. See Experimental Features Tab for additional information.

Specifies the tab to filter items on. The following values are supported for this argument:

    • Counters
    • Functions
    • Instrumented
    • Last
    • Threads

-Core <Object>

Specify either the Integer core number to view, or the string "All" to show all cores.

By Thread

This option will filter the currently visible results by only showing the results from the specified thread.

Profiler-Filter [-Tab] <ScriptingTabs> -Thread <String>

[-Tab] <ScriptingTabs>

Specifies the tab to filter items on. The following values are supported for this argument:

    • Functions
    • Instrumented
    • Last
    • Logs

-Thread <String>

The name of the thread to filter. Specify "All" to see all threads.

By Group

Note

This cmdlet variation is deprecated when the Thread-Centric Data Views option is enabled.

Profiler-Filter [-Tab] <ScriptingTabs> -Group <String>


[-Tab] <ScriptingTabs>

Specifies the tab to filter items on. The following values are supported for this argument:

    • Counters


-Group <String>

The name of the group to filter. Specify "All" to see all groups.

By System

Profiler-Filter [-Tab] <ScriptingTabs> -System <String>


[-Tab] <ScriptingTabs>

Specifies the tab to filter items on. The following values are supported for this argument:

    • Functions


-System <String>

The name of the system to filter. The name is the same as what is displayed in the  System Classification Filter . Any value that does not match one in the filter results in defaulting back to the All Systems option.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Filter Functions -RegEx "^OS"
Profiler-Filter Functions -Negate -MatchCase -FilterName "nn::profiler"
Profiler-Filter Functions -ShowSelected $true
Profiler-Filter Threads -Core 0
Profiler-Filter Functions -Thread "ThreadName"
Profiler-Filter Counters -Group "GroupName"
Profiler-Filter Functions -System "Graphics"
