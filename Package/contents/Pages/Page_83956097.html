Creating an Emitter Set

This section describes how to create an effect.
When creating effect data, start by creating an emitter set.
The dialog box appears when you perform either of the following operations. Check that EmitterSet1 is set as the name for the emitter set, and then click OK.

  • From the main menu, select File > New > Emitter Set ().
  • In the workspace, right-click Effect () or the emitter set (), and on the shortcut menu, select Add New Emitter Set ().
  • On the toolbar in the workspace, click Add New Emitter Set ().
  • Press CTRL+N.

The EmitterSet1 () emitter set is created in the workspace.

Creating Emitter Sets From Presets

This section describes how to create emitter sets from presets.
You can save effect data that you use often in a "preset" folder, which allows you to use the data when creating emitter sets.

The following example is based on JustOne, the emitter set preset that comes pre-registered as a sample.
The dialog box appears when you perform either of the following operations. Check that JustOne is set as the name for the emitter set, and then click OK.

  • From the main menu, select File > New > JustOne ().
  • In the workspace, either from Effect () or from the shortcut menu that opens when you right-click , select Add from preset () > JustOne ().

The JustOne () emitter set is created in the workspace.

Adding Emitters

This section describes how to add emitters.
Select either EmitterSet1 () or the Emitter1 () emitter immediately below this emitter set in the workspace, and perform any of the following operations. In the dialog box that opens, confirm that the name of the emitter is Emitter2, and then click OK.

  • From the main menu, select File > New > Emitter ().
  • Right-click, and select Add New Emitter () from the shortcut menu.
  • On the toolbar in the workspace, click Add New Emitter ().
  • Press CTRL+SHIFT+N.

The Emitter2 emitter () is added under the Emitterset1 emitter set () in the workspace.
Look for the mark to the right of EmitterSet1 (), indicating that the content of the emitter set has been changed.

Info

Multiple emitters can be created directly under an emitter set.

Related Links For more information about the emitter node, see Emitter Node.

Adding Emitters From Presets

This section describes how to add emitters from presets.
Emitters can be used as presets if they are used by emitter sets in the presets folder.
Under this emitter set in the workspace, select either EmitterSet1 () or Emitter2 () and then perform any of the following operations. In the dialog box that opens, confirm that the name of the emitter is JustOne (), and then click OK.

  • From the main menu, select File > New > JustOne () > JustOne ().
  • In the workspace, select EmitterSet1 , right-click to open the shortcut menu, and select Add from Preset () > JustOne () > JustOne ().
  • In the workspace, select Emitter2 (), right-click to open the shortcut menu, and select Add from Preset () > JustOne () > JustOne ().

The JustOne emitter () is added below the EmitterSet1 emitter set () in the workspace.
