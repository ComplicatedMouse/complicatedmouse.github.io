Alpha Test

This section describes the Alpha Test feature used to test the alpha value of the particle to be drawn.

Overview

This feature tests whether the alpha value of the framebuffer and the alpha value of the particle to be rendered is totally transparent.
The following overview describes the settings ().

Alpha Test

Enable this setting to display the Alpha Test () settings.
The following overview describes the settings ().

Conditions for Passing the Test

Select the alpha test passing conditions from among the presets.

Reference Value

This setting is a value to compare against the alpha value in the alpha test.
