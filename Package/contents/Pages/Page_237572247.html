5. Texture SRT Animation

Create a texture translation animation.

Texture SRT

Use Frame translation to create animation keys.

Using the attribute editor, open the place2dTexture node that is being used with the Color attribute for the TextureSRTMt material, and create animations keys in Frame translation of the 2D texture placement attributes.

Create a key like the one in the following table.
0 frames
50 frames
Frame translation 0.000 0.000 1.000 0.000

Check that the texture becomes an animation that moves sideways as in the following figure.

Export Target

The texture SRT animation export targets are the key frames of Frame translation, Frame rotation, and Repeat UV of the 2D texture placement attributes.

This corresponds to C, Texture SRT Animation Data, in the Output File Selection of Nintendo Export Options.

Related Links For more information, see Texture SRT Animation.

Info

Combines each of the color, texture SRT, and texture pattern animations into a single FMAB file and exports the file.
Texture SRT animations can also be exported as stand-alone FTS files.

Related Links For more information, see Using the Export Plug-In.
