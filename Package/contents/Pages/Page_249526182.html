How Do I Initialize Multiple Development Hardware Devices?

The standard way to initialize multiple development hardware devices is to perform sequential execution of InitializeSdev.exe or InitializeEdev.exe for the number of devices you want to initialize.

For example, if you have three SDEV devices with the IP addresses 192.168.10.10, 192.168.10.11, and 192.168.10.12 that you want to initialize, you would script a BAT file as follows.

@echo off

Tools\CommandLineTools\InitializeSdev.exe --target 192.168.10.10
Tools\CommandLineTools\InitializeSdev.exe --target 192.168.10.11
Tools\CommandLineTools\InitializeSdev.exe --target 192.168.10.12

Initialize multiple EDEV devices in the same way.

The BAT file codes for sequential execution of the initialization tool.

InitializeSdev also has a mechanism for parallel execution to initialize multiple devices at the same time. To use it specify the --keep-targetmanager-alive option as follows. This is a trial feature and sometimes fails. Please make a note of this fact.

InitializeSdev --target 192.168.10.10 --keep-targetmanager-alive

NX Add-On 4.1.0 and later versions provide a tool script named DevKitSetup.ps1 for this task. That can also be used.
