Position Follows Emitter

Feature Description Template

The section describes the Position Follows Emitter feature that makes the position of convergence follow the emitter when the emitter moves.

Overview

The Position Follows Emitter feature makes the position of convergence follow the emitter when the emitter moves.
This setting is useful when the Particle follow type of an emitter is set to Do not follow emitter.

Position Follows Emitter

Use the check box () to select whether to make the position of convergence follow the emitter when it moves.
The following formula calculates the value added to the position of a particle when Position Follows Emitter is enabled.

  • Value to add to position: (Position of the emitter at that time + Convergence position - Position of the particle at that time) × Convergent force
