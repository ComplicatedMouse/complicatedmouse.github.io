Advanced Settings for Placing Text in a Circular Shape

This section describes how to configure the options for placing text in a circular shape, including setting the origin point for conversion as the center of the text box, and placing text at fixed intervals.

Overview

Use Place text in a circular shape to automatically configure the text box pane parameters and per-character transformation curve.
The following sections are for the various text box pane settings for placing text in a circular shape.

Set the origin point for conversion as the center of the text box

This feature allows you to use the coordinates at the center of the text box as the coordinates of the origin when placing text in a text box using the per-character transformation feature.
It is mainly used when placing text in a circle to temporarily place characters in the middle of a text box pane, before setting x- and y-coordinates for the text and rotating it.

The settings that can be configured for per-character transformation change depending on whether this feature is enabled or disabled.
When the feature is enabled, the following settings are unavailable, and cannot be configured.

  • Origin (V Direction)
  • Height Offset

In contrast, one of the following settings becomes available.

  • Place characters at fixed intervals
  • Insert left and right margins

Place characters at fixed intervals

This feature allows you to set kerning to a fixed value, and set how the text is justified when the center of the text box serves as the origin of the transformation.
It is mainly used to prevent excessive spacing between characters when a short text string is placed on a circle with a large radius.

You can configure the following options when Place characters at fixed intervals is enabled.

Kerning (by duration)

Configured using kerning values.

Alignment

Select left, center, or right for the character alignment.

Insert left and right margins

The Insert left and right margins feature inserts margins based on the text width compared to the pane width.
It is mainly used to retain proportional information for characters and set equal spacing between them when placing text in a circle, even when the number of characters is reduced.

Enabling Insert left and right margins also enables setting the following option.

Alignment

Select left, center, or right for the character alignment.
