Overview

DevKitSetup.ps1 and DevKitSetup.bat are CUI tool scripts that provide an efficient way to initialize the firmware for multiple SDEV, EDEV, and HDEV development hardware devices.
They perform the following tasks.

  • Initialize multiple SDEV/EDEV devices specified at the same time.
  • Batch initialize SDEV devices connected to a PC.
  • Batch initialize EDEV and HDEV devices connected to a PC.
  • Try to setup multiple SDEV devices at the same time (a trial feature).

The setting up of multiple devices can be more safely performed by calling InitializeSdev.exe and InitializeEdev.exe using the recommended procedures.
The tool scripts are helpful for setting up and auto-testing multiple devices.

You can use DevKitSetup.ps1 from the PowerShell console, and DevKitSetup.bat from the command prompt.

Package

DevKitSetup is included in the NintendoSDK System Updater for NX packages.

Location

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/DevKitSetup.ps1
$NINTENDO_SDK_ROOT/Tools/CommandLineTools/DevKitSetup.bat

To run these scripts, you can also use the InitializeSdev.exe and InitializeEdev.exe files included in the SystemUpdater package.

Command Definitions

Option Format

Tools\CommandLineTools\DevKitSetup.ps1 [-TargetsSdev <String[]>] [-TargetsEdev <String[]>] [-ConnectSdevAll] [-ConnectEdevAll] [-SdevTryConcurrent]
 


Tools\CommandLineTools\DevKitSetup.bat [-TargetsSdev <String[]>] [-TargetsEdev <String[]>] [-ConnectSdevAll] [-ConnectEdevAll] [-SdevTryConcurrent]


Option Overview 
-TargetsSdev <IP address, serial number, MAC address>
 Specifies a number of SDEV devices using either IP address, serial number, or MAC address of each SDEV, separated by commas. 
-TargetsEdev <serial number>
 Specifies a number of EDEV or HDEV devices by using their serial numbers, separated by commas. 
-ConnectSdevAll
 Sets up all SDEV devices that can be recognized from the PC. 
-ConnectEdevAll
 Sets up all EDEV and HDEV devices that can be recognized from the PC. 
-SdevTryConcurrent
 Tries to conduct a concurrent setup of SDEV devices. (This is a trial feature.)

When you specify -ConnectSdevAll or -ConnectEdevAll, you cannot use -TargetsSdev or -TargetsEdev.

Determining Whether Execution is Successful

You cannot set an exit code to determine the result of these scripts.

Refer to the log.

How to Use

The following examples show different ways to use these scripts.

Setting Up SDEV Devices by Using Serial Numbers

Tools\CommandLineTools\DevKitSetup.ps1 -TargetsSdev XAW02121000001,XAW02121000002
 


Tools\CommandLineTools\DevKitSetup.bat -TargetsSdev XAW02121000001,XAW02121000002

You can specify multiple devices by separating the entries with commas.

Setting Up SDEV Devices by Using IP Addresses

Tools\CommandLineTools\DevKitSetup.ps1 -TargetsSdev 192.168.10.10,192.168.10.11
 


Tools\CommandLineTools\DevKitSetup.bat -TargetsSdev 192.168.10.10,192.168.10.11

Setting Up SDEV Devices by Using MAC Addresses

Tools\CommandLineTools\DevKitSetup.ps1 -TargetsSdev 7C:BB:8A:29:F5:0F
 


Tools\CommandLineTools\DevKitSetup.bat -TargetsSdev 7C:BB:8A:29:F5:0F

Setting Up EDEV Devices by Using Serial Numbers

Tools\CommandLineTools\DevKitSetup.ps1 -TargetsEdev XAL03100000001,XAL03100000002
 


Tools\CommandLineTools\DevKitSetup.bat -TargetsEdev XAL03100000001,XAL03100000002

Setting Up HDEV Devices by Using Serial Numbers

Uses the same execution procedure as for EDEV.

Setting Up SDEV and EDEV Devices by Specifying Simultaneously

Tools\CommandLineTools\DevKitSetup.ps1 -TargetsSdev XAW02121000001 -TargetsEdev XAL03100000001 


Tools\CommandLineTools\DevKitSetup.bat -TargetsSdev XAW02121000001 -TargetsEdev XAL03100000001

Use a command such as this to set up SDEV and EDEV devices at the same time.
Specifies the HDEV serial number in TargetsEdev to execute HDEV setup.

Setting Up All SDEV Devices Recognized From the PC

Tools\CommandLineTools\DevKitSetup.ps1 -ConnectSdevAll
 


Tools\CommandLineTools\DevKitSetup.bat -ConnectSdevAll

Note that this method will set up all SDEV devices that are connected to the company LAN.

Setting Up All EDEV/HDEV Devices Recognized From the PC

Tools\CommandLineTools\DevKitSetup.ps1 -ConnectEdevAll
 


Tools\CommandLineTools\DevKitSetup.bat -ConnectEdevAll

Trying to Concurrently Setup Multiple SDEV Devices

Tools\CommandLineTools\DevKitSetup.ps1 -TargetsSdev 192.168.10.10,192.168.10.11 -SdevTryConcurrent
 


Tools\CommandLineTools\DevKitSetup.bat -TargetsSdev 192.168.10.10,192.168.10.11 -SdevTryConcurrent

Use the -SdevTryConcurrent option to try to set up a number of SDEV devices all at the same time. This feature may sometimes fail. Please take note of this fact.


Notes

  • To run DevKitSetup.ps1 from PowerShell, you must set the execution policy to Unrestricted.
  • The script sometimes fails when the -SdevTryConcurrent option is specified. When that occurs, run the script without this option.
