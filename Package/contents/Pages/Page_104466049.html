Specify Model File

This section describes the Specify File feature for models.

Overview

Registers a model generated by the DCC tool, and displays it in the viewer.
The size and color of the model can be adjusted with the registered model and effects linked together in the viewer.
This section provides an overview of the settings ().

Specify File

Registers, updates, and deletes primitive files. The following are button types.

Load Open the file selection window and select a model file (FMDB file).
Reload Updates the specified model file (FMDB file).
Clear Releases the specified model file (FMDB file).

Related Links For more information, see the following pages.

Model Files For Linking This page describes model files that can be linked with EffectMaker and how to work with the files referenced by these model files.
Model to Constrain Selects which model to link from among the model files registered in the Model Settings node.
Bone to Constrain Selects which bone you want to emit the effect from the linked model selected in Model to Constrain.
