Load the template effect into EffectMaker using the following procedure.

2

When a window opens, select and open the emitter set file (ESET file) from the following location.

3

Selecting the Debug Tab

In the Workspace, select Viewer node ().
In the tabs () in the Property window, select the Debug tab.
The settings for the viewer's debug feature are displayed.

Changing the Glare Intensity

To adjust the glare intensity, configure the following settings item ().

Item Value Description
Glare Intensity 2 Changes the glare intensity to the second level.

From the tabs () in the Property window, select the Color tab.

Item Value Description

Pattern split

U = 8

Equally divides the texture into eight horizontally and into four vertically.

Use the pattern split setting to divide the RGB and alpha images for texture 0. The results are assigned numbers.

Configuring Texture Pattern Animations

In Texture Pattern Animation (), configure the texture pattern using the following items ().

Item Value Description
Playback Methods Loop Plays back the texture pattern in a loop.
Looping and pattern table options become available. Configure them as shown.
Sequential Frames per Pattern 2 Displays a single texture continuously for two frames.
Number of Pattern Tables To Use 32 Sets to the number of images that divided the number of texture patterns.
Table Easy Input 0-31

Continually plays back, in numerical order, from 0 to 31, the divided texture 0.
