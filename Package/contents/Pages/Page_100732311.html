NintendoWare 3D News

This page introduces features that were added to the 3D module in NintendoSDK 0.8.0.

3DEditor

  1. Added a feature for inserting keys to the Curve View shortcut menu.
  2. Added the Snap Keys feature to Curve Editor.
  3. Changed the shortcut key for Translate Current Frame and other functions in Curve Editor.
  4. Addition of Feature to Switch Between Color Spaces in the 3DEditor Color Picker.

Added a feature for inserting keys to the Curve View shortcut menu.

Added a feature for inserting keys to the selected curve in the Curve View shortcut menu.
The key is inserted in the current frame position.

Added the Snap Keys feature to Curve Editor.

Added the Snap Keys feature to Curve Editor.

When Snap Keys is selected and you move the key in Curve View, frames and values are snapped for other keys in Curve View.
A dotted line displays in the Curve Editor for snapped keys.

Possible Uses

This feature is useful when editing curves to translate the same value as a key in another curve or to translate a key to a frame.

The C key enables or disables Snap Keys.

Changed the shortcut key for Translate Current Frame and other functions in Curve Editor.

Changed the shortcut keys that can be used with Curve Editor.

Shortcut Keys of Eliminated Functionality

The following shortcut keys can no longer be used with Curve Editor.

Operation Description State
UP/DOWN ARROW KEYS (↑ ↓) Edits the value of the selected key. No longer available in version 0.8.0.

Shortcut Keys for Added/Changed Functionality

The following shortcut keys have been added or changed.

Operation Description State
ANGLE BRACKETS ( < > ) For all the keys displayed in Curve View, jumps to the position of the key before or after the current frame. The addition of a new feature.
LEFT/RIGHT ARROW KEYS (← → ) Shifts the selection state of the keys in Curve View to the next key to the left or right. In previous versions, this shortcut functioned to edit the frame of the selected key.

Addition of Feature to Switch Between Color Spaces in the 3DEditor Color Picker.

Added a feature to switch between color spaces in the 3DEditor color picker.

The presets sRGB, linear RGB and HSV are the available options for the color space.

Possible Uses

This feature is useful when you want to incorporate textures with material colors into different color spaces.

Example: You use a non-linear color space in your DCC tool and you want to know the LinearRGB value for a texture pixel color of 0.4.

  1. For the material color in 3DEditor, select sRGB as the color space in the color picker and then enter the value 0.4.
  2. Change the color space to LinearRGB and see that the value in linear space changes automatically to 0.13.
  3. You can see that for an sRGB value of 0.4 the LinearRGB value is 0.13, which is a useful way of matching colors.
