Diffusion Speed

This section describes the Diffusion Speed feature, which is used to specify the diffusion speed of particles in relation to the axis.

Overview

The Diffusion Speed feature is used to specify the speed at which particles diffuse from the axis.
Positive values cause particles to move away from the axis. Negative values cause particles to move toward the axis.
The following overview describes the settings ().

Diffusion Speed

The speed at which particles diffuse from the axis can be set using a dial () or text box ().

Info

Press the Particle Time Animation button () to display the Rotation Speed (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().
