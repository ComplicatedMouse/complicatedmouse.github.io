Emission

This tab configures aspects of emitter sets that can be controlled in the game application, and settings linking 3D models and effects.

Multiply Emission Settings by Scale

The Multiply Emission Settings by Scale group () is a set of features for applying scale values to the particle emission settings.
This group has the following settings.

Emission Rate　

Multiplies all particles in the emitter set by the value set for the emission rate.
Emission Interval Multiplies all particles in the emitter set by the value set for the emission interval.

Multiply Initial Settings by Scale

The Multiply Initial Settings by Scale group () is a set of features for applying scale values to the particle's initial velocity settings.
This group has the following settings.

All-Direction Initial Velocity Multiplies the All-Direction Initial Velocity for all particles in the emitter set by the specified value.
Specified-Direction Initial Velocity Multiplies the Specified-Direction Initial Velocity for all particles in the emitter set by the specified value.
Random Initial Velocity Multiplies the Random Initial Velocity for all particles in the emitter set by the specified value.

Add to Motion Settings

The Add to motion setting () feature adds a value to the velocity of particles.
This group has the following settings.

Direction (±) Adds a value in the specified direction to the velocity for all particles in the emitter set.
