Running Batch Files Before Opening a File

This feature runs specific batch files when you open a project file, sound set file, bank file, and reference file.

It can be used in applications like managing the state of a file before opening that file.

Batch File Search

Batch files are searched in the following order and only the batch file that is found first is executed.

  1. Project Name.preopen_command.bat in the same folder as the project
  2. SoundMaker_preopen_command.bat in the same folder as the project
  3. SoundMaker_preopen_command.bat in the same folder as SoundMaker.exe

Batch File Arguments

When a batch file is run, SoundMaker passes the first argument and second argument to the batch file.

The following values are passed as arguments to batch files.

Parameter Value
First parameter Path to the file to be opened.
Second parameter Path to the project file.

When attempting to open a project file, both the first and second arguments are the path to the project file.

Patch File Execution Timing

File Operation
Project file (FSPJ file)

When opening a project.
Sound set file (FSST file)

When opening a project.

When adding an existing sound set.
Bank file (FBNK file)

When opening a bank.
Reference file (such as a WAV file) When executing an open reference file.
