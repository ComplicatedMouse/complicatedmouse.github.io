Mii Helper Library

Overview

The Mii Helper Library provides features to assign nn::mii::CharModel to nn::g3d::ResFile.

Namespace

All of the features provided by the Mii Helper Library, such as shapes, constants, and variables, belong to the nn::g3d::mii namespace.

Header

You can use all of the features by including the nn/g3d/g3d_Mii.h header.
nn/g3d/g3d_Mii.h is located in the following directory.

$NINTENDO_SDK_ROOT/Include/nn/g3d/g3d_Mii.h

The other header files are located in the following directory. nn/g3d/g3d_Mii.h includes these files.

$NINTENDO_SDK_ROOT/Include/nn/g3d/mii/

Library Files

The name of the library file provided by the Mii Helper Library is libnn_g3dMii.

Sample Programs

A list of Mii Helper Library samples.

Warning

The Face Library, distributed separately, is required before you can build the sample program. If you need the Face Library, contact your Nintendo Developer Support Group.

Sample Program Name Description
MiiG3dSimple

A simple program that displays Mii face models using g3d.

Uses a single FDM file integrating all of the bones so the processing load of bone calculations is lighter.

Mii parts cannot be changed dynamically, but the program is suited for the rendering of large numbers of Mii characters.
MiiG3dHeadwear

A sample program for drawing Mii characters wearing hats.

This sample program demonstrates the following procedures.

  • Switching nn::mii::CreateModelType to types with and without a hat.
  • Switching the visibility of parts.


How to Use

By passing the nn::g3d::ResFile and the nn::mii::CharModel that you want to assign to the Mii Helper Library, you can create a nn::g3d::ResFile that contains the Mii model data. For more information, see the following page.
