File Operations

Opening Folders

Select File > Open Folder to start Windows Explorer and open a folder.

From Windows Explorer, you can open the folders of reference files for selected regions, instruments that reference waveform files, and items in the sound set that have waveform files, standard MIDI files, or other external reference files.

The only new files that you can create by using SoundMaker are sound project files, sound set files, and bank files.
SoundMaker does not delete files except when a newly created file overwrites an old one.
Use this feature when you want to save project files under another name or when you want to organize and check waveform files.

Reloading Project Files

To reload project files that were edited outside SoundMaker, select File > Reload Project.
The load targets are sound set files (FSST files) and bank files (FBNK files).

Reloading references the sound project, and checks every individual file to confirm that it is a target file.
The reloading method depends upon whether there are unsaved edits in SoundMaker.

Case 1: When there is no update (that is, unsaved edits) in SoundMaker

Loads the newest file.

Case 2: When there is an update (unsaved edits) in SoundMaker

Confirms by using a dialog box, and then loads the newest file.
When this file is loaded, the unsaved edits in SoundMaker are lost.
Also in this case, the undo history in the corresponding pane is cleared.

If the task is canceled in the dialog box, nothing is done if the file has not been updated.
(The next time the file is saved in SoundMaker, the changed file overwrites the existing file in SoundMaker.)

The CTRL+L shortcut has been assigned to Reload Project.

Loading Bank Files From Outside the Project

To open and edit bank files (FBNK files) that are not registered in the project, select File > Open File.
The Open File submenu is available only when a project is open.

You can also open an unregistered bank file by dragging the file to a view other than a bank list view.
If you drag the bank file to the bank list view, the bank file is added to the bank list.

Drag-Loading an External File

Supports loading files by dragging project files and waveform files to the application window (including sub windows).

When external files are dragged to the application window, a [+] is displayed in the mouse pointer.
When a file is dragged to a list or the Sample Map, the insert location is highlighted.

An error appears when a file that cannot be registered is dragged to a list.
During the selection and loading of several files, if there are illegal files, only the valid files are registered, and a single error appears for all of the illegal files.

Warning When Registering a File That Is Outside of the Project Folder

If you attempt to register a file that is outside of the project folder, a warning dialog box appears to confirm whether it is okay to register the file.
This prevents the mistake of unintentionally using a file that is outside of the project folder.

Select No to cancel the file registration.

Verifying the File Version

When you load a file, a warning dialog box is displayed if the version of a project file, sound set file, or bank file is earlier than the current version.
This warning dialog box offers three choices: Upgrade the file to the current version and open, Open the file in read-only mode, or Cancel.

If you select to upgrade the file, the earlier versions of SoundMaker can no longer open the file.
