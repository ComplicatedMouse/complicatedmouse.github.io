[User Settings]: Textures

Textures

Configures environment settings for using texture files.

Show warning dialog on loading when textures are not accessible

Displays or hides a warning when an emitter set (ESET) file opens, and the texture (FTXB) file added to an emitter or child is in a location that cannot be accessed.

Automatically reload when referenced textures are changed

Select whether to automatically reload primitives when the corresponding texture (FTXB) files that have been added to emitters or children are updated.

Display a warning message when loading textures that differ from the Linear Edit Mode settings

Displays or hides a warning when the texture (FTXB) file added to an emitter or child differs from that in the Linear Edit Mode settings.
For example, you can hide warnings by disabling the setting when intentionally using non-linear texture files using the Get Square of RGB feature in a linear environment.
