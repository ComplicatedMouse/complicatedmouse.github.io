Settings UI

Each setting has a dedicated UI.
There is also a UI to display registered file information when using settings to register textures and primitives.

This section describes the dedicated UI for the various effect-related settings.

UI for Value Settings

The UI for entering values into the EffectMaker feature. The UI includes the following UI elements.

Dials (), text boxes (), spinner buttons (), and adjust buttons ()
Random Setting
UI for setting a random number as the value for a setting.
Attribute Information for Primitives
This UI displays detailed information about registered primitives for emitter shapes and particle shapes.
Texture Preview
This UI previews texture files that have been registered in the Texture tab.
Pattern Tables
This UI can be used to configure the number of patterns to use in a texture pattern animation and the order of the pattern animations.
Color Editing UI
This UI provides a way to edit selected color and alpha values.
Gradient Editor

This UI provides a way to edit particle time animations using the selected color and alpha gradients.
Random Color Settings UI
This UI can register multiple colors. The registered colors are applied at random as the particle color at emission.
Global Color Animation UI
This UI provides a way to create and edit animations, with time in the horizontal direction and values in the vertical direction, for global color and alpha.
Color Replacement Button
This UI swaps the behavior type settings for Color 0 and Color 1 or for Alpha 0 and Alpha 1.
Particle Time Curve Editor
This UI provides a graphical way to edit particle time animations, with time on the horizontal axis and values on the vertical axis.
