Orthogonal Projection Mapping

Orthogonal Projection Mapping

Drag the FLYT files to LayoutEditor to load and check these features.

Sample Layouts

  • Projection_Sample_00.flyt

Orthogonal projection mapping is used to depict a background reflected in a water droplet.

Depicting Water Drop

  • The P_Drop_00 Picture pane:
    • Mask using the stage 0 texture.
    • For stage 1 texture, select Mapping Settings > Mapping Method and specify Orthogonal Projection Mapping.
    • Use indirect texture for stage 2 and spherically distort the background.
    • Use SRT animation to make it drop.
  • The P_DropLight_00 Picture pane:
    • Superimpose to provide luster to the water droplet.
      →The fragment operation is Additive.

About Projection_Sample_01.flyt

Depicts the same water droplet as Projection_Sample_00.flyt, but in a way that involves a lighter processing load.
Insert an alpha texture for the indirect texture, and reduce the number of texture stages.

Differences From Projection_Sample_00

  • The P_Drop_00 Picture pane:
    • Stage 0 texture is set to Orthogonal Projection Mapping.
    • Indirect texture is used for stage 1, and the background is spherically distorted.
    • Also, an alpha image is added, and Texture Combiners > Alpha Settings is set to Use Minimum Value for masking.

Behavior Is Different for Each Texture Format

With indirect textures, the amount of image distortion is different depending on which format you choose.
