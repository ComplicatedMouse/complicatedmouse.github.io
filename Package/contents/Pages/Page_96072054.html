Randomness Application Interval

This section describes the feature for setting the interval over which to apply randomness.

Overview

The Randomness application interval feature enables you to specify the interval of time (frames) over which to apply random values.
When the Randomness application interval value is smaller, particles move in a wiggling motion.

This section provides an overview of the settings ().

Randomness Application Interval

Sets the interval to apply the set Random width value using either the dial () or by entering the value in the text box ().
The value uses an integer of at least one frame.
