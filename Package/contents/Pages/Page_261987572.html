kpr Library

Introduction

Purpose of This Documentation

This documentation describes the kpr (keycode processing) library features included in NintendoSDK.

Intended Audience

This documentation is intended for application developers using NintendoSDK.

Related Documentation

Use this document along with kpr Library API Reference.

kpr Library Features

The kpr library provides a group of functions for handling keyboard key codes. It provides the following features.

  • Converting to Character Codes
  • Encoding assembly for character codes, which includes dead-key processing, Roman letter input, and more.
