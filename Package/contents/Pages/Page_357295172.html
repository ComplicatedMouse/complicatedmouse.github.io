3DEditor

The timeline now allows a playback range to be set.

The control bar has been improved to allow selecting a limited playback range.
You can drag a range slider to adjust when playback starts and ends.

Animation Controls This section describes animation controls.

TGA files can now be assigned to samplers normally.

Previously, it was only possible to assign TGA files to samplers temporarily; now you can assign them normally like intermediate texture files (FTXA and FTXB files). To turn them into BNTX files to load in your game application, you must use a texture converter.

Note: You cannot specify additional options to the TGA texture converter during real-time editing in 3DEditor. This constraint is expected to be removed in the future.

All selected layer comps can now be exported.

You can now export all selected layer comps using the Photoshop texture export plug-in.
With layer comps selected in Photoshop, select File > Automate > Nintendo Export Selected Texture to export.
