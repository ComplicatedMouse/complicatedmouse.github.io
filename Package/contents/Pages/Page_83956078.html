Render Target State

Render target state represent states relating to the color target and the depth stencil target.

Information

Use nn::gfx::RenderTargetStateInfo to define the render target state information.

Depth Stencil Format

Set the depth stencil format using the SetDepthStencilFormat function.
This format must match the actual depth-stencil format.

Color Target State

Set the array of color target states using the SetColorTargetStateInfoArray function.

This array of color target states correspond, in order, to the color targets.

Color Target Format

Set the format of the color target using the SetFormat function.
This format must match the actual color target format.

Setting the Render Target State

If you are using a pipeline execution model, the render target state information is included in the pipeline information.

If you are using an execution model that does not use a pipeline, you do not need to set render target state.
