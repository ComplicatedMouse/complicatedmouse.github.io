Tool Reference

Overview

This section describes the GFX tools.

First, we cover information that is common to all of the GFX tools.

For more information about a specific tool, see the page for that tool.

Location

The GFX tools are located in the following directory.

Tools/Graphics/GraphicsTools/

How the Tool Is Provided

The tools are provided either as a DLL file or an EXE file.
The DLL tools are intended for use from a program, while the EXE files are intended to be used at a command prompt.

32-Bit Tools

The 32-bit EXE files have a 32 appended to the end of the tool name to differentiate them from the 64-bit versions. The 32-bit DLL files are located in the x86 subdirectory.
If a tool does not follow these rules for the 32-bit version, it means the tool is a 64-bit version.

Tool List

This list includes the tools provided in the gfx library.
For more information, see the page for each tool.

Icon Tool Description

Shader Compiler

Outputs a shader compiled from shader source code and variation information.

Shader Converter (DLL)

Outputs a GFX shader resource file from shader source code and variation information in a variation file.

Shader Converter (EXE)

A command-line version of the DLL Shader Converter tool.

Texture Converter

Tool for converting image files to the various DDS formats.
Also converts one or many DDS files to a texture binary file.
