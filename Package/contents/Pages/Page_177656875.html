Cutoff Threshold for Traverse Distance

This section describes the Cutoff threshold for traverse distance feature, which limits particle emission to ensure that no excess particles are emitted when emitters have a high movement speed.

Overview

The Cutoff threshold for traverse distance feature treats movement distance in a single frame as being equal to 0 if that movement distance is less than or equal to the specified value.
This feature enables you to prevent an emitter from emitting particles when it is moving slowly but allow the emitter to emit particles when it is moving at a higher speed.

Cutoff Threshold for Traverse Distance
