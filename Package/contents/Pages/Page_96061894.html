Fresnel Alpha

This section describes the Fresnel Alpha feature, which is used to change the alpha values for polygon normal vectors with respect to the view vector.

Overview

The Fresnel Alpha feature changes alpha values for polygon faces, using the angle formed by the view vector and that of the normal vector for a particle polygon as a settings value.
It is effective when using primitives for the particle shape, where there are changes to the normal vector.

The following figures indicate the relationship between settings values and the angle formed by N, the normal vector, and V, the view vector.

The following overview describes the settings ().

Fresnel Alpha

Select the method for processing Fresnel alpha from among the following presets.

None Fresnel alpha is not used.
Processing by Vertex Fresnel alpha is processed in the vertex shader.
Processing by Fragment Processes Fresnel alpha on a per-pixel basis in the fragment shader.
