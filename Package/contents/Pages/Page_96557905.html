Specified Direction

This page describes the feature for setting the direction taken by the emitted particle.

Overview

This feature sets the direction to use for the speed configured using Specified-Direction Initial Velocity.
The following overview describes the settings ().

Specified Direction

The direction that particles move in when emitted is configured using x, y, andz.
You can set this value in a range from –1 to +1, using either the radial wheel () or the text box field ().
