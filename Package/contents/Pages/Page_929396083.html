Cache Operations Feature

Feature Overview

The cache operations feature enables you to perform cache maintenance operations.

Cache Line Size

The nn::os::CacheLineSize constant provides the cache line size in the development hardware environment.

Info

The cache line size on the NX platform is 64 bytes.

Flushing the Data Cache

To flush the data cache (to write the data stored in the cache line to memory and then invalidate that cache line), use the nn::os::FlushDataCache() function.

If the memory space specified by the nn::os::FlushDataCache() function is not aligned to nn::os::CacheLineSize, what gets flushed is the minimum region that is both contained in that memory space and aligned to nn::os::CacheLineSize.

The larger the memory space specified by the nn::os::FlushDataCache() function, the longer the process takes to finish.
