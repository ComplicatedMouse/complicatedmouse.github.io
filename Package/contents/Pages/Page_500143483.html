Additional Information

This section provides additional information about the settings of the parent material (IceCream_Base) included in the sample.

In this example, toppings are set for the child materials and the flavor (color) of the ice cream and cone are adjusted for the parent material.

Sampler

Our intention is to use the textures of the parent material to adjust the ice cream and cone, so select Yes for Required for Child Materials.

Related Links For more information, see Material Property Window: Sampler Page.

Value Reference

IceCream_Base itself is the parent, so no value reference is set.

Child Material Restrictions

The items are set so that only multi-textures can be operated for the child materials.

Vertex Attributes

Vertex attributes are not shown in SampleViewer, so the restriction for this item is set to Fix using the parent material value.

Related Links For more information, see Material Property Window: Shader Page > Material Referencing Settings.

Multi-textures

Multi-textures are set to the child materials as the toppings, so the restriction for this item is set to Always set using the child material.

Other items

For the other items, the restriction is set to Fix using the parent material value so the settings are not changed in the child materials.

How the Child Materials Look

Shaders

If Fix using the parent material value has been set, this settings item is not displayed for the child materials.

You can only see this item if Always set value using child material has been set.

The Parent’s Material Reference Settings

Select Material Referencing in the Material property window to view a list of the material reference settings.
You can also change and delete settings in this list.

Related Links For more information, see Material Property Window: Material Reference Page.
