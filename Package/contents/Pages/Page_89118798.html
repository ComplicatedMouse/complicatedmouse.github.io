Change to Names of NintendoWare Tool EXE Files

The NW4F prefix has been deleted from the names of the EXE files for NintendoWare tools.
Example: NW4F_EffectMaker.exe ⇒ EffectMaker.exe

Info

Note: For the layout tools, this name change will be implemented in 0.7.0.
