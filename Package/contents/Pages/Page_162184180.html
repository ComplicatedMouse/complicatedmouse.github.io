amiibo Settings Applet

The amiibo settings applet provides the following features.

  • Register amiibo owners and nicknames
  • Delete amiibo-compatible game data
  • Restore amiibo-compatible data
  • Reset amiibo

Situations in Which the amiibo Settings Applet Starts

  • The amiibo settings applet has been installed and directly launched from a menu.
  • Called by the application using the nfp library.
    • For occasions when you want to perform actions such as setting an amiibo nickname or Mii character.
