Introduction

This documentation describes the individual features provided by the util library.

It starts by describing topics that are common throughout the util library.

util Library Namespace

The types, constants, classes, functions, and other resources provided by the util library are all located in the nn::util:: namespace.

Do not use types, constants, functions, or other resources located in deeper-level namespaces unless explicitly specified.
Below that level, the hierarchy and resources are subject to change without notice.

util Library Header Files

include the following header files as needed.

  • To use vector-matrix computations, include nn/util/util_Vector.h, nn/util/util_Matrix.h, and nn/util/util_Quaternion.h.

The headers in the directories below nn/util/ are subject to change without notice.
