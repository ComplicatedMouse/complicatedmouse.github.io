Error Notification Policy

When a bcat library function with nn::Result-type return values is called and the return value indicates failure (when nn::Result::IsFailure() returns true), pass the return value to the error viewer applet and notify the user of the error.

In the case of an exception, as described in Nintendo Switch Guidelines > 0079: Displaying errors using the error viewer applet, you do not need to send an error notification to the user.
