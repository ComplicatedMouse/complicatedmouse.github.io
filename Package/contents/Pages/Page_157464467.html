Preparing the Resource Object

Prepare an instance of the Resource class that will hold the shapes and the textures to be used for rendering the Mii face model. The other classes become available for use when you have initialized Resource.

Initializing an IconBody Object (Only For Creating an Icon With a Body)

If the Mii icon will be rendered with a body and not just a head, you will need an IconBody object to hold the body part resources.

Getting the CharInfo Data Structure

Get the CharInfo data structure with the parameters for the Mii you want to display. For more information about getting CharInfo, see Getting the Mii Data.

Creating the CharModel, Faceline, and Mask Objects

Create instances of the CharModel, Faceline, and Mask classes based on Resource and the obtained nn::mii::CharInfo. In addition, use the created Faceline and Mask objects to render the face outline texture and mask texture held in CharModel. For more information about the procedure, from initializing CharModel to rendering the textures, see Rendering the Face Model.

Creating an Icon Texture

Create a texture for rendering the icon.

Creating a Depth-Stencil Texture

Create a depth-stencil texture to use when rendering the icon.

Rendering the Icon Texture Using CharModel

Render the icon to the icon texture using the initialized CharModel object. If the icon has a body, use the initialized IconBody object to render the body.

If you want to create the same icon as the Mii created with MiiEditor, you can use the IconShader class for convenient icon rendering. You can also opt to render using your own shader.

Drawing the Icon Texture Using the Application's Shader

Use the application's shader to draw the texture for the icon.
