Audio Waveform Panel

This panel displays the waveforms output by the audio library.

UI Reference


 Item Description
A Waveform list Displays a list of the waveforms for each channel.
B Vertical orange line Represents the current time.

When configuring the volume for each output device using the nn::audio function, the output volume to the TV is considered.

It is possible that the waveform that is ultimately output could be mixed down at some later stage, but Spy records the waveform before mixing down.

How to Operate

Operation Target Operations Procedure
Waveform display area for the waveform list. Click Moves the current time to the specified position.
Right-click + drag The waveform display scrolls horizontally.
CTRL+mouse wheel Enlarge or reduce the waveform display in the horizontal direction.
ALT+mouse wheel Magnifies and shrinks in the vertical direction.
Waveform list header. Left-click drag the boundary. Change the height of the display by dragging the header boundary.
Info

Use the Main Menu, Audio, and Save Waveform buttons to export the waveform data recorded by this panel to a WAV file.
