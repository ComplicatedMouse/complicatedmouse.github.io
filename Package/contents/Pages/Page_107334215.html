Tessellation States

Tessellation States

Tessellation states refer to several states within the tessellation feature in the rendering pipeline.

Information

Use nn::gfx::TessellationStateInfo to define the tessellation state information.

Number of Patch Control Points

You can set the number of patch primitive control points with the SetPatchControlPointCount function.

Object

Use nn::gfx::TessellationState to define the tessellation state object.

Initialization and Destruction

Tessellation states are initialized and finalized according to the rules for objects.

Setting Tessellation States

Set tessellation states by calling the SetTessellationState function of the nn::gfx::CommandBuffer class.
