This page describes the Adjust Time Scale feature.

Adjust Time Scale

　 　　 　
A

Set new start and end points

Set the key start point and an end point and scale the selected key along the time axis.
B Specify the origin and scale

Apply scaling to the values or frames (time) of the key.

  • For Origin, select one of the presets for the scaling origin.
    To directly specify a value, select the Input value preset.
  • For Ratio, set the ratio to apply for scaling.
