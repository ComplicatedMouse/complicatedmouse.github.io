Bitmap Font Properties

Bitmap Font Property Window

This section describes the UI of the Bitmap Font property window.

A Comments You can add comments.
These comments are added to the BitmapFont node in the FCPX file.
B Character Range Group

Specifies the character range used in the composite font.
Up to 16 character ranges can be set.
C Hexadecimal check box When selected, displays the start and end positions in hexadecimal.
When cleared, displays the range start and end positions in decimal.
D Enabled check box When selected, enables the range setting, making it possible to edit start and end positions.
E Start Position entry text box Specifies the start position of a character range.
F End Position entry text box Specifies the end position of a character range.
