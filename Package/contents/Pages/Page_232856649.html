album Library

Introduction

Purpose of This Documentation

This documentation describes the album library features included in NintendoSDK.

Intended Audience

This documentation is intended for developers that are using NintendoSDK to develop applications that use the album feature.

Related Documentation

Also see the API Reference for the album Library and Sample Programs.

Operating Environment

The album library can be built in both the development hardware environment and the Windows environment.

The album feature cannot be used in the Windows environment. The various functions that use the album features, will always return nn::album::ResultNotSupported.

Handling Image Files and Movie Files

The album library can handle the following image and movie files.

  • Image files and movie files created and stored in "albums" by this feature.

album Library Features

The album library provides the following features.

  • Screenshot Feature (NX Add-On 3.x and later)
    • Saving screenshots: The feature for saving images, created by the application, as screenshots in the album.
    • Saving and posting screenshots: The feature for saving images, created by the application, as screenshots in the album and posting them to a social media site.
