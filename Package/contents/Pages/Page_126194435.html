Unused Files

This section describes the Unused Files feature for searching for files that are not referenced by emitter sets located in the hierarchy below the search origin path.
This feature has the following dedicated settings.

A Move unused files to the Unused folder

Click the buttons for this setting to move unused files to the Unused folder.
B Move selected files Moves files selected from the file list to the Unused folder.
C Move All Moves all files displayed in the file list to the Unused folder.
D Open "Unused" folder Click the Open button in this setting to open the Unused folder in Windows Explorer. 
\Tools\Graphics\EffectMaker\UnUsed

Overview

The following table describes this preset.

A Search origin path The path specified in the address bar. The search begins at this location.
B Emitter sets ().

Emitter sets below the search origin path.

Searches for unused files not referenced by the emitter set in this range.
C Unused files ()

These are unused files that are not referenced by any emitter sets inside the range of B.
Not a search target of this feature.
