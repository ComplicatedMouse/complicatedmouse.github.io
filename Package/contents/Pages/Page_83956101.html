Property Windows: Common Pages

Property Windows: Common Pages

These pages are common to all of the Property windows.

Related Links The information displayed on the File Information page is different for the Project Property Window.

File Information Page

This page is for confirming file information of the selected file.

A The full path of the file.
B The project path of the file.
C The project file base directory.
D
Related Links Displays the additional reference path of the name or the additional reference path when files are opened using an additional reference path for the full path.
E The tool used to create the file.
F The full path to the source file if there was one when the file was created.
G The tool used to change the file.

User Data Page

User Data

This page is used to check and edit user data for the selected object.

A Moves the selected data up when there is more than one item of user data.
B Moves the selected data down when there is more than one item of user data.
C Adds user data.
D Removes the selected user data.
E Edits the content of the selected user data.
F The list of configured user data.
User data is ordered by name, type, size, and value.
You can double-click selected user data to edit its content.
G Select this check box to display integer value user data in hexadecimal.

User Data Dialog Box

This dialog box is used to add and edit user data.

A Select this check box to edit the name or type of user data.
This check box is not displayed if add user data has been selected.
B Specifies the user data name.
C Selects the user data type.
D Enter the user data value here.
Multiple values can be entered by inserting a carriage return between each value.

Skeletal Animation User Data

This user data is only displayed for skeletal animation.

A

Select to edit the bone for which to add user data.

Select the uppermost root node to edit user data of the skeletal animation itself.

B

Add, delete, and edit user data.

Select the node to display the user data list.
