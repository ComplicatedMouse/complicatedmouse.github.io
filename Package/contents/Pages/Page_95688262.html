Obtaining Environment Variables

Overview

This feature is used to get the environment variables set in the host PC.
It gets the system environment variables specified at the time the API is called.
For this reason, even if the application is running, the values can be dynamically changed by overwriting the system environment variables.

Instructions to Get Environment Variables

1. Initialize the HTC library.

#include <nn/htc.h>

nn::htc::Initialize();

2. Get the size of the environment variable values as needed.

#include <nn/htc.h>

size_t size;
if( nn::htc::GetEnvironmentVariableLength( &size, "NINTENDO_SDK_ROOT" ).IsSuccess() )
{
    // Success processing
}
else
{
    // Failure processing.
}

3. Get the environment variable values.

#include <nn/htc.h>

size_t readSize;
const size_t bufferSize = 32;
char buffer[bufferSize];
if( nn::htc::GetEnvironmentVariable( &readSize, buffer, bufferSize, "NINTENDO_SDK_ROOT" ).IsSuccess() )
{
    // Success processing
}
else
{
    // Failure processing.
}

4. Run the completion process when not using the HTC library.

#include <nn/htc.h>

nn::htc::Finalize();
