Switching the UV Origin of the Framebuffer and Depth Buffer

Info

The following information describes how to switch the origin of the texture if you are using NX Add-On 4.1 or a later version.

The origin for a set of texture coordinates can be set to the upper left or the lower left in the graphics API settings, but it is set by default to the lower left for UV coordinate calculations in the Effect module shaders so texture values can be referenced from the framebuffer and the depth buffer. To switch the origin for texture coordinates, make the change in the project configuration file.

Switching the Origin For Texture Coordinates

Info

The following information describes how to switch the origin of the texture if you are using NX Add-On 4.0 or an earlier version.
This method will continue to be supported for some time after NX Add-On 4.1. However, you should change to the method of specifying <TextureOriginMode> in the project configuration file.

To switch this setting, you can explicitly specify the following settings in the Additional preprocessor definitions of the project configuration file.

  • _TEXTURE_ORIGIN_LOWER_LEFT: UV origin is lower left
  • _TEXTURE_ORIGIN_UPPER_LEFT: UV origin is upper left
