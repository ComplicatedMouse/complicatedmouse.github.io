NintendoWare Audio News

This page describes the changes in NintendoSDK 3.0.0.

Added documents and samples about resource management.

Atk provides a system for separating and creating audio source data into the units that you want to load, and loading that data into the game.

Added a sample program for loading resources using the group feature to the AtkGroup sample in the Resource Management guide in the resource management documentation.

For more information about resource management, see below.

Added stream jumps.

Added a stream jump feature for playback of waveforms while jumping between regions that have waveforms that were specified in advance.
For more information, including an overview of the stream jump feature and rules about data creation, see Stream Sounds in the guide.
In addition, for more information about embedding, see Stream Sounds in the Programming Guide and the AtkStreamSound Sample.
