UV Set to Apply

This section describes the feature for selecting which UV set to use for the registered texture when the particle shape is Primitive.

Overview

The UV Set to Apply feature is for changing the UV set held by the model when Primitive has been set for the particle shape.
You can select a UV set separately for each texture and use it for the particle shader.
This setting is displayed when Primitive is selected for Particle Shape on the Particle tab.
The following overview describes the settings ().

UV Set to Apply

Up to two of the UV sets for the registered model file are displayed, and you can select one of them.
Related Links For more information about primitive shape particles, see Primitives.

For a model to have multiple UV sets, you must configure the settings in the DCC tool.
