Added an Option to Apply in World Coordinates for Specified-Direction Initial Velocity

All emitted particles can fall downwards, even during emitter rotation animations.

This example applies a z-axis rotation animation to the emitter and sets Specified direction (±) to the y direction.

ON

Particles are always emitted in the y direction in the world coordinate system.
OFF Particles are emitted in the y direction relative to the emitter itself.

Added Presets to Project Settings

The presets feature can now be used for individual game projects.

By adding effect data that you use frequently to a particular folder, you can call on and use that data as a template when creating new effect data.
Using this presets features in your projects has the following advantages.

  • Effects are standardized within the project.
  • Similar data does not need to be present in multiple different locations.
  • The portions of effects that you can turn into presets no longer need to be created from scratch each time, which improves workflow efficiency.
