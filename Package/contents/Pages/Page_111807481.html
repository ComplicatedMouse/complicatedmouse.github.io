Parts Palette Window

Parts Palette Window ()

The parts root is specified by the parts palette.
The parts palette displays a list of the parts layouts below the parts root.

The Parts Palette window can perform the following operations.

Parts Palette Window
A Parts Folder

Click the icon to open a dialog box for browsing for folders.

Use this dialog box to select a parts folder.
B Parts Folder Path

Displays the path to the current parts folder.

It also stores the paths to parts folders used in the past.

Drag a folder here to set it as the parts folder.
C Clear History Clears the history, except for the current parts folder.
D Updating

Refreshes the parts palette.

The time of the update is shown to the right.
E Display Parts Layouts Only In the ON state, displays only parts layouts in the parts folder.

The ON state.

Only parts layouts in the specified parts folder are shown.

The OFF state.

All layouts in the specified parts folder are shown.
F Search

Set search keywords and the search target category.

  • All
  • Part Name
  • Filename
  • Type
  • Origin
  • Description
  • Update
  • Number of panes
  • Total pixels
G List of Parts Layouts

Displays information for the parts layouts.

  • Part Name
  • Filename
  • Type
  • Origin
  • Description
  • Update
  • Number of panes
  • Total pixels
H Display Specifications

You can change the size of the item displayed in the parts layout list.

Click the left mouse button, or use the drop-down menu, to change the display size according to the following levels.

  • Maximum ()
  • Large ()
  • Medium ()
  • Small ()
  • Minimum ()
I Displaying Thumbnails

For the selected file, displays the thumbnail image in a large size to the right of the Parts Layout list.

Click the left mouse button to alternate between Display () and Display Off ().
Info

To display the thumbnail image, prepare a .NwThumbs folder, and place a thumbnail image with the name layout_file_name.flyt.png in that folder.

Related Links Use the project configuration file to set whether to save the thumbnail images in LayoutEditor. For more information, see here.


Right-click List of Parts Layouts to display the following menu.
Open Opens the selected file in the Layout window.
Position as Parts Pane Positions the selected file as a part in the Layout window that is currently open.
Open with Windows Explorer Opens the selected file with Windows Explorer.
Create Derivative Part Creates a derivative part from the selected file, and opens it in the Layout window.
Update All Derivative Parts Updates derivative parts with the latest data from the selected file.
Update Overwriting in All Body Layouts When changing the overwrite settings of the parts layout, destroys all previous overwrite information no longer used by the body.
Modify Relationship With Base You can remove the association of a derivative part, and change the derivative, in the Modify Relationship with Base window.
