Purpose of This Documentation

This documentation describes the features provided by the operating system library included in NintendoSDK.


Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Related Documentation

os Library Features

The os library provides a useful mechanism for controlling the multithreaded execution of software.

Multithreading refers to a technique by which a single program can be used by multiple independent processes at different stages of execution. The os library provides features to manage threads, mutual exclusion mechanisms to avoid resource conflicts from multiple threads, and features to synchronize control between multiple threads.

Features Provided by the os Library

The operating system library provides the following features.

  • Threads
  • Thread-Local Storage
  • Fiber
  • Semaphores
  • Event
  • Light Events
  • System Events
  • Message Queues
  • Timer Events
  • Multiple Wait
  • Mutexes
  • Condition Variables
  • Barriers
  • Readers-Writer Lock
  • Busy Mutex
  • Time Management
  • Memory Management
  • User Exception Handler
  • Cache Operations


For more information about each feature, see the manuals.
For specific usage instructions, see Sample Programs.
