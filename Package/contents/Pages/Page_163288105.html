Library Types

Types of Spy Libraries

The Spy library comprises the following library files.
The dependent libraries differ based on which features you want to use, so refer to the following table and link the library files you need.

Library Library File Dependent Library Files Description
nn::spy libnn_spy Provides the Spy infrastructure features and basic functionality like time management and log output.
nn::spy::audio libnn_spyAudio libnn_spy Collects nn::audio information and sends it via nn::spy.
nn::spy::atk libnn_spy

libnn_spy

libnn_spyAudio

 Collects nn::atk information and sends it via nn::spy and nn::spy::audio.
