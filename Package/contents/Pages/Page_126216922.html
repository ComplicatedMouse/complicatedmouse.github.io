Animation

Overview

This page describes the features of NintendoWare 3D animation. To read how to create animation, see the relevant sections in the pages for the various plug-ins and 3DEditor.

Animation Types

The g3d library supports the following types of animations.

Type Description
Skeletal Animation

This type of animation defines a skeleton, which is a model with a connected hierarchy of bones, and animates that skeleton into prescribed poses (keyframes) so the positions of the vertices that comprise the character follow suit.

Material Animation

This type of animation is applied to materials.
Shape Animation

This type of animation blends several key shapes (basic shapes).
Bone Visibility Animation

This type of animation changes the visibility of bones, showing and hiding them. Multiple shapes bound to the same bone can be shown and hidden simultaneously.
Scene Animation There are three types of scene animation: camera animation, light animation, and fog animation. The g3d library only supports animation calculations.
