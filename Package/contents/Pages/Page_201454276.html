The atk Sequence Variables Panel

This panel displays the atk sequence variables.

This feature is under evaluation. It may be changed or removed without notice.
Sequence variables will change at the audio frame level. However, this data is missing from the current implementation because transferring from the application can only occur at longer intervals than this.

UI Reference

Option Description
A Sequence Sound List Displays a list of played back sequence sounds.
B Variable Display

Displays a list of the various values for global, local, and track variables.

The column number is the variable index.

How to Operate

Operation Procedure
Click Select a sequence sound that displays a local or track variable.
