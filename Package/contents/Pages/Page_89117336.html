Programming Guide

This section contains materials for sound developers.

Info

If you are using the NintendoWare audio development environment for the first time, start by reading the Atk Start Guide.

Overview

The NintendoWare Audio Toolkit library (atk library) is used for sound playback in game programs. Using the library enables sound data created with NintendoWare SoundMaker (SoundMaker) and other tools to be played in the game applications.

This document describes the requirements for using the atk library to assemble programs that play sounds. It first describes how to build the development environment, and then walks through the sample program by using examples to describe how to use the library.

For more information about the individual classes and functions, see the API Reference Manual.

Content

Quick Start

Uses the sample application to show how to build a development environment and put together a simple sound application.
Sound Handles Describes how to use sound handles to control the playback of sounds.
Sound Actors

Describes how to use sound actors to manage sounds on a per-actor basis.
Filter Describes how to use filters to apply tone changes to sounds in real time.
Audio Effects Describes audio effects for signal processing on sounds.
3D Sound

Describes how to use 3D sounds, which are sounds for which the volume and pan change according to the location of the sound source in 3D space coordinates.
Stream Sounds Describes the features that allow playback while loading sound files.
Mix Describes the mix feature that determines the sound volume that is played through each channel.
Measuring Processing Time Describes how to use the sound profile feature to measure processing time.
Loading Resources Describes the methods for loading sound data from resource files and placing them into memory.
Memory Management

Provides information about managing memory.
Threads Describes the threads used in the atk library.
Submixes Describes the submix feature for managing sound output sent to buses.
nn::atk Sample Description Describes the atk library sample programs.
