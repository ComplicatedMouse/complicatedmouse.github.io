Inheritance Settings: Scale

This section describes the Scale inheritance setting for child emitters.

Overview

The Scale inheritance setting is a feature that makes child emitters inherit the scale value of the parent emitter in the highest node.
The following overview describes the settings ().

Scale

The feature to inherit Scale from the parent emitter is enabled or disabled using the check box ().
Enable this setting to display the Scale group () of settings. The group has the following settings.

Inherited scale ratio

This setting is displayed when the Scale inheritance setting is enabled.
When enabled, the initial scale is multiplied by the parent emitter particle scale using the specified ratio.
