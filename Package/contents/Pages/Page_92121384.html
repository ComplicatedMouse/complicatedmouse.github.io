Viewer Settings

You can configure some aspects of the viewer.

Connection

The Connection group () is used to select whether to connect to the viewer automatically when EffectMaker starts.
The viewer that was used when EffectMaker was last closed will be selected.

Frame Rate

The Frame Rate group () is used to specify the animation frame rate in the viewer from a preset.

Size of Windows Viewer Window

The Size of Windows viewer window group () is used to select the window size of the Windows version viewer from presets.

Info

You can configure the settings in Size of Windows viewer window while EffectMaker and the Windows version of viewer are linked,
and then click OK to close the dialog box and confirm. The confirmed settings are applied the next time the viewer starts.
