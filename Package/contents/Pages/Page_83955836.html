Files Used in the Library

This section describes the files that are implemented in the application but placed in storage and used via the atk library .

Sound Archive File

This file is the body of sound data used by the atk library. The file has the .bfsar extension and is created by SoundMaker.

It contains the various sounds for stream sounds, wave sounds, and sequence sounds, together with information about players, groups, and waveform archives. It also contains sequence data and waveform data for banks and wave sounds, but only the waveform data for stream sounds is referenced as external files (called stream files, and described in the following section).

Sound archive files can be located anywhere in storage.

Items

The elements of a sound archive file are called items. Stream sounds, wave sounds, sequence sounds, banks, waveform archives, players, and groups are all items in the sound archive file.

All items are managed using the strings by which they are named. As many as around 4.3 billion separate items (2^32) can be used in a single sound archive file.

Reference File Path

The sound archive file references stream files to play stream sounds, and these files are external files.

The paths to these stream files are stored using relative paths to the directory holding the sound archive file.

Stream File

These files store the waveform data for the stream sounds used by the sound archive file. A stream file has the .bfstm extension and is created by SoundMaker.

Stream files must be located somewhere that can be referenced from the file path for the sound archive file.

Output From SoundMaker and Placement in Storage

When stream files are created with SoundMaker, they are usually output to the output folder, located at the same level of the hierarchy as the sound project file being converted. You can use the Project Settings of SoundMaker to specify any directory below the output folder.

The location of stream files below the output folder as specified in the Project Settings is stored as the stream file reference file path in the sound archive file.

Everything below the output folder can be stored anywhere in storage and used.
