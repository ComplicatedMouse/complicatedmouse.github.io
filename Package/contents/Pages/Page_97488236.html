Perfectly Follow Emitter

This section describes the Perfectly follow emitter feature, which specifies whether stripe particles should follow their parent emitter.

Overview

The Perfectly follow emitter feature specifies whether stripe particles should follow their parent emitter.
When the Perfectly follow emitter feature is enabled, stripe particles follow changes in emitter attributes, such as position and rotation.
The following overview describes the settings ().

Perfectly Follow Emitter

Use the check box () to select whether stripe particles should follow the emitter when it moves.
When this feature is disabled, stripe particles will not follow changes in emitter attributes such as position and rotation.
