Color Settings

This section describes the Color 0 Behavior Type and Color 1 Behavior Type features that set the Color 0 and Color 1 colors, which are used in the Color Combiner calculation formula.

Overview

The Color 0 Behavior Type is a feature that can set the color animation and the Color 0 color, which are used in the Color Combiner calculation formula.
The Color 1 Behavior Type feature can set the color animation and the Color 1 color, which are used in the Color Combiner calculation formula.
The following overview describes the settings ().

Color 0 Behavior Type, Color 1 Behavior Type

Select among the following presets for the behavior type to use for Color 0 or Color 1 in the Color Combiner calculation formula.

Fixed
Select a color thumbnail () for a fixed color when not animating, and use the color editing UI to set it.
8-Key Animation Creates animation using the Gradation Editor US, which can register up to eight key frames and can animate.
Random
Creates a random table using the Random Settings UI, which can register up to 8 colors.
The color with the check box selected () is applied.
Particle colors are selected at random from this table when they are emitted from the emitter.

Related Links For more information, see the following pages.

Color Editing UI This UI provides a way to configure selected color and alpha values.
Gradient Editor Provides a UI to edit particle time animations, using the color and alpha gradients.
Random Color Settings UI

This UI can register multiple colors.
The registered colors are applied at random as the particle color at emission.
