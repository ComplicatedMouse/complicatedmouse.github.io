Selecting the Low-Level Graphics API

Selecting the Low-Level Graphics API

This information is not useful for standard implementations.

This functionality is designed to be used in exceptional cases, such as while creating resources when a tool needs to specify the low-level graphics API.

If there is no specific use for this functionality, do not use this feature.

Selection Method

The chosen low-level graphics API is statically selected by describing it in the preprocessor macro before including nn/gfx.h, or by having the compiler define the preprocessor macro.

#define NN_GFX_CONFIG_TARGET_API NN_GFX_CONFIG_API_GL4
#include <nn/gfx.h>

This macro is described in all uppercase characters.

What May Be Specified

The following may be selected.

  • Target low-level graphics API
  • Low-level graphics API being used
  • Target version of the low-level graphics API

Target Low-Level Graphics API

Specify the name of the target low-level graphics API in the NN_GFX_CONFIG_TARGET_API macro.

Use the following syntax to specify GL4.

#define NN_GFX_CONFIG_TARGET_API NN_GFX_CONFIG_API_GL4

It is possible to specify only the type of low-level graphics API.

#define NN_GFX_CONFIG_TARGET_API NN_GFX_CONFIG_API_GL

In this case, the target version is the specified target low-level graphics API version.

Default

The default low-level graphics API is used if a target low-level graphics API is not specified.

The default low-level graphics API depends on the platform.

Low-Level Graphics API Being Used

Setting NN_GFX_CONFIG_INCLUDE_* macro to 1 will add additional low-level graphics APIs.

The name of the low-level graphics API is substituted for the *.

A target low-level graphics API is always added, making an explicit definition not required.

Use the following syntax to add GL4.

#define NN_GFX_CONFIG_INCLUDE_GL4 1

It is possible to specify only the type of low-level graphics API.

#define NN_GFX_CONFIG_INCLUDE_GL 1

In this case, the target version is the specified target low-level graphics API version.

Target Version of the Low-Level Graphics API

Set a version number to the NN_GFX_CONFIG_*_TARGET_VERSION macro to specify the target version of a low-level graphics API.

This feature is only meaningful when used in conjunction with the feature to target a low level graphics API, or the additional features of the low-level graphics API being used.

The name of the type of low-level graphics API is substituted for the *.

Use the following syntax to add specify 4 as the target version of GL.

#define NN_GFX_CONFIG_GL_TARGET_VERSION 4

Default

The default version of the low-level graphics API is used if a target version of the low-level graphics API is not specified.
