Introduction

Purpose of This Documentation

This page describes the image library feature included in NintendoSDK.

Intended Audience

The following people are the intended audience of this library.

  • Developers using or planning the use of the NintendoSDK image library in an application
  • Developers using or planning the use of the NintendoSDK image library in middleware
  • Developers using or planning the use of the NintendoSDK image library in other uses

Related Documentation

Also see the API reference for the image library and the image sample program.

image Library Features

The image library is a collection of APIs for processing JPEG data, related data structures, or PNG data. It provides the following features.

  • Decoding JPEG data into pixel data
  • Encoding pixel data into JPEG data
  • Analysis of Exif information in JPEGs
  • Building Exif information in JPEGs
  • Decoding PNG data into pixel data
  • Encoding pixel data into PNG data

License Terms

Processing JPEG data and related data structures in the image library uses the open source libjpeg software as the back end.
For more information about the libjpeg software license, see the included license file Readme (Documents/Licenses/libjpeg_8c/README.txt) in the SDK package.

Info

The features of the image library are included in the binary for all applications, so the license notice is in the system settings. The license notice does not need to be shown in the application.
