Drop Shadows

This section describes the drop shadows feature used to add shadows to picture panes.

Overview

Drop shadows allow you to add shadows to picture panes.
Unlike interior shadows, these shadows are cast on the outside of the pane.
The following sections summarize the drop shadow features.

Rendering Mode

Sets the blend mode between the drop shadow and the background.
The following images compare the effects of changing the rendering mode.

Normal

Uses translucent blending between the shadow and background.
Multiplicative

The default setting.
Uses multiplicative blending between the shadow and background.
Additive

Uses additive blending between the shadow and background.
Subtractive

Uses subtractive blending between the shadow and background.

Knockouts

Determines whether to render a shadow in overlapping areas of a picture pane that has a transparency value set in the Basic tab of its property window.
The following images compare the effect of changing knockout settings.

Selected

The default setting.
Do not render a shadow in overlapping areas of the pane.
Off

Render a shadow in overlapping areas of the pane.

Color

Sets the shadow color.

Angle

Sets the shadow angle. Using the left edge of the picture pane as 0°, the position of the shadow rotates counterclockwise around the pane when the value is increased.

Distance

Sets the shadow distance. As this value increases, the shadow is shown farther away from the pane.

Size

Sets the shadow size. Increasing this value produces a lighter shadow that covers a larger area.

Shadow Type

Sets the shadow type.
The following images compare the effect of changing the shadow type. A size of 25 has been used to make it easy to see the visual differences.

Weak

The shadow appears lighter than it does with the default setting.
Medium

The default setting.
Strong

The shadow appears darker than it does with the default setting.

Usage Examples

The following examples show how to use drop shadows.

Angle

Applies an animation to the angle, changing the direction in which the shadow is cast.
Distance

Applies an animation to the distance, changing the distance the shadow is cast.
Size

Applies an animation to the shadow's size, changing how dark it is and how large of an area it covers.
