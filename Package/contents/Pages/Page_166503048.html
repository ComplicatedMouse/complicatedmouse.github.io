Custom Field node

Created a page describing the Custom Field node.

Described next is an example of configuring a custom field for implementing wind and shock wave fields in a game application.
The wind field is a custom field that applies a force in a fixed direction in the game and affects the moving speed of particles.
The blast wave field is a custom field that applies a radial force emanating from the center of an explosion, affecting the moving speed of particles.

UI Example Image Used in Game
A

Apply Wind Effect selected

As shown in the image, the particles depicting tree leaves are influenced by wind and move in the direction of the arrow.
B Apply Blast Wave Effect cleared As shown in the image, the particles depicting tree leaves move in the direction of the arrow and are not influenced by blast wind.
C
Wind Force Is 0.8 The speed of the moving tree-leaf particles is augmented by 0.8 of the wind speed.

Custom Action Node

Created a page describing the Custom Action node.

Next, an example of configuring a custom action for implementing collision detection in a game application will be described.
Here, “collision detection” refers to an action that defines what happens when particles collide with objects or the map in a game.

UI Example Image Used in Game

A Action ID = “Use collision detection” The emitter uses the “collision detection” action.
B Detect collisions with map = ON
Detect collisions with object = OFF		 As shown in the image, the particles collide with the map.
They do not collide with objects (characters).
C Bounce coefficient = 0.8 When particles collide with the map, they bounce off at 0.8x speed.

Updated Intermediate File Version

The version for 3D intermediate files and graphics intermediate files has been updated to 3.7.0.
The primitive files (FMDB files) and texture files (FTXB files) referenced from EffectMaker (VFX) must be updated.

Use the Intermediate File Updater as follows to update the version of existing intermediate files.

Tools\Graphics\GraphicsTools\IntermediateFileUpdater.exe
