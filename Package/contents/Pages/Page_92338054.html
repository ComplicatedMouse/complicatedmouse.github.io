File View Basic Operations

File View displays the files that have been loaded into 3DEditor in tree format.

This section describes the basic features of the File View in 3DEditor.

Related Links To open the property window for a file, or to perform operations like starting and pausing an animation, double-click the file.

File View Features

File view has the following features.

Basic Features

  • A red star (★) is displayed on the icons of edited files.
  • Filenames associated with an animation or other effect bound to an unreferenced texture or model are displayed in red.
  • If the link to any of the textures used in a texture pattern animation is broken, the texture pattern animation's filename is displayed in red.
  • If you double-click a property item, the property window for that item opens.
  • Right-click a file or folder to display the associated shortcut menu.
  • You can create new animation files.
  • You can move animation files to animation sets and different models by dragging the files.
  • Rename the selected file.
  • The Startup icon () is displayed for files opened in the startup path.
  • The number of animations, animation sets, and models used in a loaded project file (F3PJ files) are displayed for each node.
  • If you right-click a file in File View and then click Reload File, 3DEditor reloads the file as it was last saved.
    Note: Any unsaved edits made before reloading the file are lost.
    If there are any unsaved edits, a warning dialog box appears.

File Renaming Feature

Files loaded into 3DEditor can be renamed.

Info

You can use alphanumeric characters (a-zA-Z, 0-9), hyphens (-), underscores (_) and dots (.) in filenames.
A warning message is displayed if you use any other characters.

Playing and Pausing Animations

You can operate animations in the animation set being previewed in file view by clicking the Play/Pause icon ().

Related Links You can play and pause all animations in an animation set from the shortcut menu of the animation set.

Enabling and Disabling Animation Preview

Click the eye icon () to enable or disable previews of individual animations in an animation set.

The icon is dimmed and becomes unavailable when the preview for an animation is disabled.

Enabling or Disabling Animation Set Previews

If there are multiple animation sets, use the button () to select which animation set to preview.

Showing or Hiding Model Data in Object View

To show or hide model data for a model in object view, click the Show/Hide icon () next to the model filename in the file view.

The model file icon is dimmed and becomes unavailable when you hide the model in object view.

Related Links You can also show or hide model data in object view from the shortcut menu of the model file.

Loading TGA Files

Loaded TGA files are listed just like other texture intermediate files, but the icon () shows that they cannot be edited or saved.

Related Links You can add any of the texture files (including any TGA files) in the list to the Material Property Window: Samplers Page by dragging the texture thumbnails to the page.
(If the texture file does not have any material search paths, you cannot drop it into the sampler page.) (Reference Links Additional Reference Path)
