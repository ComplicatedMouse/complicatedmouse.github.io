Help

This page describes the Help menu in LayoutEditor.

Help Displays the LayoutEditor User’s Manual.
Open News Displays the NintendoWare graphics news.
Quick Start Displays the LayoutEditor Quick Start Guide.
Tutorial Displays the LayoutEditor tutorials.
Troubleshooting Opens troubleshooting for LayoutEditor.
FontConverter Help Opens the FontConverter (GUI Version) User’s Manual.
Version Displays the LayoutEditor version information.
