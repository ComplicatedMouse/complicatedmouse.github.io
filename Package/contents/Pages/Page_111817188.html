What are the restrictions on Window panes?

Window panes have the following restrictions.

  • Diagonal gradation is not applied accurately when vertex color is applied to the entire window.
  • When operating in horizontal mode, all frame and content parts must have the same number of multi-textures.
  • When operating in horizontal mode, textures must be set in the content parts of panes.
  • The combiner settings sometimes look different in the layout window and the viewer. The settings are shown correctly in the viewer.
