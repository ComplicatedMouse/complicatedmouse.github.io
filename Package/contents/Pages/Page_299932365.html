Scissor Pane

This section describes the Scissor pane, which clips the rendering targets of panes lower in the Hierarchy window in rectangular shapes.

Overview

The Scissor Pane () is where panes lower in the hierarchy are treated as the render target and clipped in a rectangular shape.
The rendering process is skipped for the area outside the rectangle region, which reduces the processing load.

The following table shows the settings for scissor panes.

Base Pane Sets information including the pane name, position, shape, and center position.
Extended User Information Sets user-defined extended information for the pane.
