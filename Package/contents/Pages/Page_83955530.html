File View

File View

The file view enables you to view and select intermediate files loaded into 3DEditor.
The top portion of the file view displays a list of files, and the bottom portion displays information about the selected file.

Related Links Operations on loaded files (for example, using the shortcut menu).

A The project file being loaded.
B Scenes
Displays a list of scene files included in the project.
Scene files are created inside animation sets.
Scene animations (camera, light, and fog animations) are displayed as children of a scene file.
C Model
Displays a list of model (FMDB and FMDA) files included in the project.
The filename extension .fmdb is for binary intermediate files, while .fmda is for text intermediate files.
Bound animations can be stored in an animation set and played simultaneously in the connected application.
Note: Filenames for model files are displayed in bold text.
D

Material

Displays a list of material files (FMTB files) in the project.
E Animation
Displays a list of animation files included in the project.
F Texture
Displays a list of texture intermediate files (FTXA, FTXB, TGA, PNG, DDS, and EXR files) included in the project.
G Shader definition
Displays a list of shader definition files (FSDB files) included in the project.
H Click the triangle symbol to show or hide the information for a file.
To change the size of the display region for file information, drag the portion below the triangle.
I Displays information that is common to all files.
J Displays information unique to each file type.

Creating Animations

Right-click the model file (FMDB or FMDA file) displayed in the File View, right-click an animation file group, and select Create New Animation from the shortcut menu that appears.

The dialog box for creating animations appears if selected.

  • Name.
    Enter a name for the animation file.
    If you are creating an animation using the shortcut menu associated with a model file, the same name used by the model file is entered by default.
  • Initial animation settings.
    Used to set the number of frames for an animation and enable or disable loop playback.

    Curve Editor supports animations that are anywhere from 0 to 16,777,215 frames in length.

Shortcut Menu for Frame Number Settings

In the new animation dialog box, if you right-click the box where you can specify the number of frames, the following shortcut menu appears.

  • Copy.
    Copy the number of frames currently entered in the box.
  • Paste.
    Paste frames copied from somewhere else.
  • Maximum number of frames.
    Automatically specifies the maximum number of animation frames supported by Curve Editor in 3DEditor (16777215).

Animations With the Same Names

3DEditor can load animation files that have the same name but different file paths.
In addition, when an animation file with a definite path is already loaded, another animation with the same name can be created.

In the File View, [1/2], [2/2], and so on are appended to files that have the same name so that the files can be distinguished from one another.

However, note the following file view restrictions on animations with the same names.

  • An animation with the same name cannot be created under the same model.
  • You cannot use the Save As operation or overwrite an existing file.
  • You cannot move or copy an animation to a model if an animation with the same name is bound to that model.
  • You cannot move an animation set to a model if the set includes an animation with the same name as an animation that is bound to that model.
  • You cannot create a new animation with the same name as an animation having an indefinite path.

Animation Sets

An animation set is a collection of multiple animations.
If played in the application, all animations included in the animation set are played simultaneously.

There are two types of animation sets: freely defined animation sets created by the user, and a default animation set.

Default Animation Set
A default animation set is provided because 3DEditor requires at least one animation set for a scene or model.
If a user-defined animation set has not been created, any animations loaded into or created with 3DEditor are stored in the default animation set.

Animations bound to a scene or model are displayed as children of an animation set.

You can use the icons next to each child animation to select whether to play the animation or whether to show or hide it in the preview.

Changing the Order of Applying Animations in an Animation Set

Animations in an animation set are applied and previewed starting from the last animation in the list.

To change the order in which animations are applied, simply drag them to the desired position in the list.

Creating an Animation Set

Right-click in the scene group or model file (FMDB or FMDA file) displayed in the File View, and select New Animation Set.

Enter a name in the dialog box that opens for creating an animation set.
Note: The dialog box opens with New animation set x as the default entry. Numbers are assigned automatically for x.

When you click OK, the animation set is created as a child of the associated scene group or model file. After it is created, you can add animations by dragging them to the set.

After you click the eye icon next to an animation set to enable playback, all animations play simultaneously when you connect to the application.
