Maya Plug-In


You can now specify to export with only BaseAnimation enabled when the export item animation layer name is left blank in the multi-export plug-in.

Added a feature to export with only BaseAnimation enabled when the export item animation layer name is left blank in the Maya multi-export plug-in.
Specify the following line in the config file to enable this feature. 

multi_export_base_anim_layer_is_default="true"

If multi_export_base_anim_layer_is_default is set to true, only BaseAnimation is enabled if only a single export item is selected in the main window and the animation layer name is left blank.
In addition, when the Set button is clicked in Anim Layer in the Input Export Items window, only BaseAnimation is enabled if this is left blank.

3D Command-Line Tools

Changed the shader cache output directory for 3D Shader Converter.

Changed the output directory for the shader cache is when using the --shader-cache-directory option in 3D Shader Converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dShaderConverter.exe).
More specifically, if you specify a directory using the --shader-cache-directory option, a directory named "shader archive name_shading model name_API name_code type name" is now created immediately below that directory, and the shader cache is exported there. Shader caches created using this converter in a previous version of the SDK can no longer be used.
