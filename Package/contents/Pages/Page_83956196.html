Copying and Pasting Samplers

Copy and paste samplers from the material Property Window, on the Samplers page, by right-clicking a sampler in the list and then using the shortcut menu.

  • Inserts the copied data at the position selected in the list.
  • Adds data to the end of the list when you paste without a position in the list selected.
  • Copies the sampler to all selected materials if copy and pasting a sampler with more than one material panel selected.
Note

Data is overwritten if the material being pasted to already has a sampler of the same name.
