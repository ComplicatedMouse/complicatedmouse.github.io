Property Window: Additional Reference Path Page

This page allows you to set and edit additional reference paths for the selected file.

A Moves the selected additional reference path up or down (if there is more than one additional reference path).
B Adds a new additional reference path.
Click the Add button to display the Edit Additional Reference Path window.
C Deletes the selected additional reference path.
D Edits the selected additional reference path.
Click the Edit button to display the Edit Additional Reference Path window.
E Displays all of the additional reference paths that have been set.
F Displays whether subfolders beneath the configured additional reference path are also included in the search.
G Displays the search target.

Edit Additional Reference Path Window

This window allows you to add and edit additional reference paths.

A Displays the reference folder.
B

You can select a reference folder by clicking the ... button and selecting a folder from the window that appears.
C Displays the absolute path of the reference folder.
D Displays the base part of the path.
E Select this check box to include subfolders under the reference folder when searching.
F Select Texture or Material as the search target.
G Click OK to add the reference folder as an additional reference path.
H Cancel the additional reference path edit process and close the window.

Info

You can use wildcards like CommonTexture/*/output in additional reference paths.
