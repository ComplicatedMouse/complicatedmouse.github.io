Background Image - Specify File

This section describes the Specify File feature for registering background images.

Overview

The Specify File feature can be used to register a background image.
This section provides an overview of the settings ().

Specify File

Displays the path of the background image and allows you to register it anywhere. The following are button types.

Load Opens the file selection window to select a background image.
Reload Updates the specified background image.
Clear Releases the specified background image.
