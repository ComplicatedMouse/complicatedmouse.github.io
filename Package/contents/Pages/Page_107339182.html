Introduction

This documentation describes the features provided by the oe library.

It starts by describing concepts that are common throughout the oe library.


oe Library Namespace

The types, constants, functions, and other resources provided by the OE library are all defined in the nn::oe:: namespace.

Do not use types, constants, functions, or other resources located in deeper-level namespaces unless explicitly specified.
Below that level, the hierarchy and resources are subject to change without notice.


oe Library Header Files

Include only the nn/oe.h header unless explicitly specified otherwise.
The structure of header files other than nn/oe.h is subject to change without notice.


oe Library Files

Automatically linked by the build system, so linking does not have to be specified separately.
