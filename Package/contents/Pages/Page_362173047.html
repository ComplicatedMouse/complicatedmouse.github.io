Made it possible to sort export items with the Maya Multi-Export Plug-In.

Added a feature to the Maya Multi-Export Plug-In for sorting export items.
Perform this operation from the Sort menu in the main window.


It is possible to sort with specific options using the Sort menu in the main window.
Sort > Ascending > Option Name (sorting in ascending order)
Sort > Descending > Option Name (sorting in descending order)
The custom attribute is changed along with the display, and the export order also changes.

Specifiable Option Name

  • Output File Name
  • Output Folder
  • Merge fmd File
  • fdes File
  • Node (Display Layer)
  • Anim Layer
  • Comment
  • Start Frame
  • End Frame
  • Loop
  • Output File Selection

Added an Undo button to the main window of the Maya Multi-Export Plug-In.

Added an Undo button to the main window of the Multi-Export Plug-In.
Clicking the button restores the state prior to the last action.

Additionally, the Move Up and Move Down buttons were changed to Up and Down buttons.


Actions other than Multi-Export Plug-In ones are undone, similar to when using the Edit > Undo menu in the main window of Maya.

Right-click the Undo button and use the displayed Redo menu to perform the action that was undone.
