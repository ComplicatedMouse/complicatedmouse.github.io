To use stripes with child emitters, enable Emitter Assignment for Particles.
Consider the following example, where a consolidated stripe has been created for the string that is the child of the balloon. In this case, enable Emitter Assignment for Particles.

Emitter Assignment for Particles Setting Cleared Emitter Assignment for Particles Setting Selected

The string emitters are combined together, making the strings join. Each string emitter is at the center of the parent balloon, so the effect displays as expected.

Removed the Linear Edit Mode feature from the Debug tab of the viewer node. This is a feature that determines whether EffectViewer will be set to display in the linear environment.

This setting is now configured using the project configuration file, which can be switched for the environment for each game application when EffectMaker starts.

Added a Feature Search and Replace Window

Added a Search window that enables you to search among a specified range of emitter set (ESET) files for files that have the specified settings and to batch-edit those settings.
Previously, if multiple files had incorrectly configured settings interspersed among a large number of other effect files, for example, you had to manually search for the files and then open them one by one to fix them.
Using this feature, you can search for the problem files from the Search tab and then batch-edit the matching files from the Replace tab.

Known Bugs The current version has the following known bugs.
The Search window is still under development and currently contains the following bugs. These bugs will be fixed in future releases.

  • Executing a search or replace operation when there are more replace expression conditions on the Replace tab than there are search conditions on the Search tab causes a fatal error.
  • Executing a search for an expression containing an alphabetical item that is not specified in the search conditions on the Search tab causes a fatal error.
  • After specifying multiple conditions in Search Conditions on the Search tab or in Replacement Expression on the Replace tab, using the trashcan button to delete a condition automatically changes the rightmost drop-down list to Matches Exactly.

Also, some conditions prevent searches from being executed.

  • The feature does not currently support searching the shapes of curve data for animation curves as data sequences.

Added a Template and Presets

Added a template, for effects often observed in games, to Favorites in the Effect Browser.
You can use the template and change the textures, and so on, to fit the game you are developing.

In addition, added presets for general-purpose parts used in effects.
You can load emitters from the presets to the effect data that you are creating.

The Hit effect template, "Hit."
The group of "Hit" presets.
Flash (the point of collision) Sparks (the direction of collision) Shock wave (the scope of collision) Speckles (the lingerings of collision)
This template can be used in combination with these presets for effects from the time they are generated, until the time they die.
Future improvements are planned for this feature for creating effects using a combination of presets.
