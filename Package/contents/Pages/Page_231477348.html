Web Applet and Offline Web Applet

The web applet and Offline Web Applet are library applets that use a web browser including basic HTML5 features and video playback features to display webpages specified by an application.

The web applet is a library applet for displaying webpages associated with specified URLs on the Internet.
The Offline Web Applet is a library applet for displaying offline HTML content included in applications.

Purpose of This Documentation

This documentation provides a general overview of the main features and specifications of the web applet and Offline Web Applet and also includes procedures for opening and closing them, techniques for customizing them for use in applications, and several usage examples.
This documentation covers information that applies to both applets in addition to information that is specific to each.
