Profiler-Exit

Overview

Forces the profiler UI to exit either immediately or once the current script has finished executing. By default, calling this will delay exiting until the script has finished running.

Profiler-Exit [-ExitCode] <Integer> [-Now] [-WarningsAsErrors]


[-ExitCode] <Integer>

The exit code to use when exiting the profiler UI.


[-Now]

Exit immediately rather than waiting for the script to finish.


[-WarningsAsErrors]

When exiting, if there are any active warnings, treat those as errors and increment the exit code by the current number of warnings.


Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Exit 0
Profiler-Exit -ExitCode 1 -Now
Profiler-Exit 0 -WarningsAsErrors
