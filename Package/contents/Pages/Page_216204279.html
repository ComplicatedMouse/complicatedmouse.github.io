NintendoWare Spy News

This page introduces the changes in NintendoWare Spy library for NintendoSDK 3.0.0.

It is now possible to cancel loading data files while they are being loaded.

The UI no longer stops responding while data files are being loaded.

Also, you can now cancel loading while the files are being loaded. In this case, the information that was loaded prior to cancellation will be displayed.

The stream sound information is now displayed correctly in SpyAtk.

Stream sound information is now displayed as a SpyAtk panel.

The stream sound panel is a panel that shows information related to the stream sound that is being played back. Currently, it displays the fill rate of the stream buffer.

For more information about the panel information, see the Atk Stream Sound Panel page.

The data displayed in this panel is sent using the nn::spy::atk::AtkSpyModule::PushStreamSoundInfo function in the SpyAtk library.

The volume output to the TV is now correctly applied to the audio waveform.

Added the volume output to the TV to the audio waveform data recored with nn::spy::audio::AudioSpyModule::PushWaveform() or nn::spy::atk::AtkSpyModule::PushWaveform().

It is applied to the audio waveform panel and the audio loudness emitter.

Added support for loudness calculation at a sample rate of 32 kHz.

Even if the sample rate is set to 32 Hz in the nn::audio and nn::atk sample rate, the loudness can still be displayed.

You can now control connections from the command line.

You can open or close connections using the command-line function in Spy.exe.

Start Spy.exe ahead of time and open a window. Select the connection target and display the panels you are collecting data for.

If Spy.exe specifies and executes the following command-line functions, commands will be sent to the Spy.exe that was started ahead of time.

  • Opening a Connection
Spy.exe --command connect
  • Closing a Connection
Spy.exe --command disconnect
