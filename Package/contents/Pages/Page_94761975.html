Alpha Combiner Formula

This section describes the equations for the Alpha Combiner.

Overview

The alpha combiner formula feature allows you to configure the equations used when alpha elements are combined.
By modifying the equation you can enhance the quality of the effect you are trying to create.

The following table provides an image and a brief description for each of the alpha elements that are used as inputs in the alpha combiner formula.

 Comb_A The alpha for the previous combiner setting.
The specified texture alpha value is output as is.
A0 The A value for the alpha 0 value specified in the Color tab.
This example uses an animation that changes this value from 0.0 (black) to 1.0 (white) and then back to 0.0 (black).
█
A1 The A value for the alpha 1 value specified in the Color tab.
This example specifies a fixed value of A = 0.5.

Alpha Calculation Formula

Select which formula among the presets to use to combine the alpha value in the previous stage (Comb_A) with the alpha values set in the Color tab.
The following table contains images of the blend results and the corresponding formulas.

 Comb_A × A0

Outputs the result of multiplying together the output result from the previous combiner stage and the output result for alpha 0.
Comb_A × A0 × A1 Outputs the result of multiplying together the output result from the previous combiner stage, alpha 0, and alpha 1.
The following formula can be used to remove effects created by animating a texture’s alpha channel threshold value.
Related Links For more information about using this technique to remove effects, see Making Effects Disappear Naturally > C Dissipate.
(Comb_A - (1 - A0) ) × 2 Subtracts the inverse of alpha 0 from the output result from the previous combiner stage and multiplies the resulting value by 2.
This formula was kept to maintain compatibility with data created using NintendoWare for CTR.
(Comb_A - A0) × A1

Subtracts alpha 0 from the output from the previous combiner stage and multiplies the resulting value by alpha 1.
This formula enables you to use alpha 1 to achieve more control over the result than is possible with the preceding formulas.
Clamp( (Comb_A - A0) × 4) × A1 Subtracts alpha 0 from the output result from the previous combiner stage and then multiplies the resulting value by 4.
This value is then clamped to a value in the range from 0 to 1, and multiplied by alpha 1.

(Comb_A - A0) × A1


This formula enables you to control the white range via the value used for A0.
In addition, you can use the value for A1 to control the contrast.

Clamp( (Comb_A - A0) × 4) × A1


In this example, increasing the value of A1 enables you to adjust the width of the boundary between white and black.
This formula is particularly suitable for use with distance field textures.
