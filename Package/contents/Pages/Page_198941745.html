Export Binary File Window

This page describes the Export Binary File window that can be opened by selecting Export Binary from the File menu in LayoutEditor.

Important

This dialog box is only intended to be used by developers for purposes such as exporting binary files for testing.
When building production layouts, we recommend that designers use a workflow that does not use this dialog box.

Commands specified in this dialog box support many (but not all) of the options available in Layout Binary Converter.

Export Binary File Window

A Output Directory Specifies the directory to which the binary files should be exported. The Export button becomes active when a directory is specified.
B Export Filename Specifies the name to use for the exported binary files.
C Create a folder with the export filename in the export directory Select this check box to create a folder with the name specified as the export filename in the output directory. Then export the binary files to this folder.
D Bake curve value outside the key region and export.

•Use the tool (compatibility mode)

•Use the converter (recommended)
E Convert escape characters in text boxes into control codes

Controls the option that determines whether the control codes used for processing tags in text in text boxes are converted (--no-convert-cvtrchar).

  • For more information about tag processing, see Font Tag Processing.
F Use tag information Clearing this check box makes it possible to select the export file types individually in H.
G Key Controls

• None

• Do not output keys if values do not change within the frame segment (excluding the first animation segment tag)

• Do not output keys if values do not change within the frame segment
H Export File Type

Select the export file type.

This element is only enabled if the Use tag information check box in F is cleared.
