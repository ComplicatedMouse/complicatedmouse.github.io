Basic Settings

Contains model file and animation file settings to verify on the viewer.

Model

The Model group () is a set of features for controlling the connected viewer.
This group has the following settings.

Specify Model File This feature loads a model file (FMDB file).

Character Animations

The Character Animation group () is a set of features for loading the exported binary file to check how effects look in the viewer.
This group has the following settings.

Specify Animation Folder Allows you to select a folder and show the animation (FSKB) files contained in the folder.
This feature is used to choose an animation file to use from the displayed list.

Visibility Animation

The Visibility Animation group () is a set of features for loading the exported binary file to check how effects look in the viewer.
This group has the following settings.

Specify Animation Folder If you select a folder that contains animation files (FVBB files), the animation files in the folder are displayed.
Use this feature to choose an animation file to use from the displayed files.
