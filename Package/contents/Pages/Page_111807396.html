Group Window

This section describes the names and uses of the various parts of the Group window.

The grouping feature enables you to group a number of related panes.
It is used in cases such as the following when it is difficult to represent a group using a parent-child hierarchy.

  • When the panes in the group straddle multiple hierarchies.
  • When panes belong simultaneous to multiple groups.

Use the Group window to confirm the groups within a layout.

Group Window The following operations are in use.
