Meter Panel

This panel shows detailed information for the current time.

Currently, this panel only supports information about performance loads.

UI Reference

Option Description
A nn::audio Displays the current sum of the various processing loads.
B Meter

Depicts the current processing loads as bar graphs.

nn::audio

Displays the current value of the processing load.
The sum value of the processing loads for Voice, SubMix, FinalMix, and Sink, in microseconds.

Meter

Displays the various component processing loads as colored bar graphs.
The number inside each bar represents the processing load. The unit is microseconds.

The following table provides the correspondence between colors and components.

Bar Colors Component
Red Voice
Yellow

SubMix
Green FinalMix
Blue Sink

How to Operate

Operation Procedure
Right-click + drag Scrolls horizontally.
CTRL+mouse wheel Magnifies and shrinks in the horizontal direction.
