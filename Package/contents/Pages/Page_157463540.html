EffectMaker

This section introduces UI elements newly added to EffectMaker.

Added a Feature to Open the Auto-Backup Folder

You can now use the main menu to directly open the folder where emitter set files are automatically backed up.

Auto backup saves emitter set files (ESET files, ) to the AutoBackup folder, which is in the same location as EffectMaker.exe.
The following procedure is an easy method for opening the AutoBackup folder.

  • From the menu bar, select Help > Open auto backup location.
  • The Option Settings dialog box () opens. In Advanced settings, under Auto Backup, click the Open folder button ().

Enhanced the Feature for Showing the Custom Shader Source

Added an option to the feature for showing the custom shader source code to select whether to preprocess.
Developers can use this option to check the effect data created by artists.
