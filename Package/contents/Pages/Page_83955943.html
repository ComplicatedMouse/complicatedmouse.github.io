Noise

This section describes the Noise feature for creating behavior without an apparent pattern for particle movement and settings values.

A waveform made from sine wave synthesis, as in the following figure, is called "noise."
When this waveform is used for the trajectory of the particle, it provides an appearance of randomness, but strictly speaking the behavior is not random.

Info

Fields that use noise have something called "GPU noise." This feature uses the waveform to move particles when they are generated.
Although the cost of trajectory calculations is not cheap, the position can be determined based on the time variable alone. This allows the particle to be implemented as a GPU particle because it eliminates the need to retain the previous state, which is a GPU bottleneck.
