Extended User Information

Extended User Information is used to edit extended user information in LayoutEditor.

The Extended User Information feature can hold user-specific information.
The editing options are the same for various types of panes.

Extended User Information Page

Enter information about the Extended User Information page.

Extended User Information

Changes the list order of the extended user information.

Deletes extended user information from the Extended User Information page list.

Copies all extended user information.

Name Displays the name of the extended user information.
Type

Displays the type of the extended user information.

  • String
  • Integer value list
  • Real number value list
  • One-dimensional integer animation
  • Two-dimensional integer animation
  • RGBA color animation
  • One-dimensional real number animation
  • Two-dimensional real number animation
  • Three-dimensional real number animation
  • RGBA color animation (with gamma correction)

Enabling the extended user information from the Base pane makes it possible to select the animation type.
Size Displays the size of the extended user information.
Value

Displays the value of the extended user information.

Enter the settings using the Extended User Information window.

Enter one value per line for each numerical type.
Info

Specifying RGBA color animation (with gamma correction) as the type, applies gamma correction to each of the V key values of the curve in the layout binary converter, and converts them to 255*(v/255)^2.2.
