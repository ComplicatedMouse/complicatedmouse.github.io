Emitter Transform: Scale

This section describes the Scale feature for setting the angle when placing emitters.

Overview

The Scale feature sets the scale when placing emitters.
The following overview describes the settings ().

Scale

Use X Y Z to set the emitter coordinates that will be the standard for placing emitters.
This feature will be applied to both the speed and scale of particles. However, it does not affect the particle size.
To change the size of emitters and particles uniformly, specify the same value for the particle scale's Standard value.
The following table shows the properties to which scale is applied.

Emitter Size Particle Size Particle Speed
Yes × Yes
