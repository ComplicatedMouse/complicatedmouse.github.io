With this tab, you can use the viewer to check the settings of the Scale feature of the Emitter node that can be controlled by the game application.

Scale Particle Size

The Scale Particle Size group () is a set of features for multiplying the size of all emitter shapes in the emitter set by a specified value.
This group has the following settings.

Size When Emitted Multiplies the size of all particles in the emitter set by the specified value when they are emitted.
Size After Emission Multiplies the size of all particles in the emitter set by the specified value after they have been emitted.

Difference Between Size When Emitted and Size After Emission

The following images compare what happens when the values for Size when emitted and Size after emission change.
For the emitter that is emitting particles, the scale x, y, and z values for the preview node Size when emitted and Size after emission change in the order of 1 → 2 → 1.
