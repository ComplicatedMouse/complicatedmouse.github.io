Specifies the blending method for panes and backgrounds, from presets.

Current Settings

The settings selected from the presets appear.

Displays Custom if you change the selected settings from the presets.
Presets
Opaque This setting allows you to render completely opaque matter without the alpha test.
Transparency Settings that allow for the expression of transparent objects with clear outlines.
Translucent (default)

Settings to blend the pane and background while maintaining colors for each.
Additive

Settings to add the pane and background images.
Subtractive

Settings to subtract the pane from the background.
Multiplicative

Settings to multiply the pane and background images.
Translucent: Max alpha

Settings that enable precise blending of the alpha portion when using a capture pane.

Alpha Test

You select alpha value comparison criteria from the drop-down list.
Use the default to disable alpha tests and reduce the binary size.

Use Default Settings

This check box is selected when using the default settings.

The default settings disable alpha tests and reduce the binary size.

Condition　　　

Never pass

The test is never passed when this setting is selected.

Alpha < Value

Passes if the alpha is less than the value.

Alpha <= Value

Passes if the alpha is the same as or less than the value.

Alpha == Value

Passes if the alpha is the same as the value.

Alpha != Value

Passes if the alpha is the not the same as the value.

Alpha >= Value

Passes if the alpha is the same as or greater than the value.

Alpha > Value

Passes if the alpha is greater than the value.

Always pass

The test is always passed if this settings is selected.

Value　　　

Values are configured either with numerical input or the slider.
Note

This does not work in the GL version.

Blend

This feature sets Pixel (the output result of the texture combiner) and FB (the framebuffer for the background).
Use default settings and configure standard translucency to reduce the binary size.

Use Default Settings

This check box is selected when using the default settings.

With the default settings, the standard translucency settings are used. You cannot make detailed settings, but the binary size is reduced.
