Apply in World Coordinates

This page describes Apply in world coordinates, the feature for choosing whether to apply the collision plane using the world coordinate system.

Overview

Apply in world coordinates sets the coordinate system for simple collisions.
Use the world coordinate system to always place the collision plane using the world coordinate system, regardless of emitter transform settings.
The particle emission location and direction can be freely adjusted, regardless of the collision plane.

Apply in World Coordinates

Either specify Apply to world coordinate system or enable or disable the check box () to switch the settings.
Collisions are affected when you rate emitters because emitters are the basis for applying collision planes when disabled.
There is no effect from emitter translation or rotation when enabled because the world coordinates are the basis for applying collision planes.
