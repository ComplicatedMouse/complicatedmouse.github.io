Fixing the Random Seed

This section describes the Fix the Random Seed feature for setting a fixed random number for Texture 0, Texture 1, and Texture 2.

Overview

The Fix the Random Seed feature is for fixing the random seed used when values are randomized in the sub-tabs for textures 0, 1, and 2 ().
By using the same random seed for all textures, you can animate texture coordinates under specific conditions, such as with matching albedo maps and normal maps.
The following overview describes the settings ().

Fixing the Random Seed

Info

Make sure that you configure the same settings for texture 0 and texture 1.
In the preceding example, the random numbers will not match even if Fix the Random Seed is enabled if texture 0 is configured in the following ways.

  • An image halved lengthwise () is registered.
  • Repetition count is set to 2x1.
  • Clamp is set to Mirror image ().
