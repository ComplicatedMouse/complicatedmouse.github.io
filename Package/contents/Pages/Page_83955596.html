Preview File

This page describes the preview file used in the preview node.

Saving Preview Files

You can save previews as preview files (PREV files). You can use EffectViewer to check the results of your preview settings.
Preview files contain the following information.

  • Preview settings
  • The name of an emitter set file (ESET file)

Location of Save Emitter Set Files

Emitter set files (ESET files) are saved in the same directory as the associated preview files (PREV files).
Files that are saved in other locations cannot be loaded when the preview file (PREV file) is opened.

Opening Preview Files

When a preview file (PREV file) is opened, one or more associated emitter set files (ESET files) are always loaded at the same time.
When opening a preview file (PREV file), a dialog box is displayed if the associated emitter set files (ESET files) are not found, and you can no longer load the preview file (PREV file).
When an emitter set file (ESET file) is loaded, a new preview file (PREV file) is added.

Effects Using Previews

You can create multiple previews for a single emitter set.
By using multiple previews, you can display the same effect in different colors in the Viewer.
With previews, you can display the link status between effects and models in the Viewer.
