Specify Animation Folder

This section describes specify folder for the animation.

Overview

The animation generated by the DCC tool may be used with the registered model file.
The timing of the effect emission or animation can be adjusted by linking the registered model's animation and effect in the viewer.
This section provides an overview of the settings ().

Specifying Folders

Allows you to select the folder where animation files are stored. The following are button types.

 Load A folder can be selected by pressing this button to open the select folder window.
Reload Click this button to update animation files in the selected folder.
Clear Unselect all folders.
