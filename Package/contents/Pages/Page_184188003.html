3DEditor

This section describes features that are newly added to 3DEditor.

The Material Referencing Feature Has Been Made Public on a Trial Basis

Note

When using this feature, note that its specifications and behavior are subject to change in the future without advance notice.

When this feature becomes officially public it will be reported in News.

Material referencing, which is a feature where a material's shader parameter settings can be referenced by a different material, has been made public on a trial basis. This feature enables multiple materials to share the same settings and facilitates the management of materials.

Related Links For more information about incorporating this feature into your workflow, see Material Referencing.

Now You Can Select ASTC Format When Exporting Textures at 32 Bits Per Channel

You can now select ASTC format when exporting texture intermediate files at 32 bits per channel.
