DirtyWord Filter

Introduction

This section describes the DirtyWord filter feature, which is expressly enabled for Chinese-market Nintendo Switch consoles (consoles for China).

Console for China refers to any system for which the platform region has been set to China.
For more information, see Testing Environment.

API

The nn::ngc namespace in the NintendoSDK provides a general-purpose feature for checking profanity. In addition to that feature, you can also use the DirtyWord filter, a special feature for Chinese-market Nintendo Switch consoles (consoles for China).
For more information, see the Include/nn/ngc/ngc_ProfanityFilterForPlatformRegionChina.h header file and the API references for nn::ngc::CheckProfanityWordsInTextForPlatformRegionChina and nn::ngc::MaskProfanityWordsInTextForPlatformRegionChina.

Info

The API listed in ngc_ProfanityFilterForPlatformRegionChina.h filters dirty words unique to consoles with the platform region set to China.
Use the standard nn::ngc API separately if you need a common profanity checking functionality for a different region.

Special Profanity

These DirtyWord functions always recognize the following words as profanity.
You can use these words to confirm that the application is using the DirtyWord filter because the filter identities them as profanity.

  • walumoxi
  • 瓦鲁莫西 　

Development Support Feature

Disabling the DirtyWord Filter

By disabling the preceding functions, you can change the behavior of consoles for China so they do not identify any entered words as DirtyWord profanity.
Use the following procedure.

  1. Start DevMenu on a console for China.
  2. Go to Device > System Settings and change DW Filter for PlatformRegion=China from Auto to Manual.
    • Restart the device when prompted.
  3. Confirm in DevMenu that DW Filter for PlatformRegion=China is set to Manual and also Disabled.

For more information about the settings as they relate to the enabling and disabling of the DirtyWord filter, see DevMenu > Changing Device Settings.
