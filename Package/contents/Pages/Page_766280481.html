Overview

You can check the send status of play reports and change play report settings. To send play reports, you must complete the procedures in Registering Network Connection Settings and Linking With Nintendo Accounts (or Linking With Accounts for China) in advance.

Info

Available in NintendoSDK Firmware for NX 10.0.0-2.0 and later.

How to Use

Checking the Transmission Status

The transmission status of report data is as follows.

Item Description
Transmission Status: Idle Report data that must be sent does not exist in storage.
Transmission Status: Pending

Report data that needs to be sent exists in storage, but sending is not taking place.
This problem may occur for the following reasons.

  • Not connected to the network.
  • The application has issued a network connection request.
  • The server is down.
  • There is no user consent for sending data.
Transmission Status: Processing Send processing is underway.
Info

Report data is also saved on the system, so the transmission status could change even if the application did not save report data.

Checking the Storage Capacity

You can check the storage capacity of report data saved in storage and not sent to the server.

Info

Because reports and the system share storage space, storage capacity could change even when the application does not save reports.

User Agreement Check Setting

This setting determines whether to check whether the user has agreed to saving and sending report data.

To enable this setting, select the User Agreement Check check box.

Item Description
Enabled The default setting.
Report data is sent according to the settings of the Nintendo Account linked to the system.
Disabled

The setting for debugging.
Report data is sent regardless of the settings configured in the Nintendo Account linked to the system.

Report data is sent even if there is no Nintendo Account linked to the system.
Info

Play reports are only sent if the Nintendo Account has opted in to send information.
For more information, see the Play Report Guide (日本語/English).

Allowing the Saving of Reports for Users Not Linked to Nintendo Accounts

This setting configures whether to save reports for users not linked to a Nintendo Account.

Configure the setting by setting the Allow to save unlinked user report check box.

Reports for users linked to Nintendo Accounts are not affected by this setting.

Item Description
Enabled

The setting for debugging.

Reports from users not linked to Nintendo Accounts are saved as reports from unspecified users.
Disabled

The default setting.

Reports are not saved from users not linked to a Nintendo Account.
Info

Available in NintendoSDK Firmware for NX 12.0.0-3.0 and later.

This setting is available by simply updating the firmware and does not require an update to the SDK for the application.

Clearing Storage

You can clear the storage that report data has been saved to.

To do this, click the Clear Storage button.

Request Immediate Transmission

Immediately transmits saved report data.

To do this, click the Request Immediate Transmission button.

Checking the Sent Report

You can see the total size of report data and the number or reports sent by the application. Restarting the system clears this information.

This information is displayed in Uploaded.

Checking the Number of Saves

A graph shows how many times the application saved report data each minute. Restarting the system clears this information.

This information is displayed in Save Count / min.
