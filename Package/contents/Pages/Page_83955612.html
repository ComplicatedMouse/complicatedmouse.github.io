Texture Pattern Animation

Texture pattern animations are created by using IFL files.

IFL Files

The IFL file is the standard text-format file supported by 3ds Max and is used on 3ds Max to replace the image file shown in every frame.
Specify IFL files the same way you assign image files as bitmaps for the diffuse lights of materials.

Texture names are registered in the IFL file in the order in which the image files will be replaced.

In the following example, the IFL file has been configured to play animation in the order p1, p2, p3, p1, p2, p3....

p2.tga

p3.tga

In this second example, the IFL file has been configured to play animation in the order p1, p1, p2, p3, p3, p1, p1, p2, p3, p3....

Info

p1.tga 2

p2.tga 1

p3.tga 2

The 3ds Max IFL Manager is a useful tool for creating IFL files.
For more information about IFL files, see the 3ds Max Help.

Repeating

Use the Time setting of the Material Editor to control how IFL files are played on 3ds Max.

All items in the Time category other than Sync Frame to Particle Age are applied in the texture pattern animation file.

Notes

  • The space is the only valid delimiter in IFL files.
