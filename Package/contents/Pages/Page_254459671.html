Table Easy Input

This section describes the Table easy input function for specifying which pattens to use for a texture pattern animation, and for changing the order in which the patterns are animated.

Overview

The Table easy input feature allows you to specify which pattens to use for a texture pattern animation and to change the order in which the patterns are animated.
The following overview describes the settings ().

Table Easy Input

Use hyphens to delineate a sequence of patterns to use in Texture pattern animation in numeric order. Use commas to delineate when specifying animation in frame units.
When entering frame units, specify the order to animate each pattern in the available white pattern () linked to the Number of pattern tables to use.
