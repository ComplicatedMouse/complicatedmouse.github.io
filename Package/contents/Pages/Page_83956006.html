Intermediate File Converter

This page covers usage of Intermediate File Converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dIntermediateFileConverter.exe) as well as other important information about the tool.

Overview

This utility converts generic format files into standard intermediate files.

It provides a limited set of intermediate file output features for use in environments that do not support the DCC tool.

This converter is strictly for limited use in special use cases. It does not support freely converting between generic file formats.
To create standard intermediate files from generic format files, load data into a DCC tool first and then export.

Dragging a file to be converted onto the executable file (3dIntermediateFileConverter.exe) or a shortcut to it converts that file to an intermediate file.
Files that cannot be converted are ignored.

If a problem is found, an error message appears and waits for key input.
If no problems are found, the checker exits silently.

Command-Line Options

3dIntermediateFileConverter.exe [--args-file ""] [--version] [-h] [-s] [-j ""]
[-o ""] [--error-detail-output ""] [--magnify ""] [--disable-invert-uv] [--texture-format-srgb] [--gpu-encoding ""] [--disable-texture] [--disable-original-image] [--3d-texture-converter-options ""] [file]...

Converts generic 3D data files into standard intermediate files.

--args-file ""

Specifies the file.
You can use the contents in the parameter file to specify arguments.
The parameter file allows you to specify each option over several lines.
Lines starting with a number sign (#) are treated as comments.
Newline characters cannot be included in escape sequences used in option specifications that appear between quotation marks (").

--version

Displays version information.

-h --help

Displays Help.

-s --silent

Suppresses the display of messages during processing.

-j "" --jobs ""

Specifies the number of jobs. When unspecified, the processor count is assigned.

File

Specifies the file to be converted. To merge a material intermediate file with a model intermediate fils and save the result, enter a model intermediate file and material intermediate file.

Supported files: FBX, OBJ, PNG, BMP, JPG, JPEG, TGA, FMDB, FMDA, FMTB, FMTA

-o "" --output ""

Specifies the output filename.
If this option is not specified, the filename extension is changed to an appropriate intermediate file extension, and the file is saved in the same folder.

--error-detail-output

Specifies a folder for saving detailed information when an error occurs. If not specified, this information is written to the default folder.

--magnify ""

Specifies the magnification to the output file's vertex coordinates and translation value.
Supported files: FBX, OBJ, FMDB, FMDA, FSKB, FSKA

--disable-invert-uv

Disables inversion of UV coordinates.
Use this option for textures with a UV coordinate system in which the upper-left corner is the origin and down is positive.
Supported files: FBX, OBJ

--texture-format-srgb

Use this option to convert a texture as an SRGB texture.
Supported files: FBX, OBJ, PNG, BMP, JPG, JPEG, TGA

--gpu-encoding

Specifies whether to use the GPU to encode the texture. For more information, see the description of the --gpu-encoding option in 3DTextureConverter.

Supported files: FBX, OBJ, PNG, BMP, JPG, JPEG, TGA

--disable-texture

Disables the conversion of the referenced texture files when converting 3D model files.

--disable-original-image

Suppresses output of the original image during texture file conversion.

--separate-materials

Separates materials included in an input model file into material intermediate files and exports them.

Supported files: FMDB, FMDA

--3d-texture-converter-options

Specifies additional options for the 3D Texture Converter used when converting textures.

Supported files: FBX, OBJ, PNG, BMP, JPG, JPEG, TGA



Restrictions

FBX File Restrictions

  • Can only be converted to model intermediate files (animations are not supported).
  • Only tested for FBX files exported from Maya as FBX 2016/2017 (7500) files.
  • Only supports conversion of triangular polygons.
  • Skinning is always treated as smooth skinning, even when the number of influenced bones is one. This restriction will be removed in the future.
  • For the following model intermediate file attributes, when using the DCC plug-in, the settings configured in the DCC tool are used. However, when using Intermediate File Converter, these attributes are fixed to the following values.
Element Attribute Name Value
<bone> visibility true
scale_compensate false
motion_mirroring_type auto
compress_enable true
<material_info> compress_enable true

Example Use

Describes procedures for using the binary intermediate file converter.

Converting OBJ Files

Converts target.obj to target.fmdb.

3dIntermediateFileConverter.exe target.obj

Converting JPG Files

Converts source.jpg to destination.ftxb.

3dIntermediateFileConverter.exe source.jpg -o destination.ftxb

Converting FMD Files

Doubles the size of source.fmdb and converts it to destination.fmdb.

3dIntermediateFileConverter.exe source.fmdb --magnify "2.0" -o "destination.fmdb"

Separating Materials From Model Intermediate Files

Separate materials included in a source.fmdb file into material intermediate files and save them. Material intermediate files are written to the same folder as the resulting model intermediate file.

3dIntermediateFileConverter.exe source.fmdb --separate-materials -o "destination.fmdb"

Sample Description
