NintendoWare Spy News

This page introduces the changes in NintendoWare Spy library for NintendoSDK 1.2.0.

The Spy debug panel has become an official feature.

Evaluation has finished for the Spy debug panel and it is now an official feature.

The Spy debug panel shows the operating state of the Spy library. Currently, it can monitor the use rate for the data send buffer.

For more information, see Spy Debugging Panel.

The data that appears in this panel us sent using the nn::spy::DebugModule class.
