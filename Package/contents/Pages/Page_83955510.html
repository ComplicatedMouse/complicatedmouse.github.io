Color Animation

Standard materials are set by animating the material nodes' Color, Transparency, Ambient Color, Incandescence, and Specular Color.

  • If a file node is connected to the Color, Incandescence, or Specular Color attribute, animations for which the Color Gain attribute of the file node has been set are applied in the intermediate file.
  • If a layeredTexture node is connected to the Color, Incandescence, or Specular Color attribute, the animation set using the Color Gain attribute of the file node connected to the top layer is applied in the intermediate file.
  • If a file node is connected to the Transparency attribute, the animation set using the Alpha Gain attribute of the file node is applied in the intermediate file.

For the color animation, data is exported only for those attributes that have been set for animation.
Attributes for which data was not output are shown in the FMD file. Even if an attribute has a set value, if it differs from the state in the CMDL file, make sure that you set at least one key to the attribute before exporting the color animation.

Info

If the Compress Material export option is enabled, even materials for which color animation is not set sometimes get animated as a result of united materials.
If this occurs, use the plug-in for setting material attributes and set the flag so that materials with color animations are not compressed.

Info

For extended materials, set the animation for the attributes specified in the config file.
The behavior when textures are connected is the same as standard materials.
