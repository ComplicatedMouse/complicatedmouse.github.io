Overview

This section describes the Render Particle feature for determining whether particles will be rendered.
You can also use the feature to enable the display of particles emitted from child emitters while disabling the display of particles from the parent emitter.
The following overview describes the settings ().

Rendering Particles

You can enable or disable rendering of particles emitted from the emitter by selecting or clearing the check box.
Particle calculations are performed even when this check box is not selected.
