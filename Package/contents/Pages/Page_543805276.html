This page lists the known bugs in this release.

  • nifm

movie

The 32-bit Windows version of MoviePlayerSimple does not work right on Windows 10.

Symptoms

The version of MoviePlayerSimple built as a 32-bit application does not work right when run on Windows 10. Movies stop while playing and there is no audio.

Workaround

No workaround is currently available.

Fix Schedule

This bug is being investigated. There is no specific fix schedule.

MoviePlayerSimple cannot play 1080p movies correctly.

Symptoms

When a movie with a resolution of 1080p is played in the Windows version of MoviePlayerSimple, the video stops and only the audio continues to play.

Workaround

No workaround is currently available.

Fix Schedule

This bug is being investigated. This bug will be fixed in a future version.

nifm

Cannot connect to WEP access points in the Windows environment.

Symptoms

With the Generic version of the SDK, cannot connect to WEP access points in the Windows environment, and network use requests return error 2110-1100.

Workaround

Connect using a method other than WEP, such as unencrypted, WPA-PSK (AES), or WPA2-PSK (AES).

Fix Schedule

This bug was fixed in NintendoSDK 8.3.0.

perf

The PerfSimple sample crashes in some PC environments.

The PerfSimple sample exits with an assertion in certain PC environments.

This bug will be fixed in a future release.
