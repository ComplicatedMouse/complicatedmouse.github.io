Rotation Speed

This section describes the Rotation Speed feature for setting the speed at which particles rotate.

Overview

The Rotation Speed feature can be used to rotate emitted particles around the axis specified for Rotation Axis.
The overall effect is a twister.
The following overview describes the settings ().

Rotation Speed

Sets the rotation speed. The rotation direction depends on the sign of this value.
The configured value is added to the positions of the particles in every frame, so you can achieve effects of equal rotational speed.

Info

Press the Particle Time Animation button () to display the Rotation Speed (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().
