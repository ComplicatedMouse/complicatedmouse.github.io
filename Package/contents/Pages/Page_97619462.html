Matrix Settings Type

The section describes the Matrix Settings Type feature for configuring how scale is applied to particles.

Overview

The Matrix Settings Type feature allows you to select how the developer can scale emitter sets.
You can use EffectViewer to check the results of changing these settings.
Designers can use this feature to preview the actual appearance in the game application of the settings used by the developer.
The following overview describes the settings ().

Matrix Settings Type

Select how to apply scale to particles from among the following presets.

setMtx The standard setting.
setMtxFast

Accelerates the process.

Select setMtx if particle scaling is not being applied correctly in the Viewer.
