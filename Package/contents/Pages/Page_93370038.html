Emitter Shape - Circle

Describes the settings for Emitter Shape - Circle, where particle emission locations are set around the circumference of a circle.
These are the particles when this emitter shape is being used.

Overview

With the Circle emitter shape, particles emit in the direction of the normal in a widening circle shape from random positions around the circumference of a circle located in world coordinates.
The following overview describes the settings ().

When the emitter shape is set to Circle, the Circle () settings are displayed. The group has the following settings.

Emitter Radius

Set the circle radius using x y z.

Arc Width (Angle)

Sets the rotating arc by the angle set in Arc starting point (angle) on the axis through the center of the circle.

Arc Starting Angle

Sets the arc starting point with an angle. For the default value, the z-coordinate in the positive direction serves as the starting place.

Random Arc Starting Angle

You can enable or disable the randomization of the arc starting angle by selecting or clearing the check box ().
