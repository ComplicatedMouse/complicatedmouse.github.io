Gradient Overlays

This section describes the gradient overlays feature for filling a picture pane with a gradient.

Overview

Gradient overlays fill the inside of a picture pane with a gradient.
The following sections describe the various gradient overlay features.

Rendering Mode

Configures the method used to render gradients in a picture pane.
The following images compare the effects of changing the rendering mode.

Normal

The default setting.
Performs translucent blending with the gradient.
Multiplicative

Performs multiplicative blending with the gradient.
Additive

Performs additive blending with the gradient.
Subtractive

Performs subtractive blending with the gradient.

Number of Control Points

You can configure up to four control points for a gradient.
The following images compare the effects of changing the number of control points.

1

Set color at a single location and fill the entire picture pane with that color.
2

The default setting.
Colors are set in two locations. The colors are interpolated between the control points.
4

Colors are set in four locations. The colors are interpolated between the control points.

Gradient Settings at Each Control Point

Sets the gradient color and adjustment value for each control point.
Assuming that the default angle is used (0.00), the left edge of the picture pane would be set as 0.00 and the right edge would be set as 1.00.

Angle

Sets the angle at which to apply the gradient to the picture pane.
When the adjustment value for the control point is 0.00, the adjustment location is set as a counter-clockwise angle with the left edge of the picture pane as 0 degrees.

The following images compare the effect of changing the angle when you create a gradient between control point adjustment values of 0.00 to 1.00.

0

The gradient is applied from left to right.
45

The gradient is applied from the lower-left corner to the upper-right corner.
90

The gradient is applied from the bottom to the top.
