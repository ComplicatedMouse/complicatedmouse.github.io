Searching and Replacing Settings

This section describes how to use the Search window to search for effect data with the specified settings and then batch-edit those settings.

Using Search Criteria

Specifying a Search Folder

Drag the folder to search into the text box, or click the folder button and select the folder to search.

Specifying the Search Criteria

Select search criteria in the combo box and click the Add button.
Example: Select Basic Settings > Particle follow type in the combo box and click Add.

For search items requiring three values, such as direction or position, enter them separated by spaces or commas, for example, as in 0.1 1.0 1.5.

Searching

Click the Search button to find the effect files that match the search criteria and display them in the Search Results tab in the bottom of the window.

Using Combinations of Search Criteria

The search is conducted on the combinations of criteria, specified as expressions and used for evaluation in order from the top.

Specifying Multiple Search Criteria

Example: Select Basic Settings > Particle follow type in the combo box and click Add.
Example: Select Basic Settings > Process type in the combo box and click Add.

Combine search criteria

Click Add and set a combination expression.
Example: Set expression 1 as A　AND　B.

Searching

Click the Search button to find the effect files that match the search criteria and display them in the Search Results tab in the bottom of the window.

Replacement Expressions

You can use the Replace tab to replace the settings in the effect files shown in the search results.
To prevent data loss, back up your data before replacing any settings.

Specifying a Replacement Expression

Add a replacement expression for the part you want to replace. For example, select Basic Settings > Particle follow type in the combo box and click Add.

Performing the Replacement

Click Replace to apply the replacement expression to all emitter files found in the search.

For Special or Difficult Searches

  • Searching by emitter type (CPU/GPU/GPU-SO).
    • Select Basic Settings > Basic Settings > Process Type.
  • Determining whether the emitter is enabled.
    • Select: Other > Check Enabled > Emitter.
  • Search based on presence of child emitters, stripes, or custom actions.
    • Select: Other > Check presence.
  • Search based on selected custom shader settings in a custom shader.
    • Select: Other > Custom Shaders > Selected index.
  • Search based on custom action callback ID selection content.
    • Select: Other > Custom Action > Selected index.
