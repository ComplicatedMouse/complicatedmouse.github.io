Base Pane

The base pane is used to edit panes in LayoutEditor.
The editing options are the same for various types of panes.

Pane Position and Shape

Sets the pane position and shape.

Position

Enter the position where you want to place the pane.
Size

Enter the pane size.
Scale Enter the pane scale.
Rotate

Enter the pane rotation.

Transparency

Sets the pane transparency.

Transparency

Enter the pane transparency. (000 to 255)

Apply the transparency to children ()

Specifies whether to apply the transparency to the child hierarchy.

Select the check box () to apply.

Pane Center

Sets the center position of the pane.

Center of Parent Pane

Sets the center of the parent pane.


Info

Set the parent pane to enable input.

Other

Configures other settings.

User Information (maximum 8 characters)

Enter user information in the text box (). (Maximum 8 characters)

Show/Hide ()

Use the check box () to switch between showing and hiding a pane.

(Shows the pane if selected.)

Target for position adjustment ()

Enables runtime position adjustment.

(Position adjustment can be used for creating widescreen layouts.)

Prohibit thinning as result of pane tree compression (experimental) ()

Setting affects panes controlled from the program. (Experimental)

Enable extended user information animations ()

Enables extended user information animations.
Part Scaling Effect

You can specify a size scaling for the parts pane and adjust the size.

The part scaling (size scaling of parts) described in this section affects the parts layout and can be specified for each pane separately.
For more information, see Effects of Part Scaling.
Specify the horizontal alignment margin

The margin (horizontal) can be set individually for the panes under the Align Pane using the check box ().
Specify the vertical alignment margin

The margin (vertical) can be set individually for the panes under the Align Pane using the check box ().
Exclude from Align Pane processing

Excludes panes from processing, even if a pane is placed under the Align Pane, using the check box ().

If this functionality is enabled in the Parts Layout, it will be disabled in the Target Layout.

Part Scaling Effect

This setting controls how the layouts where parts are arranged are affected by the size scaling setting configured for parts panes ().

Ignore

Ignores size scaling.
Specify this option when you want the pane to maintain a fixed size, regardless of the size of the part.
Scale size

Scales the size.
Maintain distance from boundaries of parts containing self

This setting is the default.
The pane size is changed so that a set distance is maintained from the edge of the part to which size scaling has been applied.
