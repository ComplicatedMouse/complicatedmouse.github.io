Fresnel Alpha, Process as Fragment

This section describes the Process as Fragment feature that handles Fresnel Alpha processing on a particle fragment basis.

Overview

The Fresnel alpha feature Process as Fragment processes the Fresnel Alpha at the level of the primitive polygon face's pixel.
The alpha gradation will appear cleanly, even for primitives with few vertices.
The following overview describes the settings ().

To display Process as Fragment (), in Fresnel Alpha, select Process as Fragment. The group has the following settings.

Angle of minimum alpha

Sets the angle (0.0) where the alpha value is at a minimum, specified in terms of the dot product of the view vector and normal vector.

Angle of maximum alpha

Sets the angle (1.0) where the alpha value is at a maximum, specified in terms of the dot product of the view vector and normal vector.

Example Configurations

The following figures show how to look at effects when using Process as Fragment, and the output value corresponding to the specified input value.
The horizontal axis represents the angle (the input value) between the view vector and polygon normal vector.
The vertical axis represents the alpha value (the output value) for the polygon face fragment.

Viewing the Effect Figure

Set the following two values as input values.

A Angle of minimum alpha 0.2

When set to 0.2, the minimum alpha value is set to 0.0.
B Angle of maximum alpha 0.8 When set to 0.8, the maximum alpha value is set to 1.0.
