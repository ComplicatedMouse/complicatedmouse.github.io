Emission Interval Type - Distance

This section describes the Distance emission interval type.

Overview

The distance type emission interval allows the particle emission to be set by distance.
Setting the emission interval to distance allows the particle emission interval to be set independent of the emitter's movement speed.

The following overview describes the settings ().

When you set the emission interval type to Distance, the Emission Interval Type - Distance () settings is displayed. The group has the following settings.

Emission interval Describes the emission interval feature, which sets the interval at which an emitter emits particles in terms of the emitter’s movement distance.
Maximum Traverse Distance per Frame Describes the maximum traverse distance per frame feature, which limits particle emission to ensure that no excess particles are emitted when emitters have a high movement speed.
Minimum Traverse Distance Per Frame Describes the minimum traverse distance per frame feature, which treats an emitter as having moved, even when it barely moves or does not move at all to ensure that particles get emitted.
Cutoff Threshold for Traverse Distance Describes the Cutoff threshold for traverse distance feature, which limits particle emission to ensure that no excess particles are emitted when emitters have a high movement speed.
Maximum Number of Particles Emitted Describes the Maximum Number of Particles Emitted feature, which limits the number of particles that can be emitted.
