What Are Controls?

This mechanism defines the functionality of layouts.
More specifically, the controls consist of feature panes that define panes with specific roles, and feature animations that define animation segment tags with specific roles.

What Can Be Done?

By adding control definitions you can define new types of functionality.
For example, if developers prepare an extended program, the designers can create buttons and other parts with unique capabilities.

Also, if developers create a viewer for the custom parts, designers can check the ultimate capabilities of the parts and efficiently try out different ideas without needing help from the developers.
