Emission Interval Type - Time: Emission Rate

Emission Rate

This section describes the emission rate setting, which determines how many particles are simultaneously emitted from an emitter.

Overview

The emission rate feature can be used to configure the number of particles that are simultaneously emitted from an emitter.
The following overview describes the settings ().

Emission Rate

Sets the number of particles that the emitter emits simultaneously.
If a value less than 1.00 is set, that value is added in each frame, and one particle is emitted after the sum reaches 1.

Note

Set a nonzero value for the emitter time animation key.

Emission Rate Randomness (%)

Randomly changes the particle emission rate.
The value set for Emission Rate is used as the upper limit of the random value. The lower limit changes randomly. The following formula calculates a lower limit for the random number.

  • Random Number Lower Limit: Calculated as Emission Rate minus Emission Rate Randomness.
