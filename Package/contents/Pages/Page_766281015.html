Play Report Feature

Use the prepo command. For more information about play report settings, see Play Report Settings.

  • User Agreement Check Setting
  • Allowing the Saving of Reports for Users Not Linked to Nintendo Accounts
  • Clearing Storage
  • Report Data Log Output

$ RunOnTarget.exe DevMenuCommand.nsp -- prepo
usage: DevMenuCommand prepo set-user-agreement-check <enable|disable>
       DevMenuCommand prepo set-unlinked-user-report-save-allowed <enable|disable>
       DevMenuCommand prepo clear-storage
       DevMenuCommand prepo print-reports
       DevMenuCommand prepo dump-reports <absolute_path> [--json] [--force]
Info

Available in NintendoSDK Firmware for NX 10.0.0-2.0 and later.

User Agreement Check Setting

For saving and sending play reports, set whether to check if the user consented.

$ RunOnTarget.exe DevMenuCommand.nsp -- prepo set-user-agreement-check <enable|disable>

Allowing the Saving of Reports for Users Not Linked to Nintendo Accounts

This setting configures whether to save reports for users not linked to a Nintendo Account.

When it is enabled, reports from users not linked to Nintendo Accounts are saved as reports from unspecified users.

When this setting is disabled,and in the case of the retail product, reports are not saved from users not linked to a Nintendo Account.

Reports for users linked to Nintendo Accounts are not affected by this setting.

$ RunOnTarget.exe DevMenuCommand.nsp -- prepo set-unlinked-user-report-save-allowed <enable|disable>
Info

Available in NintendoSDK Firmware for NX 12.0.0-3.0 and later.

This setting is available by simply updating the firmware and does not require an update to the SDK for the application.

Clearing Storage

Clears the storage that report data has been saved to.

$ RunOnTarget.exe DevMenuCommand.nsp -- prepo clear-storage

Report Data Log Output

Report Data File Output

Generates report data saved to a file. If no format is specified, the output format is MessagePack. If you select the --json option, the output is in JSON format.

$ RunOnTarget.exe DevMenuCommand.nsp -- prepo dump-reports <absolute_path> [--json] [--force]
