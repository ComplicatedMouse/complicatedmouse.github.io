Per-Character Transformation

Per-Character Transformation

Capabilities

Translation and rotation transformation processes can be applied on a per-character basis in a text box pane.
The transformation processes can also be animated.
This makes it possible to more easily achieve special effects like positioning characters along an arch or having characters undulate.

Usage Examples

Overview

This feature sets attributes such as color, position, scale, and rotation that vary for each individual character in the string in the text box pane.
Create a curve to reference whenever a character is placed in a text box pane. Attribute values such as color, position, scale, and rotation are applied to the character using the curve.

Settings

The following table shows the settings for per-character transformations.

Enable Enables or disables per-character transformations.
You can set other parameters after this setting has been enabled.
Offset/time animation simple settings

Opens the Offset/Time Animation Simple Settings for Per-Character Transformations window to set time animations.
For more information, see the linked tutorial.
Offset time Sets the time at which to start evaluating the curve.
Duration Sets the evaluation range of the curve.
Loop Configures how the curve repeats.
Origin (V Direction) Sets the center point to use when applying scaling or rotation using per-character transformations.
Height Offset Sets the height offset value from the origin.
Divide Based on Character Width When dividing a curve, divide using the size corresponding to the character width instead of dividing equally based on factors such as the number of characters.
Automatically track shadow alphas to character alphas When enabled, this setting causes the alpha values set for text shadows to track the alpha values of the top and bottom colors for per-character transformations.
Place Text in a Circle Shows the Place Text in a Circle dialog box, and sets the position of that circle.
For more information, see the linked page.
Set the origin point for conversion as the center of the text box When enabled, each character treats the center of the text box pane (its base position) as the origin.
When disabled, each character treats its own center as the origin.
Place characters at fixed intervals Determines whether to place characters at fixed intervals when Set the origin point for conversion as the center of the text box is enabled.
When enabled, this setting configures both the character intervals and justification.

Important Information

  • In per-character transformation, the transformation curve is treated as an animation curve, so there are restrictions on the use of parts similar to the restrictions on animations.
    For example, if a per-character transformation has been set in a parts layout, the per-character transformation will not work if that parts pane is placed in a body layout.
    Also, the per-character transformation for a parts pane cannot be overwritten.
  • The settings for the per-character scale, position, rotation, and color curves are shared by all segment tags.
    The range to use is determined by the offset time and the duration.
  • The layout binary converter exports the per-character scale, position, rotation, and color curves to the BFLYT file, which has the layout data, not to the BFLAN file, which has the animation data.
    It exports the offset time and the duration curves to the BFLAN file, which has the animation data.
