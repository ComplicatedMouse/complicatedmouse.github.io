Timeline Panel

Timeline Panel

This panel displays the data on a timeline.

Currently, it only supports the display of audio performance and number of voices.

UI Reference


 Item Description
A nn::audio Performance

Indicates the processing time for Voice, SubMix, FinalMix, and Sink.

  • This time starts when the first item starts and continues until the last item ends for Voice, SubMix, and Sink.

The legend represents the time passed from the start until the end of the processing as a numerical value.

The graph shows the scope from the start time until the end time of the processing.

The unit is microseconds.


IsOverTime indicates whether the rendering time has exceeded the maximum value.

  • The value is either 0 or 1. 1 indicates that the value has exceeded the maximum.
  • If the value exceeds the maximum, noise may occur in the rendering results due to processing slowdowns.

For more information about each library component, see Audio Performance Information.
B nn::audio Number of voices

Displays the voices that are playing and the dropped voices as both a numerical value and a graph.

  • VoiceCount indicates the number of voices that are being played.
  • DropCount indicates the number of dropped voices.
C Vertical orange line Represents the current time.

Operations

Operation Procedure
Click Moves the current time to the specified position.
Right-click + drag Scrolls horizontally.
CTRL+mouse wheel Magnifies and shrinks in the horizontal direction.
ALT+mouse wheel Magnifies and shrinks in the vertical direction.
