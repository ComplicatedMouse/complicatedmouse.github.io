SystemEvent Class

Premise

The system event feature is functionality that enables synchronous waiting for notifications from the system as events.

Feature Overview

The SystemEvent class wraps the System Event Features by using the nn::os::SystemEventType object and contains the following member functions. For more information, see the API reference.

Name Description Comments
SystemEvent Constructor

Use a constructor without parameters to create an uninitialized instance.
~SystemEvent Destructor. Destruction equivalent to DestroySystemEvent.
Wait Waits for a system event System event wait equivalent to WaitSystemEvent.
TryWait System event polling System event polling equivalent to TryWaitSystemEvent.
TimedWait Timed wait for a system event Timed wait for a system event equivalent to TimedWaitSystemEvent.
Clear Clears system events System event clearing equivalent to ClearSystemEvent.


Examples of Using SystemEvent Classes

The system event feature presumes that it will be used while using functionality provided as NintendoSDK modules. Examples are not included due to the difficulty of disclosing such information as a single sample. For more information, see the sample programs, among other things, that describe how to use each module.
