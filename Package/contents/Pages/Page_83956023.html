Termination Processing

This section describes the "end process" feature for forcing force effects to stop playing.

Overview

The Termination Processing feature enables a developer to use the Fade command in a game application to force particles to stop being emitted or force emitted particles to die.
Even particles that have been set to Infinite lifespan can be forced to die. In addition, you can specify alpha and scale values to control how the particles extinguish when they die.
The following are settings for termination processing.

Termination Processing

Settings for configuring the behavior of particles when the Fade command executes.

Stop Emission This feature stops the emitter from emitting particles. Already emitted particles are displayed for their lifespans.
Alpha Fadeout With this feature, all particles grow gradually more transparent over the period of time specified in Fade-out duration, after which they disappear.
Scale Fadeout With this feature, all particles gradually shrink in size over the period of time specified in Fade-out duration, after which they disappear.
Fadeout Duration This feature sets the duration, in frames, required for the fadeout. Alpha Fade and Scale Fade may be used by turning it ON.
Info

With the Termination Processing feature, you can check the result of calling the Fade command in the Viewer by clicking the Fade button () in the preview controls.
