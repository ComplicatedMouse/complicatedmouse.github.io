Getting the Play Activity

Use the playdata command to get the record of play activity for an application.

$ RunOnTarget.exe DevMenuCommand.nsp -- playdata --help
usage: DevMenuCommand playdata query-playstatistics <application-id> [--account <index>]
       DevMenuCommand playdata export <absolute_output_file_path> [--total] [--application-id <application_id>]
[SUCCESS]

Displaying the Play Activity

To display the play activity, specify the application and the account.
The following example command gets the play activity for the first registered account (index 0) playing the application with the application ID 0x1007dc000069ffd.

$ RunOnTarget.exe DevMenuCommand.nsp -- playdata query-playstatistics 0x1007dc000069ffd --account 0

Exporting the Play Activity

The play activity for an application can be exported as a CSV file.
The play activity includes the following information.

Data Recorded Every Time the Application Starts Cumulative Recorded Data
  • Application ID
  • Play time (seconds)
  • Startup time (console clock)
  • Start time (network clock)
  • Application ID
  • Cumulative play time (seconds)
  • Number of times played

Play time is rounded to the nearest second. If the power is turned off while the application is running, the play time will be treated as 0 seconds and the number of times played will be incremented in the cumulative recorded data.

Exporting Play Activity for All Applications

$ RunOnTarget.exe DevMenuCommand.nsp -- playdata export C:\playdata.csv
Export the play log to "C:/playdata.csv".
[Success]

Exporting Play Activity for a Certain Application

$ RunOnTarget.exe DevMenuCommand.nsp -- playdata export C:\playdata.csv --application-id 0x1007dc000069ffd
Export the play log to "C:/playdata.csv", with application-id(0x1007dc000069ffd).
[Success]

Exporting Cumulative Application Play Activity 

$ RunOnTarget.exe DevMenuCommand.nsp -- playdata export C:\playdata.csv --total
Export the play log to "C:/playdata.csv".
[Success]
