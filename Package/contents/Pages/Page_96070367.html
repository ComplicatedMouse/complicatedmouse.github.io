Initial Rotation

This section describes the Initial Rotation feature which sets the tilt angle when a particle is emitted.

Overview

The Initial rotation feature allows you to set the initial tilt angle of a particle when emitted.
The following overview describes the settings ().

Initial Rotation

Sets the tilt angle when particles are emitted.
Setting a negative value tilts in the clockwise direction, and setting a positive value tilts in the counterclockwise direction.

Use Random Settings () to specify the ranges within which to randomize the values.
Randomly changes the tilt angle when particles are emitted.
For example, specifying 60 degrees means a range of 30 degrees variation with respect to the standard value.
