nn::g3d News ()

Deprecated the texture converter's texture intermediate file features.

The texture converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\GraphicsTools\TextureConverter.exe) features that are related to texture intermediate files, will be deprecated in the next or a subsequently released version. To convert texture intermediate files, you should henceforth use the 3D texture converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dTextureConverter.exe), which has been newly added to the NintendoWare package.

Intermediate File-Related Tools Have Been Moved

The paths have been changed for the following tools. The old paths will be deprecated in the next or a subsequently released version. You can continue to use these tools during the migration process, but after the migration period has ended, they will be deprecated

Creating an empty file with the name Use3dTools under %NINTENDO_SDK_ROOT%\Tools\Graphics\GraphicsTools causes an error to occur when using a pre-migration tool. Make sure that you confirm whether they have been migrated before use.

nn::ui2d News ()

Added a Fill Optimization Feature for the Layout Converter.

Added a feature to create mesh shapes that do not draw transparent pixels for fill optimization of the picture pane in Layout Converter. For more information, see Mesh Trimming Optimization.

Made some adjustments to members of the Pane class that are related to expanded user information to reduce memory usage.

Members of the Pane class that are related to expanded user information have been adjusted to reduce memory usage. The reduction of a single pointer member reduces memory usage by 8 bytes per Pane in a 64-bit environment.

As a result of this change, any pane that uses expanded user information animations will be unable to swap pointers with other user information via SetExtUserDataList. (This operation causes a memory leak, so it includes an assertion to prohibit this process.) Please take note of this change, even if it is difficult to imagine a scenario where this operation would be used intentionally, and there probably will not be many cases where it would cause any issues.
