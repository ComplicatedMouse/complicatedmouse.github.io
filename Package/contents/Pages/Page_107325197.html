Introduction

diag Library

The diag library is a library that provides functions used for program debugging, such as log output, assert, and abort.

Intended Audience

This documentation is intended for application developers using NintendoSDK.

Related Documentation

Many functions are provided as macros in the diag library. See the Macro Reference.

Also see the diag API Reference.

For more information about the character codes input into these macros and functions, see Considerations for Log and Assert Character Codes.

For more information about the fixed character strings that are output when there is an assertion failure or the application has aborted, and the reasons why the strings are output, see Unexpected Exit of Application.
