Startup and Initial Settings File

Starting SoundMaker

To start the application, run \Tools\Audio\SoundMaker\SoundMaker.exe.

　： SoundMaker.exe icon

Initial Settings File

When SoundMaker starts, it creates a file named SoundMaker.xml in the %APPDATA%\Nintendo\SoundMaker\<version_number> folder.
This file is called the initial settings file.

Info
%APPDATA% is an abbreviation for
C:\Users\<username>\AppData\Roaming\folder (Windows 7).

Role of the Initial Settings File

SoundMaker saves information such as the operational settings in this initial settings file.
For example, information about previously opened project files is saved in the initial settings file, so that these same project files can be opened later by selecting File > Recent Projects.

The initial settings file can continue to be used even after an upgrade to the version of SoundMaker.

When there is no initial settings file the first time SoundMaker is started, copying is performed in the following order.

  1. In the same location as SoundMaker.exe.
  2. From the latest directory in which initial settings files would be placed.

Example:

If %APPDATA%\Nintendo\SoundMaker\ has the folders:
1.0.0.0
2.0.0.0
1.5.0.0
In this case, the SoundMaker.xml folders are searched in the following order.
2.0.0.0
1.5.0.0
1.0.0.0
