Curve Editor Window

This section describes the Curve Editor window, which provides a graphical way to create and edit emitter time animations, with time on the horizontal axis and values on the vertical axis.

A List View

Displays a list of emitter-time animation features.
Animation is set in Curve View for the Animation ON () item.
B Toolbar

A group of tools for operating the curve editor.

Click Move All () to open a window and specify an amount by which to move the entire curve along the time axis.
C Curve View

Displays the nodes that constitute an effect in tree format.

Right-clicking opens the workspace Edit menu.
