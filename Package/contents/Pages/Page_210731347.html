This section introduces new features that have recently been added to EffectViewer.

Maintaining Color Length When Hue Is Changed in RGB Mode

You can now maintain the color length when the hue is changed in RGB mode.
The following table shows the difference when setting R,G,B = 5,0,0 in RGB mode and changing the hue in the color editing UI.

Before Support After Support

R,G,B = 0,1,0

Becomes R,G,B = 0,5,0 and stored as a value of 1 or greater.

Related Links For more information, see the following pages.

Color Editing UI This UI provides a way to edit selected color and alpha values.

Custom Action Fixes

You can now omit the callback function selection operation in a custom action.
The callback function that was registered at the point when the custom action node was added will be selected by default.
This change means that you no longer need to select the custom action program from Callback ID every time.
