Z Buffer and Alpha Test

This section describes the feature for testing depth and alpha when rendering particles.

Overview

This feature allows you to configure the settings for the depth test and alpha test when particles are being rendered.
The presets offer combinations of frequently used settings. You can also customize you own combinations of settings.
The following is an overview of the setting.

Z Buffer and Alpha Test
