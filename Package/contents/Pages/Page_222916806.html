Animation Settings for Frame Size in the Window Pane

You can now configure animation for the frame size in the window pane.

In the window pane, in the property window, you can configure Frame Size just like other animations.
This window pane frame size animation is edited using the Curve Editor window.

Added support for Overwrite textures only, a part overwrite feature.

Added support for Overwrite textures only, a part overwrite feature.
Overwrite textures only is a feature that only overwrites textures, similar to Overwrite color interpolation only.

Use Overwrite textures only to only overwrite the textures, without overwriting the color interpolation.

Previously, it was necessary to use Overwrite Picture Pane, Overwrite Window Pane, etc., to overwrite the texture using the part overwrite feature, and all elements, including color interpolation and other material information, was overwritten.
As a result, settings such as color interpolation for the body layout parts pane changed to a state where they could not be used as is (because they would be overwritten).
Use the Overwrite textures only feature to enable the body layout Parts Pane settings to use all settings, other than the texture overwrite, as is.

Using the Overwrite textures only part overwrite feature, enables more efficient work processes.
