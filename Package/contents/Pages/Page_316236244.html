Mapping Settings

This section describes the Mapping Settings feature for setting how to apply texture images to panes.

Overview

Select the mapping method from Mapping Method when applying a texture image registered to Texture () to a pane.
Mapping Method has the following presets.

UV Mapping (Using UV Coordinate 0) A mapping method able to apply texture images along the texture coordinates (UV coordinates) of the pane.
Texture images are always affected by changes to the position, size, and direction of the pane.

Orthogonal Projection Mapping

A mapping method that projects the texture image on a pane.
Projection Mapping Relative to Pane Position Orthogonal projection mapping to project the texture image based on the translated position when the position of the pane changes.
Perspective Projection Mapping

A mapping method that projects the texture image on a pane.

During perspective projection () by the layout camera, the projected texture image is projected in the screen space, so perspective can be disabled.
Perspective Projection Mapping Relative to Pane Position Perspective projection mapping to project the texture image based on the translated position when the position of the pane changes.
