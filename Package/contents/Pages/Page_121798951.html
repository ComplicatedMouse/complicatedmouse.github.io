gfxutil Library

Introduction

Purpose of This Documentation

This documentation describes the gfxutil library, which provides supplementary graphics features and is included in NintendoSDK.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Chapter Descriptions

The Manual describes how to use the features provided by the gfxutil library.

Related Documentation

Also see the GfxUtil API Reference.

Features Provided by the gfxutil Library

The current version of the gfxutil library provides the following features.

  • DebugFontTextWriter
  • DescriptorPoolAllocator
  • MemoryPoolAllocator
  • TextureCompressor

For more information about each feature, see the Manual.
