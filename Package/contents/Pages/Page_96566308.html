Random Noise Offset

This page describes the Random Noise Offset feature for randomly changing the magnitude of the positional offset specified for the Noise Offset feature.

Overview

The Random Noise Offset feature makes it possible to randomly change the magnitude of the offset for the noise table specified with the Noise Offset (±) feature.
When this random offset is enabled, the magnitude of the offset for the noise table is randomly determined within a range from 0 to the value specified for the Noise Offset (±) feature.
As a result, enabling a random offset makes it possible to assign unique movement characteristics to each particle.
The following overview describes the settings ().

Random Noise Offset

Use the check box () to enable and disable the random noise offset.
To create variations in the motion of particles each time one is emitted from the emitter, set Noise Offset to 1 and enable this feature.
