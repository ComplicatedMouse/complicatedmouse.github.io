Setting for Key Items Only

Key Items Only is available when the animation is in separated mode.
This feature determines the scope of evaluation when Key Items Only is selected for the filtering method.
This selection is retained when you restart Layout Editor.

Toolbar

G


Curve Operation: Standard

The basic operation.


Curve Operation: Adjust Nearest Key

Adjusts the key nearest the mouse pointer from among multiple selected keys.


Curve Operation: Insert Key Following Curve

Inserts a key at the intersection of the line extending vertically from the mouse pointer and the curve.


Curve Operation: Add Key to Click Position

Adds a key to the position right-clicked in the graph display area.
H Key Time

Displays the selected key time (horizontal axis).
I Key Value

Displays the selected key value (vertical axis).
J Snap Increment

Specifies the snap increment of the values (vertical axis) in the range of from 0.01 through 10 when editing keys with the mouse.
K


Previous Key Selects the previous key.


Next Key Selects the next key.


Create Key

Creates a key in the current frame. When there is no curve, creates a key using the value of the selected item.

When there is a curve, creates a key on the curve.
L


Set Slope to Spline Sets the slope of the selected curve to spline.


Set Slope to Linear Interpolation Sets the slope of the selected curve to linear interpolation.


Set Slope to 0 Sets the slope of the selected curve to 0.


Set Slope to Step Sets the slope of the selected curve to step.

Selected Pane Automatic Display in Curve Editor Enabled to automatically display the curve of the selected pane in the graph display area.


Selected Pane Display in Curve Editor Available when automatic display is disabled. Click the button to display the curve of the selected pane in the graph display area.
O


Arrow Key Movement Speed Setting Sets the movement amount when the arrow key is pressed.
P

Adjust Time Scale

Click the button to open the Adjust Time Scale dialog box.
Scale the selected curve, which has Time on the horizontal axis and Value on the vertical axis.
