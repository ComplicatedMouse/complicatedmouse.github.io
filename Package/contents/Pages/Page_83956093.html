Command-Line Options

LayoutEditor has several command-line options for performing common tasks such as exporting layout files and saving screenshots of layouts.

Format

Use a format like the following.

LayoutEditor.exe [input file path] [option]

Options

Options Description
-o, --output Specifies the folder to export intermediate files to.
--close-if-loaded Closes the layouts loaded in the tool.
--parts-root-path Specifies the root parts folder.
--current-frame Specifies the LayoutEditor time frame value.
--save-screen-capture Saves a screenshot. Supports saving screenshots in BMP format (.bmp) and PNG format (.png).
--capture-size Specifies the size of the screen capture. Use comma delimiters, as in --capture-size=1280,720. Saves using the layout screen size when not specified.
--capture-file-path Specifies the file to which the screenshot will be output. If --all-frame is added to the options, the file is saved with a name that includes "_{tag name}_{number of frames from the start frame}" between the filename and the filename extension.
--animation-tag-name Specifies the animation segment tag name to evaluate during screen capture.
--all-frame Outputs all frames in sequential order. Valid only when screenshots are enabled. If an animation segment tag name has not been specified, all tags will be output.
--send-viewer Transfers the layout to the viewer. If no input file path is specified after this command-line argument and LayoutEditor is currently running, the currently selected layout is transferred.
--save-thumbnail Save the thumbnail file. It saves a file named Layout_filename.flyt.png in a .NwThumbs folder in the folder that contains the layout files. This is not included in the --current-frame setting.
--update-parts-layout Synchronize all layout files in the folder specified by the input file path. It is also possible to specify a different output folder with --output.

Example

Using the following format, you can specify a parts folder, load a file, specify the screenshot size, and then save a screenshot.

LayoutEditor.exe "D:\input.flyt" --parts-root-path="D:\parts_dir" --save-screen-capture --capture-file-path="D:\out\test.bmp"
