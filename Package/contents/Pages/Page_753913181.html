Profiler-Sample

Overview

Sets both the Sample Strategy and Sample Rate for Sampled profiles.

Profiler-Sample [-Strategy] <ScriptingSampleStrategy> [-Rate] <Integer> [-NoWarning]


[-Strategy] <ScriptingSampleStrategy>

These strings denote sampling strategies in a script. These values correspond with the strategies presented in the Sampling Strategy drop-down on the Sampled Profile tab.

    • Time

[-Rate] <Integer>

The rate at which to collect data when sampling. Only values in the Rate drop-down on the Sampled Profile tab are valid.

For the sample strategy Time, the value specified is the number of samples per 60 Hz frame.

    • 5
    • 10
    • 25
    • 50
    • 75
    • 100
    • 250
    • 500
    • 750
    • 1000
    • 2000
    • 4000


[-NoWarning]

This flag keeps a warning from being generated when an invalid Strategy and Rate are combined.

Returns

A Boolean value indicating whether the Strategy and Rate were set correctly.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Sample Time 1000
$success = Profiler-Sample -NoWarning -Strategy Time -Rate 100
