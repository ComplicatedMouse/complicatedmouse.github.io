3dShaderGraphConverter

This section describes how to use the 3dShaderGraphConverter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dShaderGraphConverter.exe).

Overview

This tool embeds source code generated from a shader graph (FCMB) file in a shader definition (FSDB) file.

Command-Line Options

3dShaderGraphConverter.exe [--version] [-h] [-s] [--args-file ""] [-o ""] [--shader-graph-source-path ""] [--material-search-path ""] [--shader-graph-search-path ""] [shader definition file]

--version
Displays version information.

-h --help
Displays Help.

-s --silent 　
Suppresses the display of messages during processing.

--args-file ""

Specifies the file.
You can use the contents in the parameter file to specify arguments.
The parameter file allows you to specify each option over several lines.
Lines starting with a number sign (#) are treated as comments.
Newline characters cannot be included in escape sequences used in option specifications that appear between quotation marks (").

-o "" --output ""
Specifies the output file path. If not specified, the input file will be overwritten.

Shader definitions file
Specify the shader definition file when making assignments.

--shader-graph-source-path ""
Specifies an FMDB intermediate model file or an FMTB intermediate material file that references the FCMB file from which the source code to embed will be generated. You can use semicolons as delimiters to specify multiple paths.

--material-search-path ""
Specifies the folder containing the parent material to search when resolving material references. The specified folder is searched recursively. You can use semicolons as delimiters to specify multiple paths.

--shader-graph-search-path ""
If model intermediate files (FMDB files) or material intermediate files (FMTD files) have been specified by the --shader-graph-source-path option, this option specifies the folder to search when resolving references to their shader graphs.
You can use semicolons as delimiters to specify multiple paths.

--sub-graph-search-path ""
If sub-graph intermediate files (FCSB files) have been specified in ShaderGraph, this option specifies the folder to search when resolving references to their sub-graphs.


Usage Example

3dShaderGraphConverter.exe input.fsdb --output output.fsdb --shader-graph-source-path "%TARGET_MODE_DIR%/input.fmdb" --material-search-path "%PARENT_MATERIAL_ROOT%"
