Managing Downloadable Content

Use DevMenuCommand, the target tool included in the SDK, to manage downloadable content.

DevMenuCommand allows you to use the following management features.

  • Installing downloadable content
  • Listing downloadable content
  • Uninstalling Downloadable Content

Installing

Installation assumes that there is downloadable content that has already been packaged according to Creating Downloadable Content.

You can install the downloadable content using the following command when connected to the development hardware.

# Install to NAND.
%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent install <AocPath>
 
# Install to SD card.
%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent install <AocPath> -s sdcard

The full path must be specified in AocPath.


The following output is obtained if installation succeeds.

Preparing to install D:\NintendoSDK\Samples\Outputs\NX-NXFP2-a64\Applications\AocSimple\Develop\SampleAoc.nsp...
5120 / 5120
[SUCCESS]



Enumeration

Use the following command to list downloadable content installed on the development hardware.

%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent list


If execution is successful, the output includes the following kinds of things, such as the application ID and index.

2 add-on-content(s) found.
id                  index  release-version  required-app-release-version  installed-storage
0x0100326002324000      1                0                             0            builtin
0x0100326002324000      2                0                             0             sdcard
[SUCCESS]



Uninstalling

Use the following command to uninstall the downloadable content installed on the development hardware.

# Deletes from NAND.
%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent uninstall <ApplicationId> [-i <index>]


# Deletes from the SD card.
%NINTENDO_SDK_ROOT%\Tools\CommandLineTools\RunOnTarget.exe %NINTENDO_SDK_ROOT%\TargetTools\NX-NXFP2-a64\DevMenuCommand\Release\DevMenuCommand.nsp -- addoncontent uninstall <ApplicationId> [-i <index>] -s sdcard

Specify the ID obtained from enumerating the downloadable content or the application ID specified in the metadata of the downloadable content as the ApplicationId.

For index, specify the index of the downloadable content you want to delete. If no index is specified, all stored downloadable content is deleted.

Note that installed storage must be properly specified for deletion to work.

The following output is obtained if uninstall succeeds.

[SUCCESS]
