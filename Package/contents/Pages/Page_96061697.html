Far Alpha

This section describes the Far Alpha feature that applies translucency to particles when they are in far distance of the camera.

Overview

The Far Alpha feature applies translucency to particles far away from the camera.
Unnecessary particles moving away from the camera can be naturally erased even when they are displayed on the screen.

The following overview describes the settings ().

Far Alpha

To enable or disable far alpha, select or clear the check box ().
Enable this setting to display the Far Alpha () settings.

The following overview describes the settings ().

Distance where Alpha is '1'

Sets the distance from the camera where the alpha value becomes 1.

Distance where Alpha is '0'

Sets the distance from the camera where the alpha value becomes 0.
