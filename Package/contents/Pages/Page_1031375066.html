Align Pane Describes the align pane, which is used for aligning and adjusting the positions and sizes of panes in the Hierarchy window.
I Can't Animate a Translate Animation for a Pane Below the Align Pane

Even though you have set an animation that moves in the direction of the alignment for the pane below the align pane, the pane does not move because the position is overwritten by the alignment process.

This page uses a dummy picture pane to describe how to avoid this issue.

Added I Can't Animate a Translate Animation for a Pane Below the Align Pane to Troubleshooting.

Added the cause and workaround for when an animation that is supposed to make a pane fade out off the screen does not work.

Uses a sample to describe how to use the capture pane, which captures the rendering results of the frame buffer or layout, and converts it into an image.
