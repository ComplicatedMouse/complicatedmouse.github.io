Bank Tab

An overview of banks and a description of the Bank tab are presented.

Overview

Role of the Bank Tab

Use the Bank tab to edit bank files (FBNK files).
Bank files are referenced from the bank of a sound set. They are used by sequence sounds during playback.

Structure of a Bank File

A bank file consists of instruments that corresponds to musical instruments and the instrument list that groups those instruments and assigns program numbers.
Detailed descriptions of the list elements are provided in later sections.

Tab Labels for Bank Tabs

Normally, for bank files belonging to projects, the tab label includes the bank name and the filename separated by a hyphen: bank name - file name. In some cases, such as when a bank file that is not registered in a project was loaded using the File menu, only the filename is displayed.
