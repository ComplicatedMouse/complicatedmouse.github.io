Repeat Count / Offset Per Repeat

This section describes the feature for repeatedly displaying particles inside the box.

Overview

Repeat Count is a feature for repeatedly rendering the emitted particles.
The Offset Per Repeat feature is for slightly shifting the display position of the particle each time it is rendered by adding an offset after it has been rendered.
You can avoid repetitiveness in the pattern by setting the offsets to sufficiently irregular values.
This section provides an overview of the settings ().

Repeat Count

The particles are rendered repeatedly inside the box for the specified number of times.

Offset Per Repeat

An offset is applied to the copied particle position.
Each particle is rendered at a new location by adding an offset every time it is drawn.
You can avoid repetitiveness in the pattern by setting the offsets to sufficiently irregular values.

Info

If a single emitter is drawn multiple times, you only need to run the calculation processes for the emitter and particles once.
This lessens the load on the CPU compared to using multiple emitters. However, the rendering cost itself is incurred multiple times.
