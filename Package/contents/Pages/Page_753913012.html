Profiler-Connect

Overview

Establishes an initial connection to the dev kit without performing the full Sync operation. This allows for obtaining information from the dev kit before a Sync has been completed. This step does not need to performed manually as the Profiler-Sync operation will automatically do this if it has not already been done. It is recommend to only use Profiler-Connect if required, otherwise continue to use Profiler-Sync.

Profiler-Connect [-Target <Identifier>]


[-Target <Identifier>]

Specifies a target dev kit to connect to if there is more than one valid target. The identifier should be a way to uniquely identify a kit.

    • IP Address
    • Serial Number
    • Target Name

Returns

A Boolean value indicating if the connection was successfully established or not.

Examples

Basic Usage

Provided here are some basic usage examples.

$connected = Profiler-Connect
$connected = Profiler-Connect -Target "123.4.56.78"
