Displaying Page and Group Names in a Tree

The tree view on the Material Property window shows the page names set in the shader definition file (FSD file) and the group names of grouped parameters.

Info

For more information about shader definition files (FSD files), see the Graphics Intermediate File Specifications, provided separately.

  1. Page Name
    If the shader definition file (FSD file) includes any <page_array> elements, page names are displayed under the shader.
    A predefined page titled Vertex Attributes is listed first under Shader. This page provides information about vertex attributes and items in <group> and <page_array> elements in the shader definition file (FSD file) with an empty page_name field.
  2. Group name
    If the shader definition file (FSD file) does not include any <page_array> elements, group names are displayed under the shader.
    If a corresponding <page_array> element has been defined, group names are displayed in the tree for that page.

You can use the shortcut menu for pages and groups to configure the following settings.

  • Copying or pasting the group custom label, or reverting it to the default value
  • Selecting Showing or Hiding in the Material List
  • Editing Custom Labels
  • Setting Value References and Child Material Restrictions

Material Configuration File Export Dialog Box

Performs operations relating to the export of the material configuration file (F3MT file).

A

Selects the default applicable items for importing.
Selected items are applied as defaults at time of import.

Assigns items to apply to the Sampler and Shader groups separately.

B

Displays the settings for items to be exported.
C

Expands and displays the settings of all parameters to be applied.
D

Collapses the expanded display of all settings.
E

Adds comments to the material configuration file to be exported.
F

Exports.
G Cancels the export.

Dialog Box for Importing the Material Settings File

Performs operations relating to the import of the material settings file (F3MT file).

A

Selects which material to overwrite during import.
You can mark multiple check boxes to select multiple materials to overwrite during import.
B

The models that use the materials to overwrite during import.
C

Select check boxes to select the items to apply when importing.
Default settings that are set in the file export dialog box are applied as default.
Note: Assigns items to apply to the Sampler and Shader groups separately.

D

Displays the settings for items to be imported.
E

Expands and displays the settings of all parameters to be applied.
F

Collapses the expanded display of all settings.
G

Shows comments included in the material configuration file (F3MT file).
H

Starts importing.
I

Cancels importing.


When Animation Files With the Same Name Are Bound

  • If an animation with the same name is bound to a model in the material to overwrite during import, that animation is edited.
  • If an animation with the same name is not bound to a model in the material to overwrite during import, that animation is created anew.
    However, if the animation with the same name has an indefinite file path, the animation is edited.

Note About Sampler and Shader Applicable Items

Samplers and shader groups can be individually assigned as applicable items in dialog boxes.

Sampler

You can use the shortcut menu to select either Replace or Add as the method of assigning the sampler items to apply.

Shaders

For Shader, you can select the shader settings and what items to apply when importing each group.

Info

When using shader settings, a shader definition and shading model are assigned.
Parameters not belonging to a group are grouped together as "other."
