Scaling Overall Curve Time

This page describes how to use the Adjust Time Scale feature to apply scaling to an overall curve.

Adjusting the Time Scale

This is the procedure for scaling a created curve.

1

In the Curve Editor toolbar, select Adjust Time Scale to open the Adjust Time Scale dialog box.
2

Specify which curves you want to scale.

Two or more keys must be selected for each curve to adjust the time scale.
3

Specify which scaling method to use.

  • Set new start and end points: Specify a new start point and end point for the curve with the adjusted time scale.
  • Set origin and scale: Select Value or Frame as the target and directly specify a scaling factor.
4

Click the Apply button to close the dialog box.

Check the graph display area to confirm that the configured scaling was applied to the curve.
Info

The Set new start and end points scaling method works as follows.

  • If there is no key frame before frame 0 of the selected curve, frame 0 is treated as the start point for scaling.
  • If there is no key frame after frame 0 of the selected curve, frame 0 is treated as the end point for scaling.
  • The time is scaled to the maximum width of the selected curve.
