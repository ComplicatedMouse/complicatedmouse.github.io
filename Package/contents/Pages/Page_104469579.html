Primitive Files

Primitive files are model files for primitives used in emitter set files.
This page describes how to use primitive files in EffectMaker.

Where to Save Primitive Files

When primitives are used for the particle shape, emitter set files (ESET files) store the names of model files but do not store path information.
As a result, when opening an emitter set file, EffectMaker searches for a model file with the same name as the one stored in the emitter set file.

Save model files in a searchable folder on the same drive as the associated emitter set file.

The following image shows the search order and the searchable region of folders for EffectMaker, where EmitterSet is the emitter set file (ESET file) to be opened and Primitive is the model file corresponding to the name stored in the emitter set file.

  1. When the EmitterSet.eset emitter set file is opened, EffectMaker first searches for model files that are in the same folder.
  2. EffectMaker then searches in the Primitives folder directly below the folder that the emitter set file is in.
  3. EffectMaker then searches in the Primitives folder above the folder that the emitter set file is in until it finds a model file.
  4. Finally, EffectMaker searches in the highest level Primitives folder on the drive. This folder is the last folder that can be searched.
    If no model file has been found by this point, EffectMaker displays a warning.
Info

EffectMaker makes no distinction between uppercase and lowercase characters in the "Primitives" folder name.
