Rendering Features

This section describes the rendering-related features provided by the Face Library.

Overview

These are main features for rendering Mii characters.

  • Creation of a Mii face model
    Creates CharModel data based on CharInfo to render the Mii face model.
  • Creation of a face outline (faceline)
    Creates Faceline data based on CharInfo to create a face outline texture (a faceline texture) for the Mii character.
  • Creation of a mask
    Creates Mask data based on CharInfo to create a mask texture for the Mii character.
  • Creation of a body model
    Creates IconBody data for rendering the body model of the Mii character.
  • Shader for creating face outline and mask textures
    The library provides a shader (the TextureShader class) for creating face outline textures and mask textures. You can also create these textures using your own shader instead of the one provided by the Face Library.
  • Shaders for creating Mii icon textures.
    The library provides shaders (IconShader and VariableIconBodyShader) for creating the icon textures for Mii characters.
