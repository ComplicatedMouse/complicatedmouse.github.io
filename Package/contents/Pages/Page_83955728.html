Project Tree

This section provides an overview of sound projects and a description of project trees.

Overview

Role of the Project Tree

The project tree is where sound sets are registered to sound projects.

Select nodes in the sound set to display a list of them in the tab pane.

You can archive multiple sound sets by adding them to a folder.

Structure of Sound Projects

You can register more than one sound set with a sound project.
You can also register bank files, waveform files, and standard MIDI files in the sound sets for use as sound resources.

You can use the Bank tab to open and edit bank files that have not yet been registered in the sound project.
There are two kinds of bank files in SoundMaker: (1) registered bank files that are subject to conversion, and (2) bank files that open but are not unregistered and are not converted.
Within this documentation, the word project refers to the former type; that is, those files that are subject to conversion. Likewise, the term workspace refers to all files that are displayed within SoundMaker, including any unregistered files that may be open.

For more information about how to load bank files that are not registered in a project, see Loading Bank Files from Outside the Project.


Key Operations

Project Tree Keyboard Operations

In addition to the menu Display > Project Tree and the toolbar Project Tree Display button, you can also use the ALT+1 shortcut key to enable or disable the project tree display option.

You can perform the following keyboard operations inside the project tree.

  • UP/DOWN ARROW keys — Move the selected node up or down.
  • LEFT ARROW — Close node. If the node is already closed, move to the parent node.
  • RIGHT ARROW — Open node. If the node is already open, move to the first child node.
  • BACKSPACE — Keep the node open and move to the parent node.
  • PAGE UP/DOWN — Scroll one page up or down.
  • HOME/END — Move to the top or bottom of the tree.
  • ENTER — The same operation as a double-click.
  • DELETE — Delete the selected item.
  • SHIFT + Mouse wheel — Scroll left and right.
  • Application key — Show the shortcut menu (this key is not present on all keyboards).

Tree View

Synchronize With Tabs

Select View > Synchronize Trees to Tabs to synchronize the node selection in the project tree with the tabs.

When the Sound Set Tab Is Active

Selects the node in the project tree that corresponds to the list that is currently displayed on the active Sound Set tab.
After this operation, the focus moves to the tree.
If a corresponding tree node happens to be closed or located where not shown in the project tree, the node is opened, or the display scrolls to show the node on the screen.
Nothing happens if the corresponding tree node cannot be identified (that is, no tabs are open).

When the Bank Tab Is Active

Selects the node in the project tree that corresponds to the location where the active Bank tab is opened.
Nothing happens if the Bank tab is open, but the original bank item has been deleted from the sound set.
If the tab is reopened from a bank that references the same bank file, the project tree view synchronizes with the tab opened last. (Synchronizes to the most recently accessed bank node.)
Other operations are the same as the Sound Set tab.

View Filters

Filtering by Unused Nodes

Child nodes in the sound set that are not being used can be hidden in the project tree.

To hide the display of child nodes, right-click a sound set node in the project tree to open the shortcut menu, and then select Hide Empty Nodes. Select this option again to show the nodes.

You can also apply the Hide Empty Nodes setting for all sound sets by selecting Hide All Empty Nodes from the shortcut menu used for project nodes.

If an item is added to an empty node while the hide setting is enabled, the node tree display is immediately restored.
However, if the parent node of this node is closed or positioned off the screen, it does not automatically open the parent node or scroll to show it on the screen.
The show/hide setting is saved for each sound set in the settings file for the project.

Filtering by Character Input

You can filter to display specific sound set nodes when you enter text in the text boxes that appear above the Project Tree.
Nodes will appear for the file when that entered text includes at least a part of the name for the sound sets, wave sound sets, sequence sound sets, or sound set banks that appear as nodes in the Project Tree.

Close All Nodes

To close and collapse all nodes in a project tree, select Collapse All from the shortcut menu after right-clicking in a project tree.

The default setting is to expand all nodes in a project tree. This setting can become cumbersome when the tree has a large number of sound sets.
In that case, close all nodes with Collapse All, and then open only the necessary sound sets to perform operations.

Minimize View

Click Minimize on the right side of the title bar of the project tree to minimize the display and show only the tab portion of the project tree.

To restore the project tree to its original size, click the button on the tab or double-click the tab.

Using Folders to Archive Sound Sets

To create a folder in the project tree, you can right-click a project in the project tree, and from the shortcut menu, select Add Folder.

You can visibly manage multiple sound sets by dragging sound sets to a folder.

You can also drag folders to other folders.
