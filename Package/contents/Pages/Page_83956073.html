Vertex State

Vertex State

The vertex state collectively represents several states of the vertex fetch before the vertex shader executes.

Information

Use nn::gfx::VertexStateInfo to define the vertex state information.

Vertex Attribute States

Set the array of vertex attribute states using the SetVertexAttributeStateInfoArray function.

You must code as many vertex attribute states as the number of vertex attributes you want to use.

Semantic Index

Set the semantic index for the element using the SetSemanticIndex function.

Slot Number

Set the slot number to assign in the shader using the SetShaderSlot function.

Buffer Index

Set the index of the vertex buffer to fetch using the SetBufferIndex function.

Offset

Set the offset from the start of the vertex buffer using the SetOffset function.

Format

Set the format of the vertex attribute using the SetFormat function.

Symbol Name

Set the name of the vertex attribute to assign in the shader using the SetNamePtr function.
If NULL is set for this parameter, the value of the Slot Number parameter is used.
The value referred to by the pointer can be freed after the object has been initialized.

Vertex Buffer State

Set the array of vertex buffer states using the SetVertexBufferStateInfoArray function.

You must code as many vertex buffer states as the number of vertex attributes you want to use.
By interleaving vertex attributes, there can be fewer states than there are vertex attributes.

Stride

Set the stride of the buffer using the SetStride function.

Instance Divisor

Set the instance divisor, which defines how many elements to advance for each instance, using the SetDivisor function.
Set 0 to handle data in units of vertices instead of instances.
The specified value is applied to all vertex attributes that reference this buffer.

Object

Use nn::gfx::VertexState to define the vertex state object.

Initialization and Destruction

The vertex state is initialized and finalized according to the rules for objects.

Setting the Vertex State

If you are using a pipeline execution model, the vertex state information is included in the pipeline information.

If you are using an execution model that does not use a pipeline, you set the vertex state by calling the SetVertexState function of the nn::gfx::CommandBuffer class.
