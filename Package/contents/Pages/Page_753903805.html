Supported Versions of MSVC v142

The following table provides the versions of MSVC v142 supported by the various NintendoSDK versions. Use the specified MSVC v142 version.

Earlier than 12.3.0 (Not supported) N/A

Using the Specified Version of MSVC v142

This section describes how to use Visual Studio 2019 to build the Visual C++ project to use the specified version of MSVC v142.

Installing the Specified Version of MSVC v142

Use the Visual Studio 2019 installer to install the components indicated by the Component IDs in the following table.

If you use the Visual Studio Installer, you can install MSVC v142 by going to the Individual components tab and selecting components.

For more information about Visual Studio Installer, see https://docs.microsoft.com/en-us/visualstudio/install/install-visual-studio?view=vs-2019.

Also see the Visual Studio Installer sample program, which uses Visual Studio Installer to prepare the required environment. (SDK 14.0.0 and later)

SDK Version MSVC v142 Version VCToolsVersion Component ID Component Name
12.3.0 and later, but earlier than 13.1.0 v14.27 14.27.29110 Microsoft.VisualStudio.Component.VC.14.27.x86.x64 MSVC v142 - VS 2019 C++ x64/x86 build tools (v14.27)
13.1.0 and later, but earlier than 14.1.0 v14.28-16.9 14.28.29910 Microsoft.VisualStudio.Component.VC.14.28.16.9.x86.x64 MSVC v142 - VS 2019 C++ x64/x86 build tools (v14.28-16.9)
14.1.0 and later v14.29-16.11 14.29.30133 Microsoft.VisualStudio.Component.VC.Tools.x86.x64 MSVC v142 - VS 2019 C++ x64/x86 build tools (Latest version)
Info

There is no Microsoft.VisualStudio.Component.VC.14.28.16.9.x86.x64 component in Visual Studio 2019 16.9.x. v14.28-16.9 will be installed as the latest MSVC version.

Setting the Project's MSVC Toolset Version

To specify the version of the MSVC compiler to use, you must set the value of the project's MSVC toolset version.

Open the properties for the project, go to Configuration Properties > Advanced > MSVC Toolset Version, and set that value to the VCToolsVersion indicated earlier.

Using MSVC v141 (Versions Earlier Than SDK 12.3.0)

In versions earlier than SDK 12.3.0, NintendoSDK does not support builds using MSVC v142.
If you use Visual Studio 2019 as your development environment with versions earlier than SDK 12.3.0, you must set the MSVC used for the build to MSVC v141.

For more information, see Using MSVC v141 With Visual Studio 2019.

Warning

Visual Studio 2019 can use the following with NintendoSDK 9.5.0, or NintendoSDK 10.2.0 and later.

  • VSI
  • VsHelp
  • Wizard
  • NintendoSDKVsExtension
