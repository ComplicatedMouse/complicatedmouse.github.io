Creating Bitmap Fonts

This section describes the procedure for creating an original bitmap font like the following using the GUI version of FontConverter.

This tutorial uses the following procedure to create a font.

1 Preparing to Create the Font Start the GUI version of FontConverter and prepare an image that will serve as the template for font creation.
2 Creating the Font Using an Image Editor Load the template image into your image-editing software and use it as a base for creating an original font.
3 Outputting the Font File From FontConverter Load the newly created original font into the GUI version of FontConverter, convert the data into a font file, and then output that file.

1. Preparing to Create the Font

Info

It is easy to use the GUI version of FontConverter to create the data for the bitmap font, but it is not suited for creating the final data as a batch process.

Ultimately, it is more convenient to create the data in FFNT format and use the console version of FontConverter rather than the GUI version to create the bitmap font data, because you can then manage the data in a platform-independent way.

For more information about the console version of FontConverter, see FontConverter (Console Version). For more information about creating FFNT files, see Bitmap Fonts Not Dependent on the Target Platform.


Start the GUI version of FontConverter, and prepare a template image.

Related Links For more information about starting the GUI version of FontConverter, see the workflow.

Selecting the Windows Font Tab

In Input, select the Windows Font tab.
Configure the following properties.

Font Name Size Description
MS Gothic
 		16 px You can select any font that has been installed to the Windows PC.
This tutorial uses MS Gothic in a size of 16 pixels as the input font.

Selecting the Image Tab

In Output, select the Image tab.
To open the dialog box and specify the name and output location of the file, click Select File ().
In this tutorial, name the file Template.bmp.

The font selected in Input can be output as an image file.
You load the image file that gets output as a template in your image editor and use it as the basis to create an original font.

Converting to a Template Image

To output the file, click Convert ().

2. Creating the Font Using an Image Editor

Open the template image in your image editing software. This tutorial uses Photoshop as its image editor.
As depicted, the template image becomes the lower layer, and that layer is used as a base for creating an original font in a different layer.

Related Links When working in Photoshop, it can be useful to separate the character font data in the template image into separate layers for the glyphs and grid, for example.
For tips about how to separate the data into different layers, see Separating Layers for Creating Fonts.

3. Outputting the Font File From FontConverter

This step uses the GUI version of FontConverter.
Load the newly created original font (IconFont_16.tga), convert the data to a font file, and then output that file.

Selecting the Image Tab

In Input, select the Image tab.
Select the image file (IconFont_16.tga) created using the image editor.

Converting the Color Format

This step changes the preset of the color format in the Image tab settings to LA8( –> BC5 ).

Related Links For more information, see the following pages.

Color Format Describes the frequently asked questions about selecting the color format.
Selecting the color format. Describes the image color formats that bitmap fonts can use.

Selecting the bffnt / bffna Tab

In Output, select the bffnt / bffna tab.
To open the dialog box and set the name and output location for the file, click Output File ().
In this tutorial, name this file IconFont_16.bffnt.

You can output the image file that was selected for Input as a font file usable by LayoutEditor.

Converting to a Font File

To output a font file, click Convert ().
