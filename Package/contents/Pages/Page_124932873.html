Target Platform

Describes the platforms that support FontConverter.

What Is the Target Platform?

If a target platform has been specified, the bitmap font binary file (BFFNT file) is output in a form optimized for that platform.
The target platform is specified by the preferences file.

Preferences File

A folder for each platform is created in Tools\Graphics\FontConverter\Preferences\ and the preference.xml file is placed in there.

─ Preferences ┬ Generic
               └ PlatformAAA

If there are multiple platform folders, Generic is ignored and the first other discovered folder is used.
If you do not want to specify a target platform, delete or rename the preference.xml file in all folders other than the Generic folder.
