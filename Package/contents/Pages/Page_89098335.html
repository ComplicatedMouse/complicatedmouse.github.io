Introduction

Purpose of This Documentation

This documentation describes the audio features included in NintendoSDK.

Intended Audience

This documentation is targeted at developers who write audio-related code using NintendoSDK.

Related Documentation

Along with this documentation, also see the Audio API Reference.

audio Library Features

The audio library provides features to input and output sound, to input to and output from devices handling sound data, and to process sounds (rendering).

How the audio Library Is Provided

See the following pages.

  • How the Libraries Are Provided and How to Use Them

Features Provided by the audio Library

The audio library provides the following features.

  • Audio Output
  • Audio Input
  • Rendering

For more information about these features, see the manual, the API reference, and the sample program.
