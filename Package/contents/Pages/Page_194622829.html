Attribute Tree

The page describes the features displayed in the Attribute Tree in the Curve Editor window.

The Attribute Tree displays the pane elements to be animated.
Animations for the elements selected here are created and edited in the graph display area.

Attribute Tree

A



Display All or No Filter Displays all items without filtering.
Basic items only Displays only the basic settings of position, size, scale, rotation, transparency, and show or hide.
Key items only

Displays only the key items.


  • When using separated mode, you can specify the search range from Filter > Settings for Key items only.
Custom Filter

Displays only the items you select.

You can customize an item filter by selecting Add New from the list on the right.
B Attribute Tree

Displays the elements being animated in the pane.

Items displayed other than the basic settings differ depending on the type of pane.
C Segment Name

When using separated mode, the tag names of segments to which animation keys have been set are displayed at the top of the tree.

Click the segment tag name to switch the selected segment.


  • These names can be shown or hidden from View > Show Names of Segments with Keys.

Attribute Tree

The following operations can be performed by right-clicking the attribute tree in the Curve Editor.

Mute

Mutes the animation.
Attaches a mute symbol () to the disabled animation in the Curve Editor.

Muted animations are not reflected in the game data.
Unmute Unmutes the animation.
Lock

Locks the animation.
Attaches a lock symbol () to the locked animation in the attribute tree.

Locking helps prevent unintended editing of animation curves.
Unlock Removes the lock.


Info

A muted or locked animation curve cannot be edited.


Creating Filters

If you press the the Custom Filter button () and click the text box on the right, a list appears.
From the list, select Add New open the Create New Filter dialog box.

Create New Filter Window
A Filter Name

Displays the list by the name input here.

If no filter name is entered, the OK button is not enabled.
B Select All Use to select (or clear) check boxes for all items.
C Check Selected Curves Use to select (or clear) check boxes for the selected curve.
D Selected Curve Attribute Tree Displays the items selected here in the attribute tree.
