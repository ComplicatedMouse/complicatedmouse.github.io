Command Line

This section describes the command-line arguments when SoundMaker is started from the command line.

Starting From the Command Line

You can execute actions to start a new instance of SoundMaker or to change the state of an instance of SoundMaker that is already running.

Specifying a Sound Project File

To open a sound project file when SoundMaker is started, specify a sound project file as the file path.

SoundMaker.exe file path (FSPJ file)

Saving a Project in a Running Instance of SoundMaker

You can save a project in an instance of SoundMaker that is already running,

SoundMaker.exe file path(fspj) --command "saveproject"

Selectively Displaying Items for a Running Instance of SoundMaker

By starting a new instance of SoundMaker from the command line with the option that specifies an item name, you can make that item active (that is, the item is selected in the UI shown to the user) in an already running instance of SoundMaker.

Start from the command line, using the following method.

SoundMaker.exe file path (fspj) --command "showitem=item name"

Example:

SoundMaker.exe C:\NintendoSDK\Samples\Sources\Resources\Atk\Common\Common.fspj --command "showitem=SEQ_COIN"

If SoundMaker is not running or was started with a different specified project, running this command from the command line starts a new instance of SoundMaker and displays the specified item.

Displaying Bank Files for a Running Instance of SoundMaker

By starting a new instance of SoundMaker from the command line with the option that specifies a bank name, you can make that bank active (that is, the Bank tab is viewable to the user) in an already running instance of SoundMaker.

Start from the command line, using the following method.

SoundMaker.exe file path (fspj) --command "openitem=bank name"

Example:

SoundMaker.exe C:\NintendoSDK\Samples\Sources\Resources\Atk\Common\Common.fspj --command "openitem=BANK_BGM"

If SoundMaker is not running or was started with a different specified project, running this command from the command line starts a new instance of SoundMaker and displays the specified bank.

Starting and Stopping Previews

You can preview sound files and end all current playing sounds using the following command line options.

Specify the following from the command line.

  • Playback: SoundMaker.exe file path (fspj) --command "playitem=item name"
  • Stop all: SoundMaker.exe file path (fspj) --command "stopallitem"

If executed while SoundMaker is not running or with a different project specified, a new SoundMaker instance starts and plays the specified item.

Specifying User Parameter Values

You can specify user parameter values when starting SoundMaker from the command line.

Start from the command line, using the following method.

SoundMaker.exe file path (fspj) --command "edititem=item name, user parameter name, user parameter value"

Item Name:

Specify the names, such as STRM_MARIOKART or SE_YOSHI.

User Parameter Name:

  • userparam
  • userparam1
  • userparam2
  • userparam3

Specify one of these values.

User Parameter Value:

Specify a number in decimal format.

Example: Setting the value of user parameter 1 for the STRM_MARIOKART item name to 255.

SoundMaker.exe C:\NintendoSDK\Samples\Sources\Resources\Atk\Common\Common.fspj --command "edititem=STRM_MARIOKART,userparam1,255"

If SoundMaker is not running or was started with a different specified project, running this command from the command line starts a new instance of SoundMaker and sets the user parameter.

Outputting Sound Items To Waveform Files

You can use the command line to specify a sound item to output to a waveform file.

SoundMaker.exe file_path (fspj) -o, --output <output-folder> 　--command "outputwavefile=item_name[, options...]"

-o, --output : Specifies where to output the file.

  • output-folder

Specifies where to output the file.
This option is enabled when the outputwavefile command is specified.

--command : Specifies various commands.

  • "outputwavefile=item_name[, options...]"

Item Name:

Makes the item a waveform file.

Optional:

"time=seconds"

Specifies the maximum output time (in seconds). [Default: 30]

"channel=output_channel"

Specifies the output channel. [Default: stereo]

　"mono" : Monaural output.
　"stereo" : Stereo output.

"openfolder=boolean value"

Opens the folder where the waveform file is output. [Default: true]

　"true" : Open the output folder.
　"false" : Do not open the output folder.

Example:

SoundMaker.exe C:\NintendoSDK\Samples\Sources\Resources\Atk\Common\Common.fspj --command "outputwavefile=SE_YOSHI,time=5,channel=mono"
