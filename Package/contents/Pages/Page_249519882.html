Using Separated Mode With Animations

This section describes how to use separated mode with animations.

Merged Mode and Separated Mode

Merged mode is the conventional way of working with animations, using a number of segment tags as demarcations in a single animation.
Separated mode is a feature that creates a 1:1 association between animations and segment tags.
By switching to separated mode, you can create a separate animation for each segment tag.
