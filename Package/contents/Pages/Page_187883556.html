New Effect Expressions

This section introduces features that broaden the scope of what can be depicted with effects.

Feature to Control Mipmap Levels

Restrict Mipmap Levels is a feature to control the mipmap levels.
This feature enables you to avoid referencing mipmaps with resolution that is too low.

The following images compare when mipmaps are set and not set for textures added to a plane particle.
Rotate the particle on the y-axis to change the horizontal display area for textures on the screen.

A No Mipmap Texture appears as is. Compare to B, With Mipmaps.
B With Mipmaps Mipmap levels at low resolutions are referenced as the area gets smaller.
The ← indicates places where referencing a resolution that is too low will distort the vertical appearance.
This phenomenon is not a desirable appearance for game applications.

The following image compares the enabling and disabling of Restrict Mipmap Levels.
The issue is controlled because the referencing of mipmap levels that cause the above phenomenon, does not occur.

Applied Mipmap Levels
ON Applied mipmap levels are restricted to levels 0 through 2.
The displayed mipmap level remains at 2 even when the display area decreases.
OFF

The default setting. Corresponding mipmap levels appear as the display area gets smaller.

Info

The following features were removed when this feature was added.

  • Effective Mip level, a feature to specify the maximum mipmap level at the emitter level.
  • Mipmap Bias Value, a feature that can shift mipmap levels in both the positive and negative directions, at the emitter level.
