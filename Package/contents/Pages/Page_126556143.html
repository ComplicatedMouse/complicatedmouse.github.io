How do I make outline fonts?

Question

How do I make outline fonts (fonts designed to show a border)?

Answer

Create outline fonts using the following method.

  • Create a font that can be input as a bitmap image. (The color channel for this image is used for the border design.)
  • Create a font in a format like LA4 or LA8 that has separate values for brightness and alpha.
    We recommend using the LA4Packed format (described in the answer to the next question) to create outline fonts that are small and flexible to use.
Info

There is no feature to automatically render the outlines of bitmap fonts, so the bitmap image must be processed directly to render the outlines.

LA4Packed Format

The LA4Packed format is suitable for use with outline fonts.
This font format stores brightness and alpha values in 8 bits, cutting memory size in half in comparison with the normal format (2-channel BC5).

Outlines Can Be Enabled or Disabled

Font outlines can be enabled and disabled, allowing you to use both an outlined and no-outline version of the font.
Previously, you had to have two entirely separate fonts to achieve this result. The new feature for enabling or disabling font borders reduces the memory required by one third (A8+LA8 vs. LA4Packed).

About the Source Image


For the convenience of performing special interpretation internally, use black fonts - white background as the source image for the fonts.
For images not like the example, the gradation of the glyph boundary is not suitable because pixels that do not have alpha values of 255 are considered to be white (255, 255, 255).
