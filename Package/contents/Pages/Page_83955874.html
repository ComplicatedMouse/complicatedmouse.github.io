Plug-In for Setting the Camera Angle of View

Configures the camera's angle of view in the vertical direction (Fovy).
The angle of view displayed in the Camera Attribute Editor is for the value in the horizontal direction, and differs from the value used by NintendoWare for the angle of view.

Use this plug-in if you want the angle of view as set with 3DEditor to be applied on Maya.

Usage

Select Nintendo > Set Camera Fovy to open the window.

The transform nodes of the selected camera are displayed in the Selected Camera column.

Info

If four or more nodes are selected, names are not displayed and only the number of nodes selected is displayed on the right edge.

Setting Method

When you select one or more camera nodes or transform nodes for the camera, the current settings are displayed in the Set field.
If multiple selections were made, the setting for the first selected camera is displayed.

Set Fovy by entering a numeric value or moving the slider.
You can make multiple selections and change all cameras at the same time.

A different camera can be set by selecting another node without closing the window.

Fovy is affected by the Vertical Film Aperture attribute of the camera node. Before configuring Fovy, make sure to configure the Vertical Film Aperture attribute.
To animate Fovy, set a value with this plug-in, then enter a key to the Focal Length attribute in the camera node.
