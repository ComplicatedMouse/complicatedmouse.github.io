Particle Sort

This section describes the feature for determining the order in which emitted particles are sorted.

Overview

Particle Sort determines the order in which emitted particles are sorted.
Changing the sort order changes the order in which emitted particles are rendered.
The following overview describes the settings ().

Particle Sort

Selects the particle rendering order from among the following presets.

Do not sort
Sorting is not performed, so this setting has the lowest load.
Use this preset for effects that display without issue, even when the rendering order is incorrect.
Ascending order Sorts so that particles that are emitted later are rendered closer.
Descending order Sorts so that particles that are emitted first are rendered closer.
Z-Sort Renders particles from farthest to nearest based on their distance from the camera.
Info

For GPU emitters, z-sort is performed based on the positions where particles are emitted.
Related Links For more information about GPU emitters, see Effect Emitters.
