tool

Using DirectXShaderCompiler to compile source code larger than 64 KiB causes an internal compiler error.

Symptoms

Using DirectXShaderCompiler to compile source code larger than 64 KiB causes the following message to be displayed.

Internal compiler error: access violation. Attempted to read from address 0x[address]

Workaround

Divide up the source file and use #include so each chunk is less than 64 KiB.

Fix Schedule

This bug will be fixed in a future version.
