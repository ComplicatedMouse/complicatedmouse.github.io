Performance Guide

This page describes how to gain optimal performance in the NintendoWare effects development environment.
Implementing some of these methods requires a trade-off with what can be depicted with effects.

Reducing CPU Load

The following describes how to decrease the CPU computational load.

Optimizing CPU Processing This section describes how to optimize CPU processing.
Reducing the Number of Emitters Using Emitter Shape Describes the use of the Circle (evenly divided) emitter shape to emit from multiple locations using one emitter.
Culling This section uses the culling callback function to show how to cull the calculation and rendering processes for off-screen emitters.

Reducing the Rendering Load

The following describes how to decrease the GPU rendering load.

Rendering Textures Without Waste Describes how to reduce wasteful volume by improving the way you render textures.
Using Primitives Describes ways to save rendering volume by using primitives as a particle shape.
Curbing Particle Overlap Describes how to adjust particle emission so there is little overlapping of particles.
Implementing Refraction Over an Appropriate Area

Describes ways to reduce rendering load by adjusting how particles overlap when using Color Buffer Refraction.
Taking Advantage of Glare and Bloom Describes ways to reduce load by using glare effects for luminescent effects.
Near Alpha and Camera Distance Scale Restriction (Near) Describes how to use the Near Alpha and Limit Scaling in Front of the Camera features to reduce the rendering load.
Reducing Texture Size Describes how to reduce the cost of loading textures by reducing the texture size.
