Audio Loudness Meter Panel

The audio loudness meter panel displays the loudness calculated for waveforms output from the audio library.

UI Reference


The NintendoSDK 1.0.0 screen.


 Item Description
A Show/Hide

Select the items to display in the graph and meter.
B Graph

Displays graphs of the average loudness, peak loudness, and RMS value.


When you only display one graph of the average loudness value or peak value, the target value appears in color in the graph's background.


The calculated True Peak value appears on the peak values graph when True Peak is enabled.
C Meter

Displays the average loudness, peak loudness for each channel, and RMS value at the current time.

Colored target values are displayed above and below the current average loudness value and peak value.


Values for the current time appear below. The peak hold appears above.


You can choose whether to display a peak value or the RMS value for a channel.

Calculated true peak values appear in the peak values when True Peak is enabled.
D Current value Displays the current average loudness in an easily visible larger font size.
Colored target values are displayed above and below the current average loudness value.

When configuring the volume for each output device using the nn::audio function, the output volume to the TV is considered.

It is possible that the waveform that is ultimately output will be mixed down at some later stage, but Spy records the waveform before mixing down.

For more information about target loudness values, see the Sound Production Guide for NX applications.

How to Operate

Operation Target Operations Procedure
Show/Hide Click

Selects the items to display in the graph and meter.


You can select either peak or RMS values for the meter. Select 2ch, 6ch, or off.
Graph Click Moves the current time to the specified position.
Right-click + drag Scrolls the graph horizontally.
CTRL+mouse wheel Horizontally zooms the graph in and out.
