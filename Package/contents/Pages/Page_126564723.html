Workspace File

Describes the workspace file, which records work in the workspace.

Opening Workspace Files

When you open another workspace file, a dialog box appears and prompts you to select whether to delete all the files you are currently editing.
If you choose to delete your current files, all of the files in the workspace file are opened.

When a workspace file is loaded, a message is displayed if any of the files associated with the workspace were not found.
Files that cannot be found are excluded from the workspace.

When a workspace file is open, you cannot have other workspace files or any other files open at the same time in EffectMaker.

Note

When a workspace file is open, you cannot have other workspace files or any other files open at the same time in EffectMaker.

Saving Workspace Files

You can save your work in EffectMaker as a workspace file (WORK).
When you open a saved workspace file, you effectively restore the workspace to the state it was in when the workspace file was saved.

A workspace file contains the following information:

  • Settings content for preview nodes
  • Link information for the viewer settings and background texture files (FTXB)
  • Link information for model files (FMDB) and animation files (FSKB)
  • Link information for emitter set files (ESET)

You can save files that are associated with a workspace file anywhere up to the highest level directory on the same drive as the workspace file.
Files that are saved in other locations are not loaded when the workspace file is opened.
