Camera Distance Scale Restriction (Near)

This section describes the Camera Distance Scale Restriction (Near) feature, which applies a limit to the sizes of particles that are close to the camera.

Overview

The Camera Distance Scale Restriction (Near) feature limits scaling so that particles in front of the camera are no larger than necessary.
This feature can be used to reduce rendering loads that arise when particles that have large rendered surface areas overlap with the camera.
The following overview describes the settings ().

Camera Distance Scale Restriction (Near)

Select or clear the Camera Distance Scale Restriction (Near) check box ()to enable or disable the feature.
When this check box is selected, the Scaling limit distance (near) group () is displayed. The group has the following settings.

Scaling limit distance (near)

Limits the size of particles shown on camera that are closer to the camera than the value set using the dial ().
