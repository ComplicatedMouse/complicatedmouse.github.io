What are the rules governing pane drawing order?

Question

What order are the panes drawn in?

Answer

Panes are drawn in the following order.

1 Rendering occurs in order from parent panes to child panes. In other words, child panes are drawn on top of the parent pane.
2

If multiple child panes exist, rendering occurs in the order of the number assigned to the child pane.

To be specific, the child panes are drawn in order of their positions in the tree node in the Hierarchy window.
3

Separate from this described drawing order, you can also change the display priority by setting Z values for the panes.

Note that the tool and the viewer cannot check Z value settings. You must have the program run its own comparison of Z value settings.
