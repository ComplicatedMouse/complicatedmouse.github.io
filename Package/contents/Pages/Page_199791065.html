Important Information

Important information for this version.

Workaround for Breaking Change Due to Improvements to Stripe Feature

You need to be aware of certain points when playing data that was created using a version of the stripe feature prior to this release.

  • When stripes are used, the Masking Check preset is selected for Soft particles in the Color tab.
  • When stripes are used, z-axis rotation is enabled in the Rotate tab.

As a result of these changes, the results of rendering data that was created with prior versions, may change in unintended ways.

New Effect Expressions

This section introduces features that broaden the scope of what can be depicted with effects.

Addition of Features That Can Be Used In Combination With Stripes

Features have been added that can be used in combination with Historical Stripe and Historical Stripe2.

Related Links Stripes now can be used together with the following features.

Masking Check This feature applies translucency to particles masked by polygon models, using soft particle processing.
Rotate

Configures settings related to particle rotation. Only rotation around the Z-axis works with stripes.

In the stripe settings, select an option other than Billboard Stripe for the Calculation Method.

The following images compare the results of using Historical Stripe used alone and in combination with these features.

Used With Masking Check Used With Particle Z-Axis Rotation Animation

OFF : There is no change in translucency if Masking Check is not selected.

ON : The floor becomes translucent when masked if Masking Check is selected.

OFF : The effect without any rotation.

ON : Rotation around the Z-axis gives the appearance of a twisting ribbon.

Historical Stripe A particle shape where position information in each frame is stored as "past samples" after a particle is generated. This information forms the basis to the generation of particles connected together in a ribbon shape.
Historical Stripe2 A particle shape that combines the feature Historical Stripe with the ability to control the movement of the last of the past samples.
