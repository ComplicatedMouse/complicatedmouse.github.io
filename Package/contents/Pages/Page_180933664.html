Creating and Managing Nintendo Accounts in the Development Environment

Introduction

To develop and check network features in the development environment, in some cases you must have a user account linked to a Nintendo Account created for the development environment.

This section describes the procedures for creating and managing Nintendo Accounts in the development environment (dd1).

For more information about linking the created account with a user account on the development hardware, see Linking With Nintendo Accounts.

To create a new Nintendo Account, use the browser on a PC or similar device to access the Nintendo Account login page for the development environment.
If you access the URL provided in https://developer.nintendo.com/group/development/g1kr9vj6/tech-info/information-for-network-title, and enter the login ID and password, the Nintendo Account login page is displayed, similar to as follows. Click Create a Nintendo Account to go to the page for creating a new Nintendo Account. Note that the actual screen might look somewhat different from the following screen.

Completing the Creation of a Nintendo Account

Normally, you must verify the email address to finish creating a Nintendo Account, but in the development environment you can complete the process by entering the authentication code displayed in the upper-left corner of the screen.

This authentication code is displayed as follows in the upper-left corner of the screen.

If an authentication code is not displayed in the upper left of the screen, the email address may already be registered to another Nintendo Account. If this happens, retry creating a Nintendo Account using a different email address.
Besides the confirmation of email communication, you can proceed with other processes that require receiving email by using the information displayed in the upper left of the screen.

Managing Nintendo Accounts

When you log in to a Nintendo Account from the login page, you can access the following management page. Here, you will see how to change your profile. When changing your profile, follow the instructions displayed on the screen.

Linking With Other Accounts

Nintendo Accounts can be linked with accounts for external services, but in the development environment, Nintendo Accounts can only be linked with a Nintendo Network ID. Nintendo Accounts cannot be linked with Facebook, Google+, or Twitter.
The only Nintendo Network IDs that can be linked are ones from the development environment that were created from the 3DS and Wii U development hardware. Nintendo Accounts cannot be linked with Nintendo Network IDs in the production environment.

Go to the Link another account menu in the Nintendo Account management page to link a Nintendo Network ID.
You must log in using a Nintendo Network ID to link accounts. Follow the instructions on the screen to log in.
