Emitter Shape - Circle (Volume)

Describes the settings for Emitter Shape - Circle (volume), where particle emission locations are set on the surface of a circle.
These are the particles when this emitter shape is being used.

Overview

With the Circle (Volume) emitter shape, particles emit in the direction of the normal in a widening circle shape from the entire surface of the circle.
You can create a donut-shaped emitter by using the surface Hollowness setting to configure regions from which particles cannot be emitted.
The following overview describes the settings ().

When the emitter shape is set to Circle (volume), the Circle (volume) () settings are displayed. The group has the following settings.

Hollowness

The region from which particles are not emitted is configured is set as a percentage of the distance from the center to the perimeter of the circle.
