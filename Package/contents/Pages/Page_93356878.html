Workspace

The workspace displays the created nodes in tree structure and allows them to be manipulated.
This documentation describes the UI and features for nodes.

A Show/hide nodes The check box () sets whether the Viewer shows or hides the nodes that are displayed in the upper right of the workspace.
B Expand/collapse tree Use this button to expand or collapse the tree
A large number of nodes in the workspace is easier to view when the tree is collapsed.
C Nodes Representative icons depict the various features of EffectMaker.
Click a node to view the related settings.
Right-click a node to view an action menu.
