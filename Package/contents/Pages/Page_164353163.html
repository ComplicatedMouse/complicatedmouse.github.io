Texture patterns

Texture patterns

For a texture pattern, you create a texture pattern list from the textures registered in a pane in LayoutEditor.

You use this texture pattern list to create a texture pattern animation.

Texture Pattern List

To display the screen to produce the texture pattern list, select the Texture Pattern List tab.

Texture Pattern List Creation Screen
Texture Pattern List

Texture list used with texture pattern animations.

Lists the textures used with animations and gives them registration numbers.

Texture pattern animations are created based on these registration numbers.
Add

To add textures to the texture pattern list, open the Add Texture Pattern window.
Delete Deletes registered textures from the Texture Pattern List.
Number The registration number of textures entered in the Texture Pattern List.
↑ Provides a higher registration number for the selected texture in the Texture Pattern List.
↓ Provides a lower registration number for the selected texture in the Texture Pattern List.
Clear Deletes all textures from the Texture Pattern List.

Add Texture Pattern

To add textures to the texture pattern list, use the Add Texture Pattern window.

Add Texture Pattern Window
Add Texture Pattern Displays the textures registered in the Textures window in a list.
Select the textures to add by selecting their check boxes, and then click OK.
(If you select multiple textures at the same time, their numbers in the texture pattern list do not match the order in which you select them.)
