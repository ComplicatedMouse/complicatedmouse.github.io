Advanced Combiner Mode

Advanced Combiner Mode

Advanced combiner mode provides more advanced combiner settings than standard combiner mode.

Number of Stages

Sets the number of combiner stages used in advanced combiner mode.
One to six stages can be used.

Switching to Standard Combiner Mode

Changes the display mode from advanced combiner mode to standard combiner mode.

Selecting Stages

Select the combiner stages to edit.
Click the arrow buttons to replace the selected stages with the stages on the right and left.

Stages

Edit the stages selected in Stage Selection.

Constant Register RGB
Constant Register A

When Constant is selected in the source, you can individually select the RGB and A constant registers to reference.

Use the RGBA that comprises the selected results as the shared Constant Colors for color and alpha.

The following list shows the color register selections and where the colors referenced by that register are specified.

  • Black Color: References the color specified for Constant Colors or Black Color in Color Interpolation ().
  • White Color: References the color specified for Constant Colors or White Color in Color Interpolation ().
  • Constant Color 0: References the color specified for C0 in Constant Colors.
  • Constant Color 1: References the color specified for C1 in Constant Colors.
  • Constant Color 2: References the color specified for C2 in Constant Colors.
  • Constant Color 3: References the color specified for C3 in Constant Colors.
  • Constant Color 4: References the color specified for C4 in Constant Colors.

Color Specify the color combination method to use in the stage.
Buffer input source

Use the check box to specify whether to overwrite the buffer for the current stage with a copy of the calculation results from the previous stage.

This option can only be selected for stages 1 to 4.
Combine mode

Specify the calculation formula for combining sources.
Texture images are treated as numerical values here.

  • Replace with the color of Source 0.
  • Source 0 × Source 1
  • Source 0 + Source 1
  • Source 0 + Source 1 - 0.5
  • [Source 0 × Source 2] + [ Source 1 × (1 - Source 2)]
  • Source 0 - Source 1
  • (Source 0 × Source 1) + Source 2
  • (Source 0 + Source 1) × Source 2
Source

Select the targets to enter for the sources in the calculation formula specified in Combine mode.

  • Texture 0: Can be used correctly when set to the 0 texture.
  • Texture 1: Can be used correctly when set to the 1 texture.
  • Texture 2: Can be used correctly when set to the 2 texture.
  • Texture 3: This option cannot be used.
  • Constant: The value configured using constant register RGB or A.
  • Polygon color: The value set for the vertex color.
  • Output from previous stage: The calculation result from the previous stage.
  • Buffer from previous stage: The value from the previous stage.
Operands

Specify the components to use as colors for the channels in the source.

  • RGB
  • 1 - RGB
  • Alpha
  • 1 - Alpha
  • RRR
  • 1 - RRR
  • GGG
  • 1 - GGG
  • BBB
  • 1 - BBB
Scale Specify a scale value to apply to the calculation results from Combine mode.
Three values can be specified: 1.0, 2.0, or 4.0.
Alpha Specify the alpha combination method to use in the stage.
Buffer input source Use the check box to specify whether use a copy of the output from the previous stage as the buffer input source, similar to for colors.
Combine mode

Specify the combine mode.

Accepts the same settings as for colors.
Source

Select the targets to enter for the sources in the calculation formula specified in Combine mode.

Accepts the same settings as for colors.
Operands

Specify the components to use as alpha values for the channels in the source.

  • Alpha
  • 1 - Alpha
  • R
  • 1 - R
  • G
  • 1 - G
  • B
  • 1 - B
Scale Specify a scale value to apply to the calculation results from Combine mode.
Three values can be specified: 1.0, 2.0, or 4.0.

Constant Colors

Displays the black color, the white color, and constant colors 0 to 4 as a list.
For each color, you can manually enter R, G, B, and A values or change them using the color picker.
