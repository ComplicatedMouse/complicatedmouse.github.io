Separating Layers for Creating Fonts

Tips for separating the information required for creating fonts into separate layers in Photoshop.

1. FontConverter Settings

Output the image file that will function as a template from the GUI version of FontConverter.
When doing this, in the Output > Image tab, under Color Settings, set the color for the data you want to convert to a layer to a color that is different from the colors used for other data.

The following example shows a FontConverter configuration.
This example sets the color for Null to ■Greens to distinguish it from the color used for Glyph, and sets the color for Grid to ■Blues to distinguish it from the black color used for text.

Click the Convert button to output the template image.
FontConverter outputs the following image.

Info

The FontConverter settings are stored in the following directory.

\Users\(user name)\AppData\Local\Nintendo\FontConverter.exe_Url_(random number)\(version)\user.config

Sharing this configuration file with the other members of your team makes it possible to ensure that everyone is using the same settings.

You can also use this file to specify settings. For example, the grid color can be specified in the <GridColor> tab of this file.
Make sure that you specify both types of values for each color channel (that is, both for the <R> value (from 0 to 255) and the <SCR> value (from 0 to 1), for example).

2. Photoshop Settings

Load the template image from FontConverter in Photoshop.
Use the same colors used in Color Settings in FontConverter in the Select > Color Range tool in Photoshop.

  1. Select Select > Color Range.
  2. In the dialog box, set the Select combo box to Blues, and then select the grid to add it as the current selection.
  3. Select Layer > New > Layer via Copy.
  4. Select the original image layer that was loaded as the original layer.
  5. Select Select > Color Range.
  6. In the dialog box, set the Select combo box to Reds, and then select the glyphs to add them as the current selection.
  7. Select Layer > New > Layer via Copy.

Your workflow efficiency will increase if you add this procedure as a Photoshop Action.

This procedure produces the following result. The grid (cell width) (A) and the glyphs (B) are each split into separate layers.
