Overview

This section describes the overview for NintendoWare 3D.

Elements of NintendoWare 3D

Intermediate File

Assets such as models, shaders, or animations in NintendoWare 3D.

Related Links 3D Intermediate File Specification Manual

DCC Plug-Ins

Plug-ins used to export intermediate files from the DCC tool.

Related Links Maya Plug-In, 3ds Max Plug-In, Photoshop Plug-In
3DEditor

The GUI tool for creating and editing intermediate files.

Related Links 3DEditor
3D Command-Line Tools

Command-line tool for editing and converting intermediate files.

Related Links 3D Command-Line Tools

3D Runtime Library

Runtime library for embedding NintendoWare 3D features in game applications.

Related Links 3D Runtime Library Overview
Personal Resources

Model, shader, and animation binary files referenced by the 3D runtime library.

They are converted from intermediate files using the 3D binary converter and 3D shader converter.

Related Links 3D Binary Converter, 3D Shader Converter
Viewer Library

Runtime library for creating viewers.

Embed the viewer library into the game application to link the 3DEditor and game application, to edit models in the game application, and preview and edit models and animations loaded into 3DEditor.

Related Links Viewer Library
Mii Helper Library

Provides the feature for assigning nn::mii::CharModel to NintendoWare 3D binary resources.

Related Links Mii Helper Library
