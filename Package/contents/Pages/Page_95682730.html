Repetition Count

This section describes the feature for setting how many times to repeat the registered texture in the facing surface of the particle.

Overview

The Repetition count feature sets how many times to repeat the registered texture across the entire facing surface of the particle.
Map the created texture in lower resolution than it would be in its actual size, and use the Wrap and Repetition count features in combination to create the desired effect on the screen.
The following overview describes the settings ().

Repetition Count

Select the repetition count from among the presets, which take the format U x V (horizontal x vertical).
There are also buttons for configuring these settings together.
The table shows the icon buttons and their corresponding selections.

1 × 1 The default setting. The texture is not repeated.
2 × 1 The texture is repeated twice in the horizontal direction (U).
1 × 2 The texture is repeated twice in the vertical direction (V).
2 × 2 The texture is repeated twice in both directions (U, V).

Example Configurations

In the following example, the Repetition count feature is used to make a quarter-circle texture look like a full circle in the viewer.
Wrap has been set to Mirror together with U, V.

A 1 × 1
 The registered quarter-circle texture () is displayed unmodified.
B 2 × 1
 The registered quarter-circle texture () is flipped horizontally.
C 1 × 2 The registered quarter-circle texture () is flipped vertically.
D 2 × 2 The registered quarter-circle texture () is repeated by flipping vertically and horizontally.

Wrap has been set to Mirror, so the texture appears as a circle.
