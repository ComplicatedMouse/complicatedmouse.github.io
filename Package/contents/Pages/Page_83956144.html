Troubleshooting

Common problems that occur when you use SoundMaker, and how to resolve them are described.

SoundMaker no longer starts.

Delete the (project_name).fspj.settings file and try to start it again.

List items to include in a sound set or on the Bank tab no longer appear.

Delete the (project_name).fspj.settings file and try to start it again.

Conversion fails.

Try reconverting.

  • After importing a project for another platform, use Reconvert rather than Convert.
  • If the maximum number of parallel conversions set under Tools Options is two or more, set it to one and try reconverting.

When you convert a sound project imported using NintendoWare SoundMaker, an invalid sound archive is exported.

When an NW4C/NW4F sound project (still with a cache file for conversion purposes) is imported and the normal conversion process is performed, the binary source files for NW4C/NW4F get mixed in with the NintendoSDK binary sound archive files (BFSAR files) and the nn::atk runtime cannot properly load the data.

This problem is due to the use of old cached data, so run Reconvert after you import the sound project.

Characters no longer display in the list.

In lists in the Sound Set tab and the Bank tab, the numbers shown in the items on the far-right side are sometimes overwritten by the display of the scrollbar and hidden from view.

Display Example

When the screen looks like the upper screen and then you set the Comments to hidden, the screen changes to look like the lower screen, and the values in User Settings can no longer be seen.

You can avoid this problem by placing items of low importance like the Comments on the far right side.

SoundMaker is slow to start.

SoundMaker can take a long time to start the first time it is started after the PC has started.
If you plan to use SoundMaker, start the application once after starting your PC so that it starts smoothly the next time you start it.

Starting Multiple Instances of SoundMaker

Although you can start multiple SoundMaker applications simultaneously, if you start multiple instances of SoundMaker and open the same project, but then edit and save that project only under one instance of SoundMaker, a dialog box for reloading the project automatically appears for the other instances of SoundMaker.

The specified note is not voiced with sequence sound

If a specified note is not voiced with sequence sound, a waveform file may not be assigned to the region that is supposed to be voiced with the note on the instrument sample map.

Check the SoundMaker instrument sample map.

Region with no assigned waveform file

When a sound waveform is output from NintendoWare SoundMaker or SoundProjectWaveExporter, silence is occasionally added to the beginning of the waveform.

When a sound waveform is output from NintendoWare SoundMaker or SoundProjectWaveExporter, sometimes silence is added to the beginning of the output waveform.

If silence has been added to the start of the waveform output, delete that portion using a waveform editor.
