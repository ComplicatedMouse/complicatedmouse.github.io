Position(±)

This section describes the Position feature for specifying the position of convergence.

Overview

The Position feature is for specifying the position at which particles converge.
The following overview describes the settings ().

Position(±)

Sets the position of convergence for particles based on the xyz of the emitter's coordinate system.
The following equation derives the value added to the position of particles as a result of position of convergence.

  • Value to add to position: Convergence position - Position of the particle at that time) × Convergent force
