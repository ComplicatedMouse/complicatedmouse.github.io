This section describes the Far feature for configuring the farthest position (far clipping), from the camera's viewpoint, in the camera's coordinate system.

Overview

The Far feature configures the position of the far clipping plane in the camera's coordinate system.
This section provides an overview of the settings ().

Far

Use the dial () or the text box () to set the value of the farthest position (far clipping), from the camera's viewpoint, in the camera's coordinate system.
