Specified Direction Dispersion Angle

This section describes Specified Direction Dispersion Angle, which can be used to disperse particles when Specified-Direction Initial Velocity is being used.

Overview

Specified Direction Dispersion Angle can be used to disperse particles in the range of the specified angle when Specified-Direction Initial Velocity is being used.
The following overview describes the settings ().

Specified Direction Dispersion Angle

Sets the dispersion angle for particles in the specified direction.
For example, if the value has been set to an angle of 20 degrees, then particles will disperse in a range of 40 degrees (B), with the specified direction A as the central axis of dispersion.

Particles are emitted from the place where the Circle (evenly divided) emitter shape is divided into four parts.
Because particles disperse from where they are emitted at these places of division, a single emitter can be made to look like multiple emitters.
