NintendoWare Audio News

This page introduces the changes in the NintendoWare audio development environment for NintendoSDK 1.1.0.

Added a feature for prefetching stream sounds.

Added a prefetching feature that pre-loads the data at the beginning of stream sounds into memory to eliminate the loading wait time when the stream sound is played.

You can enable this feature in SoundMaker in the Stream Sounds tab by selecting the Pre-fetch check box, which pre-loads the sounds in the program.

See Stream Sounds in the SoundMaker manual and Stream Sounds in the Programming Guide for more information.
