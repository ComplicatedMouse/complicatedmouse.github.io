NVIDIA GameWorks

Overview

GameWorks is a package of graphics libraries and samples using NVN.

Packages

GameWorks is provided as a separate package from the NintendoSDK NX Add-On. It may also be released later than other NintendoSDK packages.

The following table lists the packages.

Package Name Description
NVIDIA GameWorks Graphics libraries and samples using NVN.

Location

GameWorks is located in the following location in the directory where NintendoSDK is installed.

NintendoSDK/
GameWorks/
    extensions/
    Samples/
    tools/

Manual

For more information about the package, see the documentation in the package.
