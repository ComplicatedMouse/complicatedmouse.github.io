Texture Property Window: General Page

This window is used to check and edit the general properties of the selected texture.

A The full path to the selected texture.
B The color image for the selected texture.
Clicking the color image opens a details window.
C The alpha image for the selected texture.
Clicking the alpha image opens a details window.
D The index of the array texture to display. For more information, see Display Depth.
E DCC tool presets.
F The texture preset format.
G The texture format.
H Displays the filter type when creating mipmaps for textures.
There are three filters types available: point filtering, linear interpolation, and cubic interpolation.
I The color component selector to use when when sampling textures.
J Select this check box if you are applying a weight according to the human eye to RGB errors when using texture compression (bc1, bc2, or bc3).
K

Displays whether the R, G, B and A channels of the selected texture have been linear-converted.

For data output from the DCC tool, linear conversion is applied according to the settings in the configuration file.

For more information, see the description of the linear attribute of the texture_info element in ftx graphics in the 3D intermediate file specifications.
L Displays the dimension of the texture.
M The texture width.
N The texture height.
O The texture depth.
P The data size of the texture are displayed (including padding and mipmaps).
Q The mipmap level of the texture.
Specify 1 if there is no mipmap.
R The minimum mipmap size value.
S Specifies a hint about how the texture is used.
T The hash value for texture's source data.
U The 3D Texture Converterattribute of the texture is displayed.
V

The path information for the source files for channel merging.
This information is for the source files when combining multiple input file channels in 3D Texture Preset Converter and exporting them as a single file.

Display Depth

This setting is active when a texture array is loaded.

Displays the texture image corresponding to the index selected from the drop-down menu.
Displays arranged texture images when Display All is selected.



Texture Depth

Note the following when using texture dimensions or a cube map.

  • Specify the number of textures for one-dimensional and two-dimensional texture arrays.
  • In the case of cube maps, specify a multiple of 6 according to the number of cube maps.

Texture Mipmap Level

Set a maximum value calculated from the raw image size when exporting from the DCC tool.

Details Window

This window is used to confirm color or alpha details about the selected texture image.

For more information about the shortcut keys that are available for use in the texture Details window, see here.

A The scaling factor that is used for display in the details window.
B Enlarges or reduces the texture image size.
C You can check the texture image by switching RGBA among channels.
D Display depth
This setting is activated when the an array texture is selected.
It displays the texture details corresponding to the index selected from the drop-down menu.
E Shows a mipmap display of the texture image.
F Switches the display method for mipmap.
G Displays Help.
H The texture image.
You can confirm the color image and alpha image.
I The position the mouse is pointing to.
J The color value of the position the mouse is pointing to.
K The texture format.
L The texture size.
M The mipmap level.
N The total number of bytes of the texture.
O

Displays whether the previewed texture is a texture that has been encoded.

If it is an encoded texture, EncodedImage is displayed. If it is not, OriginalImage is displayed.
If the texture is for no particular platform, or if it is a texture that the SDK does not support, OriginalImage is displayed.
