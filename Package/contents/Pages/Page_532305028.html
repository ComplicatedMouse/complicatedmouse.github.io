Locking Functionality for Animations

Locking animations helps prevent unintended editing of animation curves.
You can configure animation locks by right-clicking the attribute tree in the Curve Editor.

The following images compare the results before and after locking animations.
Locking the R and G channels from the RGB color channel set removes editing abilities, and allows operations only for the animation curve in the B channel.
