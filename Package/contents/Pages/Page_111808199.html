Pane Information Paste Controls Window

Pane Information Paste Controls Window ()

The Pane Information Paste Controls window lets you select any information about the pane and paste that information to a different pane.

  • Window > Pane Information Paste Controls

Open the Pane Information Paste Controls window, select the pane that will be the source of the data to paste, and then select Edit > Copy. The selectable items in the window will vary, depending on the type of the source data.
Select the pane you want to paste the data to, and then select the information you want to paste in the Pane Information Paste Controls window. To actually paste the information, select Paste to the Selected Pane.
Items that cannot be selected are unavailable.

Pane Information Paste Controls Window
A Copy Source Pane Name
 Displays the name of the pane that is the source of the data to be pasted.
B Basic Pane Information Select the basic pane information you want to duplicate.
C Picture Pane Set if data in the picture pane is being duplicated.
D Text Box Panes Set if data in the text box pane is being duplicated.
E Window Pane Set if data in the window pane is being duplicated.
F Parts Pane Set if data in the parts pane is being duplicated.
G Material Set if material data is being duplicated.
H Vertex color

Use when copying vertex color to a different type of pane.

You can use this to reciprocally copy text box pane text color, picture pane vertex color, and window pane vertex color.

I Vector Components Paste Control (common for all information)

Controls the pasting of vector components such as those for position, size, scale, and rotation.

You can use this for common x, y, and z control over all vector component information.

Information that is not selected in Vector Components Paste Control is not applied when you paste to the selected pane.
J Color Components Paste Control (common for all information)

Controls the pasting of the RGBA color components.

You can use this for common R, G, B, and A control over all color component information.

Information that is not selected in Color Components Paste Control is not applied when you paste to the selected pane.
K Animation Set if animation data is being duplicated.
L Overwrite Key Information at Paste Target When this item is selected, all of the paste target's key information is overwritten.
If this item is cleared, the paste target's key information will be revised.
M Select All Selects all data that can be pasted.
N Paste to the Selected Pane Executes the process of pasting the data.
