Noise Offset (±)

This page describes the Noise Offset (±) feature for shifting the position of the noise table.

Overview

The Noise Offset (±) feature makes it possible to adjust the position of the noise table used to generate curl noise.
Particle motion is determined based on the position in the noise table at which the particle is generated.
Changing this offset value shifts the position and makes it possible to adjust particle motion.
The following overview describes the settings ().

Noise Offset (±)

Use the dial () or the text box () to specify the offset for the noise table.

Example Configurations

The following images provide a comparison of what happens when the Noise Offset (±) value is changed.

● Red 1 The default setting.
● Green -1 Particles move in a different direction compared to when the default setting is used.
● White 0 Particles move in the same way as when no noise table offset is applied.
● Light blue 0.01 The particles slowly move in a different direction compared to when the default setting is used.
