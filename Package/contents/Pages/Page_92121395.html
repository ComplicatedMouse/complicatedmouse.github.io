You can set how to manage the memory and data used by EffectMaker.

Memory Management

The Memory Management group () is a set of features used to set the memory size of the cache used by EffectMaker.
When the cache is full, earlier items are erased in the order that they were added.

Auto Backup

The Auto Backup group () is a set of features for automatically saving EffectMaker data at the specified interval.

Run Auto Backup Automatically saves the open emitter set file (ESET file) at the interval specified in Backup Interval.
Delete Files Earlier Than a Specific Time at Startup Select whether to delete during startup files for which the specified number of days has passed.
Info

Auto backup saves emitter set files (ESET files, ) to the AutoBackup folder, which is in the same location as NW_EffectMaker.exe.
The following procedure is an easy method for opening the AutoBackup folder.

  • From the menu bar, select Help > Open auto backup location.
  • The Option Settings dialog box () opens. In Advanced settings, under Auto Backup, click the Open folder button ().
