Infinite Lifespan

This section describes the feature for setting the particle to have an infinite lifespan.

Overview

This feature can set an infinite lifespan to all particles emitted from an emitter.
One way to use this feature is to keep just one particle displayed for an emitter.

Info
  • The Infinite lifespan features work when One Time is selected in Emission Settings.
    Related Links For more information about One Time, see One Time.
  • To extinguish particles set to have infinite lifetime, select Alpha fadeout in the feature for forcing effects to stop playing.
    Related Links For more information about this feature that forces effects to stop playing, see here.
  • In EffectViewer, you can forcibly delete an infinite lifespan particle by clicking the Run Fade Command button () in the preview controls.
  • When Infinite lifespan is used, settings like Initial velocity can be set by the following methods.
    • Press the F5 key to resend to the viewer.
    • Use the preview controls to stop () and then play () animation.
    Related Linka For more information, see Preview Controls.

Infinite Lifespan

Select or clear the check box () to enable or disable infinite lifespan.
Infinite lifespan influences features such as scale and rotation that are animated according to lifespan.

When cleared, the Infinite lifespan (OFF) Specify lifespan () settings are displayed. The group has the following settings.

Lifespan Describes the Lifespan feature, which can be used to configure particle lifespans.
