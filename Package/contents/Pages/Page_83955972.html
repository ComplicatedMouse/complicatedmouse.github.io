Indirect Texture

This section describes how to use the Indirect Texture feature.
The Indirect Texture feature warps one texture image using a different texture image.
Complex effects can be created through the combined use of the texture causing the warping and the texture being warped.

Structure of Indirect Processing

This section describes the indirect processing mechanism.

Handling Each Component of an Indirect Texture

The examples show how the various indirect texture components are handled.
The basic amount of warping in the horizontal and vertical directions is determined by the values of the R and A components, respectively.

Example １

If you have a 2-channel format indirect texture (the texture shown in the middle) and its R component and A component are both set to 128, no distortion occurs as a result of indirect processing.

Example ２

Warping occurs, but only in the region of the indirect texture (the texture shown in the middle) where its R and A components are set to values other than 128 (the circle in the center of the image of the indirect texture).
(Both the distorted texture and the indirect texture have a size of 256 × 256, with the center circle of the indirect texture having a size of 128 pixels.)

A The R and A components are both set to 128.
B The R and A components are both set to 160 (= 128 + 32).
C The portion shifted 32 pixels to the lower right is shown.

Modifying the Degree of Warping Using the Indirect Matrix

You can apply scale and rotational transformations to the amount of warping affected by the indirect texture in the separate horizontal and vertical directions.
Because you can set animation to Scale and Rotate, you can use this to create interesting effects.

Using the Texture Warp Feature

Main Steps of Setting Indirect Textures

The following three steps make up the majority of the workflow for configuring indirect texture settings.

I. Load the texture.
II. Configure the texture settings for the material in the property window. (Specify at least two textures, a standard texture and an indirect texture.
III. Configure the texture combiner settings for the material in the property window.
