Emitter Plug-In Nodes

エミッタプラグインノード

This page describes Emitter Plugin Node features.

Consolidated Stripe Configures the settings for Consolidated Stripe, which a feature for generating a string-like particle by linking together the particles being generated.
Historical Stripe Configures the settings for Historical Stripe, which a feature for generating a string-like particle based on the historical positions of the particles.
Historical Stripe 2 Configures the settings for Historical Stripe 2, which has extended features when compared to Historical Stripe for finer control over the behavior of string-like particles.
Loop-In-Range Configures the settings for Loop-in-Range, which is feature that loops particles within the range of the "box" specified for displaying them.
