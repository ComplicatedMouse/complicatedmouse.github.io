Time Slider (Curve Editor)

The area above the time slider in the curve view of Curve Editor shows information about the selected curve and allows you to edit the animation frame by frame.

The following animations are supported.

  • Color Animations, Scene Animations (Light and Fog), and Shader Parameter Animations (Color)
  • Visibility Animation
  • Texture Pattern Animation

Color Animations, Scene Animations (Light and Fog), and Shader Parameter Animations (Color)

When editing color animations, scene animations (light or fog), or shader parameter animations (color), a color bar showing the color in each frame is displayed at the bottom of the curve view.

Color icons are displayed above frames containing a selected key.

You can move color icons to any frame by dragging them with the mouse.

Setting Colors Using the Color Picker and Adding Keys

Related Links You can use the Color Picker to change colors by clicking the color bar or the color icon.
If there is no key in the frame on which the color bar was clicked, a key is added, and then the color selected with the Color Picker is set to the key.

Copying and Pasting Color Values

Right-click in the color bar to copy the color at the cursor position.

Then you can move the cursor to the position where you want to apply the color and paste it.

Color Values Not Between 0 and 1

Curve Editor handles color animations with color values that are not between 0 and 1 as follows.

All key values for the RGBA curve are between 0 and 1 Curve Editor displays the color bar and a box showing the key color.
Key values are individually within the 0 to 1 range, but the overall curve is not Curve Editor does not display the color bar.
Instead, Curve Editor displays a color icon showing the color of the key (normalized with respect to a key on the RGBA curve).

Visibility Animation

A bar indicating what is shown (in white) or hidden (in black) is displayed in the Time Slider when editing a visibility animation.

  • Clicking on this bar adds a key to the frame that was clicked.
  • Clicking the eye icon toggles between show and hide.
  • You can move eye icons to any frame by dragging them with the mouse.

Texture Pattern Animation

Textures are displayed as thumbnails in the Time Slider when editing texture pattern animations.

  • A texture icon is displayed in frames where the texture changes.
  • You can move texture icons to any frame by dragging them with the mouse.
  • Resting the mouse pointer on a texture thumbnail displays a larger texture image.

Info
Related Links You can open the Change Texture Image dialog box and edit the texture used in a texture pattern animation either by clicking a larger texture image being displayed or by clicking a texture icon.

Editing Texture Pattern Animations

When editing a texture pattern animation using Curve Editor, you cannot edit parameters associated with a loaded texture other than its index when Clamp values on the toolbar is enabled.

Clamping values is a handy feature to use when you are editing a texture pattern animation and want to prevent accidentally setting an index outside a specified range of valid values.

  • Values are always clamped during real-time editing.
  • During real-time editing, if the link to a texture being referenced is lost or a key not included in the texture pattern animation is selected, a warning message is displayed and the connection to the application is terminated.
  • A warning message is displayed and you cannot connect to the application if you attempt to transfer an invalid texture pattern animation when you start connecting to the application.

Adding Textures by Dragging

Related Links A texture using a pattern animation can be added by dragging the texture from File View or Windows Explorer to Curve View.

When dragged in Curve View, a key is added by adding to the existing pattern number with the maximum value in the frame.

Example:
When there are existing patterns up to number 4, the dragged texture is automatically added as pattern number 5 and a key is also added on the curve.
