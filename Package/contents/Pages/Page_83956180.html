Change Texture Image Dialog Box

Changes the textures used by the sampler.

Perform this operation from the Change Texture Info dialog box that is opened from the Samplers page of the material Property Window.
A The color image of the texture.
B The alpha image of the texture.
C Indicates the display depth. This setting is activated when a texture array is selected.
D Displays a list of textures loaded into 3DEditor. Select the textures to be used by the sampler from the list.
E Displays information about the selected texture.
F Confirms the use of the texture selected from the list in the sampler.
G Cancels changes made to the texture.
