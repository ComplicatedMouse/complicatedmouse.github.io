SRT Transform

This tab configures settings for the position, tilt, and size of the registered 3D model.

Offset Matrix

The Offset Matrix group () is a set of features for offsetting the position, tilt, and size of the registered model in the viewer.
This group has the following settings.

Coordinates (±)

Sets the offset to the position set for the model.
Rotation (±) Sets the offset to the angle set for the model.
Scale

Sets the offset to the scale set for the model.
