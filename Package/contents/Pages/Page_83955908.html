Emitter

This section describes the feature for using the viewer to check those settings of the Emitter feature of the Emitter node that can be controlled in the game application.

Emitter Tab

The Multiply Emitter Settings by Scale group () is a feature for configuring settings for the scale of all emitter shapes in the emitter set.
The following table shows the properties to which scale is applied.

Emitter Size Particle Size Particle Speed
Yes No No

The following is an overview of the setting.

Emitter Shape Scale (±)

Sets the value to multiply all emitter shape scales in the emitter set by for the x, y, and z axes.
