Firmware Information

Overview

Displays the firmware version information and network information for the development hardware.

Information

System Firmware Version

Version of the System Updater that the development hardware was initialized with.

Touch Screen Firmware Version

Version of the development hardware's touch screen firmware. Only displayed in NintendoSDK Firmware for NX 9.0.0-2.0 and later.

Startup Time

When the development hardware is powered on, this is the time between when startup processing for the development hardware begins and DevMenu startup processing completes (in milliseconds).

SDEV Network Status

Network information for the SDEV system. Displays the IP address, subnet mask, and MAC address.
Not displayed on hardware other than the SDEV.
