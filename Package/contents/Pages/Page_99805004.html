ADPCM Decoder

Feature Overview

The ADPCM decoder provides the functionality for decoding ADPCM data and converting it to PCM data.
This same functionality is provided by the Voice component of the audio renderer module of the audio Library, but the decoding functionality provided in the nn::codec namespace is provided as a stand-alone feature.

nn::codec provides the following functions as ADPCM decode features.

  • nn::codec::DecodeAdpcm()
  • nn::codec::GetAdpcmLoopContext()

Decoding is the complete work of the nn::codec::DecodeAdpcm() function. The nn::codec::GetAdpcmLoopContext() function is provided as a means to get the context information needed for looped playback.

These respective functions require the parameter information and the context information contained in the nn::codec::AdpcmParameter and nn::codec::AdpcmContext structures in the header of the ADPCM data.
