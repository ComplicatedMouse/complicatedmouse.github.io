TTF Font (Outline Font) Input/Output

Kerning Information Input/Output

Question

Can TTF fonts (outline fonts) be input and output to and from the GUI version of FontConverter?

Answer
TTF fonts can be input from the Windows Font tab. (However, make sure that the input font is installed on the PC).
The GUI version of FontConverter cannot output TTF fonts.

In the console version of FontConverter, TTF file paths can be specified as is.
