Audio News

NintendoWare Audio News

This page introduces features that were added to the audio module in NintendoSDK 0.10.0.

SoundMaker

Opus files can now be registered as stream sounds.

Opus files can be registered as Stream Sounds by dragging Opus files encoded in the OpusEncoder in advance to the stream sound list in SoundMaker.
However, it currently only supports up to single track and 2ch.

For more information, see the Compressed Audio page.

AtkPlayer

Effects can now be previewed.

An Effect Panel that allows sound effects to be applied and enables adjustment of parameters for each effect has been added to the AtkPlayer.

For more information, see the AtkPlayer page.

Waveforms can now be previewed.

Added a SoundOutput Panel that can monitor the waveform of the audio playback to the AtkPlayer.

For more information, see the AtkPlayer page.

Documentation

Notification of Document Structure Changes

The document structure under Audio has been revised.

It is now easier to navigate to the intended document from the tree structure to the left of the documentation.
