Rotation Axis

This section describes the feature for individually selecting the x-axis, y-axis, and z-axis for particle rotation.

Overview

Axial Rotation is a feature for separately enabling and disabling particle rotation around the x-axis, y-axis, and z-axis.
This group () displays the settings for each enabled axis. Use this group of settings to configure the angle and the rotation animation for the selected axis.
The following overview describes the settings ().

Rotation Axis

The setting for axial rotation is determined by how the flags for the x-, y-, and z-axes are combined.

The order of axial rotation is set to the default setting of y → z → x.
