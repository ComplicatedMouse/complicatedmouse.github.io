Particle Time Curve Editor Operations

This section describes the UI operations for the Curve Editor, used to add animation to particle time using curves in the property window.

Related Links For a more detailed procedure, see the Animating the Particle Scale section of the Making Flames tutorial.

Scale Animations

The image below provides a comparison of the before and after of configuring a scale animation.

Before configuring animation After configuring animation

The following properties animate the particle scale.
Animations use Curve Editor.

Item Name Time (%) Value Description
Scale Animation 0 1.0

Sets the value during emission to 1.0. The standard value is applied directly, unchanged.
50 1.0

Sets the value halfway through the lifespan to 1.0. This maintains the same size at the time of emission.
100 0.5

Sets the scale value to 0.5 in the final frame.
Matching the transparency animation creates an impression of a reduction in combustion energy.

The following video shows a method for using Curve Editor.

  • By clicking this, you can add the animation key.
  • You can set the key scale by vertically dragging the added animation key.
  • You can set the key timing corresponding to particle lifespan by horizontally dragging the added animation key.
  • To delete the animation key, right-click the key.
Info

For particle time animations, the time input values are specified as a percentage of the particle lifetime.
If you want to operate the key directly, the actual frame count is displayed in the key frame display ().
