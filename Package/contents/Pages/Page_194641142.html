Graph Display Area

This section describes the features displayed in the Graph Display Area of the Curve Editor window.

The curve of the selected pane is displayed in the Graph Display Area.

Graph Display Area
A Key Keys are displayed as black dots.
B Curve Curves are displayed in different colors depending on the selected item.
Click with the mouse to select a curve.
C Key value (vertical axis) This scale represents the key value.
D Key time (horizontal axis) This scale represents the key time.
E Color

When a color curve is selected, thumbnail icons with the configured colors are displayed at locations where several RGB keys are arranged at the same time.

Clicking with the mouse displays the color picker, where the color values can be changed.

Colors can also be set by dragging a color from the Color Samples window to an icon.

The reference color icon () is displayed at places where reference color keys have been set.

Right-click to display the Select panes having common color keys menu. From that menu, you can select panes that are using common colors.
