Terminology

A glossary of terms used in EffectMaker.

Emitter An emitter is the source of particles.
Particle A particle is a polygon object with an initial velocity that is emitted from an emitter.
Fields

A field can be used to add movement to the particle in directions other than the initial velocity direction.
Child Emitter A child emitter is an emitter located below another emitter in the hierarchy in the workspace.
Emitter Time and Particle Time This page describes the two time axes used in EffectMaker animations: emitter time and particle time.
Sine Wave A waveform that can be made into particle movements or settings animations that have a pattern with EffectMaker.
Noise A waveform of combined sine waves.
Random and Noise Describes the random and noise features for creating behavior without apparent pattern for particle movement and settings values.
Glare

Describes the glare feature that mixes the effect color into one or more pixels to create a shiny appearance.
Soft Particles Describes the Soft Particles feature that applies translucency to the borders of particles that overlap with models.
Distance Fields Describes the distance field feature, which makes it possible to achieve smooth outlines in images at lower than normal resolutions by using dedicated textures.
Normal Map Describes normal mapping in EffectMaker.
Z-Fighting Describes Z-fighting, which is a flickering effect that occurs when multiple polygons are drawn at the same position.
