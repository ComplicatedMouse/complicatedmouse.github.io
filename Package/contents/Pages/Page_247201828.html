nn::vfx News ()

Switching Texture Origin and Normalized Device Coordinates

You can now switch the following settings related to the graphics API from the project settings file.

  • The position of the texture origin in the framebuffer and texture buffer
  • The normalized device coordinate system

When these settings are switched, the behavior of the active effect viewer also changes.

nn::ui2d News ()

FFNT files used internally when converting FCPX files, are now automatically converted.

LayoutConverter now automatically converts the FFNT files that are used internally when FCPX files are converted.

nn::font News ()

Fixed a bug where scaleWidth of the multi-scalable font was not reflected in kerning.

Fixed a bug where the scaleWidth parameter, which applies scaling to the character width for each font in a multi-scalable font, was not reflected in the kerning.

Note that if you have used the scaleWidth parameter, the render result may be different now, than it was in the past.

Added scaleHeight for scaling the character height for multi-scalable fonts.

Added the scaleHeight parameter for scaling the character height for each font in a multi-scalable font.
