Material Editing Basic Operations

This sections uses the sample data included in the package to describe the operations of 3DEditor and the viewer.

Adjusting Sample Data

This section describes the process for editing materials and creating material animations for the sample model Tripod.
If you perform procedures according to the table, the resulting model will resemble the model in the following image.

1 Preparation This section describes the startup procedures for various tools that are required to begin work.
2 Loading Samples

Load each of the sample files in the package (shaders, models, and skeletal animations).
3 Linking With the Viewer Linking with the viewer, starting/stopping animations, and changing the file being played back.
4 Editing Materials

Add the sample textures in the package and adjust the materials to adjust the texture of the model.
5 New Material Animation Using diffuse color RGB animation to make the Sample Tripod's eye blink.
6 Saving Data Describes the procedure for saving the data that was created.
