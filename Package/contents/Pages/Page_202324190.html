The nn::spy Communication Buffer

Adjusting Buffer Size

You must adjust the size of the buffer configured when initializing nn::spy to suit the volume of data to send using nn::spy from the application.

The communication buffer size is too small for the size of the sent data when the following message appears for an application with the nn::spy library embedded.
Consider increasing buffer size when sending larger data, like waveform data, or when sending data at a high frequency.

[PushDataPacketAt] out of memory.


You can check the actual amount of data in use to assist in determining the buffer size. Periodically using nn::spy::DebugModule to send the rate of the nn::spy communication buffer use to Spy.exe, lets you observe that information in the Spy debug panel.
