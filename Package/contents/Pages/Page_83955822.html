TextureViewer

Introduction

TextureViewer is a tool for previewing texture intermediate files (FTXA and FTXB files).
For installation instructions, see Setup Manager.

Startup

TextureViewer is installed to %ProgramFiles%\Nintendo\ThumbnailExplorerExtensions\Tools\TextureViewer.
To start the tool, run TextureViewer.exe.

If you set up the file associations using Setup Manager, the tool starts when you open an intermediate texture file in Windows Explorer.

Screen Descriptions

  1. Title Bar (A)
    Displays the full file path name of the file that loaded in TextureViewer.
  2. Main menu (B)
  3. Changes the zoom setting of the preview. (C)
  4. Selects the channel to preview. (D)
  5. Shows or hides the mipmap display. (E)
  6. Displays detailed texture information. (F)
  7. Texture preview area. (G)
    Note: You can also preview compatible images with a drag-and-drop operation.
  8. Texture position (H)
    Displays the position of the mouse pointer relative to the loaded image file.
  9. RGBA settings (I)
    Displays the RGBA values of the part of the image file that the mouse pointer is resting on.

Main Menu

  • File
    • Open
      • Selects the file to load in TextureViewer.
        Note: You can also open an image file by dragging the file to the preview area.
    • Exit
      • Exits TextureViewer.
  • Help
    • Version Information
      Displays the version information for TextureViewer.
