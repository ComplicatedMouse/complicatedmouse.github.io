Soft Particles

This section describes soft particles.
The Soft Particles feature applies translucency to the borders of particles when models and particles overlap.
As a result, border lines fade and the effect becomes more natural.

The following figure compares what happens when the Soft Particles feature is enabled and disabled.
You can see the differences by checking the area circled in red (〇) that overlaps the floor where the snow fell.

A Unused (default) The overlap between the snow and the floor stands out, looking unnatural.
B Used The overlap does not stand out because translucency was applied to where the snow and floor overlap.

Soft particles are processed after the feature that offsets the particle depth value.
The following images compare the effects when using or not using Offset Z value and Z Fighting Prevention.

A Not in use (default) Only Soft Particles processing is applied. No Offset Z value processing occurs.
B Offset value (±) = 1 Soft Particles processing is applied at a position one step before offset application, as a result of Offset Z value processing.
C Z-fighting prevention enabled Soft Particles processing is applied at a location one before applying Offset Z value, as a result of Z Fighting Prevention processing.
