Features

This section describes the features provided by the perf library and how to use the API resources.

Features

Features Overview
Performance Metrics Tabulation Functionality for tabulating and managing performance metrics for the CPU and GPU/
CPU Performance Metrics Functionality for implementing CPU performance metrics. Tabulates the differences for system ticks and acquires performance data.
GPU Performance Metrics Functionality for implementing GPU performance metrics. Tabulates the differences for GPU timestamps and acquires performance data.
Info

Depending on the nature of the application, the CPU or GPU processing load can often vary considerably from frame-to-frame.
When visualizing measurements as numbers or graphs on the screen, take note of ghosting or flickering on the screen depending on the platform.
Drawing for debugging purposes during development is not an issue, but care must be taken when this is included in full retail versions.
