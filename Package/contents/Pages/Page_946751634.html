Photoshop Plug-In

You can now run the Post-Export Script before saving the PSD file.

Revisions were made so that you can now run the Post-Export Script before saving the PSD file in the Photoshop texture export plug-in.
If you set save_psd and execute_script_before_saving_psd to true in the config file, you can run Post-Export Script before saving the PSD file.
If the PSD file is large and takes time to save, you can shorten the time until the script is run.

Maya Plug-In

You can now export FMD files for polygons that have been converted to triangles in the same way as Maya.

Added a feature to the Maya export plug-in to convert polygons with 4 or more corners to triangles using the same pattern as Maya.
Enable this feature by specifying optimize_triangulation="dcc" in the config file.
(Set optimize_triangulation="angle" to use the previous triangle conversion patten.)
If you specify dcc, you can also generate concave faces and faces with holes.

Using Pre-Export Script to overwrite using something like NintendoOverwriteExportOption("optimize_triangulation", "dcc") is also supported.

You can now prioritize the Maya texture SRT when merging FMD files.

Added a feature to the Maya plug-in to prioritize the Maya texture SRT when merging FMD files.
If you specify texture_srt_merge_priority="true", the Maya texture SRT is prioritized.

In the multi-export plug-in, FDES filenames are now processed as preset names.

The Maya multi-export plug-in now can process the FDES filename as the preset name when you use the FDES file in the presets folder (the preset="" file).

To process the FDES filename as the preset name, specify multi_export_use_fdes_name_as_preset="true" in the config file. It is used for branching the processing of Pre-Export Script and Post-Export Script based on the preset name.

Config Settings multi_export_use_fdes_name_as_preset
true false
Output preset name FDES filename

Null character

If preset="filename" is written in the FDES file, this will be the name of the file.
