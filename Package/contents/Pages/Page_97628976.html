Size When Emitted

This section describes the feature for multiplying the size of all particles emitted from the emitter set by the specified value.

Overview

The Size when emitted feature multiplies the particle scale during emitter emission by the specified value.
When the value changes, the particle size when the emitter emits particles changes.
This does not affect the size of particles already emitted from the emitter.
This section provides an overview of the settings ().

Size When Emitted

Use the dial () or text box () to set the value to multiply the scale of particles emitted from the emitter by.

Example Configurations

The following figure compares what happens when the value of Size when emitted changes.
The Emitter tab's scale value is set to X,Y,Z = 2.

A X,Y,Z=1 The default setting. The size of the particles when emitted is the same as the Emitter tab's scale value.
B X,Y,Z=2 Compared with A , the size of the particles when emitted increases 200% for all (X,Y,Z).
C X=2, Y,Z=1
 Compared with A , the size of the particles when emitted increases 200% only for X.
