Features

The VI library has two main feature sets revolving around displays and layers.

  • How To: Create Layers

Displays

With the VI library, applications will be able to perform the following actions:

  • Enumerate the displays available
  • Open and close displays for use in other operations
  • Query display capabilities

Layers

Application content is controlled through layer handles.  With these handles, applications can expect to perform the following actions:

  • Create or destroy layers
  • Retrieve the native window for use with graphics APIs
