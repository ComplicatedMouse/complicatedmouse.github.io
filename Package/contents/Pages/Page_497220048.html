3 Parent Material Settings

This section describes how to set a single parent material (IceCream_Base) to all of the models.

Material Referencing

Click the split screen icon () to display a list of materials.

Press the SHIFT key in this material list and select all three materials other than IceCream_Base, then open the shortcut menu and select Material property window.


In Material Referencing, click Add.


In the property window for the parent material to add, select the reference source check box, then select IceCream_Base and click OK.


Confirm that IceCream_Base has been added to the Material Referencing part of the Material property window, and then close the window.

Select the check box for View the parent material. in schematic display mode to display the parent material and confirm the connections by the dotted lines.
