3D News

This document is a graphical introduction to the features that were added in NintendoSDK 4.1.0.

3DEditor

Changed the specifications for the material reference feature.

Created a menu named Child Material Restrictions in the material properties window that integrates what used to be the two separate menus Child Material Display and Child Material Edit.

Intermediate files that have been set already are converted to this new format automatically when they are loaded by 3DEditor or the 3D command-line tool. For more information about these menu items, see the documentation for the material properties window.

Maya Plug-In

Added a Feature to Control How to Export Vertex Color

You can now configure the plug-in so that Maya color sets for vertex color are not exported to the intermediate files even if they are set as the current color.
Vertex color sets with names that start with nw_ignore will not be exported even if they are set as the current color set for vertex color.

This feature is useful when you do not want editing data (such as the vertex information set to the vertex color for use with a different DCC tool) to be exported to the intermediate files.

Added a Feature for Configuring a Maya Startup Batch File to Only Set the Plug-In Environment Variable

You can now set only the environment variable for the Maya plug-in without starting Maya.
This feature is useful when using the plug-in with mayapy.exe.
