How Library Files Are Provided

Windows Environment

  • The library files are broken down at the feature level into static library files (LIB files).
  • Applications make use of the library files they need with static links.

Development Hardware Environment

  • Library files are basically available as SDK DLLs, and are dynamically linked, automatically, to applications.
  • That said, applications must explicitly link to some libraries to use them without including them in SDK DLLs, depending on library specifications and how they are used.

Specifying Library Files

For more information, see Project Parameters, the Build System Manual page.
