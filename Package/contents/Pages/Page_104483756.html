NX Performance Management Features

This section describes the features provided with the NX system to manage system performance.

Features Provided in the Current Version

The Performance Management feature provides separate sets of performance presets that can be used in console mode and in handheld mode, together with a method for specifying these presets.

The feature uses the oe library. For more information, see OE Library > Manual > Features > Selecting Performance Configurations
