Creating a Parts Layout

Describes the procedure for creating the following Select button, which can be used as a button part on the menu screen.

Info

The completed data for this tutorial is in the following location.
\Samples\Sources\Resources\Ui2d\Menu\Tutorial_Menu_Bar.flyt

Preparing to Create the Layout

To prepare to create the layout data, have LayoutEditor running in advance.

Related Links For more information about starting LayoutEditor, see Workflow.

Use the following procedure to specify the layout color space.

1 In LayoutEditor, from the menu in the main window, select Edit > Tool Environment Settings.
2 To set the color space, in the Tool Environment Settings window, select Gamma Characteristics > Linear Color Space Mode and select and clear check boxes.
After making these selections, the binary converter converts both textures and color parameters to a linear color space.
3 Click OK to confirm the settings.
4 If the Layout window was already open, close it and then open it again.

Related Links For more information about linear color space mode, see Specifying the Color Space.

Opening Windows

Begin the task after opening the following windows.

  • Textures
  • Hierarchy
  • Property
  • Parts Palette
  • Parts and Control Settings

Related Links For more information about the UI of each window, see Windows.

Setting the Parts Root

Complete the following procedure to specify the folder where the project configuration file used in the tutorial is configured.

1

In the Parts Palette window, click the Open Folder () button to open the Browse Folders dialog box.
2 Specify a lower directory as the Parts root folder.
\Samples\Sources\Resources\Ui2d\
The project setting file is created in the folder specified as the Parts root folder.
3 Click the OK button to close the Browse Folders dialog box.

Related Links For more information about the project settings file, see here.

Creating a New Layout

From the main menu, select File > New to create a new layout.

Related Links For more information about creating a new layout, see Creating a Layout.

Load the texture.

Load the following textures to be used into the Texture window from the following location.
Samples/Sources/Resources/Ui2d/Common/Textures/

Switch the Texture window to Detail View and configure the texture format as follows.

Thumbnail

 Filename Format Description

Band.tga A8:(uncompressed_A)

Format that uses a texture as alpha without compression.

Used as the select button frame.

Select.tga BC4A:(A) Format that uses a texture as alpha with compression.
Used as the select button cursor.

Related Links For more information about registering textures, see here.

Creating a Picture Pane　

From the Texture window that displays the textures, create a picture pane ().

In the Texture window, double-click Band.tga ().
In the Hierarchy window, select the created picture pane () and change the pane name to Band.
In the property window, specify the following settings.

Color Interpolation ()

 White Color Blending R,G,B,A = 8,8,8,200 This pane becomes the button frame.

In the Texture window, double-click Select.tga ().
In the Hierarchy window, select the created picture pane () and change the pane name to Band_Fade.
In the property window, specify the following settings.

Color Interpolation ()

 White Color Blending R,G,B,A = 8,8,8,200 This pane becomes the cursor when selecting the button.

In the Texture window, double-click Select.tga () again.
In the Hierarchy window, select the created picture pane () and change the pane name to Select.

Related Links For more information about changing pane names, see Editing Layouts.
For an overview of picture panes, see here.

Creating a Text Box Pane

Create the text box pane () that will be the string placed on the button.

From the main menu, select Pane > Create Pane > Text Box Pane.
In the Create Text Box dialog box that opens, configure the settings to the following.

Font File Specification Samples/Sources/Resources/Ui2d/Common/Fonts/sample.ffnt
String Tile Name

Click the OK button in the Create Text Box dialog box.
In the Hierarchy window, select the created text box pane () and change the pane name to Text.

Creating a Null Pane

Create the null pane () for grouping the created panes together.

From the main menu, select Pane > Create Pane > Null Pane.
In the Hierarchy window, select the created null pane () and change the pane name to N_Menu.

Related Links For an overview of null panes, see here.

Creating Pane Hierarchy

In the Hierarchy window, place the created panes in a hierarchy.
In the Hierarchy window, check that the created panes are in the following order.

Band (), Band_Fade (), Select (), and Text () are set in the child hierarchy of N_Menu (), as shown in the following image.

Related Links For more information about creating hierarchy in the Hierarchy window, see here.

Creating a Boundary Pane

Create a boundary pane () in a specified detection area.

From the main menu, select Pane > Create Pane > Boundary Pane.
In the Hierarchy window, select the created boundary pane () and change the pane name to Bounding.
In the property window, specify the following settings.

Base Pane ()

 Size X,Y = 400,50 The pane is configured to be slightly smaller than the button.
Part Scaling Effect Maintain distance from boundaries of parts containing self. Linked to button size changes.

Related Links For an overview of boundary panes, see here.

Making a Layout Part

In the Parts and Control Settings window, set this layout to Parts Layout.
Check that the setting options in the Parts and Control Settings window are disabled.
Click the Browse button () and make the following settings in the Control Settings dialog box.

Control Type Locator part [Locator] Can be used as a placed part in the body layout that is the placement target.
Name Menu_Bar

In the Control Settings dialog box, click OK to close the dialog box.
The Parts and Control Settings window settings are now enabled.
The pane can be used as the body layout parts pane that is the placement target.
