Overview

The command-line tools are in Tools/Graphics/3dTools.
Make sure that you include the path that contains this folder when supplying a path to the command-line tool.

The 32-bit versions of the tools have a 32 appended to the end of the tool name. Tools without it are 64-bit versions of the tools.

List of Command-Line Tools

This page describes the command-line tools for graphics provided in NintendoWare.

Icons Executable Filename Description

3dIntermediateFileOptimizer.exe Intermediate file optimizer
Performs various optimization processes on intermediate files.

3dIntermediateFileConverter.exe Intermediate File Converter
This utility converts generic format files into standard intermediate files.

3dBinaryConverter.exe 3D Binary Converter
Converts intermediate files to runtime-use binary files (BFRES files).

3dShaderConverter.exe 3D Shader Converter
Converts shader configuration intermediate files (FSCA) and shader source code (GLSL) to shader definition intermediate files (FSDB).
Also converts shader definitions intermediate files (FSDB files) into runtime-use shader archive files (BFSHA files).

3dIntermediateFileAssignUtility.exe Assign Utility
Assigns shaders in model intermediate files.

3dTextureConverter.exe

3D Texture Converter

It creates texture intermediate files (FTXA and FTXB files) from image files (TGA, PNG, DDS, and EXR files).
It also converts one or more texture intermediate files to a texture binary file.

3dTexturePresetConverter.bat

3D Texture Preset Converter

Uses an XML preset file to convert an image file to a texture intermediate file.

3dCmdlConverter.exe

CMDL Converter
This tool converts NW4C graphics intermediate files to NintendoSDK intermediate files.

3dIntermediateFileShaderVariationGenerator.exe

Shader Variation Generator

Compiles shader assign information from the model intermediate file (FMD file) and generates a shader variation (FSV file).

3dIntermediateFileVertexAttributePacker.exe

3D Vertex Attribute Packer

Merges multiple vertex attributes of model intermediate files (FMD files) based on the shader definition (FSD file) vertex attribute merge information.
3dIntermediateFileUpdater.exe

Intermediate File Updater

Updates intermediate files to the latest version.
3dIntermediateFileChecker.exe

Intermediate File Checker

Determines whether intermediate files are valid.
3dIntermediateFileBinaryToAscii.exe

Text Intermediate File Converter

Converts binary intermediate files to text intermediate files.
3dIntermediateFileAsciiToBinary.exe

Binary Intermediate File Converter

Converts text intermediate files to binary intermediate files.
3dIntermediateFileFormatter.exe

3dIntermediateFileFilter.exe

Processes intermediate files using scripts.

Using Command-Line Options

This documentation uses the following syntax for command-line options.

Example.exe [--args-file ""] [-v] [-h] [-s] [-j ""] [--search-path ""] [file|folder]...
  • To specify a parameter in the command-line options, use "" notation.
  • Omittable command-line options are enclosed in square brackets ([ ]).
  • The [ ]... notation indicates that you can specify the option inside the square brackets ([ ]) multiple times.

Parameter File

With the graphics command-line tools, you can pass the content of a file as a string of arguments by specifying the filename --args-file as one of the command-line options.

Each option can be specified on a separate line in the parameter file.
Note that you cannot include any newlines inside escape sequences enclosed in quotation marks.

Lines in the parameter file starting with a hash symbol (#) are treated as comments.

Increasing Processing Speed

For the command-line tools that support the --job-list option, you can create a text file that includes the paths of multiple files and then specify that text file for the --job-list option to convert all of the files at the same time. It makes it possible to convert files more quickly.

For example, assume that you used the assign utility to assign the simple shading model in shader.fsdb to inputA.fmdb and inputB.fmdb and wanted to output the results to inputA_out.fmdb and inputB_out.fmdb.

3dIntermediateFileAssignUtility.exe --shader shader.fsdb --shading-model simple --job-list jobList.txt

To do this, you could create a jobList.txt of the following format and then specify it for the --job-list option.

inputA.fmdb -o inputA_out.fmdb
inputB.fmdb -o inputB_out.fmdb

You can also specify different options to be used for each conversion process in this file. Any options specified separately from the --job-list option will be applied to all of the conversion processes.

For the command-line tools that support parallel processing, specifying the --job-list option creates a number of parallel processes equal to the number of files specified for the --job-list option.

If you execute a command-line tool from the system for building game resources, you can reduce processing time by executing the tool after waiting for a particular amount of data to build up.

Limitations

Note the following limitations when using command-line tools.

Binary File Identity

The resource files output by the binary converter and the shader converter differ depending on whether they are the 32-bit version or the 64-bit version. This difference is due to an error in floating point arithmetic. We recommend you use the 64-bit version.
