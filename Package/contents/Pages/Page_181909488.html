Use PC GamePad

Describes the Use PC GamePad feature that enables you to select whether you will use a PC gamepad or the controllers to control EffectViewer.

Overview

Use PC GamePad is a debugging feature for selecting whether to use the PC GamePad and controller to control the EffectViewer.
You can turn this feature off to prevent conflicts with EffectViewer when using the gamepad to play games on the PC.
