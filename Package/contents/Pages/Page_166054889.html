Mii Resources

Mii resources are graphic resources, such as parts textures and shape data, used when rendering a Mii character. The resources are separated into shape resources and texture resources. Both have several types.

Each Mii resource file is located in the following location of NintendoSDK.

NintendoSDK\Resources\Mii\Common\resource

Shape Resources

The following types of shape resources are available.

Resource Filename Description
ShapeHigh.dat High quality models.
ShapeMiddle.dat Middle quality models.

Texture Resources

Texture resources are provided for each type of platform (WinGeneric/WinNvn/NX), quality (Low/Mid), and gamma (linear/sRGB).

The following types of texture resources are available.

Resource Filename Description
WinGenericTextureLowLinear.dat Low-quality texture for linear gamma for Win/Generic use.
WinGenericTextureLowSRGB.dat Low-quality texture for sRGB gamma for Win/Generic use.
WinGenericTextureMidLinear.dat Mid-quality texture for linear gamma for Win/Generic use.
WinGenericTextureMidSRGB.dat Mid-quality texture for sRGB gamma for Win/Generic use.
WinNvnTextureLowLinear.dat Low-quality texture for linear gamma for Win/Nvn use.
WinNvnTextureLowSRGB.dat Low-quality texture for sRGB gamma for Win/Nvn use.
WinNvnTextureMidLinear.dat Mid-quality texture for linear gamma for Win/Nvn use.
WinNvnTextureMidSRGB.dat Mid-quality texture for sRGB gamma for Win/Nvn use.
NXTextureLowLinear.dat Low-quality texture for linear gamma for the NX development hardware.
NXTextureLowSRGB.dat Low-quality texture for sRGB gamma for the NX development hardware.
NXTextureMidLinear.dat Mid-quality texture for linear gamma for the NX development hardware.
NXTextureMidSRGB.dat

Mid-quality texture for sRGB gamma for the NX development hardware.
