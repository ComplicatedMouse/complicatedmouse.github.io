List of Samples

This page provides a sample data list of rendered effects that combines various EffectMaker features.

Info

The data is located in the following directory. The samples can be loaded at any time in the Effect Browser.
\Samples\Sources\Resources\Vfx\Eset\

Filename Sample Image Description Related Links
Fire_01

This effect uses texture pattern animation.

Effects can be easily created by using the actual video resources.
  • Using Pattern Animations
  • Texture Pattern Animation Playback Methods
Fire_02 Adds a sparks effect to Fire_01.
Fire_03 Adds a smoke effect to Fire_01.

Fire_04

This sample represents a single fire by overlapping multiple particles without using the video resource.
By setting the Particle follow type to Do not follow emitter, you can keep the fire effect in the location when moving the emitter.
  • Making Flames
  • Combining Flames With a 3D Model
Fire_05 This data has been adjusted to be more fire-like by adding smoke and sparks to Fire_04.
  • Embellishing the Flames
