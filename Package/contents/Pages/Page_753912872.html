Example 7: Create a Dump of the Detailed Profile Info
# Create a dump of the detailed profile info.
#   1. Expand the detailed info tree.
#   2. Copy the detailed info.
#   3. Write the detailed info to a file.
# Note: This example assumes there is already a profile loaded.


# 1. Expand the detailed info tree.
Profiler-TreeControl -Tab Info -Action ExpandAll

# 2. Copy the detailed info.
$detailedInfo = Profiler-Copy -Tab Info -NoClipboard

# 3. Write the detailed info to a file.
$outputFile = Join-Path -Path $env:TEMP -ChildPath "example-7.txt"
Out-File -FilePath $outputFile -Encoding "utf8" -InputObject $detailedInfo
