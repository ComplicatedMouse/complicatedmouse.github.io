Warnings & Errors

Displays the Warnings & Errors dialog box of the FontConverter (GUI) version.
You can jump from the dialog box to document pages that contain more detailed descriptions.
The following is the Warnings & Errors dialog box.

A Describes the error content and simple resolution method.
You can select text by dragging the mouse, and copy text using the keyboard shortcut CTRL+C.
B Related Links Click here to navigate to the List of Warnings and Error Messages page.
Describes the conditions for errors and warnings. Also describes resolution methods in more detail, along with diagrams.
Paste the copied error content text into the search feature on the browser page, and perform a search.
