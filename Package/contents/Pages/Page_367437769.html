Layout and Font News

This is a graphical introduction to the features added in NintendoSDK 6.3.0.

New Layout Expression

This section introduces features that broaden the scope of what can be expressed in the layout.

Improved Rounded Corners Feature

The Rounded Corners feature has been added to round the four corners when the pane shape is a quadrilateral.

Set Four Corners Separately

This feature sets the shape and the rounded corner radius separately for each of the four corners of the pane.

Shape Presets

This feature sets presets for the rounded corner shape.

The following video shows a preset that configures the four corners individually.

  • Upper Left: Rounded Corner (Interior) shape preset, rounded corner radius = 25.00
  • Upper Right: Corner Cut shape preset, rounded corner radius = 50.00
  • Lower Left: Rounded Corner (Exterior) shape preset, rounded corner radius = 100.00
  • Lower Right: Free shape preset, shape = 3.00, rounded corner radius = 200.00

Fixed a bug that caused the interior line of the border to look jagged.

Fixed a bug that caused the interior line of the border with rounded corners to look jagged.

Before Fix After Fix

When a pane with rounded corner borders was tilted at an angle, the interior line of the border looked jagged.
Fixed this bug by applying anti-aliasing to the interior line of the border.
The image on the left is before the fix, and the image on the right is after the fix.

Improved editing of color animation in the Curve Editor.

When you select a color curve in the Curve Editor, icons are displayed at the same time at the locations where RGB keys have been set to the curve.
The following example edits a color animation using color icons.
  • Drag using the middle or right mouse button to change the time.
  • Right-click the mouse to display the color picker and change the value.
  • Drag the reference color from the Color Sample window to the icon to set the reference color.
    The reference color icon () is displayed at places where reference color keys have been set.
