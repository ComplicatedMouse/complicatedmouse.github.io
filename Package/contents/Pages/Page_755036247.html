LayoutEditor

This section describes LayoutEditor features.

Added functionality for the Align Pane.

Added functionality to specify the horizontal and vertical alignment margin. This functionality can separately set margins for each pane affected by the Align Pane.

The following pane is located under the Align Pane in the Hierarchy Window and is subject to alignment.
In the Align Pane, Alignment Direction (horizontal) is set to left and Default Margin is set to 20.
We compare the difference between when the setting to specify the horizontal alignment margin is used and not used in the Base Pane of the pane to be aligned.

Horizontal Alignment Margin Not Specified

A : Horizontal alignment margin not specified. B : Horizontal alignment margin not specified.

The pane is aligned using the margins specified in Default Margin.

Horizontal Alignment Margin Specified


A: Horizontal alignment margin specified with a value of 50. B: Horizontal alignment margin specified with a value of 100.

The pane is aligned using the individually specified margins.
