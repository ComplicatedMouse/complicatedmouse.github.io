3D News

This section introduces new features added to the 3D module.

  • Updated Intermediate File Version
  • 3DEditor
    • Enabled use of additional reference paths for resolving material references.
    • The FMTB files in the additional references path for each file are now displayed as a parent material candidate.
    • You can now select Use the default value. when it is required to set values for the child material.
    • The startup file is now maintained between opening and closing the project file.
  • Photoshop Plug-In
  • Maya Plug-In
    • Added configuration settings that can be overwritten with the Pre-Export Script.
  • 3D Command-Line Tools
  • Runtime Libraries

Updated Intermediate File Version

Updated the intermediate file version for FMDB, FMTB, FSKB, and FMAB. For more information about the changes, see the revision history for the Intermediate File Specification Manual.

Update existing intermediate files using Intermediate File Updater.

3DEditor

Enabled use of additional reference paths for resolving material references.

You can now select a material from Search Targets when setting additional reference paths.

You can check whether the search target is a texture or material from the list of additional reference paths.

Related Links For more information, see the following pages.

Additional Reference Path

Describes an overview of additional reference paths.

The FMTB files in the additional references path for each file are now displayed as a parent material candidate.

When adding a parent material, you can now see a list of materials with additional reference paths.
You can now view only a list of materials in (Additional search path for this file.) for the reference path.

You can now select Use the default value. when it is required to set values for the child material.

Made improvements to 3DEditor so that when Child Material Restrictions and Require the value to be set for the child material are set from the parent material, you can now specify Use the default value for the Value Reference.

Related Links For more information, see the following pages.

Material Referencing Settings Describes the material reference settings.

The startup file is now maintained between opening and closing the project file.

The file that was opened at startup now remains open when re-opening the project file.
If the same file that was opened at startup is loaded into the project file, a confirmation dialog box is now displayed.
Even if the project file is closed, the file that was opened at startup does not close.
