Customizing Item Filters

This page provides an overview of how to use the custom filters feature to customize item filters and filter the items displayed in the attribute tree.

The custom filters feature makes it possible to customize item filters to filter the items that are displayed in the attribute tree.
It involves creating and adding named sets of items you want to be displayed. After these sets are added, they can be selected from a list and reused.
This makes it possible to efficiently create arbitrary item views, which can be useful when editing animations.

Adding and Using Custom Filters

Use the following procedure to add and use custom filters.

1

In the Curve Editor window, click the Custom Filter button ().
2 Click the text box on the right to display the Add New (from displayed items) button.
3 Select Add New (from displayed items) to open the Create New Filter dialog box.
4

In the dialog box, in Name, enter a name for the custom filter.

Make sure to choose a name that makes the purpose of the filter easy to understand.
5

Select the check boxes () for the items you want to sort and display.
6 Click the OK button to close the dialog box.
7 Select the custom filter added to the list in step 2, and check that the items in the attribute tree are filtered and sorted as expected.

Deleting Custom Filters

This section describes how to delete added custom filters.

1

In the Curve Editor window, click the Custom Filter button ().
2 Click the text box on the left to display a list of added custom filters.
From the list, select the custom filter that you want to delete.
3 From the Curve Editor window menu, select Filter > Remove Custom Filter.
4

The custom filter selected in step 2 is deleted.

To delete all added custom filters, select the following menu item in the Curve Editor window.

  • Filter > Remove All Custom Filters

Importing and Exporting Filters

Select the following menu items to import or export filters as files.

  • Filters > Import
  • Filters > Export
