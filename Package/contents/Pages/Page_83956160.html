About Instrument Parameters

This section describes the types of instrument parameters, and associated concepts.

Instrument Parameters and Region Parameters

The Bank tab of SoundMaker is divided internally into parameters for instruments (shared by all regions), and region-specific parameters.

The former are called instrument parameters, while the latter are called region parameters.

Using Parameters

This section looks at the various parameters that are set in the Bank tab, and describes in each case whether they are handled as instrument parameters or region parameters.

Envelope Command

Can be configured as either an instrument parameter or a region parameter.

You can select per instrument whether the envelope value is to be treated as an instrument parameter (configured at the instrument level), or as a region parameter (configured at the region level).
Although envelope is normally handled as an instrument parameter, you can configure envelope values separately at the region level by configuring a region parameter envelope for a multi-region instrument.
For more information about configuring an application target, see Selecting an Envelope Application Target.

The current envelope application target is indicated by the envelope display on the Instrument List.
The Instrument List items that show a numeric value use per-instrument envelopes. Those that show an asterisk (*) use per-region envelopes.

You cannot simultaneously set both an instrument-parameter envelope and a region-parameter envelope for a single instrument.

Pan

Handled as a region parameter and always specified in units of regions.

Volume

Can be configured independently as both an instrument parameter and a region parameter.

Pitch

Can be configured independently as both an instrument parameter and a region parameter.

About the Pan Mode

The pan mode for instruments is fixed to Dual.
