GPU Noise

This section describes the GPU Noise feature for moving particles in an orderly way that nevertheless appears random.

Overview

The GPU Noise field makes particles move along determined trajectories when they are emitted.
The trajectories can be created from combinations of as many as four waveform components.
By staggering the waveform phase, amplitude, and periodicity, you can make the particles appear to move randomly, even though their motions have regularity and are controllable.

Blue For more information, see the following pages.

Random and GPU Noise Describes the Random and Noise features, which can be used to express patternless behavior for specified particle motions and settings.

Basic Settings

The Basic Settings group () is a set of features for configuring the basic settings for the waveforms that will be used for the particle trajectories.

Noise strength

This feature sets the strength of the noise.

Basic period

This feature sets the basic value on which to determine the period for the components of the combined waveform.
Feel air resistance This feature sets whether the particles are affected by the Air Resistance set in the Emission tab of the emitter node.

Equalize phases of waveform components

The Equalize phases of waveform components group () is a set of features for setting whether to put the waveforms in phase when they are combined, and for configuring advanced settings for the phase.

Align Phase of Each Element

This feature sets whether to equalize the phases of the waveforms used each time particles are emitted.

Waveform Component Advanced Settings

The Waveform Component Advanced Settings () group is a set of features for configuring each of four waveforms before they are combined.

Settings for each component

This feature sets whether to configure advanced settings for each waveform component before they are combined.
