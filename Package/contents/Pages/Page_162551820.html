atk Sound Play-Log Panel

This panel displays a list view with information about the sounds that have played in the atk library.

UI Reference

Description
A The toolbar. Use it to change how the list is displayed.
B A list view showing the sounds that have started playing.
C The item in the list at the current time is displayed in orange.

Toolbar

Element Description
Name Filter Enter a string to filter the items in the list display. Names and players can be filter targets.

Sets the visibility of the list columns.

The column named Time / AppFrame cannot be hidden.

List View

You can sort by ascending or descending order by clicking each column header in the list.

List Column Description
Time / AppFrame

The time at which playback began.

The column name switches between Time and AppFrame depending on which unit of time has been selected.

  • Time shows the time in hh:mm:ss:ms format.
  • AppFrame shows the time as a number of application frames.
Name The sound name.
Players The player name.
Volume

The volume.

The current value is shown as a number and represented with a bar. This bar represents a ratio of the maximum value.

Players

Priority

The player priority.

The current value is shown as a number and represented with a bar. This bar represents a ratio of the maximum value.
Sound Type The sound types.
