Overview

What is In-Game Editing?

This feature edits and adjusts the content of sound archives while the application is running by including atk::viewer in the application and connecting to SoundMaker.

The following kinds of editing are possible.

  • Adjustments to sound volume and other parameters
  • Replacing waveform data and sequence data and banks

Include atk::viewer

For more information about including atk::viewer and about in-game editing, see In-Game Editing.

In-Game Editing

Connecting to the Application

This section uses the AtkViewer sample program to describe how to connect to an application that includes atk::viewer.
This description assumes you have already built the AtkViewer sample program.
For information about building the sample program, see the atk Start Guide.

Starting an Application

Make sure that TargetManager is running.
Start the AtkViewer sample program and connect to TargetManager.
The connection to TargetManager occurs automatically.

Connect to the Application

Connect to the application (AtkViewer) by selecting Preview > Connect, or on the toolbar by clicking Connect.

When the connection is established, "connected" is displayed in the AtkViewer screen.

If not, make sure that the toolbar's Platform and Destination settings are appropriate.

In-Game Editing Operations

In-Game Editing Operations

In SoundMaker, open NintendoSDK\Samples\Sources\Resources\Atk\Common\Common.fspj as the in-game editing sample project.

Add and remove sound items as targets for in-game editing using the Edit check boxes in the lists that display in the Sound Sets tab.

When a check box is selected, that item becomes a target for in-game editing. After conversion, its edited data is automatically sent to the connected application.

You can also make items automatic targets for in-game editing when parameters change in the list in the Sound Set tab.
The setting for this operation can be changed from the In-game editing tab in the Options dialog box.

Select the Edit check box to add items to the In-Game Editing window (described in more detail later) or clear the Edit check box to remove items from the In-Game Editing window.

The Edit cell displays a green check mark icon for items that are active targets for editing.
This icon changes to the Transferring Data icon while data is being sent.

: Edit enabled and sent.
: In progress (converting, waiting to send).
: Sending.
: Error.

In-Game Editing Window

To open the In-Game Editing window, select View > In-game editing from the menu or click the Show in-game editing icon on the toolbar.
These operations are used to switch between showing and hiding windows.

Switch between Floating and Docking by double-clicking the window title bar.

When the window is docked, you can minimize it so that only the tab is displayed by clicking the Minimize icon on the right side of the title bar, or by double-clicking the splitter.

If you clear the Enabled check box for a list item in the In-Game Editing window, that item remains on the list but is no longer a target for in-game editing.

Items that are targets for in-game editing can be deleted or immediately re-sent by right-clicking and selecting options from the shortcut menu in the In-game editing window, or from the toolbar.

Playing, Stopping, and More

Use the Play, Pause, and Stop buttons in both the Sound List and the toolbar to playback, pause, and stop sounds.

Applying the Edits

Items that are enabled targets for in-game editing can be sent to and applied in the application by simply changing their parameters in the list, or by using an external editor to alter and save the waveform file.

The edits are applied in the application after the data is sent, and then sounds play the next time.
Practically speaking, the edits are applied after you stop a sound and then play it one more time.
The exceptions are edits to the volume for sounds and Stream Sounds tracks, which are immediately applied to the playing sounds.

For example, when editing volume, if in-game editing is enabled and you edit the volume of a stream sound that is playing back, after the volume's value is changed and that change is committed, the Edit enabled check mark icon changes to Transferring. The changed information is transferred, and the new volume is applied as the stream sound plays.

Editable Items

The following types of sound set items can be targets of in-game editing.

  • Stream sounds
  • Wave Sounds
  • Sequence sounds
  • Bank
Info

Waveform archives, groups, and players are excluded.

If you want the data that is attached to the name of a target item to also be a target for editing, you must select the Output the string table option in the Sound Archive tab of the Project Settings dialog box.

Disconnect from application

Disconnect from the application (AtkViewer) by selecting Preview > Disconnect, or on the toolbar by clicking Disconnect.

When the connection is dropped, "disconnected" is displayed in the AtkViewer screen.

Shortcut Keys

F6 has been assigned as a shortcut key to connect to and disconnect from the application.
You can use the shortcut to switch between connecting and disconnecting.

If you select PC Application or another target for the Connect to setting in the toolbar, that selection will be the only target of the connect and disconnect operations in the menu, toolbar, and shortcut key.
