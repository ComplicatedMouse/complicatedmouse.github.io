Audio News

This page describes the changes in NintendoSDK 5.2.0.

SoundMaker

Added a feature to dynamically add and remove sound items in combination with a standard sound archive.

The add-on sound archive feature to dynamically add and remove sound items in combination with a standard sound archive has been publicly released.

It allows you to add and delete the necessary sound items while continuing to use the standard sound archive.
Add-on sound archives are suitable for use with sounds that are needed only in specific situations, and sounds that are included in downloadable content.

For more information, see the following pages.

  • Guide > Add-On Sound Archives
  • Practical Guide > Add-On Sound Archive Feature

Officially released the single playback feature.

The official version of the single playback feature for restricting simultaneous playback of a specified sound has been publicly released.
For more information, see the information about single playback on the following pages of the SoundMaker manual.

  • Practical Guide > Sound Set Tab > Wave Sounds
  • Practical Guide > Sound Set Tab > Sequence Sounds
  • Practical Guide > Sound Set Tab > Stream Sounds

Officially released the loudness measurement feature.

The official version of the feature for measuring the average loudness in SoundMaker has been publicly released.
For more information, see the information about average loudness on the following pages of the SoundMaker manual.

  • Practical Guide > Sound Set Tab > Wave Sounds
  • Practical Guide > Sound Set Tab > Stream Sounds
  • Practical Guide > Sound Set Tab > Sequence Sounds

In addition, see the information about loudness on the following page.

  • Practical Guide > Sound Set Tab > Option Settings

Officially released the downsampling feature.

The official version of the feature for downsampling the sample rate set for the sound without changing the original waveform file, and including it in the sound archive, has been publicly released.
For more information, see the Downsampling section on the following pages of the SoundMaker manual.

  • Practical Guide > Sound Set Tab > Stream Sounds
  • Practical Guide > Sound Set Tab > Wave Sounds
  • Practical Guide > Sound Set Tab > Instrument List

In addition, see the Resample section on the following page.

  • Practical Guide > Option Settings

Officially released the downmixing feature.

The official version of the feature for converting stereo to mono without changing the original waveform file, and including it in the sound archive, has been publicly released.

For more information, see the information about downmixing on the following pages of the SoundMaker manual.

  • Practical Guide > Sound Set Tab > Stream Sounds
  • Practical Guide > Sound Set Tab > Wave Sounds
  • Practical Guide > Sound Set Tab > Instrument List

In addition, see the Resample section on the following page.

  • Practical Guide > Option Settings

SoundProjectConverter Command-Line Tools

Officially released the feature for filtering items during conversion.

Official versions of the --include-list and --exclude-list options to filter the items and bank instruments in a sound project during conversion, and control whether they are included in the resulting sound archive, have been publicly released.

For more information, see the following manual.

  • Command-Line Tools > SoundProjectConverter

SoundProjectUtility (Command-Line Tool)

Officially released the feature for updating the version of sound projects.

An official version of the update-project subcommand for updating the version of sound projects has been publicly released.

For more information, see the following manual.

  • Command-Line Tools > SoundProjectUtility
