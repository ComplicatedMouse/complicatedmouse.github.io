Plug-In for Setting Shape Animations

You can use this plug-in to set the vertex attributes to be updated during a shape animation. The vertex attributes updated by default are: vertex coordinates, normal vectors, tangent vectors, and binormal vectors.
These attributes, however, are not applied on the 3ds Max screen.

Select Nintendo > Set Shape Animation to open the window.

Selected nodes are displayed under Selected Nodes.
Nodes without an added Morpher modifier are not displayed.

Setting Method

Select one or more nodes with an added Morpher modifier to display the current settings for those nodes in the Set field.
If you select multiple nodes or groups, the setting for the first selected node is displayed.

Settings are changed by controlling each item in the window. If multiple selections were made, selected nodes can be changed at the same time. (Only modified attributes are applied.)
You can select a different node by selecting another node without closing the window.

  • Position
    Select this option to update vertex coordinates. This option is enabled by default.
  • Normal
    Select this option to update normals, tangents, and binormals. This option is enabled by default.
    Tangent vectors and binormal vectors are only updated when normal mapping or an anisotropic shader is being used for the base shape.
  • Color
    Select this option to update vertex color This option is disabled by default.
    If no vertex color is set for the base object, vertex colors are not updated even if this option has been enabled.
Info

If Position, Normal, and Color are all disabled, shape animation is not exported for the selected node.
