Mii Face Models

The term Mii face model refers to the model of the head portion of the Mii character.

The Face library includes specifications for rendering Mii face models according to the parameters set in the Mii Data (the CharInfo data structure).

The Parts Comprising a Mii Face Model

A Mii face model is composed of several shapes that are combined together to render a Mii face model according to the configured parameters. The nn::mii::CharModel class serves to manage the shape and texture data for rendering the Mii face model of an individual Mii character.

The relationships between the various Mii face model parts and the corresponding nn::mii::CharModel::DrawType enumerated types are presented below.

Some of the parts that comprise the Mii face model are not necessary for every Mii character. For example, some Mii characters are rendered without parts such as eyeglasses and beards. For information about implementation-related cautions about unnecessary parts, see the description of CharModel.

Face Model Types

In addition to the normal Mii face model, there are also several other types of models designed for covering with hats and other embellishments.

The Face Library provides features for creating Mii face models for different purposes.

Face Model Types Sample Description
nn::mii::CreateModelType_Normal
Creates a normal Mii face model.
nn::mii::CreateModelType_Hat
Creates a Mii face model for use with a hat. The hat,hair, and forehead parts differ from those of a normal Mii face model.
nn::mii::CreateModelType_FaceOnly
Creates a Mii face model for use with a mask. The hat,hair, and forehead parts are not rendered.

You can also create a Mii face model with a shorter nose, for use with a full face mask.

Nose Types Sample Description
nn::mii::CreateNoseType_Normal
Creates a model with a nose of normal length.
nn::mii::CreateNoseType_Flatten

Creates a model with an adjusted nose length. The length of the nose is restricted, creating a model adjusted to below a set value.

You can specify different combinations of Mii face model type (nn::mii::CreateModelType) and nose-length type (nn::mii::CreateNoseType). For more information, see the section on the conditional creation of Mii face models in the description of CharModel.

Notes

Beard Types

Beards can be depicted by a shape or by a texture.

When a shape is used, the beard is depicted as one part of the Mii face model. When a texture is used, the beard is depicted as a portion of the Face Outline Textures.

Hat Types

The objects rendered as hat parts, include not only hats but also hair ties.

That is, hair ties are handled as hats. Note that a hair tie can also be rendered on a hat model.
