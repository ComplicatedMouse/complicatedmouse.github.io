Starting Point Alpha, End Point Alpha

This section describes the starting point alpha and the end point alpha, which configure the alpha for both ends of a stripe.

Overview

Starting point alpha and end point alpha are features to configure the vertex alpha values for the starting and end points of a stripe.
The starting point or the end point of a stripe can be made transparent without using a texture.

The following figure shows the starting and end points of a stripe generated by past samples in order from 1 to 3.

A The leading end of the stripe.
The alpha is 1.0 in the example.		 B The trailing end of the stripe.
The alpha is 0.5 in the example.

The following overview describes the settings ().

Starting Point Alpha

Use the dial () or text box () to set the alpha value in the range from 0 through 1 for the first past sample that is generated.
When the value is 0, the stripe gradually disappears from view, starting from the end and proceeding to the beginning.

End Point Alpha

Use the dial () or text box () to set the alpha value in the range from 0 through 1 for the last past sample that is generated.
When the value is 0, the stripe gradually disappears from view, starting from the beginning and proceeding to the end.
