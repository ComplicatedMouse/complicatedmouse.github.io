Sample Database For Debugging

A number of sample Mii database files have been prepared for the convenience of development.

These sample database files are located in the following directory in the SDK.

$(NINTENDO_SDK_ROOT)/Resources/Mii/Common/sampledb
Note

Use these database files for debugging purposes only. Be careful not to include them in the retail product.

Importing the Database Files

To import the sample database files, use the nn::mii::Database::Import() debugging function, DevMenu, or DevMenuCommand. To use the nn::mii::Database::Import() function, the Mii Testing Mode setting must be enabled in DevMenu. To enable this setting, go to the Debug page of DevMenu and set Mii Testing Mode to Enabled. For more information about importing databases using DevMenu or DevMenuCommand, see the ImportDatabase section in the DevMenu documentation and the Importing a Mii Database section in the DevMenuCommand documentation.

A combination that uses the maximum amount of memory when the shape resource quality is Middle, there is a texture beard, and the model is the normal type.

4 SMBSMH A combination that uses the maximum amount of memory when the shape resource quality is Middle, there is a shape beard, and the model is the type for headgear (nn::mii::CreateModelType_Hat).
5 SMBTMH A combination that uses the maximum amount of memory when the shape resource quality is Middle, there is a texture beard, and the model is the type for headgear.

The same combination of parts uses the maximum amount of memory regardless of the quality of the texture resource (Middle or Low). When the shape resource quality is High, the same combination of parts uses the maximum amount of memory regardless of whether the model type is normal, or the type for headgear.

When the beard is depicted by a texture, the same amount of memory is used regardless of the type of beard. When the beard is depicted by a shape, the amount of memory will vary depending on the type of beard.
