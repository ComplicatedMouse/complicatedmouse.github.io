Waveform Reference Count

The Waveform Reference Count window displays a list of the number of times that waveform files contained in instruments and used by all sequence sounds in a sound project are referenced.

Waveform Reference Count Operations

Opening the Waveform Reference Count Window

To open the Waveform Reference Count window, select View > Waveform Reference Count on the menu or click Waveform Reference Count in the toolbar.
These operations are used to switch between showing and hiding windows.

From the shortcut menu of this window title bar, you can choose to float, dock, or hide the window.
You can also switch between floating and docking by double-clicking the title bar.

When the window is docked, you can minimize it so that only the tab is displayed by clicking the Minimize icon on the right side of the title bar, or by double-clicking the splitter.

Operations in the Waveform Reference Count Window

Sounds can also include looped waveforms, so the play time must also be specified.

Click the Measure button in the upper-left part of the window to open the Measure Reference Count of Instrument Waveform dialog box, and then set a maximum value for the measurement time and click the Measure button there.
The measurement is made starting from the Starting Playback Position of each sound.

You can perform the same operation by selecting Preview > Measure Reference Count of Instrument Waveform.

When you run the measurement, the referenced waveform files are listed, giving the file path and the reference count for each. To display details about the referencing instruments, select a waveform from the list.

Double-click a numbered index cell in this list of referencing instruments to open the instrument of the corresponding bank.
