Shader Definitions Property Window: Shading Model Page

This page displays information about shading models generated using a shader source and shader configuration file.

A Displays the revision number of the shading model.
B Displays whether it is a shading model for materials.
C Displays information about the shader source used to generate the shading model and each component. Information is managed by using tabs.
