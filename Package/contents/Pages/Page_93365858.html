Height of Clipping Plane

Clipping Settings / Height of Clipping Plane

This section describes the feature for clipping particles displayed inside the box used for the Loop-In-Range feature.

Overview

This feature allows you to clip (hide) the range in which particles display inside the box above or below the world y-coordinate specified by Height of Clipping Plane.
You can hide particles above a specified y-coordinate to achieve the following kinds of effects.

  • Hide bubbles after they reach the water line.
  • Hide rain while the player is underwater.

Using this feature to hide the display of particles is also a way to lower the particle rendering load (fragmenting).
This section provides an overview of the settings ().

Clipping Settings

The following presets are available to enable or disable clipping and to set the range in which particles display after clipping.

Do not clip Clipping is not performed.
Clip top Clips the top of the Clipping Standard Plane so only particles below that surface are displayed.
Clip bottom Clips the bottom of the Clipping Standard Plane so only particles above that surface are displayed.
Height of Clipping Plane

If a direction for clipping has been selected in the clipping settings, the location of the clipping surface is specified in world y-coordinates.

Info

Clipped particles do not loop. As a result, the number of visible particles will decrease.
To cut wasteful particle calculations, configure the setting to lay within the clipping range.
