Material Property Window: General Page

This page is used to check and edit the general properties of the selected material.

A Specifies the visibility of shapes bound to the material.
B Specifies whether to enable material compression by the intermediate file optimizer for the selected material.
C

Displays optimized shader compiling information (see Note 1).

Use to check statistics for the selected variation number.
Note

Note 1: This information can be displayed during shader conversion when the connection target is NX and optimized shaders are enabled.

To display this information, the shader compilation information setting must be enabled in user settings.
