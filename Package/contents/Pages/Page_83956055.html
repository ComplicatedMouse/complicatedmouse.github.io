Rasterizer State

Rasterizer State

The rasterizer state collectively expresses several states primarily for the fixed features for rasterizing in the pipeline.

Information

Use nn::gfx::RasterizerStateInfo to define the rasterizer state information.

Fill Mode

Set the polygon fill mode using the SetFillMode function.

Front Face Settings

To specify whether the polygons that are front-facing have clockwise or counterclockwise winding, use the SetFrontFace function.

Culling Mode

To set the culling mode, use the SetCullMode function.

Primitive Topology Types

To set the primitive topology type, use the SetPrimitiveTopologyType function.

Rasterizer

To enable or disable the rasterizer stage, use the SetRasterEnabled function.

Multisampling

To enable or disable multisampling, use the SetMultisampleEnabled function.

Depth Clipping

To enable or disable clipping by depth, use the SetDepthClipEnabled function.

Scissor Test

To enable or disable the scissor test, use the SetScissorEnabled function.
This setting must match the scissor test parameter of the view port scissor state.

Depth Bias

Several different functions set the value to add to the depth.
When you set the value using SetSlopeScaledDepthBias, the value is scaled by the maximum depth slope before being added.
When you set the value using SetDepthBias, the value is scaled by the smallest number expressible for the depth format before being added.
When you set the value using SetDepthBiasClamp, the maximum value is set.

Conservative Rasterization

Set the mode for conservative rasterization using SetConservativeRasterizationMode .

Multisample State

Edit the multisample state information using the EditMultisampleStateInfo function.

Number of Samples

Set the number of samples for multisampling using the SetSampleCount function.

Sample Mask

Set the mask for each sample when multisampling using the SetSampleMask function.

Converting an Alpha Value to a Coverage Value

Use the SetAlphaToCoverageEnabled function to set whether to convert alpha values to coverage values when multisampling.

Object

Use nn::gfx::RasterizerState to define the rasterizer state object.

Initialization and Destruction

Rasterizer states are initialized and finalized according to the rules for objects.

Setting a Rasterizer State

When you are using an execution model that uses a pipeline, information about the rasterizer state is included in the pipeline information.

When you are using an execution model that does not use a pipeline, set the rasterizer state by calling the SetRasterizerState function of the nn::gfx::CommandBuffer class.
