Color

With this tab, you can use the viewer to check the settings of the Color feature of the Emitter node that can be controlled by the game application.

Apply Scale to Color Settings

The Apply Scale to Color Settings group () is a set of features for multiplying the color of all emitter shapes in the emitter set by a set value.
This group has the following settings.

Multiply by the Render Color Multiplies the color and alpha of all particles in the emitter set by the specified value.
Multiply by the Emitter Color Multiplies the value set for Global Multiplier Color for all emitters in the emitter set with this value.
