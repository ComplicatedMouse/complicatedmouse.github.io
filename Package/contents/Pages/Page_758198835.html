Keyboard Shortcuts

The following is a list of keyboard shortcuts supported by the profiler.

Actions Available Anywhere

Some actions within the profiler have keyboard shortcuts associated with them. When a keyboard shortcut is available, it is shown in the upper right of the button's tooltip. Below is a list of supported keyboard shortcuts.

Shortcut Description
Ctrl + O Opens a profile from disk. The same as clicking the Open button on the Home tab.
Ctrl + S Saves the current profile. The same as clicking the Save button on the Home tab.
F1 Opens the manual. The same as clicking the Manual button on the Home tab.
F5

If no profiling is active: Starts taking a sampled profile. The same as clicking the Start button on the Sampled Profile tab.

If profiling is active: Stops taking the current profile. The same as clicking the Stop button on either the Sampled Profile tab or the Instrumented Profile tab.
Shift + F5 Starts taking an instrumented profile. The same as clicking the Start button on the Instrumented Profile tab.

Actions Available on Specific Tabs

Some tabs have different actions depending on the modifier keys on the keyboard being pressed. Below is a list of those actions.

Left Window Group

Functions tab, Threads tab, Instrumented tab, Counters tab, Gfx Groups tab, and Shaders tab

Shortcut Description
Shift + Left Click Select the range of items between this click and the previous click.
Alt + Left Click Deselect all items and select only the item clicked on.
Esc When editing the Filter, clears the current text.

Right Window Group

Overview Tab, Screenshot (click screenshot to open window)

Shortcut Description
+ Zoom into the screenshot.
- Zoom out of the screenshot.

Call Tree tab, Code Coverage tab

Shortcut Description
Esc When editing Find or Filter, clears the current text.

Timeline

Shortcut Description
Ctrl + Mouse Wheel Scroll the graph up and down.
Ctrl + Left Click

If the mouse mode is Scroll Graph: Switches to Measure Time mouse mode while Ctrl is pressed.

If the mouse mode is Select Frames, Select Time, or Measure Time: Switches to Scroll Graph mouse mode while Ctrl is pressed.
Alt + Left Click If the mouse mode is Select Frames or Select Time: Switches to deselecting while Alt is pressed.
+, Up Arrow Key Zoom into the graph.
-, Down Arrow Key Zoom out of the graph.
Left Arrow Key Page the graph left.
Right Arrow Key Page the graph right.
