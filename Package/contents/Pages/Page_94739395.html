Longitude

This section describes the Longitude setting for a Sphere emitter shape.

Overview

The Longitude feature opens a spherical emitter in a fan shape sideways in the z-axis direction.
The emission area can be made a half-spherical dome while maintaining the particle density.

The following overview describes the settings ().
To display the Longitude () settings, set Method of Arc Opening to Longitude. The group has the following settings.

Arc Width (Angle)

Sets the arc width as an angle with the starting point at the center.

Arc Starting Angle

Sets the arc starting point with an angle. For the default value, the z-coordinate in the positive direction serves as the starting place.

Random Arc Starting Angle

You can enable or disable the randomization of the arc starting angle by selecting or clearing the check box ().
