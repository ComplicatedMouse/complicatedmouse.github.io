A Menu Bar

The menu bar contains the following items. Right-click an item to display a shortcut menu.

File The File menu contains effect creation and file import/export operations.
Edit
For Undo () and Redo () editing tasks and operations on nodes.
Display Performs operations for window display.
Preview
For operations having to do with connecting to the Viewer ().
Tool
For operations having to do with EffectMaker options ().
Help
Displays Help () and information for EffectMaker.


B Toolbar

Frequently used menu items are displayed as buttons. The following describes the toolbar.

 Open File > Open Opens a file.
Overwrite and Save All File > Overwrite and Save All
Saves all files.
When a file is saved, the symbol indicating that the file data has been edited () disappears in the property window.
Export All File > Export All
Files selected in the dialog box are exported.
Unlike when a file is saved, the symbol indicating that the file data has been edited () does not disappear in the property window when the file is exported.
Undo Edit > Undo Restores to a pre-edited state.
Redo Edit > Redo Reapplies a single edit that was undone.
Property Window Display > Property Window The Property window is composed of a UI for operations on nodes and for previewing, and a UI for configuring node features.
Effect Browser View > Effect Browser Displays the Effect Browser with its browse and search features specialized for EffectMaker files.
Curve Editor Window View > Curve Editor Displays the Curve Editor, which provides a graphical way to create and edit emitter time animations, with time on the horizontal axis and values on the vertical axis.
History Window View > History Displays the History Window, which lists the work procedures as a history.
Search Window View > Search Window Displays the Search window for searching for features in effect files in specified folders, and batch-converting the settings.
Platform - Selects the platform being developed.
Disconnected State > Connect Preview > Connect to Viewer Starts communication with the Viewer
Connected State > Disconnect Preview > Disconnect from Viewer Closes communication with the Viewer
Viewer to Connect to - Viewers available on the platform are displayed as presets, so select the viewer to connect to.
Starting the Viewer

Preview > Launch PC Viewer

Preview > Launch Hardware Viewer

 The viewer starts with the content in the Platform to Connect to and the Viewer to Connect to.
This option cannot be selected if there is no viewer set in the specified location.

Capturing Thumbnails Preview > Capture Thumbnail
The feature is used to capture thumbnails of emitter set files and model files shown in the viewer.

Captured images are saved as PNG files in a folder named .NwThumbs. The folder is created automatically in the directory where emitter set files are stored.
The saved PNG files are displayed as thumbnails of the files in the Effect Browser.

C Docking Area

Property Window (), Effect Browser (), Curve Editor Window (), History Window ()
These windows can all be displayed in this area or be removed to be independent windows.
You can modify the window configuration according to what you need to do.

You can place windows inside the docking area.
Right-click a window to select it and display the icons in the docking area.
Drag a window to one of the icons to move that window to that place in the work area.

Snaps the window to the left of the docking area.

You can place a window inside the window you want to dock.
Select a window and move it to the window you want to dock, and the icon () will appear.
Drag the window to one of the icons to move that window to that place.

Windows are arranged on top of one another.

Selects all cells. Selects all messages.
