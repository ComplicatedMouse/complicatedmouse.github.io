NintendoWare Audio News

This page introduces the changes in the NintendoWare audio development environment for NintendoSDK 1.0.0.

Spy

Displaying the True Peak of Waveform Data

The True Peak audio is now displayed in the Loudness Meter panel of Spy.exe.

  • It is possible to switch between displaying the True Peak and displaying the sample peak.

For more information, see Spy/Spy.exe/Audio Loudness Meter Panel.
