3D News

This section introduces new features added to the 3D module.

  • 3DEditor
    • Added a shortcut to RuntimeUserScript.
    • Added the ability to disable the functionality that prohibits saving automatically loaded parent material files.
    • Improved the default path in the save file dialog box for new files.
    • Added the ability to configure bone_anim user data.
  • Photoshop Plug-In
    • You can now use Pre/Post-Export Script to get the layer comp name corresponding to each output file.
    • Photoshop Plug-In for Photoshop 2020 (Version 21) supported
    • Ended the support for Photoshop CC 2015.5.
  • Maya Plug-In
    • Improved the process for calculating the slopes of keys when creating animation curves from baked data.

Added a shortcut to RuntimeUserScript.

Added the ability to disable the functionality that prohibits saving automatically loaded parent material files.

You can now disable the functionality that does not allow saving parent material files that were loaded automatically into 3DEditor. (You can now save the parent material files.)

This functionality can be disabled or enabled from Team Settings.

Maya Plug-In

Improved the process for calculating the slopes of keys when creating animation curves from baked data.

The process for calculating the slopes of keys when creating animation curves from baked data has been improved in the Maya plug-in.
This improvement reduces the margin of error when the animation is played at a higher frame precision than that at export time.
