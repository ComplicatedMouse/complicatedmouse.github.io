Logs Tab

The  Logs  tab displays all of the user logged counters that were sampled during a profile.

Logs Tab Toolbar

The following controls are in the Logs tab toolbar.

Deselect All Button

To quickly deselect all selected items, click on the Deselect All button.

Select Similar Toggle Button

When the Select Similar button is enabled, selecting or deselecting an item causes similarly named items to be selected or deselected.

Select Similar allows for selecting similar counters across threads. For example, if a user logged counter was profiled, a single click on a specific counter on a thread will select (or deselect) that counter on all profiled threads.

Stats Toggle Button

The Stats toggle button enables and disables viewing the Min, Max, Average, and Median columns on the Logs tab. When the button is enabled, these four columns will be visible and can also be used for sorting. When the button is disabled, the four columns are hidden and the sorting will revert to the visible columns.

Copy Button

The Copy button will copy to the clipboard whatever is currently shown using the current sorting and filtering. You can then paste the content into an email, document, or spreadsheet. Because the copied content uses tab delimiters, it will retain the column organization when pasted into a spreadsheet.

Thread Filtering Drop Down

Click on the Threads drop-down in the toolbar to select between All Threads and an individual thread. Only threads that were profiled will be available in the Threads drop-down. The default selection is All Threads and each new profile taken or loaded will reset the Threads drop-down selection to All Threads.

Selected Only Toggle Button

When the Selected Only button is enabled, only selected items will be visible. If there are no counters selected, then the button will be disabled and all counters will be visible.

Filtering Counters

Counters can be filtered using the Filter text box at the bottom of the Logs tab. This behaves similar to the Filter text box on the Functions tab.
