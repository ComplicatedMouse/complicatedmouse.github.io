Particle

This tab is for configuring settings for particles.

Particle Lifespan

Particle Lifespan () is a group of features for configuring the settings for the lifespan of the particle, from its creation until its destruction.
This group has the following settings.

Infinite Lifespan

Specifies an infinite lifespan for the particle.
Lifespan Randomness (%) Randomizes the lifespans of individual particles.

Particle Settings

Particle Settings () is a group of features that sets particle type and configures the axes for controlling positioning.
This group has the following settings.

Particle Shape Sets the shape of the particle.
Trimming This feature is used to trim the region in which particles are not rendered.
Particle Type

Sets the orientation of the particle.
Particle Center Sets the center position for particle scaling and rotation.
Particle Offset Adjusts the overlap of particles with objects in the game.
Z-Fighting Prevention This feature reduces the flickering effect that tends to occur when a particle is drawn at the same position as another polygon.

Flux Settings

Flux Settings () is a group of features that animates particle setting elements using a waveform.
This group has the following settings.

Apply Flux Applies flux to the particle settings such as alpha or scaling.
