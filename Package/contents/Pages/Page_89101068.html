Tool Reference

Overview

This page describes the PC tools provided by the AUDIO library.

Tool List

Tool Overview
AdpcmEncoder PC tool that encodes uncompressed audio data, including WAV and AIFF files, into Nintendo's proprietary ADPCM format.
AudioRedirector Tool for sending the audio playing on the host PC to the development hardware for playback.
SystemAudioMonitor Tool for getting and monitoring the system's final audio output.
