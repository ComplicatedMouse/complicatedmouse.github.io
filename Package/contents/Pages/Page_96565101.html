Collision Plane Y Coordinate (±)

This section describes the Collision Plane Y Coordinate (±) feature, which can be used to set the position of a collision.

Overview

Collision Plane Y Coordinate (±) enables you to offset the collision plane in the y-axis direction.
Use the Apply in world coordinates setting to change the base location for the offset.
The following diagram shows the differences between enabling and disabling Apply in world coordinates.

Apply in World Coordinates enabled Apply in World Coordinates disabled

Places the collision in the center of the emitter.

Offsets emitter coordinate system by -10 in the y direction.

Places the collision in the center of the world.

Offsets world coordinate system by -10 in the y direction.
Value for Collision Plane Y Coordinate (±) is -10, with ― emitter, ― collision, and ― world.
