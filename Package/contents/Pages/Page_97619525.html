Bone to Constrain

This section describes the Bone to Constrain feature in Model Links, which allows you to select which bone you want to emit the effect from from a list.

Overview

The Bone to Constrain feature allows you to select, from the bones in the model selected in Model Links, the bone you want to emit the effect from.
In addition, you can also select the bone from the list of in-game model bones by using the In-game editing feature.
