Enabling Outlines

This section describes the Enable Outlines feature that applies an outline to borders of a pane.

Overview

The Enable Outlines feature applies an outline to the borders of a pane.
The following is an overview of the settings.

Size

The following images compare what happens when the Size value is changed for the border.
The larger the value, the wider the glowing area.

Size = 3 (Default) Size = 10

Rendering Mode

The following images compare what happens when the Rendering Mode is changed.
This feature configures the method of blending the color in the area of the pane outline and background color.

Normal Multiplicative Additive (Default) Subtractive

Blend translucently. Blend multiplicatively. Blend additively. Blend subtractively.
