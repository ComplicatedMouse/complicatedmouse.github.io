Tool Reference

Overview

The following section describes the graphic tools provided with NVN.

It starts by describing information that is common to all of the NVN tools.

For more information about a specific tool, see the page for that tool.

Location

For more information about the tool placement, see the page for each tool.

How the Tool Is Provided

Tools are provided in either the library for development hardware, DLL, or EXE formats.

For the provision format of the library for development hardware, see the following pages.

  • How the Library Files Are Provided and How to Use Them (NX Add-On)

Tools in the DLL format are intended to be used from programs running on Windows.

32-bit Tools

The 32-bit EXE and DLL files have a 32 appended to the end of the tool name to differentiate them from the 64-bit versions.
If a tool does not follow these rules for the 32-bit version, it means the tool is a 64-bit version.
