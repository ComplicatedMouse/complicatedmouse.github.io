Normal Map

EffectMaker particles can use normal mapping by the game application, using custom shaders to implement shaders.
This page describes normal mapping.

Below is the support status for normal mapping in particles.

Particle
Stripe Particle		 Normal and tangent information is not held as data, but is generated at runtime.
Primitive Particles

Normal and tangent information is held as data.

When using normal mapping, register a model with normal and tangent information output from the DCC tool.
Related Links You can check whether the registered model has normal and tangent information using primitive shape particles.
