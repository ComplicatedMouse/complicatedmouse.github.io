Operating Environment

NintendoSDK NX Add-On Operating Environment

The NintendoSDK NX Add-On has been confirmed to run under the following environments.
If you intend to use the package in an environment not listed in the following table, make sure to test thoroughly before proceeding or prepare offline installers and other resources in advance.
(Be especially careful when attempting to use a different version of MSVC compiler in Visual Studio.)

Operating System
  • Microsoft Windows 10 Pro (64-bit) English or Japanese version
Warning

To install and use NintendoSDK, you must use an account with administrator privileges.

Warning

Operation is not ensured when the system locale of the operating system is changed to anything other than the default value.
.NET Framework
  • Microsoft .NET Framework 4.7.2 or later
Info

If not already installed, download and install from the following link.

https://support.microsoft.com/ja-jp/help/4054531/microsoft-net-framework-4-7-2-web-installer-for-windows

.NET Core SDK

  • .NET Core SDK 3.1.415
Info

If not installed, use the installer located in the following path.

  • NintendoSDK\Installers\DotNetSdk\v3.1\dotnet-sdk-3.1.415-win-x64.exe
Visual Studio

  • Microsoft Visual Studio 2017 Professional Edition version 15.9.30
    • English language pack required.
  • Microsoft Visual Studio 2019 Professional Edition version 16.11.6
    • English language pack required.
Info

Microsoft Visual Studio 2017 can be used in NintendoSDK NX Add-On 4.4.0 and later.
Operation has been confirmed for the following combinations of versions of Microsoft Visual Studio 2017 (VS2017) and the earliest version/release date of NintendoSDK NX Add-On (NX).

  • VS2017 15.9.11: NX 9.0.0 (2019-05-24)
  • VS2017 15.9.12: NX 8.2.1 (2019-06-06)
  • VS2017 15.9.14: NX 9.3.0 (2019-09-10)
  • VS2017 15.9.16: NX 10.0.0 (2019-10-31)
  • VS2017 15.9.20: NX 9.6.0 (2020-03-06)
  • VS2017 15.9.21: NX 10.4.0 (2020-04-07)
  • VS2017 15.9.22: NX 11.0.0 (2020-06-04)
  • VS2017 15.9.24: NX 10.6.0 (2020-07-14)
  • VS2017 15.9.25: NX 11.2.0 (2020-09-09)
  • VS2017 15.9.27: NX 11.3.0 (2020-10-08)
  • VS2017 15.9.30: NX 12.1.0
Info

Required if building and debugging an NX application.
Not required if building but not debugging an NX application.

Info

For important information about using Visual Studio 2019, see here.
Windows SDK
  • Windows SDK 10.0.15063.0 (when using Visual Studio 2017)
  • Windows SDK 10.0.19041.0 (when using Visual Studio 2019)
Info

Required to build and debug the application in a PC environment.
Not necessary if not building and debugging the application.

Video Card
  • Supports NVIDIA Maxwell, first generation or later (NX hardware is Maxwell second generation).
Info

Needed when using NVN in a PC environment.

Depending on the GPU architecture, there may be some functionality limitations or it may not be recommended.

Some video card model numbers are not supported.

For more information, see NVN Description.
Development Hardware
  • SDEV, SDEV-D (When connecting the SDEV(-D) to a PC, we recommend either connecting it directly via a network interface card, or via a local area network.)
  • EDEV, EDEV-D
  • HDEV (For more information about the supported features on the HDEV, see here.)
  • ADEV
Warning

You cannot use a prototype.

Info

Development hardware is unnecessary if developing only in the PC environment.

It is required to run NX applications on development hardware.

Info

EDEV-D and ADEV can be used in the same way as EDEV for various functionality unless specifically described otherwise. If there are differences in functionality, the differences are described in the applicable section.

EDEV-D is available in FW 10.0.0-6.0 and later.

ADEV is available in FW 13.0.0-3.0 and later.

SDEV-D can be used in the same way as SDEV for various functionality unless specifically described otherwise. If there are differences in functionality, the differences are described in the applicable section.

SDEV-D is available in FW 12.0.2-1.0 and later.

Controller
  • Joy-Con
  • Switch Pro Controller
Info

For the controller firmware revision history, see Controller Firmware Release Notes.

Info

A PC requires Bluetooth when using either the Joy-Con or a Nintendo Switch Pro Controller in a PC environment.

The Planex BT-Micro4 is the Bluetooth dongle that has been tested to work using NintendoSDK NX Add-On.

Software Versions at Times of SDK Releases

The firmware versions required for testing, described in Master ROM Submission Information on the Nintendo Developer Portal, are backward compatible with ROMs that use an NX Add-On version that can be used to submit master ROMs. (However, this is not necessarily the case when using it for an unexpected purpose or in an unexpected way.)

ROMs that use versions not corresponding to the above are compatible only with the combinations listed in the table in Software Versions at Times of SDK Releases.
