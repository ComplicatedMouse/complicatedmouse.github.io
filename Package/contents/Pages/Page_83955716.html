About NintendoWare

NintendoWare is a collection of libraries and tools used to develop graphics and sounds for game software.
It provides the basic tools and libraries used by most games so that developers can focus on the content of the games themselves.

 Set of modules to handle 3D models, UIs and fonts, and 3D effects.
NintendoWare Sound is a development environment for handling sound effects (SE), background music (BGM) and other game sounds.

Modules Beyond Graphics and Audio

Module
spy Spy is a set of libraries and tools for collecting application information and making it viewable on a PC tool (Spy.exe) timeline.

NintendoWare Characteristics

Performance

NintendoWare is optimized for Nintendo platforms.
Use NintendoWare to get the maximum performance out of the hardware.

Easy to Include

NintendoWare is provided as a collection of individual features that can be included in a variety of genres and workflows. Game developers can combine the necessary features to create their own game engine or embed them into an existing workflow.

Stability

NintendoWare is used for development both inside and outside of Nintendo and is always kept in working order. Backward compatibility is considered when adding new features or extending existing ones.

Expandability

New features are added to NintendoWare based on requests from first-party and third-party developers.

Source Code Availability

The source code for NintendoWare libraries is available to game developers. Developers can check the implementation for debugging and extensibility.

Multi-Platform Support

In general, NintendoWare can be compiled for any platform without any changes to the application code.
