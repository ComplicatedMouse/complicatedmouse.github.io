Picture Panes

Picture panes in the property window are used to edit picture pane information in LayoutEditor.

Shape

Sets the shape of the picture pane.

You can select rectangle, rounded rectangle, or circle in the property window settings.
Each of these shapes can be adjusted using parameters like the number of mesh divisions.

Info

The settings items in the property window change depending on which shape has been selected.

Shape Set the shape of the picture pane from the drop-down menu.
Rectangle

Set the picture pane to a rectangular shape.

The settings in the property window change to those for a rectangular shape.
Rounded rectangle

Set the picture pane to a rounded rectangle shape.

The settings in the property window change to those for a rounded rectangular shape.
Circle

Set the picture pane to a circular shape.

The settings in the property window change to those for a circular shape.

Note
  • If a shape other than "square" is used, the polygon of the shape is automatically generated based on the Size of the Base Pane. However,
    because it is not regenerated after the mesh is generated one time during the initialization of layout data from load issues in the runtime, even if an animation curve is set for Size, it is not possible to reproduce the intended appearance in the runtime.
  • To use settings other than Rectangle in the runtime, the following settings are required in the program.
    • In the GpuBuffer region for the constant buffer, add nn::gfx::GpuAccess_VertexBuffer and nn::gfx::GpuAccess_IndexBuffer in addition to nn::gfx::GpuAccess_ConstantBuffer to the GpuAccessFlag flag to be set to GpuBuffer.
    • To use the GfxUtil features, link libnn_gfxUtil.
