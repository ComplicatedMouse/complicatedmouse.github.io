Scale Fade

This section describes the Scale Fade feature for making particles gradually shrink and disappear when the Fade command is executed.

Overview

The Scale Fadeout feature can be used to stop the emission of particles from emitters and to fade out and kill particles that have already been omitted when the game developer has executed a Fade command.
Particles gradually extinguish over the course of the configured Fadeout Duration.
The following overview describes the settings ().

Scale Fade

The Scale Fadeout is enabled or disable using the check box.
