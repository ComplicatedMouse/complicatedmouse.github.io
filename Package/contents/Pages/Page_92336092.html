What Is the NintendoWare Audio Development Environment (Audio Toolkit)?

NintendoWare Audio is a development environment for handling sound effects (SE), background music (BGM), and other game sounds as an upper layer of nn::audio.

Audio Toolkit includes the following elements.

nn::atk

A class library for playing sounds (SE, BGM, and other sounds) in games.

For more information about the differences between nn::atk and nw:snd, see Changes to NintendoWare Audio.


Note: nw:snd is the NintendoWare audio library for the 3DS and Wii U generation.
SoundMaker A GUI tool for authoring sound data handled in games.
AtkPlayer This tool is used to preview the sound data edited in SoundMaker. It can be used to adjust and preview various parameters.
Various Command-Line Tools A group of command-line tools for converting the sound resource files edited with tools like SoundMaker into binary resource files that can be handled by the runtime library.
