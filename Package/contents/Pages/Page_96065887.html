Scale Animation

This section describes the Scale Animation feature that sets the animation for particle scale using Curve Editor.

Overview

Scale Animation is a feature that can produce scale animation for a particle over the range of a particle's lifetime, using the Curve Editor.
For example, if you created an animation with Curve Editor values ranging from 100% to 0%, you can gradually extinguish particles as a whole from the standard value.
The following overview describes the settings ().

Scale Animation

You can graphically create and edit animations on a graph where the horizontal axis is the percentage of the particle lifetime (particle time; where 100% is the full particle lifetime) and the vertical axis is a standard value that represents the base size of the particle.
