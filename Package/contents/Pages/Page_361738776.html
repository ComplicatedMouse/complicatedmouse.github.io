Editing Guides

This section describes the shortcut menu that appears when you right-click a guide.

Edit Guide Shows the Edit Guide dialog box.
Default Changes the color of the guide to the default color.
Black Changes the color of the guide to black.
Light Gray Changes the color of the guide to light gray.
Light Green Changes the color of the guide to light green.
Light Blue Changes the color of the guide to light blue.

Edit Guide Dialog Box

The following table describes the dialog box that appears when you select Edit Guide from the shortcut menu.

A Method for setting guide coordinates

Choose from the following combinations of settings used to position guides.
Editable items change depending on the selected combination.

  • Direction and position
  • Position and angle
  • 2 points
B

Direction

 Select either horizontal or vertical for the direction of the guide.
C Position Sets the x- and y-coordinates used to position the guide.
D Angle Sets the angle to rotate the guide direction, counterclockwise with horizontal as 0 degrees.
E Point 1, Point 2 Sets the x- and y-coordinates of two points that the guide passes through.
