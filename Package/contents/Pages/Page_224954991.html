[User Settings] Emitters

Emitter

Configures environment settings for using emitter set files.

Display warning dialog when saving emitter sets with no configured textures

Displays or hides a warning when an emitter set (ESET) file opens, and no texture has been added to an emitter or child.
You can intentionally suppress the warnings, for example when you are not using textures.

Show warning symbol for emitters with texture settings that differ from the Linear Edit Mode settings

Displays or hides a warning symbol for the emitter when the texture (FTXB) file added to an emitter or child differs from that in the Linear Edit Mode settings.
For example, you can hide the warning symbol by disabling the setting when intentionally using non-linear texture files using the Get Square of RGB feature in a linear environment.
