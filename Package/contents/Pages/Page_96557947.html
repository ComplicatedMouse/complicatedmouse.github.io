Direction of Gravity

Direction

This section describes direction, the functionality used to set a gravitational direction for emitted particles.

Overview

Direction is a feature for determining the direction in which the force configured using gravity is applied. You can use any direction by using the x, y, and z axes.
The following overview describes the settings ().

Direction

The direction that particles move in when emitted is configured using x, y, andz.
You can set this value in a range from –1 to +1, using either the radial wheel () or the text box field ().
