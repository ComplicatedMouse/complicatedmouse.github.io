Capture First Frame Only

This section describes the Capture First Frame Only feature, which uses the rendering result of only the first frame when rendering drop shadow effects.

Overview

The Capture First Frame Only feature can reduce the processing load by applying the effect to only the first rendering result for the pane.
Even if the rendering results change for each frame due to material animation, only the rendering result of the first frame is subject to the effect.
