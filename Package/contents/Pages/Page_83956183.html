Ui2dDemo

The Ui2dDemo demo screen. It runs from Ui2dDemo CapturePane and is for developers.


The following features of the capture pane can be listed in this demo.

  • DynamicCapture: Disables the Capture First Frame Only feature, and captures all frames.
  • CaptureFirstFrameOnly: Uses the Capture First Frame Only feature.
  • UserFilterShader: Applies the blur filter in the program, including to the model rendered in the framebuffer.
  • TextureScale: Forcibly applies texture scale, lowering the resolution.
  • FrameBufferCaptureTexture: Uses framebuffer capture textures in the picture pane.
  • ClearColor(Color,Alpha): Uses a cutout texture for the background, specifying the clear color and alpha for the cutout part.

The example on the left side of the red line illustrates adding a text box pane () or a picture pane () to the capture pane hierarchy ().

The example on the right side of the red line illustrates using a texture created from the capture pane in a picture pane ().


This method of use is as follows.

A The text is filled with a texure pattern using the advanced combiner settings in the picture pane.

B The mask texture is registered in the picture pane and the texture coordinates are animated, making the text disappear from the left and right.

C The results of additive blending by fragment operations are captured and masked in the Picture Pane.

Blend:

ColorAndHighlight:

This effect changes the color of a life gauge as the gauge increases or decreases.

Normally, if you were to create the life gauge as a single pane and then apply a white highlight, both the highlight and the color would have to change.

This effect makes it possible to achieve this change using a single pane instead of two panes.

It works by mixing in the color you want to add to a constant color and then multiplicatively combining that color with a grayscale texture to add a white texture.

DistanceField:　

This sample uses the advanced combiner mode for compensating for the shortcomings of distance fields.


Used Textures

Left:Right:


Left: A characteristic of distance fields is that the ◯ parts of the triangle are missing.

Right: Using the following method, make the ◯ parts of the triangle sharp.

  1. Prepare a three-sided triangular image for each color channel of the texture.
    Make the channels of the texture image R, G, B = , , .
  2. Register the prepared textures in LayoutEditor.
  3. Use the advanced combiner mode to multiply each channel image by a scaling factor and construct.
  4. Finally, blend using the last step of the advanced combiner mode.

Flashing:

Makes buttons slightly whiter when selected.

This behavior can be used on individual panes.

This behavior can be useful when you want to achieve small color changes where standard monochrome color interpolation does not work well.

This also reduces the amount of layout work required in comparison with using a separate second pane.

Mask:

This effect can be used to create a clear distinction between metallic areas and other areas or when you want just a portion of an object to have specular qualities.

This effect can be achieved even when you are not using two separate panes.

This effect scrolls through two textures.

A texture of a different size is used for portions of the texture that overlap with the mask.
