Plug-In for Setting Texture Pattern Animations

When the texture pattern animation is set, the only textures output by default are those specified by the Image Name attribute of the file node, and those whose corresponding key value has been set for the Frame Extension attribute. Use of this plug-in allows textures corresponding to any particular Frame Extension to be forcibly output.

If expressions or animation layers are connected to the Frame Extension attribute, of if keys are not set directly, you must use this plug-in to configure the list of textures to output.

Info

If use of the image sequence for a file node (Use Image Sequence) is enabled, the configured texture is not output by this plug-in even if no animation has been set for the image number (Frame Extension attribute).

Usage

Select Nintendo > Set Texture Pattern to open the window.

Selected file nodes are displayed in the Selected File Node column.

Setting Method

Selecting just one file node (or material node) will display the current setting for that selected node under Set.

Enter a list of the image numbers (Frame Extensions) to export to the intermediate file, and then press ENTER.
An error will result at this time if the Use Frame Extension attribute of the file node is not selected.

Use the following list format:

1,2,3

To specify the texture numbers to export, delimit each number with a comma (,).

Assume by way of example that the Image Name in the file node is name.1.tga, so you have these three textures.

　"name.1.tga"
　"name.2.tga"
　"name.3.tga"

These three textures are forcibly output, even though keys are not set for the Frame Extension attribute. A warning is displayed at the time of export if the textures that correspond to the specified numbers do not exist. Negative numbers cannot be specified.

If numbers are continuous, then rather than writing:

1,2,3,4,5

You can specify the same range as follows:

1-5

A hyphen (-) connects numbers, so consider the following specification:

3-6,10

This is equivalent to writing 3,4,5,6,10.

It is also possible to delimit texture numbers using an ASCII space rather than a comma (,).

The following formatting mistakes result in an error:

1-3-5 (two hyphens are connected to the same number)
-3 (starts with a hyphen)
5- (ends with a hyphen)

Writing "01,02" has the same effect as writing "1,2."
If files that have the names name.1.tga and name.01.tga both exist, the texture without a 0 in the filename is always the only texture exported.

You can select a different node by selecting another node without closing the window.
