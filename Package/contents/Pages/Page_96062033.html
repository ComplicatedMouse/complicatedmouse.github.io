Decals

This section describes the Decals feature, which changes the alpha value of all areas except the edges of particles that overlap with a model.

Overview

Decals writes particle pixels within a configured range from a model and particle overlap.
It makes it look as if particles are being projected onto the model.
Particles outside of the range do not appear, but there is a rendering load.

Decals

This feature creates effects as if pasting particles to models.
The Decals feature is enabled or disabled using the check box.
Decals () is displayed when the setting is enabled.

The following overview describes the settings ().

Range where Alpha is '1'

Configured range where the particle overlaps the model, defining where particle pixels are written to the model.
