Custom Shaders

You can add programmable shaders, created with game applications, and review the content of added shaders.
Game developers will likely be the ones to use this setting.

Custom Shader Definition File Path

The Custom shader definition file path group () is used to register shader definition files used in a game project.
You can use files in which shaders unique to game applications are written.

Settings to add shader compiler code.

Settings to add shader compiler code, , displays the content of added shader definition files.

Additional Include Directories

Any include paths available within custom shaders as denoted within the Project Settings file, appear here.

Additional preprocessor definitions.

Any preprocessoor definitions provided additionally during shader compilation, as denoted in the Project Settings file, appear here.

Enabling Preprocessing

Displays the setting that indicates whether to pass source code that was preprocessed during shader compilation, as denoted in the Project Settings file.

Custom Shader Settings

The Custom shader settings group () is used to display the content of shader definition files used in a game project.
