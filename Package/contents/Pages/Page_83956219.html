Main Window

About The Main Window

The window that appears immediately after SoundMaker starts is called the main window.
This window is the working area for SoundMaker, where you can switch among panes and tabs for various tasks.



Status Bar

A status bar is located at the bottom of the main window.

The status bar shows the file path that contains the project file that corresponds to the item selected in SoundMaker, and the waveform file referenced by that item.

The following locations are referred to in this path.

　　Project Tree
　　　　- Project node
　　　　- Sound Set node

　　Sound Set Tab
　　　　- Stream Sound List
　　　　- Wave Sound List
　　　　- Sequence Sound List
　　　　- Bank List

　　Bank Tab
　　　　- Instrument List
　　　　- Sample Map
　　　　- Percussion List

Project Tree

What Is the Project Tree?

Project Tree is the pane that first displays to the left inside the window when a sound project is loaded to SoundMaker.

Inside Project Tree, the following eight base nodes are displayed.

  • Stream sounds
  • Wave sound sets
  • Sequence sounds
  • Sequence Sound Set
  • Bank
  • Waveform archives
  • Player
  • Group

These nodes represent respective sound sets.

Displaying the Project Tree

The Project Tree is displayed automatically when a sound project is loaded into SoundMaker.

Sound Set Tab

Sound Set Tab

This tab displays sound sets on the right side of the main window.
A sound set comprises eight types of elements. To display and edit them, double-click the corresponding nodes in the Project Tree (or single-click the nodes if the Sound Set tab is open).

Role of the Sound Set Tab

This tab is used to edit sound set files (FSST files).
The Sound Set tab is at the center of the tasks that you perform in SoundMaker. On this tab, you create sounds, players, groups, and other items.

The Sound Set tab opens a tab for each sound set, and the content that displays in the tab can be switched inside the sound sets.
The tab title shows the list type and the sound set name.

　　The following notation is used for the list types.
　　　STRM = Stream sound
　　　WSDSET = Wave sound set
　　　SEQ = Sequence sound
　　　SEQSET = Sequence sound set
　　　BANK = Bank
　　　WARC = Waveform archive
　　　PLAYER = Player
　　　GROUP = Group

Bank Tab

About Bank Tab

The Bank tab is the tab used to edit sound data for sequence sounds.

Displaying the Bank Tab

The Bank tab can be displayed using the following methods.

  • Select the Bank node from the project tree, and then double-click an index cell from the Bank List in the Sound Set tab.
  • Right-click the selected Bank item from the Bank List, and then select Open Referenced File from the shortcut menu.
  • Expand the Bank node in the project tree and then double-click a child node.

The Bank tab opens a tab for each bank file, but the content displayed in the tabs is always fixed (unlike the content of the Sound Set tab).

Role of the Bank Tab

Edit Bank files (.fbnk).
Add instruments to bank files and specify program numbers. Create instrument regions and specify envelopes.

Sub-Window

What Is a Sub-Window?

A sub-window is a window with supplementary features in windows such as the Preview Player window, the Parameters window, and the Search Results window.

The following list contains the different kinds of sub-windows:

  • Parameters Panel
  • Preview Player
  • Preview Bank
  • Sequence Variables
  • In-Game Editing
  • Conversion Log
  • Bookmarks
  • Revision History
  • Waveform Reference Count

Displaying Sub-Windows

You can open a sub-window from the View menu in the main window, or from a button on the toolbar.

There are two ways to display sub-windows: docked to the main frame inside the main window (Docking) or floating outside the main frame (Floating).


When docked, Preview Player, Preview Bank, and Conversion Log appear at the bottom of the main window, and Sequence Variables and Parameters appear to the right of the Sound Set and Bank tabs.

You can make a sub-window float, dock, or hide by selecting Floating , Docking, or Hide from the title bar of the sub-window, or from the shortcut menu of the tab portion.
You can also toggle between Floating and Docking by double-clicking the title bar or tab.


To minimize a docked item so that only the tab is displayed, click a button on the right side of the title bar, or double-click the border.
Double-clicking the tab portion while the display is minimized does not switch between floating and docking displays, but restores the original window size display.
