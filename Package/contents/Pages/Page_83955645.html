Convergence

This section describes the Convergence feature, which brings all particles together to the specified position.

Overview

The Convergence field brings all particles together at the specified position.
Use this feature to create effects like a black hole.

Convergence

Convergence is a group () of features for configuring field settings.

Convergent Force

This feature sets the force at which particles are moved toward the position of convergence.

Position follows emitter

Selects whether the convergence position is followed when an emitter moves.
Position Determines whether to make the position of convergence follow the emitter when it moves.

Example Configurations
