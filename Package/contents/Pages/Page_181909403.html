Show Parent Emitter Shape

This section describes the Show Parent Emitter Shapes feature for displaying emitter shapes and axes in the viewer.

Overview

The Show Parent Emitter Shapes feature is for debugging the display of emitter shapes and axes in EffectViewer.
The settings for Emitter Setting () and Emitter Transform () are displayed in conjunction with UI use.
The following overview describes the settings ().

Show Parent Emitter Shape

Show Parent Emitter Shapes is enabled or disabled using the check box ().
