Model Property Window: General Page

This page is used to check and edit the general properties of the selected model.

A The number of materials included in the model.
B The number of bones included in the model.
C The number of shapes included in the model.
D When the quantized format of the vertex coordinate is standardized by the format with the highest precision inside the model, this check box is selected.
Note: The check box is cleared when exporting from a DCC tool.
E The number of vertices to process included in the model.
F The number of vertices to process per triangle of the model.
G The number of vertices included in the model.
H The number of polygons included in the model.
I The number of smooth skinning shapes included in the model.
J The number of rigid skinning shapes included in the model.
K The number of rigid body shapes included in the model.
L If selected, scaling is calculated when the bone matrix is calculated.
M The scaling calculation method of the bone matrix.
N The rotation calculation method of the bone matrix.
O

Verifies whether to use motion mirroring for the model.
Configured from the export options for Maya.
P Preset name for the DCC plug-in.
Q Displays the overall scale specified by the DCC tool.
R Displays the start frame of the animation specified by the DCC tool.
When exporting a rigid body that includes an animation, models are exported set to the start frame.

Bone Matrix Scale

The calculation method for each scale mode is as follows.

Scale Mode Calculation Method
Standard Method Calculates scale values using the standard method.
Maya Calculates scale values, taking into account the segment scale correction applied by Maya.
Softimage Calculates scale values, taking into account the scale calculation method used by Softimage.

Bone Matrix Rotation

The calculation method for each rotation mode is as follows.

Rotation Mode Calculation Method
Euler XYZ Calculates the Euler angle rotation in order of XYZ.
Euler angle rotation values are specified in degrees.
Quaternions Calculates using quaternions.
