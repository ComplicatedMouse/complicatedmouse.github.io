Profiler-TreeControl

Overview

Tells the profiler to perform actions on the tree present on the specified tab.

Profiler-TreeControl [-Tab] <ScriptingTabs> [-Action] <ScriptingTreeControlActions>


[-Tab] <ScriptingTabs>

Specifies the tab that contains the tree to manipulate. The following values are supported for this argument:

    • CallTree: Call Tree Tab
    • Functions: Functions Tab
    • Info: Info Window
    • Instrumented: Instrumented Profile Tab
    • Threads: Threads Tab
Note

The Functions and Threads tabs are only a valid option when Thread-Centric Data Views are enabled. See Experimental Features Tab for additional information.


[-Action] <ScriptingTreeControlActions>

Specifies the action to apply on the tree.

    • CollapseAll: Collapses all nodes in the tree.
    • ExpandAll: Expands all nodes in the tree.


Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-TreeControl CallTree ExpandAll
Profiler-TreeControl -Tab Info -Action CollapseAll
