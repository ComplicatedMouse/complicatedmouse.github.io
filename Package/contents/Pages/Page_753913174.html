Profiler-PerfCounters

Overview

Sets which performance counters to record with both Sampled and Instrumented profiles.

Although the GUI allows for setting these independently, this cmdlet will set both at the same time.

Profiler-PerfCounters [-PerfCounters] <ScriptingPerfCounters>


[-PerfCounters] <ScriptingPerfCounters>

The strings used to denote performance counter groups in a script. These values correspond with the performance counter groups.

    • Disabled: Used to specify Performance Counters Disabled.
    • InstructionsExecuted: Used to specify the Instructions Executed group.
    • L1ICache: Used to specify the L1 I-Cache group.
    • L1DCache: Used to specify the L1 D-Cache group.
    • L2Cache: Used to specify the L2 Cache group.
    • Atomics: Used to specify the Atomics group.
    • Branches: Used to specify the Branches group.
    • SystemCalls: Used to specify the System Calls group.
    • UnalignedDataAccess: Used to specify the Unaligned Data Access group.
