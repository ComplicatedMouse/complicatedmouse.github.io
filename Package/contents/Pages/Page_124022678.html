Overwrite Properties Settings Window

Use the Overwrite Properties Settings window to specify the panes to overwrite settings to.
By adding overwrite settings to a parts layout, parameters can be overwritten even after the parts layout has been placed in a pane.

In the Parts and Control Settings window, select Overwrite > Settings to open the Overwrite Properties Settings window.
The following operations can be performed in the Overwrite Properties Settings Window.

Overwrite Properties Settings Window
A Panes to Overwrite

To overwrite settings for parts panes, select the check boxes next to the names of the part panes in the list.
The selected panes are then listed in the Parts and Control Settings window.
B Description

Displays descriptions of the selected panes.

The descriptions entered here are shown in the list of comments in the Parts and Control Settings window.
