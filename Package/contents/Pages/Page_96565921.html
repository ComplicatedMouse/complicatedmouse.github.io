Emission Timing (%)

This section describes the Emission Timing (%) feature to specify when particles are emitted from emitters.

Overview

Emission Timing (%) is a feature to set the percentage for the lifespan of parent particles higher in the hierarchy when particles are emitted from emitters lower in the hierarchy.

The following overview describes the settings ().

Emission Timing (%)
