Random and Noise

This section describes the differences between Random and Noise features using the examples of the Random and GPU Noise settings.

The Random and Noise features are for creating behavior without apparent pattern for particle movement and settings values.
Random has no order, while Noise has rules.
The following table lists the features of Random (using a random algorithm) and Noise (using a noise algorithm) for particle movement.

Randomness

GPU Noise
Algorithm Random (random number)

Noise (sine wave synthesis)
