Apply in World Coordinates

This section describes Apply in World Coordinates, which is a feature for specifying the coordinate system for add-on values for the position.

Overview

Apply in World Coordinates is a feature for specifying whether to apply the world coordinates to add-on values for the position.
It can be used to move the particles in a constant direction without being affected by factors such as emitter rotation.

Apply in World Coordinates

Change the coordinate system to apply by using the check box ().
When the check box is selected, calculations always occur within the world coordinate system, regardless of the slope of the emitter.
If the feature is disabled, the tilt of the emitter affects the calculated value.
