Add Offset to Frame Value Window

The Add Offset to Frame Value window can be used to batch-edit frame values for the selected animation segment tag.

Add Offset to Frame Value Window
A Check All
 Selects all of the currently displayed animation segment tags.
B Animation Segment Tag Name Displays the name of the added animation segment tag.
C Set length of all frames at once Selecting this check box makes it possible to specify the frame lengths for all of the currently selected animation segment tags at one time.
D Frame Length The value specified here is set as the frame length for all of the currently selected animation segment tags.
E Add offset value to frame Selecting this check box makes it possible to add offset values to animation segment tag frames.
F Add Offset The value specified here is added to the animation segment tag frames as an offset.
