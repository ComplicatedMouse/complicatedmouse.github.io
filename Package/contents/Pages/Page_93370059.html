Emitter Shape - Line

Describes the settings for Emitter Shape - Line, where particle emission locations are set along a line.
These are the particles when this emitter shape is being used.

Overview

With the line emitter shape, particles emit in the direction of the line (the z-axis direction) from random positions of the line located in world coordinates.
The following overview describes the settings ().

When the emitter shape is set to Line, the Line () settings are displayed. The group has the following settings.

Length

Sets the length of the line.

Center Position

Sets the center point to use for emitter scaling and rotation.
Set the center position, where the length of the overall line is 1.
To specify an emitter center outside of the actual emitter itself, specify a value outside the range from –1.0 through 1.0.
