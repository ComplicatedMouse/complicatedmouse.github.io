Notice of Planned Deprecation of Platform Presets Files

Files under %NINTENDO_SDK_ROOT%\Tools\Graphics\3dEditor\Platforms will be deprecated as of NintendoSDK 7.0.0. From now on, write platform settings for the connected application in the <PlatformPresets> team setting.

Added support for material intermediate files.

3DEditor and the assign utility now support editing and saving material intermediate files.
It is now possible to create a new file for the FMDB material part in a dedicated file format (FMTB) only and handle the file as a parent material.
Material (FMTB) files can now be edited and saved.

Regenerate shader when material is modified can be modified.

While editing materials, it is now possible to easily modify Regenerate shader when material is modified.

When editing multiple shader options at the same time, it is now possible to do things like disable regeneration while editing to avoid the time it takes to regenerate the shader with each edit, and then enable regeneration after completing the editing work.

Photoshop Plug-In

Added and improved features for normal map filter plug-in.


The normal map filter plug-in has been improved and features have been added to it.

  • When combining the normal map of another file, the effect of the other file's pixels has been made weaker so that the pixels are almost transparent.


  • When converting a height map to a normal map, transparent (semi-transparent) pixels are now converted after blending the opacity of pixels with a gray background (RGB=128).
    The normal map can be used as a height map by making the parts of the transparent layer you want to render as recessed black, and the parts you want to render as protruding white.

  • Multiple layers of normal maps can now be combined.


Related Links For more information, see the following pages.
