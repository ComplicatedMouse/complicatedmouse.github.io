Model Property Window: Optimization Log Page

This page displays information related to optimization of the selected model.

A Displays the result of compressing materials.
Merges and compresses materials having the same settings.
B Displays the result of compressing bones.
Deletes bones at the end of the hierarchical structure that will not be used for rendering and then compresses bones.
C Displays the result of compressing bones.
Deletes bones located at the end of the hierarchical structure that will be used for rendering of rigid bodies only, and then compresses bones.
D Displays the result of compressing shapes.
Merges and compresses shapes having the same settings.
E Displays the result of compressing primitives.
F Displays the result of quantization analysis.
Info

For more information about optimization processes, see Intermediate File Optimizer , included in the 3D Graphics Command-Line Tool Help, provided elsewhere in this package.
