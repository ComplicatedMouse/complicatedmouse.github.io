Hierarchizing Emitter Nodes

This section describes both how to add a hierarchy to emitter nodes in the workspace, and how to remove a hierarchy.

Preparations

Related Links Before beginning, complete the tasks described in Quick Start > Creating Effects.

Using the steps to this point, multiple emitters are created directly beneath the EmitterSet1 emitter set (), as follows.

Adding Emitters Under the Hierarchy

This section describes how to add emitter nodes under the selected emitter hierarchy and how to create child emitters.
From the EmitterSet1 emitter set () in the workspace, select the Emitter1 emitter (), and then perform one of the following operations.

  • Right-click, and select Add New Emitter () from the shortcut menu.
  • On the toolbar, click Add New Emitter ().

Under the Emitter1 () hierarchy, Emitter3 () is added as a child emitter.

Copying Emitters Under the Hierarchy

Thissection describes how to copy emitters under the copied emitter hierarchy and how to create child emitters.

Under Emitterset1 () in the workspace, select Emitter1 ().
Right-click, and select Copy () from the shortcut menu.
Right-click Emitter1 (), and from the shortcut menu, select Paste emitter as child ().

Under the Emitter1 () hierarchy, Emitter1_Copy1 () is added as a child emitter.
Emitter1_Copy1 () duplicates the settings of Emitter1 () as a child emitter.

Completing the procedures to this point creates three emitters and two child emitters in the EmitterSet1 () hierarchy, as follows.

Removing Hierarchy

This section describes how to remove selected child emitters.
In the workspace, select child emitter Emitter3 (), and perform one of the following operations.

    • Right-click, and from the shorcut menu, select Remove Hierarchy ().
    • You can also press CTRL+SHIFT+P as a keyboard shortcut.

The parent-child relationship between Emitter1 (), the higher emitter in the hierarchy, and Emitter3 (), the child emitter in a child hierarchy, is removed.
The node is placed in the workspace as follows. Emitter3 () is in the same hierarchy level as Emitter1 ().
