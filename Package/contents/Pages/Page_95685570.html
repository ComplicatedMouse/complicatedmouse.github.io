Playback Method - Clamp

This section describes the Clamp playback method for texture pattern animations, which can clamp playback.

Overview

The Clamp playback method animates each pattern for the interval set for Sequential frames per pattern.
If the animation cycle is shorter than the lifespan, you can stop on the display of the final pattern.
The following overview describes the settings ().

Select to display the Clamp () and Pattern Tables () settings. The group has the following settings.

Sequential Frames per Pattern

Sets the number of frames to display for each pattern.

Number of Pattern Tables To Use

Example Configurations

The following image compares what happens when the playback method is Clamp and the value for the lifespan is changed. It also shows details about the settings.
Texture Pattern Image
Sequential Frames per Pattern 5
Number of Pattern Tables To Use 16
Table Easy Input 0 to 15
A

Lifespan = 50

Playback time per pattern (5 frames) / Particle lifespan (50 frames) => Only 10 patterns can be played.
After the 10 patterns have been played, the animation plays starting from the first pattern.
B Lifespan = 100

Playback time per pattern (5 frames) / Particle lifespan (100 frames) => Up to 20 patterns can be played.

All 16 patterns are played in the cycle, after which the animation stops for a duration of 20 frames (equal in time to 4 patterns).
C Lifespan = 200

Playback time per pattern (5 frames) / Particle lifespan (200 frames) => Up to 40 patterns can be played.

All 16 patterns are played in the cycle, after which the animation stops for a duration of 120 frames (equal in time to 24 patterns).
