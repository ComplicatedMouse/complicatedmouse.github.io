Profiler-Select

Overview

Selects items in the specified tab. The general form for this cmdlet is shown below. There are 3 variations available for this cmdlet. Not all variations are supported by all allowable tabs.

Profiler-Select [-Tab] <ScriptingTabs> <VariationArguments>


[-Tab] <ScriptingTabs>

Specifies the tab to select items on. The following values are supported for this argument:

    • Counters: Counters Tab
    • Files: Files Tab
    • Functions: Functions Tab
    • GraphicsGroups: Gfx Groups Tab
    • Logs: Logs Tab
    • Instrumented: Instrumented Tab
    • Shaders: Shaders Tab
    • Threads: Threads Tab
    • Timeline: Timeline Tab

Top Group

Selects groups of 50 elements at once.

Profiler-Select [-Tab] <ScriptingTabs> -Top [[-TopGroup] <Integer>]


[-Tab] <ScriptingTabs>

  • Functions
  • GraphicsGroups
  • Shaders

-Top

Specifies that this will be selecting groups of 50 elements at once.


[[-TopGroup] <Integer>]

Which group will be selected. If unspecified, the first 50 elements are selected.

List

Selects elements at the specified indices from the tab (indices run from top to bottom, with the top being 0).

Profiler-Select [-Tab] <ScriptingTabs> -List <Integer[]>

[-Tab] <ScriptingTabs>

  • Counters
  • Files
  • Functions
  • GraphicsGroups
  • Logs
  • Shaders
  • Threads

-List <Integer[]>

A list of the indices to select.

Deselect All

Deselects all currently selected elements.

Profiler-Select [-Tab] <ScriptingTabs> -DeselectAll

[-Tab] <ScriptingTabs>

  • Counters
  • Files
  • Functions
  • GraphicsGroups
  • Instrumented
  • Logs
  • Shaders
  • Threads
  • Timeline

-DeselectAll

Specified to deselect all selected elements on the tab.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Select Functions -Top
Profiler-Select Functions -Top 3
Profiler-Select Functions -List 0 1 4 5
Profiler-Select Functions -DeselectAll
[int[]] $items = 0, 3
Profiler-Select Functions -List @items
