Using Guides

This section describes how to arrange panes using guides.

Creating Guides

Create a guide. You can create a guide using one of the following methods.

  • From the Tools menu, select View - New Guide to open the Edit Guide dialog box. Guides can be created anywhere and at any angle.
  • As shown in the following image, you can also create a guide by dragging one from the ruler. If the ruler has been hidden, you can show it by opening the Tools menu and selecting View - Show Ruler.


Editing Guides

The created guides can be edited using any one of the following methods.

Adjusting the Position of a Guide

  • Select the guide and adjust the position in the Properties window.
  • Right-click the guide and select Edit Guide from the menu to open the dialog box to adjust the position.
  • Drag the guide (with the left mouse button) to adjust the position.

The following example shows left-clicking and dragging to adjust the position of the guide.

Adjusting the Angle of the Guide

  • Select the guide and adjust the angle in the Properties window.
  • Right-click the guide and select Edit Guide from the menu to open the dialog box to adjust the angle.
  • Press the ALT key and drag (with the left mouse button) to adjust the angle of the guide.

The following example shows pressing ALT and dragging to adjust the angle of the guide.

Changing the Guide Color

  • Select the guide and change the color in the Properties window.
  • Right-click the guide and change the color.

The following example shows changing the guide color using the Properties window.

Snapping the Pane to the Guide

As shown in the following images, selecting View - Snap to Guide from the Tools menu snaps panes to guides when they are moved.
