Classifications Option Tab

This tab contains options that modify how the classification system works within the profiler.

Classification Overrides

Contains tools to manage any classification overrides.

Import and Export Buttons

You can Import or Export a file that describes overrides for the classification system. This allows you to share your overrides with coworkers or use their overrides.

Override Table

If a function's classification has been overridden, that function will appear in the override table. The table only appears if one or more functions have been overridden (by importing an override file or by right clicking on a function in the Functions tab and selecting a classification override).

Within this table, you can manage individual functions by removing their override (red X button) or modifying the override (drop down box).
