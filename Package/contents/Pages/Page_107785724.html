vi Library

Overview

Purpose of This Documentation

This documentation describes the Video Interface (VI) library included in NintendoSDK.

Intended Audience

This documentation is intended for people who develop applications using NintendoSDK.

Associated Documentation

Use this document in conjunction with the nn::vi namespace API reference.

Features Provided by the VI Library

The VI library provides information about the capabilities of displays and registers layers with the display process.  Applications can associate a set of textures from NVN with a layer.  The layer is utilized by the display process to composite it on the display.
