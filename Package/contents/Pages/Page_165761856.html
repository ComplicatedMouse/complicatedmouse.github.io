Window Pane Property Editing Page

Describes the Window Pane Property Editing Page.

The Window Pane Property Editing Page contains common settings needed to configure the window pane property windows.
The Window Pane Property Editing Page is displayed on all of the window pane-related settings screens.

Window Pane Property Editing Page

Specify the area of properties to edit.

Left-click the image to select areas to edit.

The selected area is displayed in the color pink in the image, and the name of the selected area changes accordingly.

The areas that are selectable differ according to the type of window pane and the setting for the number of textures to apply.

The orientation of the texture for the frame is the upper left by default.

  • Content
  • Upper Left
  • Lower Left
  • Upper Right
  • Lower Right
  • Upper Frame
  • Lower Frame
  • Left Frame
  • Right Frame

Type

Determines the type of window pane.

There are three types of window panes, and each has a different shape.

In addition, the areas to which textures can be applied differ depending on the type of window pane.

  • All Directions (conventional behavior)
  • Horizontal
  • Horizontal - No Content

Texture Count

Specifies the number of textures to apply to the frame of the window pane.

There is a setting for applying a texture only to the upper left and then stretching it in four directions or applying a different texture to each division of all of the frames.

  • 1 (specify top left only)
  • 4 (specify corners only)
  • 8 (specify all)
