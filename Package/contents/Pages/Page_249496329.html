Template List

This section presents a list of templates that are available for the creation of effects.
These templates are for effects often seen in games. You are free to modify them, and add textures and the like, to meet your needs during game development.

In addition, presets have been prepared for general-purpose parts used in effects.
You can load emitters from the presets to the effect data you are creating.
For more information about the presets used with template data, see the Preset Used column in the following table.
Related Link For more information about adding emitters from Presets, see this page.

Info

The data is located in the following directory. It can also be opened from the Effect Browser.
