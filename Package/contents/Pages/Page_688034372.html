cstdio

void setbuf( std::FILE* stream, char* buffer );

int setvbuf( std::FILE* stream, char* buffer, int mode, std::size_t size );

Description

The buf is offset by 8 bytes.

The "size" parameter will get resolved to "size - 8" (which means that the default buffer is size "BUFSIZ - 8").
