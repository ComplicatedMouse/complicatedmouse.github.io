Overview

You can configure filters for LPF and BiquadFilter using the atk library and SoundMaker.

This section describes those filters.

Using Filters

The methods for using the filters are described in the documentation for the atk library and SoundMaker.

Info
  • For information about setting filters using the atk library, see Filter in the Programming Guide.
  • For information about setting filters using SoundMaker, see the sections for wave sounds, stream sounds,and sequence sounds in the SoundMaker documentation.


Relationship Between Filter Parameters and Cutoff Frequencies

The Audio Toolkit (Atk) reflects parameters configured both by the atk library and by SoundMaker. Cutoff frequencies are not set directly, but by using relative parameters.

Here we describe the relationship between the filter parameters and the cutoff frequencies.

Info

For more information about parameter processing, see Sound Parameter Control Operations.

Relationship Between LPF Parameters and Cutoff Frequencies

For the LPF, you can also set the following values.

  • With the atk library: In the range of -1.0f to 0.0f (the default is 0.0f).
  • With SoundMaker: In the range of 0 to 64 (the default is 64).

In either case, the default value applies no filtering, and the lower the value, the greater the filtering (the lower the cutoff frequency).

Converting to a Cutoff Frequency

The following calculation is made by using the configured value to determine the LPF parameter. (The LPF parameter is kept inside the range of 0.0f to 1.0f)

(LPF parameter) = (atk library's setting) + (SoundMaker's setting / 64.0f).

Ultimately, the LPF parameter is converted into a corresponding cutoff frequency, as shown in the table below and applied to the LPF.
(To lighten the calculation process, approximate values are used for the conversion to cutoff frequencies.)

For example, you are playing a wave sound with an LPF value set to -0.2f in the atk library and 48 in SoundMaker.
The LPF parameter calculates out to 0.55f (-0.2f + (48 / 64.0f), which falls under ~0.567807191 in the table below, and corresponds to a cutoff frequency of 1280 Hz for the LPF.
