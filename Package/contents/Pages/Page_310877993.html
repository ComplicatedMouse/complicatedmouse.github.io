Overview

Check the number of paired controllers and the status of connected controllers, and then delete pairing information.
This information is displayed on the page only if there are connected controllers. If a controller has been paired but is not connected, its information is not displayed.

How to Use

Pairing Controllers

To begin the process of pairing a controller with the development hardware, click Start Controller Pairing.
To cancel the pairing process, click Cancel Controller Paring.

Note

The maximum number of controllers that can be paired simultaneously with this method is seven.
When pairing 8 or more controllers at the same time, press the SYNC Button on an already-paired controller to disconnect.

Deleting Controller Pairing Settings

To delete the pairing settings, click Delete All Controller Paring Settings.

Displaying Controller Connection Status and Battery Level

DevMenu can check the number of paired controllers and the status of any currently connected controllers.

Device Style

DevMenu can also check the connection style of each controller connected to the system. The following styles are supported.

Style Description
Joy Left The Joy-Con (L) is connected.
Joy Right The Joy-Con (R) is connected.
Handheld The Joy-Con is attached to the joint rails on the system.
Pro Con A Nintendo Switch Pro Controller is connected.
Other Indicates that a controller not in this list is connected.

Player LED

The player LED patterns can be used to determine the correspondence between the Joy-Con controllers connected to the system and the controllers displayed on the screen.
When the connection style is handheld, none of the player LEDs are illuminated. Up to four controller numbers can be assigned to the left and right Joy-Con controllers.

Firmware Version

Displays the firmware version of the connected controller. For more information about the latest version, see the Controller Firmware Release Notes page.
Currently, the firmware version is not displayed when the connection style is handheld. To check the firmware version, connect the controller when it is detached from the development hardware.

Serial Number

Displays the serial number of the connected Joy-Con.

Battery

DevMenu can also check the controller battery level. In DevMenu, the battery charge level is expressed as icons representing the following five levels.

Definition Description
Empty The battery is completely depleted.
Critical The battery is almost depleted. The level appears red.
Low The battery level is low.
Medium The battery is approximately halfway depleted.
High

The battery level is high.

When the connection style is handheld, the battery level is displayed as follows.

  • If a single Joy-Con controller is connected, the battery level for that Joy-Con controller is displayed.
  • If two Joy-Con controllers (left and right) are connected, the battery level for the right Joy-Con controller is displayed.

Changing the Gyroscope and Accelerometer Settings

The feature is used for changing the settings for the gyroscope and the accelerometer.

You can change the resting point and sensitivity settings for each player LED lighting pattern for each senor.

  • When you click Apply Changes, the controller is disconnected. The changed values are applied upon reconnection.
  • The player LED patterns can even be set for a controller that is not connected; the settings will be applied when that controller is later connected.
  • When the development hardware is restarted, these changed settings revert back to their default values.

Changing the Vibration Settings

Select or clear Enable Vibration to enable or disable the controller vibration settings. This is enabled by default.

Changing the Pro Controller Wired USB Communication Settings

Select or clear Enable Pro Controller USB Communication to enable or disable the Pro Controller's wired USB communication settings. This is disabled by default.
