Question

Initialization takes too long when loading OpenType fonts.

Answer

Because processing for getting the kerning for OpenType fonts takes time, the processing for getting the kerning is optimized by having it performed all together at initialization.

This initialization process takes approximately several seconds for each font in the Debug build, but it is shortened to several tens of milliseconds — several hundred milliseconds in the Develop/Release build — so that it is not a significant problem for the final output.
