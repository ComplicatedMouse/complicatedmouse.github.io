Animation Overview

Automatic Creation of Animation Curve Data

If the Bake All Animation export option is selected or if the animation curve in Maya cannot be output as is, animation curve data is automatically created from the data of the baked frame values.
The baking process is executed in units of the number of frame specified for Frame Precision, so playback results may be different from those in Maya for frames that are not a multiple of the specified frame precision value.
Animation curve data is always automatically created and exported from baked data in the following cases.

  • The Weighted Tangents animation curve option is selected.
  • The infinity of an animation curve is Linear, and the animation output range includes the infinity range.
  • The animation curve uses a driven key, so that its horizontal axis represents driver attribute values instead of the time axis.
  • The animation curve is partially muted.
  • The animation is expression-dependent.
  • The animation is based on animation layers.
  • The animation was edited using the Time Editor.
  • The animation was edited using the Trax Editor.

For the rotation value, the animation curve is automatically generated after adjusting the baked data so that it is continuous and has a small absolute value.

To automatically create animation curve data, bake all infinity-based animations. The pre_wrap and post_wrap components in the intermediate file are both clamped. Repetition outside the export range of the animation cannot be reproduced.
To output an animation curve as shown in Maya, infinity is applied in pre_wrap and post_wrap in the intermediate file.

Loop Animation

When Loop is selected in Export options, specify an end frame having the same conditions as the start frame.

For example, if frame 1 and frame 10 of a loop animation have the same state, specify 1 for the start frame and 10 for the end frame.

Whether Loop is selected or cleared, the FrameSize in the intermediate file is specified by the EndFrame minus the StartFrame.

Values Output to FMD Files

If attributes are animated, the values specified by export option for the position of the start frame are output to the FMD file. Note, however, that bind pose values are exported for the translate, rotate, and scale nodes, which are influenced by skinning.
Also, the value specified by the fileTextureName attribute of the file node is output for texture pattern animations. If the fileTextureName attribute contains the string <f>, the <f> portion is replaced with the image number for the specified start frame in the export options when output to the FMD file.

Info

Note that current Maya frame values are not exported.

Time Editor

If the Time Editor was used, if a clip is not muted, it is assumed that animation has been set.
