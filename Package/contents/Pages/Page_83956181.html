Toolbar (Curve Editor)

You can change the layout of the Curve Editor toolbar to suit your taste.

A Key mode, key type, key frame, and value clamping can be selected by using icons.
Resting the mouse pointer on any icon displays a description of its function.
B These icons switch the display mode of the Curve View.
Resting the mouse pointer on any icon displays a tooltip describing its function.
C Plays/pauses animations.
Paused animations have a paused icon on the right side of the filename in the File View.
D Enables or disables current time clamping.
E Selects the paste method for curves and keys.
F Type a number to set the number of frames and values for keys that are selected in the Curve View.
G

Type a number to set the slope for the selected key.
H

Multiple Snap

The snap unit for the value and frame of the key to be added can be selected from a drop-down menu.
I

When Snap Keys is selected and you move the key in Curve View, frames and values are snapped for other keys in Curve View.
A dotted line displays in the Curve Editor for snapped keys.

The C key enables or disables Snap Keys.

The following icons appear on the Curve Editor toolbar.

Changes the key mode to Scale Operation. This allows you to scale the curve by center dragging in the Curve View.

Paste Method for Curves and Keys

The following paste methods are available in Curve Editor and can be selected from a pull-down list.

Paste Method Description
Insert Inserts and pastes the copied key.
The frames of keys that are located after the paste position are increased by the number of curve frames that are pasted.
Replace Replaces and pastes the copied key.
Keys that already exist are replaced by the pasted key or curve.
Merge Merges and pastes the copied key.
The copied key is pasted while leaving already existing keys in place.
The result is that the already existing key is merged with the pasted key.
Paste Location Description
Absolute Pastes the copy source value, regardless of the key position in the paste destination.
Relative Moves to the position relative to the paste destination key and pastes.
