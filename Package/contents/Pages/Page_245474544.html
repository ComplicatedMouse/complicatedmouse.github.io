Animation Segment Separated Mode

Animation segment separated mode is now available to help make animation production more convenient.

Previously, you could use segment tags in LayoutEditor to specify regions for playback in a single, long animation curve. But now, with animation segment separated mode, you can create a separate animation curve for each segment tag.

(The viewer references the segment tabs for playback, so even though you can now create individual animations, they can be played back in the same way as before.)

Differences Between Merged Mode and Separated Mode

There are now two modes for editing animation segment tags: the conventional animation segment merged mode, and the newly added animation segment separated mode.

The following images compare how the same three animations (Start, Wait, and End) are displayed in merged mode and separated mode.

Animation Production Using Animation Segment Merged Mode

This method is the same as the method that was used until now, with the animation created as a single, joined animation broken up by animation segment tags.

Example:

  • Start is an animation comprising a total of 20 frames, created in frames 00 to 20.
  • Wait is an animation comprising a total of 50 frames, created in frames 20 to 70.
  • End is an animation comprising a total of 30 frames, created in frames 70 to 100.
Animation Production Using Animation Segment Separated Mode

This is the newly added mode.
The animations are managed by using animation segment tags and are created separately, starting in each case from the 0th frame.

Example:

  • Start is an animation comprising a total of 20 frames, created in frames 00 to 20.
  • Wait is an animation comprising a total of 50 frames, created in frames 00 to 50.
  • End is an animation comprising a total of 30 frames, created in frames 00 to 30.

Switching Modes

You can switch between these modes to suit your preferred way of creating animations.
You can switch modes to edit data while working on the data (see note).

Icon buttons for switching the mode are located at the top of the Edit Animation Segment Tags window ().

Mode Switching Buttons

Merged mode button

 Represents merged mode.
Separated mode button Represents separated mode.

When you switch from merged mode to separated mode, the segment tags are referenced to separate the animations.
When you switch from separated mode to merged mode, the segment tags are referenced to merge the animations.

Info

Note: When you act to merge the animations, keys will sometimes be added outside of the range of segments, and this can lead to unexpected results.
　For this reason, we recommend that you basically stick with either separated mode or merged mode for your task.

Creating Animations in Separated Mode

This section describes the process of animation creation using separated mode.

Select the pane you want to edit and then, in the Edit Animation Segment Tags window (), select the segment tag representing the range of the animation that you want to edit. When you perform this step, the segment tag range of the animation is displayed in the animation window.
In separated mode, you edit the animation curve for each individual segment tag.
By switching the selected segment tag, you can create a separate animation curve.

All animations created in separated mode start from frame 0.

Fitting (only available in separated mode)

The Fitting feature adjusts the segment width based on the length of the animation curve.

To run the feature, click the Fitting icon () in the toolbar in the Curve Editor window ().
If you run the Fitting feature () after changing the length of the animation curve in the Curve Editor window (), the segment width in the Edit Animation Segment Tags window () is adjusted based on the length of the animation curve.

The segment width resulting from the fitting matches the longest segment in the entire animation curve in which the selected segment tag belongs.

Info
  • The starting frame of the tag segment is fixed to frame 0. (Division mode animations start from the 0th frame.)
  • If there are no animation keys in the tag segment, the minimum width is 1.

Animation Scaling (a Division Mode-Only Feature)

The feature for scaling the created animation shortens and elongates the segment while maintaining the spacing ratio between keys.

In the Edit Segment Tags window, the Frame Segment section contains three buttons for this animation scaling feature that you can use to switch the feature between Off, Decimal Precision, and Integer Precision.

Animation Scaling Buttons
OFF Disables the feature for scaling the created animation.
Decimal Precision Sets the result with decimal-point precision after the segment has been shortened or elongated using the animation scaling feature.
Integer Precision Rounds the result to an integer value after the segment has been shortened or elongated using the animation scaling feature.
(Rounding can result in fewer keys.)

If ratio linking is enabled, the spacing ratio between keys is maintained when the length of the segment is changed.

Locking Docking Windows

You can now lock (fix) the state of docking windows.
The Fix Docking feature aids users in their tasks by preventing operations that would unintentionally move docking windows.

From the LayoutEditor menu bar, select Window > Fix Docking.

Fix Docking

When this menu item is selected, docking windows can be fixed.

By fixing the docking window, you can eliminate the risk of accidentally moving the window while working.

To remove the lock, select Fix Docking again to clear the selection.

Disabling Layout Window Tabbing

A feature has been added that disables tabbing when multiple layout windows are open on the screen.

From the LayoutEditor menu bar, select Window > Do Not Tab Layout Windows.

Do Not Tab Layout Windows

When this menu item is selected, layout windows are opened in separate windows, rather than as tabs in a single window.
Choose to use this feature as you prefer.

Note: All layout windows must be closed to switch the tabbing setting.
