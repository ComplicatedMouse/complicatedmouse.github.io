Command-Line Tools ()

This page describes documentation for command-line tools and file formats.
It is intended to be used mainly by developers.

SoundProjectConverter The manual for the command-line tool for converting SoundMaker project files into archive files.
SoundProjectUtility This is the manual for command-line tools used to perform various tasks on SoundMaker project files.
SoundProjectWaveExporter The manual for the command-line tool for batch-exporting sounds registered in SoundMaker to waveform files.
WaveConverter The manual for the command-line tool for converting waveform files into resource files that can be used with the Atk library.
File Formats This section lists the file formats used in the audio development environment.
