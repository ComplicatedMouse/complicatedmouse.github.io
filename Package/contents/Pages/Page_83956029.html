Random number type

This section describes Random number type, which is the feature for controlling the random numbers used for particle behavior.

Overview

This feature allows you to arrange how random numbers are used by emitters for their features that have random elements. It is restricted to those emitters that all have the same settings.
This feature can be used to make multiple emitters move with the same trajectory, even if they have undefined trajectories and no parent-child relationship.
The following overview describes the settings ().

Random number type

Selects the random number type when calling effects from among the following presets.

Change every time Changes the random number for the emitter every time.
Common in ESET

Changes the default for the emitter random number every time.

You can make the appearance among emitters with the same settings remain the same because identical values are shared within an emitter set.
Fixed The default for the emitter random number is the same every time.

Set the type to Constant to display the Constant () settings.
The following overview describes the settings ().

Random Seed This feature sets the random number value so that the effect appears the same every time.
