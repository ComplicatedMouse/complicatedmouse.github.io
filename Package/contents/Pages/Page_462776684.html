NintendoSDK 7.2.1

This page lists the known bugs in this release.

  • Audio
    • Rendering results are sometimes not valid if the ExecuteAudioRendererRendering() function is simultaneously called by multiple audio renderers.
  • socket
    • The nn::socket::Select() and nn::socket::Poll() functions sometimes return EBADF.
  • PERF
    • The PerfSimple sample crashes in some PC environments.
  • Effects: VFX
    • When the Emitter Assignment for Particles inheritance setting for child emitters in effect data is disabled, specifying Historical Stripe or Historical Stripe2 as the particle type causes the VFX runtime to stop responding.
  • Layout: UI2D
    • Color pasting is not executed on the Shape page of the LayoutEditor property window or Capture Effect page.

Audio

Rendering results are sometimes not valid if the ExecuteAudioRendererRendering() function is simultaneously called by multiple audio renderers.

Symptoms

Rendering results are sometimes not valid if the nn::audio::ExecuteAudioRendererRendering() function is simultaneously called by multiple audio renderers.

Workaround

Exclude the nn::audio::ExecuteAudioRendererRendering() function from the application so that it is not simultaneously called by multiple audio renderers.

Fix Schedule

This bug will be fixed in an upcoming release.

socket

The nn::socket::Select() and nn::socket::Poll() functions sometimes return EBADF.

Symptoms

The nn::socket::Select() and nn::socket::Poll() functions sometimes return EBADF when called if the socket descriptor is closed.

Workaround

No workaround is currently available.

Fix Schedule

This bug will be fixed in an upcoming release.

PERF

The PerfSimple sample crashes in some PC environments.

The PerfSimple sample exits with an assertion in certain PC environments.

This bug will be fixed in a future release.

Effects: VFX

When the Emitter Assignment for Particles inheritance setting for child emitters in effect data is disabled, specifying Historical Stripe or Historical Stripe2 as the particle type causes the VFX runtime to stop responding.

Symptoms

When the Emitter Assignment for Particles inheritance setting for child emitters in effect data is disabled, specifying Historical Stripe or Historical Stripe2 as the particle type causes an invalid access event to occur in the VFX runtime.

Workaround

You can avoid this bug by enabling Emitter Assignment for Particles in the effect data settings.

Fix Schedule

This bug will be fixed in an upcoming release.

Layout: UI2D

Color pasting is not executed on the Shape page of the LayoutEditor property window or Capture Effect page.

Symptoms

Even if you select Paste on the shortcut menu of the color buttons on the Shape page of the LayoutEditor property window or Capture Effect page, it is not executed.

Workaround

You can set the color by dragging between the color buttons.

Fix Schedule

This bug will be fixed in an upcoming release.
