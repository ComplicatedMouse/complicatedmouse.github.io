One Time

Describes the One Time feature for determining the emission time for particles.

Overview

The One Time feature specifies the particle emission time.
Normally, particles are emitted infinitely. Turning this feature on limits particle emission to a specific time.
For example, this feature can be used in an explosion effect where it emits particles once.
The following overview describes the settings ().

One Time

You can enable or disable one-time emission by selecting or clearing the check box ().
When this check box is selected, particles are emitted for exactly the number of frames specified for the emitter's emission time.
When this check box is cleared, emission time is unlimited.

Enable this setting to display the One Time () group of settings. The group has the following settings.

Emission Time

This option is enabled when One Time is selected. It sets the time during which particles continue to be emitted.

Info

Looping is useful when you are using the viewer to check how effects look when One Time is ON.
Looping at the end of lifespan of the last particle emitted makes it easier to watch the effect.

For example, if an effect has an emission time of one frame, and a lifespan of 100 frames, you can set the loop to 100 frames plus a few extra frames.
