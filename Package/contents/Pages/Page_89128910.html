Features

This section describes the features provided by the hid library and how to use the API resources.

Features

Features Overview
DebugPad Functionality for implementing debug controller use.
Digitizer Functionality for implementing digitizer operations.
Keyboard Functionality for implementing keyboard operations.
Mouse Functionality for implementing mouse operations.
Touch Screen TouchScreen Functionality for implementing touch screen operations.
Gesture Functionality for implementing gesture operations.
Controller Npad Functionality for controlling a variety of play formats patterned after the controller models that are defined by the platform.
SixAxisSensor Functionality for implementing the operations of the six-axis sensor.
Vibration Functionality for controlling the vibration devices.
