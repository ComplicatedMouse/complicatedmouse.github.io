Overview

SystemUpdateAdev is a command-line tool for updating the firmware of the ADEV development hardware, and the firmware of HDEV and EDEV units, with a process equivalent to the system update feature.

This tool offers an advantage over initializing with InitializeAdev in that you can update the firmware version while maintaining the unit's settings.

Package

Included in the DevKitVersionUpdater for NX package.

It is available in SystemUpdater for NX 13.0.0-3.0 and later.

Location

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/SystemUpdateAdev.exe

In addition to this EXE file, there are also several dependency files for running the tool.

Command Format

Option Format

SystemUpdateAdev [target options] [other options]

target options:
    --target <serial number | HDEV name>               Specify the serial number or name of the target to update.

other options:
    --target-version                                   Specifies the firmware type to use for system updates.
    --display-available-version                        Display all available firmware types.
    --verbose                                          Generate detailed logs.
    --version                                           Display to tool version.

Exit Code

You can use the exit code to check whether the tool succeeded.

0: Successful completion
Nonzero value: Terminated with error

How to Use

The following examples demonstrate different ways of using SystemUpdateAdev.

Updating by Specifying the Development Hardware Serial Number

SystemUpdateAdev --target XAL03100000000

Updating by Specifying the Development Hardware Name

SystemUpdateAdev --target (target name)

Important Information

  • You cannot downgrade the version.
  • You can use this tool on development hardware running the following firmware versions. It does not work with earlier versions of the firmware.
    • EDEV: NX Add-On 4.0.0 and later
    • HDEV: NX Add-On 7.4.0 and later
    • ADEV: NX Add-On 11.1.0 and later
  • When specifying the development hardware serial number, you can specify with or without the check digit.
  • NX Add-On 12.2.X and earlier versions of SystemUpdateAdev do not support updating DevMenuApp. Rewrite this separately after running the update tool.
  • In NintendoSDK 10.4.0 and later, the target communication protocol (htc-gen1/htc-gen2) after update is automatically switched according to the htc-gen1/htc-gen2 setting on the host PC.
    • For more information, see Using Target Manager 2.
