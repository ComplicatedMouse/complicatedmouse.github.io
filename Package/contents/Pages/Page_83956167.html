Curve View

The Curve View of Curve Editor is used for creating and editing animation curves (Add Key / Copy and Paste).

Related Links If the animation cannot be edited, the reason is shown in the Curve View.

A

The Curve View.
This area is used to select and edit curves and keys.
You can clamp frames and values during editing.
The area outside the range set for the number of animation frames is unavailable (shaded a dark gray color).
B When a node with a curve is selected, displays the node path.
You can show or hide the node path by clicking the Display Settings icon in the lower right corner of the curve view and changing the Show the curve name setting.
C Different color assignments are used for different animation curves.
Color animations are displayed using red, green and blue curves.
Other unique colors are used to display other animation curves.
Double-clicking an animation curve expands the curve tree for that curve and scrolls the display to a location where it can be viewed.
D The current time.
A key is inserted into the frame corresponding to the current time (red line) when you add/insert a key using the S key.
You can move keys to any frame by dragging.
E These are color bar and color icons.
You can use the Color Picker to change colors, or you can hit a color key.
F The Time Slider.
The area above the time slider shows information about the selected curve and allows you to edit the animation frame by frame.
G Shows the Translate/Scale Tool panel.
H Shows the Animation Settings panel.
I Shows the Display Settings (Information to Show in Curve View) panel.

Mouse Operations in the Curve View

The following table shows mouse operations in Curve View.

Operation Description
Right-click Opens a shortcut menu for the selected curve.
Left-click Selects keys.
Select a range with the mouse button Selects keys and curves inside the selected range.
Center mouse button drag Moves a key when the key mode is "move closest key" or "move multiple keys."
Move closest key moves the closest key from the location where the center button is dragged.
Move multiple keys moves the keys that have been selected.
If the mode is Scale Operation, you can change the scale of the curve based on the position it is dragged to.
In Insert a key or Add a key mode, you can move keys that were inserted or added.
• In Insert a key mode, you can move keys along the curve.
• In Add a key mode, you can move keys freely.
Center button click If the key mode is "insert keys," keys are inserted based on the value and slope evaluated for the curve in the frame where the center button was clicked.
If the key mode is "add keys," keys are added in the location where the center button was clicked.
SHIFT+wheel button drag This operation is enabled when the key mode is "move closest key" or "move multiple keys."
This limits the movement of keys to vertical or horizontal motion.
ALT+wheel button drag Slides the view.
ALT+right-click left/right drag Changes the view scale to uniform.
ALT+SHIFT+right-click left/right drag start Changes the scale of the view in the frame direction.
ALT+SHIFT+right-click up/down drag start Changes the scale of the view in the value direction.

Shortcut Menu

Right-click Curve View to open the shortcut menu for the selected curve.
Right-click the color bar or a color icon to copy and paste a color with the mouse.

Item Description
Insert Key Inserts a key at the position of the current frame in the selected curve.
Show Only Selected Curve Hides all but the selected curve in Curve View.
Copy Color Copies the color of a color icon or color bar displayed when a color animation curve is selected.
Paste Color Pastes the copied color. Color can be pasted to any color icon or location in the color bar.


Translate / Scale Tool

Click the Translate/Scale Tool　() icon to display the panel for translating and scaling the curve in the Curve View.

A The translation tool.
Translates the curve (or curves) selected in Curve View by the specified amount in the corresponding direction.
Note: Use the UP ARROW key and the DOWN ARROW key to translate the value and the LEFT ARROW key and the RIGHT ARROW key to translate the frame.
B This tool changes the scale of the curve.
It changes the scale of the target (Value or Frame) for the curves selected in Curve View by the amount specified under Ratio.
You can select from Minimum Value, Maximum Value, and Input Value for the reference point to use when applying the scale.


Animation Settings Panel

In this panel, you can edit the number of frames and the quantization tolerance for the animation you are currently editing in Curve Editor.

To display this panel, click the animation settings icon () in the lower-right corner of Curve View.

A Edit the number of frames in the animation by entering a value or using the pop-up slider.
Curve Editor supports animations that are anywhere from 0 to 16777215 frames in length.
B When changing the number of frames, select this check box to automatically fit the curve.
C Select this check box to loop animation playback.
D

Specifies the quantization tolerance.

The larger the value, the smaller the data volume. The animation resolution falls in proportion to this value.

You can change the value to adjust the data volume and resolution.
E

Lists the reduction rates from quantization.

Shows the size of the reduction based on the specified quantization tolerance value.

Quantization Tolerance Settings

Set the allowable error when quantizing animations. For more information about animation quantization, see Animation Quantization in the 3D features description.

Item Description
Scale

Specifies the allowable scaling error for texture matrices and skeletal animations when quantizing animations.
Specifies the percentage for (Maximum curve value) - (Minimum curve value).
Rotate Specifies the allowable texture matrix rotation error or allowable skeletal animation rotation error when quantizing animations.
This value is specified in terms of degrees as an absolute value.
Move

Specifies the allowable texture matrix translation error or allowable skeletal animation translation error when quantizing animations.

Specifies the percentage for (Maximum curve value) - (Minimum curve value).

Information such as (50% 44/88) after Scale, Rotate, and Move indicates the binary size reduction rate of the animation curve after quantization.

For example, if Move (50% 44/88) is displayed, the binary size of the Move curve was originally 88 bytes, was reduced by 50% through quantization, and is now 44 bytes.

Shortcut Menu in the Animation Settings Panel

In the Animation Settings panel, when you right-click the setting where you can set the number of frames, the following shortcut menu is displayed.

  • Copy
    Copy the number of frames currently set.
  • Paste
    Paste frames copied from somewhere else.
  • Maximum number of frames
    Automatically specifies the maximum number of animation frames supported by Curve Editor in 3DEditor (16777215).

Skeletal Animation Settings Panel

When a skeletal animation is open in Curve Editor, you can use the Animation Settings panel to set the animation for looped playback and to configure the quantization tolerance settings.

Info

Note that if you merge the loop settings from the DCC tool, the settings configured in 3DEditor are overwritten because the DCC tool settings are prioritized.

Display Settings Panel (Information to Show in Curve View)

Click the display settings icon () in the lower-right corner of Curve View to display the display settings panel. You can select the information to display in Curve View by using the check boxes in this panel.
