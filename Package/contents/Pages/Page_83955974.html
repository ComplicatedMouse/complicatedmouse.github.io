Using the Animation Segment Tag Feature

This section describes how to use the animation segment tag feature.

Edit Animation Segment Tags Window

You can use the Edit Animation Segment Tags window to register, delete, and edit animation segment tags.
To open the window, from the menu, select Window > Edit Animation Segment Tags.

Adding Segment Tags

Use the following procedure to add segment tags in the Edit Animation Segment Tags window.

1

Click the Add () button to open a dialog box and set the information for the segment tag you want to add.

Verify that the segment tag name is Tag_00.
2 Click the OK button with the default settings to close the dialog box.
3. Verify that the Tag_00 tag has been added in the Edit Animation Segment Tags window.

The basic configuration process for the Edit Animation Segment Tags window is as follows.

1

Select the Tag_00 segment tag.
A mark () is displayed because this segment tag does not do anything.
2

Specify the number of frames to cut as the animation playback segment.
Verify that 0 (start frame) to 1 (end frame) is set.
3

When a segment tag is added with the default settings, a group is created in the Group window with the same name as the tag and set in the related groups (see the Note).

Click the Specify Related Groups button to display the dialog box for specifying related groups.

A mark () is displayed to indicate that the panes to be animated are not restricted because there are no Group settings.
4

Auto-update is specified by selecting the check box ().

With auto-update, only animation set for the target pane is set as the target for binary output.
Pane animation is not set here, so clear the selections for all of the check boxes ().

If the auto-update check box () is cleared, specify auto-update by clicking Auto-Configure Based on Keys of Target Panes.
Info

Note: By configuring related groups, it is possible to restrict the target panes of an animation indicated by that tag segment.
For example, with this functionality, you can create an animation that changes only a specific portion of the screen.
You can use LayoutViewer to verify animation playback with this functionality incorporated.

Switching Animation Modes

You can switch between these modes to suit your preferred way of creating animations.
You can switch modes to edit data while working on the data (see note).

Switch modes from Tools on the menu bar.
When you switch to separated mode, the current mode is displayed in the title bar.

 Indicates that the mode is separated mode.

When you switch from merged mode to separated mode, the segment tags are referenced to separate the animations.
When you switch from separated mode to merged mode, the segment tags are referenced to merge the animations.

Info

Note: When you act to merge the animations, keys will sometimes be added outside of the range of segments, and this can lead to unexpected results.
For this reason, we recommend that you basically stick with either separated mode or merged mode for your task.

Creating Animations in Separated Mode

This section describes the process of animation creation using separated mode.

Select the pane you want to edit and then, in the Edit Animation Segment Tags window (), select the segment tags representing the range of the animation that you want to edit. When you perform this step, the segment tag range of the animation is displayed in the Curve Editor window ().
In separated mode, you edit the animation curve for each individual segment tag.
By switching the selected segment tag, you can create a separate animation curve.

All animations created in separated mode start from frame 0.

Shortening and Elongating the Created Animation

The feature to shorten and elongate the created animation is only available in separated mode.
You can change the segment length while maintaining the key interval ratio.
In the Edit Animation Segment Tags window, the Frame Segment section contains three buttons for this animation scaling feature that you can use to switch the feature between Off, Decimal Precision, and Integer Precision.
The following table describes the buttons.

 OFF Disables the feature for scaling the created animation.
Decimal Precision Sets the result with decimal-point precision after the segment has been shortened or elongated using the animation scaling feature.
Integer Precision

Rounds the result to an integer value after the segment has been shortened or elongated using the animation scaling feature.

(Rounding can result in fewer keys.)

If ratio linking is enabled, the spacing ratio between keys is maintained when the length of the segment is changed.

Segment Width Before Change (Width = 100) Ratio Linking Enabled and Segment Width Changed From 100 to 50
