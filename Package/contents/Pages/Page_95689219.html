Attribute Information for Primitives

When you use Primitive for the emitter shape or the particle shape, information is displayed for the registered primitives.
Use this to check information about the attributes of polygon models, such as whether the primitives have intended data structures, and whether there is any unnecessary information.
The following example shows the settings when a primitive particle shape is loaded.

The following are descriptions of the information displayed in this figure.

Number of vertices 606 Indicates that there are 606 vertices.
Attribute pos Indicates that there are vertex coordinates.
normal Indicates that there is information for the normal.
tangent Indicates that there is information for the tangent.
uv0,uv1 Indicates that the primitive has the two UV coordinates UV0 and UV1.
Color Indicates whether there is vertex color.
UV set names ( _u0,_u1 ) Indicates the UV set names UV0 and UV1.
