Direction (±)

This section describes the feature for adding a value in a specified direction set to the velocity for all particles in the emitter set.

Overview

The Direction (±) feature adds the specified value in the specified direction to the velocities of all particles in the emitter set.
The specified value is added to the initial velocity of the particles, and changes the initial velocity in each frame.

It can change the initial velocity calculated from the emitter plug-in node and the preview node.
Initial velocity including direction is calculated as follows.

  • Initial velocity of particles: (Scale applied to the initial velocity setting of the emitter node × initial velocity setting of the preview node) + Direction (±)
