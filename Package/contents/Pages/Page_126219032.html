Color Information Window

Use the Color Information window to edit the color information registered in the Color Sample window.

Color Information Window
A Display colors

Shows the RGB color values on the left, and the alpha on the right.

Click to open the Color Picker.
B RGBA Directly set color values for RGB and A.
C A If selected, alpha settings are enabled.
D Name Used to configure the name.
E Comments Used for adding a comment.
F File Used to change file settings.
