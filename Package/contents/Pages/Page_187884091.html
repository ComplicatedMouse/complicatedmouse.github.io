Texture Export Plug-In Presets

When creating any of these recommended textures, it is convenient to use the preset feature of the Photoshop Texture Export plug-in.
To use this feature, open the ExportTexture.jsxinc config file in a text editor and replace its text with the following text to fit your intended purpose.
Related Links For more information, see the config file for the Texture Export Plug-In.

Nonlinear Environments

The following texture export plug-in presets are for use with game applications that do not have linear environments.

var format_presets =
  {
    "vfx_L"              : "unorm_8,rrrr,non,0000",
    "vfx_L_comp"         : "unorm_bc4,rrrr,non,0000",
    "vfx_LA"             : "unorm_8_8,rrrg,non,0000",
    "vfx_LA_comp"        : "unorm_bc5,rrrg,non,0000",
    "vfx_RGBA"           : "unorm_8_8_8_8,rgba,sRGBfetch,0000",
    "vfx_RGBA_comp"      : "unorm_bc7,rgba,sRGBfetch,0000",
  };

Linear Environments

The following texture export plug-in presets are for use with game applications that have linear environments.

var format_presets =
  {
    "vfx_L"         : "unorm_8,rrrr,non,0000",
    "vfx_L_comp"    : "unorm_bc4,rrrr,non,0000",
    "vfx_LA"        : "unorm_8_8,rrrg,linear,0000",
    "vfx_LA_comp"   : "unorm_bc5,rrrg,linear,0000",
    "vfx_linr_RGBA"      : "srgb_8_8_8_8,rgba,sRGBfetch,1110",
    "vfx_linr_RGBA_comp" : "srgb_bc7,rgba,sRGBfetch,1110",
  };

Shaders

These texture export plug-in presets are for use when using particle shaders or when implementing a normal map as a custom shader.
Use the presets as additions to a standard texture.

var format_presets =
  {
    "vfx_flowmap": "unorm_bc5,rggg,indirect,0000",
    "vfx_nml_map"     : "snorm_bc5,,normal,0000",
  };
