Effect Browser

Effect Browser

Describes the Effect Browser, which includes browse and search features specialized for EffectMaker files.
Related Links For more information about the file types used by EffectMaker, see Intermediate Files.

A Address Bar Display/input file paths. Specify the path to use as the search origin in the Search tab.
B Tab

To switch between the available Effect Browser features, click the tabs ().
C Navigation Pane

The tabs () contain various information about and settings available for each feature.
D Search Box Enter keywords in the text box () to filter the files displayed in the file list.
E Toolbar Includes buttons for various file operation tools.
F Options Change the filtering/file display schemes used in the file list.
G File List Displays a list of folders and files that match the selections made in the navigation pane or the currently configured settings.


A Address Bar

This section describes the file path feature.

Left-click the Back () and Forward () buttons to move back and forth through the history of paths that were entered in the address bar.
Left-click the Display History () button on the right side of the text box () to display a list of historical paths. You can then click any of the listed paths to go directly to that path.
Left-click the Browse button () to open a dialog box where you can select a file to open.

B Tabs

The various Effect Browser features.
Click the tabs () to switch between the tabs. The following features are available.

Computer

Access files and folders as displayed in a hierarchical view.
You can also add folders you use often to Favorites.
Favorites Display, add, and edit folders in your Favorites.
Use Favorites folders to group together items in your favorites.
Searching Search for effect data using the specified conditions.

C Navigation Pane

The tabs () display information about each feature and also the available settings.
The following appears for each tab () feature.

Computer Displays a hierarchical file tree.
Favorites Displays your Favorites folders () and Favorites ().
Searching Displays various search condition settings.

For the following file types, additional information and a thumbnail image are displayed when the file is selected.

Texture Files Primitive Files
Additional information and a thumbnail image are displayed for the selected texture file.
Related Links For more information, click here.

If there is a JPG or PNG file with the same name as the selected file, a thumbnail of the image is displayed.
Info

EffectViewer has a feature to output drawing results as PNG files.
You can use this feature to load primitive files to EffectMaker for use as thumbnails.
Related Links For more information, see EffectViewer Operations.

Info

Use the following procedure to create thumbnail images for primitives.

  1. Add the primitive file to a model node () in EffectMaker ().
  2. Start EffectViewer (PC) (), and make sure that the added model is displayed.
  3. To take a screenshot of the EffectViewer window, press the R key.
  4. In Windows Explorer, open the ScreenCap () folder in the directory that EffectViewer was started from.
  5. Screenshots taken in EffectMaker are stored in this folder as image files (PNG files).
  6. Next, put the newly created image file in the same directory as the primitive file, and make sure that the image file and the primitive file have the same filenames.
  7. Finally, select the primitive file from the file list in EffectBrowser (), and make sure that the thumbnail of the primitive is displayed.

D Search Box

Files are filtered according to the keywords entered in the text box (), and the file list is updated accordingly.

Info

The search box supports use of wildcards in the entered text.
For example, enter *.eset for VfxSample () in Favorites to display only ESET files () in the file list.

E Toolbar

Includes a group of tools for file or folder operations. Left-click the buttons to perform the corresponding operations.
The following table briefly describes each tool. Different tools are available within each tab.

Export Exports the selected file to the workspace.
Edit Comments Edit comments for the selected file.
Related Links For more information about comments, see Comments.
Label Color Settings

Sets a label color for the selected file.
Related Links For more information about label colors, see Label Colors.
Favorites Folders

Creates a Favorites folder in the Favorites tab.
Favorites folders () can be used to group together items in your Favorites ().
Removing Favorites Removes an item that was previously added to Favorites () from the Favorites tab.
Adding Favorites Adds the selected file or folder to Favorites ().
Display Windows Explorer Open Windows Explorer and go to the path of the selected file or folder.
Add Sample to Favorites Adds the sample data within a package to Favorites ().

F Options

Folders () can be used to limit the amount of information displayed when the file list is minimized.
Click the down arrow (▼) icons to select the items to hide in the displayed menu. Click Reset to reset to the default display.

The Scaling button () has a different effect depending on the tab, as shown in the following table.

Computer or Favorites tab Files inside the Textures and Primitives folders are displayed in their large sizes.
Search tab Files showing the search results are displayed in their large sizes.

G File List

Displays a list of folders and files that match the selections made in the navigation pane of the currently configured settings.
The following appears for each tab () feature.

Computer Displays files and folders in the folder selected in the navigation pane.
Favorites Displays files and folders in Favorites () or a Favorites folder ().
Searching Displays a list of file or folder hits resulting from the search.

The additional information displayed for each file and folder can be edited from the toolbar or the shortcut menu (right-click to open).
For a complete list of the tools available, see Toolbar.
