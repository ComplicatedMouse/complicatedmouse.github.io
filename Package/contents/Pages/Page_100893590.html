GLSLC

Overview

GLSLC outputs the compilation results for a shader for NVN, based on the input shader source and variation information.

How the Tool Is Provided

Provided in the form of NSO files for the development hardware environment, and in the form of DLL files for the Windows environment.

A LIB file is also provided as an import library for use together with the DLL.

Using GLSLC in a Production Environment

Because of the following advantages, we recommend using shaders that were pre-compiled in Windows for the production environment.

  • They can reduce the CPU time for compiling shaders on the production device.
  • They can reduce the amount of memory required for compiling shaders on the production device.

However, you can also use compile functionality for shaders on the production device within a production environment. Note that in such cases, these advantages are not realized.

Executable Tools Using GLSLC

This section introduces executable tools that use GLSLC.

GLSLC Shader Compiler (GameWorks)

The NVIDIA GameWorks package provides an executable shader compiler that uses GLSLC. For more information, see the documentation in NVIDIA GameWorks.

Shader Converter (GFX)

The gfx Library provides Shader Converter (EXE), which is an executable that uses GLSLC. For more information, see Shader Converter (EXE).
