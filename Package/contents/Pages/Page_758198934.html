Shaders Tab

The Shaders tab displays all of the graphics shaders that were recorded as in use during a profile. This tab will only appear for profiles where GPU Trace was enabled on the Sampled Profile or Instrumented Profile tabs.

Shaders Table

The Total column contains the percentage of total shader execution time that was spent in this shader.

The Average column contains the average time spent per invocation of this shader.

The Invocations column contains the number of shader invocations seen over the duration of the profile.

The Name column contains the string defined for the shader, using the function nvnProgramSetDebugLabel(). If the shader was not given a name, then the shader is given a unique number.

Clicking on a row will select the shader and draw it in the Timeline tab. Invocations will be combined into a single block as long as at least one Draw is referencing a Shader.

Shaders Tab Toolbar

The following controls are in the Shaders tab toolbar.

Deselect All Button

To quickly deselect all selected items, click on the Deselect All button.

Select Top Button

Selects the top 50 items.

Copy Button

The Copy button will copy to the clipboard whatever is currently shown using the current sorting and filtering. You can then paste the content into an email, document, or Excel spreadsheet. Because the copied content uses tab delimiters, it will retain the column organization when pasted into an Excel spreadsheet.

Selected Only Button

The Selected Only button forces the list display to only show items which are currently selected. If this mode is enabled and an item is deselected, it will be removed from the displayed list.

Filtering Gfx Groups

Shaders can be filtered using the Filter text box at the bottom of the Shaders tab. This behaves similar to the Filter text box on the Functions tab.
