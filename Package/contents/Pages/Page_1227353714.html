Info

This feature is available in NX Add-On 14.1.0 and later.

Introduction

DumpViewer is a tool that reads the information from NXDMP files in units of files or units of folders and lists the information in the host environment.

How to Use

Starting the Tool

To start the tool, run the program from the following location.

%NINTENDO_SDK_ROOT%\Tools\DumpViewer\DumpViewer.exe


After you start the program, a window like the following is displayed.


The window is divided into a left side and a right side.

The left side displays a list of the loaded NXDMP files, and the right side shows the details for a selected NXDMP file.

Reading NXDMP Files

There are two ways to read NXDMP files.

  1. From the upper-left menu, select File > Read File (or Read Folder) and then select the file or folder in the dialog box.
  2. Drag an NXDMP file or folder from Windows Explorer to the DumpViewer window.


If the file is successfully loaded, you can view the NXDMP file information, as shown below.


When a folder is loaded, only the NXDMP files in that folder are loaded.

In other words, folders are not read recursively.

Switching the Layout

There are two buttons at the bottom of the window in the middle, and clicking them switches the layout on the left side of the window.

Layout in Tabular Format

By default, the left side of the window is laid out in tabular format, in which case the layout looks like the following.

To change the information that is shown in each column in tabular format, from the menu select View > Add Column.

Exporting Screenshots

Right-click the screenshot displayed on the right side of the window and select Export as a file to export the screenshot as an image file.
