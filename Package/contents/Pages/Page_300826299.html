Copying and Pasting Nodes

This section describes the feature for copying and pasting nodes.

Pasting Nodes as Child Nodes

Select the copy-source node and then select Copy from the shortcut menu.
Select the copy-destination node and then select Paste as Child Node from the shortcut menu.
The copy-source node tree is pasted as a child node of the copy-destination node.

Pasting Nodes

Select the copy-source node and then select Copy from the shortcut menu.
Select the copy-destination node and then select Paste Node from the shortcut menu.
The copy-source node tree is pasted to the copy-destination node.

Pasting Properties

Select the copy-source node and then select Copy from the shortcut menu.
Select the copy-destination node and then select Paste Property from the shortcut menu.
The copy-source node property is pasted to the copy-destination node.
The child nodes are not changed.
