Workflow

This section describes the workflow, from starting 3DEditor to completing the task.

3dSampleViewer is used for the description.
3dSampleViewer is the viewer used for the tutorials in this documentation.
Note that it cannot be used when you are creating games.

Starting Tasks

Start 3DEditor and 3dSampleViewer and prepare the environment for previewing.

Starting 3DEditor

Double-click 3dEditor.exe, which is located where NintendoSDK was installed in the following location.

3DEditor

 %NINTENDO_SDK_ROOT%\Tools\Graphics\3dEditor\3dEditor.exe

Starting Viewer

To start SampleViewer, double-click 3dSampleViewer.exe, found in the following location.

3dSampleViewer

 %NINTENDO_SDK_ROOT%\Samples\BuiltApplications\3dSampleViewer\3dSampleViewer.exe

Info
Related Links If you cannot start 3DEditor and 3dSampleViewer, check the NintendoSDK operating environment again, as described in NintendoSDK Overview / Operating Environment.

Previewing a Model

This section describes the procedure for displaying actual sample data in the viewer. This assumes you have completed preparation of the preview environment.

(1) Load data for previewing into 3DEditor.

For example, load the sample data of the MaleA file, found in the following location.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\town\MaleA\MaleA.fmdb

To load this file, from the 3DEditor menu, select File > Open File and specify the file, or drag it into 3DEditor.


(2) Set the platform.

Use the platform toggle feature in the 3DEditor toolbar to select the platform for the connected application.



(3) Connect to the viewer.

From the 3DEditor menu, select Preview > Connect to Application, or double-click the connection icon () in the toolbar.

Info

If you cannot connect to the viewer, check the following items.

  • Nintendo Target Manager is started.
  • The viewer for the corresponding platform is running.


(4) Confirm that the loaded data displays in the viewer.

If you load MaleA and successfully connect to the viewer, the MaleA model is displayed as follows.


If MaleA is displayed in 3dSampleViewer, 3DEditor and 3dSampleViewer are properly connected.

Editing With 3DEditor

The following items can be edited (and in some cases created) using 3DEditor.

  • Model parameters (billboards, user data, and so on)
  • Material features (texture samplers)
  • Shader assignments and parameter settings for materials
  • Textures (user data, and so on)
  • Material animations
  • Bone visibility animations
  • Skeletal animations (user data, and so on)
  • Scene animations

Related Links Save the changes by either using the save methods in the File menu on the main menu, or by right-clicking the changed file and using the shortcut menu to save the data.

Exiting 3DEditor

Close all open files and exit 3DEditor by using any of the following methods.

  • From the main menu, select File > Exit.
  • On the main window, click Close.
  • Press CTRL+Q.
Info

If there are any unsaved files, the Save Changed File dialog box is displayed. Save files as needed, then exit the tool.
