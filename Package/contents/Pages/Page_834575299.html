LayoutEditor

This page introduces the features of LayoutEditor version 3.28.0.

Added functionality to render any text that sticks out beyond the pane size region.

Added functionality to render any text that sticks out beyond the pane size region when you are using the capture feature in a Text Box pane ().
You can enable this feature using <IsSpreadPaneSizeOfItalicEnabled> in the project settings.

The following Text Box pane () shows a text with a capture effect that has been italicized, and the right end of the string sticks out past the pane size region.

Project Configuration File Format

The project configuration file is an XML file used for recording common settings for the game project.

The <IsSpreadPaneSizeOfItalicEnabled> tag sets whether to clip the text to the pane size when using a capture effect in a text box pane.

When enabled, the text is not clipped to the pane size when a capture effect is used.
Capture Pane Describes the capture pane, which captures the result of rendering a framebuffer or a layout and converts it to an image.
Capture Effect (Drop Shadow) Describes the Capture Effect (Drop Shadow) feature, which provides an easy way to change the appearance of panes by generating various effects from the pane shapes.
Info

This feature does not support automatic wrapping based on per-character transformations and pane size. Depending on the usage, the following situations apply.

  • When the per-character transformation feature is being used, text that extends outside the pane size region is not shown in LayoutViewer.
  • Text that extends beyond the pane size region due to line breaks in the text is shown with lower quality in LayoutViewer.

Pane x Background

Blends the pane texture image in the standard combiner mode.

The colors in the alpha part are multiplied by the background, which does not produce the expected result.

Pane x Background

The alpha part of the pane is adjusted in detailed combiner mode to be white.
By using Multiplicative Blending in this state, it can be neatly combined with the background as seen in the Combined Results.
