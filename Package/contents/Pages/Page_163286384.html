Other Parameters

This page describes the parameters that were not described in the tutorial that uses the Town model.

Protecting the Overall Shape While Removing Small Details

In certain situations, for instance, when simplifying angular shapes such as mechanical objects, the Smoothing option can be useful for maintaining the overall shape of a model, while providing relief from small surface recesses and protrusions. However, this parameter can occasionally be difficult to use in a standardized way for all files.

Running a Reduction of a Model Composed of Only Flat Planes

Most polygon reduction algorithms prioritize minimizing the change in the volume of a model. For this reason, reductions of models composed only of faces on the same plane, where volume cannot be found, are not possible. In this case, offset the height to allow the volume to be calculated. This offset affects the calculation of the optimized position after the reduction. We recommend setting --optimize-position-policy to OptimalOnEdge.

For more information and to see the result of running a reduction, see Applying a Height Offset.

Running a Reduction Considering Blend Shapes

When blend shapes are set, you can run a reduction that accounts for them. For more information, see Reductions Considering Motion.

Aligning Triangular Shapes

In the case of extremely small triangles and triangles of differing sizes, specular and other rendering may produce unfavorable effects. In this case, you can use an option to align the size and shape of triangles after the reduction. For more information, see Controlling the Shapes and Areas of Faces Created During Simplification.
