Maya Plug-In

Added a feature to the Import plug-in for importing materials as extended materials.

You can now use the Material Type option in the Import plug-in to specify the node type of the material to create.

By default, this type cannot be changed from blinn (standard material), but now you can select another material by specifying extended material attribute information (material_attr_info_????) in the config file of the Export plug-in.

Info

However, currently you cannot select materials that cannot be created.
If a plug-in is required for the material, you must load it before you run the Import plug-in. (For Arnold, load mtoa.mll).
All materials imported at the same time have the same type.

In the example presented in the following figure, a standard surface shader (standardSurface node) is added to the config file.

material_attr_info_standardSurface="baseColor=albedo,_a,as_diffuse; opacity=opacity; emissionColor=emission; normalCamera=normal; specularColor=specular; metalness=metal,_m,no_color"
