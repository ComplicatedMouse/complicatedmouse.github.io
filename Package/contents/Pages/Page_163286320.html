5. Protecting UV Hard Edges and Performing Reduction

Open edges in UV space are called UV hard edges. In addition, enclosed shapes in UV space are known as UV shells in Maya but are more generally referred to as UV atlases. That is, UV hard edges are open edges of UV shells.

For more information about UV hard edges and the protection angle, see Open Edges and Protection Angle.

Changing the setting to --property-flag Position|UV makes it possible to protect UV hard edges. This documentation describes how to protect UV hard edges by specifying the angle of connected UV hard edges.

As shown in Figure 7, as with open edges, when two UV hard edges are consecutive, set the angle Θ to specify not making large changes to the outer circumference shape of atlases. This setting is exactly the same as with open edges.


Parameter D
3dIntermediateFileOptimizer.exe bg_WhiteTown.fmdb --polygon-reduction --polygon-reduction-options "--target-rate-lod-level1 0.5 --target-rate-lod-level2 0.25 --property-flag Position|UV --optimize-position-policy OptimalOnSpace --uv-weight 0.1 --lock-uv-hard-edge-angle 90"

Use the following batch file.

%NINTENDO_SDK_ROOT%\Samples\Sources\Tools\3dTools\3dIntermediateFileOptimizer\PolygonReductionTutorial \ExecuteParameterD.bat


This technique is essential for solving shell boundary issues, which the previous parameter configurations have only somewhat mitigated.

Strong UV hard edge protection (--lock-uv-hard-edge-angle 90 ) is added to Parameter C. This parameter is called Parameter D. For more information about set angles, see Intermediate File Optimizer.
