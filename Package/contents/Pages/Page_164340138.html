Selecting a Texture Format

This page describes how to select a recommended texture format to use in effects.

The available texture formats are listed in the Vfx preset names for Photoshop.
Related Links For more information about the Vfx presets, see Texture Format Presets.


Select the texture to use while referring to the following selection flow.

Linear Environment Textures

Use the following workflow for guidance in selecting a texture format to use in linear environments.

The following table illustrates the correspondence between the preset names and the texture formats.
