Emitter Set Files

This page describes the effect data intermediate file, called the emitter set file.

Content Saved in the Emitter Set File

Emitter sets are saved as emitter set files (.eset) and treated as data for a single effect.
Developers manipulate the rendering position, color, and other elements at the unit of the emitter set.
An emitter set file contains the following information.

  • The emitter set's settings
  • Texture filename
  • Primitive filenames

Opening Emitter Set Files

When an emitter set file (ESET file) is opened, the associated files are also loaded.
If the associated files are not found, EffectMaker displays a warning and those texture files are not loaded.

The following table shows related file types.

Texture Files Image files used in emitter set files.
Primitive Files Model files for primitives used in emitter set files.
