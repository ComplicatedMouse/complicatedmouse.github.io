Translation Distance

This section describes the Translation Distance feature for setting the distance of travel with the Move forward in the specified direction auto-translation type.

Overview

Translation Distance sets the distance repeatedly translated when Move forward in the specified direction is selected as the auto-translation type.
The distance of the translation can be increased by increasing this value.

The following overview describes the settings ().

Translation Distance

Use the dial () or text box () to specify the distance of move forward.
