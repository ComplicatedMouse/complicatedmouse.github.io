Depth Stencil State

Depth Stencil State

The depth stencil state collectively expresses several states of the constant features of the depth test and stencil test in the pipeline.

Information

Use nn::gfx::DepthStencilStateInfo to define the depth stencil state information.

Depth Comparison Function

To set the comparison function used in the depth test, use the SetDepthComparisonFunction function.

Depth Test

To enable or disable the depth test, use the SetDepthTestEnabled function.

Depth Writing

To enable or disable depth writing, use the SetDepthWriteEnabled function.

Stencil Test

To enable or disable the stencil test, use the SetStencilTestEnabled function.

Stencil Masks

To set the mask applied to stencil reading, use the SetStencilReadMask function. To set the mask applied to stencil writing, use the SetStencilWriteMask function.

Stencil State

To edit the stencil state information for front-facing polygons, use the EditFrontStencilStateInfo function. To edit the stencil state information for back-facing polygons, use the EditBackStencilStateInfo function.

Depth Bounds Test

Enable or disable the depth bounds test using SetDepthBoundsTestEnabled.

Directly set the range for the depth bounds test in the command buffer using nn::gfx::CommandBuffer::SetDepthBounds.

Stencil Operation

To set the stencil operations to apply to each situation, use the following functions.

Situation to Apply Function
When the stencil test fails SetStencilFailOperation
When the depth test fails SetDepthFailOperation
When the depth test succeeds SetDepthPassOperation

Stencil Comparison Function

To set the comparison function used in the stencil test, use the SetComparisonFunction function.

Stencil Reference Value

To set the stencil reference value used in the stencil test, use the SetStencilRef function.

Object

Use nn::gfx::DepthStencilState to define the depth stencil state object.

Initialization and Destruction

The depth stencil state is initialized and finalized according to the rules for objects

Setting Depth Stencil States

When you are using an execution model that uses a pipeline, information about the depth stencil state is included in the pipeline information.

When you are using an execution model that does not use a pipeline, you set the depth stencil state by calling the SetDepthStencilState function of the nn::gfx::CommandBuffer class.
