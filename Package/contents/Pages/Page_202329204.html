Select

This page describes the additional Select menu for LayoutEditor.

Select All Selects all placed panes.
Deselect Undoes pane selection.
Invert Selection Reverses the selection of panes.
Unselected panes are selected, and selected panes are cleared.
Select the pane in front of the current one Selects the pane that is one pane in front of the currently selected pane.
Select the pane behind the current one Selects the pane that is one pane behind the currently selected pane.
Common Selects the pane using a common color.
When run from the following menu, selects the pane with a common color.
  • Color Interpolation
  • Vertex Color
  • Text Box Pane
  • Shape
  • Capture Effect
  • Constant Colors
  • Capture Pane
