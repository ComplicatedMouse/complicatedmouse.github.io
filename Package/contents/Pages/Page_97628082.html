Multiply by the Emitter Color

This section describes the Multiply by the emitter color feature, which multiplies the Global Multiplier Color for all emitters in the emitter set by the specified value.

Overview

The Multiply by the emitter color feature multiplies the Global Multiplier Color in the emitter node's Color tab by the specified value.
The feature can control color in the emitter time animation for the specified color.
This section provides an overview of the settings ().

Multiply by the Emitter Color

Select or clear the check box () to enable or disable emitter colors.
Enabling this setting displays the Multiply by the emitter color group () of settings. The group has the following settings.

Color 0, Color 1

Color 0 multiplies the Global Multiplier Color 0 for all emitters in the emitter set by the specified value.
Color 1 multiplies the Global Multiplier Color 1 for all emitters in the emitter set by the specified value.
