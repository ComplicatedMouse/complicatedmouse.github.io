Base Period

This section describes the Basic period feature, which sets the basic value used to determine the period for the components of the combined waveform.

Overview

The Basic period feature sets the period of the waveform elements that is used when particles move.
The following figure illustrates the ↔ range being set using frames.

When a small value is set, particles will move vigorously.
When a large value is set, particles will move slowly.


The following overview describes the settings ().

Base Period

The base period value for waveform elements is set using either a dial () or a text box ().
When a decimal value is input, the decimal portion is rounded and it is converted to an integer value.

Example Configurations

The following images compare what happens when GPU noise is applied to particles and the Basic period value is changed.
Noise strength is set to (x, y, z) = (0, 2, 0).

A Base Period 1000 The default setting.
Particles move slowly along the waveform, taking 1,000 frames to move one period.
B Base Period 100 Compared to A, particles move vigorously, taking 100 frames to move one period.

The following images compare two effects with different base period values.

A Firefly Making the base period long, creates an effect that mimics the movement of a firefly.
B Bees Making the base period short, creates an effect that mimics the movement of bees buzzing around a space.
