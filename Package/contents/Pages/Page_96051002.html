Global Multiplicative Color and Alpha

This section describes the Global Multiplicative Color and Alpha feature to affect the color and alpha for all particles emitted from the emitter.

Overview

The Global Multiplicative Color and Alpha group is a feature that can multiply the values set to the color and alpha settings for all particles emitted from the emitter.
The following overview describes the settings ().

Color 0

Multiplies Color 0 for all particles emitted from the emitter.

Color 1

Multiplies Color 1 for all particles emitted from the emitter.

Alpha 0

Multiplies Alpha 0 for all particles emitted from the emitter.

Alpha 1

Multiplies Alpha 1 for all particles emitted from the emitter.
