Preview Node

This section describes Preview Node features.
You can save previews as preview files (PREV files). You can also use EffectViewer to check the results of your preview settings.

Basic Settings View file information and configure the basic preview settings.
SRT Transform Configures aspects of emitter sets that can be controlled in the game application, and settings linking 3D models and effects.
Auto-Translate Configures the Auto-Translate feature, which allows you to check how effects look on the Viewer when emitter sets are actually moved.
Emitters Configures settings of the Emitter feature of the Emitter node that can be controlled in the game application.
Emission Configures settings of the Emission feature of the Emitter node that can be controlled in the game application.
Particle Configures settings of the Particle feature of the Emitter node that can be controlled in the game application.
Color Configures settings of the Color feature of the Emitter node that can be controlled in the game application.
Scale Configures settings of the Scale feature of the Emitter node that can be controlled in the game application.
