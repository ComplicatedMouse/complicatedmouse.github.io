NX Performance Management

Introduction

Objectives and Content

This section considers ways of thinking about and using the performance characteristics of the NX platform, and describes ways to support the platform in both console mode and handheld mode.

The NX system requires a different approach to performance management than legacy systems due to its nature as an integrated dual mode device.

Intended Audience

The intended audience of this document is software developers who are writing game applications to run on the NX platform.

When to Read This Document

Because this documentation contains information about program design, we recommend that you look through it during the early stages of NX application development.
