Creating Textures for Pattern Animations

This section describes how to use the Photoshop plug-in for combining images to create textures for pattern animations (flipbook textures). The section uses a flame pattern as an example.
This plug-in creates a single texture comprised of several distinct images.
The set of distinct images can be animated as a texture pattern animation by discretely moving the texture UV.

Preparing to Create the Texture

To prepare to create the texture, start an instance of Photoshop with the plug-in installed.

Related Links For more information about setting up the Photoshop plug-in, see the Introduction to the Photoshop plug-in.

Opening the Plug-In for Combining Images

Merges the image files prepared in the sample, using the image-combining plug-in.
From the Photoshop menu, select File > Automate > Nintendo Unite Images, and open that plug-in.

Setting the Plug-In for Combining Images

Set the folder in which all the images to be combined are gathered.

Item Settings path (for the image data to be combined) Description
Input Folder
 		\Samples\Sources\Resources\Vfx \Eset\Textures\UniteImages\ Specify the folder in which all the images to be combined are gathered.

Set the image dimensions and the number of images to merge. Click the OK button to merge the images.

Item Value Description
Number of Pixels in an Image

Width : 128
Same Value : Selected

Sets the width and height of the images being combined to 128.
Number of Images in a Row (in a Column)

Horizonal: 8
Vertical: 4
Same Value: disabled

Sets the number of images to place in a row to 8 and the number of rows to 4.
Process Unite Alpha: disabled Disable this option when there is no alpha for the images to merge.
