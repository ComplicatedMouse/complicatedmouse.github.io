History Acceleration(±)

This section describes the History Acceleration(±) feature for setting the acceleration to apply to the overall history along each axis.

Overview

The History Acceleration(±) feature sets the acceleration to apply to the overall history along each of the x,y, and z axes.
By setting an acceleration for the history, you can create stripes whose tail ends gradually stream in a certain direction.
The following overview describes the settings ().

History Acceleration(±)

Set the acceleration for the history on each of the x,y, and z axes by either using the dials () or entering values in the text boxes().
The larger the value, the greater the movement of the History.
