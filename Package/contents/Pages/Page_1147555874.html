g3d

The Town scene drop-down list does not work when running the G3dDemo sample for the x64 platform under certain conditions.

Symptoms

When you create a Develop or Release build of the G3dDemo sample for x64 platforms in the Visual Studio 2019, no drop-down list appears when the Town Scene menu's Model Settings > env text box or ▼ button are selected, and no other models can be selected.
Other drop-down lists work normally.

Workaround

You can work around this problem by running the sample on a different platform, running it on a Debug build, or using Visual Studio 2017 to do the build.

Fix Schedule

This bug will be fixed in a future version.

tool

Using DirectXShaderCompiler to compile source code larger than 64 KiB causes an internal compiler error.

Symptoms

Using DirectXShaderCompiler to compile source code larger than 64 KiB causes the following message to be displayed.

Internal compiler error: access violation. Attempted to read from address 0x[address]

Workaround

Divide up the source file and use #include so each chunk is less than 64 KiB.

Fix Schedule

This bug will be fixed in a future version.
