Checkers Tab

The Checkers tab contains more than a dozen independent static and dynamic checkers. The static checkers analyze the assembly code and symbols while the dynamic checkers operate on the profile data that was just taken. Checkers are automatically run after a profile is taken and only checkers that triggered are shown.

Manage Checkers Toggle Button

The Manage Checkers toggle button allows you to view all checkers after taking or loading a profile. If the Manage Checkers toggle button is unchecked, then checkers that are disabled or have no results are hidden. If the Manage Checkers toggle button is checked, then you can view all checkers or re-enable any disabled checkers.

Reset All to Defaults Button

The Reset All to Defaults button will go through each checker, return its parameters back to their default values, and automatically rerun the checker. The change back to the defaults will be permanently saved.

Checkers

Checkers can be expanded or collapsed by clicking on their heading.

Checkers fall into one or more categories: Critical, Performance, Quality, and Info. Each checker lists the categories it belongs to on the right side of its heading.

All checkers follow a similar format, as described below.

Checker Categories

  • Critical: Problems detected with the accuracy or integrity of the data. Focus on correcting critical checkers before moving on to other categories.
  • Performance: Issues detected that affect the performance.
  • Quality: Issues detected that affect code quality.
  • Info: Provides general information about the code or application. Unlike other checker categories, these checkers do not identify problems to fix.

Top Buttons

  • Copy Text: Copies the text in the Summary, Explanation, and Suggestions to the clipboard.
  • Copy Table: Copies the contents of the Results Table to the clipboard.
  • Disable Checker: This will disable the checker in the current profile and prevent it from running on all future profiles. To re-enable the checker, enable Manager Checkers, find the checker that was disabled (marked with Checker Disabled on the right side of the header), and click the Enable Checker button in the top right corner.

Summary

Displays a summary of the checker's results. To copy this text to the clipboard, click the Copy Text button.

Checker Parameters

Below the Summary and above the Results Table there may be spinner controls to change and customize the checker's parameters. The default settings should provide the best results. If a checker does not support changing any of the parameters, this section will not appear in the checker.

If you change these parameters, the data in the Results Table will visibly shift to gray to indicate that the data is stale and the checker should be rerun. The changes made to the parameters will be permanently saved.

  • Rerun Checker: Executes the checker again. You will want to rerun the checker if you change any of the checker parameters.
  • One or more spinner controls to change a parameter.
  • Reset to Defaults: Revert all parameters in this checker to their default values.

Results Table

The Results Table contains all results from running the checker. The column headings can be clicked to resort the table. To copy the results of the table to the clipboard, click Copy Table.

Right clicking on a row might provide additional options that can be applied to that data or function.

Explanation

The Explanation text gives an overview of what the checker is testing and why it is relevant. To copy this text to the clipboard, click the Copy Text button.

Suggestions

The Suggestions text gives concrete advice on next steps to take. To copy this text to the clipboard, click the Copy Text button.


Filters

The filter bar at the bottom of the Checkers tab changes which checkers are visible. Select a checker category to see only checkers that are a part of that category. If more than one filter is applied, checkers that are in any of the selected categories will be visible. If no filters are applied, all checkers will be visible.

  • Critical: Show checkers that are in the Critical category.
  • Performance: Show checkers that are in the Performance category.
  • Quality: Show checkers that are in the Quality category.
  • Info: Show checkers that are in the Info category.
