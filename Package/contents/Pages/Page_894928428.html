Insert left and right margins

The Insert left and right margins feature inserts margins based on the text width compared to the pane width when the Place text in a circular shape feature is used, and Set the origin point for conversion as the center of the text box is enabled.
It is used to retain proportional information for characters and set equal spacing between them when placing text in a circle, even when the number of characters is reduced.

The following images compare Place text at fixed intervals and Insert left and right margins, which are available when Set the origin point for conversion as the center of the text box is enabled.

Place characters at fixed intervals Insert left and right margins

You can configure Kerning.
The space before and after characters with narrow character width like the letter "i" becomes wider.
 Kerning is determined by the text width relative to the pane width.
Even characters with a narrow width like "i" can be set to equal intervals.
