UV Scroll Animation

This section describes the feature for scrolling texture UV.

Overview

The UV Scroll Animation feature enables you to scroll a registered texture by moving its UV coordinates.
By combining this feature with features like Repetition count and Wrap, you can do things like repeatedly display a texture pattern and make a texture gradually disappear.
The following overview describes the settings ().

UV Scroll Animation

A feature for setting the position of the texture UV coordinates on each axis separately.
X corresponds to texture coordinate U, and Y corresponds to texture coordinate V.

Enable this setting to display the UV Scroll Animation () settings. The group has the following settings.

Add-on value

Sets the value to add to the texture UV positions in every frame.
This add-on value is applied after the emitter starts.

Initial value

Sets the initial texture UV positions.

Random initial values

Specifies the use of random numbers in a set range for the initial values.
The upper and lower limits of the random seed for the initial value randomly change. The following formula calculates a lower limit for the random number.
