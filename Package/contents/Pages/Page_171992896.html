Batch-Processing for Compute Shaders

This page provides an overview of a feature for batching compute shader processes to optimize GPU usage when drawing emitters.

How to Use

Use the following procedure to batch together compute shader processes.

  1. When initializing the effect system, enable compute shader batch-processing mode in nn::vfx::Config.
  2. Call the nn::vfx::System::BatchCalculationComputeShaderEmitter function to batch-process compute shader calculations.
// Initialize effect.
nn::vfx::Config  config;
   :
   :
config.SetEnableComputeShaderBatchProcess();
nn::vfx::System* pVfxSystem = new nn::vfx::System( config );


// Effect rendering process.
pVfxSystem->SwapBuffer();
pVfxSystem->SetViewParam( processingIndex, &view );

// Batch together compute shader calculations.
pVfxSystem->BatchCalculationComputeShaderEmitter( pCommandBuffer, nullptr );

// Draw emitter.
pVfxSystem->Draw( processingIndex, pCommandBuffer, groupId );
