BfttfMaker

Introduction

Purpose of This Documentation

This document describes BfttfMaker, which is a tool for creating BFTTF and BFOTF files.

How to Use

The BfttfMaker tool encrypts TTF and OTF files to generate BFTTF and BFOTF files. It is located in Tools\CommandLineTools\Fontll\BfttfMaker.exe.

The tool runs in the console and takes the following parameters.

Parameter #1 The path of the TTF or OTF file to input.
Parameter #2 The output path for the BFTTF or BFOTF file.
Parameter #3 The target platform. More than one platform can be specified, delineated by commas.

Example:

BfttfMaker.exe .\input.ttf .\output.bfttf Win

The BFTTF file is encrypted with a different key for each platform so it is not loaded by platforms other than the target.

The filename extension of the output file is .bfttf when encrypting TTF files and .bfotf when encrypting OTF files.

Handle TTC in the same way as TTF, by generating a BFTTF.

If multiple platforms have been specified, the tool loads the file and creates an encrypted BFTTF/BFOTF for each platform.
