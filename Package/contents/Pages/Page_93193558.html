Animation Controls

This section describes animation controls.

Related Links Animations can also be controlled using shortcut keys.


A Range slider for the frame at which to begin playback.
You can change the first frame by either entering a value or dragging the green handle.
B
The Time Slider.
Right-clicking the Time Slider sets the current frame at the mouse position.
Dragging with the mouse allows you to move the position of the current frame.
C Range slider for the frame at which to end playback.
You can change the last frame by either entering a value or dragging the green handle.
D The text box for the current frame.
You can enter a value to move the current frame selected on the Time Slider to the specified position.
E
Sets the animation playback method.
F
Controls animation playback.
G

The frame scale. Sets how many frames to advance every 1/60th of a second.
Example: Setting this value to 0.5 results in playback at half of normal speed, while setting a value of 10 results in playback at 10 times normal speed.

This value is also used as the value for Snap to Frame Step in the next section.
H

A check box for Snap to Frame Step.
When selected, it snaps to the frame scale value entered in G when the Time Slider is used.

Range Slider

Allows you to specify which frames of an animation to play back.
You can set the first and last frames by either entering them as values or by dragging the range sliders.

Animation Playback Methods

The following playback methods are available.

Item Description
Auto Automatically loops looped animations if any are included in the current animation set of any of the loaded models.
Loop Forcibly loops animations.
Once Forcibly plays animations one time only.

Animation Controls Icon

The following table describes the icons that are used to control the animation.

Icons Description
Stop button.

Frame reverse button. Pauses the animation and goes back one frame.

Holding the button causes the key to repeat.
Play/Pause button.

Frame advance button. Pauses the animation and advances one frame.

Holding the button causes the key to repeat.


Snap to Frame Step

You can snap to the frame step. You can change the frame scale value and value to snap.
