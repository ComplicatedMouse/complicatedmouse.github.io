Emission Interval Type - Time

This section describes the Time emission interval type.

Overview

The distance type emission interval allows the particle emission to be set by time.
Setting the emission interval to time controls the number of particles emitted at the same time, or the emission rate.
You can also make the Emission rate or Emission interval random.

The following overview describes the settings ().

When you set the emission interval type to Time, the Emission Interval Type - Time () setting is displayed. The group has the following settings.

Emission Rate Sets the volume of particles the emitter emits at one time.
Emission interval Sets the time interval (number of frames) at which particles are emitted.
