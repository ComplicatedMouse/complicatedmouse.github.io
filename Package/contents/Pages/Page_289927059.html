SoundMaker

Default Values Can be Set for Sound Parameters

Added a feature for setting the default parameter values when a new item is created.

A file with the default settings named Project name.fspj.defaultparam is output to the same folder as the sound project.

If you want to change the parameter defaults, edit this default settings file, and set the default values.

For more information, see Sound Parameter Initial Settings.
