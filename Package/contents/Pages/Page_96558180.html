Overview

This section describes the Blend Preset feature used to set how particle draw and the background is blended together.
Set to Z Buffer, Alpha Test, Normal or Ignore Z to display the Blend () settings.
The following overview describes the settings ().

Blend Preset

The following is a list of material images used in the description.

Src_RGB The RGB color of the particle which has been decided to be used by the write test.
Src_A The A color of the particle which has been decided to be used by the write test.
Fb_RGB The RGB background color of the write destination.

Select the method for blending particles with the background during rendering from among the following presets.

General alpha blend Src_RGB × Src_A + Fb_RGB × ( 1.0 - Src_A ) Setting for a blend in which both the particle and the background maintain their color.
Additive translucent Src_RGB × Src_A + Fb_RGB × 1.0 Adds the particles and the background.
Subtractive translucent Src_RGB × Src_A - Fb_RGB × 1.0 Subtracts the particles from the background.
Multiplicative Src_RGB × 0.0 + Fb_RGB × Src_RGB Multiplies the particles and the background.
Screen Src_RGB × ( 1.0 - Fb_RGB ) + Fb_RGB × 1.0 Additively mixes the particles with the background.
Additive translucent is enabled if you use backgrounds that are very bright.
Premultiplied alpha Src_RGB × 1.0 + Fb_RGB × ( 1.0 - Src_A ) Use this setting to maintain the correct brightness even for the parts of the texture with small alpha values.
It is effective when you are using glare and the translucent parts are not bright as expected.
