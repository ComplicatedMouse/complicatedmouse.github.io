Supported Features on HDEV

The HDEV development hardware supports the following features.
(There are no restrictions on using these features on the SDEV and EDEV except if explicitly mentioned in the documentation for each feature.)

  • Target Manager
  • ControlTarget
  • TargetShell
  • InitializeHdev
  • InitializeHdevWin
  • SystemUpdateHdev
  • DevKitSetup.ps1
  • RunOnTarget
  • Settings Manager
  • BackupSaveData
  • DevMenu
  • DevMenuCommand
  • DevMenuOverlay
  • VSI
Note

Some features included in the preceding list may not actually be usable on the HDEV due to performance limitations.
For more information, see the relevant documentation.
