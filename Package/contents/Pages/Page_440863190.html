3DEditor

Added the display of tessellation shader definitions.

The property window for shader definitions now can show the tessellation control shader and the tessellation evaluation shader.

To check these settings in the property window, open 3DEditor in file view, and double-click the shader definitions file (FSDB file).

You can now restrict the range of values for child materials from the parent material.

The shortcut menu that opens when you right-click label names and other elements can now be used to set restrictions on child material values.
Currently, you can only configure the setting for option variables and rendering information.

Open the window from the shortcut menu to configure the settings.

Maya Plug-In, 3ds Max Plug-In

With the Maya plug-in, baked animations are now exported more quickly and in parallel.

Export time has been reduced by speeding up the process of creating animation curves and performing the process in parallel.

To disable this kind of parallel processing, specify the following line in the config file. (This feature is enabled by default.)

make_anim_curve_parallel="false"


You can also disable parallel processing of pre-export scripts.

NintendoOverwriteExportOption("make_anim_curve_parallel", "false")

3D Command-Line Tools

Fixed a bug where an invalid vertex attribute was generated if the quantization type of the blend weight vertex attribute was type unorm16.

Fixed a bug where the 3D binary converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe) generated an invalid vertex attribute when it converted intermediate model files (FMDB and FMDA files) if the files contained a vertex attribute indicating a blend weight set with a unorm16 quantization attribute. (That is, the vertex attribute with blendweight contained in hint.)

This bug occurred when you used the feature to change the skinning weight precision, which was added in 7.1.0.

The vertex attribute packer no longer includes vertex attribute variables with nonzero component values in the shader archive file.

Vertex attribute variables (nn::g3d::ResAttrVar) are no longer included in nn::g3d::ResShadingModel if the component of the layout modifier in the shader is set to a value other than 0.

3D Vertex Attribute Packer

Describes the feature for packing vertex attributes.

Polygon reduction can now take vertex color into consideration when reducing polygons.

You can now specify Color representing the vertex color in a --property-flag sub-option. With this specification, vertex color is taken into consideration when reducing polygons. Use --color-weight if you want to specify a weight when calculating the cost of vertex color in polygon reduction.

In addition, added the --allowable-color-gradient-threshold sub-option to protect edges with differences in vertex color.

3dIntermediateFileOptimizer.exe --polygon-reduction --polygon-reduction-options="--target-rate 0.5 --allowable-color-gradient-threshold 0.1" input.fmdb
