Maya Plug-In

Changed the Default Value of adjust_smooth_skinning in the Config File

Changed the default value of adjust_smooth_skinning in the Maya plug-in config file from false to true. When set to true, the skinning method among contiguous shapes is standardized, eliminating the problem of gaps occurring among contiguous shapes.

There is no effect if adjust_smooth_skinning is already specified in the config file. Note that if adjust_smooth_skinning is not specified in the config file, the behavior changes.

Export options can now be overwritten with a pre-export script.

A feature (NintendoOverwriteExportOption procedure) has been added for overwriting the export options being exported using a Maya plug-in pre-export script. This feature can be used to temporarily disable optimization to shorten the time it takes to export and to make previewing faster. However, the following options cannot be overwritten.

  • preset
  • lod_export
  • pre_export_script
  • post_export_script

Some config settings can also be overwritten.

  • optimize_primitive (none / default / force / brute_force / forsyth)
  • quantization_analysis (true / false)
  • delete_near_vertex (string)

A temporary file can now be saved when an export error occurs.

A feature has been added to prevent deletion of the intermediate files output to the temporary folder when the post-export script or the intermediate file optimizer generate an error. This feature can be used for debugging in cases such as when the post-export script fails.

Specify the following in the config file.

delete_temp_file_on_error="false"

The default value when this is not specified is true (delete).

Photoshop Plug-In

Added Default Settings for Unify Image Options and Unify Converter Options

Default settings for Unify Image Options and Unify Converter Options have been added. You can set the default values when settings are not included in the PSD file.

var default_unify_image_options = true;
var default_unify_converter_options = false;
