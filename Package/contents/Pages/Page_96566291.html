Apply in World Coordinates

This page describes Apply in world coordinates, the feature for choosing whether to apply curl noise using the world coordinate system.

Overview

Apply in world coordinates sets the coordinate system for curl noise.
When you use the world coordinate system, curl noise is affected in terms of world coordinates regardless of the emitter transform settings.

World Coordinate System and Emitter Coordinate System This page describes the coordinate systems handled in effects.

The following overview describes the settings ().

Apply in World Coordinates

Use the check box () to specify whether to apply curl noise in terms of world coordinates.
When the check box is cleared, curl noise will be applied in terms of emitter coordinates so it will be effected by the position and the rotation of the emitter.
When the check box is selected, curl noise will be applied in world coordinates so it will not be effected by the position and the rotation of the emitter.
