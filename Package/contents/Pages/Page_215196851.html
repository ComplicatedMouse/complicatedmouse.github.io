Maya Plug-In

This document is a graphical introduction to the features that were added to the Maya plug-in.

Multi-Export Plug-In Edit Additional Options Window

The additional options for the multi-export plug-in can now be edited from the UI.

Info

In the Enter Export Items window, if FDES file is left blank and the export plug-in option settings are not saved in the scene, the Edit Additional Options window will not be displayed and an error will occur.
Select Nintendo > Export with Setting to display the Options window, and in the upper-left menu, select Edit > Save Settings to save the option settings.


Related Link　For more information, see the Edit Additional Options Window section in the Multi-Export Plug-In documentation.
