Question

Even though the font file (BFFNT file) was swapped with one of a different size, the information displayed in the text box did not change.

Answer

You must adjust the font scale settings in the Text Box pane.
If you set the value to 1.0, the specified font displays in its original size.
By default, the editor adjusts the font scale so the size of the text box does not change in appearance.

If the file has been saved in version 2.4.1 or later of LayoutEditor, when it is loaded any change in size is detected and a warning appears.
When the final file is saved, the font size is stored to the fontSizeOriginal element in the layout intermediate file (.flyt). Use this for batch-revision in the program.
