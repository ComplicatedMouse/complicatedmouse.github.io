Preview

Menu for previewing models and animations in applications in which the viewer library is embedded (on a PC or the development hardware).

Info

The viewer library does not support previewing or real-time editing of shape animations.

A

Connects to the application (or disconnects when already connected).
B If Connect Automatically is selected, 3DEditor automatically connects to the specified application (located either on a PC or development hardware) during startup.
The connection can be changed using Select Connection Target.
C

You can select either PC or development emulator as the connection target.
Note: You must temporarily disconnect from the application before switching the connection target.
D

Enables or disables shader optimization.
When shader optimization is enabled, optimized shaders that include only those shaders being used by each model are transferred at run time.
E Regenerates the optimized shaders.
The optimized shaders can be regenerated if connected to the application.
