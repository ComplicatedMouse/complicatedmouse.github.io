Profiler-GroupBy

Overview

Note

This cmdlet is only available with Thread-Centric Data Views are enabled. See Experimental Features Tab for additional information.


Set the function grouping mode, as used in the Thread-Centric Data Views feature. This will be reflected in how the Functions and Call Tree tabs are grouped.

Profiler-GroupBy [-GroupBy] <ScriptingGroupByMode>

[-GroupBy] <ScriptingGroupByMode>

Specifies the grouping. See Experimental Features Tab for an explanation of the values.

    • FunctionThenThreads
    • ThreadThenFunctions
    • CoreThenThreads

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-GroupBy -GroupBy ThreadThenFunctions
Profiler-GroupBy FunctionThenThreads
