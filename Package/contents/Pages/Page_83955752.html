Consolidated Stripe

This section describes the Consolidated Stripe feature, which connects generated particles together like a string.

Overview

A consolidated stripe is a particle in which the first emitted particle is connected together with subsequently emitted particles in a string-like shape.

The following image shows an example of a consolidated stripe when the first emitted particle 1, has been connected to the subsequently emitted particle 2.
The shape of the resulting stripe particles can be controlled by using the particle connection calculation method to adjust the positions of the individual particles.


Info

To use this feature with child emitters, enable Emitter Assignment for Particles.

Consolidated Stripe

The Consolidated Stripe group () is a set of features for creating particles with string shapes.

Calculation Method

Determines the orientation of the particle face.

Option

Determines the shape of the particle.
Texturing Determines how texture is mapped to the particle.
Number of Divisions Sets the number of particle divisions and smooths out the 'string'.
Connection Type Determines how the starting particle and ending particle are connected.
Starting point alpha, End point alpha Sets values for the starting point alpha and the end point alpha.
Info

Global color is applied as the color for consolidated stripes.
You can set this using the Global tab on the emitter node's Color tab.
