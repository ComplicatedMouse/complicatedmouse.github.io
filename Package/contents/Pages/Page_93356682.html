Preview Controls

These controls are used to control operations like playing and stopping when connected to the Viewer.

Move to first frame Moves to the first frame and plays the animation.
Stop Extinguishes all emitters and particles being displayed in Viewer.
Play/Pause

Pressing this button while an animation is playing pauses the animation.

Pressing it again will play the animation from where it was paused.
Step Plays the animation one frame at a time.
Execute fade command Executes the fade command to end playback as configured in the Termination Processing settings.
/ Enable/Disable Looping

Use the enable/disable looping feature (/) to enable/disable looped playback.
When looping is enabled (), the effect is played each time playback begins.

The fade command is run every time playback ends.

Specify frame Use the text box field () to specify the number of frames when looping.
Info

After you click one of these buttons, you can use the same button again by pressing the Enter key.
You can use Step Playback () to easily play back effects frame-by-frame.
