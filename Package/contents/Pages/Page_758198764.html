Thread State: Synchronization Object Details Window

The Synchronization Object Details window provides information about a synchronization object. This window will appear when you click the View button in a System Call Details window. The following information is displayed:

  • The inferred type of the synchronization object.
  • An identifier of the synchronization object (on NX, it is the pointer to the synchronization object).
  • A table containing every system call that referenced the given synchronization object.

Exploring Synchronization Object References

You can double-click any of the system calls in the table to open a System Call Details window referencing it. This lets you get a general idea of what this synchronization object is being used for.
