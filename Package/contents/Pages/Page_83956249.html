Using Text Box Panes

This section describes how to use text box panes, which are used for positioning strings on the screen.

Preparing to Create the Layout

From a state in which the LayoutEditor has been started and a layout has been created, make preparations to create a text box pane.
Related Links For more information about starting LayoutEditor, see Workflow, and for more information about creating layouts, see Creating Layouts.

Creating Text Box Panes

Use the following procedure to create a text box pane.

1 From the main menu, select Pane > Create Pane > Text Box Pane.
2 In the Create Text Box window, in Specify Font File, click Browse.
Samples/Sources/Resources/Ui2d/Common/Fonts/sample.ffnt
3 In String, enter Textbox and click OK.
4 A text box pane with the entered text string is created in the layout.

Info

If text is not displayed or if there are characters that are not displayed, check whether characters not included in the font file are being used.
For example, you might specify kanji characters in font data that contains only alphabetic characters.

Related Links For more information about which kinds of font files you can use in this tutorial, see here.

Adjusting Text Box Pane Properties

Use the following procedure to edit the text box pane properties.

1

Select the text box pane to display the property window.
Related Links For more information about displaying the property window, see Editing Layouts.
2 In the property window, from the tree on the left, select Text Box Pane.
3 Edit the properties in the page on the right side.
