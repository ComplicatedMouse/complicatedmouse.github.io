Advanced View Items
Name Displays the texture name.
Format

Displays the texture format.

Info

When you click the format for each texture, you can specify a format from a drop-down list.

You cannot specify the format for textures that are not referenced by the layout.
Width Displays the texture width (x-axis).
Height Displays the texture height (y-axis).
Size

Shows the texture data size, in bytes, after converting to the texture format to be used on the device.

When multiple platforms are supported, a column is displayed for each platform.

Info

Changing the format causes the data size to change, even for textures with the same width and height.

You cannot display the size of textures that are not referenced by the layout.
File path Displays the texture file path.

Restriction on TGA Image Files

For TGA image files, use regular TGA files.
If the image uses alpha information, create a 32-bit format TGA file; otherwise, use the 24-bit format.

Specifying the Texture Format

By specifying a texture format on LayoutEditor, you can control the data size in the game and the image display quality.
To specify the texture format:

  1. Switch how files are listed from Image View to Detail View.
  2. Select a texture from the list.
  3. Click Format for the selected texture and specify a format from the drop-down list.
