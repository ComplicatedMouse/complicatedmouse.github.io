Material Property Window: Material Reference Page
Note

When using this feature, note that its specifications and behavior are subject to change in the future without advance notice.

This page describes configuring and checking the parent material of the material referencing feature.

A

Moves the selected material up and down when there are multiple parent materials. If there are duplicate parameter settings among multiple parent materials, the order of the parent materials affects the priority of the settings to use. For more information, see Multiple Material Referencing in Material Referencing.
B Adds a new parent material.
Click the Add button to display the Edit Parent Material window.
C Deletes the selected parent material.
D Edits the selected parent material.
Click the Edit button to display the Edit Parent Material window.
E

Displays the material name of the configured parent material.

Note

Currently, only material names are replaced with the specified labels when the parent material is selected. Names are displayed in the format "Parent material label (child material name)".
F Displays the model file path with a configured parent material.
G

Lists configured parent materials.
When you double-click a parent material from the list, it opens the material property window for the parent material.

An error message is displayed in red when you cannot correctly browse the parent material.
H Deletes the selected material reference information.
I Edits the selected material reference information.
Click the Edit button to display the Edit Material Reference Information window.
J
Displays the type of shader parameter with material referencing information configured.
K

Displays the ID of the shader parameter with material referencing information configured.
L

Displays what was specified for the shader parameter value references. Displays what has been specified for setting the shader parameter values. This item is specified as either referencing the parent material's values, overwriting the values, or using the defaults. Note 1
M Displays the restrictions on child materials. Possible settings include fixing the value to the parent material and always setting the value using the child material. (See Note 1.)
N

Displays the restrictions on child material values. Displays the ID numbers selected for restrictions on the range of values or restrictions on choices. Note 1.
O The group for the shader parameters set for the material reference information is displayed.
P Lists the material reference settings. Hold down SHIFT and left-click to select multiple items, and press CTRL+A to select all.

Note 1: For more information, see Material Referencing Settings.


Edit Parent Material Window

You can use this to add and edit parent material models and parent materials.

A

Displays a list in C of the material files under the additional reference path of the path and model described in Team Settings.

In the windows for adding and changing parent materials, you can select parent materials from among categories specified in the team settings.
B

Displays a list in C of the material files under the reference source path.
C Select the parent material from the list.
D

Click OK to add the material reference.
E Cancels the edit of the material reference.


Edit Material Reference Information Window

Use this window to edit the material reference information.

A Specifies what to set for the shader parameter value references.
B Sets restrictions on value references for child materials.
C Click OK to update the material reference information.
D Cancels the changes made to the material reference information.
