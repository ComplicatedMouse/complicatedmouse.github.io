Apply Flux to Scale

This section describes the feature for fluctuating the scale of particles.

Overview

The Apply Flux to Scale feature is a way to apply a periodic animation to the scale of the particle using a waveform pattern.
You can apply flux to a scale to express softer objects, such as a bouncing ball.
This section provides an overview of the settings ().

Waveform Types

When you select Scale for Apply Flux, the settings for Waveform Type and the Apply Flux group () appear. The group has the following settings.

Waveform types This feature determines the type of waveform to use to animate flux.

The following table lists the settings for the Apply Flux group ().

Amplitude Sets the magnitude of the flux.
Period Sets the length of one period of flux.
Initial Phase Sets the initial position of the flux.
Random Phase Randomly shifts the position where each particle begins to fluctuate.

Set Scale Y Separately

With Apply Flux, you can select the scale. Use the check box () to select whether to set the y-axis scale separately.
This feature provides a way to create effects like the gentle movement of bubbles that retain their bulk.
