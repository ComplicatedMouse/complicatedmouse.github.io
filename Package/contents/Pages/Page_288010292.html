Specifying the Color Space

This section describes the procedure for configuring whether to handle the color space linearly in the layout.

Specifying the Color Space Using the Tool Environment Settings

Use the following procedure to specify the layout color space.

1 In LayoutEditor, from the menu in the main window, select Edit > Tool Environment Settings.
2 To set the color space, in the Tool Environment Settings window, select Gamma Characteristics > Linear Color Space Mode and select and clear check boxes.
After making these selections, the binary converter converts both textures and color parameters to a linear color space.
3 Click OK to confirm the settings.
4 If the Layout window was already open, close it and then open it again.

Adjusting Color Space Using the Color Picker

When operating in linear color space mode, the displayed colors differ if you get color with the eyedropper in a nonlinear environment and set RGB values in LayoutEditor.
Use the following procedure in this case to adjust RGB values.

1

Enter the nonlinear environment RGB values using the Color Picker.
2

In the Color Picker, click the Gamma Conversion button (), and then click the icons to adjust the color.
