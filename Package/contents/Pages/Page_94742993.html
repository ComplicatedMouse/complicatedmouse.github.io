Update Random Width

This section describes the Update random width feature that determines whether the random numbers set with Position(±) and Rotation(±) in Emitter Transform are updated each time particles are emitted.

Overview

The Update random width feature is a way to randomize the position and rotation emitter transform values for every particle when these values have been set to a random number for emitter emissions.
It provides more variety to the look of the overall effect by randomizing the emission of every particle.
The following overview describes the settings ().

Update Random Width

You can enable or disable the random width update timing by selecting or clearing the check box ().
When enabled, random values are updated every time particles are emitted.
When disabled, the update occurs only once, when the emitter is set.

Info

This feature will not work if any of the emitter transform's position, rotation, or scale include an animation.
