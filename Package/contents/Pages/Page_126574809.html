Texture Resource Files

Texture Resource Files

Overview

Texture file resources represent binary images created using the Texture Converter tool.

Texture resource files provide a mechanism to handle multiple textures.

Texture File Resources

Texture file resources are defined as nn::gfx::ResTextureFile.

Texture file resources represent binary images themselves created using the Texture Converter tool and can use common features of resource file classes.

Before using texture file resource features, you must first initialize the texture file resources.

Initialization and Finalization

Initialize texture file resources using Initialize, and finalize texture file resources using Finalize.

If you did not specify the memory pool during initialization, the memory pool in the resources is used. When the --disable-memory-pool option is specified during resource creation by the Texture Converter tool, no memory pool is held in the resources, so initialization is not possible using this method.
When the memory pool on the user side contains texture file resources, specify that memory pool and call Initialize to use the memory pool on the user side instead of the memory pool in the resources.

Getting the Texture Dictionary

To get nn::util::ResDic that manages all the texture resources in the texture file resources, use GetTextureDic.

Getting Texture Resources

To get the specified index texture resources, use GetResTexture
When you want to get texture resources from texture names, search for the target index using the texture dictionary nn::util::ResDic::FindIndex.

Texture Resources

Texture resources are defined as nn::gfx::ResTexture.

Getting Texture Information

To get gfx texture information, use GetTextureInfo.

Getting Texture Objects

To get gfx texture objects, use GetTexture. Before initialization, the texture object region is padded with zeros.

Getting Texture View Objects

To get gfx texture view objects, use GetTextureView. Before initialization, the texture view object region is padded with zeros.

Initialization and Finalization

Call Initialize and use the information obtained by GetTextureInfo to initialize the texture objects obtained by GetTexture and the texture objects obtained by GetTextureView.

Call Finalize to finalize the texture objects obtained by GetTexture and the texture objects obtained by GetTextureView.

Getting Names

To get texture names, use GetName.

Getting the Data Size

To get the storage data size of a texture, use GetDataSize.

Getting the User Data Dictionary

To get the nn::util::ResDic that manages all the user data resources in the texture resources, use GetUserData.

Getting User Data

To get the nn::gfx::ResUserData of the specified index, use GetUserData.

Setting and Getting the User Descriptor Slot

Texture resources have an allocated descriptor slot region that is freely usable by the user.
Use SetUserDescriptorSlot to set the user descriptor slot, and use GetUserDescriptorSlot to get the user descriptor slot.
