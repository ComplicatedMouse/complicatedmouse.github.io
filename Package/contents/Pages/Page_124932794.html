Preferences File Format

This section describes the preferences file.
The preferences file (preference.xml) is installed automatically by the installer.
You can use elements similar to the following elements to configure a GUI version of FontConverter.

Note

The settings in the preferences file (preference.xml) are not applied when the console version of the font converter is executed. Use command-line arguments to specify the necessary settings.


<PlatformPreference>

<PlatformPreference> is the preferences file route element.

Description

The user config file includes the following elements.

  • <ConverterPlatformName>: The name of the platform. At most one may appear.

<ConverterPlatformName>

<PlatformName> is the platform name setting.

Description

Specifies the platform name.

<IsGpuEncodingEnabled>

Indicates whether GPU encoding is used. true = use GPU is selected when omitted.

Description

Specifies true (= use GPU) or false (= do not use GPU).

Example of preference.xml

<?xml version="1.0"?>
<PlatformPreference>
  <ConverterPlatformName>Generic</ConverterPlatformName>
  <IsGpuEncodingEnabled>true</IsGpuEncodingEnabled>
</PlatformPreference>
