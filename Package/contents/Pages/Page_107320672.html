Tool List

The following table lists the tools provided in the hid library.

Tool Category Path Description
HID Shell Windows tool (EXE file) Tools/CommandLineTools/Hid Logging, playback, and autopilot from the PC is possible for those devices that can be handled from the hid library.
Input Director Windows tool (EXE file) Tools/Hid/InputDirector

Sends the HID input from the PC to the development hardware. The transferred values are treated as the HID input for the development hardware.
Vibration Converter Windows tool (EXE file) Tools/Hid/VibrationConverterConsole

Creates vibration binary files from audio files.
Vibration Collection

Target tools (NSP files)

 TargetTools/NX-NXFP2-a64/VibrationCollection/Release

The Vibration Collection is a tool for previewing how vibrations and sounds work together on targets that can play vibrations and audio at the same time.

The Vibration Collection provides a set of vibration files and sound files as presets. Select a vibration and a sound from the menu to try different combinations and test how they feel.
Vibration Demo Collection Target tools (NSP files) TargetTools/NX-NXFP2-a64/VibrationDemo/Release Demos that show how to use the HD rumble feature effectively. They can be run on the EDEV or SDEV.
