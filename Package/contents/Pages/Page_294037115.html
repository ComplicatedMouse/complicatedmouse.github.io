For more information, see the Texture Converter manual.

Changes to the Specification for GPU Encoding

An error now occurs if GPU encoding is not possible, and false is not specified for -- gpu-encoding.
If you want to encode in the GPU, if possible, and in the CPU otherwise, specify --gpu-encoding auto.

The gpu-encoding option is an option specified for the converter by the developer.
Consult with a lead developer, and use the project settings in LayoutEditor or the preference settings in FontConverter to specify the options to pass to the converter when previewing.

Typically, all members of a team specify common values, so the display results do not vary from person to person.

Improved the Quality of the Alpha Component When Compressing to the bc3, bc7, and ASTC Formats

The improvements show up in the texture converter.

LayoutEditor

This section introduces the new features in LayoutEditor.

Align Pane

Added the Align Pane for aligning and adjusting the positions and sizes of all panes in the Hierarchy window.

The align pane () is a pane for aligning and adjusting the positions and sizes of subordinate panes in the Hierarchy window with their arrangement relative to one another being maintained.
The feature can be used when a pane size is dynamically changed in a game application to accomplish runtime processing in the program.

For example, if the string in a text box pane () changes when the application is localized, this feature helps to maintain the relative position of the icon placed to the side, as shown in the following table.

A The Align Pane feature is not used.

B The Align Pane feature is used. The spade icon (♠) moves, following exactly along the string.

Mask

Added the Mask feature for clipping using a texture image registered to a pane.

Mask () uses the registered texture file as the image for clipping (the clipping mask image).
You can use this feature to clip the mask image to the shape of the pane, or conversely to use the alpha value of the mask image to clip the pane.

In the following example, the mask image registered to Mask () is clipped using the shape of the Text Box Pane ().
