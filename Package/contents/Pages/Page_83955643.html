Simple Collision

This section describes the feature for placing simple flat collisions making particles collide together.

Overview

Simple Collision places simple flat collisions with the ground and makes particles collide.
You can apply a bounding effect, for example, as shown in the following image, when a ball falls and collides with the ground.

Simple Collision

Simple Collision () is a collection of features to configure behavior when particles hit collision planes set using this functionality or a single collision plane specified with the game application.

Apply in World Coordinates

Determines whether to apply the field based on the world coordinate system.

Behavior When Colliding

Selects the behavior of particles when they collide.

Collision Plane Y Coordinate (±)

Sets the position of the collision in the Y direction.
Bounce Rate Sets a coefficient of bounce relative to the speed of particles when they collide.
Bounce Count (±) Sets the number of times the particle bounces when a collision is detected. The number includes the bounce from the collision.
Friction Coefficient Accelerates or decelerates particles when they collide.
