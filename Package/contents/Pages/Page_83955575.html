Nintendo SDK 0.3.0

The following known bugs are confirmed to occur as of Nintendo SDK 0.3.0.

  • The PackageManager version update feature does not work correctly.
  • A dialog box appears when you install NintendoSDK-Documents.
  • Cannot perform a full text search of the documentation.
  • Changes to the NINTENDO_SDK_ROOT environment variables are not applied.

The PackageManager version update feature does not work correctly.

The features to Update and Replace earlier versions do not work correctly in PackageManager.
If you use these features, packages will become invalid, with no consideration of package dependencies or existing versions.
To install the latest versions, first uninstall all packages and then install the new versions.

A dialog box appears when you install NintendoSDK-Documents.

The following dialog box appears when you install NintendoSDK-Documents from the package manager, and the installation process is suspended.

Click Yes to proceed with the installation.

Cannot perform a full text search of the documentation.

When you open the index.html file included in NintendoSDK-Documents, an entry field appears to the left for a full text search (as shown below in the figure on the left).
However, this feature cannot be used in Nintendo SDK 0.3.0.

You can search using the entry field displayed in the API Reference (as shown above in the figure on the right).
However, searches are limited to the content in the API Reference.

Changes to the NINTENDO_SDK_ROOT environment variables are not applied.

Sometimes the changes to NINTENDO_SDK_ROOT are not applied in the package manager, and the old setting for NINTENDO_SDK_ROOT remains in place.
If this bug occurs, you must log out and then log back in.
