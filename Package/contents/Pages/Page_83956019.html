Introduction

The method to create sequence data used by SoundMaker is described.

What Is Sequence Data?

Sequence data corresponds to the music data represented in a standard MIDI file (referred to as “SMF” in this document).

Commands that process changes in attributes, such as sound generation and volume change over time, are described.

Because sequence data is simply a set of commands, sounds are actually produced using bank data that corresponds to the sound source data.

Text Sequences and Standard MIDI Files

The sequence data used by SoundMaker can be classified into two main types.

The first type is sequence data where each file is a single sequence (track), such as in an SMF file. The second type is where multiple sets of sequence data are gathered into a single file.

SoundMaker supports both of these types of sequence data in the form of files called text sequence files (FSEQ files).

SMF belongs to the first type of sequence data. At the time of conversion, it is converted into text sequences using sequence data belonging to the first type. It is then possible to create sequence data of the second type by directly editing the text sequences with a text editor.

This document describes how to create these two types of sequence data.
