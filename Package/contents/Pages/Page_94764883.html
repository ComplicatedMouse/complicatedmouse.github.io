Texture Preview

Use Texture Preview to preview texture files that have been registered in the Texture tab ().
Texture preview is linked to the contents of the settings, and makes it easier to see the effect of the settings.
A Show RGB
 Displays the RGB components of the registered texture.
B Show alpha Displays the alpha component of the registered texture.
C Frame Number The frame number of the registered texture.
This item is mainly used to specify frames when configuring texture pattern animations.
Info
Related Links Texture Preview is provided in Texture Details. Double-click Texture Viewer to display the Texture Details Window.
