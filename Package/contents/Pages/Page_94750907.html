Inherited emitter velocity ratio

This section describes the feature that inherits the emitter's movement velocity as the particle emission initial velocity.

Overview

The Inherited emitter velocity ratio applies the velocity of the emitter in a motion animation to the particle at the time it is emitted.
Space will open up between particles, and the homogeneity of the effect might be lost, when the emitter is moving at a high speed and the particle follow type is using the Do not follow emitter setting.
You can use this functionality to see the effect all together, reducing the space between particles.

Info

The inherited value is calculated from the movement value from the previous frame, causing all inherited values on the first frame of the emitter time to be 0.

The following overview describes the settings ().

Inherited emitter velocity ratio

The inherited velocity is added to the initial velocity at the time of particle emission.
The value is specified as a ratio where the emitter velocity is 1.
