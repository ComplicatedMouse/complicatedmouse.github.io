nn::g3d News ()

Added a feature to the 3D texture preset converter for partially overriding presets.

You can now use the 3D texture preset converter to partially override presets. An option for resizing has been added for purposes such as changing the size to 50% for a preset using preset_override after it has been coded, as shown in the following example using the Normal1 preset.

<presets>
    <preset name="Normal1" format="unorm_bc5" comp_sel="rg01" hint="normal" />
</presets>
<preset_overrides>
    <preset_override name="Normal1" resize_w="50%" />
</preset_overrides>


Added a feature to the intermediate file converter for converting FBX files to model intermediate files.

You can now use the intermediate file converter to convert FBX files to FMDB files. However, there are restrictions on the FBX files that can be converted. For more information, see Intermediate File Converter.

nn::ui2d News ()

Added a setting to make the Capture Pane and Scissor Pane work properly when the origin is set to the upper-left in window coordinates.

Added a setting to make the Capture Pane and Scissor Pane work properly when the origin is set to the upper-left in window coordinates.

If you set nn::ui2d::DrawInfo::SetLeftTopWindowOrigin() to true when the origin is set to the upper-left in window coordinates, the runtime does not process the proper region.

nn::atk News ()

Changed the timing used when the function that gets the number of times the stream sound has looped increments the loop count.

Changed the timing when nn::atk::StreamSoundHandle::GetPlayLoopCount() (the function that gets the number of times the stream sound has looped) increments the loop count.

When increasing the number of loops, the playback position was previously obtained with nn::atk::StreamSoundHandle::GetPlaySamplePosition(), shortly after rollback due to the loop.
The playback position is now obtained with nn::atk::StreamSoundHandle::GetPlaySamplePosition() when the timing of a rollback due to the loop matches the timing when the loop count is increased.

Added documentation descriptions for functions that require mutual exclusion.

The functions that require mutual exclusion have been added to the Threads page of the Programming Guide.
