LOD Setting to Use During Sampling

The LOD to use during texture sampling is specified from a drop-down list on the Samplers page of the material Property Window.

Item Description
Minimum value Sets the maximum value for LOD.
Maximum Value Sets the maximum value for LOD.
Bias Value Sets the bias value for LOD.
