Material Property Window: Sampler Page

The content in this page is used to check and edit the sampler of the selected material.
The page also supports adding textures to the sampler and changing existing textures.

A Changes the order of samplers by moving the selected sampler up and down in the list when there is more than one sampler.
B Adds a new sampler.
C Deletes the selected sampler from the list.
D Sets Restriction for Child Material to None or Do not allow additional proprietary samplers.
E

The sampler list.
Displays a list of sampler information (you can copy and paste individual samplers and configure the Require for Child Materials setting).

If "Outside the traversal path" is displayed, see here.
F

The name of the sampler selected in the sampler list.

You can edit the sampler values and set Child Material Restrictions.
G
Displays a hint about how the selected sampler is handled by the DCC tool.

You can edit hints by disabling editing restrictions.

Select from among candidate reserved-name samplers added in the Tools when the parent material was created.
H Texture information for the selected sampler. You can also change the texture.
I

The wrap setting to use for the selected sampler during sampling.
J

The filter setting to use for the selected sampler during sampling.
K

The LOD setting to use for the selected sampler during sampling.

Material Referencing Settings

Right-click a label name to display a shortcut menu where you can configure the material referencing.

You can also configure material referencing settings by right-clicking the wrap, filter, and LOD settings.

Related Links For more information about the parent material settings, see the Material Reference Page.

Restriction for Child Material

Set Restriction for Child Material to None or Do not allow additional proprietary samplers, based on the parent material sampler.
When set to Do not allow additional proprietary samplers, adding samplers to child materials that are not in the parent material is prohibited.

Setting Whether to Require Samplers for Child Materials

Specify whether to require a sampler for child materials, based on the sampler of the parent material.

If Required for Child Materials is set to Yes, the key icon () is displayed for the child material sampler and cannot be deleted.

Editing Sampler Values

Specifies the sampler values to set. You can switch between referencing the parent material's values, overwriting the values, and using the defaults.

Value Reference Description
Reference Parent Material Values

Reference and use the values set by the parent material. (Its own configured values are disabled.)
Overwrite Overwrite the referenced parent material values with its own configured values.
Use Default Values An empty string.

Child Material Restrictions

Sets restrictions on values for child materials.

Restriction for Child Material

Description
None

No restrictions on the child material's Value Reference setting. Both Value Reference and Restriction for Child Material can be set for child materials.
Fix to the Parent Material Values The child material's Value Reference is always set to Reference the value of the parent material. Child material shader parameters are fixed to the values of the parent materials.
Require the value to be set for the child material. The child material's Value Reference is always set to Overwrite. Neither Value Reference nor Restriction for Child Material can be configured for child materials.
Require the child material to reference the value. The child material's Value Reference is always set to Reference the value of the parent material. Neither Set Value nor Restriction for Child Material can be configured for child materials.

Related Links For more information, see Material Referencing.

Disabling Editing Restrictions

In team settings presets, setting the Use sampler names that conform to the DCC tool's naming conventions setting to Yes restricts editing of samplers created using the DCC tool.
To forcibly edit samplers in 3DEditor, navigate to the Sampler property window and click the Clear sampler edit restrictions button.

This makes it possible to edit sampler names and hints that have the DCC mark.

Related Links For more information, see Team Settings and FollowDccSamplerNameRule.

Warning Messages

You cannot add textures by dragging from a folder that is not on the same hierarchical level as the model folder or from a location that is not specified as an additional reference path.
An "Outside the traversal path" message is displayed if a registered texture cannot be found, such as in cases where it was moved or had its name changed.
If no textures have been registered, a "No image has been specified" message is displayed.


Related Links For more information about texture searches, see Additional Reference Paths.
