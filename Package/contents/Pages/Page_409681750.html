LayoutViewer Operations

The following table summarizes the LayoutViewer operations.

Controller Operations
Left Button Steps back the animation frame. (Only in animation mode.)
Right Button Advances the animation frame. (Only in animation mode.)
Up/Down Buttons Changes the animation that is being played. (Only in animation mode.)
A Button Plays the animation. Stops the animation if it is playing. (Only in animation mode.)
B Button Resets the animation frame to the starting frame. (Only in animation mode.)
X Button Switches the menu view mode between Hide, View, and View Details.
L Button Switches the view mode for borderlines between Hide, View Null Panes Only, and View All.
ZR Button Switches alignment on and off.
SELECT Button Changes the size of the framebuffer.
Left Stick Changes the position of the cursor in the debug information menu.
ZL Button + A Button Selects the debug information menu. After selection, change the setting using the Left Stick.
ZL Button and B Button Closes the debugging information menu.
R Button Displays the operation description window.
Keyboard Operations
LEFT ARROW Steps back the animation frame. (Only in animation mode.)
RIGHT ARROW Advances the animation frame. (Only in animation mode.)
UP ARROW and DOWN ARROW Changes the animation that is being played. (Only in animation mode.)
A Plays the animation. Stops the animation if it is playing. (Only in animation mode.)
B Resets the animation frame to the starting frame. (Only in animation mode.)
X Switches the menu view mode between Hide, View, and View Details.
L Switches the view mode for borderlines between Hide, View Null Panes Only, and View All.
E Switches alignment on and off.
D Changes the size of the framebuffer.
Mouse Pointer and Touch Screen Operations
(?) Place the mouse cursor over (?). Displays the operation description window.
Right-click Feature Animation is displayed when the button is clicked. (Only in Control Mode in the PC version.)
Click the pane Displays the extended user information pane. (When PaneInfo is enabled for debugging.)
Right-click the pane Displays a list of panes at the position of the mouse pointer. (When PaneInfo is enabled for debugging.)
Click a pane in the list to display the extended user information.
Touch the screen Feature Animation is displayed when the button is clicked. (Only in Control Mode in the development hardware version.)
