Using the Per-Character Transformation Simple Settings Feature

Describes how to set an animation using the simple settings feature for per-character transformation.

About Per-Character Transformation

  • The per-character transformation feature provides a way to animate the individual characters of the text in a text box pane, as shown in the following figure in frames 1 through 5.


  • Until now, this feature has been complicated to use because it requires manually creating animation curves for the Offset and Duration parameters. But now, with the availably of the Simple Settings feature, per-character transformation is easier to configure.

Making the Settings in LayoutEditor

  • This section describes the procedure for actually creating the animation that was presented previously in LayoutEditor.

Step 1: Enabling the Per-Character Transformation Feature in the Text Box

  1. Double-click the text box to open the property window.
  2. From the tree on the left, select Text box pane and then in the Per-character transformation group of settings, select Enable. Selecting this option enables the per-character transformation feature.

Step 2: Setting an Animation Curve in the Text Box

  1. Open the curve editor.
  2. Create the curve you want to animate in the text box for which the per-character transformation feature has been enabled. If want each character to move for a duration of 50 frames, create a curve of 50 frames.


  • In the following figure, the text ABCDE has been set in the pane, and a curve has been created that moves up the y-axis from 0 to 100 during frames 0 to 25, and down the y-axis from 100 to 0 during frames 61 to 66.

Step 3: Using the Simple Settings to Set the Animation for Each Character

  1. Double-click the text box to open the property window, select Text box pane from the tree, and then in the Per-character transformation group, click Offset/duration animation simple settings.
  2. Use the Add button in the dialog box to add as many settings as there are animations.
  3. If you want to move the characters starting from the character furthest left in the text box, select Move characters from left. If you want to more the characters starting from the right, clear this check box.
  4. Enter values in Start frame and Frame where first character stops moving to set the frames in which the first character starts moving and stops moving.
  5. Enter a value in Frame where last character starts moving to set the frame in which the last character begins to move.


  • In the following figure, the first character (A) is set to move in frames 0 to 25 and frames 61 to 66, just like in the animation curve that was just created.
  • The animation proceeds in order of A, B...E, and in this example the movement of the last character (E) is set for frames 25 to 50 and frames 95 to 100.

  • If you select Automatically calculate end frame, the end frame for the animation will be set automatically. If you want to stop the animation while the last character is still moving, clear this check box and explicitly set a value for End frame.

Completion

  • Per-character transformation Offset and Duration animations are created automatically by the procedure from the previous section.
  • The following figure presents an example of animations set to a created text box.

  1. During frames 0 to 25, the first character (A) moves upward from 0 to 100 in the y-direction.
  2. At frame 25, the last character (E) begins to move in the same way. All movement ends in frame 50.
  3. During frames 51 to 60 the characters all remain stopped because there are no per-character transformation settings.
  4. During frames 61 to 66, the first character (A) moves downward from 100 to 0 in the y-direction.
  5. At frame 95, the last character (E) begins to move in the same way. All movement ends in frame 100.
  • For more information about the per-character transformation feature, see Per-Character Transformation.
