Plug-In for Converting Custom Attributes

This plug-in is for converting custom attributes for NintendoWare for Revolution (NW4R) to custom attributes for the current NintendoWare.

Usage

Select Nintendo > Convert Old Attribute to open the window.

How to Use

Click Search to display a list of nodes in the scene for which custom attributes for NW4R have been configured.

Note that nothing is displayed in the following cases.

  • When the values of custom attributes are the same as default values
  • When custom attributes for the current NintendoWare have already been set
  • When there is no corresponding custom attribute for the current NintendoWare.

Click Apply to run the conversion.
Custom attributes for NW4R remain undeleted after conversion.

The following custom attributes for NW4R are conversion targets.

  • Compress of NW4R Set Material Attribute
  • No Compress Node
  • Frame Extension List
  • Position, Normal and Color of NW4R Set Shape Anim Attribute
