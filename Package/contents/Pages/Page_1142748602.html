Deleting Unused Textures

This page describes how to use the Texture window to collect and delete textures that are not being used in the layout.

Selecting Unused Textures

If there are any unused textures in the layout data, Select Unused Items () is enabled in the Texture window.

Click Select Unused Items () to select all unused textures.

Click Delete Selection () to delete all of the unused textures.

Info
  • The targets of the Select Unused Items operation are the textures that have been loaded into the body layout.
    The textures for parts placed in the body layout are made targets by enabling them using the overwrite settings.
  • The Select Unused Items operation is only available when Within the layout is set to be displayed in the Texture window.
    If anything other than Within the layout is selected to be displayed, the Select Unused Items operation becomes disabled and unavailable.
