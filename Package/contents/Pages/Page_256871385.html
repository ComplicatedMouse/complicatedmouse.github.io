Improved Mouse Wheel Operations in the Layout Window

Operations using the mouse wheel in the Layout window have been improved, making tasks easier.

You can now use the mouse wheel, in combination with keys, to easily change how objects are displayed.
Tasks are faster by using the mouse wheel in combination with other keyboard shortcuts.

Operations in the Layout Window Using the Mouse Wheel

Improved Operation Behavior in the Layout Window
Mouse wheel Vertically scrolls the display.
CTRL + mouse wheel Horizontally scrolls the display.
ALT + mouse wheel Zooms.
Preceding operations + SHIFT Increases the amount of change.

Added a Feature to Show Neighboring Segment Keys in the Curve Editor (Only in Separated Mode)

A feature has been added that assists the creation of animation curves. Now shown are the nearest keys in the animation segment tags located before and after the curve you are editing.

You can now consult the nearest keys on either side of the animation segment you are working on in Curve Editor and directly check the neighboring animations while working.
Note: To use this feature, the same type of animation must be in the neighboring animation segment tags.

How to Use

To enable the feature, click the Show Neighboring Segment Keys icon in the Curve Editor window.

The feature shows the animation keys in the segment tags on either side of the currently selected animation segment tag. This is for reference purposes with regard to the order of display in the Edit Animation Segment Tags window.
You can snap the first and last values of the animation curve being edited to the animation keys that are being shown for reference.

Added Thumbnail Save Feature

A feature has been added to LayoutEditor for saving images to use for thumbnails.

When the setting is enabled in the project settings file, thumbnail images are saved as LayoutFilename.flyt.png in the .NwThumbs folder next to layout files when layout files are saved, during batch synchronization, and at other times.
The Update Thumbnail Image in the Specified Folder menu is also added to the File menu in LayoutEditor.

Saved images can be displayed using the Thumbnail Explorer Extension.

File Menu in the LayoutEditor Window With Added Menu Item

A Update Thumbnail Image in the Specified Folder option added to the File menu.
