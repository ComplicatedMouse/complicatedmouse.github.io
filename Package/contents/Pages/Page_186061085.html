Global Color Animation UI

Use the Global Color Animation UI to configure color and alpha animations along the axis of Emitter time.

A Add row Adds a new row under the selected row.
B Delete row Deletes the selected row.
C Copy row Copies the selected row.
D Playback Methods Selects whether to loop the animation.
E Interpolation type Selects the animation interpolation type.
F Clear Resets the edited animation content and returns it to a default state.
G Tables Sets one value per line and edits the animation.
