Sampler Texture Information

The following displays texture information for the sampler selected in the list on the Samplers page of the material Property Window.

A

The name of the texture used by the sampler.
B Opens the Change Texture Image dialog box, where you can change the texture being used.
C The full path of the texture used by the sampler.
D The color image of the texture. Click to display a detailed image.
E The alpha image of the texture. Click to display a detailed image.
F Opens the Texture Property Window.
G Indicates the display depth. This setting is activated when a texture array is selected.
H Displays DCC tool presets.
I Displays the preset texture format.
J Displays the selected texture format.
K Displays the type of filter used to create a texture mipmap.
L Displays the status of the color component selector when sampling a texture.
M Displays whether a weight adjusted for the human eye is being applied to RGB errors caused by texture compression (bc1, bc2, or bc3).
N Gamma correction Displays the gamma correction mode being used for the R, G, B, and A channels of the selected texture.
O Displays the dimensions of the texture.
P The texture width.
Q The texture height.
R

The texture depth.
S The data size of the texture will be displayed (including padding and mipmaps).
T

The mipmap level of the texture. Specify 1 if there is no mipmap.
U The maximum size of the texture used for mipmapping.
V The hash value specified when creating a image for a production device.
W The 3D Texture Converter attribute of the texture is displayed.
X The path information for the source files for channel merging.
This information is for the source files when combining multiple input file channels in 3D Texture Preset Converter and exporting them as a single file.
