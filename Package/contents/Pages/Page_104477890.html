Y Swing Speed

This section describes the Y swing speed feature for configuring the speed when the selected node is swinging up and down using Y swing amplitude.

Overview

Y swing speed sets the translational speed of the selected node's vertical oscillation motion as it moves along its circular route, using Y swing amplitude.
The speed is increased by increasing the value.
