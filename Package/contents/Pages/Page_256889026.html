Using #pragma comment(lib, )

This pragma can be used with the lld linker.

You can specify the libraries to link using absolute paths or use the -L option for a relative path to the specified directory.

To link libnn_profiler.a to a relative path, for example, you can code with either of the following methods.

#pragma comment(lib, "nn_profiler")
#pragma comment(lib, ":libnn_profiler.a")

Effective from SDK 11.0.0 (Clang for NX 1.9.0), library names without a leading colon (:) are also supported.

#pragma comment(lib, "libnn_profiler.a")

Note that the format where the name is prefixed with a colon might be deprecated in the future.
