Animation Property Window: Original Information Page

On this page, you can check the original information for the selected animation.

A The material to which the animation is bound.
B The hint used when working with the animation in the tool.
C The targeted animation information.
D The initial value set on the original animation curve.
E The number of keys set in the original animation.
