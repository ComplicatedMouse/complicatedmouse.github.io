Layout, Font News

NintendoWare Layout, Font News

This is a graphical introduction to the features that were added in NintendoSDK 0.13.0.

Added the Layout Settings window.

Added the Layout Settings window to LayoutEditor.

Added support for setting extended user data to segment tags.

Extended user data can now be set to segment tags in LayoutEditor.

Added support for moving keys by a factor of 10 in Curve Editor using SHIFT+ arrow keys.

In the Curve Editor window in LayoutEditor, when editing the animation for a pane, you can now press SHIFT+ arrow keys to move keys that have already been configured by a factor of 10.

Added the "Scalable fonts are not displayed correctly" section to the troubleshooting guide.

Added a workaround to use when scalable fonts are not displayed correctly in LayoutEditor and "?” is displayed instead.

LayoutEditor

This section introduces the new features added to LayoutEditor.

Added the Layout Settings window.

Added the Layout Settings window to LayoutEditor.

The Layout Settings window can be used to add comments and configure extended user data in a layout.
The layout environment can also be configured in this window.

Main menu > Window > Layout Settings

Switch the screen from the left side of the window.

Extended user data can now be set to segment tags.

Extended user data can now be set to segment tags in LayoutEditor.

Main menu > Window > Edit Animation Segment Tags

Edit Animation Segment Tags Window

In Curve Editor, you can now press SHIFT+ arrow keys to move keys by a factor of 10.

In the Curve Editor window, when editing the animation for a pane, you can now press SHIFT+ arrow keys to move keys that have already been configured by a factor of 10.

This can be helpful when you need to move keys by a large amount.

A Frame and value for the currently selected vertex Displays the frame and value for the currently selected animation curve vertex.
B Set arrow key shift amount Select to open the arrow key shift amount window.
This value is used when using SHIFT+ arrow keys to move animation keys.

This value is used when using the mouse to edit animations.
