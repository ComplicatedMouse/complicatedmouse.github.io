NintendoTelemetry

Overview

We would like to collect usage statistical data for some of the tools included in NintendoSDK. When data is collected, it is done with the consent of all users. However, aside from collection by Target Manager, the setting for this consent is centrally managed by the NintendoTelemetry.exe tool.

The collected data is anonymous and does not contain any personal information. It is only used to improve the development environment.

Specifications

Installation Target

NintendoTelemetry.exe is located at the same level of the hierarchy as some of the tools that collects usage statistics. Regardless of the hierarchy it is in, the setting for whether to collect data for reference is common.
The path varies depending on the structure of the installed package, but the typical path is C:\Program Files\Nintendo\NintendoSdkDaemon\NintendoTelemetry.exe.

Edit Config

Initial Setting

When a tool collects usage statistical data, a dialog box appears requesting permission as long as NintendoSDK has never asked whether to collect data.

To agree to data collection, select the check box.

Set at Any Time (GUI)

This setting can be changed at any time by starting NintendoTelemetry.exe.

Set at Any Time (CUI)

You can change this setting at any time using the following commands.

NintendoTelemetry.exe --setstatus yes
NintendoTelemetry.exe --setstatus no
NintendoTelemetry.exe --setstatus notset

You can also get the current status of this setting using the following command.

NintendoTelemetry.exe --statusonly
