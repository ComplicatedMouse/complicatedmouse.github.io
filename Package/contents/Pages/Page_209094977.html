Assign UV

This section describes the Assign UV feature for specifying how to assign UV to a texture applied to a stripe.

Overview

Assign UV is a feature that allows you to set how texture is applied to stripes.
Assign UV operates on textures when Texturing is set to Fit Paste to Rendering Range.
Related Link For more information about Texturing, see this page.

The following overview describes the settings ().

Assign UV

Select Assign UV for stripes from the following presets.

Fixed ratio per history Pastes a texture to one entire stripe.
Depends on history distance Pastes a texture depending on the distance of the History used for a stripe.
