Keyboard (NX Add-On)

Installing NX Add-On enables the Keyboard mechanism to be used in the following environments.

Structure in the SDEV Environment

USB Keyboard

The input state is determined from the state of the keyboard connected via USB.

When multiple keyboards are connected, the logical OR of their pressed state becomes the input state and the key lock state is shared.

A maximum of 8 keyboards are recognized, and any keyboards connected after that are ignored.

When other HID devices are connected via USB, the maximum number of keyboards that are recognized decreases by the number of other HID devices.

Input Sent by Input Director

The input state is determined by the state of the keyboard as sent from the host PC by Input Director.

When input from a USB keyboard exists, that logical OR becomes the final input state.

Structure in the EDEV Environment

The same as in the SDEV environment.
