Using Window Panes

This section describes how to use the window pane feature, which places windows in the layout.

Preparing to Create the Layout

Start LayoutEditor, create a layout, and then make preparations to create a window pane.
Related Links For more information about starting LayoutEditor, see Workflow, and for more information about creating layouts, see Creating Layouts.

Creating Window Panes

Use the following procedure to create a window pane.

1 From the main menu, select Pane > Create Pane > Window Pane.
2 A window pane is created in the layout.

Configuring Textures

Displaying the Texture Window

Use either of the following methods to display the Texture window.

  • From the main menu, select Window > Textures.
  • Press F8.

Advantages and Disadvantages of Window Panes

Advantages

Unlike with picture panes, if one texture happens to be greatly expanded in size it does not cause stretching of the texture being used for the frame.
As a result, the texture size is reduced.

Related Links For a comparison of frames created in the window pane and picture pane, see here.

Disadvantages

Configuring complex settings in the window pane causes a high processing load.
You can lighten the load by reducing the latitude in the settings. Do this by specifying horizontal windows or the sharing of material settings.
