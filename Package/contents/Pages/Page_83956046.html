Curve Editor

You can check and edit the curve of the selected animation.
Related Links The Curve Editor opens when you select a curve in the animation property window.

Related Links For more information about the basic Curve Editor operations, see here. For the shortcut keys used in Curve Editor, see the Shortcut List.

A Animation properties are displayed.
B Curve Tree View
Displays a tree view of nodes (settings) that can be selected for the animation being created.
You can copy and paste at the node or group level using the shortcut menu that appears when right-clicking.
C Displays the elements of the curve that is being edited.
This area is also used to specify Pre-wrap and Post-wrap for the curve.
There are items shown only for bone visibility, and items shown only for skeletal animation.
D Toolbar
Displays icons that represent features used to edit curves and keys.
Resting the mouse pointer on an icon shows a tooltip that briefly describes its function.
E Curve View
Displays the animation curve for the node selected in the Curve Tree View.
In Curve View, you can create and edit animation curves.
F Time Slider
Information for the selected curve is displayed above the Time Slider.

Items Displayed Only For Bone Visibility

The following additional items appear in the bone visibility curve editor.
Enable compression and Export binary are shown for the curve element that is being edited.

A

You can specify whether to allow compression in the bone compression for the intermediate file optimizer.
B You can specify whether to include the binary source after exporting the target curve when converting using the 3D binary converter.

Items Displayed Only For Skeletal Animation

The following additional items appear in the skeletal animation curve editor.
The binary settings appear for the curve element that is being edited. (These settings cannot be edited)

A Displays whether compression is enabled.

Set using the Maya plug-in for disabling node compression.
B

Shows whether the skeletal animation's translate, rotate, and scale values will be exported to the binary resource.

Set using the Maya plug-in for setting animation binarization.
