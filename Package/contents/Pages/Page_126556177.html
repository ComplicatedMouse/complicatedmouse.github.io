Question

A warning appears when outputting the BFFNT file if the kerning cannot be obtained.

Answer

  • The warning confirms whether the input data has kerning information.
  • If an image is being input, check whether the kerning information (BFFKN) file exists in the same folder as the image file.
  • If a Windows font is being input, check whether the font has kerning information.
    For example, check the font specifications, ask the creator of the font, or try using other software.
