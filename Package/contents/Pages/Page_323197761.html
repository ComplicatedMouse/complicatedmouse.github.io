Set Angle of Move Forward

This section describes the Set Angle of Move Forward feature for setting the direction of translation with the Move forward in the specified direction auto-translation type.

Overview

Set Angle of Move Forward sets the direction of translation when Move forward in the specified direction is selected as the auto-translation type.
Use H Direction Angle to set the angle in the direction of the xz-plane, and V Direction Angle to set the angle in the direction of the yz-plane.
The Move Forward () group appears when Move forward in the specified direction is selected as the auto-translation type.
The following description provides a summary of the settings () for this group.

Horizontal Angle

Use the dial () or the text box () to specify the angle in the direction of the xz-plane.
As shown in the following figure, the angle is 0 when moving forward in the positive direction of the z-axis, and can be set in the range from -360 to 360.

Vertical Angle

Use the dial () or the text box () to specify the angle in the direction of the yz-plane.
As shown in the following figure, the angle is 0 when moving forward in the positive direction of the z-axis, and can be set in the range from -360 to 360.
