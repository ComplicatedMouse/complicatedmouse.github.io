Guides

Guides is the page that configures the guides displayed in LayoutEditor.

Position

Enter the center position of the guide using x- and y-coordinates that take the layout window's origin as the center point.

Rotation

Enter the rotation angle that takes the center position of the guide as the origin.

Color

Set the guide color either by using the color picker or by entering a numeric value.
