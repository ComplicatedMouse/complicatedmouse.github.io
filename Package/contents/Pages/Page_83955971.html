Creating Texture Pattern Animations

This section describes how to set a texture pattern animation effect for a picture pane.

Creating Texture Pattern Animations After Adding Image Files

Use the following procedure to add an image file and create a texture pattern animation in a Picture Pane.

Preparation

To prepare for creating the layout data, start LayoutEditor in advance.

Related Links For more information about starting LayoutEditor, see Workflow.

Opening Windows

Begin the task after opening the following windows.

  • Textures
  • Curve Editor
  • Properties

Related Links For more information about the UI of each window, see Windows.

Loading the Image File

Load the image file to be used in the Texture window from the following location.
Samples/Sources/Resources/Ui2d/Common/Textures/

Thumbnail Filename

Digital.0.tga

Digital.1.tga

Digital.2.tga

After adding the image file, select Digital.0.tga and create a picture pane called P_pict_00.

Related Links For information about adding textures and creating picture panes, see here.

Registering Texture Pattern Lists

Use the following procedure to add the texture pattern list used for texture pattern animations.

1 Select the P_pict_00 picture pane and open the property window.
2

In the property window, in the tree on the left, select Textures ().
3 Select the 0 thumbnail.
4 In Settings, click the Texture Patterns tab.
5 Click Add.
6

Select the following check boxes, click OK, and add them to the pattern list.

  • Digital.0
  • Digital.1
  • Digital.2

Creating Texture Pattern Animations

Open Curve Editor and create a key for the P_pict_00 pane using the following settings.

Pane Name Element Key Time = Key Value Slope
P_pict_00 Material and Texture Pattern 0 0f = 0.00, 20f = 1.00, 40f = 2.00 Step. (See the Note.)

Note: Curve slopes for texture pattern animations are automatically set to Step.

Creating a Texture Pattern Animation Using Load Sequential Textures

The Load Sequential Textures window is a feature for loading texture files with certain names within a specified folder as pattern animations for the created pane.
Animations are automatically generated in the order of the numbers appended to the end of texture filenames.


Info

The following naming convention is used for texture files.
TextureName.XXX.tga (where XXX is any number)


Using Load Sequential Textures

The sequential texture sample is located in the following directory.
Samples/Sources/Resources/Ui2d/Common/Textures/

Use the following procedure to create a texture pattern animation.

1 Select Menu > Tools > Load Sequential Textures.
2 In Target Load Folder, specify the following folder.
Samples/Sources/Resources/Ui2d/Common/Textures/
3 Set Pattern Frame Interval to 10.
4 Select Load.
5

The following types of files are loaded, and a picture pane and animation are created.

  • Digital
  • mark
  • texpat
