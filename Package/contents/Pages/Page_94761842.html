Color Combiner

This section describes the feature for combining the emitter's color elements.

Overview

The Color Combiner is a feature for combining the emitter's color elements.
The Color Combiner handles the following color elements.

The texture's color elements. The color information for the textures to register on the Texture 0, Texture 1, and Texture 2 tabs.
Use from Texture 0 to Texture 2.
The warp component of Texture 0. This information is used with the Particle Shader feature to warp and refract RGB values.
Textures for distortion can be used if they are registered to Texture 0.
The color elements of the particle.

The color information to set on the Color tab.

Use for Formula.
The vertex color elements of the primitive. The vertex color information in the primitive file registered on the Particle tab.
Use for Primitive.

Texture 2
 Operators Select which operator to use for combining with the previous stage (× (multiply), ＋ (add), or - (subtract)).
Channel Select the color channel usage from the presets as with Texture 0, and then set it.
Formula

Combines the RGB of Color 0 (C0) and Color 1 (C1) as set in the Color tab with the RGB of the previous stage (Comb_RGB).

Related Links For more information about the equations for combining colors, see Color Combiner Formula.
