Plug-In for Setting Sampler Attributes

Sets sampler attributes for NintendoWare to bitmaps. These attributes, however, are not applied on the 3ds Max screen. It is also possible to search for settings.
Select Nintendo > Set Sampler Attribute to open the window.

You can use this window to change the settings for multiple bitmaps simultaneously or to search for materials. Selecting multiple nodes selects the materials assigned to those nodes.

The Selected Material field shows the bitmaps included in the materials assigned to the selected nodes.

Setting Method

When one or more bitmaps are selected, the Set field displays current settings for the selected bitmaps.
If multiples are selected, the bitmap setting at the top is displayed.

Enter values for items in the window and click Set to make the changes. You can make multiple selections and change all bitmaps at the same time.
By selecting another node without closing the window, you can set a different bitmap.

  • Name
    Enter the name of the sampler.
    If left blank, data is exported using a name corresponding to the bitmap. The default is a blank field.
  • Hint
    Enter a hint for the sampler.
    You can enter a tool tip in the Hint field if a sampler name is entered.
    If left blank, data is exported using hint information corresponding to the bitmap. The default is a blank field.
Info

The names and hints of samplers used by the same material must not conflict with each other.

Info

Sampler attributes are set for Max maps. They are not actually set for the bitmap files themselves.

Searching for Attributes

You can select all bitmaps that include a sampler name string by entering that string under Name in the Search field and clicking Search on the right.
Entering an asterisk and performing a search selects all bitmaps whose sampler name field is not blank.
Entering nothing and performing a search selects all bitmaps whose sampler name field is blank.

Behavior is the same when conducting a Hint search.
