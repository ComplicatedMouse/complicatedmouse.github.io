Offset Time

This page describes the Offset Time feature that determines the starting position of the curve referenced by per-character transformations.

Overview

Offset Time is a feature for setting the frame that is the starting location for the reference curve used when placing characters.
The appearance of the text box can be changed by setting different offset times even when referencing the same curve.
In addition, you can animate the offset time to create a wave effect for the characters in the text box.
