Basic Operations of the Property Window

The property window is used to view and edit the parameters of models and animations loaded into 3DEditor.

This document describes the basic operations of the Property Window.

Related Links The items in the property window change depending on the selected object. For more information, see Property Windows.

Open Property Window

The property window can be opened by doing any of the following.

  • Double-click the filename in the File View.
  • Double-click the object in the Object View.
  • Right-click the object selected in object view. On the shortcut menu that appears, select Property Window.
Info

You can also open the property window by doing the following.

  • Go to the main menu and select View > Property Window.
  • Click Open Property Window () in the toolbar.
  • Use the keyboard shortcut: CTRL+P

Property Window Icon

The property window's basic operations are performed using the icon in the upper right of the dialog box.

Icon Function Description
Copy Properties Copies the properties of the item selected in tree view. (When the selected item has child items, it also copies the child items. )
Paste Properties All items copied from the properties, including the child items, are pasted.
Material Configuration File Export Used to export material configuration to a material configuration file (F3MT file). (This feature is only enabled for the material property window.)
The exported content may be imported into a different material.
Related Links The items to export can be configured using the Export dialog box.
Importing the Material Settings File

Used to import material configuration from a material configuration file (F3MT file) to a chosen material. (This feature is only enabled for the material property window.)
Related Links The items to import can be configured using the Import dialog box.
Duplicating the Property Window

You can open a copy of the selected property window in a separate window.
Open Help Click the icon to open the HTML Help for the property window.
