Loop-In-Range

This section describes the features of the Loop-In-Range node.

Overview

The loop-in-range feature keeps the particles inside a virtual "box."
As illustrated below, particles are only displayed within the range of the "box" specified in the editor.
When particles go beyond the range, they return to the opposite side of the box and are shown again.

Parameter

The Parameter group () is the group of features for setting ～.
This group has the following settings.

Box Position, Scale, and Rotation Describes the settings for configuring the box position, scale, and rotation.

Proportion of Alpha Fade at Edge

Describes the feature for changing the alpha of particles at the edges of the box.
Fixing the Box in Front of the Camera Describes the feature for fixing the box in front of the camera.
Repeat Count / Offset Per Repeat Describes the feature for repeatedly displaying particles inside the box.
Clipping Settings / Height of Clipping Plane Describes the feature for clipping particles displayed inside the box.

Example Configurations

The following are example configurations of the Loop-In-Range feature.

A Using a Fixed Box Position

Particles display within the bounds of the box.

This method can be used for effects such as having rain fall in only a part of a large map.
B Using a Fixed Box in Front of the Camera

Keeps the box at a fixed distance from the camera, even when the camera is moved forward and backward.

This method can be used for effects such as having rain fall only around the character in a scene where the camera is following the character.
C Reusing Particles Within the Box This method reduces the CPU load by repeating a small number of particles within the bounds of a box.
