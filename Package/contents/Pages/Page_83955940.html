1. Introduction

The following documentation describes the process of using layouts.

2. Creating Layout Data

Layout data is created using the included LayoutEditor tool.

The created layout data is converted to binary files using the binary converter. The achiver collects these into a single archive file, which is what gets loaded by the program.

2.1. Using LayoutEditor

Start the LayoutEditor by running Tools\Graphics\LayoutEditor\LayoutEditor.exe.

Figure 2: LayoutEditor

  • To create a new layout, select File > Create New > Layout.
  • To convert the created layout to a binary file using the binary converter, select File > Export Binary.
  • To add a pane to the layout, select Pane > Create pane.
    A pane is an element in the layout, for example a text box.
  • To set animation to a pane, select Window > Curve Editor.
  • Press the F5 key to transfer to the viewer and to confirm animation playback.
    • You can select which viewer starts after selecting Tools > Viewer Target.
    • When it is the PC Viewer, running a preview will automatically start LayoutViewer. When it is the Hardware Viewer, start by setting Tools > Launch Hardware Viewer in advance.

Related Links For more information about the LayoutEditor, see the LayoutEditor page.

2.2. Binary Converter

The layout data is converted to binary files by the binary converter in LayoutEditor, so a description of how to use the binary converter on its own is omitted.

Related Links For more information about using a standalone binary converter, see Binary Converter.

2.3. Archiver

The exported binary files must be collected together into a single archive file by the archiver. This uses commands on the console.
For example, assume the following binary files have been exported.

./simple
./simple/anim
./simple/blyt
./simple/font
./simple/timg

To create an archive file from these, you enter something like the following command in the current directory. Be sure not to forget the period at the end.

Tools\Graphics\LayoutTools\LayoutArchiver.exe -D simple ../simple.arc . --tile-mode linear --api-type Gl --code-type Source

Related Links For more information about the archiver, see Archiver.
