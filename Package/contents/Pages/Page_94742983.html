Emitter Transform Position(±)

This section describes the Position(±) feature for setting the position when placing emitters.

Overview

The Position(±) feature sets the emitter position and moves emitters by using animation.
The following overview describes the settings ().

Position

Set the position to use as a standard when placing emitters using the X Y Z world coordinate system.

You can use Random Settings () to set the area where placement location is random.
For example, you can set the standard placement position as 10, and scope of random placement as 20.
As indicated in the following figure, emitted particles are placed randomly in a range of 20 that is ±10 from the placement coordinate 10.
