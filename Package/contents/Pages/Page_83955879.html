Historical Stripe2

This section describes Historical Stripe2, which, in addition to the features of Historical Stripe, can also control the movement of the last of the past samples.
Related Links For more information about historical stripes, see Historical Stripes.

Overview

Past samples in Historical Stripe2 are sets of information recorded for each frame about the position and movement direction of a particle emitted from an emitter.
As a result, an historical stripe2 is a string-like set of connected particles in which this recorded positional information is used for the polygon vertices.

The following image provides an example of an historical stripe2 when the previously recorded past samples have been connected together with a subsequently emitted particle.
The past samples of individual particles can respectively be connected together in different ways, at the same time, as shown in 1 and 2.

Historical Stripe2 records past samples using time.
For this reason, the past samples are not affected by changes in playback speed caused by the game application, and stripe length is maintained.
However, stripe length is not maintained when the playback speed is greater than 1.

Historical Stripe2

Historical Stripe2 () is a group of settings for creating particles with string shapes.

Calculation Method

Determines the orientation of the particle face.
Option Determines the shape of the particle.
Assign UV This feature determines the method for assigning UV to string-shaped particles.
Perfectly Follow Emitter Determines whether past samples follow the emitter when it moves.

History Length

Sets the length of the stripe.
Texturing Determines how texture is mapped to the particle.
Historical Stripe2 is the same as the Historical Stripe feature, but it can be individually specified for textures 0, 1, and 2.
Number of Divisions Sets the number of particle divisions and smooths out the 'string'.
Move History Using Waveform

This feature specifies the waveform for moving the stripe history.
Starting point alpha, End point alpha Sets the alpha values for each end of a string-shaped particle.
Starting Scale, Ending Scale Sets the scale values for each end of a string-shaped particle.
Info

Two or more past samples must be calculated before you can display Historical Stripe2 in the viewer.
However, when one or more number of divisions have been set, a total of three or more past samples is required.
