Model Property Window: LOD Model Page

This page displays the number of polygons and vertices for each level of detail when the selected model file (FMD file) includes a mesh LOD.

A Indicates the LOD level.
B Indicates the number of polygons used in the LOD level.
C Indicates the number of vertices in the LOD level.
D Indicates the number of vertices to process in the LOD level.
E Indicates the number of vertices to process per triangle in the LOD level.
