Support for Environments With Visual Studio Not Installed

Overview

This section describes use cases where you want to build a project, even though debugging with a build machine and the like is not necessary.
In that type of environment, you can build the NX project by installing Visual Studio Build Tools, and various VSI-related files, without installing Visual Studio.

Environment Using Visual Studio Build Tools 2015

  1. Install NintendoSDK\Installers\Vsi\VsiForNX_v140.*.msi .
  2. Build in a form similar to MSBuild.exe NintendoSDK\Samples\Sources\Applications\OsThread\OsThread-spec.NX.autogen.vs2015.sln /p:Configuration=Develop /p:Platform=NX64.

Environment Using Visual Studio Build Tools 2017

  1. Install NintendoSDK\Installers\Vsi\VsiForNX_VSBuildTools2017.*.msi.
  2. Build in a form similar to MSBuild.exe NintendoSDK\Samples\Sources\Applications\OsThread\OsThread-spec.NX.autogen.vs2017.sln /p:Configuration=Develop /p:Platform=NX64.
