HTML/CSS/ECMAScript Support List

This page provides lists of the HTML/CSS/ECMAScript specifications supported by the web applet and Offline Web Applet in each version of the SDK.

In NintendoSDK NX Add-On 9.1.0 (Firmware for NX 9.0.0-2.0), only the web browser included in the web applet was updated. The web browser included in the offline web applet was not updated to preserve compatibility. As a result, the HTML/CSS/ECMAScript support lists for the web applet and Offline Web Applet are now different. For versions earlier than NintendoSDK NX Add-On 9.1.0 (Firmware for NX 9.0.0-2.0), the support lists are the same.

Web Applets

  • NX Add-On 9.1.0 (Firmware 9.0.0-2.0) and Later (Web Applet)
  • NX Add-On 4.4.0 (Firmware 4.0.0-5.0) and Later (Common)
  • NX Add-On 3.4.0 (Firmware 3.0.0-11.0) and Later (Common)

Offline Web Applet

  • NX Add-On 4.4.0 (Firmware 4.0.0-5.0) and Later (Common)
  • NX Add-On 3.4.0 (Firmware 3.0.0-11.0) and Later (Common)
