3D News

This section introduces new features added to the 3D module.

  • 3DEditor
    • You can now check the Binarize values of each bone.
    • It is now easier to drag the slider in the seek bar.
    • Added the ability to control the default value of each material reference settings for the additional sampler.
  • Maya Plug-In
    • You can now specify to export cameras with the visibility attribute disabled in the config file.
    • You can now specify to export camera, light, and environment fog with node compression disabled.
    • Added config keyword values to environment variables that can be obtained with Pre-Export Script and Post-Export Script.

3DEditor

You can now check the Binarize values of each bone.

The values set for the Set No Compress Node and Set Animation Binarization Maya settings can now be checked in 3DEditor.

To check these values, check the curve elements in the animation property window.

Maya Plug-In


You can now specify to export cameras with the visibility attribute disabled in the config file.

Added a setting to the config file in the Maya plug-in to set whether to export cameras that have disabled the visibility attribute.
Configure the setting as follows. 

export_invisible_camera="true"


Related Links For more information, see the following pages.

You can now specify to export camera, light, and environment fog with node compression disabled.

This export is possible by going to Set No Compress Node and selecting Don’t compress this node for cameras that have disabled the visibility attribute.

If export_invisible_camera="true" is not specified in the config file, a camera with the visibility attribute disabled is not exported but can be exported by disabling node compression.
Light with the template attribute enabled and environment fog with the visibility attribute disabled can also be exported by disabling node compression.
(Although camera, light, and environment fog are not subject to node compression, disabling node compression allows them to be forcibly exported regardless of the attribute value.)
