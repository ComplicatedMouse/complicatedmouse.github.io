Interfaces

You can configure elements of the EffectMaker user interface.

Dialog Boxes

The Dialog Boxes group () is used to configure dialog boxes.
This group has the following settings.

Use XP Style

Toggles the XP UI design of EffectMaker's Save and Load dialog boxes on or off.
Display "Hint of the Day" at Start Selects whether to display the Hint of the Day dialog box when EffectMaker is started.

Behavior When Clicking Textbox

The Behavior when clicking textbox group () is used to set the behavior of the text box when clicked.
Selects whether to select all when right-clicking in a text box.

Text Editor to Use

The Text Editor to use group () specifies the text editor to use in EffectMaker.
The text editor specified using this feature will be used in the following features of EffectMaker.

  • Display error dialog box
  • Display shader code of the custom shader
