Command-Line Options

You can specify the file to be loaded by specifying the command-line option when starting Fcpx.

Format

Use a format like the following.

FcpxMaker.exe [--flpj=path] [--platform=arg] [input FCPX file path]

Options

Options Description
--flpj Specifies an FLPJ file. Settings for file loading pre-commands and file saving pre- and post-commands are loaded.
-p, --platform Specifies the target platform. Use commas as delimiters to specify multiple entries. Used for the call to BfftfMaker.

Example

Load the layout project settings file NW_LytProject.flpj and open IconFontBase.fcpx.

FcpxMaker.exe --flpj=NW_LytProject.flpj Common\Fonts\IconFontBase.fcpx
