XZ Translation Speed

This section describes the XZ translation speed feature that controls the speed the selected node travels on an XZ plane.

Overview

XZ translation speed controls the speed the selected node travels on an XZ plane.
The speed is increased by increasing the value.
The auto-translation type is the same as none when the value is 0.
The following overview describes the settings ().
