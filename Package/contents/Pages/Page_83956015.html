Features

This section describes the features provided by the operating system library and how to use the API resources.

Features

The os library provides the following features. Some of the features are also provided with class utilities.
For more information, see Classes.

Feature Overview Utility Classes

Threads

Functionality that provides the smallest unit of execution that can be used to run multiple processes simultaneously on a processor. -------
Thread-Local Storage Functionality that provides the smallest memory region capable of recording the different content for each thread. ThreadLocalStorage Class
Fiber A feature of threads that enable further switching between multiple context. -------
Semaphores Functionality to exclude shared resources, using an enumeration counter. Semaphore Class
Event Functionality for synchronizing among threads, based on two values corresponding to either being in a signaled state or being in a nonsignaled state. Event Class
Light Events Lightweight version of the functionality for synchronizing among threads, based on two values corresponding to either being in a signaled state or being in a nonsignaled state. LightEvent Class
Message Queues Synchronization functionality for sending and receiving any data among threads. MessageQueue Class
Timer Events Synchronization functionality that adds delay signalization functionality from the timer to event functionality. TimerEvent Class
Multiple Wait Synchronization functionality for waiting for any one of several synchronization objects to enter a signaled state. -------
Mutexes Synchronization functionality for mutual exclusion of shared resources that are being used among multiple threads. Mutex Class
Condition Variables Synchronization functionality for waiting on certain shared resource conditions. Used with mutexes. ConditionVariable Class
Readers-Writer Lock Synchronization functionality for locking in conjunction with the reading or writing objective of shared resources. ReaderWriterLock Class
Busy Mutex

Synchronization functionality for mutual exclusion that the calling thread can use to run with priority on its own CPU core.

 BusyMutex Class
Barriers Synchronization functionality for waiting among multiple threads. Barrier Class

System Events

Synchronization functionality for waiting on system notification event signalization.

SystemEvent Class

Time Management Functionality providing a tick counter. -------
Memory Management Functionality related to memory, including the allocation of dynamic memory. -------
User Exception Handler Functionality for handling user exceptions with your own functions. -------
Cache Operations Functionality for cache maintenance. -------
