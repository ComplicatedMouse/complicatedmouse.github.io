Curve Tree View

Curve Tree View

The Curve Tree view shows a tree view of nodes included in the specified animation.
This view is used for operations such as selecting a node to edit the curve for, and changing the display mode of the curve.

  • Basic Operations of the Curve Tree View
  • Icons of the Curve Tree View
  • Shortcut Menu in the Curve Tree View

A The following icons change the display mode of the Curve Tree View.
B Finds and filters the nodes in the Curve Tree View.
C Displays the nodes that you can select.
When you select a node that has a curve, the curve is displayed in the Curve View.
You can show and hide curves by selecting the check box for each node.

Operations of the Curve Tree View

This section describes the basic operations of the curve tree.

Searching in the Curve Tree View

You can filter the nodes that are displayed in the curve tree by searching for a string.

The string texture is used to filter the nodes in the following example.

Info

You can filter on multiple words separated by whitespace.
Example
If you type the string texture SRT, the tree view searches for both texture and SRT.

Selecting a Node

A node is selected when you click the square (indicated by the red arrow in the following screenshot) in the Curve Tree View.
When a node is selected, the square turns blue.

You can select multiple nodes and edit them simultaneously.

The preceding screenshot represents what is displayed on the screen when simultaneously editing or adding keys to the diffuse R, G, and B.

Info

You can change the pre-wrap or post-wrap setting of a selected node.
If multiple nodes are selected, curve settings for all selected nodes can be changed at the same time.

Selecting a Node and Focusing on the Animation Curve

After clicking a node to select it, double-click in the selected node to fill the Curve View with that animation curve.

Sorting Nodes in Alphabetical Order

Curve Editor has a feature for sorting nodes in skeletal animations and bone visibility animations in alphabetical order.
Click the following icon to enable or disable alphabetical sorting.

By default, nodes are sorted by index.
