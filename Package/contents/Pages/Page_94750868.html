Specified-Direction Initial Velocity

This page describes the Specified-Direction Initial Velocity feature that sets the particle initial velocity.

Overview

Specified-Direction Initial Velocity sets the initial velocity of particles in the direction set by Specified direction.
You can move particles in a specified direction using this feature.
You can randomize the direction of the particle's initial velocity by using diffusion angle.
The following overview describes the settings ().

Specified-Direction Initial Velocity

Sets the initial velocity to apply in the specified direction.

The Specified-Direction Initial Velocity group () includes the following settings.

Apply in World Coordinates This feature determines whether to apply the direction in which initial velocity is applied to the slope of the emitter.
Specified direction This feature sets the direction of the emitted particle.
Specified Direction Dispersion Angle This feature sets the angle at which particles disperse in the specified direction.
