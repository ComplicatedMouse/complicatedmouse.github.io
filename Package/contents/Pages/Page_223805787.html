How Can I Display Double-Byte OTF Characters in Monospace Like TTF?

Question

I am having trouble displaying double-byte OTF characters in monospace like you can do with TTF fonts. How can I display them using monospace?

Answer

You can use proportional matrix (PALT) data, included in OTF files, to proportionally display double-byte character fonts naturally and as intended.

You can also, however, ignore the PALT data and display text with a fixed width when you want to do so for double-byte characters. For example, you could differentiate between double-byte and single-byte symbol widths. Set ignorePalt to true using the <scalableFontDescription> tag within the FCPX file, as follows.

<scalableFontDescription boldWeight="0" borderWidth="0" ttcFontIndex="0" ignorePalt="true">
