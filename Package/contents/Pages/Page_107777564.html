Camera Distance Scale Restriction (Far)

This section describes the Camera Distance Scale Restriction (Far) feature, which applies a limit to the sizes of particles that are far away from the camera.

Overview

The Camera Distance Scale Restriction (Far) feature keeps particles from getting any smaller than necessary as they move away from the camera.
This allows you to set display sizes for game character data or other data needed for the game so that they can be recognized by the player regardless of the camera distance.
The following overview describes the settings ().

Camera Distance Scale Restriction (Far)

Select or clear the Camera Distance Scale Restriction (Far) check box () to enable or disable the feature.
When this check box is selected, the Scaling limit distance (far) group () is displayed. The group has the following settings.

Scaling limit distance (far)

Limits the size of particles shown on camera that are farther from the camera than the value set using the dial ().
