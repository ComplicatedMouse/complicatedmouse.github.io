Specify Binary File

This section describes the Specify File feature for registering binary files.

Overview

The Specify File feature can be used to register a binary file.
Check the effects on the Viewer by loading the exported binary file.
The following overview describes the settings ().

Specify File

Displays the path of the binary file and allows you to register it anywhere. The following are button types.

Load Opens the file selection window for selecting a binary file.
Reload Updates the configured binary file.
Clear Clears the configured binary file.

The following buttons control the playback of emitter sets in the binary file.

Play Previous Emitter Set Plays the previous emitter set in the Viewer.
Play Next Emitter Set Plays the next emitter set in the Viewer.
