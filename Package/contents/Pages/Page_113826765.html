NintendoWare News

This section provides news topics related to each release that affect all of NintendoWare and various modules.
For considerations when migrating to a specific version, see Changes for Developers.

The news about various modules is mainly written for artists and sound designers, but it is also suggested reading for developers because it contains beneficial information.

Note 1: In 7.0.0 and later, news for developers is migrated to each module's news.
