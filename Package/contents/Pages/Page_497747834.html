3D News

This section introduces new features added to the 3D module.

  • 3DEditor
    • Hidden groups are no longer displayed in the shader tree in the material property window.
  • Photoshop Plug-In
    • Sharp cubic interpolation can now be specified in Mip Gen Filter options.
    • Slice names can now be checked in a list in the Select Target Slice window.
  • Maya Plug-In
    • The Overwrite Confirmation dialog box can now be displayed when exporting from the multi-export plug-in.
    • In the export plug-in, added a feature for specifying a script to be executed after the overwrite confirmation dialog box is canceled.
    • Added a feature to cancel output from a script specified with pre_multi_export_script in the multi-export plug-in.
    • In the main window of the Maya multi-export plug-in, added a feature for executing a MEL procedure when items are selected from the export item list.
    • You can now automatically enable a specified animation layer when the item is selected in the multi-export plug-in.
    • Added a feature for using a pre-export script to change the maximum number of nodes to which skinning weights can be assigned.
    • You can now also select nodes not in the LOD group and included in output targets when exporting LOD.
    • You can now import FMD files that reference image files (TGA, PNG, DDS, or EXR files).
    • The material blending of mesh nodes is now set to multiplicative if any vertex colors are present in the FMDB shape to be imported.
  • 3D Command-Line Tools
    • Improved the algorithm for distributing shader variations for distributed compilation with 3dShaderConverter.

Maya Plug-In

The Overwrite Confirmation dialog box can now be displayed when exporting from the multi-export plug-in.

In the Maya multi-export plug-in, added a feature for displaying an overwrite confirmation dialog box.
Overwriting multiple items can now be confirmed in a single dialog box.

Specify the following option in the config file to enable overwrite confirmation.

multi_export_confirm_overwrite="true"

The timing of the overwrite confirmation is after the script specified by pre_multi_export_script has been executed.

If post-processing is necessary after cancellation by the overwrite confirmation, you can specify a script to be executed after cancellation from pre_multi_export_script.

NintendoSetPostCancelMultiExportScript(string $cmd)


An overwrite cannot be properly determined in the following cases.

  • You are using an environment variable for the path of Output Folder and the FDES file, and have set the value of that environment variable in Pre-Export Script. (Does not apply when the value of the environment variable is set with pre_multi_export_script.)
  • You are overwriting the value for the output_file_name, output_folder, or remove_namespace options in Pre-Export Script.

In the main window of the Maya multi-export plug-in, added a feature for executing a MEL procedure when items are selected from the export item list.

In the main window of the Maya multi-export plug-in, added a feature for executing a user-specified MEL procedure when the selection state of the export item list changes.
Specify the following in the config file. 

multi_export_user_item_select_callback="project name"

The procedure argument is the index array (int array) of the selected output items.

For example, the following kind of procedure is available.

global proc MyNwMultiListOnSelect(int $itemIdxs[])
{
    print("item count: " + size($itemIdxs) + "\n");
    print $itemIdxs;
}

Specify in the config file.

multi_export_user_item_select_callback="MyNwMultiListOnSelect"

This example displays the item numbers and total number of selected items.

LOD Export

Describes the use and features of LOD Export mode.

You can now import FMD files that reference image files (TGA, PNG, DDS, or EXR files).

In the import plug-in, when an FTX file referenced by an FMD file does not exist but a corresponding image file (TGA, PNG, DDS, or EXR file) does exist, it is now possible to import the FMD file.

This capability enables you to create the same type of image files as the image files (in the src_path_r path) originally produced when the FMD file referred to FTX files.


Priority order of folders searched for image files:

  1. (fmd file folder)/textures
  2. (fmd file folder)
  3. sourceimages folder at the import destination

Priority order when there are image files in the same folder that have the same filename but different filename extensions:

  1. TGA
  2. PNG
  3. DDS
  4. EXR

3D Command-Line Tools

Improved the algorithm for distributing shader variations for distributed compilation with 3dShaderConverter.

Improved the algorithm for distributing shader variations when the --variation-begin-ratio and --variation-end-ratio options have been specified for 3dShaderConverter.
There are sometimes situations where the number of shader variations in the exported shader archive (BFSHA) file differs even if the value specified for --variation-begin-ratio or --variation-end-ratio in earlier versions of the SDK is the same as the value specified in the current or later versions of the SDK.
