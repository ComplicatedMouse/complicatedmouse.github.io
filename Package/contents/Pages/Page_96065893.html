Transform Vertex by Velocity

This section describes the Deform vertex based on speed feature that stretches a particle in the speed vector.

Overview

Deform vertex based on speed is a feature that applies a scale in the particle's y-axis direction in proportion to the particle's movement speed.
Applying this feature to particles with speed can provide the impression of an afterimage as objects appear to be elongated along the speed vector as they move at high speeds.
The following overview describes the settings ().

Transform Vertex by Velocity

Select to apply Deform vertex based on speed.
Enable this setting to display the Deform vertex based on speed () settings. The group has the following settings.

Add-on value

The specified value is added to the particle y-axis direction. The following equation derives the scale in the direction of the y-axis.

  • Scale along y-axis: Size along the y-axis + (Speed x Value specified by adding the speed to the y scale)
