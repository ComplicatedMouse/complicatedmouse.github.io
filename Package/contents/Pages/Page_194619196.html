Vulkan Library

Introduction

Vulkan is a real-time graphics conversion API developed by the Khronos Group.

Vulkan is supported in the NintendoSDK NX Add-On and is provided with the assumption that NintendoSDK users will want to migrate existing titles and frameworks that use Vulkan.

PC Operating Environment

Conforms to the NintendoSDK operating environment.

Installing the GPU Driver

Set up the Vulkan environment on the PC you are using.

The NVIDIA Graphics Drive package available from Nintendo Dev Interface supports Vulkan.

Supported Versions

Nintendo Switch™ is based on Vulkan® 1.1.
