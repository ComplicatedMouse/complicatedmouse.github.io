Using the Capture Button

The following section describes the functionality available from the Capture Button.

Feature Overview

Screenshot Capture Functionality

You can save a screenshot of the running application by briefly pressing the Capture Button.

You can use the screenshot capture functionality from its API. For more information, see Screenshot Feature.

Video Capture Functionality

Info

The video capture functionality is available to applications enabled in NintendoSDK Firmware for NX 4.0.0-x.x and later.

You can save image and sound from the running application as a video file when you hold down the Capture Button.

You can save a maximum of 30 seconds of video, back to the point where the system determined that the Capture Button was being held down.

This video capture functionality is enabled by default in applications built with NX Add-On 4.2.0 and later (or 3.6.0 and later in the NX Add-On 3 series).

Saved video files have the following specifications.

Item Description
Format MP4
Videos H.264 / 720p / 30fps
Audio AAC / 48kHz / stereo
Playback time Maximum 30 seconds
Note

Video can only be captured when the running application is in the foreground.

Any content that includes transitioning to the HOME Menu, calling the library applet, or entering sleep mode will be lost.

File Save Locations

If an SD card is inserted, use that as the save location.
An SD card must be inserted before the system is started. The system must be restarted if the SD card is removed or inserted while the system was starting.

You can view captured screenshots and videos from album in the console features. For more information about the album, see the System Feature Guide.

Enabling and Disabling This Functionality

The following methods are available for controlling screenshot and video captures.

  • Controlling with NMETA file settings Writing the NMETA File
  • Specifying no capture segments with the API Operating Environment Controls

Effect on Applications

There is a performance hit for applications that have enabled video capture.

For more information, see Important Implementation Information (NX Add-On).
