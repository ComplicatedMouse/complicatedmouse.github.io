Matching the sizes of bitmap fonts and scalable fonts.

Question

How do I match the sizes of a bitmap font and a scalable font?

Answer

Specifying the same value for font size produces almost the same render result.
Set the following items to the same value in the font converter.

  • Size value of the bitmap font.
  • Size value of the scalable font.
    (This is specified by the project file in the layout editor and by the developer in the program.)
