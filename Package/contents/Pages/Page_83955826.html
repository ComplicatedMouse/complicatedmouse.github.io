Setup Manager

Introduction

The setup manager is a tool for changing the settings of the thumbnail explorer extension feature.
Use the tool to change the settings for previewing thumbnails of files like FTX, TGA, and PSD files and their associated applications.

Note

If application associations are created without using the setup manager, thumbnail and preview display may not be enabled.
In such cases, use the setup manager to change the settings again.

Default Settings

After installation, using the installer, all features are automatically enabled using standard settings.
Use Setup Manager to modify these standard settings.

Those standard settings used during installation are as follows.

How to Use

Setup Manager is installed in %ProgramFiles%/Nintendo/ThumbnailExplorerExtensions/Setup/.
To use Setup Manager you must have Windows administrator rights.

  1. To start Setup Manager, run ThumbnailExplorerExtensions/Setup/ExtensionManager.exe.
  2. From the tree menu on the left, select which file type (FTX, TGA, PSD, AI, ESET, EPS, FLYT, FMD) to configure settings for.
    Select All Files to configure all file types at the same time.
  3. Check that the explorer extension features (such as thumbnails and previews) are enabled for all desired file types.
    The Select all/Deselect all button switches between enabling and disabling all check boxes.

  4. Select the application to open the file type when the file is double-clicked from the pull-down menu. When All Files is selected, only None, Reset, or Custom can be used.

    Item Description Notes
    None No application is selected to open the file.
    Reset Reverts the settings to their states prior to using Setup Manager.
    TextureViewer Opens a file in TextureViewer. FTX only.
    Photoshop Opens the file using Photoshop.
    The installed versions of Photoshop are automatically displayed.    		 FTX, TGA, PSD only.
    Custom Manually specified application is used to open the file.
    Click Browse and select the application to use.
  5. Click OK and exit Setup Manager.
Info

Windows Explorer will restart as part of the installation process. Note: You do not need to restart the PC.
The installation may take several minutes, depending on the installation environment.
Do not move or delete the folder in which the plug-in has been installed.

Edit Config

To change the Explorer extension settings, restart the Setup Manager.

An asterisk (*) appears in the Setup Manager title bar when settings have been changed.

To change the settings for the thumbnail explorer extension feature, change them and then click OK.
