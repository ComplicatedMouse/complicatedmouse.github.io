Fixed Size Camera Offset

This section describes the Fixed Size Camera Offset feature for pass negation when offsetting the particle render position either forward or backward relative to the camera.

Overview

Fixed Size Camera Offset is a feature for Offset toward camera processing to move the camera to the foreground after adjusting the size to the vertex level on the screen.
In the Offset Z value implementation, the camera's near and far settings may cause display issues.
Fixed Size Camera Offset resolves these issues with Offset toward camera processing, after which pass negation is possible.

The following overview describes the settings ().
When you set Particle Offset to Fixed Size Camera Offset, the Particle Offset settings () are displayed. The group has the following settings.

Particle Offset Value

You can set the offset value for camera direction with the dial () or text box (), based on the emission position.
To apply the offset further back behind the position of emission, specify the negative direction.

Multiply Offset by Scale

Use the check box () to select whether to multiply the z value by the scale applied to the emitter and the final result of applying the scale to the particle.
When this setting is selected, you can multiply by the scale to remove overlapping polygons.
