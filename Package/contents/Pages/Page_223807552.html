2. Prepare the Maya Scene File

This section describes the procedure for importing the model and the animation.

In this tutorial, an existing FMDB file is imported to Maya for preparing the Maya scene. Normally you would open a Maya scene file.
Load the files (model and skeletal animation) from the sample in the package.

Nintendo Import

Use Nintendo Import to import the model file (FMDB) and the skeletal animation file (FSKB) to Maya.

Open the Nintendo Import Window

From the Maya menu, select Nintendo > Import.

Configure the Settings in the Nintendo Import Window and Load the File

Configure each setting and load the file.

Load the file by performing the operations in the order described in the following images and table.

Order Category Operation
1 General Options

Set Magnify to 1.0000. Leave Namespace empty.
Select Create Material, Create Texture Node, and Create Texture File.
Keep LDR and HDR set to Auto.

Import

Click the Import tab.

Click the folder icon () and specify the following FSKB file.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod\ExportSampleTripod.fskb

3 Import Options Import All

Click Import All. Imports all of the specified files to Maya.
