3D News

This document is a graphical introduction to the features that were added in NintendoSDK 0.14.0.

Enabling Configuration of an Additional Reference Path to Material Animations

As with additional reference paths for models, you can configure additional reference paths for material animations in 3DEditor or the assign utility.

Related Links For more information, see the following pages.

3DEditor A UI tool that allows you to edit intermediate files that you handle using 3D. You can edit model and animation intermediate files.
Assign Utility
Command-line tool that assigns the different types of data for intermediate files using 3D.
Additional Reference Path

You can add a depth range for a file that is referenced by a different file when the latter file is loaded and at other times.
Additional Reference Path Page

Sets the additional reference path page for a model or material animation in the 3DEditor property window.

Support for Photoshop CC 2015.55

The Photoshop plug-in is now compatible with Photoshop CC 2015.5.

Enabling the Use of Relative Paths for Paths Exported to src_path for Intermediate Files

You can now export a relative path from a folder specified in the src_path attribute for the <create> element for an intermediate file.
To export using a relative path you make configure settings using the various tools.

Tools Specification Format
Maya Plug-In

Specify the folder that will serve as a root for the relative path using project_root in the config file.
Photoshop Plug-In Specify the folder that will serve as a root for the relative path using project_root in the config file.
Intermediate File Converter Specify the folder that will serve as a root for the relative path using the --project-root option.
Texture Converter Specify the folder that will serve as a root for the relative path using the --project-root option.
Info

As an example, you can now tell from the relative path from the root folder which Photoshop (PSD) file was used to create a texture intermediate (FTX) file.
