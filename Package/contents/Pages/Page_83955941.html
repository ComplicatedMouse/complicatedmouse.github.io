Animation Property Window: General Page

Animation Property Window: General Page

This page is used to check the general properties of the selected animation.
The information displayed changes depending on the type of animation.

Note: For scene animations, you can enter fixed parameter values (constant values so there is no animation).

Info
Related Links The number of frames and looped playback option can be edited with Curve Editor.

A The number of frames in the animation.
B Indicates the frame resolution used when baking, quantizing, or creating frames.
C Whether looped playback is enabled for the animation.
D Displays information set using a DCC tool.
The information displayed changes depending on the type of animation selected.
E Specifies the allowable color error when baking animations as an absolute value.
F Specifies the texture matrix when baking animations.
G

Specifies the quantization tolerance to use when quantizing animations.
H Displays a list of reduction rates based on the value specified for quantization tolerance.

Animation Frame Count

3DEditor supports animations with a frame count in the following range.

Info

0 through 16777215

Setting the Number of Frames and Looping

You can configure the number of frames and looping in the General page of the animation node.

Related Links The number of frames and the loop playback settings can be changed in Curve Editor Animation Settings.

Items Displayed Only for Skeletal Animation

The following settings have been added to the display of general properties for skeletal animations.

A The scaling calculation method of the bone matrix.
B The rotation calculation method of the bone matrix.
Info
The bone matrix calculation method has changed with how data is created using the DCC tool.
Related Links For more information, see scale mode and rotate mode in the 3D Intermediate File Specification Manual.
