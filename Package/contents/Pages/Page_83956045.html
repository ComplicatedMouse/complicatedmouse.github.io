Pop-Up Slider

The pop-up slider bar in 3DEditor allows you to enter and input values in an analog fashion.
Related Links For information about the shortcut keys that correspond to slider operations, see here.

A Directly enter a number within the range supported by the slider.
B Click this button to display a slider window.
Click somewhere other than the slider window to close it.
C The value is restored to what it was when the slider window was displayed.
D Reduces the range of settings by one level.
This operation is available only when editing values that have a variable setting range.
The same operation can be performed using the minus (-) key.
E Increases the range of settings by one level.
This operation is available only when editing values that have a variable setting range.
The same operation can be performed by using the plus (+) key.
F Changes the value by using the slider bar.
Resting the mouse pointer on the slider displays the value at that position.

Reduced Slider Range

If you used the feature for reducing the slider range and the value that was already set is now outside of the range, that value is not changed. Also, a red frame is displayed on the slider.

For example, a value of 1 is set, but the range is reduced to 0.1.

In this example, you can move the slider to bring (clamp) the value back within the available range.
