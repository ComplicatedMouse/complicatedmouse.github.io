Emitter Extended Parameter

Specifies files for extended emitter parameters used on a per-emitter and per-child basis by EffectMaker.

Extended Emitter Parameter Definition File Path

The Extended emitter parameter definition files group () is used to add action definition files used in a game project.
You can use files in which particle actions unique to game applications are written.

Extended Emitter Parameter Settings

The Extended emitter parameter settings group () is used to display action definition file descriptions.
