Emitter Time and Particle Time

Effect animations can be created along two different axes of time: emitter time and particle time.
The following figure is used as an example to describe these two kinds of time axes.

The figure depicts the following effect settings.

Emission Start Frame

 10 frames Emission interval 20 frames Lifespan 20 frames

The following table describes Emitter time and Particle time, using the figure and the settings as an example.

A Emitter Time Emitter time is the time axis for the time since the effect was called. The animation influences emitted particles as a whole.
In the figure, the animation starts in frame 0, where the effect was called, and continues to play according to the animation settings.
B Particle Time

Emitter time is the time axis representing the time from when a particle is emitted from the emitter until it dies. This axis can be used to animate individual particles.

In the figure, frame 10 has been specified as the Emission start frame, so the animation starts when the particle is emitted 10 frames after the effect is called.
Particle lifespan has been set to 20 frames, so the animation is set to 100% between frames 0 and 20.
