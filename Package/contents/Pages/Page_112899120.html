Font Size Specifications

Introduction

This page describes font size specifications.

Font Size

Font size is a concept only encountered when inputting a Windows font with the font converter or when using a scalable font. Font size is measured in pixels.

In the font converter, if you specify the use of the character width specified by the font as the character width when inputting a Windows font, the font size value specified by the bitmap font cell width is set.

The value of a scalable font obtained with GetWidth() in the program matches the font size value. A value about the same as the font size is rendered and seems to be drawn as the font width, but the width of individual characters does not necessarily match the font size value.

The font size specified by the font converter and the font size in scalable fonts share the same concept. As a result, when you set the same value, you get almost the same rendering results.
