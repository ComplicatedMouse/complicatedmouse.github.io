Audio News

This page describes the changes in NintendoSDK 4.0.0.

SoundMaker

You can now use the application used for Open Referenced File for all file types.

You can now specify the application used for Open Referenced File when opening MIDI files and waveform files in SoundMaker.

  • Previously, you could specify the application only for text sequence files.

You can use this feature to preview waveform files in Windows Explorer and edit them when opening them in SoundMaker, etc.

Info

Along with this change, the external editor settings for the text sequence file were moved from the General tab of the option dialog box to the External Editor tab.

For more information, see Option Settings.

Enabled the selection to use the hardware accelerator when decoding Opus stream sound.

A feature to configure Opus decoding using the hardware accelerator was added to SoundMaker.

From Project > Project Settings > Convert, select Specify use of hardware in OPUS decoding in the target.
The setting to use the hardware accelerator will be applied to all Opus stream sound.

For more information, see Compressed Audio.

Command-Line Tools

Added the SoundProjectUtility command-line tool.

Added the Tools/Audio/SoundMaker/SoundProjectUtility.exe command-line tool.

This tool provides a variety of features for the sound project. The following features are available for this release.

  • Displaying a list of all files referenced by the sound project
  • (Under evaluation) Updating the sound project to the latest version compatible with SoundMaker

For more information, see SoundProjectUtility.

Added a feature to SoundProjectWaveExporter that splits the sequence sound into individual tracks and converts each track to a waveform.

Added a feature to Tools/Audio/SoundMaker/SoundProjectWaveExporter.exe that splits the sequence sound into individual tracks and converts each track to a waveform.

  • --split-sequence-sound-tracks

This feature is expected to be used to easily convert past resources created as sequence sounds into a waveform format that would be easier to edit in DAW.

For more information, see SoundProjectWaveExporter.
