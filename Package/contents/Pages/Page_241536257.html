Playback Method - Surface

This section describes the Surface feature for pattern animation using array textures.

Overview

The Surface playback method is a feature for pattern animation using array textures.
Textures are played in the order that they have been registered to an array texture, where the particle lifespan is one cycle.
It even works where the lifespan is random and the lifespan varies for every particle.

Surface differs from other playback methods in that it lets you do the following.

  • Use cross-fading when switching patterns.
  • Animate more than 32 patterns.
Info
Surface involves a heavier processing load than other playback methods.
In addition, the processing load is even heavier when cross-fading is enabled.

You can create array textures using the 3D texture converter command-line tool.

Crossfading

Enable or disable cross-fading using the check box ().
When this box is selected and the feature is enabled, you can animate cross-fading between patterns.

Number of Loops

Use the dial () or the text box() to set how many times to loop the playback in the range of the lifespan.
