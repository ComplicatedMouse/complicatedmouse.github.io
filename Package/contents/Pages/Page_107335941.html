Controlling the Development Hardware

Introduction

This documentation describes tools to perform the following tasks for development hardware.

  • Managing the development hardware (SDEV)
  • Initializing the development hardware (SDEV, EDEV, HDEV, ADEV)
  • Running your application
  • Automated testing
  • Installing Applications
  • Managing save data
  • Managing controllers
  • Managing albums
  • Receiving, saving, and browsing logs
  • Managing the system settings.

Using Target Manager 2

Each of the tools used to control development hardware can be used even if the host PC is configured to use Target Manager 2.
For more information about using Target Manager 2, see Guide to Setting Up the Environment > Using Target Manager 2.
