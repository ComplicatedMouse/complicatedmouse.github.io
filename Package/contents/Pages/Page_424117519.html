Audio Settings

Use the audio command.

$ RunOnTarget DevMenuCommand.nsp -- audio --help
usage: DevMenuCommand audio mute
       DevMenuCommand audio set-output-mode --target <target, tv/sphp> --mode <mode, auto/1ch/2ch/6ch>
       DevMenuCommand audio get-current-output-target
[SUCCESS]

Muting Speaker Output

This command mutes the speaker output. Designed for automated test environments and the like.

$ RunOnTarget DevMenuCommand.nsp -- audio mute
[SUCCESS]

Setting the Number of Output Channels

Set the number of output channels for the audio output target.

Use --target to set the audio output target, and --mode to set the number of output channels.
If sphp has been set for --target, you can only set 1ch or 2ch for --mode.

<Versions earlier than NintendoSDK 13.3.1>
RunOnTarget DevMenuCommand.nsp -- audio set_output_mode --target tv --mode 6ch
[SUCCESS]

<NintendoSDK 13.3.1 and later versions>
RunOnTarget DevMenuCommand.nsp -- audio set-output-mode --target tv --mode 6ch
[SUCCESS]

Getting the Current Audio Output Target

Get the current audio output target. The following values can be obtained as the audio output target.

  • BuiltInSpeaker
  • BuiltInHeadphoneJack
  • Tv
  • Usb
  • Bluetooth
$ RunOnTarget DevMenuCommand.nsp -- audio get-current-output-target
BuiltInSpeaker
[SUCCESS]
