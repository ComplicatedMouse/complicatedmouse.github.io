Layout, Font News

This document is a graphical introduction to the features that were added in NintendoSDK 3.0.0.

LayoutEditor

This section introduces the new features in LayoutEditor.

Automatically Resending Layouts Layouts can now be automatically sent again to the viewer each time a command or other operation is executed.
Unified Layout Environment Settings Window The Layout Environment Settings window has been unified into a single location.
Switching the Order of Stage Formulas The order of stage formulas can now be switched in combiner mode.
Changed Coordinates Settings in the Properties Window The coordinates settings for the properties window have been changed.


Automatically Resending Layouts

Layouts can now be automatically resent to the viewer each time a command or other operation is executed.

Layouts are sent again each time a command is executed, which allows the user to directly and immediately check the results.
The automatic resend feature can also be manually enabled and disabled.
(The feature is automatically disabled when a transfer error occurs.)

Feature for Automatically Resending Layouts
A Switch Platform Combo Box Use this to switch the platform that is used.
B Switch Auto Resend Icon

The icon to switch automatic resending.

Click the icon to switch the feature.
Does not send the layout to the viewer during editing.
Sends the layout to the viewer during editing.
C Switch Connection Target Combo Box Specifies where to connect to from the combo box.
D Switch Transfer to Viewer Animation Playback Icon

Icon to enable and disable animation playback when transferring the layout to the viewer.

Click the icon to switch the feature.
Does not play the animation in the viewer (button behavior can be checked).
Plays the animation in the viewer.

Unified Layout Environment Settings Window

The Layout Environment Settings window has been unified into a single location.

The Layout Environment Settings window was previously split between two locations. It is now unified in a single location. Unifying the settings window makes the menu simpler and makes the environment easier to configure.

The Layout Environment Settings window can now be accessed from LayoutEditor Window > Windows > Layout Settings > Environment Settings.
The LayoutEditor Window > File > Layout Environment Settings route is no longer available.

Merged with LayoutEditor Window > Window > Layout Settings Window > Environment Settings.

The LayoutEditor Window > File > Layout Environment Settings route is no longer available

The LayoutEditor Window > File > Layout Environment Settings route is no longer available, and so is no longer displayed.

Switching the Order of Stage Formulas

The order of stage formulas can now be switched in Detailed Combiner Mode.

Added a button to switch the stage formula order in the LayoutEditor properties window and the Texture Combiner Detailed Combiner Mode.

The button for switching the stage formula order that was added to the detailed combiner mode.
A Button for Switching the Order of Stage Formulas

Button for switching the stage formula with the previous or next stage formula.

Switches the stage selected by Stage Selection in B (when it is displayed in the window) with the previous or next stage.
B Stage Selection

Selects the stage displayed in the window.

Changed Coordinates Settings in the Properties Window

The texture coordinate display has been changed to improve visualization and make it more intuitive.

The coordinate settings in the Base Pane of the properties window have been changed.
The texture coordinate settings for Picture Pane have been changed in the properties window for the picture pane.
