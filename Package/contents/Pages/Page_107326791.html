Layout, Font News

NintendoWare Layout, Font News

This document is an introduction to the graphical features that were added from NintendoSDK 0.6.0 to 0.9.0.

LayoutEditor

The LayoutEditor UI is now more user-friendly.

  • LayoutEditor
    • New Logos and Icons
    • Improved to allow other files to also be specified using Reconfigure the File Path for a texture.
    • Added a feature to restore the size of picture panes to that of the texture.
    • Improved to play animations at the designated frame rate
    • Feature That Changes Expanded User Information for Parts

New Logos and Icons

LayoutEditor has a new logo, and the icons in its toolbar are new as well.

A The new LayoutEditor logo. B The toolbar icons.

Improved to allow other files to also be specified using Reconfigure the File Path for a texture.

You can now replace textures already used in the LayoutEditor with textures with other names.

In the texture window, select the textures you want to change and select Reconfigure the File Path (). The texture selection window opens.
Even if the texture size changes, the picture pan size will not change.

The following image shows the texture window.

Usage Examples

You can use this feature to change the texture to one with a different name using Reconfigure the File Path () while working on a picture pane.

Before, only textures with the same name could be reconfigured.
There are no more name restrictions, making LayoutEditor easier to use.

Added a feature to restore the size of picture panes to that of the texture.

A new feature enables you to restore the size of picture panes that have been changed to the texture's original size.

You can do this from the main menu by selecting Pane > Resize to Texture Size.
The size changes in the direction of the designated center in each picture pane's property window.

This feature affects the size. Scale is not affected.

1. Picture Pane With the Size Changed (Before Resizing) 2. Picture Pane Resized to the Texture's Original Size
A Picture pane with size changed The picture pane has been changed to a landscape-orientation size. The texture is stretched.
B Resized picture pane Resize to Texture Size resizes the picture pane to the original texture size.
C Picture pane center (lower right) This picture pane's center is set to the lower right, so it has been resized to make the lower right the center.

Usage Examples

When editing the picture pane size, you can reset the size using Resize to Texture Size to restore a picture pane to the original texture size.

Before, you needed to enter the numerical value if you wanted to restore a picture pane to the original texture size after changing its size.
With one click of a button, you can reset a picture pane's size to the actual texture resolution.

Improved to play animations at the designated frame rate

You can now play animations at the designated frames per second (fps).

From the main menu, select Edit > Tool Environment Settings, and select the Frame Rate check box to play animation at 30 fps.
The current animation frames per second is displayed in the Curve Editor window.

The following image shows the Curve Editor window.

A FPS display location
 Displays the current frame rate (fps). The default value is 60 fps.

Usage Examples

The difference between how animations look at 30 fps and 60 fps can be checked by changing the frame rate.

Feature That Changes Expanded User Information for Parts

In the Parts pane, the editing feature for overwriting the expanded user information was improved as follows.

In the Parts pane, there is now an Overwrite setting to partially change Property settings inherited from the Parts layout.
Any future reference to "overwrite" refers to the editing ability for expanded user information that can be overwritten.

A Add button Adds user information to the list.
a is the newly changed or added user information in the layout.
B Delete button Disables the selected user information. The information is not deleted from the reference parts data.
b is user information that is not used in the layout.
C
Reset button Restores user information to the reference parts data state.
c is unchanged user information in the layout.
D Edit button

Opens the Expanded User Information dialog box so you can edit the selected user information.

d is the edited user information in the layout.
E Rearrange button for selected items Vertically rearranges the selected user information in the list.
Note

After editing expanded user information in the Parts layout, changes made to the Parts layout will appear in the shaded user information when selecting Reload Parts Pane (CTRL + R) > Pane.
It will not appear, however, in other user information.

Info

Expanded User Information is a feature that can set additional original user information. The data can be used for game application features and workflow purposes.

Usage Examples

In a game application's flow for example, you can use Expanded User Information to set originally created shader values.
Because partial changes can be made to user information for reference parts panes on the layout side that uses parts, this feature can be used to change only the shader color.
