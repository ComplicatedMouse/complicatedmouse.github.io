Minimum Traverse Distance Per Frame

This section describes the Minimum traverse distance per frame feature which treats an emitter as having moved, even when it barely moves or does not actually move at all, in order to ensure that particles get emitted.

If the emitter moves by less than this distance, it is treated as if it moved this minimum amount, even if it barely moved or did not move at all.

For example, if the movement distance is 0, the emission interval is 1, and the minimum traverse distance per frame is 0.1, the distance moved is treated as 0.1, and emissions occur 1 time per 10 frames.

This setting is used to display a particle even when the emitter has stopped.

Overview

The Minimum traverse distance per frame feature treats the total movement of an emitter in a single frame as being equal to the specified value.
If the emitter moves by less than this distance (such as when the emitter barely moves or does not actually move at all), the total movement distance is treated as being equal to this minimum movement distance.
Use this feature when you want to ensure that particles get displayed even when the emitter is not moving.
