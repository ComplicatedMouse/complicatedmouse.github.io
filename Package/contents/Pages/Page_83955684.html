Color

This tab makes settings related to the color and alpha of the effect.

The Particle tab () contains the following sub-tabs ().

Particle Configures settings related to the color and alpha of individual particles.
Global Use to configure color and alpha settings that effect the emitter overall.

The following settings are available in the Particle sub-tabs ().

Color Calculation

Color calculation () is a feature that sets a value by which to multiply all particle colors used by an emitter.
This group has the following settings.

Color Scaling

Applies a scale to color settings values.

Color/Alpha Settings

Color/Alpha Settings () is a group of features that set colors to be used for the Color Combiner calculation formulas.
This group has the following settings.

Color settings. Sets the behavior type to use for Color 0 and Color 1 in the Color Combiner calculation formula.
Alpha Settings Sets the behavior type to use for Alpha 0 and Alpha 1 in the Color Combiner calculation formula.

Alpha Processing

Alpha Processing () is a group of features that apply effects to the alpha values of particles.
This group has the following settings.

Soft Particles This feature applies translucency to particles that overlap with models.
Near Alpha This feature applies translucency to particles when they are in near distance of the camera.
Far Alpha This feature applies translucency to particles far away from the camera.
Fresnel Alpha This feature changes the alpha of the primitive normal vector relative to the view vector.
Decals This feature changes the alpha value of all areas except where the model and the particles overlap.
This feature creates effects as if pasting particles to models.

The Global sub-tab () has the following settings.

Global Multiplicative Color and Alpha

Global Multiplicative Color and Alpha () configures color and alpha settings that affect all particles emitted from the emitter.
This group has the following settings.

Global Multiplicative Color and Alpha
Multiplies the color of all particles emitted from the emitter.
