Profiler-Callstacks

Overview

Sets whether to record callstacks in a Sampled profile or not. This setting will modify the state of the Leaf Only and Callstack toggle buttons on the Sampled Profile tab.

Profiler-Callstacks [-ShowCallstacks] <Boolean>


[-ShowCallstacks] <Boolean>

Specifies if callstacks should be recorded.
