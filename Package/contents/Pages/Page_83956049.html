Animation Sets

Animation Sets

Overview

An animation set is a collection of multiple animations.
An animation set is bound to a model. When played using viewer library, all animations included in an animation set are played simultaneously.

Animation set information is saved in a project file.

Info

In this section and on 3DEditor menus, binding an animation to an animation set refers to the act of including an animation in that animation set.

Animation Set Names

Animation set names must consist of printable ASCII characters.

Info

Names consisting of strings of only tabs or spaces are prohibited.

  • Animation sets bound to the same model must each have different names.
  • Animation sets can have the same name as long as each is bound to different models.
  • Animation sets having the same name but bound to different models can be distinguished from each other.

Binding Animation Sets

A single animation set must be bound to a single model.

Note: The animation set itself may be empty.

Info

However, you can bind multiple animation sets to the same model.

When you close a model to which animation sets have been bound, all bound animations are removed.

Using Animation Sets

Animation sets are manipulated in the File View.

Operations performed in File View can be undone.
