Managing microSD Cards

Use the sdcard command to get the current microSD card status and to initialize microSD cards.

$ RunOnTarget.exe DevMenuCommand.nsp -- sdcard --help
usage: DevMenuCommand sdcard status
usage: DevMenuCommand sdcard cleanup
usage: DevMenuCommand sdcard format [--dry-run]
 [SUCCESS]

Displaying the Status

Use this command to display the current microSD card status and information necessary for the system to use the card.

$ RunOnTarget.exe DevMenuCommand.nsp -- sdcard status
SdCard is initialized on the different machine. Please cleanup sdcard and reboot the system.
[SUCCESS]

Initialization

Use this command to initialize a microSD card so that it can be used by the system. The captured image data will be retained.

$ RunOnTarget.exe DevMenuCommand.nsp -- sdcard cleanup
[SUCCESS]

After initialization, restart the development hardware.
After you have run the sdcard cleanup command, the status will not be updated, even if you run the sdcard status command, unless you restart the development hardware or remove and reinsert the microSD card.

Format

This command formats the microSD card. Note that all files on the microSD card will be deleted.
Restart the development hardware after formatting.

$ RunOnTarget.exe DevMenuCommand.nsp -- sdcard format
[SUCCESS]

Run with the --dry-run option to check the conditions required for formatting without actually performing the formatting. The files on the microSD card will not be deleted.

$ RunOnTarget.exe DevMenuCommand.nsp -- sdcard format --dry-run
[SUCCESS]
