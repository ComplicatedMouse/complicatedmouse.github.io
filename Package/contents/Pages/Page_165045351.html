A Fade Out

This section describes how to make particles fade away.
This method has the following advantages and disadvantages.

Advantages
  • Suited for effects that do not have depth and will disappear in a short time.
  • Can be used to extinguish effects while maintaining the size.
Disadvantages
  • Not suited for effects that are voluminous, like three-dimensional objects and explosions.
  • Translucent rendering is involved, so you must pay attention to the render priority.
  • If the display area is big, the rendering load can be substantial.

B Shrink Away

This section describes how to make particles shrink to nothing using scaling.
This method has the following advantages and disadvantages.

Advantages
  • Suited for effects that do not have depth and will disappear in a short time.
  • Can be used to extinguish effects while maintaining the size.
  • Can be used when the Z Buffer and Alpha Test setting is Transparency or Opaque and the effect cannot be made transparent.
Disadvantages
  • Not suited for effects that are voluminous, like three-dimensional objects and explosions.
  • Translucent rendering is involved, so you must pay attention to the render priority.
  • If the display area is big, the rendering load can be substantial.
