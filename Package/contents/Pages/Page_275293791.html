Preventing Texture Blurring When Scaling

Textures can blur when items such as the picture pane are scaled.

One workaround is to create texture mipmaps, and have your program switch them out for the texture in the picture pane.

The LOD level is determined by the UV density in the region where the texture is to be rendered. In some cases, the selected mipmap will be larger than the rendering region so it is shrunk, and in other cases, it will be smaller than the rendering region so it is enlarged. But in all cases, LOD bias is applied so a large mipmap is selected, and a crisp rendering result is obtained.

For this process, a negative value (for example, -0.5f) is set for the lodBias parameter of the Setup member function in GraphicsResource.

Note, however, that the use of large mipmaps increases the processing load.
