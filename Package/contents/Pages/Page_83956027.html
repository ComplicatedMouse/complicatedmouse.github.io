Plug-In for Setting Sampler Attributes

This plug-in sets attributes for any NintendoWare sampler attached to a file node. These attributes, however, are not applied on the Maya screen.

It is also possible to search for settings.

You can restrain the export of file nodes by setting attributes to a specific value.

How to Use

Select Nintendo > Set Sampler Attribute to open the window.

Selected file nodes are displayed in the Selected file Node column.

Info

If four or more file nodes are selected, node names are omitted, but the number of nodes selected is displayed on the right.

Setting Method

When one or more nodes are selected, Set displays the current settings for the selected nodes.
If you select multiple nodes or groups, the setting for the first selected node is displayed.

Attributes are changed by controlling each item in the window.
If multiple selections were made, selected nodes can be changed at the same time. (Only modified attributes will be applied.)

You can select a different node by selecting another node without closing the window.

Name Enter the name of the sampler.
Samplers are exported using the name of the attribute to which the file node is attached if left blank. (For more information, see below.) It is blank by default.
If sampler_merge_priority="true" is set in the config file, you can only set names that start with "_" (underscore). (Entering names that do not start with an underscore results in an error.)
Hint

Enter a hint for the sampler.
You can enter a tool tip in Hint if a sampler name is entered.
Samplers are exported using the hint information of the attribute to which the file node is attached if left blank. (For more information, see below.) It is blank by default.

You can restrain export of a file node by setting a string starting with nw_ignore. Exporting is the same as when the specified file node does not exist. However, the color gain of the file node or the gain attribute of the alpha value is applied in the intermediate file.
Info

The names and hints of samplers used by the same material must be unique. The same names or hints cannot be used for multiple samplers.

Info

For more information about merging samplers, see the Help documentation for the intermediate file optimizer.

Info

Set sampler attributes by selecting the file node attached to the front face when a file node has been attached to each face of the encumber node (environment cube).
Set sampler attributes by selecting the file node of the right face when using a cube map DDS file for the file node attached to the right face of an envCube.

Info

Textures attached to the Translucence Depth attribute of a standard material are not exported by default but are exported if a sampler name is set.
If Hint is left blank for a texture attached to a Translucence Depth attribute, the value entered for Name is also exported as the hint.
Samplers specified under <sampler_array> in the intermediate file for textures attached to a Translucence Depth attribute are exported last. (Such UV sets are also exported last among UV sets.)


Searching for Attributes

You can select all file nodes that include a sampler name string by entering that string under Name in the search box and clicking Search.
Entering an asterisk and performing a search selects all file nodes that do not have a blank sampler name.
Entering nothing and performing a search selects all file nodes with a blank sampler name.

Behavior is the same when searching for a hint.

Default Name and Hint Information

For standard materials, if the sampler name is left blank, samplers are exported with the default names in the following table according to the attribute that the file node is attached to.
When setting your own sampler names, make sure to not use names that could duplicate names created by the default naming pattern.
The same goes for sampler hint information.

Attached Attribute Default Name Default Hint Information
Color _a0, _a1, ... albedo0, albedo1, ...
Transparency _o0, _o1, ... opacity0, opacity1, ...
Incandescence _e0, _e1, ... emission0, emission1, ...
Specular color _s0, _s1, ... specular0, specular1, ...
Reflected Color _r0, _r1, ... reflection0, reflection1, ...
Bump Mapping _n0, _n1, ... normal0, normal1, ...

If the sampler name (hint information) of the file node attached to an extended material is blank, the sampler name (hint information) specified by the config file is exported.
