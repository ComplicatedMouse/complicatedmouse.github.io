Data Creation Plug-Ins

The data creation plug-ins are a group of plug-ins for assisting in creating data for NintendoWare.

The plug-ins are used to set attributes and search for nodes with the specified settings in Maya when using NintendoWare.
The data creation plug-ins are executed from the Nintendo menu.

Types of Data Creation Plug-Ins

The following table lists the data creation plug-ins.

Plug-In Name Description
Plug-In for Setting Material Attributes
(Set Material Attribute) 		Sets material attributes in a shading group.
Plug-In for Setting Sampler Attributes
(Set Sampler Attribute) 		Sets sampler attributes for file nodes.
Plug-In for Disabling Node Compression
(Set No Compress Node) 		Sets the compression disable flag for nodes.
Plug-In for Setting Animation Binarization
(Set Animation Binarization) 		Sets the animation binary export flags for nodes.
Plug-in for Setting Motion Mirroring
(Set Motion Mirroring) 		Sets the motion mirroring attributes for a node.
Plug-In for Setting Texture Pattern Animations
(Set Texture Pattern) 		Sets textures to be forcibly exported to the file node for texture pattern animation.
Plug-In for Setting Shape Animations
(Set Shape Animation) 		Sets the vertex attributes of blendShape nodes that are to be updated during shape animation.
Plug-In for Setting the Camera Angle of View
(Set Camera Fovy) 		Configures the camera's angle of view in the vertical direction (Fovy).
Plug-In for Setting Light Attributes
(Set Light Attribute) 		Sets the attributes of a light's shape node.
Plug-In for Setting User Data
(Set User Data) 		Sets the user data for the node or shading group.
Plug-in for Setting Animation User Data
(Set Animation User Data...) 		Sets the user data for each bone in skeletal animation.
Plug-In for Converting Custom Attributes
(Convert Old Attribute) 		Converts custom attributes of NW4R to custom attributes for the current NintendoWare.
