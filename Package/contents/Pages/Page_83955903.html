File Information

Use this tab to view file information and to configure the basic settings for emitter sets.

Basic Settings

File Information () is a group of features for viewing basic information about emitter set files and setting comments and label colors for those files.
You can also use this feature to set comments and label colors as information to accompany files.
This information can be viewed from the Effect Browser file list and edited.
This group has the following settings.

Filename Shows the name of the file.
File path Shows the path to where the file is stored.
Comments

A feature for editing comments.
Label Color

Sets the label color.

Example Configurations

Comment and Label Color can also be verified and edited in the Effect Browser.
