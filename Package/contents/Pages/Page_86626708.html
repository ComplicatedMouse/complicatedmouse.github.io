Inheritance Settings: Draw Path

This page describes the Draw Path inheritance setting for child emitters.

Overview

The Draw Path inheritance settings inherit the draw path information from a parent emitter in a higher node.
The following overview describes the settings ().

Draw Path

This check box () determines whether to consider the Draw Path settings of the parent emitter when drawing itself. When enabled, always render after the parent emitter.
Enable this setting to display the Draw Path group () of settings. The group has the following settings.

Render before the higher-level hierarchy

This setting is displayed when the Rendering path inheritance setting is enabled.
This check box () determines whether to render before the parent emitter.
When enabled, the parent emitter particle is rendered first in Viewer.
