Overview

Force calculate before start is a feature that can be used to perform calculations on an effect for a specified duration before it is called.
Use it, for example, when you are using a snow effect and want the sky to be filled with snow from the start.
The following overview describes the settings ().

Force Calculate Before Start

Sets a duration in terms of a number of frames over which to perform calculations ahead of time on an effect.

Example Configurations

The following images provide a comparison of what happens when the Force Calculation Before Start value is changed.

A 0 The effect is called normally.
B 300 300 frames of the effect are calculated before it is called.
This can be used to create a situation in which snow is already falling when the game starts, for example.
