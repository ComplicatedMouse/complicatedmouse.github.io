Adding Distortion Effects to Captured Images

By registering parts pane under the capture pane hierarchy, you can convert the image in the parts pane into a captured image.
This section describes the process for loading the captured images as textures into the picture pane, then adding a distortion effect.
Refer to the CaptureFirstFrameOnly Ui2dDemo.

Related Links First go through Using Capture Panes and get into a state where you can use capture panes.

Info

For this tutorial, reference CaptureFirstFrameOnly in Ui2dDemo.
The Ui2dDemo data is located in the following directory.
\Samples\Sources\Resources\Ui2d \CapturePane\data\CapturePane.flyt

Starting LayoutEditor

Start LayoutEditor and create layout data.

Related Links For more information about starting LayoutEditor, see Workflow.

Creating a New Layout

From the main menu, select File > New to create a new layout.

Related Links For more information about creating a new layout, see Creating a Layout.

Registering Parts

In the Parts Palette window, load the Parts layout in the Body layout, and place it as a Parts pane ().

In the Parts Palette window, double-click CapturePane_Parts_00.flyt.
In the Hierarchy window, L_parts_00 () is added.
The following are the placed parts panes.

Related Links For more information about parts panes, see here.

Creating Capture Panes　

On the Pane menu, select Create Pane, then Capture Pane () to create a capture pane.

On the main menu, select Pane > Create Pane > Capture Pane.
In the hierarchy window, select the created capture pane (), and change the pane name to C_CaptureFirstFrameOnly.

Specifying the Capture Area

In the property window for C_CaptureFirstFrameOnly (), specify the following settings.

Base Pane ()

 Size X,Y = 512,512

Capture Pane ()

 Enable Framebuffer Capture OFF
Texture Scale 50%
Clear Color R,G,B,A = 0,0,0,0

The following is the capture pane with the size changed.
Captures the parts pane in the area indicated by the yellow box.

Creating Pane Hierarchy

In the Hierarchy window, place L_parts_00 () under the created capture pane () in the hierarchy.

Use the capture area to change the pane under the capture pane () hierarchy into a texture.
Set L_parts_00 () as a child of C_CaptureFirstFrameOnly (), as follows.

Related Links For more information about setting up hierarchies, see Creating Parent/Child Hierarchies.

Moving the Capture Pane Outside of the Screen

If you place the capture pane () within the screen, it interferes with movement and verification work, so move it outside the screen.
In the property window for C_CaptureFirstFrameOnly (), specify the following settings.

Base Pane ()

 Position X,Y = 0,618

Loading Capture Textures

Specify the capture texture from the Texture window and load it into the Layout window.

Double-click C_CaptureFirstFrameOnly.tga () in Textures.
If you double-click the texture, a picture pane () is created with the texture registered.

In the Hierarchy window, select the created picture pane () and change the pane name to P_CaptureFirstFrameOnly.
In the Properties Window, specify the following settings.

Picture Pane ()

 Vertex Colors

[ LB ] [ B ] [ RB ]
R,G,B,A = 255,255,255,32

Load the distortion texture.

Add the texture for distorting the P_CaptureFirstFrameOnly () capture texture.

Load the texture (TGA) file that use the distortion into the Texture window from the following location.
Samples/Sources/Resources/Ui2d/Common/Textures/

Thumbnail Filename Format

Distortion.tga

BC5 : (I+A)

Registering the Distortion Texture

In the Hierarchy window, select P_CaptureFirstFrameOnly () and then click the Add button in Texture ().
From the dialog box, select Distortion.tga ( ) and the click OK to add 1 to the Texture () window.

Setting Distortion Effects

Sets the distortion for the Distortion.tga () texture that was registered to P_CaptureFirstFrameOnly ().

In the Properties Window, specify the following settings for Texture ().

Texture 1 Image Settings Repeat Method U-direction, V-direction = Repeat, Repeat
Combiner Settings Combiner Settings

Click the Combiner Settings () button, and switch the combiner mode.
Set the method of blending the registered textures 0 and 1.

Configuring Combiner Settings

Configure the combiner as follows.

Color Blending Settings Type

Indirect
[Indirect affects previous stage. Current stage is distortion source.]
Amount of warp U,V = 0.01,0.01

Setting Texture Matrix Animations

Use Curve Editor to set texture matrix animation for P_CaptureFirstFrameOnly ().

Setting Curve Editor Filters

Verify that Display All / No Filter () is selected in the attributes tree filter.

Set an animation.

In the Hierarchy window, select P_CaptureFirstFrameOnly ().
Create animation by setting keys for the various elements in the Curve Editor window, as shown below.

Pane Name Hierarchy Element Key Time = Key Value Slope Setting
P_CaptureFirstFrameOnly

Texture Matrix 1/Position/U 0f = 0 , 600f = -5 Linear interpolation.
Texture Matrix 1/Position/V 0f = 0 , 600f = -5 Linear interpolation.
