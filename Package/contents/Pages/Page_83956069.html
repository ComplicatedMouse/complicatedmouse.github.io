gfx Library Initialization and Finalization

Overview

Before using the features in the gfx library, you must initialize the library.
The gfx library must be finalized after the gfx library has been initialized or if the gfx library features are no longer required. Using other features of the gfx library after gfx library finalization without initializing the gfx library again causes undefined behavior.

The gfx library initialization and finalization features are different from the other gfx features, and are provided as functions rather than classes.
Note that these functions are not thread-safe.

Initialization

Initialize the gfx library using nn::gfx::Initialize.

Exiting the Tool

Finalize the gfx library using nn::gfx::Finalize.
