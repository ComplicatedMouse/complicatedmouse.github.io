Spy.exe

Spy.exe is a tool for visualizing data sent or read from applications in which the Spy library has been embedded.
By saving the received data as a file with the .spydat filename extension, you can load and display it later.

Main Window

The Spy.exe main window has a menu, a status bar, a global timeline, and several panels.

Main Menu

Tab Item Description
Home Connect

Click to enable or disable this option.

If enabled, the tool automatically connects to the applications that can be connected to on the platform specified in Selecting a Connection.

If disabled, disconnects from the currently connected applications.
Select a Connection Specifies the connection target platform.
Save

Saves the currently opened data.

If you specify a folder, a folder named with the date will be created under that folder and the .spydat file will be saved to the new folder.
Open

Opens a saved .spydat file.

You can open multiple files at the same time by dragging a folder containing multiple .spydat files.


You cannot simultaneously open .spydat files in a different folder, so close all of the open files and reopen them.

Files cannot be opened if the Connect option is selected. Clear the selection of the Connect option, and then open files.
Close All

Destroys the currently opened data.
Play / Stop

The Play button starts playback of waveform data.

The button will change to Stop during playback.


If there is no waveform data in the Audio waveform panel (that is, if the Target combo box is disabled), this button will not do anything.

Only L/R 2ch playback is supported.
Jump to Start Moves the current time to the start.
Jump to End Moves the current time to the end.
Automatic Scrolling

Enables or disables automatic scrolling.

If enabled, the tool scrolls so that the latest data is displayed when data is received from the application.
Data (check box)

Selects the panels that can be used for the platform specified using Select connection point.

Spy.exe receives the required data from the application in the selected panels.
Options

Displays the Settings window.
Audio Save the waveform The waveform data that was recorded to the audio waveform panel is exported as a WAV file.

Global Timeline


 Description
A

The value furthest to the left is the start time, and the value to the right is the end time.

The vertical orange line indicates the current time.
B

Displays the current time.

You can change the current time by directly entering a value in this location.
C

Click to switch the unit of time between Time and AppFrame.

  • Time shows the time in hh:mm:ss:ms format.
  • AppFrame shows the time as a number of application frames.

Panels

Timeline Panel This panel displays the data on a timeline.
Meter Panel Shows the time information in detail as a meter.
Sound Panel for Sounds Playing in atk Library This panel displays information about the sounds currently playing in the atk library.
atk Sound Play-Log Panel This panel displays a log listing information about the sounds that have played in the atk library.
Atk Performance Panel The atk performance panel displays atk performance information.
The atk Sequence Variables Panel

This panel displays the atk sequence variables.

This feature is under evaluation. It may be changed or removed without notice.
Atk Stream Sound Panel This panel displays information about the stream sounds currently playing in the atk library.
Audio Waveform Panel This panel displays the waveforms output by the audio library.
Audio Loudness Meter Panel The audio loudness meter panel displays the loudness calculated for waveforms output from the audio library.
Spy Debugging Panel

This panel displays information about the working state of the Spy library.
