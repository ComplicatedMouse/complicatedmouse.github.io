Added vertical alignment functionality to the Align pane.

Added vertical alignment functionality to the Align pane to adjust the positions and sizes of panes at the lowest level in the Hierarchy window.

The align pane () is a pane for aligning and adjusting the positions and sizes of subordinate panes in the Hierarchy window with their arrangement relative to one another being maintained.
The processing of this feature takes place when the game application is running, so it can be used to dynamically change the sizes of panes in the program.
For example, when the size of the Picture pane has changed as follows, the relative positions of the icons situated along the pane are maintained.

A The Align Pane feature is not used.

B The Align Pane feature is used. The relative positions of the icons situated along the side of the diamond icon are maintained.

Displaying TGA Files in a Folder

Added functionality to display the TGA files in a folder in the Texture window.
You can switch the path where textures are placed, and add textures to layouts as follows.

The folder to display in the Texture window can be specified in the project configuration file.
The TextureDirectories element of the project configuration file is described as follows.

Label The name displayed in the Texture window.
Base Specifies the location of the path origin from any of the following.
  • Project (location of the project configuration file)
  • Layout (location of the layout files)
  • Absolute (location specified as an absolute path)
AllDirectories Specifies whether the textures under child directories are also displayed.
ShowDirectories Specifies whether to display child directories in the hierarchy.
The settings for ShowDirectories is ignored if AllDirectories is false.

Fonts

Added functionality to forcibly render fcpx at the same width.

Added functionality to forcibly render fcpx at the same width.

Use this feature when you want numeric glyphs to be the same width.

Set the forceMonospacedEnabled attribute to true and specify a character width in the forceMonospacedWidth attribute to force characters to be rendered at that width. For more information, see Composite Font Definition File Format.
