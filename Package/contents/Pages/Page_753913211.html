Profiler-ShowTab

Overview

Brings a tab into view.

Profiler-ShowTab [-Tab] <ScriptingTabs> [-ShowArgs <Boolean>]


[-Tab] <ScriptingTabs>

Specify one of the tabs in the UI to show.

    • Analysis: Analysis Tab
    • Assembly: Assembly Tab
    • CallTree: Call Tree Tab
    • Checkers: Checkers Tab
    • CodeCoverage: Code Coverage Tab
    • Counters: Counters Tab
    • Files: Files Tab
    • Functions: Functions Tab
    • GraphicsGroups: Gfx Groups Tab
    • Heartbeats: Heartbeats Tab
    • Home: Home Tab
    • Info: Info Window
    • Instrumented: Instrumented Tab
    • InstrumentedProfile: Instrumented Profile Tab
    • Last: Last Tab
    • Logs: Logs Tab
    • Overview: Overview Tab
    • SampledProfile: Sampled Profile Tab
    • Shaders: Shaders Tab
    • Start: Start Tab
    • Threads: Threads Tab
    • Timeline: Timeline Tab
    • Warnings: The Warnings window accessible from the Overview tab.
Note

Only tabs that are enabled can be shown. For example, specifying CallTree will only show the Call Tree tab when a Callstack profile is loaded.


[-ShowArgs <Boolean>]

Specifies if the function arguments should be shown or hidden. This works on the following tabs:

    • Functions
