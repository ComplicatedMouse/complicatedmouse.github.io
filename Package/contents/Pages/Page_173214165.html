An indirect texture is used to depict letters as ice and flames.
Both use texture tr_noise_00 as the indirect texture, but different scaling and animation produce the different effects.

Flaming Letter

  • An indirect texture is added to just the outline of the letter A, and then SRT animation is applied.
  • The picture pane below N_Flare_00, N_Edge:
    • The fragment operation is Additive.
    • Texture Combiners > Blending Settings > Type is set to Indirect [Affects prev stage. Current stage is distortion source.].
      ----> Distortion is adjusted by the indirectly adjusted scale value.
    • Moved upward and rotated by SRT animation of the indirect texture.

Ice Letter

  • An indirect texture is added to just the outline of the letter A, and then SRT animation is applied.
  • The Picture pane below N_Light_00:
    • The fragment operation is Additive.
    • SRT animation of texture, masked by the stage 2 texture.
  • The Picture pane below N_Ice_00:
    • Same configuration as flame.
      (SRT animation and indirect texture have different scaling.)
