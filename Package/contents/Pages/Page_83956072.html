Handling Errors

If an error occurs during output, it is displayed in the output window and in the script editor.
Descriptions are provided for errors that need them.

  • The combination of texture and UV set differs among objects.
    For more information, see the description in Multi UV.

Handling Warnings

If a warning is generated during output, it is displayed in the output window and in the script editor.
Descriptions are provided for warnings that need them.

  • The scale of the parent node in the joint node is not bound to the inverseScale attribute.
    Segment scale compensation is not being applied in Maya, so bind this.
    Executing the following MEL connects the scale of the parent node to the inverseScale attribute of all joint nodes in the scene.

    {
    for ($j in `ls -typ joint`)
    {
        string $p[] = `listRelatives -p -pa $j`;
        if (size($p) != 0 && nodeType($p[0]) == "joint")
        {
            string $ins[] = `listConnections -p 1 -s 1 -d 0 ($j + ".inverseScale")`;
            if (size($ins) != 0)
            {
                if ($ins[0] == $p[0] + ".scale")
                {
                    continue;
                }
                print ("disconnect: " + $j + " <-- " + $ins[0] + "\n");
                disconnectAttr ($ins[0]) ($j + ".inverseScale");
            }
            print ("connect: " + $j + " <-- " + $p[0] + "\n");
            connectAttr -f ($p[0] + ".scale") ($j + ".inverseScale");
        }
    }
}
