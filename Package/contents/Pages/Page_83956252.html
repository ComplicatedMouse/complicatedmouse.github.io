Display

Display

Performs display-related settings.

A Displays the Object View submenu.
B Sets the order in which items are shown when items are listed in the Object View, and shows the column item setting submenu that sets hide settings.
C Opens the properties window for the selected object.
D Toggles the Adjust Size on Page Change option.
E Displays or hides the toolbar.
F Displays or hides the status bar.
G Displays or hides the log pane.

Object View Submenu

Performs operations that relate to the Object View display.

A Sets the Object View to a single-screen layout.
B Sets the Object View to a two-screen layout, with upper and lower sections.
C Sets the Object View to a two-screen layout, with left and right sections.
D Sets the Object View to a triple-screen layout, with an undivided upper section, and a lower section divided into left and right sections.
E Sets the Object View to a triple-screen layout, with an undivided lower section, and an upper section divided into left and right sections.
F Sets the Object View to a triple-screen layout, with an undivided left section, and a right section divided into upper and lower sections.
G Sets the Object View to a triple-screen layout, with an undivided right section, and a left section divided into upper and lower sections.
H Sets the Object View to a quad-screen layout.
I Sets the Object View split size to the default status.
Valid only for layouts with two or more screens.

Column Item Setting Submenu

Saves or loads column item settings.

A Shows the Column Item Setting dialog box.
B Saves the column item settings as a preset file (.3deuipreset).
C Loads the saved the column item settings preset file (.3deuipreset).

Column Item Settings Dialog Box

Sets the order in which items are shown when items are listed in the Object View, and whether to show or hide items.

A The setting for displaying list of models.
B Material list display settings.
In the material list settings, you can configure which columns to display for each category.
C The setting for displaying a list of shapes.
D The setting for displaying a list of textures.
E The setting for displaying a list of bones.
F Column items are displayed in the current order, from top to bottom.
Clear the check box next to an item to hide the item so it does not display.
G Switches the selected item with the item displaying at the top of the display order.
H Switches the selected item with the item displaying at the bottom of the display order.
I Restores the order and display of column items to the default settings.
J Selects all items.
K Clears all items.
L Saves the column item settings file.
M Loads the column item settings file.
N Executes the change of settings to the column items.
O Cancels the change of settings to the column items.
The setting for the column is restored to the previous state.

Material List Column Settings

The materials list contains a large amount of information. You can choose which columns to display for each category so that only the information you want to see is displayed.

A In this area you can select the categories of information you want to display in the materials list.
B This area displays the fields within the selected category.
Select the fields you want to be displayed in the materials list.
