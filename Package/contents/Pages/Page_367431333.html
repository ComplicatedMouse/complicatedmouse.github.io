Height Offset

This page describes the Height Offset feature, which is used to change the character origin position when scaling or rotating characters for per-character transformation.

Overview

The Height Offset feature offsets the origin coordinates from the Origin (V Direction) position when scaling and rotating characters for per-character transformation.
You can set the position for Origin (V Direction) as 0, and only offset the origin coordinates by the value of the Height Offset.
