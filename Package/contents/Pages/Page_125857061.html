New Tag Window

The New Tag window is used to add animation segment tags.

New Tag Window

A Name Specifies the name for the animation segment tag.
B Segment

Specifies the start frame and end frame of the segment.

Nothing is displayed in separated mode.
C Length

Specifies the length of the segment.
D Comments

Specifies comments about the animation segment tag.
E Related Groups

Sets the related groups.

  • Create New (Creates a group with the same name as the tag.)
  • Select From Existing Group
F Playback mode

Sets the playback mode.

  • One Time
  • Loop
  • Wait (Frame length is set to 0.)
