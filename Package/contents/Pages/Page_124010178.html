Terminology Definitions

Sound

The NintendoWare audio development environment (Audio Toolkit) contains the elements needed to play sounds.

Sound System

The system available internally for the library to play sounds.

Sound Project

A SoundMaker project file.
See Sound Projects.

Sound Set

A group of sounds added to a sound project.
See Sound Sets.

Sound sets include elements such as stream sounds and wave sounds. For more information about these elements, see Elements of a Sound Set.
