Fresnel Alpha, Process as Vertex

This section describes the Process as Vertex feature that handles Fresnel Alpha processing on a particle vertex basis.

Overview

The Fresnel alpha feature Process as Vertex processes the Fresnel Alpha at the level of the primitive polygon's vertices.
The alpha gradation will appear cleanly, even for primitives with many vertices.
The following overview describes the settings ().

To display Process as Vertex (), in Fresnel Alpha, select Process as Vertex. The group has the following settings.

Angle of minimum alpha

Sets the value (0.0) where the alpha value is at a minimum, specified in terms of the dot product of the view vector and normal vector.

Angle of maximum alpha

Sets the value (1.0) where the alpha value is at a maximum, specified in terms of the dot product of the view vector and normal vector.

Example Configurations

The following figure shows how to view the effect when using Process as Vertex, and the output values for the corresponding input values.
The horizontal axis represents the angle (the input value) between the view vector and polygon normal vector.
The vertical axis represents the alpha value (the output value) for the polygon vertex.

Viewing the Effect Figures

Set the following two values as input values.

A Angle of minimum alpha 0.2

When set to 0.2, the minimum alpha value is set to 0.0.
B Angle of maximum alpha 0.8 When set to 0.8, the maximum alpha value is set to 1.0.
