I Want to Combine Multiple Scalable Fonts and Handle Them as One Font

Question

Is there a way to combine multiple scalable fonts and handle them as one font?

Answer

You can do this by using multiple fonts (FCPX files) and specifying multiple scalable fonts in the <multiScalableFont> tag.

To use multiple fonts separately according to their character code, use the <charCodeRange> tag to specify which fonts use which character code range.

Alternatively, configure the attributes in the <scalableFontDescription> tag to apply the width and height scale for each font and adjust the vertical baseline offset.

For more information, see the pages for Composite Fonts and Composite Font Definition File Format.
