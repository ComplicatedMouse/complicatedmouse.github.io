Display the Frame in the Layout Initial State With the Camera Button

The number of frames in the initial state of the animation is displayed in the Playback Confirmation Panel of the Animation Segment Tags window.
As seen in the following movie, it displays the frame count for the initial animation state that was recorded using the camera button () while the animation type is Separated Mode.

For example, the fade-in animation In often shows nothing on initial appearance at the 0th frame.
Layout state is not understood until the animation is played back and has fully appeared.
In such cases, press the camera button (), and record the point at which the layout is displayed as the initial state.
You can return to the recorded state at any point by pressing the reset initial animation state button ().
This improvement allows you to determine at what point (frame count) the initial state of the animation was recorded.
