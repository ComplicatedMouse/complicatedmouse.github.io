Manipulating Each Cell in the Window Pane

Question

How do I set a different UV animation for the cell on each side in a window pane () split horizontally into three parts, as shown below?

Layout Window LayoutViewer

A window pane divided horizontally into three parts.

Share Top Left Material is enabled by default.
With this setting, cells A and B use common UV coordinates.

Answer

In the default settings for window panes (), the texture coordinates for the upper-left cell () are shared.
To set the texture coordinates for cells A and B separately, clear the Share Top Left Material check box in the property settings.

LayoutViewer

Clear the Share Top Left Material check box.
This option enables UV to be animated separately for each cell.
Cell A is uv-scaled with a fade-in animation.
Cell B is animated in the v-direction after it appears.
