Example 6: Take a Code Block Only Profile
# Take a code block only profile
#   1. Turn on recording marked code.
#   2. Take a profile.
# Note: This example assumes there is already a dev kit synced.


# 1. Turn on recording marked code.
Profiler-Marked $true

# 2. Take a profile.
Profiler-Start Instrumented
Profiler-Wait 0.5
Profiler-Stop
