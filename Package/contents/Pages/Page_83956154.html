Barrier Class

Feature Overview

The Barrier class is used to synchronize multiple threads.
When the predetermined number of threads reaches a particular stage, all of the waiting threads are released and resume execution.

The Barrier class wraps the Barrier Features using the nn::os::BarrierType object. The Barrier class contains the following member functions. For more information, see the API reference.

Name Description Comments
Barrier Constructor Initialization equivalent to InitializeBarrier.
~Barrier Destructor Destruction equivalent to FinalizeBarrier().
Await Waits for other threads Lock equivalent to AwaitBarrier.

You may specify the number of threads to wait for in the constructor. For more information about each feature, see the Barrier Feature.
