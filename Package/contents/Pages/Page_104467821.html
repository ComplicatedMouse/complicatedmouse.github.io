Scale

This section describes the Scale feature for applying an offset to the scale of the model.

Overview

Scale sets the offset to the scale set for the model.
Because this setting affects the size and speed of both the model and the particles, you can use it to modify the overall size of the effect.
This section provides an overview of the settings ().

Scale

Use the dial () or the text box () to specify the offset for each axis of the model's scale.
