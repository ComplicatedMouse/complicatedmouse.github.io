Add-On Sound Archive Feature

An add-on sound archive is a sound archive that sound items can be added to by combining the add-on sound archive
with a standard sound archive (listed as the main sound archive).

It allows you to add and delete the necessary sound items while continuing to use the main sound archive.
Add-on sound archives are suitable for use with sounds that are needed only in specific situations, and sounds that are included in downloadable content.

The following section describes the specifications for add-on sound archives and how to use them.

Configuring Add-On Sound Archives

  1. Add a new sound set that you want to make an add-on sound archive.
  2. Right-click the sound set, select Sound Archive Output Settings > Add-On Sound Archive,
    and set the sound set to be output in the add-on sound archive during output.
    (The output settings can also be changed using the Sound Archive Settings 2 tab in the project settings dialog box.)
  3. The sound set that in the add-on sound archive is displayed as follows.

    Select the sound archive from step 2 to return the output in the sound archive to its original state.
  4. Afterward, add and convert sounds to output the sounds in the add-on sound archive.
    (Items that can be added are subject to the restrictions described in Add-On Sound Archive Specifications.)
Info

You must output the string table if you are using an add-on sound archive.
For more information about the setting for outputting the string table, see the Sound Archive Settings section in Option Settings.
