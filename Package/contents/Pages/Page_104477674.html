XZ Translation Radius

This section describes the XZ translation radius feature, which sets the radius of the circular route of movement for the selected node.

Overview

This section describes the XZ translation radius which sets the radius of the selected node's circular movement route.
The radius is increased by increasing the value.
The following is an overview of the setting.

XZ translation radius

Set the radius of the circular route using the dial.
