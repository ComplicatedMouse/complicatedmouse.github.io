New Effect Expressions

This section introduces features that broaden the scope of what can be depicted with effects.

Addition of Combiner Channel Settings

Presets have been added as the channel for the color combiner and the alpha combiner.
The channel value determines what component of the registered texture to apply as the input value for the combiner formula.
This enables you to expand the use of textures and is convenient when you want to reuse the same texture over multiple emitters.

The following sections describe each channel preset for the color combiner and the alpha combiner.

Color Combiner

These are the channel presets for the color combiner.
The following table shows the icons and names for the channel presets and provides a description of each.

RGB The color value of the registered texture is applied as the input value.
1.0 - RGB The texture color value subtracted from constant 1.0 is applied as the input value.
A Value of the registered texture's alpha channel.
1.0 - A The texture alpha value subtracted from constant 1.0 is applied as the input value.
1.0 Constant 1.0 is applied as the input value.

Alpha Combiner

These are the channel presets for the alpha combiner.
The following table shows the icons and names for the channel presets and provides a description of each.

A The alpha value of the registered texture is applied as the input value.
1.0 - A The texture alpha value subtracted from constant 1.0 is applied as the input value.
R Value of the registered texture's R channel.
1.0 - R The texture R channel value subtracted from constant 1.0 is applied as the input value.
1.0 Constant 1.0 is applied as the input value.

Related Links For more information, see the following pages.

Color Combiner Channel The value to input for the combiner formula is determined by running the color component of the texture through a predetermined equation.
Alpha Combiner Channel The value to input for the combiner formula is determined by running the alpha component of the texture through a predetermined equation.

Get Square of RGB

Added the Get Square of RGB feature, which squares the brightness of the registered texture to approximate the result of linear conversion.

Apply Get Square of RGB

 Linear conversion is performed using Get Square of RGB when the texture is fetched.
You can see that when this feature is selected, the result resembles the linear texture A.

When the registered texture is a grayscale image with alpha, using this feature has the following benefits.

You can expect a smaller texture size.

If the same image is used for color and alpha, color will be held in linear and alpha will be held in nonlinear, so the texture requires two channels worth of information.
This feature converts the color to linear color in the effect shader, so just one channel of information is enough for color and alpha.

EffectMaker

This section introduces UI elements newly added to EffectMaker.

Linking the Documentation to EffectMaker

The links between the documentation and EffectMaker have been enhanced.

Click a setting () to open the documentation in your default browser and display the Help for that setting.
You can change which document file to open from EffectMaker by changing the document file path in the User Settings option or the Project Settings option.

Display of Actual Frame Count on Particle Time Animation Keys

The actual time of the key frame is now shown as a frame count () in the Curve Editor and color animation UIs for a particle time animation.
This is handy when several emitters are emitting particles of different lifespans and you want the timing to match the animation.
