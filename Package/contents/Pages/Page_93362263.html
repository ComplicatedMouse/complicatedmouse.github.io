Draw Path

This section describes the feature for configuring the draw path specified in the game application.

Overview

The Draw Path feature allows you to select the draw timing for the game application coded in the file, as long as a project settings file has been specified.
By setting a draw path for each emitter, you can sandwich the emitted particles within the draw path in the game.
The following overview describes the settings ().

Draw Path

Select a preset coded in the project settings with the particle render timing registered.
