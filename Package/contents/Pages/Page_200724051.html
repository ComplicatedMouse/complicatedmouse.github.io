Using Data

Loading Data

This section describes the step for loading data stored in the data delivery cache storage into the application.

  1. Call the nn::bcat::Initialize() function and initialize the socket library.
  2. Call the nn::bcat::MountDeliveryCacheStorage() function to declare to the system features the start of using the data delivery storage cache.
  3. Call the nn::bcat::EnumerateDeliveryCacheDirectory() and enumerate the directory.
    • You can omit this step and directly specify the directory names in a later procedure.
  4. Call the nn::bcat::DeliveryCacheDirectory::Read() function for each directory and enumerate the files.
    • You can omit this step and directly specify the filenames in a later procedure.
  5. Call the nn::bcat::DeliveryCacheFile::Open() and nn::bcat::DeliveryCacheFile::Read() functions to load the files.
  6. Close the file using the nn::bcat::DeliveryCacheFile::Close() function, and close the directory using the nn::bcat::DeliveryCacheDirectory::Close() function.
  7. Call the nn::bcat::UnmountDeliveryCacheStorage() function to declare the end of the use of data distribution cache storage to the system features.


Also see the BcatSimple sample demo and BcatPractical sample demo.

Writing Data

The application cannot directly write data to the data delivery cache storage.
