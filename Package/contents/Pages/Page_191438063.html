Linking With Nintendo Accounts

Advance Preparations

Getting Nintendo Accounts for the Development Environment

In the development environment, you must get a Nintendo Account created for the development environment.
For more information about creating Nintendo Accounts for the development environment, see the Creating and Managing Nintendo Accounts in the Development Environment page.

Linking With Nintendo Accounts

Use any of the following procedures to link a user account with a Nintendo Account.

  • Using DevMenu.
    1. Open the Account detail screen for the user account in the Accounts menu.
    2. Click Ensure NSA with UI.
      • Performing this displays the system's user interface, where you can link the user account to a Nintendo Account.
  • Using the HOME Menu.
    • Follow the prompts in the User Details Screen, or the system settings, to link a Nintendo Account.

Nintendo Account Linkage State

Updating the Nintendo Account State

The Nintendo Account state is updated immediately on the server, but there is a time lag before the Nintendo SDK becomes aware of the linkage.
This time lag in the Nintendo SDK is described as follows.

The update to the Nintendo Account state will be reflected in the Nintendo SDK within around 24 hours, if a network connection is maintained during that time.

Manually Updating the Nintendo Account State

In the development environment, you can use the following manual procedure to immediately apply the updated Nintendo Account state.

  • Open the Account detail screen for the user account in the Accounts menu of DevMenu.
  • Click Update NA info cache.
    • The system communicates with the Nintendo Account Service and gets the updated state, which is applied in the Nintendo SDK.

Corrupted Nintendo Account Linkage State

Depending on the states of the Nintendo Account and the system, the link between a user account and a Nintendo Account can become corrupted.
A corrupted linkage state can have the following effects.

  • The Nintendo Account becomes unusable.
  • Network-service accounts become unusable.

If this situation accidentally arises, you can use the following procedure to restore the link with the Nintendo Account.

  • Follow the procedure, described in the section for restoring the availability of network service accounts, in Using Network Service Accounts.
