EffectMaker

This section introduces new features that have recently been added to EffectViewer.

Added Items to Option Settings

Added Show warning symbol for emitters with texture settings that differ from the Linear mode settings to the Options.
Added a setting to enable or disable the display of the warming symbol for the emitter node, if the registered textures have different settings from the Linear mode settings.
For example, you can hide the warning symbol by disabling the setting when intentionally using non-linear texture files using the Get Square of RGB feature in a linear environment.

Updated the Description of Child Emitters

The following information was added to the Child Emitter page.

Child Particle Birth Position

Describes the birth position of child particles.

A ● Parent particle Generated from a circle-shaped emitter.
The particle follow type is Perfectly follow emitter SRT, so the particle tracks the movement of the emitter.
B ● Child particles The particle follow type is Do not follow emitter, so the particles do not track the movements of their emitters.
C Child emitter
offset width

The position of the circle-shaped child emitter is set to Y = 10, and Emitter Assignment for Particle is cleared.
The child emitter is positioned 10 units away from the parent emitter along the y-axis.
The child particles (●) are generated 10 units away along the y-axis from the parent particle (●).

Child Emitter Settings and Differences in Behavior

The Child Emitter page now includes a table that presents the combinations of settings for which the child emitter behavior is the same when the parent emitter or particles emitted from the parent emitter are moved.

In the following examples, the setting for Emitter Assignment for Particle is either selected or cleared, and the behavior of child particles (█) is either the same as the parent particle (█) or not the same.
For matching behavior, we recommend clearing the setting for Emitter Assignment for Particle because this involves a lower processing load.

Matching Behavior Different Behavior

Moving object: Particles

Parent particle follow type: Do not follow emitter.

Child particle follow type: Do not follow emitter.

Moving object: Particles

Parent particle follow type: Do not follow emitter.

Child particle follow type: Perfectly follow emitter SRT
