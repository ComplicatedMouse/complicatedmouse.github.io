Checking Feature Animations

Describes a sample procedure for confirming the feature animation for a SELECT Button in a parts layout.
Because feature animation allows actual menu operations in the LayoutViewer, the animation can be revised while it is being checked.

Preparation

To prepare for creating the layout data, start LayoutEditor in advance.

Related Links For more information about starting LayoutEditor, see Workflow.

Opening Windows

Begin the task after opening the following windows.

  • Parts Palette
  • Parts and Control Settings
  • Hierarchy
  • Property
  • Curve Editor

Related Links For more information about the UI of each window, see Windows.

Opening the Body Layout

Open the Body Layout from the Parts Palette.

In the Parts Palette list, right-click Menu.flyt, and from the shortcut menu, select Open.
Menu.flyt is loaded in the LayoutEditor.

Opening the Parts Layout

Open the Parts layout from the Body layout.
In the toolbar at the bottom of the Layout window, click the Show Related Parts List button ().
Select Menu_Bar () in the displayed Parts Layout to open the file.

Using Feature Animation

Use feature animation to check the results of animation while performing actual menu operations in the LayoutViewer.
To run feature animation, select Tools > Transfer to Viewer from the main menu, or press SHIFT+F5.

Adding Animation

Add animation to brush up the Down segment tag.
In the Edit Animation Segment Tags window, select the Down animation segment tag.

Confirm that merged mode is selected in the toolbar of the Edit Animation Segment Tags window.
If the mode is separated mode, open the Tools menu in the Edit Animation Segment Tags window and select Switch to merged mode.
