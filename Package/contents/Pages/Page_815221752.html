Editing Animation Live (Editing Attached Model)

Editing Attached Model

This sample shows the method of editing an attached model where the model in the application is associated with nodes in Maya to preview the animation.
This description uses the G3dDemo sample (%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo), which must be built in advance.

Import File

Open a new scene in Maya. Use the Import Plug-In to import the following two intermediate files.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\human\human.fmdb
%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\human\human_run.fskb

G3dDemo

Open the Edit scene for the G3dDemo sample. (The target platform for G3dDemo can be anything, but the build type must be either Debug or Develop.)

Executing MEL

Run the following MEL script in the script editor and start the test tool.
(From the Maya menu, select Window > General Editors > Script Editor and execute the following command.)

NintendoCallPython("NintendoLiveEditingTestTool.NintendoLiveEditingTestToolForG3dDemo()", false);

Click Connect to enable Send model.

Change Destination Type to Normal PC application and then click Connect to connect Maya and the application.

(Change the connection type to match the target platform of the running sample application, and then connect.)

Connecting

In the Edit scene of G3dDemo, select human and click Attach.

On Maya, the Select FMDB File to Attach (human) dialog box appears. Select the human.fmdb file that was first imported to Maya.

When the Select Namespace dialog box appears, leave the selection blank (default) and click OK.

This step completes the process of associating the human model in the Edit scene with the human imported to Maya.

Confirm that the orientation of human in the Edit scene matches the orientation of the human in Maya.

Confirming

While in this state, confirm the following.

When animation is played on Maya, the human in the Edit scene is animated in the same manner.

Changing the joint node rotation for the human in Maya results in the same orientation being applied to the human in the Edit scene.
