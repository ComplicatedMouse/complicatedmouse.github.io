NintendoWare Spy News

This page introduces the changes in NintendoWare Spy library for NintendoSDK 1.0.0.

Improved the Audio Loudness Meter Panel

Improved the functionality of the Audio Loudness Meter panel.

  • It can now display true peaks.
  • Meter display now corresponds to peak holds.

For more information, see Spy/Spy.exe/Audio Loudness Meter Panel.

(Testing) Added the Spy Debug Panel

Note

This functionality is currently in testing. Note that this functionality could change or be removed without warning.

Added the Spy debug panel.

It displays information about the working state of the Spy library. Currently, it can monitor the use rate for the data send buffer.

For more information, see Spy Debugging Panel.

The data that appears in this panel us sent using the nn::spy::DebugModule class.

Changed the Time Axis

Changed the handling of the time axis for graph displays where time appears as the horizontal axis in panels like Timeline and Audio Waveform.

Grid lines now appear for the time recorded by the application frame when the time unit is AppFrame.

  • Previous reductions were made so that the application frame had equal intervals, but that functionality has been changed.
