Curl Noise

This section describes the Curl Noise feature, which places turbulence in space to control the movement of particles.

Overview

Curl Noise controls the movement of particles by moving a noise table that describes elements of turbulence placed in three-dimensional space.

The following image depicts the noise table. The arrows indicate the direction of the force applied to particle speed.
Placing forces that affect particles like this in three-dimensional space enables the application of changes to particle movement after emission.

Info

The force-directional arrows in the noise table have been fixed.
You can use curl noise to make the path of particle movement different every time by moving the table or applying a scale.

Curl Noise

The Curl Noise group () is a set of features for making particles fluctuate when they move.

Parameter weight (±)

Sets how strongly particles are affected by the noise.

Noise Table Interpolation

Determines whether to use interpolation to reduce the jerky particle motion that can occur with a low-density noise table.

Apply in world coordinates

Determines whether the table will be affected by a leaning emitter.
Noise Scale (±) Adjusts the scale of the noise table.
Noise Movement Speed (±) Sets the speed at which the noise table moves.
Noise Offset (±) Use this feature to adjust the movement of particles by shifting the position of the noise table.
Random Noise Offset Determines whether to randomize the width of the Noise Offset.

The Run on CPU setting for emitters changes how curl noise gets applied.
