Auto-translate type

This page describes the auto-translation type for selecting the automatic translation type in automatic translation of the emitter set.

Overview

The auto-translation type feature is for selecting the type of automatic translation.

The following overview describes the settings ().

Auto-translate type

Select the auto-translate type type from the following presets.

Translate XZ plane The emitter set will be automatically rotated to the xz-plane of the Viewer.
Move forward in the specified direction The emitter set will be automatically moved forward at the specified angle.

The following image compares the auto-translation type presets.

Translate XZ plane Move forward in the specified direction

The emitters will be automatically moved along the circumference of a circle of the specified size. The emitters will be moved forward in the specified direction.
