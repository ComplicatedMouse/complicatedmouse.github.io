Project Property Window

Used to check path information, or the creation log, about the selected project.

A Project
Currently, nothing is displayed when this page is selected.
B Displays the File Information page.
This page displays path information, creation information, and change information for the project.

File Information Page (Project Property Window)

This page is used to check information about the selected file.

A The project path.
B The creation history.
Displays information about the creation of project files, including the author and creation date of the project.
C The edit history.
Displays information about the changes to project files, including the author and date of changes to the project.
