In the Texture Exporter plug-in, you can now export PNG files.

A feature has been added to the Photoshop texture export plug-in for exporting 8-bit/channel PSD files as PNG files.
Specify the following setting in the config file to export as PNG instead of TGA.

3D Command-Line Tools

Fixed a bug in 3D Shader Converter where an error occurred when converting a shader that included vertex attributes without the layer qualifier.

Fixed a bug where an error occurred when converting a shader that included vertex attributes without the layer qualifier in 3dShaderConverter.

With this modification, conversion succeeds even if the vertex attribute does not have a layout qualifier, but switching between shader variations may change the location of the vertex attributes. For this reason, we recommend using the layout qualifier to fix the location of vertex attributes.

// location is fixed.
layout(location=0) in vec4 vertex;


// location is not fixed, so you must re-create the vertex state after switching the shader variation.
in vec4 vertex;
