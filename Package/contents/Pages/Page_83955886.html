Overview

SoundProjectWaveExporter.exe (SoundProjectWaveExporter) is a command-line tool for batch-exporting the waveform files for all sounds that have been registered to a NintendoWare for SoundMaker project file.

This document describes how to use SoundProjectWaveExporter and its features.

Using SoundProjectWaveExporter

Running SoundProjectWaveExporter

To run SoundProjectWaveExporter, enter a command line in the following format.

SoundProjectWaveExporter.exe [Options] Project_file_path

Examples:

$ cd NintendoSDK/Tools/Audio/SoundMaker
$ ./SoundProjectWaveExporter.exe -o temp ../../SampleData/Sound/simple/simple.fspj

The waveform file is exported to the specified directory.

Project File Path

Specifies the project file (FSPJ) path.

Command-Line Options

The SoundProjectWaveExporter options are specified by a character or string that begins with a hyphen (-).

Option Description
-o, --output <output-folder> Specifies the output folder.
-t, --time <time>

Specifies the maximum export time (in seconds) .

[Default value: 30]

-r, --sampling-rate <rate>

Specify 32 for a sample rate of 32 kHz.
Specify 48 for a sample rate of 48 kHz.
The default is 48.
-m, --monaural Monaural output.
-s, --silent Do not export a log.
--sound-type

Limits the type of output sound waveforms.

You can specify wave, sequence, or stream.

Multiple types can be specified by separating them with commas.

Example: --sound-type wave,sequence
--split-sequence-sound-tracks Exports each track of a sequence sound to a separate file.
--enable-high-quality-resampler Specifies the use of a high-sound-quality resampler for pitch conversion in Voice processing.


SoundProjectWaveExporter Features

SoundMaker already has a feature for specifying one or many sounds for export to a waveform file. This tool can simultaneously export all of the sounds that have been registered to a SoundMaker project file to a waveform file.

Info

All sounds are saved as WAV files using 16-bit PCM compression regardless of the compression format that has been set for each sound.

Actions When Exporting

  • The log is exported to standard output.
  • Errors are exported even if the -s option has been specified.
  • If the folder specified with the -o option does not exist, a folder with the specified name is created.
  • Wave files that have been set to Disable Item in SoundMaker cannot be exported.
  • If the --sound-type option is used to limit the types of output sound waveforms, only sounds of the specified type are output.

Converting Sequence Sound Tracks to Waveforms

When the --split-sequence-sound-tracks option is specified, each track of the sequence sound is output to a separate waveform file.

  • The waveform files are named {sequence sound name}_01.wave with track numbers appended starting from 1.
  • If a track is silent, the corresponding waveform file is deleted.

Additional Information

  • For more information about using SoundMaker, see the SoundMaker Manual.
  • For more information about the text sequence files (FSEQ files) handled by SoundMaker, see the Sequence Data Manual.
  • SoundProjectWaveExporter.exe cannot be used by itself. There must be one in each installation_directory\Tools\Audio\SoundMaker folder.
