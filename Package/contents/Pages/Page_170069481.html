Audio News

NintendoWare Audio News

This page introduces the changes in the NintendoWare audio development environment for NintendoSDK 0.15.0.

SoundMaker

Sounds Can Be Exported as Waveforms

SoundMaker and SoundProjectWaveExporter can now convert any sound into a waveform and export it to a WAV file.

For more information, see SoundMaker / Waveform File Output Feature and SoundProjectWaveExporter.

Note

One known bug is that in some cases a period of silence is inserted at the start of the exported WAV file. This bug will be fixed in a future release.

AtkPlayer

I3DL2 Reverb Effects Can Be Previewed

I3DL2 Reverb has been added to the kinds of effects that can be previewed in the Effect panel of AtkPlayer. You can apply the effects and adjust the parameters just like you can for Reverb and Delay effects.

For more information, see the AtkPlayer page.
