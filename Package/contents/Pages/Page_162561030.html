VariableIconBody

VariableIconBody

Overview

The nn::mii::VariableIconBody class contains the body shape data for rendering an icon with a certain physique.

This class is required if you want to create a Mii icon with a certain physique.


Loading Resources

Body part data is not included in the Mii resource. You must load a separate file.

Load the following file.

$(NINTENDO_SDK_ROOT)/Resources/Mii/Common/iconbody/MiiBody.dat

The file contains body male and female model data.


Initializing and Finalizing

The following notes are for initialization and finalization. You can also view the API reference for each function.

Initialization

Use the nn::mii::VariableIconBody::Initialize() function to perform initialization.

You must load the body resource data required during initialization to the memory before initialization.

Use the nn::mii::VariableIconBody::CalculateMemorySize() and nn::mii::VariableIconBody::CalculateMemoryAlignment() functions to get the initialization memory size and alignment.

Use the nn::mii::VariableIconBody::CalculateMemoryPoolSize() and nn::mii::VariableIconBody::CalculateMemoryPoolAlignment() functions to get the initialization memory pool size and alignment. If the memory pool properties passed during initialization are CpuCached, you must flush the memory pool after initialization.

The memory and memory pool passed during initialization, must be maintained until you call termination processing. You can destroy the body resource file after initialization.

Finalization

Call the nn::mii::VariableIconBody::Finalize() function to perform termination processing. After termination processing, you can destroy the memory and memory pool used in initialization.

DrawParam

Use the nn::mii::VariableIconBody::GetDrawParam() function to get the parameters for body rendering.

Pass nn::mii::CharInfo to get the gender-specific parameters set internally. You can also specify the gender directly.

Matrix

Use the following functions to get the Matrix for variable icon body rendering.

  • nn::mii::VariableIconBody::CalculateWorldMatrix()
    Calculates the world matrix based on the specified gender and body type.
  • nn::mii::VariableIconBody::StoreSkeleton()
    Gets the matrix for skinning based on the world matrix obtained using CalculateWorldMatrix().
  • nn::mii::VariableIconBody::StoreHeadWorldMatrix()
    Gets the world matrix to place the Mii face (CharModel).
  • nn::mii::VariableIconBody::StoreCameraMatrix()
    Gets the matrix for the camera.
