Scene Animation

Overview

This page describes the NintendoWare 3D scene animation feature. For more information about creating animation, see the relevant sections in the pages for Plug-Ins and 3DEditor.

Scene animations can include the following types of animations.

Type Description
Camera Animation

This type of animation animates a camera.

For the camera type, you can select either projection mode or rotation mode. For projection mode, you can select either perspective projection or orthographic projection. For rotation mode, you can select either to spin around the look-at point or ZXY rotation.
Light Animation

This type of animation animates a light.

For the light type, you can select between ambient, directional, spot, and point light.
Fog Animation

This type of animation animates fog.

You can animate such parameters as distance attenuation and fog color.

NintendoWare only supports calculations for these types of animations, so you must implement processes in the application to apply the animation calculation results to camera, light, and fog.
