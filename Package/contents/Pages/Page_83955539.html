UI Descriptions

This section describes the user interface for 3DEditor.

Main Window

Describes 3DEditor's main window.
Main Menu

Describes the main menu in the main window.
Toolbar

Describes the icons used on the toolbar in the main window.
File View

Describes the file view in the main window.
Property Window

Describes the panel used to edit objects that are loaded into 3DEditor.
*The content of this panel depends on the object being edited.
Curve Editor Describes the editor used on animation files loaded into 3DEditor and newly created animations.
Color Picker Describes the color picker used when modifying color animations and blend colors.
Pop-up Slider Describes the pop-up slider that appears when editing data in 3DEditor.
