3. Linking With Viewer

Note

3dSampleViewer is a viewer for the 3DEditor tutorial. When you are actually using 3DEditor to produce a game production, you must incorporate the viewer library into the game application and prepare an environment that can be used to preview 3DEditor models.


This section describes how to link 3DEditor and the viewer (3dSampleViewer).
Linking with the viewer, starting/stopping animations, and changing the file being played back.

This tutorial involves the following UI.

Connect to PC Used to connect to the viewer (3dSampleViewer).
Animation Controls Starts and stops animation playback.
Changing Playback Animation Changes the animation that is being played.

Connecting to Viewer

On the 3DEditor toolbar, click Not connected.

If the connection is successful, "Connected" () is displayed, and the SampleTripod model is displayed in the viewer.

Info

If the connection process fails, check the following and then try connecting again.

  • From the Windows Start menu, select Nintendo Target Manager.
  • Check that “Win” is displayed as the platform in the toolbar.
  • Check that the connection target is set to PC ().

Starting and Stopping Animation Playback

Play back animations using Animation Controls at the bottom of the 3DEditor screen.

To start the animation, click the Start/Pause button (). Clicking the button again pauses the playback.

To stop the animation, click the Stop button ().

You can verify the animation in the viewer.

Related Links For more information, see Animation Operations.

Changing Playback Animation

The file that is played back is the last file in the Default Animation Set in File View.

Info

Make sure that the correct file is displayed in Animation.


To change the file that is played back, drag the file you want to play back so that it is the last file in the list.


Verify that the animation being played back in the viewer has changed.

Info

You can also choose the animation that is shown by selecting Show/Hide animation preview ().
