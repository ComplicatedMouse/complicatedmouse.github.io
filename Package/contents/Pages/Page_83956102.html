Depth Stencil View

Overview

Depth stencil view provides features for setting a texture as a depth stencil.
The properties of this view are basically the same as for texture view. See texture view.

The depth stencil view expresses a portion of the source texture with an array range and a specific mipmap level.

Information

Use nn::gfx::DepthStencilViewInfo to define the depth stencil view information.

Dimension, Format, Texture Pointer

See texture view, which has almost identical settings.

The source texture format must be usable as a depth stencil.

The source texture must be initialized in a way that includes GpuAccessFlag_DepthStencil as a method for GPU access.

Sub Resource Range

The array range is specified by the EditArrayRange function, and the specific mipmap level is specified by SetMipLevel.

Object

Use nn::gfx::DetphStencilView to define the depth stencil view object .

Initialization and Destruction

Depth stencil views are initialized and finalized according to the rules for objects.

Setting a Depth Stencil View

You can set the depth stencil view as the depth stencil by specifying it together with a color target view array using the SetRenderTargets function of the nn::gfx::CommandBuffer class.
The format of the depth stencil to set must match the depth stencil target format of the render target state.

Clearing a Depth Stencil View

Although depth stencil view can freely write through depth writing and stencil writing, a clear function is provided to allow you to write the same value uniformly to all pixels in the depth stencil.

Use the ClearDepthStencil function of the nn::gfx::CommandBuffer class to clear the depth stencil. Specify the value to clear for the depth value and stencil value for all pixels.
At this time, you can also clear only a specific array range in the depth stencil target, by specifying the desired range in pArrayRange. When NULL is specified for this parameter, the entire array range is cleared.

Restrictions on Clear

The render target settings are disabled after a clear. Set the render target again, if needed.
