Preparing the SDEV

The following three steps are required to set up the SDEV.

  • Turn the SDEV on.
  • Connect the SDEV to a PC.
  • Connect a Wii Classic Controller Pro.


Info

Before taking any of these steps, check the DIP switches on the lower left of the back of the SDEV (framed in red in the image), and confirm that they all are set in the up position.

Turning the SDEV On

Connect the included Wii U console AC adapter to the AC adapter port on the top of the SDEV (circled in red in the figure).

The SDEV does not have a main power switch, so you must connect and remove the Wii U console AC adapter to turn the main power on and off.

When you hold down the POWER Button on top of the SDEV console for seven seconds, the system turns off, but the SDEV's main power remains on.

You can check whether the SDEV has been turned on using the following two methods.

  • The LEDs on the Ethernet port on top of the SDEV light up (and then turn off after approximately 10 seconds).
  • The middle of the three LEDs on the back of the SDEV in the lower-left corner lights up.
Info

Some SDEV sets include the following instead of the Wii U AC adapter.

  • AC adapter (UNI336-1524)
  • Conversion cable for UNI336-1524

To use, attach the conversion cable to the AC adapter.

Connecting the SDEV to the PC

Connect an Ethernet cable to the Ethernet port on the top of the SDEV (circled in red in the figure).


Info

You can check the connection between the SDEV and a PC by following the procedure in Initializing the SDEV.

There might be a problem with the connection if the SDEV is not displayed in the initialization UI.


Use one of the following methods to connect the PC and the SDEV console via Ethernet.

  1. Connect the PC and the SDEV directly using the Ethernet cable.
    • The setting for the network card directly connected to the SDEV must be to set to Obtain an IP address automatically.
    • For more information about configuring this setting, see Direct Connection IP Address Settings.
  2. Use the Ethernet cable to connect the SDEV to a network device (switch or router) for the network to which the PC is connected.
    • This works for networks with and without a DHCP server.
    Note

    If you are connecting the SDEV directly to the Internet, you must use the iptables feature to configure the appropriate network connection restrictions.
    For more information, see NintendoSDK Documents > Guide to Setting Up the Environment ~NX Addon~ > More Information.

Improving Communication Speed With the SDEV

The communication speed with the SDEV can be improved if the PC's network adapter supports jumbo frames.

Open Device Manager, select Network adapters > Properties (for the target adapter) > Advanced, and then set Jumbo Frame to 4 KB or 4088 bytes.

Note: For some network adapters, this setting is labeled Jumbo Packet.

Note

You may not be able to use Jumbo Frame depending on the environment, so please disable it if communications are not smooth.

Connecting the Wii Classic Controller Pro

Connect the Classic Controller Pro to the debug controller terminal on top of the SDEV (circled in red in the figure).

The Classic Controller Pro can be used as a debugging controller.
