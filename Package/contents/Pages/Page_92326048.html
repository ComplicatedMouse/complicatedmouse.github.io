Emitter Set Node Operation

This page describes how to operate emitter set node features in the workspace.

Preparations

Related Links Before beginning, complete the tasks described in Quick Start Create Effects.

Using the steps to this point, multiple emitters are created directly beneath the EmitterSet1 emitter set (), as follows.

Changing the Name of the Emitter Set

This section describes how to change the name of an emitter set that has already been created.
With the emitter set EmitterSet1 () selected in the workspace, do one of the following.

  • From the main menu, select File > Rename ().
  • Right-click and select Rename () from the shortcut menu.
  • Press F2.

The name can now be changed. Type Flow as the emitter set name. Press ENTER to save the change.
EmitterSet1 () is changed to Flow ().

Duplicate Emitter Set

This section describes how to duplicate the selected emitter.
With the emitter set Flow () selected in the workspace, do one of the following.

  • From the main menu, select Edit > Duplicate ().
  • Right-click and select Duplicate () from the shortcut menu.
  • On the toolbar, click Duplicate ().
  • The keyboard shortcut is CTRL+D.

The emitter set Flow_Copy1 () is added to the workspace.

Reorder Emitter Set

This page describes how to reorder multiple emitter sets.
Select Flow () when the Flow emitter set () is on top of Flow_Copy1 ().
Do the following.

  • Press and hold the ALT key, and drag Flow to below Flow_Copy1.

Flow () is moved to below Flow_Copy1 ().
