Bounce Count (±)

This page describes Bounce Count (±), a feature for configuring the number of times a particle will collide with collisions.

Overview

Bounce Count (±) enables you to configure the number of bounces that include the particle colliding with collisions.
This feature is applied when the emitter type is CPU and Behavior When Colliding is set to Bounce.
The following overview describes the settings ().

Bounce Count (±)

This feature is applied when Behavior When Colliding is set to Bounce.
The count for bounces against a collision, during particle collisions, is set to an integer value, using the dial () or text box ().

You can set infinite bouncing using a value of -1.
