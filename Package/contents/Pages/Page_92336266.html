Object View

This section describes the Object View in 3DEditor.

Switching the Object View Display Mode

You can change the display mode of the main window by using the following buttons.
The default mode is the Schematic Display mode.

Info

Each view can change its display mode when the object view is split.

Display Modes

Related Links For more information about the shortcut keys that are available in each display mode, see here.

Modes Description
Schematic Display Displays the structures of models and associated nodes (bones, shapes, and materials) schematically.
This display allows you to make edits from the shortcut menu by right-clicking the image being displayed, and entering comments by pressing F3.
Model List Displays a list of models loaded into 3DEditor.
Material List Displays a list of materials included in the model.

The material list has the following features and displays the following information.

  1. A filter for viewing pages one at a time because there is a large amount of information displayed
  2. Displays whether the optimization shader is used for each material

If improper data is being used, the material name and improper data appear in red.
Bone List Displays a list of bones included in the model.
Shape List Displays a list of shapes included in the model.
The names of the materials used for each shape are also included in the list.
Texture List Displays a list of textures loaded into 3DEditor.

Operations When Using the Schematic Display

When a schematic display is used for the object view, you can control the display using icons and sliders.

A Selecting this check box displays the parent material set as the material reference. (See View the parent material.)
B

Aligns objects and trees vertically and displays the combination at the center. Note: See each alignment mode
C

Aligns objects vertically and trees horizontally and displays the combination at the center. Note: See each alignment mode
D

Aligns objects and trees horizontally and displays the combination at the center. Note: See each alignment mode
E

Displays the display scale of the view.
F Use the slider to adjust the display scale of the view.

G

Adjusts the view scale between minimum and maximum in 10% steps.


Note: View the parent material.

If a material reference has been set, this displays the parent material.

View the Parent Material Disabled View the Parent Material Enabled

(The parent material is displayed in a darker color. If a material file is being referenced, it is connected with a dotted line.)

Operations During List Display

The following operations are available when the object view is in list display mode.

A

The currently selected display mode. Use the display mode select button on the right to change the display mode.
B

A description of the selected display mode is displayed for each item. Displayed information can be sorted by clicking an item.

Column Item Setting for List Display

You can also open a dialog box for setting column items from the shortcut menu that appears when you right-click an item.

Display Invalid Data in Material List

When you change the object view to material list mode, invalid data is displayed in red text.

Material List Filter Feature

You can now filter items in the material list, as follows.

All

Displays all material information.
General

Displays the same information listed on the General page in the materials property window.
Preview

Displays the same information listed on the Preview page in the materials property window.
Sampler Displays the same information that is listed on the Sampler page in the material property window.
Shaders Displays the same information that is listed on the Shader page in the material property window.
Parent material

Displays the same information listed on the Material Property Window: Material Reference Page.

Using Optimized Shaders

In the material list, you can check the color of the icon for each material loaded in 3DEditor to determine whether it has been optimized.

Info

● Yellow icon: Shader not sent.
● Blue icon: Optimized shader sent.

You can also widen the column where the optimization icon is displayed to check for the accompanying comment.

Operations Using Divided Screens and Display Mode Switching

To demonstrate how useful this can be, this example splits Object View horizontally into two screens, with the display mode set to Schematic Display in one and Material List in the other.

As shown in this example, by splitting the screen in two you can check both the schematic display and the names of all materials at the same time.
If you show only the schematic display, you can only check some the material names, so you can see that it is convenient to split the screen in two.
