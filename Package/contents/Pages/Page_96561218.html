Add-On Value (±)

This section describes the Add-On Value (±) feature, which specifies the coordinate value to add to the position of the particles.

Overview

The Add-On Value (±) feature is used to specify the value to add or subtract from the particle position along with the XYZ coordinates.
Increasing this value increases the distance that the particles move per frame.
The following overview describes the settings ().

Add-On Value (±)

The values to add or subtract from the particle location for each frame can be specified using a dial () or text box ().

Info

Press the Particle Time Animation button () to display the Add-On Value (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().

Example Configurations

The following figure compares different values specified for the y-coordinates in Add-On Value (±).

A 0.2 The particles move in the positive direction along the y-axis.
B 0.5 The particles move in a greater distance toward the positive direction of the y-axis when compared to A.
C -0.2
 The particles move in the opposite direction (negative direction of the y-axis) compared to A.

Animating the Add-On Value (±) value makes it possible to create effects similar to gravity fields.
This can be used to create animations in which different gravity forces are applied to each individual particle.
