Album Applets

This is an applet library for calling the album features of the console.

For more information about albums, see System Application > Capture Features - Album in the Console Features Guide.

Provided Features

The following table lists the supported function calls and provided functionality.

Functionality Corresponding Function Notes
Displays a list of files in the album nn::album::ShowAlbumFiles

Only displays files that are associated with the application calling this function.

Including Header Files

Add the following code to your program.

#include <nn/album/album_Applet.h>

Starting the Album Applet

The functionality of the album applet can be accessed by making the supported function calls.
The function that starts the album applet blocks until the applet is closed.
