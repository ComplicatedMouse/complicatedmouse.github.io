This section describes the Comments feature for editing and checking text comments written to the preview file.

Overview

The Comments feature is for editing and checking text written to the preview file selected in the workspace.
The following overview describes the settings ().

Comments

Preview comments are displayed in a text box (). The comments may be edited by clicking on it.
To copy the file path, right-click it and select Copy from the shortcut menu.
