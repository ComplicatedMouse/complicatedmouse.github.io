Plug-In for Disabling Node Compression

Sets a flag so that objects and nodes are not compressed even if the node compression feature is enabled. It is also possible to search for settings.
Select Nintendo > Set No Compress Node to display the window.

Setting Method

When one or more objects or nodes are selected, the Set field displays current settings for the selected nodes.
If you select multiple nodes or groups, the setting for the first selected node is displayed.

Click one of the Set list items to change the flag for disabling node compression. You can select multiple nodes to change the settings for all selected nodes at the same time.

If Compress this node if possible is set, compression will be performed when the node compression function is enabled. This setting is the default.
If Don’t compress this node is set, compression will never be performed even if the node compression function is enabled.

You can select a different node by selecting another node without closing the window.

Searching for Attributes

Click one of the options in the Search column and click Search to select the corresponding objects or nodes.
