CaptureVisualizer

Overview

CaptureVisualizer is a tool for controlling the CaptureProcessor image processing parameters.

The parameters controlled using this tool enable you to check the behavior results when executing the CaptureProcessor API, in addition to the source code for executing that process.

How to Use

Starting the Tool

You can start the tool by running the program in the following location.

Tools\AutoTestAssistTools\Tools\CaptureVisualizer\CaptureVisualizer.exe


The following two windows are displayed after starting the tool. The window displaying the target screen capture is the preview window, and the other one is called the main window.


The basic procedure is to select the screen process item you want to check from the main window's top tab. You can then check the process results by using the UI for parameter controls.

Warning

Behavior is not assured when using a capture board if the capture board input resolution and the resolution specified for Target are not the same.

Use with a Target selected that has the same resolution as the input resolution.


Saving Captured Images

When you drag to select an area in the preview window, you can save the selected area in PNG format. You can select the directory it is saved to from the Config tab in the main window.

Config

You can configure overall settings for the tools in the main window's Config tab. The following items are supported for configuration.

Option Overview
Target

The target for getting a screen capture.

You can select any of the following.

  • A video input device, like a capture board, connected to the host PC
  • A target registered in TargetManager
  • A selected area on the desktop
    (* For more information, see Desktop Capture.)
Output Image Path The output directory path that the clipped image is saved to when you select an area in the capture preview window.
Code Type Type of sample code displayed.
Border for capture rect The color of the border of the selected region when saving the captured image.
Border for result rect The color of the border displaying the area for processing results like Labeling, OCR, and Detect Object.

Desktop Capture

When you specify Desktop Capture as the screen capture target, you can capture an area of the desktop screen.

You can specify a screen area to capture by moving or resizing the blue Select Region window. You can switch between showing and hiding the Select Region window by using the Select Region button in the main window.

Info

If you capture the desktop while the Select Region window is displayed, the capture applies the blue filter. After you have specified the target area in the Select Region window, we recommend that you click Select Region to hide the window.

Adjusting Parameters

By opening each tab of the main tab, you can execute the image processes corresponding to each tab. The results of the filter processing are applied to the preview window, and you can check how the screen changes when applied.

Detection results are also displayed in the text box at the bottom of the main window for detection processing such as labeling or OCR.


Each tab supports the following processes.

Tab Function Notes
Color Range SetColorRangeFilter You can manipulate the lower and upper limit values of the color area filter range and check the applied results in the preview window.
Binarization SetBinarizationFilter You can manipulate the binarization filter threshold value and check the applied results in the preview window.
Morphology SetMorphologyFilter You can manipulate the type of morphology conversion or number of processes and check the applied results in the preview window.
Labeling ExecuteLabeling The area coordinates detected when labeling is executed can be checked in the text box at the bottom of the main window.
OCR ExecuteOcr The area coordinates detected when OCR is executed can be checked in the text box at the bottom of the main window.
Detect Object DetectObject You can check the coordinates where a template image is detected when template matching is executed in the text box in the lower part of the main window.

For more information about each process, see Image Processing Features.

Color Range

You can check the color area filter processing results in the Color Range tab.

The slider displayed in the main window corresponds from the top in the order of Red, Green, and Blue. The left and right correspond to the minimum and maximum values.

The results where the color area filter is applied are displayed in the preview window.

Value Image

lowerColor = [0, 0, 0]

upperColor = [255, 255, 255]

lowerColor = [0, 200, 0]

upperColor = [180, 255, 180]


Binarization

You can check the binarization filter processing results in the Binarization tab.

You manipulate the binarization filter threshold value by using the main window slider, and the results of applying the binarization filter are displayed in the preview window.

Value Image
binarizeThreshold = 128

Morphology

You can check the morphology conversion filter processing results in the Morphology tab.

You can manipulate the type of morphology conversion or number of conversions in the main window and check the results with the morphology conversion filter applied in the preview window.

Value Image

type = MorphologyConversionType_Erosion

iterations = 2

type = MorphologyConversionType_Dilation

iterations = 1


Labeling

You can check the value you can get when executing the labeling process in the Labeling tab.

As a result of the labeling process, the detected area within the bounding rectangle is displayed in the text box at the bottom of the main window.


You can display the area detected using the labeling process in the preview window by clicking the Show result rects button in the main window. The color of the displayed border can be changed from the Config tab of the main window.


For more information about executing image processing filtering in labeling preprocessing, see About Preprocessing Filters.

OCR

You can check the string that you can get when OCR is executed in the OCR tab.

The target language can be selected with the drop-down menu from the main window, and the execution results are displayed in the text box at the bottom of the main window.


You can display the area of the string detected using OCR in the preview window by clicking the Show result rects button in the main window. The color of the displayed border can be changed in the main window from the Config tab.

For more information about executing image processing filtering in OCR preprocessing, see About Preprocessing Filters.


Detect Object

In the Detect Object tab, you can check the coordinates where a template image is detected using template matching.

Before you do this, you must open the main window, click the Please click and select an image button, and specify the template image.


You can display the area detected using template matching in the preview window by clicking the Show result rects button in the main window. The color of the displayed border can be changed in the main window from the Config tab.

For more information about executing image processing filtering in template matching preprocessing, see About Preprocessing Filters.


About Preprocessing Filters

In tabs for detection features, such as Labeling and OCR, you can click the + Add Filter button in the main window to run filter processing before running the detection processing.

Up to five filters can be added by clicking the + Add Filter button. You can delete filters by clicking the X button to the left of the filter. You can also select the processing to perform from the drop-down menu.


When multiple filters have been added, processing occurs in order from top to bottom.
