Emission Process Count

This section describes the Emission Process Count feature for setting how many times to call the emitted particles.

Overview

The Emission Process Count feature provides a way to set the number of calls that the program makes to the emission process.
The emitter's emission process is called the specified number of times.
The following overview describes the settings ().

Emission Process Count

Use the dial () or the text box () to set the number of times to calculate particle emissions.
