Using a Parts Layout

This describes the procedure for loading and placing a Parts Layout in a Body Layout that is the placement target.
The following are the Parts Layout and the Body Layout that is the placement target.

Parts Layout Body Layout

Select button created in Creating a Parts Layout. The menu screen background created in Creating a Body Layout.

Preparing to Create the Layout

To prepare to create the layout data, have LayoutEditor running in advance.

Related Links For more information about starting LayoutEditor, see Workflow.

Opening Windows

Begin the task after opening the following windows.

  • Hierarchy
  • Property Window
  • Parts Palette

Related Links For more information about the UI of each window, see Windows.

Updating the Parts Palette

If you are performing the update through the tutorial, the created layout will not appear in the Parts Palette window.
Click the Update () button in the Parts Palette window. Updating the Parts Palette to the latest version displays the created layout.

Related Links For more information about the Parts Palette window, see here.

Opening the Body Layout

Open the Body Layout from the Parts Palette.

In the Parts Palette list, right-click Menu.flyt, and from the shortcut menu, select Open.
Menu.flyt is loaded in the LayoutEditor.

Registering Parts

In the Parts Palette window, load the Parts layout in the Body layout, and place it as a Parts pane ().

In the Parts Palette window, double-click Menu_Bar.flyt.
In the Hierarchy window, L_parts_00 () is added.

Related Links For more information about Parts Pane, see here.

Moving Parts Panes

In the Hierarchy window, select L_parts_00, and then in the Property window, configure the settings as follows.

Base Pane ()

 Position X,Y = -275,260

Creating a Null Pane

Place the registered Parts Pane () in the hierarchy, and create a Null Pane () to animate it.

From the main menu, select Pane > Create Pane > Null Pane.
In the Hierarchy window, select the created null pane () and change the pane name to N_InOut.

Creating Pane Hierarchy

In the Hierarchy window, place L_parts_00 () under the created N_InOut () in the hierarchy.

Duplicating a Parts Pane

Use the following procedure to duplicate the Parts Pane ().

1

In the Hierarchy window, select L_parts_00 ().
2

From the main menu, select Edit > Copy to copy L_parts_00 ().
3

In the Hierarchy window, select N_InOut ().
4

In the Hierarchy window, select Operations > Paste to copy L_parts_00.
In the Hierarchy window, check that L_parts_01 () has been added.

Moving a Duplicated Parts Pane

This procedure moves L_parts_01 () in the Layout window.
Select L_parts_01 () and press the down arrow (↓) key while checking the location in the Layout window.

Repeat the procedures in Duplicating a Parts Pane. Parts 02 to 07 will be appended to the ends of the duplicate pane names.
Set the y-position spacing of the parts panes to 70 pixels, and arrange the duplicated panes at equal intervals.
The following table shows the final placement of each pane.
