Layout, Font News

This document is a graphical introduction to the features that were added in NintendoSDK 0.17.0.

Configuring the Default Texture Format You can now select the LayoutEditor default texture format. Functionality added from version 0.11.2
Color Picker Hexadecimal Input You can now enter Color Picker values using hexadecimal entries. Functionality added from version 0.11.2

LayoutEditor

This section introduces the new features in LayoutEditor.


You can now select the default texture format.

Added functionality that enables you to set the LayoutEditor default texture format.

Specify the DefaultTextureFormat element, using Project Settings.

Info

The project configuration file is a file in XML format to record common settings for the game project.
It is created under the parts root folder.

Related Links For more information, see the following pages.

Project Configuration File Format Describes the specifying of a default texture format.


You can now enter Color Picker values using hexadecimal entries.

Added a form that allows entry of RGB values for the LayoutEditor Color Picker in hexadecimal.

You can now enter RGB values in the text box, starting with a number sign (#).
