Using Per-Character Transformation

Describes how to use per-character transformation.

Making the Settings in LayoutEditor

Step 1: Enable the Per-character transformation property for the text box pane.

  • Select the text box, and then double-click it to open the property window.
  • In the tree on the left, select the text box pane, and select the check box to enable in the per-character transformation group. Selecting that check box enables the various configuration parameters.
  • If Enable is not selected, there is no significant additional processing load.
  • Set the Offset Time and Duration as needed. They can be left as is if their use is unclear.
  • If you are using this feature for the first time, try using more characters than usual and not using line breaks to make the effects easier to understand.

Step 2: Configure a per-character transformation curve for the text box.

  • Open the curve editor.
  • Select a text box for which you enabled the Per-character transformation property in step 1.
    Select the following items from the tree on the left. Pane Name > Text Box > Text Box > Per-character transformation: Position

  • Configure the key frames and transformation curves. If configured correctly, the change to the characters appear on the editing screen.
    The following example shows the display when setting a curve for Per-Character Transformation: Y-Position.
    The curve is divided by the number of characters, and characters are applied in order to the frames at the corresponding positions.

  • Specify the range of the configured transformation curve to evaluate by adjusting the Offset Time and Duration set in step 1.
    By default, Offset Time is 0, Duration is 100, and the per-character transformation is performed on a curve from frame 0 to 100 (the range outlined in blue).
    If Offset Time is 50, a curve from 50 to 150 is used to perform the per-character transformation.
    If Offset Time is 0, and Duration is 50, a curve from 0 to 50 is used to perform the per-character transformation.

Step 3: Animation (Optional)

  • Set the key for the Per-character transformation: Time Offset in Curve Editor (Pane Name > Text Box > Text Box > Per-character transformation: Time Offset).
    Example with setting the Time Offset 0th frame key to 0, and 100th frame key to 50.

  • These settings animate the Offset Time from step 1 and change the offset of the evaluation of the transformation curve. If you play this animation, the characters are usually animated.

  • Select a transformation curve configured in step 2 to check its evaluation offset as the blue line shown in Curve Editor. (Normal animation playback time is the red line.)
    Example of animating a time offset. The evaluation start time is indicated with a blue line.
    The A character shows the frame value corresponding to the evaluation start time.
