Error Codes

Error Codes of the account Library

The errors codes that are passed via the account library for display in the error viewer are defined in the following range.

Error codes passed via the account library are represented as 2124-NNNN, where NNNN is a value in the range from 0000 to 8191.

This section describes the causes of errors codes passed via the account library.
Note that this document does not describe every error code passed by this library.

Note

The error codes and their descriptions in this document differ from the descriptions presented to end users.
It is assumed that this document will be referenced only during the process of application development by developers using the Nintendo SDK.

account Library Error Codes and Result Values

The Result values returned via the account library are defined as result with a Module value of 124.
For Result values with a Module value of 124, the Description value conforms to the NNNN part of the 2124-NNNN error code passed via the account library.

Refer to this document if there is a Result value pertaining to the account library that you cannot find using the Find Result feature of the API Reference.

Note

The Description values of the account library's private Result values may vary depending on the system version.
For this reason, do not branch on Description values in your application.

Error Codes Assumed to Be Displayed in Error Viewer

The following range of error codes defined in the account library (2124-NNNN) are assumed to be displayed in the Error Viewer.

Name Start End Description
Authentication and Communication Errors 2124-3000 2124-8191 Refer to nn::account::ResultNetworkCommunicationError in Handling Errors.

Error codes outside of this range indicate an application program error or a problem in the error handling or firmware.
Check the application implementation or contact support.

FAQ

Generation of 2124-7400 or 2124-7403 Error Code When Getting Nintendo Account Authorization

If you get a 2124-7400 or 2124-7403 error code when you call one of the following functions, it might be because the request to use the authorization service was not approved, or because the specified Application ID differs from the submitted one.

  • nn::account::NintendoAccountAuthorizationRequestContext class
    • GetResultWithInteractionIfRequired() function
    • GetResult() function
