Skinning

The Maya plug-in supports animation that uses smooth skinning.

Animation using rigid skinning is not supported.

  • Skinning Settings
  • Cancel Skinning

Skinning Settings

Smooth skinning is set using Skin > Bind Skin.

If skinning is set, the FMD file's translate, rotate, scale, and polygon shapes are all output in the bind pose.

Info

Do not perform any of the following operations after using Bind Skin.

• Operations that change the number of vertices.
• Change the Rotate Order or Segment Scale Compensate of nodes that affect skinning.
• Change the transform offset parent matrix for the transform node of an object set for skinning.

If you must perform any of these operations, use Detach Skin first.
To change the number of vertices, select Edit > Delete by Type > Non-Deformer History to make output possible.

Vertex positions may also be changed after you use Bind Skin.

Info

tweak nodes must be located upstream from skinCluster nodes.
Normally, tweak nodes are created upstream of skinCluster nodes, but tweak nodes could be downstream of skinCluster nodes if the deformer order is changed in the pop-up window opened by selecting Inputs > All Inputs.

When you want to perform transformations such as translation, rotation, and scaling on the parent node of the object set for skinning, clear Inherits Transform for the transform nodes of the object set for skinning.

Do not change the names of original mesh nodes (~Orig) created after performing Bind Skin.

Make sure that skinning weights for a single vertex add up to 1.0.
The rendering processing load can be lowered if the weights for each node of all vertices is 1.0.

Info

Up to eight nodes per vertex can be set to a weight other than zero. If weights are assigned to more than nine nodes, up to eight nodes in the order of largest weight are applied in the intermediate file.
You can also specify the maximum number of nodes over which a weight can be distributed in the config file of the export plug-in.

Info

By default, weights are snapped to a multiple of 0.01 during export (in Maya, weights less than 0.005 are treated as 0).
You can use the config file for the export plug-in to change the export settings to prevent snapping weights.

Info

Do not use multiple bind poses. When you bind multiple objects to the same skeleton, use a common bind pose.

Set Envelope for the skinCluster node to 1.0.

Shapes are not exported if the Primary Visibility attribute on the influence object mesh node is set to OFF.
Clear the Use Geometry option when you run Add Influence.

If you use UV projection mapping (such as planar mapping, cylindrical mapping, spherical mapping, or automatic mapping) after binding skin, make sure that you select the Insert Before Deformers option for each projection mapping.

If you combine skinned objects while History is enabled, the skinning animation will display in Maya but can no longer be output to an intermediate file. In this case, delete the history and reset skinning.

Cancel Skinning

Normally, set the History of Unbind Skin to Delete History.

If you set the History option to Keep History, weight can be retained, but Detach with this setting does not detach skin completely and only skinCluster nodes become invalid.

Info

After performing Detach with the Keep History option, do not perform any operation that changes the vertex count.

After performing Detach with the Keep History setting, you may perform operations such as changing the bind pose.
