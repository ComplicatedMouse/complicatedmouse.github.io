Deprecated Features

ld and gold Linker Removal

The ld linker and the gold linker, which have been deprecated since NX Add-On 10.4.0, have been removed from the package starting with NX Add-On 15.0.0.

Any use of the ld and gold linkers will need to be changed to use the lld linker.

End of Support for Pre-UAL (Unified Assembler Language) Syntax

Starting with Clang for NX 1.6.0, which is planned to be a part of NX Add-On 7.0.0, attempts to compile code using earlier pre-UAL syntax result in an error.

For this reason, you must migrate to UAL syntax.

Specifically, you must replace the pre-UAL instructions on the left with the supported UAL instructions on the right.
