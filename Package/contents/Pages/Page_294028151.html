Multi-Scalable Font Properties

Multi-Scalable Font Property Window

This section describes the UI of the Multi-Scalable Font property window.

A Comments You can add comments.
These comments are added to the MultiScalableFont node in the FCPX file.
B Font Size Specifies the font size.
C Alternate Characters Sets alternate characters.
D Line Feed Offset

Specifies the line feed offset.
E Reorder Scalable Font Setting Sequence Button Changes the positions of items selected in the Scalable Font Settings List.
F Scalable Font Settings List Displays a list of the scalable font settings for a child node.
The higher an item is displayed in the list, the higher the priority.
