Sequence Sound Set

Select the Sequence Sound Set node in the project tree to show a list of the sequence sound sets in the Sound Set tab.
The list is split and displayed in upper and lower sections in the Sound Set tab.
The upper part of this split tab is called the Sequence Sound Set List, and the lower part is called the Sequence Sound List.

Role of Sequence Sound Sets

The data that you can register in a sequence sound set is the same as the data that is registered in sequence sounds. However, you can collect numerous sequence sounds together and register them in a single sequence sound set.
Group registration can also occur at the sequence sound set level, so using the sequence sound set can make the group management of sequence data easier. (You can also register a single sequence in a sequence sound set to a group.)
Sequence sounds are primarily for background music, and sequence sound sets are typically for sound effects.

Creating and Adding Sequence Sound Sets

To create or add a sequence sound set, select the list region in the Sound Set tab and then select Project > Add Item > Sequence Sound Set.

With sequence sound sets, you can create or add a new sequence sound set by using the shortcut menu, which opens when you right-click the sequence sound set list.
