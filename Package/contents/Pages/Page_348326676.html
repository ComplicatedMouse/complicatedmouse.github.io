Rounded Corners

Describes the Rounded Corners feature to round the four corners when the pane shape is rectangle.

Enable Rounded Corners Sets whether to add rounded corners to the pane.
Enable Outlines Sets whether to add an outline to the pane.
Enable Shadows (Interior) Sets whether to add a shadow inside the pane.
Enable Color Overlay Sets whether to add a color overlay to the pane.
Enable Gradation Overlay Sets whether to add a gradation overlay to the pane.
Enable Drop Shadows Sets whether to add drop shadows to the pane.

Overview

Rounded Corners is a feature for changing the shape of the pane corners, without adding resources such as textures or meshes.
For example, the Rounded Corners feature allows you to make the rounded parts contract into the interior of the pane, make the rounded parts circle shaped, and set animations to animate the shape.
