LayoutEditor

This section introduces the new features in LayoutEditor.

Feature to Display the Curves In Segments

The feature in the Animation Segment Tags window for displaying all of the curves included in a segment can now be used also in merged mode.
The following video clip shows this feature in action after a segment has been selected from the list in the Animation Segment Tag Settings window and the button is clicked.

FcpxMaker

Feature to Run BfttfMaker

Added a feature to call the BfttfMaker tool of FcpxMaker. To run the tool, from the Tools menu, open the BfttfMaker dialog box and specify the input file and the output file.

Runtime Libraries

Describes changes to the runtime libraries.

Temporarily Disabling Vector Graphics Texture Functionality

Vector graphics functionality is not available in NintendoSDK 7.3.0. A warning message appears in the log when using vector graphics. Nothing is rendered for the texture. Wait for subsequent releases.

Adding the Ability to Visualize the font Library Texture Cache

Added functions and samples to visualize how the texture cache of the font library is used.

For more information about using it, see FontViewingTextureCache in the FontDemo samples.
