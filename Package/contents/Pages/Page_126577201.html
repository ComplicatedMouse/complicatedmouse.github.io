Load Favorite Samples

This section describes how to use the Effect Browser () for loading files using favorite samples.

Preparations

Before beginning this task, start EffectMaker and EffectViewer.

Related Link For more information about starting EffectMaker and EffectViewer, see Workflow.

Launch Effect Browser

Start Effect Browser by clicking the Effect Browser button () on the EffectMaker toolbar.

Using Effect Browser

The following describes how Effect Browser works.
Related Links For more information, see Effect Browser.

Select the Favorites tab

From the Effect Browser window, select the Favorites tab.

Select your favorites

As a sample favorite, select VfxSample (), from the navigation pane within the Favorites tab.

Select a file

Select Bubble_Fish.work () displayed in the file list.

Export the selected file

Click Export to workspace () from the toolbar while the file is selected.
Doing so loads a file associated with Bubble_Fish.work () to the workspace.

Info

You can save your work in EffectMaker as a workspace (WORK) file ().
When you open a saved workspace file, you effectively restore the workspace to the state it was in when the workspace file was saved.
Related Links For more information, see Workspace Files.

Using Main Window

This section describes working in the main window.

Check the workspace

Check the workspace.
Make sure that a file associated with Bubble_Fish.work () has been loaded to the workspace node.

The following is a list of node types to check and the files that are added to those nodes.

Node Types File
Emitter set () Bubble_Fish.eset (), Drop_02.ftxb (), SphereMap.fmdb ()
Preview () Bubble_fin1.prev (), Bubble_fin4.prev (), Bubble_head.prev ().
Model Settings () Fish.fmdb (), Fish.fskb ().
