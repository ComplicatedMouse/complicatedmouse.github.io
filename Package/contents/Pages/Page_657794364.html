Overwriting extended user data in panes within parts disables some functions.

Question

Overwriting extended user data in panes within parts disables some of the functions set on the source part.

Answer

The following features use the extended user data region. If the extended user data on the pane is changed by being overwritten, they can no longer be used.

  • Capture Effect
  • Shape (Rounded Corners)
  • Align Pane
  • Mesh Trimming Optimization
Info

Effects are not disabled by capture panes () that use capture features comparable to capture effects ().
