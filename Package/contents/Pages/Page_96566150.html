Inheritance Settings: Speed

This page describes the Speed inheritance setting for child emitters.

Overview

The Speed inheritance settings inherit the initial velocity from a parent emitter in a higher node.
The following overview describes the settings ().

Speed

The feature to inherit speed from the parent emitter is enabled or disabled using the check box ().
Enable this setting to display the Speed group () of settings. The group has the following settings.

Inherited velocity ratio

This setting is displayed when the Speed inheritance setting is enabled.
When enabled, the initial speed is multiplied by the parent emitter particle speed using the specified ratio.
