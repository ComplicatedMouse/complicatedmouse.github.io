Files Used in EffectMaker

Describes the file types used in EffectMaker.

Workspace File WORK

The workspace file records work in the current workspace.
A workspace file contains the following information:

  • Preview file (PREV) settings ()
  • Link information for the viewer settings and background texture files (FTXB) ()
  • Link information for model files (FMDB) and animation files (FSKB) ()
  • Link information for emitter set files (ESET) ()
Emitter Set Files .eset A file for an effect. Contains the following information.
  • The emitter set's settings
  • The name of the texture file (FTXB) that determines the particles’ appearances ()
  • The name of the primitive files (FMDB) that can be used as emitters or particles ()
Preview File PREV Preview files allow you to use EffectViewer to check settings of emitter sets that can be controlled in the game application.

Contains the following information.

  • Preview file (PREV) settings ()
  • The names of emitter set files (ESET) ()
Model Files For Linking .fmdb

Models files that are linked to check how they work with effects in the viewer. Contains the following information.

  • Names of texture files (FTXB) used by the model ()
  • Names of skeletal animation files (FMAB) used by the model
