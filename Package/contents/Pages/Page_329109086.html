Capture Effects (Drop Shadow)

The Capture Effects (Drop Shadow) page is used for setting drop shadows in panes.

Use Knockouts

Configure whether the drop shadow effect where panes overlap will be knocked out.
This feature is mostly used in combination with Render Effects Only.

Use Knockouts

When cleared

Renders the drop shadow effect where it is overlapped by the pane.

Use Knockouts

When selected (default)

Omits the drop shadow effect where it is overlapped by the pane.

Render Effects Only

Configure whether to render only the drop shadow effect.

Render Effects Only

When cleared (default)

Both the pane and the drop shadow effects are rendered.

Render Effects Only

When selected

Only the drop shadow effects are rendered, without rendering the pane.

Perform gamma correction of the effect if working in linear color space

When enabled in the linear workflow, this option applies degamma correction so that the midtones of the gradient created using Capture Effect are displayed correctly.

The default settings for this function can be changed in the project configuration file.
Related Links For more information, see Project Configuration File Format > Default Capture Effect Degamma Setting.

Perform gamma correction of the effect if working in linear color space

When cleared

Does not apply degamma correction.

Perform gamma correction of the effect if working in linear color space

When selected (default)

Applies degamma correction so that the midtones of the gradient created using Capture Effect are displayed correctly.

Capture initial frame only

Sets whether to use the only results of the first pass of frame rendering to draw the drop shadow effects.

Capture first frame only

When cleared (default)

The result of each frame rendering is used to render the drop shadow effects.

Use this setting when applying material animation to a pane or when applying animation to the drop shadow effect.

Capture first frame only

When selected

Use only the initial frame rendering result when rendering the drop shadow effects.

This feature can reduce the processing load compared to when it is not selected.
Info

If the Enable Outlines, Enable Brilliance, and Enable Drop Shadows check boxes are all cleared, the settings for Capture First Frame Only, Use Knockouts, Render Effects Only, and Perform gamma correction of the effect if working in linear color space are not saved in the layout intermediate (FLYT) file. As a result, settings may return to their initial state if the file is reopened.

Clear Color

Added a clear color feature to the capture effects.
There is an issue where unintended lines appear between text box panes and capture effects when effects are applied to text box panes.
You can reduce this issue by specifying the same color as the text for Clear Color.

Related Links For more information about this issue, see the troubleshooting here.


Enable Outlines

Sets whether to add an outline to the pane.

The following Borders settings are displayed when the check box is selected.

Borders

Sets how the outline is drawn.

Size

Sets the size of the border.

Rendering Mode

Sets the rendering mode type.

  • Standard (Default)
  • Multiplicative
  • Addition
  • Subtractive

Color

Sets the color and alpha of the border.

Enable Brilliance

Sets whether to add an effect where it appears the pane is emitting light.

The following Brilliance settings are displayed when the check box is selected.

Brilliance

Sets how brilliance is rendered.

Rendering Mode

Sets the rendering mode type.

  • Standard
  • Multiplicative
  • Additive (Default)
  • Subtractive

Color

Sets the color and alpha of the brilliance.
Spread

Sets the spread of the brilliance.

When set to the maximum value, the brilliance completely obscures the background.
Size Sets the size of the brilliance.

Enable Drop Shadows

Sets whether to add a shadow behind the pane.

The following drop shadows settings are displayed when the check box is selected.

Drop Shadows

Sets how the drop shadow is rendered.

Rendering Mode

Sets the rendering mode type.

  • Standard
  • Multiplicative (Default)
  • Additive
  • Subtractive

Color

Sets the color and alpha of the shadow dropped from the pane.
Angle

Sets the angle at which the shadow is dropped.
Distance Sets the distance between the pane and the shadow.
Spread Sets the spread (gradation) of the shadow.
Size

Sets the size of the shadow.
