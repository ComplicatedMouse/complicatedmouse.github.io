Field Node

This section describes the field nodes features.

Random Configures the settings of the Random field, which moves particles about randomly.
GPU Noise Configures the settings of the GPU Noise field, which moves particles in accordance with the Random Trajectory.
Magnetic Force Configures the settings of the Magnetic Force field, which attracts and repels particles so they move as if on springs.
Spin Configures the settings of the Spin field, which sets all particles moving like a twister.
Convergence Configures the settings of the Convergence field, which moves particles so they converge at a specified location.
Add to Position Configures the settings of the Add to Position field, which moves particles in the specified direction.
Simple Collision Configures the settings of the Simple Collision field, which sets simple collisions and makes particles collide together.
Curl Noise Configures the settings of the Curl Noise field, which controls particle movement by placing turbulence in space.
Custom Fields These fields can be added to game applications.
