Link the Angle of View With the Scaling Restrictions

This section describes the Link the angle of view with the scaling restrictions feature that suppresses the effects of changing the camera angle-of-view on the particle display size when using the Camera Distance Scale Restriction feature.

Overview

This feature suppresses the effects of changing the camera angle-of-view on the particle display size when using the Camera Distance Scale Restriction feature.
You can reduce the change in size of particles on the screen when the angle of view is dynamically changing in the game.
The following overview describes the settings ().

Link the Angle of View With the Scaling Restrictions

You can enable or disable the Link the angle of view with the scaling restrictions feature by using the check box ().
The feature requires setting the "standard in-game angle of view" and "current angle of view" in the game application.
