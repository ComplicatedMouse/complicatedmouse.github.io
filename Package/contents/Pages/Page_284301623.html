Tick Class

Premise

The Tick class is for handling system ticks. A system tick is an interval determined by the system. This interval is a monotonic 64-bit signed integer that is represented by the nn::os::Tick class. The system tick rate is determined by the target environment, but there is a feature for getting the frequency (as a number of counts per second). For more information, see Time Management.

The initialization value for the console system tick is undetermined. Do not make any implementations with the assumption that this initialization value is 0 (zero).

Feature Overview

The nn::os::Tick class contains the following member functions. For more information, see the API reference.

Name Description Notes
Tick(int64_t tick) Constructor

Initializes the object with the tick value parameter.
Tick(nn::TimeSpan timespan) Constructor Initializes the object with a TimeSpan value.
GetInt64Value() Get the tick value. Returns the current tick value as a std::int64_t.
ToTimeSpan() Converts the tick value to an nn::TimeSpan. Converts the current tick value and returns it as an nn::TimeSpan.
Other
For more information about other member functions, see the nn::os::Tick class API reference.



Important Information About Using the Tick Class

The system tick value can normally be obtained using the following code.

#include <nn/os.h>

auto tick = nn::os::GetSystemTick();

Even though the calling tick.GetInt64Value() on the tick value that was obtained returns a int64_t typed immediate value, do not convert this value to a float or an int32_t for the purposes of equality or inequality checks. When you convert the values to types that are smaller and less precise than int64_t, information in the original system tick values can get lost and comparison operations can be incorrect.
