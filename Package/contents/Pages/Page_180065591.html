Reflect Alpha Value

This section describes the Reflect Alpha Value feature for controlling the amount of refraction with texture 1 or 2 alpha elements when Color Buffer Refraction is applied.

Overview

The Reflect Alpha Value feature multiplies the image alpha element registered to Texture 1 or 2 by the Color Buffer Refraction coefficient and controls the amount of refraction.
The following overview describes the settings ().

Reflect Alpha Value

To change whether the image alpha element registered to Texture 1 or 2 is affected by the amount of refraction, select or clear the check box ().
