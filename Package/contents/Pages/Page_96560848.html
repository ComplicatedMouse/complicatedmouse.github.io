Convergent Force

This section describes the Convergent Force feature, which is used to specify the force at which particles move toward the position of convergence.

Overview

The Convergent Force feature sets the force at which particles are moved toward the position of convergence.
The greater the value the greater the force at which particles move toward the position of convergence.
The following overview describes the settings ().

Convergent Force

Sets the coefficient to apply in each frame for the distance between the current position and convergence position of the particle.
The particle is translated toward the convergence position by the amount of the calculation result.

  • Value to add to position: Convergence position - Position of the particle at that time) × Convergent force
Info

Press the Particle Time Animation button () to display the Convergent Force (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().
