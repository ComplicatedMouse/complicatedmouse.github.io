Comments

This section describes the feature for editing and checking text comments written to the emitter set file.

Overview

Comments is a feature for editing and checking text written to the emitter set file selected in the workspace.
The following overview describes the settings ().

Comments

The emitter set comments is displayed in the text box. The comments may be edited by clicking on it.
To copy the file path, right-click it and select Copy from the shortcut menu.
