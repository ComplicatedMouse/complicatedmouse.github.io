Connecting Controllers to the SDEV and EDEV.

Overview

To use a controller that is not attached to the SDEV, you must first register that controller with that SDEV through a process called pairing.
You can register the information for up to 10 controllers in the NX system.

There are four ways to pair a controller with a system.

  1. (Only Joy-Con) Attach the Joy-Con to the rail of the system to pair.
  2. Connect the controller and system together with a charging cable to pair.
  3. Register the controller using DevMenu.
  4. (Only When Using the SDEV) Register the controller using a debug controller.

Also use DevMenu to delete the information for a controller registered in the system.

  • Deleting Controller Registration

(Only Joy-Con) Attaching the Joy-Con to the Rail of the System to Pair

  1. Start the SDEV and confirm that either DevMenu is displaying or the application is running.

  2. Attach the Joy-Con (R) to the right rail of the NX system (SDEV or EDEV), or attach the Joy-Con (L) to the left rail.

  3. The pairing process for registration normally completes within one second.
    If the controller that is attached to the system rail is working, the pairing process has completed.
    In the retail product, a message appears informing the user that the controller has been paired.


Connecting the Controller and System Together With a Charging Cable to Pair

  1. Start the SDEV MP and EDEV MP and confirm that either DevMenu is displaying or the application is running.

  2. Attach the SDEV cradle that came bundled with the SDEV MP (EDEV MP).

  3. Attach the Joy-Con (R) and Joy-Con (L) to the Joy-Con charging grip.

  4. Use the controller charging cable to connect the SDEV cradle to the Joy-Con charging grip or the USB Type-C terminal of the Nintendo Switch Pro Controller.

  5. The pairing process for registration normally completes within a few seconds.
    In the retail product, a message appears informing the user that the controller has been paired.

  6. Press a button on the registered controller to connect wirelessly to the system.

Note

This procedure will not work correctly if the Joy-Con charging grip and Nintendo Switch Pro Controller USB firmware is out of date.

Before doing anything else, see this page and update the USB firmware.

Registering the Controller Using DevMenu

  1. Start the SDEV and verify that DevMenu is displayed.

  2. In DevMenu, select the Controller menu.

  3. In the Controller menu, select Start controller pairing. The system will begin waiting for controller pairing.
  4. Press and hold the controller's SYNC Button for more than one second.
    When the controller pairing process begins, the player LED starts flashing in a sequential animation.
  5. When the animation stops and only the player LED representing the controller's own connection port is lit, the pairing has succeeded.
    If the player LED goes off, pairing has failed. Repeat the procedure starting from step 4.
  6. To pair additional controllers, repeat the procedure starting from step 4.

  7. After pairing has completed, select Cancel controller pairing to quit the registration process.

    Note

    If you select Start controller pairing for multiple SDEV/EDEV systems and they are all waiting for pairing to take place, there is a chance that systems and controllers will be paired in unintended combinations.
    After controller pairing has completed, make sure that you select Cancel controller pairing to quit the registration process.

(Only When Using the SDEV) Registering the Controller Using a Debug Controller

  1. Start the SDEV and verify that DevMenu is displayed.
  2. Press and hold the controller's SYNC Button for more than one second.
    When the controller pairing process begins, the player LED starts flashing in a sequential animation.
  3. Press and hold the HOME button on the debug controller (Classic Controller) for one second or longer, and put the SDEV into pairing mode.
    Pairing mode on the console only lasts for approximately five seconds, so make sure to perform step 2 and place the controller in pairing mode first.

  4. When the animation stops and only the player LED representing the controller's own connection port is lit, the pairing has succeeded.
    If the player LED goes off, pairing failed. Restart from step 2.

Note: You cannot pair multiple controllers at the same time. Perform the procedures in steps 2 through 4 for each controller.
