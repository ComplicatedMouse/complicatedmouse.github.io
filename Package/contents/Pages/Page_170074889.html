Mouse (NX Add-On)

Installing the NX Add-On enables the mouse API to be used in the environments described below.

Structure in the SDEV Environment

USB Mouse

The input state is determined from the state of the USB connected mouse.

If more than one mouse is connected, the pressed states is the logical OR of all mice input states, and the cursor position and wheel delta are the sums of all mice input states.

Up to eight mice are recognized, and any other connected mouse is ignored.

If other HID devices are connected via USB, the maximum number of mice that are recognized decreases by the number of other HID devices that are connected.

Input Sent by Input Director

The input state is determined by the state of the mouse as sent from the host PC by Input Director.

Structure in the EDEV Environment

The same as in the SDEV environment.
