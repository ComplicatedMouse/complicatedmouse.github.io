Automatically Track Shadow Alphas to Character Alphas

This section describes the Automatically Track Shadow Alphas to Character Alphas feature, which makes the alpha of the shadow linked to the character track the per-character transformation when an alpha curve has been set for the per-character transformation.

Overview

The Automatically Track Shadow Alphas to Character Alphas feature makes the alpha of the Shadow property of the text box pane track to the alpha of each character when an alpha curve has been set for the per-character transformation.
