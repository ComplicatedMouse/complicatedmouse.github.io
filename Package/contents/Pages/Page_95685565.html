Playback Method - Fit to Lifespan

This section describes the Fit to Lifespan playback method for texture pattern animations, which cycles the animation to match the particle lifespan.

Overview

The Fit to Lifespan playback method animates a texture pattern in a way that matches the animation cycle to the particle lifespan.
It even works where the lifespan is random and the lifespan varies for every particle.
The following overview describes the settings ().

Select to display the Fit to Lifespan () and Pattern Tables () settings. The group has the following settings.

Number of Pattern Tables To Use

Sets the number of patterns to use for the animation.
Related Link For more information, see here.

Info

There are some limitations for Fit to Lifespan.

  • When the particle is set to Infinite lifespan, the animation will not play because one cycle would be endless.
  • When the number of patterns to use is larger than the number of frames of the lifespan, the patterns will be thinned out and played.
    For example, if 16 has been set as the number of patterns to use but the lifespan is 8 frames, patterns will be skipped as follows for the pattern animation.
    Pattern playback order = 0, 2, 4, 6, 8, 10, 12, 14.
