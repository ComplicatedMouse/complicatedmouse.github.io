Filter Mode

This section describes the feature for configuring the processes that are performed on the texture pixels.

Overview

The Filter Mode sets the process that is performed on the texture pixels displayed in the viewer when textures are mapped to particles.
You can use this feature to change the processing method used when determining the texels corresponding to the display size of the polygons, and to control what the effect looks like in the viewer.

Info

Set the mode to Linear when you are using Distance Field.

The following overview describes the settings ().

Filter Mode

Select the filtering mode for the texture from among the following presets.

Linear Interpolates to smooth out pixel edges.
Near Displays each pixel distinctly.
