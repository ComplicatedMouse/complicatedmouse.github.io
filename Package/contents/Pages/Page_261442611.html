SSL Protocols and Encryption Suites

This section describes the information for the SSL protocols and encryption suits supported by the web applet.

Note that the status of this support is different from the support for the SSL library in the Nintendo SDK. For more information, see Features > Network > SSL Library.

The video player also uses the SSL library in the NintendoSDK, so see the SSL library support status.

Supported SSL Protocols

Supported Protocols Key Exchange Algorithms Encryption Algorithms
  • TLSv1.0
  • TLSv1.1
  • TLSv1.2
  • ECDHE
  • RSA
  • AES-CBC (128-bit and 256-bit)
  • AES-GCM (128-bit and 256-bit)
  • SHA1
  • SHA-2 (256-bit and 384-bit)

Encryption Suites

The web applet sends the following encryption suites to the server.

  • TLS_ECDHE_ECDSA_WITH_AES_256_GCM_SHA384 (0xc02c)
  • TLS_ECDHE_ECDSA_WITH_AES_128_GCM_SHA256 (0xc02b)
  • TLS_ECDHE_ECDSA_WITH_AES_256_CBC_SHA384 (0xc024)
  • TLS_ECDHE_ECDSA_WITH_AES_128_CBC_SHA256 (0xc023)
  • TLS_ECDHE_ECDSA_WITH_AES_256_CBC_SHA (0xc00a)
  • TLS_ECDHE_ECDSA_WITH_AES_128_CBC_SHA (0xc009)
  • TLS_ECDHE_RSA_WITH_AES_256_GCM_SHA384 (0xc030)
  • TLS_ECDHE_RSA_WITH_AES_128_GCM_SHA256 (0xc02f)
  • TLS_ECDHE_RSA_WITH_AES_256_CBC_SHA384 (0xc028)
  • TLS_ECDHE_RSA_WITH_AES_128_CBC_SHA256 (0xc027)
  • TLS_ECDHE_RSA_WITH_AES_256_CBC_SHA (0xc014)
  • TLS_ECDHE_RSA_WITH_AES_128_CBC_SHA (0xc013)
  • TLS_RSA_WITH_AES_256_GCM_SHA384 (0x9d)
  • TLS_RSA_WITH_AES_128_GCM_SHA256 (0x9c)
  • TLS_RSA_WITH_AES_256_CBC_SHA (0x35)
  • TLS_RSA_WITH_AES_128_CBC_SHA (0x2f)
