Particle Offset

This section describes the Particle Offset feature for adjusting the overlap of particles with objects in the game.

Overview

The Particle Offset feature allows you to set an offset to a position based on the particle's depth information and the camera direction.
You can adjust particle overlap by setting this feature on a per-emitter basis.
The following overview describes the settings ().

Particle Offset

Select the particle offset method from among the following presets.

Offset Z value Controls the depth value of the particle.
The size of the particle does not change if the path is something like Offset toward camera.
Offset toward camera Offsets the position where a particle is rendered either forward or backward relative to the camera.
Fixed Size Camera Offset Pass negation occurs when offsetting particle render position either forward or backward relative to the camera.
None This feature is not used, so there is no extra processing load.

Example Configurations

The following image compares what happens to a cube-shaped particle with a letter F applied to it when each preset is specified.
In this comparison, a plate-shaped particle with a checkered pattern is moved forward and backward from the background to overlap with the letter F particle.

A Offset Z value The depth of the particle is set to 20 due to the specified offset.
The particle does not overlap with the particle in the background, and the size of the particle does not change.
B Offset Toward Camera

The particle is moved 20 units in the Z direction in the world coordinate system due to the specified offset.

The particle does not overlap with the particle in the background. However, perspective is applied to the particle, and so the size of the particle changes.
C Fixed Size Camera Offset The particle is moved 20 units in the Z direction in the world coordinate system due to the specified offset.
The size does not change because pass negation occurs when adjustments are made at the vertex level using the screen coordinate system.
D Do not use The particle overlaps with the particle in the background.
