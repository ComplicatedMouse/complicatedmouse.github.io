Z-Fighting

Describes z-fighting.

Z-fighting is a phenomenon that occurs when several opaque polygons are rendered at the same position (with z-compare and z-write enabled). As a result, the depth relationships of pixels rendered at that position become indeterminate, as shown in the following image.
EffectMaker now includes a z-fighting suppression () feature to combat z-fighting.

Z-Fighting Suppression Describes the feature for suppressing z-fighting, which reduces the flickering that occurs when a particle is drawn at the same position as another polygon.
