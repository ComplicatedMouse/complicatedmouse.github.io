Shader Definitions Property Window: Shader Definitions Page

This page is used to check the information of the selected file.

A Displays the code page for the shader source code.
<shader_definition_info> found in the intermediate file is specified for the code page.
For more information about <shader_definition_info>, see the separate documentation about the intermediate file format.
B The path to the shader configuration file used to generate the shader definition file.
Displays the filename or the relative path from the root path, set with the shader converter.
C Displays the shader (GLSL file) used by the program and the program included in the shader definition file.
D Displays the shader source code of the vertex shaders, fragment shaders, geometry shaders, compute shaders, tessellation control shaders, and tessellation evaluation shaders of the program selected from the list described in C. This information is managed by using tabs.
