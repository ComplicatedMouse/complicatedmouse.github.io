LayoutEditor

This section introduces the new features in LayoutEditor.

You can no longer edit muted curves.

You can no longer edit muted curves using CurveEditor in LayoutEditor.
Larger icons are also now used in the Hierarchy window and the Curve Editor to indicate that audio is muted for a curve.

The Apply Parent Translations to Children icon is now a button.

The Apply Parent Translations to Children icon that was shown at the bottom of the Layout window has been changed to a button that can be clicked to select and clear the option.

Runtime Libraries

A feature has been added to allocate area for the user pointer for each pane.

A feature has been added to allocate area for the user pointer for each pane.

To use it, enable isSystemDataUserPointerPerPaneEnabled for nn::ui2d::Layout::BuildOption and build a layout.

You can then use the user pointer using nn::ui2d::Pane::GetSystemExtDataForUserPointer() or nn::ui2d::Layout::GetSystemExtDataForUserPointer() in the built layout.

nn::ui2d::Layout::GetSystemExtDataForUserPointer() calls GetSystemExtDataForUserPointer() for the root pane of the layout.
