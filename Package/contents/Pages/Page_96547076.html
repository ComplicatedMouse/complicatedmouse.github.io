Amplitude for Element 0 to Element 3

This section describes the feature for adjusting the amplitude of waveform elements 0 (±) to 3 (±).

Overview

The element 0 (±) to element 3 (±) settings allow you to weight the amplitudes sent in Noise Strength.
Larger values increase the amplitude of the elements used to generate GPU noise, and smaller values decrease the amplitude.
The following overview describes the settings ().

If the Element settings feature has been enabled, the Set a weight to each element's variability group () is displayed. The group has the following settings.

Element 0 (±) to Element 3 (±)

Sets the weight to apply to the amplitude of each of the waveform elements used to generate GPU noise relative to the noise strength setting.
The following formula is used to calculate the amplitude of each element.

  • Element amplitude = Noise strength × Weight of element’s amplitude
