Plug-In for Setting Shape Animations

You can use this plug-in to set the vertex attributes to be updated during a shape animation. The vertex attributes updated by default are: vertex coordinates, normal vectors, tangent vectors, and binormal vectors.
These attributes, however, are not applied on the Maya screen.

Usage

Select Nintendo > Set Shape Animation to open the window.

Selected blendShape nodes are displayed under Selected Blend Shape.

Info

If four or more nodes are selected, only the number of nodes selected is displayed on the right edge without displaying their names.

Setting Method

When one or more blendShape nodes or the transform nodes of the object set by BlendShape is selected, the current settings for the selected nodes are displayed in the Set field.
If you select multiple nodes or groups, the setting for the first selected node is displayed.

Settings are changed by controlling each item in the window.
If multiple selections were made, selected nodes can be changed at the same time. (Only modified attributes are applied.)

You can select a different node by selecting another node without closing the window.

Position Enable this option to update vertex coordinates. This option is enabled by default.
Normal Enable this option to update normals, tangents, and binormals. This option is enabled by default.
Tangent vectors and binormal vectors are only updated when normal mapping or an anisotropic shader is being used for the base shape.
Color Enable this option to update vertex color. This option is disabled by default.
If no vertex color is set for the base object, vertex colors are not updated even if this option has been enabled.
Info

If Position, Normal, and Color are all disabled, shape animation for the selected blendShape nodes cannot be exported.
