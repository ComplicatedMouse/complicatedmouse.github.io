Glare

This section describes the Glare feature, which blurs the effect color into one or more pixels to create glare.

Overview

Glare is a screen effect for reproducing the optical phenomenon that occurs when a light source is viewed directly by the eye or camera.
In EffectViewer, for example, a luminescent effect is applied just to the brightest areas of an image (areas where the RGB values are greater than 1.0) to make the colors blur accordingly in those areas.

Glare-Related Features

Related Links The following features are related to glare effects.

Color Scaling Color scaling is a feature that is used for setting values by which colors in the Color tab can be multiplied.
Color Editing UI Configures the selected color and alpha values.
Setting RGB values greater than 1.0 makes it possible to create glare effects in EffectViewer.
