Property Editing

Open the Property window using one of the following methods.

  • Select a pane, and from the shortcut menu, select Properties.
  • Double-click a pane.
  • Select a pane and then press CTRL+P.
  • From the main menu, select Window > All Property Windows.

Redoing

LayoutEditor supports Undo and Redo operations (but with some exceptions).
If you have entered incorrect settings, you can undo them, or reapply (redo) the settings after undoing them.

Undoing an Operation

Make the layout window the active window and then perform either of these actions to undo an operation and restore the previous state.
By repeating the Undo action you can take back a series of operations.

  • Undo From Menu
    From the menu, select Edit > Undo.
  • Undo Using Shortcut
    Press CTRL+Z.

Redoing an Operation

Make the layout window the active window and then perform either of these actions to redo an operation that was undone.
This action can also be used to redo a succession of undone operations.

  • Redo From Menu
    From the menu, select Edit > Redo.
  • Redo Using Shortcut
    Press CTRL+SHIFT+Z.

Saving Files

Specify a filename and save.
The extensions for layout files are .flyt and .flan.

Save All

Use either of these operations to save all of the created layout data.
If data is being saved for the first time, a dialog box appears and prompts you to enter a filename.

  • Save All from Menu
    From the menu, select File > Save > All.
  • Save All Using Shortcut
    Press CTRL+S.

Save As

Use either of these operations to save the created layout data using a different name.
If data with the same name already exists, a dialog box opens to confirm that you want to overwrite that data.

  • Save All from Menu
    From the menu, select File > Save As > All.
  • Save All Using Shortcut
    Press CTRL+SHIFT+S.
