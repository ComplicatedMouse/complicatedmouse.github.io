Overview

The Photoshop plug-in is a group of tools for creating texture intermediate files using Adobe Photoshop.

Notation

This description uses the following simplified notation.

Official Name Simplified Name
Adobe Photoshop Photoshop

Operating Environment

The Photoshop plug-in has been confirmed to run under the following environments.

Photoshop Photoshop CC 2018 (19.1.9)
Photoshop CC 2019 (20.0.10)
Photoshop 2020 (21.2.12)
Photoshop 2021 (22.5.3)
Photoshop 2022 (23.0.2)
Operating System Microsoft Windows 10 Pro (64-bit)
Info

Photoshop 2022 is supported by NintendoSDK 14.1.0 and later.


Setup Method

Info

If you install the NintendoWare package using the Nintendo Dev Interface (NDI), the Photoshop plug-in will be automatically set up if Photoshop is installed on your PC.
In addition, the Photoshop plug-in will be removed when uninstalling the NintendoWare package.

Note

The Photoshop plug-in will only be set up if you install the NintendoWare package with administrator rights.
If not, use the following procedure to install the plug-in.

Info

If multiple NintendoWare packages are installed on one PC, the Photoshop plug-in will be set up for the last installation.
To change the Photoshop plug-in to set up, use the following procedure.

Exit Photoshop and Adobe Bridge if they are currently running.

To set up plug-ins, execute the following batch files located in the %NINTENDO_SDK_ROOT%\Tools\Graphics\PhotoshopPlugins folder, with administrator privileges.

SetupPluginsPhotoshop-All.bat (To set up plug-ins for all installed versions of Photoshop)

SetupPluginsPhotoshop-Cc2018.bat (To set up just for Photoshop CC 2018 (version 19))
 
SetupPluginsPhotoshop-Cc2019.bat (To set up just for Photoshop CC 2019 (version 20))
 
SetupPluginsPhotoshop-2020.bat (To set up just for Photoshop 2020 version 21)

SetupPluginsPhotoshop-2021.bat (To set up just for Photoshop 2021 version 22)

SetupPluginsPhotoshop-2022.bat (To set up just for Photoshop 2022 version 23)
Info

Use the following procedure to set up the plug-ins manually without a batch file.

Place a shortcut to the %NINTENDO_SDK_ROOT%\Tools\Graphics\PhotoshopPlugins\NintendoPhotoshopPlugins folder named NintendoPhotoshopPluginsShortcut in the following directory.

(Photoshop installation directory)\Plug-ins\

Place a shortcut to the %NINTENDO_SDK_ROOT%\Tools\Graphics\PhotoshopPlugins\NintendoPhotoshopScripts folder named NintendoPhotoshopScriptsShortcut in the following directory.

(Photoshop installation directory)\Presets\Scripts\

If you want to display thumbnails of texture intermediate files using Adobe Bridge, copy NintendoFtx.8bi (found inside the %NINTENDO_SDK_ROOT%\Tools\Graphics\PhotoshopPlugins\NintendoPhotoshopPlugins folder) to the following location.

(Adobe Bridge installation directory)\Plug-ins\

This concludes the set up process.

If set up correctly, Nintendo Export Texture is available under File > Automate in the Photoshop main menu, and Nintendo is available under Filters.

Assigning Keyboard Shortcuts

To assign keyboard shortcuts for the Photoshop plug-in, select Edit > Keyboard shortcuts on the Photoshop main menu.

We recommend assigning keyboard shortcuts for File > Automate > Nintendo Export Texture and Nintendo ExportTexture Settings, because both are used frequently.

Setting a Smart Filter

You can apply a filter non-destructively by using a smart filter.

To use a Photoshop plug-in filter as a smart filter, you must first select File > Script > Browse, execute the following script, and then click Yes. (This setting will not be required again when starting Photoshop in the future.)

Download Scripting Listener Plug-In for Windows from the Adobe website.

Extract the ZIP file.

The script is in the following location.

Scripting_Win64\Sample Scripts\JavaScript\EnableAllPluginsForSmartFilters.jsx

When used as a smart filter, the Photoshop plug-in filter is applied after converting layers to smart objects using Filters > Convert for use as smart filter.
