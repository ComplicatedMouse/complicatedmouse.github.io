User Data

This tab sets additional user-specific information on a per-emitter set basis.
The data can be used for game application features and workflow purposes.

Bits

The Bits group () is used to set user data with flags.
This group has the following settings.

Bits 0-15

A group of 16 flags that can be individually selected using the check boxes ().

Numbers

The Numbers group () is used to set user data with sliders.
This group has the following settings.

Number 1 & Number 2

Set the value as a positive integer with either the dial () or text box ().

Example Configurations

The following example shows user data being used in a game application.

A Flag [0] bit = ON　 Enemy characters are placed in the game at the positions of the particles of this emitter set.
B Number = 128
 Used as an ID number to specify the type of the enemy character to follow in the game.
