Behavior When Colliding

This page describes Behavior When Colliding, the feature to configure behavior when a simple collision and particle collide.

Overview

Behavior When Colliding enables you to use presets the configure the behavior when a particle collides with a collision.
You can have particles do things like die or bounce when you set their behavior for colliding with a collision plane.
The following overview describes the settings ().

Behavior When Colliding

Select among the following presets for the behavior to use when particles collide.

Reflection Particles bounce, reversing direction, when they collide.
Die.

Particles expire when they collide.

If child particles are configured to emit when parents reach the end of their lifespans, they are emitted when parent particles collide and expire.
Process with the custom shader

Particles behave according to the code in the custom shader when they collide.

Disable Run on CPU in the Basic Settings tab for the emitter node because calculations occur using the GPU-SO emitter type.
