How To: Create Layers

Please refer to the graphics samples as an example for how to create layers.  For NVN, this initialization sequence can be found in TutorialBaseClass.Nx.cpp in the NvnTutorial library.

1. Initialize VI library.

nn::vi::Initialize();

2. Open display and obtain the display handle.

nn::vi::Display* pDisplay;
nn::vi::OpenDefaultDisplay(&pDisplay);

3. Create layer.  Note the visibility is set to true upon creation.  However, until the application renders content to the layer, it will be blank and ignored by the display.

nn::vi::Layer* pLayer

nn::vi::CreateLayer(&pLayer, pDisplay);

4. Get the NativeWindowHandle for use with a graphics API.  Please refer to the graphics samples to demonstrate how to use the NativeWindowHandle.

nn::vi::NativeWindowHandle nativeWindow; 
nn::vi::GetNativeWindow(&nativeWindow, pLayer);
