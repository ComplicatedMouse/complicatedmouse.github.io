Magnetic Force (±)

This section describes the Magnetic Force (±) feature, which is used to specify the strength of the magnetic force.

Overview

The Magnetic Force (±) feature can be used to set the magnitude of the magnetic force.
Increasing this value increases the magnitude of the force that draws the particle to the position of the magnetic force.

The following overview describes the settings ().

Magnetic Force (±)

Sets the magnitude of the magnetic force, using ±. Specify a value in the negative direction for particles to be repelled by the magnetic force.
The further a particle is from the position of the magnetic force, the faster its speed toward the axis.

Info

Press the Particle Time Animation button () to display the Magnetic Force (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().
