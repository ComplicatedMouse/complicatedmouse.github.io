Emitter Transform: Rotation(±)

This section describes the Rotation(±) feature for setting the angle when placing emitters.

Overview

The Rotation(±) feature sets the angle when placing emitters.
The following overview describes the settings ().

Rotation

Set the angle to use as a standard when placing emitters using the XYZ emitter coordinate system.
You can use the Random setting () to set an area where the placement angle is random.

Info
Related Links For more information about randomizing emission of every particle, see Update Random Range.
