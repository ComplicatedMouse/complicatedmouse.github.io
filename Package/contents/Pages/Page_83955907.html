Emitter Set - Auto-Translate

This tab configures the Auto-Translate feature, which allows you to check how effects look on the Viewer when emitter sets are actually moved.

Auto-Translate

Auto-Translate group () is a set of features for checking how effects look in the viewer when emitter sets have been moved.
This group has the following settings.

Auto-translate type　

A feature for selecting the auto-translate type.
XZ translation radius Sets the radius of the arc over which the emitter set is translated.
XZ translation speed Sets the speed of the emitter set when it moves on the XZ plane.
Translation Distance Sets the distance when the emitter set moves in a straight line.
Translation Speed Sets the speed when the emitter set moves in a straight line.
Straight Line Angle Sets the direction when the emitter set moves in a straight line.
Y Swing Amplitude Sets the amplitude of the up and down movement when the emitter set is translated vertically.
Y swing speed Sets the speed of the up and down movement when the emitter set is translated vertically.
Z roll speed Sets the speed of rotation when the emitter set is rotated around the z-axis.
Info

When you use the Auto-Translate feature of the Preview node, the Model Links settings made on the SRT Transform tab are ignored.
