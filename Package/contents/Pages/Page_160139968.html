Hats

This section describes how to create hats.

Verifications Before Creating a Hat

To display Mii characters wearing a hat, you must display the face model for hats that has the upper part of the head removed.

The developer displays the face model for hats using the Face Library. For more information, see the face model types section in the Face Model Composition page.

An example of the face model for hats is below.

Examples of wearing hats are below.

Baseball Cap (Visible Bangs) Knit Cap (Hidden Bangs)

Creating a Hat

This section describes the procedure for creating a hat.

First, we describe the procedure where the designer creates a model of a hat and the developer implements it.

Creating the Hat Model Data (Designer Task)

Samples that provide standards when creating a hat are available. Display and check Cap_template in your 3DCG software.

Cap_template has two hats, designed to fit all Mii hairstyles. As a result, when creating your own hat, make sure that the hat is not smaller than these two hats.

When creating a hat such as a baseball cap where some bangs are visible, refer to Cap_template00. When creating a hat such as a knit cap where bangs are not visible, refer to Cap_template01. There is a baseball cap that has been created using Cap_template and the Cap_sample knit cap sample. You can also reference these hats.

Note the following points when creating hats.

  • The hat model is positioned slightly above the origin point. Create the hat using this position. After creating the hat, make sure that you align the pivot point to the origin.
  • Attach a polygon to the bottom of the hat. There is a risk that a hair section will be visible when viewed from below.

Displaying the Hat Model (Programmer Task)

Use the Face Library to display the Mii face model, and display the hat to match the face model.

Attach the hat to the face model for hats with the upper part of the head removed. To create a face model for hats, specify nn::mii::CreateFlag_Hat for CreateFlag in the nn::mii::CharModelInfo, which is passed during initialization of nn::mii::CharModel. You can get the parameters used to display a face model for hats by specifying nn::mii::CreateModelType_Hat for nn::mii::CreateModelType in nn::mii::CharModel::GetDrawParam().

The hat is attached with reference to the Null value set_hair prepared in the face outline model. set_hair is the Null value indicating the position of the hair model. Use the nn::mii::CharModel::GetHatPartsTranslate() function to get the translation information. For information about set_hair, see the verifications before creating headwear on the Headwear page.

Verifications After Creating a Hat

Place the hat on all of the hairstyles of Mii characters in the actual application for verification purposes.

If the hat is too small, check again that the hat is not smaller than the hats in the Cap_template sample, which provides the standards for creating hats.
