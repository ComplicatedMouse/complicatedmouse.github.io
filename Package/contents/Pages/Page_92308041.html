NintendoWare Effect News

This document is a graphical introduction to the features that were added in NintendoSDK 0.7.0.

Viewer Startup Files

The startup file for EffectViewer, which is the viewer for checking the results of editing with EffectMaker, is provided in versions for each platform.

Limitations of EffectViewerNw4f

The EffectViewerNw4f viewer has a problem with crashing when FMDB files registered to the Model Settings node of EffectMaker are transferred.
Use the following workaround for this problem.

  1. Close EffectMaker and the viewer ahead of time.
  2. Run \NintendoSDK\Tools\Graphics\3dTools\SwitchToNwBinaryMode.bat to create MarkerCreateOldBinary.
  3. Copy this created MarkerCreateOldBinary to \NintendoSDK\Tools\Graphics\EffectMaker.
  4. Start EffectMaker and EffectViewerNw4f and connect them together.
  5. Register the FMDB file by selecting the Model Settings node, and then selecting Specify File.
  6. Confirm that the registered model is displayed in the viewer.

Changed the Controls to Reorder Nodes

The mouse drag operation to move nodes in the workspace has been changed to require an ALT+mouse drag.
This was done to prevent accidental and unintentional changing of the rendering order caused by operations that rearrange the order of nodes.

Related Links For more information, see the following pages.

Added the Standard Gravity Feature

Added a Standard Gravity button in the Emitter tab that calculates and sets a physically correct gravity value based on the setting in the Options Project Settings.
This feature makes it easier to enter physically correct values for gravity than was previously possible when effect designers made individual adjustments by eye.
Deciding on a unique, uniform value for gravity also enables consistency among the movements of the various effects in the game.

Changed the Command-Line Interface Options

The command-line interface of the binary converter used by EffectMaker was changed to make it more consistent with the specification for the NintendoSDK command-line interface.
