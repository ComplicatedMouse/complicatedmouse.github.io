Particle Shape

This section describes the feature for setting the shapes in which particles are displayed.

Overview

For the particle shape, you can configure either Particle or Primitive.
By changing the particle shape, you can use a variety of different shapes to express a wider range of effects.
The following overview describes the settings ().

Particle Shape

Select one of the following presets to specify which shape to use for particles.

Particle Flat, plate-shaped polygons generated by the effect runtime.
Primitive Polygons with free shapes created using the DCC tool.
This setting is also used for polygons that have thickness.
