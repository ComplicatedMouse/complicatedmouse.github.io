PROD MODE Restricted Functions

The following functions are restricted on retail units and on development hardware initialized with the PROD MODE firmware.

Note

This list shows only the functions included in the public packages. The functions included in some limited-publication packages are not shown.

Prohibited Functions

The following functions are prohibited in the retail product ROM, and also affect behavior in PROD MODE.

time

nn/time/time_ApiForClaimPeriodicBenefit.h

The following functions in the header.

  • nn::time::PeriodicBenefitClaimContext::GetEndOfPenaltyAtLastClaimForDebug()
  • nn::time::PeriodicBenefitClaimContext::GetSpanToEndOfPenaltyAtLastClaimForDebug()

GetEndOfPenaltyAtLastClaimForDebug() became available starting in NX Add-On 6.x.x but is scheduled for deprecation in NX Add-On 8.x.x.

Starting in NX Add-On 8.x.x, use GetSpanToEndOfPenaltyAtLastClaimForDebug() instead.

Reference

  • You can use either of the following methods to check for prohibited APIs.
    • Run AuthoringTool with the --error-unpublishable option to create an error log.
    • Open the NSP file in AuthoringEditor to display errors preventing master ROM submission.
