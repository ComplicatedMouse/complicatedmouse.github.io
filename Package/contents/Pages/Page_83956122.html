Linear Edit Mode

This section describes the Linear Edit Mode feature of EffectMaker.

Overview

Linear Edit Mode is an option setting that tailors the EffectMaker color editing environment, and the display states of textures, to the gamma correction for accurate lighting calculations by the shaders.

Linear Edit Mode

Enabling Linear Edit Mode switches the following features to linear edit mode.

Get Square of RGB A UI is displayed in linear edit mode.
This feature squares the brightness of the registered texture to approximate the result of linear conversion.
Color Editing UI The color editing UI switches to a gamma-corrected state.
Texture Preview Corrects the texture preview, depending on the format and according to the linear workflow.

Color UIs

Enabling linear-edit mode switches color-related UIs to gamma-corrected status.
The gamma correction is applied to the following UIs.

  • Color Slider
  • Color Picker
  • Color Animation UI

Texture Preview

When linear-edit mode is enabled, an appropriate gamma correction is applied to linear textures, and the way they appear in the viewer is also corrected. Also, when using linear-edit mode, a warning is displayed if non-linear textures are loaded. From Options, you can choose not to display the warning.

Linear Texture

Describes linear textures.

Types of Linear Textures

In most cases, the following three texture formats are used for effects.
The way textures are handled in EffectMaker also changes based on these formats.

Format Modes Description
unorm Normal Normal texture.
linear unorm Linear Linearized when exported from PhotoShop.
sRGB Linear sRGB texture. Linearized when fetched in shader.
