Element Settings

This section describes the feature for configuring the waveform that defines the trajectories of moving particles.

Overview

The waveform that defines the trajectory of movement of a particle is formed from a combination of four waveform components.
Use the Element settings feature to separately configure the period and amplitude of the four waveform components (elements 0 to 3).

In the following figure, the four waveforms elements 0 to 3 are added together to form a single waveform.

Info

To make the particles move randomly from the start of playback, the initial phase of each waveform element is always staggered.

The following overview describes the settings ().

Settings for Each Component

Enable and disable waveform settings for each component by selecting or clearing the check box ().
Enabling this feature displays the related groups. The groups have the following settings.

Period for Element 0 to Element 3

The Ratio relative to the base period of each element group () of settings.
This feature sets the period (the number of frames) for each waveform element.

Amplitude for Element 0 to Element 3

The Set a weight to each element's variability group () of settings.
This feature sets a weighted amplitude for each element by setting a value by which to multiply the amplitude.
