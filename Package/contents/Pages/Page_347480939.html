Effect News

This is a graphical introduction to the features added in NintendoSDK 6.1.0.

EffectMaker

This section introduces new features that have recently been added to EffectViewer.

Model Linking Supports Visibility Animation

Visibility animation can now be used with model linking.
Showing and hiding models to link in the viewer, can be animated using a visibility animation file (FVBB).

Bone Visibility AnimationDescribes animation that switches the visibility of bones between showing and hiding them.

Added Move Forward in the Specified Direction to the Auto-Translation Type

In the Auto-translation type in the Preview Setting, you can now use Move forward in the specified direction. for repeated translation moving straight in the specified direction.
You can specify the direction, distance, and speed of the emitter set or model sent to the viewer, and have it repeatedly move in a straight line.

Move Forward in the Specified Direction

The emitters will be automatically moved along the circumference of a circle of the specified size. The emitters will be moved forward in the specified direction.
