Random Phase

This section describes the Random Phase feature for setting the magnitude of the flux.

Overview

These waveform settings can be configured when sine wave is selected for the waveform type.
This section provides an overview of the settings ().
When you select either setting in Apply Flux, the Apply Flux group () of settings appears. The group has the following settings.

Random Phase

Use the check box () to switch between randomly shifting the position where each particle begins to fluctuate.
Random phase formula: Initial Phase + Period × Random number.
