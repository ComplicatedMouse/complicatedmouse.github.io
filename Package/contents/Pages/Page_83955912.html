This section describes the feature for using the viewer to check those settings of the Particle feature of the Emitter node that can be controlled in the game application.

Multiply Lifespan Settings by Scale

The Multiply Lifespan Settings by Scale group () is a feature for configuring settings for the scale of all emitter shapes in the emitter set.
This group has the following settings.

Lifespan

Multiplies the Lifespan of all particles in the emitter set by the specified value.
