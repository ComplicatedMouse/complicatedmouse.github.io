What Are Remote Video and Remote Input?

In Target Manager 2, you can use features like remote video that display images on a PC that are displayed on the target, or remote input for operating a target from a PC.
This document describes remote video and remote input.

Remote Video

You can use remote video to display video output on Target Manager 2 from an application that is currently running. You can also save that video as an MP4.

Note that the remote video feature depends on the Video Capture feature. When the Video Capture feature is disabled, Target Manager switches to an alternate mode but is subject to the following limitations. This alternate mode is only available with the SDEV.

  • The frame rate drops significantly for videos displayed on Target Manager 2.
  • The video recording feature using Target Manager 2 is disabled.
  • In TV output mode, the screen is black.
    • As a workaround, you can use DevMenCommand to run a command like debug set-operation-mode-policy handheld to send the output to the LCD screen.

Also note that all applets disable the video capture feature. Although DevMenu is an applet, DevMenuApplication, which can be started from HomeMenu, is an application rather than an applet, so the video capture feature is enabled.

Info

In remote video mode, the overlays for changes in volume and the like are not rendered.
In the alternate mode, the overlays also are rendered.

Info

In Nintendo Target Manager 2 version 12.2.0.0 and later, a recording prohibited icon is displayed in the status bar of the remote video window when the Video Capture feature is disabled.

Info

By specifying debug set-force-disable-app-chain-la-recording false in DevMenuCommand (available in NX Add-On 14.1.0 or later environments), library applets started from the app can also be rendered on the EDEV remote video.

For more information about this command, see Development Support Features.


Functionality

Display Magnification

For the magnification, you can choose whether to display the video in its actual size, or fit it to the window.

In Nintendo Target Manager 2 version 14.0.0.0 and later, you can adjust the slider to any multiplier.
If touch operations are enabled, you can also adjust it with the mouse wheel.

Muting Audio

You can mute the application audio played back on your PC.

Saving Videos

For saving a video, you can record the last 30 seconds of a video or any segment using the Start and Stop recording buttons.

Info

In Nintendo Target Manager 2 version 13.2.0.0 and later, the recording time is displayed in the status bar on the bottom of the remote video window when recording with the Start and Stop buttons.

You can configure the following options for saving videos from Target Manager 2 > Tools > Settings.

  • Naming Conventions
  • Output Folder
  • Action When Saving

Remote Input

With remote input, you can operate a target from Target Manager 2.
To operate the controller, click the controller image displayed in the view or connect it to the PC.
That input is recognized as either the input coming from the existing controller or the additional controller, depending on the connection status of the controllers.
In addition, you can perform touch operations and mouse operations on the remote video screen.
To switch between touch operations and mouse operations, use the buttons on the toolbar on the remote video screen.

Input Director starts automatically when you start the screen.
You cannot use this feature if Input Director is not found.

Depending on the Input Director settings, moving the pointer off the screen while this window is open could activate mouse capture mode. To disable capture mode, in the Input Director options, clear the Toggle when mouse pointer moves to edge of screen check box.

Functionality

Adding Controllers

Info

This functionality is available in Nintendo Target Manager 2 version 13.2.0 and later.
In previous versions, switching the controller in the combo box changes the controller that is operated.

This feature adds the controller displayed in the combo box menu to the targets.
Assignment for the added controller follows normal Npad specifications. For more information, see Npad.
If you want to use a Joy-Con dual-controller grip state, add both a left and right Joy-Con.

If the target is already at the maximum number of controllers when adding a controller, or when the target controller is not supported, nothing happens and the controller is not added.

Note

If DebugPad is connected to the development hardware, DebugPad is not displayed on the remote operations screen unless you disconnect the target, then operate the DebugPad connected to the development hardware upon reconnecting.
In this state, even if you add a DebugPad from the remote operations screen, it is not actually added.
It is displayed again if you restart the target and reconnect.
This behavior will be improved in a future version.

Disconnecting Controllers

Info

This functionality is available in Nintendo Target Manager 2 version 13.2.0 and later.

You can disconnect the target controller by clicking the Disconnect icon ().
Note that the physical controller connected to the target is also disconnected.

However, the behavior is different for the following cases, so see each explanation.

  • When you perform the disconnect operation with the Joy-Con attached to the console
    • The controllers are disconnected, but they are reconnected if you operate the Joy-Con controllers attached to the console
  • When you perform the disconnect operation while the DebugPad is connected
    • The controller is not disconnected, and you can still operate the DebugPad connected to the console

Button Operations

You can click and operate the button that appears.

Pressing Multiple Buttons Simultaneously

  1. Hold down the CTRL key and simultaneously click the buttons you want to press.
  2. Release the CTRL key.
  3. When you click one of the buttons you want to press simultaneously, all buttons selected with CTRL are entered.
  4. The selection is cleared when you click a button other than one you wanted to press simultaneously.

Keeping a Button Held Down

If you select a button while holding down SHIFT, that button is maintained in the selected state.
To release that button, select it again while holding down SHIFT.
You can release all buttons from the pressed state by clicking Deselect all in the toolbar or by pressing ESC.

Holding Down the Stick

By right-clicking the stick, you can hold down the stick.

Touch Operations

This functionality is available when touch operations are enabled in the Remote video view.
While holding down the left mouse button on the Remote video screen, the location of the pointer is detected as receiving touch input.

Mouse Operations

This functionality is available when mouse operations are enabled in the remote video view.
If there is a mouse pointer on the Remote video screen, the various mouse operations are transferred to the target.

Note

The locations of the mouse pointer on the Remote video screen and on the target may be misaligned.
If the pointer positions becomes misaligned, use the following procedure to restore the alignment.

  1. Move the mouse pointer off the Remote video screen and take the Remote video window out of focus.
  2. Move the pointer back into the Remote video screen.

Keyboard Operations

Info

This functionality is available in Nintendo Target Manager 2 version 11.3.0.0 and later.

This functionality is available when keyboard operations are enabled in the Remote video view.
If the Remote video view is active, keyboard operations will be transferred to the target.

Note

In Nintendo Target Manager 2 versions earlier than 13.0.0, the mouse is treated as connected while keyboard input is being transferred.
If mouse operations are not enabled, the system treats the previous location of the mouse pointer as the current location of the pointer.

Controller Operations Using Keyboards

Info

This functionality is available in Nintendo Target Manager 2 version 11.1.0.0 and later.

You can operate mapped controller buttons and sticks using keyboard input on the remote video screen and remote input controller display.
Keyboard operations are disabled by default with Remote video, so select the Controller Operations Using Keyboards button in the toolbar.
You can check and configure mappings in the remote operations tab under Tools > Settings.

In Nintendo Target Manager 2 version 13.2.0 and later, the behavior changes as follows.

  • The controller for which the keyboard icon () on the remote operation screen is selected becomes the controller that is operated.
  • For remote video, the NpadID, Handheld, or DebugPad controller specified in the combo box becomes the controller that is operated.
    • However, operations cannot be performed if you select a controller that is not connected.

Using External Controllers

You can perform remote input using an external controller that is connected to the PC.
For more information, see Input Director.
