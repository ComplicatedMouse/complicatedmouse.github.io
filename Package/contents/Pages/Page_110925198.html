Question

The Parts Pane is not displayed.

Answer

Check whether the parts layout size is zero.

When an animation is attached, part size calculations occur in the frame 0 state.
If it is a scale animation, or otherwise has scaling effects, we recommend creating your data so that zooming starts from a negative frame and reaches the normal state in the 0th frame.
From NW4F version 2.5.0 on, when you save a parts layout with a size of zero in LayoutEditor, there is a check and a warning is displayed.
