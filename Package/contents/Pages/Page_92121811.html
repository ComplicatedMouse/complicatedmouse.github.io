EffectViewer Operations

The following table provides a summary of EffectViewer (Windows version) operations.

Operations With the Classic Controller
R Button
Outputs the drawing result every four frames as a PNG file to ScreenCap () in the same hierarchy as the Viewer's EXE file.
Camera Operations With the Classic Controller
Left stick Rotates around the look-at point.
Right stick Moves in parallel to the XZ plane.
X/Y Button Zooms in and out.
L + right stick Moves in parallel to the Y direction or left-right direction.
L Button Shows and hides Viewer Info.
HOME Button Resets the camera.
Mouse and Keyboard Operations
Left-click + drag Rotates around the look-at point.
Middle-click + drag Moves the line of sight parallel to the up-down or left-right direction.
Right-click + drag Zooms in and out.
Wheel button Zooms in and out.
L Shows or hides the debugging information.
S Resets the camera to the default state.
G
Related Links Enables or disables the Show Ground feature.
This debugging feature shows or hides the ground.
E
Related Links Enables or disables the Show Parent Emitter Shapes debugging feature.
This debugging feature shows or hides parent emitter shapes.
C
Related Links Enables or disables the Show Child Emitter Shapes debugging feature.
This debugging feature shows or hides child emitter shapes.
F
Related Links Enables or disables the Display Field Information debugging feature.
This debugging feature shows or hides fields.
Keypad values 4 / 8 Zoom in and out when NUM LOCK is disabled.
Mouse and Keyboard Camera Operations
R

Outputs the drawing result every four frames as a PNG file to ScreenCap () in the same hierarchy as the Viewer's EXE file.
SHIFT+R
Outputs the drawing result every four frames as a PNG file to ScreenCap () in the same hierarchy as the Viewer's EXE file.
