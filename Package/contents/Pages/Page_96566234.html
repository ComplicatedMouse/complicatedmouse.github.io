Inheritance Settings: Color

This section describes the Color inheritance setting for child emitters.

Overview

The Color inheritance setting is a feature that makes child emitters inherit the color value of the parent emitter in the highest node.
The following overview describes the settings ().

Color

The feature to inherit Color from the parent emitter is enabled or disabled using the check box ().
When this check box is selected, the parent emitter's Color value is multiplied by the color when particles are emitted.
You can separately specify Color 0 and Color 1 as the colors to inherit.
