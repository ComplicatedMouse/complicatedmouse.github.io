Bone Property Window: General Page

Bone Property Window: General Page

This window is used to check and edit the general properties of the selected bone.

A Shows Bone Information.
Used to edit properties such as the shape display related to the bone.
B

Shows Bone Matrix information.
C

Billboard settings.
Used to select the bone's billboard setting.

Bone Information

Used to confirm and edit information for the selected bone.

A

The index of the bone in the model.
B

The name of the parent bone of the bone.
This field is blank if the bone is the root bone.
C

The bone side value.

When using motion mirroring or similar, this value is the value configured in the DCC tool.
D

Select this check box if displaying a shape associated with the bone.
E

Select this check box if it is okay to remove the bone during bone compression and optimization.

Bone Matrix

This section is used to confirm matrix information about the selected bone.

A

The smooth skinning matrix index.
Set to –1 if the bone is not used in smooth skinning.
B

The rigid skinning matrix index.
Set to –1 if the bone is not used in rigid skinning.
C

Indicates whether the bone is used when rendering rigid bodies.
D

Indicates whether to eliminate the scale of the parent bone.
E

The scale value of the bone.
F

The rotation value of the bone.
G

The translation value of the bone.

Billboard

Specifies rotation control using billboarding.
For a DCC tool, the result is always no billboarding.

A Selects billboarding using an option.
B A description of the selected billboarding operation.

Billboarding

Billboarding Processing Description
No billboards Billboarding is disabled.
World Constrains the z-axis to remain parallel to the camera view.
It is influenced by scaling and rotation of the parent bone.
World viewpoint Constrains the z-axis in the camera direction.
It is influenced by scaling and rotation of the parent bone.
Screen Constrains the y-axis to be parallel to the camera's up vector and the z-axis to be parallel to the camera view.
It is influenced by scaling of the parent bone.
Screen viewpoint Constrains the y-axis to be parallel to the camera's up vector and the z-axis to the camera direction.
It is influenced by scaling of the parent bone.
Y-axis Rotates only the y-axis, and constrains the z-axis to be parallel to the camera view.
It is influenced by scaling and rotation of the parent bone.
Y-axis viewpoint Rotates only the y-axis,and constrains the z-axis to remain pointed in the camera direction.
It is influenced by scaling and rotation of the parent bone.
