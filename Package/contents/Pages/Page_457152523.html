Overview

InitializeHdev is a command-line tool for initializing the firmware on HDEV units.

Currently, this is the same tool as InitializeEdev . For more information, see that page.

Package

InitializeHdev is included in the SystemUpdater for NX package.

Location

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/InitializeHdev.exe
