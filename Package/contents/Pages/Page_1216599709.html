Profiler-BufferMode

Overview

Sets the buffer mode to be used when profiling.

Although the GUI allows this value to be set independently for both Sampled and Instrumented profiles, this cmdlet will set the same value for both.

Profiler-BufferMode [-Mode] <BufferMode>


[-Mode] <BufferMode>

Specifies the buffer mode to use when profiling. See Take a Profile for more information on the buffer modes.

  • StopWhenFull: Stops profiling when the buffers are full.
  • RingBuffer: Continue profiling when the buffers are full, overwriting the old data.


Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-BufferMode StopWhenFull
Profiler-BufferMode -Mode RingBuffer
