Configuring Waveform Archives

Configuring Waveform Archives

This section describes how to configure waveform archives.

Waveform archives are collections of waveform data that use banks and wave sound sets.
Waveform archives are configured automatically by default, so if you are not changing any waveform archive settings there is no need to reconfigure them.
We will describe the connection with banks here as well.

About Waveform Archives

The waveform archive groups multiple waveforms into one file.

Waveform data is much larger than bank data. Accordingly, bank data and waveform archive data are handled separately.
This allows for more efficient memory usage, such as referencing a single waveform archive used by multiple different banks.
You must register a bank and a waveform archive to register a single bank.

Displaying Waveform Archives

You can configure waveform archives from the Waveform Archive list in the Sound Set tab.
Select the Waveform Archive node in the project tree. The waveform archive list is displayed in the Sound Set tab.

Creating Waveform Archives

To create a waveform archive, right-click within the Waveform Archive list, and then select either Add Item or Insert Item from the shortcut menu.

The Create New Waveform Archive dialog box appears. Enter the required information.
