[User Settings]: Primitive

Primitive

Configures environment settings for using primitive files.

Show warning dialog when loading inaccessible primitives

Displays or hides a warning when an emitter set (ESET) file opens, and the model (FMDB) file for use with the particle shape primitive registered to an emitter or child is in a location that cannot be accessed.

Automatically load when referenced primitives are changed

Select whether to automatically reload primitives when the corresponding model (FMDB) files that have been added to emitters or children are updated.
