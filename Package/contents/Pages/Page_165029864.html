Starting Scale, Ending Scale

This section describes the Starting Scale and Ending Scale features to set the scale values for each end of a string-shaped particle.

Overview

You can use these features to set the Starting Scale, which is the alpha for the first particle vertex that is generated, and the Ending Scale, which is the alpha for the last particle vertex that is generated.
This section provides an overview of the settings ().

Starting Scale

Use Dial () and Text Box () to set the alpha value for the first particle vertex that is generated.
The stripe gradually disappears from view, starting from the end and proceeding to the beginning.

Ending Scale

Use Dial () and Text Box () to set the alpha value for the last particle vertex that is generated.
The stripe gradually disappears from view, starting from the beginning and proceeding to the end.
