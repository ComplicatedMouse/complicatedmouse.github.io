Tools

Overview

The NvPerf library provides tools (nvperf.exe) for processing that can be computed in advance, like generating ConfigImage and CounterDataPrefix offline on a PC.

For specific usage methods, see the PerfWorks User Guide or Help content displayed when a tool is run.

Location

Install the NvPerf library tools in the following directory.

%NINTENDO_SDK_ROOT%/Tools/Graphics/NvPerf
