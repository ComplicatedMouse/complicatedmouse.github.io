Emitter Shape - Line (Evenly Divided)

Describes the settings for Emitter Shape - Line (evenly divided), where particle emission locations are set equidistantly along a line.
These are the particles when this emitter shape is being used.

Overview

With the Line (Evenly Divided) emitter shape, particles emit in the direction of the line (z-axis direction) from the evenly divided locations on the line emitter.
The following overview describes the settings ().

When the emitter shape is set to Line (Evenly Divided), the Line (Evenly Divided) () settings are displayed.
The following settings are specific to even division in this group.

Emission Type

Selects the behavior type when a particle is emitted in the emitter normal direction.

Together Particles are emitted from all division locations in the direction of the normal.
Random Particles are emitted from random locations of the divisions in the direction of the normal.
By Index Particles are emitted from the division locations in the order of rendering in the direction of the normal.

Number of Divisions

This setting for the number of divisions becomes the number of particles that are emitted, and the setting for Emission rate is ignored.

Random Number of Divisions

Changes the number of divisions randomly every time particles are emitted.
The upper and lower limits for the random number of divisions changes randomly. The following formula calculates a lower limit for the random number.

Random Number Lower Limit: Number of Divisions - Random Number of Divisions.

Example Configurations

In the following descriptive drawings for Emission Type, the number of divisions is set to 6, and the emission rate is set to 2.

A Together Particles emit together from six evenly divided locations.
The emission rate cannot be set in this preset.
B Random Particles are emitted randomly two at a time from six evenly divided locations.
