Project Restore Feature

At the times such as when the reference file path of a project has changed, you can check for the presence of the files and automatically restore project files by selecting Project >Restore Project.

There are no restore settings. Restore by clicking Yes on the dialog box.

If the reference file cannot be found, you can specify a new file path and choose a method to search for the file in the specified path.
