Color Replacement Button

A button that replaces the values for the following settings.

C0 <-> C1

 Swap the behavior type settings for color 0 and color 1.

A0 <-> A1

Swap the behavior type settings for alpha 0 and alpha 1.


The following table provides example settings and an example case using the CO <> C1 button.

Textures Combiner Expressions
C0 × Comb_RGB ＋ C1 × (1 － Comb_RGB)

The colors specified for C0 and C1 are swapped in EffectViewer when the button is activated.
