This section introduces the new features in 3D.

Updated Intermediate File Version

Updated the 3D intermediate file version to from 3.7.0 to 4.0.0.

Sampler References in Material References

Material references now reference samplers.

Improved Quantization Tolerance Configuration UI

Improved the UI so that it is easier to determine changes in the data size reduction rate based on the quantization tolerance settings for the curve editor.

Edit Team Settings From the GUI

Team settings can now be easily edited using buttons in the GUI.

Optimized Shader Compiling Information

You can now check optimized shader information for each material at compilation time.

Material Label Settings

You can now specify labels for materials, models, bones, textures, and animations.

Thumbnail Explorer Extension

Added support for displaying thumbnails of AI and EPS files.


Updated Intermediate File Version

The file version for 3D intermediate files was updated from 3.7.0 to version 4.0.0. For the major changes, see the update history in 3D Intermediate File Specification Manual.

Make sure that you update existing intermediate files using Intermediate File Updater. This intermediate file version update does not affect the binary resource files (BFRES files) output by 3D Binary Converter.

Material References Now Reference Samplers

Material references now reference samplers. This change means that the parent material can control sampler parameters and sampler structures of the child material.

Optimized Shader Compiling Information

You can now check optimized shader information for each material at compile time.

Note

This feature has problems in NintendoSDK 5.0.0, and does not output the correct information in the following situations.

  • When loading multiple models to which one shading model has been assigned
  • When Undo is performed in real-time shader editing
  • When material options have been edited, and you return to the previous state


Material Property Window: General Page Describes the page for verifying the general properties of the material and then editing them.


Material Label Settings

You can now specify labels for materials, models, bones, textures, and animations.

Note

Currently, only material names are replaced with the specified labels when the parent material is selected.

Label (Material name) is displayed when there is a label. Otherwise, Material Name is displayed.
