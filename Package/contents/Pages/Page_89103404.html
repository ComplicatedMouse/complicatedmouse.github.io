NintendoWare 3D News

This page introduces several new features that were added to the 3D module in NintendoSDK 0.6.0.

3DEditor

Support for Loading Textures of the Same Name in Different Paths

NintendoWare 3DEditor can now load textures of the same name, but in different directory paths.

However, multiple identically named textures may not be applied to the same model.

List View Settings Save and Load

The display settings for the list view may be saved and loaded.

Target Name Filter Feature Added to List Display

Target names can now be filtered with list display.

Blinking Feature for the Selected Model

Specifying a model in 3Deditor now makes the model blink in the connected application.

This feature depends on the connected application not using the display list.

Photoshop Plug-In

Added Resize Interpolation Algorithm Selection Feature

The interpolation algorithm during resizing with the texture export plug-in can now be selected.
