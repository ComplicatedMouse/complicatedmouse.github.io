Plug-In for Disabling Node Compression

This plug-in sets flags so transform nodes and joint nodes are not compressed even if the compression feature is enabled.
If compression is disabled this way for the transform nodes of camera, light, and environment fog, cameras and environment fogs can be exported to the FSN file even if the Visibility attribute is not selected, and lights can be exported even if the Template attribute is selected. (This feature is in Nintendo SDK version 9.1.0 and later.)

The plug-in also has a feature for searching for the settings.

How to Use

Select Nintendo > Set No Compress Node to display the window.

Selected transform nodes and joint nodes are displayed under Selected Transform Node.

Info

If four or more nodes are selected, names are not displayed and only the number of nodes selected is displayed on the right edge.

Setting Method

When you select a transform node or joint node, the current setting for the selected node is displayed under Set. You can select multiple nodes.
If you select multiple nodes, the setting for the first selected node is displayed.

Click the combo box to change the flag for disabling compression.
You can select multiple nodes to change the settings for all selected nodes at the same time.

If you set the flag to Compress this node if possible, the node will be compressed if compression has been enabled. This setting is the default.
If you set the flag to Don’t compress this node, the node will never be compressed, even if compression is enabled.

You can configure the setting for a different node by keeping the window open and selecting a different node.

Searching

In Search > Target, specify the search target. Specify Model to search for the settings of the transform nodes for models. Specify Camera / Light / Fog to search for the settings of the transform nodes for cameras, lights, and environment fogs.
To select the corresponding transform nodes and joint nodes, in Search, select the appropriate check boxes and then click Search.

Info

If you select multiple check boxes, the search looks for all nodes that satisfy any of the settings.
