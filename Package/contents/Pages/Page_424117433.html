Common Options Argument Values Description
-s Name of the target storage. Specifies the storage that will be the target of the process. For more information about the storage names that can be specified with the argument, see the description of the application install command.

Installing

Option Description
--dynamic Reports to the currently running application that the available downloadable content has changed.
For more information, see Enabling the Use of Downloadable Content Downloaded in the Background While the Application Is Running.

--force

 Forcibly overwrites and installs if the downloadable content being installed is already installed.
-i

Specifies an index.

To specify multiple indexes, separate them by commas, or use a hyphen to specify a range, as in x-y. Commas and hyphens can be used together.

Normal Installation

$ RunOnTarget.exe DevMenuCommand.nsp -- addoncontent install C:\addoncontent.nsp
Preparing to install C:\addoncontent.nsp...
0 / 2010624
2010624 / 2010624
[SUCCESS]

Forcibly Installing Multiple Specified Indexes

$ RunOnTarget.exe DevMenuCommand.nsp -- addoncontent install C:\addoncontent.nsp -i 2,3,5-15 --force
Preparing to install C:\addoncontent.nsp...
0 / 1537536
1537536 / 1537536
[SUCCESS]

Uninstalling

Option Description
-i

Specifies an index.

To specify multiple indexes, separate them by commas, or use a hyphen to specify a range, as in x-y. Commas and hyphens can be used together.
