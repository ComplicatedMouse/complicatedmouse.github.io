Sound Panel for Sounds Playing in atk Library

This panel lists information about the sounds currently playing in the atk library.

UI Reference

Description
A The toolbar. Use it to change how the list is displayed.
B The list view displaying the sounds that are playing.

Toolbar

Item Description
Name Filter Filters list items using the entered text.
Sound and player names are filter targets.
Check this option to display all vocalized sounds.
Check this option to display all muted items.

Sets the visibility of the list columns.

Cannot hide the vocalization and sound name columns.
Displays the number of sounds playing currently.

List View

Sounds only appear as list items while they are being played back. After playback ends, sounds disappear from the list.
When multiple sounds are played back at one time, an item for each sound appears on the list.

List Column Description

Audible

Indicates the audible status of the sound.

If the playing sound is set to a volume of 0, then it will be shown as muted.
