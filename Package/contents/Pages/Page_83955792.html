Thumbnail Preview Display Feature

Introduction

This feature displays thumbnails and previews of image files in Windows Explorer.
To install or uninstall, see the Thumbnail Explorer Extension User Manual page.
To change the extension settings, see the Setup Manager page.

How to Use

A thumbnail is displayed in a way that fits the Windows Explorer Icon, List, Details, Tiles, and Content items in the View menu.
Supported formats: Targa files (TGA files), PSD files, AI files, EPS files, texture intermediate files (FTXA and FTXB files), and files with extensions specified as command-line arguments of the installer.

For textures with an alpha channel, thumbnails are displayed using the product of the color channel and the alpha channel, except when displaying as large icons or larger.
If the alpha channel is zero, nothing will be displayed.

The thumbnails of ESP files are only displayed if the files have been exported from Adobe Illustrator.

Thumbnails Can Now Be Displayed for Intermediate Model Files, Layout Files, and Emitter Set Files

To display thumbnails of model intermediate files (FMDB and FMDA files), layout files (FLYT files), and emitter set files (ESET files) in Explorer, you must create thumbnail images in a specified folder. Specifically, create a folder named .NwThumbs in the directory that contains the files for which thumbnails are being created. Then, to be able to display thumbnails, create image files in that folder named <filename including filename extension>.(JPG or PNG).

For example, if you want to display a thumbnail for an intermediate model file in the path C:\Models\human.fmdb, create an image file in the path C:\Models\.NwThumbs\human.fmdb.jpg.

Displaying Thumbnails of Files in Optional Formats

To use the Thumbnail Explorer Extension to display thumbnails of files in formats that are not the standard supported file formats, you must register the file extensions during installation. For more information, see Registering Optional Extensions.

To show the thumbnails, create a folder named .NwThumbs in the directory containing the files with the thumbnails you want to show. In that folder, create image files with filenames that include the extensions (such as .jpg and .png).

Icon Display

When displaying as a large icon or larger, the color and alpha channels are displayed separately.
Texture format, texture width and height, mipmap level, and data size are also displayed.

Info

If mipmaps have not been generated to the fullest possible extent, the Mipmap level display will be in red.

Tooltip Display

When the mouse points to a thumbnail displayed in Windows Explorer, a tool tip appears showing the texture information.
