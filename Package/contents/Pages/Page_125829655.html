Favorites

This page describes the Favorites tab UI.

A Address Bar

Specify a folder path.
The content of the specified path is displayed in the file list.
B Navigation Pane

Displays items that have been added to Favorites ().
Favorites folders () can be used to group together items in your Favorites () for improved accessibility.
Info

Double-click a Favorites folder () or Favorites () to rename them.

Available Features

The following features are available in the Favorites tab.

Export Exports the selected file to the workspace.
Edit Comments Edit comments for the selected file.
Related Links For more information, see Comments.
Label Color Settings

Sets a label color for the selected file.
Related Links For more information, see Label Colors.
Favorites Folders

Creates a Favorites Folder () in the Favorites tab.
Favorites folders () can be used to group together items in your Favorites ().
Remove Favorites Removes an item that was previously added to Favorites () from the Favorites tab.
Add to Favorites Adds the selected file or folder to Favorites ().
Display Windows Explorer Opens Windows Explorer and goes to the path of the selected file or folder.
Add Sample to Favorites Adds the sample data within a package to Favorites ().
