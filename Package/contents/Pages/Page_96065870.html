Scale Standard Value

Standard value

This section describes the Standard value feature that sets the base size of particles upon emission.

Overview

The Standard value feature can set the base size of particles upon emission.
The specified size becomes the base, to which scaling can be applied.
The following overview describes the settings ().

Standard value

Sets the scale for particle emission.
