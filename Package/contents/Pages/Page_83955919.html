Property Window

Property Window

The Property window is used to edit objects that are loaded into 3DEditor.
The fields that can be edited depend on the object that is selected.

Property Window Types

The following types of Property windows are available in 3DEditor.

Property Window Name Description
Project Property Window

Used to check path information or the creation log for the project.
Model Property Window

This window is used to check and edit the properties of the selected model.
Bone Property Window

This window is used to check and edit the properties of the selected bone.
Shape Property Window

This window is used to check and edit the properties of the selected shape.
Material Property Window

Used to check and edit the properties of the selected material.
Texture Property Window

This window is used to check and edit the properties of the selected texture.

Animation Property Window

This window is used to check and edit the properties of the selected animation. (The fields that can be edited depend on the animation type.)
Shader Definitions Property Window

Used to check and edit the selected shader definition file.

Basic Operations of the Property Window

The following basic operations can be performed in the various property windows.

A Fixes, or pins, the selected state of the target object.
If the selected state has been pinned this way, the property window for other objects is opened in another window.
B
Displays the name of the object being configured.
Displays the name of the model being used after the name of the object to distinguish between objects with the same name in the property window.
Example: Names now appear as Material name - Model name.
C

Copies properties page settings, including the child hierarchy, for a properties page selected by an active property window.
D

Pastes the copied property page settings.

Pastes each selected item when multiple items are selected.
E
Used to export material configuration files.
Related Links This operation is enabled in the Material Property Window.
F
Imports the material configuration file.
Related Links This operation is enabled in the Material Property Window.
G Open a copy of the selected property window in a separate window.
H Opens the Help file.
I The property window tree list.
Selects the property page for the object.
J Edits the task color.
K Edits the task label.
L

Edits the task comments.

You can create multiline comments. Press CTRL+ENTER to insert a newline character, and press ENTER to confirm the comment.
For comments longer than three lines, use the up and down arrow buttons to scroll.
M Used to check and edit the content of the selected property page.
N
A red star (update mark) is displayed next to items that have been edited but not saved.
Related Links The star is also displayed next to edited items in the File View.
