Profiler-NX-SampleMode

Overview

Sets the sampling mode to use for taking a profile.

Profiler-NX-SampleMode [-Mode] <ScriptingSampleMode>


[-Mode] <ScriptingSampleMode>

These values correspond with the sampling mode buttons on the Sampled Profile tab.

    • InProcess: Sets the In-Process profiling mode.
    • OutOfProcess: Sets the Out-of-Process profiling mode.
