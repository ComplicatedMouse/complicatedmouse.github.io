Creating Sound Sets

1. Overview

The following example shows how to create a sound set.

You can also create and add a sound set when creating a new sound project. This page describes how to create and add a sound set when working on an existing sound project.

For more information, also see SoundMaker > Basic Guide > Creating Sound Projects > Creating Sound Sets.

2. Example

2.1. Creating a Sound Set

1. Select Project > Add Sound Set > New Sound Set.

2. Set the sound set name and path and click OK.

3. The sound set is added to the project tree.

2.2. Specifying an Existing Sound Set

1. Select Project > Add Sound Set > Existing Sound Set.

2. Select the sound set file (FSST file) to add and click Open.

3. The sound set is added to the project tree.
