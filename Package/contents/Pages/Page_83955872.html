Plug-In for Setting Light Attributes

This plug-in sets NintendoWare light attributes for the shape node of point lights or spot lights.

Usage

Select Nintendo > Set Light Attribute to open the window.

The transform nodes of the selected light are displayed in the Selected Light column.

Info

If four or more nodes are selected, names are not displayed and only the number of nodes selected is displayed on the right edge.

Setting Method

When you select one or more shape nodes or transform nodes for a point light or spot light, the current settings for the selected light are displayed in the Set field.
If you select multiple lights, the setting for the light selected first is displayed.

Attributes are changed by controlling each item in the window. If you selected more than one light, you can change all of the selected lights at the same time. (Only modified attributes are applied.)

You can select a different light by selecting another node without closing the window.

Distance Attenuation

Enable this option to use distance attenuation.
This item is tied to the decay rate attribute of the shape node of a light.
If this option is enabled, the decay rate changes linearly if No Decay is selected, quadratically if Quadratic is selected, or cubically if Cubic is selected.
The intermediate file exported is the same, regardless of whether the decay rate is Linear, Quadratic, or Cubic.
Start Distance

Specifies the distance at which to start attenuation.
This option is available if Distance Attenuation is enabled.
End Distance

Specifies the distance at which to end attenuation.
This option is available if Distance Attenuation is enabled.

Start Distance and End Distance are not applied on the Maya screen.

To set a key in the current frame, right-click Start Distance or End Distance and select Set Key from the menu.

Custom attributes are added to the shape node of the selected light when this plug-in is executed.
Custom attributes are indicated by the names NW4F Attn Start Dist and NW4F Attn End Dist in the Channel Box and in the Attribute Editor (in Extra Attributes).

Info

Custom attributes and animations connected to custom attributes are cleared if the light type is changed.
If you change the light type, make sure that you use this plug-in to re-configure the Start Distance and End Distance.
