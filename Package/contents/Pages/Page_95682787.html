Setting Constant Texture Density

This section describes the feature for setting a fixed, constant display size for the texture.

Overview

The Set Constant Texture Density feature can be used to display textures in the same size, unaffected by the particle scale.
This section provides an overview of the settings ().

Constant Texture Density U

When particle scale is animated, the animation is not influenced by the particle texture U coordinate.

Constant Texture Density V

When particle scale is animated, the animation is not influenced by the particle texture V coordinate.
