This window is used to check and edit the properties of the selected model.

A Select when copying and pasting all model properties.
B

Displays the General page.
C

Displays the Preview page.
D

Displays the LOD Model page.
E

Displays the File Information page.
F

Displays the Additional Reference Path page.
G

Displays the Optimization Log page.
H

Displays the User Data page.
