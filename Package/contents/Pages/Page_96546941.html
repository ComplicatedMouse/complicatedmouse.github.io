Phase Scatter (%)

The Phase scatter feature varies the GPU noise waveforms for each particle.

Overview

The Phase scatter (%) feature specifies a percentage by which to vary the phases of waveforms aligned using the Align phase of each element feature.
When phase is varied, even if phases are equalized in an overall way, each particle will have its own unique trajectory.
The following overview describes the settings ().

Phase Scatter (%)

Use the dial () or the text box () to set the range in which to vary the phase (where 100% is equal to a full period).
If 100% is specified, it is the same as if phases were not equalized.

One period is the upper limit of the random number, and the lower limit is changed randomly. The following formula calculates a lower limit for the random number.
