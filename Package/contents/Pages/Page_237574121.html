7. Export Animation

3. Export

This section describes how to export animations from Maya.

Selecting a Model

Select and set the model to export.

Select TopL from the outliner.

Export

From the Maya menu, select Nintendo > Export with Setting.

To export, perform operations in the order indicated by the following diagram and table.


Order Category Operation
1 Export Target Configure Selection.
2 Output Filename

Enter TrafficSignal.
3 Output Folder

Click the folder icon and specify the following AnimationSample folder.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal
4 General Options For Magnify, enter 1.000. For Texture SRT Mode, select Maya.
5 Animation Options

For Start / End Frame, select All. Export the animation with the playback range (0F-50F) set in Maya.

Set looping playback for the animation. Select Loop. (You must select this option for one of the next 7 animation data items.)
6 Output File Selection Select Texture Data.
7 Output File Selection

Click Search Model Animation. A valid export target for the animation key will be selected.

Check that the following four output files are selected.

Bone Visibility Animation Data

Color Animation Data

Texture SRT Animation Data

Texture Pattern Animation Data

8 Export

Click Export. Files will be output according to the specifications set in items 1 to 7.
