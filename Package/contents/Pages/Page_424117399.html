Checking the Firmware Information

Use the firmware command.

$ RunOnTarget.exe DevMenuCommand.nsp -- firmware --help
usage: DevMenuCommand firmware version
       DevMenuCommand firmware revision
[SUCCESS]

Checking the Firmware Version

$ RunOnTarget.exe DevMenuCommand.nsp -- firmware version
NintendoSDK Firmware for NX 0.1.0
[SUCCESS]

Checking the Firmware Revision

$ RunOnTarget.exe DevMenuCommand.nsp -- firmware revision
305f25d731cd3fafe059435ccb9bd508aff02f5c
[SUCCESS]
