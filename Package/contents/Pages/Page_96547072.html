Period for Element 0 to Element 3

This section describes the feature for adjusting the period of waveform elements 0 to 3.

Overview

This feature sets the period for each waveform element (elements 0 to 3).
Changing the ratio of the period of each waveform element relative to the base period makes it possible to generate non-uniform waveforms and randomize the paths along which particles move.
The following overview describes the settings ().

If the Element settings feature has been enabled, the Ratio relative to base period of each element group () is displayed. The group has the following settings.

Element 0 to Element 3

Sets the period of each waveform element used to generate GPU noise.
The specified values are multiplied by the base period value to calculate the period (number of frames) of each element.

Larger values produce waveforms with longer periods that change more slowly. Smaller values produce waveforms with shorter periods that change more rapidly.
The following formula is used to calculate the period of each element.

  • Element period = Base period × Ratio relative to base period
