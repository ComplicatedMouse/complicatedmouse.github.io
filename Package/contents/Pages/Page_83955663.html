Toolbar

Displays various 3DEditor features as icons.


Icons Corresponding Menu Item Operation
File > Open Opens a file.　
File > Save Saves the selected file, and overwrites any existing file in the save location that has the same name.　
File > Save All in Specified Location Saves all loaded files in the specified location. A save dialog box allowing you to specify the save location opens if selected.
File > Overwrite and Save All Saves all files that have been edited using 3DEditor. Unedited files are not saved.
Edit > Undo Restores to a pre-edited state.
Edit > Redo Reapplies a single edit that was undone.
Display > Object View > One Screen Sets the Object View to a single-screen layout.　
Display > Object View > Two Screens: Horizontal Sets the Object View to a two-screen layout, with upper and lower sections.　
Display > Object View > Two Screens: Vertical Sets the Object View to a two-screen layout, with left and right sections.　
Display > Object View > Triple Screen: Split Bottom Sets the Object View to a triple-screen layout, with an undivided upper section, and a lower section divided into left and right sections.　
Display > Object View > Triple Screen: Split Top Sets the Object View to a triple-screen layout, with an undivided lower section, and an upper section divided into left and right sections.　
Display > Object View > Triple Screen: Split Right Sets the Object View to a triple-screen layout, with an undivided left section, and a right section divided into upper and lower sections.　
Display > Object View > Triple Screen: Split Left Sets the Object View to a triple-screen layout, with an undivided right section, and a left section divided into upper and lower sections.　
Display > Object View > Four Screens Sets the Object View to a quad-screen layout.
Display > Property Window Opens the property window for the object selected in the File View or Object View.
-- Selects the platform of the connection target application from a drop-down list.
Preview > Connect to Application (while not connected) Connects to (or disconnects from) the application specified as the connection target.
Preview > Connect to Application (while connected)

PC (P)

 Preview > Select Connection Target (PC connection) Selects the connection target.
You can select development emulator or PC. You cannot switch the connection target while connected with either.
To switch the connection target, first disconnect from the application (running on the PC or development hardware).
Note: You can also select the connection target by specifying which target to connect to by using a command-line option when starting 3DEditor at a command prompt.
Note: You cannot change the selection when connected.

Production device (C)

 Preview > Select Connection Target (while connected to development hardware)
Preview > Optimize shaders (disabled state) Enables or disables shader optimization.
When shader optimization is enabled, only shader programs being used by each model are transferred at run time.
Preview > Optimize shaders (enabled state)
Preview > Optimized Shader Regeneration Use this option when you need to manually regenerate your optimized shaders for a material.
Note: You cannot use this feature when shader optimization is disabled.
Automatic Shader Regeneration --

Select a setting for automatic shader generation from the drop-down list.

(Usage example: Select Disable to avoid spending time regenerating the shader while working, and then switch to Enable after the work is complete.)

  • When set to Enable, the settings for individual materials are ignored and the process is forcibly enabled.
  • When set to Disable, the settings for individual materials are ignored and the process is forcibly disabled.
  • If set to Use the material's setting, the value set in the Preview page is used.

Tools > Auto Animation Bind Mode Enables or disables Auto Animation Bind Mode.
You can also change this option under Tools on the main menu.
Help > Help Opens the 3DEditor Help documentation.
Help > Version Information Displays version information.
