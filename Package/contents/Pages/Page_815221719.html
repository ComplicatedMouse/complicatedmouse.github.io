Animation Live Editing (Preparation)

Preparation

The following sample shows the steps for previewing animation by sending models from Maya to your application. In practice, you use the Maya scene files from which the intermediate files are exported, but this example prepares a sample Maya scene using intermediate files included in the SDK.

This section describes the following procedure: Import FMDB files > Save MA files > Reference MA files > Import animation.

1. Creating a Project

Create a new Maya project in any folder.

2. Importing and Saving the Models

Import the models to Maya and save them as Pierrot.ma and Robot.ma.

Save as Pierrot.ma Save as Robot.ma

Importing and Saving Pierrot

Open a new scene, use the Import Plug-In to import the following model intermediate file to Maya and save it as Pierrot.ma.
(In the Maya menu, select Nintendo > Import > Import > fmd File and specify the following Pierrot.fmdb file for import.)

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SkeletalAnimDemo\Pierrot.fmdb

Importing and Saving Robot

Open a new scene, use the Import Plug-In to import the following model intermediate file to Maya and save it as Robot.ma.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SkeletalAnimDemo\Robot.fmdb

3. Creating References

Open a new scene and create a references for the Pierrot.ma and Robot.ma files you just saved.
(From the Maya menu, select File > Create Reference) to create the references.)

Select the Pierrot:Root node, and use the Import Plug-In to import the following intermediate skeletal animation file to Maya.
(From the Maya menu, select Nintendo > Import > Robot.ma > fsk File and specify the following Pierrot_Run.fskb file for import.)

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SkeletalAnimDemo\Pierrot_Run.fskb

Verify that the imported animation is applied to Pierrot.

Pierrot and Robot are overlapping, which makes them harder to see. Slightly shift the Robot's position by changing the Translate X value of the Robot:Root node.
