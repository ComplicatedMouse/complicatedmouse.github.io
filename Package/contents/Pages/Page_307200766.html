NX Add-On 3.4.0 (Firmware 3.0.0-11.0) and Later (Common)

This page provides lists of the HTML/CSS/ECMAScript specifications supported by the web applet and Offline Web Applet in NX Add-On 3.4.0 (Firmware 3.0.0-11.0). The supported specifications may be subject to changes in future system updates.

The information listed for the web applet in this document also applies to the offline web applet unless specifically noted otherwise.

HTML5

The following is a list of the supported W3C HTML5 (https://www.w3.org/TR/2014/REC-html5-20141028/) specifications.

Note the following differences in operation compared to a general browser.

  • The video element does not support the autoplay attribute.
  • resize events do not affect the root element. resize events do affect resources loaded in iframe objects.
