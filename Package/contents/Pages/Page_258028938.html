Searching for Shader Parameters

You can now search for shader parameters by pressing CTRL + F while in the Shader page of the material Property window.
Items with the entered text string are highlighted.

Hide Materials in Curve Editor That Are Not Set With Keys

It is now possible to hide and not show materials in the curve editor that have not been set with keys.

Improved Operations for Opening the Curve Editor From the Material Property Window

A button for opening the curve editor has been added to each channel for colors and vectors, enabling the curve editor to be opened with a curve selected for each channel.

In addition, when multiple materials have been selected, clicking the buttons opens the curve editor with the curves selected for all selected materials.

Change to the Material Referencing Specifications

The specifications have been changed for resolving references according to the Child Material Restrictions setting when there are multiple references for a material. For more information, see the Multiple Material Referencing section of Material Referencing.

Thumbnail Explorer Extension

Thumbnails Can Now be Displayed for Model Intermediate Files, Layout Files, and Emitter Set Files

It is now possible to show thumbnails for model intermediate files (FMDB and FMDA), layout files (FLYT), and emitter set files (ESET). To show thumbnails for model intermediate files, place image files in the prescribed folder. (An example is presented below.)

For more information about how to display the thumbnails, see Displaying Thumbnails for Model Intermediate Files, Layout Files, and Emitter Set Files in Thumbnail Preview Display Feature. LayoutEditor and EffectMaker have a feature for capturing thumbnails of layout files and emitter set files. For more information, see the documentation for those tools.

Thumbnail Preview Display Feature Describes the feature for displaying thumbnails and previews of image files in Windows Explorer..
