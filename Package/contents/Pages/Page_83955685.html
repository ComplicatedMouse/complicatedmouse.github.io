Textures

This tab is used to configure the settings for texture images mapped to particles.

The Textures tab () contains the following sub-tabs ().

Textures 0 to 2

Three subtabs: Texture 0, Texture 1, and Texture 2.

Up to three textures (Texture 0 to Texture 2) can be configured for use one by one with a single emitter.
Shared Settings Configure settings shared by Texture 0 to Texture 2.
Info

You can register texture files by directly dragging them from the Effect Browser or Windows Explorer to one of the three texture sub-tabs (Texture 0 to Texture 2) ().

The following settings are for the Texture 0, Texture 1, and Texture 2 sub-tabs ().

Texture

Texture () is a group of settings for configuring the textures used by emitters.
This group has the following settings.

Specify File

Displays the path of the texture file and allows you to register a texture.
There is a texture preview feature and useful preset buttons.
UV Set to Apply When the particle type is Primitive, this setting allows you to select the UV set for the texture being registered.
Get Square of RGB Squares the brightness of the registered texture to approximate linear space.
Pattern Division

Sets the number of divisions of the texture.
Wrap Sets how to repeat the texture.
Repetition Count Sets how many times to repeat the texture.
Random Flip Sets whether to flip the texture coordinates when particles are emitted.
Sphere Map Uses registered textures for sphere mapping.
Filter Mode Sets the method of texture filtering.
Restrict Mipmap Levels Sets texture mipmaps.

Texture Coordinate Animation

Texture Coordinate Animation () is a group of settings for configuring the UV coordinates of registered textures.
This group has the following settings.

UV Scroll Animation Animates the position of the texture UV coordinates.
UV rotation animation Animates the angle of the texture UV coordinates.
UV Scale Animation Animates the scale of the texture UV coordinates.

Texture Pattern Animations

Texture Pattern Animation () is a group of settings for configuring the animation of numerous images (patterns) rendered to a single texture.
This group has the following settings.

Playback Methods Animates texture patterns by moving texture UV in a stepwise fashion.

Texture Options

Set Constant Texture Density () is a group of settings for configuring the feature that fixes display size of textures.
This group has the following settings.

Setting Constant Texture Density

Displays textures in a fixed and constant size.

The Shared Settings sub-tab () contains the following settings.

Random Number Setting

Random Number Setting () is a group of settings for configuring the feature that fixes a random seed for textures 0 to 2.
This group has the following settings.

Fixing the Random Seed

Fixes the random seed in the Texture 0, Texture 1, and Texture 2 sub-tabs.
