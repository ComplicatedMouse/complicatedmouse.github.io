Multiply by the Render Color

This section describes the feature for multiplying the color and alpha of all particles in the emitter set by the configured value.

Overview

The Multiply by the render color feature multiplies the color and alpha of all particles in the emitter set by the set RGBA value.
The feature performs multiplication on the result obtained in combiner calculations.
This section provides an overview of the settings ().

Multiply by the Render Color

Set the color and alpha to multiply the particles by.
