Workflow

This section describes the flow from starting the FontConverter (GUI version) to creating the bitmap font binary (BFFNT file) and completing the procedure.
The following subsections describe the procedures involved in performing a task.

1 Starting the Task Start FontConverter (GUI version) and begin the task.
2 Input Specifications

From Input Specifications, select the tab that is the conversion source and perform input settings in that tab.
3 Filter Specifications Specify filters as necessary. If you are not going to use a filter, clear the Use Filter check box.
4 Output Specifications

From Output Specifications, select the tab that is the output target and perform output settings in that tab.
5 Convert

Click the Convert button to execute conversion processing.
6 Completing the Task Complete the FontConverter (GUI version) task.

１ Starting the Task

To begin the process, start the FontConverter (GUI version) that configures the effects.

Starting FontConverter (GUI Version)

To start FontConverter (GUI version), double-click the following startup file.

\Tools\Graphics\FontConverter\FontConverter.exe

2 Input Specifications

Load the font file for creating fonts into the FontConverter (GUI version).
Describe each tab.

Input Specifications in the Image Tab

To load the font file, do one of the following.

  • Click the Select File button and specify the image file conversion source from the selection dialog box.
  • From Windows Explorer, drag the image file conversion source to the text box ).

Input Specifications in the WindowsFont Tab

To load the font file, do one of the following.

  • Select the font to use as the conversion source from the list of fonts installed on Windows PCs.
  • Click the Select File button and specify the file from the selection dialog box.

3 Filter Specifications

Specify filters as necessary. If you are not going to use a filter, clear the Use Filter check box.
To load the font file, do one of the following.

  • Click the Select File button and specify the XLLT file from the selection dialog box.
  • Directly enter the path in the text box () and specify the XLLT file.
  • Drag the converted file to the text box () and specify the XLLT file.
  • Click the ▼ button in the text box () and specify the XLLT file from the history.

4 Output Specifications

Create a font file.
Do one of the following.

  • Click the Select File button and specify the save destination folder and filename from the selection dialog box.
  • Directly enter the path in the text box () and specify the save destination folder and filename.
  • Drag and drop the converted file in the text box () and specify the save destination folder and filename.
  • Click the ▼ button on the text box () and specify the save destination folder and filename from the history.

5 Conversion

Click the Convert button and convert the tab using the selected input tab settings.
The file types generated in the conversion are different for each tab.
Related Links For more information, see Output Files.

6. Completing the Task

This completes the font creation process.

Exit FontConverter (GUI Version)

Close all open files and exit FontConverter (GUI version) using any of the following methods.

  • From the main menu, select File > Exit.
  • On the main window, click Close.
  • The keyboard shortcut is CTRL+Q.
  • Click the Close () button in the upper-right corner of the main window.

If there are any files that have not been saved, a dialog box appears. Specify a save destination in the dialog box that opens, and then click OK.

Directly enter the path in the text box () and specify the save destination folder and filename.
