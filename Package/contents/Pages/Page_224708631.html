Overlap Check for Particles

Describes the Overlap Check for Particles feature for verifying particle overlap.

Overview

Overlap Check for Particles is a debugging feature for checking the degree of overlap among particles, using EffectViewer.

Overlap Check for Particles

You can enable or disable this option by selecting or clearing the check box ().
