Edit

This menu includes several editing operations.

A

Undoes any edits made and returns to the pre-edited state.
B

Applies the edits again to content that was previously canceled by the Undo operation, returning the content to its post-edited state.
C

Deletes the edit history.
After the edit history has been deleted, no Undo and Redo operations can be performed on the edits made to this point.

D

The edit history is automatically deleted when the file is saved.

Related Links You can also configure the edit history settings in the user settings file (3dEditor.user.config).
