Curbing Particle Overlap

Describes how to adjust particle emission so there is little overlapping of particles.

It can be impossible to control particle overlap when particle emission locations are completely random.
The following two methods can reduce overlap using emitter features.

A

Set the emitter shape to Circle.

Emission can occur in an overlapped way, from the same location, because the particle source locations are random.
B

Set the emitter shape to Circle (evenly divided).

You can control overlapping in identical locations, while continuing to emit using a certain level of random positions, because Random Angle is set to 10.
