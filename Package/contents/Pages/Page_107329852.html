NintendoWare 3D News

This page introduces features that were added to the 3D module in NintendoSDK 0.9.0.

Maya Plug-In, 3ds Max Plug-In

  • Deprecated Use 3DEditor.
  • Added a feature for creating a custom UI to the Options window of the Maya export plug-in.

Deprecated Use 3DEditor.

The Use 3DEditor option has been deprecated.

For more information about alternatives, see NintendoSDK Migration Guide for NintendoWare 3D.

Added a feature for creating a custom UI to the Options window of the Maya export plug-in.

Added a feature for creating a custom user-designed UI to the Options window of the Maya export plug-in.

For more information, see The user_custom_ui Setting in the Export Plug-In.

3DEditor

  • Added a function to show or hide the Debug Log Window.
  • Added a feature for switching the platform to the toolbar.
  • Added the 3DEditorBridge tool.

Added a function to show or hide the Debug Log Window.

Shows or hides the Debug Log Window.

To show or hide, in the main menu, select Tool > Display Debug Log Window.

Info

This window is mainly used to investigate problems.
Displaying the window is useful if a bug always occurs in a particular environment.

Added a feature for switching the platform to the toolbar.

Added a feature for selecting the target platform for the connected application to the 3DEditor toolbar.

Info

If the incorrect platform is selected, you may experience issues such as not being able to connect to the viewer. Make sure to select the correct platform.

Added the 3DEditorBridge tool.

Added 3DEditorBridge, which is a command-line tool for using 3DEditor features from outside of 3DEditor itself.

3DEditorBridge can be used to link 3DEditor with other tools.

For more information, see 3DEditorBridge.
