Fade-In Duration

This section describes the Fade-In Duration, which is the feature for setting the duration for particle fade-in.

Overview

When particles are emitted, they fade in over the course of time set for this value.
The Fadeout Duration feature is available when Alpha Fadeout or Scale Fadeout are being used.
The following overview describes the settings ().

Fade-In Duration

Set how much time elapses after the Fade command executes.

Example Configurations

The following figure compares what happens when different values are set for Fade-in Duration when Alpha Fade is enabled.

A 0 Particles appear at the moment the Fade command is executed.
B 100 Particles gradually appear over the course of 100 frames, with fade-in for both alpha and scale.
