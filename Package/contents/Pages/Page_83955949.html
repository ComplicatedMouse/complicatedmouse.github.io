Texture Property Window: Usage Status Page

This page is used to check the usage status of the selected texture.

A Lists materials that reference the selected texture.
B Lists pattern animations that use the selected texture.
It displays nothing if the texture is not used in a texture pattern animation.
Info
Related Links Double-click a material or texture pattern animation in the list to open the Samplers page of the material property window or the Curve Editor.
