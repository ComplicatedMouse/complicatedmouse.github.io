Bitmap fonts and scalable fonts display in different sizes.

Question

My bitmap fonts and scalable fonts display with slight differences.

Answer

Kerning calculation methods differ.
Kerning calculation methods differ. When you draw text using a bitmap font (BFFNT file) created from a scalable font, the result can differ slightly from text drawn by ScalableFont with the same-sized scalable font.

When creating your bitmap fonts, specify the FontConverter option Uses the scfont module as source from which kerning information is extracted.
With FontConverterConsole, specify -isck.
