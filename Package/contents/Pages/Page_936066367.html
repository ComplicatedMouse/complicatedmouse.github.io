Describes the features for configuring detailed alignment settings for individual panes.

Items set with animation in the parts layout cannot be overwritten in the body layout.

The specification is designed such that items set with animation in a parts layout cannot be overwritten in the body layout.
As a result, the animations in the parts layout are played in the game application even if you overwrite the settings in the body layout.

Related Links For more information, see Troubleshooting > Items set with animation in the parts layout cannot be overwritten in the body layout..
