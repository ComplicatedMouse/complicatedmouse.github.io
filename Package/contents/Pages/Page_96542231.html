Noise Strength

This section describes the Noise Strength feature for setting the amplitude of the waveforms used by particles.

Overview

The Noise strength feature sets the amplitude of the waveforms that are used when particles move.
The following figure illustrates the ↕ range being configured.

The larger the value, the wider the range of movement for the particles.


The following overview describes the settings ().

Noise Strength

The Noise strength value is set as x-, y-, and z-coordinates with the emitter as the origin, using either the dials () or by entering values in the text boxes ().

Info

Press the Particle Time Animation button () to display the Noise Strength (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().
