Align Phase of Each Element

This section describes the Align phase of each element feature, which makes it possible to set whether to align the phases of the waveforms used by particles.

Overview

The Align phase of each element feature makes it possible to set whether to equalize the shifts in phase and thereby align the phases of the waveforms used each time particles are emitted.
Aligning the phases makes it possible to unify the movements of large numbers of particles.

The following overview describes the settings ().

Equalize Phases of Components

Use the check box () to select whether to enable equalization of the phases of the waveform components.
Enabling the feature displays the Align phase of each element group (). The group has the following settings.

Phase change rate This feature sets the speed at which to move the phase of the equalized waveforms.
Phase variability (%) This feature sets an amount of variability for the phases of the equalized waveforms.
