bcat Library

Introduction

Purpose of This Documentation

This documentation describes the features of the background contents asymmetric sync delivery and transmission (bcat) library in the NintendoSDK.

Intended Audience

This documentation is intended for people who develop the BCAT features of applications using NintendoSDK.

Related Documentation

For more information, see the following documents.

  • BCAT API Reference
  • Nintendo Switch BCAT Data Delivery Guide (English, Japanese)
  • BCAT Management Tool Usage Manual (English, Japanese)

bcat Library Features

The bcat library provides the following features.

  • Data delivery
    • Features for downloading data registered in Nintendo servers to the system

Using the bcat Library

  • NX Add-On 3.x.x is required in order to use the bcat library. The bcat library cannot be used with NX Add-On 1.x.x.
