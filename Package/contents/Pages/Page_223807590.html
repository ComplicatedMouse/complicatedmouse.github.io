3. Export

This section of the tutorial describes the procedure for exporting models, textures, and animation data from Maya.

Selecting a Model

Select and set the model to export.

Select TopL from the outliner.

Export

From the Maya menu, select Nintendo > Export with Setting.

To export, perform operations in the order indicated by the following diagram and table.


Order Category Operation
1 Preset

Select TutorialDefault.
2 Export Target Configure Selection.
3 Output Filename

Enter ExportTripod.
4 Output Folder

Click the folder icon and specify the following SampleTripod folder.

%NINTENDO_SDK_ROOT% \ Samples\Sources\Applications\G3dDemo\Resources\SampleTripod

5 General Options For Magnify, enter 1.000. For Texture SRT Mode, select Maya.
6 Animation Options

Set Start / End Frame to Range and enter 0 and 220. Outputs animations in the range from 0 to 220 frames.

Set looping playback for the animation. Select Loop. (You must select this option for the skeletal animation data items to be selected in 7.)
7 Output File Selection Select Model Data, Texture Data, and Skeletal Animeation Data.
8 Export

Click Export. Files will be output according to the specifications set in items 1 to 7.

Confirming

Verify that the following files are output.　The model (FMDB) and animation (FSKB) files will be output with the same name.

Folder
%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SampleTripod
File Content
ExportTripod.fmdb Model files
ExportTripod.fskb Skeletal Animation
/textures/SampleTripodA01Albedo.ftxb Texture
