2. Loading Samples

This section describes how to load the models and the parent material in 3DEditor.

Drag the File

Drag the following sample project file to 3DEditor.

Location %NINTENDO_SDK_ROOT% \ Samples\Sources\Applications\G3dDemo\Resources\SampleIceCream\TutorialSample.f3pj

This loads the following models and materials.

Connect to the Viewer

Click the Not connected icon () to connect to 3dSampleViewer. If the process is successful, the icon changes to Connected () and the models are displayed in the viewer.

Info

Nintendo Target Manager must be running.

The models are displayed in 3dSampleViewer.
