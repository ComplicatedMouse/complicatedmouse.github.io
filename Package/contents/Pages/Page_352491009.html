LayoutEditor

This section introduces the new features in LayoutEditor.

Guide

A guide feature has been added to be used as a scale when placing a plane.
Show/hide, and snap to guidelines can be freely toggled.

Guides can be shown using the following methods.

  1. If the ruler is not shown, select View > Show Ruler (shortcut: CTRL+R).
  2. Place the mouse cursor over the ruler on the left side to create a guide in the vertical direction, and place the mouse cursor over the top ruler to create a guide in the horizontal direction.
  3. Drag the ruler to any position in the layout window.

Reference Colors

A reference color feature has been added to assign colors consolidated between multiple layouts.

The reference colors are stored in a reference color file (FLRC) that is placed under the Parts root. The reference color file, and reference colors, can be created from the Color Sample window.
Reference color files placed under the Parts root, are automatically loaded when LayoutEditor is started.

You can assign reference colors by dragging them from the Color Sample window to color buttons in the property window.
If you change the color of a reference color in the Color Sample window, it automatically updates the color assigned to the pane.
To change the color, open the color picker from the property window for an item assigned a reference color. Alternatively, delete the reference color assignment.
You can specify whether to include alpha settings in a reference color. If they are not included, you can set the alpha in the property window.

The following is a video of dragging a reference color to a color button in the property window.

Note: The following are notes about this version.

  • Reference colors do not support color animations in Nintendo SDK 6.2.0.
    Reference colors may not be displayed properly if they are assigned to items that also have an animation configured. This problem has been corrected in Nintendo SDK 6.3.0.
  • If a reference color is renamed or deleted in the Color Sample window after reference colors have been assigned to a layout, errors may occur when layout editor is opened or a binary conversion is executed without resolving the references.
  • Reference color is not one of the individual settings displayed in the Color Sample window. When assigned by drag-and-drop, only the value of the color is overwritten.

Developer Information

You must specify a directory that contains a reference color file (FLRC file) in the parameter when performing binary conversion on a layout that uses a reference color.
A binary version of the layout is generated using the color values that have been assigned to it from the reference color file.

Text Layout with Fixed Kerning

A feature has been added where, when placing text in a circle, you can lay out text with the kerning set to a fixed value, and set the typographic alignment.
When arranging text with a small number of characters on an arc with a large radius, you can set the kerning so that characters are not too far apart.
You can also use the typographic alignment to set where on the arc to set the text.
