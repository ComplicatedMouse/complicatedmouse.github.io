Creating Part Animations

This section describes a sample procedure for creating parts animation for a SELECT Button in a parts layout.

Preparation

To prepare for creating the layout data, start LayoutEditor in advance.

Related Links For more information about starting LayoutEditor, see Workflow.

Opening Windows

Begin the task after opening the following windows.

  • Parts Palette
  • Parts and Control Settings
  • Hierarchy
  • Property
  • Group Window

Related Links For more information about the UI of each window, see Windows.

Opening Parts Layout

Open Menu_Bar.flyt for parts related to Menu.flyt.

In the Parts Palette list, right-click Menu_Bar.flyt, and then in the shortcut menu, select Open.
Menu_Bar.flyt is loaded in the LayoutEditor.

Using a Part as a Button

In the Parts and Control Settings window, set the Menu_Bar.flyt part to be used as a button.
Click the Browse button () to open the Control Settings dialog box.
Change the Control Type as follows.

Control Type Standard button [ NormalButton ]

In the Control Settings dialog box, click OK to close the dialog box.
The settings in the Parts and Control Settings window will have been changed.
You can use the Parts Pane as a button to use the Feature Animation features.

At this stage, confirm that the Warning mark () is displayed in the Feature Pane and the Feature Animation windows.

Setting Cursor and Button Alignment

To determine the cursor alignment in the Feature Pane, in the Boundary Pane, specify Bounding ().
The rectangular range of the Boundary Pane is the button determination area.

Name Pane Name
Hit Bounding

At this stage, confirm that the Warning mark () that had been displayed in the Feature Pane has disappeared.

Setting Animations for Button Features

In Feature Animation, set the animation to be played and the playback range corresponding to button features (as a number of segment tags).

Click the Generate Segment Tag button ().
In the Segment Tag Generation and Settings to Use in Feature Animations window, select the check boxes for all feature animation names, and then click OK.
The segment tags are automatically set according to the button features, as follows.

At this stage, red text indicates error states, as shown in the following table.

Name Segment Tag
OnOff OnOff
Down Down
Disable Disable

Related Links For more information about creating and setting segment tags for use in feature animations, see here.

Configuring Groups

The following procedure uses the Groups window to create a group of panes for which animation is set.
The procedure for creating groups is as follows.

1

In the Hierarchy window, select the following nodes.
N_Menu Band Band_Fade Select Text

2 Right-click, and in the shortcut menu, select Create Group.
3

In the Groups window, right-click G_group_00 () and in the shortcut menu, select Edit Selected Group.
The Edit Selected Group dialog box is displayed.
4 Change the group name to Button and click OK to close the Edit Selected Group dialog box.
5

In the Groups window, check for Button ().

Related Links For more information about the group feature, see here.

Configuring Animation Segment Tags

Configure segment tags in the Edit Animation Segment Tags window.
For each segment tag, set the playback range for animation and the pane in which to play animation.

Confirm that merged mode () is selected in the toolbar of the Edit Animation Segment Tags window.
In the frame segment on the right side of the window, change the animation segment tag settings as follows.

Number Name Segment Comments

0

 OnOff 0 - 4 Sets the animation to use when the mouse is moved over a button.
1 Down 14 - 22 Sets the animation to use when a button is clicked.
2 Disable 32 - 34 Sets the animation to use when a button is disabled.

Use the following procedure to set the groups for an animation segment tag.
This operation is executed with all of the OnOff, Down, and Disable animation segment tags that were created.

1 Select an animation segment tag, and then click the Specify Related Groups button to open a Group Selection window.
2 In the Group Selection window, select the Button check box and click OK to set the related group.

Related Links For more information about the Edit Animation Segment Tags window, see here.

Set an animation.

Create animation according to the animation playback range set in the Edit Animation Segment Tags window.

Binary Export Settings

In the Edit Animation Segment Tags window, set the animation items to target for binary output.
Confirm that the following Binary Export settings are used.

Number Name Auto-Update Playback mode

0

 OnOff On One Time
1 Down On

One Time
2 Disable On One Time

If Auto Update is selected ( On), animations that do not have key information are excluded from the export targets, which cuts down on memory and computational load.
