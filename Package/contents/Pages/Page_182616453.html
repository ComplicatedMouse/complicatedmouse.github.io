Color Picker

The Color Picker provides a visual way to set colors.
When colors are dragged in the Color Picker, the display of the layout window or property window is updated accordingly.

A Current Color Displays the current color.
B

Adds the current color to the color palette.

Note: The alpha component is not added.
Color Palette

Click a color in the color palette to select that color.
The selected color will be set as the current color.
C Color Hues Changes the hue of the color.
D Color Saturation/Brightness Changes the saturation and brightness value of the color.
E Hexadecimal Input Fields Use these fields to enter RGB hexadecimal values.
F

Use the Convert to linear gamma feature to convert sRGB colors to linear.

Use the Convert to sRGB gamma feature to convert linear colors to sRGB.

G
Switches between RGB format and HSB format for the display of color components.
Color Component Display Enter a numeric value for the color or set it with the slider.
