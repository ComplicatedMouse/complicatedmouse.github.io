Textures

Textures

Sets texture settings in the LayoutEditor pane.

Add/delete

Adds or deletes textures.

Switches the order of registered textures.

Add...

Adds a texture from the textures registered in the Textures window.
You can also add textures by dragging from the Textures window to the thumbnail.

Delete...

Deletes a registered texture.
Change... Changes a registered texture and switches it with a different texture.

Texture Thumbnail Display

 Displays thumbnails of the textures registered from 0 to 2.
Click a thumbnail image to display the settings for that texture.

Display Combiner Settings ()

Click the Display Combiner Settings button () to display the settings.

Total Number of UV Coordinates Enter the number of texture coordinates to use.

When you add textures, you can increase the total number of UV coordinates by the number of added textures.

From the Image Settings tab, use Mapping Method to set which coordinates to use for each of the textures.
Info

You can also drag a thumbnail to another thumbnail to change the order.
