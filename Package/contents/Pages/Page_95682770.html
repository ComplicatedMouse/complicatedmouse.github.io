UV Scale Animation

This section describes the feature for scaling the UV of the registered texture.

Overview

The UV Scale Animation feature enables you to scale a registered texture by scaling its UV coordinates.
The following overview describes the settings ().

UV Scale Animation

A feature for setting the texture UV scale on each axis separately.
X corresponds to texture coordinate U, and Y corresponds to texture coordinate V.

Select this option to display the UV Scale Animation () settings. The group has the following settings.

Add-on value

Sets the value to add to the texture UV scale in every frame.
This add-on value is applied after the emitter starts.

Initial value

Sets the initial scale value for texture UV.

Random initial values

Specifies the use of random numbers in a set range for the initial values.
The upper and lower limits of the random seed for the initial value randomly change. The following formula calculates a lower limit for the random number.
