NintendoWare Spy News

This page introduces the changes in NintendoWare Spy library for NintendoSDK 3.1.0.

Made it possible to display the remaining size of the file read cache in SpyAtk.

When using the file read cache feature, for the SpyAtk stream sound panel, it is now possible to display the remaining cache.


To display the remaining cache, pass nn::atk::SoundArchivePlayer when calling the nn::spy::atk::AtkSpyModule::PushAtkProfiles() function.

For more information about the panel information, see the Atk Stream Sound Panel page.
