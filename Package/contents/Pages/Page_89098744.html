Manual

Introduction

This manual describes the individual features provided by the mem system library.

It begins by describing topics that are common throughout the mem library.

mem Library Namespace

The types, constants, classes, functions, and other resources provided by the mem library are all located in the nn::mem:: or nn::lmem:: namespace.

Do not use any of the types, constants, functions, or other resources located deeper than this level unless explicitly specified.
Below that level, the hierarchy and resources are subject to change without notice.

mem Library Header Files

include the following header files as needed.

  • To use the expanded heap: nn/lmem/lmem_ExpHeap.h
  • To use the frame heap: nn/lmem/lmem_FrameHeap.h
  • To use the unit heap: nn/lmem/lmem_UnitHeap.h
  • To use the StandardAllocator: nn/mem/mem_StandardAllocator.h
  • To use NumberLineAllocator: nn/mem/mem_NumberLineAllocator.h

The directory structure below nn/lmem/ and nn/mem/ is subject to change without notice.
