Time Slider

This section describes the features displayed in the Time Slider in the Curve Editor window.

The Time Slider displays the current frame, animation range, and key frames for the element selected in the attribute tree.

Time Slider
Displays the range between the playback start frame and playback end frame specified in the Range Slider.

A Current Frame The current frame is indicated with a blue line.
B Current Frame (a numerical value)

The current frame is shown as a number.

You can enter a number here to move the current frame.
C Key Frames The key frames are displayed as orange lines.
D Key Range

The range where keys have been set is shown in light blue.

(In the figure, keys are set in the range between frame 0 and frame 80.)
