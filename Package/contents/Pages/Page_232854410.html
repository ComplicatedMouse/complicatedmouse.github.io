Manual Emission Interval

This section describes the Manual Emission Interval feature for setting the interval of time (frames) for the emission of particles from a manual emitter.

Overview

The Manual Emission Interval feature provides a way to set the interval for particle emission from a placed emitter.
When Manual Emitter Emission is selected, the emission interval configured in the Emission Setting group () in the Emission tab is ignored, and instead the setting configured for Manual Emission Interval is applied.

Info

Manual emission takes place when the program issues the command for emission, and it is not effected by the emitter's emission settings or the emission interval of the preview feature.

The following overview describes the settings ().

Manual Emission Interval

Use the dial () or text box () to set the particle emission interval.

Example Configurations
