Latitude

This section describes the Latitude setting for a Sphere emitter shape.

Overview

The Longitude feature opens a spherical emitter in a dome shape in the y-axis direction (zenith) by opening the south (base).
The emission area can be made a half-spherical dome while maintaining the particle density.

The following overview describes the settings ().
Set Method of Arc Opening to Latitude to display the Latitude () settings. The group has the following settings.

Axis

Sets the axis direction in the world coordinate system. The direction of the vertex is + (plus).

Arc Width (Angle)

Sets the arc width as an angle with the starting point at the vertex.
