Emitter Set Using Specified File

This section describes the Emitter Set Using Specified File feature to search for an emitter set using specific reference files.
This feature has the following dedicated settings.

Emitter Set Using Specified File Select File Dialog Box
A Display Dialog Box ()

Click to open the Select File dialog box.
B Select File Dialog Box

The dialog box lists the following files under the search origin path.
C Text Box If the files selected in the dialog box are registered, their filenames are displayed here.
D Clear () Click to clear a registered file.

Overview

The following table describes this preset.

A Search origin path The path specified in the address bar. The search begins at this location.
B Specified file ()

The file selected in the dialog box.

Searches for the emitter set that references the file in this range.
C Emitter set ()

Not a search target of this feature.
