MessageQueue Class

Feature Overview

The MessageQueue class is a synchronous communication feature for sending and receiving data between threads.

The MessageQueue class wraps the message queue feature using the nn::os::BarrierType object and contains the following member functions. For more information, see the API reference.

Name Description Remarks
MessageQueue Constructor Executes initialization equivalent to InitializeMessageQueue.
~MessageQueue Destructor Executes destruction equivalent to FinalizeMessageQueue.
Send Transmits to the end of the message queue Standby transmission to the end of the message queue equivalent to SendMessageQueue.
TrySend Same as above (polling) Try sending to the end of the message queue equivalent to TrySendMessageQueue.
TimedSend Same as above (time limited) Time limited send to the end of the message queue equivalent to TimedSendMessageQueue.
Jam Transmits to the front of the message queue Standby transmission to the front of the message queue equivalent to JamMessageQueue.
TryJam Same as above (polling) Try sending to the front of the message queue equivalent to TryJamMessageQueue.
TimedJam Same as above (time limited) Time limited send to the front of the message queue equivalent to TimedJamMessageQueue.
Receive Receive from the message queue Standby reception from the message queue equivalent to ReceiveMessageQueue.
TryReceive Same as above (polling) Try receiving from the message queue equivalent to TryReceiveMessageQueue.
TimedReceive Same as above (time limited) Time limited reception from the message queue equivalent to TimedReceiveMessageQueue.
Peek Reference message queue Standby referencing the message queue equivalent to PeekMessageQueue.
TryPeek Same as above (polling) Try referencing the message queue equivalent to TryPeekMessageQueue.
TimedPeek Same as above (time limited) Time limited referencing of the message queue equivalent to TimedPeekMessageQueue.

Examples of Using the MessageQueue Class

This example implements a StackBuffer class using the MessageQueue class.

#include <nn/os.h>

class StackBuffer
{
public:
    StackBuffer() : m_MessageQueue(m_Buffer, s_Count) {}

    void Push(uintptr_t data)
    {
        m_MessageQueue.Jam(data);
    }

    uintptr_t Pop()
    {
        uintptr_t data;
        m_MessageQueue.Receive(&data);
        return data;
    }

    bool IsEmpty()
    {
        uintptr_t data;
        return m_MessageQueue.TryPeek(&data);
    }

private:
    static const int       s_Count = 8;
    uintptr_t              m_Buffer[s_Count];
    nn::os::MessageQueue   m_MessageQueue;
};

The StackBuffer class is a class that provides a buffer to perform stack operations that use the MessageQueue class.

First, the constructor initializes an instance as a message queue to handle data for eight messages. The m_Buffer[] region is used as a queue buffer.

The Push and Pop functions are used to accumulate and get data. Push accumulates data passed to parameters in the start of the buffer. At this time, if the buffer is already full, the process waits until there is space in the buffer. On the other hand, the Pop function extracts and returns data from the buffer in the reverse order (stack operations) of the Push order. At this time, if the buffer does not have any data, the process waits until data accumulates in the buffer.

Use the IsEmpty function to check if the buffer has any data. If the return value is true, then there is accumulated data, if false, then there is space in the buffer.
