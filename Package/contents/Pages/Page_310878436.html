Overview

This setting checks for network connections and sets the testing feature. You must configure the network settings before a network connection can be established.
For more information, see Registering Network Connection Settings.


How to Use

Settings

Info

This is available in NX Add-On 14.0.0 and later.

It starts the Internet Settings screen of the HOME Menu.

Export/Import/Delete

Info

This is available in NX Add-On 8.1.0 and later.

It performs the following operations.

Buttons Functionality
Export

Exports the network connection settings that have been registered to the device to an SD card. The file path for the export is NintendoSDK/network_settings.json. If a file already exists in that same path, it is overwritten.

Warning

When exporting, the access point and proxy passwords or passphrases are sent as plain text, so be careful when handling the output files.
Import Imports to the device any network connection settings saved to an SD card. The file path for the import source is the same as for the export: NintendoSDK/network_settings.json. The device must be restarted for the imported settings to take effect.
Delete Deletes the network connection settings that have been registered to the device. You must restart the system to apply any changes that result from deleting those settings.

Submit network request

Click this button to attempt to make a network connection. The availability of a network is displayed to the right of the button.

Status Display
Network is available. Available
Network is not available. Not available (<Result value>, <error code>)

The Result value is presented in hexadecimal by the GetInnerValueForDebug() function. The error code is an eight-digit number presented in the format xxxx-yyyy.

Testing mode

Use this setting to specify whether to use the test feature for enabling and disabling wireless and wired communications. For more information, see the nifm Library API Reference. This is disabled by default.

  • nn::nifm::SetWirelessCommunicationEnabledForTest()
  • nn::nifm::SetEthernetCommunicationEnabledForTest()

Acceptable Network Type

Info

This feature is available in NX Add-On 8.1.0 and later.

Any unselected network types become unavailable. This setting is temporary. Restarting the device reverts to the default settings. This is enabled by default.
