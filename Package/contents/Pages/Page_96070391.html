Random Spin Direction

This section describes the Random spin direction feature to randomize the rotation direction for particles upon emission.

Overview

The Random spin direction feature allows you to randomly set the direction in which particles rotate when emitted.
The following overview describes the settings ().

Random Spin Direction

You can enable or disable the randomization of the spin direction during particle emission by selecting or clearing the check box ().
You can set spin direction randomly for axes for which this feature is enabled.

Example Configurations

The following figure compares what happens when the various Random spin direction settings are selected and cleared.

ON The spin direction of particles when they are emitted is random.
OFF The spin direction of particles when they are emitted is fixed at counterclockwise.
