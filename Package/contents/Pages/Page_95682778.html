Texture Pattern Animation Playback Methods

This section describes the playback methods for Texture Playback Animation, a feature for using textures as pattern animations.

Overview

A texture "pattern" refers to the individual images of a texture that is comprised of many images, divided using the Pattern Division feature.
Playback Method is the feature for setting how to play an animation of the divided pattern that is animated by moving texture UV in a stepwise fashion.

Multiple images for a single texture are switched out, so compared to switching multiple textures, the resource load for the loading of textures is lighter.

Playback Methods

Select the animation playback method from among the following presets.

None Patten animation is not conducted.
Fit to Lifespan Plays the animation so it cycles to match the particle lifespan.
Clamp This method animates each pattern for the interval set for Sequential frames per pattern.
If one cycle is shorter than the lifespan, the final pattern is displayed.
Loop This method animates each pattern for the interval set for Sequential frames per pattern.
If one cycle is shorter than the lifespan, the animation loops.
Random A pattern to display is selected randomly each time a particle is emitted.
Surface Pattern animation using array textures. This can be used for pattern cross-fading.
