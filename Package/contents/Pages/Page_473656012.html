FcpxMaker

This section introduces the new features in FxpxMaker.

Comments Feature

You can now add comments to fonts in FcpxMaker.
These comments get added to the ScalableFontDescription, BitmapFont, and MultiScalableFont nodes in the FCPX file.
However, any font comments that are added cannot be viewed in LayoutEditor.
The following image shows the comment settings added to the property windows for bitmap fonts, multi-scalable fonts, and scalable fonts.
