Target Axes

This section describes the Target axis feature for restricting the direction in which the magnetic force is applied to the specified axis.

Overview

The Target axis feature allows you to restrict the direction in which magnetic force is applied separately for the x-axis, y-axis, and z-axis.
The following overview describes the settings ().

Target Axes

Enable or disable which directions (X, Y, Z) the magnetic force is applied by using the check box. ().
