Color Animation

Color animation is set by animating a material's Diffuse Light, Specular Light, Transparency and Self-illumination.

For color animation, data is exported only for parameters for which animation has been set.

Parameters for which data is not exported are shown in their state in the model file.
Even if a parameter has a fixed value, if that value differs from the value in the model file, make sure that you export the color animation after hitting at least one key for that parameter.

Info

If the Compress Material export option is enabled, even materials for which color animation is not set sometimes get animated as a result of united materials. If this happens, use the NW4F Set Material Attribute plug-In to change settings so that materials for which color animation is set are not compressed.
