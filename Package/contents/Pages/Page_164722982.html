Extended User Information Settings Window

The Extended User Information Settings window is used to edit the extended user information in LayoutEditor.

Extended User Information Settings Window

Name

Enters the name of the extended user information.

Click to display a list of the extended user information presets described in the project settings files located below the Parts root. You can use the list to configure settings.
The presets are set in the FLPJ and FLUP files.


Related Link For more information, see Extended User Information Presets.

Type

Configures the type of the extended user information.

  • String
  • Integer value list
  • Real number value list
  • One-dimensional integer animation
  • Two-dimensional integer animation
  • RGBA color animation
  • One-dimensional real number animation
  • Two-dimensional real number animation
  • Three-dimensional real number animation
  • RGBA color animation (with gamma correction)

Enabling the extended user information from the Base pane makes it possible to select the animation type.

Value:

Displays the value of the extended user information.

Enter one value per line for each numerical type.

If choices have been prepared in the extended user data presets, select an item and set the value.
Info

Specifying RGBA color animation (with gamma correction) as the type, applies gamma correction to each of the V key values of the curve in the layout binary converter, and converts them to 255*(v/255)^2.2.
