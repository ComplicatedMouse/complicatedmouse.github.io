Animation Property Window: Pattern Page

This page is used to check and edit the textures and texture numbers used in texture pattern animations.
Note: This page is displayed only in the texture pattern animation Property window.

A Moves up and down to select patterns in the texture pattern table (D).
B Opens the Add Texture Pattern dialog box and allows you to add textures loaded in 3DEditor to patterns in the texture pattern table.
C Removes the selected texture from a pattern in the texture pattern table (D).
D The Texture Pattern List.
This table number is the texture number when texture pattern animation is used.
E Filename of the texture selected in the texture pattern table.

G Displays the full path of the texture.
H The color image of the texture.

J Opens the Texture Property Window.

Adding Texture Patterns

  1. Open the texture file used by the pattern in the 3DEditor.
  2. Open the FMAB material animation file for the preview or create a new material animation.
  3. In the Property window, click the pattern.
  4. Click Add (B in the table in the preceding section) to open the Add Texture Pattern dialog box, and add a texture pattern. (For more information about dialog boxes, see the following section.)
    (Textures can also be added by dragging them.)

Add Texture Pattern Dialog Box

This dialog box displays a list of the textures currently loaded in 3DEditor. You can select and add multiple textures to a pattern at the same time.

Info

Select the check box for a texture to add it. Unselected textures are not added.
Textures that have already been added to the texture pattern table are not included in this list.

Adding and Replacing Textures by Dragging

You can add or replace textures by dragging them from the file tree view or Windows Explorer to the pattern list in the Property window.

When textures are added with this method, the Add Texture dialog box is not displayed.

When a texture is dragged to an empty region in the pattern list, the texture is added by adding to the pattern number with the maximum value.
When a texture is dragged to a pattern number that has a texture set, that texture is replaced.

Change Texture Image Dialog Box
