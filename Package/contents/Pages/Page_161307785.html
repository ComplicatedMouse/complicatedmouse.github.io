Manual

Introduction

This manual describes the various features provided by the ro library.

It starts by describing topics that are common throughout the ro library.

ro Library Namespace

The types, constants, classes, functions, and other resources provided by the ro library are all located in the nn::ro:: namespace.

Do not use any of the types, constants, functions, or other resources located deeper than this level unless explicitly specified.
Below that level, the hierarchy and resources are subject to change without notice.

ro Library Header Files

Include only the nn/ro.h header unless explicitly specified otherwise.
The directory structure below nn/ is subject to change without notice.

ro Library Files

ro library features are available as SDK DLLs, and are automatically dynamically linked to applications.
