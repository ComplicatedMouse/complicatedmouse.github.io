Layout Window

★ Layout Window

This section describes the names and roles of the various parts of the Layout window.
  • A Layout screen region
  • B Coordinate system origin
  • C Toolbar
  • D Additional menu bar items (LayoutEditor window)
    When opening the Layout window, items are added to the menu bar of LayoutEditor.

A Layout Screen Region

The layout screen region corresponds to the screen shown on the actual hardware.
Create your layout by first setting a screen region of the size required for your task.

Info

Use the Layout Settings window to configure colors, grid lines, and other settings for the Layout window.

Related Links For more information, see the following pages.

Layout Settings Window Describes the UI for layout environment settings.

B Coordinate System Origin

The red line drawn in the center of the screen is the coordinate origin of the layout screen region.
(The layout coordinate system is a three-dimensional coordinate system consisting of a two-dimensional xy-axis plus a one-dimensional z-axis.)

C Toolbar

The menu bar contains the following items.

Displays the state of undo and redo operations. Each action performed in the layout is counted, incrementing the number that is displayed.
Undo operations are counted and the number is shown in parentheses. The greater the number of undo operations, the smaller the number of tasks done in the layout.
The coordinates of the cursor. "Tweaking" is displayed while a pane is being moved. "Selecting" is displayed while a region is selected.

Displays statistical information for the layout.

Resting the pointer over the icon displays the statistical information. Clicking the icon displays the statistical information in a window.

Displays whether translation of parent panes affects children.
Clicking the icon switches the icon shape and changes the behavior.

To have changes affect children, select Pane > Apply Parent Translations to Children.

Switches between perspective projection and orthogonal projection.
Clicking the icon switches the icon shape and changes the projection.

The icon during orthogonal projection.

A projection method where the object size is not affected by distance from the camera viewpoint.

The icon during perspective projection.

A projection method where objects in the near distance from the camera viewpoint look larger, and objects far away look smaller.

To display a list of related parts, click the icon.

To change the editing target, click a target from the displayed list of related parts.

To open a separate window, drag the target from the displayed list of related parts.
Combo box and slider. Use +/- to zoom in and zoom out.

D Additional Menu Bar Items

When opening the Layout window, items are added to the menu bar of LayoutEditor.
These items are executed for the active Layout window.

Panes

Operations to create panes, set hierarchies, lock panes, and configure other pane settings.
Select

Select or clear the selection for panes to display them in the Layout window.
Display

Switch the display state of the Layout window.
Tools

Operations for the viewer and the tool.
