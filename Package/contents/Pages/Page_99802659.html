Playback Speed

This section describes the Playback Speed feature, which can be used to change the playback speed.

Overview

Use the Playback Speed feature to change the playback speed on the viewer by multiplying the speed by the specified value.
Use the viewer to check what the specified playback speed looks like in the game.
The following overview describes the settings ().

Playback Speed

The playback speed is multiplied by the value specified using the dial ().
