Changing the Bluetooth Driver Software (Windows 7 Only)

Overview

Use Microsoft's Bluetooth driver software to connect the controller via Bluetooth and use it in a PC environment.
When using a commercially-available Bluetooth dongle in a Windows 7 environment, Bluetooth driver software for the company that manufacturer the dongle (and not Microsoft) will be installed by default. You must change to use Microsoft's software instead.

Note: The Microsoft Bluetooth driver software is used automatically in both the Windows 8.1 and 10 environments. In those cases, this procedure is unnecessary.

Changing the Bluetooth Driver Software in a Windows 7 Environment

  1. Open Control Panel. Click Hardware and Sound.
  2. Click Device Manager.
  3. From the device list, double-click Bluetooth Devices. Right-click the device that you want to update, and then select Update Driver Software.
  4. Select Browse my computer for driver software.
  5. Select Let me pick from a list of device drivers on my computer.
  6. Make sure that the Show compatible hardware check box is selected. From the model list, select Generic Bluetooth Radio, and then click Next.
  7. The driver software starts installing.
  8. The following screen completes the switch in Bluetooth driver software. Click Close to end the wizard.
