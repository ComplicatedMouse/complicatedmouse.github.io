Load Sequential Textures Window

The Load Sequential Textures window is a feature for loading texture files with certain names within a specified folder as pattern animations for the created pane.
Animations are automatically generated in the order of the numbers appended to the end of texture filenames.

Info

The following naming convention is used texture files.
TextureName.XXX.tga (where XXX is any number)

Menu bar > Tools > Load Sequential Textures.

Load Sequential Textures Window
A Target Load Folder

Enter the path of the folder with the sequential textures you want to load.
For the filenames of the texture files you provide, use the following format: TextureName.XXX.tga, where XXX is any number.
B Start Frame Enter the frame at which to begin sequential texture pattern animation.
C Pattern Frame Interval Enter the interval, in frames, at which to switch the sequential texture pattern animation textures.
D Repetition Count

Enter the number of times to repeat the sequential texture pattern animation.

(If you are using mirroring to repeat the animation, include the mirror playback portion in the repetition count.)
E Repetition Method
Repeat
 Plays the textures in the order of the texture name numbers, sequentially.
Mirror
 Plays forward in sequential order, and then reverses that sequence in playback.
F Load Locks in the settings and loads.
The sequential textures are loaded from the folder into the Textures window.
Info

The sequential texture animation stops before finishing and starts over when the length of the animation set using the Curve Editor is shorter than the length of the animation playback for the configured sequential textures.
