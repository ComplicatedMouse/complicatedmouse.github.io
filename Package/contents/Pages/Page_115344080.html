NintendoWare Effect News

This document is a graphical introduction to the features that were added in NintendoSDK 0.10.0.

Starting Process Added a feature to determine behavior when particle emission begins.
Starting EffectViewer From the GUI You can now start EffectViewer from the EffectMaker GUI.
Browsing for Samples From the Effect Browser You can now load effect sample data from the Effect Browser.

Node movement control is now possible by dragging the mouse in the workspace without using the ALT key.

New Effect Expressions

This section introduces features that broaden the scope of what can be depicted with effects.

Starting Process

Added a feature to determine behavior when particle emission begins.
It can reduce the unnatural feeling from particles suddenly being displayed by displaying them gradually using alpha and scale at the time the effect appears.

Usage Examples

Click the animation Stop button () to stop the particle emission.
You can check the Start Processing effect by clicking the animation Play button () to start particle emission.
The following figure compares what happens when the various Start Processing features are selected and cleared.

A Fade off The effect without fading.
B Alpha Fade on The particles start out transparent and gradually appear over the course of the specified Fade-In Duration.
C Scale Fade on The particles start from a size of zero and and gradually enlarge over the course of the specified Fade-In Duration.

Browsing for Effect Samples Using the Effect Browser

You can now load effect sample data from the Effect Browser.
This section describes the loading procedures.

1 Select tab Select the Favorites tab in the Effect Browser window ().
2 Select Favorites tab Select the Favorites tab.
3 Select VfxSample () Select Favorites VfxSample (). Sample data is displayed in the list.
4 Select sample data Select the sample data file (ESET or PREV file) you want to load in EffectMaker.
5 Export to workspace Click the toolbar's Export to workspace button ().
You can also double-click a file to load it.
Info

To add VfxSample () to your favorites, in the Effect Browser () toolbar, click Add Sample Folder to Favorites () .
