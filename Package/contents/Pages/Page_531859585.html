Overview

You can configure the SDEV development hardware using a standard web browser.

How to Use

Use your browser to access the IP address of the SDEV development hardware.

The following items are displayed on the main page in the browser.

  • Show Information
  • Control SDEV
  • Configure SDEV
  • Capture LCD
  • Show Logs


Info

If you enable your computer to search for the network the SDEV is connected to, the device appears in Windows Explorer in Network > Other Devices.

Double-click the SDEV that is displayed to view information about that device in your browser. Click SDEV Menu to jump to the top page for the SDEV.


Show Information

Displays the following information.

SDEV Name The name that has been set for this SDEV development hardware unit. The default name is the serial number of the unit.
Model A string representing the model type of the unit. It is sometimes used when you make inquiries to verify the device.
Serial Number The serial number of the SDEV unit.
MAC Address This MAC address is assigned to the LAN port that connects the SDEV unit and the host PC.
HostBridge Firmware

Information about the firmware of the interface that connects the SDEV unit and the host PC.

Note that this information is not about the firmware of the target.

Control SDEV

This set of features checks the SDEV when inquiries are made about a unit that seems to be malfunctioning.

It includes the following items.

Status Displays the status of the DIP switches and the Recovery button on the back of the SDEV unit.
Target Control This setting is for controlling the target.
Button Control This setting is for controlling the buttons on the SDEV unit that operate the target.
Video Setting This setting is for configuring the HDMI port on the SDEV unit.
LED Control This setting is for controlling the LEDs on the back of the SDEV unit.

Status

Displays the status of the DIP switches and Recovery button on the back of the development hardware SDEV.

It includes the following items.

Wi-Fi 1 / Blue Tooth The status of the Wi-Fi 1 / Bluetooth DIP switch.
Wi-Fi 2 The status of the Wi-Fi 2 DIP switch.
DIP Switch The status of other DIP switches.
Recovery Button The status of the Recovery button.
Target Power The status of the target power.

Target Control

Can be used to confirm the ability to control the target. The basic use is for queries and for determining whether there has been a fault.

It includes the following items.

Reset Resets at a specific time via the target hardware.
Battery Connects and disconnects the power source connected to the target.
USB Port Enables or disables the USB port.
Boot Mode Used to determine whether there has been a fault in the firmware update functionality.
Shutdown Used to determine whether there has been a fault in power.

Button Control

Can be used to control the POWER Button and the volume buttons.

Power / Sleep Press the POWER Button for the specified time.
Volume Up Press the button to increase the volume for the specified time.
Volume Down Press the button to decrease the volume for the specified time.

Video Setting

Can be used to control the HDMI output image for the development hardware SDEV console.

It includes the following items. To apply changes, select Apply.

Frequency Select the refresh rate.
Rotation Rotates the screen. (This feature might be removed in the future.)

LED Control

Can be used to test the LEDs on the back of the development hardware SDEV.

It includes the following items.

Power LED

ON: On, OFF: Off, Blink: Flashing, Normal: Normal operation
SD Card LED ON: On, OFF: Off, Blink: Flashing, Normal: Normal operation
Game Card LED ON: On, OFF: Off, Blink: Flashing, Normal: Normal operation
SD and Game Card LED Blink Alternately: Turn on SD and GC alternately, Normal: Operate normally

Configure SDEV

This setting is for configuring the SDEV development hardware.

Current Setting Displays the current settings of the SDEV development hardware.
Configure Setting This setting can be used to change the SDEV settings.
Firmware Update

Can be used to update the firmware of the interface that connects the development hardware SDEV and the host PC.

There are also these other buttons at the bottom.

Reboot HostBridge Restarts the SDEV.
Restart Service Restarts the connection service between the SDEV and the host PC.

Current Setting

Displays the current settings of the SDEV development hardware.

It includes the following items.

Name The development hardware SDEV name.
DHCP Whether to use DHCP or Auto-IP. Whether to use a fixed IP address.
DNS Whether to automatically get the DNS IP address or to specify it manually.
Jumbo Frame Enables or disables the jumbo frame feature.
MTU The MTU size when using a jumbo frame.
TCP port The TCP port number to use for communication with the target.
UPnP Discovery Enables or disables UPnP discovery.
Overlay The overlay display settings.

Configure Settings

This setting can be used to change the SDEV settings.

It includes the following items. After selecting Apply, settings are applied either by pressing the Reboot HostBridge button at the bottom of the unit or by plugging and unplugging the development hardware SDEV.

Name The development hardware SDEV name.
DHCP Whether to use DHCP or Auto-IP. Whether to use a fixed IP address.
DNS Whether to automatically get the DNS IP address or to specify it manually.
Jumbo Frame Enables or disables the jumbo frame feature.
MTU The MTU size when using a jumbo frame.
Data Transfer The TCP port number to use for communication with the target. (This setting does not normally need to be changed.)
UPnP Discovery Enables or disables the UPnP discovery feature.
Overlay Display Whether to display the overlay display always or only temporarily during sleep.
Target log to syslog Can be used for verification queries.

When using jumbo frames, the default MTU size is 4082.
If an MTU size of 9000 or more can be specified on the host PC, specifying a value of around 7000 may improve transmission performance.

Firmware Update

Can be used to update the firmware of the interface that connects the development hardware SDEV and the host PC.

Update the firmware using the file included in NintendoSDK (Resources\Firmwares\NX\HostBridgeFirmware.nhf).

The target firmware is not updated.

Capture LCD

This feature captures screenshots of the images displayed on the SDEV screen.

Supports the BMP and PNG formats. It is also possible to get a vertical or horizontal screen.

Because of the hardware configuration of the SDEV, images are captured fastest as BMP images in portrait mode.

Show Logs

Displays information for inquiries about SDEV problems.
