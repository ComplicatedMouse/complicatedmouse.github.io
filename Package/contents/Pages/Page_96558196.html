Alpha Fadeout

This section describes the Alpha Fade feature for making particles gradually turn transparent and disappear when the Fade command is executed.

Overview

The Alpha Fadeout feature can be used to stop the emission of particles from emitters and to fade out and kill particles that have already been omitted when the game developer has executed a Fade command.
Particles gradually extinguish over the course of the configured Fadeout Duration.
The following overview describes the settings ().

Alpha Fadeout

Alpha Fadeout is enabled and disabled using the check box.
