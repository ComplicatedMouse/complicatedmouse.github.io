Bookmarks - Features and Operations

Overview

The nodes that can be viewed in the Sound Sets tab and the Bank tab of the Project Tree can be registered as bookmarks and called up later for viewing.

Features and Operations

Viewing Bookmarks

To open the Bookmarks panel, select View > Bookmark from the menu or click the Show Bookmarks icon in the toolbar.
These operations are used to switch between showing and hiding windows.

Adding Bookmarks

To add bookmarks, use one of the following methods.

  • Select a node from the Project Tree and then select Edit > Add to Bookmarks from the menu.
  • Right-click a node on the Project Tree, and then select Add to Bookmarks on the shortcut menu.
  • Right-click either the Sound Sets tab or the Bank tab and select Add to Bookmark from the shortcut menu.

Creating a Bookmark Folder

The Bookmarks feature supports folders so you can manage them in a hierarchy.

To create a bookmark folder, use one of the following methods.

  • On the Bookmarks panel, select Edit > Create Bookmark Folder.
  • Right-click anywhere in the Bookmarks panel and select Create Bookmark Folder from the shortcut menu.

In the Create Bookmark Folder dialog box that appears, specify a name for the folder.

Deleting Bookmarks and Bookmark Folders

To delete added bookmarks or bookmark folders, use one of the following methods.

  • In the Bookmarks panel, select a bookmark or bookmark folder, and then select Edit > Delete.
  • In the Bookmarks panel, select a bookmark or bookmark folder, and then press DELETE.
  • In the Bookmarks panel, right-click a bookmark or bookmark folder, and then select Delete from the shortcut menu.

Moving Bookmarks and Bookmark Folders

You can rearrange bookmarks in the Bookmarks panel by moving them in to and out of bookmark folders.

To move bookmarks, select one or more bookmarks or bookmark folders and drag them up or down.

Using Bookmarks

If you double-click a bookmark that has been added to the Bookmarks panel, that registered node (in the Sound Sets tab or the Bank tab) is displayed.

Rename

To change the names of added bookmarks and bookmark folders, in the Bookmarks panel, right-click the bookmark or bookmark folder and select Rename from the shortcut menu.

History - Features and Operations

Overview

SoundMaker has a feature for keeping a history of which nodes were shown in the Sound Sets tab and the Bank tab.

Features and Operations

History

To open the History panel and view a list of the nodes, select View > History or click the Display History icon in the toolbar.
These operations are used to switch between showing and hiding windows.

The program keeps a history log of up to 100 entries.

You can sort the list in ascending or descending order by clicking Name or Date.

Registering to History

When nodes are displayed in the Sound Sets tab and the Bank tab of SoundMaker they are automatically added to the history.

Deleting History

Click the Clear button at the top of the History panel to completely delete the history.

Using History

If you double-click an index cell in the history list, that node (in the Sound Sets tab or the Bank tab) is displayed.
