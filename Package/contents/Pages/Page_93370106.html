Emitter Shape - Circle (Evenly Divided)

Describes the settings for Emitter Shape - Circle (evenly divided), where particle emission locations are set equidistantly around the circumference of a circle.
These are the particles when this emitter shape is being used.

Overview

With the Circle (Evenly Divided) emitter shape, particles emit in the direction of the normal in a widening circle shape from evenly divided locations around the circle.

Info

The emission rate is a fixed value for Circle (evenly divided) and Line (evenly divided) for emitter Shapes and data using emission rate animation.
Also, the value the emission rate at the beginning of the animation is inherited by Number of divisions as a fixed value.

The following overview describes the settings ().
When the emitter shape is set to Circle (Evenly Divided), the Circle (Evenly Divided) () settings are displayed. The following settings are specific to even division in this group.

Random Angle

Emits particles from random locations along circular arcs, defined by the divisions of the emitter shape.

Related Links For more information, see Random Angle.

Emission Type

Selects the behavior type when a particle is emitted in the emitter normal direction.

Together Particles are emitted from all division locations in the direction of the normal.
Random Particles are emitted from random locations of the divisions in the direction of the normal.
By Index Particles are emitted from the division locations in the order of rendering in the direction of the normal.

Number of Divisions

This setting for the number of divisions becomes the number of particles that are emitted, and the setting for Emission rate is ignored.

Random Number of Divisions (%)

Changes the number of divisions randomly every time particles are emitted.
The upper and lower limits for the random number of divisions changes randomly. The following formula calculates a lower limit for the random number.

A Together

A emitter with a circle shape is divided into six parts, and particles are emitted all together from all locations each time.

The emission rate cannot be set in this preset.

B Random Particles are emitted from two locations each time, randomly selected from among the six divisions.
C By Index Particles are emitted from two locations each time, in order among the six divisions.
