Null Panes

This section describes null panes, which can control the panes with lower hierarchy in the Hierarchy window as a group.

Overview

Null panes are panes that can control, as a group, the panes lower in the hierarchy for the parent/child hierarchy window.
Animating null panes can animate panes in the hierarchy as a group.
Null panes are not displayed on the screen.

The panes lower in the hierarchy can inherit the following information from null panes.

  • Pane position, scale, and rotation
  • Pane transparency

The following table shows the null pane settings.

Base Pane Sets information including the pane name, position, shape, and center position.
Extended User Information Sets user-defined extended information for the pane.
