Historical Stripe

This section describes the Historical Stripe feature. With this feature, a particle’s positional information, after being generated, is stored as past samples. These samples are then used to generate stripe particles in which the polygons are connected together in a string-like shape.

Overview

Past samples in Historical Stripe are sets of information recording the position and movement direction of a particle emitted from an emitter at a set interval.
As a result, an historical stripe is a string-like set of connected particles in which this recorded positional information is used for the polygon vertices.

Historical Stripe

Historical Stripe () is a group of settings for creating particles with string shapes.

Calculation Method

Determines the orientation of the particle face.
Perfectly Follow Emitter Determines whether past samples follow the emitter when it moves.

Options

Determines the shape of the particle.
Texturing Determines how texture is mapped to the particle.
Number of Divisions Sets the number of particle divisions and smooths out the 'string'.
Number of Past Samples (Length of Stripe)

Sets the length of the stripe.
Starting point alpha, End point alpha Sets the alpha values for each end of a string-shaped particle.
Info

Two or more past samples are required before you can calculate a historical stripe for display in the viewer.
However, when one or more number of divisions have been set, a total of three or more past samples is required.
