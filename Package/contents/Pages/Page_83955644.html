Add to Position

This section describes Add to Position, which moves particles in the specified direction.

Overview

Add to Position adds the specified value to the positions of particles in every frame.
It can be used to change the movement direction of all particles that have been emitted.

Add to Position

Add to Position is a group () of features for configuring field settings.

Apply in world coordinates

A feature for switching the coordinate system.

Add-on value

This feature sets a value to add to or subtract from the particle's position.
