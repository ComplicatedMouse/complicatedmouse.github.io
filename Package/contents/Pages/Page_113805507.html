Introduction

This documentation describes the features provided by the perf library.

It starts by describing concepts that are common throughout the perf library.

perf Library Namespace

The types, constants, functions, and other resources provided by the perf library are all located in the nn::perf:: namespace.

Do not use types, constants, functions, or other resources located in deeper-level namespaces unless explicitly specified.
Below that level, the hierarchy and resources are subject to change without notice.

perf Library Header Files

Include nn/perf.h.

nn/perf.h is located in the following location. 

$NINTENDO_SDK_ROOT/Include/nn/perf.h
