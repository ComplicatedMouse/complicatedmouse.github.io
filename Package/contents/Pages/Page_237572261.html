6. Texture Pattern Animation

Create a texture pattern animation.

Textures for Pattern Animation

Replace the texture in the texture pattern animation.

Replace the texture set for the file attribute of the PatternMt material with the TexturePattern.0.tga file.

Location %NINTENDO_SDK_ROOT% Samples\Sources\Applications\G3dDemo\Resources\TrafficSignal\sourceimages\TexturePattern.0.tga
Info

The folder contains the following five numbered textures.

  • TexturePattern.0.tga
  • TexturePattern.1.tga
  • TexturePattern.2.tga
  • TexturePattern.3.tga
  • TexturePattern.4.tga

Like these files, texture files for texture pattern animations have a number attached to them before the extension that corresponds to the frame extension.

To check and export the pattern animation, the numbered texture must be in the same folder.


Image Sequence

Create an animation key for the Image Number file attribute of the PatternMt material.

Use the attribute editor to open the file node that is bound to the Color attribute.
Create the key by selecting the Use Image Sequence check box, right-clicking Image Number, and selecting Delete the expression.

Create a key like the one in the following table.
0 frames
10 frames
20 frames
30 frames
 40 frames
Image Number 0 1 2 3 4

Select the animation curve and change it to Step tangent.
The texture will switch at the intended time.

Using a window like the one for Hypershade, select the file node with the set animation, and then select the animation curve in the graph editor and change it into a step tangent.

Export Target

The texture pattern animation export target is the key frame of the Image Number file attribute.

This corresponds to D, Texture Pattern Animation Data, in the Output File Selection of Nintendo Export Options.

Related Links For more information, see Texture Pattern Animation.

Info

Combines each of the color, texture SRT, and texture pattern animations into a single FMAB file and exports the file.
Texture pattern animations can also be exported as stand-alone FTP files.
