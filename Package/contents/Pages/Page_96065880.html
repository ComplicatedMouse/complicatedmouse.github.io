Randomness (%)

This section describes the Randomness (%) feature that randomizes the base size of particles upon emission.

Overview

The Randomness (%) feature randomizes the scale when particles are emitted.
Randomizing the size of each particle when multiple particles are emitted at the same time adds visual texture to the effect.
The following overview describes the settings ().

Randomness (%)

The calculated in Randomness is used as the upper limit of the random value. The lower limit changes randomly. The following formula calculates a lower limit for the random number.

  • Random Number Lower Limit: Calculated as Standard value minus the calculated Randomness.

Info

To make the random number upper limit larger than the standard value, specify a percentage greater than 100% Randomness.
