3DEditor

Added a feature for previewing the mirroring of bone visibility animations.

When mirroring is enabled, the bones to which bone visibility animation is applied get flipped horizontally.
To preview this mirroring, open the animation's property window, go to the Preview page, and select the Enable mirroring playback check box.

Maya Plug-In

For LOD export, LOD models now always have fewer submesh divisions than the base model.

Previously, for LOD export, the LOD model sometimes had more submesh divisions than the base shape or the LOD shapes at a lower level than the LOD model. With this improvement, the LOD model now always has fewer submesh divisions than the lower-level models.

The following example checks 3dSampleViewer and confirms that the number of base shape divisions is greater than or equal to the number of LOD shape divisions.
