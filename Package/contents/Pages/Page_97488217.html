Options

This section describes the Options feature for changing the shapes of stripe particles.

Overview

The Options feature allows you to change the shapes of stripe particles.
Stripe particles can be configured to be flat or cross-shaped.
The following overview describes the settings ().

Options

Select the stripe shape from among the following presets.

Normal A flat polygon.
Cross Two flat polygons crisscrossing. This option provides apparent thickness to the stripe when viewed from multiple directions.
