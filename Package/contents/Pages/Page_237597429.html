World Coordinate System and Emitter Coordinate System

This page describes the coordinate systems handled in effects.
The standard used when placing particles changes depending on the coordinate system.

Coordinate System Types

In EffectMaker, two coordinate systems handle configuring velocity, direction, size, and angle: the world coordinate system and the emitter coordinate system.
The following figures illustrate the coordinate systems and their appearance in the viewer.

World Coordinate System Emitter Coordinate System

Particle Follow Type

When Perfectly follow emitter SRT is selected in Particle follow type, the emitter is followed.
