5.9. Global Packages

Overview

This section describes related packages.

These packages are provided as separate packages from the NintendoSDK NX Add-On.

List of Packages

Package Name Description
NVN Graphics Debugger A dedicated graphics debugger is for NVN. It provides features for debugging draw calls, states, and resources, in addition to features for performance analysis and capturing rendered frames.
Low-Level Graphics Debugger A dedicated low-level graphics debugger for NVN. Like the NVN Graphics Debugger, the Low-Level Graphics Debugger allows you to examine resources and the state of the graphics pipeline—at a lower level—in order to debug your application.
NVIDIA GameWorks A package of graphics libraries and samples using NVN.
NVIDIA CUDA A package that enables you to use CUDA in the NX environment.
RenderDoc For NX A package that enables you to use the RenderDoc graphics debugger in the NX environment.
