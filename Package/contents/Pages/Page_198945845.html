Tool Environment Settings

This page describes the Tool Environment Settings window that can be opened from the LayoutEditor Edit menu by selecting Tool Environment Settings.

Tool Environment Settings Window

A Render Select this check box to draw the display-safe region within the layout window.
B Frame Color Specifies the color of the display-safe region.
C Frame Margin Specifies the margin to set for the display-safe region.
D Do not show hidden panes in the viewer Select this check box disable display of hidden panes in the viewer.
E Convert escape characters in text boxes into control codes For use by developers for testing purposes. In most cases designers will not need to use this option.
F Do not export animation if there are no keys in the frame segment of the segment tag

Invokes the --omit-nokey-all option when running Binary Converter to preview data.

Configure this setting to match the binary converter settings in the game program.
G Handle unsupported vector graphics texture features as conversion errors. Unsupported vector graphics data features are handled as errors when this option is selected.
H Rendering Mode Currently only DirectX mode can be selected.
I FOV

Specifies the field of view (FOV) value.

FOV: Field of view.
J

Near

Specifies the Near value.

Near
K Far

Specifies the Far value.

Far
L Initialize perspective projection Select this check box to set perspective projections to their initial states.
M Perform depth testing

Select this check box to enable depth testing.

This is a rendering test used to determine the positional relationship of objects relative to one another in the depth direction.
N 30 FPS

Select this check box to set the frame rate to 30 FPS.

Clear this check box to set the frame rate to 60 FPS.
O Enable the default value of "Apply transparency to children" setting for null panes Select this check box to set the default value for the Apply transparency to children setting to Enabled for null panes.
P Arrow key shift amount You can set how much the pane shifts using the arrow keys.
Q Linear Color Space Mode

Converts textures and color parameters to linear color space in the binary converter.

Switch the rendering mode in the viewer to restore to sRGB color space for display.

If you change this setting, close and then re-open the layout window.

Related Links For more information about linear color space mode, see Linear Space in the Layout.
R Specify color parameters in linear color space

If specified, the gamma correction for color parameters in linear color space will not be performed upon conversion.

Configure this option in the team settings file after consulting with a developer.

Related Links Also see the Vertex Color and Color Correction section in Linear Space in the Layout.
S Write sRGB data to the framebuffer

Controls whether to write data to the framebuffer in the sRGB space.
This setting is usually always selected when you are using linear color space mode. (The check box cannot be cleared from within the GUI.

However, this option can be configured freely by editing it directly in the project configuration file.)


Configure this option in the team settings file after consulting with a developer.
T Fetch the sRGB for the texture

Controls whether to fetch textures in sRGB space.
This setting is usually always selected when you are using linear color space mode. (The check box cannot be cleared from within the GUI.

However, this option can be configured freely by editing it directly in the project configuration file.)


Configure this option in the team settings file after consulting with a developer.
U Display warning window for animation management mode If this option is cleared, the warning window that normally appears when switching between merged mode and separated mode in the Edit Animation Segment Tags window does not appear.
V Display a dialog box with news when updating LayoutEditor When this option is cleared, a dialog box with news is not displayed when LayoutEditor is updated.
W Move the selected panes with the layout window Enables the use of the arrow keys to move the panes that are selected in the Hierarchy window in the layout window.
X Tree view indent (hierarchy window) Configures the tree view indent for the hierarchy window.
Y Tree view indent (property window) Configures the tree view indent for the property window.
