Rendering Textures Without Waste

Rendering loads can be kept low by avoiding the use of unnecessary margins around textures, and by rendering at appropriate texture sizes.

Merits of Emitting Particles for High-Volume Particle Effects

Sometimes simply increasing the number of particles is merited, rather than writing numerous grains on a single texture.
If you can use GPU particles, the CPU cost is not an issue. Reducing the number of parts to remove based on alpha value also helps reduce the GPU load.
