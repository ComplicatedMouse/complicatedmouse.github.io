LayoutEditor

This section introduces the new features in LayoutEditor.


Added Link Playback Range Button

The Link Playback Range button has been added to the Edit Animation Segment Tags window.

Link Playback Range Button

The Link Playback Range button is a feature for linking the ranges of segment tags, created in the Edit Animation Segment Tags window, with
the ranges of the time slider, and the range slider displayed in the Curve Editor window.
Toggle the button on and off to support animation editing in the Curve Editor window.

Link Playback Range Button Is ON
A Link Playback Range The Link Playback Range button is turned on.
B Segment Tag Start and End Frame Display The start and end frames specified by this segment tag are applied in Curve Editor.
C Segment Range to Graph Display Area Display The segment tag range is shaded light gray in the graph display area.
D Time Slider, Range Slider The segment tag range is applied in the time slider and the range slider.
E Segment Tag Start and End Frame Display When you switch to another segment tag, the range display is linked in Curve Editor and also switches.
F Segment Range to Graph Display Area Display The segment tag range that was switched to, is shaded light gray in the graph display area.
G Time Slider, Range Slider The range display in Curve Editor is linked to the segment tag that was switched to, and also switches.
Link Playback Range Button Is OFF
A Link Playback Range The Link Playback Range button is turned off.
B Segment Tag Start and End Frame Display The graph display area in Curve Editor switches based on the start and end frames specified by this segment tag.
The time slider and the range slider are not linked because the Link Playback Range button is turned off.
C Segment Range to Graph Display Area Display The segment tag range is shaded light gray in the graph display area.
D Time Slider, Range Slider The segment tag range is not applied to the time slider or the range slider because the Link Playback Range button is turned off.
E Segment Tag Start and End Frame Display When you switch to another segment tag, the graph display area in Curve Editor switches.
The time slider and the range slider are not linked because the Link Playback Range button is turned off.
F Segment Range to Graph Display Area Display The segment tag range that was switched to, is shaded light gray in the graph display area.
G Time Slider, Range Slider The segment tag range is not applied to the time slider or the range slider because the Link Playback Range button is turned off.

Other Tools Started From LayoutEditor

Other tools can now be started from LayoutEditor.

When you open the Layout window in the LayoutEditor window, Tools is added to the menu bar of the LayoutEditor window.
You can select created commands in the Tools menu, under User Commands.

State configured to open Notepad in \Samples\Sources\Resources\Ui2d\NW_LytProject.flpj.

Notepad is added to the Tools menu of the LayoutEditor window, under User Commands.
