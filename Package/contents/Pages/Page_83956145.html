NintendoWare 3D News

This page introduces several new features that were added to the 3D module in NintendoSDK 0.5.0.

3DEditor

Added a feature to filter Curve Editor fields by string

It is now possible to search for fields to edit in Curve Editor.

Improved key movement and scale tool in Curve Editor

Usability improvements were made to key movement and the scale tool in Curve Editor.

Added subgrouping feature for shader parameters

It is now possible to create subgroups for shader groups in the material properties window.

If you create a subgroup, specify the subgroup of a group by using group_id shader annotation, as in the following example.

// @@ group="group_lighting" label="Lighting"
// @@ group="group_hemisphere_light" label="Hemispherical light"
// @@ group="group_directional_light" label="Directional light"
/* @@ group_lighting:
group_id="group_hemisphere_light"
group_id="group_directional_light"
*/

Added the ability to drap a texture directly to the Select Texture UI of a shader

It is now possible to register a texture by dragging it directly to the Select Texture UI element of the shader in the material properties window.
