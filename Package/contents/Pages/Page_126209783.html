Filtering to the Results in the Search Origin Path

This section describes the Filter to results in search origin path flag, which narrows the range of the search to those objects in the hierarchy below the search origin path.

Overview

The following images depict the flag. For this overview, assume that the search target file is "texture" and the search criterion is "reference files."

A Search origin path The path specified in the address bar. The search begins at this location.
B
Search range ()
  • When Filter to results in the search origin path () is disabled
    Searches for folders in the texture search range in higher- and lower-level hierarchies with the search origin path as the starting point.
  • When Filter to results in the search origin path () is enabled
    Filters the search for folders in the texture search range to lower-level hierarchies with the search origin path as the starting point.
