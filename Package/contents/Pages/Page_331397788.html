Library

Overview

The NvPerf library is currently only available in NX version. It is provided in header file and dynamic library formats.

There are two types of dynamic libraries: nvperf_host.nso and nvperf_target.nso. Applications link to one of these types. nvperf_host.nso contains all the features provided by the NvPerf library. On the other hand, nvperf_target.nso does not contain features that can be substituted offline on a PC, such as ConfigImage, CounterDataPrefix, and other structures (See Tools). When a flow is selected that uses these features on a PC, the application can link to nvperf_target.nso instead of nvperf_host.nso.
