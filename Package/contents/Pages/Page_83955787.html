Model Property Window: Preview Page

This page is used to check and edit settings associated with previewing the selected model.

A Selects the model to bind.
Select a model currently loaded into 3DEditor.
B Selects the bone to bind from among those included in the selected model.
C Selects whether to display the bones to bind in order by name or by number.
D Edits the scale of the model in terms of X, Y, and Z.
E Links the scales of X, Y, and Z for simultaneous editing when selected.
F Edits the X, Y, and Z rotation.
G Edits the X, Y, and Z movement.
H Resets modified settings to their default values.
You can reset all settings or reset scale, rotation, and movement settings individually.
I For models that have LODs configured, you can select the LOD level at which to preview the model.
J Clear this check box to disable vertex quantization when previewing models.

Selecting the LOD Level to Use for Preview

When models that have shape LOD levels configured are loaded, you can select the LOD level at which to preview the model from a drop-down menu.
