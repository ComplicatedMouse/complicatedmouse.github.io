Colors Used For Mii Characters

This section describes the colors used for Mii characters.

For information about the colors, presented both visually and as RGB values, see here.

Sheets Description
CommonColor A list of shared colors used for hair, eyes, glasses, and mouths.
CommonColorOrder The order in which all colors are shown on the edit screen of Mii Applet.
SkinColor The list of colors used for skin tone.
SkinColorOrder The order in which face colors are shown on the edit screen of Mii Applet.
FavoriteColor The list of colors used for hats and for bodies (icons with bodies and icons with certain physiques).
Info

Color RGB values are the 8-bit values in the sRGB gamma space.

The row of UpperLipColor colors in the CommonColor sheet are used when the Mii character has an upper lip part.

Note

This documentation is for reference purposes. Do not make direct use of the values presented in the documentation.
