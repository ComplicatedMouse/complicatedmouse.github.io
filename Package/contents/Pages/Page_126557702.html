Face Outline Textures

The face outline texture is rendered on the right half of the Mii face.

Skin color, wrinkles, makeup, and beards (as depicted by texture) are all rendered as components of the face outline texture.

Parts of a Face Outline Texture

A face outline texture is composed of a number of parts, combined together to render the face outline texture according to the parameters configured in the Mii Data (the CharInfo data structure). The nn::mii::Faceline class serves to manage the shape and texture data for rendering the face outline texture of an individual Mii character.

The following table shows the relationship between the parts that compose the face outline texture and the corresponding nn::mii::Faceline::DrawType enumerated types.

Type nn::mii::Faceline::DrawType Comments
SkinTone nn::mii::Faceline::DrawType_Skin
Makeup nn::mii::Faceline::DrawType_Make Not all Mii characters have a makeup part.
Wrinkles nn::mii::Faceline::DrawType_Wrinkle Not all Mii characters have a wrinkles part.
Beard nn::mii::Faceline::DrawType_Beard Not all Mii characters have a beard part.

With the exception of skin tone, not all Mii characters are rendered with all of these parts. For more information about important implementation considerations for unnecessary parts, see the description of Faceline.

The following sections provide the kinds of modulations and rendering parameters for the various parts, and examples of textures. In this case, modulation refers to the mechanism for altering the color of the various parts. For more information, see Variations.
