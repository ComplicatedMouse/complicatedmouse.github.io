Capture Pane

The capture pane is used to capture the rendering results of the framebuffer and layouts for use as textures.
The LayoutEditor capture pane information is edited in the capture pane, inside the property window.

Info

To use the capture pane feature, you must enable the feature in the project settings file.
Related Links For more information, see Project Configuration File Format > Capture Texture.

Capture Pane Properties

Configure the capture texture settings in the capture pane of the property window.

Capture Texture Settings

Enable framebuffer capture

 The setting for capturing the framebuffer for the capture region in the layout as a texture.

Capture first frame only

The setting for generating a texture one time only for the first frame.

Clear this check box () if you want to animate the panes in the Capture Pane hierarchy.
Texture Scale Maximizes the resolution of the capture pane and generates a texture by applying the scale at the specified ratio.
Clear color

Sets the texture color in the following cases.

  • When the framebuffer capture is not performed
  • When the capture area exceeds the texture size of the framebuffer

In particular, setting the alpha value to 0 also sets the initial alpha value to 0 for the generated texture.

These settings make the transmissive parts outside of the panes in the Capture Pane hierarchy transparent in the generated texture.
Info

In the runtime, cannot reproduce any data where the size of captured textures changes.
Do not animate the size and scale in the capture pane.
