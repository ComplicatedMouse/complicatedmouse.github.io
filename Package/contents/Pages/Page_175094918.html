Maximum Traverse Distance Per Frame

This section describes the maximum traverse distance per frame feature, which limits particle emission to ensure that no excess particles are emitted when emitters have a high movement speed.

Overview

The maximum traverse distance per frame feature treats the total movement of an emitter in a single frame as being equal to the specified value.
Treating the total movement of an emitter in a single frame as being equal to the specified value enables you to limit the number of particles that are emitted when the emitter is moving at a high speed within the game screen.
As a result, you can prevent unnecessarily high processing loads from occurring due to too many particles being emitted.

The following overview describes the settings ().

Maximum Traverse Distance Per Frame

Use the dial () or the text box () to specify the value for the maximum traverse distance per frame.
