Changes for File Format Specifications for Model Intermediate File Material Referencing

The specifications for material referencing file formats for FMDB and FMDA model intermediate files handled by 3DEditor and the assign utility have changed. You can pass model intermediate files to the intermediate file updater to convert them to the new format. The earlier format before the specifications change can still be used, but support will be deprecated in future versions. Update to the new format.

Maya Plug-In

Added a feature to run any script before or after exporting multiple items with the multi-export plug-in.

A feature was added with the multi-export plug-in that runs any script before exporting the first output item, or after the last output item has been exported. You can register the script to be executed by adding the following lines to the config file.

pre_multi_export_script="MyPreMultiExport()"
post_multi_export_script="MyPostMultiExport()"
