Manual

This section describes how to use the features provided by the gll library.

The description begins with the location of the gll library.

Namespace

The features provided by the gll library are contained in the nn::gll:: namespace when using a C++ interface. The prefix nngll is added when using a C interface.

Header

All features of the gll library can be used by including the nn/gll.h header file.

nn/gll.h is located in the following directory.

$NINTENDO_SDK_ROOT/Include/nn/gll.h

Other GLL header files are located in the following directory, and are included in nn/gll.h.

$NINTENDO_SDK_ROOT/Include/nn/gll/
