Noise Scale (±)

This page describes the Noise Scale (±) feature to configure the scale for the noise table.

Overview

Noise Scale (±) adjusts the scale for the noise table used with curl noise.
The images depict what happens when the Noise Scale value is adjusted for planar curl noise.

The following overview describes the settings ().

Noise Scale (±)

Set the scale of the noise table using the dial () or in the text box ().
Smaller values expand the table. Particle movement is smoother.
Larger values contract the table. Particle movement is rougher.
