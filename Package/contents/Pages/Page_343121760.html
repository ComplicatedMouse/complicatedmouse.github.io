New Layout Expression

This section introduces features that broaden the scope of what can be expressed in the layout.

Added Rounded Corners

The Rounded Corners feature has been added to round the four corners when the pane shape is Rectangle.

Using the settings of Rounded Corners, you can change the shape of the pane corners without adding resources such as textures or meshes.
Rounded Corners allows you to, for example, make the rounded parts contract into the interior of the pane, make the rounded parts circle shaped, and set animations to animate the shape.

In addition to changing the pane shape, you can also apply effects to the pane corresponding to the rounded shape, such as borders and shadows inside the shape.
Compared with the similar Drop Shadow Effects feature, the Rounded Corners feature allows a range of expression with relatively lower GPU load.
