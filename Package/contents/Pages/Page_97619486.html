Matrix Application Type

This section describes the Matrix Application Type feature, which determines whether to constrain the emitter set to the rotation and scale of the bound bone.

Overview

The Matrix Application Type feature can configure the influence on the emitter set rotation and scale when animating the bound bone rotation and scale using the model linking feature.
The following overview describes the settings ().

Matrix Application Type

From the following presets, select whether to constrain the emitter set with just the coordinates (T) of the bound bone, or also the rotation (R) and scale (S).

TRS The emitter set is influenced by the scale and rotation.
T only The emitter set is not influenced by the scale and rotation.
