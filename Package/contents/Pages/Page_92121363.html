User Settings

These settings can be configured for each user.

Undo

The Undo group () is used to set the number of times the undo command can be used to reverse actions.
This group has the following settings.

Unlimited Enables and disables unlimited undo operations.
Maximum Number Sets the maximum number of operations that are logged. Use this option when the Unlimited check box is cleared.

Click the OK button to confirm the settings, and they are applied when EffectMaker restarts.

Workspace Files to Auto-load

The Workspace Files to Auto-load group () is used to set whether the workspace file is automatically loaded when EffectMaker is started.
Sets a workspace file.
Click the OK button to confirm the settings, and they are applied when EffectMaker restarts.

Preset folder path

The Preset folder path group () is used to change the folder containing emitter set files (ESET files) used when creating effects.
Sets the location of the folder.
Click the OK button to confirm the settings, and they are applied when EffectMaker restarts.

Related Links For more information, see the following pages.

Creating Emitter Sets From Presets This section describes how to create emitter sets from presets.

Options

The Options group () is used to set the files and useful features that EffectMaker will use.

Emitters Can set the display or hide state of the emitter file dialog box.
Texture Can set the display or hide state of the texture file dialog box.
Primitive Can set the display or hide state of the primitive file dialog box.
Preview Can set the behavior of Preview when a file is saved.

Documentation file path

The Documentation File Path group () is used to set the location of the documents referenced by EffectMaker.
Sets Documentation File.
Click the OK button to confirm the settings, and they are applied when EffectMaker restarts.

This setting is found in both the User Settings and the Project Settings.
Related Links For more information about how behavior differs depending on the settings, see Settings Common to User Settings and Project Settings.
