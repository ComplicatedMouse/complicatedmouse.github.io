Picture Panes

This section describes picture panes, which can use registered texture images as layout assets.

Overview

Picture panes allow you to place texture images as layout assets.
Picture panes can use multiple texture images to create a complex expression by blending the images in different ways.
Blending can use polygon colors (black and white) and constant color.

The following are the main settings that can be configured in picture panes.

  • The pane position, scale, rotation, and shape
  • The texture image to use
  • The UV settings to use
  • The polygon color and polygon alpha values
  • Texture and color blending
  • Blending with the background and with other panes

The following table shows the picture pane settings.

Base Pane Sets information including the pane name, position, shape, and center position.
Picture Pane Shape Sets the picture pane's shape and shape-specific values.
Texture Sets the texture image, texture image settings, and texture pattern.
Color Interpolation Settings Sets the input values that affect color and alpha.
Texture Combiner Sets the blending method for two or more images.
Fragment Operations Sets the blending method with a background or other panes.
Extended User Information Sets user-defined extended information for the pane.
