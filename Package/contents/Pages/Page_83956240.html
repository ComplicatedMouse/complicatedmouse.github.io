Creating Layouts

This section describes how to create a layout used for creating layout data in the LayoutEditor.

Creating a New Layout

Start LayoutEditor to open the LayoutEditor window.

Create a new layout file by selecting File > Create New > Layout.
An untitled window opens in the LayoutEditor window. This window is called the layout window.

Related Links For more information, see the following pages.

Creating a Pane in Layout

In LayoutEditor, position panes for images, text, and frames in this Layout window to create the layout for your game screens.
The following pages describe how to use panes.
