Modify Relationship With Base

The following dialog box appears when you select Modify Relationship With Base from the shortcut menu in the Parts Layout list.

Modify Relationship With Base Dialog Box

A Current name

Shows the parts layout filename that is the current derived form.
B New name

From the drop-down menu, specify the parts layout file to be the new derived form.

Alternatively, delete the derivative relationship to make them standard parts.
C Edit target

Use check boxes to specify the derivative parts layout file to be the target of editing.

All the parts layout files in the parts folder derived from the Current Derived Form will be displayed in the edit target.
