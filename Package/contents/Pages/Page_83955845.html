Overview

This section describes the components of effects that occur within games and for NintendoWare effects.

Result of Effects in Games

Using effects in games provides the following advantages.

They add dynamism to the screen.

In combination with backgrounds, they can be used to explain the situation in a particular stage.

In combination with player actions, they can enhance the sense of responsiveness in the game.

Basic Terminology of EffectMaker

EffectMaker is an editor that can be used to create and edit game effects.
The following terms are used in EffectMaker.

Emitters

An emitter is the source of particles.

Various parameters are set on the emitted particles, including initial velocity, lifespan, scale, and color.

The position of the emitted particles is determined by the emitter shape and emitter transform.
Particle

A particle is a polygon object that is emitted from an emitter.

The particle is emitted with an initial velocity and its behavior is affected by fields until it reaches the end of its lifespan and disappears.

With EffectMaker you can collectively control the movements of large volumes of particles.
Fields

A field can be used to add movement in directions other than the initial velocity direction.

Various types of fields are available, including gravity, magnetic force, and spin.

Structure of NintendoWare Effect

NintendoWare Effect supports creating effects data and playback in applications.
NintendoWare Effect comprises four main components.

EffectMaker

This Windows-based tool is used for creating effect data. It also converts the effects to binary.

Artists use EffectMaker to create effects and save them as emitter set intermediate files (ESET files).

Related Links EffectMaker

EffectViewer

This tool is used to preview emitter set intermediate files that were created using EffectMaker.

A Windows version of the application is provided along with versions that run on the various platforms.

Related Links EffectViewer

Effect Runtime

The runtime library is embedded into the game program.

Emitter set intermediate files created in EffectMaker are converted to binary data, and this binary effect data is loaded in the runtime to manage playback of effects in the target application.

Related Links Overview of the Effect Runtime

Viewer Library

The runtime library for creating your own EffectViewer.

By embedding the viewer library in the game application and linking the game application to EffectMaker, you can play and edit effects in the game application itself.

Artists can use this feature to attach prototypes of effects to characters in the game and adjust the nuances of these effects based on how they are ultimately rendered.
