Introduction

Purpose of This Documentation

This documentation describes the features provided by the human interface device (hid) library included in NintendoSDK.

Intended Audience

This documentation is intended for application developers using NintendoSDK.

Chapter Descriptions

Manual describes how to use the features provided by the hid library.

Related Documentation

Also see the hid API Reference and the Sample Backgrounder.

For an overview of the controllers and the console, see the Controller Guide. When developing applications, pay particular attention to the Application Controller Support section.

hid Library Features

The hid library provides an API for controlling various human interface devices including gamepads, keyboards, touch screens, and accelerometers.
