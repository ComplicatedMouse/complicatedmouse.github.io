Animation Segment Tags

This section describes the Animation Segment Tags feature.
With LayoutEditor you can assign names to portions of animations and manage them separately. This involves using what are known as animation segment tags.
Animation segment tags consist of a tag name, frame segment (start frame and end frame), and settings for binary output.
In the separate page that introduces the Parts feature, animation segment tags are used to specify button behavior animations.

Advantages of Using Animation Segment Tags

By using animation segment tags, you can cut a single animation file into segments to manage several animations.
The binary converter can interpret the tag information and generate a separate binary file for each.

Animation Segment Modes

There are two animation segment modes: merged mode, in which segment tags specify ranges in a single, long animation curve, and separated mode, in which the animation in each range of segment tags is edited in a separate window.

(Segment tags are referenced for the playback of animations in the viewer, so the animation plays the same no matter which mode is set.)


Info

Note: The merging of animations when the mode is switched will sometimes add keys outside the segment range, and this can lead to unintended results.
For this reason, we recommend that you basically stick with either separated mode or merged mode for your task.

Animation Production Using Animation Segment Merged Mode

In this method, the animation is created as a single, joined animation that is broken up by animation segment tags.

Example:

  • Start is an animation comprising a total of 20 frames, created in frames 00 to 20.
  • Wait is an animation comprising a total of 50 frames, created in frames 20 to 70.
  • End is an animation comprising a total of 30 frames, created in frames 70 to 100.
Animation Production Using Animation Segment Separated Mode

The animations are managed by using animation segment tags and are created separately, starting in each case from the 0th frame.

Example:

  • Start is an animation comprising a total of 20 frames, created in frames 00 to 20.
  • Wait is an animation comprising a total of 50 frames, created in frames 00 to 50.
  • End is an animation comprising a total of 30 frames, created in frames 00 to 30.

Base Values

When no segment tags have been selected in separated mode, the pane values are their so-called base values.
When you select segment tags and set animations, these base values are applied to items for which keys have not been set.

Creating Animations in Separated Mode

This section describes the process of animation creation using separated mode.

Select the pane linked to the animation that you want to edit and then select the segment tag representing the range of the animation that you want to edit in the Edit Animation Segment Tags window (). When you perform this step, the segment tag range of the animation is displayed in the Animation window.
In separated mode, you edit the animation curve for each individual segment tag.
By switching the selected segment tag, you can create a separate animation curve.

All animations created in separated mode start from frame 0.
