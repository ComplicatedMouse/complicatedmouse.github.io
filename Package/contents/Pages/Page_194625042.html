Data Delivery

Data delivery is a feature for delivering data to multiple consoles simultaneously. The data has been registered to a server that is managed by Nintendo. Delivery is performed in a format that synchronizes the server data on the console.


Multiple directories, that have one level of hierarchy depth, can be registered to the server. Multiple files can be registered in a directory. The start date and time and the end date and time, of delivery for each file, can be reserved in the settings.

During synchronization, the directory and file tree structure is copied 'as is' to the console. If the directories and files are deleted from the server, they will also be deleted from the console.

This document refers to directories and files collectively as data.


When the application is running, synchronization can take place at any time. When the application is not running, the system synchronizes automatically.
