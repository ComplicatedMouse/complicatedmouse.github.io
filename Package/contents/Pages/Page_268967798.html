Creating a Layout

To create a layout, from the main menu, select File > New.

Related Links For more information about how creating a new layout, see Creating a Layout.

Screen Size Settings

Set the size of the menu screen to be created.
A layout of the specified size is created.

In the main menu, select Window > Layout Settings.
The Layout Settings dialog box opens. Enter the following settings.

Environment Settings Screen Size Width 1280.0
Height 720.0

Load the texture.

Load the following texture (TGA) files from the following location into the Texture window.
Samples/Sources/Resources/Ui2d/Common/Textures/

Switch the Texture window to Detail View and configure the texture format as follows.

Thumbnail

 Filename Format Description

Format that uses a texture as grayscale with compression.

Used to express the scan lines of a background.

Used as background decoration.

Creating a Background

Create a background for the menu screen in the picture pane.

From the Texture window that displays the textures, create a Picture pane ().
In Texture, double-click BG_Line.tga ().
In the Hierarchy window, select the created Picture pane () and change the pane name to BG.
In the property window, specify the following settings.

Base Pane ()

Size

X,Y = 1280,720

According to the screen size.

Picture Pane ()

Vertex Color

[ALL] R,G,B = 0,0,0

[ LT ] R,G,B = 8,99,85

[ RT ] R,G,B = 3,20,16

Set the background color with the vertex colors.

Because a texture is not used, gradations are smooth and low-cost.

Texture ()

Image Settings



Repeat Method

U-direction, V-direction = Repeat, Repeat

Expresses the scan lines.

Can animate a texture by repeatedly scrolling it.

Filtering

When expanding, when contracting = Near

 Changes the texture interpolation method so scan lines are clearly visible.
SRT Transformation Matrix

Scale Link = Off

U, V = 1.000, 100.000

 Scales texture coordinates in the vertical direction to align multiple scan lines in the background.

Combiner Settings ()

Advanced Combiner Mode

Click the Switch to advanced combiner mode button to switch modes.
Set the method of blending the registered textures 0 and 1.

Detailed Combiner Mode Settings

Blend two Deco_Scroll.tga textures () registered in the Deco_Scroll background decoration () in Detailed Combiner Mode.

Use the following procedure to set Detailed Combiner Mode for the texture ().

1 In Number of stages used, enter 2.
In Stage Selection, select Stage 0 and Stage 1.
2

To set Stage 0, in Stage Selection, click the 0 button.

3

Apply (Source 0 + Source 1) * Source 2 as the alpha combiner mode.

The alpha values for Texture 0 and 1 are added and then multiplied by the alpha value for Constant 0.
4

Set the alpha source.

Apply Texture 0 as Source 0 and Texture 1 as Source 1.
5

Set the color and alpha for contrast color C0 as follows.
R,G,B = 0,128,128 (), A = 30 ()
6 To set Stage 1, in Stage Selection, click the 1 button.
7

Apply Source 0 * Source 1 as the alpha combiner mode.

The alpha value for Stage 0 in the previous stage is multiplied by the alpha value for the polygon color.
8

Apply the Polygon color in alpha Source 1.
Apply the set alpha value in the Vertex color of the picture pane ().

Creating a Null Pane

The created pane is placed in a hierarchy and two null panes are created for animation.
Panes can be easily substituted or added in the Null Pane hierarchy.

From the main menu, select Pane > Create Pane > Null Pane. Do this step twice.
In the Hierarchy window, select N_Null_00 () and change the pane name to N_Deco.
Places the LayoutEditor mark in this pane hierarchy and moves it up and down.

In the Hierarchy window, select N_Null_01 () and change the pane name to N_Samples.
Place the menu screen title "Samples" in this pane hierarchy and animate the opacity.

In the Hierarchy window, check that the created panes are in the following order.
