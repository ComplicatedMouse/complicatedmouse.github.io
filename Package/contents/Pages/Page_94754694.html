Random Initial Velocity and Random Momentum

This page describes the differences between two features for randomizing particle movement paths: Random Initial Velocity and Random Momentum.

Overview

The Random Initial Velocity feature randomizes the initial particle velocity each time a particle is emitted.

The Random Momentum feature randomizes the momentum of individual particles as they are emitted.
This feature also affects other features such as fields that apply changes to particle movement after particles are emitted.
