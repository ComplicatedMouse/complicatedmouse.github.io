Curve Editor Operations

This section describes how to create an animation.

Basic Workflow for Animation

In LayoutEditor, animations are creating using the method of keyframe animation.
To create the animation, you set key frames in the created layout.

Creating a Pane Position Animation

Create an animation that moves the position of the pane along the x-axis (left/right).

Curve Editor

Use either of these operations to open the Curve Editor in the LayoutEditor window.

  • Click the Curve Editor icon ().
  • From the menu, select Window > Curve Editor.

The following image shows the Curve Editor window.

Selecting a Pane

Select the pane you want to animate in the LayoutEditor window.
Select a pane from the items shown in the tree on the left side of the Curve Editor. The selected pane becomes the target for animation editing.

Creating a Starting Key Frame

From the attribute tree in the Curve Editor window, select Pane > Position > X.
Use either of the following operations to create a starting key frame.

    • To create the starting key frame, select Toolbar > Create Key in Current Frame.
  • Press S as a keyboard shorcut.

The key frame (black dot) is created in the graph display space, and the mark showing a set key is displayed in the position X.

Info

If you do not see anything, the display area may be misaligned. To show all, press A as a keyboard shortcut.

Moving the Current Frame

Use either of the following operations to change the current frame to frame 50.

  • Drag the time slider.
  • Enter a numerical value in the text box next to the time slider.

Creating an Ending Key Frame

Use either of the following operations to create an ending key frame.

  • To create the ending key frame, select Toolbar > Create Key in Current Frame.
  • Press S as a keyboard shortcut.

Moving the Key Frame

Drag a rectangle to select the END key.
Drag using the middle mouse button to move the key frame.

Info

Hold down the SHIFT key and drag with the middle button to move the key frame along a fixed axis (time or value axis).

The direction in which you first move the mouse becomes the fixed axis along which the key frame moves.

Animation Playback

On the playback controls, click the Play button.
The animation starts playing. Confirm that the pane moves on the x-axis along the curve.
To stop the playback of the animation, click the Stop button in the playback controls.

Editing Slopes

Edit the slopes of the key frames to change the animation.

Editing the Slope of a Key Frame

Drag a rectangle to select a key. The selected key is shown in green.
Drag a rectangle to select a control point shown to the left or to the right of the key.

When a slope is selected, the color changes to yellow-green.
Drag with the middle button to edit the shape of the slope.

You can create animations by editing the shape of the curve in this way.

Checking in the Viewer

Use either of the following operations to check your changes in the viewer.

  • From the menu in the LayoutEditor window, select Tools > Transfer to Viewer.
  • Press F5 as a keyboard shortcut.

The viewer starts and the transfer of data begins.
After the transfer has completed, the pane is displayed and the animation plays.
