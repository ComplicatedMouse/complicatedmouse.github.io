Setting the Width of Each Line

This section describes the Specify the Width of Each Line feature for changing the width of each row for multiple lines of text.

Overview

The Specify the Width of Each Line feature changes the width of each row for multiple lines of text aligned to the upper left.
The width of each line is reduced by taking the size set in size x for the base pane as the maximum width of that row, and setting negative values for the amount of movement from the left and the right.
You can configure the settings for Specify the Width of Each Line to move the starting position and the line break position of the text box pane to match the background.
The following overview describes the settings for Setting the Width of Each Line.

Enable

Enables Setting the Width of Each Line. When selected, additional settings are displayed.
Select the text box pane in the layout window. The ■ symbol is displayed to indicate the position of movement left or right for each row, as shown below.

The following additional settings are displayed when the feature is enabled.

Specify Row Movement From the Left

Sets how many pixels to move the left-most text from the beginning of the row, in the range from 0 to any negative value.

Specify Row Movement From the Right

Sets how many pixels to move the line feed position of the text from the right edge, in the range from 0 to any negative value.
