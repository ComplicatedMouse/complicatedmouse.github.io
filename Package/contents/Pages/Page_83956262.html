Skinning - 3ds Max

Skinning

Skinning data can be exported.

The supported objects are Bone, Biped, Dummy, and Point Helper.

Note About Bone Structure

If nodes that are not used for skinning are mixed in the hierarchy of nodes used for skinning, the animation may not be exported correctly.
This is because for the posture that is exported to the model file, skin is bound to the nodes that are used for skinning but the nodes that are not used for skinning are exported to the posture in the starting frame.

To avoid this problem, make sure that nodes in the node hierarchy that are not needed for skinning are registered to the skinning modifier.
As an alternative approach, export the model file as a bind pose and export the animation as a skeletal animation file.

Info

Do not perform the following operations after you apply the skin modifier:

  • Operations that change the number of vertices.
  • Delete nodes that are affected by skinning.
