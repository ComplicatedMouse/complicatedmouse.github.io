Overview

About PC Emulation

SoundMaker supports the PC emulation-based playback of sounds from computer audio output.
PC emulation runs as an independent program from the SoundMaker GUI, and enables the playback of instruments in the stream sounds, sequence sounds, wave sounds, and bank files of sound projects.

Playing Instruments

There are two ways to play instruments by using PC emulation.
The first is playback by using GUI operations where playback is performed from the bank window. This corresponds to "Playback Information Route 1" in the following conceptual diagram.
The second is playback based on MIDI signals from an external MIDI device. This corresponds to "Playback Information Route 2" in the conceptual diagram.

Note that completely different operations are performed under PC emulation for these two methods of playback.
GUI-based selection of instruments does not affect the playback of any particular MIDI channel.

GUI-Based Sound Playback

You can play back sounds by using the graphical user interface (GUI) after making a selection in the Sound Set tab.

For more information, see Sound Playback Operations.

GUI-Based Instrument Playback

You can play back instruments from the GUI by using the Bank tab.
When multiple Bank tabs are open, instruments always play from the active Bank tab.

The following section describes how to play instruments by using PC emulation in the Bank tab.

Instrument List

You can open this list by pressing the SPACEBAR, F9, and F10 on the computer keyboard.
Pressing and releasing the SPACEBAR corresponds to the note on and note off MIDI commands, respectively.
Press F9 to trigger playback and F10 to stop playback.
This is true of all playback performed in Bank tabs using the SPACEBAR, F9, and F10.
In the instrument list, pressing the SPACEBAR, F9, and F10 plays the key selected within the virtual keyboard in the Sample Map pane (usually C4) for the selected instrument.

Sample Map

To play back, click the keyboard shown in the GUI and press the SPACEBAR, F9, and F10 on the computer keyboard.
Click the virtual keyboard to play the key that corresponds to the instrument that you are editing.
The SPACEBAR and F9 are used for playback of a region.
The waveform of the region selected by key stroke will play using the original key for that region.

With the slider to the left of the keyboard, you can specify the velocity during playback through a keyboard.

Percussion List

You can open this list by pressing the SPACEBAR, F9, and F10 on the computer keyboard.
Pressing the SPACEBAR or F9 plays the key corresponding to the selected instrument being edited in the list.

MIDI-Based Instrument Playback

PC emulation functions as a 16-channel, multi-timbre sound source for MIDI input from the configured MIDI port.
Channels that produce sound from MIDI input support all instruments of the bank tab that is active in the channel. Switching between instruments is performed according to the program numbers set on the instrument list.

For each channel, make sure to specify a program number for the instrument to play at the MIDI program change.

For the selection of an instrument from the instrument list, the program changes of the corresponding instrument are sent only to MIDI channel 1.

Instruments from separate banks cannot be played simultaneously on multiple MIDI channels.
The bank is switched when the Bank tab becomes active.
Note that after a program number is specified for a channel, it is not reset even if the active Bank tab is switched.

MIDI Playback

Use the Options dialog box to specify the devices that SoundMaker uses as the keyboard and the sequencer for MIDI input.

MIDI Playback From the Selected Keyboard

This option is mostly used to check the instruments that are registered for a particular bank.

If the Bank tab is active, and an instrument within the instrument list is selected, MIDI signals from the selected keyboard are played back by using that instrument.
If a different instrument is then selected, the sound switches to that instrument.

Because the currently active instrument is used for playback, playback of MIDI signals from the selected keyboard requires that the Bank tab is active and that an instrument is active as well.

Regardless of whether an instrument is selected, if a "Program Change" MIDI command is received from the keyboard, the instrument that corresponds to the new program is selected.
Incoming "Bank Select" commands are ignored.

MIDI Playback From the Selected Sequencer

This is mostly used to incorporate MIDI data that was created with commercially available sequencing software into SoundMaker, and to check the corresponding banks.

MIDI is played back by using the banks that are registered in the Preview Bank pane.

You can set up to four banks for a sequence sound.

Confirming the State of PC Emulation

The upper-right part of the main window always shows two statuses associated with PC emulation: (1) MIDI Reception and (2) Actual Sound Production.

MIDI keyboard input produces only the sound for the active bank panel.
If the bank panel is not active, no sound is produced.

Meanings of Displayed Items

The following sections explain the displayed items.

MIDI

Lit green when PC emulation has received some kind of MIDI signal.

SOUND

Lit green when PC emulation has sent some kind of sound output.

Display Updates

The displayed information is updated every 1/30 of a second.
Accordingly, it may not be possible to confirm the output of extremely brief sounds.

Stop All Channels

During PC emulation you can forcibly stop all sounds being played back by selecting Tools > Stop all channels.

Use this feature when you want to stop the playback of an instrument that has an extremely long release time that was set by using an envelope.

MIDI Reset

To reset all PC emulation MIDI parameters, select Tools > MIDI reset.
This operation resets all parameters such as volume that were specified previously by external MIDI input.

Use this feature when you play another MIDI sequence under PC emulation, when you play a MIDI sequence from a midway point, or when you do not want to keep the settings for parameters that are based on MIDI messages such as control change.

Cautions and Supplemental Notes

This page provides cautions and other information that is useful during PC emulation.

Using PC Emulation From Sequencers Running on the Same Computer

To perform PC emulation playback by using the MIDI output of sequencers that run on the same computer that runs SoundMaker, it is necessary to loop the MIDI output from the sequencer back into the MIDI input of the computer.

Looping back a MIDI signal in this manner is called loopback. Loopback can be accomplished in the software by using a freeware virtual MIDI device driver.

Reproducibility of Sounds Using PC Emulation

When you play sounds using PC emulation, you cannot re-create the degradation of audio quality due to compression of ADPCM and other waveform data, nor the characteristics of the platform-specific audio circuits.
Consequently, the sound quality of the output can differ on some platforms.

As with MIDI, PC emulation does not support theoretically impossible playback such as loops or sequence command playback.

Be sure to perform final testing of each sound using the platform consoles.
