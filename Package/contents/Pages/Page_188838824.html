nn::ui2d and nn::font News

Added Options to LayoutArchiver for Specifying the Texture Tiling Method (tile-optimize and tile-size-threshold)

Added options to LayoutArchiver for specifying the texture tiling method (tile-optimize and tile-size-threshold).

The specifications conform to the specifications for the same options in the texture converter.

Differences in file size will occur in small textures and textures of a size that are not a 'power of 2'.

For example, specifying tile-optimize=size_auto for all of the textures in the Samples\Sources\Resources\Ui2d\cannonButton\CannonButton_00.flyt layout, results in the following sizes.

  • tile-size-threshold=5 ⇒ 96.1 KB
  • tile-size-threshold=100 ⇒ 72.1 KB

(The larger the value set for tile-size-threshold, the more size is prioritized during conversion.)
