Left Window Group

The Left Window contains lists of data collected during the profile:

  • Functions: List of the recorded functions.
  • Last: Recorded functions from the last loaded profile.
  • Threads: List of recorded threads.
  • Instrumented: Instrumented functions and user logged Marked code blocks.
  • Counters: Performance counters and user logged counters.
  • Logs : User logged counters.
  • Gfx Groups: Debug Marks and Debug Groups recorded as part of the GPU Trace data.
  • Shaders: Shader information recorded as part of the GPU Trace data.
  • Files : File information recorded as part of the File System Trace data.
