Using Picture Panes

This section describes how to use picture panes, which are used for positioning images on the screen.

Preparing to Create the Layout

From a state in which the LayoutEditor has been started and a layout has been created, make preparations to create a picture pane.
Related Links For more information about starting LayoutEditor, see Workflow, and for more information about creating layouts, see Creating Layouts.

Loading Textures

Before you can position the picture pane, the image file that will be used by the picture pane must be registered as a texture.
Textures can be registered in the Texture window.
This section describes the procedure for registering the following image file as a texture.

Displaying the Texture Window

Use either of the following methods to display the Texture window.

  • Select Menu > Window > Texture.
  • Press F8.

Loading the Image File

Use the following procedure to load an image file and register it as a texture.

1

In the Texture window, click Load ().
2 In the Load dialog box, select the following image file and click Open.
Samples/Sources/Resources/Ui2d/Common/Textures/f.tga
3. The loaded image is displayed in the Texture window.

You can also load an image file with the following procedure.

  • Drag a TGA file from the Explorer window to the Texture window.

Specifying a Texture and Creating a Picture Pane

Use the following procedure to create a picture pane.

1

 Select the image file f registered in the Texture window.
2

Click the Create Pane button ().
3. A picture pane that displays the texture selected in the Layout window is created.

You can also create a picture pane with the following procedure.

  • Double-click an image file in the Texture window.
  • Drag the image file from the Texture window to the Layout window.

This section describes picture panes, which can use registered texture images as layout assets.

Specifying the Format of the Registered Texture

In the texture window, specify the format for binary conversion of textures.
You can change the format to make color texture monochromatic or to make a texture using only the alpha.
The following description uses the process of converting the color format "BC1" to the monochrome format "BC4L" as an example.

1

In the Texture window toolbar, change Image View () to Detail View (). You can view the details of the registered textures as a list.
2 Verify that f is registered in the BC1:(RGB+1A) format.
3. Right-click BC1:(RGB+1A), and then select BC4L:(I) from the drop-down list.
4 The picture pane display changes to monochrome in the Layout window.
