Loop

This page describes the Loop feature, which sets how to loop a curve referenced by the per-character transformation feature.

Overview

The Loop feature specifies how to loop the curve when the interval between the beginning and ending frames of the referenced curve is narrower than the duration.
The following overview shows the Loop settings.

Loop

Use the radio buttons to set the looping method for the reference curve.

One Time The values before and after the curve segment are fixed as the start key value and end key value, respectively.
Loop The curve loops before the start key and after the end key, respectively.
