Is there a way to run my own commands when saving a file?

Question

How do I run my own commands when saving a file?

Answer

You can write your own commands to be executed before and after a file is saved or deleted. To do this, specify the path of the file in which the commands are scripted in the project settings file.
For example, you could use this together with version control software.

  • Get edit rights when a file is opened.
  • Release edit rights when a file is saved.

For file paths, you can use absolute paths, paths that are relative to the project file, or environment variables, designated in %XXX% format.
Specify the file paths (UTF-8 without BOM) of all the files you want to be passed when the command is executed.

<PreSaveCommandPath>%NINTENDO_SDK_ROOT%\PreSave.bat</PreSaveCommandPath>
<PostSaveCommandPath>%NINTENDO_SDK_ROOT%\PostSave.bat</PostSaveCommandPath>
<PreDeleteCommandPath>%NINTENDO_SDK_ROOT%\PreDelete.bat</PreDeleteCommandPath>
<PostDeleteCommandPath>%NINTENDO_SDK_ROOT%\PostDelete.bat</PostDeleteCommandPath>
<PreLoadCommandPath>%NINTENDO_SDK_ROOT%\PreLoad.bat</PreLoadCommandPath>
