Configuring Groups

We describe how to configure groups.

Groups provide an efficient way to load waveform data into memory when the application is playing sounds.

What Is a Group?

A group is one constituent element of a sound set. Each item in created sounds, in addition to other entities, are registered. Data (such as wave data) is gathered together at the group level during conversion.

Categorize and register sounds used by the same scene or stage to the same group for more efficient loading of waveform data by the application.
This also makes more efficient use of memory, because duplicate waveform data is not stored within a group.

Configuring Groups

Group settings are configured in the group list and the group item list in the Sound Set tab.
These settings are used first to display these lists, then to create groups, and then to register items to those groups.

Displaying Groups

Double-click the Group node in the project tree (or single-click if the Sound Set tab is open). The Sound Set tab is split into upper and lower sections that display two kinds of lists.
The upper section displays a list of groups. If you select a group from the upper section of the screen, a list of items registered with that group is displayed in the lower part.
The list in the upper section is called the group list, and the list in the lower section is called the group item list.

Creating Groups

To create a group, select Group in the project tree, click inside the Sound Set tab to change the focus, and then select Project > Add Item > Group.
The Create New Group dialog box appears. Enter the required information.

Registering Items in Groups

Select the group to which you want to register items from the group list that is displayed in the upper part of the Sound Set tab. Next, select Project > Add Item > Group Item.
The Create New Group Item dialog box appears. Select the items that you want to register.
