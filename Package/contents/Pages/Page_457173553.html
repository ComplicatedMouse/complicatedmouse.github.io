Overview

OpusDecoder is a PC tool that decodes files created in the opus format by OpusEncoder and outputs them as WAV files. It is located in Tools/Codec/AudioCodec/.

This tool is dependent on the following DLLs. These are contained in NintendoSDK.

  • Tools/Codec/AudioCodec/AudioCodec.dll
  • Tools/Codec/AudioCodec/NAudio.dll
  • Tools/Codec/AudioCodec/Nintendo.ToolFoundation.CommandLine.dll
  • Tools/Codec/AudioCodec/ja/OpusDecoder.resources.dll

How to Use

How to use: OpusDecoder.exe [-h] [--version] [-o=<arg>] [-v] <arg>

Available options:
  -h, --help                    Displays help messages.
  --version                     Displays version information.
  -o, --output <arg>            The output file path.
  -v, --verbose                 Verbose mode.

Available positional arguments:
  <arg>                         The input file path.

Usage Examples

Decodes input.opus to output.wav

$ OpusDecoder.exe -o output.wav input.opus
