Depth Test

This section describes the Depth test feature used to test the depth of the particle to be drawn.

Overview

This feature tests a particle for depth using selected conditions, whether the depth value of the particle to be rendered is in front of or behind the value of the Z buffer (depth buffer) as seen from the viewpoint.
The following overview describes the settings ().

Depth Test

Use the check box () to enable or disable Depth Test.
Enable this setting to display the Depth Test ().
The following overview describes the settings ().

Conditions for Passing the Test

Select the depth test passing conditions from among the presets.

Z-buffer write

Select or clear this check box () to specify whether to write depth values that pass the test to the depth buffer.

Example Configurations

The following images compare the different depth test presets.
The planar particle with the checkered pattern (in the red frame □) is moved backward and forward to make it appear that it is passing through the planar particles mapped with the letter F.
All check boxes are selected () for Z buffer write.

A Never pass. The test is never passed when this setting is selected.
The letter F never appears.
B Depth value < Z buffer Passes if the depth value is smaller than the Z buffer value.
The letter F appears when the red frame □ is in front.
C Depth value = Z buffer Passes if the depth value is same as the Z buffer value.
The letter F appears when the letter and the red frame □ are in the same position.
D Depth value ≦ Z buffer Passes if the depth value is same as or smaller than the Z buffer value.
The letter F appears when the red frame □ is in front of or in the same position as the letter.
E Depth value > Z buffer

Passes if the depth value is greater than the Z buffer value.

The letter F appears when the red frame □ is behind the letter.
F Depth value ≠ Z buffer

Passes if the depth value is different from the Z buffer value.

The letter F appears when the letter and the red frame □ are not in the same position.
G Depth value ≧ Z buffer Passes if the depth value is same or greater than the Z buffer value.
The letter F appears when the red frame □ is behind or in the same position as the letter.
H Always passes The test is always passed if this settings is selected.
The letter F always appears.
