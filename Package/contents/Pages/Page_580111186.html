Changes From Previous Firmware

Updated the rendering engine and JavaScript engine included in web applets.
The main new features that are now supported are as follows.

  • HTML5
    • Forms > input > minlength
  • Web API
    • Fetch
    • Resource Timing Level 1
    • Shadow DOM
  • CSS
    • CSS Animations Level 1
    • CSS Font Loading Module Level 3
    • CSS Grid Layout Module Level 1
  • ECMAScript
    • ECMAScript 6
    • ECMAScript 2016+

The following features are no longer supported.

  • CSS Font Loading Module Level 3 > 3.2 Events
    • onloading
    • onloadingdone
    • onloadingerror
