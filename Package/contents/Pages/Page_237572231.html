Export Target

The color animation export target is the specular shading key frame of the material.

Note

The curve applied for the color or other animation of the material changes depending on the shader to define.

The demo.fsdb file uses color. The export plug-in corresponds to the following curve.

  • Color (color gain when the texture nodes are connected)
  • Transparency (alpha gain when the texture nodes are connected)
  • Ambient color
  • Incandescence
  • Specular color

This corresponds to B, Color Animation Data, in the Output File Selection of Nintendo Export Options.

Related Links For more information, see Color Animation.

Info

Combines each of the color, texture SRT, and texture pattern animations into a single FMAB file and exports the file.
These animations can be exported individually using the export plug-in config settings.
