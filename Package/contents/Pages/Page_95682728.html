Wrap

This section describes the Wrap feature for setting how to repeat texture UV.

Overview

The Wrap feature has presets for selecting how textures are mapped when they are repeated to cover the facing surface of the particle.
For example, when Repeat is selected, the texture is repeated using the same pattern.
The following overview describes the settings ().

Wrap

Select among the following presets for wrapping the texture in the horizontal direction (U) and vertical direction (V).
For each wrapping method there is a button for configuring the settings for U and V together.
The table shows the icon buttons and their corresponding selections.

Mirror Repeatedly mirrors the texture.
Repeat Repeats the texture.
Clamp Extends the color at the edge of a texture.
