Output Image Tab

Image

The Image tab can be used to output fonts as image files.
These image files can then be used as an image in Input Specifications.
This section describes the WindowsFont tab UI.

A Image Files Specifies the path of the image file to output.
B Character Order Selects the character order used in the image file.
C Cell Size

Specifies the cell size and glyph output position.

When Position glyph in center is selected, the glyph output position is the center of the cell.
Related Links For more information, see Cell Size.
D Image Format

Specifies the image format of the image file to output.

When Match the specified file extension is selected, the output format is determined based on the extension.
E Draw Grid Draws the grid when selected.
F Null / Width Line / Grid / Margin

You can specify colors to output for the respective areas in the image file.

Related Links For more information, see Specifying Output Image Colors.
