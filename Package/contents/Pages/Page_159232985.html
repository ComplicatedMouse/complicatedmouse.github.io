Major Changes to atk

User Effect Functionality Added

Added user effect functionality, enabling signal processing for waveforms. When the user creates a class derived from nn::atk::EffectAux, that class can then have user effects added.

For more information, see the AtkEffect samples and the Audio Effects documentation for the programming guide.

Changes Specific to Memory Pools

Changed the Way Samples Support Memory Pools

To improve memory efficiency in the samples, the buffers that are needed to support memory pools are now attached as a group instead of individually.

Added Data for Which Memory Pools Must Be Managed on Application Side

Some information was missing about the types of memory regions needed for memory pool management within the NintendoSDK 0.11.0 documentation. Specifically, the following two types were added.

  • Memory sound archive
  • User management group

For more information, see the Memory Management documentation in the Programming Guide.
