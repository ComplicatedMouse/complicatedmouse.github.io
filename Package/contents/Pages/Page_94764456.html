Texture - Specify File

This section describes the feature for registering and previewing textures.

Overview

The Specify File feature is used for registering the textures to use with emitters.
The following overview describes the settings ().

Specify File

Displays the path of the texture file and allows you to register a texture. The following are button types.

Load Opens a file selection window where you can select a texture file.
Reload Reloads the specified texture file.
Clear Clears the specified texture file.

Presets (Random Vertical or Horizontal Flipping)

Click the Presets icon () to change the settings to frequently used combinations.
This changes the Texture Pattern Animation settings and some of the settings in the Texture group.

The content of the registered texture can be separately previewed as RGB elements and alpha elements.
