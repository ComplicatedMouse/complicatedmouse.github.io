Number of Past Samples (Length of Stripe)

This section describes the Number of past samples (Length of stripe) feature for configuring the string length of a stripe.

Overview

The Number of past samples (Length of stripe) feature can configure the string length of a stripe.
The larger the value the longer the stripe becomes, but larger values also increase processing load.
The following overview describes the settings ().

Number of Past Samples (Length of Stripe)

Information about the position and movement direction from particle generation is recorded for the specified number of times.
Stripes are generated from the recorded positional information, so the larger this value, the longer the stripe.

Example Configurations

The following figure compares what happens when the Number of past samples (Length of stripe) value is changed.

A 10 The stripe is shorter than the default setting.
B 60 The default. This length is frequently used.
C 128 The stripe is longer than the default setting.
