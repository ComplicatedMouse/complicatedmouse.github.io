The settings vary depending on the type of pane.

UI for Selecting Panes Using a Common Color

In the region of the settings in the property window where color settings can be configured, you can select panes that are using a common color.
Right-click to display the shortcut menu. The shortcut menu contain the following options.

Color Interpolation

Selects panes using common color interpolation.

  • Select panes with common black color interpolation.
  • Select panes with common white color interpolation.
Vertex Colors

Selects panes with common vertex colors.

  • Select panes with common LT.
  • Select panes with common RT.
  • Select panes with common LB.
  • Select panes with common RB.
Text Box Panes

Selects panes that are using a common color in text box panes.

  • Select panes with a common top edge text color.
  • Select panes with a common bottom edge text color.
  • Select panes with a common top edge shadow.
  • Select panes with a common bottom edge shadow.
  • Select panes with common black shadow color interpolation.
  • Select panes with common white shadow color interpolation.
Shape

Selects panes with common colors for shapes.

  • Select panes with a common shape stroke color.
  • Select panes with a common shape shadow (inner) color.
  • Select panes with a common shape color overlay color.
  • Select panes with a common shape gradient 1 color.
  • Select panes with a common shape gradient 2 color.
  • Select panes with a common shape gradient 3 color.
  • Select panes with a common shape gradient 4 color.
  • Select panes with a common shape drop shadow color.
Capture Effects

Selects panes with common capture effect colors.

  • Select panes with a common capture effect stroke color.
  • Select panes with a common capture effect glow color.
  • Select panes with a common capture effect drop shadow color.
Textures

Selects panes with common colors for constant colors.

  • Select panes with a common black color.
  • Select panes with a common white color.
  • Select panes with a common CO.
  • Select panes with a common C1.
  • Select panes with a common C2.
  • Select panes with a common C3.
  • Select panes with a common C4.
Capture Panes

Selects panes with common colors for clear colors.

  • Select panes with a common capture texture clear color.

Animation UI

You can configure animation for any settings with an animation marker () in the property window.

To display a shortcut menu, right-click an item name with Menu Visible or an animation marker ().
You can operate animation features from the shortcut menu. Shorcut menus contain the following options.

Add Key Adds an animation key for the corresponding item, using the current frame.
Add Keys to All Adds an animation key for all items in the corresponding group.
Add Key to Red Adds an animation key to the red component of a color.
Add Key to Green Adds an animation key to the green component of a color.
Add Key to Blue Adds an animation key to the blue component of a color.
Add Key to Alpha

Adds an animation key to the alpha component of a color.

Mute Mutes the animation.
Unmute Unmutes the animation.
Mute All Mutes the animation for all selected items.
Unmute All Unmutes the animation for all selected items.
Open Curve Editor Open the Curve Editor to display the corresponding item.
Copy

Copies the selected color.

Copies the selected animation key or curve of the selected item.
Paste

Pastes the selected color.

Pastes the curve of the selected animation.


Info

The menu options vary, depending on the item.

Muted animations are not included in binary output when transferring from LayoutEditor to the viewer.
Muted animations are not included in binary output when you select the --discard-muted-animation option in the Layout Binary Converter.


You can also check the state of an item from the color of the animation marker.
The following table lists the display statuses and their descriptions.

 State where no animation key has been configured for this item.

State where an animation key has been configured for this item.
The value of an item has been confirmed in the current frame of the timeline.

State where the value for the item is unconfirmed.
The value that has been set is discarded if you operate the timeline or save the file.
Adding a key to the current frame of the timeline changes an item to the confirmed state ().

State where the animation for this item is muted. The animation stops in the viewer.
A muted curve cannot be edited in Curve Editor.

State where the animation for this item is locked.
Locking prevents unintended editing of the animation curve.
