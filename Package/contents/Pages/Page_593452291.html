Overview

InitializeHdevWin is a GUI tool for initializing the firmware on the HDEV and EDEV development hardware.

Currently, this is the same tool as InitializeEdevWin. For more information, see that page.

Package

InitializeHdevWin is included in the SystemUpdater for NX package.

Location

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/InitializeHdevWin.exe
