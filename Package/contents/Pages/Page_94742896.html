Emitter Shape - Cube (Volume)

Describes the settings for Emitter Shape - Cube (volume), where particle emission locations are set within a cube.
These are the particles when this emitter shape is being used.

Overview

With the Cube (Volume) emitter shape, particles emit in the direction of the normal in a widening cube shape from the entire surface of the cube.
The following overview describes the settings ().

When the emitter shape is set to Cube (volume), the Cube (volume) () settings are displayed. The group has the following settings.

Hollowness

Use the dial () or the text box () to set the region where particles are not emitted as a percentage from the center of the emitter.
