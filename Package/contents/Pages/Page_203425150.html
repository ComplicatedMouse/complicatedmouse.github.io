List of Warning Messages

The following list contains the warnings exported from the VFX runtime and their resolution methods.

Timing of Occurrence Message Reason for Occurrence How To Resolve the Issue
When the effect system is generated.

[Stripe] Memory Allocate Error!!

[ConnectedStripe] Memory Allocate Error!!

[SuperStripe] Memory Allocate Error!!

 Could not allocate a buffer from the static heap when initializing the stripe system (when initializing the vfx system). Use detail::SetStaticHeap to increase the configured heap buffers. detail::SetStaticHeap resides in the nn::vfx namespace.
When an effect binary is being registered.

vfx system deleted the registered binary.

resource id : XX.

 Binary was registered already with the specified resource ID, so that binary was deleted.

Clear the registered effect binaries (nn::vfx::System::ClearResource) before destroying the vfx system.
When an effect binary is being removed. resource to clear does not exist.
resource id : XX.

When deleting the effect binary from the vfx system, the specified resource ID already does not contain a registered effect binary.

 When deleting effect binaries from the vfx system, verify whether the specified resource ID is still in use.

When an effect is being generated.

 A resource appointed at the time of emitter set generation is not registered.

A resource that is not registered was specified when the emitter set was generated.

Verify the resource ID that is passed when calling CreateEmitterSet or verify that the effect binary being used is registered in the system.
There is not emitter set information to be formed within an appointed resource. The specified resource does not have information for creating an emitter set. Verify the resource ID that is passed when calling CreateEmitterSet or verify that the effect binary being used is registered in the system.
EmitterResource Setup Failed.

Setup of the resource for playing the emitter failed due to insufficient buffers, etc.

Setup of the shader or rendering primitive failed.
Check whether other warnings or errors occur in the effect binary that is used.
CustomShader Callback not unestablished. CustomShaderID:XX

A custom shader is configured to be used for the emitter parameters, but the custom shader callback to use is not specified.

 Verify whether there are any problems with the settings for the parameters and callback functions for the custom shader.

CustomAction Callback is not unestablished.

CustomActionID:%d

A custom action is configured to be used for the emitter parameters, but the custom action callback to use is not specified.

 Verify whether there are any problems with the settings for the parameters and callback functions for the custom action.
There is no available EmitterSet instance. Could not find an available emitter set instance. Use nn::vfx::Config::SetEmitterSetCount to increase the emitter set instances.
There is no available Emitter instance. Could not find an available emitter instance. Use nn::vfx::Config::SetEmitterCount to increase the emitter instances.

DynamicHeap Alloc Failed.

Alloc size : XX Byte.

 Insufficient dynamic heaps. Use detail::SetDynamicHeap to increase the configured heap buffers. detail::SetDynamicHeap resides in the nn::vfx namespace.

GpuBuffer Alloc Failed. Alloc size : XX Byte.

GpuBuffer Alloced : AllocedSize/BufferSize.

 Insufficient GPU buffer. Increase the size of the GPU buffer set by nn::vfx::Config::SetGpuBufferSize.
When an effect is being deleted. EmitterSet has already been deleted. Emitter sets that were already deleted were deleted. Check whether there is a path where emitter sets that have already been deleted are deleted.
When an effect binary is being registered.

vfx system deleted the registered binary.

resource id : XX.

 Binary was registered already with the specified resource ID, so that binary was deleted.

Clear the registered effect binaries (nn::vfx::System::ClearResource) before destroying the vfx system.
When an effect binary is being removed. resource to clear does not exist.
resource id : XX.

When deleting the effect binary from the vfx system, the specified resource ID already does not contain a registered effect binary.

 When deleting effect binaries from the vfx system, verify whether the specified resource ID is still in use.
When rendering an effect. Particle sort has failed. More buffer size is needed. Insufficient particle sort buffers. Use nn::vfx::Config::SetParticleSortBufferCount to increase the configured sort buffers.

TemporaryBuffer Alloc Failed. Alloc size : XX Byte.

TemporaryBuffer Alloced : AllocedSize/BufferSize.

 Insufficient temporary buffer. Increase the size of the temporary buffer set by nn::vfx::Config::SetTemporaryBufferSize.
Stripe There is no available Stripe instance. Could not find an available stripe instance. Use nn::vfx::Config::SetStripeCount and SetSuperStripeCount to increase the number of stripes that can be generated.
Info

For more information about nn::vfx::Config and buffers, see Memory Consumption in the Runtime.
