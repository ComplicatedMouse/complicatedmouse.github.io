Place Text in a Circle

This section describes the feature for placing text in a circle using character-based transformations.

Overview

The feature for placing text in a circle can place text configured in a text box pane along an arc.
This section describes the settings for placing text in a circle.

A Starting angle Specifies the starting and ending angles of the arc.

The value 0 defines the right side, and values move counterclockwise around the circle in 360 units.
B Ending angle
C Circle radius X

Specifies the radius of the circle in the horizontal and vertical directions.

Change the horizontal and vertical sizes to have the text follow along an oval shape.
D Circle radius Y
E Number of Divisions Specifies the number of keys when the circle is expressed as a curve.
More keys make for a smoother arc shape, but increase the processing load.
F Character orientation

Specifies the angle, relative to the slope, for each character placed along the circle.
In the figure, the text origin is set to center, and the character orientation is set to 90.
