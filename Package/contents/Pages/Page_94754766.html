Directional Polygon

This section describes the particle type Directional Polygon.

Overview

When a Directional Polygon type particle moves, it refers to its position in the previous frame in which it moved and tilts in that direction of motion.
The effect can be made flat like a board because, unlike billboard, the particle's direction of rotation is not affected by the direction of the camera.

Info

Note About Using Directional Polygon Type Particles
Particles will not be displayed if Specified-Direction Initial Velocity is set to a nonzero value and the Specified direction is XYZ = 0,1,0 because in this case the direction of the displayed face is indeterminate.
If this happens, set Specified Direction Dispersion Angle to 0.0001. The particles are displayed by adding a little velocity in a direction away from the y-axis.
