Workflow

This tutorial describes the workflow from the starting of EffectMaker to the completion of the task.

1 Starting the Task Start EffectMaker and start the task.
2 Loading a File Load the emitter set file (ESET) in EffectMaker.
3 Connecting to the Viewer Start EffectViewer and connect to it.
4 Completing the Task Complete the EffectMaker task.

1. Starting the Task

To start the task, start EffectMaker to configure the effect settings.

Starting EffectMaker

To start EffectMaker, double-click the following startup file.

\Tools\Graphics\EffectMaker\EffectMaker.exe

2 Loading the File

Load the emitter set file (ESET) that serves as the effect data.
For example, you can open an emitter set file stored in the following location.

\Samples\Sources\Resources\Vfx\Eset\Fire_01.eset

To open the emitter set file Fire_01.eset, do one of the following.
The Fire_01 () emitter set is shown in the workspace.

  • From the main menu, select File > Open (), select the emitter set file Fire_01.eset in the dialog box, and then click OK.
  • Select Fire_01.eset in Windows Explorer, and then drag it to the workspace window.
  • Press F10 to open the Effect Browser (), and then, after selecting Fire_01.eset, click Export to Workspace ().

Select the platform and the viewer to connect to from the presets, and start and connect the viewer.

The content of the selections changes based on the selected platform.

Reconnecting to the Viewer

If you have been disconnected from the Viewer, use either of the following methods to reconnect.

  • From the main menu, select Preview > Resend ().
  • On the toolbar, click Disconnected > Connect ().
  • Press the F5 shortcut key.

4 Completing the Task

Complete the EffectMaker task.

Exiting EffectMaker

To close all open files and exit EffectMaker, do one of the following.

  • From the main menu, select File > Exit.
  • On the main window, click Close.
  • The keyboard shortcut is CTRL+Q.

If there are any files that have not been saved, a dialog box appears. Specify a save destination in the dialog box that opens, and then click OK.

Exiting the Windows Viewer

To exit the Windows Viewer, do one of the following.

  • Right-click the Windows Viewer title bar, and select Close from the menu that opens.
  • Click the Close () button in the upper-right corner of the Windows Viewer window.
  • With the Windows Viewer activated, use the shortcut key ALT+F4.
