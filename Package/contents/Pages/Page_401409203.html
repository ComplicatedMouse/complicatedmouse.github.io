Plug-In for Setting Animation User Data

Sets the user data for each bone in skeletal animation to transform nodes (joint nodes).

How to Use

Select Nintendo > Set Animation User Data to open the window.

Selected transform nodes and joint nodes to be displayed under Selected Transform Node.
Camera, light, and fog transform nodes are not subject to settings in this plug-in, so they are not displayed.

Setting Method

If only one transform node (joint node) is selected, the current settings for the selected node are shown in the window.

A list of configured user data is displayed in the lower part of the window.

Operation methods are the same as Plug-In for Setting User Data.
