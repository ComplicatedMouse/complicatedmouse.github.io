Introduction

This page describes the graphics APIs that are available with the NX for the following purposes.

  • Understanding the features of each graphics API
  • Providing a quideline to use when considering which graphics API to use

The following section describes the APIs.

GFX

GFX is a graphics API that abstracts the NVN and OpenGL low-level graphics APIs. For more information, see gfx Library.

This API is suited to the following purposes.

  • High performance and easy migration to a future low-level graphics API
  • Using NintendoWare

It is highly likely that future platforms will provide new graphics APIs optimized to the platform. Even in this case, the plan is for GFX to support this functionality in the form of more supported low-level graphics APIs. This support would enable the use of a fast low-level graphics API provided for each platform using shared code.

When using NintendoWare, you must also use the gfx library.

In addition, GFX supports interoperability with low-level graphics APIs. This support allows conversion from gfx library objects to low-level graphics API objects, and conversion in the other direction. You can use this feature to use the gfx library API and a low-level graphics API together.

NVN

NVN is a low-level 3D graphics API supported by the NX. For more information, see nvn Library.

This API is suited to the following purposes.

  • Maximum performance on the NX

NVN is designed to enable maximum performance on the NX. Higher performance than any other graphics API is possible. However, it is highly likely that future platforms will provide new graphics APIs optimized to the platform, and NVN is not easily migrated.

NVN supports interoperability with OpenGL. This support allows conversion from OpenGL texture objects to NVN texture objects. Reverse conversion is not possible. You can use this feature to use the NVN library API and the OpenGL API together through textures.

The API supports GLSL, SPIR-V, and binary formats as shader formats.

OpenGL

This graphics API was established by the Khronos Group. For more information, see OpenGL Library.

This API is suited to the following purposes.

  • Migrating existing titles, and using frameworks that use OpenGL

The NX supports OpenGL mainly for the purpose of migrating from other platforms.

Vulkan

This graphics API was established by the Khronos Group. For more information, see Vulkan Library.

This API is suited to the following purposes.

  • Migrating existing titles, and using frameworks that use Vulkan

The NX supports Vulkan mainly for the purpose of migrating from other platforms.
