Added support for the improvements made to access log output.

The following improvements were made to the access logs in NX Add-On 10.2.0.

  • Added entry_count and entry_buffer_count to the nn::fs::MountSaveDataReadOnly() access log.
  • Added nn::fs::GetDirectoryEntryCount() access log output.
  • Added read_size to the nn::fs::ReadFile() access log.
  • Added nn::fs::GetFileSize() access log output.
  • Added nn::fs::GetFileOpenMode() access log output.
  • Added entry_type to the nn::fs::GetEntryType() access log.
  • Added open_mode to the nn::fs::OpenDirectory() access log.
  • Added size to the nn::fs::GetFreeSpaceSize() access log.
  • Added size to the nn::fs::MountRom() access log.
  • Added nn::fs::CanMountRomForDebug() access log output.
  • Added the save_data_size and save_data_journal_size items to the nn::fs::GetCacheStorageSize() access log.
  • Added cache_storage_count to the nn::fs::ReadCacheStorageList() access log.
  • Added size to the nn::fs::MountAddOnContent() access log.
  • Added save_data_size and save_data_journal_size to the nn::fs::GetSaveDataSize() access log.
  • Added nn::fs::GetFileTimeStampForDebug() access log output.
  • Changed function of the nn::fs::MountSdCardForDebug() access log from MountSdCard to MountSdCardForDebug.
  • Added root_path to the nn::fs::MountHost() access log.
  • Changed function of the nn::fs::UnmountHostRoot() access log from Unmount to UnmountHostRoot.
