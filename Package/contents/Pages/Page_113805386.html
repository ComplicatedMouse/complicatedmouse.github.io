Introduction

Purpose of This Documentation

This documentation describes the perf library used to provide performance metrics included in NintendoSDK.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Chapter Descriptions

The Manual chapter describes topics that you must understand when using the features of the perf library.

Related Documentation

Also see the perf API Reference.

perf Library Features

The perf library provides performance metrics and tabulation features for the metrics.
Using this library enables you to tabulate the difference for system ticks and to check performance data.

How the perf Library Is Provided

See the following pages.

Features Provided by the perf Library

The current perf library provides the following features.

  • Performance Metrics Tabulation
  • CPU Performance Metrics
  • GPU Performance Metrics
Info

The perf library performs measurements and tabulations at the application level.
CPU and GPU profiling tools, in which registers are read to conduct profiling at the hardware level, will be provided separately in a future release.

For more information about each feature, see the Manual.
