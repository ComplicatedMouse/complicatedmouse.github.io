Multi-Export Plug-In

This section describes the use and features of the multi-export plug-in.

Features

You can use the multi-export plug-in to export more than one model and more than one animation all at the same time by saving multiple export settings for a scene.

The multi-export plug-in exports data based on a combination of the following settings. These combinations of settings are called export items.

  • The filename of the intermediate file to be exported
  • The path to the folder to export intermediate files
  • The path to the FMD file to merge
  • The path to the FDES file to use
  • The node name to be exported
  • The scene layer node name to be exported
  • The name of the effective animation layer
  • Comment text for the intermediate file
  • The range of use when exporting an animation
  • A loop playback flag
  • Export flags for each intermediate file
  • Additional options
Info

Exported items are saved as custom attributes of the root node of the scene.

How to Use

Select Nintendo > Multi Export to display the window.

A list of export items set for the current scene is displayed in the window that appears.

  • Output File Name
    The name of the intermediate file to export.
  • fdes File
    The name of the FDES file to use.
  • Node
    The name of the node to export.
  • Layer
    The name of the display layer to export.
  • Comment
    Editing comments for the intermediate file.
  • Start End
    The range of animation to export.
  • Loop
    The flag for looping playback.
  • Selection
    Export flags for each intermediate file. The following letters are shown if they are enabled.
    M: FMDB file
    T: FTXB file
    S: FSKB file
    V: FVBB file
    C: FCLB file
    R: FTSB file
    P: FTPB file
    H: FSHB file
    N: FSNB file
  • Options
    Additional options.
    After the path of the export folder, the path of the FMD file to merge, and the name of the effective animation layer have all been set, select an option from the list to display it below the list.
    Click Add to add new user data. If an item in the list is selected, data is inserted in the selected location; if no item is selected, data is added to the end of the list. (The selection can be canceled by clicking "-----" in any location where there is nothing in the listed.)
    The setting made for an existing export item can be changed by selecting an item in the list and clicking Edit or by double-clicking the item.
    To duplicate an item (or items), select one or more items from the list and then click Duplicate.
    Export items can be deleted by selecting items and clicking Delete.
    You can change the order of export items in the list by selecting one or more items and clicking Move Up or Move Down. (The order does not effect the intermediate file being exported.)
    Select an item in the list and click Load fdes to Scene to load the settings from the fdes file entered into the scene and display the export plug-in Options menu. (Option settings for the current scene are displayed if the path is left blank.)
    Select an item from the list and click Get Playback Range to get the animation playback range on 3ds Max and use it to set the start frame and end frame for the export item.
    Select an item from the list and click Set Playback Range to change the playback range of the animation used by 3ds Max so that the playback range specified by Start / End Frame for the export item is played.
    Items selected in the list can be exported by clicking Export Selected Items.
    You can export all export items by clicking Export All Items.
    A window for entering export items can be brought up by clicking Add or Edit.
  • Output File Name
    Enter the name of the intermediate file to export (excluding the filename extension).
    Click Scene on the right to set the current scene name.
    Clicking Node allows you to enter a name for the node currently selected in Maya (or the first node selected if more than one is selected).
  • Output Folder
    Enter the folder to export the intermediate files to.
    If this setting is left blank, the settings of the FDES file are used. (If the File setting in the FDES file is also blank, the current scene's settings are used instead.)
    If the specified path is not a full path, the system treats it as a relative path from the folder storing the current scene file.
  • Merge fmd File
    Merges the specified FMD files if you select this option and specify an FMD file path.
    If the path is left blank, the export file to be overwritten is merged.
    If the specified path is not a full path, the system treats it as a relative path from the folder storing the current scene file.
  • fdes File
    Enter the path to the FDES file you are using.
    If left blank, the current scene's option settings are used for export.
    If the specified path is not a full path, the system treats it as a relative path from the folder storing the current scene file.
    You can select an FDES file from the history of recently imported FDES files by clicking Recent fdes at the bottom.
    Click Load fdes to Scene to load the settings from the FDES file entered into the scene and display an export plug-in option window. (Option settings for the current scene are displayed if the path is left blank.)
  • Node
    Enter the name of the node to export if you are exporting data for only a particular node. (Nodes in the hierarchy below this node will also be exported.)
    You can enter multiple node names by separating them with spaces.
    You do not need to enter anything if the FDES file is being used to export the entire scene.
    Conversely, only the node entered will be exported if you enter a node name even though the FDES file is set to export the entire scene.
    Clicking Get located on the right enters the name of the node currently selected in 3ds Max.
  • Click Select to select the entered node on 3ds Max.
  • Layer
    To export a particular layer, enter the name of that layer to export.
    You can enter multiple layer names by separating them with spaces.
    You do not need to enter anything if the FDES file is being used to export the entire scene.
    But even if the entire scene is being exported, if you enter layer names here, only those layers will be exported.
    Clicking Get located on the right enters the name of the node currently selected in 3ds Max.
    Clicking Set makes the entered layers visible on 3ds Max.
  • Anim Layer
    To enable a particular animation layer as an active layer, enter the name here.
    You can enter multiple animation layers by separating them with spaces.
    Enter Base Layer to deactivate all non-base animation layers.
    If this setting is left blank, all currently active animation layers are exported.
    Click Get on the right to enter the name of the animation layers currently active into 3ds Max.
    Click Set to enable only the entered animation layers on 3ds Max.
  • Comment
    Specifies comments about editing the intermediate file.
    Comments cannot include semicolons (;) or quotation marks (").
  • Start / End Frame
    Specifies the range (start frame and end frame) of the animation to output.
    This setting works the same way as the Start / End Frame option of the Export plug-in.
  • Loop
    Sets the loop playback flag.
    Select this option to export the data as looped playback data.
    If this option is cleared, the animation range is exported as one-time playback data
  • Output File Selection
    Sets the export flags for the intermediate files.
    Click Get from fdes to export only the intermediate files that are set for export using the specified FDES file in fdes File. (If the path is left blank, all intermediate files set for output in the current scene are enabled for export.)

  • Additional Options
    Additional options can be specified using the same format described for Using Option Strings, separating them by spaces without using double quotation marks.
    You can enter multiple options either on a single line or by using returns and entering them on multiple lines.
    These additional options take priority over the options specified in the FDES files.
    For example, you can enter specifications as follows if the setting for Frame Precision in the FDES file is 1.0, but you are exporting it as 0.5.

    -fp 0.5
    		 
    Click OK to add or modify export items.
Info

If the FDES file for the exported items uses presets, locked settings in the preset file have priority over export item settings.
For example, if the Output Folder specified by preset has been locked as D:/data/, data is exported to D:/data/ even if D:/tmp/ is set as the Output Folder for the export item.
The same is true for other option settings such as Merge fmd File, Start / End Frame, Loop, and Output File Selection.
