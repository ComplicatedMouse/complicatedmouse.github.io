Managing Patches

Use the patch command. To run the patch after it has been installed, use the application command.
For more information about the handling of patches, see Building and Running a Patch.

  • Installing
  • List
  • Uninstalling 

$ RunOnTarget.exe DevMenuCommand.nsp -- patch --help
usage: DevMenuCommand patch install <absolute_nsp_path>
       DevMenuCommand patch list
       DevMenuCommand patch uninstall <application_id>
[SUCCESS]
Common Options Argument Values Description
-s Name of the target storage. Specifies the storage that will be the target of the process. For more information about the storage names that can be specified with the argument, see the description of the application install command.

Installing

$ RunOnTarget.exe DevMenuCommand.nsp -- patch install C:\patch.nsp
