Camera Type

This section describes the Camera Type feature for setting the camera type.

Overview

For the camera type, select either Perspective (perspective applied) or Ortho (no perspective applied).
This section provides an overview of the settings ().

Camera Type

Selects the camera type from the following presets:

Perspective A perspective projection camera. This camera applies perspective to give depth to an object based on the angle of view.
Ortho An orthographic projection camera. A camera that does not apply perspective to an object based on the angle of view.
