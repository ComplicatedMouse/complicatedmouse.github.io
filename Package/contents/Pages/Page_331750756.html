nn::font News ()

Removing the fontll2 Library

Due to problems with the delivery method for the fontll2 library, it has been deleted from the package.

If you have been using the fontll2 library, use the fontll library instead.

Increased the maximum number of fonts that can be combined in one multi-scalable font typeface

The maximum number of fonts that can be combined in one multi-scalable font typeface has been increased from 8 to 16.
Because this change has also increased the size of nn::font::TextureCache::InitializeArg from 45 KB to 90 KB, a stack overflow may occur when allocating to a stack. In this case, change the allocation to a heap.

Fixed a bug where stopping a sound with the release value set to 127 stopped playback of the sound before the fadeout was finished.

This fix affects the following processing.

  • Waveform playback with the atk runtime
