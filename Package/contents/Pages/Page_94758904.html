Offset Z value

This section describes the Offset Z value feature for eliminating overlapping polygons by manipulating particle depth values.

Overview

The offset Z value feature is a particle offset feature that can be used to manipulate the depth values of particles.
The depth information of the particle is overwritten by a value that is the product of this setting and the offset toward the camera.
Because the particle is not actually offset toward the camera, the taken path does not change the particle size, and overlapping polygons can be removed.

This section provides an overview of the settings ().
When you set Particle Offset to Offset Z value, the Particle Offset settings () appear. The group has the following settings.

Particle Offset Value

You can set the offset value for camera direction with the dial () or text box (), based on the emission position.
To apply the offset further back behind the position of emission, specify the negative direction.

Multiply Offset by Scale

Use the check box () to select whether to multiply the z value by the scale applied to the emitter and the final result of applying the scale to the particle.
When this setting is selected, you can multiply by the scale to remove overlapping polygons.
