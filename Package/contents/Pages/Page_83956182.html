Shortcut List

This page lists the shortcut keys available for use in 3DEditor.

  • Main Window
    • Object View
    • Animation Controls
    • Log Bar
  • Curve Editor
  • Texture Details Window
  • Slider Control

Main Window

These shortcut keys correspond to items on the main menu in the main window.

Operation Corresponding Menu Item Description
CTRL+O File > Open Opens a file.
CTRL+W File > Close Closes the selected file.
A save file dialog box appears if there are any edited files.
CTRL+SHIFT+W
CTRL+ALT+W		 File > Close All Closes all loaded files.
A save file dialog box appears if there are any files being edited.
CTRL+S File > Save Saves files.
CTRL+SHIFT+S File > Save As Saves the current files with a new name.
CTRL+ALT+S File > Overwrite and Save All Overwrites and saves all loaded files.
F2 File > Change Name Renames the selected file.
CTRL+M File > Merge > Model Merges the selected file.
This shortcut key displays a select file window that allows you to select model files to be merged.
CTRL+Q File > Quit Closes the 3DEditor.
CTRL+Z
Z		 Edit > Undo Undoes the previous operation.
CTRL+Y Edit > Redo Redoes the previously undone operation.
CTRL+P Display > Property Window Opens the property window for the selected file.
CTRL+1 Display > Object View > One Screen Displays the object view in a single screen.
CTRL+2 Display > Object View > Two Screens: Horizontal Displays the object view in two screens, top and bottom.
CTRL+3 Display > Object View > Two Screens: Vertical Displays the object view in two screens, left and right.
CTRL+4 Display > Object View > Triple Screen: Split Bottom Displays the object view in a three-screen layout by splitting the bottom screen.
CTRL+5 Display > Object View > Triple Screen: Split Top Displays the object view in a three-screen layout by splitting the top screen.
CTRL+6 Display > Object View > Triple Screen: Split Right Displays the object view in a three-screen layout by splitting the right screen.
CTRL+7 Display > Object View > Triple Screen: Split Left Displays the object view in a three-screen layout by splitting the left screen.
CTRL+8 Display > Object View > Four Screens Displays the object view in a four-screen layout.
F5 Preview > Connect to/Disconnect from Application Connects to the application.
Disconnects from the application if already connected.
F1 Help > Help Opens 3DEditor Help.

Object View

These shortcut keys are available when Object View is active in the main window.

Operation Description
CTRL+A Selects all objects in Object View.
PLUS SIGN (+) Increases the view magnification. (Only available when a schematic is displayed.)
MINUS SIGN (-) Reduces the view magnification. (Only available when a schematic is displayed.)
A Display all in center. (Only available when a schematic is displayed.)
F Displays the selected objects in the center of the View. (Only available when a schematic is displayed.)
F3 Opens the comment associated with the selected object for editing. (Only available when a schematic is displayed.)

Animation Controls

These shortcut keys correspond to the items in the animation controls in the main window.

Operation Description
V Plays or pauses the animation.
SHIFT+V Stops the animation. (The current frame becomes the animation start frame.)
ALT+COMMA Reverses the animation frame-by-frame.
ALT+PERIOD Advances the animation frame-by-frame.

Log Bar

These shortcut keys can be used to perform actions on the log displayed in the Log Bar in the main window.
These shortcut keys can only be used while the Log Bar is selected.

Operation Description
CTRL+A Selects the entire log.
CTRL+C Copies the selected log.
DELETE Clears the entire log.

Curve Editor

These are the shortcut keys available for use in Curve Editor.

Operation Description
S Adds a key to the frame associated with the current time (red line) displayed in the Curve View.
DELETE
BACKSPACE		 Deletes the selected key.
CTRL+C Copies the selected key.
CTRL+V Pastes the copied or cut key.
CTRL+X Cuts the selected key.
A Fits the entire curve in the view.
F Fits the selected keys in the view.
CTRL+PLUS SIGN (+) Increases the view magnification.
CTRL+MINUS SIGN (-) Reduces the view magnification.
ANGLE BRACKETS ( < > ) For all the keys displayed in Curve View, jumps to the position of the key before or after the current frame.
LEFT/RIGHT ARROW KEYS (← → ) Shifts the selection state of the keys in Curve View to the next key to the left or right.
E Changes the key mode to Move the closest key.
I Changes the key mode to Insert a key (while pressing the I key).
R Changes the key mode to Scale Operation.
W Sets the key mode to Move multiple keys.
C Enables or disables the feature to snap-to another frame or value.

Texture Details Window

The following shortcut keys are available for use in the Texture Details Window.

Operation Description
CTRL+PLUS SIGN (+) on the numeric keypad Increases the view magnification.
CTRL+MINUS SIGN (-) on the number keypad Reduces the view magnification.
CTRL+1 Displays the RGB channels.
CTRL+2 Displays the R channel.
CTRL+3 Displays the G channel.
CTRL+4 Displays the B channel.
CTRL+5 Displays the A channel.
CTRL+M Shows or hides the mipmap.
CTRL+D Switches the display direction for the mipmap between vertical and horizontal.

Slider Control

The following shortcut keys can be used when using a slider control.

Operation Description
R Resets the value of the slider control to the default value.
PLUS SIGN (+) Increases the range of values on the slider.
MINUS SIGN (-) Decreases the range of values on the slider.
