Instrument List

The Bank tab is divided into upper and lower sections. The upper section is called the Instrument List.


Role of the Instrument List

The Instrument List is used to create instruments and specify program numbers. The Instrument List functions as a tone set when a sequence references a bank.
Single-region instruments can be edited by using only the Instrument List.

Creating Instruments

With the Bank tab in the active state, select Project > Add Item > Instrument.

To create or add a new instrument, use the shortcut menu that appears when you right-click the Instrument List.


The original key for the default state of the new instrument is set automatically based on the filename of the specified waveform file, and additional information.
(For example, the A3 portion of the Test.A3.aif filename.)

For notes written to filenames, follow the sequence command rules. Use the following order for settings.

  1. If the waveform filename contains characters that specify a key, use this key as the original key.
  2. If it does not contain such characters, the original key information is obtained from the waveform file itself.
  3. If step 2 fails, the key is set to cn4.

Selecting Targets for an Envelope

For a description of envelopes, see Instrument Parameters.

You can make target settings separately for each instrument.
You can switch targets by using the shortcut menu for the Instrument List.

Opening a Folder or Reference File

To perform operations on the files, select either Open Folder or Open Referenced File from the shortcut menu.

Renumber Programs

When Renumber Programs is selected from the shortcut menu, the items selected by the instrument can have their programs renumbered.

The dialog box for renumbering programs appears. Enter the starting number.

If an existing number is duplicated, an alert dialog box appears.

Renumbering programs enables you to easily view items from the same series by lining them up later, and intentionally arrange instruments that are already registered by specifying program changes that use variables.

Editing the Instrument List

This section describes the meaning of each of the elements that define the Instrument List.

Program Numbers

Specifies the instrument assigned to a program change number when a bank is referenced by a sequence.
Although only 128 program numbers can be specified for MIDI program changes, some numbers are likely to be specified for text sequences. Use the instrument list to specify program numbers in excess of this limitation.

Name

Specifies the name of the instrument.

The characters that can be used in a name are restricted. The following characters can be used:

  • Single-byte alphabetic characters and the underscore [a-z, A-Z, _] for the first character
  • Single-byte alphanumeric characters and the underscore [a-z, A-Z, 0-9_] for subsequent characters

Compression Format

Specifies the compression format that is used by the converter to convert reference waveform files.

  • ADPCM
    Converts the waveform file to the proprietary ADPCM format.
  • PCM16
    Converts the file to 16-bit PCM data.

File path

Specifies the path to the waveform file referenced by an instrument.
This specification can be made from the Instrument List only for single-region instruments. For multi-region instruments, an asterisk (*) is displayed in this column to indicate that editing is impossible.

The characters that can be used in a filename or path name are restricted. The following characters can be used.

  • Single-byte alphabetic characters and the underscore [a-z, A-Z, _] for the first character of the filename.
  • Single-byte alphanumeric characters and the underscore [a-z, A-Z, 0-9, _] for subsequent characters in the filename.

Playback time

Displays the playback time for sounds (the length of the waveform data expressed as time).
The units are displayed in mm:ss. ms and the value in milliseconds are rounded to the second digit.

For looped waveform data, this shows the time to the end of the loop, not the length of the waveform itself.
In addition, an L is displayed at the front of the cell.

Ticks

Displays the length of the required note commands.
Ticks are units for specifying the length of notes and rests within text sequences.

The number of ticks per minute (tpm) and ticks per second (tps) depend on both the tempo and the quarter note resolution. Assuming that the default tempo value of 120 quarter notes per minute (qpm) is set, and that the quarter note resolution of the text sequence is set to the default value of 48 (which differs from the value of the SMF Quarter Note Resolution setting under Project Settings), the tpm and tps rates can be calculated as follows.

  • 1 minute = 120 * 48 = 5,760 ticks
  • 1 second = 120 * 48 / 60 = 96 ticks

The tempo and quarter note resolution values correspond to these rates.

Knowing the number of ticks is useful when writing text sequences.

Note that changing the pitch of an instrument causes the actual playback duration to increase or decrease, and the number of actual ticks taken for playback to change accordingly.

Downsampling

Specifies whether downsampling is enabled or disabled.

You can downsample and include waveform files in the sound archive, using the value specified as the sample rate, without changing that original waveform file, when this option is enabled.

Sample Rate

Displays the sample rate of the waveform file when downsampling is disabled.

The downsampling value can be specified in a range from 2,000 to 192,000 Hz when downsampling is enabled.

Bit Rate

Displays the bit rate of the waveform file.

Number of Quantization Bits

Displays the quantifying bit number of the waveform file.

Number of Channels

If there are one or two waveform channels, either Mono or Stereo appears. If there are three or more channels, just the number of channels appears.

Downmixing

This feature enables and disables downmixing.

When enabled, this feature downmixes stereo into mono without changing the original waveform, and includes it in the sound archive.

Data Size

Shows the size of the waveform data after compression and conversion.
If the instrument references multiple waveform files (for example, a multi-region instrument), the total size of the reference files is shown.
The content on display is updated when files are converted.

Original Key

Specifies the original key (that is, the position on the keyboard where the waveform file is played at its original pitch), for the referenced waveform file. Specifies the seconds.
This specification can be made from the Instrument List only for single-region instruments. For multi-region instruments, an asterisk (*) is displayed in this column to indicate that editing is impossible.

Env A

Specifies the envelope attack time as a value in the range from 0 through 127.
This specification can be made from the Instrument List only when envelope settings are applied at the instrument level. If the settings are applied at the region level, an asterisk (*) is displayed in this column to indicate that editing is impossible.
This operation is the same for each of the following phases of envelope specifications.

Env H

Specifies the envelope hold time as a value in the range from 0 through 127.

Env D

Specifies the envelope decay time as a value in the range from 0 through 127.

Env S

Specifies the envelope sustain level as a value in the range from 0 through 127.

Env R

Specifies the envelope release time as a value in the range from 0 through 127.

Volume

Specifies the volume of the instrument as a value in the range from 0 through 255.

If sound data volume is set to 127, playback is at the same volume (1.0 times) as the source data.

However, volume can be set in many ways (data or program). The result after all setting values are applied is clamped between 0.0 times and 2.0 times the source volume. For more information, see Sound Parameter Control Operations in the Sound System Manual.

Pitch (Semitones)

Specifies the pitch of the instrument in semitones.

Pitch (Cents)

Specifies the pitch of the instrument in cents (1 cent = 1/100th of a semitone).

Interpolation Type

Specifies the interpolation type at the instrument level.

Configure the voice interpolation method. Select 4-point interpolation or Linear interpolation. The default interpolation value is 4.

Currently, even if this interpolation type setting is changed, the change is not applied to the interpolation method.
