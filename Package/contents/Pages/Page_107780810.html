Introduction

Purpose of This Documentation

This documentation describes the crypto library features included in NintendoSDK.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Related Documentation

Refer to this information together with the crypto API Reference.

crypto Library Features

The crypto library provides encryption features such as the following.

  • Date encryption and decryption
  • Generation and authentication of digital signatures
  • Calculation of hash values using a cryptographic hash function
