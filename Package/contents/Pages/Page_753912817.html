Example 1: Take a Profile and Save it to Disk
# Take a profile with the current settings and save it to disk.
#   1. Start profiling with the existing settings.
#   2. Stop profiling after a set time.
#   3. Once the profile is loaded, save it to disk.
# Note: This example assumes there is already a dev kit synced.


# 1. Start profiling with the existing settings.
Profiler-Start Sampled

# 2. Stop profiling after a set time.
Profiler-Wait 2.0
Profiler-Stop

# 3. Once the profile is loaded, save it to disk.
$outputPath = Join-Path -Path $env:TEMP -ChildPath "example-1.nprof"
Profiler-Save -FileName $outputPath
