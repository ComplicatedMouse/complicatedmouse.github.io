Changed it so that the TGA file footer is exported when exporting a TGA file

Changed it so that the TGA file footer is exported when exporting a TGA file with the Photoshop Texture Export Plug-In.

It did not export footers in NX Add-On 5.3.0 or later, but it was changed to always export them to match the behavior through NX Add-On 5.2.0.
(Because TGA was exported with the Photoshop feature until NX Add-On 5.2.0, the footer was exported.)

Maya Plug-In

You can now restrain file node output.

You can now restrain export when you set a string starting with nw_ignore in Hint Information with the Plug-In for Setting Sampler Attributes.

Multiple file nodes are connected to layered textures for display verification on Maya, but it has been changed so you can configure and not export them in cases where you only want to export part of it.
UV is also no longer exported. When exporting UV, configure the UV set names to start with nw4f_uv or nw4f_fix.
