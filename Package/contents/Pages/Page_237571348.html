Create a bone visibility animation.

Visibility

From the outliner, select the Visibilty_Model model and create an animation key for Visibility.

Create a key like the one in the following table.

0 frames 25 frames
Visibility Selected Off

Check that the model is like the bone visibility animation in the following figure.

Export Target

The visibility animation export target is the Visibility key frame.

This corresponds to A, Bone Visibility Animation Data, in the Output File Selection of Nintendo Export Options.
