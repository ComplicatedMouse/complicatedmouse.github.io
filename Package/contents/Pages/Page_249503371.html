Do Not Tab Layout Windows

Disabling Layout Window Tabbing

The Do Not Tab Layout Windows feature lets you choose how to display layout windows when more than one is opened at the same time.
There are two choices: Tabbed View and Separate Windows.

From the menu bar in the LayoutEditor window, select Window > Do Not Tab Layout Windows.

Do Not Tab Layout Windows

When this menu item is selected, layout windows are opened in separate windows, rather than as tabs in a single window.
Choose the mode to match your task.

Note: All layout windows must be closed to switch the tabbing setting.
