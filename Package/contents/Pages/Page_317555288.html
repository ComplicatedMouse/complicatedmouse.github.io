Sample Mii Data for Debugging

Several sample Mii data (CharInfo) files have been prepared for the convenience of development.

These sample data files are located in the following directory in the SDK.

$(NintendoSDK)/Resources/Mii/Common/charinfo
Note

Use these Mii data files for debugging purposes only. Be careful not to include them in the retail product.

Using Mii Data Files

Mii data files use the ROM file system and the file system on the host PC to load files, which are cast and used in the nn::mii::CharInfo type. For more specific usage examples, see the MiiSimple Sample Description.

Descriptions of Sample Files

The following list shows the sample Mii data files provided by the Face library. Each file contains one Mii data (CharInfo) set.

Filename Overview
GenderFemaleBuild0Height0.dat Mii data for a female with the smallest build and shortest height.
GenderFemaleBuild0Height127.dat Mii data for a female with the smallest build and tallest height.
GenderFemaleBuild127Height0.dat Mii data for a female with the largest build and shortest height.
GenderFemaleBuild127Height127.dat Mii data for a female with the largest build and tallest height.
GenderMaleBuild0Height0.dat Mii data for a male with the smallest build and shortest height.
GenderMaleBuild0Height127.dat Mii data for a male with the smallest build and tallest height.
GenderMaleBuild127Height0.dat Mii data for a male with the largest build and shortest height.
GenderMaleBuild127Height127.dat Mii data for a male with the largest build and tallest height.
SampleBroken00.dat

Corrupted Mii data. Use for purposes such as checking error handling.

The two files types SampleBroken00.dat and SampleBroken01.dat have different types of corrupted data, but it is fine to use only one of the files to check error handling.
