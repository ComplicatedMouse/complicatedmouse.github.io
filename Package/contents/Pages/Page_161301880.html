Enable the feature for preventing blurring due to centering text

This feature is available in the Text Box Pane (Advanced) window.
When this feature is enabled, coordinates are converted to integer values by rounding the decimal portions so that centered text does not get blurry.

Although this prevents text from getting blurry, the text may get drawn slightly off-center, as shown in the images.


This image shows the resulting text when the feature is enabled (1) and disabled (2).



This feature was more effective in legacy platforms that had lower resolutions. On the more modern platforms that have much higher resolutions, text does not usually get noticeably blurry even when using decimal value coordinates.

This feature is only included to maintain backward compatibility. Leave it disabled normally and enable it if you observe noticeably blurry text.

By default, this feature is disabled.
Draw text on left in front

This feature is available in the Text Box Pane (Advanced) window.

When this feature is enabled, text on the left is drawn in the foreground.

Enable when you want to change the order in which text overlaps.

Text Box Pane (Advanced)

The Text Box Pane (Advanced) window provides more advanced settings than the Text Box Pane window.

Setting Widths at the Row Level

Edits the width of rows in text box panes at the row level.
The width of rows is reduced by taking the horizontal size (the x-value) set in the base pane as the maximum width of that row, and setting a negative value as the amount to move it left and right.
Values can be set for up to 16 rows.

Setting Widths at the Row Level
Enable

Size X The size of the text box pane in the X direction.

Shows the size of the text box pane in the X direction.
This is the size in the x-direction configured for the base pane in the property window.

When you adjust the width on a per-line basis, the size of each line changes, so this number serves as a reference value.
A Settings for narrowing the row from the left.

Setting a negative value in pixels moves the left edge of the row toward the right edge.

Per-Character Transformation

Enter information about per-character transformations.

Translation and rotation transformation processes can be applied on a per-character basis in a text box pane.
The transformation processes can also be animated.
This makes it possible to more easily create special effects such as positioning characters along an arch or having characters undulate.

Enable

Select the check box to enable per-character transformations.
Offset/Duration Animation Simple Settings
A feature for easy configuration of the per-character transformation settings.
Select Enable to activate this button.
Click the active button to open the Offset/Duration Animation Simple Settings window.
Offset time

This setting specifies a time frame to use for character animations when animating text on a per-character basis.

Specified as “from frame 00 to the specified frame”.


The specified animation time frame functions as an offset.

This offset can be used to specify playback speed and timing when the Per-character transformation: Offset time option is enabled in Curve Editor.

Info

By editing the Per-character transformation: Offset time curve for the offset animation in Curve Editor, you can freely specify the playback speed and timing for the animation.
Duration

Applies a time difference to the animation of each character.
Use this option to specify the timing at which each character animation is played.

Info

If you specify a value of -60 for the duration, it will take 60 frames longer for all character animations to be completed.

That is, the time at which the last character animation is completed will be 60 frames later.

If there is a large number of characters to be animated, an average duration to apply to each character is calculated by starting at the last character and working backward.
Looping

Specifies the loop setting.

  • One Time
  • Loop
Origin (V Direction) Sets the origin of the v direction for each character.
This origin is used for animating characters.
  • Center
  • Down
Height Offset A numerical value entered for the height offset from the origin when characters move.
It is an offset from the position of the origin as set in Origin (V Direction).
Divide Based on Character Width

While characters are placed with even spacing regardless of the standard character width, select this option to adjust character intervals according to the character width.
When lining up fonts for which the character width differs for each character, decreasing the interval between characters with smaller width makes the text look neater.

Automatically track shadow alphas to character alphas

 Selecting this option when using the shadow feature for characters makes the shadow alphas track to character alpha values.
When animating the character alpha values (to make it vanish), the shadow also vanishes naturally.
Place Text in a Circle

This feature allows you to place text in a circle.
To properly place text in a circle, select the Use the origin of conversion as the center of the text box check box.

Use the origin of conversion as the center of the text box

While normally the center of each character is the origin, selecting this option changes the origin for each character to a single location at the center of the text box.

Do not select this option if you want the animation start position to begin at the origin for each character.
