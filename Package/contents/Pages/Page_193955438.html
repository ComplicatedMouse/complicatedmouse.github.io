EffectMaker

This section introduces new features that have recently been added to EffectViewer.

Added Debug Display Feature

The following information has been added to the debug display of the viewer node.

Show GPU Metrics

The Show GPU Metrics feature displays the GPU processing load for each emitter.
Use this feature when you want to know the GPU load of effects in more detail.

GpuCost shows the GPU processing load for all emitters.
Shows the following information.

-Root Shows the total GPU processing costs of the entire viewer in milliseconds.
-VfxGpu Shows the total GPU processing costs of all of the effects in milliseconds.
-Compute Shows the GPU processing cost of the compute shaders.
-EmitterSet XXXX Shows the GPU processing cost of the emitter set currently being played back.
-Emitter XXXX Shows the GPU processing cost in milliseconds of each emitted currently being played back.
Info

Information is not displayed in the EffectMaker workspace for emitters for which the View icon () has been switched to Hide ().

GpuStatistics shows the GPU internal counter.

This shows the counter value not for each emitter, but for overall effect rendering.
Shows the following information.

|- InVert

The number of input vertexes.
|- InPrim The number of input primitives.
|- VtxShaderInvoke The number of vertex shader calls.
|- ClipInPrim The number of primitives processed by the clipping stage.
|- ClipOutPrim The number of primitives generated by the clipping stage.
|- PixShaderInvoke The number of pixel shader calls.
|- SamplePassed The number of samples that passed the fragment test.
Info

GpuStatistics is not displayed when the platform is the Windows environment.

Display Field Information

The Display Field Information feature displays the field and emitter plug-in node information in the viewer.
The following information is displayed when Display Field Information is enabled.

Magnetic Force The position of the magnetic force is displayed when the Magnetic Force field is used.
Spin

The rotation axis and rotation direction are displayed when the Spin field is used.
Convergence The position of convergence is displayed when the Convergence field is used.
Simple Collision The collision plane is displayed when the Simple Collision field is used.
Loop-In-Range The box that limits particle display is displayed when the Loop-In-Range emitter plug-in node is used.
