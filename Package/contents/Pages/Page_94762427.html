Shader Type

This section describes the feature for applying special effects to the rendering of particles using registered textures.

Overview

Shader Type is a feature using registered textures to apply special effects like distorting the background and other textures.
Distortion is an effective way to express effects like water.
The following overview describes the settings ().

Shader Type

Select the particle shader type from among the following presets.

Normal In this state, no shader type has been selected.
Color Buffer Refraction A feature for distorting the background using the RGB values of the texture registered in the Texture 0 tab.
Texture Distortion A feature for distorting the texture registered in the Texture 1 tab using the RGB values of the texture registered in the Texture 0 tab.
Coefficient Animation The Coefficient animation feature that animates the amount of refraction in Color Buffer Refraction, and the amount of distortion in Texture Distortion, will be explained.
