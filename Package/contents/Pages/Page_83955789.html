Overview

What Is the Parameters Panel?

The Parameters panel provides a GUI for editing various sound parameters.

For a sound list that has been selected in the Sound Set tab, you can use this panel to enter values for parameters like volume and priority, and to select or clear various check box options. For an instrument selected in the Bank tab, you can use this panel to enter values for parameters like envelope and pan.
For envelope values you can view a graph for visual confirmation of your operations.

Role of the Parameters Panel

The Parameters panel operates in conjunction with the Sound Set tab and the Bank tab.
There are sliders and up and down buttons to the side of the text boxes for entering numerical values, providing greater flexibility than editing in lists.

For the Bank tab, the combined use of the instrument and percussion lists helps when setting the envelope and other parameters that are hard to imagine with just a numeric display.
By using the Parameters panel together with the sample map, you can specify settings for envelope and pan that cannot be edited on the sample map.

The Parameters panel is displayed as a floating dialog box that can be moved outside of the main window of SoundMaker.

Parameters Panel Operations

Opening the Parameters Panel

To open the Parameters panel, either select View > Parameters Panel or click the Parameters Panel icon in the toolbar of the main window.
The shortcuts CTRL+I and ALT+8 can also be used for this operation.
These operations are used to switch between showing and hiding windows.

If a list is not active, its settings values will not be displayed in the Parameters panel.

Targets of Parameters Panel Operations

The Parameters displayed for editing are always the parameters of the active Sound Set tab or Bank tab.

Editing Parameters

The following example uses the Bank tab to describe the editing operations in the Parameters panel.

Editing Envelopes

By selecting an instrument or region anywhere inside the Bank tab and using the envelope operations in the Parameters panel, you can edit the envelope values for the selected instrument or region.

Edit values by using sliders for each phase.
The created envelope curve can be checked with the graph displayed in the Parameters panel. The times for Attack, Hold, Decay, and Release are accurately displayed in milliseconds at the top of the graph.

Slider operations have no effect if the envelope cannot be edited. This may occur in cases such as when a multi-region instrument is selected on the instrument list.

Editing the Volume

The volume can be specified independently for each instrument and region.

By selecting an instrument or region anywhere inside the Bank tab and using the volume operations in the Parameters panel, you can edit the volume values for the selected instrument or region.

Editing the Pitch

The pitch can be specified independently for each instrument and region.

By selecting an instrument or region anywhere inside the Bank tab and using the pitch operations in the Parameters panel, you can edit the pitch values for the selected instrument or region.

Pitch can be specified in units of half tones or in units of cents (1 cent = 1/100 half tone), or as a combination of both.

Editing the Pan

By selecting a region anywhere inside the Bank tab and using the pan operations in the Parameters panel you can edit the pan values for the selected region.
Edit values by using sliders to adjust the pan.

Slider operations have no effect if the pan cannot be edited. This may occur, for example, when a multi-region instrument is selected on the instrument list.
