Initial Diffusion Velocity

This page describes the All-Direction Initial Velocity feature that sets the particle initial velocity in the emitter normal direction.

Overview

The Initial Diffusion Velocity randomizes the emission direction and initial velocity of each particle on the specified axis.
Randomizing the initial velocity of each particle while keeping a fixed movement axis will add texture to the effect.
The following overview describes the settings ().

Initial Diffusion Velocity

Sets the axis along which emitted particles disperse and their velocity in the dispersion direction.

Diffusion Axis

The diffusion axis is specified in the emitter coordinate system.
The following image is a side view of a Circle (evenly divided) emitter with a tilted diffusion axis.

A Initial Diffusion Velocity X Fixes the initial diffusion velocity for emitted particles to the x-axis, based on a standard emitter that is inclined 45° about the z-axis.
B Initial Diffusion Velocity Y Fixes the initial diffusion velocity for emitted particles to the y-axis, based on a standard emitter that is inclined 45° about the z-axis.
C Initial Diffusion Velocity Z Fixes the initial diffusion velocity for emitted particles to the z-axis, based on a standard emitter that is inclined 45° about the x-axis.

Initial Diffusion Velocity

Particle emission velocity is calculated as the sum of the initial velocity and the initial diffusion velocity.
The summed value changes randomly with the specified value as the upper limit. A particle’s initial velocity when emitted is calculated using the following formula.

  • Initial emission velocity = Initial velocity + [value from -(initial diffusion velocity) to +(initial diffusion velocity)]
