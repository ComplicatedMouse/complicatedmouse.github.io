Features That Cannot Be Used With Capture Panes

Question

Some features do not yield the intended result when used with capture panes.

Answer

The following features make internal use of the capture pane feature, and if the features are used in a way that straddles layouts, such as for parts overwriting, the capture texture drawing order cannot be resolved so they are not displayed properly.

  • Capture Effect (Mask)
  • Capture Effect (Drop Shadow)
