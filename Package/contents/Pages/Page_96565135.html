Parameter Weight (±)

This page describes Parameter Weight (±), functionality for configuring the level of effect on particles from curl noise.

Overview

Parameter Weight (±) is functionality that allows you configure the level of effect on particles from curl noise independently for the x-axis, y-axis, and z-axis.
You can set the direction for affecting particle movement.
The following overview describes the settings ().

Parameter Weight (±)

The values for the level of effect on particles from curl noise is set using the dial () or text box ( ).
The larger the value, the greater the effect on particle speed. Negative values result in movement in the opposite direction. There is no effect when the value is 0.
