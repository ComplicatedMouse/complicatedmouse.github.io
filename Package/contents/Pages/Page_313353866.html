3DEditor

Added a feature for referencing custom labels using material referencing.

You can now use material referencing to reference shader parameter custom labels.


When the custom label for a child material is being edited, the custom label for the parent material is ignored and the custom label for the child material is applied.

Related Links For more information, see the following pages.

Editing Custom Labels Describes the material referencing settings in the shortcut menu.


Improved the file view scene animation operations.

Scene animation operations have been improved.

  • You can now show or hide scene animation files.
  • You can now drag to move scene animation files.
  • You can now drag to move animations in scene animation files.


Related Links For more information, see the following pages.

Bone sides are now displayed in the property window.

You can now display the values for bone sides used for motion mirroring.
You can check this General page of the window, in Bone Information.



Related Links For more information, see the following pages.

Bone Property Window Describes the property window for bones.

Photoshop Plug-In

Added support For Photoshop CC 2018 (version 19).

The Photoshop plug-in is now compatible with Photoshop CC 2018 (version 19).

If you install the NintendoWare package using the Nintendo Dev Interface (NDI), the Photoshop plug-in will be automatically set up if Photoshop is installed on your PC.
