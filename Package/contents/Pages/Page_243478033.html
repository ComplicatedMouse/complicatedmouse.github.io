Offset/Duration Animation Simple Settings

Per-Character Transformation Simple Settings

The Per-Character Transformation Offset/Duration Animation Simple Settings feature provides an easy way to configure per-character transformation settings.

The Per-Character Transformation feature is complicated to use because it requires manually creating animation curves for the Offset and Duration parameters.
These settings are easier to configure if you use the Per-Character Transformation Offset/Duration Animation Simple Settings.

Open the property window and select Text Box Pane > Per-Character Transformation > Offset/Duration Animation Simple Settings.

Per-Character Transformation Offset/Duration Animation Simple Settings Window
A Add

Click Add to add a Simple Settings animation setting.

Each click of the button adds a Simple Settings item.
B Remove

Click Delete to remove a Simple Settings animation setting.

Each click of the button removes a Simple Settings item.
C Simple Settings (1)

The settings screen for (1).

Shows a Simple Settings animation setting that was created by clicking Add.
D Simple Settings (2)

The settings screen for (2).

Shows a Simple Settings animation setting that was created by clicking Add.
Additional clicks of Add will add additional screens (3), (4), and so on.
E Move characters from left ()

Select this option to move the characters starting from the character furthest left in the text box.

Clear this option to move the characters starting from the character furthest right in the text box.
F Automatically calculate end frame ()

Sets the end frame of the animation automatically.

Clear this check box if you want to stop the animation while the last character is still moving.
G Start frame Enter the frame in which the first character starts to move.
H Frame where first character stops moving Enter the frame in which the first character stops moving.
I Frame where last character starts moving Enter the frame in which the last character start to move.
J End frame Use this to set the end frame if it is not being calculated automatically.
