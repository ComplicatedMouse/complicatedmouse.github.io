Overview

This page describes the PC tool for updating the USB firmware.

This tool can be used to update the USB firmware installed in the Joy-Con charging grip and Nintendo Switch Pro Controller.

Updating the USB firmware makes it possible to use a controller charging cable to pair the controllers to the SDEV and EDEV over the cable connection, and to charge the controllers.

Note

The standard controller firmware is not updated with the USB firmware update tool.

Refer to the following documentation for updating this firmware.

Guide to Setting Up the Environment > Preparing the ControllersUpdating Controller Firmware

Necessary Tools

  • Controller charging cable
    This is the controller charging cable bundled with the development hardware.
    Use this cable to connect the Joy-Con charging grip and the Nintendo Switch Pro Controller to the PC.

  • USB Controller Firmware Updater
    A tool distributed with the NintendoSDK NX Add-On for updating the USB firmware. 
Tools/Hid/ChargingGripFirmwareUpdater


Procedure


  1. Installing Drivers

    Move to "Dfu driver" and "Virtual comport driver" under Tools/Hid/ChargingGripFirmwareUpdater/driver.

    Click dpinst_amd64.exe to install each of the drivers.


  2. Connecting Equipment

    Connect the Joy-Con charging grip and Nintendo Switch Pro Controller to the PC using the controller charging cable.

    At this point, make sure to connect only one piece of equipment.


  3. Starting the Updater
    Execute
    Tools/Hid/ChargingGripFirmwareUpdater/UsbFirmwareUpdater.exe

  4. Start firmware update.

    Under the following circumstances, select Target and click Start.

    If Virtual COM Port is displayed.



    If Need Update is displayed.


  5. If the update starts successfully, the following screen is displayed.


    The update takes approximately one minute to complete.

    Note

    Make sure to not disconnect the controller charging cable while updating the firmware.
    In the worst-case scenario, if you disconnect the PC and the cradle while the firmware is updating, the firmware could be damaged beyond the point of recovery.

  6. After the “Finished!!” message is displayed, press the Exit button to close the application.
    　



  7. Disconnect the equipment from the PC. This concludes the firmware update process. Thank you for your cooperation.
