Overview

There will be restrictions when using a Joy-Con in a Windows PC environment because of differences in Bluetooth performance.

Restrictions on Using the Joy-Con in a PC Environment

Note the following differences from a production environment when using the Joy-Con in a PC environment.

Overview Restrictions in a PC Environment Product Specification
Maximum Number of Connected Devices

Bluetooth specifications indicate that the maximum number of all devices that can be connected is seven. Depending on the environment of the PC and the Bluetooth dongles, however, that maximum number can change.
The maximum number of sets of right and left Joy-Con controllers that can be run at the same time is four.

 Connecting eight controllers
Up to four sets of Joy-Con (R) and Joy-Con (L)
Communication Period

Fixed at 15 milliseconds.

 The actual time varies between 5 and 15 milliseconds depending on the number of connections and the features used by the Joy-Con.

Cautions for Use in a PC Environment

Depending on the wireless signal in the PC environment, delays can occur for input data from controllers, such as buttons or control sticks, and for vibration controls. The communication period is set to 15 milliseconds, and the data is sent again if it fails to send and receive in that period. Delays can occur when these re-sent commands stack up.
If the delay is particularly severe, it may be caused by wireless noise from the computer. Try using a USB extension cable to increase the distance between the computer and the Bluetooth adapter.
