Font Pair Properties

Font Pair Properties Window

This section describes the UI of the Font Pair Properties window.

A Font 1 Type The type of Font 1. This cannot be edited in the properties window.
Set when a child node is added to a font pair node from the Tree View.
B Font 2 Type The type of Font 2. This cannot be edited in the properties window.
Set when a child node is added to a font pair node from the Tree View.
C Swap

Swaps the Font 1 Type and Font 2 Type.
