Texture SRT Animation

When using texture SRT animations, you can animate the offset UV, tiling UV, and angle W of coordinates.
No other parameters are supported.

For texture SRT animation, data is exported only for parameters for which animation has been set. Parameters for which data is not exported are shown in the model file. Even if a parameter has a fixed value, if it differs from the value in the model file, make sure that you export a texture SRT animation after hitting at least one key for that parameter.

Info

If the Material Compression export option is enabled, even materials for which texture SRT animation is not set are sometimes animated as a result of united materials.

If this happens, use the NW4F Set Material Attribute Plug-In to change settings so that materials for which a texture SRT animation is set are not compressed.
