Search Conditions

This section describes each preset for search conditions.
In the following table, the assets referenced by effects such as textures and primitives are collectively referred to as files.

Reference File Searches for files referenced by the emitter set beneath the search origin path.
Unused Files Searches for the unused files not referenced by the emitter set beneath the search origin path.
Emitter Set Using Specified File Searches for the emitter set using the selected file.
