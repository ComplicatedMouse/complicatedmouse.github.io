Overview of FontConverter (GUI Version)

The section describes the overview of FontConverter (GUI version).
The font converter creates a bitmap font binary (BFFNT file) that is processed by the font library.

What Can Be Done With FontConverter

  • Generate image files in BMP, TGA, and TIFF formats from Windows fonts.
  • Generate font binaries from BMP, TGA, or TIFF format images.
  • Generate font binaries from Windows fonts.
  • Filter to create font binaries that extract only the necessary characters.

Cautions

Describes cautions when using FontConverter.

Font Licenses

FontConverter does not include any font licenses.
To sell software that uses proprietary fonts, you must sign a license agreement with each font publisher.

Disabled Automatic Font Links

Windows has a feature called automatic font links.
This feature automatically displays a character not included in a font by using a different font.
The automatic font link feature has been disabled in the font converter in consideration of the danger of using an unlicensed font in a product inadvertently.

Internal Unicode Conversion

Encoding conversion is performed based on the Windows conversion tool.
For example, the JIS character code 0x8160 for "～" is defined in Unicode as U+301C, but as U+FF5E in Windows (=FontConverter).
