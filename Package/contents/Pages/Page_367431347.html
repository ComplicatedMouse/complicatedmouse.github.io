Divide Based on Character Width

This page describes the Divide Based on Character Width feature, which divides the curve for per-character transformation based on positions that consider character width.

Overview

The Divide Based on Character Width feature divides the string considering character width, and places the string on the referenced curve.
Enabling this setting creates an aesthetically pleasing curve but also causes more processing load.
The example shows a string consisting of the characters "A" and "I" (which have different widths) placed on the following curve.
