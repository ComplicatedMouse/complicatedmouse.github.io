Basic Settings

This tab is used to configure the overall settings for the emitter.

File Information

Basic Settings is the group of features for configuring the basic behavior of the emitter and the emitted particles.
This group has the following settings.

Filename　 Shows the name of the file.
File path Shows the path to where the file is stored.
Comments A feature for editing comments.
Label Color Sets the label color.

Playback Settings

The Playback Settings group () is a set of features for configuring the playback settings for effects.
This group has the following settings.

Call Start Frame A feature for setting when to call the effect.
Force Calculate Before Start A feature for calculating effects ahead of time so particles are being displayed when the game starts.
Manual Emitter Emission A feature that provides game developers a way to emit emitters at any time.
