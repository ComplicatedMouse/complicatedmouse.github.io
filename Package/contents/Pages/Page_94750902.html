Gravity

This section describes the Gravity feature that applies a force on emitted particles.

Overview

The Gravity feature applies acceleration in a specified direction on emitted particles.
Like real gravity, this feature can apply an acceleration in the downward direction. It can also be used to represent a force being applied in a different direction.
Animations may be set to change the way gravity is applied.
The following overview describes the settings ().

Gravity

Adds the specified value to the velocity of the emitted particles in every frame.

Info

Press the Emitter-Time Animation button () to display the Curve Editor Window and configure an animation for this value.
Animations can be enabled by selecting Run on CPU ().

Related Links For more information, see the following pages.

Emitter Time and Particle Time Describes the two types of time axes used in effect animations.
Curve Editor Window Describes the Curve Editor window, which provides a graphical way to create and edit emitter time animations, with time on the horizontal axis and values on the vertical axis.
Run on CPU This section describes the Run on CPU feature, which allows you to switch between using the CPU and the GPU for calculating particle behavior.

Gravity can be animated as an emitter-time animation.
However, the animation will only work on a CPU emitter because the data cannot be passed to the GPU.

This is for the same reason as with particle time animations for the other field features.

Apply Standard Gravity

This button is enabled if the Apply Standard Gravity option has been selected.
If you click the button, the gravity value will be entered based on the Project Settings so that you can re-create free-falls just like they happen on Earth.

Related Links For more information about the option settings, see Standard Gravity in Project Settings.

Info

The Standard Gravity feature is a way to enter physically correct values for gravity. Previously, the effect designer had to make individual adjustments by eye.
Deciding on a unique, uniform value for gravity also enables consistency among the movements of the various effects in the game.


The Gravity group () includes the following settings.

Apply in world coordinates Determines whether to apply the direction in which gravity is applied to the slope of the emitter.
Direction Applies gravity to the velocity of the emitted particle in the specified direction.
