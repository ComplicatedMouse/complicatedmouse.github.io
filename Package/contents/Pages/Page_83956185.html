Overview

The 3ds Max plug-in is a group of tools for exporting graphics intermediate files from Autodesk 3ds Max.

Notation

This description uses the following simplified notation.

Official Name Simplified Name
Autodesk 3ds Max 3ds Max
Graphics Intermediate Files Intermediate File

Operating Environment

The 3ds Max plug-in has been confirmed to run under the following environments.

3ds Max Autodesk 3ds Max 2018.4 Update
Autodesk 3ds Max 2019.3 Update
Operating
System		 Microsoft Windows 10 Professional 64-bit version

To use all of the features, you must also install the graphics tools for the corresponding platform, and appropriately configure the NINTENDO_SDK_ROOT environment variable.

The plug-in works in both the Japanese and English language versions.

However, the NintendoWare plug-in dialog boxes are displayed in English even for the Japanese-language version.

Installing the 3ds Max Plug-Ins

Follow this procedure to install the 3ds Max plug-ins.

Shut Down 3ds Max

Exit 3ds Max if it is currently running.

Prepare the Plug-In Files

The batch file in the Tools\Graphics\3dsMaxPlugins folder is used for setting up the 3ds Max plug-in.
There are different batch files for different versions of 3ds Max. Make sure that you use the batch file that matches the version of 3ds Max you are using.

3ds Max Version Batch File
3ds Max 2018 SetupPlugins3dsMax-2018.bat
3ds Max 2019 SetupPlugins3dsMax-2019.bat

The following files are installed during set up.

Folder File
(3ds Max root folder)\plugins NintendoExport.dle
(3ds Max root folder)\scripts\startup NintendoExport.ms
NintendoExportUtilities.ms.inc
NintendoExportNodeOptions.ms.inc
NintendoExportMaterialOptions.ms.inc

Testing

Start 3ds Max.

If Nintendo has been added to the 3ds Max menu, the installation was successful.

If the Plug-ins Do Not Start

If an error such as "Failed to initialize DLE file" appears when you start 3ds Max and you cannot use the plug-ins, use Windows Update to install all of the update programs.

Updating the Plug-Ins

Set up using the same installation method as for the most recent plug-ins.

Uninstalling

Exit 3ds Max normally, and then manually delete the installed files.

Then start 3ds Max, and remove Nintendo from the menu by executing NintendoExportRemoveMenu.ms in the Tools\Graphics\3dsMaxPlugins\scripts folder by selecting maxscript > Execute script.

Restrictions on Usable Characters

  • Prohibited characters
    Only single-byte uppercase and lowercase alphabetic characters, numeric characters, and underscores (_) can be used in node names and material names. Use of any other character or symbol is prohibited. If a prohibited character is used, it is converted to an underscore (_) at time of output.
  • Uppercase and lowercase
    Although either uppercase or lowercase characters can be input for notation used in Windows, they are overwritten when output because ABC.fmdb and abc.fmdb are recognized as the same file during internal processing.
    On the other hand, the names ABC and abc (when used for a node name, for example) with 3ds Max are treated as different names. In other words, such names can co-exist.
    There is a possibility that such naming with 3ds Max might collide during data clipping.
    When you export data from Maya to the external operating system as files, make sure that you use names that do not conflict even when uppercase characters are treated as lowercase characters.

Coordinate Systems

Because the coordinate systems differ between 3ds Max and NintendoWare, data is converted and then exported.


 Coordinate System Horizontal Direction Vertical Direction Depth Direction
3ds Max Right-hand system X Z Y
NintendoWare Right-hand system X Y -Z
