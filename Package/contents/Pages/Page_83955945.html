Overview of 3DEditor

Overview of 3DEditor

3DEditor is a tool for editing intermediate files.
It communicates with the viewer library and supports real time editing so you can edit while checking how the edits appear.

3DEditor Workflow

This section presents the general workflow of 3DEditor.


Items Edited Using 3DEditor

The following items can be edited (and in some cases created) using 3DEditor.

  • Model parameters (billboards, user data, and so on)
  • Material features (texture samplers)
  • Shader assignments and parameter settings for materials
  • Textures (user data, and so on)
  • Material animations
  • Bone visibility animations
  • Skeletal animations (user data, and so on)
  • Scene animations

Viewer Library

The viewer library is a runtime library embedded into the game program and used for previewing graphics.
The viewer library communicates with 3DEditor in real time to display the edits that are made.

Embedding the viewer library has the following advantages.

  • You can quickly switch between the game and preview environments without starting the game program or the viewer program.
  • You can adjust the parameters of game-specific rendering processes, such as lighting and full-screen filters, after they have been applied.

Supported Animation Types

3DEditor supports the following animations.

Filename Extension Description
.fskb / .fska Used for skeletal animation intermediate files.
The .fskb filename extension is for binary intermediate files, while .fska is for text intermediate files.
.fmab / .fmaa

Used for material animation intermediate files.
The .fmab filename extension is for binary intermediate files, while .fmaa is for text intermediate files.
.fspb / .fspa Used for shader parameter animation intermediate files.
The .fspb filename extension is for binary files, while .fspa is for text intermediate files.
.fclb / .fcla Used for color animation intermediate files.
The .fclb filename extension is for binary intermediate files, while .fcla is for text intermediate files.
.ftsb / .ftsa Used for texture SRT intermediate files.
The .ftsb filename extension is for binary intermediate files, while .ftsa is for text intermediate files.
.ftpb /.ftpa Used for texture pattern animation intermediate files.
The .ftpb filename extension is for binary intermediate files, while .ftpa is for text intermediate files.
.fvbb / .fvba Used for bone visibility animation intermediate files.
The .fvbb filename extension is for binary intermediate files, while .fvba is for text intermediate files.
.fvmb / .fvma Used for material visibility animation intermediate files.
The .fvmb filename extension is for binary intermediate files, while .fvma is for text intermediate files.
.fshb / .fsha Used for shape animation intermediate files.
The .fshb filename extension is for binary intermediate files, while .fsha is for intermediate text files.
.fsnb / .fsna Used for scene animation intermediate files.
The .fsnb filename extension is for binary intermediate files, while .fsna is for text intermediate files.
