Introduction

Purpose of This Documentation

This documentation describes the nfc library included with the NintendoSDK, and the features for accessing the available NFC tags.

Note

To use this library, you must set up your development environment correctly for using the NFC features.

Note

The nfc library contains similar features to the nfp library. For more information about redundant parts of this documentation, see nfp Library. When referring to this content, read NFP or nfp as NFC or nfc.

Intended Audience

This documentation is intended for people who develop applications using the NintendoSDK nfc library.

Chapter Descriptions

This Manual describes how to use the individual features included in the nfc library.

Related Documentation

Also see the API Reference, the Sample Backgrounder, and nfp Library.

What the nfc Library Can Do

The nfc library supports the following features.

Features Notes
NFC Tag Detection Allows detection of NFC tags. This documentation refers to all detectable IC tags and cards as NFC tags.
Read and Write MIFARE tags Provides dedicated functions for reading and writing MIFARE tags. MIFARE is a contactless IC tag standard used for Taspo cards and the like.
Sending Commands to a NFC Tag Enables sending commands to NFC tags and receiving reply data from NFC tags.

Only one tag can be used at a time. Simultaneous detection of two or more tags is not supported and may result in problematic application behavior.
