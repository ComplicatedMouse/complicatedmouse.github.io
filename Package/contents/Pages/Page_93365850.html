Proportion of Alpha Fade at Edge

This section describes the feature for changing the alpha of particles at the edges of the box used for the Loop-In-Range feature.

Overview

The Proportion of Alpha Fade at Edge feature enables you to set the opacity of particles inside the range of the box.
This feature helps improve the look of effects where particles would otherwise suddenly appear and disappear at the edges of the box.
This section provides an overview of the settings ().

Proportion of Alpha Fade at Edge

Sets the proportion from the edge to the center of the box as a value in the range of 0 to 1, where 1 is the center and 0 is the edge.
The particle's alpha value declines from 1 to 0 between the set position and the edge of the box.
Set the values for the X, Y, and Z axes. The final alpha value is determined by multiplying the alpha value of each axis.
