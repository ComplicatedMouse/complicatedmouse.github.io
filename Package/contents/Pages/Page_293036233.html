Property Window

This section describes the UI for various properties.
The types of property windows are listed below.

Common Properties

Properties window for editing the basic information in a composite font definition file (FCPX).
Font Pair Properties Properties window for editing font pair settings.
Bitmap Font Properties

Properties window for editing bitmap font settings.
Scalable Font Properties Properties window for editing scalable font settings.
Multi-Scalable Font Properties Properties window for editing multi-scalable font settings.
Scalable Font Settings Properties Properties window for editing the settings for scalable font settings.
Font Properties Properties window for editing font settings.
