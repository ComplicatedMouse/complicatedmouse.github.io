Model to Constrain

This section describes the Model to Constrain feature in Model Links, which allows you to select which model you want link from a list.

Overview

The Model to Constrain feature allows you to select, from the models registered in the Model Settings Node, the model you want the effect to be emitted from.
In addition, you can also select the bone from the list of in-game model bones by using the In-game editing feature.
