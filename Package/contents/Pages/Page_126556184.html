Bitmap fonts and scalable fonts are rendered differently.

Question

The same font displays with a slight difference when rendered from a bitmap font and scalable font.

Answer

Kerning information is handled differently in these cases. Depending on the font and the drawing size, there are differences in the string pixel size.
