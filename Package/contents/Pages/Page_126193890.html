Searching for Reference Files

This section describes how to use the search feature of the Effect Browser. By way of example, we describe the procedure for searching for the textures referenced by an emitter set.

Opening the Effect Browser

Click the Effect Browser icon in the toolbar () to open the Effect Browser window.

Selecting the Search Tab

Select the Search tab () in the Navigation pane to display the item for setting the search criteria.

Configuring Search Settings

Configure the search settings using the following procedure.

Setting the Search Origin Path

Use the text box field () to specify the path of the folder that will serve as the basis for the search.
For the purpose of our example, enter the following in the package.

\Samples\Sources\Resources\Vfx\Eset\

Selecting the Search Target Files

Using the radio buttons (), select the type of file you want to find using the search feature.
This example searches for textures, so select the Texture option ().

Setting the Search Criteria

Select from among the presets for the search criteria.
Select Referenced Texture.
With this condition, the feature searches for texture files referenced by the emitter set file beneath the search origin path.

Related LinkFor more information about search conditions, see here.

Running the Search

To start a search, click the Search button.
Confirm that texture files are shown in the File List of search hits.

Setting the Search Range

Select Filter to results in the search origin path to run the search again.
When this is done, the range of files shown in the File List is narrowed down to those files below the search-origin path.
