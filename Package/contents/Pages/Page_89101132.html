Features

This section describes the features provided by the AUDIO library and how to use the API resources.

Features

Feature Overview
Audio Output This module is for outputting sound to an audio output device.
Audio Input This module is for getting sound from an audio input device.
Audio Renderer This module is for audio signal processing and output.
Other Other features, including a feature for setting the devices to which the audio will be output.
