Alpha Settings

This section describes the Alpha 0 Behavior Type and Alpha 1 Behavior Type features that set the Alpha 0 and Alpha 1 alpha values, which are used in the Color Combiner calculation formula.

Overview

The Alpha 0 Behavior Type is a feature that can set the alpha value animation and the Alpha 0 alpha value, which are used in the Color Combiner calculation formula.
The Alpha 1 Behavior Type is a feature that can set the alpha value animation and the Alpha 1 alpha value, which are used in the Color Combiner calculation formula.
The following overview describes the settings ().

Alpha 0 Behavior Type, Alpha 1 Behavior Type

Select among the following presets for the behavior type to use for Alpha 0 and Alpha 1 in the Color Combiner calculation formula.

Fixed Sets the color when is there no animation using the Alpha Editing UI.
8-Key Animation Creates animation using the Color Animation UI, which can register up to eight key frames.
