Tree View

This section describes the Tree View UI.

A Add button

Add child nodes to a selected node.
The type of node that can be added differs depending on the type of node that is selected.
Bitmap Font nodes and Scalable Font Description nodes can be added when a node has a Font node as a child node.
There are also node types to which child nodes cannot be added.
B Delete button Deletes the selected node and all child nodes.
Font nodes cannot be deleted. When you want to delete nodes, delete the parent node.
C Node Tree Allows you to check the parent-child hierarchy of a node.
If an invalid parameter is set for a property, the node name is displayed in red.
