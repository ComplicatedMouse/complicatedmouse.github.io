Features

This section describes the features of 3DEditor.

Project

Only one project exists at a time when 3DEditor is running, and it is used to manage 3DEditor tasks.
Animation Sets

An animation set is a collection of animations.
User Settings

The user settings file can be edited by to customize 3DEditor operations.
Team Settings

The operations of 3DEditor can be partially modified by placing various settings files in a team settings folder whose path is specified in an environment variable.
Additional Reference Path

Additional reference paths are specified for the file I/O settings file (FileIo.config) set in the team settings folder and each model, and are used when searching for reference files.
Startup Path

Configures a startup path so 3DEditor opens certain specified files when it starts.
Attach Path

The attach path is used to search for reference files during runtime collaboration model attach and shader attach.
Parent Material Reference Path The parent material reference path is used to search for models containing the material shown in the Add/Edit Material dialog boxes.
Material referencing

References to external materials can be configured in the 3DEditor UI.

Command Line

You can change the startup behavior of 3DEditor using command-line options when starting 3DEditor from a command line.
3DEditorBridge 3DEditorBridge is a command-line tool that connects 3DEditor with external tools.
