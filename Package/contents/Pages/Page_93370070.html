Primitive emitter shapes.

Describes the settings for primitive emitter shapes for model vertices created using a DCC plug-in for the particle emission locations.

Overview

With the Primitive emitter shape, freely shaped primitive model files created with the DCC tool can be loaded into EffectMaker and used as emitters.
Particles are emitted from each vertex of the loaded primitive model placed in the world coordinate system in the normal direction.
The following overview describes the settings ().

When the emitter shape is set to Primitive, the Primitive () settings are displayed. The group has the following settings.

Specify File

You can load a model file. The following are button types.

Load Opens the file selection window to select a model file.
Open Sample Primitives Opens the Primitive folder. You can put frequently used model files in this folder.
The Primitive folder is set to the following location.
Folder_where_EffectMaker_starts\Primitives
Reload Updates the specified model file.
Clear Releases the specified model file.

When you load a primitive, the following information about that primitive is shown in comments.
Use this information to check the load of the primitive and information about its attributes.

  • Number of vertices
  • Included attributes (pos (vertex positions) / normal / tangent / color (vertex color) / uv (texture coordinates))

Emission Type

Selects the behavior type when a particle is emitted in the emitter normal direction.

Together Particles are emitted from all primitive vertices in the direction of the normal.
Random Particles are emitted from a random location in the primitive vertices in the direction of the normal.
By Index Particles are emitted from primitive vertices in the direction of the normal in the order in which the vertices are drawn.
