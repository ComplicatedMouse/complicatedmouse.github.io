Using the Copy and Paste Feature

This section describes the copy and paste feature.

Copying Panes

Use the following procedure to perform copy and paste operations.

1 Select the pane using an operation such as clicking with the mouse.
2 From the main menu, select Edit > Copy, or press CTRL+C to copy.
3 From the main menu, select Edit > Paste, or press CTRL+V to paste.
4 A pane with the same parameters is duplicated at the same location as the copied pane.

The following parameters are not copied from the source pane when duplicating (copying and pasting) panes.

  • Animations (curves, keys, and so on)
    To copy animation information, use the Copy and Paste Controls window.
  • Groups

Using the Pane Information Paste Controls Window

You can paste only part of the copied pane information in the Pane Information Paste Controls window.
It can be used in copy and paste situations such as the following.

  • Copy and overwrite only some of the pane information to existing panes.
  • Perform paste processing together for multiple panes.
  • Paste animation data.

Overwriting Part of Pane Information by Pasting

This section describes the procedure for pasting only the Rotation information of the Heart pane to other panes in a layout like the following.

1 Select the Heart pane copy source, and copy it using the standard method.
2 Select the range of Spade and Diamond panes.
3 From the main menu, select Window > Pane Information Paste Controls to open the Pane Information Paste Controls window.
4 Verify that the name of the pane copied in step 1 is displayed at the top of the Pane Information Paste Controls window.
5 Clear the Check All check box at the bottom of the window.
Clear the check box for all items other than Overwrite Key Information at Paste Target.
6

Select the check boxes for the following item.

  • Basic Pane Information: Rotation
  • Vector Components paste control: X Y Z
7 Click the Paste to the Selected Pane button.

In this procedure, only Rotation information is pasted to other panes.


Tips Using pane information paste controls, you can control the paste parameters.

  • As follows, you can paste only the Size X information from the Heart pane to another pane.

Perform the same operations as for Overwriting Part of Pane Information by Pasting.
In step 6, check only the following items.


  • Basic Pane Information: Size
  • Vector Components paste control: X
 Only Size X can be pasted.


  • You can paste animation keys in the Pane Information Paste Controls window.
    If you paste while the Overwrite Key Information at Paste Target check box is selected, all of the keyframes set for the copy target pane are deleted prior to pasting.
    The information of the paste destination is completely replaced by information from the paste source.
