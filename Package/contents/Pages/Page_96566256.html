Inheritance Settings: Rotation

This section describes the Rotation inheritance setting for child emitters.

Overview

The Rotation inheritance setting is a feature that makes child emitters inherit the rotation value of the parent emitter in the highest node.
This section provides an overview of the settings ().

Rotate

The feature to inherit the particle rotation angle from the parent emitter is enabled or disabled using the check box ().
When this feature is selected, the rotation angle of the parent emitter is inherited as the initial value when particles are emitted and is multiplied by the rotation angle.

Note

For child emitters to use this feature, make sure that you select the inheritance check box () for each axis of rotation.
