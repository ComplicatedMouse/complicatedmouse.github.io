Image Settings

Image Settings

Image settings determine how textures appear in panes in LayoutEditor.

Repeat Method

Configure the repeat method for picture data.

U Direction

Sets the repeat method to the u direction.

V Direction

Sets the repeat method to the v direction.

Select the repeat method for both the u direction and the v direction from among the following presets.

Clamp
 This setting stretches the top, bottom, left, and right edges of the picture data.
Repeat
 This setting repeats the picture data.
Mirror

This setting repeats the picture data while flipping the data.

Filtering

Select the filtering for the texture from the following presets.

When Expanding

Configures the filtering when expanding the texture.

When Shrinking

Configures filtering when shrinking the texture.

Select the filtering for both expanding and shrinking the texture from among the following presets.

Near ()
 Displays each pixel distinctly.
Linear ()
 Interpolates to smooth out pixel edges.

Mapping Method

Use the drop-down list to configure the texture mapping method.

UV Mapping (Using UV Coordinates 0–2)

Configured using UV mapping.

Orthogonal Projection Mapping

Configured using orthogonal projection mapping.

Projection mapping relative to pane position

Configured using project mapping relative to pane position.
Perspective projection Configured using projection mapping. When the layout camera is in perspective view, the texture image to project is rendered in screen space.
Perspective projection mapping relative to pane position Configured using project mapping relative to pane position.

SRT Conversion Matrix

Enter information about SRT conversion matrices.

U Scale:

Sets the scale value for the U direction.

V Scale:

Sets the scale value for the V direction.

Rotation:

Rotation values.
U Position: Sets the translation value for the U direction.
V Position: Sets the translation value for the V direction.


Note

Repeating the display of large textures with large values set for parameters like the UV scale puts a heavy load on the GPU.

UV Coordinates

Configure the UV coordinates. Select UV Mapping for the mapping method.

The following table shows frequently used UV coordinate presets.

The following buttons are used to edit UV coordinates.

Vertically flips the texture coordinates.

Rotates the texture coordinates 90 degrees counterclockwise.

Select the UV Coordinate Advanced Settings check box () to display the UI.
As shown in the following table, you can edit the vertices of the four UV corners of the rectangle using coordinates and pixel values.
The setting is unavailable when multiple textures of differing sizes are registered to one texture coordinate.

Vertex UV Coordinates Coordinates in Vertex Pixel Values
Sets the UV coordinates of the texture vertices for the entire pane. Sets each vertex coordinate of textures for the entire pane using pixel values.
The default value changes depending on the texture size.
Note

When the property window is in the inactive state and texture coordinate control in the Texture properties is clicked, the title bar of the property window sometimes disappears.
To restore the display to the correct state, click somewhere outside of the property window to move the focus.

Translation

Brick Repeat

Configures the Brick Repeat feature. Select and display Brick Repeat for the mapping method.

Repeat Count:

Sets the number of repetitions of a texture in the vertical and horizontal directions of a pane.

Offset Shift:

Starting from the element at the center of the pane, set the amount of UV that is shifted each time the repeat count is increased or decreased by one.
Local Conversion Matrix Sets the scale and rotation applied to the UVs in each repeated cell.
Scale Sets the scale value applied to the UVs in each repeated cell.

Random Offset

Use the check box to set whether to apply random offset to the scale of the local transform matrix.
When turned on, a random offset is applied to the scale in each element according to the UV.
Range Sets the upper and lower limits of the random offset value applied to the scale.

Rotation:

 Sets the rotation applied to the UV in each repeated cell.

Random Offset

Use the check box to set whether to apply random offset to the rotation of the local transform matrix.
When turned on, a random offset is applied to the rotation in each element according to the UV.
Range Sets the upper and lower limits of the random offset value applied to the angle.

Random Direction of Rotation

Use the check box to set whether to randomize the direction of rotation.
When turned on, the direction of rotation of each repeated cell is randomized.
