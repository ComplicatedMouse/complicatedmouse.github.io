Overview

SystemUpdateHdev is a command-line tool for updating the HDEV development hardware firmware with a process equivalent to the system update feature.

Currently, this is the same tool as SystemUpdateEdev . For more information, see that page.

Package

Included in the DevKitVersionUpdater for NX package.

Location

$NINTENDO_SDK_ROOT/Tools/CommandLineTools/SystemUpdateHdev.exe
