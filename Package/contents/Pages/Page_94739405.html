Random Angle

This section describes the Random Angle feature for emitting particles from random locations.

Overview

The Random Angle feature emits particles from random locations along circular arcs, defined by the divisions of the emitter shape.
When an emitter shape is divided into equal parts, those parts can serve as the central positions of randomly located particle emissions.
The following overview describes the settings ().

Random Angle

You can vary the locations where generated particles are emitted by using Random Angle to specify a range across which random numbers are applied, based on the positions of the divisions in the emitter shape.
For example, if you set Random Angle to 90 degrees, the location of particle emission changes within a range of 180 degrees around the specified value.

Example Configurations

Random angle

The following example settings and image show what happens when the emitter shape is Circle (Evenly Divided), the number of divisions is set to 4, and Random Angle is set to 45 degrees.

B Random angle An arc spanning 90 degrees, or 45 degrees on either side of one place where the shape is equally divided into four parts.
C Location of Particle Generation

Particles are generated at random locations along arc B.
D Direction of particle movement Particles move in the direction of the normal of the Circle emitter shape.
