Plug-In for Setting Material Attributes

This plug-in sets the attributes for materials that are used by NintendoWare in a shading group.
These attributes, however, are not applied on the Maya screen.

You can also search for the settings.

How to Use

Select Nintendo > Set Material Attribute to open the window.

Selected materials and shading groups are displayed under Selected Material.

Info

If four or more nodes are selected, names are not displayed and only the number of nodes selected is displayed on the right.

Setting Method

When one or more material nodes (such as a lambert node) or shading groups (shadingEngine node) are selected, the current setting for the selected node is displayed under Set.
If you select multiple nodes, the setting for the first selected node is displayed.

Attributes are changed by controlling each item in the window.
If multiple selections were made, selected nodes can be changed at the same time. (Only modified attributes will be applied.)

You can configure the setting for a different node by keeping the window open and selecting a different node.

Compress

Sets a flag that prevents compression even when the Compress Material export plug-in option is selected.

  • Select Compress if possible to use compression if other materials have the same attributes. This setting is the default.
  • Select Don’t compress to export the material as an independent material without compression, even if the attributes are the same as other materials.

Searching for Attributes

To select the corresponding shading group, materials connected to the shading group, and polygons belonging to the shading group, select the appropriate check boxes under Search, and then click Search.

Selecting Rendering > Show Shading Groups under Hypergraph makes the results easy to see.
