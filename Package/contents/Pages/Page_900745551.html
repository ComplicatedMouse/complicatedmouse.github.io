Introduction

Purpose of This Documentation

This document describes the GoogleTest library included in NintendoSDK and how to use it.

About GoogleTest

GoogleTest is an individual C++ programming language test library based on the xUnit architecture.

Intended Audience

This document is intended for developers of applications and middleware using NintendoSDK.

How GoogleTest Is Provided

Both PC and NX versions of the GoogleTest header files, source files, and projects used to build the GoogleTest library are provided. When using GoogleTest, use the library built and generated by the project.

Source Files

NintendoSDK provides the GoogleTest source files.

They are located in the following directory.

$NINTENDO_SDK_ROOT/Externals/googletest

Header Files

NintendoSDK provides the GoogleTest header files.

They are located in the following directory.

$NINTENDO_SDK_ROOT/Externals/googletest/googletest/gtest

Projects for Building the GoogleTest Library

NintendoSDK provides the projects for building the GoogleTest library.

They are located in the following directory.

$NINTENDO_SDK_ROOT/Samples/Sources/Libraries/gtest

Built Library

The built library is saved to the following directory.
Create and use a link to the generated libgtest_sample library.

$NINTENDO_SDK_ROOT/Samples/Libraries/

GoogleTest Included in NintendoSDK

NintendoSDK includes GoogleTest v1.10.0. There are no changes to the source code.
