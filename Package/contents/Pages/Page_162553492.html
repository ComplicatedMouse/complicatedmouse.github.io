Audio Performance Information

nn::audio Performance

The nn::spy::audio library collects performance information from the nn::audio library and sends it to Spy.exe.

This performance information provides insight on the processing loads on each of the following components of the audio library.
For more information about each audio component, see the page that describes the audio renderer.

Component Description of Content Displayed in Spy.exe
Voice

Indicates the processing load of the Voice that is currently being played back.

Indicates the numbers of voices that are played and voices that are dropped.
SubMix

Indicates the SubMix processing load.

It includes the processing load for effects that are associated with SubMix.
FinalMix

Indicates the FinalMix processing load.

It includes the processing load for effects that are associated with FinalMix.
Sink

Indicates the Sink processing load.

Use the nn::audio::PerformanceInfo::IsRenderingTimeLimitExceeded() function to get information about whether the rendering time has exceeded the maximum, and display this information as IsOverTime in Spy.exe.

  • If the rendering time exceeds the maximum, noise may occur in the rendering results due to processing slowdowns.

nn::atk Performance

The nn::spy::atk library collects performance information from the nn::atk library and sends it to Spy.exe.

Internal nn::atk thread processing loads are displayed in addition to the nn::audio performance information.
For a description of each thread, see the Atk thread documentation.

Threads Item Description
Sound thread NwFrame Displays the processing loads associated with calculating parameters and updating the internal state of the nn::atk library.
Task thread bfstm load Displays the start and end times for loading BFSTM files.
Task thread opus load Displays the start and end times for loading Opus files.
Task thread opus decode Displays the start and end times for Opus decoding processes.
