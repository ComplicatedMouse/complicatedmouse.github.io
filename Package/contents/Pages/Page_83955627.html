Shader Definitions Property Window: Source Code Page

This menu bar is used to search for source code that contains a particular string.

A Enter the string to search for.
B Search for previous.
C Search for next.
D Search all sources.
E Search without case distinction.
F Search using regular expression.

This window is used to check and edit the selected source code (GLSL file).

A Displays the include paths in the shader source code.
B The relative path from the include paths in the shader source code.
C Used to edit the shader source code.
Clicking Edit opens an external text editor, allowing you to edit the shader source code.
D Displays whether to force-include the shader source code.
E Displays the shader source code for checking and editing.

Editing Shader Source Code

Shader source code can be edited by connecting 3DEditor to the runtime.

Use the following procedure to edit shader source code.

  1. Load the model and shader definition file in 3DEditor.
  2. Specify a shader definition file for the model materials.
    Related Links The shader definition file settings are configured on the Shader page of the material properties window.
  3. Connect to the runtime.
  4. Related Links From the File View, open the shader definition properties window .
    To open the shader definition properties window, double-click a shader definition file (FSDB file), or from the main menu, select View > Property Window.
  5. Related Links From the list of GLSL files shown in the source code of the shader definition properties window, select the shader source to edit.
  6. The shader source (GLSL file) selected in step 5, can be edited with an external editor by selecting Source Code > Edit.
    You can select any external text editor by defining it in 3dEditor.user.config.
  7. Edits are applied when you edit and save the shader source using an external editor.
Info

The edits are not applied in the original shader definition file that is loaded in 3DEditor.
You must save the edited shader definition file separately if you want to keep it. (Save by copying the shader source code or by saving with a different name with the external editor.)

Info

The shader source code edit can also be attached to a runtime shader.
Related Links To execute an attachment, you must implement QueueAttachShaderArchiveCommand() on the runtime side.
