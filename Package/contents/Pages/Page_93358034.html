Custom Render Settings

This section describes the feature for customized configuration of the depth and alpha tests when rendering particles.

Overview

To display the Depth Test and Alpha Test items, select Custom in the Z Buffer and Alpha Test presets.

Depth test Feature to test the depth value of the particle being drawn.
Alpha Test Feature to test the alpha value of the particle being drawn.
