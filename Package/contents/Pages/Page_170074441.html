DebugPad (NX Add-On)

Installing NX Add-On enables the DebugPad mechanism to be used in the environments described below.

Structure in the SDEV Environment

Classic Controller Pro

The input state is determined from the Classic Controller PRO state.

Input Sent by Input Director

The input state is determined by the state of the XInput-compatible gamepad as sent from the host PC by Input Director.

Structure in the EDEV Environment

Input Sent by Input Director

The input state is determined by the state of the XInput-compatible gamepad as sent from the host PC by Input Director.
