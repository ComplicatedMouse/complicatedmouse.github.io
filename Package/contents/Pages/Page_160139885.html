Creating Headwear

This section describes how to create your own headwear to use in an application.

Overview

The following decorative items can be attached to a Mii head.

  • Accessory
  • Hats
  • Headwear

The content describes how to create accessories, hats, headwear, and is intended mainly for designers. Detailed descriptions of the implementation procedures used by developers are not included.

But developers should first read this section about how to create Mii character headwear to get an understating of the process flow. After reading this manual, see the sample demos, API Reference, and implement the headwear.

The following sample demos for attaching decorative items to Mii heads are available.

$(NINTENDO_SDK_ROOT)/Samples/Sources/Applications/MiiHeadwearExample

Sample Models

Sample model data has been prepared in a format for Maya (MB files) and in a format for other 3DCG software (FBX files).

When Maya data is exported to binary data, the scale is increased by a factor of 10. If you create models or Mii headwear in the same workspace (scale) as the samples, create them according to the output scale.

Sample models are in the following location.

$(NINTENDO_SDK_ROOT)/Resources/Mii/headwearsample

A partial folder structure follows.

headwearsample/
    +-- scenes/                   // Maya data.
        +-- Cap_sample.mb         // 3D models for hat reference, created based on Camp_template.mb (two types).
        +-- Cap_template.mb       // Template for making hats (two types).
         +-- Faceline_sample.mb    // Mii face outline model (12 types). Use these samples to create headwear.
         +-- Hair_sample.mb        // Hair model where you can see the null positions for accessories (1 type).
         +-- HairAcce_sample.mb    // 3D model for accessory reference (three types).
         +-- Headgear_sample.mb    // 3D model for headwear reference (three types) and 3D model with example of limited nose height.
    +-- Fbx/                      // 3D CG software data other than Maya.
        +-- Cap_sample.fbx        // 3D model for hat reference (two types), created based on Cap_template.fbx.
        +-- Cap_template.fbx      // Template for making hats (two types).
        +-- Faceline_sample.fbx   // Mii face outline model (12 types). Use these samples to create headwear.
        +-- Hair_sample.fbx       // Hair model where you can see the null positions for accessories (one type).
        +-- HairAcce_sample.fbx   // 3D models for accessory reference (three types).
        +-- Headgear_sample.fbx   // 3D model for headwear reference (three types) and 3D model with example of limited nose height.
Note

Secondary use of these sample demos is prohibited. Do not use them for anything other than creating Mii head coverings.
