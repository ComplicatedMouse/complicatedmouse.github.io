Time Management

Time Management by Spy

Spy manages time in two ways: in units of real time, and in units of application frames.

The time information received from the Spy library can be displayed by Spy.exe in terms of either real time or in terms of application frames.

Time Handling by the Spy Library

When data is sent from the Spy library to Spy.exe, it is basically sent as a set together with the time (in microseconds) of transmission.

When application frames are used as the units of time, an application frame value is set in the Spy library for each frame from the application.
When the Spy library receives a change in application frame, it associates the application frame with a time in microseconds and sends that to Spy.exe.

Based on this information, Spy.exe determines the application frame for which to show data.
