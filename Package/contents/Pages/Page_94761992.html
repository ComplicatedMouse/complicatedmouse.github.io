Alpha Combiner

This section describes the feature for combining the emitter's alpha elements.

Overview

The Alpha Combiner is a feature for combining the emitter's alpha elements.
The following table shows the alpha elements handled by the Alpha Combiner and their names in the formulas.

The alpha component of the texture. The alpha information for the textures to register on the Texture 0, Texture 1, and Texture 2 tabs.
Use from Texture 0 to Texture 2.
The alpha component of the particle. The alpha information to set on the Alpha tab.
Use for Formula.
The vertex alpha elements of the primitive. The vertex alpha information in the primitive file registered on the Particle tab.
Use for Primitive.

Alpha Combiner

The following table summarizes the Alpha Combiner settings.
Combining is a process that runs from earlier stages to later stages, in the order shown in the UI.

Texture 0 Channel
The value to input for the combiner formula is determined by running the color component of the texture through a predetermined equation.
Related Links For more information about the preset formula, see Alpha Combiner Channel.
Texture 1
 Operators Select which operator to use for combining with the previous stage (× (multiply), ＋ (add), or - (subtract)).
Channel Uses Texture 1's alpha component as the alpha channel.
Texture 2
 Operators Select which operator to use for combining with the previous stage (× (multiply), ＋ (add), or - (subtract)).
Channel Uses Texture 2's alpha component as the alpha channel.
Primitives
 Operators Select which operator to use for combining with the previous stage (× (multiply), ＋ (add), or - (subtract)).
Channel Uses the Vertex Color's alpha component as the alpha channel.
Formula Combines the alpha values of A0 and A1 set on the Alpha tab with the alpha value of the previous stage (Comb_A).
Related Links For more information about the equations for combining alpha values, see Alpha Combiner Formula.
