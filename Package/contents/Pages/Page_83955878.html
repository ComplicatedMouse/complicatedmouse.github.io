Basic Settings

This tab is used to configure the overall settings for the emitter.

Basic Settings

Basic Settings is the group of features for configuring the basic behavior of the emitter and the emitted particles.

Particle Follow Type This section describes the setting for how particles follow the transform animation of the emitter.
Run on CPU This section describes the feature for switching between the CPU and GPU for particle behavior calculations.
Starting Process Describes the Starting Process feature for determining the processing when the effect first starts to play.
Termination Processing Describes the Ending Process feature for determining the processing when a playing effect is forcibly stopped.

Render settings

Render Settings is the group of features for configuring settings related to the rendering of emitted particles.
This group has the following settings.

Z Buffer and Alpha Test This section describes the Z buffer and alpha test features for testing depth and alpha when rendering.
Rendering Particles This section describes the feature for determining whether particles will be rendered.
Display Face This section describes the feature for determining which faces of the polygon to show particles on.
Particle Sort This section describes the feature for determining the order in which emitted particles are sorted.
Draw Path This section describes the feature for configuring the draw path that is specified in the game application.

Random Number Setting

Random Number () is the group of features for controlling the random numbers used for particle behavior.
This group has the following settings.

Random number type This section describes the feature for controlling the random numbers used for particle behavior.

Inheritance Settings

Inheritance Settings () is the group of features for configuring the settings that child emitters inherit from their parent emitter.

Related Links For more information about child emitters, see Child Emitters.
This group has the following settings.

Emitter Assignment for Particles Enables or disables the option to make particles emitted from the parent emitter become child emitters.
Draw Path Sets whether to consider the parent emitter rendering path during rendering.
Speed Sets whether to multiply the speed of the parent particle to its own speed.
Color Scaling Sets whether to multiply the color scale of the parent particle to its own color scale.
Color Sets whether to multiply the color of the parent particle to its own color 0 and color 1.
Alpha Sets whether to multiply the alpha of the parent particle to its own alpha 0 and alpha 1.
Scale Sets whether to multiply the scale value of the parent particle to its own scale value.
Rotate Sets whether to multiply the rotation value of the parent particle to its own rotation value.
