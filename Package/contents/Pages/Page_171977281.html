Threshold-Based Alpha Blending

Switches to entering settings for threshold-based alpha blending.

Threshold-Based Alpha Blending ()

When cleared (default)

The values entered for black color blending and white color blending affect the color of the texture.


  • The alpha set for black color blending affects the opacity of the transparent parts of the texture.
  • The alpha set for white color blending affects the opacity of the opaque parts of the texture.
Threshold-Based Alpha Blending ()

When selected

The alpha values set in black color blending and white color blending blend the alpha values as threshold values.


  • The alpha set for black color blending affects the opacity of the transparent parts of the texture.
    The number set here is taken as the threshold value, and all texture alpha values below this threshold are treated as transparent.
  • The alpha set for white color blending affects the opacity of the opaque parts of the texture.
    The number set here is taken as the threshold value, and all texture alpha values below this threshold are treated as opaque.
Texture borders are less blurry when Threshold-Based Alpha Blending is set.
Info
  • This feature can be used as a high-performance alpha test (with detailed control and smooth borders).
  • You can reduce blurring around the borders of a texture that has been scaled by applying an alpha channel that has been converted to a distance field to the texture.
    Related Links The alpha channel can be converted to a distance field using the NintendoWare Photoshop plug-in Distance Field Filter plug-in.
  • This feature cannot be used in tandem with Advanced Combiner Mode.
