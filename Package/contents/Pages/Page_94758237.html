The following videos show how to operate the color animation setting UI.

  • To add an animation key (), click the gradient bar ().
  • Click and hold the animation key () and move the mouse pointer left and right.
    Use the overall length of the gradient bar () as the particle lifespan to set the key timing (time).
  • Set the key color by clicking the added animation key ().
  • To delete the animation key, click the key delete button () on the animation key ().
Info

For particle time animations, the time input values are specified as a percentage of the particle lifetime.
If you want to operate the key directly, the actual frame count is displayed in the key frame display ().
