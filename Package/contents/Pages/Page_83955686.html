Rotate

This tab is for configuring settings for rotating particles.

Rotation Settings

Rotation Settings () is a collection of features that set axes for particle rotation and configure settings that affect rotation settings overall.

Rotation Axis

Feature that individually selects the x, y, and z axes that particles rotate around.

Rotational order

Functionality to configure the processing order when using two or more of the x, y, and z rotational axes.
Rate of Deceleration

Feature that increases or decreases the rotation speed of emitted particles.

Rotation Axis

Axial rotation () is a collection of features that set the animation for particle scale using the Curve Editor.
This group has the following settings. The settings are the same for the x-axis, y-axis, and z-axis.

Initial Rotation (±)

Feature that sets the tilt angle for particle emission.
Random spin direction Feature that randomizes the rotation direction for particle emission.
Rotation Speed (±) Feature that creates a rotation animation, adding each frame value to the rotation direction.
