Fixing the Docking Windows

The Fix Docking feature locks or fixes the state of docking windows to prevent operations that would unintentionally move them.

From the menu bar in the LayoutEditor window, select Window > Fix Docking.

Fix Docking

When this menu item is selected, docking windows can be fixed.

By fixing the docking window, you can eliminate the risk of accidentally moving the window while working.
To remove the lock, select Fix Docking again to clear the selection.
