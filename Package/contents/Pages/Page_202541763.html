Layout, Font News

This document is a graphical introduction to the features that were added in NintendoSDK 1.2.0.

LayoutEditor

This section introduces the new features added to LayoutEditor.

Changed the default to requiring the specification of target groups when using animation segment tags. Specifying the target group when using animation segment tags is now required by default.
Made it possible to perform group settings for animation segment tags using batch selection. You can now set related groups for multiple animation segment tags at the same time when multiple tags are selected.
Added automatically linking the aspect ratio to the font size scale input. A button for automatically linking the aspect ratio has been added to the Size Scale.
Made changes so that decimal values are not included when dragging a pane. Decimal coordinate values are no longer included when dragging a pane to adjust its position.


Changed the default to requiring the specification of target groups when using animation segment tags.

Specifying the target group when using animation segment tags is now required by default.

It is now required by default because animation segment tags are used with groups specified in the majority of cases, and there have been many cases where the user forgets to specify groups.
(It is possible to handle target group specifications in the same manner as before by using the predefined specifications.)

・A warning icon is displayed on the animation segment tag window if a target group is not specified.

・An error notification is sent when the tags are converted to binary, such as when transmitting to the viewer.

・It is now possible to control error checking in the LayoutEditor settings file.

The warning icon displayed in the Edit Animation Segment Tags window
Example of the error notification when sending to the viewer

Reverting to Previous Behavior

It is possible to handle target group specifications in the same manner as before by using the predefined specifications.

To handle the specifications in the same manner as before, specify the AllowNoGroupAnimationTag element in the project settings file.

Made it possible to perform group settings for animation segment tags using batch selection.

You can now set related groups for multiple animation segment tags at the same time when multiple tags are selected.

Previously, if there were multiple animation segment tags, the related groups had to be set individually even if the settings were the same.
From now on, you can set all the related groups at the same time when multiple animation segment tags are selected.

Added automatically linking the aspect ratio to the font size scale input.

A button for automatically linking aspect ratios has been added to the Size Scale options in Properties Window > Text Pane > Text Settings.

The button was added next to the location for entering the size scale values.
It is the Link Aspect Ratio button for the size scale.
The button switches between Link and Unlink for the size scale.

Link Aspect Ratio button added to Properties Window > Text Settings > Size Scale
Link Aspect Ratio button is Linked The aspect ratio ratio size scaling values will be locked for both the x and y directions.
The same value will be locked and used for the other coordinate when you enter a size scaling value for either the x or y direction.
Link Aspect Ratio button is Unlinked The aspect ratio ratio size scaling values will be unlocked for the x and y directions.
Separate values can be entered for size scaling in both the x and y directions.

Made changes so that decimal values are not included when dragging a pane.

Decimal coordinate values are no longer included when dragging the pane to adjust the position.

Previously, decimal values were included in the coordinates when dragging the pane to adjust its position. (If not using snap to grid)
Work can be more efficient without including the decimal coordinates.
The operation for dragging the pane to register the coordinates in the Curve Editor will be smoother.
