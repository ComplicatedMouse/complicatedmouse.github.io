Manual Emitter Emission Feature Limitations

This section describes features that cannot be used and features that we do not recommend using with Manual Emitter Emission.

Features That Cannot Be Used

The following features cannot be used with Manual Emitter Emission.

Node Description
Emitter

The following emission-related features cannot be used.

  • One Time
  • Emission Start Frame
  • All features for the Distance emission interval type.
  • Emission interval types Time and Emission Interval.
Child Emitter Manual Emitter Emission cannot be used with emitter sets that contain child emitters.
Preview

The following features are not available in NintendoWare for CTR 1.2.x.

  • Force Calculate Before Start
  • Emission Interval

Features We Do Not Recommend Using

We do not recommend using the following features with Manual Emitter Emission.

Node Description
Emitter

None of the following emitter-time animations will work properly.

  • Emitter Shape / Scale
  • Emitter Transform / Position
  • Emitter Transform / Rotation
  • Emitter Transform / Scale
  • Emission Settings / Emission Rate
  • Initial Velocity Setting / All-Direction Initial Velocity
  • Initial Velocity Setting / Specified-Direction Initial Velocity
  • Other Motion Settings / Gravity
  • Particle Lifespan / Lifespan
  • Global Multiplier Color and Alpha
  • Basic Scale / Standard value
Historical Stripe
Historical Stripe2		 These particles will not work correctly if the particle lifespan is shorter than the manual emitter emission interval.
If you plan to use historical stripes, set the particle lifespan to be longer than the emission interval.
