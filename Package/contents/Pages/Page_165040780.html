Emission Interval Type - Time: Emission Interval

Emission interval

This section describes the Emission Interval feature, which specifies the interval at which particles are emitted as specified in terms of time (frames).

Overview

The emission interval feature can be used to configure the interval at which particles are emitted.
The following overview describes the settings ().

Emission interval

Sets the time interval (number of frames) at which particles are emitted.

Random Emission Intervals

Randomly changes the particle emission range.
The emission range changes randomly according to random number upper and lower limits. The following calculation method is used when seeking a random number upper limit.

  • Random Number Upper Limit: Calculated as Emission Interval plus Emission Rate Randomness.
