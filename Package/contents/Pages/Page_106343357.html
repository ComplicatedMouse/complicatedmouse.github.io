UI Descriptions

This section provides overviews of the various elements of the user interface for the editor.

 LayoutEditor Window Describes the names and uses of the windows that open when LayoutEditor starts.
Layout Window Describes the names and roles of the various parts that comprise the main window of LayoutEditor.
Property Window Describes the Property window, used to configure the settings for the various nodes of LayoutEditor.
Curve Editor Window Describes the Curve Editor window, used to create and edit animations for LayoutEditor.
Texture Window Describes the Texture window that is used to register LayoutEditor textures and select the format.

Load Sequential Textures Window

Describes the Load Sequential Textures window that is used to automatically create animations from sequential textures.
Group Window Describes the Group window, used to group data according to the type of layout.
Hierarchy Window Describes the Hierarchy window, used to build layout structures.
Pane Information Paste Controls Describes the Pane Information Pasting Control window, used to control items to copy between multiple panes.
Edit Animation Segment Tags Window Describes the Edit Animation Segment Tags window, used to specify the animation range.
Search for Panes Window Describes the Search for Panes window, used to search for the desired panes.
Color Sample Describes the Color Sample, used to create a color set to use in the layout.
Color Picker Describes the Color Picker, a tool that enable you to visually specify colors.
Animation Sharing Information Window Describes the Animation Sharing Information window, used to share animations that are frequently used.
Parts and Control Settings Window Describes the Parts and Control Settings window, used to register and specify layout data as parts.
Parts Palette Window Describes the Parts Palette window, used to select optional parts that are listed.
LayoutViewer Describes LayoutViewer, which is the viewer for LayoutEditor.
Layout Settings Window Describes the environmental settings of the layout window.
