Random Initial Velocity (%)

This section describes the Random Initial Velocity (%) feature to randomize the initial velocity of emitted particles.

Overview

The Random Initial Diffusion Velocity (%) randomizes the initial velocity of each particle at the time of emission.
Randomizing the initial velocity of each particle when there are multiple particles emitted at one time will add visual texture to the effect.
The following overview describes the settings ().

Initial Velocity Randomness (%)

Randomly changes the particle emission initial velocity.
The upper and lower limits of the random numbers randomly change the initial velocity. The following formula calculates a lower limit for the random number.

  • Random Number Lower Limit: Calculated as Initial velocity - Initial-Speed Randomness.

Example Configurations

The following images compare what happens when an initial velocity of 0.5 is set in the Y direction and the Random Initial Velocity (%) value is changed.

A 0% (default) The initial particle velocity when particles are emitted remains the same, so particles are emitted at a constant interval.
B 50 % The initial velocity of each particle when emitted varies randomly in the range from 0.25 to 0.5.
C 100 % The initial velocity of each particle when emitted varies randomly in the range from 0 to 0.5.
