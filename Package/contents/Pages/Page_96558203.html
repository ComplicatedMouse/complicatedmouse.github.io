Fadeout Duration

This section describes Fadeout Duration which sets the time required for a fade out when executing a Fade command.

Overview

You can use this feature to extinguish particles over the specified amount of time when the Fade command is executed.
The Fadeout Duration feature is available when Alpha Fadeout or Scale Fadeout are being used.
The following overview describes the settings ().

Fadeout Duration

Set how much time elapses after the Fade command executes.

Example Configurations

The following figure compares what happens when different values are set for Fadeout Duration when Alpha Fadeout is enabled.

A 0 Particles die at the moment the Fade command is executed.
B 100 Particles gradually disappear over the course of 100 frames, with fadeout for both alpha and scale.
