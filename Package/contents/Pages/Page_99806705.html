util Library

Introduction

Purpose of This Documentation

This documentation describes the util library included in NintendoSDK.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Related Documentation

Refer to the nn::util library API reference along with this documentation.

Features Provided by the util Library

The util library provides the following features.

  • Bitwise operations
  • Zooming in and zooming out
  • Endian conversion
  • String operations
  • String formats
  • Text encoding conversion
  • Constants
  • Triangle functions and arithmetic utilities
  • Vectors and matrices
  • Other utilities

For more information about each feature, see the manuals.
