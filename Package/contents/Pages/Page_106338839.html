Quick Start

This document describes the basic operating instructions for LayoutEditor.

Workflow A description of the workflow, from the starting of LayoutEditor to the completion of the task.
Creating Layouts This section describes how to create the layout data.
Editing Layouts This section describes how to edit the layout data.
Using Guides Describes how to use guides to help align panes placed in a layout.
Creating Parent/Child Hierarchies Describes how to create parent/child hierarchies to consolidate multiple panes.
Using the Grouping Features Describes how to use groups to consolidate panes used with animation segment tags.
Using the Animation Segment Tag Feature Describes how to use the animation segment tag feature.
Curve Editor Operations Describes the UI operations for the Curve Editor, used to add animation by using curves in the property window.
Using the Parts Feature Describes how to use the Parts feature.
Specifying the Color Space Describes the process for specifying the color space in the layout.
Using Reference Colors Describes how to use reference colors, which are convenient when using the same color in multiple properties.
Deleting Unused Textures Describes how to use the Texture window to collect and delete textures that are not being used in the layout.
