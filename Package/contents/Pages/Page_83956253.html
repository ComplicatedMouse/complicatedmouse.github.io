Biped

Characters that use Biped can be exported.

However, because special processing is performed internally for Biped, normal display is not possible in the following cases.

  • No support for any scaling other than 1.0
    Normal display is not possible if a scale value other than 1.0 is applied to Biped bones because the exact matrix information pertaining to the scale is unobtainable.

Consider the following points when you configure character settings.

  • Figure mode and skin mode
    Configure skin pose settings at the origin using a Dummy object or PointHelper object as the Biped parent.
