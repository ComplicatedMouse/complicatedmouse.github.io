LayoutEditor

The LayoutEditor version is 3.39.0.

Added functionality to LayoutViewer for saving screenshots as PNG files.

Added the ability to save LayoutViewer screenshots in PNG format.
To save screenshots as PNG files, use .png as the filename extension when you specify the filename with the --capture-file-path option.

In the following example, all frames of the animation segment tag named "In" of the Tutorial_Menu.flyt sample data are saved as PNG files.

In Troubleshooting, added the information that multiple animations from the same group sometimes play in the Playback Confirmation pane.

Within animation segment tags, when some panes have show/hide keys and some do not, segment tags with keys may be affected by segment tags without keys, causing them to be hidden.
Added information that describes how to avoid having multiple animations from the same group sometimes play in the Playback Confirmation panel.

The Tag_01 and Tag_02 segment tags are attached to shown/hidden animations in the same group.
When you select Tag_00 that is not attached to an animation, the animation becomes hidden.

 You can restore the recorded animation to its initial state and normalize the playback frame state.
