Basic Common Features of Sequence Data

Time Base

The default value for time base (at quarter-note resolution) is 48.
The time unit for the time base scale is a tick. In other words, the default length of a quarter note is 48 ticks.
The default tempo is 120.

Comments

You can include comments in text sequences. Under the conventions used for text sequences, when a semicolon (;) is encountered in a line, everything from that point to the end of the line (line break) is treated as a comment.

;;; comment
 
Track_0:  ; comment

Label

A colon (:) placed after a string in a text sequence file is treated as a label definition.
You can specify a jump from one location to another by defining such labels.

If a text sequence containing more than one set of sequence data is used under SoundMaker, labels are used to specify the playback starting position.

Characters That Can Be Used in Labels

Label names must begin with an alphabetic character. Each subsequent character must be a single-byte alphanumeric character or an underscore (_).

LoopStart:
test_seq:
Track_01:

Local Labels

If an underscore is used as the first character in a label name, it is treated as a local label.

A local label is only valid between other labels that are not local labels.
The same local label name can be used in a different section.

label_1:
        ;; _local of the third line can be used here
_local:
        ;; _local of the third line can be used here
 
label_2:
        ;; _local of the seventh line can be used here
_local:
        ;; _local of the seventh line can be used here
 
label3:
        ;; _local cannot be used here
