Camera

Contains settings related to the camera being used by Viewer.

View-Projection

The View-Projection group () is a set of features for controlling the connected viewer.
This group has the following settings.

Camera Position (±) Configures the position of the camera in world coordinates.
Look-At Point (±) Configures the look-at point of the camera in world coordinates.
Camera Rotation (±) Rotates the camera around the z-axis in camera coordinates.
Near Configures the nearest position (near clipping), from the camera's viewpoint, in the camera's coordinate system.
Use this together with the Far setting to configure the rendering range.
Far Configures the farthest position (far clipping), from the camera's viewpoint, in the camera's coordinate system.
Use this together with the Near setting to configure the rendering range.
Camera Type For the camera type, select either Perspective or Ortho.
