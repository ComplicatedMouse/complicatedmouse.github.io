Thumbnail Explorer Extension

Thumbnail Explorer Extension User Manual

Overview

The Thumbnail Explorer extension extends the functionality of Windows Explorer, enabling it to show previews of NintendoWare format files in Explorer. The Thumbnail Explorer extension is included in the NintendoSDK package as an installer.

List of Features

  • Displaying Thumbnails in Windows Explorer
    • NintendoWare Intermediate Texture Files (FTXB and FTXA files)
    • Targa image (TGA)
    • Photoshop image (PSD/PSB)
    • Illustrator image (AI)
    • Illustrator EPS image (EPS)
    • Emitter Set (ESET)
    • Layout Files (FLYT)
    • NintendoWare Intermediate Model Files (FMDB and FMDA)
    • Extensions specified as parameters at installation time.

  • NintendoWare Intermediate Texture File Viewer
    • NintendoWare intermediate texture files (FTXB and FTXA) can be previewed and the details checked in the dedicated viewer.

Note
  • EPS thumbnails can only be displayed for EPS files saved in the Illustrator 9 EPS format or later in Adobe Illustrator.
  • AI thumbnails can only be displayed for AI files saved in Illustrator CS or later formats with the Create a PDF Compatible File option selected in Adobe Illustrator.

Included Tools

Thumbnail Preview Display Feature An extension feature to display thumbnails of image files in Windows Explorer.
TextureViewer Intermediate Texture Files (FTXB and FTXA Files) Preview Tool
Setup Manager This tool is for changing the settings of these other tools.

Installing and Uninstalling

Note
  • Windows administrator status is required to install or uninstall the Thumbnail Explorer extension.
  • Exit any running applications before installing or uninstalling. Also confirm that no other Windows Explorer extension tool is running.
  • When installing and uninstalling, make sure that other users are not logged in to the target PC at the same time.
  • Install this software using the account that will be using the Windows Explorer Extension Features. If you install the plug-in using an account other than the user account, extension features can no longer be used when target filename extensions are manually associated.

Installation

Installer Location

The Thumbnail Explorer Extensions installer is included in the NintendoSDK package. After installing the NintendoSDK package, the Thumbnail Explorer Extensions installer is located in the <root path>/Installers/ThumbnailExplorerExtensions/ folder. To activate the standard settings functionality, run the setup_jp.exe file.

Changing Settings

To manually change some feature settings, including application binding settings, use the setup manager. If settings are changed without using the setup manager, the thumbnail/preview display feature could be disabled. After installing the Thumbnail Explorer extension, the setup manager is located in the %ProgramFiles%/Nintendo/ThumbnailExplorerExtensions/Setup folder. For more information, see the Setup Manager page.

Updates

Only one version of the Thumbnail Explorer extension may be installed on each PC. After updating, run the Thumbnail Explorer Extensions installer included in the updated NintendoSDK package.

Registering Optional Extensions

To register the extensions .aaa and .bbb, specify /custom=aaa;bbb as an installer parameter. Delimit the extensions using commas. For more information about displaying the thumbnails, see Displaying Thumbnails of Files in Optional Formats.

Deleting Optional Extensions

On subsequent installations and updates, the prior settings will be inherited and the specified optional extensions will be re-registered. To prevent these prior settings from being inherited, specify /remove-custom as an installer parameter. You can use this parameter with the /custom option. If used in conjunction, only the settings for the extensions specified with the /custom option will be inherited.

Installing Without Displaying Dialog Boxes

To hide the dialog boxes when installing and updating, specify /VERYSILENT, /SUPRESSMSGBOXES, and /no-restart-explorer as installer parameters. You can use these same options for the uninstaller (unins000.exe).

Note
  • If you install or update with the /no-restart-explorer option while Windows Explorer or another application is using the Thumbnail Explorer Extension, a nonzero error is returned and the process ends.
  • The installer and uninstaller execute as GUI applications, so you cannot get error codes even if you run the installer and uninstaller from the command line (cmd.exe). To get the error codes, use a batch version of the method.
  • The /no-restart-explorer option does not work when uninstalling or updating from an earlier version that does not support /no-restart-explorer (Explorer extension versions prior to 2.2.0).
  • If you run the program without administrator privileges, the dialog box for user account control is displayed.
  • An installation that does not show dialog boxes might not display thumbnails in Explorer immediately after the installation.

Uninstalling

You can use any of the following methods to uninstall the Thumbnail Explorer extension.

  • With the Thumbnail Explorer extension installed, run setup.exe (used for installation), and then select Delete.
  • In Windows Control Panel, select Programs and Features, and then uninstall the NintendoSDK ThumbnailExplorerExtensions.
  • Run the unins000.exe file located in %ProgramFiles%/Nintendo/ThumbnailExplorerExtensions.

Other

  • All applications using file dialog boxes must be able to display thumbnails, but this feature might not work properly on all Windows applications. Custom implementations of dialog boxes may have unexpected behavior.
  • Preview of 3D textures is not supported by the Thumbnail Explorer extension.
  • Images of intermediate texture files, displayed using Thumbnail Explorer extensions, have undergone simple conversion, according to the format and other specifications of the original image. There may be small differences with conversion results using a texture converter.
  • Thumbnails may not be displayed due to changes in association for the extension, such as when other applications are installed. In that case, you must configure the settings again using the setup manager.

Known Bugs

Thumbnails are all the same in Maya's File dialog box.

In the Maya File dialog box, the thumbnails for files in a folder are also displayed for other files. To prevent this issue, in Maya, open the Preference menu. In File/Project, for File Dialog, specify OS Native. This bug has been confirmed in the following versions of Maya.

  • Maya 2011
  • Maya 2012
  • Maya 2013
  • Maya 2014
  • Maya 2015

Microsoft Edge exits when Explorer restarts.

It has been confirmed that Microsoft Edge exits when Explorer is restarted, during installation or when settings are changed using Setup Manager.

TextureViewer crashes when displaying at high magnification.

TextureViewer sometimes stops responding or crashes when the display is set to high magnification.
