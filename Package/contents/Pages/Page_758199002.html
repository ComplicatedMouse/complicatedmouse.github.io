Right Window Group

The Right Window contains the following tabs:

  • Start: Provides easy access to useful actions and information.
  • Overview: Visual summary of the profile.
  • Checkers: Static and dynamic checks that were run on the profile and assembly code.
  • Call Tree: The function call tree. This tab only appears if profile contains callstack information.
  • Timeline: The main graphing window showing data over time.
  • Code Coverage: Shows which functions were seen/not seen during the profile.
  • Assembly: Shows the assembly line hit counts for a single function.
