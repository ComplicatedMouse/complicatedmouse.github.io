LayoutEditor

This section introduces the new features in LayoutEditor.

LayoutEditor Update Dialog Box

A dialog box now appears when starting LayoutEditor to notify the user of newly added features.
In the dialog box, select News to jump to the latest news for layouts and fonts.
You can change whether the dialog box appears in the Tool Environment Settings.

LayoutViewer

This section introduces the new features in LayoutViewer.

Added the Ability to Display Extended User Information

You can now display any extended user information that was configured for panes in LayoutViewer.

Enable PaneInfo in the Debug Data menu to display extended user information when you click a pane.
You can also right-click a pane to display a list of panes at the position of the mouse pointer, and select the pane to display its extended user information.

Runtime Libraries

ui2d

Different types of archive shaders are now bundled together in a single file.

Previously, if (for example) you were working with an archive shader to use when using three textures and an archive shader to use when using the advanced combiner, multiple archive shaders were created. Now, these multiple archive shaders are bundled together into a single file.

Also, previously the maximum number of archive shaders was only 256. Now, up to 65,536 archive shaders are supported.

This change breaks compatibility between shader files and layout binary files, so the files must be reconverted.
