Introduction

Purpose of This Documentation

This documentation describes the features of the NG word check (ngc) library included in NintendoSDK.

Intended Audience

This documentation is intended for people developing applications using NintendoSDK.

Features Provided by the ngc Library

Checking for Profanity

The ngc library provides the following profanity check features.

  • Checking strings against a profanity list to determine whether the string contains profanity
  • Counting the number of numeric characters in strings

For more information about this feature, see Checking for Profanity.

DirtyWord Filter

The ngc library provides a DirtyWord filter function enabled specifically on consoles for China.

For more information about this feature, see DirtyWord Filter.
