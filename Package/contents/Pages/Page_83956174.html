Filter Settings to Use During Sampling

The filter to use during texture sampling is specified from a drop-down list on the Samplers page of the material Property Window.

The following tables describe each setting.

Expand

Specifies a filter for enlarging the texture.

Filter Description
Point Performs point sampling.
Linear Applies a linear filter during sampling.

Shrink

Specifies a filter for reducing the texture.

Filter Description
Point Performs point sampling.
Linear Applies a linear filter during sampling.

Mipmaps

Specifies a filter between mipmaps.

Filter Description
None Do not apply a mipmap.
Point Apply a mipmap.
Linear Apply a mipmap and use linear interpolation between mipmaps.

Anisotropic

Specifies the maximum value for the anisotropic animation.
