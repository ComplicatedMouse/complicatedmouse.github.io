Tools

This page describes the additional Tool menu for LayoutEditor.
Remove Unused Fonts Deletes unused fonts.
Load Sequential Textures Displays the Load Sequential Textures window for automatically generating sequential textures for a texture pattern animation.
Colocate Textures, Fonts, and Parts

Collects the textures, fonts, and parts used in a layout and puts them in a single folder.
Send to Viewer Sends the layout to the viewer.
Use this setting to check feature animations.
Transfer to Viewer (Animation Mode) Transfers the layout to the viewer in animation mode.
Launch Hardware Viewer

Starts the Hardware Viewer.

When checking a layout in the Hardware Viewer, first start the Hardware Viewer and then transfer the layout.
Automatically start the standard PC viewer

The standard PC viewer is automatically started during transfer.

Disable this option when using a custom viewer.
Send the first layout that was opened

You can select the verification target in LayoutViewer when you open a part layout from the Body Layout's list of related parts ().
When ON, the body layout is transferred to the viewer.
Capture current frame with PC Viewer

Captures the frame currently displaying in the layout window with the PC viewer.

The captured image is saved to the desktop.
Compress Pane Tree (Experimental)

This feature is currently in testing and its description is omitted.
Automatic Scaling (Experimental) This feature is currently in testing and its description is omitted.
User Commands

Starts other tools from LayoutEditor.

You can choose the created command.
