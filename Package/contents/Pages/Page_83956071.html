User Settings File Format

This section describes the components included in user settings file.
The 3dEditor.user.config file is in XML format.

This description includes a 3dEditor.user.config example and each component that is used.

3dEditor.user.config Sample

This example uses 3dEditor.user.config.

<?xml version="1.0" encoding="utf-8"?>
<UserSetting Version="0.0.0">
    <SearchPath Name="folderB" Recursive="false">C:\folderB</SearchPath>
    <StartUpPath>%DATA_FOLDER%\modelA\modelA.f3pj</StartUpPath>
    <ReloadOptions>
        <Model Reload="true" ShowDialog="true" />
        <Texture Reload="true" ShowDialog="false" />
        <SkeletalAnimation Reload="true" ShowDialog="false" />
        <MaterialAnimation Reload="true" ShowDialog="false" />
        <ShaderParameterAnimation Reload="true" ShowDialog="false" />
        <ColorAnimation Reload="true" ShowDialog="false" />
        <TextureSrtAnimation Reload="true" ShowDialog="false" />
        <TexturePatternAnimation Reload="true" ShowDialog="false" />
        <BoneVisibilityAnimation Reload="true" ShowDialog="false" />
        <MaterialVisibilityAnimation Reload="true" ShowDialog="false" />
        <ShapeAnimation Reload="true" ShowDialog="false" />
        <SceneAnimation Reload="true" ShowDialog="false" />
        <ShaderDefinition Reload="false" ShowDialog="false" />
    </ReloadOptions>
    <EditHistory ClearAfterFileClosed="false" ClearAfterFileReloaded="false" ClearAfterFileSaved="false" MaximumItemCount="10000" />
    <ExternalProgram TextEditor="notepad.exe" TextEditorFileOpenOption="&quot;%1&quot;" EnableRuntimeDebugLog="false" />
    <IO AutoFix="false" AutoReloadUpdatedOnLoad="true" DumpShaderSource="true" TextureCompositorDisableCache="false" MaximumRecentlyUsedFileCount="10" MaximumRecentlyUsedProjectCount="10">
        <DumpShaderSourcePath>C:\dumpFolder</DumpShaderSourcePath>
    </IO>
    <UI FixedSlider="false" />
</UserSetting>

<UserSetting>

The root element of the file.

Attributes

Name Description
Version The 3dEditor.user.config version. The current version is 0.0.0.

Description

The user config file includes the following elements.

  • <SearchPath> : Additional reference path settings. Zero or more may appear.
  • <StartUpPath> : Startup path settings. Zero or more may appear.
  • <ReloadOptions> : Reload when files are updated settings. One of these elements at most.
  • <EditHistory> : Edit history settings. One of these elements at most.
  • <ExternalProgram> : External program settings. One of these elements at most.
  • <IO> : I/O settings. One of these elements at most.
  • <UI> : Control settings. One of these elements at most.


<SearchPath>

The element that includes the Additional reference path setting.

Attributes

Name Description
Name The additional reference path name. This attribute can be omitted for a relative path.
Recursive Specifies whether to include subfolders of the folder specified by the additional reference path in searches. Set to true or false. If omitted, false is assumed.

Section

The absolute or relative path to the folder. This path may include environment variables surrounded by percent signs (%).

<StartUpPath>

Startup path settings.

Attributes

Name Description
Base The name of the additional reference path used as the starting point. Omit this component if the startup path is not a relative path from the additional reference path.

Section

The absolute or relative path to the file or folder. This path may include environment variables surrounded by percent signs (%).

<ReloadOptions>

Reload when files are updated setting.

Description

The following elements appear once at most.

No files of any type are reloaded if this is omitted.

  • <Model>: Setting to reload model files.
  • <Texture>: Setting to reload texture files.
  • <SkeletalAnimation>: Setting to reload skeletal animation files.
  • <ShaderParameterAnimation>: Setting to reload shader parameter animation files.
  • <ColorAnimation>: Setting to reload color animation files.
  • <TextureSrtAnimation>: Setting to reload texture SRT animation files.
  • <TextureSrtAnimation>: Setting to reload texture pattern files.
  • <BoneVisibilityAnimation>: Setting to reload bone visibility animation files.
  • <MaterialVisibilityAnimation>: Setting to reload material visibility animation files.
  • <MaterialAnimation> : Setting to reload material animation files.
  • <ShapeAnimation> : Reloads shape animation files.
  • <SceneAnimation> : Reloads scene animation files.
  • <ShaderDefinition> : Reloads shader definition files.

<Model> <Texture> <SkeletalAnimation> <ShaderParameterAnimation> <ColorAnimation>
<TextureSrtAnimation> <TexturePatternAnimation> <BoneVisibilityAnimation> <MaterialVisibilityAnimation> <MaterialAnimation> <ShapeAnimation> <SceneAnimation> <ShaderDefinition>

Setting to reload all types of files. If omitted, files are reloaded and a confirmation dialog box appears whenever you reload an unsaved file.

Description

The absolute or relative path to the file or folder. This path may include environment variables surrounded by percent signs (%).

Attributes

Name Description
Reload Specifies whether to reload files when they have been updated by another application.
Set to true or false. If omitted, true is assumed.
ShowDialog Specifies whether to display the confirmation dialog box before reloading files.
Set to true or false. If omitted, false is assumed. This feature is enabled only when Reload is true.
Even if set to false, a confirmation dialog box is displayed when reloading unsaved files.

<EditHistory>

This component includes edit history settings.

Attributes

Name Description
ClearAfterFileClosed Sets whether to clear the edit history when a file is closed. Set to true or false. If omitted, false is assumed.
ClearAfterFileReloaded Sets whether to clear the edit history when a file is reloaded. Set to true or false. If omitted, false is assumed.
ClearAfterFileSaved Sets whether to clear the edit history when a file is saved. Set to true or false. If omitted, false is assumed.
MaximumItemCount Sets the maximum number of items that can be saved in the edit history by 3DEditor. Specify a value equal to or greater than 0. A value of 10,000 is assumed if omitted.

<ExternalProgram>

This component includes external program settings.

Attributes

Name Description
TextEditor Specifies the text editor to be called up by 3DEditor. Set the file path to the text editor executable.
The text editor defined by the system is called up if this omitted.
TextEditorFileOpenOption

Specifies options used when calling up the text editor.
EnableRuntimeDebugLog Setting for whether to output debugging logs for the application that is connected to.

<IO>

This element controls the I/O settings.

Description

The user config file includes the following elements.

  • <DumpShaderSourcePath>: The output directory for the shader compiling information.

Attributes

Name Description
AutoFix Specifies whether to auto-correct materials when a model is loaded. Set to true or false. If omitted, false is assumed.
AutoReloadUpdatedOnLoad Specifies whether to automatically reload files that are currently loaded if they have been updated when they are reopened. Set to true or false. If omitted, false is assumed. Specifying false displays a confirmation dialog box.
DumpShaderSource Specifies whether to export compiling information during shader optimization. Set to true or false. If omitted, false is assumed. You must also set the DumpShaderSourcePath option to export compiling information.
TextureCompositorDisableCach
Note

This feature is operating under test. When using this feature, note that its operation and specifications may change in the future, without advance notice.

Sets whether to disable the 3dIntermediateFileTextureCompositor.exe caching feature. Set to true or false. If omitted, false is assumed.
MaximumRecentlyUsedFileCount Sets the maximum number of recently used files that are displayed in the file menu.
MaximumRecentlyUsedProjectCount Sets the maximum number of recently used projects that are displayed in the file menu.

<DumpShaderSourcePath>

The output directory for the shader compiling information.

Description

The absolute path of the output directory for the shader compiling information. This path may include environment variables surrounded by percent signs (%).

<UI>

This component includes UI control settings.

Attributes

Name Description
FixedSlider Sets whether to display the slider control on the Shader page in the material Property window.
Set to true or false. If omitted, false is assumed.
