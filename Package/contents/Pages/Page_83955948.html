Material Property Window: Preview Page

This page is used to check and edit preview information for the selected material.

A Include in optimization
When checked, the material is included in shader optimization processing when a model is reloaded.
This option is enabled by default.
Note: Shader optimization must be implemented in the runtime to enable this feature.
B Re-create the optimized shader following the changes to the material
You can set using <DefaultValue> when Include in optimization is enabled.
You can set the default values for check boxes in <DefaultValue> in the team settings file.
C Shader send status
Displays the status of shader optimization for each material.

Regenerating the Shader When the Material is Modified

This section describes the behavior both when the Follow the material changes and re-create the shader option is selected and when it is cleared.

Check Box Is Selected

When shader parameters are edited, the optimization shader is automatically regenerated.

Check Box Is Cleared

Even when shader parameters are edited, the optimization shader is not automatically regenerated.
You can regenerate the optimization shader manually by clicking the Regenerate Optimization Shader button on the toolbar in the main window.
