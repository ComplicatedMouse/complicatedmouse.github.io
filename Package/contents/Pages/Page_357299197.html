Guide

This section provides a description of guides, which can be used to align panes efficiently.

Overview

Guides are a feature for aligning panes efficiently.
By enabling View / Snap to Guides in the Layout window, you can place panes along these guide lines.
Guides can be selected, moved, rotated, copied, pasted, deleted, and operated on just like panes. The following graph depicts the coordinate system when operating with guides.

 The red circle (○) is the center of the guide.
Position sets the center of the guide in world coordinates.
Rotation sets a point of reference for the center of the guide.
If you specify a value in the positive direction, the guide rotates counterclockwise.
