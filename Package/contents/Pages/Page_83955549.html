Command Line

Overview

You can change the startup behavior of 3DEditor using command-line options when starting 3DEditor at a command prompt.

Starting 3DEditor at a Command Prompt

When the directory that 3DEditor.exe is installed in is the current directory, or if a symbolic link has been established, you can use the following command to start 3DEditor.

Info

3dEditor.exe

Command-Line Options

Info

3dEditor.exe [@ParameterFile][ - - version] [ --team-settings-file=TeamSettingsFile [-r] [ProjectFile|IntermediateFile|Folder]...

@Parameter File

You can use the contents in the parameter file to specify arguments. The parameter file allows you to specify each option over several lines. Lines starting with a number sign (#) are treated as comments. Newline characters cannot be included in escape sequences used in option specifications that appear between quotation marks (").

--version

Displays the 3dEditor version. When this option is selected, 3DEditor is not started and terminates.

--team-settings-file

Specifies the team settings file.

-r --recursive

Recursively loads intermediate files in the specified folder if a folder is specified.

Project File|Intermediate File|Folder

Specifies the files and folder to load. If you specify a project file, do not specify any other files.

Sample Option Specifications

Loads the specified team settings and starts 3DEditor.

Info

3DEditor.exe --team-settings-file=GameProject.3dEditor.teamsettings
