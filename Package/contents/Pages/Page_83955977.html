Using the Parts Features

0. Setting the Parts Palette and the Parts Root

All the parts that are used by LayoutEditor are stored together in a folder called the parts folder, and the path to this folder is called the parts root.
The first step is to set the parts root.

The parts root is specified by the parts palette.
The parts palette displays a list of the parts layouts below the parts root.
If a new file is saved or if the folder structure is changed, the parts palette is updated to the newest state when it is reloaded.

1. Creating a Parts Layout

  1. Create a normal layout.

    The first step is to create a standard layout by using the following procedure.

    • Positioning the Pane
    • Set the animation (if necessary).
    • Set animation segment tags and the target group (if necessary).
  2. Configure the parts settings for this standard layout.

    Make parts settings for the standard layout you just created.
    The parts settings are configured in the Parts and Control Settings window.
    Set the parts type, name, comments, feature animation, feature pane, and overwritable properties settings.

  3. Save the parts layout to the parts folder.

    Save your created parts layout below the parts root you just configured.
    When you reload the parts palette, the saved layout displayed in the list.

2. Positioning the Parts Pane

  1. Create a body layout for positioning the part.

    Using the Create New feature, create a layout for positioning the part.
    This layout is called a body layout.

  2. Create a parts pane.

    Create a pane called a parts pane for positioning the parts layout.
    Configure a Size Scaling to scale the size of the parts pane relative to panes located underneath the parts pane.
    To create a parts pane, in the Parts Palette list, right-click a parts layout, and then on the shortcut menu, select the option to create a parts pane.
    This procedure positions a parts pane in the body layout.

Related Links For more information about Parts Pane, see here.

3. Adjusting the Parts Pane

  1. Adjust the size scaling.

    You can specify a size scaling for the parts pane and adjust the size.
    To set Size Scaling, select the parts pane, and then in the properties window, select the parts pane page.

    Note that, unlike other pages, you cannot a set any arbitrary size for the property. In addition, the size cannot be animated.

    Difference Between Size Scaling and Size Restriction

    The difference involves instances where the size of the parts layout is changed after the parts pane has been positioned.
    With Size Restriction, the size that was specified at the time of positioning is maintained, and the size does not change.
    With Size Scaling, the size changes.

    Types of Size Scaling

    The size-scaling described in this section affects the parts layout and can be specified for each pane separately.
    To set the size-scaling type for a pane, select the pane and then in the properties window, set the type in the Basic Pane Information.

    • A: Ignore - Size scaling is ignored. Specify this option when you want the pane to maintain a fixed size, regardless of the size of the part.
    • B: Scale size - Size scaling is applied to the size.
    • C: Maintain distance from boundaries of parts containing self - The pane size is changed so that a set distance is maintained to the periphery of the part that size-scaling has been applied to. This setting is the default.
  2. Overwritable Properties

    From the parts pane Parts Settings you can overwrite and change the content of predetermined panes.
    For overwritable panes, the category items are shown in the properties window when the parts pane is selected.
    To set a pane for overwriting, edit the Parts Settings in the parts layout.
