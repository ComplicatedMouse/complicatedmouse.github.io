Shader Resource Files

Shader Resource Files

Overview

Shader resource files express binary images created using the Shader Converter tool.

Shader resource files, in addition to expressing shaders compiled in advance, provide a mechanism to handle shader variations.

Shader File Resources

Shader file resources are defined as nn::gfx::ResShaderFile.

Shader file resources represent binary images themselves created using the Shader Converter and can use common features of resource file classes.

Getting Shader Container Resources

Use GetShaderContainer to get Shader Container Resources.

Shader Container Resources

Shade container resources are defined as nn::gfx::ResShaderContainer.

Before using shader container resource features, you must first initialize the shader container resources.

Initialization and Finalization

Initialize shader container resources using Initialize, and finalize shader container resources using Finalize.

If you did not specify the memory pool during initialization, the memory pool in the resources is used. When the --disable-memory-pool option is specified during resource creation by the Shader Converter, no memory pool is held in the resources, so initialization is not possible using this method.
When the memory pool on the user side contains shader container resources, specify that memory pool and call Initialize to use the memory pool on the user side instead of the memory pool in the resources.

Getting the Variation Count

To get the shader variation count associated with the shader container resources, use GetShaderVariationCount.

Getting Shader Program Resources

To get the shader program resources of the specified variation, use GetResShaderProgram.

Shader Program Resources

The shader program resources are defined as nn::gfx::ResShaderProgram.

Getting Shader Information

To get the gfx shader information, use GetShaderInfo.

Getting Shader Objects

To get the gfx shader objects, use GetShader. Before initialization, the shader object region is padded with zeros.

Initialization and Finalization

To initialize the shader objects obtained by GetShader, call Initialize and use the information obtained by GetShaderInfo.

To finalize the shader objects obtained by GetShader, call Finalize.
