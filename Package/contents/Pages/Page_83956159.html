Bone Visibility Animation

Bone visibility animations reflect the Visibility configured in the Rendering Control settings.
0.0 is interpreted as hidden. Values greater than 0.0 are interpreted as shown.
The visibility animations of objects hidden on the viewport (interactive hidden objects) are not displayed.

For bone visibility animations, animation curve data is always automatically created and exported using baked data.

Info

Because bone visibility animation is set on a node basis, when the Compress Bone export option has been set to Cull Useless Node, Merge Useless Node, or Unite Compressible Node, make sure that nodes for which bone visibility animation is set are not compressed. (You can set the compression disable flag using plug-in for not compressing nodes.)
Do not set Compress Node to Unite All when using bone visibility animation.
Make sure that nodes for which bone visibility animation is set are not gathered together when Unite Child is selected.
