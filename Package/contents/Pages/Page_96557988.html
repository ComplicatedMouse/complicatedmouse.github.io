Random Seed

This section describes how to use the Random Seed functionality to configure random numbers so that effect appearances are identical every time.

Overview

The Random Seed feature can be used to directly specify a random number.
By setting the Random number type to Constant, the same random seed is used so features with random elements function in a coordinated way among specified emitters and effects look the same every time.

Set the Random number type to Constant to display the Constant () setting. The group has the following settings.
The following overview describes the settings ().

Random Seed

Sets a random number value. This value is used as the seed for creating random number strings.
The same random number string is generated from a particular random seed each time the effect is called.
