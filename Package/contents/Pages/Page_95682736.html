Sphere Map

This section describes the feature for using registered textures for sphere mapping.

Overview

The Sphere Map feature enables you use textures for sphere mapping in the EffectMaker shader.
This feature is used mainly with Primitive shape particles.
Due to the nature of sphere maps, the effect is easier to understand when the primitive is divided into polygons and the normal is open.
The following overview describes the settings ().

Sphere Map

Use the check box () to select whether to apply the registered texture to a sphere map.
