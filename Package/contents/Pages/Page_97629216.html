Period

This section describes the waveform settings.

Overview

These waveform settings can be configured when sine wave is selected for the waveform type.
This section provides an overview of the settings ().

Period

Sets the period of the flux, in units of frames. The smaller this value, the finer the fluctuations.
If the period is set to a value that is shorter than the lifespan of the particle, the process loops.
