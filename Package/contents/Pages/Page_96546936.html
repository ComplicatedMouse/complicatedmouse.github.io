Phase Change Rate

This section describes the Phase change rate feature, which allows you to set the rate at which the phase of a waveform element changes.

Overview

The Phase change rate feature feature allows you to change the phase of a waveform element.
Specifying a phase change rate makes it possible to gradually change the trajectories of particles that are following the same path.
The following overview describes the settings ().

Phase Change Rate

Phase change rate specifies the rate at which to change the phase.
The phase will not change if this value is set to 0.

Example Configurations

The following images provide a comparison of what happens when the Phase change rate values are changed.

A Phase change rate = 0 The default setting.
With this setting, the phases of the particles do not change, and they move according to a constant waveform.
B Phase change rate = 0.5 With this setting, unlike in A, the phase of the waveform that affects the particles changes, and as a result the waveform path gradually changes.
