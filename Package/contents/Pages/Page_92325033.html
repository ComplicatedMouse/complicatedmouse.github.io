Emitter Node Operation

This page describes how to operate emitter node features in the workspace.

Preparations

Related Links Before beginning, complete the tasks described in Quick Start > Creating Effects.

Using the steps to this point, multiple emitters are created directly beneath the EmitterSet1 emitter set (), as follows.

Changing the Emitter Name

This section describes how to change the name of an emitter that has already been created.
In the workspace, in the EmitterSet1 emitter set (), select the JustOne emitter () and then perform one of the following operations.

  • From the main menu, select File > Rename ().
  • Right-click and select Rename () from the shortcut menu.
  • Press F2.

The name can now be changed, so type Emitter3 as the emitter set name, and press ENTER to save the change.
JustOne () is changed to Emitter3 ().

Duplicate Emitter

This section describes how to duplicate the selected emitter.
From the EmitterSet1 emitter set () in the workspace, select the Emitter3 emitter (), and then perform one of the following operations.

  • From the main menu, select Edit > Duplicate ().
  • Right-click and select Duplicate () from the shortcut menu.
  • On the toolbar, click Duplicate ().
  • The keyboard shortcut is CTRL+D.

The Emitter3_Copy1 emitter () is added below the EmitterSet1 emitter set () in the workspace.
To change the name of the emitter from Emitter3_Copy1 () to Emitter4, press F2.

Completing the procedures to this point creates four emitters directly under EmitterSet1 (), as follows.

Selecting and Editing Multiple Emitters Simultaneously

This section describes how to select multiple emitters in the workspace.
In the workspace, select Emitter1 (), and then use the following operations to select multiple emitters.

    • To select Emitter3 () in addition to Emitter1 (), press CTRL and click Emitter3 ().
    • To select Emitter1 (), Emitter3 (), and Emitter2 () between the two, press SHIFT and click Emitter3 ().

Use the following instructions to batch edit the multiple selected emitters.

1
The following example shows how to change the Position(±) X value in the Emitter Transform group to Value = 10.12 ().
2
To clear the selection of multiple emitters, click EmitterSet1 ().
3

Check the Position(±) X value of the multiple emitters that were selected, and confirm that the value is still the same Value = 10.12 ().

Changes made to settings apply to all emitters selected using these procedures, allowing you to change the settings of multiple emitters by only editing one of the emitters.
