Binary Command-Line Options

This section describes the command-line options for binary conversion of the textures, 3D binary, and shaders in the effect binary.


 Tools in Use Command-Line Options
Texture 3dTextureConverter.exe

-o EffectMaker\Converter\texture_intermediate\output.bntx

--tile-mode=Linear "input01.ftxb" "input02.ftxb" ( *2 )
3D binary 3dBinaryConverter.dll

--ignore-assign --silent --disable-attribute-alignment-sort

Vertex and pixel shaders ShaderConverter.exe

--glsl-version=430 -sGlsl -aGl --code-type=Source

-o EffectMaker\Converter\shader_intermediate\output.grsf

--vertex-shader=EffectMaker\Converter\shader_intermediate\shader.vsh

--pixel-shader=EffectMaker\Converter\shader_intermediate\shader.psh

--include-directory=EffectMaker\Converter\shader

--variation=EffectMaker\Converter\shader_intermediate\shaderVariation.xml

--print-progress --code-page 65001 ( *3)
Compute Shader ShaderConverter.exe

--glsl-version=430 -sGlsl -aGl --code-type=Source

-o EffectMaker\Converter\shader_intermediate\output.grsf

--compute-shader=EffectMaker\Converter\shader_intermediate\computeShader.vsh

--include-directory=EffectMaker\Converter\shader

--variation=EffectMaker\Converter\shader_intermediate\computeShaderVariation.xml

--print-progress --code-page 65001 ( *4 )
Info

(Note 2) There will be a list of FTXB file paths for textures that will be converted all at one time.

(Note 3) The content of the -o option for the ShaderConverter.exe output path will be replaced when the --output-shader-binary option is specified during command-line conversion.

(Note 4) The content of the -o option for the ShaderConverter.exe output path will be replaced when the --output-compute-shader-binary option is specified during command-line conversion.
