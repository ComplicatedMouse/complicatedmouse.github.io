Creating an Account For China in the Development Environment

A Nintendo Switch console that has been set for the China region cannot use the network features without an account that has been created for the China region. This section describes how to create such an account in the development environment.

Advance Preparations

Change the Platform Region to China

For more information about switching the platform region of the development hardware to China, see Testing Environment.

Create a Nintendo Account

For more information about creating a Nintendo Account for the development environment, see Creating and Managing Nintendo Accounts in the Development Environment.

Enter optional values for Country/region and the other settings for the Nintendo Account you are creating. You cannot use an Nintendo Account for a child (under the age of 13).

Creating an Account For China in the Development Environment

You need an account for WeChat, the real-time messaging app provided by Tencent, to create production environment accounts for China.

In the development environment, there is a method for creating an account for China that does not require a WeChat account. This method does not work in the production environment.

Procedure for Creating an Account For China in the Development Environment

Use the following procedure to create an account for China in the development environment.

Note
  • There are two kinds of Basic authentication. When creating an account for China, pay attention to the kind of authentication that is being requested.
    • Basic authentication for accounts for China For more information, see Basic Authentication When Linking an Account for China later on this page.
    • Nintendo Accounts For more information, see Creating and Managing Nintendo Accounts in the Development Environment.
  • Parts of the procedure may change when the server is updated.


  1. Create a local account on a Nintendo Switch console that has been set for the China region.
  2. Proceed to the step for linking the account.
  3. When the screen for Basic authentication is displayed, enter the user name and password for Basic authentication of the account for China.
    We recommend saving the Basic authentication information in the browser at this time.
  4. After authentication has completed, the following screen is displayed.
    1. Click Login with Nintendo Account. This link only appears in the development environment.
  5. When you click this link, the Basic authentication process repeats.
    1. For this round of Basic authentication, enter the user name and password for Basic authentication of the Nintendo Account.
    2. If the Basic authentication information for the account for China has not been saved, enter the Basic authentication information for the account for China before entering the Basic authentication information for the Nintendo Account.
  6. Proceed with linking a Nintendo Account just like you would on a console set to Global.
    2. If the Basic authentication information for the account for China has not been saved, enter the user name and password for Basic authentication of the account for China after signing in with the Nintendo Account.
    3. If the Basic authentication information for neither the account for China nor the Nintendo Account has been saved, enter the Basic authentication for the account for China before entering the Basic authentication information for the Nintendo Account.
  7. After you have advanced to the following screen, proceed by entering information.
    2. If the Basic authentication information for the account for China has not been saved, enter the user name and password for Basic authentication of the account for China after providing that information.
  8. When the following screen appears, select 跳过.
    2. If the Basic authentication information for the account for China has not been saved, enter the user name and password for Basic authentication of the account for China after making this selection.
  9. When the following screen appears, select the button on the right.
  10. Select 確定 to complete the process of registering the account for China.
