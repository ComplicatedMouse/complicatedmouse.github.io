Emitter Assignment for Particles

This section describes the Emitter Assignment for Particles feature that is used to assign emitters for particles emitted from the parent emitter.

Overview

The Emitter Assignment for Particles inheritance setting feature allows child emitters to assign emitters for particles emitted from the parent emitter.
It is possible to have the child emitter follow the position of the particles emitted from the parent emitter for every frame.

Info

To use stripes with child emitters, enable Emitter Assignment for Particles.

Emitter Assignment for Particles

To enable or disable the option to make particles emitted from the parent emitter be emitters, select or clear the check box ().
When the check box is cleared (the default), the emitter assignment is child emitter, which has little processing load.

The following table shows the differences in behavior when this option is enabled and disabled.

ON

While the child emitters each have an emitter, this causes computational load for each emitter.

Enable the setting if you want to use child features from before this feature was supported.
OFF The computational load of emitters is reduced because the child emitters inherit a single shared emitter.
While it is particularly effective when many child emitters are emitted, it is subject to the following restrictions.
  • Emitter-time animations for child emitters are not supported. While it is possible to set them in the UI, they will be ignored during execution.
  • Cannot follow the movement of the parent particles after the child emitters are released.
  • Cannot use Historical Stripe alongside Historical Stripe2.
Warning

When Emitter Assignment for Particle is disabled, Historical Stripe and Historical Stripe2 cannot be used.
When this setting is configured, a warning message is displayed at run time and emitter generation is canceled.

Info

If data using child emitters from before this feature was supported is loaded, the data will be loaded in the state where Emitter Assignment for Particles is enabled.
