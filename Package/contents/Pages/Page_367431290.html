Duration

This page describes the Duration feature, which sets the range of the reference curve.

Overview

The Duration feature determines the range of the curve that is referenced when a character is placed.
It specifies the number of frames of the curve to use for per-character transformation, starting at the offset time.
The following examples use Duration.

Reference Curve

Duration = 100 (Default) Duration = 50



Frames 0 to 100 of the curve are used. Frames 0 to 50 of the curve are used.
