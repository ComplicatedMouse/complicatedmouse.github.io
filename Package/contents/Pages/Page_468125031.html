async Library

Introduction

The async library is a library required for applications to use the asynchronous API functions provided by NintendoSDK.

This document is intended for application developers using the asynchronous API functions provided by NintendoSDK.

Important Information

Note
  • Currently, most asynchronous functions provided by NintendoSDK work without requiring the async library. Applications do not need to use the async library as long as they do not use other libraries that explicitly require the async library.
  • The async library is available in NX Add-On 8.1.0 and later.
