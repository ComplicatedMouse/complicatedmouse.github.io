Custom Actions

Specifies files for custom actions used on a per-emitter and per-child basis by EffectMaker.

Custom Action Definition File Path

The Custom action definition file path () is used to register action definition files used in a game project.
You can use files in which particle actions unique to game applications are written.

Custom Action Settings

The Custom action settings group () is used to display the content of action definition files used in a game project.
