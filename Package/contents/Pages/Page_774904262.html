Troubleshooting

This section provides a collection of frequently asked questions.

  • Exporting Models
    • Shapes of the same material are not compressed even though Compress Shape is enabled.
    • An error occurs when Polygon Reduction is enabled.
    • Exporting the model takes an unusually long time.
  • Exporting Animations
    • Simple animation curves get baked even though Bake All Animation is not enabled.
    • Exported animations are choppy.

Exporting Models

Shapes of the same material are not compressed even though Compress Shape is enabled.

The shapes you want to compress might have different vertex attributes. For example, if only one of the shapes has vertex color or the shapes are assigned different UV values, shape compression is not performed.

An error occurs when Polygon Reduction is enabled.

Errors can occur if there are bad polygons (such as polygons with lamina faces, non-manifold geometry, or faces with a zero map area). The situation might improve if you perform mesh cleanup to remove bad polygons before you export.

Exporting the model takes an unusually long time.

Exporting can take a long time if there are bad polygons (such as polygons with lamina faces, non-manifold geometry, or faces with a zero map area). The situation might improve if you perform mesh cleanup to remove bad polygons before you export.

Exporting Animations

Simple animation curves get baked even though Bake All Animation is not enabled.

Simple animation curves are baked and then exported if the animation cannot be expressed in an intermediate file due to factors like the parent-child hierarchy. For specific information about the bake conditions, see the pages for the various kinds of animations in Animation.

Exported animations are choppy.

If an animation is baked when it is exported, problems with errors during baking might lead to choppy playback. You can improve this situation in some cases by changing the value of Frame Precision in Animation Options, or by adjusting the error tolerance for Node Translate and Node Rotate in Animation Bake Tolerance Options.
