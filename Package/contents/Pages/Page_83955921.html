Audio News

NintendoWare Audio News

Introduces features that were added and changed in NintendoSDK 0.5.0.

Multi-Platform Support

Multi-platform support starts with this version of the audio module.
Until now, there was a "Cafe version" and a "Cafe version running on Windows." The latter version has been changed to a "Windows version" that is not dependent on Cafe or any other particular platform.

Info

Multi-platform support for NintendoWare Audio is in progress.
The specifications may change in subsequent versions.

Changes to the Binaries

No changes for 0.5.0.

Changes to Sound Project, Sound Set, and Bank Files

No changes for 0.5.0.

Changes to the Converter

No changes for 0.5.0.

Changes to SoundMaker

Added a combo box to SoundMaker for specifying the platform.

Use this combo box to specify which platform you want to connect to.
SoundMaker uses the specified platform to determine whether to connect to a Cafe version application on the CAT-DEV or to a Windows version application.

Info

Note: This has no effect on the stored intermediate files.
Note: In 0.5.0, this also has no effect on the output binaries, so you can load both the Cafe and Windows versions of the application.
　(Neither impacts the output binaries.)
