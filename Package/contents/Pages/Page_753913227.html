Profiler-Start

Overview

Starts taking a profile of the specified type.

Profiler-Start [-ProfileType] <ScriptingProfileType>


[-ProfileType] <ScriptingProfileType>

The different types of profiles that can be started.

    • Sampled: Equivalent to starting a profile from the Sampled Profile Tab.
    • Instrumented: Equivalent to starting a profile from the Instrumented Profile Tab.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Start Sampled
Profiler-Start -ProfileType Instrumented
