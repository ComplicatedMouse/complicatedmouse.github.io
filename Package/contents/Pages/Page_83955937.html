DCC Priority Mode

DCC Priority Mode

The DCC tool export plug-in may be set to DCC tool priority or 3DEditor priority.

Info

For more information about the DCC tool export plug-in sampler merge settings, see Export Plug-In > Configuration File > sampler_merge_priority in the DccPlugin documentation.

When the sampler merge setting is enabled (DCC tool priority), it is possible to change the Material Property Window Sampler page UI specifications in 3DEditor.

This section describes the UI change operations when sampler merging is enabled.

Changing the Sampler Merging UI

There are two methods for changing the UI specifications of the Sampler page: changing by using the tool team settings or changing by using the team settings file.

Changing by Using the Tool Team Settings

From the main menu of 3DEditor, open Team Settings and select Preset settings.

Click the Switch to Edit Mode button, and change the Use sampler names that conform to the DCC tool's naming conventions option to Used. (3DEditor must be restarted.)
The Team Settings File will be updated to the DCC Priority Mode.

Changing by Using the Team Settings File

To change the UI for the sampler page in the material Property window, go to the team settings file (3dEditor.teamsettings) and set either true or false for <FollowDccSamplerNameRule> in <Preset>.

3dEditor.teamsettings is found in the following location.

(NintendoSDK_ installation_folder)\NintendoSDK\Tools\Graphics\3dEditor\3dEditor.teamsettings

Info

The default setting of <FollowDccSamplerNameRule> in <Preset> of the team settings file (3dEditor.teamsettings) is false.
Change <FollowDccSamplerNameRule> to true to change the sampler merge setting of the DCC tool plug-in to DCC tool priority.

For more information about the <Preset> element in 3dEditor.teamsettings, see Team Settings File Format.

true Use the UI for enabled sampler merging.
false Use the original UI design.

Adding a New Sampler and Changing a Sampler Name

This section describes how to add new samplers and enter names for the samplers when sampler merging is enabled.

A

Adds a new sampler.

Click the Add button to add a new sampler. The default name for the first new sampler is sampler0. As you continue adding samplers, they take default names such as sampler1, sampler2, and so on.

No samplers with names starting with "_" (underscore) (such as _a1 or _a2) will be added.
B

Enter any sampler name or set from the combo box.　
C

Clear sampler edit restrictions.

Select this button to enable the adding of samplers with names that start with "_" (underscore), such as _a1 or _a2.

If this button is enabled, you can deassign texture from the right-click shortcut menu for the selected sampler.

Entering and Changing the Sampler Name

This section describes how to enter or change the name of a sampler.

Entering Text Freely (User Can Specify Any Sampler Name)

• You can freely enter sampler names. However, you cannot enter sampler names that start with an underscore (_). (Example: _a0)
• To enter sampler names that start with an underscore (_), select the Clear sampler edit restrictions button.
• If you change the name of a sampler exported from the DCC tool that already has a name that starts with "_" (such as _a0), and also has hint information, the hint information is cleared and the underscore is removed from the sampler name.

(For example, if a sampler contained hint = albedo0 and name = _a0, changing the sampler name to a0 would remove the hint and change the sampler name to a0.)

Selecting a Sampler Name From a Combo Box

You can set a list of preset sampler names to the combo box and then display them.

Info

Set any sampler names that you want to display in the combo box by editing the <SamplerNamePresets> element within <Preset> in the team settings file (3dEditor.teamsettings).

Related Links For more information about the <Preset> element in 3dEditor.teamsettings, see here.

Deassigning Textures

If the Clear sampler edit restrictions button is selected, you can Deassign texture from the right-click shortcut menu of the selected sampler.

Displaying Samplers Configured and Synchronized With the DCC Tool

A small DCC mark is displayed by the sampler's name. The mark appears in the sampler list for samplers that were configured and synchronized with the DCC tool.


Info

• For samplers synchronized with the DCC tool, you can only edit the sampler parameters.
• Samplers synchronized with the DCC tool have names that start with "_" (underscore).

Copying and Pasting When Sampler Merging Is Enabled

Sampler assignment when copying and pasting samplers operates the same as when importing materials.

Copying and Pasting Samplers Synchronized With the DCC Tool (Samplers With Names Starting With "_" and Hint Information)

Example

Copying and pasting a sampler with name a0, and hint albedo0, produces the following result.

The underscore at the front of the sampler's name is automatically removed, and the hint information is cleared. This copy of the sampler is also no longer synchronized with the DCC tool.

If removing the underscore from the sampler's name causes it to have the same name as another existing sampler, the sampler is renumbered accordingly.
In this example, if there was already a sampler named a0, the copied sampler would be named a1 after pasting.

Info

If the list you are pasting the sampler to already has a sampler with the same name and hint information, only the sampler parameters of the existing sampler in the list will be overwritten.

Copying and Pasting Samplers Not Synchronized With the DCC Tool

  • The sampler name and hint information are retained when the sampler is copied and pasted.
  • If a sampler with the same name is already in the list, both the texture data and sampler parameters (wrap, filter, and lod) of that existing shader are overwritten.
  • If the hint information for the new sampler pasted into the list matches the hint information of an existing sampler, the hint information of the new sampler is cleared.
    Note: The hint information of existing samplers is prioritized.

Important Information When Sampler Merging Is Enabled (DCC Tool Is Prioritized)

The shader parameter for a texture SRT, for which the original hint has not been cleared, may not be modified when the sampler merge setting is enabled.
