friends Library Features

The friends library offers functionality for managing the relationships among various users.

  • Friend List
    • The list that manages friends that play together online.
    • When there is a friend relationship, you can share a user's play status, or presence, with those friends.
  • Blocked-User List
    • The list that manages users that you met during online play but do not want to play with, or users that sent you a friend request that you blocked.
  • Users Played With
    • Manages the users met during online play that you then played with.

Network Service Accounts are used as user identifiers.

Provided Features

The friends library provides the following features.

  • Getting the friend list
  • Getting the blocked-user list
  • Updating presence
  • Getting any user's user profile (nickname and profile image URL)
  • Adding users played with

Creating a Friendship

Friendships are created on My Page.
Use the My Page applet to create a friendship, or a friend application, from within an application.

Restrictions

The following restrictions exist for the friends library.

Using in the Windows Environment

The library is only implemented as a stub in the Windows environment.
You can embed the friends library and build, but none of the actual functionality will be implemented.
