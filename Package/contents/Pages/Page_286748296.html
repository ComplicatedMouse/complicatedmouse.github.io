Feature for Placing Text in a Circle

This feature lets you place text in a circle, as shown in the example below.
You can also use this feature to place text in an arc or around an ellipse.

Making the Settings in LayoutEditor

Place a Text Box

Create a text box pane.

Enable Character-Based Transformations

Use the following procedure to display the Place Text in a Circle dialog box.

1

Select Text Box Pane () in the Hierarchy window.
2

Select Text Box Pane () in the property window.
3.

Select the Character-Based Transformations check box ().
4 Click Place Text in a Circle to open the dialog box.

Configure the Place Text in a Circle Settings

The following image shows the result of configuring the settings with the example values.

Starting angle 180 Specifies the starting angle of the arc.
The value 0 defines the right side, and values move counterclockwise around the circle in 360 units.
The value specified in this example, 180, defines the character on the far-left side of the arc.

Clicking OK after configuring the settings closes the dialog and applies the results.
Ending angle 360

Specifies the ending angle of the arc.
The value specified in this example, 360, defines the character on the far-right side of the arc.
Circle radius X 100

Specifies the radius of the circle in the horizontal direction.
Circle radius Y 100 Specifies the radius of the circle in the vertical direction.
Number of divisions 10 Specifies the number of keys when the circle is expressed as a curve.
More keys make for a smoother circle, but can impact performance.
Character orientation 90 Specifies the angle relative to the slope for each character placed along the circle.
If you specify 90, each character will rotate 90 degrees counterclockwise to orient toward the center of the circle.

The following image depicts the result when Ending angle is set to a value of 0 and Character orientation is set to a value of 270. The characters orient to the outside of the circle.
