Additional Reference Path

Overview

Additional reference paths are used when searching for reference files specified in the File I/O Settings file in team settings or in models or material animations.
Additional reference paths for which an absolute path is specified start from the startup path or from the path for files included in the project.

Info

When models or material animations are opened, the system searches for the referenced texture files.
If the texture file first found by the search is not opened, the texture file is opened.

If the shader definition file with the same name as the shader definition that is referenced when opening the model is not opened in 3DEditor, a search for the shader definition file is started.

In addition, you can set the material additional reference path to use for searching for parent materials when resolving material references.

Info

Searches for parent materials in the material reference.

The materials inside the folder for additional reference paths can be selected from Additional search path for this file. when adding material references.

Scope and Order of Search

The scope and order of the search is as follows.

Search Order Search Range Specified Location for Reference Paths
1 Folders containing the reference source file.　
2

The folder for the additional reference path set for the model intermediate files and material intermediate files of the reference source model.

Folders with the additional reference paths specified for the model intermediate files in which the reference source material animation intermediate files are bound.

The additional reference path for each type of intermediate file.

(Can be assigned using the assign utility.)

3

Folders and subfolders with the additional reference paths specified for Team Settings and User Settings.

Options for Team Settings, User Settings, and command-line tool
4

The textures folder (only if the reference target is a texture) or Materials folder (only if the reference target is a material) in the folder included in the reference source file.


5

The textures folder in folders that contain model intermediate files in which the reference source material animation intermediate files are bound (only when the referenced file is a texture).
6 Folders within the search range for the material intermediate files of the parent material.

When you specify an additional reference path as part of a search, the search range includes 2 and 3. 3DEditor searches 1, 4, and 5 automatically. The reference source files are the files opened in 3DEditor.
Wildcards (*, ?) can be specified in additional reference paths.
When subfolders are searched or a wildcard is specified, and multiple files have the same name, the paths are sorted alphabetically (case-insensitive) and the first file in the sorted list is applied.
If a material reference is also set for the parent material, the first file found in the search range by recursively traversing the parent material is applied.



Setting Multiple Additional Reference Paths

If more than one additional reference path has been set, the search is carried out in the order they were set.
You can set whether to include the subfolders in the folder specified in the additional reference path separately for each additional reference path.

An error does not result even if a reference file cannot ultimately be found.

Info

Specify a directory path for the additional reference path.
You can set an absolute path or a relative path from the reference source file.
Path strings may include environment variables surrounded by percent signs (%). 3DEditor interprets such paths by replacing any environment variables surrounded by percent signs (%) with the value the environment variable represents.

Specifying Absolute Paths in the Settings File

  • Make sure that you set an additional reference path name.
  • Additional reference path names are enabled only for additional reference paths set using an absolute path.
  • Each additional reference path name must be unique.

You can specify additional reference paths for intermediate files either with the assign utility, or in 3DEditor on the Additional Reference Path page in the property window.

When specifying additional reference paths for intermediate files as absolute paths, specify the path starting from the appropriate environment variable, which is enclosed in percent signs (%).
