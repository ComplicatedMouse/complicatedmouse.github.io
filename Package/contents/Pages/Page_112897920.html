Scale Fade

This section describes the Scale Fade feature for making emitted particles gradually increase in size.

Overview

Scale Fade is a feature that makes all particles begin with a scale of zero when they are emitted from the emitter and then grow larger.
The particles gradually appear over the course of the configured Fade-In Duration.
The following overview describes the settings ().

Scale Fade

The Scale Fade feature is enabled or disabled using the check box ().
