Common Settings in User Settings and Project Settings

The table summarizes how behavior differs depending on what is configured for the settings that are present in both the User Settings and the Project Settings.

User Settings Project Settings Behavior
Default Default The file in the User Settings default path has display priority.
Default User-specified The file in the path specified in the Project Settings has display priority.
User-specified Default The file in the path specified in the User Settings has display priority.
User-specified User-specified The file in the path specified in the User Settings has display priority.
