Connection Type

This section describes the Connection type feature, which specifies how the leading ends of stripe particles are connected.

Overview

The Connection type feature specifies whether the leading ends of stripe particles should be connected to the trailing ends of the stripe particle or to the emitter.
The following image illustrates the leading and trailing ends of a stripe particle when particles that are emitted in order, starting from the first particle, are connected together.

A The trailing end of the stripe. B The leading end of the stripe.

The following overview describes the settings ().

Connection Type

Select the type of connection between polygons from among the following presets.

Normal For normal stripes. At least two particles are required to calculate the stripe particle.
Connect to trailing end Connects the leading end and trailing end of the stripe in a ring shape. At least three particles are required to create a ring-shaped stripe particle.
Connect to emitter Connects the leading end of the stripe to the center of the emitter. At least two particles are required to calculate the stripe particle.
This behavior can be used to create smoke-like effects in which particles are continuously emitted from a single location.
