Description of Features

This section describes the various rendering-related features provided by the Face Library.

List of Features

Feature Overview
Resource
The class with features for managing the graphics resources used for rendering Mii face models.

CharModel

The class with features for managing the shapes, textures, and other data of the parts that comprise a Mii face model.
Faceline The class with features for managing the shapes, textures, and other data of the parts that comprise a face outline texture.
Mask The class with features for managing the shapes, textures, and other data of the parts that comprise a mask texture.
DrawParam The class with features for managing the parameters required for rendering individual parts.
TextureShader The class with features for rendering face outline textures and mask textures.
IconBody
The class with features for managing the body shape data for rendering a Mii icon with a body.
IconShader The class with features for rendering the Mii icon texture.
Material The class with features for managing the materials for an individual Mii character.

VariableIconBody

The class with features for managing the body shape data for rendering a Mii icon with a certain physique.

VariableIconBodyShader

The class with features for rendering the icon texture for a Mii icon with a body with a certain physique.
TransparentAdjuster The class with features to prevent bleeding of transparent pixel color when performing linear interpolation.
