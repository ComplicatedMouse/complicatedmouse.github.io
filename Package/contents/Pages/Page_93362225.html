Display Face

This section describes the feature for determining which faces of the polygon to show particles on.

Overview

The Display Face feature allows you to select whether to display particles on the front, back, or both faces of the polygon.
For primitive particles and other non-planar polygons, you can reduce the rendering load by selecting to not display particles on faces that cannot be seen.
The following overview describes the settings ().

Display Face

Select the particle display surface setting from among the following presets.

Both Sides Displays both sides of polygons.
Front Side Displays the front side of polygons.
Back Side Displays the back side of polygons.
