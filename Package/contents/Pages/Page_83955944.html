Color Target View

Color Target View

Overview

Color target view provides features for setting textures as color targets.
The properties of this view are basically the same as for texture view. See texture view.

The color target view expresses a portion of the source texture with an array range and a specific mipmap level.

Information

Use nn::gfx::ColorTargetViewInfo to define the color target view information.

Dimension, Format, Texture Pointer

See texture view, as the settings are almost the same.

The source texture format must be usable as a color target.

The source texture must be initialized in a way that includes GpuAccessFlag_ColorTarget as a method for GPU access.

Sub Resource Range

The array range is specified by the EditArrayRange function, and the specific mipmap level is specified by SetMipLevel.

Object

Use nn::gfx::ColorTargetView to define the color target view object.

Initialization and Destruction

Texture views are initialized and finalized according to the rules for objects.

Setting the Color Target View

You can set a color target view as a color target by specifying it along with a depth stencil view using the SetRenderTargets function of the nn::gfx::CommandBuffer class. At this time, the color target view is set as an array, and each is analyzed as a multi-render target according to the order.
The format of each color target to set must match the color target format of the render target state.

Clearing the Color Target View

Although color target view can freely write through the pixel shader, a clear function is provided to write the same value uniformly to all pixels of a color target.

Use nn::gfx::CommandBuffer::ClearColorTarget to clear color targets in floating point, signed integer, and unsigned integer formats, or nn::gfx::CommandBuffer::ClearColor to clear color targets in only floating-point format. Specify the values to clear for each of the red, green, blue, and alpha elements for all pixels.
At this time, you can also clear only a specific array range in the color target by specifying the desired range in pArrayRange. When NULL is specified for this parameter, the entire array range is cleared.

Restrictions on Clear

The render target settings are disabled after a clear. Set the render target again, if needed.

Resolving Multi-Sample Images

If the color target view is a multi-sample color target, you can resolve it to a single sample texture using the Resolve function of the nn::gfx::CommandBuffer class.

At this time, you can also resolve only a specific array range in the color target by specifying the desired range in pSrcArrayRange. When NULL is specified for this parameter, the entire array range is resolved.
