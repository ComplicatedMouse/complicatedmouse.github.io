All-Direction Initial Velocity

This section describes the All-Direction Initial Velocity feature that sets the particle initial velocity in the emitter normal direction.

Overview

The All-Direction Initial Velocity feature disperses particles away from or collects particles toward the emitter.
Particles may be emitted in the normal direction by using the primitive emitter.
This feature is useful, for example, when you want to emit particles in the normal direction of a game character.
The following overview describes the settings ().

All-Direction Initial Velocity

Sets the initial velocity at which particles disperse outward from the emitter.

To set a negative value, enable the +/- value button ().
When a negative value is set, it will emit particles inward from the emitter.
