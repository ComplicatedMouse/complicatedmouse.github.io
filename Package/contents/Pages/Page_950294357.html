NintendoSDK Support in Public Repositories

Including implementation details that support the NintendoSDK in public source code repositories is prohibited.

Including the names of APIs that are provided by the NintendoSDK in public repositories is also prohibited.
However, the following strings can be included since they are public information.
For example, using the following strings as names for settings or preprocessor macros in the source code of a public repository is allowed.

NintendoSDK Common

  • NintendoSDK

NintendoSDK NX Add-On

  • NX
  • NX32
  • NX64
