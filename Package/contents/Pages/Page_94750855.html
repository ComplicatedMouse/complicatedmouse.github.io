Emission Start Frame

This section describes the Emission start frame feature that sets the emission timing.

Overview

Emission start frame specifies the timing of when the emitter begins to emit particles.
When multiple emitters are combined in an effect, each emitter can be timed to start emitting particles at different times.
For example, with fireworks, the started emitter emits the fireball, and the firework explosion may be emitted when the emitter is at zero velocity at the top.
