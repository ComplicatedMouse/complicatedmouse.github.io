NintendoWare 3D News

This page introduces features that were added to the 3D module in NintendoSDK 0.7.0.

Shader Converter, Binary Converter

  1. Binary Export Support for nn::g3d

Binary Export Support for nn::g3d

To output binary for nw::g3d, first run %NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\SwitchToNnBinaryMode.bat, and then use the shader converter and binary converter.

To output binary for nw::g3d, first run %NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\SwitchToNwBinaryMode.bat, and then use the shader converter and binary converter.

When using 3DEditor, the mode must be changed depending on if the destination application is implemented using nn::g3d or nw::g3d.

Note that G3dDemo may not run properly without rebuilding after changing the mode.

3DEditor

This section describes the new features and the latest information about 3DEditor.

  1. Changed the functionality to enable editing of scene animation settings in the General page of the Property window
  2. Changed the functionality to enable the number of frames and looping to be configured in the General page of the animation node.

Changed the functionality to enable editing of scene animation settings in the General page of the Property window

You can now set fixed values for camera, light, and fog in the scene animation Property window, on the General page.

Info

Properties to which animation has been set are dimmed and cannot be edited.

Camera Animation

For camera animations, properties that cannot be used with rotation mode or projection mode are disabled.

Light Animation

If all parameters are set to fixed values, the light animation Property window opens by default on the General page.

Changing the light type changes the instruction parameters.

Fog Animation

If all parameters are set to fixed values, the fog animation Property window opens by default on the General page.

Changed the functionality to enable the number of frames and looping to be configured in the General page of the animation node.

Changed the functionality to enable the number of frames and looping to be configured in the General page of the animation node.

The number of frames and the loop playback settings can be changed in Curve Editor Animation Settings.
