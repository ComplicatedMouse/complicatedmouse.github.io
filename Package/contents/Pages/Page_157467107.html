Material

nn::mii::Material is the class for managing materials for an individual Mii character.

When you want to display a 3D Mii model in the same manner as with the Create or Edit Mii applet, get and render the material for each part from this class.

Initialization and Finalization

Call the nn::mii::Material::Initialize() function to perform initialization. Set the material for each part based on nn::mii::CharInfo passed during initialization. The initialized content cannot be changed until you call the nn::mii::Material::Finalize() function and perform termination processing.

Getting Parts Materials

Use the nn::mii::Material::Get() function to get the specified nn::mii::CharModel::DrawType material. Alternatively, use the nn::mii::Material::GetBody() and nn::mii::Material::GetPants() functions to get the body and pants materials.

Use each Get function to get nn::mii::DrawParamMaterial, which contains individual DrawParam materials. For information, see the API reference.
