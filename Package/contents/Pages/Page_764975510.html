Profiler-Timeline

Overview

Controls the Timeline options.

Sample Display

Control how samples are displayed.

Profiler-Timeline -SampleDisplay <SampleDisplay>


-SampleDisplay <SampleDisplay>

Controls how Callstack samples are displayed.

    • Self
    • Total
    • Mixed

Name Display

Control how names are displayed.

Profiler-Timeline -NameDisplay <FunctionNameDisplay>


-NameDisplay <FunctionNameDisplay>

Controls how function and counter names are displayed in the Timeline.

    • Hide
    • Short
    • Long
    • Full

Show Stack Graph

Controls whether the Stack Graph is visible or not.

Profiler-Timeline -ShowStackGraph <Boolean>


-ShowStackGraph <Boolean>

Sets the Stack Graph visibility to the specified Boolean value.

Show Icicle Graph

Controls whether the Icicle Graph is visible or not.

Profiler-Timeline -ShowIcicleGraph <Boolean>


-ShowIcicleGraph <Boolean>

Sets the Icicle Graph visibility to the specified Boolean value.

Show Heatmap

Controls whether Heatmap mode is enabled in the Timeline.

Profiler-Timeline -ShowHeatmap <Boolean>


-ShowHeatmap <Boolean>

Sets the Heatmap mode based on the provided Boolean value.

Deselect All Frames

Deselects all selected frames.

Profiler-Timeline -DeselectAllFrames


-DeselectAllFrames

Forces the Timeline to deselect all selected frames. It is also possible to perform this operation using the command "Profiler-Select -Tab Timeline -DeselectAll".

Show Combined

Note

This option has been deprecated.

Control whether the combined line should be visible.

Profiler-Timeline -ShowCombined <CombinedLineDisplay>


-ShowCombined <CombinedLineDisplay>

Controls how the Timeline shows the combined line.

    • Hide
    • Show
    • Only

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Timeline -SampleDisplay Mixed
Profiler-Timeline -NameDisplay Short
Profiler-Timeline -ShowStackGraph $false
Profiler-Timeline -ShowIcicleGraph $true
Profiler-Timeline -ShowHeatmap $false
Profiler-Timeline -DeselectAllFrames
