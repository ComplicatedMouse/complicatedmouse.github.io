Color Editing UI

Color Editing UI

Configures the selected color and alpha values.

A Time Set the timing of the animation as a percentage of the particle life span.
B Color saturation and brightness Changes the saturation and brightness value of the color.
C Hue Changes the hue of the color.
D Color space

Select either RGB, sRGB, or HSV as the color component display format.
Only the display format changes. The color itself is not affected.

  • RGB format: Brightness can be adjusted in linear space.
  • sRGB format: Brightness can be adjusted in non-linear space.
  • HSV format: Hue, saturation, and brightness can be adjusted.

Related Links For more information about linear space, see here.
E Color and alpha settings Sets the color and alpha values. You can set values of 1 or higher.
Info

Setting RGB values greater than 1.0 makes it possible to create glare effects in EffectViewer.
