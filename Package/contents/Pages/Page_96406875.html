Get Square of RGB

This section describes the Get Square of RGB feature for squaring the registered texture.

Overview

Get Square of RGB squares the brightness of the registered texture to approximate the result of linear conversion.
When the registered texture is a grayscale image with alpha, using this feature has the following benefits.

You can expect a smaller texture size.

If the same image is used for color and alpha, color will be held in linear and alpha will be held in nonlinear, so the texture requires two channels worth of information.

This feature converts the RGB color to linear color in the effect shader, so just one channel of information is enough for color and alpha.
