Shape

Shape

The Shape page is used for setting the shape and effects for a picture pane.

Enable Rounded Corners

Sets whether to add rounded corners to modify the shape of the corner parts of the pane.
When the check box is selected, the following Rounded Corners settings are displayed.

Set Four Corners Separately

Selects whether to set the Rounded Corners settings of Shape and Rounded Corner Radius separately or as a group for the four corners of the pane.
When the check box is selected, the Rounded Corners settings are displayed for each of the four corners.

Rounded Corners

Sets how the rounded corners are rendered.

Shape

Sets the shape of the corner parts.

Rounded Corner Radius

Sets the radius of the corner parts.

Enable Outlines

Sets whether to add an outline to the pane.

When the check box is selected, the following Borders settings are displayed.

Borders

Sets how the outline is drawn.

Size

Sets the size of the border.

Rendering Mode

Sets the rendering mode type.

  • Normal (default)
  • Multiplicative
  • Addition
  • Subtractive

Color

Sets the color and alpha of the border.

Enable Shadows (Interior)

Sets whether to add a shadow inside the pane.

When the check box is selected, the following Shadow (Interior) settings are displayed.

Shadow (Inside)

Sets how the shadow (interior) is rendered.

Rendering Mode

Sets the rendering mode type.

  • Normal
  • Multiplicative (Default)
  • Additive
  • Subtractive

Color

Sets the color and alpha of the shadow dropped from the pane.
Angle

Sets the angle at which the shadow is dropped.
Distance Sets the distance between the pane and the shadow.
Size Sets the size of the shadow.
Shadow Type

Sets the type of shadow.

  • Weak
  • Medium (default)
  • Strong

Enable Color Overlay

Sets whether to add a color overlay to the pane.

When the check box is selected, the following Color Overlay settings are displayed.

Color Overlay

Sets how the color overlay is rendered.

Rendering Mode

Sets the rendering mode type.

  • Normal (default)
  • Multiplicative
  • Addition
  • Subtractive

Color

Sets the color and alpha of the color overlay.

Enable Gradation Overlay

Sets whether to merge gradation with the pane.
When the check box is selected, the following gradation overlay settings are displayed.

Gradation Overlay

Sets how the gradation overlay is rendered.

Rendering Mode

Sets the rendering mode type.

  • Normal (default)
  • Multiplicative
  • Addition
  • Subtractive
Number of Control Points

Sets the number of colors that constitute the gradation.

Use control points to specify the positions of the colors in the gradation.
Gradation Setting Sets the color and alpha of gradation, and the control point adjustment points.
Angle Sets the direction in which gradation is applied to the pane.

Enable Drop Shadows

Sets whether to add drop shadows to the pane.

The following drop shadows settings are displayed when the check box is selected.

Drop Shadows

Sets how the drop shadow is rendered.

Rendering Mode

Sets the rendering mode type.

  • Normal
  • Multiplicative (Default)
  • Additive
  • Subtractive

Color

Sets the color and alpha of the shadow dropped from the pane.
Angle

Sets the angle at which the shadow is dropped.
Distance Sets the distance between the pane and the shadow.
Size Sets the size of the shadow.
Shadow Type

Sets the type of shadow.

  • Weak
  • Medium (default)
  • Strong
