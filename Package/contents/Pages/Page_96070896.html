Random Width

This section describes the Random width feature of the Random field for moving particles around randomly.

Overview

The Random width feature moves particles in a random direction for the specified axis.
Each time randomness is applied, a value to add to the value range specified for the particle speed is determined.

The following overview describes the settings ().

Random Width

The random width value is set using x-, y-, and z-coordinates with the emitter as the origin, using either the dials () or by entering values in the text boxes ().
The following equation derives the value added to the velocity of particles as a result of the random field.

  • Value to add to particle speed: Value set using random seed × Random width
Info

Press the Particle Time Animation button () to display the Random (particle time animation) group () and configure the animation in Curve Editor.
Animations can be enabled by selecting Run on CPU ().
