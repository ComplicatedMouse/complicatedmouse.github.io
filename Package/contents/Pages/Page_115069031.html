NintendoWare 3D News

This page introduces features that were added to the 3D module in NintendoSDK 0.10.0.

Maya Plug-In

This section describes the features newly added to the Maya plug-in.

Feature for a Custom User Interface to the Multi-Export Plug-In Window

Users can now customize the multi-export dialog box.
This allows the multi-export dialog box to be customized by adding a submit button for the version management system.

Feature for Adjusting the Width of the Output Item List Column to the Multi-Export Plug-In

The width of the output item list in the multi-export dialog box was fixed and some items would not fit, but now the width can be changed in the UI.
Use the shortcut menu of Column Width and the slider in the upper-right corner of the window to adjust the width for each item.

Displays a Detailed Description When an Export Plug-In Error (Warning) Occurs

When using Maya in a Japanese UI, the following types of descriptions now appear in Japanese.

Error Display Example in the English UI Error Display Example in the Japanese UI
Make sure that you clear the In-Between option when creating blend shapes Make sure that you clear the In-Between option when creating blend shapes.

3DEditor

This section describes the newly added UI elements of 3DEditor.

Improvements to the Key Display in Curve Editor

Improved the display to make the key status easier to see in Curve Editor.
Descriptions of the improvements follow.

Key Color Is the Same Color as the Curve

As shown in the following image, the key is now the curve color.
This makes it easy to see at a glance which curve the displayed key is for.

Highlighting When Selecting a Key

As shown in the following image, when you select a key, the key is displayed in white.
This makes it easy to tell a key's selected state.

Highlighting Nearby Keys

As shown in the following image, when multiple keys are nearby, the keys are circled.
This makes it easier to find the mistake of multiple keys overlapping at the same time.

Enable Switching the float3 and float4 Controls to Controls for Color in the Shader Page of the Material Property Window

The float3 and float4 controls, in which three or four values are lined up in the material settings, can now be switched between value control and color control.
This change also enables you to use a UI to intuitively specify colors with the Color Picker.
As shown in the following image, you can right-click the label, select Display color control from the shortcut menu, and switch the control.

Info

You must set items to auto in the shader annotation in advance.

Related Links For more information, see the following pages.

Sample Viewer for the Tutorial

A sample viewer for the tutorial is now included in the package.

Previously, you had to prepare the viewer environment to preview models and animations, and it was difficult for designers to test them. Now you can use the sample viewer to easily begin tutorials.

Connect the sample viewer to the 3DEditor to see model and animation previews.

Related Links For more information, see the following pages.

Intermediate File

Notice of changes to intermediate file specifications.

Support for the Material Animation Intermediate File Format

Specifications for the material animation intermediate file format (FMA) have been added.
The FSP (FCL, FTS), FTP, and FVM formats will be deprecated in the future. It is necessary to convert these formats to the FMA format.
