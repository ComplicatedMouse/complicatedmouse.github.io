Overview

  • NintendoWare AtkPlayer is an application tool for previewing data created with NintendoWare SoundMaker.
    • For more information about SoundMaker, see the SoundMaker Manual.

How to Start

This section describes how to start AtkPlayer.

Starting From SoundMaker

  • From SoundMaker, select Preview > Start PC AtkPlayer.
  • From the toolbar, click the Start PC AtkPlayer icon.
  • When you run Start PC AtkPlayer, processing is performed in the following order.
    1. Sound data is converted.
    2. AtkPlayer is started.

SoundSystem Panel

The panel for selecting surround, stereo, or mono as the mode for main output.

Effect Panel

This panel applies effects to the Aux bus.

The effect settings for each path displayed on the upper level is called an effect list. It is possible to select the type and order of effects applied to each bus. Using the preceding image as an example, apply the Delay effect in AuxB, and then apply the Reverb effect. The names and supported types of the displayed effects are described in the following table.

Effect Name Effect Type
None

Do not apply any effects.
Reverb

Apply the nn::atk::EffectReverb effect.
Delay

Apply the nn::atk::EffectDelay effect.
I3dl2

Apply the nn::atk::EffectI3dl2Reverb effect.

The list of effect parameters displayed on the lower level is called the parameter list. You can select effect parameters from the list and adjust them. In addition, the Memory displayed between the effect list and parameter list indicates the total size of the work buffer of the currently applied effects.

SoundOutput Panel

This panel displays the sound output waveform in real time.

Output for 6 channels is displayed. The channels are displayed in the following order from top to bottom: FL, FR, FC, LFE, RL, and RR.

Settings similar to the following settings are displayed at the very bottom.

Settings Description
[Play]

Begins monitoring the output waveform.
[Stop] Ends monitoring of the output waveform.
Time Indicates the display time of the output waveform.
Amp Indicates the display magnification of the amplitude of the output waveform.


Reloading Sound Archive Files

When editing sound data with SoundMaker after starting AtkPlayer, the edited content is not automatically applied to AtkPlayer.

To have it applied, you must reload the sound archive file.


After editing the sound data, the sound archive file is converted from SoundMaker.


After conversion is finished, you can reload the file in AtkPlayer by pressing the R + X buttons.

At this time, "Reloaded" is displayed next to "Edit" in the upper right of AtkPlayer's screen.

In-Game Editing Feature

AtkPlayer can use SoundMaker's in-game editing feature.

For more information about SoundMaker's in-game editing feature, see In-Game Editing Feature.

Connecting With SoundMaker

After starting AtkPlayer, a connection is made from SoundMaker.

When the connection is established, the color of "Edit" in the upper right of the AtkPlayer screen changes.

While the connection is established, in-game editing can be performed from SoundMaker.

Disconnecting With SoundMaker

When you exit in-game editing, SoundMaker disconnects.
When you start in-game editing again, a connection is made from SoundMaker.

Visualizing the Information for Playing Sounds

By connecting AtkPlayer to the PC tool Spy.exe, you can visualize various kinds of playback information for sounds, including the processing load of the audio renderer and the number of voices being used.

To use this feature, start AtkPlayer first and then connect to it from Spy.exe. For more information about using Spy.exe and the playback information it processes, see Spy.exe.

Info

Important Information

Spy.exe does not support changes to sounds after it has connected to them. As a result, inconsistencies may emerge in the information being displayed as a result of in-game editing or the reloading of sound archives. When this happens, disconnect AtkPlayer from Spy.exe, and then reconnect them to resolve the inconsistencies in the information.

Troubleshooting

When AtkPlayer started, I got the error message "Failed to initialize SoundSystem."

  • This error is displayed when the nn::atk::SoundSystem initialization fails. Verify whether the audio playback environment is appropriately prepared. For example, this error is displayed when no audio playback devices such as speakers or headphones are connected.
