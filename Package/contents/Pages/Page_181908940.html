Image height (={0}) is too small. Must be at least {1} pixels or greater (7×number of blocks).

Adjust the height of the conversion source image to be {1} pixels or greater.

The height of the source image for conversion is not the prescribed height.

The value, exclusive of the grid from a width per one block, the margin, and the width area (totaling six pixels), is at or below zero pixels.


The following example uses a height of 84 pixels and a vertical column of 14 blocks.

Divide the height of the source image for conversion, 84 pixels, by the number of vertical blocks, 14, to get a height of six pixels per one block.

The image is not converted because the value for the six-pixel block width is zero pixels after deducting the grid, the margin, and the width area, which totals six pixels.
Use the following formula to configure a height for the source image for conversion so that the value for a single block is not zero pixels or less after deducting the grid, the margin, and the width area, or a total of six pixels, from the block height.


(6 px + ≥ 1 px) × number of blocks in vertical direction

  • Six pixels is the total for the grid, the margin, and the width area.
  • At least one pixel is the portion needed for conversion.


The following example uses a height of 98 pixels and a vertical column of 14 blocks.

Divide the height of the source image for conversion, 98 pixels, by the number of vertical blocks, 14, to get a height of seven pixels per one block.

The image is converted because the value for the seven-pixel block width is one pixel after deducting the grid, the margin, and the width area, which totals six pixels.
