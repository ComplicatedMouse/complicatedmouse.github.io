Gfx Groups Tab

The Gfx Groups tab displays all of the graphics groups that were recorded during a profile. This includes user graphics groups and a group used by the profiler (domain: 1314407248). This tab will only appear for profiles where GPU Trace was enabled on the Sampled Profile or Instrumented Profile tabs.

Gfx Groups Table

The Domain column contains the ID as defined when the graphics group was setup.

The Name column contains the string as defined when the graphics group was setup.

Clicking on a row will select the group and draw it in the Timeline tab. Note that multiple durations can overlap in the same graphed row, which is depicted by darker colors.

Gfx Groups Tab Toolbar

The following controls are in the Gfx Groups tab toolbar.

Deselect All Button

To quickly deselect all selected items, click on the Deselect All button.

Select Top Button

Selects the top 50 items.

Copy Button

The Copy button will copy to the clipboard whatever is currently shown using the current sorting and filtering. You can then paste the content into an email, document, or Excel spreadsheet. Because the copied content uses tab delimiters, it will retain the column organization when pasted into an Excel spreadsheet.

Selected Only Button

The Selected Only button forces the list display to only show items which are currently selected. If this mode is enabled and an item is deselected, it will be removed from the displayed list.

Filtering Gfx Groups

Gfx Groups can be filtered using the Filter text box at the bottom of the Gfx Groups tab. This behaves similar to the Filter text box on the Functions tab.
