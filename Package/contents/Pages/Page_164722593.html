Added Compare (Darken) and Compare (Lighten) to the Color Blending Settings in the Texture Combiner.

Two new material blends, Blend (Darken) and Blend (Lighten) have been added, expanding the range of expression.

Blend (Darken) and Blend (Lighten) in the Properties Window

Blend (Darken)

The Blend (Darken) setting compares two overlaid colors, the base color and the composite color, and applies the darker color.

  • When the base color is darker than the composite color, the base color remains as is.
  • When the base color is lighter than the composite color, the color is replaced.
Example of Using Blend (Darken)
A Texture 0 image Set an all-yellow image for Texture 0. (Base color)
B Texture 1 image Set an image of a white to black gradation for Texture 1, and then apply Blend (Darken).
C

Result

Blend (Darken)

 The yellow portions of Texture 0 are darker than the white portions of Texture 1, and so the yellow color of Texture 0 remains unchanged.
However, the yellow portions of Texture 0 are lighter than black portions of Texture 1, and so the yellow color of Texture 0 gets replaced.

Blend (Lighten)

The Blend (Lighten) setting compares two overlaid colors, the base color and the composite color, and applies the lighter color.

  • When the base color is darker than the composite color, the color is replaced.
  • When the base color is lighter than the composite color, the base color remains as is.
Example of Using Blend (Lighten)
A Texture 0 image Set an all-brown image for Texture 0. (Base color)
B Texture 1 image Set an image of a white to black gradation for Texture 1, and then apply Blend (Lighten).
C

Result

Blend (Lighten)		 The brown portions of Texture 0 are darker than the white portions of Texture 1, and so the brown color of Texture 0 gets replaced.
However, the brown portions of Texture 0 are lighter than the black portions of Texture 1, and so the brown color of Texture 0 remains unchanged.

Added a feature to specify the italic tilt of text boxes in degrees.

You can now specify the italic tilt of text boxes in degrees, enabling a greater range of detail for expressive effects.
You can configure the settings more naturally than with the previous ratio settings.

Italic Tilt Setting Added to Properties Window
Italic Tilt (Angle): Enter an angle at which to tilt the font.
Specify a value in the range from -60.00 to 60.00.
(Percentage): Displays a percentage representing how much the font was changed by applying the tilt.
Related to the value specified for Italic Tilt (Angle).

Support for kerning from the GPOS Table for OpenType fonts is now available.

Although loading kerning data from the GPOS tables that OpenType fonts use was not supported until now, that data can now be applied when rendering.

Added support for the BC7 format.

Expanded support to enable use of the BC7 format.

This option can be specified from the texture window of LayoutEditor.

BC7 is an RGBA color format, and there is 1 byte of texture format (1/4 of RGBA, twice of BC1) for each pixel.
Compared to BC3, the format has less compression noise and displays at a slightly higher quality.
However, be aware that conversion takes more time (particularly if the size is large).
