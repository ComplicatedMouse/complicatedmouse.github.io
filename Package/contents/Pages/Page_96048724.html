Color Scaling

The following describes the Color Scaling feature that applies a scale to color settings values.

Overview

Color Scaling is a feature that can set values by which colors in the Color tab can be multiplied.
In an environment where glare is applied as a post-effect, using scale values can increase the luster of the overall effect.
The following overview describes the settings ().

Color Scaling

Use the dial () or the text box () to specify the value for the color scale.

Example Configurations

The following image compares changes to the color scale values.

A Default Setting: 1.0 No effect on the color scale.
B Value set to 2.0 Amplifies the colors from A by a factor of two. When the value exceeds 1, a gradual glare effect appears.
C Value set to 10.0
 Amplifies the colors from A by a factor of ten. The value already exceeds 1. The glare effect is applied at an extreme.
