UI Descriptions ()

This section provides overviews of the various elements of the user interface for the editor.

Main Window Describes the names and roles of each part of the EffectMaker window.
Property Window Describes the Property window, used to configure the settings for the various EffectMaker nodes.
Effect Browser Describes the Effect Browser, used to browse and search for files used with EffectMaker.
Curve Editor Window Describes the Curve Editor window, where EffectMaker emitter time animations are created and edited.
History Window Describes the History window, where the edit history on an effect is listed and you can go back and forth through that history.
Search Window This section describes the Search window for searching for features in effect files in specified folders and batch-converting the settings.
Texture Details Window Describes the Details window, which presents detailed information about the textures being used with effects.
EffectViewer Describes EffectViewer, which is the viewer for EffectMaker.
