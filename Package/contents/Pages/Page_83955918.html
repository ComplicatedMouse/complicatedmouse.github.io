Model Auto-Translate

This tab configures the Auto-Translate feature, which allows you to check how effects look on the viewer when the registered model is moved.

Auto-Translate

The Auto-Translate group () is a set of features for checking how effects look in the viewer when models are moved.
This group has the following settings.

Auto-translate type　

A feature for selecting the auto-translate type.
XZ translation radius Sets the radius of the arc over which the model is moved.
XZ translation speed Sets the speed of the model when it moves in the xz-plane.
Translation Distance Sets the distance when the model moves in a straight line.
Translation Speed Sets the speed when the model moves in a straight line.
Straight Line Angle Sets the direction when the model moves in a straight line.
Y Swing Amplitude Sets the width (the amplitude) when the model is moved up and down.
Y swing speed Sets the speed when the model is moved up and down.
Z roll speed Sets the speed when the model set is rotated around the z-axis.
