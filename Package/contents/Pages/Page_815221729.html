Animation Live Editing (Synchronization)

Synchronization

1. Starting the Viewer

Start 3dSampleViewer located in the following path.

%NINTENDO_SDK_ROOT%\Samples\BuiltApplications\3dSampleViewer\3dSampleViewer.exe

2. Executing MEL

Run the following MEL script in the script editor and start the test tool.
(From the Maya menu, select Window > General Editors > Script Editor and execute the following command.)

NintendoCallPython("NintendoLiveEditingTestTool.NintendoLiveEditingTestToolForG3dDemo()", false);

Click Connect to enable Send model.

(Verify that the connection type is Normal PC application, and then connect.)

3. Sending the Model

Click Send Model and select the FMDB model to synchronize.

A dialog box appears and prompts you to select the FMDB file corresponding to the particular namespace. Open the intermediate model file that corresponds to the specified namespace.

  • For the file that corresponds to the :Pierrot namespace, select Pierrot.fmdb.
  • For the file that corresponds to the :Robot namespace, select Robot.fmdb.

The FMDB files are located in the following directory.

%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SkeletalAnimDemo\Pierrot.fmdb
%NINTENDO_SDK_ROOT%\Samples\Sources\Applications\G3dDemo\Resources\SkeletalAnimDemo\Robot.fmdb

4. Confirming Synchronization

Confirm that Robot and Pierrot are displayed in 3dSampleViewer.

While in this state, confirm the following.

  • When you move the viewpoint on Maya, the viewpoint on 3dSampleViewer also moves in the same manner.
  • When you playback animation on Maya, the model on 3dSampleViewer also animates in the same manner.
  • When you rotate the Pierrot:Spine node on Maya, the same orientation is applied to the model on 3dSampleViewer.

5. Confirming Retargeting and Motion Mirroring

  • In the test tool, from the Retargeting Host drop-down menu, select Pierrot and confirm that Robot assumes the same postures as Pierrot in 3dSampleViewer.
  • In the test tool, select Motion Mirroring and confirm that the animations are mirrored in 3dSampleViewer.
  • In this state, play the animation and change the rotation of any bone for Pierrot and confirm that Robot assumes the same posture.
