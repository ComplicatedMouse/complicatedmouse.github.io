Profiler-Units

Overview

Specifies which units should be displayed in the GUI.

Profiler-Units [-UnitType] <ScriptingUnitType> [-ShowError] <Boolean>


[-UnitType] <ScriptingUnitType>

The type of units to display in the profiler.

    • AvgTime: Selects the Avg button on the Home tab.
    • Percent: Selects the Percent button on the Home tab.
    • Samples: Selects the Samples button the Home tab.
    • Time: Selects the Time button on the Home tab.


[-ShowError] <Boolean>

Whether the error value should be visible.

Examples

Basic Usage

Provided here are some basic usage examples.

Profiler-Units Percent $false
Profiler-Units -UnitType Samples -ShowError $true
