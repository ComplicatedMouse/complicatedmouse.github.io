NX Add-On 0.12.x Error Codes

Differences With Existing account Library Error Codes


Compared to Account Library / Error Codes, differences exist in the following range of error codes.

Name Start End Cause
Authentication Error 2124-4000 2124-4499 An error occurred during authentication.
For more information, see the Authentication Error section later in this document.

Authentication Error

Error codes depicted in this format, represent error codes with differences.

Error Codes Description
2124-4003

The development hardware is not allowed to use the authentication service.
2124-4007 The development hardware has been denied use of the authentication service.
2124-4025

The game card has been denied use of the authentication service.
2124-4026

The application is not allowed to use of the authentication service.

This includes cases where the OMAS application was unapproved or the wrong application ID was specified.
2124-4027 The development hardware is not allowed by the application service to use the authentication service.
2124-4028 The application service has not started.
2124-4029 The application service has ended.
2124-4030

The application service is not currently available.
This can happen when the service is down for maintenance.

2124-4200 The authentication service is down for maintenance.
2124-4201 The authentication service is down for maintenance.
Other error code There is a problem with the authentication service.
