Shader parameter labels are now displayed in both Japanese and English.

English label names were added to support English localization. You can switch the labels displayed in the Japanese and English versions of 3DEditor respectively by using an annotation like the following. 

label="ディフューズ:lang-en{Diffuse}"

Maya Plug-In

The expanded/collapsed state of the UI for each category in the Options window of the export plug-in is now maintained.

Maya now maintains the expanded/collapsed state of the UI for each category (such as Preset Options and Output Options) in the Options window of the Maya export plug-in.
The same revision was made to the Edit Additional Options window of the multi-export plug-in.

Related Links For more information, see the following pages.

Export Plug-In Describes the available features and how to use them.

The FMD file can now be exported with the root bone's translation and rotation values reset to 0 and the scale reset to 1.

By selecting the check boxes in the Reset Root Transform option, you can specify separately that the translate, rotate, and scale values be reset.

Example: No Reset Root Transform Options Are Selected
(Export Result So Far)

Example: All Reset Root Transform Options Are Selected


Use this feature when you have a model that is located at a position other than the origin in Maya and you want to export it just as you would if it was positioned at the origin (without rotation or scaling).
You can also use this feature when you have offset the positions of the root nodes so the models for the various LOD levels do not overlap when exporting LOD. (This is not possible if each level has multiple root nodes.)

If there are multiple root nodes in Maya and nw4f_root has been added, the translate, rotate, and scale values are not reset for the bones that correspond to the root nodes on Maya.
If a root bone is a shape for which skinning has been configured, or the root bone has skinning influence, it is always exported in the bind pose without being reset.

Even if the rotation pivot (the scale pivot) is changed, the translate value of <bone> is forcibly set to 0 and exported.
(If you display the FMD file alone, it might not be in the same position it would be in at the origin in Maya.)
In addition, even if the rotation axis or the joint orientation is not 0, the rotate value of <bone> is forcibly set to 0 and exported.
