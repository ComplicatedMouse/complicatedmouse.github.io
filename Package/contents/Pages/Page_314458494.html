Layout and Font News

This is a graphical introduction to the features added in NintendoSDK 5.3.0.

New Layout Expression

This section introduces features that broaden the scope of what can be expressed in the layout.

Added texture projection mapping mode supporting perspective projection.

Presets have been added to mapping methods for rendering the projected texture image in the screen space during the perspective projection of the layout camera.
During perspective projection of the layout camera, a mapped texture is shown projected to the screen coordinates, even when the pane is rotated on the x-axis or y-axis.

The following images compare what happens with orthogonal projection mapping and perspective projection mapping during perspective projection of the layout camera ().

A Orthogonal Projection Mapping

Because perspective is applied to the pane based on the rotation, the texture image () is distorted.
B Perspective Projection Mapping

The texture image () is projected in the screen space, so distortion based on perspective does not occur.

LayoutEditor

This section introduces the new features in LayoutEditor.

Improved the Hierarchy window operations.

The operations in the Hierarchy window () have been improved.
The association operations with the layout window and the hierarchy operations have been improved.
The following list shows the improved shortcut keys for operations in the Hierarchy window.

Key Operations That Are Now Immediately Applied In the Layout Window
A Displays the entire layout window.
F Displays all of the panes selected in the layout window.
Key Operations That Make Pane Hierarchy Tasks Easier to Perform
P Places the currently selected pane under the hierarchy of the last selected pane.
SHIFT+P Removes the hierarchy relationship for the selected pane.
G Creates a null pane and places it under the hierarchy of the selected pane.
Key Operations That Make It Easier to Show and Hide Multiple Panes

ALT + Click the Show/Hide icon ()

Displays only the panes under the hierarchy of the selected pane and hides all other panes.
Perform the same operation again to revert to the previous state.

Documentation

This section introduces some of the pages we recommend reading, which describe the new features added to the NintendoWare layout module.

Added samples.

Added samples that use the Indirect feature and the Threshold-Based Alpha Blending feature.

\indirect\indirect.flyt

This sample uses indirect expression.

The background and the "GAME CLEAR" text are indirectly distorted.

The "GAME CLEAR" portion uses the Capture Pane feature to make it possible to change the text for purposes such as localization.

\AlphaThreshold\AlphaThreshold.flyt

This sample demonstrates the threshold-based alpha blending feature.


The threshold-based alpha blending feature is used to display the text "RESULT" and ruled lines on the screen.
