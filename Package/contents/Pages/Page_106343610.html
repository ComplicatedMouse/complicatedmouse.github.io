LayoutViewer

Describes LayoutViewer, which is the viewer for checking the results of the editing in LayoutEditor.

LayoutViewer Startup File

LayoutViewer is the viewer to which content edited using LayoutEditor can be sent for previewing.

LayoutViewer Screen Description

The following information is displayed in LayoutViewer.

A: Debug information

B: Process information

C: Viewport

D: Animation window

A: Debug information

Displays the settings of the current layout.
The following items are displayed in the debug information.

Mode

Displays the mode when the layout was sent.
Animation Mode

Display when sent with Send to viewer (animation mode).
Animation can be checked per-animation or per-segment tag.

Related Links For more information about Send to viewer mode, see here. For more information about playback switching, see here.

Control Mode

Display when sent with Transfer to Viewer.
You can check feature animations using the mouse or touch screen.

Related Link For more information about feature animation, see here.

Current animation frame when sent with .
Anim

The name of the currently playing animation segment, the loop setting, the segment number, and the total number of segments when sent with .
Size Displays the size of the layout screen.
Memory Displays the amount of memory used by the layout.
FPS Displays the frame rate.
Gamma Displays the color space mode.
Proj Displays the projection mode.
FOVY Displays the angle of view in the direction perpendicular to the camera.
Near Displays the near clip value.
Far Displays the far clip value.
Align

Displays whether the alignment feature is enabled or disabled.
You can change this setting by selecting or clearing the check box.
PaneInfo Shows whether the expanded user information feature is enabled.
You can change this setting by selecting or clearing the check box.
HDR Shows whether the high dynamic range is enabled.
AnimWin

Shows whether the Animation window is enabled.
You can change this setting by selecting or clearing the check box.
Bound

Sets how to display borderlines.

  • None: Pane borderlines are not displayed.
  • Null: Displays borderlines for null panes.
  • All: Displays borderlines for all panes.
FBSize

Sets the framebuffer resolution during preview.

B Processing Statistics Results

Displays the results of measuring the processing of the current layout.

Calc Displays processing times for the whole CPU.
Draw Displays processing times for the draw command.
Gpu Displays processing times for the whole GPU.

C Viewport

Displays the layout that was sent.
If the mode has been set to Control Mode, you can play feature animations with the mouse or by tapping.
If PaneInfo has been enabled in the debug information, you can display a list of the expanded user information that has been set for a layout pane by clicking the pane in the viewport.

D Animation Window

This window can be used to check the state of animations being played back simultaneously as a group.

This window reproduces the actual playback states of the animations in the game, making it easier to find data issues while creating a layout.
The following items are displayed in the debug information.

A |<<

Moves to the first frame.
>

Plays the animation.
||

Pauses animation.
|<

Goes back one frame.
>|

Goes forward one frame.

Enables or disables group playback.
B (?) Rest the mouse pointer on the question mark to display Help.
C

Enables or disables playback of segment animation on a per-group basis.
GroupName

Displays the group name.
AnimationName

Displays the segment tag name associated with the group.
FrameSpan

Displays frames of the animation selected in AnimationName.
Info

The state in which the animation window is displayed when LayoutViewer is started can be set via the project configuration file.
The displayed window can be switched by pressing the X key on the keyboard.
