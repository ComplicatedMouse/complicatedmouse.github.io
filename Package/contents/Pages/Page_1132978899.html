Maya Plug-In

During the LOD export, influences that were not used for some base polygons are now available for LOD polygons.

As with the previous specification, influences that are not used for any of the base polygons are still unavailable for LOD polygons.
In addition, the maximum number of influences must be less than the number of base polygons.

Info

Specifically, if you use an influence not used in base polygon X, the intermediate file optimizer produces an error if one of the base polygons does not use the same influence type (rigid or smooth) as X.
(If the Compress Shape option is enabled, export is possible as long as the type after shape compression is X.)

Info

Exportable Example 1
// Base
ShapeA: Smooth skinning using joint1 and joint2
ShapeB: Smooth skinning using joint1, joint2, and joint3
// LOD1
ShapeA: Smooth skinning using joint1 and joint3 // Previously this caused an error.
ShapeB: Smooth skinning using joint1, joint2, and joint3

Exportable Example 2
// Base
ShapeA: Rigid skinning using joint2
ShapeB: Rigid skinning using joint3
// LOD1
ShapeA: Rigid skinning using joint3 // Previously this caused an error.
ShapeB: Rigid skinning using joint2 // Previously this caused an error.

Non-Exportable Example
// Base
ShapeA: Smooth skinning using joint1 and joint2
ShapeB: Smooth skinning using joint1 and joint3
// LOD1
ShapeA: Smooth skinning using joint1, joint2, and joint3 // Error (the maximum number of influences exceeds the base polygon count).
ShapeB: Rigid skinning using joint3
