IconBody

Overview

The nn::mii::IconBody class contains body shape data for rendering a Mii icon with a body. This class is required if you want to create a Mii icon with a body.

Resource Loading

Body part data is not included in the Mii resource. You must load a separate file.

Load the following file.

$(NINTENDO_SDK_ROOT)/Resources/Mii/Common/iconbody/IconBody.dat

The above file contains body model data for both male and female.

Initializing and Finalizing

The following are notes regarding initialization and finalization. You can also view the API reference for each function.

Initialization

Use the nn::mii::IconBody::Initialize() function to perform initialization.

You must load the body resource data required during initialization to the memory before initialization.

Use the nn::mii::IconBody::CalculateMemorySize() and nn::mii::IconBody::CalculateMemoryAlignment() functions to get the initialization memory size and alignment.

Use the nn::mii::IconBody::CalculateMemoryPoolSize() and nn::mii::IconBody::CalculateMemoryPoolAlignment() functions to get the initialization memory pool size and alignment. If the memory pool properties passed during initialization are CpuCached, you must flush the memory pool after initialization.

The memory and memory pool passed during initialization, must be maintained until you call termination processing. You can destroy body resource data after initialization.

Finalization

Call the nn::mii::IconBody::Finalize() function to perform termination processing. After termination processing, you can destroy the memory and memory pool used in initialization.

DrawParam

Use the nn::mii::IconBody::GetDrawParam() function to get the parameters required for body rendering. Pass CharInfo to get the gender-specific parameters set internally. You can also specify the gender directly.
