Plug-In for Setting User Data

This plug-in sets user data for a transform node (joint node) or shading group.
You can also set user data for the camera, light, and fog of a scene animation.
You can also set user data for <model> in the FMD file and <skeletal_anim> in the FSK file.

How to Use

Select Nintendo > Set User Data to open the window.

The selected transform node or shading group (material) is displayed in Selected Transform Node or Shading Group.

Setting Method

Selecting just one transform node (joint node) or shading group (shadingEngine node) displays the current setting for that node in the window.
(If a material node is selected, the setting for the corresponding shadingEngine node is displayed.)

A list of configured user data is displayed in the lower part of the window.

Name The name of the user data.
Type

The type of the user data.
Integer represents integers, Float real numbers, ASCII ASCII text strings, and Unicode Unicode text strings.
Size The size of the user data (number of items).
Value

The content of user data.
If data is too long to all be displayed on-screen, the unseen part is indicated using "..." at the end.

To add new user data, click Add. If an item in the list is selected, data is inserted in the selected location. If no item is selected, data is added to the end of the list. (The selection can be canceled by clicking ----- at the bottom of the list.)

The name and contents of existing user data can be changed by selecting an item in the list and clicking Edit, or by double-clicking the item.

Selected user data can be deleted by selecting an item in the list and clicking Delete.

The order of user data in the list can be changed by selecting an item and clicking Move Up or Move Down.

Select Display integer value in hexadecimals to display integer-value user data in hexadecimal. (Data is also entered in hexadecimal.) Clear this option to display the data in decimal.

Info

You can select a different node by selecting another node without closing the window.

To display the window for entering user data, click Add or Edit.

Name

Enter the name of user data.
Non-ASCII characters cannot be used. Symbols other than the underscore (_) cannot be used.
Type

Specifies the type of user data.
Integer is integers, Float is real numbers, ASCII String is ASCII text strings, and Unicode String is Unicode text strings.
Values

Enter the value (number or text string) of the user data.
More than one value can be entered by delimiting with a return. (Only one value per row can be entered.)

To add or change user data, click OK. If the name or value entered is invalid, an error is displayed on the command line and in the script editor.

Info

If you set your user data to a transform node using a name starting with nw4f_model_ud, it is exported as user data in the <model> element in the FMD file.
Even if that node is a locator or is not exported as a <bone> node, the <model> user data will be exported.
If there are multiple nodes for export that have names starting with nw4f_model_ud, the node with priority is the one that is displayed at the top of the list when you set the Sort Order to Scene Hierarchy in the aligner.

Info

If you set the user data to a transform node with a name starting with nw4f_fsk_ud, it is exported as user data in the <skeletal_anim> element in the FSK file. The specifications are the same as for nw4f_model_ud.

Info

Use the Plug-In for Setting User Data to set user data for each bone in skeletal animation.
