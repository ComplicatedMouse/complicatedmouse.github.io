Glare Intensity

This feature adjusts the Glare post effect embedded in the EffectViewer.

Overview

The Glare Intensity feature allows you to adjust and check the Glare post effect embedded in the viewer.
This section provides an overview of the settings ().

Glare Intensity

Select the amount of glare applied from selections 0 through 6.
The following are the display states when using each selection. The higher the value, the more intense the applied glare.

0 (none) 1 2 3. 4 5 6
