Derivative Parts

Overview

The Derivative Parts feature provides the functionality for making multiple parts layouts from a single parts layout that serves as the derivative base.
You can add new panes and animations to the parts layout of this derivative base and make new parts.
(This feature is still under development, so the specifications are subject to change.)

Benefits

By using derivative parts, you can enable adjustments that are not covered by the adjustments to Overwritable Properties and Size Scaling after a part has been positioned.
This enables you to make wider shared use of individual parts, providing even greater merit than the ease of data creation and adjustment provided by the use of parts in layout creation.

Using Derivative Parts

Adding a Pane

You can add new panes.
You can also hang panes as children below the pane of the derivative base part. (In other words, the panes can share the affects of scale animation and transparency.)

For example, create a button with one added icon on a shared button.

Adding Animations

You can add an animation to the pane of the derivative base part. (However, this is limited to elements that were not animated in the first place.)


For example, you could add color animations to create buttons that behave completely different from the shared color button.

Changing Functionality

By changing the Parts Settings you can change the functionality of derivative parts.
For example, you can change the feature animation and add an animated pane that is linked to the base part animation.

Limits on Derivative Parts

Modifying the Derivative Base Part

  • Edit pane properties.
  • Delete a pane, or delete an animation.
  • Edit hierarchy settings that alter the parent-child structure of the base part. (Example: Adding new items between parts at the same level.)
  • Edit or delete animation segment tags or groups.

Derivatives of Derivative Parts

You cannot create multi-level derivatives.

Derivative Parts

Derivative Base Pane


Displayed in red icons. You cannot select it in the Layout window. You cannot edit its properties.

Derivative Base Pane Animation


Tree items are displayed with red lock icons. Curves are shown in black. You cannot select or edit keys.

Derivative Base Groups, Segment Tags, and so On

Shown in gray text with pale blue background. You cannot delete or edit.
