  • A Menu bar
  • B Toolbar

A Menu Bar

The menu bar contains the following items. Right-click an item to display a shortcut menu.
The listed items change when the layout window is displayed.

File
Operations for creating layouts, importing and exporting files, and configuring the environment settings.
Edit Operations to redo, copy, and delete.
Window Show or hide sub-windows needed for creating layouts.
Help Displays Help and information for LayoutEditor.

C Work Area

This is the area where work on the layout is performed.
Multiple layout windows can be opened and worked on at the same time.
By using multiple layouts at the same time and switching between tabs you can compare differences among layouts and different animations.
You can also make copies of panes among layouts.

Window Docking

Dock the windows for various operations in the work area and arrange them to your liking for the task at hand.

The various windows including the Group window (), the Hierarchy window (), the Curve Editor window (), and the Parts Palette window () can be docked and displayed together, or displayed independently as floating windows.
The windows can be arranged to suit your task in LayoutEditor.

Workspace

Use the Workspace drop-down list in the toolbar (B) to change the mode for how windows are displayed to match the task at hand.
From the workspace, select Layout, Animation, or Part.

You can customize the windows for the Layout, Animation, and Part modes.
The tool stores the customized window settings for each mode, so individual users can arrange the windows to their liking.

Placement of Windows Inside the LayoutEditor Work Area

Click along the top edge of the LayoutEditor window to display icons in the LayoutEditor window.
Drag a window to one of the icons to move the window to that location in the work area.

Remember Workspace

This feature allows users to save their customized workspaces.

Window > Workspace > Remember Layout

Restore Workspace

Use this feature to restore a workspace that was remembered by the tool using the Remember Workspace feature.

Window > Workspace > Restore Layout


You can also do this from the toolbar using the Restore the workspace layout icon.
Shortcut: CTRL+SHIFT+T


Info

The Tools window cannot be placed on top of the region of the layout window.
The layout window cannot be a floating window.

If you want to open multiple layout windows, use the tabs at top of the display to switch between windows.
