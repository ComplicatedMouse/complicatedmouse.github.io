nn::atk News ()

The Addition of Evaluation Features Has Increased Both the Processing Load and Memory Requirements

With the addition of evaluation features in NX Add-On 4.2.0, the CPU processing load of the sound thread has increased, as have the memory requirements for nn::atk::SoundSystem and nn::atk::SoundArchivePlayer.
A future version will incorporate revisions to cap these increases in processing load and memory requirements, but be aware that both will swell in NX Add-On 4.2.0.
