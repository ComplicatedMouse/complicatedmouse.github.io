Search

This page describes the Search tab UI.

A Search origin path

The path specified in the address bar in the Search tab is the location where searches start.
B Search Settings
 Displays the search settings in the navigation pane in the Search tab.
C File List Displays a list of file hits from the search.

Search Settings

The search settings are as follows.

A Search Target File Select whether to search for textures or primitives.
B Search Conditions

Select from among the presets for the search criteria. A dedicated UI is displayed for each preset.
C

Filtering to the Results in the Search Origin Path

Set whether to filter the search range below the search origin path by selecting or clearing the check box ().

Clearing this setting makes it possible to search up to the highest hierarchical level on the drive.
D Execute Search Left-click this button to start the search.
E Reset Left-click this button to reset the search conditions and the contents of the file list.

Available Features

The following features are available in the Search tab.

Export Exports the selected file to the workspace.
Edit Comments Edit comments for the selected file.
Select Label Color

Sets a label color for the selected file.
Display Windows Explorer Open Windows Explorer and go to the path of the selected file or folder.
