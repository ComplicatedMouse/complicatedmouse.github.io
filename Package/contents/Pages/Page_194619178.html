OpenGL Library

Introduction

OpenGL is a real-time graphics conversion API developed by the Khronos Group.

OpenGL is supported in the NintendoSDK NX Add-On and is provided with the assumption that NintendoSDK users will want to migrate existing titles and frameworks that use OpenGL.

PC Operating Environment

Conforms to the NintendoSDK operating environment.

Installing the GPU Driver

Set up the OpenGL environment on the PC you are using.

The NVIDIA Graphics Drive package available from Nintendo Dev Interface supports OpenGL.

Supported Versions

Nintendo Switch™ is based on OpenGL® 4.6 and OpenGL® ES 3.2.

For more information, see the following pages.
