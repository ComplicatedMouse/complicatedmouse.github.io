Background Camera and Image

The Background Color - Image group () is a set of features for controlling the connected viewer.
This group has the following settings.

Background Color This feature sets the viewer background color.
Specify File This feature registers the background image to display in the viewer.
Multiplicative Color and Alpha This feature sets colors to be multiplied with the background image.
