How do I apply a common mask or alpha to multiple panes?

This can be achieved by using the framebuffer alpha with fragment operations. Use the following procedure.

  1. Write the mask and alpha gradation you want to apply universally to the alpha channel for the background image.
  2. Create a picture pane where the background image has been rendered from the deepest parts first.
  3. Configure the settings for writing the alpha to the framebuffer using the Blend settings of the Fragment operations, located in the Detailed Material Settings for the picture pane just created.

  • Set None for the color.

  • Set alpha to blend calculation = (Pixel Color x 1) + (Framebuffer Color x 0).

  1. Next, create the picture pane to which the mask and alpha will be applied on the background picture pane.
  2. Configure the settings for referencing the alpha in the framebuffer using the Blend settings of the Fragment operations, located in the Detailed Material Settings for the picture pane just created.

  • Set the color to (Pixel Color x eFB Alpha) + (Framebuffer Color x (1 - eFB Alpha))
  • Set alpha to blend calculation = (Pixel Color x 0) + (Framebuffer Color x 1).

These settings blend the picture pane created in Step 4 with the background according to the alpha value of the background picture pane.
