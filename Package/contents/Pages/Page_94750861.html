Emission Interval Type

This section describes the feature that sets the emission interval.

Overview

The emission interval type allows the particle emission interval to be set by time or distance.
The particle density can be controlled for many effects by manipulating the emission interval type.
The following overview describes the settings ().

Emission Interval Type

Particles are always emitted in the first frame of playback, but after that you can select the interval for subsequent emissions from the following presets.

Time Uses time as the base to set the emission interval for particles.
By setting the interval to be random, you can create some fluctuation in particle emission.
You can also use Emission Rate to sets the number of particles that are emitted at one time.
Distance

Uses distance as the base to set the emission interval for particles.

When the emitter moves, particles are emitted in a manner that keeps the interval the same.
Particles can be made to be emitted at the same distance regardless of how fast they are moving.
