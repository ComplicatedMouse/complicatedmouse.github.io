Manual Emitter Emission

This section describes the Manual Emitter Emission feature, which provides a way for game developers to control particle emission at the level of emitter sets.

Overview

The Manual Emitter Emission feature provides a way for game developers to control particle emission at the level of emitter sets.
By calling the same emitter set multiple times, you can control the location and timing of particle emission in a way that reduces the cost of CPU processing.
The Manual Emitter Emission feature has the following limitations.

  • When this feature is used, no particles will be emitted if the developer has not configured the settings for manual emitter emission.
  • Emitter sets that include child emitters cannot be emitted.

When Manual Emitter Emission is enabled, some additional settings are added to the Playback Settlings () group. The group has the following settings.

Manual Emission Interval Sets the time interval (number of frames) at which the emitter's particles are emitted.
Emission Process Count Sets the number of times the emission process is called by the program when the emitter is emitting particles.
