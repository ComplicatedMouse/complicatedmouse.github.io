Showing Effects

This section describes how to show or hide an effect in the Viewer.

Preparations

Begin this task after starting both EffectMaker and the viewer.

Related Links For more information about starting EffectMaker and the Viewer, see Workflow.

Loading the Emitter Set

First, load the effect used in this example into EffectMaker with the following procedure.

1 From the main menu, select File > Open ().
2

When a window opens, select and open the emitter set file (ESET file) from the following location.

\Samples\Sources\Resources\Vfx\Eset\QuickStart\NodeOperation\Viewer.eset

Verifying in the Viewer

Confirm that the loaded effect has been transferred to Viewer.
You can now edit the effect while observing the changes in the viewer.

Switching Between Showing and Hiding an Effect

This section uses an emitter node to describe how to switch between showing and hiding an effect.
The following describes the procedure in the workspace in addition to what appears in the Viewer.

1 The particles from Emitter_A () are displayed in front of Emitter_B ()
2 To hide the emitter from view (), click the View button () to the left of Emitter_A ().
Emitter_A () is hidden in the Viewer, and the particles from Emitter_B () are shown.
