Library

Overview

The PC version of NVN provides only header files. The library does not need a static link, and the NVN library entry point gets function pointers from the NVN driver at run time.
The NVN header files provided by NintendoSDK NX Add-On contain this helper function. For more information about using it, see theNVN Programming Manual and the samples.

The NX version of NVN provides a specialized library in addition to the header file.

Header Files

NVN provides nvn/nvn.h, which defines the C interface, and nvn/nvn_Cpp.h, which defines the C++ interface.
These files are placed in the following directory along with other required header files. 

$NINTENDO_SDK_ROOT/Include/nvn
