Multiplicative Color and Alpha

This page describes the Multiplicative Color and Alpha feature that sets the background color of the viewer.

Overview

Sets colors to be multiplied with the background image.
This section provides an overview of the settings ().

Multiplicative Color and Alpha

Sets a color by which to multiply the background image in the Color Editing UI.
You can adjust the alpha value A to blend the background color and background image.
