File output

This page describes the types of files output by FontConverter.
Different types of file conversions can be performed according to the settings in the tabs in Output.

Converting from the Image Tab

Converts a file to a special type of BMP file that can be used in FontConverter.
Related Links For more information, see FontConverter File Format.

The output image file can be edited using an image editing software package to achieve the desired design.
After you have completed the editing of the image file, the resulting BMP file can be used as font source data in LayoutEditor by performing the following.

  • In the Image tab in Input, specify the edited BMP image file.
  • Then, in the bffnt/bffna tab in Output, set bitmap font binary file type (BFFNT) as the file type and click the Convert button.

Converting from the bffnt/bffna Tab

The font converter converts the font into a bitmap font binary file (BFFNT) that can be processed by the font library.
Once the conversion is complete, the font can be used in LayoutEditor.
