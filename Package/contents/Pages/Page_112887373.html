Starting Process

This section describes the feature for determining the processing when the effect first starts to play.

Overview

The Starting Process feature determines the behavior when the effect first starts to play.
It can reduce the unnatural feeling from particles suddenly being displayed by displaying them gradually using alpha and scale at the time the effect appears.
The following table describes the various settings () for Starting Process ().

Starting Process

These settings configure the behavior when particles are emitted.

Alpha Fade All particles start out transparent and gradually appear over the period of time specified by the Fade-In Duration.
Scale Fade All particles start out small and gradually increase in size over the period of time specified by the Fade-In Duration.
Fade-In Duration This feature sets the duration, in frames, required for particles to fade in to appearance. Alpha Fade and Scale Fade may be used by turning it ON.

Example Configurations

To stop particle emissions, click the animation Stop button (). To begin particle emission, click the animation Play button ().
The following image compares the Start Processing feature with different settings turned on and off.

A Fade off The effect without fading.
B Alpha Fade on The particles start out transparent and gradually appear over the course of the specified Fade-In Duration.
C Scale Fade on The particles start from a size of zero and and gradually enlarge over the course of the specified Fade-In Duration.
