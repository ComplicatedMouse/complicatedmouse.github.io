Color Combiner Channel

This section describes the Color Combiner channels.

Overview

The channels of the Color Combiner determine what component of the registered texture to apply as the input value for the equations.
This enables you to expand the use of textures and is convenient when you want to reuse the same texture over multiple emitters.

The following table shows the image, name, and description of each channel.

RGB The color value of the registered texture is applied as the input value.
1.0 - RGB The texture color value subtracted from constant 1.0 is applied as the input value.
A Value of the registered texture's alpha channel.
1.0 - A The texture alpha value subtracted from constant 1.0 is applied as the input value.
1.0 Constant 1.0 is applied as the input value.
