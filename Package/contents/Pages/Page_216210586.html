Atk Stream Sound Panel

This panel displays information about the stream sounds currently playing in the atk library.

UI Reference


 Item Description
A Sound Name The name of the stream sound currently playing.
B Fill Rate The fill rate of stream sounds.
C Delay State If the stream sound playback pauses due to a load delay, Delayed is displayed.
D Fill Rate Graph Display The fill rate of stream sounds expressed as a bar graph.
E File Read Cache Remainder

When using the stream sound file read cache feature, the remaining cache amount will appear.

To display the cache, pass nn::atk::SoundArchivePlayer when calling the nn::spy::atk::AtkSpyModule::PushAtkProfiles() function.
