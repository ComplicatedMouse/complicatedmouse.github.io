1. Values From the Vertex Shader

Values such as color and alpha are provided by the vertex shader.
The following table shows the main values provided by the vertex shader.

Colors 0, 1

The vertex shader performs calculations for values whose behavior is selectable from constant / random / animation.

The emitter color is multiplied by the vertex shader, and the result is provided to the pixel shader.
Textures 0, 1, 2

The vertex shader calculates the initial value of the texture coordinates and random and animation.

Based on this value, the pixel shader gets the texture color.
Primitive Vertex Color
Provided when using a primitive.

3. Alpha Processing

An alpha processing effect such as soft particles to apply after alpha values are combined can be selected in the Alpha Processing group () in the Color () tab in EffectMaker.
The following alpha processing effects are available.

Soft Particles Near Alpha

4. Custom Shaders

Application-specific effects to apply after color values and alpha values are calculated can be configured in the Cstm tab () in EffectMaker.
Shaders that apply custom shaders perform application-specific lighting processing for the color values and alpha values calculated so far.
