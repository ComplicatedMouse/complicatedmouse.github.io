This sample shows how letters appear written when the Alpha Test fragment operation is employed.

Alpha Test

  • The P_fontN_00 Picture pane:
    • Set a gradation from white to black for the alpha image of the stage 0 texture in the desired written order.
    • In the settings for the Alpha Test fragment operation, clear the check box for Use default settings.
    • As the condition, set alpha > value. With this setting, you can use a texture to control what is shown.
    • Set an alpha test animation where the value of the alpha test declines from 1 to 0.
    • Note: Other letters take on the same structure.
  • The P_Light_00 P_Light_01 Picture pane:
    • Light shines at the leading edge of the letter being written.
      →The fragment operation is Additive.
