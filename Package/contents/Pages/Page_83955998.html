Features

You can use the multi-export plug-in to export more than one model and more than one animation all at the same time by saving multiple export settings for a scene.

The multi-export plug-in exports data based on a combination of the following settings.
These combinations of settings are called export items.

  • The filename of the intermediate file to be exported
  • The path to the folder to export intermediate files
  • The path to the FMD file to merge
  • The path to the FDES file to use
  • The name of the node to be exported (display layer name)
  • The name of the enabled animation layer
  • Comment text for the intermediate file
  • The range of use when exporting an animation
  • A loop playback flag
  • Export flags for each intermediate file
  • Additional options
Info

Export items are saved for the script node named nw4fMultiExp_Settings1 in the scene.

Info

The script to run before exporting the first export item and after exporting the last export item can be specified in the config file.


How to Use

Main Window

Select Nintendo > Multi Export to display the main window.

A list of export items set for the current scene is displayed in the window that appears.

Output File Name The filename of the intermediate file to be exported.
fdes File The filename of the FDES file to be exported.
Node (Display Layer) The name of the node to export (display layer name).
Comment Comment text for the intermediate file.
Start End The range of use when exporting an animation.
Loop The loop playback flag.
Selection Export flags for each intermediate file. The following letters are shown if they are enabled.
M: FMDB file
T: FTXB file
S: FSKB file
V: FVBB file
C: Color animation intermediate file (FMAB or FCLB file)
R: Texture SRT animation intermediate file (FMAB or FTSB file)
P: Texture pattern animation intermediate file (FMAB or FTPB file)
H: FSHB file
N: FSNB file
Options

Additional options.

  • After setting the path of the export folder, the path of the FMD file to merge, and the name of the effective animation layer, select an option from the list to display it below the list.
  • Click Add to add new user data.
    If an item in the list is selected, data is inserted in the selected location; if no item is selected, data is added to the end of the list. (The selection can be canceled by clicking ----- at the bottom of the list.)
  • The setting made for an existing export item can be changed by selecting an item in the list and clicking Edit or by double-clicking the item.
  • Select an item from the list and click Duplicate to duplicate it.
  • Export items can be deleted by selecting items and clicking Delete.
  • You can change the order of export items in the list by selecting an item and clicking the Up or Down button. (The order they appear in the intermediate file being exported does not change.)
  • Select an item from the list and click Set Playback to change the playback range of the animation used by Maya such that the playback range specified by Start / End Frame for the selected export item is played.
  • Clicking the Undo button restores the state prior to the last action. (If the last action was one other than the multi-export plug-in, it is also undone.) Right-click the Undo button, and use the displayed Redo menu to perform the action that was undone.
  • You can adjust the list column width that you selected in Column Width using the slider on the right.
  • Items selected in the list can be exported by selecting Export Selected Items.
  • You can export all export items by clicking Export All Items.
  • If multiple export items share the same filename (and extension) in the same folder, an error is generated during the export.
Info

When only one item in the list is selected and an animation layer has been specified in Export Items, only that animation layer is automatically enabled (available in NintendoSDK 9.0.0 and later).
Switching the enabled animation layer is also subject to an Undo command.

The following operations are available from the shortcut menu that appears when you right-click a selected item in the list.

When multiple items in the list are selected, only Move Start / End Frame is displayed.
The user custom menu can be added in the config file.

Select Node Selects the node in Maya that the export item will export. This menu item is available when there is an entry in Node (Display Layer).
Set Display Layer Only the display layer to be exported by the export item is displayed in Maya. This menu item is available when a display layer name has been specified in Node (Display Layer).
Get Anim Layer Gets the active animation layer in Maya and sets it as an Anim Layer export item. This menu item is available when there are animation layers.
Load FDES to Scene Loads the FDES file settings of the item into the scene and opens the export plug-in options window. (If the FDES file path is blank, options from the current scene are displayed.)
Set Playback Range Changes the animation playback range used by Maya such that the animation is played for the range specified in Start / End Frame in the export item.
Get from Playback Range

Sets Start / End Frame for the export item by getting the animation playback range used by Maya.
Get from Selected Clip Sets the Start / End Frame for the export item by getting the range containing all clips selected in the Time Editor. This menu item is available when clips have been created.
Move Start / End Frame... Moves Start / End Frame for the export item. (Multiple items are possible.) To move the frames, specify the translation value (an integer) to add to Start / End Frame in the window and click the Move button. Negative values can be set for the translation value. Start/End Frame must be a range or the frames will not move.
Info

If the FDES file setting is left blank, an error occurs during exporting if the option settings for the export plug-in are not saved in the scene.
Use Nintendo > Export with Setting to display the Options window, and then select Edit > Save Settings in the upper-left menu to save the option settings.

Input Export Items Window

To display the Input Export Items window, from the main menu, click Add or Edit.

Output File Name

Enter the filename of the intermediate file to be exported (excluding its extension).

If left blank, the current scene name is used for the intermediate filename.

Enter @node to include the name of the output destination node (the uppermost node in the hierarchy if there are multiple), specified in Node (Display Layer), in the intermediate file name. If a display layer is specified in Node (Display Layer), the node names included in the display layer are also displayed.
To enter the current scene name, on the right, click Scene.
Clicking Node allows you to enter a name for the node currently selected in Maya (or the highest node in the hierarchy if more than one is selected).
Output Folder Enter the folder to which to output intermediate files.
If this setting is left blank, the settings of the FDES file are used. (If the File setting in the FDES file is also blank, the current scene's settings are used instead.)
If the specified path is not a full path, the system treats it as a relative path from the current project folder.
You can use environment variables (of the form %NAME% or $NAME) for the file path.
Merge fmd File Merges the specified FMD files if you select this option and specify an FMD file path.
If the path is left blank, the export file to be overwritten is merged.
If the specified path is not a full path, it is treated as a relative path from the current project folder.
You can use environment variables (of the form %NAME% or $NAME) for the file path.
fdes File

Enter the path to the FDES file you are using.
If left blank, the current scene's option settings are used for export. An error occurs if option settings are not saved in the scene.

If the specified path is not a full path, the system treats it as a relative path from the current project folder.
If a full path has been entered and it is a path within the current project folder, it is converted to a relative path when you click OK.
For the file path, you can use environment variables (of the %NAME% or $NAME form).
Output is skipped if there is no FDES file in the specified path.

You can also specify the FDES file in the presets folder. Normally, the FDES file in the presets folder is preset="". The default is to process as if there is no preset. You can use the config file to change the process to use this preset name for the FDES file. (This feature is available in NintendoSDK 11.3.0 and later.) It is used for branching the processing of Pre-Export Script and Post-Export Script based on the preset name.


You can select an FDES file from the history of recently imported FDES files by clicking Recent fdes at the bottom.
Click Load fdes to Scene to load the settings from the FDES file entered into the scene and display an export plug-in option window. (Option settings for the current scene are displayed if the path is left blank.)
Node (Display Layer) To export a particular node, enter the name of the node to export. (Nodes in the hierarchy below this entered node are also exported.)
You can also enter the name for the display layer to export. (All nodes below the nodes included in the display layer are also exported.)
You can enter multiple node names (or display layer names) by separating them with spaces.
You do not need to enter anything if the FDES file is being used to export the entire scene.
Also, even if the FDES file is configured to export the entire scene, if a node name (display layer name) is entered, only the entered node (display layer) is exported.
If you entered a display layer name and the display layer is hidden during export, it is temporarily displayed for export.
Click Get, located on the right, to enter the name of the node currently selected in Maya.
Click the Select button to select the entered node within Maya. (If you entered a display layer name, all nodes included in the display layer are selected.)
Click the Get Display Layer button, located on the bottom, to enter the name of a visible, nontemplate display layer into Maya.
Click the Set Display Layer button to make only the entered display layer visible in Maya.
Anim Layer To enable and export a particular animation layer, enter the name of the animation layer to enable.
You can enter multiple animation layers by separating them with spaces.
If any of these animation layers have parents, the parent animation layers are also activated.
Enter BaseAnimation to deactivate all non-base animation layers.
If this setting is left blank, BaseAnimation or the currently active animation layer is exported. (Select in the config file.)
To enter the names of the animation layers that are currently active into Maya, click Get on the right.
Click Set to activate only the entered animation layers within Maya.
Comment

Specifies comments about editing the intermediate file.
You cannot use semicolons (;), single quotation marks ('), or quotation marks (").

If left blank, the editing comment specified in the FDES file will be used.
Start / End Frame Specify the range (the start frame and the end frame) of the animation.
This setting works the same way as the Start / End Frame option of the Export plug-in.
Click Get, located on the bottom, to enter the playback range value being used by Maya for the animation.
Click Set to change the animation playback range being used by Maya so that the range entered is played back.
Click Get Clip to enter the value for the range containing all clips selected in the Time Editor. (This button is available if clips have been created.)
Loop Sets the loop playback flag.
Select this option to export the data as looped playback data.
If this option is cleared, the animation range is exported as one-time playback data
Output File Selection Sets an export flag for each intermediate file.
Click Get from fdes to export only the intermediate files that are set for export using the specified FDES file in fdes File. (If the path is left blank, all intermediate files set for output in the current scene are enabled for export.)
Additional Options

Specifies additional options using the same format used in the third argument of NintendoExportByFdes3.

Click the Edit button to display the Edit Additional Options window, which can be used to specify additional options in the UI (described in more detail later).
You can enter multiple options either on a single line or by using returns and entering them on multiple lines.
These additional options take priority over the options specified in the FDES files. (If the preset locked in the options of the FDES file is any value except None, the setting in the FDES file option settings takes precedence.)
For example, you can enter specifications as follows if the setting for Frame Precision in the FDES file is 1.0, but you are exporting it as 0.5.

-fp 0.5

You can enter specifications as follows to execute a batch file after export.

-poes 'system "D:/data/post_export.bat"'

(If an argument includes an space, you must enclose it in single quotes (').)


We do not recommend specifying -preset as an additional option to use a different preset. The reason is because presets that have been specified using an additional option and are locked are not reflected in the UI of the multi-export plug-in.

Click OK to add or modify export items.

Info

If the following options are locked in the FDES file for the exported items, the Input Export Items window UI is locked against edits (a key icon is displayed), and the FDES file settings always take priority.

  • Output File Name
  • Output Folder
  • Merge fmd File (Merge on/off, path to the FMD file to merge)
  • Comment
  • Start/End Frame
  • Loop
  • Output File Selection
If LOD export is locked to ON in the FDES file, output options for intermediate files other than FTX files are locked against editing.


Edit Additional Options Window

In the Input Export Items window, in Additional Options, click the Edit button to display the Edit Additional Options window.

Info

In the Input Export Items window, if FDES file is left blank and the export plug-in option settings are not saved in the scene, the Edit Additional Options window does not display (an error occurs).
Use Nintendo > Export with Setting to display the Options window, and then select Edit > Save Settings in the upper-left menu to save the option settings.


Right-click the pen icons next to options that are unavailable (displayed in gray), and select Edit to place them into an editable state.
Specify the desired options and then click OK to change the additional options.

Right-click an editable option and select Use fdes to return that option to the unavailable state (displayed in gray). In this state, the values specified in the FDES file (or the options in the current scene) are applied.

Select the Show Locked Options option in the lower left corner of the window to display the options that are locked in the FDES file.
Locked options cannot be edited (and a lock icon is displayed next to them).
Display locked options when you need to check the values in the FDES file.

Options in the Input Export Items window, that have their own dedicated UIs, are not displayed in the Edit Additional Options window.
When such an option is specified in Additional Options, the Other Options setting is displayed at the bottom of the Edit Additional Options window.

Other Options takes options specified in the same format used for the third parameter of the NintendoExportByFdes3 function.

If a Merge Animation Name is specified without explicitly enabling or disabling Merge Animation Folder, intermediate animation files will be merged together.
If a Divide Submesh mode is specified without explicitly enabling or disabling Divide Submesh, submesh division will be enabled.
Similarly, if a Polygon Reduction mode is specified without explicitly enabling or disabling Polygon Reduction, polygon reduction will be enabled.

Sorting Export Items

You can sort export items with specific option values using the Sort menu in the main window.
Select Sort > Ascending > and an option to sort in ascending order based on the value of that option, and Sort > Descending and an option to sort in descending order based on the value of that option.

The order stored in the custom attribute is changed along with the display in the window.

Replacing the Settings for Multiple Export Items

You can batch-replace the option settings for export items from the Replace menu in the main window.
You can also add prefixes and suffixes to the option values.
To replace all export items, select Replace > Replace All Items.
To replace only the selected export items, select one or more items from the list in the main window, and then select Replace > Replace Selected Items.
After selecting a menu, the window for replacing export items is displayed.

Target specifies the target option.
Mode specifies the replacement mode.

Replace Replaces some strings (From) with a different string (To).
Replace (RegExp)

Replaces a string specified using a regular expression (From) with a different string (To).

If there are matches with multiple parts inside the original string, the first one is replaced.
Overwrite Completely overwrites using the string specified by Value.
Add Prefix Adds the string specified by Prefix to the start of the original string.
Add Suffix Adds the string specified by Suffix to the end of the original string.

Is Target is Output File Selection, use the check boxes to specify various intermediate file export flags. (They are always overwritten.)
Click Apply to run the replacement process.

If the mode is Replace (RegExp) and if From is ^human, only human at the start of the source string is replaced. If From is run$, only run at the end of the source string is replaced.

Exporting and Importing Multi-Export Settings

The following operations can be performed from the Settings menu in the main window.

Import Settings from ndmes File Imports the export item settings from the NDMES file.
When there are existing export items, the settings are added after these items.
Export All Settings to ndmes File Exports all export item settings to the NDMES file.
Export Selected Settings to ndmes File Exports the settings for the export item selected in the list (multiple items can be selected) to the NDMES file.

When there are export items with a blank FDES file path, the scene's standalone export settings are also exported to the NDMES file.

When a scene's standalone export settings are included in the NDMES file to import, select whether to also import those settings in the dialog box.
Note that when you import a scene's standalone export settings and the NDMES file path of existing export items is blank, those export settings will also be affected.

Multi-Exporting Using a Batch File

You can export all export items saved for a scene by using a Windows batch file as follows.

call StartMaya-2019.bat -batch -proj "C:/Project1" -file "Scene1.mb" -command "NintendoMultiExportDirect"

Do the following to export only specified export items.

call StartMaya-2019.bat -batch -proj "C:/Project1" -file "Scene1.mb" -command "NintendoMultiExportDirect2 \"out1,out3\""

Add the output filenames of the export files to the arguments of NintendoMultiExportDirect2. Place commas (,) between arguments when specifying multiple output filenames. If multiple export items have the same output filename, all of them are exported.
