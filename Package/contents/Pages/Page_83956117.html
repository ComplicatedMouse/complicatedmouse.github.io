Overview

This document describes the layout development environment in NintendoWare.

Layout Development Environment

The layout development environment is a system designed for the efficient development of primarily components that are rendered in 2D, such as status displays and menu screens.

LayoutEditor is the provided tool for visually laying out screen elements and embedding the created screens into programs.

The tool provides a rich assortment of features like buttons and animations for the simple creation of even advanced screens.

Structure

Layout includes the following components.

  • LayoutEditor
    • An editing tool that designers can use to create layout data. This tool runs on a PC.
    • You can use the tools in LayoutEditor to place and arrange image data (TGA files) and text (in your desired fonts) to create a layout. You can then animate the layout to make it a 2D graphical user interface for use in your game application.
    • This tool cannot create the image data. You must use another editing application such as Photoshop.
  • Layout Viewer
    • A program for checking the appearance of layouts created with LayoutEditor.
    • There is one viewer for development hardware and another viewer for the PC.
  • Layout Library (nn::ui2d)
    • A library that the developer can embed in the game program and use for displaying and controlling the layouts created with LayoutEditor.
    • The basics of cameras and rendering are the same as in any 3D graphics library, but NW4F Layout is special in that it treats the XY plane itself as a rectangular polygon. You can use NW4F Layout to place and arrange rectangular objects the same way as you would in a 2D graphics library simply by rendering an orthogonal projection of the layout from the Z direction.
    • Shaders are included and the shader that is optimal for rendering a layout is automatically used. Developers do not need to write their own shader code.
    • A viewer library is included to make the creation of custom viewers easier. It exists in the nn::ui2d::viewer namespace.
  • Layout Binary Converter
    • A program for converting the layout data created with LayoutEditor into the binary files that are handled by the layout library. This tool runs on a PC.
  • Layout Archiver
    • A program that groups together the layout data converted with Layout Converter and then puts it into a format that the Layout Library can handle. This tool runs on a PC.
