4. Editing Materials

This section describes how to add and configure textures in the demo shader.
Add the sample texture that is included in the package and adjust the material to adjust the textural look of the model.

This tutorial involves the following UIs.

Property Window Opens the property window for the material.
Edit Shader Parameters

Edit the shader parameters assigned to the material, using specular and normal map as examples.

Editing Shader Parameters

Edit the shader parameters assigned to the material, using specular and normal map as examples.

In the property window, from the tree list view (see the following figure), select the item.
This example shows how to edit Specular and Normals.

Adjusting Specularity

In the property window, in the tree list view, select Specular.
Change the Specular Strength and Specularity Aperture values to the values in the following table.

Changes to the values are displayed in real time in the viewer.

Specular Strength Specularity Aperture
1.0 10.0

Adding a Specular Mask

Select the Specular Mask check box ().

Drag the following texture to the Unspecified section of the specularity mask.

The specularity has disappeared from the areas with black lines, as shown in the following figure.

Info

A ★ symbol is displayed in locations that have changed since the last save.
The ★ symbol disappears if you undo the changes or save the file again.

Adding Normal Maps

In the property window, in the tree list view, select Normals.

Select the Enable normal vector map check box ().

Drag the following texture to the Unspecified section of the normal map.

Normal Map
