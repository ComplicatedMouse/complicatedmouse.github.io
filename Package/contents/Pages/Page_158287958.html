This page introduces features that were added to the 3D module in NintendoSDK 0.12.0.

Maya Plug-In

This section describes both new features and those that were improved for the Maya plug-in.

Added a feature to the Maya plug-in for exporting the RA components of vertex color.

When 5 is specified for the output components for vertex color, you can now export just the RA components.
One use of this feature is for reducing the amount of data that is exported when you bake ambient occlusion information to vertex color and the RGB components are all the same value.

The following example shows how to use this feature. By setting the name of the color set as shown, only the RA components will be exported.

Texture pattern animation now can be exported even if keys are not set directly to the Frame Extension attribute of the file node.

The Plug-In for Setting Texture Pattern Animations has been improved!
Texture pattern animation now can be exported even if keys are not set directly to the Frame Extension attribute of the file node (and instead set via utility nodes or in another way).

They can be exported even in a case like the following, where the image number of the file node (the Frame Extension attribute) is bound to the Multiply Divide utility node.
