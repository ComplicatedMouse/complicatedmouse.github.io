Is there a way to speed up data transfers to the viewer when using fonts?

Question

How can I speed up data transfers to the viewer when using fonts?

Answer

If you place the font source files (XLOR and BMP files) together with the font file (BFFNT file), only the fonts used by the layout data are automatically generated and sent to the viewer, which can increase the transfer speed.

For example, put the XLOR and BMP files in the same folder as the BFFNT file, or set up your file structure as follows: ./font/XXXX.bffnt ./xlor/XXXX.xlor ./bmp/XXXX.bmp.

The following notification is displayed in the status bar at the bottom of the window. This occurs when the preview is being transferred, and the automatic generation is successful: "Automatically generated fonts with reduced size to shorten transfer time. Processing time ---- ... ms."
