Rendering Effects Only

This section describes the Render Effects Only feature that sets whether to render only the drop shadow effect.

Overview

The Render Effects Only feature sets whether to render only the drop shadow effect.

When Render Effects Only is enabled, only the drop shadow effect is displayed, without drawing the pane itself.
