NintendoSDK Migration Guide for Migrating From NW4C Font

NW4C Font (font2) is based on NW4F Font, which enables you to follow the instructions in NintendoSDK Migration Guide for Migrating From NW4F Font to migrate from NW4C Font to NintendoSDK.

If you are migrating from font1, see the Font Overview section in the NW4C documentation for an overview of the differences between font1 and font2 before continuing to read this document.
