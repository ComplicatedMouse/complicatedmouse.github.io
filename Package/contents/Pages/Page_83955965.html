Sequence Variables

What Are Sequence Variables?

The Sequence Variables window is used for viewing and changing sequence variables on SoundMaker, and for checking them via PC emulation playback or with AtkPlayer (when SoundMaker is connected to AtkPlayer).

Sequence Variables Operations

Opening the Sequence Variables Window

To open the sequence variable window, select Display > Sequence Variable or click Display Sequence Variable on the toolbar.
The shortcut ALT6 can also be used for this operation.
These operations are used to switch between showing and hiding windows.

Sequence Variables Operations

You can automatically write changes by right-clicking in the upper part of the Sequence Variables window, and then selecting Auto-write on the shortcut menu that appears.
When Auto-write is set to off, the Write button in the lower-right part of the window becomes available and you can write at any time.

When you edit values, click the small slider button to display the larger slider.

　　　

To invert all settings at the same time, click Invert Selections. To clear all settings at the same time, click Clear.

In addition, you can Play or Stop All sound with shortcut key operations when the focus is on the sequence variable window while the sequence variable window is in a docked state.

  • F9: Playback
  • F10: Stop all
