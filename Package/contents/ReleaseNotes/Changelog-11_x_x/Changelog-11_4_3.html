Details

Changes in NintendoSDK NX Addon 11.4.3 (based on Generic 11.4.3)

Improvements

11-4-3S01: (CPU Profiler) Nintendo NX CPU Profiler Version 1.5.1

  • Improved the error messages that appear when the scripting command Profiler-Assembly fails.
  • Fixed bugs that could stop the profiler from locating nnSdk.nss.
  • Fixed a bug where the setting to disable the blinking of the Overview tab when a warning occurs was not applied.
  • Fixed a bug where the Overview tab showed the incorrect number for Seen Functions in Profile Info.
  • Fixed a bug when using C++ lambda functions that could cause a checker to produce an error.
  • Fixed a bug where the Reset All to Defaults button on the Checkers tab did not reset checkers that were not expanded.
  • Fixed a bug that caused Assembly tab tooltips to disappear too quickly.


Bug Fixes

11-4-3S02: (CPU Profiler) Fixed a crash if the profiler library was initialized and finalized hundreds of times.

Fixed a crash that occurred if the profiler library was initialized and finalized hundreds of times without restarting the profiled application.
A profiler thread remained after profiler::Finalize.

11-4-3S03: (NIFM) Fixed a bug where the program aborted when the nifm library API was called in multiple threads.

Fixed a bug where the program aborted if the nifm library was initialized and then the first call to the nifm API overlapped in multiple threads.
