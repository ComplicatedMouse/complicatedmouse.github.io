Summary

Changes in NintendoSDK NX Addon 11.4.5 (based on Generic 11.4.5)

Added Features

  • 11-4-5S01: (TOOL) Added the ability to split up an NSP file created by merging a patch and a compacted application to the AuthoringToolKit splitnsp command.

Improvements

  • 11-4-5S02: (TOOL) Made fixes to multi-application card documentation.

Bug Fixes

  • 11-4-5S03: (TOOL) Fixed a bug in AuthoringTool where files remained when a patch generation error occurred.
  • 11-4-5S04: (TOOL) Fixed a bug in AuthoringTool where NSP files for secure startup could not be created from a compacted application.
  • 11-4-5S05: (TOOL) Fixed a bug in AuthoringTool where the path specified in the -o option was ignored when running the splitnsp command.

Details

Changes in NintendoSDK NX Addon 11.4.5 (based on Generic 11.4.5)

Added Features

11-4-5S01: (TOOL) Added the ability to split up an NSP file created by merging a patch and a compacted application to the AuthoringToolKit splitnsp command.

Added the ability to split up an NSP file created by merging a patch and a compacted application to the AuthoringToolKit splitnsp command.
For more information, see the following documentation.

  • Features > Building > Creating Application Images > Tool Reference > AuthoringTool > Splitting NSP Files.


Improvements

11-4-5S02: (TOOL) Made fixes to multi-application card documentation.

Fixed documentation to specify pre-conversion additional content instead of on-card downloadable content when creating a multi-application card with DLC.
For more information, see the following documentation.

  • Features > Building > Creating Application Images > Tool Reference > AuthoringTool > Creating an NSP File for the Production of a Multi-Application Card.
  • Features > NX Development Cards > Multi-Application Cards


Bug Fixes

11-4-5S03: (TOOL) Fixed a bug in AuthoringTool where files remained when a patch generation error occurred.

Fixed a bug in AuthoringTool where the compacted application remained when patch generation using the --do-application-compaction option resulted in an error.

11-4-5S04: (TOOL) Fixed a bug in AuthoringTool where NSP files for secure startup could not be created from a compacted application.

Fixed a bug in AuthoringTool where NSP files for secure startup could not be created from a compacted application.

11-4-5S05: (TOOL) Fixed a bug in AuthoringTool where the path specified in the -o option was ignored when running the splitnsp command.

Fixed a bug in AuthoringTool where the output file was always placed in the folder where the input source NSP file was located when running splitnsp for NSP files like the following.

  • NSP files merging a compacted application and patch
