Details

Changes in NintendoSDK NX Addon 11.4.7 (based on Generic 11.4.7)

Improvements

11-4-7S01: (TOOL) Tightened the NMETA file check when running AuthoringTool.

Tightened the NMETA file check when running AuthoringTool and modified it so that an error occurs if no valid value is configured.

Bug Fixes

11-4-7S02: (GFX) Fixed a bug where eglSetBlobCacheFuncsANDROID() did not work correctly.

Fixed a bug where calling eglSetBlobCacheFuncsANDROID() caused eglGetError() to return EGL_BAD_ACCESS, and the provided callback function was not called.

11-4-7S03: (NGC) Fixed a bug where masking failed when strings with a certain string pattern were input for nn::ngc::ProfanityFilter::MaskProfanityWordsInText().

Fixed a bug where masking failed when strings with a certain pattern were passed to the <em>pText</em> argument of the nn::ngc::ProfanityFilter::MaskProfanityWordsInText() function.

11-4-7S04: (TOOL) Fixed a bug where patch creation with AuthoringTool failed on rare occasions.

Fixed a bug where the message "[Error] Failed to Build RelocatedIndirectStorageBuilder 0x001F4002" was displayed and the creation of the patch NSP file failed on rare occasions, depending on where the delta from the previous version was located.

11-4-7S05: (TOOL) Fixed a bug where the large size icon and small size icon generated from the same BMP image sometimes did not match.

Fixed a bug where the large size icon and small size icon generated from the same BMP image were judged as not matching when creating an application image in AuthoringTool.
With this fix, an error now occurs if the large size icon and small size icon do not match.
