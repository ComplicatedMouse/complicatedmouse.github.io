Changes in NintendoSDK NX Addon 10.8.2 (based on Generic 10.8.2)

Bug Fixes

*10-8-2S01: (libcurl) Fixing the issue with curl_easy_pause() that can potentially block other threads.

There was a scenario where curl_easy_pause() can block another thread from running for too long if a calling thread is given the same thread priority as the other thread. Now the issue has been fixed.
