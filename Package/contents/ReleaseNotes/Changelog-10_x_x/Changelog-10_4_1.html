Details

Changes in NintendoSDK NX Addon 10.4.1 (based on Generic 10.4.1)

Added Features

10-4-1S01: (FS) Added the function nn::fs::IsMounted() for determining whether the mount name is already mounted.

Added the function nn::fs::IsMounted() for determining whether the mount name is already mounted.

10-4-1S02: (HID) HID Shell now supports the keyboard and mouse.

You can now perform autopilot operations on the keyboard and mouse through HID Shell.

10-4-1S03: (HID) Input Director now supports transfer of input from the Nintendo Switch Pro Controller.

Input Director now supports transfer of input from the Nintendo Switch Pro Controller. However, this support is limited to Bluetooth connections. If you are using Input Director in combination with RemoteFX, use a Bluetooth USB adapter and connect to that USB adapter with RemoteFX.

Improvements

10-4-1S04: (TOOL) You can now create NSP files for secure startup from NSP files that were created with earlier versions of the SDK.

The makesecure command for creating an NSP file for secure startup now succeeds even if the major version of the SDK that was used to create the input NSP file is earlier than the major version of AuthoringTool. Previously, the major versions needed to be the same for both.

Bug Fixes

10-4-1S05: (HID) Fixed a bug where Input Director could not detect some of the devices redirected by RemoteFX.

Fixed a bug where Input Director could not detect some of the devices redirected by RemoteFX. With this change, input now can be transferred across remote desktops even from DirectInput gamepads and LCD tablets.
