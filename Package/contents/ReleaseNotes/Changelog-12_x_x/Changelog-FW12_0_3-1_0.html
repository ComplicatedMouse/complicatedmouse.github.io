Details

Changes in NintendoSDK NX Addon 12.3.4 (based on Generic 12.3.4) FW 12.0.3-1.0

Bug Fixes

*FW12-0-3-1-0S01: (SSL) Temporarily disabling TLS 1.3 due to rare connection failures.

Due to rare connection failures in specific network environments, TLS 1.3 has been temporarily disabled. When passing nn::ssl::SslVersion_Auto to nn::ssl::Context::Create(), nn::ssl::DoHandshake() only negotiates up to TLS 1.2. When passing nn::ssl::SslVersion_TlsV13 to nn::ssl::Context::Create(), nn::ssl::DoHandshake() uses TLS 1.2.

FW12-0-3-1-0S02: (EC) Fixed a bug where the dummy ec applet displayed a warning on returning from sleep mode.

Fixed a bug where when returning from sleep mode after more than 30 seconds had elapsed since entering sleep mode, the dummy EC applet displayed a "Warning: application background tasks are too heavy" message regardless of the background processing load.

FW12-0-3-1-0S03: (FS) Fixed a bug where calling the nn::fs API while having specified a particular path string caused the app to forcibly close.

Fixed a bug present since NintendoSDK Firmware for NX 12.0.0-1.0 where specifying a path string that satisfies the following conditions when calling the nn::fs API caused the app to be force-terminated.

  • Application that uses NintendoSDK 0.x.x or 1.x.x
  • Includes one of the following prohibited characters: ":", "*", "?", "<", ">", "|".
  • Includes one of the following: "..\", "\..".
  • Applications that use NintendoSDK 7.x.x through 11.x.x
  • One or more repetitions of the "/" or "\" characters after mount name + ":/" and the entry name includes one of the following prohibited characters: ":", "$".


In an application using NintendoSDK 12.x.x or later, calling the nn::fs API by specifying a path string that includes prohibited characters causes the application to be force-terminated due to precondition violations.
For more information about the conditions for valid path strings, see the API reference > nn::fs namespace > More...

FW12-0-3-1-0S04: (TOOL) Added support for update to system version 12.0.3.

Added support for update to system version 12.0.3.
