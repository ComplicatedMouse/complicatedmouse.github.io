Added Features

12-3-7S01: (SSL) Introducing nn::ssl::FixedBuiltInManager to retrieve the bundled CA certificate data

The way to retrieve the bundled CA certificate data was added in the nn::ssl. There are limitations to use this feature. Also, it is required to get an approval by Nintendo to use it. Please refer the API reference of nn::ssl::FixedBuiltInManager for details.
