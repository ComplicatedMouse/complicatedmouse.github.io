Summary

  • 12-3-4S01: (TOOL) Added the ability to split up an NSP file created by merging a patch and a compacted application to the AuthoringToolKit splitnsp command.

Bug Fixes

  • 12-3-4S03: (TOOL) Fixed a bug in AuthoringEditor where a portion of the information included in NSP files created in version 7.0.0 or earlier of the SDK could not be confirmed.
  • 12-3-4S04: (TOOL) Fixed a bug in AuthoringEditor where an error message was incorrectly displayed when indexes and tags for on-card DLC in a multi-application card (MAC) were duplicated.
  • 12-3-4S05: (TOOL) Fixed a bug in AuthoringTool where files remained when a patch generation error occurred.
  • 12-3-4S06: (TOOL) Fixed a bug in AuthoringTool where NSP files for secure startup could not be created from a compacted application.

Details

Changes in NintendoSDK NX Addon 12.3.4 (based on Generic 12.3.4)

Added Features

12-3-4S01: (TOOL) Added the ability to split up an NSP file created by merging a patch and a compacted application to the AuthoringToolKit splitnsp command.

Added the ability to split up an NSP file created by merging a patch and a compacted application to the AuthoringToolKit splitnsp command.
For more information, see the following documentation.

  • Features > Building > Creating Application Images > Tool Reference > AuthoringTool > Splitting NSP Files.


Improvements

12-3-4S02: (TOOL) Made fixes to multi-application card documentation.

Fixed documentation to specify pre-conversion additional content instead of on-card downloadable content when creating a multi-application card with DLC.
For more information, see the following documentation.

  • Features > Building > Creating Application Images > Tool Reference > AuthoringTool > Creating an NSP File for the Production of a Multi-Application Card.
  • Features > NX Development Cards > Multi-Application Cards


Bug Fixes

12-3-4S03: (TOOL) Fixed a bug in AuthoringEditor where a portion of the information included in NSP files created in version 7.0.0 or earlier of the SDK could not be confirmed.

Fixed a bug in AuthoringEditor where the following information contained in NSP files created in an environment with AuthoringToolKit 7.0.0 or earlier could not be confirmed.

  • Offline HTML
  • The list of URLs of the Internet sites that the web applet may reference
  • Software legal information


12-3-4S04: (TOOL) Fixed a bug in AuthoringEditor where an error message was incorrectly displayed when indexes and tags for on-card DLC in a multi-application card (MAC) were duplicated.

Fixed a bug where an error message was incorrectly displayed in AuthoringEditor when indexes and tags for on-card DLC for multiple applications on a multi-application card (MAC) were duplicated.

12-3-4S05: (TOOL) Fixed a bug in AuthoringTool where files remained when a patch generation error occurred.

Fixed a bug in AuthoringTool where the compacted application remained when patch generation using the --do-application-compaction option resulted in an error.

12-3-4S06: (TOOL) Fixed a bug in AuthoringTool where NSP files for secure startup could not be created from a compacted application.

Fixed a bug in AuthoringTool where NSP files for secure startup could not be created from a compacted application.

12-3-4S07: (TOOL) Fixed a bug in AuthoringTool where the path specified in the -o option was ignored when running the splitnsp command.

Fixed a bug in AuthoringTool where the output file was always placed in the folder where the input source NSP file was located when running splitnsp for NSP files like the following.

  • NSP files merging a compacted application and patch


12-3-4S08: (TOOL) Fixed a bug where a crash occurred if you entered text in the AuthoringEditor search box.

Fixed a bug where entering text into the search text box displayed when checking or editing NMETA or NSP files in AuthoringEditor caused the application to crash.
