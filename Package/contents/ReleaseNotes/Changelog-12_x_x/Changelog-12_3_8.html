Details

Changes in NintendoSDK NX Addon 12.3.8 (based on Generic 12.3.8)

Improvements

12-3-8S01: (TOOL) A message is now displayed if the AuthoringTool makepatch command is run and the SDK version of the latest application is lower than the SDK version of the previous application.

If you run the AuthoringTool makepatch command and the SDK version of the latest application is earlier than the SDK version of the previous application, an error message stating that the SDK version is earlier is displayed even if the --error-unpublishable option has not been specified.
For more information, see the following documentation.

  • Features > Patches > Building and Running a Patch > Creating the Patch.


Bug Fixes

12-3-8S02: (TOOL) Fixed a bug where patch creation with AuthoringTool failed on rare occasions.

Fixed a bug where the message "[Error] Failed to Build RelocatedIndirectStorageBuilder 0x001F4002" was displayed and the creation of the patch NSP file failed on rare occasions, depending on where the delta from the previous version was located.
