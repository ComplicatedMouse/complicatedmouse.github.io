Details

Changes in NintendoSDK NX Addon 9.3.1 (based on Generic 9.3.1)

Bug Fixes

9-3-1S01: (NVN) Fixed a bug involving GLSLC and OpenGL where invalid memory was sometimes accessed when Bindless Textures or Separate Texture was used.

Fixed a bug involving GLSLC and OpenGL included in NX Add-On 8.1.0 and later where invalid memory was sometimes accessed when Bindless Textures or Separate Texture was used after explicitly specifying LOD.
This bug could cause MMU faults and texture corruption at run time.

9-3-1S02: (VSI) Processes such as breaks no longer take as much time when debugging applications built with Clang 1.8.4 or later.

Processes such as breaks no longer take as much time when debugging applications built with Clang 1.8.4 or later.

Changes in NintendoWare NX Addon 9.3.1 (based on Generic 9.3.1)

Bug Fixes

9-3-1W01: (Audio-ATK) Fixed a bug where quickly playing and stopping stream sounds caused an assertion failure in the debug and develop builds.

Fixed a bug where quickly playing and stopping stream sounds caused an assertion failure when SubMix and FinalMix were finalized in the debug and develop builds.
