Details

Changes in NintendoSDK NX Addon 0.12.6 (based on Generic 0.17.6)

Added Features

13-7S01: (NFP) Added a tool for checking the items in the NFP guidelines.

Added a tool for checking the guidelines for applications that use the nfp library and for debugging such applications.

For more information about using this tool, see NFP Library / Tool Reference / NfpDebugTool.

13-7S02: (SWKBD) Added resources for the character set that can be used as input via the software keyboard.

Added resources (XLLT and XLOR) related to the character set that can be used as input via the software keyboard. They are located in Resources/Swkbd/NX.

Changes in NintendoWare NX Addon 0.12.6 (based on Generic 0.17.6)

Bug Fixes

13-7W01: (Audio-ATK) Fixed a bug where a particular value in the sound profile information was sometimes very large.

Fixed a bug where the value of the nwFrameProcess variable of the sound profile information in the nn::atk::SoundProfile structure obtained using the nn::atk::ProfileReader class was sometimes very large.
Also fixed a bug where nwFrameProcess included the time for processes that are conducted only when profiling is enabled.
Note that this fix might make it seem that performance has improved when in fact it has not.
