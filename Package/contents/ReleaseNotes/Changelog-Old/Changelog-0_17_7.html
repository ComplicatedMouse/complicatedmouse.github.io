Changes in NintendoSDK NX Addon 0.12.7 (based on Generic 0.17.7)

Improvements

13-8S01: (NGC) Added the nn::ngc::ProfanityFilter::Finalize() function.

Added the nn::ngc::ProfanityFilter::Finalize() function for finalizing instances of nn::ngc::ProfanityFilter.
If the nn::ngc::ProfanityFilter instance is destroyed without calling nn::ngc::ProfanityFilter::Finalize(), the function will be called automatically by the destructor.

Bug Fixes

13-8S02: (NGC) Fixed a bug where illegal memory access occurred when the features of the ngc library were used with multiple instances of nn::ngc::ProfanityFilter.

Fixed a bug that occurred when multiple instances of nn::ngc::ProfanityFilter were being used where the second and subsequent instances could use the working buffer region that was passed to the first instance.

Changes in NintendoWare NX Addon 0.12.7 (based on Generic 0.17.7)

Improvements

13-8W01: (Effect-VFX) Added an index attribute to the particle attribute buffer that is created in the VFX runtime.

Added an index attribute to the particle attribute buffer that is created in the VFX runtime so it can also be used as an index buffer.
