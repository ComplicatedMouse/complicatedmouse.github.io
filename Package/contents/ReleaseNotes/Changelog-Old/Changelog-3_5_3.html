Details

Changes in NintendoSDK NX Addon 3.5.3 (based on Generic 3.5.3)

Bug Fixes

3-5-3S01: (NGC) Fixed a bug where the profanity check feature did not work depending on the NMETA settings.

Fixed a bug in the NintendoSDK Firmware for NX 4.0.0 environment where the profanity check did not work correctly depending on the NMETA settings.

This bug was only found when you used the NMETA settings that were prohibited in versions prior to NX Add-On 4 series.

3-5-3S02: (NGC) Fixed a bug where the profanity check feature did not work depending on the NMETA settings.

Fixed a bug in the NintendoSDK Firmware for NX 4.0.0 environment where the profanity check did not work correctly depending on the NMETA settings.

This bug was only found when you used the NMETA settings that were prohibited in versions prior to NX Add-On 4 series.
