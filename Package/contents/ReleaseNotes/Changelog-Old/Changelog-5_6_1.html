*5-6-1S01: (TOOL) Revisions to the Rating Specifications

The following changes were made to the specifications for ratings, and this changes the ratings specifications in the NMETA file and in AuthoringEditor.
* For Japan’s CERO ratings, the age set for “審査予定 (Rating Scheduled)” and “規定適合 (Regulations-Compatible)” was changed from 0 to 18.
* For Taiwan’s GSRMR ratings, the age set for “審査中” was changed from 0 to 18.
* For Australia’s ACB ratings, the age set for “Check the Classification” was changed from 0 to 18.
* Ages have been standardized for the PEGI and PEGI Portugal ratings, so the entry field for PEGI Portugal has been deleted.
For more information, see Rating in NintendoSDK Documents > Features > Building > Creating Application Images > Writing the NMETA File.
