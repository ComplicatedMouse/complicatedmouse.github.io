Summary

Changes in NintendoSDK NX Addon 1.0.0 (based on Generic 1.0.0)

Specification change that breaks your code

  • *1-0-0S01: (FS) Standardized application behavior for when an application attempts to perform an operation on a read-only file system.
  • *1-0-0S02: (GFX) Added a rendering framework management feature to nns::gfx::GraphicsFramework, and added the FrameworkDemo sample.
  • *1-0-0S03: (HTCS) Improved the behavior of nn::htcs library functions when a connection to a host has not been established.
  • *1-0-0S04: (HTCS) Made several fixes to the htcs library to support the Target Manager 1.0.2.17834 update.
  • *1-0-0S05: (IRSENSOR) Fixed a bug where when using image transfer mode with the size set to QQQVGA, the leftmost column of pixels in the image were extremely dark.
  • *1-0-0S06: (TOOL) Changed the size of reflection structures in Shader Compiler.
  • *1-0-0S07: (VSI) Updated NX VSI and Target Manager to version 1.0.2.17834.

You can use the nn::vfx::System::SetGpuHeapReleaseCounter function to manually adjust how many frames to wait in the application.
