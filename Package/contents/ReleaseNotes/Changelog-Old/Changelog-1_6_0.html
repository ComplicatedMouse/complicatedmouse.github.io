Details

Changes in NintendoSDK NX Addon 1.6.0 (based on Generic 1.6.0)

Improvements

1-6-0S01: (GFX) The NVN version of the GFX library no longer caches the results of GetGpuAddress.

NX Add-On 1.5.0 included a change in which NVN was not called when the nn::gfx::Buffer::GetGpuAddress function in the NVN version of the GFX library was called. The function returned a result that had previously been retrieved and cached. This change has been undone.

1-6-0S02: (TOOL) Improved input of rating information in AuthoringEditor.

Made rating information easier to input.
Now, after the region has been selected, AuthoringEditor only displays input items related to the ratings required in that region.

Bug Fixes

1-6-0S03: (3DTool) Fixed a bug where 3D Shader Converter sometimes became unresponsive when running in a Windows 7 environment.

Fixed a bug where 3D Shader Converter (<code>%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dShaderConverter.exe</code>) sometimes became unresponsive when running in a Windows 7 environment if the --shader-cache-directory option was used.

Changes in NintendoWare NX Addon 1.6.0 (based on Generic 1.6.0)

Specification change that breaks your code

*1-6-0W01: (3D-G3D) Updated the binary resource file format.

Updated the binary resource file format to support the recent changes to the data structures in nn::gfx. This change breaks compatibility with binary resource files created in previous versions of 3D binary converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe). These resource files must be reconverted.

Improvements

1-6-0W02: (3DTool) Reduced the size of binary files exported from the 3D binary converter.

Removed unnecessary regions that were previously included in binary data exported from 3D Binary Converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe). This change may reduce the size of binary files that include nn::g3d::ResMesh and nn::g3d::ResVertex data.

Bug Fixes

1-6-0W03: (Audio-ATK) Fixed a bug where an assertion failure occurred if StartOffset was specified near the end of a looped waveform in a stream sound.

Fixed a bug where an assertion failure sometimes occurred if StartOffset was specified in a stream sound, that used a looped waveform, and the end of the specified loop region occurred at the same point as the starting playback position.

1-6-0W04: (3DTool) Fixed a bug where 3D binary converter failed to convert model intermediate files if the polygon reduction feature was used in Intermediate File Optimizer.

Fixed a bug where 3D binary converter (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dBinaryConverter.exe) failed to convert model intermediate files (FMDB files) if the polygon reduction feature in Intermediate File Optimizer (%NINTENDO_SDK_ROOT%\Tools\Graphics\3dTools\3dIntermediateFileOptimizer.exe) was used when the files were exported.
