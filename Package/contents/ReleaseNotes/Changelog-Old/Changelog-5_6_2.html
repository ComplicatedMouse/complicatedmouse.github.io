Details

Changes in NintendoSDK NX Addon 5.6.2 (based on Generic 5.6.2)

Added Features

*5-6-2S01: (OE) Changed the crash report creation feature to be enabled by default.

Changed the crash report creation feature to be enabled by default.
This is also enabled when there is no option specified for the CrashReport option in the NMETA file.

Bug Fixes

*5-6-2S02: (TOOL) Fixed a bug where an error would occur when opening a patch NSP file in AuthoringEditor.

Fixed a bug where an error would occur and file opening would fail when opening a patch NSP file in AuthoringEditor.
Also fixed a bug where data corruption was constantly displayed because corruption detection for application NSP files did not work correctly.
