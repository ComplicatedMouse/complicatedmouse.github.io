Details

Changes in NintendoSDK NX Addon 0.12.12 (based on Generic 0.17.12)

Added Features

13-13S01: (General) Added a feature to DevMenu to display the global IP address, the NAT type, and the MAC address.

The global IP address, the NAT type, and the MAC address are now displayed in the Network menu of DevMenu.

Improvements

*13-13S02: (HID) Updated the Joy-Con firmware for improved NFC communications.

Improvements have been made to the characteristics of NFC communications.
To apply these improvements to the system, you must update the Joy-Con (R) firmware to 3.48.3.0b.

13-13S03: (OE) Added a warning when building with a performance configuration that will be deprecated.

A warning is now output when you build using the following performance configurations, which are set to be deprecated.

  • nn::oe::PerformanceConfiguration_Cpu1020MhzGpu307MhzEmc1600Mhz
  • nn::oe::PerformanceConfiguration_Cpu1020MhzGpu384MhzEmc1600Mhz


Bug Fixes

13-13S04: (TOOL) Fixed a bug where InitializeSdev would sometimes fail to initialize the SDEV MP.

Fixed a bug where InitializeSdev would sometimes display [ERROR] Write serial number to HostBridge failed and fail when it tried to initialize the SDEV MP.

13-13S05: (TOOL) Fixed a bug where NSP file creation would fail depending on the names of the entries in the resource data.

Fixed a bug where creation of the NSP file would fail if any of the names of the entries in the resource data contained multiple byte characters.

Changes in NintendoWare NX Addon 0.12.12 (based on Generic 0.17.12)

Bug Fixes

13-13W01: (Audio-ATK) Fixed a bug in rolling back the sample count for the stream sound playback position when loading was delayed or there was a pause.

Fixed a bug where the value returned by nn::atk::StreamSoundHandle::GetPlaySamplePosition would be around 10000 samples less than the correct value when loading was delayed or there was a pause.
