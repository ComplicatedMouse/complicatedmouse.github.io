Specification change that breaks your code

*4-1-0S01: (NFC) Deprecated the mifare library and added the read/write MIFARE feature and the send commands with pass-through feature to the NFC library.

The following features have been added to the NFC library.

  • A feature to read/write MIFARE
  • A feature to send commands to NFC tags using pass-through



Along with this change, the mifare library has been deprecated.
The following list shows the correspondence between the deprecated mifare library functions and the functions added to the NFC library for reading and writing MIFARE.

  • nn::nfc::mifare::AttachActivateEvent() → nn::nfc::AttachActivateEvent()
  • nn::nfc::mifare::AttachDeactivateEvent() → nn::nfc::AttachDeactivateEvent()
  • nn::nfc::mifare::AttachAvailabilityChangeEvent() → nn::nfc::AttachAvailabilityChangeEvent()
  • nn::nfc::mifare::GetDeviceState() → nn::nfc::GetDeviceState()
  • nn::nfc::mifare::GetNpadId() → nn::nfc::GetNpadId()
  • nn::nfc::mifare::ListDevices() → nn::nfc::ListDevices()
  • nn::nfc::mifare::StartDetection() → nn::nfc::StartDetection()
  • nn::nfc::mifare::StopDetection() → nn::nfc::StopDetection()
  • nn::nfc::mifare::GetTagInfo() → nn::nfc::GetTagInfo()
  • nn::nfc::mifare::Read() → nn::nfc::ReadMifare()
  • nn::nfc::mifare::Write() → nn::nfc::WriteMifare()


In addition, added the following functions to the NFC library for sending commands using pass-through.

  • nn::nfc::SendCommandByPassThrough()
  • nn::nfc::KeepPassThroughSession()
  • nn::nfc::ReleasePassThroughSession()


*4-1-0S02: (TOOL) Separated the VSI for NX and Target Manager packages.

The packages (zarf) for VSI for NX and for Target Manager have been reorganized.

* Before the change: NX Visual Studio Integration and Target Manager were both included in the VSI for NX package.
* After the change: NX Visual Studio Integration is included in the VSI for NX package, and Target Manager is in the Target Manager package.

The Target Manager package is required for running applications on the NX development hardware.
The new VSI for NX package is required for building and debugging applications using Visual Studio.

Install the packages required for your purposes.

For both, if you use NDI to install the package, the installation to the Windows system will also be performed. If you want to install the package to the system at another time, execute the following batch files as the administrator.

* For VSI for NX, run Installers\Vsi\Install.bat.
* For Target Manager, run Installers\TargetManager\Install.bat.
