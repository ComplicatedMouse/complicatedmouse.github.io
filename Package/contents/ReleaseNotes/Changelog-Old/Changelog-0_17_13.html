Changes in NintendoWare NX Addon 0.12.13 (based on Generic 0.17.13)

Bug Fixes

*13-14W01: (Effect-VFX) Fixed a bug where compute shader attributes would occasionally become corrupted in the VFX runtime.

Fixed a bug where, depending on the timing of the processing of the VFX runtime, unexpected values occasionally would be written to the attributes read and written by the compute shader, causing corrupted display.
Because revisions have been made to the standard feature VFX shader of EffectMaker, you must reconvert your effect binary.
The revisions made to the internal handling of the gfx memory pool attributes and runtime attributes reduce the overall CPU load and increase the GPU load, but the GPU load only increases by approximately 0.2 percent for each 10,000 particles, so there is not much effect except when a huge number of effects are playing.
