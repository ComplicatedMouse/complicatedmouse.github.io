Details

Changes in NintendoWare NX Addon 8.2.1 (based on Generic 8.2.1)

Bug Fixes

8-2-1W01: (LayoutTool) Fixed a bug in LayoutEditor where some of the Parts and Control Settings were cut off.

Fixed a bug in LayoutEditor where some of the Parts and Control Settings were cut off.

8-2-1W02: (LayoutTool) Fixed a bug in LayoutEditor where switching segments took too long

Fixed a bug in LayoutEditor where switching segments took too long. Unnecessary processing is no longer run when the animation collision warning (<IsWarnAnimTagCollisionEnabled>) is disabled in the project configuration file. We also improved the processing speed when the animation collision warning is enabled.
