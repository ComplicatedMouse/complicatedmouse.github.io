Details

Changes in NintendoSDK NX Addon 0.12.10 (based on Generic 0.17.10)

Added Features

*13-11S01: (FRIENDS) Added to the library that calls the My Page applet.

Added the following functions for the My Page applet.

- nn::friends::ShowFriendList()
- nn::friends::ShowUserDetailInfo()
- nn::friends::StartSendingFriendRequest()
- nn::friends::ShowMethodsOfSendingFriendRequest()
- nn::friends::StartFacedFriendRequest()
- nn::friends::ShowReceivedFriendRequestList()
- nn::friends::ShowBlockedUserList()

For more information about this applet, see Library Applets / My Page Applet.

*13-11S02: (General) Added support for infrastructure mode communication in the final testing environment (PROD MODE).

Added a feature to DevMenuApp to set Debug > Network Service to Enabled so infrastructure mode communication can be used in the final testing environment (PROD MODE).

*13-11S03: (General) Integrated SystemSoftwareKit.

Integrated the content of SystemSoftwareKit into the NintendoSDK NX Add-On package and the NintendoSDK System Updater for NX package.
From this point on, SystemSoftware will be included as standard practice when the development hardware is initialized.

*13-11S04: (HID) Added to the library that calls the Controller Support applet.

Added the following functions for the Controller Support applet.

- nn::hid::ShowControllerSupport()
- nn::hid::SetExplainText()

For more information about this applet, see Library Applets / Controller Support Applet

*13-11S05: (SWKBD) Added to the library that calls the Software Keyboard applet.

Added functions for the Software Keyboard applet to the nn::swkbd namespace.
For more information about this applet, see Library Applets / Software Keyboard Applet.
In addition, the sample program SwkbdSimple was added.

*13-11S06: (WEB) Added to the library calling the Web applet and the Offline Web applet.

Added functions for the Web applet and the Offline Web applet to the nn::web namespace.

For more information about these applets, see Library Applets / Web Applet and Library Applets / Offline Web Applet.

*13-11S07: (Accounts) Added to the library that calls the Account Management applet.

Added the following functions for the Account Management applet.
Also added the AccountApplication sample program.

- nn::account::ShowUserSelector()
- nn::account::ShowUserCreator()

For more information, see Library Applets / Account Management Applet

13-11S08: (NIFM) Added a function to get the proxy settings.

Added the nn::nifm::GetCurrentProxySetting() function to get the proxy settings that have been specified for the network connection being used.

Improvements

13-11S09: (FS) Merged the FS Speed Emulation Mode selections OFF and Faster in Devmenu.

The OFF and Faster modes have the same behavior in the current version of the FS Speed Emulation Feature, so for FS Speed Emulation Mode in the Debug tab of DevMenu, these two selections have been merged as OFF / Faster. In the future, Faster mode will be faster than OFF mode, so at that time the two modes will be separated again.

Bug Fixes

13-11S10: (AUDIO) Fixed a bug in the nn::audio::SetReverbInputOutput() function.

Fixed a bug where the audio renderer would become unstable and the system would crash if the input/output was switched by the nn::audio::SetReverbInputOutput() function when the nn::audio::ReverbType structure was in use.

This bug occurred if nn::audio::RequestUpdateAudioRenderer() was called without the input/output having been set by the n::audio::SetReverbInputOutput() function after the addition of nn::audio::ReverbType.

As part of this bug fix, the state of nn::audio::ReverbType is now reset when nn::audio::SetReverbInputOutput() is called.
Note that noise can get mixed in the waveforms if nn::audio::SetReverbInputOutput() is called while nn::audio::ReverbType is active.

13-11S11: (MOVIE) Fixed a bug where control would not return from Decoder::Stop() if the function was called before End of Stream was returned.

Fixed a bug where program control would not return from the Decoder::Stop() function if it was called before the result of sending End of Stream to Decoder was returned.

13-11S12: (TOOL) Corrected typos in the rating organization names in the metafile output by the AuthoringEditor tool.

Corrected the typos in some of the rating organization names in the metafile output by the AuthoringEditor tool, which caused an error when the file was input to the AuthoringTool.
The following names were corrected.

  • (Wrong) PEGI Portugal → (Correct) PEGIPortugal
  • (Wrong) PEGI+BBFC → (Correct) PEGIBBFC


13-11S13: (TOOL) Fixed a bug where BackupSaveData would generate an import error if the total amount of data in the save data was larger than the journal size.

Fixed a bug where BackupSaveData would generate an import error if the total amount of data in the save data was larger than the journal size.

An import error is still generated if the save data contains a file that is larger than the journal size. This bug will be fixed in the future.
