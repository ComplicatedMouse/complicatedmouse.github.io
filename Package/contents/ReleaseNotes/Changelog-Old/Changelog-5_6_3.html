Details

Changes in NintendoSDK NX Addon 5.6.3 (based on Generic 5.6.3)

Specification change that breaks your code

*5-6-3S01: (TOOL) Added a feature to the makepatch command of AuthoringTool that checks whether the NSP file can be submitted.

Added a feature to the makepatch command of AuthoringTool that checks whether the NSP file can be submitted.
To check whether the NSP file can be submitted, specify the --error-unpublishable option when you run the makepatch command.
This option checks for the same items that are checked by the get-unpublishable-error command of Features > Building > Creating Application Images > Tool Reference > AuthoringTool.
The creation of the NSP file fails if any items require fixing.
If there are any fatal errors, the creation of the NSP file fails even if the --error-unpublishable option has not been specified.
For more information about the makepatch command, see Features > Patches > Building and Running a Patch.

Improvements

*5-6-3S02: (TOOL) Added a feature to check for consistency of tool versions in the application and the patch.

Added a feature to AuthoringTool to check the version of the tool that was used when the NSP file was created for the application and for the patch.
An error is displayed when get-unpublishable-error is run if the NSP files for the application and the patch were created using different versions of the tool.
Also, the replace command fails if you run it to try to revise entries in those NSP files.
For more information about these commands, see Features > Building > Creating Application Images > Tool Reference > AuthoringTool.

*5-6-3S03: (TOOL) Creation of the NSP file now fails if the SDK used by the application and AuthoringTool have different major versions.

When AuthoringTool is used to create an NSP file, the process now fails if the major version of AuthoringTool is different from the major version of the SDK used by the application.

*5-6-3S04: (TOOL) Enabled using Hangul, Simplified Chinese, and Traditional Chinese for title names, publisher names, and tags in NMETA files.

Enabled using Hangul, Simplified Chinese, and Traditional Chinese for the following elements in NMETA files.

  • Application/Title/Name
  • Application/Title/Publisher
  • AddOnContent/Tag

For more information, see Features > Building > Creating Application Images > Writing the NMETA File and Features > System > aoc Library > Creating Downloadable Content.
