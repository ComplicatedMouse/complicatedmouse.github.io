Summary

Changes in NintendoSDK NX Addon 1.3.1 (based on Generic 1.3.1) FW 2.0.0-13

Bug Fixes

  • *1-3-1FW13S01: (BLUETOOTH) Fixed a bug where you could not wake from sleep mode using the Joy-Con™ HOME Button.
  • 1-3-1FW13S02: (Display) Fixed a bug where the screen would fade to black and hang after the Switch logo disappears, when starting.
  • 1-3-1FW13S03: (General) Fixed a bug where a 2162-2000 fatal error would occur if at least 24 hours had passed after staring the Nintendo Switch™ system with an account that is associated with a Nintendo Account.

Details

Changes in NintendoSDK NX Addon 1.3.1 (based on Generic 1.3.1) FW 2.0.0-13

Bug Fixes

*1-3-1FW13S01: (BLUETOOTH) Fixed a bug where you could not wake from sleep mode using the Joy-Con™ HOME Button.

Fixed a bug where you could not wake from sleep mode using the HOME Button on a Joy-Con™ controller removed from a Nintendo Switch™ system configured with Internet Settings that had entered sleep mode.

1-3-1FW13S02: (Display) Fixed a bug where the screen would fade to black and hang after the Switch logo disappears, when starting.

Fixed a bug where the screen would fade to black and hang after the Switch logo disappears, when starting.

1-3-1FW13S03: (General) Fixed a bug where a 2162-2000 fatal error would occur if at least 24 hours had passed after staring the Nintendo Switch™ system with an account that is associated with a Nintendo Account.

Fixed a bug where the following fatal error would occur if at least 24 hours had passed after starting the Nintendo Switch™ system with an account that is associated with a Nintendo Account.
ErrorCode: 2162-0002
Program: 0x0100000000000032
