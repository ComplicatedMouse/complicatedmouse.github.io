Details

Changes in NintendoSDK NX Addon 4.0.1 (based on Generic 4.0.1)

Bug Fixes

4-0-1S01: (General) Fixed bugs where an application was not updated correctly when setting its language to either Simplified or Traditional Chinese.

The following three bugs were present in NX Add-On 4.0 for Simplified and Traditional Chinese.

  • Could not correctly get icons for Simplified and Traditional Chinese.
  • Settings for Simplified and Traditional Chinese were reversed when getting them.
  • Could not get shared fonts for Simplified and Traditional Chinese.


These bugs were fixed in NX Add-On 4.0.1.
