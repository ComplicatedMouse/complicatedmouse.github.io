Changes in NintendoSDK NX Addon 0.12.14 (based on Generic 0.17.14)

Added Features

13-15S01: (TOOL) Added support for displaying NSP file overviews in AuthoringEditor.

Added a feature for displaying NSP file overviews to AuthoringEditor.
This feature makes it possible to display overview information stored in the NSP file such as the ROM hash, the application ID, and the middleware used.

Changes in NintendoWare NX Addon 0.12.14 (based on Generic 0.17.14)

Bug Fixes

13-15W01: (Effect-VFX) Fixed a bug in the VFX runtime where particles disappeared prematurely when the random lifespan feature was used.

Fixed a bug in the VFX runtime where in rare cases, particles disappeared while being emitted when the random lifespan feature was used. This bug tended to occur more often when the particle Kill() function was called in a custom process.
This bug was caused by a particle generation-related optimization process in the runtime. Fixed this bug by simply removing this optimization process because it only yielded a small performance benefit.
